# 结构化案例卡

案例卡数量：12496

## 文件

- `case-cards.jsonl`：结构化案例卡，每行一个案例。
- `case-cards.csv`：表格版案例卡。

## 字段

| 字段 | 用途 |
|---|---|
| `case_type` | 案例类型：开篇、拆书、公开处刑、榜单、作业点评等 |
| `stage` | 对应新人课程等级 |
| `module` | 应进入的系统模块 |
| `genres` / `subgenres` | 题材和子流派 |
| `topics` / `search_terms` | 检索关键词 |
| `use_for` | 可用于训练或取证的方向 |
| `evidence_snippet` | 原文前段证据片段 |

## 检索示例

```powershell
Import-Csv .\case-library\case-cards.csv | Where-Object { $_.stage -like '*L3*' -and $_.genres -like '*都市现实职业*' } | Select-Object -First 20 title,source,module
```
