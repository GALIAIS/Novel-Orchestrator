## 521：发现自己在提问的时候，每次编辑...

人工查看范围：正文、全部 8 条评论。无附件，无图片。

### 读者承诺

这篇讲“提问”本身就是一次整理：当作者为了让别人看懂而编辑问题、补足逻辑、检查遗漏时，脑子里混乱的材料会被迫归纳成结构，很多答案会在发送前自动浮出来。提问不是只等别人救场，也可以成为费曼式自我讲解。

### 核心机制

```text
脑内混乱 -> 写问题 -> 为他人解释清楚 -> 发现遗漏和逻辑断点 -> 自己推导答案 -> 必要时再发问
```

### 期待链与情绪

```text
作者想找人解惑 -> 编辑提问时反复组织语言 -> 逻辑逐渐清晰 -> 发现其实可以这么写 -> 从求助焦虑转为自我解决
```

### 关键判断

- 发帖者发现自己编辑问题时，会不停思考如何说得有逻辑。
- 这个过程常常让她在提问前恍然大悟。
- 评论指出本质是缺少总结，脑子里东西多但混乱，提问时不自觉完成归纳。
- 三更剑舞提到费曼学习法：想象老师指导自己，自己给自己讲题，常能得出答案。
- 念之秋说许多问题在发送前胎死腹中，因为写着写着自己解决了。
- 这说明“把问题写清楚”是创作复盘的低成本工具。

### 可迁移写法

提问自检表：

```text
我遇到的问题：
我现在已知的信息：
我想让别人回答什么：
我卡在哪个具体选择：
我尝试过的方案：
每个方案的利弊：
如果不发问，我现在能得出的答案：
还需要别人补充的证据：
```

### 训练任务

下次发问前，先按 8 格表写完问题。写完后暂停 10 分钟，删掉已经自己解决的问题，只保留真正需要外部经验判断的部分。

## 522：昨晚刚刚拆完了b王陈北玄的前二...

人工查看范围：正文、全部 19 条评论。无附件，无图片。

### 读者承诺

这篇拆《重生之都市修仙》前二十章，核心是相对无敌和分层打脸：当前舞台绝不让强过主角的人直接压主角，读者知道远处有更强者，但主角在眼前每一层都能赢。实力提升后马上给打脸舞台，收服的小弟和结交的势力马上回流到爽点里。

### 核心机制

```text
当前舞台分层 -> 主角相对无敌 -> 两三章铺垫仇恨 -> 一章打脸 -> 实力/势力提升 -> 下一层舞台
```

### 期待链与情绪

```text
读者知道世界有更强者 -> 但当前敌人压不过主角 -> 仇恨快速铺垫 -> 主角用更高级方式打脸 -> 爽点密集兑现 -> 继续向上一层移动
```

### 关键判断

- 发帖者认为该书风评不一定好，但剧情构建逻辑简单粗暴、干净利落。
- 相对无敌类似魔塔分层：主角不升级就遇不到比他强的人，升级后遇到的人仍没他强。
- 读者又明确知道绝对有人比主角强，因此长线仍有上升空间。
- 节奏基本是两三章铺垫仇恨，一章打脸。
- 每次实力进步后立刻安排打脸舞台，不做无用提升。
- 收服小弟、结交势力，后续打脸马上用上。
- 高级打脸不是直接揍二代，再等二代喊人，而是打脸二代父辈都惹不起的强者，侧面震慑。
- 评论认可这本是作者圈公认神书，爽点密集到公交路上也能一波一波刺激读者。
- 对“摇头”等时代痕迹，不应扣无意义小细节，要看剧情架构逻辑。

### 可迁移写法

相对无敌分层表：

```text
当前舞台：
主角当前层级：
当前敌人层级：
远处更强者：
仇恨铺垫章数：
打脸兑现章数：
实力提升：
新收资源/小弟/势力：
下一场如何立刻用上：
是否有强者提前压主角：
```

### 训练任务

设计一个 10 章相对无敌小单元：每 3 章一个仇恨，每 1 章一个打脸。主角必须始终赢当前舞台，但每个单元末尾露出下一层更强者。

## 523：已经改好了

人工查看范围：正文、1 张图片。无评论，无附件。图片已下载并人工查看：`case-library\精拆\attachments\batch-105\case-523-image-1.jpg`。

证据限制：知识库正文只有“已经改好了”，具体改稿内容主要来自图片。

图片证据：图片显示改稿片段：谢令仪听出姐姐歉疚，两姐妹没有彻底同化；李元芳召唤成功后杀掉村民，解决完后替谢令仪解绳；李元芳说“我杀了所有人，但是你替我拖延了时间，也算是救了我，所以我给你选择，要么开始新的生活，我不会让你们饿死，要么，你和他们一样，走得安详一点，还能留个全尸。”

### 读者承诺

这篇是对“女主是否圣母”的改稿反馈：原本要让女主放过并养活想吃掉她的村民，改稿后变成李元芳先完成惩罚和清算，再给幸存者选择。这样既保留女主线里可能需要的善意空间，又不让读者觉得反派无代价逃过。

### 核心机制

```text
前情毒点 -> 村民要吃女主 -> 救援者到场 -> 村民被清算 -> 姐妹保留人性 -> 给选择而非无条件宽恕
```

### 期待链与情绪

```text
读者看到女主遭遇极端伤害 -> 期待坏人付代价 -> 李元芳杀完村民 -> 情绪得到平衡 -> 对未彻底同化的人给选择 -> 善意不再像圣母
```

### 关键判断

- 图片中谢令仪能听出姐姐歉疚，说明作者保留了人物可救空间。
- 李元芳召唤成功后立即处理村民，解决读者最在意的惩罚问题。
- 村民包围李元芳却不是对手，形成爽点和安全感。
- 李元芳没有把所有人都同等宽恕，而是先杀掉构成直接恶行的人。
- 结尾给谢令仪选择，语气是恩威并施，不是无条件原谅。
- “我不会让你们饿死”保留种田经营可能，“和他们一样”保留威慑。
- 改稿方向证明：想救人可以，但必须先让坏人付代价。

### 可迁移写法

非圣母改稿表：

```text
原毒点：
谁伤害主角：
谁仍有可救理由：
惩罚执行者：
惩罚结果：
幸存者获得什么选择：
主角保留的善意：
威慑边界：
读者情绪是否平衡：
```

### 训练任务

把一个“主角放过坏人”的桥段改成恩威并施版：先写清谁必须付代价，再写主角为什么给另一部分人机会。不得无条件原谅直接伤害者。

## 524：#思考分析 #推书试读/影视/...

人工查看范围：正文、全部 4 条评论。无附件，无图片。

### 读者承诺

这篇强调“功夫在诗外”：想写出新奇故事，不能只看网文套路，还要从科幻、推理、民俗、志怪、奇幻、武侠、心理学等外部材料里积累异常素材。无脑爽文市场成熟后，差异化更多来自素材厚度和跨类型组合。

### 核心机制

```text
外部阅读 -> 异常素材积累 -> 类型知识储备 -> 与网文结构结合 -> 新奇卖点 -> 可持续题材方向
```

### 期待链与情绪

```text
作者想写新奇内容 -> 单靠网文扫榜容易同质化 -> 外部类型书提供陌生素材 -> 素材进入题材选择和设定 -> 读者看到新鲜感
```

### 关键判断

- 发帖者认为无脑爽文市场已经饱和成熟，新人难与远古大神竞争。
- 现在更需要奇思妙想，而奇思妙想来自不同寻常的素材积累。
- 《我有一座冒险屋》的作者从小看恐怖故事，所以写恐怖游刃有余。
- 推荐书目覆盖科幻《人类补完计划》、推理《死了七次的男人》、民俗《我是一个算命先生》、志怪《金棺陵兽》、奇幻《波西杰克逊与神火之盗》、金庸武侠、心理学《重口味心理学》。
- 每本推荐都强调素材功能：短篇展示长篇、时间循环推理、民俗骗术、志怪传说、现代神话化、心理疾病表现等。
- 评论虽少，但确认这些书作为同道素材库存在。
- 该帖不是要求照搬实体书，而是训练跨类型输入能力。

### 可迁移写法

外部素材输入表：

```text
素材来源：
类型：
最有用的异常设定：
可迁移情绪：
可迁移结构：
可迁移角色：
适合的网文题材：
不能直接照搬的部分：
可生成的卖点：
```

### 训练任务

从非网文书里选一个素材，填 9 格输入表。最后把它改成一个网文卖点，不许只写“这个设定很酷”。

## 525：想问一下，我写的女主被村民绑起...

人工查看范围：正文、全部 13 条评论。无附件，无图片。

### 读者承诺

这篇直接判定“主角放过要吃自己的村民”是圣母毒点，男频铁算，女频也容易被骂。主角可以种田经商、可以救人，但要先区分直接作恶者、中立者和阻止者；坏人必须付代价，剩下的人才能恩威并施。

### 核心机制

```text
村民要吃女主 -> 救援者到场 -> 直接作恶者必须惩罚 -> 中立/阻止者区分处理 -> 给硬理由救人 -> 种田经营继续
```

### 期待链与情绪

```text
读者看到女主被绑要被吃 -> 期待反杀和清算 -> 如果女主无条件养活村民会愤怒 -> 惩罚坏人后再救剩余者 -> 情绪平衡后才能接受经营线
```

### 关键判断

- 发帖者想让女主被救后不杀村民，还在村里住一段时间给他们搞吃的。
- 星主明确：在男频铁算圣母。
- 合理做法是至少把要杀自己的那一批宰了，剩下的恩威并施。
- 评论建议要么洗白大部分村民，把锅推给少数人；要么换地图。
- 北渚从读者视角追问：女主这样做读者能赞同吗？村民值得救吗？
- 主角放过反派会爆毒。
- 李元芳的人设也不支持无条件宽恕，他通常是杀光对面再救人。
- 洗白可以，但要付出代价，否则读者心理不平衡。
- 评论补充必须给女主不杀村民一个足够硬的理由，这也关乎村民为什么杀女主。
- 女频也不天然免疫圣母毒点。

### 可迁移写法

圣母毒点排查表：

```text
谁伤害主角：
伤害程度：
直接作恶者：
中立者：
阻止者：
主角为什么不杀：
这个理由够不够硬：
坏人付出什么代价：
读者情绪如何平衡：
后续经营线如何不膈应：
```

### 训练任务

写一个“主角救村子”的桥段前，先列村民三类：作恶者、中立者、阻止者。作恶者必须付代价，中立者必须受惩罚或交投名状，阻止者才可优先被救。
