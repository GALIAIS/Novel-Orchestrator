## 1156：之前有位朋友发了一个起点的投稿...


人工查看范围：正文、全部 1 条评论。无附件、无图片。

证据限制：评论只给出 `https://t.zsxq.com/2J2NBEU` 短链，知识库未展开链接正文，无法读取起点投稿大纲模板内容；拆解重点是“投稿模板作为提交前约束，而非正文创作替代”。

### 读者承诺

这篇沉淀的是投稿大纲模板意识：新人找起点投稿大纲模板，本质是在寻找编辑能快速判断项目的结构化信息入口。模板不是神秘资料，而是把题材、卖点、主线、人物、金手指、前期事件和后续规划说清楚。

### 核心机制

```text
准备投稿 -> 找大纲模板 -> 用固定栏目梳理项目 -> 降低编辑理解成本 -> 暴露缺口
```

作者询问谁收藏了起点投稿大纲模板，评论给出星球短链。虽然模板正文不可见，但这个需求本身说明，投稿前作者需要把零散故事想法整理成编辑可评估的结构，而不是只凭正文片段和口头解释。

### 期待链与情绪

```text
作者准备投稿 -> 不知道大纲怎么写 -> 找模板 -> 填栏目 -> 发现主线/卖点/后续是否清楚
```

大纲模板的价值不只是“提交格式”，更是一次项目体检：填不出来的栏目，就是作品当前还没想清的部分。

### 关键判断

- 投稿大纲要服务编辑快速判断。
- 模板栏目能逼作者暴露主线和卖点缺口。
- 找模板不能替代作品本身规划。
- 题材、核心爽点、人物动力、金手指、前期事件都应可概括。
- 短链不可读时不能假装知道模板细节。
- 投稿前应先用模板自检，而不是临时凑文案。

### 可迁移写法

```text
投稿平台：
题材/流派：
一句话卖点：
主角目标：
金手指/核心机制：
前三章事件：
前十万字大纲：
后续长期承诺：
```

### 训练任务

用一页纸写投稿大纲自检表：一句话卖点、主角目标、金手指机制、前三章事件、前十万字主线、后续三十万字承诺。任一栏写不出，就先补项目设计再投稿。

## 1157：请问宝剑怎么看赛博朋克这个题材...


人工查看范围：正文、全部 1 条评论。无附件、无图片。

证据限制：这是题材热度问答短帖，没有展开赛博朋克写法或代表作品；拆解重点是“标签热度与成熟题材之间的风险差”。

### 读者承诺

这篇沉淀的是新兴题材判断：赛博朋克标签热度高、作品少、代表作少，说明它可能处在刚起来但尚未成熟的阶段。热度不等于稳定市场，代表作少意味着可参考模板少、试错成本高。

### 核心机制

```text
标签热度高 -> 代表作品少 -> 题材未成熟 -> 模板不足 -> 写作风险上升
```

作者问赛博朋克题材怎么看：标签热度高，但作品少，没有代表作品。评论回应宝剑前两天说过，这题材最近才起来，还不算很火，所以代表作品自然少。这里的关键不是能不能写，而是不能把“热度高”误判成“已经有成熟读者承诺和套路库”。

### 期待链与情绪

```text
作者看到热标签 -> 以为风口明确 -> 查不到代表作 -> 不确定怎么写 -> 判断题材成熟度
```

新兴题材适合有强执行力和自造模板能力的作者；新人若缺少可拆样本，容易只写表层美术风格，而不是稳定剧情承诺。

### 关键判断

- 标签热度和题材成熟度不是一回事。
- 代表作品少，说明套路库不足。
- 最近起来的题材不等于已经很火。
- 新题材要评估自己能否自造结构。
- 没代表作时要找邻近类型借承诺。
- 不要只写霓虹、义体、公司压迫等皮相。

### 可迁移写法

```text
题材标签：
热度来源：
代表作品数量：
可拆模板：
邻近类型：
核心读者承诺：
新题材风险：
是否适合新人：
```

### 训练任务

评估一个新兴题材：找 5 本可参考作品，若找不到，就列 3 个邻近成熟类型，并写出你要借用的主线承诺、爽点机制和前三章结构。

## 1158：宝剑大大，能不能仔细说一下那个...


人工查看范围：正文、全部 1 条评论。无附件、无图片。

证据限制：这是读者反馈定义问答，没有具体后台数据或评论案例；拆解重点是反馈范围和评论分流。

### 读者承诺

这篇沉淀的是读者反馈总线：反馈不只是完读率，也不只是评论，而是一切读者与作品发生的互动，包括点击、推荐、月票、订阅、完读率、评论等。评论处理要分流：无脑找茬删封，真正错误正经回应，能改就改，改不了就放下。

### 核心机制

```text
读者互动 -> 汇总成反馈 -> 区分数据/评论 -> 评论分流处理 -> 作品修正或维持
```

作者问读者反馈是否主要指完读率，是否包含评论，以及评论怎么处理。回答明确：所有读者和书的互动都叫反馈。评论方面，不必把每条都当圣旨，也不能全部无视；要按质量和可修正性处理。

### 期待链与情绪

```text
作者看到数据/评论 -> 不知道该信哪个 -> 扩大反馈定义 -> 分流评论 -> 对真问题修改
```

反馈系统的核心是“别单点迷信”。完读率重要，但点击、订阅、推荐、月票和评论共同说明读者是否愿意继续互动。

### 关键判断

- 完读率只是反馈之一。
- 点击、推荐、月票、订阅、评论都算反馈。
- 评论要分无脑找茬、真实错误、正常互动。
- 无脑找茬可删封，不必消耗作者心态。
- 真错误要正经回应，能改就改。
- 改不了的反馈要记录边界，不无限纠缠。

### 可迁移写法

```text
反馈来源：
数据指标：
评论类型：
是否真实错误：
是否能修改：
处理动作：
修改位置：
后续观察指标：
```

### 训练任务

把一周反馈分成四类：数据异常、真实错误、正常互动、无脑找茬。每类写一个处理动作，只对真实错误和数据异常安排文本修改。

## 1159：宝剑大大，现在我准备从最基础的...


人工查看范围：正文、全部 2 条评论。无附件、无图片。

证据限制：没有作者作品样本，只能依据作者自述和评论经验拆解“从辞藻叙述转向结构法则”的训练阶段。

### 读者承诺

这篇沉淀的是新人写作思维重塑：从只关注当前段落叙述、华丽辞藻，转向整体结构、法则运用和详细大纲能力。这个阶段痛苦，但它是从散写转向可控长篇的必要过程。

### 核心机制

```text
重视辞藻/段落 -> 发现长篇失控 -> 学结构与法则 -> 能写详细大纲 -> 再回到整本创作
```

作者意识到过去过分注重段落叙述和华丽辞藻，忽略整体结构和法则，准备从基础写作思维开始练，甚至发现自己连详细大纲都写不出。评论回应：打破重组开始都痛苦，前两本没想过结构，学理论再上路也会很难，但熬过去会变好。

### 期待链与情绪

```text
新人写局部文字 -> 长篇无法成型 -> 痛苦重学结构 -> 大纲能力提升 -> 整本小说更可控
```

文笔不是没用，但对网文长篇来说，结构能力决定能不能持续制造期待和兑现。

### 关键判断

- 华丽辞藻不能替代整体结构。
- 段落叙述好，不等于长篇能跑。
- 写不出详细大纲，是结构能力缺口。
- 学理论后的重组期会痛苦。
- 练基础不等于不写，而是分步练结构、事件和大纲。
- 熬过重组期后，创作会更可控。

### 可迁移写法

```text
原有强项：
当前缺口：
结构训练项：
法则训练项：
大纲训练项：
短练习：
整本开写条件：
复盘周期：
```

### 训练任务

用 7 天做结构重塑练习：每天只设计 1 个事件，写清目标、阻碍、选择、代价、结果、下一期待。第 7 天把 7 个事件串成一份小大纲，不写正文辞藻。

## 1160：经常看到有大佬说啥基本功，新人...


人工查看范围：正文、全部 5 条评论。无附件、无图片。

证据限制：评论明确“基本功没有固定标准”，因此不能把它整理成唯一权威定义；拆解重点是可教学的“术”和不可直接教的自悟边界。

### 读者承诺

这篇沉淀的是网文基本功拆分：所谓基本功没有统一标准，不同人指代不同，但大体可分为可教学的“术”和更依赖自悟的部分。可练的基础包括剧情设计、画面组织、文字表达、故事设计能力、人物刻画、情感渲染、场景刻画等。

### 核心机制

```text
基本功概念模糊 -> 拆成可训练能力 -> 剧情/画面/文字/故事设计 -> 逐项练习
```

作者困惑基本功到底是什么。评论给出一种拆法：剧情是知道接下来写什么；画面包括打斗、对话、场景；文字是准确把画面写出来。死翼耐萨里奥则说基本功没有固定标准，本质多是“术”上的东西，是可以通过教学学习的；基本功之上的东西更靠自己悟。另有评论概括为故事设计能力和文字表达能力。

### 期待链与情绪

```text
新人听到基本功 -> 觉得是默认概念 -> 发现没有统一标准 -> 拆成训练项 -> 不再空泛焦虑
```

“练基本功”如果不拆项，就会变成空话。新人需要把它变成可执行训练，而不是追问一个统一定义。

### 关键判断

- 基本功没有固定标准。
- 不同作者说的基本功可能不是一回事。
- 可教学部分多属于“术”。
- 剧情、画面、文字是一个实用拆法。
- 故事设计和文字表达是更高层概括。
- 自悟部分不能完全靠教程替代。

### 可迁移写法

```text
当前短板：
剧情设计：
画面组织：
文字表达：
人物刻画：
情感渲染：
场景刻画：
本周训练项：
```

### 训练任务

做一个基本功周练表：每天写一个 300 字片段，分别练打斗画面、对话画面、场景画面、人物刻画、情感渲染、剧情转折、文字压缩。每篇只检查一个能力，不同时追求全能。
