## 016：【一级结构-进阶】从情绪阐发单元故事剧情和核心梗

人工查看范围：文件全文。

证据限制：该 Markdown 只有收藏链接，没有保留正文和评论。本卡只记录它在体系里的位置：一级结构从基础起承转合进入“情绪阐发单元故事和核心梗”的进阶方向。

### 读者承诺

它指向的问题是：学会起承转合后，还要明白每个单元为什么能产生情绪，以及单元故事如何承载核心梗。基础结构回答“怎么排”，进阶结构回答“为什么爽、为什么能复用”。

### 核心机制

```text
单元事件 -> 情绪目标 -> 核心梗抽象 -> 情绪链复用 -> 下一个单元变体
```

### 可迁移写法

在每个单元故事卡旁边加一列“情绪阐发”：

```text
本单元表层事件：
读者期待什么：
情绪缺口来自哪里：
主角如何释放：
该单元背后的核心梗：
下次换壳时保留什么：
```

### 训练任务

选一个已写单元，不改剧情，只补写“情绪目标”和“核心梗抽象”。如果写不出来，说明这个单元只是事件堆叠，还没进入可复用结构。

## 017：【选题拆书怎么做】

人工查看范围：正文可见摘要、全部评论中智能推、标签、同人选站、起点变化、同水平拆书和基础训练讨论。

### 读者承诺

这篇解决新人“到底写什么题材容易赚钱、怎么扫榜找对标”的问题。它把选题拆书从只看榜单，转向“自身积累 + 平台智能推荐 + 标签识别 + 同类样本反向定位”。

### 核心机制

```text
自身阅读积累 -> 平台流量机制 -> 智能推/标签反查 -> 同类低榜样本 -> 判断可写题材 -> 拆开篇与留存点
```

### 关键判断

- 新人主要矛盾是积累，不是热点词不够。
- 智能推提高了流量分发效率，让很多老题材、小题材也能找到读者。
- 书名、简介、正文内容都会影响平台判断你的标签和读者池。
- 不只看榜首，也要看智能推里不在排行榜但同类型、可参考的书。
- 拆书要认清自身水平。差距太大的神书不一定适合新人硬学，同水平作品更容易看到可复用解法。

### 可迁移写法

选题拆书表：

```text
我有长期阅读积累的题材：
平台当前能推送到的同类标签：
榜单样本：
智能推样本：
同水平样本：
前三章共同承诺：
我能写长的素材库存：
不能碰的高难点：
```

### 训练任务

用一个自己熟悉的题材，找 3 本榜单样本和 3 本智能推样本。不要拆全书，只拆书名、简介、第一章承诺、前三章留存点。

## 018：笔记-宝大【矛盾和冲突】

人工查看范围：笔记全文。

### 读者承诺

这篇解决“长篇剧情为什么会换皮、乏味、可预见”的问题。长篇不是不断换敌人，而是要有主次矛盾、矛盾网、矛盾升级和特殊解决方式。

### 核心机制

```text
主要矛盾主导当前卷 -> 次要矛盾暗中升级 -> 解决主要矛盾 -> 次要矛盾自然接位 -> 新矛盾从旧矛盾中生长
```

### 冲突六要素

- 冲突必须和主角相关，最好冲着主角来。
- 冲突要有赌注、风险、强对手、渴望目标。
- 冲突可升级，但冲突对象变化时，敌对势力或矛盾链条要保持完整。
- 升级要符合逻辑，堵死后路，让主角不能轻易退出。
- 解决方式要先爽，再谈意义，三观正确是底线。
- 新颖不是新人第一优先级，稳和情绪拉扯更重要。

### 可迁移写法

卷纲矛盾网：

```text
当前主要矛盾：
主角为什么无法退：
赌注：
风险：
次要矛盾 1：
次要矛盾 2：
解决当前矛盾后，哪个次要矛盾接位：
敌对链条如何保持：
```

### 训练任务

给一个 10 万字卷纲列出 1 个主要矛盾和 3 个次要矛盾。每个次要矛盾都写清楚它何时升级、为什么不是突然跳脸。

## 019：讲一下对于金手指的分析

人工查看范围：正文完整阅读、评论中认可和系统适配补充。

### 读者承诺

这篇解决“金手指为什么有的能持续产剧情，有的只是摆设”的问题。它把金手指定义成两件事：让主角在一定范围内与众不同，并帮助主角克服困难、获得收益、走向成功。

### 核心机制

```text
金手指唯一性 -> 背景设定给出空子 -> 主角钻空子获得收益 -> 收益解决当前矛盾 -> 主角特别感增强
```

### 关键判断

- 唯一性不是绝对唯一，而是在当前范围内不能有人和主角同赛道一样快甚至更快。
- 金手指必须带来收益，不能克服困难并获得收益的困难可以删掉。
- 背景设定和金手指必须高度相关。
- 金手指决定背景中必须存在可钻的空子，背景也决定金手指的功能方向。
- 不要添加严防死守金手指发挥的设定，除非你准备换金手指或换背景。

### 可迁移写法

金手指适配卡：

```text
主角在哪个赛道特别：
谁能证明他特别：
世界有什么空子：
金手指如何钻这个空子：
钻空子得到什么收益：
收益能解决哪些常见矛盾：
哪些设定会压死金手指：
```

### 训练任务

拿自己的金手指做一次“背景空子测试”。写出 3 个该世界常见矛盾，并说明金手指收益如何自然解决它们。

## 020：从“听劝系统”再谈网文的微创新与金手指分析逻辑

人工查看范围：正文可见摘要、评论中听劝/不听劝脑洞、写长难度、反套路边界讨论。

证据限制：知识库文件只保留正文开头摘要和评论，未保留完整正文。本卡基于可见证据精拆。

### 读者承诺

这篇解决“怎么做微创新而不是瞎反套路”的问题。可见正文提出三个方向：角色标签反套路、大众化金手指反套路、核心内容另辟蹊径。

### 核心机制

```text
大众熟悉套路 -> 找一个可反转点 -> 保持读者价值观和逻辑 -> 形成新期待 -> 检查能否写长
```

评论里最关键的提醒是：一个脑洞成立不等于能写百万字。听劝系统可以推演、也可以“不听劝”打破常识，但必须追问主线如何持续、期待感是否足够强、读者还能不能继续追。

### 可迁移写法

微创新检查表：

```text
原套路是什么：
反转的是角色标签、金手指，还是核心内容：
反转后是否符合读者价值观：
反转后能制造什么持续矛盾：
能否写 3 个连续单元：
百万字主线靠什么延展：
```

### 训练任务

给一个常见金手指做微创新，必须写出 3 个连续单元。如果只能写一个高概念开头，就先不要开书。
