## 796：#参考资料 妖怪百科书-纪妖

人工查看范围：正文、全部 6 条评论、正文外链 `http://www.cbaigui.com/` 的当前首页。无附件、无图片。

证据限制：正文中的星球内《山海经》搜索链接未展开；纪妖官网当前可见首页包含“妖怪、杂谈、讨论、图绘、公告”等入口，并能看到妖怪词典与杂谈条目，但未逐条检索所有妖怪资料，因此不能证明资料库完整性。

### 读者承诺

这篇是资料工具推荐：给怪异文、玄幻仙侠、民俗奇谈、妖怪题材作者提供一个妖怪素材入口。它承诺的不是直接可用剧情，而是降低查找妖怪设定、出处和文化来源的成本。

### 核心机制

```text
发现垂直资料库 -> 按文化来源查妖怪 -> 按作品出处反查 -> 类比历史文工具 -> 写作素材入库
```

正文强调纪妖可以查中国、日本、西方妖怪，也能按《山海经》等作品出处反查。评论反馈“写一些玄幻仙侠题材就有不错的素材”，说明它的价值在于素材收集，而不是替作者完成设定转化。

### 期待链与情绪

```text
作者缺怪异素材 -> 发现专门工具 -> 查分类/出处 -> 产生题材联想 -> 保存入口 -> 后续转成故事材料
```

情绪上是“资料宝库”的兴奋感。风险在于新人只收藏工具，不把素材消化成可写的冲突、禁忌、能力、代价和人物压力。

### 关键判断

- 垂直工具适合降低查资料成本。
- 妖怪资料要按文化来源、出处、能力和禁忌二次整理。
- 怪异素材不能直接等于剧情。
- 玄幻仙侠、民俗悬疑、怪谈文都可从中取材。
- 工具推荐帖要记录入口、用途和适用题材。
- 外部站点应标注访问状态和未来失效风险。

### 可迁移写法

怪异素材入库表：

```text
妖怪名：
文化来源：
原始出处：
外形特征：
能力/习性：
禁忌/弱点：
可转化冲突：
适合题材：
可做开篇钩子的异常：
资料链接：
```

### 训练任务

从纪妖或其他妖怪资料库选 3 个妖怪，每个只保留 5 条最有用设定，并各写一个 300 字开篇异常：必须包含目击者、异常现象、危险代价和主角介入理由。

## 797：分享一下我最近拆的一本书（万订...

人工查看范围：正文、全部 3 条评论。无图片。

证据限制：原帖有 1 个附件 `file_841588881218212`，但 Markdown 只保留附件 id，本地导出目录没有对应 `meta` 或 `download_url` 文件，因此未能查看附件正文。本拆解只依据可见拆书心得与评论校正。

### 读者承诺

这篇承诺通过一部万订样本的第二卷拆解，提炼卷纲事件、反派压迫、不公平伤害、爽感强弱和拉期待手法之间的关系，帮助作者从“我觉得爽”转向“我知道爽感变量在哪里”。

### 核心机制

```text
人工分卷 -> 选取三个事件 -> 比较伤害对象/施害者/解决方式 -> 判断爽感强弱 -> 用评论校正模型
```

作者认为第一个事件最爽，因为主角和读者代入阵营同时遭受同一反派的不公平伤害，并由主角正面对决、暴力解决。宝剑补充关键校正：爽感高低更多取决于情绪煽动和拉期待是否到位，暴力手段容错率高，但不是唯一答案。

### 期待链与情绪

```text
诡异克系修仙氛围铺底 -> 第二卷事件展开 -> 不公平伤害归因到反派 -> 拉起读者愤怒 -> 主角解决冲突 -> 追问下一次升级
```

情绪核心是“归因清楚的愤怒”和“宣泄”。如果伤害主角和伤害读者代入阵营的反派分散，情绪会被拆开；如果解决方式不是正面对抗，也需要更强的期待铺垫来补足宣泄感。

### 关键判断

- 反派实力升级不等于爽感必然升级。
- 不公平伤害是制造爽感的重要燃料。
- 同一反派同时压迫主角和读者代入阵营，爽感更容易集中。
- 暴力毁灭反派是高容错宣泄方式，但不是爽感唯一来源。
- 智谋、取巧、破阵也能爽，前提是情绪煽动和期待拉得足。
- 高阶人脉可以作为底牌，用来挡住敌方王炸。

### 可迁移写法

卷纲爽感对比表：

```text
事件编号：
反派是谁：
谁遭受不公平伤害：
主角是否直接受害：
读者代入阵营是否受害：
伤害是否来自同一反派：
主角如何解决：
拉期待手段：
情绪煽动强度：
爽感兑现方式：
```

### 训练任务

选一卷 3 个连续事件，按上表拆出爽感变量。再把最弱事件重写成两版：一版增强不公平伤害归因，一版增强期待铺垫，比较哪一版更适合当前卷纲。

## 798：整个上午居然没人说话😂那我来问...

人工查看范围：正文、全部 8 条评论。无附件、无图片。

证据限制：评论中提到《后天》《2012》《流浪地球》《行尸走肉》等作品链接作为类型参照，当前拆解只依据评论中的类型归纳，不展开这些作品页面。

### 读者承诺

这篇回答“末世文从末世降临写，还是从秩序稳定后写”。核心承诺是：开篇切入点必须服务后续类型体验；读者要看的如果是末世，就要有降临、崩坏、倒计时、资源压力和持续紧张。

### 核心机制

```text
确定后续主线 -> 选择开篇时间点 -> 兑现类型情绪 -> 主角在危机中崭露头角 -> 秩序演化
```

评论分歧表面是“降临前还是降临后”，实际是“你要写什么类型”。如果世界已经稳定，读者感知会更像异界、高武、种田或争霸；如果写末日降临，则要保留灾难片式压迫感和倒计时。

### 期待链与情绪

```text
末世降临 -> 主角在混乱中求生/升级 -> 团队形成 -> 势力范围出现 -> 新秩序建立
```

情绪上先是恐惧和紧迫，再是生存优势，再是团队与秩序扩张。若开篇直接进入稳定新世界，压迫感会下降，读者期待也会转向新秩序冒险而非末世崩坏。

### 关键判断

- 开头不能脱离后续主线单独选择。
- 末世文的看点包括崩坏、倒计时、资源争夺和矛盾紧张。
- 秩序已稳定的开局容易滑向异界、高武、种田或争霸。
- 主角从个人生存到团队势力是自然升级路径。
- 类型基调要在开篇定好。
- “从哪里开始”本质是“读者第一口吃到什么体验”。

### 可迁移写法

末世开篇切入表：

```text
后续主线是生存/升级/种田/争霸：
第一章世界状态：
是否有倒计时：
资源压力是什么：
主角第一场危机：
主角优势如何展示：
秩序崩坏到什么程度：
三卷后是否进入新秩序：
```

### 训练任务

同一个末世设定写两个 500 字开头：一版从降临当天写，一版从秩序稳定后写。分别标出读者会期待的后续类型，并删除与该类型承诺冲突的开篇信息。

## 799：大家有什么神兵利器出世的描写片...

人工查看范围：正文、全部 4 条评论。无附件、无图片。

### 读者承诺

这篇解决“神兵利器出世卡壳”：神器不靠一段华丽外形描写撑场，而靠事前造势、旁人反应、争夺代价和实战效果，让读者先相信它厉害，再看它如何兑现。

### 核心机制

```text
传闻/名号造势 -> 众人争夺/畏惧 -> 读者预设价值 -> 神器出世 -> 正面效果兑现
```

评论给出两类写法：一是物品加特效、天气变化，像写人一样写气势；二是参考屠龙刀、倚天剑，先用江湖传闻和无数人争夺推高期待，出世时外观普通也能成立。

### 期待链与情绪

```text
听说神兵很强 -> 看到众人争夺/避让 -> 读者默认其价值 -> 出世瞬间期待拉满 -> 破甲/连斩/威慑兑现
```

情绪重点不是“光很亮”，而是“这东西一出现局势就变”。侧面描写负责抬价，正面描写负责兑现。

### 关键判断

- 强物写法可以借鉴强人写法。
- 先写地位和影响，再写外形。
- 外形普通不影响神器价值，前提是造势足够。
- 旁人避让、巴结、抢夺都是侧面抬价。
- 破甲、连斩、威慑等实战效果负责正面兑现。
- 天象特效只能辅助，不能替代价值铺垫。

### 可迁移写法

神兵出世造势表：

```text
神兵名号：
传闻一句话：
谁想抢：
谁害怕：
曾造成什么战绩：
出世前异常：
外观是否反差：
第一次实战效果：
局势如何改变：
```

### 训练任务

设计一件神兵，不写外形，先写 600 字侧面造势：至少包含一个传闻、两方争夺、一名旁观者反应。再用 300 字写第一次实战兑现。

## 800：扫榜的时候发现自己想写幻想类题...

人工查看范围：正文、全部 8 条评论。无附件、无图片。

### 读者承诺

这篇处理“想写的题材”和“爱看的题材”分离：作者想写仙侠修真等幻想类，却不太想看；喜欢看种田文，却没有创作欲。评论给出的方向是区分阅读偏好、创作欲、擅长和职业选择。

### 核心机制

```text
发现偏好撕裂 -> 区分喜欢看/想写/擅长/能赚钱 -> 硬读目标类型 -> 学优点 -> 避毒点 -> 决定是否职业化投入
```

宝剑的判断很明确：这种阻力能分出玩票和职业。想写什么，就得硬着头皮看这个类型的书，学优点、避毒点。也就是说，创作方向不能只靠兴趣冲动，还要补齐目标类型的读者期待。

### 期待链与情绪

```text
扫榜发现想写方向 -> 发现自己不爱看 -> 产生撕裂感 -> 判断是否愿意投入 -> 强制输入同类样本 -> 建立类型认知
```

情绪关键词是“撕裂感”。喜欢消费某类内容，不等于愿意生产；觉得某类设定很酷，也不等于掌握该类节奏和毒点。职业化写作要求作者为目标题材补足输入。

### 关键判断

- 喜欢看和擅长写是两回事。
- 想写某类题材，必须读该类样本。
- 扫榜不是找感觉，而是补读者期待。
- 创作欲可以来自设定酷感，但不能停在酷感。
- 职业化写作要跨过“不想看但要学”的阻力。
- 如果长期无法输入目标题材，应重新评估方向。

### 可迁移写法

题材偏好校准表：

```text
我喜欢看的题材：
我想写的题材：
我已经熟悉的同类样本：
我抗拒阅读的原因：
该题材读者期待：
我能学的优点：
我必须避开的毒点：
我是否愿意读 20 本同类：
最终选择：
```

### 训练任务

选一个“想写但不爱看”的题材，连续扫 10 本同类书的前三章。每本只记录 3 项：开篇承诺、主角优势、明显毒点。10 本后再决定是否继续写这个方向。
