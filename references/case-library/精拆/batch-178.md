## 886：我最近上架，高订两千，均订在推...


人工查看范围：正文、全部 2 条评论。无附件、无图片。

证据限制：只有作者自报高订、均订和时间变化，没有书名、题材、推荐位类型、章节追订曲线和具体正文，不能定位具体毒点章节。

### 读者承诺

这篇是上架后留存诊断：作者高订两千，推荐上均订一千六，一周后降到一千一，情绪上很难受。真正要拆的不是“下降是否正常”，而是均订下滑背后的追订、节奏、毒点和新爽点断档。

### 核心机制

```text
推荐曝光 -> 均订上升 -> 推荐后自然回落 -> 检查追订曲线 -> 定位拖点/毒点 -> 抛新爽点补位
```

宝剑老师给出的诊断路径很实用：如果没有大规模爆更，先看追订是不是降得快；如果是，就在下降最多的章节附近检查情节是否拖了、有没有毒点，同时尽快抛出后面的新爽点。作者也承认最近确实写得有点拖。

### 期待链与情绪

```text
读者上架付费 -> 期待持续推进 -> 章节节奏拖慢 -> 追订先掉 -> 均订后掉 -> 新爽点重新拉回期待
```

连载的关键是让读者每章都有“继续付费”的理由。旧冲突结束后，如果新目标、新奖励、新矛盾迟迟不出现，追订会比均订更早发出警报。

### 关键判断

- 均订下降要先拆追订，不要只看总数。
- 推荐后回落正常，但断崖式追订要查章节。
- 情节拖沓会直接伤付费留存。
- 毒点通常集中体现在某几章掉订。
- 新爽点要尽快抛出，不能让读者等太久。
- 上架后诊断要围绕付费章节曲线，而不是作者体感。

### 可迁移写法

追订下滑诊断表：

```text
上架时间：
推荐期均订：
当前均订：
追订最低章节：
下降前剧情：
是否拖沓：
是否有毒点：
旧爽点是否结束：
新爽点何时抛出：
补救方案：
```

### 训练任务

拿自己最近 20 章付费章节，标出追订下降最大的 3 章。逐章写出当章的目标、冲突、爽点、毒点风险和下一章钩子，再补一个新爽点投放计划。

## 887：星球里稿费月万的铁汁有多少？有...


人工查看范围：正文、全部 7 条评论。无附件、无图片。

证据限制：这是收入层级讨论，不是单书复盘；评论提到刺猬猫、榜单前几、第十名五万左右、订阅八九万等经验信息，但没有完整榜单、渠道收入和时间跨度。

### 读者承诺

这篇用于理解平台型二次元/轻小说站的收入结构：它不承诺无限天花板，而是服务明确付费读者后，中层作者相对更容易吃饱饭；月十万集中在头部，月万作者往往已经有自己的方法论。

### 核心机制

```text
平台读者精准 -> 题材适配 -> 稳定订阅 -> 榜单分层 -> 中层可生存 -> 头部冲高收入
```

宝剑老师的判断是，月万能做到的基本都有自己的方法论；月十万应该有，刺猬猫拉来的亲友团能到这个数，但集中在榜单前几。没有渠道收入导致总天花板不如主站，不过对中层来说相对容易吃饱饭。

### 期待链与情绪

```text
作者关心平台收入 -> 发现头部集中 -> 认清中层生存空间 -> 回到题材和订阅转化 -> 建立现实预期
```

新人容易只看“有没有月十万”，但更实际的问题是：平台是否适合自己的题材，能不能稳定吃到中层订阅，自己的方法论是否能支撑月万。

### 关键判断

- 月万作者通常已有稳定方法论。
- 月十万集中在榜单头部，平台内部分层很大。
- 无渠道收入会限制天花板。
- 小站/垂直站中层生存可能比主站更容易。
- 收入讨论必须回到平台读者、题材适配和订阅转化。
- 单条评论不能当当前收入事实，投稿前仍需重新核验榜单。

### 可迁移写法

平台收入预期表：

```text
目标平台：
核心读者：
主流题材：
榜单头部收入：
中层收入：
渠道收入：
我的题材适配：
订阅转化风险：
现实目标：
```

### 训练任务

选一个目标平台，记录榜单前 20 的题材、更新量、同人/原创比例和可见收入信息。把自己的目标先定为“中层可持续”，再判断是否有冲头部的条件。

## 888：下个视频想聊聊新人如何选题材、...


人工查看范围：正文、全部 10 条评论。无附件、无图片。

证据限制：这是视频/专栏预告，不是完整课程正文；只能提炼新人需求、课程结构和训练路径，不能还原具体选题步骤。

### 读者承诺

这篇指向新人开书最前置的问题：如何选题材、找亮点、发掘卖点。宝剑老师计划先用浅层视频建立概念，再根据反馈做星球专栏文字深入发掘，并配合作业实践。

### 核心机制

```text
新人准备开书 -> 不会选题材/亮点/卖点 -> 先建立概念 -> 专栏深化 -> 作业实践 -> 校准开书方向
```

评论区集中表达“需要、期待、想等视频再开书”。这说明新人卡点不只是写作技术，而是开书前就不知道自己卖什么、亮点在哪里、题材是否适合。

### 期待链与情绪

```text
新人想开新书 -> 害怕方向错 -> 等待选题方法 -> 用思路校准题材 -> 作业验证卖点 -> 再正式开书
```

对新人来说，选题阶段如果错了，后面大纲、开篇和更新都会被拖累。先建立概念，再通过作业把亮点写出来，比盲目开书更稳。

### 关键判断

- 新人选题是高需求入口模块。
- 题材、亮点、卖点要在开书前一起判断。
- 第一层先建立概念，不必一开始就复杂化。
- 深入方法要配合作业，否则只是听懂。
- 评论区反馈可用来决定专栏深化方向。
- 开书前等方法可以，但不能无限拖延实践。

### 可迁移写法

新人选题卖点表：

```text
候选题材：
目标读者：
读者已有期待：
我的亮点：
一句话卖点：
前三章兑现方式：
同题材标杆：
差异点：
作业验证结果：
是否开书：
```

### 训练任务

为 3 个候选题材各写一句话卖点、一个前三章兑现点和一个差异化亮点。发给读者或社群看反馈，只保留反馈最清楚的一个继续做大纲。

## 889：这几天在研究浴血黑帮，发现真和...


人工查看范围：正文、全部 4 条评论、图片 1 张。图片已保存并查看：`<EVIDENCE_CACHE>\batch-178\889-image-418882512428258.jpg`。无附件。

证据限制：图片只截取了《浴血黑帮》三幕式拆解的一部分，可见冲突爆发、情绪升级、小高潮、大高潮节点；未包含完整拆解图和全部剧情线。

### 读者承诺

这篇把复杂多线叙事拆回基础结构：表面上《浴血黑帮》多线复杂，但拆开后仍是普通三幕式。进一步的问题是长篇如何避免《斗破》后期那种剧情重叠。

### 核心机制

```text
复杂多线 -> 拆成三幕式 -> 长篇保持基调 -> 核心模型重复 -> 同路不同级 -> 降低疲劳
```

宝剑老师的判断很现实：一般避免不了。要保持原有基调，就肯定会重叠重复；玩新花样跳出基调，读者往往一走了之，只能两害取其轻。评论补充的解法是“同路不同级”：30 级打血色修道院，60 级打斯坦索姆，70 级打伊利丹，90 级打巫妖王，本质不变，但等级和规模升级。

### 期待链与情绪

```text
读者看见复杂作品 -> 拆解发现基础结构 -> 长篇继续维持熟悉爽点 -> 敌人/副本/等级升级 -> 读者接受重复中的变化
```

长篇网文不是完全避免重复，而是在重复中保持承诺。读者来这里就是要看某种核心循环，问题是每轮能不能抬高等级、扩大代价、换更强敌人、给新奖励。

### 关键判断

- 复杂多线也可以拆回三幕式。
- 长篇保持基调必然带来结构重复。
- 跳出基调可能比重复更伤读者。
- 同路不同级是升级流降低疲劳的常见办法。
- 后期重复无法消灭，只能用规模、等级、敌人和奖励变化缓冲。
- 拆剧集时先抓冲突爆发、情绪升级、小高潮、大高潮。

### 可迁移写法

同路不同级升级表：

```text
核心循环：
第一轮敌人/副本：
第二轮敌人/副本：
第三轮敌人/副本：
等级差异：
奖励差异：
情绪差异：
新增代价：
保持不变的读者承诺：
```

### 训练任务

拿一部长篇升级文，列出 5 个相似大事件。分别标注敌人等级、场景规模、奖励、主角代价和情绪差异，判断它是有效“同路不同级”还是无效重复。

## 890：#参考资料 国家宝藏，名言名句...


人工查看范围：正文、图片 2 张。图片已保存并查看：`<EVIDENCE_CACHE>\batch-178\890-image-418882512224488.jpg`、`<EVIDENCE_CACHE>\batch-178\890-image-818884214445582.jpg`。无评论、无附件。

证据限制：图片为《国家宝藏》名言名句截图，含大量节目台词和文物/历史表达；本次只做素材类型归纳，不逐条转录全部句子。

### 读者承诺

这篇是国风历史表达素材库入口。《国家宝藏》的名言名句适合为历史、文物、国风、文化传承类作品提供句式、意象、情绪和标题灵感。

### 核心机制

```text
文化节目语料 -> 提取句式/意象/情绪 -> 匹配历史国风场景 -> 改写成台词/标题/卷首语 -> 提升文化质感
```

图片里可见高频表达包括守护历史、华夏文明、文字和历史、上下五千年、礼乐、文物生命轨迹、时代镜子等。它们不是直接复制到小说里，而是用来学习文化类表达如何把宏大历史、个人情绪和具体器物连接起来。

### 期待链与情绪

```text
作者需要文化感 -> 找到高密度语料 -> 学习句式和意象 -> 改写到具体人物/文物/时代 -> 读者感到厚重与共鸣
```

国风表达最怕空喊大词。有效用法是把“国家、历史、文明”落到一个器物、一个人物选择、一句台词或一个卷首主题上。

### 关键判断

- 《国家宝藏》适合作为文化表达素材库。
- 素材要按句式、意象、适用场景整理。
- 名言不能生搬硬套，必须改写进剧情语境。
- 宏大表达要落到具体器物和人物。
- 可用于标题、卷首语、角色台词和宣传文案。
- 文化素材入口仍需后续分类归档，不能只截图收藏。

文化金句转化表：

```text

核心意象：
情绪类型：
适用场景：
可对应文物/历史事件：
可对应人物：

使用位置：
```
