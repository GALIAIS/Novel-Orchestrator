## 1216：#思考分析 交作业


人工查看范围：正文、全部 1 条评论。无附件、无图片。

证据限制：这是五折线作业梗概，不是完整校园格斗章节；拆解重点是简单结构如何符合标准。

### 读者承诺

这篇沉淀的是五折线作业可以很简单，只要下行、上行、虚晃、再下行和最终打脸的功能清楚，就能符合训练标准。主角有系统加持格斗技巧，死党惹上校外混混，最后主角当众打败混混，完成保护朋友和围观震惊。

### 核心机制

```text
死党被混混堵人 -> 主角因金手指有信心 -> 死党劝逃制造不信任 -> 小路虚晃 -> 被混混堵住 -> 主角出手打脸
```

正文中，死党先认为主角疯了，强调混混强大并提出小路逃跑；路上遇到抽烟学生是虚晃；即将逃出时被混混堵住，死党下跪求饶，主角拉起死党并用格斗技巧击败混混。宝剑评价“简单却都符合标准”。

### 期待链与情绪

```text
朋友惹祸 -> 主角想出头 -> 朋友不信主角 -> 好像能逃掉 -> 退路被堵死 -> 主角证明自己
```

爽点来自简单清楚的反差：死党和围观学生都认为主角不可能赢，结果主角几回合解决问题。

### 关键判断

- 作业不必复杂，折点功能清楚即可。
- 金手指要先给主角出手信心。
- 朋友的恐惧能放大混混威胁。
- 虚晃一枪可以是“以为只是路人/以为逃掉”。
- 最后要有见证人和被保护者反馈。
- 简单校园格斗也能练完整情绪链。

### 可迁移写法

```text
主角金手指：
被保护对象：
外部威胁：
同伴不信：
虚晃一枪：
退路封死：
见证人：
最终打脸：
```

### 训练任务

写一个 5 点五折线作业，每点只写一句话，但必须标明：下行、上行、虚晃、再下行、最终上行。写完检查每个点是否真的改变读者期待。

## 1217：#参考资料 某位朋友要的梗科普...


人工查看范围：正文、图片。无评论、无附件。图片外链已下载到：`<EVIDENCE_CACHE>\batch-244\1217-image-418851222425118.jpg`。

证据限制：正文只有一句“某位朋友要的梗科普up”，图片是 B 站账号页面截图；拆解重点是梗素材入口，而非具体梗内容。

### 读者承诺

这篇沉淀的是写梗、用梗要有稳定素材入口。图片显示 B 站 UP 主“梗指南”，认证为梗指南官方账号，粉丝量很高，动态里有“整一期读评论吧”等内容，说明梗科普账号可以作为作者追踪流行梗、评论梗和传播语境的来源。

### 核心机制

```text
作者需要新梗 -> 找到梗科普账号 -> 看解释/评论/传播语境 -> 判断能否入文 -> 转成段评可接的梗
```

这不是直接教怎么写章节，而是给出梗资料来源。它和榜单梗密度讨论互相补充：作者不能只凭印象塞梗，要知道梗从哪里来、什么意思、谁在用、读者能不能接住。

### 期待链与情绪

```text
读者看到熟悉梗 -> 迅速识别共同语境 -> 产生会心一笑 -> 段评接梗 -> 章节互动增强
```

梗源的价值不在“多”，而在作者能选到读者认识、语境正确、不会破坏正文情绪的梗。

### 关键判断

- 梗科普账号是素材入口，不是可直接整段搬运的内容库。
- 用梗前要理解语义和传播场景。
- 评论区本身也是梗反馈来源。
- 读者接得住，梗才有互动价值。
- 梗过密会伤害正文情绪。
- 素材入口要长期更新，不能只靠旧梗。

### 可迁移写法

```text
梗来源：
梗含义：
传播场景：
适合角色：
适合情绪：
读者识别门槛：
可放章节位置：
不用的原因：
```

### 训练任务

选 10 个近期梗，不写进正文，先做一张筛选表：含义、适用情绪、适合角色、读者识别门槛、是否会破坏剧情。只保留 3 个。

## 1218：起点榜上的文，那梗密度是真的高...


人工查看范围：正文、全部 7 条评论。无附件、无图片。

证据限制：这是榜单观察和资料讨论，没有给出具体章节；拆解重点是梗密度、段评互动和素材来源。

### 读者承诺

这篇沉淀的是高梗密度可以显著提升段评互动。作者观察起点榜文一章能塞 6、7 个流行梗，怀疑作者有“梗字典”，效果是段评直接起飞。评论补充了 B 站梗科普账号、小鸡词典、字媒体等来源。

### 核心机制

```text
作者持续收集热梗 -> 按章节情绪插入 -> 读者识别并接梗 -> 段评数量上升 -> 互动反哺热度
```

宝剑也纳闷作者是否天天盯新闻热点写书。其他人给出可行来源：B 站每天更新解释梗的账号、评论区、小鸡词典、公众号字媒体、梗指南、梗百科。这里的重点是梗不是凭空来的，而是持续收集和筛选。

### 期待链与情绪

```text
正文推进 -> 熟悉梗突然出现 -> 读者会心一笑 -> 段评接力 -> 章节显得热闹
```

梗密度的爽点偏互动和陪伴，不一定直接等于剧情爽点；它适合轻松、吐槽、都市、文娱、沙雕等语境。

### 关键判断

- 梗密度能带动段评。
- 高梗密度需要素材库支撑。
- 热点、梗科普、评论区都是来源。
- 梗要贴合章节情绪和角色口吻。
- 一章 6、7 个梗不适合所有题材。
- 梗传播链本身值得观察：谁先用、谁解释、谁扩散。

### 可迁移写法

```text
章节情绪：
可用梗：
来源：
角色口吻：
放置位置：
段评触发点：
过密风险：
替换方案：
```

### 训练任务

找一章轻松剧情，准备 8 个梗，最后只放 3 个：一个角色台词梗、一个旁白吐槽梗、一个段末引导段评的梗。删掉不贴角色口吻的梗。

## 1219：#公开处刑 请各位老师帮忙看一...


人工查看范围：正文、文件区、全部 7 条评论。文件区有 1 个附件：`这个明星想出道.doc`，id `421482111488428`，知识库未提供可读正文或可下载链接；本地按文件名和 id 未找到附件。无图片。

证据限制：投稿正文在不可读附件中，不能拆原稿具体段落；只能依据作者自述和评论诊断拆“文娱开头过度下行”的问题。

### 读者承诺

这篇沉淀的是文娱开头不能让读者连看三章主角憋屈。靠谱网文开头通常几句话说明主角已经处于“下行末端”，然后展示金手指，告诉读者接下来会进入上行情节。哪怕第一章不立刻装逼，也要让主角处在走向装逼的路上。

### 核心机制

```text
开头交代下行末端 -> 主角醒来/入局 -> 金手指或能力展示 -> 目标明确 -> 反派/导师言论衬托 -> 舞台爽点兑现
```

宝剑指出原稿前三章主角太憋屈，读者受不了。作者问能否直接从第四章开始：主角穿越醒来已经在后台写编曲，因公司穷买不起版权，前面几组就轮到他，主角快速编曲上台。宝剑进一步说明，爽点弱不是歌曲问题本身，而是前面情绪拉扯不够；选歌要和反派、导师状态和言论导向相关，才能衬托主角牛逼。选秀里也要有冲突，踩完当前对手，再去踩新年晚会背后的 boss。

### 期待链与情绪

```text
主角已在低谷 -> 马上要上台/被质疑 -> 金手指可解决当前问题 -> 反派和导师给出错误判断 -> 主角用作品打脸
```

文娱爽点需要舞台、对手、言论导向和作品选择互相咬合，不是单独换一首“更好听”的歌就能解决。

### 关键判断

- 开头不要从连续下行开始。
- 主角可以在低谷，但要处于下行末端。
- 金手指展示要尽早告诉读者后面会上行。
- 文娱选歌要对应当前冲突和评价口径。
- 选秀里必须有可踩的冲突对象。
- 长线目标可以升级：选秀对手后面接新年晚会 boss。

### 可迁移写法

```text
主角低谷：
下行末端证据：
当前舞台：
金手指展示：
当前反派：
导师/舆论导向：
作品选择理由：
下一层boss：
```

### 训练任务

重写一个文娱开头：只用 300 字交代主角低谷，然后立刻进入后台倒计时。写清反派或导师的错误评价，再让主角选择一首能精准反打该评价的作品。

## 1220：#思考分析 刚刚在写文的时候有...


人工查看范围：正文、全部 6 条评论。无附件、无图片。

证据限制：这是写作过程复盘，不是完整场景；拆解重点是同一种情绪描写过量，以及如何用配角/反派侧面承托主角从容。

### 读者承诺

这篇沉淀的是“主角从容淡定”不能靠连续写原因、动作、表情、话语来堆。针对同一种情绪，描述不能太细碎、太繁复、太集中；即使是装逼，也要让威胁、信息差和配角反应承担情绪。

### 核心机制

```text
主角需要从容 -> 先建立小BOSS压迫 -> 配角担忧/反派不解 -> 主角少量动作稳定 -> 危机转折证明从容有底气
```

作者发现自己写主角从容淡定的原因、动作、表情、话语，越写越烦。评论提醒，同样信息不要太细碎；boss 要让读者感觉到威胁和压迫，无敌文也会先写 boss 恐怖再秒。另有人建议“皇上不急太监急”：主角稳操胜券，但身边队友不知道，在情况急转直下时替主角担忧；反派也可以纳闷主角凭什么这么淡定。

### 期待链与情绪

```text
小BOSS看似有压迫 -> 配角替主角紧张 -> 反派误判主角自信过头 -> 主角仍只做少量反应 -> 后续证明他早有把握
```

读者要先感觉局面值得紧张，主角的从容才有装逼价值。

### 关键判断

- 同一种情绪不要密集重复描写。
- 从容淡定需要外部压力衬托。
- 反派压迫力不能被主角提前消解干净。
- 配角担忧是最省力的侧面写法。
- 反派不解能制造信息差。
- 主角动作越少，越需要后续结果证明底气。

### 可迁移写法

```text
主角目标情绪：
外部威胁：
配角担忧：
反派误判：
主角少量动作：
信息差：
打破淡定的可能：
最终证明：
```

### 训练任务

写一个主角从容面对小 boss 的场景，限制主角直接描写不超过 3 句；其余用配角担忧、反派不解和局势恶化来衬托。
