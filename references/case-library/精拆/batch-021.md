## 101：下期【情绪拉扯案例+技巧解读】...

人工查看范围：正文全文、全部评论、关于情绪拉扯案例、作业、拉期待和“望远镜”方法的讨论。

### 读者承诺

这条是情绪拉扯正式训练前的预告，核心价值在于提醒：不会拉期待、不会把“待人恨”的事写得让人恨，是新人爽点失败的前置问题。情绪拉扯作业不是单独技巧，而是接在“拉期待”之后的实践检验。

### 核心机制

```text
拉期待理论 -> 案例技巧解读 -> 作业检验 -> 发现不会恨/不会爽 -> 回到期待设计
```

### 关键判断

- 情绪拉扯训练需要和拉期待训练连起来。
- 很多新人不是没有冲突，而是不会把可恨之事写得足够让读者在意。
- “望远镜”式拉期待，是先让读者看见远处值得追的东西。
- 作业的作用是检验概念是否真正会用。
- 公开处刑要按规矩提交，否则样本质量会影响诊断。

### 可迁移写法

拉期待预检表：

```text
读者要期待什么：
远处可见的好处/答案/反击：
谁做了让读者恨的事：
读者为什么会恨：
情绪拉扯作业要验证哪一点：
```

### 训练任务

写一个 300 字开头前置段，只做两件事：让读者看见远处目标，让读者明确讨厌一个阻碍者。不要急着兑现。

## 102：#公开处刑

人工查看范围：正文全文、全部评论、关于飞卢仿写、冲突力度、爽点不爽、炮灰逆袭、外门测试舞台和主角反应的讨论。

证据限制：附件 `玄幻：天命女帝旺夫，我掠夺机缘仿写.docx` 只保留文件 id，没有可下载链接；本卡依据可见正文与评论精拆。

### 读者承诺

这篇指出“按套路写了仍然不爽”的根因：铺垫、打压、打脸、震惊、收益这套顺序本身没错，但如果最早冲突不发生在主角身上、没有不公平伤害、没有强烈危机，读者情绪调不起来。

### 核心机制

```text
不公平伤害 -> 主角必须反击 -> 旧敌持续拉仇恨 -> 测试舞台见证 -> 秒杀兑现 -> 收益升级
```

### 关键判断

- 爽点写法框架可以对，但事件选错仍然不爽。
- 冲突不是发生在主角身上，而是主角主动招来，会显得小打小闹。
- 主角行动动机如果只是馋女角色身子，情绪正当性不够。
- 强冲突可以是要被杀、驱逐、废修为等，让主角无路可退。
- 外门测试是天然装逼舞台：围观群众、亲友、宗门高层、敌人都在场。
- 旧敌刘贵应提前塑造成欺压主角多年的可恨对象，再在测试中被打脸。
- 主角被坑后的反应不能“无奈笑笑”，要体现杀意、压抑、克制和逼格。
- 飞卢式主角可以功利，但作者自己不认同这种三观时，代入会不足。

### 可迁移写法

强冲突爽点表：

```text
主角被谁长期压迫：
当前不公平伤害：
主角为什么必须反击：
装逼舞台：
见证者：
旧敌如何被打脸：
主角获得什么收益：
下一步目标：
```

### 训练任务

把一个“突然出现的新敌人”改成“欺压主角多年的旧敌”。写出三次拉仇恨铺垫，再写测试场合的一次公开打脸。

## 103：各位大大，我准备写一篇都市异能...

人工查看范围：正文提问、全部评论、关于都市异能三章、设定过长、金手指介绍过长、冲突偏平、扶老太太过马路、念书世界和第一章进入冒险的讨论。

证据限制：附件第一章外链返回 401，第二、三章只保留文件 id；本卡依据可见正文与评论精拆。

### 读者承诺

这篇是“有设定但开篇不吸引”的典型：主角、金手指、世界背景都有想法，但前两章抛设定太多，冲突平淡，核心卖点没有及时展示。正确修法是从能触发金手指的行动场景切入，快速进入冒险，让金手指在困境中翻盘。

### 核心机制

```text
日常行动触发积分 -> 金手指出现 -> 快速进入冒险世界 -> 使用兑换物翻盘 -> 获得奖励与期待
```

### 关键判断

- 开头介绍主角可以，但世界观和金手指介绍太长会拖。
- 设定弟弟、脑海图书馆、背景伏笔都不适合先堆。
- 学长冲突偏平淡，情绪拉扯不足，主角蒙混过关就不爽。
- 冲突要更直接，若用隐藏危机，必须让更厉害的人有压迫感。
- 可以直接从第三章“主角扶老太太过马路”开始，用行动引出金手指。
- 攒积分、兑换念书或扭蛋，是把设定嵌入事件的入口。
- 兑换物必须在接下来的冒险世界起决定性作用。
- 最迟第一章结尾，主角应进入冒险世界或明确核心卖点。
- 其他设定可以后续慢慢透露。

### 可迁移写法

设定转事件表：

```text
想介绍的设定：
能触发它的行动：
金手指第一次反馈：
兑换/能力选择：
马上进入的困境：
能力如何翻盘：
翻盘奖励：
延后说明的背景：
```

### 训练任务

把一段设定说明改成事件：不要解释金手指是什么，先写主角做一件好事获得积分，再让积分兑换物在下一场危机里救命。

## 104：#公开处刑 【忐忑jpg】

人工查看范围：正文提问、全部评论、关于双金手指、女主前世大佬、正向情绪系统、伪无敌、主动挑事和台词训练的讨论。

证据限制：附件 `天后她又在助人为乐开头小纲.docx` 只保留文件 id，没有可下载链接；本卡依据可见正文与评论精拆。

### 读者承诺

这篇给出双金手指和主动挑事的正例边界：女主前世大佬加正向情绪系统并不冲突，只要主角人设就是会算计别人并名利双收，主动挑事也可以成为爽点来源。

### 核心机制

```text
前世经验 -> 正向情绪系统 -> 主动挑事/助人为乐 -> 算计成功 -> 名利双收
```

### 关键判断

- 双金手指完全可以成立。
- 前世大佬会削弱危机感，但如果看点是算计别人和名利双收，方向仍然可行。
- 主角为了自己的目标主动挑事没问题，关键是符合主角人设。
- 配角表现夸张一点，新人阶段可以接受，后续靠练习打磨。
- 如果被欺负的是配角，拉仇恨比直接欺负主角弱。
- 但若主角机制是正向情绪系统，被迫助人为乐，配角受害也能作为触发器。
- 反派台词、对抗台词可以多研究成熟电影；好台词同时体现人设和推进剧情。

### 可迁移写法

主动挑事双金手指表：

```text
主角前世优势：
第二金手指：
主角主动挑事目标：
这件事如何符合人设：
受害者是谁：
情绪系统如何反馈：
主角获得的名：
主角获得的利：
```

### 训练任务

设计一个双金手指女主桥段：前世经验负责判断局势，系统负责给奖励。主角必须主动挑事，并在结尾同时获得名声和实际收益。

## 105：#思考分析 大致说下自己总结

人工查看范围：正文全文、全部评论、关于市场风向、情绪拉扯、主角视角、金手指、矛盾冲突、少说废话、新人节奏和大神慢开区别的讨论。

### 读者承诺

这条短总结把新人最小闭环压得很实用：先看投稿网站市场风向，再抓情绪拉扯、主角视角、金手指、矛盾冲突，一个故事接一个故事推进，基础行文别拖，最重要是少说废话然后写。

### 核心机制

```text
市场风向 -> 核心卖点 -> 主角视角 -> 金手指/冲突 -> 连续故事 -> 少废话执行
```

### 关键判断

- 投稿网站市场风向是开书前置判断。
- 情绪拉扯、主角视角、金手指、矛盾或冲突是开篇核心组合。
- 网文推进要一个故事接一个故事，不靠散点设定撑。
- 基础行文、段落、标点会影响可读性。
- 少说废话是新人最重要的执行原则之一。
- 开头多次改稿、模仿老手文风，是常见训练路径。
- 创意要遵循市场需求，不能过分追求噱头。
- 新人不要轻易模仿大神慢展开；大神能用细节真实慢慢切入世界，新人更应加快节奏。

### 可迁移写法

新人最小闭环表：

```text
目标网站：
市场主流：
主角视角：
金手指：
第一冲突：
第一故事闭环：
删掉的废话：
今天写了多少：
```

### 训练任务

给自己的项目写一张 8 行卡：目标网站、主流题材、主角、金手指、第一冲突、第一爽点、要删的废话、今天必须写的字数。
