## 252：个人写作笔记。

人工查看范围：正文。无评论，无附件，无图片。

### 读者承诺

这篇是个人写作过程中的硬约束清单，核心是前期不要慢、不要水、不要让复杂设定拖死自己。五万字前重点修改，十万字前稳住心态，断更也要保持接触文本，避免思路断线。

### 核心机制

```text
控制铺垫 -> 控制心理描写 -> 简化设定 -> 前五万字反复修改 -> 保持日常接触 -> 用理论校准但不被理论绑架
```

### 期待链与情绪

```text
作者写出初稿 -> 发现铺垫和设定拖慢节奏 -> 砍无效内容 -> 前五万字打磨质量 -> 维持写作手感 -> 读者更快进入有效剧情
```

### 关键判断

- 不要做对当前剧情没有推进作用的铺垫。
- 如果必须铺垫，尽量几十字概括。
- 最好的铺垫同时服务当下剧情和未来剧情。
- 纯心理描写尽量不要连续超过两百字。
- 推理等能推进剧情的心理描写可以保留。
- 如果心理描写只是讲设定，能砍就砍。
- 设定复杂到写出来繁琐，而自己文笔处理不了，就该改设定。
- 五万字前每章都要反复修改，因为一本书死活大多在前五万字决定。
- 十万字前不要过度被成绩影响，不同网站推荐节奏不同。
- 慢节奏新人慎用，除非已有粉丝基础。
- 断更也要码字，哪怕凑不出一章，也要保持手感。
- 写作期间少看和自己题材截然不同且影响强烈的作品。
- 写前明确目标读者，不要用“像我一样的人”含混代替。
- 单次剧情重要角色尽量不超过三人，次要角色尽量不超过五人。
- 理论不是全部，但绝大多数都不合理论时，应优先怀疑自己的文本。

### 可迁移写法

前五万字自检表：

```text
本章无效铺垫：
本章心理描写最长段：
设定是否超出表达能力：
当前章节是否慢节奏：
本章推动剧情的动作：
本章目标读者：
重要角色数量：
次要角色数量：
是否受外部作品干扰：
需要反复修改的位置：
理论冲突点：
最终以好不好看判断：
```

### 训练任务

检查自己前 3 章：每章删掉 1 个无效铺垫，压缩 1 段纯心理描写，并列出所有出场角色，判断是否超过承载上限。

## 253：新人小白想请教下怎么样写简介，...

人工查看范围：正文、全部 7 条评论。无附件，无图片。

### 读者承诺

这篇把简介问题和神话复苏题材底线问题连到一起。简介不是文案装饰，而是广告；书名、简介、开篇几章共同承担筛选读者、抛卖点和承诺题材边界的功能。卷纲式简介可以作为新人快速抓主线的方法。

### 核心机制

```text
书名吸引 -> 简介承诺 -> 开篇兑现 -> 卷纲压缩主线 -> 题材尺度提前声明
```

### 期待链与情绪

```text
读者看到神话复苏卖点 -> 简介告诉他主角要面对什么 -> 开篇几章兑现鬼怪/道士/异兽 -> 都市黑暗面不过线 -> 读者确认题材可追
```

### 关键判断

- 书名、简介、开篇几章，本质都是广告。
- 简介要服务卖点，不是把设定全部塞进去。
- 新人写不好简介时，可以参考成熟书名简介解析。
- 简介写卷纲是一种可行方法，把一卷核心矛盾压缩成文案。
- 神话复苏类在起点要注意都市题材底线。
- 鬼怪、道士、异兽会自然接触社会黑暗面，但尺度要提前控制。
- 现代都市与灵异/神话元素结合时，不要让黑暗面吞掉主卖点。
- 简介应承诺“复苏带来的核心变化”，而不是只列元素。
- 开篇几章必须兑现简介里的广告承诺。
- 如果简介像卷纲，就要保留悬念，不要把全卷结果讲死。

### 可迁移写法

卷纲式简介表：

```text
书名卖点：
目标读者：
题材核心：
主角身份：
第一卷核心矛盾：
神话复苏变化：
前三章兑现点：
都市尺度边界：
不能提前剧透的结果：
一句广告钩子：
```

### 训练任务

给一个神话复苏题材写 3 版简介：卖点直给版、卷纲压缩版、悬念钩子版；每版都标出前三章要兑现什么。

## 254：#毒点解析 最近在看《星界使徒...

人工查看范围：正文、全部 10 条评论、2 张图片。图片为《星界使徒》书友圈评论截图，读者吐槽从奇幻科幻期待跳到低级武侠/水浒式副本，以及担心后续一个世界 100 章顶不住；已下载至 `case-library\精拆\attachments\batch-051-case-254\` 并人工查看。

### 读者承诺

这篇拆的是无限流/多世界文的类型基调毒点：开篇、书名、简介和第一个世界已经筛选了一批期待奇幻、科幻、神奇世界的读者，后续突然换成低级武侠和水浒起义，会让读者感觉偏离类型定位，期待值下滑。

### 核心机制

```text
开篇定基调 -> 第一个副本筛读者 -> 第二个副本风格突变 -> 类型承诺偏移 -> 期待下行 -> 读者流失
```

### 期待链与情绪

```text
读者被星界/奇幻/科幻吸引 -> 期待更神奇的世界 -> 进入低级武侠争霸 -> 觉得变式使徒/水浒出戏 -> 主世界又割裂 -> 越筛读者越少
```

### 关键判断

- 无限流不等于可以随意大幅换基调。
- 书名、简介、开篇世界共同决定读者期待。
- 第一个世界偏奇幻，后续世界最好延续相近的惊奇感。
- 从剑与魔法突然跳到低级武侠，容易让读者感觉降级。
- 即便单个副本有装逼打脸爽点，类型不丝滑仍会形成毒点。
- 总体情绪从上行转为下行，读者期待会滑落。
- 副本世界和主世界割裂，会加重阅读断裂。
- 主世界学院戏如果突然塞太多人物和打斗，会让节奏失焦。
- 每次大幅换风格都在重新筛读者。
- 筛读者不是越多越好，网眼越小，留下的人越少。
- 喜欢第二世界的读者也可能在第三世界继续被筛掉。
- 开头家庭矛盾拖节奏且恶心部分读者，大神可试，新人慎用。
- 多世界文的关键不是世界多，而是主承诺稳定。

### 可迁移写法

无限流副本基调表：

```text
书名/简介承诺：
开篇世界基调：
目标读者期待：
第二世界候选：
是否降级：
是否换风格过猛：
主世界连接方式：
每个副本情绪方向：
读者被筛掉的原因：
需要保留的主承诺：
```

### 训练任务

为一个无限流设计 3 个连续副本：每个副本都换题材元素，但必须保留同一个主承诺；写出不能跨过的风格边界。

## 255：人设硬爽点多，硬核拆书《曹操穿越武大郎》谈如何学习高手写作手法

人工查看范围：知识库可见正文摘要、全部 25 条评论。无附件，无图片。

证据限制：正文主体以外链文章形式存在，知识库仅保留前言和第一节开头；本卡依据可见摘要与评论精拆。

### 读者承诺

这篇是拆书方法论补强：先完整读一遍，再决定拆不拆；第二遍开始批注；多次阅读会不断发现新层级。教程和模板只是参考，最终要形成适合自己题材、习惯和目标的拆书方法，而不是把“计划-意外”等模式硬套到所有题材。

### 核心机制

```text
完整阅读 -> 判断是否值得拆 -> 第二遍批注 -> 多遍深化 -> 结构图辅助 -> 按题材调整方法
```

### 期待链与情绪

```text
学习者想拆高手书 -> 第一遍先读懂 -> 第二遍标注结构和爽点 -> 第三遍看细节与整体 -> 发现模板边界 -> 形成自己的方法
```

### 关键判断

- 拆书前必须先读一遍，不是没看就闷头拆。
- 成绩好的书不一定适合自己拆，因为可能学不来。
- 成绩一般的书也可能有值得拆的优秀处。
- 多次阅读是必要的，次数越多，领悟越多。
- 第一遍可能只关注上下行、字数、铺垫和情绪拉扯。
- 第二遍、第三遍会发现之前没看到的细节和整体结构。
- WPS 批注、截图批注、结构折线图都只是工具。
- 拆书方法作为参考，按需选择即可。
- 新人想赚钱要谨慎选择赛道，功力好也可能因题材冷而签约困难。
- 学教程的意义是形成自己的方法论，不是照单全收。
- “计划-意外”适合一部分热血升级题材，但不能覆盖所有题材。
- 日常恋爱等题材硬套计划意外，会失去本题材味道。
- 目标段落模板太死，会凌驾作者思路，导致四不像。
- 不同题材要通过拉片拆书总结不同方法。
- 自己喜欢就先看，用数量覆盖，写多了才知道哪些更适合自己。

### 可迁移写法

多遍批注拆书表：

```text
第一遍阅读感受：
是否适合自己拆：
值得学习的点：
学不来的点：
第二遍批注重点：
第三遍结构重点：
使用工具：
题材适配限制：
不能硬套的模板：
最终沉淀的方法：
```

### 训练任务

选一本自己喜欢的书前三章做三遍拆读：第一遍只写感受，第二遍标注爽点和铺垫，第三遍画结构线，并写出哪些技巧不能直接套到自己题材。
