## 341：在拆书的时候，忽然有个突发奇想...

人工查看范围：正文、全部 14 条评论。无附件，无图片。

### 读者承诺

这篇讨论开篇配角的功能配套。开篇需要自然介绍背景时，配角不只是聊天工具，而是按主角状态配套出现：孤儿主角常配死党，亲缘主角常配妹妹/兄弟。配角可以当信使、资源入口、气氛调节、动力目标，也能从“信使”转成长期伙伴。

### 核心机制

```text
主角状态 -> 配套配角登场 -> 对话引出背景 -> 配角承担信使/资源/锅/互动 -> 可转伙伴或主角分身
```

### 期待链与情绪

```text
读者进入陌生开篇 -> 配角与主角对话降低说明感 -> 配角暴露关系和背景 -> 主角行为逻辑保持稳定 -> 互动产生人设和后续伙伴期待
```

### 关键判断

- 开篇配角常用于自然介绍剧情和背景。
- 主角天煞孤星时，死党同学、胖子、高富帅常承担前期功能。
- 这类死党能提供金钱、物品、人脉，甚至吸引仇恨目标。
- 主角有亲人时，妹妹或兄弟常承担气氛调节和前期动力目标。
- 这类配角俗套但有效。
- 《诡秘之主》开头也用妹妹引出背景。
- 配角可以从前期信使转为中期伙伴，重生文里不少死党贯穿全书。
- 好对话应在展露人设的同时展露信息。
- 网文常用解释型对话，是因为多数作者难以同时写信息和人设，不是因为上乘写法不需要人设。
- 伙伴可以视为主角的分身，拆出优点或缺点。
- 拆优点偏情感，拆缺点偏成长。
- 伙伴和主角大目标通常一致，伙伴私事如果不服务主线，最好少写。
- 如果要给伙伴加戏，伙伴小目标要与主角大目标形成克制。
- 不符合主角逻辑的看美女、管闲事、犯错等行为，可以交给配角承担。
- 配角犯错接剧情，主角解决问题拿功劳，是稳定的网文推进法。

### 可迁移写法

开篇配角功能表：

```text
主角起点状态：
配套配角类型：
配角与主角关系：
要引出的背景信息：
配角提供的资源：
配角承担的锅或风险：
配角是否是主角分身：
配角小目标：
是否克制主角大目标：
后续转伙伴路径：
```

### 训练任务

给一个开篇设计两个配角版本：死党版和亲人版。每版都写出他引出的背景信息、提供的资源、替主角承担的行为，以及能否转为中期伙伴。

## 342：想问一下大家仙侠和玄幻的最大区...

人工查看范围：正文、全部 4 条评论。无附件，无图片。

### 读者承诺

这篇把仙侠和玄幻的边界落到几个可操作指标：等级体系里有没有“仙”，世界文化是否保持中国古典传统，终极目标是否求长生/超脱，能力体系是否偏修炼学习而非随意 bug。玄幻限制更少，可以混合更多元素和更不讲道理的天赋能力。

### 核心机制

```text
题材承诺 -> 检查等级体系是否有仙 -> 检查文化和技术边界 -> 检查终极目标 -> 区分仙侠或玄幻
```

### 期待链与情绪

```text
读者看到仙侠 -> 期待仙气、侠意、长生、修行和古典秩序 -> 读者看到玄幻 -> 接受更自由的体系、血脉、天赋和异世界混搭
```

### 关键判断

- 现在仙侠和玄幻确实越来越难区分。
- 简单标准可以看等级体系里有没有仙。
- 仙侠更强调仙气和侠意。
- 仙侠世界更适合中国古代传统文化气质。
- 仙侠不宜随意出现欧洲人、欧洲宗教、现代科技等明显异质元素。
- 佛教相关设定要谨慎，受众不一定喜欢，也更可能敏感。
- 道教相关设定更贴合仙侠，使用风险相对低。
- 仙侠常以长生、超脱、自由为最终目标。
- 仙侠人物寿命通常较长。
- 仙侠修炼体系多为嗑药、吸纳灵气、学习法术和规则约束。
- 玄幻里长生常是附带，不一定是核心。
- 玄幻修炼体系更自由，只要大致合理即可。
- 玄幻可以设计弱小时掌握时间、空间、命运等 bug 技能。
- 玄幻更容易出现不讲道理的天赋、血脉和奇遇。

### 可迁移写法

仙侠玄幻边界表：

```text
等级体系是否有仙：
终极目标：长生/超脱/力量/争霸
文化基底：
是否有外来宗教或科技：
能力来源：修炼/学习/血脉/天赋/系统
规则技能出现阶段：
寿命尺度：
核心气质：仙气/侠意/热血/奇幻
最终题材归类：
```

### 训练任务

拿一个设定分别改成仙侠版和玄幻版。仙侠版必须写清长生目标、古典文化和修炼路径；玄幻版必须写清自由能力体系和更大的爽点尺度。

## 343：我问一下，世界观模糊是什么意思...

人工查看范围：正文、全部 8 条评论。无附件，无图片。

### 读者承诺

这篇讨论编辑反馈“世界观模糊”。虽然原词来自编辑个人理解，不能替编辑强解，但评论给出一个重要判断：鬼怪和宗门的矛盾主体不同。鬼怪强调人和鬼的矛盾，宗门强调人和人的矛盾；如果只是把鬼怪改成妖兽，而没有重建宗门与怪物之间的关系，世界观仍然会显得不清。

### 核心机制

```text
多个世界观元素 -> 各自矛盾主体不同 -> 环境描写无法统一冲突 -> 编辑感到模糊 -> 用核心组织关系统一元素
```

### 期待链与情绪

```text
读者看到鬼怪/妖兽 -> 期待人怪矛盾 -> 又看到宗门 -> 期待人际势力矛盾 -> 两条矛盾无主次 -> 读者不知道故事主要看什么
```

### 关键判断

- “世界观模糊”这类编辑词，最好直接问编辑。
- 不同编辑可能有自己的理解，不宜随便替他下定义。
- 只通过环境描写突出背景，不一定能解决世界观主矛盾。
- 鬼怪和宗门不是单纯素材差异，而是矛盾主体差异。
- 鬼怪讲的是人和鬼之间的矛盾。
- 宗门讲的是人和人之间的矛盾。
- 把鬼怪改成妖兽，只是替换怪物名词，未必解决主矛盾冲突。
- 如果宗门是专门对付鬼怪/妖兽的组织，两者就能统一。
- 世界观清晰需要说明主要冲突如何组织社会。
- 背景、怪物、宗门、主角目标要服务同一个故事问题。

### 可迁移写法

世界观主矛盾统一表：

```text
元素A：
元素A的主要矛盾：
元素B：
元素B的主要矛盾：
当前主矛盾：
谁负责解决这个矛盾：
组织/宗门为何存在：
怪物/敌人如何影响社会：
主角目标如何进入主矛盾：
是否只是换名词：
```

### 训练任务

把“宗门+鬼怪/妖兽”写成一句世界观承诺：这个宗门为什么存在，它和怪物的关系是什么，主角第一章要面对哪一个具体矛盾。

## 344：#思考分析 说点感悟。

人工查看范围：正文。无评论、无附件、无图片。

### 读者承诺

这篇强调书名是作者和读者的互相筛选。新人没有名气，书名必须表达题材、基调、卖点、噱头、金手指，让读者看到书名就能在脑中形成故事框架和期待；正文第一章则要按背景、危机、赌注、目标、金手指、阻碍等要素满足这个期待。

### 核心机制

```text
书名筛选读者 -> 读者形成故事预期 -> 第一章按要素兑现 -> 表里一致 -> 签约和追读更稳
```

### 期待链与情绪

```text
读者看到书名 -> 判断是不是自己想看的菜 -> 书名承诺题材和卖点 -> 第一章交付对应金手指和危机 -> 期待被满足继续读
```

### 关键判断

- 书名非常重要，是作者和读者互相筛选的入口。
- 新人没有名气，只能先靠书名吸引目标读者。
- 不知所云或让人误会的书名很容易被跳过。
- 书名要表达足够多信息。
- 题材、基调、卖点、噱头、金手指都可以尽量体现在书名里。
- 读者看到书名时，脑中应能产生大概故事框架。
- 书名产生期待后，正文必须满足期待。
- 书名和故事不一致，就像点炖肘子上白菜汤。
- 编辑审稿也会看背景、危机、赌注、目标、金手指、阻碍等要素框架。
- 金手指介绍有合适篇幅，过短不清，过长会拉低评价。
- 表里不一、结构不明、缺少要素、乱换地图、不能满足期待都可能影响签约。
- 作者给出的第一章结构包含：时间地点主角、主角行为、世界人物背景、新手礼包铺垫、金手指、礼包收获、使用计划和长期目标、危机和路径、赌注风险与堵死其他路径。
- 模板可以消化吸收，但要素最好齐全。

### 可迁移写法

书名第一章一致性表：

```text
书名：
书名承诺的题材：
书名承诺的卖点：
书名承诺的金手指/噱头：
第一章时间地点主角：
主角行为：
世界和人物背景：
金手指介绍篇幅：
新手礼包/初始收获：
长期目标：
当前危机：
赌注风险：
是否满足书名期待：
```

### 训练任务

给一个已有开篇反推书名承诺，再检查第一章是否兑现。若书名像“炖肘子”而正文像“白菜汤”，写出一个改书名方案和一个改第一章方案。

## 345：流浪地球的故事核心 就这么短...

人工查看范围：正文、全部 1 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-069-case-345\image-1.jpg`。

图片证据：图片为《流浪地球 2》电影剧情概况：太阳即将毁灭，人类在地球表面建造巨大推进器，寻找新家园；宇宙之路危机四伏，为拯救地球，流浪地球时代的年轻人挺身而出，展开争分夺秒的生死之战。

### 读者承诺

这篇强调故事核心越短越好。核心梗必须一句话能完整概括，雷同并不可怕；同样是少年成长为强者，也可以被不同作者反复写。真正危险的是核心太杂、不够精炼，作品还没写就已经失去清晰承诺。

### 核心机制

```text
复杂想法 -> 压成一句核心梗 -> 读者立刻理解承诺 -> 细节和执行做差异 -> 长篇围绕核心扩展
```

### 期待链与情绪

```text
读者看到一句话核心 -> 立刻明白危机、目标和主角行动 -> 接受常见母题 -> 期待作者如何讲出细节差异
```

### 关键判断

- 故事核心越短越好。
- 核心梗必须一句话就能完整概括。
- 核心雷同不一定是问题。
- “少年成长为强者”这种母题可以反复写。
- 土豆多本书都能围绕相近成长核心展开。
- 太杂、不够精炼的文，没写前就已经有扑街风险。
- 《流浪地球》的核心可以压成：太阳毁灭，人类推着地球找新家园，年轻人为拯救地球进行生死战。
- 一句话核心要包含危机、目标和行动方向。
- 差异化不必放在核心梗里塞满，而应在人物、节奏、场面和细节里体现。

### 可迁移写法

一句话核心梗表：

```text
世界/处境危机：
主角或群体目标：
核心行动：
失败代价：
一句话核心：
是否超过一句：
是否塞入太多副设定：
哪些差异放到细节：
长篇扩展方向：
```
