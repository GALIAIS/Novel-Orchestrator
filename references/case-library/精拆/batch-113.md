## 561：刚刚看了视频，发现自己有三点做...

人工查看范围：正文、全部 1 条评论。无附件，无图片。

### 读者承诺

这篇是开篇自我复盘样本：作者看完教程后发现自己在迫切性、不公平伤害和迪化情绪上都没做够。它的价值在于把“开头不对劲”拆成具体缺项，再通过文字总结让问题变清楚，而不是继续凭感觉改稿。

### 核心机制

```text
看教程 -> 对照开篇元素 -> 找到缺项 -> 写成复盘 -> 反推修改动作
```

### 期待链与情绪

```text
作者以为冲突已经成立 -> 发现没有身份没钱还不够迫切 -> 补抓捕风险和不公平对待 -> 再补看不起与大佬迪化反差
```

### 关键判断

- 作者把冲突放在“没有身份无法工作、没有收入”上，但觉得迫切性不足。
- 他意识到应增加没有身份证明会被抓等更直接危害。
- 主角来到危险世界一无所有，理论上是不公平，但正文没有让他遭到不公平对待。
- 迪化剧情只展示大佬处理负债者的残酷手段，情绪没拉起来。
- 更有效的做法是加入手下对主角的看不起，和后面大佬反应形成对比。
- 星主鼓励把反思打出来总结，因为这样自己会认识更清楚。

### 可迁移写法

开篇复盘三项表：

```text
当前冲突：
迫切性是否足够：
不公平伤害在哪里：
读者能否共情：
情绪被谁下压：
后续谁来反向抬高：
本轮修改动作：
```

### 训练任务

看完一个开篇教程后，不直接改稿。先写 300 字复盘，分别回答迫切性、不公平伤害、情绪对比三项缺在哪里。

## 562：在研究畅销榜上的金手指，看了夜...

人工查看范围：正文、全部 3 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-113\case-562-image-1.jpg`。

图片证据：图片为百度百科式角色资料，列出庆尘的身份、师父、亲人、领路人、徒弟、能力、功法、技能、禁忌物等，显示其能力和身份是多阶段、多来源叠加。

### 读者承诺

这篇讨论畅销榜金手指研究：单个设定可能都是老酒，但成绩来自剧情手法和阶段性能力叠加。若初始金手指不具备自然成长性，就要通过因果关系让主角不断获得新能力，或者阶段性增加金手指。

### 核心机制

```text
初始金手指 -> 形成优势因果 -> 获得别人得不到的能力 -> 阶段性增加玩法 -> 继续支撑长线
```

### 期待链与情绪

```text
读者看到主角初始能力 -> 期待它带来新资格 -> 新资格获取新能力 -> 能力树不断扩展 -> 长线期待不断档
```

### 关键判断

- 发帖者研究《夜的命名术》百科，认为设定本身都不是新东西。
- 成绩说明作者剧情和手法拉满。
- 评论指出主角多才多艺、身兼多种职业是主角标配。
- 关键判断是金手指不具备成长性时，后面要阶段性增加金手指。
- 也可以让已有金手指成为因果前提：因为超级记忆，才能获得别人得不到的能力。
- 图片里的身份、能力、功法、技能、禁忌物说明长篇主角能力不是单点，而是持续扩容。

### 可迁移写法

阶段性金手指扩容表：

```text
初始金手指：
它不成长的风险：
第一阶段带来的资格：
第二阶段新增能力：
新增能力的因果理由：
第三阶段新玩法：
能力扩容后的新敌人：
```

### 训练任务

给一个不成长的金手指设计三阶段扩容。每阶段都必须由上一阶段能力带来资格，而不是作者硬塞新外挂。

## 563：#毒点解析 分享一下最近看到的...

人工查看范围：正文、全部 11 条评论、3 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-113\case-563-image-1.jpg`、`case-library\精拆\attachments\batch-113\case-563-image-2.jpg`、`case-library\精拆\attachments\batch-113\case-563-image-3.jpg`。

图片证据：三张图片均为《一拳超人》相关读者评论截图，集中批评重制版剧情基础没打好、饿狼人设崩、别打着打着亦敌亦友、村田加入私货、按最近几话评分还没出完别急着下结论等，和正文“群像写嗨削弱主角”的判断一致。

### 读者承诺

这篇把《一拳超人》重制版争议转化为无敌流写作警告：群像可以出彩，但不能把主角忘了。无敌流的核心期待，是配角被压到足够绝望后，主角出场逆转战局；如果群像打得有来有回，甚至由配角或反英雄盘活死局，主角的必要性就被削弱。

### 核心机制

```text
群像受压 -> 局面一边倒恶化 -> 读者期待无敌主角 -> 主角出场逆转 -> 群像价值回流主角
```

### 期待链与情绪

```text
读者看英雄被怪人压制 -> 等待琦玉登场 -> 如果配角自己盘活局面 -> 主角期待被掏空 -> 读者转而骂战力和人设
```

### 关键判断

- 发帖者认为表面问题是饿狼人设崩、战力体系崩、剧情拖沓。
- 根本问题是布局能力弱，搞群像搞到忘了主角。
- 原作中英雄协会被怪人协会暴打，英雄倒下后等琦玉出场，读者爽。
- 重制版打得太有来有回，英雄被碾压的感觉没了，主角出场必要性下降。
- 读者骂战力崩，本质是配角实力提升快到不需要主角。
- 饿狼如果打着打着亦敌亦友，会削弱无敌老师出手的期待。
- 群像最终应为主角服务，不能削弱甚至取代主角。
- 星主评论补充：有作品后期群像写到二三十章没主角，很离谱。
- 图片评论也指出重制版饿狼相关剧情和人设基础没打好。

### 可迁移写法

无敌流群像服务主角表：

```text
群像角色：
他们承担的压力：
他们不能解决的死局：
主角出场前的最低点：
主角出场逆转方式：
群像如何反衬主角：
不能让配角替代的功能：
```

### 训练任务

设计一个无敌流群像战场：写 5 个配角努力失败的节点，最后只能由主角解决。每个配角可以高光，但不能解决核心死局。

## 564：#公开处刑

人工查看范围：正文、全部 5 条评论、附件状态。附件 `这个书生大有问题.docx` 在导出目录未发现本地文件，原文无可下载链接，未能查看附件正文。无图片。

证据限制：无法查看附件正文，本卡依据正文设定和评论反馈拆解。

### 读者承诺

这篇诊断“书娘化”设定的金手指差异化不足：旧世界诗词典籍化作可爱强大的书娘是卖点，但如果别人也能通过读书得到书娘，主角就不够不同。主角必须因为穿越者知识库获得远超常人的书娘数量、稀有度或组合优势。

### 核心机制

```text
书籍娘化世界 -> 普通人获取困难 -> 主角拥有旧世界知识 -> 轻松获得多书娘 -> 用数量/稀有度拉差距
```

### 期待链与情绪

```text
读者看到书娘世界 -> 期待主角如何不同 -> 若人人都能获得就无聊 -> 主角凭旧世界知识成规模召唤 -> 形成爽点
```

### 关键判断

- 作者设想是旧世界文学知识到新世界，诗词文章经史典籍化作书娘。
- 评论直接指出“没看点”，金手指没啥用。
- “我看书就能获得书中力量”简单直接，但作者觉得不符合书娘世界观。
- 关键建议是拉大主角与普通人的差距。
- 普通人一生只能得到一只书娘，两只已凤毛麟角，主角却能得到几十只、上百只，金手指才有意义。
- 另一条评论指出剧情过于平淡，被高人看中和加大考试难度不适合作为开篇主爽点。
- 幕后高人最好不要一开始透露，应先强调主角受不公平待遇，再用金手指翻盘。
- 设定应少量抛出，在事件中不断体现，而不是大篇幅讲设定。

### 可迁移写法

书娘金手指差异化表：

```text
世界通用规则：
普通人获取难度：
主角独有资源：
主角第一只书娘：
数量差距：
稀有度差距：
第一次翻盘事件：
设定分批展示位置：
```

## 565：提问：宝剑哥，看了你的教程，不...

人工查看范围：正文、全部 6 条评论。无附件，无图片。

### 读者承诺

这篇校准“开篇 18 元素”的使用范围：18 个要求不是塞进第一章，而是前两万字整体满足。第一章的任务通常是讲清金手指、让读者代入主角处境、拉起期待；“留住读者”与“18 元素审核”不是同一件事。

### 核心机制

```text
误解元素清单 -> 第一章硬塞信息 -> 内容杂碎 -> 明确前两万字范围 -> 第一章只完成核心留存任务
```

### 期待链与情绪

```text
作者想一章塞满矛盾金手指信息 -> 发现开头杂乱 -> 星主澄清 18 元素是前两万字合格线 -> 第一章回到亮点和代入
```

### 关键判断

- 发帖者想把十八点都塞进第一章，导致内容很杂很碎。
- 星主明确没有说都塞进第一章，而是前两万字满足十八个要求就合格。
- 一章不够把事情全部说明白。
- 通常第一章能把金手指说明白、让读者代入主角处境、拉起期待，就算完成任务。
- “一章甚至半章留住读者”说的是留存亮点，不等于 18 元素审核。
- 18 元素审核是尽量剔除可能踩的雷区。
- 前两万字要求是因为两三章直球发上来时，连完整剧情都没写完，无法判断。

### 可迁移写法

第一章与两万字分工表：

```text
第一章任务：
前3000字必须有：
前2万字必须补齐：
不能塞进第一章的信息：
留住读者的亮点：
18元素排雷项：
当前杂碎的信息：
后移位置：
```

### 训练任务

把一个想塞满信息的第一章拆成两张表：第一章只保留金手指、处境、期待；其余 18 元素分配到前两万字的具体章节。
