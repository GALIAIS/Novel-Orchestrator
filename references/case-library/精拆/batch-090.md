## 446：三幕式小想法（1）

人工查看范围：知识库可见正文摘要、全部 2 条评论。无附件，无图片。

证据限制：正文为知识星球文章摘要，完整正文在外链文章中，知识库镜像只保留前半段并以省略号截断。以下拆解以可见摘要和评论为依据。

### 读者承诺

这篇把三幕式中的中心危机和延滞危机合并理解：中心危机不必一次打完，可以用多个小危机作为中间点，用伪胜利和伪失败连接，让剧情写长写大；当主角目的达成一半甚至更多时，再让延滞危机爆发，把故事推入至暗时刻和第三幕。

### 核心机制

```text
中心危机建立 -> 小危机分段推进 -> 伪胜利/伪失败连接 -> 小高潮达成半目标 -> 延滞危机爆发 -> 至暗时刻 -> 第三幕
```

### 期待链与情绪

```text
读者知道大危机存在 -> 看主角解决连续小危机 -> 每次胜败都改变局面 -> 小高潮后以为接近成功 -> 延滞危机爆发制造下坠 -> 期待第三幕翻盘
```

### 关键判断

- 中心危机和延滞危机可以同时存在。
- 小危机可以作为中间点，帮助大剧情分段。
- 伪胜利和伪失败是连接各个小危机的节点。
- 这种结构能帮助情节持续酝酿，也就是把故事写长写大。
- 小高潮适合放在主角目的达成一半或更多的时候。
- 延滞危机要在读者以为局面转好时爆发。
- 延滞危机爆发后，剧情进入至暗时刻。
- 至暗时刻开启第三幕，给最后翻盘留空间。
- 评论认为总结清楚，例子直观。
- 对新人来说，重点不是套名词，而是让每个小危机都改变中心危机的状态。

### 可迁移写法

中心危机延滞表：

```text
中心危机：
延滞危机：
小危机1：
小危机1伪胜利/伪失败：
小危机2：
小危机2伪胜利/伪失败：
主角半目标达成点：
延滞危机爆发方式：
至暗时刻：
第三幕翻盘资源：
```

### 训练任务

给一卷故事写 1 个中心危机、1 个延滞危机和 3 个小危机。每个小危机必须有一次伪胜利或伪失败，并说明它如何改变中心危机。

## 447：我读园林的，最近已经是找工作的...

人工查看范围：正文、全部 16 条评论。无附件，无图片。

### 读者承诺

这篇不是写法技巧，而是新人是否全职写网文的现实判断。没出成绩前不建议全职，除非有足够积蓄、家庭压力低、写书本身带来由衷快乐。写网文可以锻炼文案、表达和叙事能力，但“爱看小说、想写小说、写得了小说、能靠小说糊口、能靠小说赚钱”是完全不同的阶段。

### 核心机制

```text
职业痛苦 -> 想转写作 -> 检查积蓄/家庭压力/写作快乐 -> 先找糊口工作 -> 兼职试错写作 -> 再按成绩转向文案/编剧/全职
```

### 期待链与情绪

```text
毕业迷茫想逃离专业 -> 写作像希望 -> 评论压低全职冲动 -> 先保生存底线 -> 用兼职写作验证能力 -> 再考虑转行
```

### 关键判断

- 没出成绩时一般不建议全职写网文。
- 可以例外的情况包括：积蓄足够，不担心一年没收入。
- 家里不会给予过多压力，不会因此焦虑。
- 写书能带来由衷快乐。
- 网文训练出的文案能力可以迁移到文案、视频、公众号等工作。
- 写材料、写文案也能因网文训练变得更简洁有效。
- 园林、景观、工程相关行业被多位评论者描述为活多钱少、房地产下行影响大。
- 小说试错成本低，但用职业工作时间试错仍有机会成本。
- 爱看、想写、写得了、能糊口、能赚钱是不同概念。
- 稳定月入五千并不容易，可能比普通工作更难。
- 家庭环境没压力很关键，但仍建议先找工作攒存款。
- 可以先找能糊口的工作，再写出几个作品，往编剧、文案、公众号方向转。

### 可迁移写法

全职写作风险表：

```text
当前职业压力：
存款可支撑月数：
家庭压力：
是否已有写作成绩：
写作是否带来快乐：
兼职写作时间：
目标收入：
最低生存收入：
可迁移能力：
备选工作：
全职触发条件：
止损条件：
```

### 训练任务

做一张 12 个月写作试错表。写清每月生活费、存款、兼职写作字数、投稿目标、最低收入线和止损时间。没有达到全职触发条件前，不把写作当唯一收入来源。

## 448：今天发现云上轻骑兵大佬更新了他...

人工查看范围：正文、外链状态。原文无评论、无附件、无图片。

证据限制：正文只推荐 B 站视频《网络写作月入过万15：爆款网络小说的特点》，本地知识库未保存视频内容；短链公开访问返回不可用。以下只拆该帖体现的资料定位和学习方式，不推断视频具体内容。

### 读者承诺

这篇的价值在于资料发现：爆款网络小说特点是值得补充到学习体系的外部教程主题，但外部教程必须回到自己的作品检查表，而不是停留在“值得一看”。没有视频正文时，不能把视频内容当成已吸收，只能把它登记为待学习资料源。

### 核心机制

```text
发现外部教程 -> 确认主题 -> 记录资料来源 -> 看后提炼爆款特征 -> 转成作品检查表 -> 用样本验证
```

### 期待链与情绪

```text
作者想了解爆款 -> 发现教程更新 -> 把教程作为资料入口 -> 看完提炼可执行指标 -> 回到开篇/题材/节奏检查
```

### 关键判断

- 原帖只表达“云上轻骑兵更新网文教程，值得一看”。
- 视频标题指向“爆款网络小说的特点”。
- 本地知识库没有保存视频正文或字幕。
- 公开短链访问不可用，因此不能声称已吸收具体视频内容。
- 正确处理方式是把它作为外部资料源，而不是直接生成所谓爆款规律。
- 外部教程学习后要转成可执行检查问题。
- “爆款特点”应结合具体样本验证，避免只记口号。

### 可迁移写法

外部教程吸收表：

```text
教程标题：
作者/来源：
主题：
可访问状态：
核心观点1：
核心观点2：
样本作品：
可转化检查项：
我的作品对应问题：
是否已验证：
```

## 449：想问问各位，你们平时给主角取名...

人工查看范围：正文、全部 11 条评论。无附件，无图片。

### 读者承诺

这篇把主角取名从“独特”拉回“好读好认”。主角名字首先不能奇葩、生僻、难记，其次才考虑主线谐音、诗词意境、设定音译、现实人名改造、生成器筛选。网文主角名是读者代入入口，不是作者炫设计的地方。

### 核心机制

```text
好读好认 -> 避免生僻奇葩 -> 绑定主线/人设 -> 从诗词/现实名/音译/生成器取材 -> 读者记得住
```

### 期待链与情绪

```text
读者看到主角名 -> 能顺畅读出来 -> 迅速记住人物 -> 名字略带题材或主线暗示 -> 代入不被打断
```

### 关键判断

- 星主给出的核心标准是“好读好认”。
- 评论明确提醒不要用生僻字。
- 取名可以套梗、主设定谐音、主设定翻译音译。
- 可以改现实人名，用同偏旁、同义词或缝合虚拟人物名字。
- 中文名可以从药材名或诗词中提取。
- 西方名可以找两个形容词，翻译成外语后音译，再改得像人名。
- 日文名可以从动漫、日剧等作品中拆分组合。
- 也可以从 B 站、QQ 列表、微博等现实昵称里找灵感。
- 主线谐音名能让名字好记又圈题，比如鹿行对应路行。
- 姓名生成器可用于摇出候选，再人工挑顺眼的。
- 主角名不能为了独特牺牲读者识别。

### 可迁移写法

主角取名检查表：

```text
候选名：
是否好读：
是否好认：
是否有生僻字：
是否奇葩出戏：
主线谐音：
人设/题材暗示：
来源：
读者第一眼记忆点：
是否需要换名：
```

### 训练任务

给一个主角列 20 个候选名。先删掉所有生僻、难读、奇葩的名字，再从剩余名字里选 3 个能暗示主线或人设的，读出来测试是否顺口。

## 450：插播一条消息，刚刚我在QQ群上...

人工查看范围：正文、全部 10 条评论、2 张图片引用。两处图片引用为同一图片，已下载并人工查看：`case-library\精拆\attachments\batch-090-case-450\image-1.jpg`、`image-2.jpg`。

图片证据：图片显示一本文名为《我在大魏当毒枭[穿越]》的作品页，类型为原创-纯爱-架空历史-爱情，约 1 万字，未签约；文案写主角穿越到中国广东的架空程氏家族，并决定在大魏王朝继续完成前世未竟之事业。

### 读者承诺

这篇是红线提醒：涉毒题材不是“能不能技巧化处理”的问题，而是男频连过审都不可能、平台审核也可能出严重问题。作者不要拿违法高危职业当主角事业卖点，更不能用穿越、架空、纯爱等包装去淡化红线。

### 核心机制

```text
看到高危题材 -> 判断是否违法红线 -> 不做主角事业卖点 -> 不用架空包装稀释 -> 直接删除/改成反毒立场
```

### 期待链与情绪

```text
作者发现危险样本 -> 群内提醒同行 -> 评论确认红线严重 -> 创作时主动避开 -> 平台和作者降低审核风险
```

### 关键判断

- 原帖询问《我在大魏当毒枭[穿越]》是否属于红线。
- 发帖者补充主要是提醒同行，有些线不能碰。
- 评论认为这不是红不红线的问题，而是作者明显知道问题仍故意写。
- 星主判断这类题材在男频连过审都不可能。
- 评论认为晋江审核有问题。
- 还有评论提到不止这一本，晋江此波爆出多本涉毒内容。
- 图片中文案把“毒枭”作为标题和主角事业方向，这是最危险的识别点。
- 架空历史、穿越、爱情标签不能抵消涉毒主角事业的风险。
- 创作避险不是换词，而是彻底改变价值立场和主角目标。
- 如果必须涉及毒品，应站在缉毒、反毒、受害者救助或犯罪惩戒视角，而不是主角经营。

### 可迁移写法

涉毒红线检查表：

```text
作品标题：
主角职业：
是否涉毒：
是否把违法行为当事业：
是否用架空/穿越包装：
价值立场：
平台审核风险：
可否改为反毒视角：
必须删除的卖点：
替代冲突：
```

### 训练任务

检查自己的设定里是否有黄赌毒、真实违法产业、极端价值观等高危卖点。每个高危点只能保留在反派、受害者、调查或惩戒链条里，不能成为主角经营升级线。
