## 556：#公开处刑

人工查看范围：正文、全部 11 条评论、附件状态。附件 `简介和前三章.docx` 在导出目录未发现本地文件，原文无可下载链接，未能查看附件正文。无图片。

证据限制：无法查看附件前三章全文，本卡依据发帖者自述、大纲补充和评论拆解。

### 读者承诺

这篇诊断都市异能开篇“连续被追但缺爽点”的问题：作者三章里让主角不断被跟踪、解决麻烦、进局子、再被跟踪，但读者只记住“它追，他逃”。都市异能开篇不能只靠麻烦连续发生，要把世界奇异感、主角金手指、爽点、期待和主线因果写清楚。

### 核心机制

```text
主角被盯上 -> 第一波麻烦展示能力 -> 发现怪物异常 -> 主线反派露面 -> 正派介入善后 -> 主角因秘密被卷入
```

### 期待链与情绪

```text
读者看到主角遭遇追踪 -> 期待他展现能力和压制对手 -> 如果只追逃就乏味 -> 若玩具怪物和世界奇异感放大 -> 期待主线秘密
```

### 关键判断

- 发帖者自己承认前三章可能云里雾里，世界观、设定伏笔和金手指准备后面再讲。
- 当前大纲是回家被跟踪、解决麻烦进局子，出来又被跟踪，再解决麻烦，再被跟踪。
- 评论认为文字层面和打斗还可以，但环境、动作、人物语言有割裂感。
- 有评论指出人物脸谱化，反派没嚼头，建议放大世界奇异之处。
- 关键评价是看完三章只留下“它追，他逃”，故事没写出爽点、没拉起期待。
- 另一条评论指出打塑料玩具和新手混混场面太小，没有装逼，没劲。
- 第一波工具人如果只为展示金手指，必须让读者明确看到金手指多厉害，否则工具人没有价值。
- 如果前两章和后面主线关系弱，可以考虑直接用第三章作为第一章。

### 可迁移写法

都市异能追逃开篇表：

```text
第一场麻烦：
它和主线的关系：
主角展示的能力：
场面规模：
世界奇异感细节：
反派为什么追：
正派何时介入：
读者看到的爽点：
下一章期待：
```

### 训练任务

把一个“三次被追”的开篇压成一次强事件：只保留最能展示金手指的一次追击，补上世界奇异感、反派目的和主角压制对手的爽点。

## 557：问一下大家设定都是怎么写的

人工查看范围：正文、全部 6 条评论。无附件，无图片。

### 读者承诺

这篇回答“设定写多细”的问题：地图、门派、功法、等级和升级设定不用一开始全做成百科，但也不能粗到后期战力崩坏。更稳的方式是从核心矛盾推导设定：谁压迫谁、谁争夺什么、资源如何分配、反抗如何出现，再按剧情需要逐步补全。

### 核心机制

```text
确定核心矛盾 -> 推导阶层/组织/资源 -> 生成地图门派功法等级 -> 写作中按冲突补设定
```

### 期待链与情绪

```text
作者担心设定太粗会崩 -> 从矛盾出发搭框架 -> 设定服务压迫和争夺 -> 后续补细节更有方向
```

### 关键判断

- 发帖者卡在地图、门派、功法、等级、升级设定的细致程度。
- 他意识到提前设定能缓解后期战力崩坏。
- 也担心设定太细耗时，设定没写完就失去写作激情。
- 评论“矛盾大法好”指出设定应服务核心矛盾。
- 赛博朋克例子说明：底层争夺生存空间会自然推出帮派、中间人、赏金猎人；高层争夺商业利益会推出公司和垄断体系。
- 有压迫就有反抗，有反抗就有躺平者和被抛弃者，城市之外也会出现跟不上时代的人。
- 设定不是凭空列清单，而是从社会结构和冲突逻辑里长出来。

### 可迁移写法

矛盾推导设定表：

```text
题材：
核心矛盾：
被压迫者：
压迫者：
争夺资源：
组织形态：
地图分区：
力量体系：
升级限制：
需要边写边补的细节：
```

### 训练任务

给一个题材先写核心矛盾，不写设定名词。再从这个矛盾推导出 3 个组织、3 种资源、3 个地图区域和 3 条力量规则。

## 558：该怎么理解【情绪发泄】这个词

人工查看范围：正文、全部 13 条评论。无附件，无图片。

### 读者承诺

这篇解释“情绪发泄”不等于装逼打脸，也可以是情感线的满足与平息。导师之死把情绪拉起，主角接受冒险召唤、埋葬导师、告别家乡，分别解决自身阻碍、敌人阻碍和伙伴/家人阻碍，让读者感到这一段圆满落下，并对新旅程产生期待。

### 核心机制

```text
铺垫人物和目标 -> 意外死亡拉高情绪 -> 主角转变选择 -> 收尾仪式平息情绪 -> 新旅程开启期待
```

### 期待链与情绪

```text
主角拒绝冒险 -> 导师牺牲证明危险和责任 -> 主角接受召唤 -> 埋葬和告别让情绪落地 -> 读者获得满足和新期待
```

### 关键判断

- 发帖者疑惑：明明只是埋葬导师、告别家乡的日常，却让他感到畅快和爽。
- 星主用“拉皮筋”解释：事件和铺垫把皮筋拉起，关键情节释放后，收尾让动能渐渐平息。
- 发帖者进一步用目标—阻碍—结果理解：目标是接受冒险召唤。
- 主角自身阻碍是害怕危险而拒绝召唤。
- 朋友/家人阻碍是担心主角接受冒险。
- 敌人阻碍是导师死亡证明冒险确实危险。
- 主角口头接受只解决自身阻碍，告别家人解决伙伴阻碍，埋葬导师解决敌人/死亡带来的情绪阻碍。
- 评论补充这是满足感接期待感：上一段得到圆满句号，新一段给下一卷打广告。
- 这类桥段更考验笔力和人物刻画，难度高于简单装逼打脸。

### 可迁移写法

情绪发泄拉皮筋表：

```text
被拉起的情绪：
核心目标：
自身阻碍：
伙伴/家人阻碍：
敌人/事件阻碍：
关键释放点：
收尾仪式：
平息后的新期待：
```

### 训练任务

写一个非打脸的情绪释放单元：用一个死亡、分别或失败拉起情绪，再用 3 个收尾动作分别解决自身、伙伴、事件三个阻碍。

### 读者承诺


## 560：宝剑你好

人工查看范围：正文、全部 2 条评论。无附件，无图片。

### 读者承诺

这篇确认“身份反差”也可以算金手指：负债累累的人穿越到平行世界变成世界首富，本质上获得了资源、地位、信息差和行动权限。这种金手指不一定是系统或能力，也可以是身份和资源结构的突变。

### 核心机制

```text
原世界低谷身份 -> 穿越到高位身份 -> 资源/权限暴增 -> 处理新身份问题 -> 利用反差制造爽点
```

### 期待链与情绪

```text
读者看到主角负债累累 -> 穿越成世界首富 -> 期待他如何使用财富和权力 -> 新身份带来新危机和新爽点
```

### 关键判断

- 发帖者问穿越后的身份反差是否也可以定义为金手指。
- 例子是负债累累男子穿越到平行世界，在那里是世界首富。
- 星主回答“可以啊，都算”。
- 这说明金手指的定义不局限于系统、能力、神器，也包括身份、资源、地位和世界关系。
- 身份型金手指要写出前后落差，并让读者看到新身份能带来什么行动可能。
- 同时也要有新身份问题，否则只是设定变富，缺少剧情发动机。

### 可迁移写法

身份反差金手指表：

```text
穿越前身份：
穿越前困境：
穿越后身份：
新增资源：
新增权限：
新增危机：
第一件能爽的事：
第一件必须解决的事：
身份反差的长期玩法：
```

### 训练任务

设计一个非系统金手指：只改变主角身份和资源，不给超能力。写出穿越前后反差、第一件爽事和第一件危机。
