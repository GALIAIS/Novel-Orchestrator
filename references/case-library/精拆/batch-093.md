## 461：【宝剑侍从网文教程】文字版总纲（长期更新）

人工查看范围：知识库可见正文摘要、全部 48 条评论。无附件，无图片。

证据限制：正文为知识星球文章摘要，完整正文在外链文章中，知识库镜像只保留开头并以省略号截断；评论显示该帖是旧版总纲，后续有优化版和新置顶帖。以下拆解以可见正文和评论为依据。

### 读者承诺

这篇是教程体系的文字版总纲入口。它的核心价值不是让新人顺着链接无限套娃，而是把学习目标按分类统筹起来：以总纲作为主目录，按推荐顺序翻看，遇到重复链接时逐渐融会贯通；旧帖和缺失链接要服从新的置顶帖和更新版总纲。

### 核心机制

```text
总纲入口 -> 学习目标分类 -> 按顺序阅读 -> 遇到链接回到主目录 -> 重复内容融会贯通 -> 新版置顶替代旧版
```

### 期待链与情绪

```text
新人进入知识库 -> 被大量链接冲晕 -> 总纲提供主目录 -> 按顺序建立轮廓 -> 看多后理论互相印证 -> 从迷茫变成可执行学习
```

### 关键判断

- 正文说明星球内教程更具体、更系统，以文字版为主，并包含收费文章内容。
- 作者说明总纲会长期增删补全，教程厚度会持续增加。
- 评论中作者说明当时多数内容还空着，会逐步补。
- 多位新人反馈链接套链接，容易看迷糊、找不到顺序。
- 作者明确回复：以总纲为主要目录，它是主要学习目标分类。
- 另有回复说这就是推荐顺序，可以收藏后按顺序翻。
- 有些目录链接后来缺失或未统合汇总，后续会补。
- 评论指出有些链接重复；看多后能一眼扫过去，说明知识点会融会贯通。
- 旧版总纲后续要优化掉，新的置顶帖优先级更高。
- 对已写几十万字但成绩不好的人，也可以从头学，因为系统化理解“为什么这么想”很重要。
- 情绪链条等细项后来有独立汇总贴，说明总纲承担路由作用。

### 可迁移写法

教程总纲学习表：

```text

是否有新版置顶：

本轮只读的模块：
链接是否套娃：
回到主目录位置：
重复理论：

缺失/失效链接：
下一步学习目标：
```

### 训练任务

用总纲学习时，只开一个主目录。每次选一个分类读 3 篇，做 5 条笔记后回到主目录，不继续点套娃链接。读完一轮后写出“我现在最该训练的一个能力”。

## 462：#思考分析 这几年做自由职业者...

人工查看范围：正文、全部 16 条评论。无附件，无图片。

### 读者承诺

这篇讲自由职业写作者的现实低谷：自由职业不是轻松自由，而是收入不稳定、心态高压、需要自律和计划。连续扑街后，技巧进步不等于成绩必然进步；真正的恢复路径是重新找回写作热情，输入能触动自己的作品，再带着短板去拆书、拆人设、拆感情变化和核心梗。

### 核心机制

```text
全职/自由职业期待 -> 收入和成绩落差 -> 心态崩盘 -> 情感输入恢复热情 -> 回看失败作品 -> 针对短板拆书 -> 重建大纲能力
```

### 期待链与情绪

```text
作者以为一本比一本好 -> 新书成绩低于旧书 -> 失去热情和自信 -> 被喜欢的CP重新点燃 -> 找到问题和方向 -> 从0到1走完大纲
```

### 关键判断

- 自由职业不是时间自由，而是时间灵活；灵活要求更强自律和计划。
- 没有稳定收入，干了也未必有收入，是常态。
- 第一本成绩能维持，不代表第二本、第三本会更好。
- 第三本开篇留存更好，但后续拉胯，说明开篇局部进步不等于整书成功。
- 榜单涨幅低和长期轮空会严重冲击心态。
- 不知道问题在哪、怎么改，比单纯扑街更折磨。
- 心态崩掉后，断更、请假、失约、逃避读者期待都会出现。
- 创作者热情消亡会接近写作生涯消亡。
- 重新嗑到《祝卿好》的 CP 快乐，帮助作者恢复情绪输入。
- 女频感情爽点常需要情感敏锐度，单纯理性和资料输入未必够。
- 后续作者开始从章纲、逆推大纲、人设塑造、构建逻辑、感情变化、核心梗、同题材共性、心动点、CP 等维度拆书。
- 星主总结为曲折前行、螺旋上升，最重要是维持写作热情。

### 可迁移写法

自由写作低谷复盘表：

```text
当前收入状态：
成绩落差：
心态崩盘触发点：
是否断更/逃避：
还能触动我的作品：
情感输入方式：
失败作品问题：
需要拆的样本：
章纲/人设/核心梗短板：
重新出发目标：
```

### 训练任务

写一次低谷复盘。不要只写“我扑了”，必须列出成绩落差、心态反应、逃避行为、重新触动你的作品，以及下一本要拆的 3 个具体短板。

## 463：辰东的行文有着极强的个人风格，...

人工查看范围：正文、全部 9 条评论。无附件，无图片。

### 读者承诺

这篇分析辰东式热血短句的气势来源：短句长度集中在四到八字，末字常带爆破感，词汇常见但组合有疏离感，形成宏大、中二、可传播的名台词。它能提高热血情绪，但必须有前文铺垫，否则只会显得空喊中二。

### 核心机制

```text
短句节奏 -> 爆破音收尾 -> 常见词陌生组合 -> 宏大意象 -> 前文铺垫承托 -> 热血气势爆发
```

### 期待链与情绪

```text
读者被强意象句子击中 -> 感到气势和逼格 -> 句子成为梗传播 -> 前文铺垫足时热血上头 -> 铺垫不足则变成尴尬中二
```

### 关键判断

- 辰东行文有很强个人风格，很多句子适合做表情包传播。
- 例句包括“谁在称无敌，哪个敢言不败”“我为天帝，当镇压世间一切敌”等。
- 每个短句通常不超过十个字，更多集中在四到八字。
- 不少短句末字有爆破感，更有气势。
- 使用常见词，但组合出陌生疏离感，如边荒、天渊、真命。
- 评论认为气势是真足，主角战力未必顶尖，但气势和逼格是天花板。
- 辰东作品中常有可传播梗，如“我儿王腾有大帝之资”。
- 中期星空古路等段落，谁描写多谁显得牛逼，说明气势描写本身能抬位格。
- 这种短句能帮助写热血、煽动情绪。
- 中二表达需要前文铺垫；铺垫不够，读者只会觉得中二。
- 大神句子之所以成立，是因为气势水到渠成。

### 可迁移写法

热血短句气势表：

```text
要表达的位格：
四到八字短句：
爆破感收尾：
宏大意象：
常见词陌生组合：
前文铺垫：
说话人物资格：
读者应感到的情绪：
是否中二出戏：
可传播梗点：
```

### 训练任务

给一个高潮场景写 10 句热血短句。每句不超过 10 字，至少 5 句在 4 到 8 字之间。写完标注每句前文需要什么铺垫，否则删除。

## 464：本周上架首订，曹操穿越武大郎，...

人工查看范围：正文、全部 6 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-093-case-463\image-1.jpg`。

图片证据：图片为“2022-07-01 开通上架后 24 小时首订数据（起点收藏公众号）”表格，列出排名、书名、分类、作者、等级、字数、收藏、首订、收订比、男女频等字段。《国民法医》首订 12800，《猩红降临》10500，《开局我就想退出江湖》7450，《魏主》6200；《曹操穿越武大郎》历史分类，作者神枪老飞侠，约 29.38 万字，收藏 5.54 万，首订 4300，收订比 13。下方另有 6 月 27、28、29 日开通上架后 24 小时首订数据。

### 读者承诺

这篇体现的是上架首订榜单观察法：不要只看“破 3000”这个结论，还要看分类、收藏、首订、收订比、同期作品和数据来源。《曹操穿越武大郎》能在历史分类拿到 4300 左右首订，说明强设定同人式改命、公共认知角色和开篇卖点具有市场反馈，但具体数值需注意来源差异。

### 核心机制

```text
上架榜单 -> 看首订 -> 对照收藏和收订比 -> 比同期开书分类 -> 判断题材卖点反馈 -> 注意数据源差异
```

### 期待链与情绪

```text
作者关注样本上架 -> 看到首订破线 -> 回看题材和分类 -> 用数据验证卖点 -> 再决定是否学习该方向
```

### 关键判断

- 原帖指出《曹操穿越武大郎》本周上架首订不出意外破 3000。
- 图片表格显示该书首订约 4300，评论中也有人提到另一图看到 4900，说明数据源可能存在差异。
- 图片字段包括分类、作者等级、字数、收藏、首订和收订比，不能只看单一首订。
- 该书历史分类，收藏 5.54 万，首订 4300，收订比 13。
- 同期高首订作品包括都市、科幻、仙侠等多个分类。
- 评论提供起点收藏公众号和转载文章作为数据来源。
- 刺猬猫等其他站点也有独立首订数据观察入口。
- 对创作学习来说，榜单价值在于验证题材和卖点，而不是只围观数字。
- 《曹操穿越武大郎》此前在案例中多次作为同人改命、公共认知强代入样本出现，上架数据提供了市场侧反馈。

## 465：宝剑大大，我想问一问，日常究竟...

人工查看范围：正文、全部 3 条评论。无附件，无图片。

### 读者承诺

这篇回答日常该放在哪里、怎么发挥作用：日常不是“战况急转直下”式的新紧张，而是战斗爽完后的演出收尾。它可以清点战果、让别人震惊欢呼、互动暧昧、铺设世界观、人设、金手指、伏笔和新地点，最后自然过渡到下一次激励事件。

### 核心机制

```text
战斗高潮 -> 清点收获 -> 震惊欢呼 -> 人设互动/暧昧笑点 -> 伏笔和金手指准备 -> 过渡到新舞台 -> 下一激励事件
```

### 期待链与情绪

```text
读者经历高强度爽点 -> 需要确认战果 -> 通过欢呼和互动放松 -> 在日常中吃到笑点/甜点/伏笔 -> 慢慢转向下一剧情
```

### 关键判断

- “战况急转直下”不是舒缓剧情，而是让剧情更紧张。
- 日常常插在冲突结束后、主角清点收获时。
- 战斗爽完后，要先把战果写出来，让别人震惊欢呼。
- 清点收获本身也能带来爽点。
- 明确下一个目标后，再和其他角色互动一圈、暧昧一圈，读者心情会放松。
- 放松之后再去琢磨新的惊人意外。
- 日常不只是休闲，还承担剧情推进和伏笔铺设。
- 日常中可以混合爽点、笑点、泪点。
- 《大奉打更人》可拆成舞台准备、登台唱戏、演出收尾。
- 演出收尾到新舞台准备之间，是观察日常功能的重点。
- 日常可以铺垫不重要但增加真实感的世界观。
- 日常可以描写人设、写笑点、写爽点、介绍新人物。
- 日常可以给出金手指，为下个剧情解决问题做准备。
- 日常可以铺远期或近期伏笔，最后过渡到新剧情地点和激励事件。

### 可迁移写法

日常演出收尾表：

```text
上一场高潮：
战果清点：
他人震惊/欢呼：
主角收获：
人设互动：
笑点/甜点/泪点：
世界观补充：
金手指准备：
伏笔/悬念：
新地点过渡：
下一激励事件：
```

### 训练任务

给一个战斗高潮后写 1500 字日常收尾。必须包含战果清点、他人反应、人设互动、一个伏笔、一个下个剧情会用到的道具或能力，并自然过渡到新地点。
