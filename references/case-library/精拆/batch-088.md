## 436：提问：宝剑大大，我最近结合拆书...

人工查看范围：正文、全部 2 条评论。无附件，无图片。

### 读者承诺

这篇提出一套卷纲结构：一卷由三个关联密切的事件组成，每个事件先展露强敌压迫和大危机，再分化短期危机，主角解决短期危机后直面核心危机，经历伪胜利、意外下行，最后用金手指解决。价值在于框架能帮助新人保持期待递进，但真正难点在写作时根据剧情调整、填充爽点和安排支线舒缓。

### 核心机制

```text
卷首最终危机预告 -> 事件1短危机/核心危机 -> 伪胜利下行 -> 金手指解决 -> 事件2升级 -> 事件3卷终核心危机
```

### 期待链与情绪

```text
读者先看见实力悬殊 -> 等短期危机解决 -> 发现大危机仍在 -> 小高潮后意外下行 -> 金手指翻盘 -> 期待下一事件对手和收获升级
```

### 关键判断

- 该框架将一卷拆成三个关联密切的事件。
- 每个事件开场先展示主角与对手实力悬殊的大危机。
- 大危机会分化出一个短期危机，供主角先解决。
- 短期危机解决后，大危机仍未解除，主角继续直面核心危机。
- 中段可以写伪胜利小高潮，再用意外情节迅速下行。
- 最后让主角利用金手指真正解决危机。
- 三个事件结构相似，但对手实力逐层提高，收获也逐层提高。
- 每卷开头可以表露最终危机内容，让读者知道卷终压力。
- 一套未实践的结构只能算标准框架，真正重要的是按具体剧情调整和填充。
- 新手容易只顾解决矛盾，忽略爽点延伸展示。
- 主线太密集时，容易忽略用支线舒缓情绪。
- 该结构本身没有明显问题，但写出来一定要权衡修改。

### 可迁移写法

三事件卷纲表：

```text
卷终最终危机：
事件1对手：
事件1短期危机：
事件1伪胜利：
事件1意外下行：
事件1金手指翻盘：
事件1收获：
事件2升级点：
事件3卷终危机兑现：
爽点延伸展示：
支线舒缓章节：
```

### 训练任务

给自己的作品写一卷 3 事件卷纲。每个事件必须有短期危机、核心危机、伪胜利、下行、翻盘和收获；另外单独安排 2 个支线舒缓点，避免主线压迫过密。

## 437：#思考分析 #参考资料

人工查看范围：正文。原文无评论、无附件、无图片。

### 读者承诺

这篇不是剧情技法，而是写手长期生产的身体底盘：码字需要清晰头脑、手指和关节稳定、体能能在第二天恢复。运动不能过量，体重大不宜硬跑，训练后要拉伸和深呼吸，手指疲劳要照顾小臂肌群，受伤严重要遵医嘱。

### 核心机制

```text
身体状态评估 -> 合适运动量 -> 肌肉训练辅以有氧 -> 拉伸深呼吸恢复 -> 手指/小臂保护 -> 第二天保持清醒码字
```

### 期待链与情绪

```text
写手想提高产能 -> 先避免过劳和伤病 -> 体能稳定恢复 -> 码字时头脑清晰手部不痛 -> 长期输出更可持续
```

### 关键判断

- 写手需要关注身体健康，不能只谈技巧和字数。
- 过量有氧可能导致脂肪和糖分过少，激素和能量供给不足。
- 运动过量会带来精神萎靡、易怒，影响第二天写作。
- 更推荐符合自身水平的肌肉训练，辅以有氧。
- 体重过大时，跑步对关节伤害大，可优先考虑游泳。
- 每次运动后要花 5 到 10 分钟拉伸，并做深呼吸训练。
- 恢复不足会让写手第二天饱受折磨。
- 过度劳累可能成为失眠原因之一。
- 码字前可以做手指操，疲劳时也可放松。
- 手指实际受手臂肌肉群控制，因此也要按摩小臂。
- 韧带受损严重时必须遵守医生医嘱，不要自作聪明。
- 做好取舍，不要什么都要做，否则什么都做不好。

### 可迁移写法

写手身体维护表：

```text
当前体重/关节状态：
适合运动：
不适合运动：
每周训练量：
拉伸时间：
深呼吸训练：
码字前手指操：
小臂放松方式：
第二天精神状态：
必须遵医嘱的问题：
需要取舍的任务：
```

### 训练任务

制定一个 7 天写手健康计划。每天记录运动量、拉伸时间、睡眠、第二天精神状态和码字手部疲劳程度。目标不是练猛，而是让第二天码字更清醒。

## 438：这个意思就是只有作者懂政治，才...

人工查看范围：正文、全部 4 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-088-case-438\image-1.jpg`。

图片证据：图片为七猫中文网作者招募页面截图，列出免费加广告阅读模式，并在“成为作者的标准”中写到每天至少创作 6000 字、已完成开篇 2 万字和作品大纲、热爱文字阅读且有扎实文学基础、掌握历史/政治/地理/哲学/逻辑学等人文学科知识、正确世界观价值观并对社会有正向影响、丰富社会阅历和生活见解等。

### 读者承诺

这篇的核心不是让作者去写政治，而是提醒作者要理解平台风险和知识边界：平台会期待作者有基本人文常识、正确价值取向和社会认知，但实际过稿仍首先看开篇是否好看、表达是否清楚、是否踩线。对新人来说，知识储备是避坑能力，不能替代开篇质量。

### 核心机制

```text
平台作者标准 -> 人文知识/价值观要求 -> 写作避坑意识 -> 开篇质量优先 -> 签约后按平台反馈改稿
```

### 期待链与情绪

```text
新人看到高门槛标准 -> 担心必须懂政治才能写 -> 评论降低焦虑 -> 回到开篇好不好签 -> 同时保留常识和边界意识
```

### 关键判断

- 原帖问题是：是否只有作者懂政治，才能写出不涉政的东西。
- 图片显示平台作者标准包含历史、政治、地理、哲学、逻辑学等人文学科知识。
- 这些要求更像平台对作者综合素养和风险意识的期待。
- 评论认为开头写得妙，不懂政治也能签。
- 但“你不关心政治，政治会来关心你”说明边界意识仍重要。
- 平台过稿后未必轻松，评论提到七猫过稿后新人噩梦才开始，每天改稿。
- 对新人来说，开篇质量、清楚表达和不踩线是更直接的执行点。
- 人文知识可以帮助作者避免无意识写出敏感、低级或价值观错误的内容。
- 平台标准不能被理解成“必须写政治”，而应理解为“写现实和社会时要知道边界”。

### 可迁移写法

平台知识边界表：

```text
投稿平台：
平台公开作者标准：
作品是否涉及现实社会：
可能踩线内容：
需要补的常识：
开篇是否足够好看：
价值观表达是否清楚：
签约后改稿压力：
编辑反馈处理方式：
保留/删除的敏感设定：
```

### 训练任务

拿自己的开篇做一次平台边界检查。列出所有现实机构、政策、历史、地理、价值判断相关内容，标注哪些必须补常识，哪些可以虚化，哪些应删除。

## 439：『菠萝包轻小说废了』

人工查看范围：正文、全部 20 条评论、4 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-088-case-439\image-1.jpg` 到 `image-4.jpg`。

图片证据：图片为社区截图和聊天截图，核心信息包括“取消后宫标签”“不准后宫”“不能开后宫，不能追很多女人”“多个女生追一个男生但最后只选一个”等讨论；另有评论截图提到开放式结尾、多女主追男主不可行、没有标签后新人后宫小说更难被看到、部分旧书可能改名或下架。

证据限制：本案图片是社区/聊天截图，不是平台正式公告；评论中也有人提醒不要标题党，称编辑说等通知、目前写的书不受影响。以下拆解按“平台规则风险管理”处理，不把传闻当最终规则。

### 读者承诺

这篇体现的是后宫、多女主、百合等强规则敏感题材的平台风险：当平台标签、审核或主流方向变化时，作者不能只靠侥幸开书。正确动作是确认正式规则、判断新书是否以多伴侣为核心卖点，并准备单女主、无女主、暧昧添头、转平台或换题材的退路。

### 核心机制

```text
平台规则传闻 -> 核实正式通知 -> 判断题材核心卖点 -> 降低多伴侣风险 -> 调整标签/关系线 -> 准备退路
```

### 期待链与情绪

```text
作者听到后宫规则收紧 -> 担心题材死亡 -> 评论放大/修正风险 -> 回到新书能不能承受审核不确定 -> 选择更稳的关系结构
```

### 关键判断

- 原帖称菠萝包取消后宫区，不准有后宫结局，百合区改为女性主角。
- 图片讨论里有人认为取消后宫标签会倒逼“不可追多个女生、不可开后宫”。
- 有人认为主流方向可能变成多个女生追男生但最后只选一个。
- 评论提到番茄也可能不准多女主，见到会封书。
- 评论提到刺猬猫也有书被封，但也有人说是特殊时期，过几天恢复，畅销榜仍有后宫标签。
- 有人提醒不要标题党，称目前写的书不受影响，应等正式通知。
- 对作者来说，新书赌不起，不能只抱侥幸心理。
- 如果多女主只是添头，风险可能低于以多女主为核心卖点。
- 如果后宫标签消失，新人写后宫的可见度会下降。
- 有些旧书可能通过改书名、下架或调整标签处理。
- 关系线题材要把平台规则当成创作边界，而不是写完再碰运气。

## 440：一群人背井离乡，高层全死了，主...

人工查看范围：正文、全部 10 条评论。无附件，无图片。

### 读者承诺

这篇讨论的是困境群体里主角如何获得领导地位。关键不是突然被众人拥戴，而是主角数次化解危机、展示过人之处、建立群众基础和亲信基本盘，再通过服从性测试、排除异己、持续解决问题，让众人相信他最有能力带他们活下去。

### 核心机制

```text
群体失去高层 -> 主角解决初步困境 -> 声望上升 -> 拉拢亲信 -> 服从性测试 -> 排除异己 -> 持续解决问题 -> 成为领袖
```

### 期待链与情绪

```text
群体陷入无主恐慌 -> 主角第一次力挽狂澜 -> 群众开始依赖 -> 反对者挑战 -> 主角压住内乱 -> 更大危机中确立领导权
```

### 关键判断

- “高层全死、群体背井离乡”天然提供权力真空。
- 主角获得领导地位可以通过数次化解危机、力挽狂澜、逐步提高威望完成。
- 可设置几个只有主角能解决的难题，一步步打群众基础。
- 权力来源可以理解为服从性，服从性来源于别人对你力量的想象。
- 主角必须有过人之处：智力、体力、处理事务能力至少一项超常。
- 先帮助众人初步解决困境，但不要一下彻底解决。
- 初步建立地位后，要拉拢亲信，形成基本盘。
- 有基本盘后做服从性测试，不服者要被处理，用来排除异己和杀鸡儆猴。
- 即使无法马上解决问题，也要表现出自信和从容。
- 声望可以阶段化，每个阶段给对应奖励，声望满后多数人自然服气。
- 若想快节奏，可参考“超额奖励后被吹捧上领导位”的写法。
- 如果作者不擅长权力斗争，应降低复杂度，不要硬写深权谋。

### 可迁移写法

困境领袖上位表：

```text
群体原高层为何消失：
当前共同困境：
主角过人之处：
第一次只解决一半的问题：
群众基础来源：
亲信人选：
反对者：
服从性测试：
排除异己方式：
第二次更大危机：
主角最终领导权证明：
```
