### 读者承诺

这篇沉淀的是“高体质无敌”这种金手指如果只靠主角站着挨打、反派自己吃瘪，很快会从爽点变成重复和疲劳。作者自审已经很准确：前三章内容重复，都是女反派折磨男主、破不了防、反而吃瘪；困境第六章才爆杀解决；穿越前因和世界观伏笔占了第一章宝贵篇幅；体质金手指一动不动化解折磨，乍看是爽点，但主角太被动，缺乏魅力和张力；体质数值缺少直观成长性；女反派疯癫、女性开头容易被误认为女主，削弱可恨程度。长图正文也能看到这些问题：伤害提示不断出现，女法师用各种液体、器皿、怪物、血蛭等实验，主角凭 999 体质抗住，甚至吐血也更多是被动承受；系统属性表和游戏设定占了不少解释空间。这个设定真正该做的是把“防御强”转成主动事件能力：反杀、碰瓷、硬闯、替人挡灾、诱敌误判、用身体破局。否则读者会很快知道他不会死，也知道敌人会失败，期待就塌了。

### 核心机制

```text
高体质金手指 -> 被动挨打不破防 -> 反派重复吃瘪 -> 张力衰减 -> 转成主动破局和成长用法
```

被动无敌只能用来首秀，不能连续三章当同一种桥段重复。

### 期待链与情绪

```text
读者看到虚弱男主被折磨 -> 期待金手指反转 -> 第一次不破防有爽感 -> 连续重复失去紧张 -> 需要主角主动爆杀或换玩法
```

无敌流也要有选择、误判和代价，否则只有数值没有主角魅力。

### 关键判断

- 作者自审本身很到位。
- 前三章重复同一桥段会快速疲劳。
- 穿越前因和世界观伏笔不该占第一章核心位置。
- 困境解决太晚会压低爽感。
- 高体质一动不动抗伤害，容易让主角失去能动性。
- 被动爽点缺乏持续张力。
- 单一数值缺少可预见成长。
- 女反派开头出现容易被读者误判为女主。
- 疯癫反派若不可恨，只会变成噪音。
- 奇幻背景里血腥折磨冲击力会被稀释。
- 现代背景的连续杀人狂桥段，换到奇幻未必同等有效。
- 主角名字读感也会影响代入。

## 1712：关于“公开处刑”不知能否提几点...


人工查看范围：正文、全部 3 条评论。无附件、无图片、无外链。

证据限制：这是公开处刑流程建议帖，不是具体稿件；本卡只拆“最终稿提交、疑问清单和集中修改”的评审机制。

### 读者承诺

这篇沉淀的是公开处刑不是随手扔草稿让别人替你发现基础错误，而是拿当前能力上限的最终修改版，附上作者自己的疑问、思路和取舍，让评审集中处理作者真正想不通的问题。发帖者提出两点：第一，公开小说最好是最终修改版本。未经多次修改的版本会有错字、病句、标点错误、人物行为逻辑混乱、设定不明、篇幅占比异常等问题，这些作者自查一遍就能发现。书名、简介、小纲也要修改到当前能力极限，因为这是面对读者和编辑的敲门砖。第二，公开小说要附作者疑问、考虑和想法。很多小纲 18 元素没勾上、人设逻辑剧情明显有问题，作者却不提问，等别人反馈，会消耗读者时间，也消耗作者耐心。更好的方式不是“请毒舌锐评”，而是列出 1、2、3 个具体疑问，说明自己的思路哪里卡住。这样能避免“修改-反馈-修改”循环，集中问题大改。评论也指出，有些稿件像写着玩，公开处刑门槛不高，但作者必须主动。

### 核心机制

```text
草稿自查 -> 修改到当前上限 -> 附书名简介小纲和疑问 -> 集中评审核心问题 -> 大改复盘成方法
```

公开处刑的价值不在被别人改一遍，而在暴露自己思路上想不到的盲点。

### 期待链与情绪

```text
作者求评 -> 先自查基础错误 -> 提交最终版和问题清单 -> 星友集中排雷 -> 作者形成自己的方法
```

读者和评文者更愿意回应认真准备过的问题。

### 关键判断

- 公开处刑最好提交最终修改版。
- 错字、病句、标点应先自查。
- 人物逻辑、设定、篇幅占比也要先用读者视角审一遍。
- 书名、简介、小纲是敲门砖。
- 最好不是钻牛角尖，而是当前能力上限。
- 提交时要附作者疑问和想法。
- 18 元素没勾上却不提问，会浪费反馈机会。
- 不要只写“毒舌锐评”。
- 用 1、2、3 列出具体问题。
- 集中大改比反复小修求评更省时间。
- 思考、总结、成文是作者赚钱法宝。
- 作者不主动，没人能真正帮忙。

### 可迁移写法

```text
提交版本是否最终稿：
自查过的基础错误：
书名：
简介：
小纲：
18元素缺口：
作者自己的判断：
最想问的3个问题：
希望评审聚焦处：
下一轮集中大改范围：
```

### 训练任务

为一次公开处刑准备提交包：最终稿、书名简介、小纲、18 元素自检表、3 个具体疑问。禁止只写“求锐评”。

## 1713：求书推，男频小说。


人工查看范围：正文、全部 20 条评论、微信读书搜索外链标记。外链《斗破苍穹》《不败战神》均可访问，HTTP 200，content-type 为 text/html，大小约 2656 bytes；只作为书名指向，未展开书籍全文。

证据限制：这是求书和拆书取样讨论帖，不是具体作品拆解；本卡只拆“少年感取样应聚焦同类，同时保留跨媒介长期套路参考”的方法。

### 读者承诺

这篇沉淀的是新人找“少年感”样本时，要先盯准自己要写的类型和感觉深挖，不要一开始贪多胡学；但成熟以后，可以从畅销榜和热血动画里补充更大范围的商业套路。发帖者想找男频少年感小说，例子是《斗破苍穹》萧炎“三十年河东，三十年河西，苍天不欺少年穷”的少年意气。评论有两条路线：一条是“扫榜”和看畅销榜大佬，不限类型找启发；另一条提醒新人最好盯着一种类型死薅，喜欢什么类型、什么感觉，写起来更顺手，贪多胡学容易样样稀松。推荐样本包括《不败战神》、江南《刺客王朝》、夜的命名术、穹顶之上，以及火影、七龙珠、灌篮高手、钻石王牌、排球少年等热血或运动番。关键分辨是：少年感不是单一题材，而是一组情绪结构，包含不服输、受辱后立志、遍体鳞伤仍坚持、伙伴/双雄/主角团、成长中的莽撞和赤诚。网文作者可以学长剧情套路和人物精神，但短期桥段处理要考虑载体不同，不能直接照搬动漫节奏。

### 核心机制

```text
锁定少年感 -> 先深挖同类型样本 -> 提炼不服输/热血/伙伴结构 -> 再跨媒介补套路 -> 回到网文章节节奏
```

新人拆书的第一优先级是写得出来，而不是什么都看。

### 期待链与情绪

```text
读者看到少年主角受压 -> 期待不服输立志 -> 经历训练和挫败 -> 伙伴或双雄互相点燃 -> 爆发兑现少年意气
```

少年感的核心是“我还弱，但我不认命”。

### 关键判断

- 少年感可来自单主角、双雄或主角团。
- 《斗破苍穹》的退婚立志是典型少年意气样本。
- 新人找样本应先扒同类型。
- 扫畅销榜能开眼，但不要一开始贪多。
- 喜欢的类型更容易写顺。
- 不同类型也可能提供灵感，但要有筛选能力。
- 热血动画有成熟商业套路。
- 动漫长线套路可学，短期桥段要谨慎迁移。
- 运动番常有强少年感。
- 少年感常与热血、不服输、遍体鳞伤仍坚持绑定。
- 拆样本要提炼情绪结构，不只抄桥段。

### 可迁移写法

```text
目标少年感类型：
同类型样本：
主角受压点：
不服输宣言：
伙伴/双雄关系：
训练或挫败：
第一次爆发：
可跨媒介参考：
不能照搬的载体差异：
```

### 训练任务

选 3 个少年感样本，每个只拆 5 项：受辱点、不服输动作、伙伴反馈、挫败代价、第一次爆发。再把它们改写成同一网文章节节奏。

## 1714：话说，番茄是怎么算稿费来着，我...


人工查看范围：正文、全部 15 条评论。无附件、无图片、无外链。

证据限制：这是 2022 年番茄稿费讨论，平台规则和算法可能已经变化；本卡只拆原帖中“在读、题材、完读、留存和单价错觉”的判断模型，不当作当前稿费规则。

### 读者承诺

这篇沉淀的是番茄稿费不能简单按“一万在读等于固定月入”理解，不同题材、留存、完读、字数和平台算法会造成单价错觉。发帖者听到不同网文 UP 给出不同说法，有人说一万在读月入一百，有人说三百，所以求证。评论给出几种解释：每个题材不一样，读者群体越少可能单价越高，传统武侠一万在读一天可能 200，同人可能 50；但也有人认为单价本身可能差不多，差别来自读者留存、完读等相关数据，题材粘性高就显得单价高；还有人说看完读和题材，越原创单价越好；小众文阅读量小但单价高，热门题材阅读量大但单价低，最后稿费可能差不多。这个帖子的实用点是：作者不能只拿在读排名推算收入，也不能只追所谓高单价题材。更稳的判断是，题材决定流量上限和粘性，完读留存决定广告或分发收益表现，字数和更新决定持续曝光。收入模型要看“在读量 x 留存质量 x 题材单价/算法权重 x 字数阶段”，而不是单点数字。

### 核心机制

```text
看到在读数字 -> 想换算稿费 -> 题材/完读/留存/字数影响收益 -> 固定单价失效 -> 用综合数据判断
```

在读是入口，不是收入公式本身。

### 期待链与情绪

```text
作者看见火书在读 -> 想象月入 -> 听到单价差异 -> 发现题材和留存影响大 -> 回到作品数据质量
```

真正有用的不是传闻单价，而是自己作品在目标题材里的完读和留存。

### 关键判断

- 一万在读不等于固定收入。
- 题材会影响收益表现。
- 小众题材可能单价高，但在读难做大。
- 热门题材在读容易高，但单价可能低。
- 同人常被认为单价较低。
- 原创和高留存题材可能表现更好。
- 完读和留存会制造单价差异。
- 字数上去后收入表现可能变化。
- 不要只按在读排名推算稿费。
- 传闻数字只能参考，不能当决策依据。
- 选题要同时看流量上限和留存质量。

### 可迁移写法

```text
目标题材：
在读预期：
完读/留存预期：
题材粘性：
同类收入传闻：
字数阶段：
真实后台数据：
是否值得换题材：
风险：单价高但流量小/流量大但单价低
```

### 训练任务

选三个题材，分别写“流量上限、留存可能、单价传闻、最大风险”。最后只用风险最小的一项做选题，不用最高传闻收入做选题。

## 1715：所谓文笔，是不是就是一段四五十...


人工查看范围：正文、全部 11 条评论。无附件、无图片、无外链。

证据限制：这是概念讨论帖，不含具体文本；本卡只拆“文笔、笔力、节奏和读者情绪”的边界。

### 读者承诺

这篇沉淀的是文笔不是单纯把四五十字细纲扩成三千字，而是把具体内容呈现得更清楚、更流畅、更有画面，同时不破坏小说节奏和读者情绪。发帖者提出：所谓文笔是不是把一段细纲扩成 3000 字供读者阅读，且读者读得津津有味。评论给出不同层级：文笔是正儿八经的文字描写功力；文字流畅、描述清晰、有画面感、场景立体；长短句交替熟练运用；除了遣词造句，还要能控制节奏，不能像散文一样寡淡如水，调动读者情绪才是赚钱法门。另一个关键区分是文笔和笔力：文笔更接近表面呈现，把具体内容呈现得更好；笔力是综合能力，文笔是能力表现。也就是说，文笔不是“扩写字数”，而是读者能直观看到的表达质量；笔力则包含节奏、情绪、结构、人物目的和行动，把画面情节串起来。网文写作不能把文笔训练成漂亮散文，必须服务情节节奏和读者情绪。

### 核心机制

```text
细纲内容 -> 清晰描写 -> 画面立体 -> 长短句和节奏控制 -> 调动读者情绪
```

扩写只是字数变化，文笔要让内容更好读，笔力要让故事更能牵动人。

### 期待链与情绪

```text
读者进入场景 -> 看懂人物目的和行动 -> 感到画面清楚 -> 被节奏推着读 -> 情绪被调动
```

好文笔不抢故事，而是让故事更顺、更清楚、更有情绪。

### 关键判断

- 文笔不是简单扩写细纲。
- 文字描写功力是文笔基础。
- 流畅、清晰、画面感、立体场景是可见标准。
- 长短句交替能改善阅读节奏。
- 遣词造句不能脱离情节节奏。
- 小说写成散文可能寡淡。
- 节奏跟情绪有关。
- 调动读者情绪才是赚钱法门。
- 文笔更接近表面呈现。
- 笔力是综合能力。
- 文笔是读者能直观看到的东西。
- 人物目的和行动能把画面情节串起来。

### 可迁移写法

```text
细纲一句话：
人物目的：
人物行动：
场景画面：
读者需要感到的情绪：
长句位置：
短句位置：
节奏转折：
是否像散文拖慢剧情：
```

### 训练任务

把 50 字细纲扩成 500 字，不追求辞藻，只检查四点：读者是否看懂目的、画面是否清楚、句式是否有节奏、情绪是否被推进。
