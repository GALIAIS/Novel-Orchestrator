## 201：#思考分析

人工查看范围：正文、全部 2 条评论。重点查看作者把刚发生的家庭冲突拆成双向不公平伤害，并意识到亲身经历可作为创作催化剂。

### 读者承诺

这篇的价值在于把生活小事拆成戏剧冲突：母亲忘拿钥匙、敲门打电话没人应，她觉得自己受了委屈；作者戴耳机、手机静音、没听见，被一开门就数落，也觉得自己受了委屈。双方都认为自己是“不公平的一方”，谁都有理、谁都不服，冲突因此爆发。

### 核心机制

```text
双方误差 -> 各自认为受伤 -> 情绪上行 -> 语言反击 -> 冲突爆发
```

### 期待链与情绪

```text
读者看到敲门误会 -> 理解双方理由 -> 等待谁先爆发 -> 对骂升级 -> 摔门收束
```

### 关键判断

- 亲身经历是很有效的创作催化剂。
- 不公平伤害不一定只有一方成立。
- 双方都觉得自己有理时，冲突更真实。
- 冲突常来自信息不对称：一方不知道对方为什么没回应，另一方不知道对方为何急。
- 生活冲突里，客观责任和主观感受会错位。
- 情绪爆发前要有过程：联系不上、敲门、误判、开门、数落。
- 对骂不是凭空发生，而是双方都认为对方不讲理。
- 生活素材需要下一步艺术加工。
- 真实事件不能原样搬进小说，必须提炼目标、阻碍、升级点。
- 双向不公平伤害适合写家庭、恋爱、同伴误会、阵营分歧。
- 结尾的摔门是冲突暂时收束，不是问题真正解决。
- 这种结构可迁移为“双方都站得住”的争执桥段。

### 可迁移写法

双向不公平伤害表：

```text
事件误差：
A 方认为自己受伤：
B 方认为自己受伤：
双方不知道的信息：
第一句引爆台词：
反击理由：
升级动作：
暂时收束：
后续修复/恶化方向：
```

### 训练任务

写一个 800 字争执桥段：双方都必须有合理委屈，不能把任何一方写成纯粹无理取闹。

## 202：#公开处刑

人工查看范围：正文、全部 4 条评论、附件状态。附件 `晓月魔塔.docx` 尝试下载返回 401；本卡依据正文与评论精拆。

证据限制：附件正文不可见；评论已指出书名简介谜语人、主线/金手指/身世不清、开篇爽点缺失、画面感弱、三章内金手指/人设/冲突/期待不达标等关键问题。

### 读者承诺

这篇是“模板自查找不出问题，但仍然签约失败”的反面教材：作者设想是监狱进入异世界打怪物，混合石之海和规则怪谈，但评论集中认为它不是网文写法。读者进来不知道书名意思、不知道主线、不知道金手指、不知道主角身世，开篇又没有爽点，直接劝退。

### 核心机制

```text
谜语人包装 -> 读者不知道看什么 -> 开篇爽点缺失 -> 主角弱而不讨喜 -> 签约与留存失败
```

### 期待链与情绪

```text
读者看书名简介 -> 没得到承诺 -> 进正文仍看不懂主线 -> 三章元素不达标 -> 放弃
```

### 关键判断

- 模板检查不出问题，不代表开篇有效。
- 监狱、异世界、怪物、规则怪谈混合题材本身难签约。
- 这类路子对故事成品要求很高。
- 书名和简介不能谜语人。
- 读者要知道主线是什么。
- 读者要知道书名大概是什么意思。
- 金手指、主角身世、核心卖点不能全藏。
- 开篇没有爽点会很劝退。
- 画面感弱会让小说像口头讲故事。
- 叙述太口语化、文字不连贯，会造成画面断层。
- 三章内金手指、人设建立、冲突、期待感都要达标。
- 懦弱、惊恐、歇斯底里的主角容易让网文读者讨厌。
- 现实高中生的真实反应，不等于网文主角应有反应。
- 读者想看强大、勇敢、狡黠或智慧的主角。
- 对热门书的模仿要先学网文写法，不要只学设定壳。

### 可迁移写法

谜语人开篇排雷表：

```text
书名承诺：
简介是否说清主线：
前三章金手指：
前三章人设：
前三章冲突：
前三章爽点：
读者已知信息：
读者仍不知道的信息：
主角是否讨喜：
画面感问题：
```

### 训练任务

把一个谜语人简介改成直给版：必须写清主角是谁、要去哪、靠什么活、第一章能看到什么爽点。

## 203：#参考资料

人工查看范围：正文、附件状态。附件 `一个公式拯救一个故事.pdf` 只有 file id，未提供可下载链接；本卡依据正文精拆。

证据限制：PDF 附件不可见，未能查看思维导图具体内容；正文说明该资料来自 B 站视频笔记，主题是起承转合和剧情卡顿。

### 读者承诺

这篇的核心是把外部视频学习转成自己的思维导图：起承转合和剧情卡顿不是只靠记概念，而是要整理成“卡住时能查”的公式化工具。对新人来说，资料的价值不在原视频本身，而在把它转化为自己的故障排查表。

### 核心机制

```text
B 站学习 -> 起承转合整理 -> 剧情卡顿问题 -> 思维导图 -> 新人排障工具
```

### 期待链与情绪

```text
新人剧情卡住 -> 查起承转合导图 -> 找到缺失环节 -> 补事件/转折/兑现 -> 继续写
```

### 关键判断

- 视频学习要转成可查的创作工具。
- 起承转合适合用来排查剧情卡顿。
- “公式”不是为了机械套文，而是在没思路时定位问题。
- 思维导图比散乱笔记更适合新人复习。
- 导图可以继续扩充，不应一次定死。
- 起承转合至少要回答：起什么事、承什么目标、转什么压力、合什么结果。
- 剧情卡顿常常是某个环节缺行动、缺阻碍或缺反馈。
- 资料共享要说明适用对象和使用场景。
- 没有具体项目时，公式只能提供方向。
- 新人使用公式后仍要回到读者期待和情绪变化。

### 可迁移写法

剧情卡顿排查表：

```text
当前卡住位置：
起：事件是否启动：
承：目标和行动是否明确：
转：阻碍和代价是否升级：
合：结果和反馈是否兑现：
卡顿原因：
补救动作：
下一章钩子：
```

### 训练任务

拿一个卡住的剧情，用起承转合四栏重写：每栏必须写出一个具体行动，不能只写设定说明。

## 204：分享一本书《天下刀宗》，不推荐...

人工查看范围：正文、4 张图片。图片 1、2、4 为《天下刀宗》第一章片段，展示高水准武侠文笔、意境和人物出场；图片 3 为微信读书书籍信息，推荐值 70.9%，24.6 万字。

### 读者承诺

这篇提供了一个很反常的样本：文笔一流，讲述能力强，能把不是故事的东西讲得像故事，读者阅读时欲罢不能，但读完后会骂。它说明“文笔”和“故事满足”不是同一件事，强语言可以让人读下去，却不一定给读者合格的故事回报。

### 核心机制

```text
一流文笔 -> 阅读黏性 -> 故事目标模糊/回报不足 -> 读后反噬 -> 只适合拆语言不适合学网文结构
```

### 期待链与情绪

```text
读者被文笔吸引 -> 继续追随叙述 -> 等待故事闭合 -> 回报不足 -> 欲罢不能后骂娘
```

### 关键判断

- 《天下刀宗》文笔水准很高。
- 作者讲故事能力强，能把非故事讲成故事。
- 但发帖者认为它不是合格故事。
- 阅读时欲罢不能，不代表读后满足。
- 文笔强可以制造黏性，但不能替代故事闭合。
- 豆瓣获奖、武侠扛鼎之作等评价不等于适合网文学习。
- 图片片段显示语言有意境、节奏和古典感。
- 这种书可拆语言、氛围、句式，不宜直接拆网文爽点。
- 如果作品持续制造窥探感却不给清晰回报，会造成读后反噬。
- 文学性强的作品未必适合新人仿写。
- 新人学这类文笔容易牺牲剧情推进。
- 推荐值和奖项不能替代自己的阅读目标。

### 可迁移写法

文笔强故事弱评估表：

```text
语言优势：
氛围优势：
读者被吸住的原因：
主角目标是否清晰：
故事回报是否明确：
读后满足/反噬：
可学部分：
不可学部分：
是否适合当前网文目标：
```

### 训练任务

选一个文笔强样本，只拆 500 字语言：句式、意象、节奏各摘 3 点；同时写明为什么不直接仿它的故事结构。

## 205：#思考分析

人工查看范围：正文。重点查看作者通宵写书时对“套路框架不必硬套”的体会，以及方法和自审的边界。

### 读者承诺

这篇讲的是方法用熟后的状态：套路和公式是在没思路、纠正问题、训练基本功时用的；当写得顺、情绪拉到位、自己能判断文本真好还是假好时，就不必硬套框架。关键不是反套路，而是有自审能力后敢写。

### 核心机制

```text
学习套路 -> 实践用熟 -> 自审能力形成 -> 顺手时脱稿 -> 好看优先
```

### 期待链与情绪

```text
作者通宵写作 -> 灵感顺畅 -> 不再逐项套公式 -> 情绪自然到位 -> 形成创作信心
```

### 关键判断

- 实践出真知。
- 简单套路用熟后，不必每次硬套。
- 能用则用，不适合不要强塞。
- 自己觉得通顺，情绪拉到位，是重要判断。
- 看得精彩时，不必循规蹈矩。
- 方法和公式适合没思路时使用。
- 方法和公式也适合纠正问题时使用。
- 顺手时没必要细究方法对不对。
- 真正关键是能判断文本到底是真好，还是看不出问题。
- 有自审能力后，可以大胆写。
- “文过千章，其意自会”强调大量实践后的内化。
- 这不是否定理论，而是理论内化后的使用方式。

### 可迁移写法

套路内化自审表：

```text
当前写作状态：卡住/顺手
是否需要套公式：
情绪是否到位：
读者期待是否清晰：
自审发现的问题：
是否只是自嗨：
需要回到哪个工具：
可以放开的部分：
```

### 训练任务

写一章后做两遍检查：第一遍不看公式只读流畅度和情绪，第二遍再用公式查缺口；比较两次结果，判断哪些方法已经内化。
