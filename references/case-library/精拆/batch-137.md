## 681：刚刚看见一哥们在谈《起风了》主...

人工查看范围：正文、全部 1 条评论、图片 2 张。无附件。

图片证据：两张图片均为表情图，一张写“你的脑残表现让我耳目一新，不错，我欣赏你！”，另一张是尴尬捂头表情。它们强化了作者对“贫民窟给小女孩大额金币”这一桥段的反智感受。

### 读者承诺

这篇拆的是“善良写成反智”的毒点。作者原本可能想写主角心地善良，看到贫民窟小女孩可怜就给大额金币，但在贫民窟环境下，这等于把手无缚鸡之力的小孩推入危险。读者看到的不是善良，而是没脑子的傻白甜圣母。

### 核心机制

```text
主角想施善 -> 不考虑环境风险 -> 给弱者大额钱财 -> 弱者暴露危险 -> 读者骂主角反智
```

### 期待链与情绪

```text
读者期待主角善良可靠 -> 主角出手帮助小女孩 -> 帮助方式制造更大风险 -> 读者从感动转为愤怒 -> 善良人设崩成圣母
```

情绪关键不是“给得多不多”，而是作者忽略了具体场景逻辑。评论给出高级写法：让配角犯这个错，造成伤害，再由主角解救或教育配角，锅给配角，功给主角。

### 关键判断

- 善良桥段必须考虑受助者处境。
- 在贫民窟给弱小者大额钱财，会制造抢夺和伤害风险。
- 主角不能用愚蠢方式做好事，否则善良变圣母。
- 同一桥段若让配角犯错，主角补救，就能转成主角高光。
- 更稳的帮助方式是给食物、庇护、路线、身份保护等低风险资源。

### 可迁移写法

善良避毒检查表：

```text
受助者处境：
主角给出的帮助：
帮助是否暴露受助者：
弱者能否守住资源：
环境中谁会抢夺：
读者会觉得主角善良还是反智：
能否改成配角犯错：
主角如何补救并拿功：
```

### 训练任务

写一个 500 字施善桥段：主角在贫民窟帮助小孩，禁止直接给大额钱财。必须写出主角如何判断环境风险、选择低风险帮助方式，并留下后续回报或关系钩子。

## 682：#思考分析

人工查看范围：正文、全部 5 条评论。无附件，无图片。

### 读者承诺

这篇收集大量影视剧常见套路，并讨论它们如何迁移到小说。表面是老套桥段清单，核心价值在评论：经典套路可以反着写，用读者已有预期制造反差，让剧情思路打开。

### 核心机制

```text
读者熟悉老套路 -> 作者识别固定预期 -> 反向改写关键动作 -> 产生新情绪/反差 -> 老桥段获得新用途
```

### 期待链与情绪

```text
读者看到熟悉触发器 -> 预判下一步套路 -> 作者反写 -> 读者获得惊讶、荒诞或更深情绪 -> 预期被重新利用
```

评论举例：听到噩耗，不写碗掉地上，而写平静扒饭，表达哀莫大于心死；遭遇突变冲到外面，不写雷雨，而写外面欢声笑语，表现人与人的悲欢并不相通。

### 关键判断

- 套路不是不能用，关键是知道它触发了什么预期。
- 反写不是随机颠倒，而是换出更准确的情绪。
- 老桥段可作为读者预期按钮。
- 反写要服务人物状态和主题，不只是搞笑。
- 经典套路清单也可转成训练素材库。

### 可迁移写法

套路反写表：

```text
原套路：
读者默认预期：
原套路表达的情绪：
反写动作：
反写后表达的情绪：
适用人物状态：
是否只是恶搞：
```

### 训练任务

从影视套路中选 5 条，每条写一个反写版本。要求每条都写清：原套路表达什么，反写后表达什么，为什么反写更适合当前人物。

## 683：看到宝剑大锐评，突然想到一件事...

人工查看范围：正文、全部 1 条评论。无附件，无图片。

### 读者承诺

这篇把网文卷级爽感拆成“期待感、畅快感、满足感”。作者认为网文脉络不是抽象主题或核心矛盾本身，而是主角对谁装了什么逼，让读者感觉爽。主题和思想辩证都应服务于具体装逼项目。

### 核心机制

```text
装逼清单 -> 读者期待 -> 小兑现带来畅快 -> 期待与享受交互 -> 卷末大兑现 -> 战利品/成长满足
```

### 期待链与情绪

```text
读者知道主角要装什么逼 -> 清单逐步落实 -> 每次兑现强化下一轮期待 -> 最大事件兑现 -> 主角获得明确收益
```

情绪关键在“坑要诱人”。只是挖坑不够，坑要像“某王埋下宝藏的土地”，让读者愿意等。只画大饼不发糖，会把读者冷走。

### 关键判断

- 网文卷设计要有清晰装逼清单。
- 期待和享受要交互，不宜只积压不兑现。
- 新人更适合量变产生质变，定期给甜头。
- 满足感常被忽略：装完逼后主角必须得到东西。
- 收益可以是外物、内在成长、情感、人际关系。

### 可迁移写法

装逼清单卷纲表：

```text
本卷最大装逼：
装逼对象：
读者为何期待：
小兑现 1：
小兑现 2：
卷末大兑现：
主角获得的外物：
主角获得的成长：
关系变化：
下一卷期待：
```

### 训练任务

写一个 300-500 字卷纲：包含一个诱人的装逼清单、至少两次小兑现、一次卷末大兑现，以及大兑现后主角获得的明确收益。

## 684：提问：宝剑大大，各位球友大大，...

人工查看范围：正文、全部 25 条评论。无附件，无图片。

### 读者承诺

这篇讨论主角暴走被 BOSS 控制后误杀无辜者是否毒。评论共识是：毒，最好前置改设定，不要事后洗白。主角可以暴走，但伤害对象必须有责任边界，不能让主角杀掉本该拯救的受害者后再硬圆。

### 核心机制

```text
主角被控制 -> 暴走杀本可救者 -> 理念自相矛盾 -> 读者判定伪善/弱 -> 改设定避毒
```

### 期待链与情绪

```text
反派控制他人作恶 -> 主角应救人或打反派 -> 暴走力量出现 -> 读者期待怒火指向真正恶人 -> 若误杀无辜则爽感转债务
```

更稳的情绪链是：喽啰改成死有余辜、不可救或已死亡；BOSS 以为控制主角，主角保留意识或因正义感主动爆发，反过来炸反派基地、击杀 BOSS。

### 关键判断

- “洗白”不如“别弄脏”。
- 一次性工具人属性越简单越好。
- 杀戮爽点的目标必须合法化。
- 影视特摄暴走有画面兜底，文字里更需要逻辑和责任边界。
- 主角可以失控，但最好保留后手、底线或主动动因。
- 暴走桥段应凸显主角人设和反派可恨，而不是压主角。

### 可迁移写法

暴走责任边界表：

```text
暴走原因：
主角是否保留意识：
伤害对象：
对象是否无辜：
对象是否可救：
读者会不会替主角背道德债：
反派责任是否清楚：
暴走后获得什么能力/成长：
是否需要改成主动愤怒：
```

### 训练任务

写同一桥段的三版小纲：毒点版、改设定版、改动因版。每版 300 字以内，最后标出哪一版最不容易让读者觉得主角伪善。

## 685：话说刺猬猫分A、B、C、D签，...

人工查看范围：正文、全部 1 条评论。无附件，无图片。

证据限制：评论者说明信息来自作者群大佬、问编辑和个人总结，未提供官方规则截图；本卡只按经验性口径记录，不当作官方标准。

### 读者承诺

这篇解释刺猬猫 A、B、C、D 签的大致认知：A 是神签，面向少数畅销作或天才；B 是精品签，大约 3000 均订以上；C 是潜力签，可能有 1000 均订潜力或责编看好；D 是鼓励新人、有一点商业价值、填充书库的扑街签。

### 核心机制

```text
签约等级 -> 当前成绩/潜力判断 -> 编辑认可 -> 平台商业价值 -> 作者定位预期
```

### 期待链与情绪

```text
作者听说签约分级 -> 想知道自己处于什么水平 -> 评论拆 A/B/C/D -> 新人理解等级差异和预期
```

情绪是好奇和焦虑。等级信息能帮助新人判断自己位置，但因为不是官方规则，不能机械套用。

### 关键判断

- A/B 偏高成绩或强作者。
- C 偏责编看好和潜力判断。
- D 不是完全无价值，而是新人鼓励和基础商业价值。
- 等级不只看当前均订，也看平台判断。
- 经验信息要标注来源不确定，避免当官方政策传播。

### 可迁移写法

平台等级说明表：

```text
等级：
经验标签：
大致成绩/潜力：
编辑判断：
商业价值：
新人应如何理解：
是否官方规则：
```

### 训练任务

把刺猬猫 A/B/C/D 签写成 800 字以内新人 FAQ。要求只基于可见评论，逐级解释含义，并明确提示这不是官方规则。
