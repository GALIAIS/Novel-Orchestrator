### 读者承诺

这篇沉淀的是书名简介要第一时间显出主角身份、装逼方式和核心爽点，黄金三章仿写也必须从自己的设定出发。作者先给出书名《我真不想洗白啊》和简介：宗门是修仙界最臭名昭著的宗门，犯罪率高、触犯禁忌、灵气浓度倒数，而这些都是主角干的。评论最关键的反馈是“主角的身份、装逼方式看不太出来”，以及“我”泛指太大、没有特定身份、没看出爽点。后来有人更看好《开局宗门圣子的我不可能糊》，因为“圣子”直接点明身份，“不可能糊”制造疑问。作者后续按黄金三章范例试写后，又收到反馈：设定不错，但结构太刻意仿《太监能有什么坏心思》，高武仙侠原创信息更多，不适合用日记体塞太多；所有铺垫应该服务主角“精神病”这一核心展示，金手指逻辑也要理顺。

### 核心机制

```text
书名点身份 -> 简介点装逼方式 -> 核心设定服务爽点 -> 黄金三章结构适配自身题材 -> 读者快速确认期待
```

学范例不能只搬结构。范例里的日记体、抛收节奏、信息量安排，是服务它自己的低武架空设定；换成高武仙侠和原创设定，就要重新选择展示顺序。

### 期待链与情绪

```text
读者看到书名 -> 立刻知道主角身份 -> 简介看到反差和爽点 -> 开篇展示核心异常 -> 金手指逻辑清晰 -> 愿意继续看三章
```

如果书名和简介只给“我”，读者要猜身份；如果开篇只仿别人结构，读者要处理的信息会过多。

### 关键判断

- 书名最好直接给出可识别身份。
- 简介要让读者看出主角如何装逼或解决问题。
- “我”太泛，容易降低第一眼识别度。
- 备选书名可以用身份词加反差词制造悬念。
- 设定不错不等于开篇结构可直接套用。
- 高武仙侠原创信息多，不能照搬低武架空日记体的信息压缩方式。
- 黄金三章所有铺垫要服务本书最核心的展示点。
- 金手指名称和作用逻辑要一致，否则读者会觉得怪。
- “犯罪”“洗白”等词要注意道德底线，避免读者误判主角不可接受。

### 读者承诺

这篇沉淀的是女装桥段不天然是毒点，关键看主角形象、后续价值和边界处理。作者设想主角为穿过危险区域暂时女装，被别人喜欢上，但不会发生实质关系，只当插曲。评论给出的判断很明确：后续价值越大，越能引出有趣情节，就越可用；如果主角本身是骚浪贱、能让读者气得牙痒痒的类型，也可以这么干；身份暴露后能制造“网恋对象是抠脚大汉”的幻灭喜感。也有人提醒，这种桥段放男配比放主角更稳，因为主角承担代入，处理不好容易招喷。作者解释自己用的是“虚晃一枪”：主角为了摆脱麻烦女装伪装，但伪装太成功，反而不得不出手。

### 核心机制

```text
危险区域 -> 主角女装伪装 -> 伪装过度成功 -> 引出额外麻烦 -> 身份暴露制造笑点/反转
```

女装不是目的，必须成为情节杠杆。它要么让主角绕过危险，要么引出新冲突，要么在身份暴露时回收喜感。

### 期待链与情绪

```text
读者看到主角被迫伪装 -> 担心桥段尴尬或毒 -> 伪装成功引发麻烦 -> 主角主动解决 -> 暴露后形成笑点
```

只要不写实质暧昧、不损伤主角代入、不把主角尊严持续踩低，就更容易被读者当笑点接受。

### 关键判断

- 女装桥段不天然有毒，但主角代入风险高。
- 后续能引出有趣情节，桥段价值就更大。
- 主角人设如果本来轻佻搞怪，接受度更高。
- 男配承担女装梗通常比主角稳。
- 主角女装后被喜欢，必须避免真实感情线误会。
- “不会发生什么”要在情节里明确边界。
- 身份暴露后的幻灭和怀疑人生，是可用笑点。
- 虚晃一枪要回到主线压力，不能只玩梗。

## 1593：🔊分清小说类型很重要»»


人工查看范围：正文、全部 12 条评论。无附件、无图片、无外链。

证据限制：这是作者个人对仙侠、玄幻、都市等大类的实用区分，不是平台官方分类规则；本卡只拆“分类会影响扫榜、投稿、读者期待”的方法论。

### 读者承诺

这篇沉淀的是作者必须先固定自己的题材大方向，否则扫榜、投稿和开篇承诺都会错位。正文把仙侠、玄幻、都市按生活背景和能力表现粗分：仙侠偏古代、御剑飞行、破境循序渐进；玄幻也多古代，但可多级破境、凌空飞行；都市是现代高楼大厦、手机电脑。作者强调分清类型的三重作用：扫榜别扫错，投稿别投错组，最重要是别写错。评论进一步展开：不要看《诡秘之主》在玄幻区这种幸存者偏差；缝合怪往往是大纲没打好，中后期什么科幻仙侠都揉进去，读者看不懂就流失；真正的问题不是编辑能不能改分类，而是读者看到机器人就期待科技树，如果你写成修仙书，就会让期待落空。

### 核心机制

```text
确定题材大类 -> 扫对应榜单 -> 开篇兑现对应期待 -> 投稿匹配分组 -> 避免中后期乱窜
```

类型不是标签游戏，而是读者期待合同。开头给了什么信号，读者就会按什么类型等爽点。

### 期待链与情绪

```text
读者看到题材信号 -> 形成仙侠/玄幻/都市/科技期待 -> 开篇兑现对应元素 -> 继续信任作者 -> 类型乱窜时流失
```

混搭不是不行，但前期必须让读者知道混搭规则，而不是中期突然换赛道。

### 关键判断

- 扫榜前先确认自己写的是哪个大类。
- 投稿分组要和题材信号匹配。
- 不要用幸存者偏差指导新人分类。
- 缝合多类型前，要先在大纲里确定主方向。
- 读者看见机器人会期待科技，套修仙外壳会落差。
- 无限流可以多世界，但开头要交代多世界框架。
- 编辑能改分类，不代表读者期待可以随便改。
- 前期不稳，后期设计再好也没人追更。

### 可迁移写法

```text
主类型：
副元素：
开篇题材信号：
读者会期待什么：
扫榜对象：
投稿分组：
混搭规则：
不能中途乱加的元素：
```

### 训练任务

### 读者承诺

### 核心机制

```text
想读写作书 -> 选择可获得版本 -> 不在渠道上内耗 -> 立刻阅读 -> 转成作品修改动作
```

资料学习最怕把“找资源”当成“学习”。拿到可读版本后，应该尽快进入阅读和笔记。

### 期待链与情绪

```text
作者担心版本问题 -> 获得可读渠道 -> 开始阅读 -> 对照作品做笔记 -> 学习焦虑下降
```

### 关键判断

- 资料载体不是核心，读起来才是核心。
- 同一本理论书只要内容无明显出入，就不要长期纠结渠道。
- 星球内已有 PDF 时，优先用现成资源。
- 纸质书适合长期翻阅，PDF 适合检索和分享。
- 《故事策略》在微信读书可能以不同书名出现。
- 上传资料后要归档到参考资料板块，方便后人使用。
- 资料读完必须转成写作动作，否则只是收集。


## 1595：#思考分析


人工查看范围：正文、全部 11 条评论、2 个站内书名搜索链接。无附件、无图片。

证据限制：这是女频作者分享的女性美貌描写经验帖；本卡只拆可迁移到男频/女频的角色外貌写法，不将其视为完整人物塑造教程。

### 读者承诺

这篇沉淀的是外貌描写不要只写“她很美”，而要把美拆成可记忆的特质、生活联想、动作和名字。正文提出通感描写法，借《洛神赋》说明可以用视觉之外的感受写美；例子里小玉身量不高、四肢纤细、杏眼带光、双唇像水蜜桃，带着夏天甜蜜气息。作者强调要把美的特质写进生活，让读者一看到水蜜桃就想到小玉。后续补充体香、小动作和名字：名字可以贴合角色，也可以用反差萌。评论里死翼耐萨里奥认为男频多数作者确实该向女频学习，否则一写女性就尴尬；另有评论建议提前建立人物总纲，初见不必大段细腻刻画，先挑两点特别特征，更多细节放到后续动作和互动里。

### 核心机制

```text
人物总纲 -> 初见挑两点特征 -> 通感/生活联想强化记忆 -> 动作互动继续补描 -> 名字形成贴合或反差
```

外貌描写的目的不是堆辞藻，而是让角色在读者脑中留下可召回的识别点。

### 期待链与情绪

```text
读者初见角色 -> 记住独特特征 -> 后续动作反复强化 -> 名字和气质互相印证 -> 角色形象变鲜活
```

男频写女性也要避免尬写，要用具体特征和角色功能服务读者感受。

### 关键判断

- 不要笼统写“美得无法形容”。
- 初见先抓两点特征，别堆一大段。
- 通感能把外貌和气味、季节、触感、生活物关联。
- 体香和小动作可以丰富形象，但要克制。
- 名字可以贴合气质，也可以制造反差萌。
- 人物总纲能减少临场描写尴尬。
- 更细腻的描写适合放进后续动作和互动。
- 男频女性描写可向女频学习底层逻辑，但要服务本书读者期待。
- 男性需求和女性需求不同，底层仍离不开打脸、比照、被追捧和事业成功等反馈。

### 可迁移写法

```text
角色名字：
初见两点特征：
通感联想：
生活记忆物：
体香/声音/动作：
名字贴合或反差：
后续互动补描位置：
读者应记住的标签：
```
