## 361：请教个问题，像下面这一段关于主...

人工查看范围：正文、全部 8 条评论。无附件，无图片。

### 读者承诺

这篇讨论第三人称心理描写里“他”和“自己”的代称问题。核心不是必须禁用某个词，而是一段剧情里代词指向要统一、明确、顺口；如果“他”可能被读者理解成另一个人，就会卡壳。心理活动可以单开段落，用引号或特殊符号处理，最好用语音读出来检查。

### 核心机制

```text
第三人称心理 -> 代词指向不明 -> 读者短暂停顿 -> 统一代称/单开心理段 -> 代入顺滑
```

### 期待链与情绪

```text
读者进入主角心理 -> 需要快速跟上主角判断 -> 代词跳变造成疑问 -> 统一后阅读不被语法打断
```

### 关键判断

- 一段剧情里代词最好统一，不要来回变动。
- “他”在单独段落里可能指主角，也可能指另一个人，容易主语不明确。
- “平日里向来没有人能管得了自己”比原句更顺。
- 看得快的读者遇到代词跳变会卡壳。
- 第二句用“他”，第三句回到“自己”，若上下文不足就会影响观感。
- 心理活动可以单开一段，用引号或特殊符号区分。
- 阅读语音能帮助发现不顺。
- 对话和内心独白写多后，人物语言习惯会潜移默化建立。
- 读者觉得突兀，有时不是代词问题，而是角色说了不属于他人设的话。
- 在换场景或新对话前，每个段落里人称代词指向要明确。

### 可迁移写法

心理代词检查表：

```text
当前视角：
心理描写段落：
使用的代词：
每个代词指向谁：
是否出现“他/自己”跳变：
是否可能误指其他角色：
是否需要单开心理段：
语音朗读是否顺：
最终改写：
```

### 训练任务

找一段第三人称心理描写，圈出所有“他”和“自己”。逐个写明指向谁，凡是可能产生歧义的句子，改成统一代称或主角名。

## 362：经过宝剑大大和各位大佬的指点，...

人工查看范围：正文、全部 2 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-073-case-362\image-1.jpg`。

图片证据：图片为前置提问截图，提出娱乐文主角身患绝症，系统给大师级唱功，唱歌获得影响力可续命；外界误以为主角很惨，实际主角状态很好，作者想写外界关怀和主角不需要关怀的反差喜感，但担心主角显得缺德卖惨。

### 读者承诺

这篇复盘“绝症唱歌”梗：反差喜感可以偶尔调剂，但不能当主线爽点。爽点爆发必须对应期待铺垫，让众人觉得主角牛逼、厉害、值得震惊或心疼；如果只是大家白担心一场，读者不会爽，反而会觉得浪费表情。

### 核心机制

```text
绝症设定 -> 外界担心铺垫 -> 主角唱歌仍然强 -> 众人震惊/感动/心疼 -> 音乐梦想和实力落点形成爽点
```

### 期待链与情绪

```text
读者看到主角绝症 -> 期待他在困境中证明自己 -> 唱功震惊众人 -> 乐坛关心和路人羡慕 -> 心疼与爽感同向
```

### 关键判断

- 反差喜感严格说算不上主线爽点。
- 爽点爆发要和期待感铺垫对应。
- “大家以为主角要死，结果主角没事”不会让众人觉得主角厉害。
- 这类曝光更可能让角色觉得自己白担心。
- 读者最多在未曝光过程中获得一点喜感。
- 喜感可以调节气氛，但不能一直持续。
- 主线剧情重点应落在爽点上。
- 绝症设定更适合落在“主角病了但唱歌依旧牛逼”。
- 乐坛天王天后关心、整个乐坛把主角当国宝、路人震惊羡慕，都是可用爽点。
- 煽情和感动路线相对更可靠，但要控制度。
- 可行核心是：我绝症了，但坚持音乐梦想，音乐还很牛逼，众人既震惊又感动又心疼。
- 理论落实到具体写文时容易忽视，需要通过实例复盘内化。

### 可迁移写法

爽点落点校准表：

```text
设定梗：
读者期待：
铺垫方向：
最终曝光结果：
众人是否觉得主角厉害：
读者是否获得爽感：
只是喜感还是主线爽点：
更可靠的爽点落点：
需要控制的煽情程度：
```

### 训练任务

把一个“反差喜感梗”改成“爽点梗”。要求写出铺垫、爆发、旁观者反应，并检查爆发是否让主角更强、更值得心疼或更被认可。

## 363：看完，计划意外补充。尝试把打更...

人工查看范围：正文、全部 3 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-073-case-363\image-1.jpg`。

图片证据：图片为《大奉打更人》第一个事件的计划/意外流程图：保命 -> 没卷宗 -> 表弟出现要卷宗 -> 妖物作祟无解 -> 金手指破案成功 -> 保命成功，超额收获人脉、名气。

### 读者承诺

这篇用《大奉打更人》第一事件验证“计划-意外-补充/转机”结构。主角每次想推进计划，都会遇到新意外；意外不是断路，而是逼出转机、资源或金手指，让事件继续上行，最后用破案成功结算保命和超额收获。

### 核心机制

```text
计划1 -> 意外1阻断 -> 转机1补路 -> 计划2 -> 意外2加压 -> 转机2金手指 -> 计划3兑现
```

### 期待链与情绪

```text
读者看到主角要保命 -> 没卷宗形成阻碍 -> 堂弟背默提供转机 -> 妖物铁案再加压 -> 金手指发挥 -> 翻案成功并获得人脉名气
```

### 关键判断

- 《大奉打更人》第一个事件可以拆成多轮计划和意外。
- 计划 1 是尝试从卷宗入手。
- 意外 1 是没有卷宗。
- 转机 1 是堂弟可以背默出来。
- 计划 2 是拿到卷宗找线索。
- 意外 2 是妖物作祟近乎铁案。
- 转机 2 是金手指发挥。
- 计划 3 是利用推理成功找到线索，翻案成功。
- 事件最终不只保命，还获得人脉和名气。
- 这种结构难度高，新人不必强行达到打更人水平，但可以学“每次阻断都要给新路”。

### 可迁移写法

计划意外转机表：

```text
主角总目标：
计划1：
意外1：
转机1：
计划2：
意外2：
转机2：
计划3：
最终兑现：
超额收获：
```

### 训练任务

给一个开篇危机写三轮计划。每一轮必须有一个意外阻断，再给一个合理转机，最后结算目标和超额收获。

## 364：#思考分析 现在的网文教程都这...

人工查看范围：正文、全部 13 条评论、2 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-073-case-364\image-1.jpg` 到 `image-2.jpg`。

图片证据：图片为网文教程截图，包含“开局 3000 字必须出现主角”“第一章 3000 字必须触发第一个矛盾冲突”“每章超过 3 个场景”“每章 3 个转折”“每一章有爽点”“主角不能出丑 3 次以上”“每章主题不超过 300 字”“严禁抽象描写”等大量定量规则。

### 读者承诺

这篇批判暴论式、模板式网文教程。许多规则有一定提醒价值，但不讲原理、只给死板结论，会让新人照着签也不知道为什么过，更无法支撑后续剧情。暴论能吸引外行，但真正写书仍要理解期待、节奏、结构和长期发展。

### 核心机制

```text
外行焦虑 -> 暴论教程吸睛 -> 死板模板可短期套用 -> 不懂原理无法迭代 -> 回到基础学习和实践
```

### 期待链与情绪

```text
新人想要必过公式 -> 教程给硬指标 -> 看似明确降低焦虑 -> 写过后不知道原因 -> 后续剧情跟不上
```

### 关键判断

- 面向外行时，暴论是最有效的吸引眼球手段。
- 许多号根本目的不是教写书，而是吸引外行成为粉丝。
- 只讲开头，深入一点就讲不出来，说明教程作者自己也未必会写。
- 一些硬规则本身可能提醒方向没错。
- 但不讲原理直接出结论，新人即使照做过了，也不知道为什么。
- 一本书不是只有第一卷，死板写法后续跟不上仍然没用。
- 生搬硬套不能给作者带来真正提升。
- 这类模板最多帮人签到分成，未必能拿更高收益。
- 浮躁和想一招鲜吃遍天，是新人常见问题。
- 偷懒走捷径符合人性，但写作实际省不了。
- 越学习越发现积累不够，是正常阶段。
- 慢就是快。

### 可迁移写法

教程原理反查表：

```text
教程规则：
它提醒的真实问题：
背后原理：
适用场景：
不适用场景：
如果照抄会怎样：
我自己的改写原则：
后续十万字是否还能支撑：
```

### 训练任务

选 5 条网文教程硬规则，逐条反问“它想解决什么问题”。不能只抄规则，必须写出背后原理和一个不适用场景。

## 365：#思考分析

人工查看范围：正文、全部 8 条评论。无附件，无图片。

### 读者承诺

这篇系统区分弱主线、强主线和完全主线。弱主线靠主题串联单元剧，单元之间像虚线；强主线在每一卷内部有明确主线，但卷与卷之间仍主要靠主题串联；完全主线则在卷级之上继续形成实线，前几卷还共同构成更大的三段式。

### 核心机制

```text
主题 -> 单元剧虚线 = 弱主线
主题 -> 每卷虚线 + 卷内实线 = 强主线
主题 -> 卷内实线 + 跨卷实线 = 完全主线
```

### 期待链与情绪

```text
读者看弱主线 -> 期待同主题下的新单元
读者看强主线 -> 期待每卷目标完成和换地图
读者看完全主线 -> 期待跨卷矛盾持续升级和回收
```

### 关键判断

- 弱主线代表包括无敌文、日常文、神豪文。
- 弱主线剧情由一个个单元剧组成，单元之间没有强因果主线。
- 弱主线不是没有组织，而是靠主题串联成虚线。
- 《开局账号被盗，反手充值一百万》的主题是“较真”。
- 强主线代表包括升级文、悬疑文。
- 强主线通常有短期目标、中期目标、长期目标。
- 强主线不是只有一条主线，而是有一个统摄主题。
- 分卷和换地图级别上，强主线也像弱主线一样由主题串联。
- 区别在于强主线每一卷内部有明确主线、三段式、起承转合和情绪积累。
- 一级结构：单元剧在主题作用下串联成虚线，偏弱主线。
- 二级结构：每一卷在主题作用下串联成虚线，每卷内部有明确主线，偏强主线。
- 三级结构或完全主线：在二级结构基础上，多卷又串成更大的三段式。
- 《诡秘之主》第一卷廷根结束后，矛盾没有完全解除，反而引出复仇和后续更大矛盾。
- 一二三级结构没有高下之分，只是不同架设思路。
- 强主线可以把一个故事或观点放入三幕式，再继续向下细分每一步的主题和事件。

### 可迁移写法

主线层级判定表：

```text
作品主题：
单元之间是否强因果：
每卷是否有明确主线：
卷与卷之间是否组成更大三段式：
短期目标：
中期目标：
长期目标：
超长期目标：
结构类型：一级/二级/三级
下一卷是否回收前卷未解矛盾：
```

### 训练任务

给一个题材写三个版本：弱主线单元剧版、强主线分卷版、完全主线三卷版。每版都写主题、单元/卷目标和期待来源。
