## 411：刚忙过，存了二十章了，萌新想去...

人工查看范围：正文。原文无评论、无附件、无图片。

### 读者承诺

这篇信息很短，但对应新人平台试水的一个决策点：已经有二十章存稿，想去番茄试水，却听说新书签约难，不如直发等广签。可迁移重点是把“平台传言”转成可执行试水方案：存稿量、直发节奏、广签等待、数据观察和止损点必须提前定。

### 核心机制

```text
已有存稿 -> 平台签约传言 -> 选择直发试水 -> 等待广签/数据反馈 -> 决定加码或止损
```

### 期待链与情绪

```text
新人有二十章存稿产生尝试冲动 -> 听到签约不可能而犹豫 -> 直发试水降低决策压力 -> 用数据判断是否继续
```

### 关键判断

- 作者是萌新，已有二十章存稿。
- 目标平台是番茄。
- 外部建议认为番茄新书签约难。
- 另一种策略是直发后等广签。
- 原帖没有评论，因此没有更多确认信息。
- 可执行重点不在“信不信传言”，而在直发试水的止损设计。
- 二十章存稿足够做短期连续更新测试。
- 试水前应明确每天更新量、观察指标和放弃条件。
- 如果只是“试一试”，不要把所有长期资源押上。

### 可迁移写法

番茄直发试水表：

```text
当前存稿章数：
每日更新章数：
预计可撑天数：
是否已有签约预期：
是否接受广签：
观察指标：
第几天复盘：
继续条件：
止损条件：
下一平台备选：
```

### 训练任务

拿 20 章存稿做一个 14 天直发计划。写明每日更新、观察数据、签约/广签等待时间和止损线。没有止损线不发书。

## 412：拆著名商业动画《某科学的超电磁...

人工查看范围：正文。原文无评论、无附件、无图片。

### 读者承诺

这篇人工拆《某科学的超电磁炮》第一集，重点不是复述剧情，而是学习商业动画如何在 21 分钟内完成角色登场、设定展示、事件启动、记忆点强化和高潮。它的关键方法是：有大量设定，但几乎没有纯讲设定，设定都通过导游、红绿灯事故、售货机、能力测试、角色互动和危机事件进入剧情。

### 核心机制

```text
强角色先出场 -> 一分钟内出事件 -> 设定挂在行动上 -> 角色记忆点连续强化 -> 小高潮回收
```

### 期待链与情绪

```text
观众先知道女主很强 -> 城市异常引出世界规则 -> 多角色用动作登场 -> 能力测试铺垫装逼 -> 日常互动建立亲近 -> 危机高潮兑现武力和人设
```

### 关键判断

- 有效时长约 21 分 44 秒。
- 开头御坂美琴被混混包围，观众知道她很强，可以脑补结果。
- 外传/续篇可利用观众已有认知，但新人原创不能完全照搬。
- 画外音介绍世界观后立刻转成导游讲话，避免纯旁白。
- 导游之后在高潮部分成为受害者，前置信息被回收。
- 第一个事件在约一分钟出现：红绿灯失控导致交通混乱。
- 第二分钟通过追逐罪犯引出黑子和初春。
- 简短动作体现黑子武力值和人物形象。
- 倒在御坂身边的罪犯体现女主武力值。
- OP 后对话围绕设定，但仍挂在角色互动中。
- 踢售货机和巡检机器人继续强化高科技设定。
- 超能力测试先铺普通学生、再铺黑子、再铺转校生吹牛，最后女主装逼。
- 商业日番用浴室、掀裙等桥段添加角色记忆点。
- 五分之二处已完成女一女二初步形象，并让四位主要角色登场。
- 四位角色相会时，每个人都有目的。
- 佐天送呱太让御坂重视她，日常事件完成关系推进。
- 开头导游与游客在后段危机中回收。
- 女主反差萌通过强大、喜欢呱太、焦躁、失落、喜悦连续建立。
- 时间安排大致为五分之二角色出场、五分之一过渡、五分之二高潮。

### 可迁移写法

商业第一集拆解表：

```text
有效时长/章节长度：
第一分钟事件：
第一批登场角色：
设定如何事件化：
角色记忆点1：
角色记忆点2：
中段关系目的：
日常道具/小事件：
高潮危机：
前置信息回收：
结尾留下的角色关系：
```

### 训练任务

把自己的第一章按“前 20% 角色和事件、20%-60% 关系目的、后 40% 危机高潮”重排。所有设定必须挂在动作、道具、事故、测试或对话上，不能单独成段讲。

## 413：大家帮我看看我找的这个大纲模板...

人工查看范围：正文、全部 7 条评论、附件状态。附件为 `我的大纲模板.docx`，已尝试下载但文件链接返回 401 Authorization Required，无法读取正文内容。

证据限制：无法查看 docx 模板具体内容，因此本卡依据帖正文和评论区对“大纲模板”的讨论提炼，不评价模板质量。

### 读者承诺

这篇讨论的是大纲模板该不该照搬。结论是模板因人而异，本质像草稿纸，不是论文格式；好用就是好。章纲可能卡得很死，万字纲更适合部分作者。真正关键不是模板长什么样，而是从总纲、卷纲、万字纲到章纲逐步缩小，并在每一步寻找波折。

### 核心机制

```text
参考模板 -> 个人化改造 -> 总纲到章纲逐层细化 -> 每层寻找波折 -> 服务写作
```

### 期待链与情绪

```text
新人不知道大纲怎么写 -> 想先看别人模板获得安全感 -> 评论降低格式焦虑 -> 把模板当草稿纸 -> 用波折驱动章纲
```

### 关键判断

- 作者卡在万字纲和章纲，不知道写什么。
- 评论认为章纲会卡得很死，万字纲更灵活。
- 模板因人而异。
- 模板不是论文，不需要固定格式。
- 模板像做数学题时的草稿纸，怎么用看自己。
- 先看别人怎么写，再学会这套东西，是可接受的学习方式。
- 可参考“绑架全人类”的大纲。
- 构思大纲可从大到小：总纲、卷纲、万字纲、章纲。
- 写什么都可以，最重要的是有波折。
- 章纲可以只有几句话，但每句话之间要有转折。
- 大纲真正服务的是让作者知道下一步写什么，而不是填满表格。

### 可迁移写法

大纲模板个人化表：

```text
我最容易卡在哪层：
总纲一句话：
卷纲核心变化：
万字纲阶段目标：
章纲每句转折：
必须保留的模板项：
可以删除的模板项：
是否会卡死写作：
是否让我更想写：
下一章要写什么：
```

### 训练任务

不用完整模板，只写 1 个总纲、1 个卷纲、1 个万字纲和 3 条章纲。每条章纲必须有“但是/于是/结果”式转折。写完后删掉所有不能帮助你继续写正文的模板项。

## 414：【飞卢天榜作者内投起点被拒？】...

人工查看范围：正文、全部 4 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-083-case-414\image-1.jpg`。

图片证据：图片为论坛截图，标题大意是“群里一个上过天榜的作者，内投起点被拒签了”，楼主评论“虽然知道起点比飞卢难，但也不至于难得这么离谱吧？”。

### 读者承诺

这篇强调跨平台文风错配：一个作者在渠道/飞卢成绩好，不代表起点编辑会收，也不代表起点读者会买账。某些开头风格只在特定网站赚钱；换平台前必须扫榜调研，判断自己的文风、信息包、长线能力和目标站读者是否匹配。

### 核心机制

```text
原平台成绩 -> 目标平台内投 -> 文风/结构错配 -> 被拒或扑街 -> 重新扫榜选站
```

### 期待链与情绪

```text
作者以为天榜成绩能跨站通用 -> 起点拒签制造落差 -> 发现平台口味差异 -> 回到扫榜调研 -> 选择正确网站减少单机
```

### 关键判断

- 原帖作者不确认事件真假，但认为大概率会发生。
- 渠道网站成绩很好，去起点签约不了或扑街并不罕见。
- 这不一定是水平问题。
- 有些文风尤其内投开头风格，只在特定网站能赚钱。
- 一本天榜和本本天榜的作者，水平有本质差别。
- 起点部分编辑或工作室内投门槛明显提高。
- 起点老作者新书也可能改很久，竞争激烈。
- 渠道向文风放起点可能没人看，还会被骂。
- 放到正确网站却能赚钱。
- 不要只问“我去哪个网站合适”，需要自己逐个平台调研扫榜。
- 蒙头发错网站，可能把本该赚钱的书写成单机。
- 飞卢部分作者开头很爆，但信息包可能只服务前 5 万或 10 万字爆发。
- 如果前面信息包不为后续安排，编辑追问长线时容易露馅。
- 新人哪个网站书看得多，就扫哪个网站榜，写到过签先练一本。

### 可迁移写法

跨平台文风适配表：

```text
我的原平台/参考平台：
目标平台：
开头风格：
信息包服务前几万字：
是否有长线安排：
目标平台同类榜单：
目标读者能否接受：
编辑门槛：
错配风险：
更适合的平台：
```

### 训练任务

拿自己的前三章，同时对照飞卢、起点、番茄各 5 本同题材榜单开头。只判断三项：开头爆点、信息包长线、读者耐受度。最后选一个最匹配平台，不允许凭感觉投。

## 415：拆书拆了三本了，脑子和眼睛以及...

人工查看范围：正文、全部 19 条评论。无附件，无图片。

### 读者承诺

这篇处理“拆书会了但写不出来”的执行问题。拆三本后感觉懂了，不代表能集中写正文；如果一章死磕、写几百字就玩、反复改前文，需要先区分灵感不足、大纲不满意和注意力训练不足。解决办法不是继续硬逼开书，而是休息、完善让自己想写的大纲，并用番茄钟/小黑屋/字数锁训练专注。

### 核心机制

```text
拆书获得懂感 -> 打开码字注意力涣散 -> 区分灵感/大纲/专注问题 -> 小块强制专注 -> 逐步恢复产出
```

### 期待链与情绪

```text
新人以为自己会了 -> 写作时只产几百字 -> 挫败感加重 -> 用工具和小目标降低阻力 -> 完成小段产出建立信心
```

### 关键判断

- 拆书三本后“脑子眼睛手都觉得会了”，但实写仍可能卡住。
- 一打开码字注意力难集中，说明问题不只是理论。
- 一章反复死磕、边写边玩、边改边玩，会导致一天没有推进。
- 宝剑建议先歇一歇，别把开书当任务。
- 真正灵感迸发时，不需要一直逼自己码字。
- 如果有灵感仍被小玩意吸引，就需要训练注意力。
- 手机可用专注软件，设定 2-40 分钟不打开其他 App。
- 电脑可用小黑屋类强制锁定。
- 第一次字数锁建议 1-2k，太多会降低码字意愿。
- 不建议一开始用时间锁或混锁，可能不直观。
- 竞争型作者可用闯关或拼字。
- 大纲不完善、设计不满意，也会让人不愿意动笔。
- 一个让自己满意的大纲会产生写作动力。
- 章纲和模板不是关键，关键是有让作者想写下去的内容。

### 可迁移写法

写作专注排查表：

```text
当前是否真想写：
是否只是把开书当任务：
大纲是否让我兴奋：
今日最低字数锁：
手机是否锁定：
电脑是否小黑屋：
第一次专注时长：
是否反复改前文：
完成后奖励：
明天继续点：
```

### 训练任务

连续 7 天做专注训练：每天只锁 1000-2000 字，不改前文，不开其他 App。写完记录“卡住原因是灵感、大纲还是注意力”。7 天后再决定是否开正式长篇。
