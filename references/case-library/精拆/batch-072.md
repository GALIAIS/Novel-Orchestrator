## 356：现在不完全架空的都市题材里，地...

人工查看范围：正文、全部 3 条评论。无附件，无图片。

### 读者承诺

这篇讨论不完全架空都市里的现实名词使用，以及题材归类边界。地球、欧亚、美洲、泰山、三星堆、金字塔、百慕大这类地理或文化名词通常可以用，但具体审核会随时期变化。另一个评论案例说明：即便起点在地球，后续穿越到沙之世界、蒸汽绿洲、现代枪械、诸天裂缝混杂，也不再是正统西幻，而更接近无限诸天。

### 核心机制

```text
现实名词使用 -> 判断是否敏感或真实映射 -> 题材元素混合 -> 看主驱动是否是诸天穿梭 -> 重新归类
```

### 期待链与情绪

```text
读者看到地球和真实地名 -> 形成都市或现实锚点 -> 后续出现异世界和多世界裂缝 -> 阅读期待转向冒险/无限 -> 题材标签需跟主玩法一致
```

### 关键判断

- “地球”这类基础现实名词一般问题不大，但无法保证任何时期都安全。
- 欧亚、美洲、泰山、三星堆、金字塔、百慕大等地理文化名词，通常可作为背景锚点。
- 不完全架空都市要注意真实机构、现实政治、真实人物等更高风险内容。
- 题材归类不只看开头地点。
- 主角从地球探险进入沙之世界，再通过裂缝穿梭诸天，主玩法已偏无限诸天。
- 西方中世纪、奇幻生物、蒸汽机、现代枪械混在一起，不等于正统西幻。
- 西方背景少时，对西幻标签影响更小。
- 标签应按读者实际会追的主循环来定，而不是按单个元素定。

### 可迁移写法

现实名词与题材归类表：

```text
使用的现实名词：
是否只是地理/文化锚点：
是否涉及真实机构/政治/人物：
主角起点：
主要地图：
核心玩法：都市/探险/西幻/无限/诸天
主要读者期待：
最终题材标签：
需要替换或架空的词：
```

给一个“地球探险家进入异世界”的设定写 3 个标签方案：都市探险、西幻、无限诸天。分别说明哪个标签最容易准确筛选读者，哪个最容易误导。

## 357：我深入的读了gu真人这书，

人工查看范围：正文、全部 11 条评论、3 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-072-case-357\image-1.jpg` 到 `image-3.jpg`。

图片证据：图片内容为文档局部，涉及“组织本质是资源再分配”“上位者调动积极性的价值观工具”“权力的魅力”“区别待遇肯定自身价值”等表达，被发帖者用作“夹带私货、隐喻社会体制和权利分配”的例证。

### 读者承诺

这篇继续讨论黑暗反派文的封禁风险，结论更明确：危险不只在杀人越货、侮辱尸体、拦路抢劫这类情节，而在作者夹带太多社会体制、权力分配、组织无用论等私货，并让读者和粉丝形成“魔道中人”式认同。黑暗流可以学脑洞、力量体系和故事结构，但不能把现实政治隐喻、价值导向和粉丝组织化一起写进商业网文。

### 核心机制

```text
黑暗流爽点 -> 夹带社会/权力私货 -> 粉丝形成身份认同 -> 争议和举报扩大 -> 平台一刀切封禁
```

### 期待链与情绪

```text
读者被反派爽点吸引 -> 作品持续输出黑暗价值解释 -> 一部分读者奉为圣经 -> 另一部分读者反感举报 -> 风险从剧情尺度升级为价值导向问题
```

### 关键判断

- 杀人越货、侮辱尸体、拦路抢劫等情节危险，但经过反派对象和语境处理，未必直接导致封禁。
- 真正更高风险的是夹带社会体制、权利分配、组织本质等现实隐喻。
- 国内网文对这类内容本就敏感。
- 红旗飘飘也可能被误读，平台无法替作者承担解释风险。
- 平台一刀切省事利落。
- 黑暗流粉丝自发形成“魔道中人”等身份认同，会让作品风险放大。
- 作者在章末自称魔道、引导粉丝推荐，也容易被看成引导教唆。
- 树大招风和举报，与作品本身灰色风险并不矛盾。
- 不签约扑街书没人看，可能没事；热度起来后就会被举报和平台处理。
- 审核模糊导致很多书在没人闹时睁一只眼闭一只眼，闹大后丢出去祭旗。
- 作品脑洞、力量体系、成语拟化、市井小民等可学习，但价值观和私货不可迁移。
- 组织无用论、公共价值反向正当化等属于高风险表达。
- 跳大地图牵强也被评论指出是结构问题，不能只讨论封禁。

### 可迁移写法

黑暗流私货风险表：

```text
黑暗爽点：
暴力情节是否有语境：
是否夹带现实体制隐喻：
是否讨论权力分配/组织本质：
是否贬低公共价值：
是否引导粉丝身份认同：
是否容易被解读成现实政治：
可学习的脑洞/结构：
必须删除的私货：
替代表达：
```

### 训练任务

选一个黑暗流桥段，拆成“可学脑洞”和“不可学私货”两栏。保留力量体系或反转设计，删掉现实体制隐喻和粉丝身份认同话术。

## 358：从来没考虑过铺垫和伏笔的区别，...

人工查看范围：正文、全部 6 条评论。无附件，无图片。正文含 QQ 浏览器外链，未提供本地正文内容。

### 读者承诺

这篇讨论铺垫和伏笔的区别。铺垫更偏整体布局和烘托，防止后面的人物、事物、事件突然出现；伏笔更偏关键位置的提示和暗示，让读者产生期待、往作者设计的方向猜，但不能完全猜透，最终形成前后呼应。

### 核心机制

```text
后文重要内容 -> 前文铺陈降低突兀 -> 关键细节暗示形成伏笔 -> 读者猜到开头猜不到结尾 -> 回收呼应
```

### 期待链与情绪

```text
读者先看到次要信息或细节 -> 觉得自然合理或产生疑问 -> 后文重要人物/事件出现 -> 回想前文线索 -> 获得合理和惊喜
```

### 关键判断

- 铺垫和伏笔在写卷纲、章纲时区别明显。
- 铺垫是整体布局，伏笔是关键位置的铺垫。
- 铺垫主要为了突出后面要出场的重要人物、事物或事件。
- 铺垫常通过次要人物、事物、事件的铺陈描述，烘托或引出重要内容。
- 伏笔注重对将要出现的人物或事件预先提示或暗示。
- 伏笔的核心是前后呼应。
- 铺垫防止后续人或事太突兀。
- 伏笔揭露一部分有期待感的细节，引导读者往设计方向想。
- 好伏笔应让读者猜到开头，却猜不到结尾。
- 简化到创作层面，设定和剧情好看仍是核心；设定靠积累和灵感，剧情要合理反转。
- 没有灵感时不要硬写。

### 可迁移写法

铺垫伏笔区分表：

```text
后文重要内容：
需要铺垫的原因：
用于铺垫的次要信息：
关键伏笔细节：
读者会猜到什么：
读者不该猜透什么：
回收位置：
回收后的情绪：合理/惊喜/反转
```

### 训练任务

给一个后文反转写 3 个前置信息：其中 2 个只做铺垫，1 个做伏笔。标明哪个防突兀，哪个负责让读者产生期待和猜测。

## 359：这样应该不会被夹了吧……

人工查看范围：正文、全部 8 条评论、1 张图片。图片已下载并人工查看：`case-library\精拆\attachments\batch-072-case-359\image-1.jpg`。

图片证据：图片为女频开篇片段：茅草屋、鲜血桌子、杀猪刀、被抹脖子的丫头、村民想拿穿越后的女主开刀，女主在黑暗中被女声提醒“你想跑？”并思考逃跑路线。片段仍保留血、刀、死人、开刀等较直接危险表达。

### 读者承诺

这篇延续女频饥荒敏感情节改写问题。系统夹通常是敏感词触发，尤其“吃人”这类词和情节在女频更容易被夹。解决方向不是简单问男频尺度，而是按女频和系统审核口径替换敏感词、弱化直给表达，并让危险服务角色设定和逃生动作。

### 核心机制

```text
硬核情节 -> 系统敏感词触发 -> 替换直给词和画面 -> 保留危机功能 -> 角色设定合理化危险
```

### 期待链与情绪

```text
读者看到灾荒血腥危险 -> 系统可能先拦截直给词 -> 作者转成暗示和逃生压力 -> 读者理解危机但不被猎奇细节拖住
```

### 关键判断

- 男频能过的内容，女频不一定能过。
- 被系统夹通常不是编辑判断，而是敏感词或敏感表达触发。
- 可以先找出并替换敏感词。
- “吃人”是高风险直给词。
- 人吃人比非人生物吃人更容易被 ban。
- 克系生物吃人，如果符合角色设定，读者不一定觉得奇怪。
- 人类灾荒吃人更贴近现实伦理和敏感点，风险更高。
- 图片片段虽已改成危险场景，但仍有鲜血、杀猪刀、抹脖子、死人、开刀等强刺激词。
- 女频改法应把重点放在女主逃生、判断和尊严选择，而不是血腥物件堆叠。
- 系统夹时，先做词级替换，再做画面级弱化。

### 可迁移写法

系统夹敏感词排查表：

```text
原情节：
可能触发词：
可能触发画面：
平台/频道：男频/女频
是否贴近现实伦理敏感：
可替换词：
可侧写动作：
保留的危机功能：
主角逃生动作：
角色设定是否能合理化：
```

## 360：提问宝剑巨:我最近在写一个玄幻...

人工查看范围：正文、全部 10 条评论。无附件，无图片。

### 读者承诺

这篇讨论丹道和剑道双元素设定失衡。解决办法不是必须砍掉一个，而是明确主线/支线：丹道可以做主线，剑道做支线；如果作者心理预期是剑道为主，就把 70% 戏份给剑道，丹道做金手指或辅助。无论哪条线为主，另一条都要推进主线，不能完全平行。

### 核心机制

```text
双元素设定 -> 明确主线支线 -> 分配 70/30 戏份 -> 两线互相影响 -> 支线推进主线
```

### 期待链与情绪

```text
读者看到丹帝重生和剑道修炼 -> 期待两种装逼方式 -> 作者确定主承诺 -> 主线负责长期成长 -> 支线提供辅助和差异爽点
```

### 关键判断

- 双元素不一定要删一个。
- 更适合用主线/支线思路处理。
- 类似《斗破苍穹》主角升级是主线，炼药是支线。
- 丹帝重生后，丹道作为升级主线没问题。
- 也可以丹道作为金手指装逼，剑道作为主线。
- 关键看作者心理预期到底想写丹道还是剑道。
- 如果剑道为主，就把更多戏份给剑修，删减多余丹修戏份。
- 比例可以参考 70% 主线、30% 支线。
- 两条线都可以装逼，但方式和方向要有区别。
- 支线必须对主线有推进作用，不能完全平行。
- 最好在设定上让两者相互影响。
- 融合路线如丹剑合一可以解决占比问题，但如果主题不符就不必强行融合。
- 作者擅长丹道装逼，容易写歪；这说明要用篇幅和事件功能控制主承诺。

### 可迁移写法

双元素主支线表：

```text
元素A：
元素B：
作者真正想写的主承诺：
主线元素：
支线元素：
主线戏份比例：
支线戏份比例：
两种装逼方式差异：
支线如何推进主线：
是否需要融合：
需要删减的偏题戏份：
```

### 训练任务

给一个“双元素主角”写 10 个事件，标注每个事件属于主线还是支线。主线至少 7 个，支线最多 3 个，且每个支线事件必须推动主线状态变化。
