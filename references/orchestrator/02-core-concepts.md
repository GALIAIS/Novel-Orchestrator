# 02-core-concepts

## 核心总公式

```text
读者需求 -> 具象承诺 -> 戏剧性契机 -> 主角目标 -> 阻碍与代价 -> 期待 -> 情绪拉扯 -> 兑现 -> 余波反应 -> 新期待
```

系统内所有模块都围绕这条链路工作。

## 读者需求

读者需求不是主角目标。主角想升级、赚钱、复仇、活下去，只是事件目标。读者真正期待的是目标达成后会发生什么：

- 地位改变
- 仇人后悔
- 旁人震惊
- 关系推进
- 生存安全
- 资源跃迁
- 秘密揭晓
- 新世界展开
- 情绪获得补偿

判断标准：如果目标达成后没有可想象的爽、怕、甜、燃、虐、奇、悬、笑，目标就不产生阅读动力。

## 读者承诺

读者承诺是作品对目标读者的隐性契约：

```text
你读这本书，会稳定得到哪类情绪价值和故事体验。
```

例子：

- 规则怪谈：看主角在死亡规则中识别漏洞、反杀规则、扩大认知优势。
- 升级文：看主角从低位到高位，获得力量、地位、资源和见证。
- 都市神豪：看资源跃迁后打破旧关系秩序。
- 悬疑：看信息缺口被层层揭开。
- 恋爱：看关系从误解、试探、靠近到确认。

## 期待感

期待感不是“主角有目标”，而是读者对“目标达成后的可能性”产生想象。

期待感常来自：

- 目标达成后能获得的具体好处。
- 失败后会付出的明确代价。
- 读者比角色知道更多的信息差。
- 角色比世界知道更多的信息差。
- 反派、旁观者、亲友对主角的错误预期。
- 某个戏剧性设定必然会和特定人事物发生反应。
- 旧承诺未兑现，新承诺已经出现。

## 爽点

爽点是期待被兑现后的情绪释放。常见结构：

```text
铺垫 -> 压制 -> 能力/信息/身份准备 -> 爆发 -> 见证反应 -> 收获 -> 新目标
```

只写主角变强，不等于爽。要写清楚：

- 谁认为主角不行。
- 主角为什么必须证明。
- 主角用什么能力或选择破局。
- 破局后谁的预期被打破。
- 主角获得什么显性或隐性收益。
- 这个收益如何指向下一个期待。

## 情绪

情绪不是一句“他很愤怒”“她很震惊”。情绪来自交锋后的反应。

有效情绪需要：

- 触发物：事件、话语、选择、物品、真相。
- 关系：这件事为什么影响这个人。
- 交锋：人物怎样应对、压制、误判或反击。
- 变化：状态、认知、关系、地位、危险发生改变。
- 见证：他人反应放大读者感受。

## 戏剧性

戏剧性是让读者觉得“这件事一旦发生，必然会有好看的反应”的结构条件。

常见戏剧性来源：

- 身份错位：太监掌权、废柴持有顶级资源、底层掌握高维信息。
- 关系错位：前任、老板、亲友、仇人、恩人都将在主角变化后重新评价。
- 信息错位：读者知道主角有能力，角色不知道。
- 规则错位：主角理解规则漏洞，世界按旧规则行动。
- 价值错位：主角的选择挑战既有秩序。

## 起承转合

从读者期待角度理解：

- 起：告诉读者主角将得到什么、失去什么，或必须面对什么。
- 承：主角思考、选择、执行，逐步接近期待。
- 转：出现意料之外的阻碍、反转、升级或新信息，放大期待。
- 合：兑现得失，回收铺垫，完成情绪释放，并接上新期待。

简化公式：

```text
制造期待 -> 接近期待 -> 放大期待 -> 揭晓期待
```

## 单元故事

单元故事是最小可验证结构。它不是固定章节数，而是一个事件闭环。开篇范围也不一定是三章，通常应以第一个单元故事为准。

一个合格单元至少改变一项：

- 主角状态
- 关系状态
- 读者认知
- 资源状态
- 危险等级
- 世界规则理解
- 长线目标位置

## 两长一短

为了避免断期待，开篇早期应尽快建立：

- 一个短期期待：当前马上要解决的目标。
- 两个长线期待：更大的关系、身份、力量、谜团、世界、情绪承诺。

后续任一时刻，期待库存不应低于“两长一短”。短线目标一个接一个，长线期待提前铺垫、延迟满足、阶段兑现。

## 断期待

断期待是读者失去继续阅读理由。常见原因：

- 丢失目标：高潮后不知道接下来干什么。
- 人际关系网破裂：见证体系消失，爽点没人承接。
- 长期无升级或无收获：目标悬置太久。
- 兑现太弱：期待被满足但读者没感觉。
- 发展太平：读者一眼看到结果。
- 毒点破坏承诺：虐主、偏题、人设崩、核心梗丢失。

## 拆书

拆书在本系统中的定位是“取证模块”。

拆书不负责替你决定创作方向，只负责回答具体问题：

- 这本书如何建立读者承诺？
- 第一个单元如何跑通？
- 它怎样拉期待？
- 它怎样避免断期待？
- 它的主角怎样推动事件？
- 哪些结构可以迁移到我的题材？
