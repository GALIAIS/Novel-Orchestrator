# Module：期待感与情绪

## 使用场景

- 爽点弱。
- 读者知道主角要做什么，但不想看。
- 情绪折线平。
- 高潮写完像流水账。
- 目标达成了，但读者没有满足感。

## 期待链路

```text
需求 -> 目标 -> 具体好处/失败代价 -> 戏剧性契机 -> 误判/阻碍 -> 情绪拉扯 -> 兑现 -> 反应 -> 新期待
```

## 期待感来源

1. 具体收益：钱、地位、力量、身份、关系、安全、答案。
2. 失败代价：死亡、社死、失去机会、亲友受害、阶层锁死。
3. 误判对象：反派、前任、上司、亲友、同门、公众。
4. 信息差：读者知道主角会赢，但角色不知道。
5. 戏剧性设定：某个设定一旦碰到特定场景，必然好看。
6. 长线承诺：当前事件通向更大的世界或关系变化。

## 情绪构建

有效情绪不是形容词，而是交锋：

1. 角色遭遇触发物。
2. 触发物击中需求或伤口。
3. 对方施压、误判、嘲讽、诱惑或阻拦。
4. 主角做出选择。
5. 选择带来代价、反击或收获。
6. 多方反应放大情绪。

## 断期待排查

- 当前目标完成前，有没有提前给下一个目标线索？
- 主角完成目标后，读者是否知道下一步要看什么？
- 人际关系网是否还能见证主角变化？
- 长线期待是否太久没有阶段性兑现？
- 兑现是否写出了足够差异化的收获？
- 核心梗是否重复到读者能一眼看穿？

## 修复策略

- 目标不吸引：把目标达成后的生活和反应具体化。
- 爽点不爽：增加前置压制和见证反应。
- 情绪不够：增加交锋，而不是增加心理描写。
- 期待断档：在兑现前系上下一个扣。
- 重复感强：换核心梗表达方式，或交替运行多个核心梗。
