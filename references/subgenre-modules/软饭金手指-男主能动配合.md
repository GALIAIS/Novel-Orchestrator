# 软饭金手指-男主能动配合

依据案例：`batch-049` 案例 241。

## 读者承诺

强女主可以成为男主金手指，但男主必须有主动目标、行动贡献和尊严，救场要写成关系配合，而不是男主窝囊后被动等救。

## 核心循环

```text
男主主动介入 -> 解决能力内问题 -> 危机升级 -> 女主主动补强 -> 双方共同获益
```

## 开篇必备

- 男主先做决定，不是被推着走。
- 男主先完成第一层问题。
- 升级危机来自意外后台、信息差或背刺。
- 女主出手有自己的动机。
- 救场后给男主保留贡献和收益。

## 期待来源

- 读者期待男主有胆有担当。
- 女主过度解决能带来强者爽点。
- 双方配合能带来关系甜点。
- 男主不窝囊，软饭才能硬吃。

## 断档风险

- 男主明知无能还硬莽，会显得蠢。
- 女主替男主完成全部任务，会削弱男主。
- 危机升级没有铺垫，会像强行救场。
- 女主没有动机，会变成工具人金手指。

## 训练任务

写一个强女主救场桥段，分三层：男主独立解决、危机升级、女主过度解决；每层都标出男主的贡献。
