# 初见对话-降格称呼与低烈度对立

依据案例：batch-391 案例 1954

## 读者承诺

通过故意误称职业、职级或身份建立地位差，让施压者用玩笑保留退路，目标人物则以克制回应显出边界和后续反击能力。

## 核心循环

```text
正式自我定义 -> 对方故意降格误称 -> 主角纠正 -> 对方玩笑化重申 -> 表面礼貌延续但关系账留下
```

## 开篇必备

- 双方争夺的称呼或群体第一印象。
- 施压者需要压低对方的现实动机。
- 不至于立即翻脸的可否认外壳。
- 目标人物能塑造人设的克制回应。

## 期待来源

- 一句话同时显出阶层和敌意。
- 主角守住体面但暂不彻底反击。
- 旁观者态度让称呼产生现实后果。
- 后续能力证明重新夺回定义权。

## 断档风险

- 生搬外国翻译腔。
- 贬称强到人物理应当场翻脸。
- 主角只长篇解释而没有边界动作。
- 后文从不回收称呼与地位变化。

## 训练任务

在中国职场和架空宴会各写一段降格称呼初见戏，让读者能判断敌意、地位和不翻脸的理由，并按文化语境调整强度。
