# 营销号套路-浓缩版爽文结构

依据案例：`batch-192` 案例 958。

## 读者承诺

营销号常用的是浓缩版爽文结构：不公平危机、金手指方案、快速成长、回归打脸。

## 核心循环

```text
不公平待遇 -> 补设定 -> 金手指 -> 时间跳跃 -> 回归打脸
```

## 开篇必备

- 开头立刻给不公平。
- 金手指承诺改变命运。
- 成长过程压缩。
- 结尾给群体认同和地位反转。

## 期待来源

读者想看被亏待的人如何靠金手指翻身，并回到原场景完成清算。

## 断档风险

- 不公平不够直观。
- 金手指和翻身结果关系弱。
- 成长压缩后没有可信反馈。
- 打脸场景没有围观认同。

## 训练任务

用 400 字写一个营销号爽文：开除/羞辱、金手指、三个月后、回归打脸四段齐全。
