# 仙侠探案开篇-冤案冲突与专业反制

依据案例：`batch-255` 案例 1273。

## 读者承诺

仙侠探案开篇模块用当前冤案冲突和主角专业反制建立追读，不用说明文一次性塞满设定。

## 核心循环

```text
被冤入局 -> 专业能力找漏洞 -> 小反制争生机 -> 更大势力压迫 -> 破案翻盘期待
```

## 开篇必备

- 第一场直接给当前冲突。
- 主角能力要立刻可见。
- 案情漏洞只抛最关键的一两个。
- 视角切换要克制，先稳住主角线。

## 期待来源

读者站队被冤的主角，同时看到他能用刑警经验、证据意识和人心判断在仙侠世界破局。

## 断档风险

- 开头像说明文。
- 主角连续受苦，回弹太慢。
- 朝堂势力和世界观提前解释过多。
- 前三章随意切视角，打断代入。

## 训练任务

写一个 1500 字被冤入狱开篇：300 字内给冲突，600 字内让主角反制一次，结尾只留 1 个案情漏洞和 1 个生死悬念。
