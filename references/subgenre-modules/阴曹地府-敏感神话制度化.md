# 阴曹地府-敏感神话制度化

依据案例：`batch-338` 案例 1688

## 读者承诺

阴曹地府题材要先做合规转译，把鬼神素材写成神话、仙侠、秘境、副本或官僚制度冲突。

## 核心循环

```text
阴司素材 -> 敏感风险识别 -> 制度化转译 -> 主角进入规则 -> 利用漏洞破局
```

## 开篇必备

- 明确不是直白宣扬迷信。
- 阴司以规则或机构呈现。
- 主角有进入阴司的现实目标。
- 第一章有可利用规则。
- 爽点落到破局，不是堆名词。

## 期待来源

读者期待熟悉的阴司神话被重新制度化，并看到主角利用规则反杀。

## 断档风险

- 封建迷信风险未处理。
- 只堆鬼差、判官、阎王名词。
- 没有主角目标。
- 资料味压过故事。
- 参考书单替代主线设计。

## 训练任务

把“主角下地府”转成秘境、仙侠机构或规则副本，设计一个主角利用阴司规则破局的事件。
