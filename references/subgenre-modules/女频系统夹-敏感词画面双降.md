# 女频系统夹-敏感词画面双降

依据案例：`batch-072` 案例 359。

## 读者承诺

这个模块用于女频灾荒、血腥和恐怖开篇被系统夹的问题。它承诺的是先降敏感词，再降直给画面，同时保留女主逃生和判断的危机功能。

## 核心循环

```text
被系统夹 -> 找敏感词 -> 替换直给表达 -> 弱化血腥画面 -> 强化逃生动作
```

## 开篇必备

- 区分系统夹和编辑夹。
- 先做敏感词排查。
- 人吃人等现实伦理敏感要侧写。
- 血、刀、死人等画面减少直给堆叠。
- 女主必须有判断和行动。

## 期待来源

- 暗示保留危险。
- 逃生动作提供紧张感。
- 女主判断建立能力。
- 画面弱化提高过审概率。
- 危机服务人物，不服务猎奇。

## 断档风险

- 只按男频尺度判断。
- 只替换一个词，不改画面。
- 血腥物件堆叠。
- 人类吃人直给。
- 弱化后危机消失。

## 训练任务

把一段血腥开篇先做词级替换，再做画面级侧写。最后检查女主是否仍有明确逃生目标和行动。
