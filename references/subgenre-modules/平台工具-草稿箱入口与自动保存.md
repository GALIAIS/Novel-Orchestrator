# 平台工具-草稿箱入口与自动保存

依据案例：`batch-204` 案例 1016。

## 读者承诺

创作者需要能快速找回自动保存草稿，平台工具知识要把入口路径写清。

## 核心循环

```text
写作中断 -> 自动保存 -> 找入口 -> 打开写文章 -> 右侧草稿箱
```

## 开篇必备

- 说明自动保存不是草稿入口。
- 写清重新进入编辑页的方法。
- 标出草稿箱所在位置。
- 说明适用场景。

## 期待来源

创作者相信中断后的草稿能找回，工作流不会被平台入口打断。

## 断档风险

- 只知道自动保存，不知道入口。
- 平台功能说明缺失。
- 创作者误以为草稿丢失。
- 长文中断后无法续写。

## 训练任务

写一条 PC 端找回草稿 FAQ：5 步说明从进入写文章到打开右侧草稿箱。
