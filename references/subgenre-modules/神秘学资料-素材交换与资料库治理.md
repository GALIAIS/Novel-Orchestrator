# 神秘学资料-素材交换与资料库治理

依据案例：batch-376 案例 1876

## 读者承诺

神秘学和克苏鲁资料要按完整度、来源和用途治理。资料库越大，越要避免未整理材料污染正文。

## 核心循环

```text
题材需求 -> 搜集资料 -> 标注可用性 -> 交换补缺 -> 按剧情抽取
```

## 开篇必备

- 明确缺什么资料。
- 已有素材分类。
- 可用/未整理/残缺标记。
- 准备服务的剧情场景。

## 期待来源

- 神秘学质感更具体。
- 仪式、道具、怪物有来源感。
- 设定不只是空名词。
- 素材能转成事件压力。

## 断档风险

- 资料堆砌不入戏。
- 未整理资料直接使用。
- 来源不清导致误用。
- 交换素材没有质量边界。

## 训练任务

把一个资料库分成可用、待整理、残缺三类，再抽取 5 个意象绑定具体剧情事件。
