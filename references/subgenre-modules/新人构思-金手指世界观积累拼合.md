# 新人构思-金手指世界观积累拼合

依据案例：batch-265 案例 1322

## 读者承诺

帮助新人把零散金手指灵感扩成可写故事：先补足可调用样本，再把能力、世界规则、代价、冲突、主角欲望拼成闭环。

## 核心循环

```text
灵感碎片 -> 样本复盘 -> 提取能力/规则/代价 -> 拼合世界观 -> 生成前三章事件
```

## 开篇必备

- 不要只展示金手指名字，要展示它改变了什么现实困境。
- 世界观必须给金手指制造限制和代价。
- 主角欲望要和能力使用绑定。
- 前三章至少出现一次能力试用、一次规则碰壁、一次收益反馈。

## 期待来源

期待来自“这个能力在这个世界会怎么用”。能力越能触发现有秩序的漏洞、代价和利益争夺，读者越愿意跟。

## 断档风险

- 只有能力，没有世界规则。
- 只有世界观，没有主角收益。
- 金手指太万能，冲突消失。
- 样本积累不足，设定只能停留在一句话脑洞。

## 训练任务

拿一个金手指点子，强制配 5 个样本来源：能力样本、规则样本、代价样本、敌人样本、爽点样本。每个样本只提取一个可用零件。
