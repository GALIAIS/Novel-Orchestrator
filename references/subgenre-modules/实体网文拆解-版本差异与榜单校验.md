# 实体网文拆解-版本差异与榜单校验

依据案例：batch-384 案例 1919

## 读者承诺

实体网文可以拆，但要先核对首发平台、连载时间、网站成绩和版本删改。连载版与实体版服务不同学习目标。

## 核心循环

```text
确认作品 -> 核对时间与成绩 -> 对照两个版本 -> 标记删改 -> 按目标提炼技法
```

## 开篇必备

- 作品、作者和首发平台。
- 连载与出版时间。
- 原版成绩证据。
- 章节切分和内容差异。

## 期待来源

- 连载版展示追读机制。
- 实体版展示压缩和润色。
- 对照能看见编辑选择。
- 老作品的有效结构得到保留。

## 断档风险

- 把出版年份当创作年份。
- 假定实体版等于原始连载。
- 用出版书研究实时追订却不对照原版。
- 没有正文证据就判断版本优劣。

## 训练任务

对照同一作品两个版本的前三章，标出切分、删减、润色、钩子和信息密度变化。
