# 教程总纲-主目录顺序学习

依据案例：`batch-093` 案例 461。

## 读者承诺

总纲是学习路由，不是链接迷宫。按主目录顺序建立轮廓，遇到套娃链接先回主目录，旧帖服从新版置顶。

## 核心循环

```text
进入总纲 -> 选一个分类 -> 读少量文章 -> 做笔记 -> 回主目录 -> 融会贯通
```

## 开篇必备

- 确认是否有新版置顶。
- 只把总纲当主目录。
- 每轮只读一个分类。
- 套娃链接不要无限点。
- 缺失链接先标记，不中断学习。

## 期待来源

- 新人知道学习顺序。
- 理论逐渐形成轮廓。
- 重复链接变成复习。
- 旧内容被新版内容替换。

## 断档风险

- 链接套链接迷路。
- 一次想吞完整个体系。
- 旧帖和新帖混用。
- 看了很多但不知道训练目标。

## 训练任务

每次只读总纲一个分类的 3 篇文章，做 5 条笔记后回主目录，最后写出一个下一步训练目标。
