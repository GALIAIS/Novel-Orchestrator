# 侧面塑造-他人视角钩子

依据案例：`batch-032` 案例 160。

## 读者承诺

用他人视角塑造主角时，每个观察者都要强化主角反差，并继续拉高悬念。

## 核心循环

```text
日常场景 -> 旁人发现异常 -> 亲近者强化 -> 专业者小解答 -> 更大悬念
```

## 开篇必备

- 主角有明显反差。
- 观察者视角各有功能。
- 小解答必须带出更大问题。
- 侧面描写最终仍在写主角。

## 期待来源

- 路人看出不协调。
- 亲近者证明异常不是偶然。
- 专业者解释一部分秘密。
- 后果越严重，冲突越大。

## 断档风险

- 总不给重点，读者跑掉。
- 旁人戏份盖过主角。
- 反差不够强。
- 只有描写，没有新的期待。

## 训练任务

写三段主角登场：路人、亲近者、专业者各看见一个信息，每段都必须制造新问题。
