# 爆款瑕疵-核心优点遮蔽

依据案例：`batch-104` 案例 519。

## 读者承诺

爆款也有瑕疵，但强更新、强剧情和主导爽点可以遮蔽部分问题。新人不能只学爆款的瑕疵，必须先确认自己有没有足够强的核心优点。

## 核心循环

```text
局部瑕疵 -> 核心优点 -> 读者获得价值 -> 忽略缺陷 -> 持续追读
```

## 开篇必备

- 读者当前最关心的价值。
- 作品主导优点。
- 缺陷是否影响当前理解。
- 缺陷是否影响当前情绪。

## 期待来源

- 高频更新。
- 基础剧情扎实。
- 主线推进强。
- 成长和爽点持续。

## 断档风险

- 拿爆款缺陷当借口。
- 自己没有遮蔽缺陷的优点。
- 压抑世界观超过读者承受。
- 关键设定缺解释影响当前剧情。

## 训练任务

列一本爆款的三项瑕疵和三项优点，再列自己作品的对应项。若优点不足，就先修瑕疵。
