# 双线复仇-主支线卷尾合流

依据案例：`batch-108` 案例 536。

## 读者承诺

一卷里有两条复仇线时，读者需要清楚哪条是本卷主线，哪条是在积压下一卷情绪。主线负责推进，支线负责压迫和蓄力，卷尾必须合流释放一部分情绪。

## 核心循环

```text
主线复仇推进 -> 支线压迫加码 -> 情绪下压 -> 卷尾交汇 -> 部分释放 -> 下一卷反攻
```

## 开篇必备

- 明确本卷主敌人。
- 支线只承担因果和压迫，不抢主线篇幅。
- 每次支线出现都要积累读者怒气。
- 卷尾让支线和主线产生交点。
- 留出下一卷正式反攻目标。

## 期待来源

- 读者知道主角先解决谁。
- 支线压迫越积越深。
- 主角从忍让转为反击。
- 卷尾释放后仍保留更大反攻。

## 断档风险

- 两条线平均写，导致本卷目标模糊。
- 支线只压抑不释放。
- 卷尾合流牵强。
- 下一卷反攻对象没有提前蓄力。

## 训练任务

写一张双线复仇表：主线写 5 个推进事件，支线只写 3 个压迫事件和 1 个卷尾释放点。
