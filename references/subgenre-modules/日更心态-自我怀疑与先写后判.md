# 日更心态-自我怀疑与先写后判

依据案例：`batch-234` 案例 1170。

## 读者承诺

日更作者会反复自我怀疑；先完成当天最低字数，再判断主线、爽点、人物和金手指是否微调。

## 核心循环

```text
早晨怀疑 -> 记录问题 -> 先写最低字数 -> 写后复盘 -> 明天继续
```

## 开篇必备

- 早晨不做切书决定。
- 怀疑点写成清单。
- 先完成当天更新。
- 写后再做小复盘。

## 期待来源

作者用流程吸收情绪波动，读者不断更，作品有机会在持续产出中修正。

## 断档风险

- 情绪低谷直接切书。
- 只怀疑不产出。
- 每天重判项目生死。
- 不复盘具体问题。

## 训练任务

每天早上只列怀疑点，不切书；写完最低字数后，用 10 分钟检查主线、爽点、人物、金手指各一项微调。
