# 玄幻小白文-第一页金手指应对

依据案例：batch-290 案例 1450

## 读者承诺

新人写玄幻小白文，移动端第一页就要出现危机，并让金手指介入应对。

## 核心循环

```text
第一页危机 -> 金手指触发 -> 小反转 -> 外挂承诺成立 -> 下一次升级期待
```

## 开篇必备

- 几百字内的危机。
- 主角被压迫点。
- 金手指触发。
- 可见效果。
- 一个小爽点。
- 后续升级钩子。

## 期待来源

读者迅速确认主角凭什么变强，愿意继续看后续升级和反打。

## 断档风险

- 三五千字才第一次用金手指。
- 前两个波动都靠普通办法解决。
- 模仿大神慢热。
- 解释体系太多，效果太少。

## 训练任务

写一个 500 字内的玄幻开篇：危机、触发、应对、小反转四步必须齐。
