# 理论训练-显性规则与反馈驯化直觉

依据案例：`batch-401` 案例 2002

## 读者承诺

把理论转成写前预测、短样本和前后对照，让反馈逐步校正判断并形成可复现直觉；不先用“有无天赋”给作者定性。

## 核心循环

```text
理论提出单一预测 -> 完成短样本 -> 记录作者预期 -> 收集理解与行为反馈 -> 分离症状和方案 -> 单点修改 -> 跨样本复现 -> 压缩成可复查直觉
```

## 开篇必备

- 理论命题能转成具体读者反应预测。
- 一轮只调整一个主要变量。
- 样本有明确字数和完成期限。
- 写前保存作者对目标、阻力和章末期待的判断。
- 反馈问题优先询问理解、期待和停顿位置。
- 保留初稿、修改稿与验证结果。

## 期待来源

作者从抽象的天赋焦虑转向可完成实验，每轮都能看到预期与实际差在哪里，并以跨文本复现累积可靠感。

## 断档风险

- 把理论当成功保证或身份门槛。
- 只在作品完成后进行漂亮解释。
- 一次同时修改太多变量。
- 把单个读者偏好当成普遍规律。
- 直接照抄读者方案而不核对重复症状。
- 用一次成绩证明机制已经稳定。
- 直觉形成后拒绝在新情境重新验证。

## 训练任务

选择一条理论，为 1500 字场景写三个可核对预测；让三名陌生读者复述目标、难点和章末问题，只修一个重复症状，再以同样问题测试第二版和第二个样本。
