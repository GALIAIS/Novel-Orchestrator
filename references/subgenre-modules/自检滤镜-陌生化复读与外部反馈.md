# 自检滤镜-陌生化复读与外部反馈

依据案例：batch-348 案例 1736

## 读者承诺

作者用固定流程制造陌生感，减少刚写完时的自恋滤镜，让修稿更接近真实读者视角。

## 核心循环

```text
刚写完自评 -> 时间间隔 -> 语音/朗读复检 -> 外部反馈 -> 定点修稿
```

## 开篇必备

- 写完后不要只靠当场自评。
- 至少使用一种媒介转换：语音播放、大声朗读、手机预览。
- 至少使用一种时间间隔：隔天、两天、七天。
- 外部反馈优先问第一处卡点。
- 修稿只先改最影响阅读的问题。

## 期待来源

作者能持续看见自己原先看不见的卡点。

## 断档风险

- 刚重构完就觉得自己又行了。
- 只换设备，不换读者视角。
- 外部打击只造成情绪，没有转成修稿项。
- 不愿看第二遍。
- 一次性大改太多，失去判断。

## 训练任务

每章完成后做三步：隔 48 小时复读、语音播放、请人指出第一跳读点。只记录前三个问题。
