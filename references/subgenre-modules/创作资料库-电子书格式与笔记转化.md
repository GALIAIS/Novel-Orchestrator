# 创作资料库-电子书格式与笔记转化

依据案例：`batch-211` 案例 1053。

## 读者承诺

写作资料库的价值在于可阅读、可标注、可转化，而不是单纯囤书。

## 核心循环

```text
资料收集 -> 格式筛选 -> 阅读标注 -> 方法摘录 -> 写作清单
```

## 开篇必备

- 优先选择适合标注的格式。
- 按写作问题给资料分类。
- 把书中方法转成检查表。
- 定期回流到大纲、人物、场景或修稿。

## 期待来源

创作者能在需要结构、人物、场景方法时，从资料库里拿出可执行步骤。

## 断档风险

- 只囤书不阅读。
- 粗扫 PDF 难标注。
- 笔记只有摘抄，没有使用场景。
- 资料分类只按书名，不按问题。

## 训练任务

选 3 本写作书，各整理 10 条“使用场景 + 操作步骤 + 检查问题”的可操作笔记。
