# 都市异能命名-梗化微创新

依据案例：batch-273 案例 1365

## 读者承诺

都市异能命名先服务能力效果和人物记忆点，再追求炫酷。梗、学科、文学、音乐、玩具都能成为能力微创新素材。

## 核心循环

```text
能力类别 -> 效果/限制 -> 人物个性 -> 素材来源 -> 名称记忆点
```

## 开篇必备

- 先定能力效果，不先造酷名。
- 能力名最好暗示效果或人设。
- 梗异能要有使用场景和限制。

## 期待来源

读者期待从名字猜能力、从效果看升级、从角色使用中记住这个异能。名字和效果对上，才有传播性。

## 断档风险

- 名字很炫但能力模糊。
- 梗只好笑一次，后续无法升级。
- 能力和人物个性脱节。

## 训练任务

同一个能力写 5 个名字：效果直译、人物个性、网络梗、学科术语、文学/音乐梗。选最能让读者记住的一版。
