# 五折线上行-虚晃与关系补强

依据案例：`batch-241` 案例 1205。

## 读者承诺

五折线里虚晃一枪不能替代上行；标为上行的段落必须让处境、关系或期待变好。

## 核心循环

```text
误会下行 -> 虚晃退让 -> 补关系上行 -> 反派加压 -> 爽点修复关系
```

## 开篇必备

- 明确误会来源。
- 写出虚晃动作。
- 补足女主/读者期待回升。
- 最后同时完成打脸和情感兑现。

## 期待来源

读者看到主角暂退不是变弱，而是关系已经暗中松动，等待最后爆发。

## 断档风险

- 把“我不行”当上行。
- 虚晃没有情绪回升。
- 女主心理变化缺失。
- 爽点只打脸，不修复关系。

## 训练任务

检查五折线每个上行段，写出“哪里变好了”；说不出来就补关系、信息或处境改善。
