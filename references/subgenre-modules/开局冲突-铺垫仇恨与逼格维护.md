# 开局冲突-铺垫仇恨与逼格维护

依据案例：batch-374 案例 1868

## 读者承诺

开局冲突要让读者先理解仇恨、因果和主角布局。没有铺垫的危机不刺激，只会显得突兀。

## 核心循环

```text
仇恨铺垫 -> 危机倒计时 -> 主角有逼格行动 -> 因果清楚 -> 主线推进
```

## 开篇必备

- 反派为什么该被打。
- 主角为什么知道局势。
- 倒计时危机。
- 主角行动的逼格。

## 期待来源

- 天然仇恨或充分铺垫。
- 主角预判和布局。
- 危机临近。
- 读者期待破局。

## 断档风险

- 硬加原创反派。
- 主角低逼格埋伏。
- 无关保安打脸抢节奏。
- 定时炸弹信息分散。

## 训练任务

删掉一个无关打脸桥段，补足反派仇恨和主角预判依据，把开头改成直接推进主线危机。
