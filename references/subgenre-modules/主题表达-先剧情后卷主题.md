# 主题表达-先剧情后卷主题

依据案例：`batch-102` 案例 508。

## 读者承诺

新人先把剧情写顺，再谈主题。主题不是口号，而是人物在卷级剧情中的选择、变化和代价被读者看见之后，自然逆推出的价值判断。

## 核心循环

```text
剧情顺畅 -> 卷三幕式 -> 人物成长 -> 关键选择 -> 卷主题 -> 全书主题
```

## 开篇必备

- 先有能跑通的剧情。
- 一卷内有清晰起承转合。
- 主角有可见成长或变化。
- 主题通过选择表达，不提前喊口号。

## 期待来源

- 读者关心人物如何选择。
- 选择带来代价。
- 结尾反证或兑现卷主题。
- 多卷主题共同形成全书主题。

## 断档风险

- 新人先设计宏大主题，剧情反而僵硬。
- 主题偏激或三观不稳。
- 用作者宣言替代人物行动。
- 阅历和表达能力不足时强写深刻。

## 训练任务

写一卷三幕式，只标主角三次选择和选择代价。最后用一句话逆推卷主题，不许先写主题再硬套剧情。
