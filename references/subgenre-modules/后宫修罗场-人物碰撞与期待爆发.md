# 后宫修罗场-人物碰撞与期待爆发

依据案例：`batch-230` 案例 1149。

## 读者承诺

修罗场爽点来自前期有期待的女性角色发生碰撞，释放争夺、反差、未知结果和角色特色。

## 核心循环

```text
前期期待 -> 女主会面 -> 表面和气 -> 暗中较劲 -> 特色爆发 -> 结果悬念
```

## 开篇必备

- 参与者都是读者已在意的女角色。
- 每个女角色有独立卖点。
- 会面理由自然。
- 明面礼貌和暗中冲突同时存在。

## 期待来源

读者想看重要人物碰撞后的未知结果，也享受优秀女角色围绕主角争夺的得意感。

## 断档风险

- 女角色没有前期期待，只剩吵架。
- 所有人说话同质化。
- 修罗场没有结果压力。
- 把休闲桥段写成主线阻滞。

## 训练任务

设计 1500 字修罗场：两名女角色各带一个前文承诺，表面三轮礼貌对话，暗中各出一次招，结尾留下主角必须处理的结果。
