# 三国穿越-史实锚点与知识差实证

依据案例：`batch-399` 案例 1992

## 读者承诺

用明确历史节点和身份危机锚定开篇，让穿越知识在现实制度与资源限制下完成第一次改命，并随历史偏离逐步失去万能预知。

## 核心循环

```text
历史节点既定风险 -> 身份与知识差确认 -> 现实限制下选择 -> 局部结果改变 -> 偏离削弱旧知识 -> 重新调查扩大影响
```

## 开篇必备

- 说明时间、地点、身份和当前风险。
- 选定正史、演义或架空混合口径。
- 第一条历史知识准确、具体且可执行。
- 官职、称谓和信息速度有最低核对。
- 知识应用立即改变局部结果。
- 建立偏离后失效信息台账。

## 期待来源

读者既认得熟悉历史节点，又看到主角在限制中用信息改命；每次成功都会让后续答案更不可靠，迫使他真正参与新历史。

## 断档风险

- 把穿越本身当成能力兑现。
- 只说知道历史、不采取行动。
- 正史演义口径混乱。
- 文白无功能地频繁切换。
- 历史偏离后所有人仍照原轨运行。
- 创意或考据压过当前冲突。

## 训练任务

用一条知识差写 2000 字“节点—风险—选择—偏离”闭环；结尾列三项因此失效的未来信息，下一章必须重新调查而非照答案行动。
