# 铺垫解释-删减优先与事件包装

依据案例：batch-384 案例 1917

## 读者承诺

孤立说明先删减或延后；真正必要的信息应通过行动、阻力和后果呈现，只有能形成完整冲突时才独立成情节。

## 核心循环

```text
孤立信息 -> 删除/延后判断 -> 行动化呈现 -> 必要时独立冲突 -> 回流主线
```

## 开篇必备

- 本章核心目标。
- 读者当前必须知道的信息。
- 信息影响选择的时刻。
- 事件化后的实际后果。

## 期待来源

- 读者通过结果理解规则。
- 信息直接改变人物行动。
- 支线完成后改变主线条件。
- 解释不再打断节奏。

## 断档风险

- 什么都想解释清楚。
- 为补字数单开说明章。
- 支线与主线只有表面关系。
- 信息没有行动和后果。

## 训练任务

从章纲挑五条说明，删两条、延后两条，只把一条改成会改变人物选择的事件。
