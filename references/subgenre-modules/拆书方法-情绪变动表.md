# 拆书方法-情绪变动表

依据案例：`batch-166` 案例 830。

## 读者承诺

拆书时不只记录剧情节点，还要记录情绪高低变化，用情绪落差反推节奏控制方法。

## 核心循环

```text
章节节点 -> 角色目标 -> 行动交叉 -> 情绪升降 -> 节奏判断 -> 回写创作
```

## 开篇必备

- 每个节点写清角色目标。
- 标出主角、配角、反派行动线交叉。
- 记录当前情绪升、降、平或转。
- 写明下一处期待。

## 期待来源

作者期待从优秀作品中看见商业网文“有迹可循”的部分，尤其是情绪落差如何构成节奏。

## 断档风险

- 拆书只复述剧情。
- 不标情绪变化，无法指导自己写。
- 只看主角线，忽略上层意志和配角目标。
- 表格填完不回写创作。

## 训练任务

拆 3 章，每章 5 个情绪节点。每个节点必须写明“为什么升/降/转”，最后总结 3 条可回写技巧。
