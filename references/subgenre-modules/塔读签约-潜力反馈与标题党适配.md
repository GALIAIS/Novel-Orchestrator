# 塔读签约-潜力反馈与标题党适配

依据案例：batch-308 案例 1536

## 读者承诺

平台签约反馈要看明确结果，不把“有潜力”当通过。若平台偏强标题、强卖点，书名和简介要先贴平台风。

## 核心循环

```text
提交签约 -> 收到潜力反馈 -> 判断能签/不能签/继续观察 -> 改包装或续写 -> 再提交
```

## 开篇必备

- 记录官方门槛和实际编辑要求。
- 区分潜力话术和签约动作。
- 检查书名是否够平台风。
- 继续写到要求字数时同步优化简介、卖点、前三章。

## 期待来源

期待来自明确节点。作者知道什么时候继续、什么时候重写、什么时候换包装，不靠猜编辑语气消耗心态。

## 断档风险

- 把“有潜力”当签约前兆。
- 只补字数，不改平台包装。
- 书名太平，不符合强刺激平台。
- 没有止损条件，反复等待。

## 训练任务

写一张签约反馈表：平台、当前字数、要求字数、反馈原文、明确状态、书名强化版、下一次提交节点。
