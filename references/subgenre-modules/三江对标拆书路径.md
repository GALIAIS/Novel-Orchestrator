# 三江对标拆书路径

依据案例：`batch-002` 案例 007。

## 读者承诺

给新人一条从市场样本进入写作实践的路径：先看正在被验证的书，再拆同类结构，最后才抽象理论和开自己的书。

## 核心循环

```text
扫榜 -> 选同题材样本 -> 拆前三章/第一单元 -> 提取共性 -> 找可对标点 -> 写自己的替换方案
```

## 开篇必备

- 至少 3 本同题材近期样本。
- 每本只先拆前三章和第一个闭环。
- 明确哪些是题材共性，哪些只是单本作者个人风格。

## 期待来源

- 市场样本证明该题材仍有读者。
- 多本样本交叉后，能看出稳定承诺。
- 对标书提供开篇节奏、核心梗、二级结构和爽点兑现方式。

## 断档风险

- 只看目录不读正文。
- 只抄设定，不拆情绪链和结构。
- 把单本爆点当成题材必需。
- 理论先行，导致开书没有样本支撑。

## 训练任务

建立 5 本对标表。每本记录书名、题材承诺、第一章钩子、第一单元闭环、核心梗、可迁移点、禁止照搬点。
