# 理论反应-积累实践与人设感悟

依据案例：`batch-249` 案例 1244。

## 读者承诺

理论反应模块强调新人学习不是背术语，而是让已有积累在理论框架下被重新解释，再通过实践转成能力。

## 核心循环

```text
阅读/写作积累 -> 接触理论 -> 旧经验被命名连接 -> 形成感悟 -> 小场景实践验证
```

## 开篇必备

- 先列出自己已有的阅读或写作经验。
- 只选一条理论原则，不同时套多个术语。
- 把感悟落到人物动机、压力、选择和行动。
- 用短场景验证，而不是只写心得。

## 期待来源

学习者获得的爽点是顿悟感：过去模糊知道的好坏，终于被理论解释清楚，并能进入下一次练习。

## 断档风险

- 没有积累，理论变成空话。
- 没有理论，经验停在感觉。
- 没有实践，感悟不能转成写作能力。
- 人设理解只停在标签，不进入行为选择。

## 训练任务

选择一条人设理论，配 3 个读过的角色例子，再写一个 500 字小场景验证。验收只看人物是否通过行动体现动机。
