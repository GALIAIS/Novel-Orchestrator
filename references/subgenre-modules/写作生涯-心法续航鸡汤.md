# 写作生涯-心法续航鸡汤

依据案例：`batch-166` 案例 828。

## 读者承诺

用作家经验和心法材料缓解失败感、枯燥感和动笔恐惧，帮助作者回到持续产出。

## 核心循环

```text
写作受挫 -> 读取心法 -> 失败感正常化 -> 降低完美要求 -> 完成最小动作 -> 继续产出
```

## 开篇必备

- 明确这不是技巧训练。
- 只服务长期续航和情绪恢复。
- 读完后必须接一个最小写作动作。
- 不把鸡汤当成逃避写作的理由。

## 期待来源

作者期待知道卡文、失败、不满意并不是自己独有的问题，从而获得继续写的心理余量。

## 断档风险

- 沉迷励志阅读，不回到产出。
- 用心法替代结构和技巧训练。
- 把失败感浪漫化，不做最小行动。
- 只追求完美，迟迟不动笔。

## 训练任务

做 7 天最小写作动作表：每天写一个小目标、一个完成结果、一句失败感处理，不记录宏大感想。
