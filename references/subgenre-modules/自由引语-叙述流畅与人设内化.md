# 自由引语-叙述流畅与人设内化

依据案例：`batch-345` 案例 1721

## 读者承诺

自由直接引语和自由间接引语能让人物口吻、判断和内心自然进入叙述，提升流畅性和人设表现。

## 核心循环

```text
人物念头 -> 选择引语方式 -> 减少引导语 -> 融入角色口吻 -> 叙述更顺
```

## 开篇必备

- 明确是谁在想。
- 保留人物口吻。
- 不让叙述身份混乱。
- 只在需要贴近内心时使用。
- 以读者舒服为准。

## 期待来源

读者期待自然理解角色判断和情绪，不想被频繁的“他说、他想、他认为”打断。

## 断档风险

- 为术语而术语。
- 人物和叙述者声音混乱。
- 引语方式频繁切换。
- 读者误判是谁在想。
- 语句技巧脱离人物和节奏。

## 训练任务

把一段内心戏写成直接引语、间接引语、自由直接引语、自由间接引语四版，比较流畅性和人物口吻。
