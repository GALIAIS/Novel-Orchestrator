# 颜值代入-帅感幻想与章评验证

依据案例：`batch-243` 案例 1213。

## 读者承诺

代入感不只来自普通自我，也来自理想自我；主角长得帅能增强社交、恋爱和装逼爽点。

## 核心循环

```text
主角颜值确认 -> 读者理想投射 -> 他人反应证明 -> 章评共鸣 -> 后续收益更顺
```

## 开篇必备

- 用他人反应确认外形。
- 给颜值带来实际事件收益。
- 让读者看到理想自我投射点。
- 避免主角自恋式自夸。

## 期待来源

读者愿意代入更优秀的自己，颜值是低门槛、强反馈的理想化优势。

## 断档风险

- 只写静态外貌，没有事件收益。
- 描写过度自恋。
- 颜值不影响情节。
- 忽略章评和读者反馈。

## 训练任务

写一个主角出场，用 3 个他人反应和 1 个实际收益证明颜值爽点，不直接堆形容词。
