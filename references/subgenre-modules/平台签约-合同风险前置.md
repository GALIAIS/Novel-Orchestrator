# 平台签约-合同风险前置

依据案例：`batch-046` 案例 227。

## 读者承诺

以稿费到账为目标时，签约规则就是创作准备的一部分。新人尤其要先查合同周期、签人/签作品、未成年规则和版权范围。

## 核心循环

```text
写作目标 -> 平台规则 -> 合同风险 -> 投稿决策 -> 退路平台
```

## 开篇必备

- 明确目标类型。
- 查平台签约规则。
- 查未成年签约方式。
- 准备替代平台。

## 期待来源

- 作者知道赚钱路径。
- 签约前避免长期风险。
- 平台和题材更匹配。
- 投稿不再只凭冲动。

## 断档风险

- 只准备大纲，不查合同。
- 签人网站误当签作品。
- 未成年用证件问题留下隐患。
- 没有退路平台。

## 训练任务

做一张目标平台签约风险表：签约主体、年限、版权范围、字数要求、未成年规则和替代平台必须齐全。
