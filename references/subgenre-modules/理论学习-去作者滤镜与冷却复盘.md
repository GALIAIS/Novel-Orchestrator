# 理论学习-去作者滤镜与冷却复盘

依据案例：`batch-235` 案例 1171。

## 读者承诺

学理论后要避免过度套用；作者天然有滤镜，需要冷却后站回读者视角复盘。

## 核心循环

```text
学习理论 -> 尝试使用 -> 发现读者观感错位 -> 冷却复读 -> 删除多余理论痕迹
```

## 开篇必备

- 分清理论该用和不该用。
- 不把理论当炫技工具。
- 存稿冷却七到十天。
- 复盘结构和观感，不只改病句。

## 期待来源

读者看到的是顺畅、有趣、能懂的故事，而不是作者硬塞理论的痕迹。

## 断档风险

- 刚学理论就到处套。
- 自以为读者视角，其实仍是作者滤镜。
- 复盘只改错字病句。
- 不结合成绩和反馈客观看书。

## 训练任务

把 2 万字封存 7 天，复读只标三类：理论过度、读者看不懂、读者觉得无聊。每类只改最明显的一个问题。
