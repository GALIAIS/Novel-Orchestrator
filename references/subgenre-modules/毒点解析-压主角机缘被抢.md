# 毒点解析-压主角机缘被抢

依据案例：`batch-098` 案例 486。

## 读者承诺

低调、苟、当谋士、暂时伏低都能写，但不能让读者感到主角的核心权益被配角拿走。主角可以让利，但必须换回主动权、尊重、资源或更大收益。

## 核心循环

```text
主角低调 -> 配角登场 -> 主角付出资源 -> 配角回馈主角 -> 主角获得更大收益 -> 低调不等于憋屈
```

## 开篇必备

- 主角先有可见位格。
- 配角强要有合理理由。
- 主角伏低必须有收益。
- 主角机缘不能被白拿。

## 期待来源

- 主角低调但掌控全局。
- 配角强大反衬主角更聪明。
- 主角付出后获得超额回报。
- 当小弟也能当出谋士位格。

## 断档风险

- 配角原本不如主角，却长期压主角。
- 主角资源供养配角，但没有回馈。
- 主角机缘被抢，作者还想让读者忍。
- 伏低做小没有战略意义。

## 训练任务

写一个主角主动辅佐配角的桥段。要求配角给出三项回馈：尊重、资源、后续剧情入口。少一项就重写。
