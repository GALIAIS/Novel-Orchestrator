# 家族修仙-主承诺血缘金手指

依据案例：`batch-054` 案例 267。

## 读者承诺

家族修仙要先立住家族承诺：家族危机、血缘身份、字辈称呼、利益绑定和不可替代金手指共同支撑主角成长。

## 核心循环

```text
家族危机 -> 主角绑定利益 -> 金手指破局 -> 家族奖励 -> 个人与家族互相反哺
```

## 开篇必备

- 第一矛盾最好是家族层面。
- 主角身份要能进入家族权力结构。
- 家族细节要有字辈和亲属称呼。
- 金手指不能随便替换成系统/灵眼。
- 家族成长线和个人成长线要明确主次。

## 期待来源

- 家族一荣俱荣、一损俱损。
- 主角解决家族困境会放大成群体爽点。
- 家族反哺主角，成长更有根。
- 血缘和权力结构提供长期矛盾。

## 断档风险

- 个人升级太强，家族变成背景板。
- 金手指没有独特性，会显得换皮。
- 仙侠写成武侠罩门，会让受众出戏。
- 外来户身份强行掌权，容易逻辑降智。

## 训练任务

设计家族修仙前三章：家族危机、主角身份、金手指独特用法、家族奖励四项必须出现。
