# 长篇大纲-支线减法

依据案例：`batch-025` 案例 122。

## 读者承诺

长篇进入大地图后，保留能推动主线高朝的支线，砍掉只增加风味却拖慢节奏的支线。

## 核心循环

```text
地图扩大 -> 支线膨胀 -> 判断影响 -> 删除低价值线 -> 回收必要元素 -> 主线提速
```

## 开篇必备

- 当前主线高朝。
- 每条支线的事件作用。
- 每条支线绑定的角色或势力。
- 删除后的影响评估。

## 期待来源

- 读者期待集中到主线。
- 角色戏份围绕事件存在。
- 地图扩展不等于节奏变散。
- 必要元素用更短路径回收。

## 断档风险

- 只会加角色、加势力、加事件。
- 支线删除后忘记回收关键要素。
- 舍不得删，导致高朝拖沓。
- 角色离开支线就失去存在理由。

## 训练任务

给一卷大纲做支线审计：每条支线写“保留理由”和“删除代价”。删除一条对高朝影响最小的线。
