# 震惊值-见证圈扩大

依据案例：`batch-064` 案例 317。

## 读者承诺

这个模块用于飞卢说书、直播、曝光、科普、点评、排名、学霸解题等见证型爽点。它承诺的是通过扩大见证者范围和反应可信度，放大同一个能力展示的爽感。

## 核心循环

```text
能力展示 -> 权威/大众见证 -> 可信震惊 -> 震惊圈扩大 -> 名气资源收割 -> 下一轮更大舞台
```

## 开篇必备

- 主角展示的东西必须让读者认可。
- 第一层见证者要有评价资格。
- 震惊反应要具体，不能只写“卧槽”。
- 有直播、榜单、会议、赛场、课堂等扩大载体。
- 震惊后要带来名气、资源、关系或新任务。

## 期待来源

- 读者期待更多人发现主角厉害。
- 见证圈从小圈扩到大圈。
- 权威误判被打破。
- 弹幕、水友、观众、专家连续反应。
- 震惊带来下一阶段收益。

## 断档风险

- 角色震惊但读者不认可。
- 只有尬惊，没有具体评价。
- 主角展示没有后续收益。
- 一直扩大观众但能力不升级。
- 平台风格不适配，盲学飞卢。

## 训练任务

同一个主角能力写三版展示：私下、公开、直播。分别写见证者、反应可信点、震惊后收益和下一轮更大舞台。
