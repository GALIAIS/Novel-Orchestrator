# 微表情素材-触发动作与多义校准

依据案例：batch-393 案例 1961

## 读者承诺

微表情只提供可观察证据，不直接宣判心理；通过触发、人物基线、台词矛盾和后续选择，让读者参与判断真正立场。

## 核心循环

```text
情境触发 -> 动作偏离基线 -> 台词形成一致或矛盾 -> 观察者作出可错解释 -> 后续选择验证
```

## 开篇必备

- 人物平常的动作与说话基线。
- 触发异常动作的具体事件。
- 同一动作至少两种合理含义。
- 能验证或推翻解释的后续选择。

## 期待来源

- 表面台词与细小动作不完全一致。
- 读者先于或落后于观察者猜真相。
- 对手根据破绽进行试探。
- 最终行动重新解释早期细节。

## 断档风险

- 把表情写成读心术词典。
- 每个动作后立刻翻译心理。
- 所有人都用相同眉眼套话。
- 动作没有触发和后续回收。

## 训练任务

建二十条“触发、基线、动作、三种含义”素材，再用两条写谈判场景，禁止直接说明心理并用结尾选择验证。
