# 同人台词-原汁原味与照搬边界

依据案例：batch-366 案例 1829

## 读者承诺

同人可用短台词、口头禅和招牌句保持角色原汁原味，但要控制比例，避免大段照搬原作或其他同人。

## 核心循环

```text
角色识别 -> 短台词/口头禅 -> 原汁原味 -> 新剧情推进 -> 控制复制比例
```

## 开篇必备

- 明确同人对象。
- 保留少量识别性台词。
- 新场景和新冲突。
- 不照搬其他同人。

## 期待来源

- 熟悉角色回来了。
- 口头禅触发记忆。
- 新剧情让二创成立。
- 不 OOC 的同时有新内容。

## 断档风险

- 台词和原文占全文大头。
- 轻小说原文大段复制。
- 抄其他同人桥段。
- 只有还原，没有新故事。

## 训练任务

写同人角色登场，只允许两句原作短台词，其余用新动作、新场景和新冲突完成还原。
