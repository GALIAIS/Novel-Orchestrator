# 作者收入认知-幸存者偏差校准

依据案例：batch-271 案例 1353

## 读者承诺

帮助新人把收入幻想校准成可执行阶段目标：先稳定写、稳定更、稳定签约和获得反馈，再谈高收入。收入认知不是劝退，而是防止因为错把可见大神当平均水平而反复太监。

## 核心循环

```text
行业信息 -> 排除幸存者偏差 -> 目标分层 -> 更新纪律 -> 作品质量反馈 -> 收入再校准
```

## 开篇必备

- 明确当前阶段：练笔、签约、稳定更新、首收、冲高。
- 设定能由自己控制的指标，如字数、天数、完本节点。
- 把榜单作者和大神收入从日常对标中剥离。

## 期待来源

作者端期待来自“我知道下一步该做什么”。当目标从月入幻想落到稳定产出、卖点优化、节奏训练，写作心态会更稳。

## 断档风险

- 把月入过万当普通新人基准。
- 看见社群成功样本后误判行业平均水平。
- 几万字没反馈就多开马甲、换题材、太监。
- 只讨论收入，不回到作品卖点和更新能力。

## 训练任务

写一张五档目标表：10 万字、签约、稳定更新 30 天、首个稳定收入、挑战收入。每档只写行动和验收标准。
