# 都市泄压流-不公平伤害与较真反击

依据案例：`batch-248` 案例 1240。

## 读者承诺

都市泄压流用现实不公平伤害激发压抑情绪，再让主角较真到底完成释放。

## 核心循环

```text
现实不公 -> 普通人忍让预期 -> 主角较真 -> 规则/资源反击 -> 读者泄压
```

## 开篇必备

- 不公平伤害具体。
- 读者现实中能共鸣。
- 主角反击有规则或资源。
- 控制尺度不过火。

## 期待来源

读者现实里常忍的事，主角替他们不忍，情绪立刻有释放口。

## 断档风险

- 纯情绪宣泄。
- 尺度过火被封禁。
- 对立过强不利于持续。
- 反击方式不合规。

## 训练任务

设计一个都市较真开篇：一个现实不公、一个合法反击路径、一个泄压点、一条尺度红线。
