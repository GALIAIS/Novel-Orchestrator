# 内投开头-金手指潜力明示

依据案例：batch-300 案例 1499

## 读者承诺

开头冲突平淡或金手指第一功能普通时，必须尽早明示升级潜力，让读者知道这只是冰山一角。

## 核心循环

```text
第一功能亮相 -> 旁观者震惊/迪化 -> 暗示多功能体系 -> 主角设定升级目标 -> 读者期待后续
```

## 开篇必备

- 第一功能解决当下问题。
- 旁观者反应放大价值。
- 明示还有更多功能或更高层级。
- 给出升级条件或下一阶段计划。

## 期待来源

- 读者知道金手指不止当前用法。
- 异象、误判、迪化增强卖点。
- 主角执行计划能解锁新能力。

## 断档风险

- 情绪折线清楚但冲突平淡。
- 核心卖点藏太深。
- 第一功能普通，读者以为全书都普通。

## 训练任务

给一个普通金手指加三条明示：一次异象、一个旁观者误判、一个主角升级计划。
