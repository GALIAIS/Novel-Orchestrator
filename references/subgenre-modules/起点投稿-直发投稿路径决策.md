# 起点投稿-直发投稿路径决策

依据案例：`batch-128` 案例 637。

## 读者承诺

新人投稿前先解决路径判断，而不是迷信“选哪个组”。直发和投稿都可能触发审核，投稿未必更容易；真正决定签约潜力的仍是书名、简介、前三章和卖点。

## 核心循环

```text
准备发书 -> 查官方规则 -> 比较直发/投稿 -> 评估更严格风险 -> 回到文本补强
```

## 开篇必备

- 书名和简介可用。
- 前三章完成。
- 核心卖点清楚。
- 查过官方投稿规则。
- 知道投稿可能比直发更严格。

## 期待来源

- 新人降低路径焦虑。
- 选择更符合当前准备程度。
- 官方规则压过传言经验。
- 注意力回到文本质量。

## 断档风险

- 把签约希望押在编辑组选择。
- 没看官方规则就投稿。
- 以为投稿一定比直发容易。
- 前三章弱却只研究流程。

## 训练任务

写一页投稿路径决策卡：直发优点、投稿优点、投稿更严格的风险、官方规则来源、自己前三章最弱的一项。
