# 轻小说人设-日常主线调和

依据案例：`batch-074` 案例 370。

## 读者承诺

这个模块用于轻小说、轻幻想、都市异能日常和角色卖点文。它承诺用强人设先吸住读者，再用日常互动提供陪伴感，用主线和设定保证长期方向。

## 核心循环

```text
人设亮相 -> 关系互动 -> 日常反应强化人设 -> 主线事件压入 -> 人设在事件中再证明
```

## 开篇必备

- 至少一个能被读者记住的核心人设。
- 角色反应要比设定说明更早出现。
- 日常桥段不能纯闲聊，要强化关系或埋主线。
- 主线压力要早露头，避免只有角色展示。
- 世界设定服务人设和冲突，不要单独堆。

## 期待来源

- 读者期待角色面对不同场景时会怎样反应。
- 女性角色卖点或群像关系提供持续互动。
- 日常带来陪伴感和节奏缓冲。
- 主线提供长期目标和升级空间。
- 角色在主线中被迫选择，能加深人设。

## 断档风险

- 只做人设贴标签，没有行动证明。
- 日常太散，和主线互不相干。
- 逻辑弱到破坏读者信任。
- 模仿大神级复杂结构，超出当前积累。
- 收集样本很多，却不判断自己要学哪条路线。

## 训练任务

为一个轻小说开篇写三章功能表：第一章人设亮相，第二章日常互动强化关系，第三章主线压力进入。每章都注明角色卖点、日常功能和主线推进。
