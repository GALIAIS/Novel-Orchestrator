# 悬念期待整活-断章张力组合

依据案例：batch-359 案例 1794

## 读者承诺

用一个清楚问题、一个即将发生的动作和一个结果前断章制造阅读拉力。整活儿负责让问题更显眼，断章负责把答案留到下一段或下一章。

## 核心循环

```text
异常局面 -> 清楚问题 -> 动作逼近答案 -> 整活放大注意力 -> 结果前断章
```

## 开篇必备

- 读者立刻知道想看什么结果。
- 动作已经启动或马上启动。
- 可能结果可被想象。
- 断章点卡在答案揭晓前。

## 期待来源

- 可视化动作。
- 荒诞或危险感。
- 多个可能结果。
- 下一段必须兑现答案。

## 断档风险

- 问题没成立就硬断。
- 整活和主线无关。
- 结果兑现拖太久。
- 悬念只有“神秘”，没有行动。

## 训练任务

写 5 个章末钩子，每个只允许一个动作和一个问题。检查读者能否在 3 秒内说出“我想看什么结果”。
