# 番茄开篇-前三章冲突打脸

依据案例：`batch-313` 案例 1562

## 读者承诺

番茄新人开篇先兑现刺激：前三章内让读者看到冲突、系统或能力、主角反击、情绪上行，而不是用长铺垫证明世界观完整。

## 核心循环

```text
主角入局 -> 危机压迫 -> 系统/能力出现 -> 主角主动反击 -> 打脸回收 -> 背景后补
```

## 开篇必备

- 第一章要有可感知危机。
- 前三章要有主角完成的反击。
- 系统、金手指、秘境关键规则尽量前置。
- 背景只解释当前冲突必需的信息。
- 情绪不能持续下行。

## 期待来源

读者期待“这个主角马上能爽起来”。压迫可以有，但必须迅速出现破局可能和实际回收。

## 断档风险

- 前几章只铺设定、铺身世、铺秘境背景。
- 主角长期挨压，没有动作。
- 系统发声太晚，能力兑现太晚。
- 反击只停在口号，没有具体打脸。

## 训练任务

把一个慢热开篇拆成三段：危机 300 字、能力 300 字、反击 400 字，删除暂时不影响冲突的背景说明。
