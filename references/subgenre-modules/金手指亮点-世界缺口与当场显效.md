# 金手指亮点-世界缺口与当场显效

依据案例：batch-350 案例 1750

## 读者承诺

金手指亮点要绑定世界缺口，并在开头事件中当场显效，让读者清楚它为什么有价值。

## 核心循环

```text
世界需要力量 -> 主角拥有对应金手指 -> 当场显效 -> 反馈升级 -> 第一爽点兑现
```

## 开篇必备

- 金手指定义简洁。
- 世界观力量价值明确。
- 主角力量正好对准缺口。
- 首次战斗中持续体现作用。
- 第一爽点不能拖太晚。

## 期待来源

读者看到世界需要这份力量，主角恰好有，并且马上用它改变局势。

## 断档风险

- 金手指又复杂又模糊。
- 反派能力比金手指更有存在感。
- 激活过程过长。
- 第一战拖 6000 字。
- 金手指获得的初始技能无法带来爽点。

## 训练任务

重写金手指开篇：300 字世界缺口，800 字激活，1500 字内改变局势。
