# 虚拟网游-主舞台副本工具与标签分流

依据案例：batch-389 案例 1944

## 读者承诺

先判断游戏是全书主舞台还是现实主线的工具，再让进入、选择、反馈、代价和现实后果形成主循环，并使标签与高频正文一致。

## 核心循环

```text
定位核心快感 -> 判定游戏主副地位 -> 进入并操作规则 -> 虚拟结果改变现实 -> 标签与长期循环对齐
```

## 开篇必备

- 主角必须进入游戏的理由。
- 一条能当场操作的规则。
- 选择、反馈和失败代价。
- 游戏结果对现实的不可逆影响。

## 期待来源

- 规则允许真实操作和策略。
- 虚拟胜负改变钱、身份、关系或身体。
- 现实压力迫使主角再次进入。
- 每轮主循环升级玩法或谜题。

## 断档风险

- 设定菜单替代玩法实证。
- 工具型游戏副本与现实主线脱节。
- 标题承诺纯网游，正文长期不进游戏。
- 用包装掩盖核心循环不足。

## 训练任务

为同一设定分别写游戏主舞台版与都市谜题工具版三章纲，按高频目标、最大高潮和标签承诺选择唯一主版本。
