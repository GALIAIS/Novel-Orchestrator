# 番茄数据-首秀判断与切书阈值

依据案例：batch-291 案例 1452

## 读者承诺

番茄数据用来辅助判断写、改、切，不是让新人提前陷入指标焦虑。

## 核心循环

```text
签约/开书 -> 看首秀前数据 -> 判断继续/修改/切 -> 对标拆书 -> 下一本迭代
```

## 开篇必备

- 是否签约。
- 完读率。
- 评分。
- 放量情况。
- 收益趋势。
- 读者反馈。
- 行动判断。

## 期待来源

作者理解数据含义后，能避免误切好苗子，也避免在明显无效的书上硬耗。

## 断档风险

- 一开始就沉迷数据。
- 不懂指标导致误判。
- 只讲坚持，不做复盘。
- 想做亚文化但基本功没过关。

## 训练任务

做一张番茄开书判断表，最后只给一个动作：继续、修改、切书。
