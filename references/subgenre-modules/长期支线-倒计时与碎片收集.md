# 长期支线-倒计时与碎片收集

依据案例：batch-347 案例 1735

## 读者承诺

倒计时和碎片收集可以制造长期压力与阶段进度，但更适合做支线积累和锦上添花，不宜单独撑主线。

## 核心循环

```text
长期缺口 -> 阶段寻找/倒计时推进 -> 每段带来新事件 -> 小爆发兑现 -> 回到主线
```

## 开篇必备

- 明确长期目标服务哪条主线。
- 每个碎片或倒计时节点都带来新信息。
- 阶段推进要有即时冲突。
- 每隔一段给小爆发。
- 最终拼合要改变局势，而不只是完成收集。

## 期待来源

读者看到进度条推进，同时每次推进都有新价值。

## 断档风险

- “还差几块”机械拖篇幅。
- 老式收集缺乏新鲜感。
- 倒计时时间线拉太长。
- 支线喧宾夺主。
- 收集物不改变能力、信息、关系或敌人。

## 训练任务

设计 5 段碎片收集支线，每段写出获得方式、即时冲突、新信息、阶段奖励和主线关系。
