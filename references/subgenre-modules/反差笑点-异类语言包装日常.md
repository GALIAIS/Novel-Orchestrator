# 反差笑点-异类语言包装日常

依据案例：`batch-097` 案例 481。

## 读者承诺

用高烈度类型语言包装低烈度日常事件，让读者同时看见“小事本体”和“大场面叙事”，由错位产生笑点。包装语言必须有前提支撑，否则只是空喊。

## 核心循环

```text
日常小事 -> 建立夸张前提 -> 借用热血/末日/谍战语言 -> 现实互动打断 -> 笑点落地
```

## 开篇必备

- 读者要先知道事件本体很普通。
- 高烈度语言要服务同一个前提。
- 现实对白或动作要把夸张拉回日常。
- 不能只堆华丽比喻，必须有笑点落点。

## 期待来源

- 类型错位的新鲜感。
- 小事被大词包装的夸张感。
- 日常角色突然进入热血/末日语境。
- 现实吐槽打破宏大叙述。

## 断档风险

- 没有前提，夸张语言不成立。
- 借用类型语言太长，笑点来得太晚。
- 原事件不清楚，读者不知道在反差什么。
- 所有人都夸张，没有正常锚点。

## 训练任务

把“妈妈炒菜”写成 300 字末日桥段。必须先给黑暗料理前提，最后用一句现实对白把末日叙述打断。
