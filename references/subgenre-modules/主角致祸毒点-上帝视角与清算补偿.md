# 主角致祸毒点-上帝视角与清算补偿

依据案例：batch-363 案例 1814

## 读者承诺

主角无意导致灾难时，读者会站上帝视角追责。后续不能轻易和伤害己方阵营的人真交好，必须有清算、补偿、假合作或洗白反转。

## 核心循环

```text
无意致祸 -> 读者知情 -> 主角发现/未发现 -> 清算补偿/反转洗白 -> 避免道德反噬
```

## 开篇必备

- 明确主角是否知情。
- 灾难责任不能糊。
- 后续关系选择要谨慎。
- 读者道德期待要被处理。

## 期待来源

- 主角承担或补偿。
- 真凶揭露。
- 假合作互相试探。
- 背锅侠反转翻盘。

## 断档风险

- 用“不知情”逃避全部责任。
- 和国仇敌人称兄道弟。
- 大量无辜者死亡被轻描淡写。
- 没有伏笔强行洗白。

## 训练任务

设计一个主角无意闯祸桥段，写出补偿行动、真正责任人和读者不骂主角的反转证据。
