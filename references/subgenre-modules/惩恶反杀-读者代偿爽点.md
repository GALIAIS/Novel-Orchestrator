# 惩恶反杀-读者代偿爽点

依据案例：`batch-109` 案例 545。

## 读者承诺

惩恶反杀的爽点来自读者代偿：读者希望现实中厌恶的不公、欺软怕硬和恶意，被主角果断消灭。反派越能承载读者厌恶，主角清算越痛快。

## 核心循环

```text
反派作恶 -> 厌恶积累 -> 主角受压 -> 果断反杀 -> 情绪释放
```

## 开篇必备

- 先写反派为什么该被惩罚。
- 恶行要能唤起现实经验。
- 主角反击必须果断。
- 反派求饶不能轻易稀释爽点。
- 释放后给出余波或代价。

## 期待来源

- 读者想替主角出气。
- 反派越恶，清算越爽。
- 主角做了读者现实中不能做的事。
- 打脸同时打掉读者生活中的相似不如意。

## 断档风险

- 反派还没招恨就被秒。
- 主角反杀犹豫，爽点软掉。
- 惩恶缺乏正当性。
- 只写暴力，不写读者情绪投射。

## 训练任务

写一个惩恶反杀桥段，前半段只负责让读者讨厌反派，后半段再写主角清算。检查反派恶行是否有现实代偿感。
