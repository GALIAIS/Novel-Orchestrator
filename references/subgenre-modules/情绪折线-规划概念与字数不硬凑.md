# 情绪折线-规划概念与字数不硬凑

依据案例：`batch-247` 案例 1231。

## 读者承诺

情绪折线是规划和转折意识，不是固定字数比例。

## 核心循环

```text
定情绪方向 -> 找关键转折 -> 按重要性分配篇幅 -> 写作中保留转折意识
```

## 开篇必备

- 明确本章主情绪方向。
- 标出关键转折。
- 重要情节详写。
- 不重要情节略写。

## 期待来源

读者感到情绪在变，而不是作者机械套比例。

## 断档风险

- 硬凑上下行字数。
- 为折线牺牲情节自然度。
- 忽略不同情节差异。
- 一章只有事件，没有转折意识。

## 训练任务

给一章大纲只标主情绪方向、关键转折、详写处，不规定上下行字数。
