# 认知异能-误判能力与规则边界

依据案例：batch-349 案例 1744

## 读者承诺

认知型异能靠误判、迪化和反误导制造爽点，但必须有清楚规则边界，避免前期爽后期崩。

## 核心循环

```text
对手误判 -> 主角获得能力 -> 主角诱导误判升级 -> 破局 -> 误判被修正或反噬
```

## 开篇必备

- 谁的认知有效。
- 认知强度和持续时间明确。
- 能力能否叠加有边界。
- 主角能误导，但不能无限刷。
- 揭穿后有代价。

## 期待来源

读者期待主角如何操纵对手想象，又担心规则反噬。

## 断档风险

- 能力过于万能。
- 对手随便脑补就能刷神技。
- 规则不严谨导致后期崩。
- 照搬日轻翻译腔。
- 碎片化叙事破坏中文网文阅读。

## 训练任务

设计三场误判战：第一次小胜、第二次升级、第三次反噬。每场写清误判来源和修正时刻。
