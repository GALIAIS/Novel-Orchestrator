# 新人扑街-复盘拆书再开

依据案例：`batch-158` 案例 790。

## 读者承诺

新人扑街后先复盘作品问题，再决定平台和新书方向；换平台不能替代拆书、做大纲和实践积累。

## 核心循环

```text
作品扑街 -> 复盘踩坑 -> 选方向 -> 拆同平台样本 -> 做新大纲 -> 多写多发验证
```

## 开篇必备

- 原平台和题材。
- 数据节点。
- 开篇问题。
- 节奏问题。
- 爽点问题。
- 更新问题。
- 新方向。
- 要拆的样本。

## 期待来源

期待来自下一本比上一本文更清楚地避坑，而不是靠换平台获得奇迹。

## 断档风险

- 失败后只怪平台。
- 不总结踩坑。
- 越急越乱，反复开新坑。
- 不按平台节奏写。

## 训练任务

对扑街稿列 5 个踩坑点，拆 3 本同平台同题材成绩书开篇，再写下一本前 3 章改法。
