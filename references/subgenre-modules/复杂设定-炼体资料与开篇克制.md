# 复杂设定-炼体资料与开篇克制

依据案例：batch-373 案例 1863

## 读者承诺

复杂等级设定适合作为素材库，不适合开篇一次性展示。每个能力都要被剧情调用，解决眼前问题。

## 核心循环

```text
资料库 -> 按剧情抽取 -> 能力可视化 -> 解决问题 -> 留下升级钩子
```

## 开篇必备

- 主角当前目标。
- 第一冲突。
- 一个能立刻看懂的能力效果。
- 暂不展示的设定边界。

## 期待来源

- 身体变化可视化。
- 能力解决具体困难。
- 境界提升带来明确奖励。
- 下一层能力有想象空间。

## 断档风险

- 开局沉迷等级树。
- 名词密度高于事件密度。
- 设定不服务冲突。
- 读者看不到升级收益。

## 训练任务

从复杂修炼资料中只抽 3 个能力，分别服务逃命、反击、收获，不解释完整体系。
