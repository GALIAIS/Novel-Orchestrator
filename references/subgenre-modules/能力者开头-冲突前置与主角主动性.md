# 能力者开头-冲突前置与主角主动性

依据案例：batch-360 案例 1800

## 读者承诺

能力者开头要尽早引爆和主角有关的冲突，让主角主动行动并兑现第一个爽点。弱书名时尤其不能慢热。

## 核心循环

```text
前200字冲突 -> 主角主动选择 -> 克服阻碍 -> 小爽点兑现 -> 背景后补
```

## 开篇必备

- 书名或前 200 字给兴趣点。
- 冲突直接关乎主角利益。
- 主角不是被安排着走。
- 第一个爽点在读者离开前出现。

## 期待来源

- 最强能力者的准备和判断。
- 被压制后的主动反击。
- 冲突意外性。
- 小 boss 或阻碍被处理。

## 断档风险

- 第一次冲突千字后才出现。
- 悬念暂时不关主角。
- 主角悲观、自嘲、被动。
- 前期配角专场抢主线。
- 提前揭露幕后安排降低意外性。

## 训练任务

把一个慢热能力者开头改成 600 字：前 200 字冲突，前 400 字主动选择，前 600 字小爽点或反击方向。
