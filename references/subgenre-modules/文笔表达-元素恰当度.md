# 文笔表达-元素恰当度

依据案例：`batch-144` 案例 716。

## 读者承诺

把“文笔好不好”从玄学变成可检查的表达问题：作者先明确想表达什么，再选择动作、语言、旁白、心理、神情等元素，让读者准确接收到意思和情绪。

## 核心循环

```text
表达目的 -> 选择元素 -> 组合句子 -> 检查恰当度 -> 删除华丽噪音 -> 更清楚地表达
```

## 开篇必备

- 一个明确要表达的事实或情绪。
- 至少一种能快速让读者看见的动作。
- 不超过需要的心理和旁白。
- 避免先堆漂亮词，再补表达目的。

## 期待来源

读者期待的是“我立刻懂这个人怎么了”，不是作者证明自己词汇量多。清楚、快、情绪准，比单句华丽更重要。

## 断档风险

- 作者不知道自己要表达什么。
- 动作、心理、旁白重复说同一件事。
- 句子很华丽，但读者不知道人物想要什么。
- 为了文笔拖慢冲突进入。

## 训练任务

用 5 种方式表达“主角很饿”：动作版、语言版、旁白版、心理版、动作加环境版。每版 80 字以内，最后删掉所有不影响表达的华丽词。
