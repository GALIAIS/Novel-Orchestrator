# 爱情鸿沟-主线结合拉大

依据案例：`batch-062` 案例 306。

## 读者承诺

这个模块用于单女主、恋爱日常、轻幻想恋爱线。它承诺的是：男女主不是靠误会硬拖关系，而是因为主线规则、身份、世界或价值差异形成鸿沟；两人越靠近，主线矛盾越大，甜感和危机同步升级。

## 核心循环

```text
互补吸引 -> 关系鸿沟 -> 主线事件触碰鸿沟 -> 靠近一次 -> 鸿沟拉大一次 -> 共同选择 -> 新关系期待
```

## 开篇必备

- 男女主各有残缺或缺口。
- 两人的缺口能互补，但不能立刻无成本在一起。
- 鸿沟要绑定题材主线，如身份、规则、世界、立场、创作者与角色等。
- 第一单元要让读者看见“靠近很甜，但靠近会出事”。
- 鸿沟不能只靠误会或嘴硬维持。

## 期待来源

- 读者期待两人如何跨过当前阻碍。
- 甜感兑现后，会出现更高一级的关系代价。
- 主线危机让恋爱线不散。
- 恋爱推进会改变世界、规则或双方处境。
- 两人共同选择能制造关系确认感。

## 断档风险

- 只撒糖，没有主线压力。
- 只写危机，恋爱承诺消失。
- 鸿沟只是误会，解释清楚就没故事。
- 每次阻碍都同级重复，没有拉大。
- 为拖关系强行降智。

## 训练任务

写一个三阶段爱情鸿沟：第一阶段是个人关系阻碍，第二阶段扩大到身份或规则阻碍，第三阶段扩大到世界或阵营阻碍。每阶段都写一个甜感兑现点和一个新代价。
