# 角色口癖-人设外显与场景兑现

依据案例：`batch-214` 案例 1068。

## 读者承诺

口癖是人设外显，不是单独贴标签；要在剧情和情绪到位时兑现。

## 核心循环

```text
人设底色 -> 场景铺垫 -> 冲突对峙 -> 口癖出口 -> 记忆点形成
```

## 开篇必备

- 口癖符合角色价值观。
- 第一次出现有场景支撑。
- 使用频率克制。
- 禁止在不合适场景硬说。

## 期待来源

读者在关键场景听到角色标志性回应，会把台词和人设绑定。

## 断档风险

- 没有人设支撑。
- 剧情没铺垫就喊口号。
- 频繁重复消耗台词。
- 套路过老导致尴尬。

## 训练任务

设计 3 句口癖：日常、对峙、胜利宣告。每句写触发条件、禁用场景和一次 500 字兑现片段。
