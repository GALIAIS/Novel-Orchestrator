# 无敌文-相对无敌迪化幕后

依据案例：`batch-190` 案例 949。

## 读者承诺

无敌文优先写相对无敌，用层级差、迪化和幕后流维持爽点，避免绝对无敌后期断档。

## 核心循环

```text
当前层级无敌 -> 外界误判 -> 主角解决矛盾 -> 众人迪化 -> 更高层级展开
```

## 开篇必备

- 明确是相对无敌还是绝对无敌。
- 主角仍有更高层级或更大局。
- 配角压力服务主角出手。
- 爽点反馈回到主角身上。

## 期待来源

读者期待主角轻松解决别人眼中的难题，并通过迪化、幕后身份和层级推进获得持续优越感。

## 断档风险

- 绝对无敌没有后续空间。
- 配角展开导致主角边缘化。
- 用主角无知或犯错制造阻碍。
- 限制主角显得主角蠢。

## 训练任务

写一个相对无敌小剧情：反派压迫配角，主角轻松解决，配角误判主角深不可测，并引出更高层级。
