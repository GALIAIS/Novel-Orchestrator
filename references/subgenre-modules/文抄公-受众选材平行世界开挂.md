# 文抄公-受众选材平行世界开挂

依据案例：`batch-087` 案例 433。

## 读者承诺

文抄公题材承诺的是主角用前世作品或系统内容在平行世界制造轰动。核心不是抄得多，而是抄对受众爱看的东西，并让胜负、热度和奖励连续兑现。

## 核心循环

```text
确认受众 -> 选择作品 -> 设置兑换/记忆限制 -> 发布对打 -> 舆论轰动 -> 多维奖励 -> 更大舞台
```

## 开篇必备

- 平行世界或其他规避现实抄袭感的设定。
- 明确目标平台和受众口味。
- 第一轮要有对手或榜单。
- 作品发布后要有讨论、专业认可、金钱收益或系统奖励。

## 期待来源

- 下一次主角会拿出什么作品。
- 对手如何被打败。
- 舆论如何发酵。
- 专业人士如何合作或跪舔。
- 舞台如何从小圈子扩大到全行业。

## 断档风险

- 选的作品不符合平台受众。
- 只写搬运，不写轰动和奖励。
- 无差别踩原创者，引发毒点争议。
- 平行世界差异不清，现实抄袭感太重。

## 训练任务

设计一个文抄单元：写清受众、作品、兑换限制、对手、发布舞台、热度反馈、金钱收益和下一轮更大舞台。
