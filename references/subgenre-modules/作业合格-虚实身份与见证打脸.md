# 作业合格-虚实身份与见证打脸

依据案例：`batch-252` 案例 1260。

## 读者承诺

虚实身份打脸模块用提前放出的底牌、敌人误判、被保护者受辱和当众证明完成爽点。

## 核心循环

```text
底牌伏笔 -> 旁人不信 -> 敌人半信半疑 -> 二次施压 -> 行动加证据打脸
```

## 开篇必备

- 主角先说出底牌。
- 亲近角色不相信。
- 敌人给出半信半疑反应。
- 最终证明必须有见证者。

## 期待来源

读者等着主角证明“我真认识老大”或“我真有底牌”，证明瞬间同时释放武力爽和身份爽。

## 断档风险

- 底牌没提前放，打脸突兀。
- 敌人只会莽，没有误判层次。
- 被保护者没有承压，主角出手正当性不足。
- 打完没有见证反应。

## 训练任务

写一个校园打脸桥段：提前放底牌、二次施压、被保护者承担屈辱、主角行动加证据证明。
