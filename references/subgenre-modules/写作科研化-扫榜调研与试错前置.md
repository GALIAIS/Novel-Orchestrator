# 写作科研化-扫榜调研与试错前置

依据案例：batch-359 案例 1792

## 读者承诺

写作前置调研不是拖延，而是降低项目路线错误。通过扫榜、看竞品、积累题材知识，先判断读者需求和差异点，再投入长篇写作。

## 核心循环

```text
题材冲动 -> 扫榜读竞品 -> 提取稳定承诺 -> 设计差异点 -> 小规模验证开头
```

## 开篇必备

- 至少知道同题材榜单样本。
- 清楚读者默认爽点。
- 明确自己和竞品的差异。
- 开头要验证调研结论。

## 期待来源

- 题材承诺踩准市场。
- 差异点不是凭空幻想。
- 开篇更快对接读者预期。
- 作者避免十几万字后才发现方向错。

## 断档风险

- 不看市场就谈创新。
- 把竞品当抄袭模板。
- 只码字不复盘路线。
- 调研过度但不开小样验证。

## 训练任务

连续三天扫同一题材榜单，每天记录 3 本竞品的主角目标、亮点、第一冲突和章末钩子，再写自己的 800 字开头验证。
