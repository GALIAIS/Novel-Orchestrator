# 宏大奇观开篇-现实锚定与尺度阶梯

依据案例：batch-391 案例 1952

## 读者承诺

用读者熟悉的现实参照逐级确认异常，让宏大奇观击穿可信世界，并把文明尺度谜团尽快移交给人物主线。

## 核心循环

```text
古老疑问定主题 -> 现实探索建立尺度 -> 微小异常出现 -> 多重观察复核 -> 奇观分级揭示并改变世界
```

## 开篇必备

- 一个贯穿全书的宏大秘密。
- 可核验且服务推进的现实参照。
- 异常信号、复核者与状态升级。
- 奇观对主角线的具体移交后果。

## 期待来源

- 人类渺小与宇宙孤独持续蓄压。
- 多层观察让不可能之物显得真实。
- 物理细节逐步放大奇观尺度。
- 揭示后留下明确来源和去向谜团。

## 断档风险

- 主角迟到且奇观不能移交主线。
- 现实资料独立科普、不改变异常状态。
- 错误事实反噬开篇权威感。
- 只复制古文、航天和巨物表面。

## 训练任务

按现实、异常、复核、揭示、后果五段写 1500 字奇观开篇，再补主角接线场景；奇观若不改变主角下一步就重做移交。
