# 战斗描写-克制下风破局击败

依据案例：batch-268 案例 1339

## 读者承诺

把影像打斗中可文字化的兴奋点提炼出来：敌方克制、主角下风、能力破局、击败瞬间。

## 核心循环

```text
敌方优势 -> 克制主角 -> 主角短暂下风 -> 核心能力破局 -> 击败瞬间释放
```

## 开篇必备

- 先建立敌方优势和具体克制。
- 主角下风要短，但要真实。
- 破局必须来自主角核心能力或前文伏笔。
- 击败瞬间要清楚写，不要一笔带过。

## 期待来源

期待来自压力和释放：读者先相信敌人难打，再看到主角用自己的优势反破局。

## 断档风险

- 试图复制镜头和配乐，文字效果弱。
- 敌人没有具体克制，胜利没重量。
- 主角下风太长变憋屈。
- 击败瞬间写虚，情绪没回收。

## 训练任务

写一场 800 字战斗，按“克制、下风、破局、击败”四段分配篇幅。每段都必须有一个具体动作。
