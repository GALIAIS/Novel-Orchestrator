# 套路内化-顺手脱稿自审

依据案例：`batch-041` 案例 205。

## 读者承诺

套路和公式是训练与排障工具。写顺时可以脱稿，但前提是已经具备判断文本真好还是自嗨的自审能力。

## 核心循环

```text
学习公式 -> 实践内化 -> 顺手脱稿 -> 自审校准 -> 必要时回工具
```

## 开篇必备

- 判断当前是卡住还是顺手。
- 情绪和期待要到位。
- 自审能发现问题。
- 需要时回到公式排查。

## 期待来源

- 写作不被框架卡死。
- 理论内化成直觉。
- 好看优先。
- 自审防止自嗨。

## 断档风险

- 没学会就反套路。
- 把通顺误当好看。
- 不会自审还脱稿。
- 顺手时忽略读者期待。

## 训练任务

写一章后先凭阅读感自审，再用公式复查，记录哪些问题靠直觉发现，哪些必须靠工具发现。
