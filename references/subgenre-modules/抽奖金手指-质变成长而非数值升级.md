# 抽奖金手指-质变成长而非数值升级

依据案例：`batch-119` 案例 595。

## 读者承诺

抽奖金手指的吸引力来自“下一次会出现什么新解法”，不是单纯抽到更强道具。长线要靠奖励类型、使用场景和主线解法质变维持期待。

## 核心循环

```text
随机奖励 -> 眼前用途 -> 意外副作用/伏笔 -> 阶段条件升级 -> 奖励类型质变 -> 主线解法变化
```

## 开篇必备

- 第一次抽奖要有新奇感。
- 随机范围要可控，不能什么都能出。
- 奖励要立刻服务一个事件。
- 无用物品要有伏笔、交易或笑点回收方式。
- 后续成长维度不能只是数值变大。

## 期待来源

- 下一次抽奖的不确定性。
- 特殊升级条件带来的解谜感。
- 奖励从物品、能力、关系、权限等维度变化。
- 看似无用的奖励后续被回收。

## 断档风险

- 奖励只从低级变高级。
- 抽奖只是附带彩蛋，不能推动主线。
- 随机性太强，作者像临时救场。
- 特殊条件很花，但奖励没有本质变化。

## 训练任务

列三阶段抽奖池：初期 5 个保命/解围奖励，中期 5 个资源/关系奖励，后期 5 个改变主线解法的奖励。每阶段至少一种奖励形态不同。
