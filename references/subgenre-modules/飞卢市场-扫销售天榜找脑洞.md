# 飞卢市场-扫销售天榜找脑洞

依据案例：`batch-333` 案例 1663

## 读者承诺

判断飞卢市场先扫销售天榜，再拆脑洞、书名、简介、金手指和开篇钩子。

## 核心循环

```text
销售天榜 -> 高频题材 -> 脑洞包装 -> 开篇兑现 -> 选题库
```

## 开篇必备

- 看当前销售榜。
- 记录书名关键词。
- 拆简介承诺。
- 拆金手指和开篇事件。
- 提炼可迁移脑洞。

## 期待来源

作者从正在卖的书里提取市场证据，而不是靠抽象印象选题。

## 断档风险

- 不扫榜就问市场。
- 只记题材名，不拆爽点。
- 把脑洞理解成乱设定。
- 不看付费节点。
- 选题和平台读者错配。

## 训练任务

扫飞卢销售天榜前 20，记录书名、简介承诺、金手指、开篇事件和付费节点。
