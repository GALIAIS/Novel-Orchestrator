# 反派压迫-伤害具体化与民怨激发

依据案例：`batch-203` 案例 1011。

## 读者承诺

想让读者愤怒，不能只写抽象压迫，要把伤害落到具体受害者、具体损失和不可逆后果上。

## 核心循环

```text
抽象恶行 -> 具体受害者 -> 家破人亡 -> 反派冷漠话术 -> 读者期待惩戒
```

## 开篇必备

- 给出一个具体受害者。
- 写清他失去了什么。
- 让反派漂亮话与惨状形成反差。
- 给主角一个必须出手的触发点。

## 期待来源

读者看到受害者真实代价后，会自然期待主角惩戒压迫者。

## 断档风险

- 只写概念，不写人。
- 反派坏得抽象，没有情绪落点。
- 围观者反应替代了受害者损失。
- 主角出手理由不够具体。

## 训练任务

把“横征暴敛”改成 500 字场景：一个家庭、一个被夺物、一个冷漠回应、一个主角出手理由。
