# 外部教程-爆款特征资料吸收

依据案例：`batch-090` 案例 448。

## 读者承诺

外部爆款教程只能算资料入口。看完后必须提炼观点、绑定样本、转成作品检查表，否则只是收藏链接。

## 核心循环

```text
发现教程 -> 确认主题 -> 提炼观点 -> 匹配样本 -> 转检查项 -> 回查作品
```

## 开篇必备

- 记录教程来源和可访问状态。
- 不在没看正文时编造结论。
- 每条观点配一个样本。
- 每条观点转一个检查问题。

## 期待来源

- 爆款特征变成可执行标准。
- 教程观点能被样本验证。
- 自己作品能据此修改。

## 断档风险

- 只说“值得一看”。
- 短链失效仍假装吸收。
- 记口号不做检查表。
- 不回到作品实践。

## 训练任务

找一个爆款教程，摘 5 条观点。每条观点配 1 本样本和 1 个作品检查问题，写成自己的检查表。
