# 一级新人-先扫书学理论

依据案例：`batch-096` 案例 476。

## 读者承诺

一级新人最常见的问题是凭兴趣开书，但不熟平台、不懂套路、没有后续方向。此时最有效的不是找人夸优点，而是先暂停开书，补足扫书、理论、拆书和大纲基础。

## 核心循环

```text
凭兴趣写几章 -> 后续编不出来 -> 暂停开书 -> 扫大火作品 -> 学理论 -> 拆开篇套路 -> 建立大纲方向 -> 再开书
```

## 开篇必备

- 开书前至少看过同平台大火样本。
- 知道目标平台读者想看什么。
- 第一章有明确卖点、主角困境和后续期待。
- 有大方向，不靠当天灵感硬编。

## 期待来源

- 热门作品提供类型承诺。
- 理论提供判断框架。
- 拆书提供可复用样本。
- 大纲提供持续写下去的方向感。

## 断档风险

- 把小额打赏当成路线成立。
- 只求鼓励，不补基本功。
- 不熟平台风格就开书。
- 没有大纲，几章后原地打转。

## 训练任务

连续拆 5 本同平台热门书的第一章。每本只记录 5 项：主角、困境、卖点、第一爽点、章末期待。拆完再写自己的第一章。
