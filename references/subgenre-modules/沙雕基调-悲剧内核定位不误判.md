# 沙雕基调-悲剧内核定位不误判

依据案例：`batch-097` 案例 483。

## 读者承诺

沙雕文可以有悲剧内核，但第一屏必须让读者判断对作品定位。悲剧背景提供人物厚度，轻松句子、台词、选择和严肃环境里的不正经，负责兑现沙雕承诺。

## 核心循环

```text
轻松承诺 -> 悲剧背景暗线 -> 严肃环境 -> 主角不正经选择 -> 反差笑点 -> 逐步揭露内核
```

## 开篇必备

- 第一章先确认轻松/沙雕基调。
- 悲剧背景只露一角，不抢主承诺。
- 周围环境可以严肃，主角反应制造喜感。
- 台词和句子要参与定调。

## 期待来源

- 严肃环境中的不正经主角。
- 悲剧内核带来的故事感。
- 轻松表层与沉重过去的反差。
- 后续揭露时的角色深度。

## 断档风险

- 开头像复仇苦情文，读者误判定位。
- 所有人都搞笑，反差消失。
- 悲剧背景过重，压垮沙雕感。
- 只靠设定反差，没有句子和台词配合。

## 训练任务

写一个沙雕主角在严肃会议中出场。500 字内只能暗示一次悲剧过去，必须用一句台词和一个选择确认轻松基调。
