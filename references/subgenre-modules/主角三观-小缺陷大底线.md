# 主角三观-小缺陷大底线

依据案例：`batch-123` 案例 612。

## 读者承诺

主角可以有小缺陷，但大是大非必须站稳。缺陷负责真实和趣味，底线负责代入和安全感。

## 核心循环

```text
小缺陷出场 -> 日常制造反差 -> 遇到原则事件 -> 主角守住底线 -> 对反派释放狠劲
```

## 开篇必备

- 缺陷不能伤害无辜者。
- 大是大非要明确。
- 对反派可以狠，对普通人要有分寸。
- 关键场景要证明三观。
- 屑感和恶趣味要控制尺度。

## 期待来源

- 小缺陷带来角色辨识度。
- 底线让读者放心代入。
- 关键选择证明主角值得追随。
- 对恶人清算提供爽点。

## 断档风险

- 主角单纯恶，读者无法共鸣。
- 为达目的不择手段伤及无辜。
- 缺陷没有边界，变成毒点。
- 完美无缺又缺少真实感。

## 训练任务

写主角 3 个小缺陷和 3 条底线。再写一个场景，让主角带着缺陷做出正确选择。
