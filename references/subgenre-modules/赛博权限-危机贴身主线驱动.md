# 赛博权限-危机贴身主线驱动

依据案例：`batch-118` 案例 589。

## 读者承诺

赛博权限类开篇的卖点是用信息优势解决贴身危机。城市危险不能只是背景板，必须直接压到主角身上，并把金手指首次使用接成主线目标。

## 核心循环

```text
身份漏洞 -> 贴身危险 -> 权限解围 -> 得到新信息 -> 主线敌人追压 -> 权限继续升级爽点
```

## 开篇必备

- 无身份危机具体可见。
- 第一个敌人直接威胁主角。
- 金手指首次使用解决眼前问题。
- 解围后接身份/金钱/帮派目标。
- 摸尸和背景信息压缩。

## 期待来源

- 最高权限立刻兑现。
- 读者看到主角不是旁观者。
- 危机推动主角行动。
- 主线敌对势力自然出现。

## 断档风险

- 危险不针对主角。
- 摸尸和解释太长。
- 金手指限制缺乏说服力。
- 主角像游戏人间，没有迫切目标。

## 训练任务

写一场赛博开篇：通缉或身份危机直接逼近主角，主角用权限解决一个具体敌人，并得到下一个必须处理的主线目标。
