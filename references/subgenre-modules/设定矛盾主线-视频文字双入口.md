# 设定矛盾主线-视频文字双入口

依据案例：`batch-165` 案例 821。

## 读者承诺

把名作教程入口整理成可复查资料卡，明确它讨论的是设定、矛盾和主线剧情的联动，而不是把未读附件强行当成已掌握方法。

## 核心循环

```text
教程入口 -> 样本标题 -> 可见议题 -> 访问边界 -> 后续复查 -> 系统索引
```

## 开篇必备

- 写清教程样本名。
- 写清可见证据来自正文、图片、评论还是附件。
- 未读视频和未下载附件必须单独标注。
- 只抽取已确认的议题方向。

## 期待来源

读者期待从名作中学到整体结构方法，尤其是设定如何导出矛盾，矛盾如何牵引主线。

## 断档风险

- 把教程标题误当教程正文。
- 未读附件却复述成确定结论。
- 只收藏链接，不建立后续复查入口。
- 把设定、矛盾、主线拆成三个孤立概念。

## 训练任务

整理 3 个教程入口，每个只写“可确认内容”和“待复查内容”。待复查内容不能进入正式方法论，只能进入复查清单。
