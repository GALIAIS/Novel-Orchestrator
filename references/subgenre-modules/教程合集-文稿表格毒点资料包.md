# 教程合集-文稿表格毒点资料包

依据案例：`batch-114` 案例 570。

## 读者承诺

教程合集不是收藏夹，而是训练路由入口。面对开篇、毒点、拆书、爽感、人物设定等问题时，从资料包中取一个当前最相关文件，转成判断标准和训练动作。

## 核心循环

```text
当前问题 -> 定位资料文件 -> 提取判断标准 -> 转成训练动作 -> 产出复盘文件 -> 补案例样本
```

## 开篇必备

- 先写当前问题。
- 只选一个主题资料。
- 文稿负责解释，表格负责自检。
- 毒点资料用于排雷。
- 拆书资料用于找样本证据。

## 期待来源

- 零散教程变成可查入口。
- 新人知识焦虑变成任务清单。
- 表格让判断可复核。
- 毒点解析能直接指导删改。

## 断档风险

- 一次性全读，读完不会写。
- 只收藏资料，不产出训练。
- 教程概念压过原文章节事实。
- 缺少同题材案例补证。

## 训练任务

选一个当前写作问题，从资料包或系统目录里找 3 个相关文件名。每个文件只输出一条判断标准和一个动作，最后合成一张本周训练表。
