# 刺猬猫签约-等级分层认知

依据案例：`batch-137` 案例 685。

## 读者承诺

帮助新人理解刺猬猫 A/B/C/D 签的大致经验认知：等级对应成绩、潜力、编辑判断和商业价值，但不是官方规则。

## 核心循环

```text
签约等级 -> 成绩/潜力 -> 编辑判断 -> 商业价值 -> 作者预期
```

## 开篇必备

- A/B/C/D 逐级解释。
- 标注经验来源。
- 区分成绩和潜力。
- 说明 D 签也有基础价值。
- 明确非官方规则。

## 期待来源

- 新人知道自己大概位置。
- 等级焦虑被拆开。
- 平台判断逻辑更清楚。
- 避免误传官方政策。

## 断档风险

- 把经验当官方标准。
- 只看均订，不看编辑判断。
- 低签约等级被误解成无价值。
- 用过时信息做硬判断。

## 训练任务

写 800 字以内新人 FAQ，解释刺猬猫 A/B/C/D 签含义，并注明只是评论经验总结，不等同官方规则。
