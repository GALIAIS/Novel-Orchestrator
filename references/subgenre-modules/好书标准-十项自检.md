# 好书标准-十项自检

依据案例：`batch-157` 案例 783。

## 读者承诺

一本书是否写得不错，可以用人设、主线、情绪、金手指、任务收益和动态世界做系统自检。

## 核心循环

```text
人设动机 -> 主线任务 -> 场景情绪 -> 金手指成长 -> 任务收益 -> 世界反作用
```

## 开篇必备

- 主角是什么人。
- 主角为什么做主线任务。
- 主线是否贯穿。
- 场景情绪设计。
- 金手指成长性。
- 任务收益和责任。
- 世界动态反作用。

## 期待来源

读者期待主角每一步行动都有动机、情绪、收益和后续影响。

## 断档风险

- 主角只做事，没有人设强调。
- 任务原因薄弱。
- 场景没有情绪。
- 金手指不成长。
- 完成任务没有收益。
- 世界静止不反作用于主角。

## 训练任务

用“有/弱/无”检查自己的第一卷十项标准，凡是弱或无的项，补一个具体桥段。
