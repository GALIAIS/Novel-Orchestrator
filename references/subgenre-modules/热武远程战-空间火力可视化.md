# 热武远程战-空间火力可视化

依据案例：`batch-334` 案例 1670

## 读者承诺

热武远程战要写清距离、掩体、视野、火力和目标，让射击变成可视化战术变化。

## 核心循环

```text
战术目标 -> 火力压制 -> 位置/掩体变化 -> 装备技能破局 -> 命中关键目标
```

## 开篇必备

- 双方距离清楚。
- 掩体和视野清楚。
- 武器差异影响选择。
- 每轮交火改变局势。
- 主角有明确战术目标。

## 期待来源

读者期待主角在火力压制下通过位置、装备和技能反杀，而不是互相开枪流水账。

## 断档风险

- 只堆武器名。
- 双方站桩互射。
- 不写掩体和移动。
- 弹药、冷却、换弹没有节奏。
- 反派没有战术。

## 训练任务

写 800 字远程战，三轮交火分别改变位置、掩体、弹药、视野或战术目标。
