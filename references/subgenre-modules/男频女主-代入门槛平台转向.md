# 男频女主-代入门槛平台转向

依据案例：`batch-117` 案例 585。

## 读者承诺

男频女主文的关键不是能不能写，而是目标读者是否吃这一口。主站男频代入门槛高，轻小说站、变身文、橘子文等方向更可能有对应受众。

## 核心循环

```text
想写女主 -> 查同类样本 -> 评估代入门槛 -> 选择平台/方向 -> 设定关系边界
```

## 开篇必备

- 先找同类样本。
- 看成绩和读者评论。
- 明确目标平台。
- 给代入感补偿。
- 男性角色关系边界清晰。

## 期待来源

- 样本证明方向可行。
- 轻小说/变身/橘子受众更明确。
- 女主卖点与平台口味一致。
- 关系边界减少读者雷点。

## 断档风险

- 起点飞卢主站硬写原装女主。
- 没样本就开长篇。
- 把电影女主成功直接类比网文。
- 男性角色和女主关系过密踩雷。

## 训练任务

找 5 本同类样本，记录平台、成绩、标签、读者评论、男女关系边界。样本不足时，先写短篇试水。
