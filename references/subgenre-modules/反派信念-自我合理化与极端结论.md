# 反派信念-自我合理化与极端结论

依据案例：batch-384 案例 1918

## 读者承诺

强反派会否认自身恶意，把错误归因和极端方案包装成公共利益。作品必须同时揭示具体受害者、隐去代价和更困难的替代选择。

## 核心循环

```text
真实不满 -> 错误归因 -> 否认恶意 -> 公益包装极端方案 -> 代价揭露
```

## 开篇必备

- 反派真实不满或创伤。
- 错误但自洽的归因。
- 他自称保护的公共利益。
- 一名具体受害者。

## 期待来源

- 反派不把自己当坏人。
- 话术初听似乎务实。
- 剧情逐步揭开被删除的代价。
- 主角提出有成本的替代方案。

## 断档风险

- 直接拿现实受保护群体作消灭对象。
- 只让主角口头骂反派。
- 反派逻辑比作品价值判断更完整。
- 不呈现方案伤害的具体人物。

## 训练任务

用完全虚构派系写一套三步极端论证，同时呈现具体受害者和隐去代价，让主角提出更困难的替代方案。
