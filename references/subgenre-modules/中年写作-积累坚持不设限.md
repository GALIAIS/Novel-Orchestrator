# 中年写作-积累坚持不设限

依据案例：`batch-326` 案例 1630

## 读者承诺

写作入行不由年龄单点决定；中年作者更该看已有积累、持续投入、身体约束和能否坚持长期训练。

## 核心循环

```text
年龄焦虑 -> 核查积累和机会 -> 设定长期训练 -> 保护健康 -> 稳定输出
```

## 开篇必备

- 不把年龄当唯一门槛。
- 评估已有阅历、阅读量和笔力。
- 明确每日稳定投入时间。
- 不幻想一夜暴富。
- 把健康和家庭约束写进计划。

## 期待来源

作者期待确认自己是否值得开始，并把中年阅历转成写作资产。

## 断档风险

- 把知乎成功故事当轻松捷径。
- 忽略时代机会和十年坚持。
- 熬夜透支健康。
- 只盯赚钱，失去生活规划。
- 没有第一年训练目标。

## 训练任务

写一张晚入行评估表：已有积累、每日投入、健康约束、目标收入、第一年只练的三个能力。
