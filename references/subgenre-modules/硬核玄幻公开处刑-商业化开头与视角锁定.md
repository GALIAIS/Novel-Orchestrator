# 硬核玄幻公开处刑-商业化开头与视角锁定

依据案例：batch-358 案例 1790

## 读者承诺

硬核玄幻也必须先给商业化开头：主角、视角、危机、目标、行动、亮点。设定、背景、氛围和伏笔都不能替代故事推进。

## 核心循环

```text
主角入场 -> 视角锁定 -> 危机/目标出现 -> 金手指或亮点验证 -> 背景设定后置
```

## 开篇必备

- 前 500 字出现主角行动或对话。
- 主视角锁主角。
- 第一危机和目标明确。
- 设定只保留必要信息。

## 期待来源

- 读者立刻知道看谁。
- 危机让故事动起来。
- 金手指/同人看点快速兑现。
- 粉丝期待被简介和正文回应。

## 断档风险

- 用老白/硬核当慢热借口。
- 前几章全是背景和设定。
- 英雄联盟世界观却不兑现粉丝想看点。
- 无用环境描写和早期伏笔劝退读者。

## 训练任务

删改一个硬核开头：保留主角、危机、目标、亮点，删除不影响剧情的环境描写，重写前 500 字。
