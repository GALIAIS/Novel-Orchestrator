# 结构图拆书-情绪折线大局观

依据案例：`batch-039` 案例 193。

## 读者承诺

结构图把文本拆成节点、分支和情绪折线，让原本模糊的跌宕起伏变得可查、可复盘、可反向写大纲。

## 核心循环

```text
文本阅读 -> 节点拆分 -> 情绪标注 -> 支线回流 -> 大纲反推
```

## 开篇必备

- 事件节点清楚。
- 标注上行下行。
- 支线和主线分层。
- 记录回收点。

## 期待来源

- 新人看懂节奏变化。
- 大局观提升。
- 模糊结构变成可视化。
- 拆书结论能指导大纲。

## 断档风险

- 只画剧情流水账。
- 不标情绪变化。
- 图太大但没有高清版本。
- 画图替代文本阅读。

## 训练任务

拆前三章，画 10 个节点，每个节点标明事件功能、情绪方向、服务的读者期待。
