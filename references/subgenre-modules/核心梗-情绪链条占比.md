# 核心梗-情绪链条占比

依据案例：`batch-010` 案例 049。

## 读者承诺

一本书长期留人的不是单个桥段，而是反复变形的核心情绪链条。核心梗必须占据大多数关键情节。

## 核心循环

```text
核心梗 -> 情绪链条 -> 场景替换 -> 爽点复现 -> 长线承诺
```

## 开篇必备

- 题材标签要和核心梗一致。
- 前期尽快展示核心情绪。
- 金手指、人物和事件都服务核心梗。
- 偏离内容要能回流主线。

## 期待来源

- 读者知道下一次还会爽在同一个主题上。
- 场景和人物变化避免重复腻味。
- 情绪链条稳定提供熟悉满足。
- 核心梗占比越高，书越不散。

## 断档风险

- 书名简介承诺一种核心梗，正文写另一种。
- 核心梗占比低于一半。
- 只抄桥段，没有提取情绪链条。
- 为创新改掉题材标签的核心乐趣。

## 训练任务

列前 20 个情节，逐个标注是否命中核心梗。命中不足 10 个，重做题材承诺。
