# 长篇规划-主线字数弹性控制

依据案例：`batch-164` 案例 820。

## 读者承诺

在主线大纲阶段预估体量，同时保留按成绩增减、支线扩展和番外补完的空间。

## 核心循环

```text
估主线容量 -> 定卷级矛盾 -> 预留伏笔成长 -> 标出可删可扩 -> 按成绩调整
```

## 开篇必备

- 先规划主体内容，不先锁死字数。
- 每卷有明确主矛盾。
- 伏笔和人物成长有回收位置。
- 支线分成可删和可扩两类。

## 期待来源

作者期待大纲既能控制主线承诺，又能根据成绩和连载状态调整体量。

## 断档风险

- 新人硬估精确字数。
- 为凑字数拖主线。
- 支线无法删减。
- 成绩变化时大纲没有弹性。

## 训练任务

写一个 150 万字弹性大纲：5 卷主线、3 条可删支线、3 条可扩支线、番外补完池。
