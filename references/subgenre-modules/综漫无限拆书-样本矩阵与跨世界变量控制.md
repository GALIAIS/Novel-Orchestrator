# 综漫无限拆书-样本矩阵与跨世界变量控制

依据案例：`batch-407` 案例 2033

## 读者承诺

不按单本好看程度选拆书样本，而比较节奏、IP依赖和主角知识差不同的作品；用知识、力量、角色、长线四类变量保证跨世界公平且有累积。

## 核心循环

```text
明确学习问题 -> 选择差异样本 -> 统一矩阵拆入口规则退出 -> 提炼四类变量 -> 原创副本迁移一项机制 -> 本地角色反制预知 -> 退出余波进入下一世界
```

## 开篇必备

- 写清本轮具体拆书问题。
- 样本覆盖不同节奏和前置知识要求。
- 主角原作知识有错误或过期风险。
- 不同能力体系有共同尺度和失效条件。
- 本地角色拥有独立目标与选择。
- 每个世界改变长线资产、关系或信念。

## 期待来源

熟悉世界快速建立期待，剧情偏移和本地人物让预知失效；副本结果带着资产与债务进入下一世界，形成真正长线。

## 断档风险

- 2022 年书单被当作当前标准样本。
- 只拆一部便宣布类型规律。
- 主角全知且永不出错。
- 本地角色只等主角收编。
- 力量体系跨世界随意换算。
- 通关后关系和后果全部清零。
- 使用受保护角色却未核验授权。

## 训练任务

选三份差异样本用同一矩阵拆解；不用任何现成角色名，写一个原创副本入口，只迁移一个机制，并让主角的一条预知因本地角色选择而失效。
