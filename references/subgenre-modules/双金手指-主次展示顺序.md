# 双金手指-主次展示顺序

依据案例：batch-290 案例 1447

## 读者承诺

双金手指要先确定主次，第一章只让一个能力解决当前冲突，另一个能力后置或只留症状。

## 核心循环

```text
怪异钩子 -> 主能力承诺 -> 高筹码展示 -> 解决冲突 -> 副能力后置
```

## 开篇必备

- 第一眼怪异钩子。
- 主金手指。
- 当前冲突。
- 能力展示筹码。
- 使用过程。
- 副金手指的后置伏笔。

## 期待来源

读者先对一个能力形成稳定期待，并在当场冲突里看到它确实有用。

## 断档风险

- 两个能力同时解释，期待分散。
- 先展示的能力后面不用。
- 能力展示像突然出题突然答题。
- 简介把悬念解释光。
- 视角太高，主角代入弱。

## 训练任务

把一个双外挂开篇改成单外挂主线：第一章只展示主能力，副能力只出现代价或异常。
