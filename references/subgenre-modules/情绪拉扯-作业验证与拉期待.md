# 情绪拉扯-作业验证与拉期待

依据案例：`batch-227` 案例 1132。

## 读者承诺

情绪拉扯要通过作业验证：能不能让读者恨、期待、等待反击。

## 核心循环

```text
案例学习 -> 拉期待复习 -> 写作业 -> 检验情绪 -> 修正场景
```

## 开篇必备

- 明确读者该恨谁。
- 写清恨的理由。
- 延迟释放，保留期待。
- 章尾给反击信号。

## 期待来源

读者先被压住情绪，再等主角释放；作者用作业确认自己是否真正会写。

## 断档风险

- 只听课不练。
- 只写事件，没有情绪。
- 不会拉期待，直接给结果。
- 公开处刑不按规则提交。

## 训练任务

写 800 字情绪拉扯小场景：让读者讨厌一个人，同时期待主角反击。包含压迫理由、旁人反应、主角暂忍、章尾反击信号。
