# 小说真实感-现实锚点与虚幻风筝

依据案例：`batch-208` 案例 1037。

## 读者承诺

好故事要让幻想飞起来，同时用现实经验、人物动机和因果逻辑把它拴住。

## 核心循环

```text
现实锚点 -> 虚构夸张 -> 可信动机 -> 情节推进 -> 现实线回拉
```

## 开篇必备

- 给出足够吸引人的虚幻卖点。
- 安排一个读者熟悉的现实锚点。
- 让人物反应符合真实需求。
- 标出不能崩坏的因果线。

## 期待来源

读者想看超出现实的白日梦，但需要相信人物、世界和情节仍然能被理解。

## 断档风险

- 过于现实，没有白日梦。
- 过于虚幻，逻辑和人物前后矛盾。
- 只发泄情绪，不提供理解。
- 高概念开篇后圆不回来。

## 训练任务

为一个夸张设定列出 5 个幻想爽点和 5 个现实锚点，再写 500 字开篇，要求每 150 字至少出现一个锚点。
