# 榜单选题-热度关键词扫描

依据案例：batch-296 案例 1476

## 读者承诺

用榜单、书名、关键词和频道变化辅助新人选题，识别当前读者愿意点击的题材承诺。

## 核心循环

```text
扫榜单 -> 提取关键词 -> 判断频道/活动影响 -> 过滤伪分类 -> 落成自己的选题
```

## 开篇必备

- 明确目标平台和频道。
- 记录高频书名词、金手指词、开局词。
- 区分真实题材和披皮分类。
- 把热词转成自己的主角处境和开篇承诺。

## 期待来源

- 读者看到熟悉的高需求承诺。
- 热词降低点击理解成本。
- 题材趋势帮助作者减少盲选。

## 断档风险

- 只抄热词，不做自己的冲突设计。
- 把征文、活动导致的短期热度当长期趋势。
- 只看分类名，不看书名、简介、开篇。

## 训练任务

扫一个榜单，提取 20 个关键词，组合成 3 个新选题；每个选题写一句开篇读者承诺。
