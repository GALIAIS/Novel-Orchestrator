# 期待杠杆-物品装逼链

依据案例：`batch-017` 案例 085。

## 读者承诺

主角获得物品后，要通过功能、误判、克制、失败参照和围观反应多次加杠杆，才能让最终装逼有层次。

## 核心循环

```text
物品功能 -> 配角误判 -> 反派适配 -> 他人失败 -> 主角方案 -> 围观不看好 -> 秒杀兑现
```

## 开篇必备

- 物品功能要具体。
- 配角要有误判。
- 反派特性要能被物品克制。
- 先安排他人失败作参照。
- 围观者反应要分层。

## 期待来源

- 读者期待物品使用。
- 读者期待误判被打脸。
- 读者期待克制反派。
- 读者期待他人失败后主角成功。
- 读者期待马后炮和收获。

## 断档风险

- 只说物品强，没有展示适用场景。
- 没有配角误判，缺少信息差。
- 反派和物品没有针对性。
- 最终只赢一层，缺少连锁满足。

## 训练任务

设计一个物品装逼单元，至少写出 5 次期待加杠杆：功能、误判、克制、他人失败、围观不看好。
