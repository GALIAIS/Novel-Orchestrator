# 游戏王同人-第一章先打牌

依据案例：`batch-334` 案例 1667

## 读者承诺

游戏王类同人第一章要先进入决斗/考核，直接兑现新玩法和“玩坏”亮点。

## 核心循环

```text
熟悉原作规则 -> 直接进入对局 -> 主角拿出新玩法 -> 对手/读者震惊 -> 规则边打边解释
```

## 开篇必备

- 500 字内进入对局或考核。
- 第一章出现核心玩坏点。
- 设定随行动露出。
- 少吐槽少说明。
- 主角第一场赢法有新意。

## 期待来源

读者期待熟悉规则被新组合玩出花，而不是先读世界说明。

## 断档风险

- 第一章海量设定。
- 剧情靠嘴推动。
- 同人卖点迟迟不出现。
- 吐槽注水。
- 牌佬还没看到打牌就退出。

## 训练任务

把第一章改成先决斗后解释，第一章内必须出现一次“被玩坏”的召唤或组合。
