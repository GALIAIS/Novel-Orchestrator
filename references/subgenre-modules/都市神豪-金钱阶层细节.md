# 都市神豪-金钱阶层细节

依据案例：`batch-013` 案例 063。

## 读者承诺

都市神豪的爽点不是数字变大，而是让读者看到有钱以后能做什么，以及不同阶层如何被金钱、资源和信息差改写反应。

## 核心循环

```text
金钱目标 -> 专业赚钱 -> 真实消费 -> 阶层反应 -> 超额收获 -> 新资源/新关系
```

## 开篇必备

- 明确金钱欲望承诺。
- 赚钱环节要有专业细节。
- 花钱环节要有真实流程。
- 人物反应要体现阶层差异。

## 期待来源

- 读者想看普通人无法想象的消费流程。
- 读者想看钱如何转成身份、人脉、资源和关系优势。
- 读者想看专业信息带来的长见识爽点。
- 读者想看不同阶层人物被主角改写判断。

## 断档风险

- 只写余额和价格，没有真实细节。
- 不懂行业却硬装专家。
- 花钱没有超额收获，只是流水账消费。
- 照抄万订书毒点，尤其前期挨打、尴尬互动和延迟兑现。

## 训练任务

写一个 1500 字消费场景大纲，必须包含消费门槛、行内流程、超额收获和三类阶层反应。
