# 强主线-长短钩交替

依据案例：`batch-066` 案例 327。

## 读者承诺

这个模块用于强主线、升级、冒险、悬疑推进文。它承诺的是让长线目标和短线危机交替工作：短钩负责拉紧和爆发，长钩负责缓冲、成长和持续行动。

## 核心循环

```text
长钩牵引 -> 短钩危机 -> 爆发兑现 -> 收获补偿 -> 长钩成长 -> 下一短钩
```

## 开篇必备

- 卷主线目标要清楚。
- 至少有一个长期目标推动主角。
- 短线危机出现后要主导行动。
- 爆发后要给收获或缓冲。
- 不连续三次使用同一种节奏。

## 期待来源

- 明确危机带来紧张。
- 长线目标让无危机段也能动。
- 收获补偿读者。
- 小伏笔短周期兑现。
- 每卷目标拆成小目标。

## 断档风险

- 连续爆发导致疲惫。
- 连续过渡像鸡肋。
- 长线目标太虚，主角不动。
- 大伏笔埋太久没有小爽点。
- 短钩和长钩命名混乱，作者自己也误判。

## 训练任务

设计 6 个事件，按“长钩/短钩/收获”标注。检查是否有三次同类节奏连用，有就插入缓冲或危机。
