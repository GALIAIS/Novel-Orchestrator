# 教学文档合集-可下载知识库

依据案例：`batch-148` 案例 739。

## 读者承诺

把分散教程、读书笔记、拆书案例、表格和视频文字版整理成可下载合集，让新人能反复查找、对照拆书、稳扎稳打学习。

## 核心循环

```text
新教程产出 -> 合集更新 -> 目录截图 -> 整包下载 -> 单篇下载 -> 拆书结合使用
```

## 开篇必备

- 更新时间。
- 新增文件列表。
- 整体下载地址。
- 单篇下载方式。
- 目录截图。
- 新人优先阅读说明。

## 期待来源

读者期待资料不散、不丢、可复习。目录和网盘降低检索成本，示例图和拆书结合降低理解成本。

## 断档风险

- 文件分散，找不到。
- 只有网盘包，没有新增清单。
- 理论不配案例。
- 缺失资料不补档。

## 训练任务

整理自己的写作资料库，分成理论文档、拆书案例、表格模板、视频笔记四类。写一张本周新增目录，标注每篇解决的写作问题。
