# 终局兑现-智取后蛮力碾压

依据案例：`batch-149` 案例 744。

## 读者承诺

前面靠计谋分割、埋伏、逐个击破铺垫成长后，最终面对原本不可战胜的强敌，要换成蛮力和实力兑现，让读者看到等待已久的碾压。

## 核心循环

```text
强敌压迫 -> 智取小敌 -> 成长奖励 -> 终局正面遭遇 -> 假下行虚晃 -> 实力全开碾压
```

## 开篇必备

- 强敌不可战胜的早期压迫。
- 至少一段智取铺垫。
- 成长奖励或关键武器。
- 终局解决方式和前段不同。
- 敌人误判主角的瞬间。
- 干脆的实力兑现。

## 期待来源

读者期待前面所有铺垫最终变成“现在能打穿他”。假下行可以制造转折，但不能真的破坏兑现感。

## 断档风险

- 前段靠计谋，终局仍然靠计谋，口味重复。
- 成长奖励没有参与最终战。
- 真下行压太久，读者觉得前面铺垫白等。
- 碾压不够干脆，爽点被拖散。
- 敌人败因不是主角成长，而是外力救场。

## 训练任务

写一场 1000 字最终战：敌人先误以为主角底牌耗尽，随后主角展示前文获得的武器或境界，用蛮力完成斩杀。禁止用第三方救场。
