# 科幻参考-设定亮点转写作

依据案例：batch-264 案例 1319

## 读者承诺

影视、游戏、设定集可以进资料库，但必须提炼成写作亮点：阵营、道具规则、身份反差、技术设定，而不是只搬资源链接。

## 核心循环

```text
看参考作品 -> 提炼设定亮点 -> 写推荐理由 -> 对接自己的世界观/角色/道具
```

## 开篇必备

- 参考作品名。
- 可借鉴设定。
- 阵营设计。
- 关键道具规则。
- 身份反差。
- 可用于自己的位置。

## 期待来源

作者期待从成熟视听作品里获得可迁移创意，而不是只收藏素材。

## 断档风险

资源链接没有方法价值；未完结作品可以先写阶段性感想，不要等到全看完才入库。

## 训练任务

看一集科幻剧，只写 5 条可迁移设定，每条说明能放进自己的哪类剧情。
