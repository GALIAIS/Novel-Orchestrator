# 番茄望远镜-十万字爆点

依据案例：batch-286 案例 1430

## 读者承诺

先设计十万字巨大悬念或爽点，再用简介、开头和中段持续递望远镜，最终集中兑现。

## 核心循环

```text
十万字爆点 -> 简介承诺 -> 开头望远镜 -> 中段加压 -> 集中爆发
```

## 开篇必备

- 十万字爆点。
- 简介里的明确承诺。
- 第一章望远镜。
- 中段提醒方式。
- 爆点兑现后的新问题。

## 期待来源

读者从一开始就知道未来有大场面，于是持续追到承诺兑现。

## 断档风险

- 爆点只藏不递。
- 中段不提醒，期待断掉。
- 老套路没有集中兑现。
- 爆点兑现后无新钩子。

## 训练任务

设计一个十万字修罗场或大反转，倒推前三章望远镜和中段三次提醒。
