# 日常文-样本检索拆解

依据案例：`batch-106` 案例 528。

## 读者承诺

学日常文先找样本，再按关系、平台、主角设定和轻冲突拆。日常不是没事写水，而是用小事件推动关系和人物状态。

## 核心循环

```text
样本收集 -> 平台分类 -> 关系核心 -> 小事件 -> 关系变化 -> 下一章期待
```

## 开篇必备

- 明确日常文子方向。
- 至少 3 本同类样本。
- 主角设定和关系核心。
- 每章小冲突来源。

## 期待来源

- 关系变化。
- 轻松互动。
- 生活细节。
- 时间差、长生、恋爱游戏等设定反差。

## 断档风险

- 把日常理解为没剧情。
- 不分平台口味。
- 只看推荐书名，不拆事件。
- 小事不带来关系变化。

## 训练任务

拆 3 本日常文前 3 章，每章只记：小事、关系变化、下一章期待。
