# 穿古日常-现代融入生活流

依据案例：`batch-145` 案例 725。

## 读者承诺

古代角色进入现代，现代角色帮助其融入社会。主线不是打怪升级，而是通过连续生活事件完成社会适应、关系推进和感情升温。

## 核心循环

```text
时代错位 -> 现代生活困难 -> 主角帮助 -> 琐事互动 -> 情感反馈 -> 融入进度推进
```

## 开篇必备

- 开头直接进入关系和处境。
- 一个古代角色不懂的现代事项。
- 现代角色提供帮助但不全包。
- 琐事必须推动融入或感情变化。

## 期待来源

读者期待细腻陪伴感：看古代角色一点点理解现代生活，也看两个人在琐事中慢慢靠近。

## 断档风险

- 日常变流水账。
- 琐事不推动关系和主线。
- 开头太啰嗦，不进剧情。
- 新人只学慢节奏，没学生活事件的目标感。

## 训练任务

写一个 1000 字穿古日常开头：古代角色在现代遇到一个具体生活困难，现代角色帮忙但保留角色主动学习空间；结尾必须出现一次关系变化。
