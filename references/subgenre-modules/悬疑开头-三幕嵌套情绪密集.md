# 悬疑开头-三幕嵌套情绪密集

依据案例：batch-262 案例 1307

## 读者承诺

悬疑开头可以用老套路，但要靠三幕式嵌套、密集转折和意外推进制造紧张感。读者看的是压力和问题升级。

## 核心循环

```text
危机入场 -> 短幕转折 -> 新问题出现 -> 情绪再上扬 -> 小阶段收束
```

## 开篇必备

- 明确悬疑危机。
- 每章至少一个情绪转折。
- 转折后产生新问题。
- 旧套路里的新操作。
- 节奏与题材匹配。

## 期待来源

读者期待每次以为看懂时，都出现新意外，推动主角进入更危险的局面。

## 断档风险

硬搬快节奏会疲劳；只学语言不学结构，会失去内在逻辑；附件缺失时不能假装已拆完整样本。

## 训练任务

拆一个悬疑开头前三章，把每次情绪转折标成“起、承、转、合”，再看是否形成三幕套三幕。
