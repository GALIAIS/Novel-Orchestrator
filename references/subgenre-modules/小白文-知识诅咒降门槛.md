# 小白文-知识诅咒降门槛

依据案例：`batch-155` 案例 771。

## 读者承诺

小白文用简单直接绕开“知识的诅咒”，让读者快速理解、快速共情，同时用更剧烈的冲突保留张力。

## 核心循环

```text
复杂信息 -> 降低门槛 -> 读者秒懂处境 -> 快速共情 -> 强冲突兑现
```

## 开篇必备

- 一眼能懂的主角处境。
- 一眼能判断的不公平或利益冲突。
- 少量必要设定。
- 明确的敌我关系。
- 更集中的冲突动作。

## 期待来源

读者不需要花精力理解复杂设定，能直接进入共情和期待反击。

## 断档风险

- 把简单写成空洞。
- 降低门槛后没有加强冲突。
- 写手只鄙视小白文，不拆其留存机制。
- 设定解释压过读者共情。

## 训练任务

把一个复杂设定改成 300 字小白版，要求读者立刻知道谁受害、谁占便宜、主角要怎么反击。
