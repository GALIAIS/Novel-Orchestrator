# 末世鬼屋-核心毒点隔离

依据案例：`batch-027` 案例 132。

## 读者承诺

末世鬼屋开篇要集中展示无法逃避空间、怪物威胁和原罪真相，隔离绿帽等无关强毒点。

## 核心循环

```text
封闭空间 -> 怪物威胁 -> 生存规则 -> 原罪线索 -> 主角破局
```

## 开篇必备

- 人类为什么逃不了。
- 巨人/丧尸的具体威胁。
- 导致灾难的原罪线索。
- 主角第一生存目标。

## 期待来源

- 鬼屋规则带来压迫。
- 怪物和空间形成稳定看点。
- 原罪拉长线悬念。
- 主角改变格局。

## 断档风险

- 用绿帽压主角。
- 就近时间线出现无法挽回损伤。
- 末世写着写着转种田，核心散掉。
- 只有紧迫感，没有看点和亮点。

## 训练任务

把一个末世开篇的无关人格羞辱删掉，改成墙内规则、感染风险或基地驱逐造成的生存压迫。
