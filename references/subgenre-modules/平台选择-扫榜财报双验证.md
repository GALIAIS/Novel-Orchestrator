# 平台选择-扫榜财报双验证

依据案例：`batch-124` 案例 616。

## 读者承诺

选平台先验证，不靠传言。榜单回答读者喜欢什么，财报和作者口碑回答平台值不值得投。

## 核心循环

```text
目标平台 -> 查收益风险 -> 扫榜前排 -> 归纳题材偏好 -> 对照自身题材 -> 决定试水
```

## 开篇必备

- 平台前排榜单必须看。
- 作者口碑和稿费风险要查。
- 题材匹配不能靠猜。
- 试水前确定投入成本。
- 陌生平台先小样本验证。

## 期待来源

- 榜单给出真实读者偏好。
- 收益风险影响长期投入。
- 平台匹配降低开书失败率。
- 扫榜结论能指导题材选择。

## 断档风险

- 只问别人，不扫榜。
- 忽略稿费和平台风险。
- 用其他平台经验套书旗。
- 题材不匹配还硬投。

## 训练任务

扫一个陌生平台榜单前 20，统计题材和更新状态；再查作者口碑。最后写是否试水和理由。
