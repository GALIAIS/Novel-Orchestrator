# 情绪作业-帮人正当性

依据案例：`batch-023` 案例 113。

## 读者承诺

让主角帮人出头之前，读者先认可被帮者值得帮，主角出手才顺民意。

## 核心循环

```text
被帮者受害 -> 普通办法带来希望 -> 新信息打落 -> 逃离再上行 -> 背刺下行 -> 主角爆发
```

## 开篇必备

- 被帮者不是纯自作自受。
- 不公平伤害清楚。
- 主角第一套普通解法。
- 普通解法失败后再用金手指。

## 期待来源

- 读者站在主角一边。
- 上行下行交替制造追读。
- 普通办法失败抬高金手指价值。
- 最终对手有明确强点。

## 断档风险

- 被帮者太损，读者不希望主角救。
- 金手指太早解决，折线没跑起来。
- 反派强度只靠作者说。
- 围观敬佩没有前置压迫。

## 训练任务

写一个五折线帮人桥段，第一行先写“被帮者为什么值得救”。写不出就先改受害原因。
