# 包袱结构-误判埋设与延迟兑现

依据案例：`batch-245` 案例 1225。

## 读者承诺

包袱可以是笑点也可以是爽点，关键是先制造合理误判，再延迟兑现反差。

## 核心循环

```text
铺危险 -> 配角误判 -> 信息差发酵 -> 主角回归 -> 一句话解包袱
```

## 开篇必备

- 前提清楚。
- 误判合理。
- 信息差能持续。
- 兑现时有一句利落台词。

## 期待来源

读者知道误会正在发酵，等待主角回归拆掉它，得到反差笑点或爽点。

## 断档风险

- 没铺前提就抖包袱。
- 误判不合理。
- 太快解释，没有延迟。
- 包袱只好笑，不服务剧情。

## 训练任务

写一个延迟包袱：配角误判主角失败，中间隔一场剧情，再让主角用一句话拆误会。
