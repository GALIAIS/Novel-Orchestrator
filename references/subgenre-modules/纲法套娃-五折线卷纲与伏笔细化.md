# 纲法套娃-五折线卷纲与伏笔细化

依据案例：`batch-252` 案例 1257。

## 读者承诺

纲法套娃模块把大高潮逆推、计划-意外、五折线放在不同层级上协同使用。

## 核心循环

```text
大高潮定终点 -> 计划意外推事件 -> 五折线控情绪 -> 细化伏笔回收
```

## 开篇必备

- 先写卷目标和大高潮。
- 每段有计划和意外。
- 每个关键事件有上行下行。
- 伏笔和回收成对出现。

## 期待来源

读者既能感到长线目标推进，也能在每个局部事件里获得情绪拉扯和伏笔期待。

## 断档风险

- 把方法当同义词混用。
- 只有框架，没有细化事件。
- 五折线不加伏笔，后续缺回收。
- 卷纲和章纲层级混乱。

## 训练任务

用一个卷目标写出大高潮、3 个计划-意外、1 个五折线事件和 2 组伏笔回收。
