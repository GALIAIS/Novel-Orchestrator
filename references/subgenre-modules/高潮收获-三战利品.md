# 高潮收获-三战利品

依据案例：`batch-142` 案例 706。

## 读者承诺

每次高潮后都给读者清晰收获：马上能用的奖励、后续回收的伏笔奖励、可做人情或交易的资源奖励，让爽点不落空并自然引出新剧情。

## 核心循环

```text
矛盾爆发 -> 高潮解决 -> 即时战利品兑现 -> 伏笔战利品挂钩后续 -> 资源战利品打开关系线
```

## 开篇必备

- 第一矛盾清楚。
- 三万字内有首个高潮。
- 高潮后有明确收获。
- 收获分即时、伏笔、资源三类。

## 期待来源

- 努力必须有回报。
- 即时奖励给爽感。
- 伏笔奖励给长线期待。
- 人情/交易奖励带出新人物和新事件。

## 断档风险

- 高潮后没有实质收获。
- 收获只有数值，不带剧情。
- 三件奖励功能重复。
- 类似桥段反复使用超过三次。

## 训练任务

给一个高潮设计三件战利品：下一章马上用、10 章后回收、用来送人情或换资源。写 300 字说明它们分别引出哪三条剧情。
