# 低武复古升级-主线刻进事件

依据案例：`batch-044` 案例 220。

## 读者承诺

复古低武文可以老套，但主线必须清楚刻进事件：试炼、地位、资源、认可和下一层任务要一环扣一环。

## 核心循环

```text
低位起点 -> 试炼压力 -> 经验破局 -> 地位资源 -> 更高任务
```

## 开篇必备

- 主角处于低位。
- 第一试炼明确。
- 失败代价具体。
- 破局后有地位和资源变化。

## 期待来源

- 读者看主角摆脱压迫。
- 前世经验带来优势。
- 师兄或组织认可主角。
- 新任务承接更大主线。

## 断档风险

- 只学仙味和氛围。
- 弱主线变成无主线。
- 打脸装逼没有地位/资源变化。
- 结构不清，写成流水账。

## 训练任务

设计一个低武第一单元：试炼、失败代价、前世经验、资源获取、认可者、下一任务必须齐全。
