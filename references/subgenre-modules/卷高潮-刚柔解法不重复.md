# 卷高潮-刚柔解法不重复

依据案例：`batch-125` 案例 623。

## 读者承诺

一卷里的爽点解法最好有变化。前面用巧劲，后面就可用刚劲；读者看到的是主角懂变通，而不是同一招反复用。

## 核心循环

```text
起终点拉开 -> 巧劲破局 -> 反派仇恨加码 -> 刚劲清算 -> 刚柔并济证明主角
```

## 开篇必备

- 本卷起点和终点差距要大。
- 每次意外的解法类型要标清。
- 反派恶行可用于拉高仇恨。
- 卷高潮不能重复前面解法。
- 最终要体现主角识时务、懂变通。

## 期待来源

- 差距制造阻碍和情绪。
- 巧劲带来智取爽。
- 仇恨加码带来清算爽。
- 刚柔变化让卷高潮升级。

## 断档风险

- 全卷都用同一种巧妙解法。
- 反派仇恨不足，正面清算不爽。
- 只追求操作复杂，忽略情绪。
- 爽点变化没有服务主角成长。

## 训练任务

给一卷设计三次危机，分别用巧劲、借力、刚劲解决。每次写清不同爽感和反派仇恨来源。
