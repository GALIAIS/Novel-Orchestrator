# 飞卢天榜-榜单含金量与市场取样

依据案例：batch-373 案例 1864

## 读者承诺

扫榜前先确认榜单含金量。越接近真实付费反馈的榜单，越适合用来取样题材、卖点和开篇节奏。

## 核心循环

```text
确认榜单规则 -> 判断含金量 -> 取样热书 -> 拆包装与开篇 -> 反推市场反馈
```

## 开篇必备

- 平台名。
- 榜单名。
- 规则是否明确。
- 样本代表的数据类型。

## 期待来源

- 减少选题盲猜。
- 看到付费读者正在选什么。
- 拆出热书共同卖点。
- 校准书名简介前三章。

## 断档风险

- 把所有榜单等量看待。
- 规则不清仍下结论。
- 用点击榜推断付费喜好。
- 只跟风题材，不拆包装和节奏。

## 训练任务

对同一平台三个榜单做含金量判断，只从最可靠榜单取 10 本书拆书名、简介和前三章。
