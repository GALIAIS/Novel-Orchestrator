# 下行剧情-咬牙过渡与视角换压

依据案例：`batch-237` 案例 1184。

## 读者承诺

高潮后的下行剧情不一定要推倒重写；现写连载优先过渡到上行，用关系期待、反派布局和信息差维持张力。

## 核心循环

```text
高潮后回落 -> 张力不足 -> 现写成本高 -> 补期待/换视角 -> 转入上行
```

## 开篇必备

- 明确下行段功能。
- 判断存稿是否允许大改。
- 设计关系期待或反派压力。
- 尽快安排上行恢复点。

## 期待来源

读者在平淡过渡中仍能看到下一轮危机、暧昧关系或反派布局，愿意等转折爆发。

## 断档风险

- 现写状态下反复第三版大改。
- 下行段只降温，不蓄压。
- 用暧昧日常完全替代主线。
- 反派视角铺太长，迟迟不转折。

## 训练任务

为一段下行剧情写三场：反派布局、主角察觉、危机转折。每场都标明它如何把读者带回下一段上行。
