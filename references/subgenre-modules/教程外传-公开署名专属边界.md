# 教程外传-公开署名专属边界

依据案例：`batch-122` 案例 608。

## 读者承诺

学习资料可以交流，但要分清公开内容和会员专属内容。公开教程可引用需署名，付费专属资料不能外传。

## 核心循环

```text
想分享资料 -> 判断公开/专属 -> 公开内容署名引用 -> 专属内容自用 -> 用公开链接替代外传
```

## 开篇必备

- 标注资料来源。
- 判断是否公开发布。
- 引用时保留作者署名。
- 会员专属资料不转发。
- 小范围讨论也要守边界。

## 期待来源

- 学习者可以合法交流。
- 原作者权益得到尊重。
- 社群资料边界清楚。
- 公开资料成为共同讨论入口。

## 断档风险

- 把付费资料当公开资料传播。
- 截图搬运不署名。
- 小范围外传破坏信任。
- 资料来源混乱，后续无法追溯。

## 训练任务

给自己的资料库加四个标签：公开可引、公开需署名、付费自用、禁止外传。每次分享前先检查标签。
