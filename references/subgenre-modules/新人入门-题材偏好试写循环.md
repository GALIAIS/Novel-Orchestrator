# 新人入门-题材偏好试写循环

依据案例：`batch-158` 案例 786。

## 读者承诺

新人不用先找到完美入门法，从自己最常看、最喜欢、最熟悉的题材开始，边拆边写边改。

## 核心循环

```text
阅读偏好 -> 题材选择 -> 拆同类书 -> 建素材库 -> 试写 -> 暴露问题 -> 再改
```

## 开篇必备

- 最近一年常看题材。
- 喜欢的看点。
- 同类高成绩样本。
- 可借鉴背景框架。
- 试写目标。
- 卡点记录。

## 期待来源

期待来自真实动手后问题变具体，学习不再停在抽象焦虑。

## 断档风险

- 一直找最优题材，不动笔。
- 背景框架凭空原创，负担过大。
- 仿写只抄表面。
- 不建素材库。

## 训练任务

列 3 个常看题材，拆一本同类书 10 章，再写 3000 字试稿并记录 5 个卡点。
