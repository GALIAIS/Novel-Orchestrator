# 玄幻等级势力-决策差异与剧情实证

依据案例：`batch-402` 案例 2006

## 读者承诺

不用生僻名称制造表面新颖，而让每个等级改变行动、成本、权限或反制，让每个势力以不同资源和制度逼出不同选择；金手指只偏离一条常规，并在事件中证明优势与代价。

## 核心循环

```text
最低力量与资源基线 -> 等级质变打开新行动 -> 势力制度分配资源权限 -> 金手指制造有限偏差 -> 主角利用规则获利 -> 反制与社会反应出现 -> 新层级重设问题
```

## 开篇必备

- 只介绍当前冲突所需的最低层级。
- 相邻等级有可观察的行动质变。
- 能力资源、成本和反制同时清楚。
- 势力控制不同稀缺资源或合法性。
- 招募、晋升、分配和惩罚制度一致。
- 金手指保留成本、盲区或暴露风险。

## 期待来源

读者先理解常规，再看主角用一项偏差改变方案；成长既带来能力与权限，也会唤醒更强反制和制度压力。

## 断档风险

- 用大量专有名词代替功能差异。
- 升阶后只是同样招式数值变大。
- 势力只靠服饰和善恶标签区分。
- 金手指直接抹平全部体系。
- 开篇一次性倾倒完整设定表。
- 规则从未改变人物选择。
- 反制只在作者需要时临时出现。

## 训练任务

不用专有名称设计三级力量和三个势力，让同一场护送在每级、每个势力下都有不同方案；再写一场交易或实战，证明金手指只违反一条规则且必须付出代价。
