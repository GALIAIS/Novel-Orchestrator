# 结构套娃-起承转合与折线层级

依据案例：`batch-248` 案例 1237。

## 读者承诺

起承转合可以套在卷、章、段和情绪折线里，用小结构持续吊住期待。

## 核心循环

```text
大起承转合 -> 小起承转合 -> 折线起承转合 -> 小转折服务大目标
```

## 开篇必备

- 先定上层目标。
- 再拆下层结构。
- 每层都有转折。
- 小结构不偏离主线。

## 期待来源

读者在大目标推进中不断吃到小期待和小变化。

## 断档风险

- 只套结构名词。
- 小结构脱离大目标。
- 转折过密导致混乱。
- 作者忘记主线。

## 训练任务

写一章两层起承转合：整章一层，高潮前段落一层，确认小层级服务整章目标。
