# 新技能爽点-获得期待使用兑现

依据案例：`batch-052` 案例 258。

## 读者承诺

新技能先拉期待，再用关键场景兑现；获得本身是否爽，取决于它是否满足主角欲望、是否被铺垫为珍贵、是否带来情绪宣泄。

## 核心循环

```text
获得技能 -> 价值铺垫 -> 拉低/拉高期待 -> 实战使用 -> 解决麻烦 -> 情绪宣泄
```

## 开篇必备

- 说明主角为什么想要这个技能。
- 交代技能效果和限制。
- 安排一个看低技能或看低主角的人。
- 给技能一个必须使用的麻烦。
- 使用后要有敌人失败或配角态度转变。

## 期待来源

- 技能等级和效果会给读者放望远镜。
- 被看低后反杀，会产生反差爽。
- 特别想要的技能，获得时也能爽。
- 技能帮助挣脱压迫，也能成为情绪宣泄。

## 断档风险

- 只学不用，期待会悬空。
- 技能太强又无限制，会破坏后续冲突。
- 技能价值没人见证，爽点会变弱。
- 获得技能没有欲望铺垫，只像发道具。

## 训练任务

写一个新技能桥段：获得、被看低、第一次使用、解决麻烦、见证者反应五步必须齐全。
