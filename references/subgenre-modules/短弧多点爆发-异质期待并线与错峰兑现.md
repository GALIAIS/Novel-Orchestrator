# 短弧多点爆发-异质期待并线与错峰兑现

依据案例：`batch-408` 案例 2040

## 读者承诺

短弧同时建立任务、群体、价值与关系等异质承诺；局部胜利保留可描述的未完成量，高潮由一个关键选择并线、不同角色分别回收，再以余波接续下一弧。

## 核心循环

```text
建立核心任务与异质次级承诺 -> 局部兑现但保留明确余量 -> 不同角色与铺垫分别蓄压 -> 关键选择使因果并线 -> 多方动作错峰完成结果 -> 认可与价值形成余波 -> 胜利后果开启下一弧
```

## 开篇必备

- 一个短弧核心不可能任务。
- 二至四条性质不同的次级承诺。
- 每条期待可准确描述的未完成量。
- 各维度独立的因果和负责回收者。
- 使多线汇合的关键选择。
- 最迟回收点与胜利后果钩子。

## 期待来源

局部突破证明问题可解，却让救人、制裁和身份等倒计时继续；高潮分别解除不同焦虑，群体结果和关系认可再扩大胜利意义。

## 断档风险

- 把连续战胜敌人误当多维爆发。
- 小回报只拖延、不留下明确余量。
- 主角一击自动解决所有问题。
- 所有反馈都只是旁观者重复震惊。
- 多条期待没有独立铺垫和负责角色。
- 下一弧钩子与本次胜利没有因果关系。

## 训练任务

设计一个八章短弧，为任务、群体、价值和关系四条期待分别填写建立、局部偿还、剩余量、负责回收者和最终结果；高潮至少用三条不同因果兑现，删除任一条后主弧都必须发生可见变化。
