# 都市异能-追逃缺爽点诊断

依据案例：`batch-112` 案例 556。

## 读者承诺

都市异能开篇不能只让主角连续被追。追逃必须展示金手指、世界奇异感、反派目的和主角压制力，否则读者只会记住“它追，他逃”。

## 核心循环

```text
被盯上 -> 触发麻烦 -> 展示能力 -> 揭开异常 -> 主线势力介入
```

## 开篇必备

- 第一波麻烦和主线有关。
- 金手指在事件中看得懂。
- 对手不能只是低级工具人。
- 有一两个世界奇异感细节。
- 结尾给出反派继续追的理由。

## 期待来源

- 主角能力压制对手。
- 玩具怪物等异常细节拉起世界观。
- 正反势力都盯上主角。
- 读者想知道主角身上有什么秘密。

## 断档风险

- 连续追逃但没爽点。
- 三章都在铺后面。
- 第一波工具人和主线断开。
- 场面太小，主角强度不明。

## 训练任务

把三次追逃合并为一次强事件，要求同时展示金手指、异常世界观和反派目的。
