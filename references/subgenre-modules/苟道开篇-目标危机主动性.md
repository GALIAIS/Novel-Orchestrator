# 苟道开篇-目标危机主动性

依据案例：batch-285 案例 1422

## 读者承诺

苟道开篇必须先给目标、危机和主动选择，让主角在压力下展示谨慎，而不是只说明他很苟。

## 核心循环

```text
小任务 -> 卷入危机 -> 想躲但选择出手 -> 保命准备破局 -> 新支线地图
```

## 开篇必备

- 主角明确要做的小任务。
- 外部危机。
- 主角不想惹事的理由。
- 必须出手的价值判断。
- 至少一个保命准备兑现。

## 期待来源

读者想看谨慎主角如何靠准备、判断和底牌解决危险，而不是被金手指拖着走。

## 断档风险

- 四章还不知道主角想干什么。
- 金手指接管主角。
- 信息铺太多，危机不足。
- 苟只停留在嘴上。

## 训练任务

写一个苟道开篇小任务，1000 字内完成“想躲、必须出手、用准备破局、打开新地图”。
