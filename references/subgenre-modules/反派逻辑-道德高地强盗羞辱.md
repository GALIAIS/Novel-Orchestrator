# 反派逻辑-道德高地强盗羞辱

依据案例：batch-265 案例 1321

## 读者承诺

让读者迅速站队并产生怒火：主角或正方明明有理，却被占便宜的人用强盗逻辑反过来羞辱、教育、道德绑架。

## 核心循环

```text
正当权益 -> 被侵犯 -> 对方拒不认错 -> 对方歪理上价值 -> 主角反击回收怒火
```

## 开篇必备

- 先让读者看见主角的正当性。
- 反派伤害要具体，不只笼统说坏。
- 歪理要像现实中真的有人会说的话。
- 第一轮冲突最好短、脏、准，快速激怒读者。

## 期待来源

期待来自“这人凭什么这么理直气壮”的不公平感。读者不是单纯想看反派输，而是想看被倒置的价值秩序重新摆正。

## 断档风险

- 反派只坏不狡辩，怒火不够持久。
- 主角正当性没铺好，读者站队不稳。
- 歪理太假，会像作者硬造冲突。
- 怒火长期不回收，会变成憋屈。

## 训练任务

写三个低级反派歪理，每个都要包含“占便宜行为、倒打一耙说法、主角一句反击”。从中挑一个扩成 800 字冲突。
