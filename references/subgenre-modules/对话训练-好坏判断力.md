# 对话训练-好坏判断力

依据案例：batch-289 案例 1443

## 读者承诺

台词训练先建立好坏判断力，再用拉片和改写把信息、关系、冲突、潜台词落到句子里。

## 核心循环

```text
尬硬台词 -> 判断功能 -> 学习原理 -> 拉片校准语感 -> 删除/改写无效句
```

## 开篇必备

- 当前台词问题。
- 每句台词功能。
- 信息推进。
- 关系变化。
- 潜台词。
- 对话后的行动。

## 期待来源

读者在对话中看到信息和关系变化，而不是听角色互相说明设定。

## 断档风险

- 只靠语感，不建立判断标准。
- 台词只抒情或说明，不推动情节。
- 翻译书例子直接照搬到中文网文。
- 对话占比不高就忽略质量。

## 训练任务

拿一段尴尬对话逐句标功能，没有功能的删掉，只有说明的改成潜台词。
