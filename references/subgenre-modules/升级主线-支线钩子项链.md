# 升级主线-支线钩子项链

依据案例：`batch-144` 案例 720。

## 读者承诺

主角的真正长线是升级成长，家人失踪、仇敌、约定等支线像项链上的珍珠，缠绕主线并迫使主角不断变强。金手指可以是明面系统，也可以是穿越信息差和现代知识。

## 核心循环

```text
隐性优势 -> 初期资源 -> 缓解危机 -> 升级成长 -> 支线钩子压迫 -> 阶段回收 -> 更高地图
```

## 开篇必备

- 主角低位处境。
- 一个能兑现的隐性金手指。
- 第一笔资源或资格获取。
- 一个未完全解决的危机。
- 一个会长期逼迫主角成长的支线钩子。

## 期待来源

读者期待主角每次变强都有现实原因和目标牵引。支线不是散点任务，而是不断提醒主角“你还不够强”的压力源。

## 断档风险

- 把金手指理解得太窄，只等系统出现。
- 支线和成长无关，变成杂事。
- 阶段回收完全不是主角造成，爽感弱。
- 提纲按固定章数硬切，正文节奏失真。

## 训练任务

设计一个穿越武道开头：主角只靠现代知识获得第一笔资源，并借此进入更安全或更有上升空间的位置。再写一个长期支线钩子，说明它如何在前三卷分别逼迫主角升级。
