# DND资料片-魔法船星际化扩展

依据案例：`batch-345` 案例 1724

## 读者承诺

魔法船这类交通设定能把传统奇幻从固定地图扩展成星界或星际冒险，带来新事件循环。

## 核心循环

```text
固定奇幻地图 -> 新交通工具 -> 地图尺度放大 -> 未知空间 -> 船队冒险循环
```

## 开篇必备

- 交通设定能改变行动半径。
- 新地图带来新资源和敌人。
- 船只自带团队、补给、航线。
- 设定要产生事件，不只做背景。
- 冲突类型同步升级。

## 期待来源

读者期待角色离开旧地图，进入未知空间、文明、遗迹和航线风险。

## 断档风险

- 只加设定名词。
- 交通工具不改变剧情。
- 新地图没有新资源。
- 船队没有团队关系。
- 奇幻科幻融合后冲突仍旧。

## 训练任务

给一个固定地图奇幻故事增加一种交通工具，设计招募、补给、探索、遭遇战、升级五个事件。
