# 成绩差-先查开头再改节奏

依据案例：`batch-122` 案例 607。

## 读者承诺

推荐成绩差时，先查开头，不要立刻把全书基调改成过山车。开头承诺、第一冲突和继续读理由不成立，后文节奏再强也接不住。

## 核心循环

```text
成绩差 -> 想大改节奏 -> 回到开头排查 -> 找承诺/冲突/情绪缺口 -> 定点修开篇
```

## 开篇必备

- 第一章给出清楚承诺。
- 第一冲突尽早发生。
- 日常感也要有继续读理由。
- 推荐反馈先定位读者流失点。
- 改稿前列出问题再求评。

## 期待来源

- 读者知道这本书要看什么。
- 日常中有稳定关系、目标或悬念。
- 开头情绪启动后，后续节奏才有效。
- 定点修复降低大改风险。

## 断档风险

- 成绩差就全书改风格。
- 过山车剧情破坏原有日常承诺。
- 不分析开头就追求强刺激。
- 把推荐失败归因过宽。

## 训练任务

复盘前 3 章，每章写一个读者继续看的理由。写不出来的章节，只改承诺、冲突和情绪启动。
