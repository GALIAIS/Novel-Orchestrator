# 人物塑造-外貌行为心理三层递进

依据案例：`batch-210` 案例 1050。

## 读者承诺

人物塑造先给外貌记忆点，再用言行显性格，最后用心理、目的和立场解释选择。

## 核心循环

```text
外貌记忆点 -> 言语行为 -> 性格判断 -> 动机心理 -> 立场选择
```

## 开篇必备

- 外貌只抓一两个记忆点。
- 台词和动作要能显露性格。
- 心理描写要服务关键选择。
- 立场和心态要与行动一致或形成张力。

## 期待来源

读者先记住人物，再判断人物，最后理解人物为什么这样选择。

## 断档风险

- 外貌流水账。
- 性格全靠作者标签。
- 心理描写脱离行动。
- 人物没有立场和代价。

## 训练任务

写 800 字人物出场：外貌记忆点、言行性格、关键选择中的目的和立场三层都要出现。
