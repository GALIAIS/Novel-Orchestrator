# 番茄玄幻-传统脑洞分流

依据案例：batch-311 案例 1555

## 读者承诺

番茄能写玄幻，但要分传统玄幻和脑洞玄幻。免费站更吃强钩子、强刺激和榜单正在验证的题材。

## 核心循环

```text
想写玄幻 -> 看番茄榜单 -> 区分传统/脑洞 -> 加强第一章卖点 -> 决定入场
```

## 开篇必备

- 先看当下榜单热题材。
- 判断自己是传统升级还是脑洞融合。
- 第一章给直观卖点。
- 可融合惊悚游戏、龙王战神等强钩子元素。

## 期待来源

期待来自题材包装贴合平台。读者一眼知道有什么新鲜刺激，就更愿意点下去。

## 断档风险

- 按传统玄幻慢热写免费站。
- 不看榜单，只凭印象判断热度。
- 脑洞元素只是贴皮，不进主线。
- 第一章卖点不够直观。

## 训练任务

把一个传统玄幻设定改成番茄脑洞版：保留升级核心，加入一个强钩子，并写第一章卖点画面。
