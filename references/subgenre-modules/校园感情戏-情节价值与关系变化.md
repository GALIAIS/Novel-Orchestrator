# 校园感情戏-情节价值与关系变化

依据案例：`batch-207` 案例 1032。

## 读者承诺

校园暧昧可以轻，但情节不能空；一次互动必须带来关系、心理、处境或矛盾变化。

## 核心循环

```text
日常事件 -> 暧昧互动 -> 代价/误会 -> 心理变化 -> 关系推进
```

## 开篇必备

- 事件本身有价值或代价。
- 互动后关系状态改变。
- 角色心理有明确变化。
- 后续有可承接的钩子。

## 期待来源

读者期待小事件带来关系推进，而不是只看两个人说了几句话。

## 断档风险

- 情节只是发生，没有变化。
- 事件太普通，没有附加代价。
- 男频感情戏过轻飘。
- 外貌细节多，关系推进少。

## 训练任务

把校园日常暧昧桥段改两版：一版加外部代价，一版加关系误会，结尾都要改变关系状态。
