# 网文综合症-自我归因与现实校正

依据案例：`batch-239` 案例 1194。

## 读者承诺

新人扑街后要把“怀才不遇”转成可验证问题；数据和读者反馈会校正自我想象。

## 核心循环

```text
自信过高 -> 作品反馈差 -> 倾向甩锅 -> 数据打脸 -> 回到作品问题
```

## 开篇必备

- 记录真实数据。
- 对照同类榜样书。
- 分离外部原因和作品问题。
- 把抱怨改成可验证假设。

## 期待来源

作者不再把失败解释成世界不懂自己，而是找到能改的题材、开头、节奏、爽点问题。

## 断档风险

- 自认开创流派却没扫榜。
- 把编辑和读者都当成不懂。
- 成绩差只怪推荐。
- 长期沉浸怀才不遇叙事。

## 训练任务

把一次扑街的 3 个外部借口，改写成 3 个可检查问题：题材、开头、节奏、爽点、平台适配或更新。
