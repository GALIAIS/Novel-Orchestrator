# 网文基本功-低门槛表达

依据案例：`batch-185` 案例 922。

## 读者承诺

网文基本功的最低承诺是读者能顺畅理解，并在关键处被情绪带动。

## 核心循环

```text
语句通顺 -> 标点正确 -> 表达清楚 -> 重点可归纳 -> 情绪可传达
```

## 开篇必备

- 主语清楚。
- 事件说完整。
- 标点不干扰阅读。
- 少用提高门槛的抒情和氛围堆叠。

## 期待来源

读者不被文字绊住，才能进入人物、冲突和爽点。

## 断档风险

- 句子不通。
- 标点混乱。
- 抒情过度赶走小白读者。
- 重点散，读者不知道这一章在干什么。

## 训练任务

检查 1000 字正文：修正不通顺句、错标点、主语不明句，并用 3 句话概括事件、情绪、追读点。
