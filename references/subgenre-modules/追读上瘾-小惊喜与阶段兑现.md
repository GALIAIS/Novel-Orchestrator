# 追读上瘾-小惊喜与阶段兑现

依据案例：batch-378 案例 1887

## 读者承诺

追读上瘾来自未完成期待和阶段满足交替。每章给一点反馈，但保留下一轮不确定。

## 核心循环

```text
期待 -> 拉扯 -> 小惊喜 -> 阶段兑现 -> 新目标
```

## 开篇必备

- 当前目标。
- 未确定问题。
- 小反馈点。
- 章末新钩子。

## 期待来源

- 读者想知道结果。
- 情绪被拉扯。
- 伏笔逐步揭开。
- 目标达成后还有下一个目标。

## 断档风险

- 只吊不兑现。
- 只爽不留疑问。
- 小惊喜和主线无关。
- 阶段目标不清楚。

## 训练任务

设计 5 章追读链，每章给一个小反馈并保留未完成期待，第 5 章兑现后打开新目标。
