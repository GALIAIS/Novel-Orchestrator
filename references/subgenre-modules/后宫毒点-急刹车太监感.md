# 后宫毒点-急刹车太监感

依据案例：`batch-134` 案例 666。

## 读者承诺

后宫文要让读者看到男主对妹子有欲望、有行动、有推进。拉期待可以设置阻碍，但不能让男主在该推进时无理由退缩。

## 核心循环

```text
好感升温 -> 推进期待 -> 外部阻碍/代价 -> 男主保持主动 -> 延后兑现
```

## 开篇必备

- 明确男主有欲望和行动力。
- 女主反馈要清楚。
- 阻碍来自外部或代价，不来自男主怂。
- 延后兑现要挂下一次期待。
- 不照搬日轻长期不推。

## 期待来源

- 女主好感逐步升温。
- 男主主动推进关系。
- 欲拒还迎带来张力。
- 阻碍有现实代价，读者不骂男主。

## 断档风险

- 关键时刻男主无理由拒绝。
- 女主已经到位却强行急刹车。
- 把日轻亚撒西当后宫文模板。
- 读者把克制理解成太监。

## 训练任务

写 800 字后宫推进桥段：好感到位但暂不完全兑现，阻碍必须来自外部危机、身份代价或女主试探，不能写男主无理由退缩。
