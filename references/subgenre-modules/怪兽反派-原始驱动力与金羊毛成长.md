# 怪兽反派-原始驱动力与金羊毛成长

依据案例：`batch-210` 案例 1046。

## 读者承诺

主角从纯粹原始动机出发，对抗有压迫性的怪兽反派，并在追目标途中获得成长和更大使命。

## 核心循环

```text
笼中怪兽 -> 原始动机 -> 追寻目标 -> 旅途收获 -> 使命升级
```

## 开篇必备

- 反派具备压迫性和威胁性。
- 主角动机简单到读者本能认可。
- 目标能自然指向最终 boss。
- 旅途收获伙伴、能力、身份或责任。

## 期待来源

读者先认同主角救亲人、求自由或求生，再接受他逐步成为英雄。

## 断档风险

- 普通主角开局就喊拯救世界。
- 反派没有压迫性。
- 金羊毛收获和原始目标脱节。
- 私人动机没有升级节点。
