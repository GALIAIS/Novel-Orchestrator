# 情绪机制-不公平伤害价值浪费

依据案例：`batch-181` 案例 904。

## 读者承诺

不公平伤害叠加价值无谓浪费，可以快速制造郁气、血压和反击期待，适合压迫开局、熊孩子冲突和反派正当清算。

## 核心循环

```text
珍贵价值 -> 低劣理由破坏 -> 价值没有转移只被浪费 -> 旁人轻描淡写 -> 主角反击获授权
```

## 开篇必备

- 先让读者知道被毁之物为什么珍贵。
- 施害理由要低劣且不对等。
- 损失不能只是钱，要承载努力、希望、生命或稀缺性。
- 反击前给足读者郁气。

## 期待来源

读者期待有人把被轻描淡写的价值重新放回秩序中心，让浪费者承担代价。

## 断档风险

- 没写出价值，只写出争吵。
- 施害者太蠢导致桥段假。
- 旁人包庇过度但主角反击不足。
- 反击不匹配损失，情绪落空。

## 训练任务

设计一个价值浪费开局：写清被毁对象的形成成本、承载意义、施害者低劣理由、旁人包庇话术和主角反击的正当性。
