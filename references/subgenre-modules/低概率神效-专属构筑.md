# 低概率神效-专属构筑

依据案例：`batch-142` 案例 708。

## 读者承诺

别人避开的低概率、低评价、冷门机制，在主角手里变成稳定收益；但金手指必须绑定专属职业或机制，形成可成长构筑路线。

## 核心循环

```text
冷门职业 -> 垃圾低概率技能 -> 主角稳定触发 -> 反常识装逼 -> 解锁更高阶低概率机制
```

## 开篇必备

- 世界随机机制。
- 主角职业选择理由。
- 被轻视的低概率技能。
- 金手指适用范围和不能生效范围。
- 初始限制。

## 期待来源

- 捡漏感。
- 信息差暗爽。
- 职业加金手指形成专属路线。
- 构筑逐步成型。

## 断档风险

- 金手指对所有随机都生效。
- 任意职业都一样强。
- 没有限制，后续无成长。
- 职业选择没有戏剧意义。

## 训练任务

写 800 字现实网游化开局：主角选冷门职业，金手指只绑定该职业一类低概率机制。第一章出现一个被旁人认为没用的技能，并证明这是专属路线。
