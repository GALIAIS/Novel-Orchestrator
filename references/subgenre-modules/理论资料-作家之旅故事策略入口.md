# 理论资料-作家之旅故事策略入口

依据案例：`batch-109` 案例 542。

## 读者承诺

《作家之旅》《故事策略》这类资料提供的是结构理论底层，不是网文速成公式。读者需要把理论章节转成大纲字段，再放回具体网文案例和自己的项目中验证。

## 核心循环

```text
阅读理论 -> 标注概念 -> 对照案例 -> 转成字段 -> 应用到大纲
```

## 开篇必备

- 明确当前要解决的创作问题。
- 不只收藏 PDF。
- 阅读时做标注。
- 每个概念都对应一个网文案例。
- 最后转成可填写模板。

## 期待来源

- 抽象理论变成可操作字段。
- 大纲不再只靠感觉。
- 结构问题有参考坐标。
- 反复翻阅能持续复用。

## 断档风险

- 只囤资料不读。
- 只背理论名词，不落到剧情。
- 用经典理论替代平台样本。
- 电子资料不方便标注导致长期闲置。

## 训练任务

读一个理论章节，提炼 3 个字段，填进自己的大纲。不能只写概念名，必须写“我的项目里对应哪一段”。
