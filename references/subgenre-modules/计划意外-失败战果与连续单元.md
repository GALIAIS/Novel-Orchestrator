# 计划意外-失败战果与连续单元

依据案例：batch-367 案例 1834

## 读者承诺

计划-意外是可连续组合的剧情单元；每次失败都要带回阶段战果或线索，并预留新危机。

## 核心循环

```text
目标 -> 计划 -> 意外 -> 失败下行 -> 战果线索 -> 新计划
```

## 开篇必备

- 明确目标。
- 看似可行的计划。
- 打断计划的意外。
- 失败后的阶段收获。

## 期待来源

- 计划可行带来期待。
- 意外制造下行。
- 失败仍有线索。
- 下一轮计划更接近成功。

## 断档风险

- 失败没有任何收获。
- 意外只拖延剧情。
- 战果不指向下一计划。
- 连续单元之间没有升级。

## 训练任务

写 5 个计划-意外单元，每个单元都要有失败、战果、遗留危机和下一轮计划。
