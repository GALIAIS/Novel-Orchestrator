# 玄幻迭代-名利力量多点爆发

依据案例：`batch-190` 案例 948。

## 读者承诺

玄幻爽点不只来自力量升级，也可以让名、利、力量在同一小高潮里互相放大，形成多点爆发。

## 核心循环

```text
价值观应用 -> 获得名 -> 获得利/力量 -> 小高潮汇合 -> 多重收获增幅
```

## 开篇必备

- 主角价值观能落到行动。
- 至少两条收获线各有阻碍。
- 收获类型可区分为名、利、力量或权。
- 世界设定能解释这些收获如何互相绑定。

## 期待来源

读者不只看到主角变强，还看到他获得声望、资源、地位，并让这些收获互相推动。

## 断档风险

- 只写力量，不写名利反馈。
- 多条线各自爆发，互不相干。
- 价值观硬讲，不通过冲突应用。
- 世界设定无法支撑名利力量联动。

## 训练任务

设计一个小高潮，安排主角同时获得名和利，且名推动利、利反过来放大名。
