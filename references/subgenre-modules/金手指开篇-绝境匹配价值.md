# 金手指开篇-绝境匹配价值

依据案例：`batch-329` 案例 1644

## 读者承诺

金手指第一次显价值时，要解决一个没有它就过不去的绝境。

## 核心循环

```text
重罚困境 -> 普通方法失效 -> 金手指介入 -> 暂时过关 -> 新麻烦下行
```

## 开篇必备

- 失败惩罚足够重。
- 危机和金手指强度匹配。
- 主角处在压力中心。
- 金手指解决具体问题。
- 章末给下一轮压力。

## 期待来源

读者期待外挂不是摆设，而是把主角从真实困境里拉出来。

## 断档风险

- 失败也能混下去，危机变轻。
- 日常关系戏抢在绝境前面。
- 主角像旁观者。
- 金手指出现太随意。
- 没有上行下行，章末不追读。

## 训练任务

重写金手指第一次使用前的 800 字：先写失败代价，再写普通方法无效，最后用金手指破局。
