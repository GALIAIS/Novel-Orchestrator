# 反差冷笑话-严肃错位搞笑

依据案例：batch-272 案例 1356

## 读者承诺

用正常语境和错位回答制造低成本笑点，让轻松文获得角色趣味。笑点来自读者预期与角色认真逻辑的偏移，不是随机怪话。

## 核心循环

```text
正常问题 -> 常规答案预期 -> 角色认真错答 -> 旁人反应 -> 气氛变轻松
```

## 开篇必备

- 场景语境要正常，读者知道通常该怎么答。
- 角色回答要一本正经，不能像作者硬塞段子。
- 错位回答要符合角色性格。

## 期待来源

读者期待的是轻松、意外和角色可爱。越严肃的提问，越需要角色以另一套逻辑认真接住，形成反差。

## 断档风险

- 只为搞笑而破坏角色可信度。
- 严肃剧情里频繁插段子，削弱危机。
- 错位没有常规预期垫底，读者看不出笑点。

## 训练任务

列 10 个正常问答，先写常规回答，再写错位回答，最后只保留符合角色性格的 3 个。
