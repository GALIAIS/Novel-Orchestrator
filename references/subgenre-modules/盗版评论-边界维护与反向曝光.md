# 盗版评论-边界维护与反向曝光

依据案例：`batch-233` 案例 1161。

## 读者承诺

盗版读者闯入正版评论区是越界行为；作者可以顺势调侃形成曝光，但要守住正版边界。

## 核心循环

```text
盗版越界 -> 正版站队 -> 轻度调侃 -> 引导正版 -> 维护评论区
```

## 开篇必备

- 不提供盗版线索。
- 不和盗版读者长线争吵。
- 保护正版读者观感。
- 将荒诞点导向正版支持。

## 期待来源

正版读者看到作者有边界、有态度，围观者也可能因荒诞评论产生好奇。

## 断档风险

- 为营销主动鼓励盗版。
- 评论区被盗版话题污染。
- 和盗版读者争吵过久。
- 书友群索要原文件时没有拒绝。

## 训练任务

写三条盗版评论处理话术：调侃、边界、正版引导。每条不超过 60 字，不提供盗版信息。
