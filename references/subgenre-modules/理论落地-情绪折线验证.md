# 理论落地-情绪折线验证

依据案例：`batch-036` 案例 178。

## 读者承诺

理论要用作品数据验证。情绪折线的价值，是把凭感觉的爽点与追读，变成可设计、可复盘、可迭代的章节结构。

## 核心循环

```text
旧书问题 -> 理论学习 -> 新书应用 -> 上架验证 -> 追订复盘
```

## 开篇必备

- 记录旧书追订问题。
- 标出新书情绪折线。
- 观察上架数据。
- 继续看追订是否断档。

## 期待来源

- 作者知道方法是否有效。
- 情绪上行下行有证据。
- 读者留存反向验证结构。
- 经验转化成可复制能力。

## 断档风险

- 只庆祝成绩，不复盘方法。
- 精品靠运气，下一本不可复制。
- 理论停在笔记，没进章节。
- 只看首订，不看后续追订。

## 训练任务

为一章正文画情绪折线，标出铺垫、兑现、新期待，并写 3 条可能导致追订下跌的断档点。
