# 练习互评-五折线标记与同伴校正

依据案例：`batch-243` 案例 1215。

## 读者承诺

五折线训练要靠写作和互评一起完成；标错虚晃一枪时，同伴指出具体位置比泛泛鼓励更有价值。

## 核心循环

```text
写作业 -> 标折点 -> 同伴认真看 -> 指出标注偏差 -> 作者修正结构理解
```

## 开篇必备

- 给每个折点标功能。
- 写清机会出现的位置。
- 写清机会暂时落空的原因。
- 接受互评后回改标注。

## 期待来源

作者通过他人视角发现自己以为成立的结构点，其实在读者眼里承担了别的功能。

## 断档风险

- 把下行误标成虚晃。
- 只写鼓励，没有具体校正。
- 作者不说明原本标注理由。
- 机会和兑现之间缺误会或阻断。

## 训练任务

重标一个五折线梗概：每个虚晃都写出“读者以为会发生什么”和“为什么暂时没发生”。
