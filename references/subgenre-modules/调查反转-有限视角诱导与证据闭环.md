# 调查反转-有限视角诱导与证据闭环

依据案例：`batch-415` 案例 2071

## 读者承诺

读者从信息最少的基层人物进入一场真假难辨的接触，先共享对陌生调查者的怀疑，再通过可验证身份完成地位翻转；真正目标因恐惧暴露而作出异常选择，调查者利用差异信息逐句诱导其补充可核细节。记录设备保存过程，但最终结算仍依赖权限、证据保管和程序闭环，同一结果同时改写入口人物、腐败目标与公共状态。

## 核心循环

```text
基层入口人物遇到可疑请求 -> 按职业常识质疑并启动身份核验 -> 权威验证改变信任与现场地位 -> 真正目标因害怕暴露主动露出异常 -> 调查者制造错误安全感并逐句锁定细节 -> 记录物保存言行 -> 证据与程序交叉验证 -> 同一结果结算误判、权力反转和公共后果
```

有限视角保护第一重反转，四方信息差支撑诱导，逐句状态变化提供智力感，程序闭环防止“录到一句话就万能定罪”。

## 开篇必备

- 调查者、入口人物、目标和核验机构四方角色表。
- 每方掌握的信息、即时目标和最怕失去的东西。
- 入口人物合理怀疑陌生人的职业依据。
- 可被独立验证的身份凭据与核验回路。
- 目标最先露出的异常动作及既有恐惧。
- 八句以内会改变信任、事实或行动的关键台词。
- 记录设备、证据来源、保管和交叉验证。
- 需要另查的权限、录音、诱导与逮捕程序。
- 三项共享同一因果的结算。

## 期待来源

- 陌生调查者究竟是真是假。
- 入口人物何时发现真正被调查的是上级。
- 目标为何比其他人更早慌乱。
- 哪句话会让目标误以为自己仍然安全。
- 记录内容能否通过其他证据验证。
- 调查者是否越过程序或伦理边界。
- 同一证据会怎样改变个人、组织和受害者状态。

## 断档风险

- 为反转让基层人物放弃基本核验常识。
- 调查者凭空知道一切，没有信息来源。
- 对话只显得聪明，却不改变任何状态。
- 目标无缘无故自曝完整罪行。
- 把执法记录仪直接等同于可采铁证。
- 打脸基层人物压过真正权力滥用。
- “维护正义”只有口号，没有受害者或制度变化。
- 用现实机构名称却不核对司法程序。

## 训练任务

写一场 2200 字有限视角调查谈话。提前制作四方信息表，安排一次身份核验和八句状态变化台词；记录设备只能保存证据，不能直接完成定罪。结尾分别结算入口人物误判、目标权力和公共后果，并列出三项待核现实程序。
