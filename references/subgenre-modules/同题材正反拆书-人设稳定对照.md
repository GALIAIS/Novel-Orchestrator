# 同题材正反拆书-人设稳定对照

依据案例：`batch-047` 案例 232。

## 读者承诺

同题材正反样本最适合看人设稳定。公共认知强的主角，第一章必须符合读者已有期待，否则立刻出戏。

## 核心循环

```text
同题材样本 -> 正反对照 -> 人设承诺 -> 崩点定位 -> 写法迁移
```

## 开篇必备

- 选同题材两本书。
- 先拆第一章主角行为。
- 对照公共认知承诺。
- 标出立住点和崩塌点。

## 期待来源

- 正样本强化读者认知。
- 反样本暴露不能写的行为。
- 同题材差异清楚。
- 拆法更容易校准。

## 断档风险

- 只看设定相似，不看人设执行。
- 公共认知角色写崩。
- 对话荒诞破坏信任。
- 观察力和格局不符合角色。

## 训练任务

找两本同题材开篇，分别列出第一章 5 个立住人设的动作和 5 个毁人设的动作。
