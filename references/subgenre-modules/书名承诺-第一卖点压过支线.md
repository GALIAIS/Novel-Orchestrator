# 书名承诺-第一卖点压过支线

依据案例：`batch-339` 案例 1695

## 读者承诺

书名里的第一卖点必须前三章兑现，支线再合理也不能抢走读者点进来的核心期待。

## 核心循环

```text
书名第一承诺 -> 前三章兑现 -> 支线服务主危机 -> 第一卖点破局 -> 后续再扩展支线
```

## 开篇必备

- 明确书名第一承诺。
- 第一章点出核心卖点。
- 支线篇幅受控。
- 主角有主动计划。
- 重要女角色帮助有理由。

## 期待来源

读者期待点击书名后马上看到对应卖点，而不是被世界观、对话和第二金手指支线带偏。

## 断档风险

- 三章不出书名卖点。
- 支线占前三章大头。
- 对话无有效信息。
- 世界观铺垫太慢。
- 女角色无理由对主角好。

## 训练任务

重排前三章：第一章点出书名卖点，第二章危机逼近，第三章用第一卖点破局；删掉不服务主危机的支线。
