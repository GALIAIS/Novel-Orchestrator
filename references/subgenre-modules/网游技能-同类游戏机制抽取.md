# 网游技能-同类游戏机制抽取

依据案例：batch-268 案例 1338

## 读者承诺

让网游小说技能像真实游戏机制，而不是只有炫酷名字：从同类游戏和规则书中抽机制，再改造成自己的职业体系。

## 核心循环

```text
确定职业 -> 找同类游戏 -> 抽技能机制 -> 改名改数值 -> 接入战斗选择
```

## 开篇必备

- 先定网游风格和职业定位。
- 技能要有冷却、消耗、范围、控制或连招功能。
- 参考游戏要和作品风格相近。
- 技能必须能改变战斗选择。

## 期待来源

期待来自读者相信这套技能真的能玩：有克制、有配合、有操作、有翻盘空间。

## 断档风险

- 只用取名器生成名字。
- 不同风格游戏技能乱混。
- 技能没有机制，只是光效描写。
- 参考后不改职业语境和数值。

## 训练任务

从一款同类游戏里拆 8 个技能，只保留机制，全部重命名并改成自己的职业技能。
