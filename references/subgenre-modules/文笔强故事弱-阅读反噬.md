# 文笔强故事弱-阅读反噬

依据案例：`batch-041` 案例 204。

## 读者承诺

文笔能把读者吸住，但故事回报不足会造成读后反噬。语言、氛围和故事满足要分开评估。

## 核心循环

```text
语言吸引 -> 阅读黏性 -> 等待回报 -> 回报不足 -> 反噬
```

## 开篇必备

- 区分语言优势和故事功能。
- 判断主角目标是否清楚。
- 检查故事回报。
- 明确可学和不可学部分。

## 期待来源

- 高水平语言带来沉浸。
- 氛围和意象有审美收益。
- 读者期待故事闭合。
- 拆解时只取需要的能力。

## 断档风险

- 把文笔好误认为故事好。
- 新人仿写导致节奏变慢。
- 读后没有满足。
- 奖项和口碑替代自己的判断。

## 训练任务

拆一个文笔强样本：只拆语言 500 字，再写清为什么不照搬它的故事结构。
