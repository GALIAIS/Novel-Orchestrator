# 历史权谋反派-利益网络与合法性转换

依据案例：`batch-408` 案例 2039

## 读者承诺

反派通过需求、收益、权威、制度和公共声望建立分层利益共同体；主角追踪资源转换接口、改变参与者激励，使网络先失去合法性与庇护，再进入最终对决。

## 核心循环

```text
控制需求或稀缺资源 -> 收益绑定中间人 -> 权威制造可信外观 -> 制度分享利益或风险 -> 公共声望提高质疑成本 -> 证据与内部矛盾积累 -> 主角切断转换边 -> 联盟分裂暴露 -> 反派失去庇护
```

## 开篇必备

- 事业表面的公共价值和隐藏代价。
- 至少三层利益不同的参与者。
- 财富转成背书、背书转成庇护的具体场景。
- 符合时代制度的传播节点。
- 各层退出成本与一个证据接口。
- 网络崩塌后的秩序替代问题。

## 期待来源

反派越被调查越显出多层合法外观，压迫感持续上升；主角每切断一条资源转换边，就让一层盟友因利益变化而松动，形成可推理的层层破局。

## 断档风险

- 用“手眼通天”替代参与者激励。
- 所有盟友都只因恐惧服从。
- 慈善和工程只有标签，没有受益与包装双重作用。
- 主角只找罪证，不改变网络激励。
- 顶层倒下后忽略受害者和制度空缺。
- 直接照搬现实违法流程形成操作性描写。

## 训练任务

为架空王朝设计商人、权威、官员、公共受益者和受害者五层网络，逐层写清收益、风险、退出成本与证据接口；再让主角只切断一条资源转换边，触发联盟连锁分裂。
