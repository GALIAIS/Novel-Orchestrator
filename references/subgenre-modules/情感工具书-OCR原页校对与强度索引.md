# 情感工具书-OCR原页校对与强度索引

依据案例：`batch-401` 案例 2003

## 读者承诺

把 OCR 当作情感资料的搜索层，以原始页、稳定页码和人工校对保住证据，再按触发、身体、行动、关系与强度建立能服务场景的索引。

## 核心循环

```text
保存原件 -> 建立页码稳定的 OCR 层 -> 分层抽检 -> 标记可信等级 -> 建立情绪与场景索引 -> 搜索后回查原页 -> 按角色语体和处境转写
```

## 开篇必备

- 原件、OCR 文件、来源和版本信息同时保存。
- 处理后仍能稳定回到具体原页。
- 抽检覆盖前中后段与复杂版式。
- 区分“可引用、仅定位、待核验”。
- 索引连接情绪、触发、身体信号、行动冲动和强度。
- 每个正式方法结论都保留原页锚点。

## 期待来源

作者可以快速找到候选表达，又不会被 OCR 错误或孤立词表误导；搜索结果最终转成符合人物习惯的动作与选择。

## 断档风险

- 原件不可读仍宣称完成内容吸收。
- 依据书名或推荐语概括书中观点。
- 把 OCR 文本当作原始证据。
- 页码在转换后丢失。
- 只抽检整洁页面，忽略表格和脚注。
- 直接粘贴情绪词句而不做角色转写。
- 没有校对状态和来源链。

## 训练任务

对一份原件可读资料人工抽检 10 页，登记识别错误并建立 20 条带原页锚点的情绪索引；随后把同一情绪按低、中、高强度写成三个不同角色的动作场景。
