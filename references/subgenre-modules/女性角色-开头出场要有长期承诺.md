# 女性角色-开头出场要有长期承诺

依据案例：`batch-329` 案例 1643

## 读者承诺

开头可以出现女性角色，但越早出现越像承诺；必须有长期作用、身份预期和主线承接。

## 核心循环

```text
早期出场 -> 读者判断重要性 -> 长期作用兑现 -> 关系线稳定服务主线
```

## 开篇必备

- 标清她是否女主。
- 明确 20 章内核心作用。
- 避免非女主抢在女主前制造误判。
- 角色牵出的家族/父亲/支线要能接入主线。
- 临时加戏后回到大纲检查。

## 期待来源

读者期待早期重要角色不是随手加的支线，而是会反复出现并改变主线。

## 断档风险

- 作者写爽了，打乱总体节奏。
- 角色被读者误判为女主。
- 父亲、家族等支线挤压主线。
- 早期角色后续没有兑现。
- 删改时只删剧情，不处理读者预期。

## 训练任务

列出前三章女性角色，标注“女主/长期配角/工具人/误导项”，没有长期作用的后移、合并或删除。
