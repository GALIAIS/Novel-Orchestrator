# 后悔流-三线轮动

依据案例：`batch-139` 案例 695。

## 读者承诺

让读者持续看到男主离开前任后越过越好：前任逐步破防，金手指持续给收益，新女主关系不断升温。

## 核心循环

```text
前任嘲讽 -> 男主收益 -> 女主被吸引 -> 前任后悔 -> 男主收益升级 -> 感情升温 -> 前任不甘愤怒
```

## 开篇必备

- 分手、离婚或退婚带来的羞辱点。
- 前任明确低估男主。
- 金手指或事业收益线。
- 新女主初次关系钩子。
- 前任第一次后悔但不彻底崩溃。

## 期待来源

- 打脸/后悔线提供解气。
- 奖励/赚钱线提供数据正反馈。
- 感情线提供关系奖励。
- 三线轮流推进，慢节奏也不空。

## 断档风险

- 前任一章内彻底破防，后续无拉扯。
- 金手指收益和情绪线脱节。
- 感情线长期不推进，只剩赚钱流水。
- 三线过早正面碰撞，提前耗尽期待。

## 训练任务

写一个五轮后悔流大纲。每轮必须包含男主收益、女主关系进展、前任情绪升级；前任情绪按嘲讽、怀疑、后悔、不甘、愤怒分阶段推进。
