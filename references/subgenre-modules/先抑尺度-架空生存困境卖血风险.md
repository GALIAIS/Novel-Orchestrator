# 先抑尺度-架空生存困境卖血风险

依据案例：`batch-200` 案例 1000。

## 读者承诺

“欲扬先抑”可以写主角低谷，但卖血、卖肾等现实敏感行为要架空化、规则化，并尽快转入翻盘期待。

## 核心循环

```text
生存压迫 -> 架空隔离 -> 付出代价 -> 触发规则 -> 翻盘信号
```

## 开篇必备

- 低谷要短，不能只卖惨。
- 敏感行为不写现实流程。
- 用异世界、血族、不死等规则隔离现实。
- 500-1000 字内给出翻盘信号。

## 期待来源

读者接受短暂压抑，是因为相信主角很快会把代价转化成能力、身份或规则优势。

## 断档风险

- 写成现实可模仿教程。
- 压抑时间过长，爽感迟迟不来。
- 主角只受苦，没有主动选择。
- 架空规则不明确，审核和读者体验都不稳。

## 训练任务

把“主角卖血求生”改成架空桥段：只写规则、代价和翻盘信号，不写现实流程。
