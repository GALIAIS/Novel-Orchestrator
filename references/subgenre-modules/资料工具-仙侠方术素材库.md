# 资料工具-仙侠方术素材库

依据案例：`batch-169` 案例 841。

## 读者承诺

把方术、术数、仪式、禁忌、器物类资料转成仙侠作品中的术法质感、门派知识壁垒和冲突弹药。

## 核心循环

```text
资料条目 -> 术法效果 -> 施术条件 -> 禁忌代价 -> 剧情冲突 -> 主角破解/升级
```

## 开篇必备

- 不要开篇堆资料名词，先让一个术法后果发生。
- 每个术名至少绑定一个条件和一个代价。
- 资料中的古奥词汇要通过人物行动解释。
- 术法最好牵出门派、传承、敌我信息差或禁忌反噬。

## 期待来源

读者期待仙侠设定有出处感，同时期待这些出处感进入可感知的斗法、解谜、修炼和反噬场面。

## 断档风险

- 只收藏资料，不转化剧情。
- 名词很多，但效果、条件、代价都不清楚。
- 资料设定与主线敌人无关。
- 把古奥感误当成爽点，导致读者看不懂。

## 训练任务

选 5 个方术/术数条目，各写一张剧情卡：术名、效果、条件、禁忌、谁掌握、第一次出场场面、三章内的反噬或破解。
