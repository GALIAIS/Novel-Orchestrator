# 架空都市-灵魂轮回与现实规避

依据案例：`batch-218` 案例 1088。

## 读者承诺

用近似现实的架空都市承载灵魂轮回等超自然设定，同时避开现实机构和地点绑定。

## 核心循环

```text
现实都市代入 -> 架空声明 -> 超自然规则 -> 日常碰撞 -> 都市奇谈
```

## 开篇必备

- 明确这是架空都市。
- 不直接使用现实机构、国家、地点、人名。
- 写一句与主角记忆都市不同的声明。
- 第一章让超自然规则介入现实生活。

## 期待来源

读者既能获得现代生活代入，又能接受灵魂轮回、摆渡等差异规则。

## 断档风险

- 现实绑定过强导致设定别扭。
- 架空只说不做，机构地点仍照搬。
- 超自然规则没有代价。
- 为规避而换外国环境，反而削弱代入。

## 训练任务

设计一个架空都市灵魂轮回设定，写城市名、三条差异规则、五个规避名词和第一起灵魂事件。
