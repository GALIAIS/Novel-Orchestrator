# 轻松玄幻-微搞笑梗与主线克制

依据案例：batch-379 案例 1894

## 读者承诺

轻松玄幻可以有梗和吐槽，但升级、资源、胜负和压力不能丢。梗是调味，要服务人物关系、局势反差和爽点兑现。

## 核心循环

```text
玄幻压力 -> 轻松反应 -> 梗服务反差 -> 主线推进 -> 爽点兑现
```

## 开篇必备

- 明确的玄幻压力。
- 主角当前目标。
- 可承载梗的人物关系。
- 不被梗打断的胜负或奖励。

## 期待来源

- 读者看到压力但不压抑。
- 人物互动更讨喜。
- 旁观者误判制造轻松反差。
- 胜负和升级仍然兑现。

## 断档风险

- 梗冲掉主线目标。
- 关键危机强行破气氛。
- 只学口吻不学结构。
- 轻松写成无压力。

## 训练任务

写一个玄幻测验或比斗场景，保留胜负和资源奖励，只在人物反应、旁观者误判和赛后吐槽中加入三个轻松梗。
