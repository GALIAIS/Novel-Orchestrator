# 单场情节-正戏反写与情绪折线

依据案例：batch-360 案例 1799

## 读者承诺

单场情节要有核心对抗、转折变化、承接线索和人设一致。正戏反写用反向行为制造误判，再用转折回到真实诉求。

## 核心循环

```text
本场诉求 -> 反向行为 -> 读者误判 -> 转折揭示 -> 情绪折线完成
```

## 开篇必备

- 目标、困境、对手至少有一个。
- 有承上启下线索。
- 行为贴合人设。
- 转折能回到本场诉求。

## 期待来源

- 反向表现制造疑问。
- 转折带来预期差。
- 真实人设或关系被揭示。
- 下一场被线索带出。

## 断档风险

- 反写变成绕远。
- 转折不服务诉求。
- 人物行为跳出人设。
- 场景只有信息，没有变化。

## 训练任务

写三个正戏反写场景：反派坏、关系差、队友不可靠。每个先给反向表现，再用转折揭示真实诉求。
