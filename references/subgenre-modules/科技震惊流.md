# 科技震惊流

依据精拆：`case-library/精拆/batch-001.md` 案例 004。

## 上级题材

都市现实职业、科幻、两界穿梭、学霸文、文娱/产业文。

## 读者承诺

主角掌握超出现代认知的科技或知识，在现代专业场景中击穿权威判断，获得震惊、资源、身份和产业机会。

## 核心循环

```text
现实专业问题 -> 现代权威无法解决/误判 -> 主角拿出跨界科技 -> 专家学霸震惊 -> 资源反馈 -> 更高阶项目
```

## 开篇必备

- 一个能被普通读者理解的专业问题。
- 一个足够权威的见证人。
- 主角科技优势的最小展示。
- 震惊后的现实收益：论文、专利、项目、投资、职称、合作。

## 期待来源

- 专家越懂，震惊越有效。
- 普通人先不懂，专业人再解释价值。
- 技术能解决现实卡点。
- 技术背后暗示更大来源或更高文明。

## 断档风险

- 科技解释太硬，失去爽文节奏。
- 只有震惊，没有资源反馈。
- 技术没有进入事件，只是设定展示。
- 过度追求严谨，反而筛掉目标读者。

## 操作卡

```text
现实问题：
权威误判：
主角技术：
普通人反应：
专家反应：
资源反馈：
下一项目：
```

## 训练任务

写一个 1500 字“专家震惊”场景。要求普通人看热闹、专家看门道、主角拿到一个现实资源。
