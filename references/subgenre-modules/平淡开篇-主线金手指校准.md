# 平淡开篇-主线金手指校准

依据案例：`batch-032` 案例 159。

## 读者承诺

平淡开篇要从主线、动机、金手指、强矛盾四处改，不是只润句子。

## 核心循环

```text
老题材 -> 新切入点 -> 主角目标 -> 明确金手指 -> 强矛盾 -> 跟读理由
```

## 开篇必备

- 主线一句话。
- 金手指一句话。
- 第一章强矛盾。
- 主角主动目标。

## 期待来源

- 老题材出现新入口。
- 金手指能立刻产生收益。
- 危机迫使主角行动。
- 读者知道后续看什么。

## 断档风险

- 流水账。
- 手机等现代物件破坏修仙基调。
- 主角被剧情带着走。
- 小妖怪小打小闹，矛盾太淡。

## 训练任务

重写一个平淡仙侠开篇小纲，必须先写主线、金手指、主角目标和第一冲突。
