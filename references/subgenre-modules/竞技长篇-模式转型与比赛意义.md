# 竞技长篇-模式转型与比赛意义

依据案例：`batch-339` 案例 1692

## 读者承诺

竞技长篇从日常装逼转向职业联赛或群像时，要提前铺新看点，并保证每场比赛有剧情意义。

## 核心循环

```text
前期轻快承诺 -> 转入职业模式 -> 群像目标接棒 -> 每场比赛改变局势 -> 新阶段承诺稳定
```

## 开篇必备

- 明确前半核心承诺。
- 转型前预告新模式。
- 群像角色有独立目标。
- 比赛不能重复同一赢法。
- 赢输必须改变主线。

## 期待来源

读者期待比赛不只是打斗过程，而是承诺、复仇、成长、荣誉或资源争夺的兑现。

## 断档风险

- 中途像换一本书。
- 主角戏份骤降未铺垫。
- 密集比赛无意义。
- 赢法重复。
- 原来只看主角装逼的读者流失。

## 训练任务

设计 5 场比赛，每场写清为什么必须打、赢输改变什么、和上一场赢法有什么不同。
