# 大纲执行-激情写作控偏

依据案例：batch-265 案例 1324

## 读者承诺

让连载作者在现场灵感和既定大纲之间保持可控：允许正文期产生新剧情，但每个新剧情都要回到主线、人物动机和后续成本上检查。

## 核心循环

```text
大纲目标 -> 正文激情 -> 新剧情冒出 -> 主线检查 -> 保留/改写/删除 -> 回写大纲
```

## 开篇必备

- 每章开写前只看三个点：本章目标、情绪兑现、结尾钩子。
- 现场新增内容先不急着删，写完后统一筛。
- 保留的新增内容必须服务主线或制造有效成本。
- 大纲允许更新，但不能被一时自嗨拖走。

## 期待来源

期待来自连载稳定性。读者不在乎作者是否严格照大纲写，只在乎剧情是否持续推进、承诺是否兑现、人物是否没有突然跑偏。

## 断档风险

- 完全按大纲写，正文干硬无趣。
- 完全不管大纲，主线断裂。
- 新灵感只爽一段，后续无法回收。
- 写完不复盘，偏航会在几章后爆雷。

## 训练任务

建立“章后偏航表”：每章写完列出新增剧情，标记保留、改写、删除，并写清楚它服务哪条主线或哪次情绪兑现。
