# 意识异空间-肉体安全处理

依据案例：`batch-168` 案例 836。

## 读者承诺

处理主角意识进入异空间后现实肉体的安全问题，决定它只是设定补丁，还是成为双线压力。

## 核心循环

```text
意识离体 -> 肉体滞留 -> 安全弱化/风险剧情 -> 两边联动 -> 主角付代价
```

## 开篇必备

- 明确意识停留时长。
- 说明现实肉体状态。
- 如果不想写风险，用时间流速或冬眠弱化。
- 如果想写风险，让现实危险和异空间目标有关。

## 期待来源

读者期待设定自洽；如果肉体有危险，则期待主角两边兼顾，形成更强悬念。

## 断档风险

- 肉体问题不处理，读者出戏。
- 只补设定，不服务剧情。
- 现实危险和异空间目标无关。
- 学《黑客帝国》《盗梦空间》只学壳，不学双线压力。

## 训练任务

写同一设定的两个版本：一个用时间流速差快速解决肉体安全，一个让肉体危险反过来影响异空间决策。
