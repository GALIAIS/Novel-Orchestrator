# 资料工具-怪异妖怪素材库

依据案例：`batch-160` 案例 796。

## 读者承诺

把妖怪、怪异、民俗资料工具转化为可写素材库，服务怪异文、玄幻仙侠、民俗悬疑和规则怪谈。

## 核心循环

```text
找到资料入口 -> 按出处检索 -> 提取能力/禁忌/代价 -> 转成异常事件 -> 放入题材素材库
```

## 开篇必备

- 资料不能只收藏，要转成冲突。
- 每个妖怪至少提炼外形、能力、禁忌、代价。
- 开篇先写异常后果，再揭示来源。
- 素材来源要记录链接和访问状态。

## 期待来源

读者期待民俗或怪异设定有出处感，同时能进入具体危险、谜团和人物选择。

## 断档风险

- 只堆百科，不形成剧情。
- 妖怪设定与主线无关。
- 资料来源失效后无法追溯。
- 把素材新奇感误当成故事推进。

## 训练任务

选 3 个妖怪，各写一张素材卡，并为每张卡设计一个开篇异常、一个禁忌代价、一个可持续三章的追查目标。
