# 创作动机组合-收入表达认同与最低回报线

依据案例：`batch-406` 案例 2028

## 读者承诺

允许收入、表达、成长、认同和完本同时存在，但只选一项主目标和少量次目标；为每项定义最低回报，并让项目范围服从时间、健康与合同边界。

## 核心循环

```text
列多种创作动机 -> 选一主两次 -> 定义最低回报 -> 加入现实约束 -> 设计兼容的作品范围 -> 阶段核对实际回报 -> 调整权重缩短或退出 -> 保留仍有价值的创作通道
```

## 开篇必备

- 主动机与次动机明确。
- 收入、成长、表达和认同各有验收。
- 不使用统一金额、时薪和字数标准。
- 时间、健康、学业和家庭边界清楚。
- 外部反馈与内部满足互为备份。
- 动机变化在检查点显式更新。

## 期待来源

作者无需在赚钱和情怀之间自证纯度；每个项目知道为什么值得投入、最低需要得到什么，也能在回报变化时调整而不否定初心。

## 断档风险

- 把赚钱与表达设成互斥。
- 所有目标都要求越多越好。
- 总字数替代全部回报。
- 只靠数据波动维持行动。
- 只求自我满足却期待大规模传播。
- 动机改变后仍被旧承诺绑住。

## 训练任务

给七类动机排序并只保留一主两次，为三项分别写最低回报；比较长篇商业、短卷训练和私人表达三种范围，选择与现实约束兼容的一种。
