# 画面感-视角动作细节落点

依据案例：`batch-335` 案例 1671

## 读者承诺

画面感要让读者通过固定视角看见关键变化，并理解这个变化对情绪、信息或下一步剧情的作用。

## 核心循环

```text
固定视角 -> 选关键动作/物件 -> 暗示信息或情绪 -> 人物做判断 -> 推动下一步剧情
```

## 开篇必备

- 当前视角明确。
- 画面细节能被看见。
- 细节和人物判断有关。
- 感官描写只保留最有用的一处。
- 每段画面后都有动作或信息变化。

## 期待来源

读者期待自己像在现场一样看见危险、尴尬、压迫或转机，并能顺着画面理解人物为什么这样行动。

## 断档风险

- 堆形容词。
- 五感乱塞。
- 景物和剧情无关。
- 画面很满但没有因果。
- 视角频繁乱跳。

## 训练任务

把一个抽象情绪句改成 300 字场景段，只用一个视角、三个动作和一个感官细节，最后一句必须推动剧情。
