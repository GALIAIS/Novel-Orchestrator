# 神豪开篇-不公平伤害与快速打脸

依据案例：batch-351 案例 1754

## 读者承诺

神豪文开局用低门槛不公平伤害拉共鸣，再用金手指和财富快速反转打脸。

## 核心循环

```text
被穷羞辱 -> 不公平伤害加压 -> 金手指出现 -> 财富可视化 -> 对方震惊后悔
```

## 开篇必备

- 前 300 字给伤害和低谷。
- 前 500 字出金手指。
- 第一笔钱要直观。
- 打脸对象要明确。
- 避免复杂金融和敏感现实抢戏。

## 期待来源

读者先憋屈，再立刻看到钱权反转。

## 断档风险

- 开头先讲宏观金融，理解门槛高。
- 金手指太晚。
- 伤害不够不公平。
- 打脸不可视化。
- 涉及时政暗喻导致风险。

## 训练任务

写 800 字神豪开头：300 字伤害，500 字金手指，800 字完成一次花钱打脸。
