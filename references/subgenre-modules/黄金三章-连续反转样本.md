# 黄金三章-连续反转样本

依据案例：`batch-143` 案例 714。

## 读者承诺

用强情绪开场和连续反转建立黄金三章式追看感：观众刚形成预期，剧情就偏移，再建立新预期，再次反转。

## 核心循环

```text
强钩子 -> 建立预期 -> 快速反转 -> 新预期 -> 再反转 -> 追看欲增强
```

## 开篇必备

- 强情绪入口。
- 第一预期。
- 第一反转。
- 第二预期。
- 第二反转。
- 反转后的新钩子。

## 期待来源

- 读者不断猜错。
- 每次反转都有情绪增量。
- 节奏快但因果清楚。
- 浮夸样本可转成小说桥段。

## 断档风险

- 只快不反转。
- 反转无因果，像胡闹。
- 照搬短视频浮夸表演。
- 推荐样本不拆反转点。

## 训练任务

找一个短视频、广告或小说片段，记录至少 3 次“以为会 A，结果变 B”的反转点。写出预期、反转动作、情绪，再改成 100 字推荐动态。
