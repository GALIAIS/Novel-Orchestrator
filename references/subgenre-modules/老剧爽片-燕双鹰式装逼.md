# 老剧爽片-燕双鹰式装逼

依据案例：batch-311 案例 1552

## 读者承诺

老爽剧可作为网文结构样本：不公平待遇、队友怀疑、敌人轻视、主角装逼证明、坏人受罚。

## 核心循环

```text
主角受压 -> 队友劝怂 -> 敌人轻视 -> 主角出手 -> 装逼证明 -> 惩罚坏人
```

## 开篇必备

- 先给主角一个不公平待遇。
- 安排队友或群众不信。
- 敌人要轻视主角。
- 主角出手要既爽又符合底层逻辑。

## 期待来源

期待来自低估后的证明。读者越清楚配角误判，就越期待主角出手。

## 断档风险

- 只模仿夸张表现，不保留逻辑。
- 主角装逼没有前置压力。
- 队友怀疑太蠢，变成降智。
- 坏人惩罚不够明确。

## 训练任务

拆一段老剧装逼桥段，改成网文章纲：不公平待遇、低估、出手、证明、惩罚五步。
