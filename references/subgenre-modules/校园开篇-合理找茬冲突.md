# 校园开篇-合理找茬冲突

依据案例：`batch-023` 案例 112。

## 读者承诺

让校园开篇的第一冲突早出现、能看懂、能升级，并让读者相信主角不能简单走开。

## 核心循环

```text
校园关系 -> 利益/面子触发 -> 找茬合理化 -> 主角被迫应对 -> 关键角色见证 -> 反击期待
```

## 开篇必备

- 找茬者的利益或面子。
- 主角无法无损离开的理由。
- 女主、男二或社团元素进入同一冲突。
- 对话符合人物身份。

## 期待来源

- 读者知道冲突为什么现在爆发。
- 配角不是纯工具人。
- 主角第一选择暴露性格。
- 关键人物见证主角表现。

## 断档风险

- 为冲突而冲突。
- 找茬理由太小，主角可直接离开。
- 后援团、女神、富二代都是古早符号，没有新压力。
- 人物说话不像正常人。

## 训练任务

把一个“路人突然找茬”的校园桥段改成利益冲突：写清对方损失了什么、主角为什么被卷入、主角为什么不能走。
