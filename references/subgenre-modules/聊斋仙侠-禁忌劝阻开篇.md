# 聊斋仙侠-禁忌劝阻开篇

依据案例：batch-285 案例 1423

## 读者承诺

聊斋仙侠开篇可以用“别人劝主角不要去”制造期待，再用禁忌、异象和保命法术兑现未知危险。

## 核心循环

```text
禁忌劝阻 -> 主角必须去 -> 异象出现 -> 保命术应对 -> 获得规则信息
```

## 开篇必备

- 禁忌地点或事件。
- 劝阻者。
- 主角必须前往的理由。
- 第一处异象。
- 一个有用但有限制的保命术。

## 期待来源

读者想知道那里为什么不能去，以及主角进入后会遇见什么怪异规则。

## 断档风险

- 只靠劝阻不兑现危险。
- 长生目标太远，当前目标不具体。
- 保命技能只列设定，不进事件。
- 氛围变尬，缺少真实威胁。

## 训练任务

写 1000 字禁忌开篇：300 字内有人劝阻，600 字内主角进入，1000 字内保命术解决小危机。
