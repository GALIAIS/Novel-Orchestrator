# 金手指-低风险高回报

依据案例：`batch-006` 案例 028。

## 读者承诺

主角用较低成本和风险获得超出常人的回报，让读者获得安全感、收获感和持续期待。

## 核心循环

```text
低成本动作 -> 低风险执行 -> 高回报收益 -> 安全感建立 -> 更大收益路径
```

## 开篇必备

- 金手指使用成本简单明了。
- 第一次使用不能显得无脑作死。
- 回报必须高于读者预期。
- 如果开局高风险，必须尽快补回安全感。

## 期待来源

- 读者想看主角如何低成本滚雪球。
- 读者想看弱任务如何量变到质变。
- 读者相信主角有安全垫后，愿意看更大冒险。

## 断档风险

- 作者视角知道不会死，读者视角却觉得主角作死。
- 为使用金手指强行挑战强敌。
- 高回报低于读者预期。
- 读者阅读成本过高，爽点太迟。

## 训练任务

把一个高风险金手指桥段改成“两个低风险积累 + 一次强敌兑现”的结构。
