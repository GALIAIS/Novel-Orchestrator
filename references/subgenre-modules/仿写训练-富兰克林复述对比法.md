# 仿写训练-富兰克林复述对比法

依据案例：`batch-203` 案例 1015。

## 读者承诺

理论看不懂时，用样本书切块、记录、复述、对比，学习结构和取舍，但不能抄袭文本。

## 核心循环

```text
选样本 -> 切 500 字板块 -> 记录重点 -> 脱稿复述 -> 对比原文 -> 修正写法
```

## 开篇必备

- 选 2-3 本同方向样本。
- 先拆前 20 章。
- 每 500 字标注功能。
- 用自己的角色、能力、事件复述。

## 期待来源

作者通过对比原文和复述稿，看到自己在节奏、信息量、爽点和啰嗦度上的差距。

## 断档风险

- 抄原句，变成侵权。
- 只换名词，不换事件逻辑。
- 不做对比，看不出问题。
- 样本方向和自己要写的类型不一致。

## 训练任务

拆同类型书前 3 章，每 500 字写一句功能说明；合上原文后复述，再对比节奏、信息量和爽点差距。
