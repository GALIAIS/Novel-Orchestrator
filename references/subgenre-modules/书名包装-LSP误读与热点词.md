# 书名包装-LSP误读与热点词

依据案例：`batch-200` 案例 999。

## 读者承诺

书名要在快节奏扫榜中完成第一眼卖点传递，同时检查谐音、断句和擦边误读是否会造成预期错配。

## 核心循环

```text
关键词组合 -> 第一眼识别 -> 欲望/机制预期 -> 简介验证 -> 前三章兑现
```

## 开篇必备

- 书名至少有一个明确类型词。
- 最好叠加机制词和角色欲望词。
- 简介要验证标题承诺。
- 前三章要兑现最强卖点。

## 期待来源

读者通过标题快速捕捉“我能在这本书里看到什么”，再用简介确认是否值得点进正文。

## 断档风险

- 标题制造的擦边预期正文不兑现。
- 谐音误读带来错误读者。
- 热点词堆砌但核心卖点不清。
- 书名和简介互相脱节。

## 训练任务

给同一设定写 10 个书名，每个包含角色欲望词、机制词和类型词，并标出可能误读。
