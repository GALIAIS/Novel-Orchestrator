# 核心立意-逻辑链藏进故事

依据案例：`batch-032` 案例 158。

## 读者承诺

主题要先经过作者消化，再藏进人物选择和剧情后果里，不要用旁白说教。

## 核心循环

```text
作者思考 -> 逻辑链验证 -> 人物成长 -> 情节承载 -> 读者认可
```

## 开篇必备

- 一句明确立意。
- 三条支撑逻辑。
- 对应主角成长节点。
- 删除说教表达。

## 期待来源

- 读者先被故事吸引。
- 人物选择体现主题。
- 成长带来情绪认可。
- 作品比纯爽文更有余味。

## 断档风险

- 主题幼稚，经不起责问。
- 作者不自觉输出油腻三观。
- 角色对话硬讲道理。
- 新人硬塞立意拖慢爽点。

## 训练任务

为一条人物成长线写立意逻辑链，再把所有旁白结论改成角色行动和代价。
