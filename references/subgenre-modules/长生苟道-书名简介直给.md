# 长生苟道-书名简介直给

依据案例：`batch-064` 案例 319。

## 读者承诺

这个模块用于长生、苟道、熬死对手、属性加点、老六无敌流。它承诺的是书名和简介直接告诉读者主角靠寿命、时间和苟道策略碾压世界，让第一章用高信息密度和自然快节奏兑现核心梗。

## 核心循环

```text
长生优势 -> 时间收益 -> 苟道选择 -> 熬死对手 -> 属性/资源增长 -> 更大世界见证
```

## 开篇必备

- 书名直接暴露核心爽点。
- 简介交代寿命优势和成长机制。
- 主角行为风格要鲜明：苟、老六、稳、记仇。
- 第一章信息密度高但条理清楚。
- 前三章要保持稳定，不能只有简介强。

## 期待来源

- 读者期待主角靠时间碾压。
- 短寿世界和永生主角形成反差。
- 每年或每段时间都有收益。
- 仇敌老去，主角仍在成长。
- 老六行为制造轻松爽感。

## 断档风险

- 书名有梗，简介不交代机制。
- 只苟不行动，剧情不动。
- 第一章快但乱。
- 后两章明显掉质量。
- 长生优势没有阶段性收益。

## 训练任务

写一个长生苟道书名、150 字简介和第一章前 5 个信息点。信息点必须依次交代主角处境、寿命优势、成长机制、行为风格和第一个目标。
