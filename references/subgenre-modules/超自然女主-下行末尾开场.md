# 超自然女主-下行末尾开场

依据案例：batch-262 案例 1310

## 读者承诺

超自然女主可以遭遇压迫，但开头应从下行末尾切入，尽快复活、觉醒或反击，让读者看到强设定兑现。

## 核心循环

```text
困境低点 -> 特殊身份启动 -> 第一次反击 -> 情绪上行 -> 建立翻盘期待
```

## 开篇必备

- 女主的特殊身份。
- 当前被压迫的低点。
- 复活/觉醒触发。
- 第一次具体胜利。
- 后续危机的克制规则。

## 期待来源

读者期待强设定主角不是一直挨打，而是在被压到底后快速显示特殊性，并赢回东西。

## 断档风险

下行时间过长会疲惫；连续下行没有翻盘，会让情绪发力落空；女主强却不兑现，会削弱代入。

## 训练任务

写一章下行末尾开场：前三段困境，第 4 段觉醒，章尾赢回一个具体资源或身份优势。
