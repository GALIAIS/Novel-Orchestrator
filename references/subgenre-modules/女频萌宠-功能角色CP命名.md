# 女频萌宠-功能角色CP命名

依据案例：`batch-103` 案例 515。

## 读者承诺

女频萌宠不是装饰，而是能承担儿女、导师、伙伴、追随者和氛围烘托的功能角色。CP 名字则是关系传播标签，让读者更容易记住、讨论和嗑。

## 核心循环

```text
压抑主线 -> 萌宠出场 -> 陪伴/导师/氛围功能 -> 情绪缓冲 -> 关系标签化 -> 读者传播
```

## 开篇必备

- 萌宠与主角的明确关系。
- 至少一个事件功能。
- 一个情绪缓冲功能。
- 一个反差萌点。
- 可传播的 CP 或关系名字。

## 期待来源

- 陪伴感。
- 反差萌。
- 主线压抑后的轻松出口。
- 萌宠参与关键事件。
- CP 名字带来的饭圈式辨识度。

## 断档风险

- 萌宠只卖萌，不参与事件。
- 功能和主线无关。
- 压抑主线没有情绪缓冲。
- CP 名字生硬，不体现关系特征。

## 训练任务

给主角设计一个萌宠，写清它在三场戏中的功能：安抚、提示、推动事件。再给主 CP 起 3 个短名字，选最有辨识度的一个。
