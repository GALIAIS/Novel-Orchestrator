# 拉扯内核-戏剧性信息差对比

依据案例：`batch-015` 案例 075。

## 读者承诺

网文吸引力可以拆成戏剧性、情绪拉扯、信息差和对比。爽文重快速调动情绪，剧情文重铺垫地基，两者都要服务期待和爽点。

## 核心循环

```text
戏剧性 -> 信息差/对比 -> 情绪拉扯 -> 期待增强 -> 爽点释放
```

## 开篇必备

- 本场必须有戏剧性。
- 明确谁和谁形成对比。
- 明确谁知道、谁不知道。
- 情绪来源要能被读者立刻理解。

## 期待来源

- 戏剧性够，读者自然想看下去。
- 拉扯补足戏剧性不足。
- 信息差让读者等误判被揭穿。
- 对比让主角优势更清楚。

## 断档风险

- 快情绪没有铺垫地基，爽点悬浮。
- 剧情文铺垫太慢，情绪调动不足。
- 只有嘲讽打脸，没有戏剧性。
- 信息差和对比没有服务主角行动。

## 训练任务

拿一场装逼戏，分别写快情绪版和剧情铺垫版，标出两版的戏剧性、信息差、对比和地基。
