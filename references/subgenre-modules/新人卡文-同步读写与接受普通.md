# 新人卡文-同步读写与接受普通

依据案例：`batch-234` 案例 1167。

## 读者承诺

新人写不出理想感觉时，不要乱换题材和阅读对象；写网文就读网文，边学边写，接受可用版本。

## 核心循环

```text
写不出感觉 -> 路线变乱 -> 回到目标网文 -> 同步阅读写作 -> 持续改进
```

## 开篇必备

- 阅读对象对齐目标受众。
- 不用换背景逃避能力缺口。
- 每天读大神文。
- 每天写自己的正文。

## 期待来源

作者不追求一次写出脑海中的完美感觉，而是先写出可用稿，再通过同步训练提高。

## 断档风险

- 过度看教程不写正文。
- 读传统文学替代网文样本。
- 频繁换题材。
- 因不满意停笔。

## 训练任务

连续 7 天：每天读 3 章同平台大神文，写 100 字节奏笔记，再写 1000 字自己的正文；不允许换项目。
