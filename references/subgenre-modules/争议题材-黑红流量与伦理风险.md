# 争议题材-黑红流量与伦理风险

依据案例：`batch-216` 案例 1080。

## 读者承诺

识别高争议噱头带来的曝光、观众提纯和伦理反噬，不把现实伤害当廉价卖点。

## 核心循环

```text
争议噱头 -> 舆论吵架 -> 曝光增加 -> 观众提纯 -> 伦理反噬
```

## 开篇必备

- 明确争议来自哪里。
- 判断目标受众和反感来源。
- 识别未成年人、弱势处境等平台风险。
- 给人物尊严和主题处理留空间。

## 期待来源

创作者知道黑红可以带来点击，但也会筛掉读者、伤害口碑，并增加平台与伦理风险。

## 断档风险

- 把社会悲剧娱乐化。
- 用未成年人或弱势处境做猎奇。
- 只追求吵架流量。
- 没有主题补偿和人物尊严。

## 训练任务

把一个争议桥段写成猎奇版和伦理处理版，对比点击点、反感点、平台风险、人物尊严和长期口碑。
