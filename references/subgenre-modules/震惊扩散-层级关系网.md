# 震惊扩散-层级关系网

依据案例：`batch-003` 案例 013。

## 读者承诺

一个装逼点不只释放一次，而是沿着同层、中层、高层关系网扩散，让同一事件形成多轮反应、多轮质疑和新矛盾。

## 核心循环

```text
同层震惊 -> 同层传播 -> 中层质疑 -> 中层验证 -> 高层介入 -> 新冲突升级
```

## 开篇必备

- 主角所在层级。
- 同层见证者和质疑者。
- 中层权威或管理者。
- 高层权威、势力或更大环境。

## 期待来源

- 读者想看消息传出去后别人怎么反应。
- 读者想看更高层的人从不信到震惊。
- 读者想看主角一次表现如何撬动更大环境。

## 断档风险

- 所有人只会重复震惊。
- 没有层级差，反应无法升级。
- 只写反应，不让反应带来新矛盾。
- 人物关系太薄，读者不关心谁震惊。

## 训练任务

为一个主角胜利事件设计三层传播链：同层 2 人、中层 2 人、高层 1 人。每个人写不同反应和后续动作。
