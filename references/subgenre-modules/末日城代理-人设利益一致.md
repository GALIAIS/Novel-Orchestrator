# 末日城代理-人设利益一致

依据案例：batch-286 案例 1426

## 读者承诺

末日城代理系统的爽点在现实压力、异界经营、任务发布和抽成收益；主角底线可以有，但情绪和行为必须一致。

## 核心循环

```text
现实债务 -> 接管末日城 -> 绑定幸存者 -> 发布任务 -> 抽成收益 -> 扩大经营
```

## 开篇必备

- 现实债务或生存压力。
- 异界代理身份。
- 第一个幸存者。
- 第一笔收益。
- 主角明确但不过度的底线。

## 期待来源

读者想看主角如何用异世界劳动力、资源差价和系统权限解决现实困境。

## 断档风险

- 主角突然哭泣立宏愿。
- 同情幸存者后马上压榨，行为打架。
- 长线目标过早锁死。
- 催债反派行为失真。

## 训练任务

改写一个主角共情段：只用一个行动体现底线，同时保留经营、抽成、赚钱的爽点。
