# 历史穿越素材-技术装逼转剧情

依据案例：`batch-052` 案例 257。

## 读者承诺

把历史穿越技术素材转成具体剧情问题：粮食、盐、钢、玻璃、医药、军队都要服务民生、财富、权力或战争爽点。

## 核心循环

```text
古代痛点 -> 现代知识 -> 低成本试制 -> 群体收益 -> 装逼见证 -> 新利益冲突
```

## 开篇必备

- 先给出古代痛点，不先讲技术课。
- 技术材料和工艺要受时代限制。
- 第一次成功要小而可见。
- 成果必须改变某个利益格局。
- 技术后果要引出新冲突。

## 期待来源

- 现代知识在古代发挥作用，天然有装逼爽。
- 粮食、盐、钢、药这些基础物资会放大群体收益。
- 技术改变生产力，会带来权力资本。
- 新利益冲突让种田不变流水账。

## 断档风险

- 技术资料堆太多，会像科普视频搬运。
- 主角全知全能，会削弱真实感。
- 没有材料和组织限制，成功太轻飘。
- 技术只赚钱不引发冲突，后劲不足。

## 训练任务

选一个土法技术，写出古代痛点、试制过程、见证者反应、收益和后续争夺。
