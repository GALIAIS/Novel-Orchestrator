# 大纲模板-草稿纸波折化

依据案例：`batch-083` 案例 413。

## 读者承诺

这个模块用于新人选择和改造大纲模板。它承诺把模板当草稿纸，只保留能帮助继续写正文、制造波折的部分。

## 核心循环

```text
参考模板 -> 删除无用项 -> 总纲到章纲 -> 每句加波折 -> 写作推进
```

## 开篇必备

- 先写总纲一句话。
- 再写卷纲变化。
- 万字纲写阶段目标。
- 章纲只写几句也可以。
- 每句话之间必须有转折。

## 期待来源

- 模板降低起步焦虑。
- 个人化避免被格式卡死。
- 波折让章纲可写。
- 大纲让作者知道下一步。
- 满意的大纲会产生动笔欲。

## 断档风险

- 把模板当固定格式。
- 章纲写太死。
- 只填人物性格和剧情梗概。
- 没有波折。
- 模板越填越不想写。

## 训练任务

写一份极简大纲：总纲 1 句、卷纲 3 句、万字纲 5 句、章纲 3 句。每句之间必须有“但/于是/结果”。
