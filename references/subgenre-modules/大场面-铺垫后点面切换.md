# 大场面-铺垫后点面切换

依据案例：`batch-028` 案例 140。

## 读者承诺

大场面靠前置铺垫和结果改变撑起规模感，再用局部人物感受承载整体震撼。

## 核心循环

```text
利害铺垫 -> 各方入场 -> 局部切入 -> 整体扩展 -> 关键碰撞 -> 格局改变
```

## 开篇必备

- 大场面为什么发生。
- 各方为什么必须到场。
- 主视角或关键观察点。
- 胜负会改变什么。

## 期待来源

- 读者提前理解利害关系。
- 局部人物的恐惧、选择或牺牲。
- 点面切换带来的规模感。
- 场面结束后的局势重排。

## 断档风险

- 没铺垫就堆人数和招式。
- 视角散，读者不知道跟谁看。
- 只有热闹，没有胜负代价。
- 结束后世界没有变化。

## 训练任务

为一场围攻、战争或大会写细纲，先列铺垫、参与方、主视角、局部切入点、胜负代价和结果改变。
