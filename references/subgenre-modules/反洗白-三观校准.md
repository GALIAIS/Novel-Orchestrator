# 反洗白-三观校准

依据案例：`batch-198` 案例 986。

## 读者承诺

角色有悲惨经历不等于能洗掉伤害无辜的行为；三观校准要靠同等处境下的选择对照。

## 核心循环

```text
角色受挫 -> 选择伤害无辜/守住底线 -> 对照高下 -> 读者自然判断
```

## 开篇必备

- 明确角色遭遇的不公。
- 明确他是否伤害无辜。
- 设置同等处境对照角色。
- 不用作者强行洗白。

## 期待来源

读者看到角色选择后，自然理解谁值得认同、谁失去正当性。

## 断档风险

- 为流量强行反转洗白。
- 把屠杀包装成真性情。
- 忽略无辜受害者。
- 作者强行站队恶人。

## 训练任务

写两个同样失败的角色，一个守底线，一个伤害无辜。只用行动对照，不写价值观旁白。
