# 仿古文风-易懂产能一致

依据案例：`batch-338` 案例 1687

## 读者承诺

古风题材的文风要易懂、稳定、写得动；古意服务场景，不应拖慢冲突和产能。

## 核心循环

```text
题材古意 -> 保留必要称谓制度词 -> 绕句改白 -> 全文统一 -> 冲突顺畅推进
```

## 开篇必备

- 从第一章统一风格。
- 句子一眼能懂。
- 古意来自少量词汇和场景。
- 作者产能可持续。
- 不中途突然变文风。

## 期待来源

读者期待进入古代氛围，同时不用被仿古腔调拦住阅读。

## 断档风险

- 硬凹古文导致低产。
- 文风中途切换。
- 腔调抢过剧情。
- 读者看懂成本高。
- 为风格放弃更新节奏。

## 训练任务

把 1000 字仿古段落改成白文，保留称谓、官职、制度词，删除所有影响理解的绕句。
