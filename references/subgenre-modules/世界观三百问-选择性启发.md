# 世界观三百问-选择性启发

依据案例：`batch-154` 案例 770。

## 读者承诺

世界观资料是作者思考工具，不是正文设定清单；只选择能启发故事冲突和大纲盲区的问题使用。

## 核心循环

```text
资料问题池 -> 选择相关问题 -> 启发盲区 -> 服务故事冲突 -> 正文只露必要信息
```

## 开篇必备

- 当前故事缺口。
- 与冲突有关的问题。
- 只给作者知道的答案。
- 必须写给读者看的最小设定。
- 后续可延展的信息。

## 期待来源

期待来自世界观对人物行动、资源、权力、规则和冲突的支撑，而不是设定本身的复杂度。

## 断档风险

- 机械回答全部问题。
- 把问卷答案整段塞进正文。
- 资料复杂度超过故事需要。
- 设定替代剧情。

## 训练任务

从任意世界观问卷中只选 10 个和当前冲突有关的问题，标注“开篇必须写、后续再写、只给作者知道”。
