# 单期待主线-道歉认错情绪债

依据案例：`batch-231` 案例 1154。

## 读者承诺

单期待主线用一个极具体的情绪债拉动前段阅读，例如让亏欠者向主角道歉认错。

## 核心循环

```text
误解/羞辱 -> 读者站队 -> 加深亏欠 -> 等待认错 -> 情绪清算 -> 切新期待
```

## 开篇必备

- 明确谁亏欠主角。
- 亏欠事件足够清楚。
- 读者快速站主角。
- 认错动作具体可期待。

## 期待来源

读者想看误解主角的人知道真相、低头认错、偿还情绪债。

## 断档风险

- 期待对象模糊，不知道谁该认错。
- 兑现前分散到太多支线。
- 认错不够公开或不够有力。
- 兑现后没有新期待承接。

## 训练任务

设计前三章“道歉认错”单期待：写明亏欠事件、三次加压、认错场景、兑现后新期待。
