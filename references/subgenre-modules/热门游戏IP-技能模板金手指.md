# 热门游戏IP-技能模板金手指

依据案例：`batch-194` 案例 966。

## 读者承诺

热门游戏标签只负责吸引注意，真正留人的是玩家熟悉的技能机制、成长路线、克制关系和名场面被小说化兑现。

## 核心循环

```text
玩家认知 -> 技能模板绑定 -> 异界危机 -> 熟悉机制破局 -> 标签爽点兑现
```

## 开篇必备

- 明确主角获得的是单英雄或单体系模板。
- 第一场危机能用核心技能解决。
- 技能表现符合玩家基本认知。
- 原创世界规则能承接技能成长。

## 期待来源

读者期待熟悉游戏机制在新世界产生新用法，同时享受“我懂这个技能所以我知道主角能怎么秀”的参与感。

## 断档风险

- 只贴热门标签，不懂受众爽点。
- 技能细节写露怯。
- 大杂烩堆英雄导致体系崩。
- 把版权问题误当创作有效性问题。

## 训练任务

选一个热门游戏角色，写出 3 个小说化技能、1 个升级条件、1 个第一章破局场景，避免直接照搬名称和设定。
