# 都市日常系统-目标女角前置

依据案例：`batch-043` 案例 212。

## 读者承诺

都市日常、神豪、暧昧、后宫类开篇要尽快让目标受众看到女角色、任务和小爽点。生活惨状和系统说明只能为关系期待服务。

## 核心循环

```text
标签承诺 -> 系统任务 -> 目标女角 -> 日常不爽 -> 暧昧/装逼兑现
```

## 开篇必备

- 前期出现系统任务。
- 第一位女角色有看点。
- 男主核心人设不毒。
- 日常不爽事件化。

## 期待来源

- 读者想看任务如何撩动关系。
- 神豪资源改变日常处境。
- 女配反应放大爽点。
- 暧昧和修罗场有后续预告。

## 断档风险

- 2500 字没有目标女角色。
- 系统一次介绍太多。
- 心理活动和惨状拖慢。
- 不公平伤害只靠背景口水话。

## 训练任务

压缩一个都市系统开篇：前 500 字给系统任务，前 1000 字给目标女角，第一章末完成一次小爽点。
