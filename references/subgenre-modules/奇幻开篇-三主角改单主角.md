# 奇幻开篇-三主角改单主角

依据案例：`batch-327` 案例 1632

## 读者承诺

新人奇幻开篇不要三主角；先用单主角完成第一段情节、冲突和断章，再逐步引入配角。

## 核心循环

```text
单主角入场 -> 异常事件 -> 情绪波动 -> 冲突引爆 -> 断章钩子
```

## 开篇必备

- 明确唯一主角。
- 延后两个配角或降为功能配角。
- 删除无剧情铺垫。
- 第一章必须不止悬念，还要有冲突。
- 人设在事件中体现。

## 期待来源

读者期待迅速知道跟谁走、看什么冲突、下一章为什么要点。

## 断档风险

- 三主角让代入分散。
- 实体味铺垫太长。
- 金手指没亮点。
- 笔力好但节奏慢。
- 6000 字只有后 2000 字有剧情。

## 训练任务

把三主角开篇改成单主角，压缩前三分之二铺垫，把真正冲突并到第一章末。
