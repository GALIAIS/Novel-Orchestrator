# 敏感红线-现实政治儿童伦理避坑

依据案例：`batch-115` 案例 571。

## 读者承诺

敏感元素不该让作品承担无谓发布风险。现实政治、真实国名地名、未成年性化和血缘伦理，如果不是核心卖点，优先规避或架空化。

## 核心循环

```text
识别敏感元素 -> 判断是否核心必需 -> 问编辑/查平台口径 -> 架空转译 -> 保留剧情功能 -> 降低发布风险
```

## 开篇必备

- 不碰真实政治热点。
- 不用真实国家冲突当随手背景。
- 不写儿童色情和低俗未成年描写。
- 血缘伦理关系先避险。
- 具体灰区问对应编辑。

## 期待来源

- 作者不被规则焦虑拖住。
- 故事功能保留，敏感外壳替换。
- 编辑口径让风险变具体。
- 架空表达保留冲突张力。

## 断档风险

- 用现实热点蹭情绪。
- 把敏感元素当世界观真实感。
- 不问编辑就赌平台容忍。
- 规则漂移后没有替代表达。

## 训练任务

列出作品中 10 个高风险名词或关系，逐个改成架空表达。改完检查原剧情功能是否仍成立。
