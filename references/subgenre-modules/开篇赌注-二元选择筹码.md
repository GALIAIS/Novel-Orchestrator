# 开篇赌注-二元选择筹码

依据案例：`batch-154` 案例 767。

## 读者承诺

开篇赌注不是抽象设定，而是主角被迫选择时押上的筹码；选错失去，选对获得收益。

## 核心循环

```text
主角陷入处境 -> 不得不做选择 -> 选错损失筹码 -> 选对获得收益 -> 读者期待兑现
```

## 开篇必备

- 主角必须面对的二元选择。
- 选错会失去的具体筹码。
- 选对会获得的具体收益。
- 赌注的紧迫性或重要性。
- 高层信息只留伏笔。

## 期待来源

读者期待看到主角在高压选择中押上筹码，并用能力、智慧或金手指赢回收益。

## 断档风险

- 只有危险，没有主角选择。
- 只有收益口号，没有具体筹码。
- 高层博弈开篇铺太满。
- 敏感元素叠加造成影射风险。

## 训练任务

把一个开篇事件改成选择题：列出两个选项、选错损失、选对收益、赌注紧迫性，再把高层博弈压缩成 3 个伏笔。
