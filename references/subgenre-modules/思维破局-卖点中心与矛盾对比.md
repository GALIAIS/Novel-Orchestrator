# 思维破局-卖点中心与矛盾对比

依据案例：`batch-247` 案例 1233。

## 读者承诺

打破中庸先围绕卖点写，再用矛盾和对比制造亮点。

## 核心循环

```text
明确卖点 -> 剧情围绕卖点 -> 扫榜记录亮点 -> 加矛盾对比 -> 反复改写
```

## 开篇必备

- 开篇明确卖点。
- 类型爽点不偏离。
- 每段服务卖点。
- 用对比放大反差。

## 期待来源

读者因为卖点进来，就要持续看到卖点带来的剧情和爽点。

## 断档风险

- 东一笔西一笔。
- 后期偏离初始卖点。
- 只想改变思维，不改具体段落。
- 没有矛盾和对比。

## 训练任务

给一章大纲标唯一卖点，每段写出如何服务卖点；标不出来就用矛盾或对比重写。
