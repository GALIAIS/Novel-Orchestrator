# 灭世预言-权威追杀循环

依据案例：`batch-067` 案例 333。

## 读者承诺

这个模块用于长线悬疑、幻想、异能、反英雄或世界误判型故事。它承诺的是让主角背上世界级预言罪名，在权威压迫、追杀逃亡和命运改写中持续拉期待。

## 核心循环

```text
证明预言权威 -> 预言主角灭世 -> 世界震惊围剿 -> 主角逃亡破局 -> 真相或能力升级 -> 新预言再压迫
```

## 开篇必备

- 先证明预言者为什么可信。
- 预言影响普通人或公共秩序。
- 主角被预言命中后立刻产生外部压力。
- 追杀方有正当性，不只是无脑反派。
- 主角每轮破局后，问题必须升级。

## 期待来源

- 预言权威带来的可信恐惧。
- 主角被误判成灭世者的身份炸弹。
- 世界围剿造成高压逃亡。
- 主角改写命运的反抗感。
- 每次预言内容不同，形成新谜团。

## 断档风险

- 预言者只靠作者口头吹。
- 每次循环都只是“又有人追杀”。
- 主角只逃不查真相。
- 预言内容没有升级变量。
- 宏大问题讨论过深，拖慢网文节奏。

## 训练任务

设计 3 轮预言循环，每轮写：权威证明、预言内容、追杀方、主角破局、下一轮升级点。禁止三轮使用同一种追杀理由。
