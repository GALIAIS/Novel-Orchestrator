# 超英开头-主动动机补强

依据案例：`batch-168` 案例 838。

## 读者承诺

超英/怪人开头不仅要尽快给危机和高光，还要让主角拥有非去现场不可的主动动机。

## 核心循环

```text
世界背景 -> 主角迫切欲望 -> 非去不可 -> 危机爆发 -> 主动救人 -> 金手指兑现
```

## 开篇必备

- 100 字内交代世界压力。
- 让主角有短期欲望。
- 给主角去现场的必要理由。
- 危机和金手指兑现要服务这个目标。

## 期待来源

读者期待主角主动踏入事件，靠选择和能力获得结果，而不是被作者安排到展示台上表演。

## 断档风险

- 只提前危机，不补主角动机。
- “顺便测试系统”导致主动性弱。
- 人物过多，主角模糊。
- 冲突突兀，缺少伏笔和过渡。

## 训练任务

写一份 1800 字超英开头小纲：主角必须有迫切欲望、任务奖励、主动选择、金手指兑现和见证者震惊。
