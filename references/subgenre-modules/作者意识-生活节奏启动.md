# 作者意识-生活节奏启动

依据案例：`batch-155` 案例 772。

## 读者承诺

作者意识不只在脑子里，也落在每天能否启动、能否抵抗吞时间的诱惑、能否维持精力。

## 核心循环

```text
低门槛启动 -> 拖过娱乐冲动 -> 管理身体精力 -> 有意识输入 -> 持续写作学习
```

## 开篇必备

- 明确当天目标。
- 一个低门槛启动动作。
- 记录娱乐冲动。
- 替代动作。
- 输入素材的卖点/爽点拆解。

## 期待来源

期待来自每天可执行的稳定节奏，而不是一次性鸡血。

## 断档风险

- 只喊自律，没有启动动作。
- 把娱乐全当敌人，长期失去输入。
- 冲动来时没有替代动作。
- 忽视睡眠、饮食、运动对产出的影响。

## 训练任务

连续 7 天早起先写日记，并记录一次娱乐冲动的处理方式；每天拆一个娱乐素材的卖点、节奏和欲望满足。
