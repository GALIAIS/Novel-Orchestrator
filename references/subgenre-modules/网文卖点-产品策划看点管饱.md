# 网文卖点-产品策划看点管饱

依据案例：batch-270 案例 1349

## 读者承诺

用产品策划思维写网文：一句话卖点吸引读者，持续看点管饱，每章内容围绕痛点痒点。

## 核心循环

```text
一句话卖点 -> 主角非常人 -> 持续强反馈 -> 主线分阶段兑现 -> 有效内容占比提高
```

## 开篇必备

- 先写出一句话卖点。
- 明确无法割舍的核心看点。
- 主角要快速做出常人做不到的举动。
- 每章围绕卖点安排有效内容。

## 期待来源

期待来自读者能清楚知道“这本书卖什么”，并且每章都持续收到对应的爽点反馈。

## 断档风险

- 只有剧情，没有卖点。
- 看点不够管饱。
- 叙事不分主次。
- 模仿大神只仿皮，不抓卖得好的原因。

## 训练任务

给作品写三条广告语式卖点，选一个最强的，再用它检查第一章每一段是否有效。
