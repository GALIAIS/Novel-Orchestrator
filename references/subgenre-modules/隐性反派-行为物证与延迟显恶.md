# 隐性反派-行为物证与延迟显恶

依据案例：`batch-398` 案例 1986

## 读者承诺

不用主角直接宣布谁可疑，而以具备双重解释的动作和物品逐级互证，让读者参与怀疑；显恶后再由旧证据支持主人公反制。

## 核心循环

```text
轻微行为偏差 -> 双解物证 -> 跨场景复现 -> 线索互证并造成损失 -> 真实目标显露 -> 旧证据支持反制与问责
```

## 开篇必备

- 嫌疑对象有可信的正常身份。
- 第一条线索可被无辜解释。
- 同类偏差在新场景中再次出现。
- 线索可靠度和实际后果逐步提高。
- 主角只掌握部分信息并允许误判。
- 显恶不新增此前不存在的决定性证据。

## 期待来源

读者先自己察觉不协调，随后观察不同证据能否组成模式，并在揭晓时确认早期判断既有依据又并非唯一答案。

## 断档风险

- 阴森形容词直接锁定反派。
- 第一条线索只有恶意解释。
- 只堆疑点、不提高可靠度。
- 嫌疑人没有正常社会功能。
- 显恶靠临时新增证据。
- 主角反制与旧线索无关。

## 训练任务

设计六条具备无辜/恶意双解的线索，分成偏差、模式、损失三档；显恶场景禁止新增关键证据，主人公只能使用已出现的物证完成反制。
