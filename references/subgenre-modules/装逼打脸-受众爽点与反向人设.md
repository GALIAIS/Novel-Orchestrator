# 装逼打脸-受众爽点与反向人设

依据案例：batch-356 案例 1776

## 读者承诺

装逼打脸要先匹配细分题材的受众爽点。神豪文要用钱打脸，战力文要用力量打脸；主角开局低位最好反着金手指设计，让第一反击自然兑现题材承诺。

## 核心循环

```text
题材爽点 -> 反向低位人设 -> 嘲讽者铺垫 -> 金手指就位 -> 立刻打脸
```

## 开篇必备

- 明确题材受众想看的爽点。
- 主角低位与金手指反向对应。
- 打压者人设适配爽点。
- 主角已有立刻反击资本。

## 期待来源

- 读者知道主角能反击。
- 嘲讽越精准，反击越爽。
- 钱、力量、身份等爽点不跑偏。
- 低位和上行形成强对比。

## 断档风险

- 爽点塑造偏离题材。
- 打压太早，主角没法回击。
- 嘲讽者人设和金手指不匹配。
- 只写憋屈，不写兑现。

## 训练任务

给 5 个金手指各配一个反向低位人设和一个打压者，写清第一章如何当场打脸。
