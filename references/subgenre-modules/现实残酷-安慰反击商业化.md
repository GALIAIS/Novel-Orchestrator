# 现实残酷-安慰反击商业化

依据案例：batch-264 案例 1320

## 读者承诺

商业网文可以写残酷现实，但多数时候要给读者安慰、反击和补偿。只压抑不回升，要先确认平台和受众。

## 核心循环

```text
现实压力 -> 读者代入 -> 主角反击 -> 情绪补偿 -> 爽点兑现
```

## 开篇必备

- 目标平台。
- 读者想要的情绪。
- 现实压力来源。
- 主角能反击的路径。
- 补偿收益。

## 期待来源

读者期待从小说里获得爽快、甜美、安慰、轻松，哪怕先看见残酷，也要看到回升。

## 断档风险

教育读者、单纯暴露痛苦，会偏出版或表达型；若目标是商业回报，容易劝退。

## 训练任务

把一个现实不公桥段改成网文爽点：保留压力，必须写主角反击和补偿回收。
