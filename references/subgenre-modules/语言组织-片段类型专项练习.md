# 语言组织-片段类型专项练习

依据案例：`batch-070` 案例 347。

## 读者承诺

这个模块用于情节在脑子里有、落笔却卡壳的问题。它承诺的是把语言组织能力拆成打斗、对话、细节、过渡等片段类型，逐类找样本和仿写。

## 核心循环

```text
表达卡壳 -> 判定片段类型 -> 找同类样本 -> 拆句式功能 -> 小段仿写 -> 积累表达库存
```

## 开篇必备

- 先判断卡在哪类片段。
- 明确这一段的剧情功能。
- 找同类优秀样本。
- 只模仿组织方式，不照搬内容。
- 每次只练 300 到 500 字。

## 期待来源

- 片段分类降低难度。
- 样本提供句式和节奏。
- 小练习能快速反馈。
- 表达库存逐渐增加。
- 卡壳从抽象能力变成具体动作。

## 断档风险

- 泛泛说自己文笔差。
- 一上来想练完整章。
- 找错样本类型。
- 只背好词好句，不拆推进功能。
- 公式化后没有剧情推进。

## 训练任务

选一个最卡的片段类型，找 3 个样本，各拆 5 句。把句式功能标出来，再仿写一段自己的 300 字情节。
