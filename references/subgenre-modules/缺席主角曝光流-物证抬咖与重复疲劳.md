# 缺席主角曝光流-物证抬咖与重复疲劳

依据案例：batch-363 案例 1815

## 读者承诺

主角可以长期缺席，但每一轮曝光都要围绕主角的秘密、资产、过去行为抬高形象，并给下一轮更大秘密。

## 核心循环

```text
主角缺席 -> 物证曝光 -> 众人震惊 -> 主角形象抬高 -> 下一轮更大秘密
```

## 开篇必备

- 核心期待始终指向主角。
- 曝光物能证明主角很牛。
- 配角反应用来放大。
- 下一轮钩子明确。

## 期待来源

- 私房钱/资产/秘密。
- 节目或直播曝光。
- 他人替主角震惊。
- 主角身份越来越离谱。

## 断档风险

- 误以为配角戏本身能火。
- 每轮只是重复震惊。
- 主角存在感不够。
- 曝光层级不升级。

## 训练任务

设计 5 轮缺席主角曝光，每轮写物证、旁人反应、主角形象升级点，五轮不能同一种震惊。
