# 冒险修仙-老题材必须有新钩子

依据案例：`batch-331` 案例 1651

## 读者承诺

老式冒险修仙可以写，但开头必须用新角度、明确危机、奖励对比和章末下行拉住读者。

## 核心循环

```text
老题材场景 -> 明确危险和目标 -> 主角特殊能力破局 -> 奖励/危机升级 -> 追读
```

## 开篇必备

- 主角为什么到冒险地点。
- 地点危险和背景清楚。
- 术语符合修仙/玄幻类型。
- 奖励大小有对比。
- 章末不是结束副本，而是开启更大危机。

## 期待来源

读者期待主角在明确危险里凭特殊性拿到高价值奖励，而不是看氛围和谜语。

## 断档风险

- 无提纲随手写。
- 冒险题材老旧且没有新钩子。
- 人物关系和地点背景不清。
- 危机解决太快。
- 情绪上行处直接断章。

## 训练任务

重写第一章小纲：危险、到场原因、奖励对比、主角特殊性、章末危机五项必须齐。
