# 描写资料OCR-可搜索素材库与情节关联

依据案例：batch-354 案例 1766

## 读者承诺

描写资料库可搜索后，作者能更快找到外貌、神态和气质素材。但素材必须被改写成情节相关细节，并在后文用事件兑现，否则只是漂亮废话。

## 核心循环

```text
OCR 可搜索资料 -> 按状态检索 -> 选择情节相关细节 -> 改写压缩 -> 后文行为兑现
```

## 开篇必备

- 可检索关键词。
- 角色当下状态。
- 描写服务的场景功能。
- 后文可兑现的行为或情节。

## 期待来源

- 外貌线索让读者预判性格。
- 气质描写在后文被验证或反转。
- 描写素材成为伏笔。
- 搜索资料提升写作速度。

## 断档风险

- 直接复制资料原句。
- 只写更美，不服务剧情。
- 第一印象和后续行为脱节。
- PDF/资料不可读时臆造内容。

## 训练任务

做一张 30 条“描写关键词索引”：每条含关键词、角色状态、情节功能、后文兑现。只写索引，不抄原句。
