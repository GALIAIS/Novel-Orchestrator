# 西幻镜头感-氛围沉浸流

依据案例：`batch-151` 案例 752。

## 读者承诺

小众西幻可以不靠大众题材热度，而靠文字镜头感、运镜、氛围感和沉浸感吸引特定读者。学习时要区分文字质感价值和商业成绩价值。

## 核心循环

```text
西幻场景 -> 镜头推进 -> 氛围压住 -> 读者沉浸 -> 结合成绩判断模仿边界
```

## 开篇必备

- 清晰的场景镜头。
- 镜头移动顺序。
- 稳定氛围关键词。
- 读者能感到的沉浸细节。
- 作品成绩和作者履历。

## 期待来源

读者期待像看电影一样进入场景：画面连续、氛围统一、动作与环境互相托举。

## 断档风险

- 只写氛围，不写事件。
- 只谈个人喜欢，不查成绩。
- 题材小众却误当大众商业模板。
- 镜头感堆砌成慢节奏拖沓。

## 训练任务

选一段西幻或悬疑开头，拆成 5 个镜头：远景、角色入场、环境压力、动作推进、章尾氛围。再补充该作品成绩，判断只学文风还是可学结构。
