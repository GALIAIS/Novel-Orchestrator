# 文字教学-单作者风格追踪

依据案例：`batch-258` 案例 1290。

## 读者承诺

文字教学模块用单作者追踪和段落精拆学习视角、重心、心理描写和流畅度。

## 核心循环

```text
选定作者 -> 精拆段落 -> 标注句子功能 -> 模仿一段 -> 对照修正
```

## 开篇必备

- 初学只追一个目标作者。
- 拆具体段落，不空谈文笔。
- 标注视角、重心、心理密度、动作环境比例。
- 模仿后对照原文修正。

## 期待来源

学习者抓住一个作者稳定的文字脉络，减少多风格混学造成的混乱。

## 断档风险

- 想用统一公式解决所有文字问题。
- 同时混学太多作者。
- 只看词句漂亮，不看句子功能。
- 不做模仿验证。

## 训练任务

选目标作者 1000 字，标注每句功能：动作、心理、环境、解释、对话、反应。再按同样比例写自己的 1000 字。
