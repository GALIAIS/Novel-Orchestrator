# 金手指定位-书名承诺与滚雪球兑现

依据案例：batch-382 案例 1909

## 读者承诺

书名和简介以金手指引流，前三章就要兑现。金手指提供决定性资源，主角用意志和才智组合资源滚雪球。

## 核心循环

```text
宣传承诺 -> 金手指生效 -> 获得资源 -> 主角主动运用 -> 滚雪球爽点
```

## 开篇必备

- 清楚的金手指规则。
- 前三章首次收益。
- 主角主动选择。
- 下一次升级期待。

## 期待来源

- 读者看到标题噱头落地。
- 资源持续增加。
- 主角组合资源解决问题。
- 金手指与胜利保持因果联系。

## 断档风险

- 宣传金手指却长期写凡人流。
- 金手指每次直接给答案。
- 主角没有意志和抉择。
- 获得资源后不再强调来源。

## 训练任务

设计三章：展示规则、获得两个资源、由主角组合资源解题；胜利不能由金手指直接代办。
