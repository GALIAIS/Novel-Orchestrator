# 五折线训练-虚晃一枪与爽点爆发

依据案例：`batch-239` 案例 1195。

## 读者承诺

五折线训练用上行、下行、再上行、再下行、爽点爆发，帮助作者掌握情绪拉扯和目标兑现。

## 核心循环

```text
确认目标 -> 画五折线 -> 设计虚晃一枪 -> 避免主角背锅 -> 金手指爆发
```

## 开篇必备

- 主角金手指。
- 明确事件和目标。
- 对立情绪上下摆动。
- 最后一段完成目标。

## 期待来源

读者被短暂缓和和再次下行调动，最后在主角能力爆发中获得爽点。

## 断档风险

- 下行让主角背锅。
- 没有虚晃一枪。
- 最后爽点没有完成目标。
- 梗概写太长，说明没想清楚。

## 训练任务

任选一个小事件，只写五句五折线梗概；检查虚晃一枪、主角背锅、目标兑现三项。
