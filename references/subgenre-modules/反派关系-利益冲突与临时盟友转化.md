# 反派关系-利益冲突与临时盟友转化

依据案例：`batch-205` 案例 1021。

## 读者承诺

反派不必永久纯恶，只要在当前事件中与主角利益对立；利益变化后可以转成临时盟友。

## 核心循环

```text
利益对立 -> 反派施压 -> 主角破局 -> 利益重排 -> 敌友关系转化
```

## 开篇必备

- 明确当前事件的利益冲突。
- 给反派理性选择。
- 写清是否有不可调和理由。
- 事件后重新评估合作可能。

## 期待来源

读者期待聪明反派带来有效压力，并在利益变化后产生更复杂的关系走向。

## 断档风险

- 把反派等同无脑坏人。
- 没有利益冲突，只靠标签对立。
- 明明可合作却强行死仇。
- 事件反派和长期反派混淆。

## 训练任务

设计一个聪明反派转临时盟友的小剧情：当前冲突、阻止理由、主角破局、后续合作理由。
