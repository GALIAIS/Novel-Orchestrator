# 压抑情节-相关人物情绪对比

依据案例：`batch-248` 案例 1236。

## 读者承诺

压抑情节里的旁观者情绪只写和最终高潮相关的人，用对比放大反转。

## 核心循环

```text
判断相关性 -> 写战前误判 -> 高潮反转 -> 战后态度变化
```

## 开篇必备

- 明确最终高潮。
- 挑相关人物。
- 控制篇幅和时机。
- 无关人群略写。

## 期待来源

读者看到相关人物被压低或误判，后续态度反转会放大爽点。

## 断档风险

- 所有人都写一遍。
- 旁观者情绪抢主线。
- 战前战中战后停不下来。
- 写了对高潮没贡献的人。

## 训练任务

只挑 3 个相关旁观者，写他们的战前误判和战后反转，不写无关人群。
