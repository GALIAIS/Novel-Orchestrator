# 不公平伤害-强制不等价交换

依据案例：`batch-191` 案例 954。

## 读者承诺

不公平伤害的本质是强制的不等价交换，用极端失衡的付出和收益制造情绪下行与反击期待。

## 核心循环

```text
等价交换破坏 -> 受害者巨大付出 -> 加害者低成本获利 -> 无法拒绝 -> 愤怒蓄压
```

## 开篇必备

- 写清受害者付出什么。
- 写清加害者获得什么。
- 强制条件无法轻易反抗。
- 失衡程度足够明确。

## 期待来源

读者看到价值交换被强行扭曲，期待主角打破规则、讨回代价、完成清算。

## 断档风险

- 只有坏人作恶，没有交换失衡。
- 受害者损失不具体。
- 加害者收益不清楚。
- 强制条件太弱，读者不信。

## 训练任务

设计一个不公平伤害场景，分别列出双方的付出、获得、强制条件和主角打破方式。
