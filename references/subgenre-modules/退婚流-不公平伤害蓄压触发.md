# 退婚流-不公平伤害蓄压触发

依据案例：`batch-189` 案例 945。

## 读者承诺

退婚流和废柴流开局用不公平伤害蓄压，再用金手指或反击希望点触发爽点。

## 核心循环

```text
高位跌落 -> 群体嘲讽 -> 关系疏离 -> 公开羞辱 -> 希望点出现 -> 打脸期待
```

## 开篇必备

- 主角曾经有高位或天赋。
- 伤害覆盖能力、地位、关系或尊严。
- 羞辱足够不公平。
- 压抑后尽快给希望点。

## 期待来源

读者积累对不公平的愤怒，等待主角获得工具后清算、打脸和夺回尊严。

## 断档风险

- 只压抑，不给希望。
- 现代读者耐心不足，照搬早期长蓄压。
- 伤害太散，不服务反击。
- 金手指出现太晚，读者先弃书。

## 训练任务

设计一个废柴流开局，写 3 个不公平伤害和 1 个希望点；希望点必须在前三章出现。
