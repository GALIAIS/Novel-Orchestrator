# 奇幻大航海-反殖民领主争霸

依据案例：`batch-148` 案例 740。

## 读者承诺

主角穿越到被殖民侵略的原始部落，利用奇幻种族模板和领主经营，带落后部落反杀大航海列强，完成弱者文明逆袭。

## 核心循环

```text
殖民追兵 -> 部落弱势 -> 种族模板亮相 -> 特色战斗破局 -> 部落脱困 -> 反殖民争霸
```

## 开篇必备

- 直接从追兵危机切入。
- 被压迫部落处境。
- 架空但可识别的大航海原型。
- 金手指第一次亮相有种族特色。
- 前四章压缩成两章。

## 期待来源

读者期待弱者翻盘和奇幻特色，不只是数值增强。蜥蜴人模板必须打出另类战斗方式。

## 断档风险

- 开头先进山洞找草药，压力不足。
- 种族模板只加速度力量。
- 真实地名太直白或历史负担太重。
- 世界观冷门焦虑掩盖了卖点展示不足。

## 训练任务

写一个 1500 字开篇，从发现殖民者追兵开始。设计 3 个蜥蜴人模板非数值战斗用法，并把大航海殖民美洲改成架空奇幻背景。
