# 复杂设定-赛博仙侠铺设定钩子

依据案例：`batch-100` 案例 496。

## 读者承诺

给愿意读复杂世界的老白读者一个新鲜、密集、有问题可追的世界。重点不是把设定讲全，而是让读者在场景里看见异常，并不断想知道“这东西为什么存在、背后是谁、主角为什么被卷入”。

## 核心循环

```text
现实锚点 -> 异常设定物件 -> 主角行动 -> 暂不解释的问题 -> 新人物/势力露头 -> 更深设定钩子
```

## 开篇必备

- 先给时间、地点、人物，降低复杂设定的进入成本。
- 第一屏必须有可感知场景，不能只有设定名词。
- 赛博元素和仙侠元素要放进同一个动作或物件里。
- 每段设定说明后必须留下一个具体问题。

## 期待来源

- 陌生世界的新鲜感。
- 主角身份和异常状态。
- 保护者、敌人、鬼怪、法器、技术来源等未解问题。
- 老白读者对复杂规则的求知欲。

## 断档风险

- 只解释世界观，不推进主角当前处境。
- 名词过多但没有悬念。
- 高信息密度压垮非目标读者。
- 打斗高潮被设定说明切碎，爽点减弱。

## 训练任务

写一个 800 字赛博仙侠场景，必须包含 1 个现实城市锚点、1 个仙侠术语、1 个技术化物件、1 个主角危机和 3 个暂不解释的钩子。删掉所有不能引出问题的设定句。
