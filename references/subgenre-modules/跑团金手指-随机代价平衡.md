# 跑团金手指-随机代价平衡

依据案例：batch-275 案例 1374

## 读者承诺

用随机性制造紧张和可玩性，但保证收益与代价公平。跑团式翻车可以有，不能让读者觉得作者强行坑主角。

## 核心循环

```text
随机判定 -> 收益出现 -> 代价触发 -> 公平感检验 -> 主角逐步掌控
```

## 开篇必备

- 先展示一个核心功能。
- 副作用要和收益匹配。
- 功能多就分批解锁，不要开头全讲。

## 期待来源

读者期待赌注、翻盘和主角成长。随机不是失控，而是让主角在规则里学会降低风险。

## 断档风险

- 误伤友军等副作用代价过大。
- 收益小、代价重，直接变毒点。
- 跑团规则解释太多，小说不如视频好看。

## 训练任务

写三次金手指使用：小赚、大赚、翻车。每次都标出收益、代价和主角学到的下一步策略。
