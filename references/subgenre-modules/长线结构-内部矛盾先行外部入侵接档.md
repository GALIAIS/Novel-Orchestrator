# 长线结构-内部矛盾先行外部入侵接档

依据案例：`batch-095` 案例 473。

## 读者承诺

王朝、阶层、底层压迫类开局，前期承诺是主角从内部矛盾里积累资本。如果外部入侵太早出现，所有内部矛盾都会被“一致对外”冲淡。更好的长线结构是前期内部矛盾先行，中期用外部入侵接档。

## 核心循环

```text
内部压迫 -> 主角破局 -> 积累资本 -> 埋外敌伏笔 -> 内部高潮 -> 外部入侵接档 -> 新卷压力升级
```

## 开篇必备

- 前期只让读者看清内部压迫和主角利益。
- 外部入侵只能做传闻、异象、边境消息，不正面抢戏。
- 主角的资本积累要来自解决内部矛盾。
- 中期旧矛盾解决后，外部入侵立刻提供新追读。

## 期待来源

- 底层翻身的压迫和补偿。
- 主角一步步积累资源。
- 伏笔兑现时的世界扩大感。
- 外敌来临后主角从地方人物变成关键人物。

## 断档风险

- 一开篇就写世界级危机，底层矛盾失焦。
- 外敌过强，主角早期完全无能为力。
- 伏笔太少，中期入侵像强行换书。
- 内部高潮结束后没有新压力接上。

## 训练任务

写一个两卷接档表：第一卷 5 个内部矛盾事件，第二卷 5 个外部入侵事件。第一卷每个事件必须给主角一项资本，第二卷每个事件必须用到前卷资本。
