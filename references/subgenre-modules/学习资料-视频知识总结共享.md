# 学习资料-视频知识总结共享

依据案例：`batch-192` 案例 957。

## 读者承诺

把视频教程整理成文字笔记并共享，可以降低复习成本，也能把听懂转成可执行写作动作。

## 核心循环

```text
观看视频 -> 整理笔记 -> 提炼动作 -> 分享反馈 -> 补充完善
```

## 开篇必备

- 标明视频主题。
- 记录核心概念。
- 转成写作动作。
- 保留疑问和后续补充位置。

## 期待来源

学员能快速复盘教程重点，并把概念用于开篇、人物、爽点、设定或改稿。

## 断档风险

- 只搬运，不理解。
- 笔记只有概念，没有动作。
- 附件不可检索。
- 社群反馈热闹但没有复盘。

## 训练任务

看一个教程视频，整理 10 条“我下一次写作要做什么”，禁止只写名词解释。
