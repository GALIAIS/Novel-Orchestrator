# 同人长篇-分卷门槛校准

依据案例：`batch-133` 案例 663。

## 读者承诺

第二卷可以承接第一卷和原作，但要让读者快速知道当前目标、关系和冲突。看不懂会被误判成节奏慢。

## 核心循环

```text
承接前文 -> 补足必要信息 -> 给出本卷目标 -> 推进新事件 -> 控制原作门槛
```

## 开篇必备

- 当前主角目标。
- 第一卷必须知道的信息。
- 原作背景的最低补充。
- 本卷独立冲突。
- 专有名词控制。

## 期待来源

- 老读者看到延续。
- 新读者能抓住当前事件。
- 原作背景增强理解，而不是成为门槛。
- 分卷目标让节奏可判断。

## 断档风险

- 默认读者都懂原作。
- 让读者跳过第一卷却不给重入信息。
- 专有名词密度过高。
- 把看不懂误当成慢节奏。

## 训练任务

给一份第二卷大纲，列出所有依赖前文/原作的信息，并各用一句话补足。最后判断慢感来自事件少、回报迟还是信息门槛高。
