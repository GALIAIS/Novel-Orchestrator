# 性转变身-CP倾向前置与平台适配

依据案例：batch-359 案例 1791

## 读者承诺

变身/性转题材要先承诺平台、读者阵营和关系方向。读者进入前需要知道这是变百、变嫁、变单还是其他路线，避免简介含糊造成预期诈骗。

## 核心循环

```text
题材设定 -> 平台匹配 -> CP/单身声明 -> 开篇兑现变身趣味 -> 固定读者持续反馈
```

## 开篇必备

- 明确变身或性转触发方式。
- 标签/简介提前说明关系方向。
- 避开平台敏感元素和表达。
- 第一卷不要摇摆多路 CP 预期。

## 期待来源

- 稳定题材读者群。
- 变身后的身份差和行动差。
- CP 或单身路线的确定性。
- 平台读者对题材的包容度。

## 断档风险

- 变百、变嫁、变单预期混写。
- 简介故意含糊，几边流量都想吃。
- 敏感年龄/称谓触发审核或反感。
- 只靠猎奇，没有剧情和人物承诺。

## 训练任务

为同一变身设定写三版 100 字简介，分别服务变百、变嫁、变单读者，并标出每版绝不能混入的误导承诺。
