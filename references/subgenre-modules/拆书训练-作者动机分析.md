# 拆书训练-作者动机分析

依据案例：batch-282 案例 1408

## 读者承诺

拆书不止记录剧情，还要分析作者表达动机：为什么这样安排、承担什么功能、让读者产生什么反应。

## 核心循环

```text
剧情记录 -> 作者动机 -> 章节功能 -> 读者效果 -> 可迁移写法
```

## 开篇必备

- 事件摘要。
- 作者安排意图。
- 对应读者反应。
- 可迁移手法。

## 期待来源

训练者从“我看到了什么”升级到“作者为什么让我这样看”，拆书才真正转化为写作能力。

## 断档风险

- 只做剧情复述。
- 表格栏目很多但没有动机分析。
- 不判断读者效果。
- 附件不可见时强评具体内容。

## 训练任务

拆 5 章，每章写四项：事件、作者动机、读者效果、可迁移写法。
