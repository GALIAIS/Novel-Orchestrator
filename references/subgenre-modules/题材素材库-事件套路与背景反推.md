# 题材素材库-事件套路与背景反推

依据案例：`batch-231` 案例 1153。

## 读者承诺

题材素材库应是同题材同流派的事件套路集合，不是零散资料堆。

## 核心循环

```text
确定流派 -> 收集事件套路 -> 套入自身背景 -> 推演前后事件 -> 反推格局
```

## 开篇必备

- 先定题材和细分流派。
- 拆同流派作品的事件。
- 每个事件提炼成可替换套路。
- 用事件反推人物关系和势力格局。

## 期待来源

作者不再空想新点子，而是从已有流派事件模型中生成符合自己背景的新事件。

## 断档风险

- 素材库只存名词资料，不能生成剧情。
- 没细分流派，素材泛而无用。
- 只追求创新，不先建立可用套路。
- 背景格局和事件互相脱节。

## 训练任务

拆 10 个同流派事件，写成“触发条件-参与角色-冲突规则-结果收益-可替换背景”。再用 3 个事件反推作品格局。
