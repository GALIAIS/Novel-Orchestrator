# 剑修神医金手指-治病爽点与友方讨喜

依据案例：`batch-203` 案例 1013。

## 读者承诺

剑修神医型金手指要把治病破局写成前三章核心爽点，同时保证友方角色登场讨喜。

## 核心循环

```text
异界危机 -> 金手指剑显效 -> 治病破局 -> 友方改观 -> 长期主线勾连
```

## 开篇必备

- 书名和简介给出清晰卖点。
- 金手指外观、手感、效果写具体。
- 治病环节详写怀疑、过程、结果。
- 友方不要突兀攻击败坏眼缘。

## 期待来源

读者期待主角用特殊剑解决别人解决不了的问题，并在异世界站稳脚跟。

## 断档风险

- 简介承诺的文风开篇没兑现。
- 金手指是重点却写得模糊。
- 语言教学占用前三章黄金位置。
- 友方角色讨厌，读者不想看互动。

## 训练任务

写 1000 字治病爽点：病症展示、众人怀疑、金手指细节、治疗过程、结果反转、友方态度变化。
