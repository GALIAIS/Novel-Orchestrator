# 传统玄幻-系统潮流下的新意替代

依据案例：batch-360 案例 1798

## 读者承诺

传统玄幻可以写，但必须尽早给出区别于老套路的新意。系统是最容易的显性创新，不用系统也要有替代亮点。

## 核心循环

```text
传统题材 -> 同质化压力 -> 显性新意/系统机制 -> 第一章兑现差异 -> 留住读者
```

## 开篇必备

- 读者熟悉的传统元素。
- 第一眼差异点。
- 主角或能力机制新意。
- 第一冲突不能照旧。

## 期待来源

- 系统或替代机制的明确承诺。
- 传统爽点被新方式打开。
- 读者知道这本凭什么不一样。

## 断档风险

- 只复刻传统玄幻旧套路。
- 没有系统也没有别的新意。
- 把“文好可破”当新人赌注。
- 商业目标和为爱发电混淆。

## 训练任务

为一个传统玄幻开头设计三种不靠系统的新意：身份、冲突、能力机制。每种写 100 字第一章呈现方式。
