# 主线层级-主题虚线实线结构

依据案例：`batch-073` 案例 365。

## 读者承诺

这个模块用于判断弱主线、强主线和完全主线。它承诺的是用主题虚线、卷内实线、跨卷实线三个层级判断结构，而不是简单问“有没有一条主线”。

## 核心循环

```text
主题串联单元 -> 弱主线
主题串联卷 + 卷内明确主线 -> 强主线
多卷共同构成更大主线 -> 完全主线
```

## 开篇必备

- 先写出作品主题。
- 判断单元之间是否强因果。
- 判断每卷内部是否有明确主线。
- 判断卷与卷是否共同组成更大三段式。
- 目标分短中长和超长期。

## 期待来源

- 弱主线靠同主题新单元。
- 强主线靠卷内目标兑现。
- 完全主线靠跨卷矛盾回收。
- 主题统一让虚线不散。
- 实线主线让读者持续追。

## 断档风险

- 把弱主线误当没主线。
- 强主线只剩一条最终 boss 线。
- 换地图后主题断裂。
- 每卷没有内部起承转合。
- 跨卷伏笔太多但不回收。

## 训练任务

给同一个题材写三版结构：5 个单元剧、3 卷强主线、3 卷完全主线。分别标出主题虚线和实线目标。
