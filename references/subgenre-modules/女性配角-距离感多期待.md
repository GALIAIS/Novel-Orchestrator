# 女性配角-距离感多期待

依据案例：`batch-053` 案例 264。

## 读者承诺

女性配角靠笔墨、性格、与男主的关系和距离感制造长期期待，不靠美貌词堆砌。

## 核心循环

```text
魅力原型 -> 笔墨出场 -> 性格行动 -> 关系距离 -> 期待持续 -> 阶段更替
```

## 开篇必备

- 作者脑中先有具体魅力形象。
- 出场笔墨足够让读者记住。
- 性格必须体现在行动和对话里。
- 与男主保持距离感。
- 不同阶段安排不同期待角色。

## 期待来源

- 求而不得会形成长期期待。
- 性格推动剧情，角色才像活人。
- 团队互动能放大主角装逼。
- 多阶段角色配置能延长读者兴趣。

## 断档风险

- 只写美貌，读者记不住。
- 旁白人设和实际行为矛盾，会崩角色。
- 过早攻略会削弱期待感。
- 后期高配置角色前期写太多，会压低前期角色吸引力。

## 训练任务

设计 6 名女性角色，按前中后三阶段分布；每名写魅力点、性格动作、距离感和一次团队互动。
