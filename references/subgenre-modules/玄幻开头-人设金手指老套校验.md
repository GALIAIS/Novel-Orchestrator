# 玄幻开头-人设金手指老套校验

依据案例：`batch-228` 案例 1138。

## 读者承诺

玄幻开头要在第一二章立住主角人设，并让金手指有趣，不只是把旧套路讲清楚。

## 核心循环

```text
开头求评 -> 查人设 -> 查金手指趣味 -> 查剧情新变化 -> 决定改/毙
```

## 开篇必备

- 主角人设一章内可感知。
- 金手指一句话能让人想看。
- 旧套路必须有新变化。
- 老化开局要谨慎使用。

## 期待来源

读者期待新鲜主角、新鲜能力或熟套路的新解法。

## 断档风险

- 金手指清楚但无趣。
- 强行介绍能力。
- 舔狗开局过老。
- 三万字才发现核心承诺无亮点。

## 训练任务

给玄幻开头三项打分：主角人设、金手指趣味、旧套路新变化。任一项低于 3 分写最小改法；三项都低再考虑毙稿。
