# 正义桥段-主动性收益闭环

依据案例：`batch-188` 案例 937。

## 读者承诺

主角可以见义勇为，但必须有计划、有自保、有收益或至少真正解决冲突，不能让正义行为变成降智拉仇恨。

## 核心循环

```text
正义事件 -> 主角预判风险 -> 主动布局 -> 反派反扑入套 -> 冲突解决 -> 收益落袋
```

## 开篇必备

- 主角介入理由清楚。
- 反派后台和反扑路径提前可见。
- 主角不是临时莽上去。
- 解决冲突后有收益或局面推进。

## 期待来源

读者期待主角既三观正，又聪明可靠，能用布局把不公平事件变成打脸和收获。

## 断档风险

- 只拉仇恨，不解决冲突。
- 救人后主角白吃亏。
- 正义行为没有后手。
- 为冲突升级牺牲主角智商。

## 训练任务

写一个冤案桥段，标出主角介入、延缓判决、收集铁证、借势钉死反派四步；缺一步就重写。
