# 平台合约-追读倒贴与止损

依据案例：`batch-237` 案例 1181。

## 读者承诺

新人练笔也要先看平台规则；追读、全勤、补贴、扣款和版权风险不清楚，就不要用主力题材试错。

## 核心循环

```text
想练笔 -> 随手签约 -> 结算/追读风险暴露 -> 评估止损 -> 下本换平台
```

## 开篇必备

- 明确目标平台。
- 查清签约和收益规则。
- 识别倒扣、断更、追读门槛。
- 预设切书止损线。

## 期待来源

作者不再只看“能不能签”，而是把平台稳定性、合同边界、收益规则当作写作项目的生存条件。

## 断档风险

- 只为练笔就随便签。
- 不看合同和结算规则。
- 低估追读不够、断更、倒扣风险。
- 想搬稿时才发现版权或扯皮成本。

## 训练任务

开书前列一张平台风险表，写清签约、全勤、追读、版权、切书处理和扣款规则；有一项查不明白，就换低风险试验方案。
