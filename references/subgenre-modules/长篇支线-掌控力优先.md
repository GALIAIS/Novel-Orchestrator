# 长篇支线-掌控力优先

依据案例：`batch-038` 案例 188。

## 读者承诺

支线数量不由字数决定，而由篇章目标和作者掌控力决定。质量和回收能力比“30 万字塞几条”更重要。

## 核心循环

```text
篇章目标 -> 支线功能 -> 掌控难度 -> 回收点 -> 回流主线
```

## 开篇必备

- 先定篇章目标。
- 每条支线有功能。
- 先写回收点。
- 控制在自己能掌控的数量。

## 期待来源

- 支线补强主线。
- 每条支线都有回报。
- 读者不担心散线。
- 作者能稳定收束。

## 断档风险

- 按字数机械塞支线。
- 支线太多收不回。
- 支线质量低。
- 支线不回流主线。

## 训练任务

为一个卷纲列 5 条支线候选，删到 2-3 条，并给每条写明功能和回收点。
