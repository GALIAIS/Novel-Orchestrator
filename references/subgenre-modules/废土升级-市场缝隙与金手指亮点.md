# 废土升级-市场缝隙与金手指亮点

依据案例：batch-358 案例 1789

## 读者承诺

新人选题材要看流量、竞争和自身优势。传统玄幻竞争强且市场变小，废土升级若有亮眼金手指和足够味道，反而可能有机会。

## 核心循环

```text
题材候选 -> 市场/竞争判断 -> 选择缝隙 -> 配金手指亮点 -> 拆迁旧素材
```

## 开篇必备

- 题材流量判断。
- 竞争强度判断。
- 金手指亮点。
- 废土味道。

## 期待来源

- 废土环境带来生存压力。
- 金手指提供升级路线。
- 市场缝隙降低新人竞争压力。
- 旧人物或梗可换身份复用。

## 断档风险

- 被旧大纲沉没成本绑架。
- 传统玄幻无变化硬写。
- 废土只有壳，没有味道和爽点。
- 风格差异太大还强行迁移素材。

## 训练任务

把旧大纲素材拆成“能迁移/需改造/应封存”三类，再为废土升级设计一个第一章可验证的金手指。
