# 反派误判-借势离间局

依据案例：batch-281 案例 1405

## 读者承诺

让反派基于有限信息误判主角背后有靠山，主角顺水推舟制造两边互斗，比单纯装逼更有智斗和暗爽。

## 核心循环

```text
主角异常变强 -> 反派合理误读 -> 主角放大误会 -> 反派互相猜忌 -> 主角收利
```

## 开篇必备

- 主角真实底牌。
- 外界能观察到的异常。
- 两个反派之间本就有压力。
- 主角推动误会的动作。

## 期待来源

读者知道真相，看反派一本正经脑补错误结论，再互相消耗，获得智斗式装逼爽。

## 断档风险

- 反派误判没有逻辑。
- 身边人公开贬低主角。
- 主角没有主动推动局面。
- 读者不知道主角真实底牌。

## 训练任务

写 1500 字借势局：两个反派都误以为主角靠对方，误判必须各有证据，最后两边互相试探。
