# 番茄玄幻-顿悟金手指快节奏模板

依据案例：`batch-107` 案例 533。

## 读者承诺

用顿悟金手指把“主角比天才更天才”做成直观感官爽点。读者不需要复杂推导，开篇就要看到高逼格人物、夸张耗时对比、连续变强舞台和被碾压的天才强者。

## 核心循环

```text
高位设定 -> 顿悟触发 -> 数字对比 -> 新舞台任务 -> 天才登场 -> 主角碾压
```

## 开篇必备

- 圣地、老祖、神魔等高逼格锚点。
- 顿悟对象和普通人耗时。
- 主角极短时间突破。
- 一个继续变强的比赛、秘境或冒险活动。
- 捧哏角色抬高主角差距。

## 期待来源

- 突破速度远超常人。
- 每个阶段能力极限继续拔高。
- 天才妖孽被主角衬成普通人。
- 老骨头或高位角色亲自震惊。

## 断档风险

- 强行嘲讽导致读者出戏。
- 只堆天才，不给舞台规则。
- 只有金手指，没有继续变强的由头。
- 番茄式情绪节奏直接搬到不适合的平台。

## 训练任务

写一个顿悟桥段：先给功法历史顶点，再给主角顿悟耗时对比，最后把“如何继续变强”指向一个必须参加的活动。
