# 神秘侧开篇-主动选择触发

依据案例：`batch-020` 案例 099。

## 读者承诺

让神秘侧开篇不再把主角拖着走。主角必须做选择，金手指必须尽早改变局面，组织和怪异必须有合理行动逻辑。

## 核心循环

```text
神秘事件 -> 主角选择 -> 组织反压 -> 金手指触发 -> 主角反击 -> 追杀升级
```

## 开篇必备

- 神秘侧危险。
- 组织为什么接触主角。
- 主角的主动选择。
- 金手指触发条件。
- 反击后的新危机。

## 期待来源

- 主角从被害者变成反击者。
- 影咒或异常能力和主角绑定。
- 组织从观察者变成追杀者。
- 不公平伤害被主角回收。

## 断档风险

- 神秘组织动机薄弱。
- 主角被动五章。
- 金手指展示不到位。
- 神秘侧描写只炫设定，不服务冒险。

## 训练任务

写一个神秘侧三章小纲：第一章异常入身，第二章主角做选择，第三章金手指触发并反杀一次。
