# 情绪折线-层层上扯

依据案例：`batch-015` 案例 074。

## 读者承诺

情绪拉扯不是一次嘲讽一次震惊，而是反复下拉和上扯，每一层都用新质疑和新证明把期待抬高。

## 核心循环

```text
误判 -> 小证明 -> 新质疑 -> 权威证明 -> 更大危机 -> 最终引爆
```

## 开篇必备

- 初始误判要清楚。
- 每次下行都有新理由。
- 每次上行都有新证据。
- 最终引爆回收前面所有质疑。

## 期待来源

- 旁观者第一次被镇住。
- 权威人物摇摆或证明。
- 官方/专业系统介入。
- 主角身份或能力最终被确认。

## 断档风险

- 重复“众人不信/众人震惊”。
- 折线只上下，没有整体抬升。
- 下行太多，上行太弱。
- 最终引爆没有汇总前面铺垫。

## 训练任务

写一场三层拉扯戏，每层写一个不同质疑理由和一个不同证明方式，最后做一次总引爆。
