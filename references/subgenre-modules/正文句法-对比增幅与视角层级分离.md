# 正文句法-对比增幅与视角层级分离

依据案例：batch-392 案例 1960

## 读者承诺

先按陈述、解释、问答和对比的叙事功能组织信息，再以共同尺度放大关键差值；叙事人称与内心表达分层处理，保持稳定认知锚。

## 核心循环

```text
确定信息功能 -> 选择表达形式 -> 用共同尺度拉对比 -> 人物选择回应差值 -> 稳定视角承接结果
```

## 开篇必备

- 本段唯一观察视角与已知边界。
- 读者理解行动所需的基准事实。
- 最值得放大的一个关键差值。
- 对比后人物会作出的实际选择。

## 期待来源

- 基准让身份、资源和结果差距可见。
- 对比把静态信息转成情绪估值。
- 人物选择让反差进入因果链。
- 稳定视角让情绪持续归属于同一锚点。

## 断档风险

- 把功能标签当互斥句型公式。
- 每句话都拉满反差导致失真。
- 为强情绪临时切换叙事人称。
- 配角视角切入却没有新增信息。

## 训练任务

标注 800 字正文的句子功能，只保留两处有共同尺度的关键对比；全文固定一个限制视角并删除所有无意越界信息。
