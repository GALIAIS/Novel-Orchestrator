# 写作训练-付费验证

依据案例：`batch-140` 案例 698。

## 读者承诺

写作训练要进入有效读者场景，用签约、完读、追读、付费转化和掉点章节判断作品是否兑现承诺，而不是用免费练手逃避市场反馈。

## 核心循环

```text
连续扑街 -> 想免费练手 -> 识别反馈失真 -> 进入真实验证场 -> 按读者行为复盘下一本
```

## 开篇必备

- 作品原本承诺。
- 实际读者掉点。
- 完读率或追读问题。
- 是否经过签约/上架验证。
- 下一本修正目标。

## 期待来源

- 作者知道哪里崩，而不是只知道扑。
- 付费反馈暴露真实阅读意愿。
- 番茄等免费平台也通过签约和读者反馈验证。
- 训练方向从“写完一本”改为“让读者读下去”。

## 断档风险

- 免费收藏、点击带来虚假安全感。
- 用低压环境训练高压能力。
- 复盘停留在文笔不好、题材不行。
- 完读率太低却不定位承诺断裂点。

## 训练任务

选一部表现差的作品，列出三个真实掉点，并对应读者行为：哪里不追、哪里完读低、哪里收藏后不看。写 300 字复盘：作品承诺什么，读者为什么不再相信。
