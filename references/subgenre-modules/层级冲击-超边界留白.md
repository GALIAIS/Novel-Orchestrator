# 层级冲击-超边界留白

依据案例：`batch-026` 案例 129。

## 读者承诺

用清晰层级建立危险边界，再让主角突然进入超出边界的层级，并用留白放大冲击。

## 核心循环

```text
分级标准 -> 已知危险 -> 想象边界 -> 超边界数字 -> 戛然而止 -> 回归兑现
```

## 开篇必备

- 层级越高越危险的标准。
- 已知危险层级的具体场景。
- 读者认可的极端边界。
- 突然出现的超边界提示。

## 期待来源

- 数字推进像进度条。
- 已知边界被打破。
- 留白迫使读者脑补。
- 主角归来展示变化。

## 断档风险

- 只报数字，不写层级差异。
- 极端边界没建立就抛超边界。
- 超边界后马上解释，削弱冲击。
- 主角归来没有实力变化。

## 训练任务

设计一个 1、5、10、13、999 的地下城层级表，写清每层危险升级点，主角落到 999 后立刻切到队友视角。
