# 连载防崩-大纲路线与惯性

依据案例：batch-303 案例 1512

## 读者承诺

开书后停和改都变难，大纲要提供最低路线，防止连载靠惯性写崩。

## 核心循环

```text
开书连载 -> 每日推进 -> 对照大纲 -> 修正偏航 -> 阶段兑现
```

## 开篇必备

- 主线目标明确。
- 当前卷目标明确。
- 阶段节点不超过三到五个。
- 每周检查是否偏离承诺。

## 期待来源

- 读者看到稳定推进。
- 作者减少临场乱写。
- 卷尾高潮有预设方向。

## 断档风险

- 无大纲莽长篇。
- 写具体情节全靠惯性。
- 发现偏航太晚，大改成本过高。

## 训练任务

写一页防崩大纲：主线、当前卷、三阶段节点、卷尾高潮。每写完三章对照一次。
