# 三翻四震-情绪折线尺度

依据案例：`batch-009` 案例 041。

## 读者承诺

通过多次铺垫和转折，让读者的期待逐层升高，最后在爆发点获得更强满足。重点是情绪阶梯，不是固定数字。

## 核心循环

```text
铺垫 -> 小转折 -> 期待升高 -> 再铺垫 -> 大转折 -> 爆发
```

## 开篇必备

- 先确定当前情节尺度。
- 明确读者最初期待。
- 每次转折都要让期待更具体。
- 爆发点必须回收前面的铺垫。

## 期待来源

- 读者看到矛盾逐层升级。
- 每一次反应都比上一次更强。
- 多次铺垫让高潮不是突然砸下来。
- 爆发后还能接新问题。

## 断档风险

- 死数三次反转四次震惊。
- 小章节硬塞大高潮结构。
- 转折只变事实，不变情绪。
- 铺垫太多但没有爆发。

## 训练任务

给同一个打脸点画三条折线：500 字、3000 字、两万字。每条只标期待上升、转折、爆发，不写正文。
