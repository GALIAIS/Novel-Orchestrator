# 教程分发-门槛分层实践闭环

依据案例：`batch-080` 案例 396。

## 读者承诺

这个模块用于判断教程内容该怎么学、怎么做、怎么内化。它承诺把低门槛入门内容、进阶长文内容和实写实践分开，避免把“看懂教程”误当成“会写”。

## 核心循环

```text
低门槛视频入门 -> 长文进阶理解 -> 拆书找样本 -> 实写验证 -> 复盘再学
```

## 开篇必备

- 先判断自己在门槛外、门槛边缘还是门槛内。
- 初级概念可用视频建立方向。
- 复杂概念要用长文和图表反复研究。
- 每个概念必须绑定一个拆书样本。
- 最后必须进入自己的正文或章纲。

## 期待来源

- 新人通过视频获得方向感。
- 长文提供更高信息密度。
- 案例图解降低抽象难度。
- 实写让理论暴露真正短板。
- 复盘让教程从爽感变成能力。

## 断档风险

- 把暴论教程当捷径。
- 教程看很多遍但不实践。
- 只记笔记，不写正文。
- 用视频学习复杂结构，反复回看成本太高。
- 没有写过足够字数就急着评价理论无用。

## 训练任务

选一个教程概念，按“视频级一句话、长文级一页拆解、样本一处、自己正文一处”四步完成。缺少实写验证时，不允许标记为掌握。
