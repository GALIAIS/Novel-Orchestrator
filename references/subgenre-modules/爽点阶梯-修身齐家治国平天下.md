# 爽点阶梯-修身齐家治国平天下

依据案例：`batch-316` 案例 1577

## 读者承诺

爽点可以按影响范围升级：先改变自己，再让家庭、队友、势力、国家、世界依次受益，并获得不同角色的价值肯定。

## 核心循环

```text
自我突破 -> 家庭受益 -> 队友认可 -> 敌人破防 -> 势力扩张 -> 世界改变
```

## 开篇必备

- 先有可见的个人变化。
- 早期让家人或身边人受益。
- 用敌人、队友、旁观者给反馈。
- 每轮扩大影响范围。

## 期待来源

读者期待主角行为不只改变自己，还能逐步改变身边世界，并被别人承认价值。

## 断档风险

- 影响范围跳太快，缺少台阶。
- 只有战力提升，没有外部改变。
- 旁人没有反馈，爽点落不实。
- 世界级目标太早出现，显空。

## 训练任务

把一个个人升级桥段补齐五层反馈：自己、家人、队友、敌人、势力。
