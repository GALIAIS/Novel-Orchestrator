# 新人学习-作业分区路径

依据案例：`batch-174` 案例 870。

## 读者承诺

新人进入写作社群后，要先弄清作业分区、视频作业、公开处刑和置顶规则，按路径提交才能获得有效反馈。

## 核心循环

```text
新人入圈 -> 读置顶 -> 找作业分区 -> 按视频提交 -> 公开处刑按规则发
```

## 开篇必备

- 先看置顶帖。
- 作业一般伴随视频发布。
- 过期作业可能不能补交。
- 公开处刑按栏目规则提交。

## 期待来源

新人期待快速获得反馈。清楚入口和格式后，反馈成本会降低，训练也更稳定。

## 断档风险

- 不看规则直接发正文。
- 把作业和公开处刑混在一起。
- 过期作业还按当前任务提交。
- 求评问题不明确，反馈发散。

## 训练任务

进入一个写作社群后，先整理置顶帖、作业分区、公开点评规则、提交格式和截止时间，再提交你的第一次作业。
