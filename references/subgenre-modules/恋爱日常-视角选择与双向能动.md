# 恋爱日常-视角选择与双向能动

依据案例：`batch-416` 案例 2078

## 读者承诺

恋爱日常按“主动追求、被看见、共同成长或观察暧昧”的核心体验选择男性、女性或双视角，而不是按主角性别机械预测代入。双方都有独立生活、欲望、边界和否决权；普通日常事件暴露需求差，迫使人物承担拒绝风险并让信任、公开度、共同计划或距离改变一格。

## 核心循环

```text
确定核心恋爱体验 -> 选择拥有最多关键选择和代价的主视角 -> 为双方设置独立目标、边界与否决权 -> 日常事件暴露需求差 -> 主角主动表达并承担拒绝风险 -> 对方按自身利益回应 -> 关系状态改变一格 -> 余波进入各自生活并制造下一次相处 -> 必要时用另一视角提供独有重释
```

体验决定焦点，双向能动提供真实心动，关系状态格保证日常有进度，边界与修复让亲密可信，有限视角切换避免重复。

## 开篇必备

- 主动追求、被看见、共同成长或观察暧昧中的主体验。
- 主视角拥有的关键选择、未知和代价。
- 双方各自的生活目标、社交关系和关系欲望。
- 双方不可越过的边界与真实否决权。
- 主角的关系能力、盲点和被拒绝风险。
- 一个会暴露需求差的普通日常事件。
- 对方基于自身目标作出的主动回应。
- 本场改变的一格关系状态。
- 余波进入工作、家庭或朋友线的方式。

## 期待来源

- 主角会不会清楚表达而不是操纵。
- 对方在没有作者强迫时为何主动靠近。
- 哪项边界会让最优恋爱方案失效。
- 普通日常怎样暴露深层价值差。
- 误解后谁会承担修复成本。
- 关系公开或共同计划会引来什么现实压力。
- 另一视角何时能重释而非重复场景。

## 断档风险

- 用主角性别直接代替代入设计。
- 把轻浮、贬低或制造依赖当作恋爱能力。
- 恋爱对象只有人设标签，没有独立目标。
- 对方不能拒绝、离开或提出条件。
- 连续甜蜜互动后关系状态完全不变。
- 双视角重复同一信息，拖慢日常节奏。
- 用热门角色模板替代具体关系因果。
- 依据旧评论处理当前年龄、同意和平台规则。
- 按分区刻板印象改掉作者真正擅长的视角。

## 训练任务

用同一前提分别写男性、女性和双视角三版 1200 字开场。双方都要有独立目标和一次否决权，场末必须改变一格关系状态；将三版交给目标读者复述体验，依据核心体验和关键选择选择视角，而非统计主角性别偏好。
