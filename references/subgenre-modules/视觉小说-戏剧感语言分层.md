# 视觉小说-戏剧感语言分层

依据案例：`batch-056` 案例 277。

## 读者承诺

这个模块用于提升对话和背景语言质感。它承诺的不是把台词写得华丽，而是让语言强度、生活感、戏剧感和叙事层级匹配角色功能，使人物关系和世界压力通过语言显形。

## 核心循环

```text
场景压力 -> 角色立场 -> 台词互攻 -> 背景句暗示 -> 语言层级区分 -> 人物和主题同时立住
```

## 开篇必备

- 场景必须有明确关系压力，如争产、分赃、审判、背叛。
- 每个说话人有不同利益立场。
- 台词至少有一句能戳痛点。
- 普通人、权威者、超然者的语言质感不同。
- 背景语言不只写景，还要暗示命运或主题。

## 期待来源

- 读者从台词里看到人物互相攻击和防守。
- 角色形象通过语言动作快速立住。
- 背景句埋下结局暗示。
- 现实层和幻想层的语言差异制造层次。
- 日常逐渐混入异物，带来不安和悬念。

## 断档风险

- 台词只有信息交换，没有立场交锋。
- 所有人说话一个腔调。
- 戏剧感过量，普通场景显得浮夸。
- 背景描写只堆形容词，不服务命运暗示。
- 借鉴视觉小说时忘记网文阅读节奏。

## 训练任务

写一场 800 字多人争执。先给每个人写一句“他最怕别人提到什么”，然后让台词围绕这些痛点互相逼迫；最后用一句环境描写暗示后续灾难。
