# 旧威名迪化-虚张声势

依据案例：`batch-164` 案例 816。

## 读者承诺

利用主角过去威名和当前虚弱之间的信息差，让旁人自我脑补、自我恐惧，形成迪化喜剧和悬念。

## 核心循环

```text
旧战绩 -> 权威抬价 -> 旁人误判 -> 主角虚张声势 -> 短期渡劫 -> 长期补实力
```

## 开篇必备

- 读者知道主角当前变弱。
- 旁人知道主角过去很强。
- 至少一个懂行者抬高主角旧战绩。
- 主角必须承担穿帮风险。

## 期待来源

读者期待主角如何用旧威名撑过眼前危机，并等待误会兑现、穿帮或转化为真实实力。

## 断档风险

- 读者不知道主角虚弱，反差不足。
- 旁人误会没有权威来源。
- 主角只享受误会，不付代价。
- 旧威名长期不兑现。

## 训练任务

写一个旧威名迪化场景：权威角色提旧战绩，敌人误判，主角当前虚弱但必须维持架势。
