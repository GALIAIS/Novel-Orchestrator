# 综漫系统-能力模板启动章

依据案例：`batch-135` 案例 673。

## 读者承诺

综漫系统同人开篇要尽快展示能力模板的压场效果。系统解释和空间设定服务事件，不能抢走第一章爽点。

## 核心循环

```text
获得能力模板 -> 遭遇压迫 -> 主角第一次反制 -> 旁人反应 -> 后置系统解释
```

## 开篇必备

- 能力模板一句话说明。
- 第一场冲突尽快出现。
- 主角第一次能力使用。
- 敌人或旁人反应。
- 系统说明压缩到必要信息。

## 期待来源

- 读者熟悉能力模板。
- 中二幻想被具体行动兑现。
- 强敌压迫后被反制。
- 旁人反应放大逼格。

## 断档风险

- 第一章系统长篇解释。
- 空间规则分析过早。
- 主角只冷淡命名强敌，没有情绪。
- 中二姿态不绑定行动。

## 训练任务

重排一个综漫系统开篇：300 字内说明系统，写出一次能力使用、一个对手压迫、一次主角反制、一个旁人反应。
