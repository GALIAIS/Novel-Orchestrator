# 望远镜期待-全文远景拉满

依据案例：batch-284 案例 1416

## 读者承诺

用连续望远镜让读者始终看见远处更大的结果、秘密、危机或收益，靠“现在的小事件通向未来的大兑现”维持追读。

## 核心循环

```text
当前事件 -> 远端结果 -> 短期验证 -> 阶段回收 -> 更大远景
```

## 开篇必备

- 当前事件必须能落地。
- 第一层远景要足够具体。
- 短期必须有验证。
- 每个远景都要留回收位置。

## 期待来源

读者追的不是眼前小事，而是小事背后的大局、大变化和大兑现。

## 断档风险

- 只会抛神秘词，不给验证。
- 远景过大但人物成长不足。
- 题材混合导致读者不知道按科幻还是玄幻阅读。
- 望远镜长期不回收。

## 训练任务

写一个三章开篇，每章至少一个望远镜，并标注该望远镜在第几章或第几卷回收。
