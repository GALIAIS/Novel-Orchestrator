# 开书方法论-微创新与反套路拆解

依据案例：`batch-194` 案例 970。

## 读者承诺

拆潜力新书时，不学作者光环，而学标签风向、金手指机制、首章卖点、微创新入口和反套路处理。

## 核心循环

```text
潜力新书 -> 旧套路底盘 -> 微创新点 -> 反套路亮点 -> 新人可复制动作
```

## 开篇必备

- 先识别作品的旧套路底盘。
- 找到它换新的入口或机制。
- 判断反套路是否服务爽点。
- 提炼新人能执行的动作。

## 期待来源

新人能从已验证作品里提取可迁移结构，而不是盲目模仿大神表达、节奏和复杂度。

## 断档风险

- 只看大神书，忽略新人可学性。
- 把反套路当成目的。
- 只总结概念，不落到首章动作。
- 不区分标签风向和个人笔力。

## 训练任务

拆一本潜力新书，列出旧套路底盘、微创新点、反套路点、金手指机制、首章卖点，再筛掉新人学不了的部分。
