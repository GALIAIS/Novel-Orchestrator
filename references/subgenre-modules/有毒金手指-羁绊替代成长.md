# 有毒金手指-羁绊替代成长

依据案例：`batch-167` 案例 833。

## 读者承诺

让主角前期依赖高收益金手指，逐步发现代价，最终用传承、伙伴和自我成长替代金手指。

## 核心循环

```text
金手指爽感 -> 代价暴露 -> 被迫再用 -> 主导权争夺 -> 替代能力铺垫 -> 羁绊放逐
```

## 开篇必备

- 前期金手指要先兑现爽感。
- 代价要清晰且逐步加重。
- 主角不能长期失去主体性。
- 替代能力和伙伴羁绊必须提前铺。

## 期待来源

读者期待主角既能享受外挂升级，又能在代价压迫下完成真正成长，最后不是靠外挂赢，而是靠自己和关系网赢。

## 断档风险

- 金手指太复杂，新人驾驭不了。
- 后期突然换力量来源，显得生硬。
- 主角长期被金手指操控，读者失去代入。
- 多个金手指关系没有卷级收线。

## 训练任务

写三卷弧线：第一卷靠金手指赢，第二卷因金手指付代价，第三卷用前两卷铺好的替代力量摆脱金手指。
