# 脱口秀手册-逐字稿与情绪节奏训练

依据案例：batch-359 案例 1795

## 读者承诺

把脱口秀训练迁移到网文：先把洞察和情绪写成可检查的逐字稿，再用情境、节奏和反馈修改到读者愿意相信。

## 核心循环

```text
洞察/负面情绪 -> 逐字稿落地 -> 情境故事化 -> 节奏编排 -> 读稿反馈改稿
```

## 开篇必备

- 一个明确洞察。
- 一个真实情绪源。
- 一个具体小情境。
- 一处节奏点或反应点。
- 能被读者复述的卖点。

## 期待来源

- 情绪真实。
- 表达具体。
- 故事承载价值观。
- 反馈让文本持续变准。

## 断档风险

- 只讲概念不写逐字稿。
- 准备很多但文本没有反应点。
- 情绪是假的，节奏是平的。
- 不愿让别人读、不接受反馈。

## 训练任务

把一个卖点写成 300 字逐字稿式开场白，必须包含洞察、负面情绪、具体情境和一句能被读者记住的节奏句。
