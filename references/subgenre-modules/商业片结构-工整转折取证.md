# 商业片结构-工整转折取证

依据案例：`batch-042` 案例 207。

## 读者承诺

商业片可以作为结构训练样本：看主角团如何计划、遇到意外、转向新方案，并在上下行情绪中保持行动推进。

## 核心循环

```text
计划 -> 意外 -> 失败代价 -> 改计划 -> 情绪回升
```

## 开篇必备

- 目标明确。
- 计划路径清楚。
- 意外能破坏原计划。
- 新方案不是凭空出现。

## 期待来源

- 读者知道计划成功后会得到什么。
- 意外打断制造下行。
- B 方案带来新的行动期待。
- 多轮转折让事件不平。

## 断档风险

- 只学热闹场面，不拆结构功能。
- 转折工整但人物无魅力。
- 普通商业片被误当顶级范本。
- 计划失败没有代价。

## 训练任务

拆一段商业片桥段，写出三轮“计划-意外-改计划”，每轮标明情绪上行或下行。
