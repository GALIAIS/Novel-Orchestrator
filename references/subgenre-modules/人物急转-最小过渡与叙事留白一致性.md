# 人物急转-最小过渡与叙事留白一致性

依据案例：`batch-407` 案例 2032

## 读者承诺

新信息足以改变优先级时，人物可以快速变脸；只用停顿、目光、称呼或动作补一拍重估，并保证策略变化仍服从稳定价值和作品长期叙事风格。

## 核心循环

```text
旧信息下的姿态 -> 新信息改变优先级 -> 最小外显或内写重估 -> 行动急转 -> 稳定价值不变 -> 关系重新判断 -> 后续事件验证
```

## 开篇必备

- 稳定价值与不可退让项明确。
- 新信息确实改变安全、利益或关系。
- 一拍因果桥避免读者漏段感。
- 过渡方式服从长期视角风格。
- 喜剧急转避免解释过量。
- 后续行为验证灵活而非随机变脸。

## 期待来源

读者先按旧姿态建立预期，新机会出现后人物迅速改称或转向制造笑点；一拍细节让转折既快又可回溯。

## 断档风险

- 新信息不足以支持态度翻转。
- 把心理说明当作唯一补桥方式。
- 一段解释改变人物道德含义。
- 作品长期留白却单点突然内心剖白。
- 解释过长泄掉喜剧节奏。
- 只看一条退出评论判断普遍毒点。

## 训练任务

写原版、动作桥版和两句心理版，让读者分别判断转向原因、人物价值和关系含义；保留与长线人设和叙事风格最一致的一版。
