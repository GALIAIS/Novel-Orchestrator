# 历史人物现代-合规疑问与故事看头

依据案例：`batch-243` 案例 1211。

## 读者承诺

历史人物来到现代只是入口，真正要兑现古今错位、历史遗憾、现代事件和人物故事。

## 核心循环

```text
历史人物到来 -> 现代冲突 -> 历史回望 -> 主角介入 -> 离开/留下资源或麻烦
```

## 开篇必备

- 明确人物为何来到现代。
- 给出现代事件压力。
- 让历史身份带来具体冲突。
- 设计离开、留下或持续来访机制。

## 期待来源

读者认识历史人物，因此期待他在现代环境里制造反差，也期待他回望原本命运。

## 断档风险

- 只有人物噱头，没有事件。
- 古今错位笑点很快耗尽。
- 历史人物反而不影响主线。
- 单元模式故事功底不足。

## 训练任务

选一位历史人物，写 3 个现代单元：错位笑点、历史遗憾、离开后留下的资源或麻烦。
