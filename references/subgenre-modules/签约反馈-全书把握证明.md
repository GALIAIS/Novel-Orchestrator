# 签约反馈-全书把握证明

依据案例：batch-284 案例 1419

## 读者承诺

编辑看到潜力但不放心时，作者要证明自己对全书、人物成长和后续事件有把握，而不是只交一个开头。

## 核心循环

```text
少量正文 -> 编辑疑虑 -> 补大纲成长线 -> 能签就写 -> 签不了就改
```

## 开篇必备

- 一句话卖点。
- 前十章细纲。
- 三十万字主线。
- 主角成长三阶段。
- 后续核心矛盾。

## 期待来源

编辑需要相信作者能把题材写长、写稳，不会四千字后临时想。

## 断档风险

- 把反馈误读成单纯不过审。
- 没有后续人物成长安排。
- 大纲只有设定，没有事件。
- 签不了后不改也不复盘。

## 训练任务

给自己的开篇写一份“编辑信心包”：卖点、三十万字主线、成长线、前三个大事件和长期矛盾。
