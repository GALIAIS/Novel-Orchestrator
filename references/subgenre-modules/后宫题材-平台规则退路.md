# 后宫题材-平台规则退路

依据案例：`batch-088` 案例 439。

## 读者承诺

后宫、多女主、百合等关系线题材要先确认平台规则，再决定是否作为核心卖点。规则不明时，新书要准备单女主、无女主、弱暧昧或换平台退路。

## 核心循环

```text
规则传闻 -> 核实通知 -> 判断核心卖点 -> 改关系结构 -> 调整标签 -> 选择退路
```

## 开篇必备

- 不把聊天截图当最终规则。
- 明确后宫是否为核心卖点。
- 准备单女主版本。
- 准备无明确关系版本。
- 判断标签消失后的曝光风险。

## 期待来源

- 关系线能否稳过平台边界。
- 读者是否还能获得情感期待。
- 多女主卖点损失后是否有其他爽点支撑。
- 转平台是否更合算。

## 断档风险

- 抱侥幸开新书。
- 后宫是核心卖点却硬投收紧平台。
- 标签消失导致新人不可见。
- 传闻未核实就恐慌切书。

## 训练任务

把一个后宫构思改成三版：后宫版、单女主版、无明确关系版。分别写平台风险、卖点损失、替代爽点，再选当前最稳版本。
