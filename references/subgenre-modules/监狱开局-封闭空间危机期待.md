# 监狱开局-封闭空间危机期待

依据案例：`batch-039` 案例 195。

## 读者承诺

监狱开局天然提供封闭空间、规则压迫、逃脱目标和翻案期待。关键是出狱后仍有新期待接上。

## 核心循环

```text
入狱原因 -> 监狱规则 -> 第一敌友 -> 求生/逃脱 -> 翻案/改命 -> 出狱新期待
```

## 开篇必备

- 主角为何入狱。
- 监狱规则清楚。
- 短期目标明确。
- 第一敌人和盟友出现。

## 期待来源

- 自由被剥夺。
- 规则压迫。
- 越狱或翻案计划。
- 外部世界阴谋。

## 断档风险

- 只借监狱地点，不写规则压迫。
- 出狱后目标断档。
- 影视样本只学氛围，不学结构。
- 封闭空间人物关系太散。

## 训练任务

拆一个监狱开局：标出入狱原因、规则、敌友、短期目标、长期目标、出狱后新期待。
