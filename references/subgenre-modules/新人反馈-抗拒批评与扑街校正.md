# 新人反馈-抗拒批评与扑街校正

依据案例：`batch-246` 案例 1227。

## 读者承诺

新人要把反馈当作低成本市场校正，而不是把“不吸引读者”理解成人身贬低。

## 核心循环

```text
拿作品求反馈 -> 听到负面判断 -> 接受并验证 -> 修改书名简介 -> 再让市场验证
```

## 开篇必备

- 反馈对象具体。
- 问题指向读者点击。
- 不急着反驳。
- 用重复反馈决定修改。

## 期待来源

作者越早习惯负面反馈，越能在真正发布前改掉最伤点击的问题。

## 断档风险

- 把反馈当攻击。
- 只想听夸奖。
- 不拿成绩和点击验证。
- 靠扑街才承认问题。

## 训练任务

把书名简介给 3 个人看，只记录“不想点开”的理由，隔天按重复问题改一版。
