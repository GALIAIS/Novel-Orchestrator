# 赛博硬汉-网文结构承载文艺

依据案例：batch-275 案例 1371

## 读者承诺

用网文场景结构承载赛博硬汉、浪漫、文学性和个人审美。先让读者看懂主角欲望、世界矛盾和场景对抗，再让风格慢慢浮出来。

## 核心循环

```text
硬汉主角 -> 原始欲望 -> 世界矛盾 -> 场景对抗 -> 文艺余味
```

## 开篇必备

- 主角要有清楚的欲望。
- 世界要有亮眼且事件化的矛盾。
- 第一场戏必须有目标、对抗和转化。

## 期待来源

读者期待一个强烈的人带他们进入有张力的世界。文学性、引用和硬汉腔要成为余味，不应成为门槛。

## 断档风险

- 配角比主角更可怜、可信、可爱。
- 世界只介绍设定，没有背景矛盾。
- 情感戏只有对话，没有关系变化。
- 战斗戏只炫场面，不提供信息或破局。

## 训练任务

把一个文艺开篇改成场景：动作进入、信息揭示、对抗升级、人物转化、结尾疑问五步齐全。
