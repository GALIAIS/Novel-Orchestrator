# 编辑工作室-同名机构与合同关系核验

依据案例：batch-393 案例 1964

## 读者承诺

先识别“工作室”究竟是平台部门、认证合作方还是第三方机构，再用合同、资金流、权利和退出条款判断合作，不跨平台类推。

## 核心循环

```text
发现同名说法 -> 核验机构与平台关系 -> 确认合同对手和服务 -> 画清费用稿费权利流 -> 审查退出违约
```

## 开篇必备

- 机构法定或官方主体及平台关系。
- 服务内容、交付方式和验收标准。
- 稿费支付、收费与抽成路径。
- 版权、账号、独家、违约和退出条件。

## 期待来源

- 名称混淆被实体关系图消除。
- 口头培养承诺变成可验收服务。
- 资金与权利交换透明可计算。
- 最坏退出后果在签署前显现。

## 断档风险

- 用另一平台工作室经验类推。
- 只听口头“资源”和“指导”。
- 不确认稿费由谁支付。
- 忽略版权、账号和退出控制。

## 训练任务

为两份工作室合作说明分别画实体图和资金流图，任何主体、服务、费用、版权或退出项无书面证据都标红待核验。
