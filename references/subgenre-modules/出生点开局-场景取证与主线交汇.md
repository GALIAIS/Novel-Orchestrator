# 出生点开局-场景取证与主线交汇

依据案例：`batch-399` 案例 1994

## 读者承诺

从常见而有特点的具体场景进入陌生世界，让主角通过环境和人物互动取得身份、规则与危险信息，并在首章完成接入主线的不可撤销选择。

## 核心循环

```text
具体出生点 -> 环境人物给浅层信息 -> 主角按当前需求观察试探 -> 深层危险显出 -> 第一行动 -> 不可撤销变化接入主线
```

## 开篇必备

- 场景用途无需复杂解释。
- 至少两个细节提示世界或身份。
- 主角有明确当前需求。
- 浅层信息可见，深层信息靠互动。
- 身份或职业自然生成第一事件。
- 章末局面发生不可撤销变化。

## 期待来源

读者先快速理解空间，再随主角从细节推断陌生世界，并等待第一项选择造成的主线后果。

## 断档风险

- 场景成为静态风景导览。
- 所有身份信息靠长篇回忆。
- 特殊细节不影响任何判断。
- 随机路遇与主线无接口。
- 多条长线在第一章等量抢戏。
- 章末仍未发生状态变化。

## 训练任务

用病房、牢房、办公室分别写 500 字出生点，只能靠两个环境细节和一次互动让主角推断身份风险；保留最自然导向不可撤销选择的一版扩写。
