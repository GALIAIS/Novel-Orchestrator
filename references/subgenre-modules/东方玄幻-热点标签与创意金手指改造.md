# 东方玄幻-热点标签与创意金手指改造

依据案例：`batch-206` 案例 1030。

## 读者承诺

传统东方玄幻可以写，但纯老派升级很难；要用热点标签和创意金手指建立新鲜入口。

## 核心循环

```text
传统框架 -> 热点标签叠加 -> 创意金手指 -> 开篇展示 -> 升级期待
```

## 开篇必备

- 识别老派套路风险。
- 加入 2-3 个热点标签。
- 金手指必须有新鲜机制。
- 第一章展示差异点。

## 期待来源

读者看到熟悉东方玄幻框架，同时发现足够新的核心卖点。

## 断档风险

- 只复刻斗破式旧套路。
- 标签堆砌但正文不兑现。
- 金手指不够新。
- 开篇仍按废材退婚老桥段走。

## 训练任务

改造一个老派东方玄幻梗概：3 个热点标签、1 个创意金手指、第一章展示方式、前三章升级钩子。
