# 脑洞训练-学校母题与同类校验

依据案例：`batch-242` 案例 1209。

## 读者承诺

用“学校”这类日常母题承接类型元素，产出课程、规则、考试、老师、学生和冲突段子，再用同类作品校验新鲜度。

## 核心循环

```text
日常母题 -> 类型化改造 -> 制度细节 -> 段子/冲突 -> 同类校验 -> 差异化承诺
```

## 开篇必备

- 选一个人人懂的日常母题。
- 套入明确类型元素。
- 写出制度和课程细节。
- 查同类作品确认差异。

## 期待来源

读者熟悉学校秩序，因此当修仙、魔法、异世界元素进入学校制度时，会立刻产生会心一笑和设定好奇。

## 断档风险

- 只有概念，没有可写细节。
- 误把旧梗当新花样。
- 段子不能支撑长线。
- 类型元素和学校制度只是并列，没有碰撞。

## 训练任务

把一个日常母题改造成 3 个类型方向，每个方向写 3 个制度细节、1 个冲突和 2 个同类作品。
