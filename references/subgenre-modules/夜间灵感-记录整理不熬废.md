# 夜间灵感-记录整理不熬废

依据案例：`batch-332` 案例 1658

## 读者承诺

深夜灵感高发可以利用，但要记录、整理、转任务，不能只靠熬夜透支。

## 核心循环

```text
深夜灵感 -> 快速记录 -> 次日整理 -> 进入大纲/章节 -> 控制作息损耗
```

## 开篇必备

- 夜间灵感卡。
- 记录关键词和情绪。
- 第二天固定整理。
- 白天安排低创造任务。
- 设定夜间停止线。

## 期待来源

作者把夜间高沉浸转化成可持续产能，而不是第二天报废。

## 断档风险

- 灵感不记录，醒来忘光。
- 熬到四五点影响白天。
- 夜里兴奋，白天无执行。
- 只写设定名词，没记情绪。
- 长期作息崩坏。

## 训练任务

做一张夜间灵感卡：关键词、情绪、场景、下一步；第二天用 20 分钟整理进大纲。
