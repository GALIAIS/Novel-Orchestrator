# 男频宿命虐恋-相杀牺牲复活

依据案例：`batch-180` 案例 896。

## 读者承诺

宿命虐恋提供强命运感、相杀张力、牺牲情绪和轮回复活希望，但会筛掉偏轻松爽的男频读者。

## 核心循环

```text
血脉相冲 -> 相杀互动 -> 感情升温 -> 牺牲解咒 -> 巅峰战 -> 轮回追寻
```

## 开篇必备

- 相冲规则清晰。
- 相杀不只是伤害，还要制造互相理解。
- 复活/轮回希望提前埋。
- 女主牺牲必须有必要性。

## 期待来源

读者期待两人在不可调和的命运里建立信任，并在牺牲后看到重逢希望。希望感比单纯虐更重要。

## 断档风险

- 男频受众被重虐刷掉。
- 女主死得突然，没有复活伏笔。
- 感情没写够就牺牲。
- 只制造悲剧，不兑现希望。

## 训练任务

写一条宿命虐恋线，列出 3 个升温节点、3 个复活希望伏笔、女主牺牲必要性和最终重逢方式。
