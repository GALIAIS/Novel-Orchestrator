# 分支叙事-未知后果与多线因果汇流

依据案例：`batch-407` 案例 2034

## 读者承诺

让人物在信息不足时面对两项合理选择，后果未知却可回溯；选择改变隐藏状态，并经共享物件、信息或空间进入其他人物线，最终带着差异汇流。

## 核心循环

```text
多线独立目标 -> 两项合理选择 -> 隐藏状态变化 -> 跨线媒介传播 -> 局部后果 -> 延迟重释 -> 共同危机汇流 -> 差异化代价
```

## 开篇必备

- 多条线共享时间或共同危机。
- 每条线有独立目标和局部结果。
- 选择时两个选项都合理。
- 未知后果有早期可回溯线索。
- 跨线影响依赖物件、信息、交通或目击。
- 汇流保留资源、伤势和信任差异。

## 期待来源

读者先把选择看作局部决定，随后在另一条线发现影响；汇流时所有“巧合”回到人物行动，产生结构性顿悟。

## 断档风险

- 只有一个显然正确选项。
- 结局完全随机无法回溯。
- 支线只为主线搬运线索。
- 线性小说重复展示所有未选路线。
- 多线汇流后状态全部归零。
- 纯巧合替人物解决共同危机。

## 训练任务

设计三人同城十二小时故事，每人做一次两难选择，并通过物件、交通或信息改变另一条线；画状态图后写差异化汇流场景。
