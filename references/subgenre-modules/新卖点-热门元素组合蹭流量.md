# 新卖点-热门元素组合蹭流量

依据案例：`batch-188` 案例 939。

## 读者承诺

新人可以把 SCP、克苏鲁、赛博朋克、异能、修仙等热门元素组合成新皮，让读者先看到作品，再用故事执行留住人。

## 核心循环

```text
热门元素采样 -> 元素组合 -> 外显卖点 -> 开篇承接 -> 故事留存
```

## 开篇必备

- 热门元素在书名、简介或第一章可见。
- 组合后有新规则，不只是贴词。
- 第一章有具体冲突。
- 三章内兑现元素带来的不同体验。

## 期待来源

读者被熟悉元素的新组合吸引，期待看到不同规则下的危机、爽点和破局。

## 断档风险

- 只有概念，没有故事。
- 只换皮，不改变规则。
- 热点情报过时。
- 入口吸引强，正文留存弱。

## 训练任务

用两个热门元素组合一个题材，写出一句卖点、一个第一章冲突、一个三章内兑现点，证明它不是空概念。
