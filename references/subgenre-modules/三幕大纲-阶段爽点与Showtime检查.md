# 三幕大纲-阶段爽点与Showtime检查

依据案例：`batch-209` 案例 1045。

## 读者承诺

三幕大纲不只排事件，还要在每个阶段检查读者现在爽不爽、何时能爽。

## 核心循环

```text
三幕节点 -> 爽点标记 -> 计划失败 -> 至暗时刻 -> 主角showtime -> 新平衡
```

## 开篇必备

- 第一幕完成入局、目标、金手指。
- 第二幕安排计划与反计划。
- 中点要有倒计时、联手或脆弱暴露。
- 第三幕给主角 showtime 和下一卷线索。

## 期待来源

读者跟随目标、阻碍和钩子前进，并在至暗之后获得主角爆发式解决问题的回报。

## 断档风险

- 三幕节点只有结构，没有爽点。
- 第二幕只是拖长。
- 至暗后没有主角主动爆发。
- 结尾没有新平衡或下一卷线索。

## 训练任务

写一个 12 节点三幕大纲，每节点填目标、阻碍、爽点、钩子四项，并标注爽点类型。
