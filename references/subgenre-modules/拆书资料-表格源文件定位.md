# 拆书资料-表格源文件定位

依据案例：`batch-106` 案例 530。

## 读者承诺

学习拆书工具时，先定位原文件，不凭视频截图重造表格。表格可能嵌在 Word 文档里，准确文件名比模糊记忆更重要。

## 核心循环

```text
视频展示 -> 文件包 -> 准确文件名 -> 打开 Word -> 找到表格 -> 记录字段
```

## 开篇必备

- 视频或教程来源。
- 文件包链接。
- 准确文件名。
- 文件类型。
- 表格所在位置。

## 期待来源

- 直接复用成熟表格。
- 避免重复造轮子。
- 减少找资料时间。
- 形成资料定位习惯。

## 断档风险

- 只记得视频里有表。
- 文件包扫过但没找准确文件名。
- 把 Word 内嵌表格误判为没有。
- 凭记忆重做导致字段缺失。

## 训练任务

找一个拆书表源文件，记录文件名、文件类型、表格位置和可复用字段。下次拆书先打开源文件对照。
