# 轻小说日常-人物驱动主线

依据案例：`batch-174` 案例 867。

## 读者承诺

宅味日常和恋爱轻小说看似主线弱，但常靠人物关系、角色碰撞和感情线驱动持续事件。

## 核心循环

```text
主角人设 -> 核心角色群 -> 意外建立联系 -> 人物碰撞 -> 新角色带来新故事
```

## 开篇必备

- 主角人设要能和角色群产生反应。
- 第一章用意外连接第一个核心角色。
- 角色性格要鲜明，能自动制造事件。
- 恋爱文要把感情线当主线看。

## 期待来源

读者期待喜欢的角色继续碰撞、关系变化、新人物登场带来新事件。

## 断档风险

- 弱主线同时弱人物，日常变流水账。
- 角色只是标签，碰撞不出事件。
- 新人物只增加人数，不增加期待。
- 没有关系推进，读者追读动力下降。

## 训练任务

设计主角和 4 个核心角色，每个角色写一个和主角碰撞出的事件。第一章只写主角与第一个角色的意外联系。
