# 屑感角色-立场边界校准

依据案例：`batch-030` 案例 146。

## 读者承诺

角色可以屑，但必须让读者知道他的锋芒对准压迫者，底线不伤弱者。

## 核心循环

```text
恶劣特质 -> 对象限定 -> 价值底线 -> 反差补偿 -> 缺点变魅力
```

## 开篇必备

- 明确角色讨厌特质。
- 明确这些特质只对谁生效。
- 给出不欺弱者的证据。
- 给角色一项真实本事或担当。

## 期待来源

- 读者等他去坑更坏的人。
- 双标变成站队爽点。
- 傲慢只对上位压迫者释放。
- 小缺点让人物更真实。

## 断档风险

- 屑感无差别伤害所有人。
- 只有缺点，没有本事。
- 大是大非站错队。
- 反差只是口头设定，没有场景证明。

## 训练任务

写一个腹黑主角坑压迫者的场景，同时安排他对弱者宽容一次，证明屑感有边界。
