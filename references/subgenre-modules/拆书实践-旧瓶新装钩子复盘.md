# 拆书实践-旧瓶新装钩子复盘

依据案例：`batch-123` 案例 611。

## 读者承诺

拆书要持续实践，重点学技巧、钩子和处理方式，不照搬老套情节。旧套路能赚钱，通常赢在断章、节奏和稳定兑现。

## 核心循环

```text
连续拆章节 -> 记录章末钩子 -> 标出争议毒点 -> 区分技巧与情节 -> 回填自己的大纲
```

## 开篇必备

- 每次拆固定少量章节。
- 每章记录一个章末钩子。
- 优点和争议点都要拆。
- 老套情节只学处理，不学皮相。
- 拆完要回到自己的大纲。

## 期待来源

- 章末钩子制造追读。
- 争议情节提供避坑样本。
- 稳定拆书保持学习惯性。
- 旧瓶新装证明套路仍可用。

## 断档风险

- 只拆优点，不拆毒点。
- 学情节不学处理。
- 一次拆太多后中断。
- 不把拆书结论转回自己的大纲。

## 训练任务

连续三天拆同一本书，每天 3 章。每章写章末钩子、可学技巧、不可照搬情节各一条。
