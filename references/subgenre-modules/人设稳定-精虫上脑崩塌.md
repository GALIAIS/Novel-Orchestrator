# 人设稳定-精虫上脑崩塌

依据案例：batch-310 案例 1547

## 读者承诺

主角可以有欲望，但不能因为美女突然失控、舔狗化或做出和既有人设严重不符的决定。

## 核心循环

```text
既有人设 -> 欲望/诱惑场景 -> 行为仍符合性格 -> 关系推进 -> 读者信任保持
```

## 开篇必备

- 先明确主角对女性和欲望的基本态度。
- 诱惑场景要有符合人设的反应。
- 禁欲系只能逐步动摇，不能突然色中饿鬼。
- 好色人设也要有底线和逻辑。

## 期待来源

期待来自人物稳定。读者相信主角不会被作者强行降智，才愿意继续投入。

## 断档风险

- 见美女就走不动路。
- 禁欲系男主突然精虫上脑。
- 为了暧昧桥段牺牲主角智商。
- 女性相关情节缺少铺垫和逻辑。

## 训练任务

写一个主角面对美女诱惑的桥段，标出既有人设、合理反应、低毒行动和需要避免的崩人设动作。
