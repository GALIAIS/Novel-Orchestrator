# 电影感表达-画面对话转场

依据案例：batch-262 案例 1308

## 读者承诺

电影感表达用画面、台词、人物出场和转场承载信息，让读者不用读大段解释，也能理解目的、阻碍和解决路径。

## 核心循环

```text
画面交代局势 -> 台词交代目的 -> 人物承接阻碍 -> 动作推进结果
```

## 开篇必备

- 一个能说明局势的画面。
- 三句以内关键对话。
- 解决阻碍的人物。
- 立刻发生的动作。
- 删掉冗余分析。

## 期待来源

读者期待顺畅推进：看见、听见、理解、发生变化，而不是被作者按住解释。

## 断档风险

精练不是省信息；如果画面和台词没承载足够信息，读者会看不懂。高阶样本不能只模仿短句。

## 训练任务

把一段局势分析改成电影感场景，只保留一个画面、三句台词、一个动作，然后标注每个信息点。
