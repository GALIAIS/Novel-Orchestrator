# 自由写作-低谷热情重建

依据案例：`batch-093` 案例 462。

## 读者承诺

自由职业写作不自由，收入和成绩波动会打穿心态。恢复路径不是硬逼自己，而是重新获得情感输入，再带着短板拆书和重建大纲能力。

## 核心循环

```text
成绩落差 -> 心态崩盘 -> 情感输入 -> 找回热情 -> 回看失败作品 -> 针对短板拆书 -> 重建写作
```

## 开篇必备

- 承认收入不稳定。
- 区分开篇进步和整书成功。
- 记录心态崩盘触发点。
- 找能重新触动自己的作品。
- 拆章纲、人设、感情变化和核心梗。

## 期待来源

- 作者重新想写。
- 失败作品能看出问题。
- 情绪输入反哺女频感情线。
- 大纲从 0 到 1 重新可控。

## 断档风险

- 以为自由职业等于轻松。
- 扑街后逃避读者和作品。
- 只看干货不补情绪。
- 没有稳定收入还持续全职硬撑。

## 训练任务

写一次低谷复盘：成绩落差、逃避行为、能触动你的作品、失败作品短板、下一本要拆的 3 个维度。
