# 五折线作业-严格审题模板

依据案例：`batch-019` 案例 091。

## 读者承诺

把情绪拉扯训练压成可验收格式：五段梗概、一个以上爽点、一个虚晃一枪、下行不让主角背锅。

## 核心循环

```text
选题 -> 定目标 -> 画五折线 -> 逆推下行 -> 加虚晃 -> 查毒点
```

## 开篇必备

- 主角金手指。
- 明确事件目标。
- 对立情绪：积极在上，消极在下。
- 最后由金手指完成的爽点。

## 期待来源

- 读者知道主角有能力，但暂时被阻隔。
- 虚晃一枪让读者短暂放松，再遇到意外下行。
- 下行压力不是主角犯错，而是外部角色加锅。

## 断档风险

- 没审题，写成别的题目。
- 五段梗概写成长篇解释。
- 虚晃不明显。
- 下行靠主角蠢、逃跑、捅娄子。
- 爽点没有使用金手指完成目标。

## 训练任务

写一个五句梗概，每句只承担一条折线。完成后删掉所有解释性废话，只留下读者能看懂的事件变化。
