# 奇幻出路-热度套皮小众风险

依据案例：`batch-054` 案例 269。

## 读者承诺

传统奇幻和龙文商业风险高；若要赚钱，优先考虑热度主题套皮，把奇幻标签服务于学院、神秘、种田或身份压制等明确爽点。

## 核心循环

```text
奇幻想法 -> 判断小众风险 -> 寻找热度标签 -> 套皮转译 -> 明确核心爽点
```

## 开篇必备

- 区分正统剑与魔法和热度套皮。
- 判断龙文是否只是小众兴趣。
- 第一章必须给出商业爽点。
- 不要只靠“奇幻”标签吸引读者。
- 目标是赚钱时，要敢于换题材。

## 期待来源

- 魔法学院有成长和身份期待。
- 诡秘式神秘有未知和规则期待。
- 龙类种田有成长和建设期待。
- 热度套皮能降低冷门风险。

## 断档风险

- 正统奇幻赛道太窄。
- 龙文小众，不等于稳定赚钱。
- 套皮没有核心爽点，只会变成空标签。
- 异兽吞噬进化和龙文爽点不同，不能混用。

## 训练任务

把一个正统奇幻设定分别改成学院版、神秘版、龙类种田版，比较前三章卖点和商业风险。
