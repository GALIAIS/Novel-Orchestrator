# 起点都市开篇-事件危机替代设定堆叠

依据案例：batch-349 案例 1745

## 读者承诺

起点都市或赛博异能开篇要用事件危机承载世界观、金手指和人设，避免长篇日常与设定说明。

## 核心循环

```text
异常事件 -> 主角陷入危机 -> 世界规则显形 -> 金手指改变局势 -> 章尾生死选择
```

## 开篇必备

- 第一屏出现异常。
- 500 字内出现明确危机。
- 金手指或血统尽快发挥作用。
- 设定边行动边给。
- 书名简介、正文卖点、编辑期待对齐。

## 期待来源

读者立刻想知道主角如何活下去，以及这个能力能带来什么变化。

## 断档风险

- 7000 字日常堆设定。
- 世界危险只靠旁白说。
- 金手指卖点与简介错位。
- 主角强能力却长期底层没有解释。
- 土著主角硬塞现代吐槽。

## 训练任务

把设定型开篇压成 1500 字事件开篇：异常、危机、能力作用、生死选择四项必须全部出现。
