# 码字节律-早段打底与拼字推进

依据案例：batch-347 案例 1731

## 读者承诺

稳定产能来自固定节律、提前准备、外部推进和低成本复听，而不是每天临时靠意志力硬顶。

## 核心循环

```text
早睡早起 -> 中午前字数打底 -> 卷纲章纲降低卡文 -> 拼字房推进 -> 语音复听改文
```

## 开篇必备

- 中午前设置基础字数。
- 每卷先做大纲，每章写 3 到 6 个细节点。
- 开写前准备资料和爽点。
- 拼字工具要配合外部备份。
- 码后用间隔重读或语音复听制造陌生感。

## 期待来源

作者每天都知道先写什么、写到哪里、写完如何检查。

## 断档风险

- 熬夜赶稿导致白天崩盘。
- 没有章纲，写到现场才找方向。
- 拼字软件吞稿没有备份。
- 只追字数，不复听尴尬对白。
- 存稿不足时没有快速改文办法。

## 训练任务

执行 7 天早段打底实验，每天记录开写时间、中午前完成字数、复听后修改点。
