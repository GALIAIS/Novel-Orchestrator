# 神豪开局-上下行极限与装逼不浪费

依据案例：batch-353 案例 1765

## 读者承诺

神豪装逼打脸要先把下行压足，再让羞辱者在场承受上行反转。每一次付款、震惊、怀疑、包场、拒绝赠予都要服务情绪递进，不能让装逼机会白白流失。

## 核心循环

```text
羞辱低位 -> 危机升级 -> 系统/财富救场 -> 小打脸但对方不服 -> 二次场景彻底放大
```

## 开篇必备

- 明确的主要羞辱者。
- 进退两难的下行压力。
- 金手指救场的可视化财富。
- 羞辱者在场的态度变化。

## 期待来源

- 第一场只让对方震惊，留下不服。
- 第二场用更大金额或更强资源碾压。
- 虚晃一枪让读者期待下一次装逼。
- 现实细节可信，爽点不被常识打断。

## 断档风险

- 女配或反派提前离场，打脸对象消失。
- 下行不够，金手指来得太轻。
- 上行只付款，没有情绪反转。
- 美元、现金厚度、餐饮细节失真。

## 训练任务

写一段 1200 字神豪双场景打脸：餐厅小反转，服装店大反转。每场必须有下行、上行、对方态度变化和现实细节检查。
