# 开头多线-同一困难观众视角

依据案例：`batch-328` 案例 1639

## 读者承诺

开头双线不是讲两件事，而是用主角线解决问题，用旁观线解说、唱衰、抬高难度并震惊回收。

## 核心循环

```text
同一困难 -> 旁观线抬高难度 -> 主角线解决 -> 旁观线震惊
```

## 开篇必备

- 两条线围绕同一核心困难。
- 主角线负责行动和解决。
- 旁观线负责解说、设定、伏笔、拉情绪。
- 旁观者有不能出手的理由。
- 章末用震惊回收主角价值。

## 期待来源

读者期待大佬都解决不了的问题被主角解决，从而产生装逼爽感。

## 断档风险

- 多线各讲各的，分散注意力。
- 旁观线喧宾夺主。
- 旁观者明明能出手却不出手。
- 唱衰没有回收。
- 主角解决的问题没有被旁观线提前抬高。

## 训练任务

设计一个双线开头：主角线破局，旁观线远端唱衰，两条线必须围绕同一困难。
