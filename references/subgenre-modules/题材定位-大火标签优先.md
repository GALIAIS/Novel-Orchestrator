# 题材定位-大火标签优先

依据案例：`batch-020` 案例 098。

## 读者承诺

帮新人避开“男女频都不吃香”的混合题材。先确定平台、频段和大火标签，再设计开篇。

## 核心循环

```text
选平台 -> 选频段 -> 贴大火标签 -> 配开篇套路 -> 验证受众入口
```

## 开篇必备

- 目标读者能一眼识别的标签。
- 不公平伤害或明确困境。
- 主线目标。
- 金手指或前世优势。
- 感情线或助力关系入口。

## 期待来源

- 女频重生看虐渣复仇和关系重组。
- 男频升级看能力、资源和地位跃迁。
- 读者知道这本书会稳定兑现自己想看的东西。

## 断档风险

- 同门冒险、重生、炼药炼器、剑术天才都放进去，但没有明确受众。
- 自以为新颖，实际先筛掉读者。
- 开篇没有主线目标。
- 感情线和爽点都不够明确。

## 训练任务

把一个混合设定拆成男频版和女频版，各写一句卖点。只保留更清晰的一版，再写前三章必须出现的 5 个标签场面。
