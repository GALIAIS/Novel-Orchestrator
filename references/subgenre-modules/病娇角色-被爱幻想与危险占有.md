# 病娇角色-被爱幻想与危险占有

依据案例：batch-366 案例 1827

## 读者承诺

病娇角色卖点不是单纯失控，而是极端被爱、被唯一化和被强烈需要；危险占有必须配安全阀。

## 核心循环

```text
被爱匮乏 -> 唯一化爱意 -> 占有越界 -> 危险张力 -> 安全阀/后果
```

## 开篇必备

- 主角对病娇的特殊意义。
- 日常可爱面。
- 一次轻度越界。
- 越界后的后果或约束。

## 期待来源

- 强烈爱意反馈。
- 不可替代感。
- 可爱与危险反差。
- 关系边界被试探。

## 断档风险

- 只写疯狂，不写爱欲。
- 越界行为没有后果。
- 角色变成纯恐惧源。
- 主角被动承受，缺少关系选择。

## 训练任务

写病娇出场的两面：300 字日常可爱，300 字危险占有，再写一个读者能接受的安全阀。
