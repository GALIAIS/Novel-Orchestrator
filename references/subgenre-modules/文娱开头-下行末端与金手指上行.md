# 文娱开头-下行末端与金手指上行

依据案例：`batch-244` 案例 1219。

## 读者承诺

文娱开头可以让主角处在低谷，但必须很快展示金手指和上行方向，让读者知道爽点即将兑现。

## 核心循环

```text
低谷末端 -> 后台倒计时 -> 金手指展示 -> 对手/导师误判 -> 作品精准反打
```

## 开篇必备

- 用少量文字交代低谷。
- 让主角马上进入舞台压力。
- 金手指能解决当前问题。
- 作品选择对应反派和导师言论。

## 期待来源

读者能接受主角苦，是因为他已经走到反击路上，而不是继续被压三章。

## 断档风险

- 开头连续下行太久。
- 只换歌曲，不补情绪拉扯。
- 选秀里没有可踩对象。
- 金手指太窄，撑不住文娱多场景。

## 训练任务

写 300 字文娱开头：低谷一句带过，立刻进入后台倒计时，并让作品选择精准反打一个错误评价。
