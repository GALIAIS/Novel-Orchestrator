# 写作粒度-句段章降维打磨

依据案例：`batch-336` 案例 1676

## 读者承诺

写作从章降到段、再降到句，是控制力细化；每个句段都要有明确功能，最后再回到章节推进。

## 核心循环

```text
章节目的 -> 段落功能 -> 句子功能 -> 删除空转 -> 回看整章节奏
```

## 开篇必备

- 先知道本章目的。
- 每段有信息、动作或情绪落点。
- 每句能标出功能。
- 打磨句子不脱离章节节奏。
- 慢写后要恢复推进。

## 期待来源

作者期待把“写不顺”拆成具体句段问题，而不是靠状态硬顶。

## 断档风险

- 只按章粗写，段落失焦。
- 只抠句子，章节停滞。
- 用情绪听歌替代结构判断。
- 反复删除但不定位功能。
- 把写慢误判为退步。

## 训练任务

选一个卡住的段落，逐句标注信息、动作、情绪、转场或伏笔，删掉没有功能的句子。
