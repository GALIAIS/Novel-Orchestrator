# 同人金手指-原著互动与技能新用法

依据案例：batch-369 案例 1845

## 读者承诺

同人金手指要绑定原著角色、技能和互动代价，让熟悉能力产生新用法，而不是只给主角刷资源。

## 核心循环

```text
原著角色技能 -> 借用/交易/收集 -> 付出代价 -> 新用法 -> 反转收益
```

## 开篇必备

- 原著熟悉能力。
- 互动对象。
- 借用代价。
- 不同于原著的新用法。

## 期待来源

- 熟悉角色被重新激活。
- 熟悉技能有新脑洞。
- 代价制造互动。
- 新用法带来反转。

## 断档风险

- 同人题材和金手指分开设计。
- 签到过于烂大街。
- 情绪值商城不新鲜。
- 金手指只刷数值，不牵动原著人物。

## 训练任务

列 5 个原著角色技能，每个写一个借用代价和一个不同于原著的新用法。
