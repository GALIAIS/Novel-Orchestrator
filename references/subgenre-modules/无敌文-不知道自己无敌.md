# 无敌文-不知道自己无敌

依据案例：batch-296 案例 1477

## 读者承诺

通过主角认知错位解释他不称霸、不乱杀、不主动统治，同时让读者享受外界过度解读的反差。

## 核心循环

```text
主角实际无敌 -> 主角自认普通 -> 外界脑补升级 -> 反差兑现 -> 新误会/新矛盾接上
```

## 开篇必备

- 主角真实强度与自我认知明显错位。
- 外界有足够理由误读主角行为。
- 第一次反差能当场兑现。
- 埋下设定红利耗尽后的变化机制。

## 期待来源

- 读者比主角更知道真相。
- 外界越认真，反差越好笑或越爽。
- 主角不称霸的动机天然合理。

## 断档风险

- 一直重复“主角不知道”，没有新信息。
- 设定新鲜感耗尽后没有矛盾网。
- 强行升级打 boss，偏离纯无敌文承诺。

## 训练任务

写三次认知错位：同一个普通动作，分别被村民、宗门、国家误解成越来越高层级的威慑。
