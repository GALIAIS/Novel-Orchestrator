# 支线占比-主线参与校准

依据案例：`batch-029` 案例 145。

## 读者承诺

支线可以存在，但必须服务核心卖点、让主角参与或受影响，不能长期挤掉主线。

## 核心循环

```text
核心卖点 -> 主线目标 -> 支线短目标 -> 主角参与 -> 推进主线 -> 回收成果
```

## 开篇必备

- 明确整本书卖点。
- 区分长期目标和短期目标。
- 标注支线是否由主线导致。
- 控制无主角支线比例。

## 期待来源

- 支线调节主线节奏。
- 支线提供资源、情报、关系或世界观。
- 主角通过支线得到主线收益。
- 读者始终知道自己在看哪本书。

## 断档风险

- 支线写得像另一条主线。
- 长时间没有主角。
- 支线不推进核心卖点。
- 支线占比过高引发读者抗议。

## 训练任务

给前 30 章标色：主线、主线相关支线、脱离主线支线。脱离主线支线超过三分之一时，必须删改。
