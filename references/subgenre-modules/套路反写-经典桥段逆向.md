# 套路反写-经典桥段逆向

依据案例：`batch-137` 案例 682。

## 读者承诺

利用读者对经典桥段的熟悉预期，反向改写关键动作，制造更准确或更新鲜的情绪。

## 核心循环

```text
经典套路 -> 读者预判 -> 反向动作 -> 新情绪 -> 桥段翻新
```

## 开篇必备

- 知道原套路触发什么预期。
- 反写要服务人物状态。
- 新情绪比原套路更准。
- 不只是恶搞。
- 反差要读者能识别。

## 期待来源

- 熟悉套路带来预判。
- 反写打破惯性。
- 情绪表达更有层次。
- 老桥段变新鲜。

## 断档风险

- 为反而反。
- 反写后情绪不成立。
- 只做段子，剧情无用。
- 读者没识别原套路。

## 训练任务

选 5 个影视老套路，每个写一个反写版本，并说明原套路情绪和反写后情绪。
