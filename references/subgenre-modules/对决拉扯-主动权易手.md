# 对决拉扯-主动权易手

依据案例：`batch-017` 案例 083。

## 读者承诺

一波三折的对决，不是反派一路破防，而是主动权反复易手，让读者安全感和危机感交替变化。

## 核心循环

```text
反派优势 -> 主角抢势 -> 反派反制 -> 主角底牌失效 -> 陷阱反杀
```

## 开篇必备

- 开局明确谁占优。
- 每段都要改变局势。
- 主角至少一次看似失去主动权。
- 最终胜利要解释前面的布局。

## 期待来源

- 反派轻敌给主角机会。
- 主角底牌初次见效。
- 反派化解底牌。
- 主角利用反派反制反设陷阱。

## 断档风险

- 主角全程优势，反派像小丑。
- 只有反派情绪下行，没有局势变化。
- 底牌无风险，读者不紧张。
- 最终反杀没有前置铺垫。

## 训练任务

写一场 4 段对决，主动权至少易手 3 次，并让主角一个底牌看似失效。
