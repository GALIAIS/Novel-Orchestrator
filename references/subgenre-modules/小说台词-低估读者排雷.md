# 小说台词-低估读者排雷

依据案例：`batch-044` 案例 219。

## 读者承诺

小说台词不能把读者当成需要被反复解释的人。高级感来自潜台词、关系压力和信息缺口，不是把设定和情绪说满。

## 核心循环

```text
直白台词 -> 功能检查 -> 删除解释 -> 补潜台词 -> 关系推进
```

## 开篇必备

- 台词有目的。
- 台词服务关系或冲突。
- 设定说明尽量藏进行动。
- 情绪用反应表达。

## 期待来源

- 读者能从缝隙里理解角色。
- 角色话里有话。
- 沉默、动作、反问产生压力。
- 对话推进事件而不是停下来讲解。

## 断档风险

- 台词重复读者已知信息。
- 角色替作者解释设定。
- 情绪全靠直说。
- 没有潜台词和关系变化。

## 训练任务

删改一段 20 句对话：删除解释句和直说情绪句，用动作、反问、沉默补足压力。
