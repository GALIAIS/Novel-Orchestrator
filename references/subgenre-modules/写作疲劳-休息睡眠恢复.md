# 写作疲劳-休息睡眠恢复

依据案例：`batch-314` 案例 1570

## 读者承诺

当细纲清楚、剧情也知道怎么走，但身体停机、盯屏发呆时，优先按疲劳处理，而不是立刻怀疑大纲和能力。

## 核心循环

```text
长时间码字 -> 注意力掉线 -> 识别睡眠/疲劳 -> 停止硬撑 -> 休息睡觉 -> 恢复后小段重启
```

## 开篇必备

- 先确认睡眠和连续码字时长。
- 区分情节卡壳和身体疲劳。
- 疲劳时先休息，不强行改纲。
- 恢复后从小段落重启。

## 期待来源

作者期待保护长期产能。睡眠恢复比硬挤当天字数更能维持连载。

## 断档风险

- 把疲劳误判成灵感枯竭。
- 睡眠不足还硬撑。
- 靠咖啡因替代休息。
- 疲劳状态下大改大纲。

## 训练任务

做一张盯屏发呆自查卡：睡眠、连续码字时长、是否知道下一段、是否身体疲劳、立即休息动作。
