# 综漫副本-基调防偏航

依据案例：`batch-141` 案例 701。

## 读者承诺

综漫或无限流副本必须持续兑现书名、分类、简介和主角能力承诺，不能因为作者无大纲乱跑副本，把作品写成另一个类型。

## 核心循环

```text
确定主线基调 -> 筛选候选副本 -> 检查副本是否放大主角能力 -> 副本结束推进主线 -> 淘汰偏航副本
```

## 开篇必备

- 书名承诺。
- 分类和标签承诺。
- 主角核心能力。
- 第一阶段主线。
- 副本选择边界。

## 期待来源

- 读者想看核心能力在不同二次元世界发挥。
- 每个副本都加深主线，而不是只换地图。
- 副本差异带来新鲜感。
- 主角实力恢复后仍有新目标牵引。

## 断档风险

- 没大纲，想到哪写到哪。
- 后期副本短且不深挖。
- 热门玄幻副本破坏二次元基调。
- 主角实力恢复后失去方向。

## 训练任务

列 10 个候选副本，逐个判断是否符合书名承诺、是否放大主角能力、是否会偏成别的类型。删掉 3 个最容易偏航的副本。
