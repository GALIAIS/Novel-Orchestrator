# 生活化恐惧-不可暴力破解

依据案例：`batch-161` 案例 805。

## 读者承诺

用贴近日常、暴力无效、认知混淆的方式制造恐惧，让读者觉得异常可能发生在自己身边且无法简单解决。

## 核心循环

```text
熟悉场景 -> 异常入侵 -> 常规手段失效 -> 规则不明 -> 认知混淆 -> 恐惧升级
```

## 开篇必备

- 场景要足够日常。
- 先给读者熟悉动作，再放入异常。
- 武器或常识必须失效。
- 规则要不完整、不可信或带代价。

## 期待来源

读者期待主角能理解怪物，但怪物越不可杀、不可交流、不可理解，恐惧越强。

## 断档风险

- 怪物能被火力解决。
- 场景离读者太远，代入弱。
- 规则太清楚，变成解谜游戏。
- 主角太强，恐惧无处承载。

## 训练任务

写一个日常地点恐怖开篇，必须包含：熟悉动作、异常细节、一次失败的常规解决、一个不可信规则、一个比主角更弱的受害者。
