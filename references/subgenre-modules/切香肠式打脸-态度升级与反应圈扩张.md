# 切香肠式打脸-态度升级与反应圈扩张

依据案例：`batch-411` 案例 2053

## 读者承诺

把一个公开清算拆成态度、反应圈和代价三条升级轴；每阶段先兑现局部态度或反应，又把主角推向更明确、更公开且不可撤回的选择，最后以真实后果完成状态改变。

## 核心循环

```text
具体不公与目标铺垫 -> 流程小回报 -> 第一次损失 -> 隐性态度 -> 外围反应扩散 -> 第二次刺激 -> 公开拒绝跨过门槛 -> 核心权力回应 -> 资源关系后果进入主线
```

## 开篇必备

- 被扭曲前的规则价值。
- 操纵证据与主角可失去目标。
- 态度、反应圈、代价三条升级轴。
- 与主线相关的流程小回报。
- 不可逆选择及下一章首个反应。
- 爆发后的资源、关系和身份后果。

## 期待来源

主角先以隐性动作给一口释放，外围人物察觉压力；刺激再次升级后，他公开拒绝并承担代价，读者继续等待核心权力与公众无法回避的回应。

## 断档风险

- 同一句立场换方式重复多次。
- 不公只有标签，没有规则和损失。
- 主角没有可失去东西却随意耍酷。
- 流程小甜点与主线蓄压无关。
- 旁观反应始终停留在边缘人物。
- 断章后重复回放，不立即兑现后果。

## 训练任务

为公开会议画态度、反应圈和代价三条五级轴，写 1800 字使每 400 字至少推进一轴；停在不可逆选择后，再用下一章前 300 字立即兑现核心反应和一项资源后果。
