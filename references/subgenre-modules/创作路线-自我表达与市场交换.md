# 创作路线-自我表达与市场交换

依据案例：batch-385 案例 1922

## 读者承诺

为爱写作和市场创作可以兼顾，但必须明确当期首要目标、目标读者、作品亮点和交换规则，并用真实反馈定位短板。

## 核心循环

```text
明确目的 -> 接受交换规则 -> 突出目标读者亮点 -> 完成作品 -> 反馈定位短板
```

## 开篇必备

- 不可放弃的创作核心。
- 首要目标和目标读者。
- 前三章可感知亮点。
- 最优先解决的单一短板。

## 期待来源

- 喜欢的内容被更好地表达。
- 读者快速看见继续理由。
- 小完成和反馈恢复行动感。
- 技术提升逐步扩大受众。

## 断档风险

- 想赚钱却要求读者降低标准。
- 渴望反馈又否定不理解的读者。
- 为所有人修改导致定位消失。
- 用行业吐槽替代短板诊断。

## 训练任务

写一份三个月路线声明：首要目标、创作核心、目标读者、前三章亮点和唯一优先短板，期末按完成量与反馈复盘。
