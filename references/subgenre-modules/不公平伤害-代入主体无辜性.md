# 不公平伤害-代入主体无辜性

依据案例：batch-269 案例 1344

## 读者承诺

写不公平伤害前，先确保被伤害者值得代入：无辜、有尊严、有翻身可能，而不是又脏又弱又蠢。

## 核心循环

```text
主角无辜 -> 亲密背叛/秩序倒置 -> 尊严受损 -> 反派责任明确 -> 报复/补偿承诺
```

## 开篇必备

- 清掉主角劝退标签。
- 反派主动恶意要明确。
- 主角受损要伤到尊严和利益。
- 快速承诺翻身、报复或更优关系补偿。

## 期待来源

期待来自读者觉得“这不该发生在他身上”。主角越值得，伤害越不公平，反击越有爽感。

## 断档风险

- 主角自己也有明显道德问题。
- 主角弱到让读者鄙视。
- 现实狗血新闻直接搬成主角开篇。
- 反派责任不清，读者不站队。

## 训练任务

把一个狗血新闻改成主角开篇。先列原始劝退标签，再逐条改成无辜性和反派恶意。
