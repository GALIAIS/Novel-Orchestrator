# 全球流缩小-熟悉载体降复杂

依据案例：`batch-096` 案例 479。

## 读者承诺

全球流、创世流、文明培养这类题材很容易被庞大前置设定拖死。更稳的做法是把超大范围缩小到读者熟悉的载体里，如超级英雄、校园、城市、名人圈，让大影响用小场景先兑现。

## 核心循环

```text
大范围承诺 -> 前置复杂过高 -> 选择熟悉载体 -> 缩小第一舞台 -> 影响关键人物 -> 逐步扩大世界
```

## 开篇必备

- 第一章只写一个小舞台，不开全球设定大会。
- 借读者已有认知降低说明成本。
- 主角先影响一个关键人物或组织。
- 保留全球感的远景，但不抢第一事件。

## 期待来源

- 小场景撬动大世界的杠杆感。
- 熟悉角色/职业/城市带来的低门槛。
- 影响关键人物后的连锁反馈。
- 从局部到全球的扩张期待。

## 断档风险

- 一开篇就解释文明、神灵、全球格局。
- 没有熟悉载体，读者认知成本过高。
- 只缩小范围，却丢掉大承诺。
- 同人和原创边界混淆，错估读者基础。

## 训练任务

把一个全球流脑洞压缩成“一个城市的一件事”。写清一个地点、一个关键人物、一次影响、一个远景伏笔。正文不超过 800 字。
