# 误读喜剧-中二脑补与日常揭底

依据案例：`batch-394` 案例 1969

## 读者承诺

用同一动作的两套解释制造短促笑点：观察者把暧昧日常行为脑补成宏大中二事件，当事人再以极普通的现实原因揭底。

## 核心循环

```text
暧昧日常动作 -> 角色滤镜触发误读 -> 中二推演升级 -> 观察者采取行动 -> 现实原因短句揭底 -> 关系或场景留下后果
```

## 开篇必备

- 可见动作先于解释出现。
- 动作同时容纳现实与夸张解释。
- 观察者有稳定、可辨认的思维滤镜。
- 脑补内部自洽并逐级抬高预期。
- 揭底短、具体且平静。
- 笑点至少塑造人物或改变场景。

## 期待来源

读者既理解误读为何成立，又等待宏大解释怎样被现实击穿；长期期待来自滤镜不变、误读领域与后果持续变化。

## 断档风险

- 无动作证据就突然脑补。
- 现实答案与前置动作对不上。
- 每次都用同一危险误判和同一揭底节奏。
- 揭底解释过长，落差被说明耗尽。
- 把即时误读机械写成所有人降智吹捧。
- 关系发展后角色仍毫无学习。

## 训练任务

分别以战斗、阴谋、恋爱、超自然和身份为误读领域写五组桥段，每个揭底不超过十五字；再让其中一场误读引发真实行动，证明笑点可以进入剧情因果。
