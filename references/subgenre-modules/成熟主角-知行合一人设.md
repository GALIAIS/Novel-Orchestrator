# 成熟主角-知行合一人设

依据案例：`batch-115` 案例 575。

## 读者承诺

成熟主角的爽感来自稳定信念和高执行力。敌人诡辩、威胁、嘴炮都不能让主角反复破防；主角用选择和行动证明人设。

## 核心循环

```text
敌人诡辩 -> 主角不动摇 -> 直接行动 -> 行动兑现信念 -> 读者认同人设 -> 爽感稳定
```

## 开篇必备

- 主角有明确核心信念。
- 第一场冲突能让信念接受测试。
- 敌人有诡辩或威胁。
- 主角少废话，用行动回应。
- 行动后让旁观者或结果证明人设。

## 期待来源

- 读者知道主角不会被嘴炮带偏。
- 稳定信念带来安全感。
- 高执行减少内耗。
- 每次选择都强化人设。

## 断档风险

- 主角成长半本还在犹豫。
- 用台词宣称人设，不用行动证明。
- 敌人嘴炮轻易说破防。
- 成熟人设写成无脑莽。

## 训练任务

写一场 800 字冲突：敌人用道理和威胁拖住主角，主角最多两句话，然后用行动兑现自己的核心信念。
