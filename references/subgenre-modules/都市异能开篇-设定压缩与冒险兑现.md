# 都市异能开篇-设定压缩与冒险兑现

依据案例：`batch-215` 案例 1072。

## 读者承诺

都市异能开篇要让金手指在事件、危机和奖励里被理解，而不是先用大段设定解释。

## 核心循环

```text
生活事件 -> 金手指触发 -> 兑换能力 -> 冒险危机 -> 翻盘脱险 -> 奖励回收
```

## 开篇必备

- 用生活事件触发金手指。
- 第一章结尾给出冒险方向或明确危机。
- 第二、三章展示能力功效和收益。
- 伏笔延后，不压过当下爽点。

## 期待来源

读者看到主角每次选择都能带出能力、危机、装逼和奖励，设定自然变成追读动力。

## 断档风险

- 前两章连续解释世界观。
- 隐藏危机绕太远，读者没有压迫感。
- 打脸装逼刻意，冲突不够强。
- 金手指只被介绍，没有解决问题。

## 训练任务

把一个设定介绍型开头改成三章事件流：触发积分、兑换能力、冒险翻盘，并列出每章删除或延后的设定。
