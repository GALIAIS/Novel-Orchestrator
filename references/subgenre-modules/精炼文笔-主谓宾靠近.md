# 精炼文笔-主谓宾靠近

依据案例：batch-274 案例 1368

## 读者承诺

通过删赘词、减限定、改主动、换具体动词，让读者快速知道谁做了什么，并保留必要画面感。

## 核心循环

```text
臃肿句子 -> 删除赘词 -> 主谓宾靠近 -> 动词具体化 -> 阅读变顺
```

## 开篇必备

- 删除口头语和评书用语。
- 减少名词前堆叠限定词。
- 泛动作词换成具体动作。

## 期待来源

读者期待顺滑、清楚、不断句。文笔精炼的结果不是华丽，而是阅读阻力低。

## 断档风险

- 形容词堆太多，主语宾语距离太远。
- 否定句、被动句拖慢节奏。
- 连词和重复情绪词让句子发虚。

## 训练任务

拿 500 字正文逐句问：谁做了什么？这个词是否让画面更清楚？不能回答就删或改。
