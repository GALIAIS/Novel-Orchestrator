# 情绪下行-激励事件多样化

依据案例：batch-285 案例 1425

## 读者承诺

情绪下行不必每次靠意外触发；激励事件可以是选择、反制、代价、资源变化或规则变化。

## 核心循环

```text
计划推进 -> 激励事件触发 -> 大局变难 -> 主角个人上行 -> 反制期待
```

## 开篇必备

- 当前计划。
- 下行触发源。
- 大局变难的具体表现。
- 主角上行动作。
- 与前一次下行的差异。

## 期待来源

读者想看局势变难后主角如何调整，而不是机械地被一连串意外砸中。

## 断档风险

- 把术语当硬规则。
- 每次下行都是意外，形成疲劳。
- 整体下行时主角也跟着无能。
- 普通冲突没有转成行动压力。

## 训练任务

同一章写五种下行源：意外、反制、选择代价、资源耗尽、规则变化；每种都给主角一个上行动作。
