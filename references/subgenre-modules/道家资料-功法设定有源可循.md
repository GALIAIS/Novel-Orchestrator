# 道家资料-功法设定有源可循

依据案例：`batch-110` 案例 547。

## 读者承诺

仙侠和道门设定需要有气质来源。查道家典籍不是为了照抄古文，而是让功法、咒法、仪式、门派和禁忌有可追溯的质感。

## 核心循环

```text
查典籍 -> 提取术语 -> 改造设定 -> 白话解释 -> 放入剧情
```

## 开篇必备

- 资料来源清楚。
- 只提取气质和名目。
- 改造成原创功法或规则。
- 给读者白话解释。
- 设定必须服务剧情压力。

## 期待来源

- 功法名有道门味。
- 世界观更厚实。
- 仪式和禁忌更可信。
- 读者感到设定不是凭空编的。

## 断档风险

- 直接堆古奥术语。
- 照抄原文导致难读。
- 资料量太大，不筛选。
- 设定只好看，不影响剧情。

## 训练任务

从道家典籍目录选 5 个词，分别改造成一个功法、法器、仪式、禁忌、门派名，并各写一句白话功能。
