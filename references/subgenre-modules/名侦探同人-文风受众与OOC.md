# 名侦探同人-文风受众与OOC

依据案例：`batch-060` 案例 297。

## 读者承诺

这个模块用于福尔摩斯、名侦探、推理名人同人或类同人。它承诺的是先守住原型人物的读者认知，再判断文风是否服务目标平台和目标受众。

## 核心循环

```text
原型公共认知 -> 平台文风校准 -> 不 OOC 边界 -> 第一场智力压制 -> 神秘/超自然设定适配
```

## 开篇必备

- 写出读者默认相信这个人物强在哪里。
- 第一单元要展示推理或智力压制。
- 神秘力量不能破坏侦探核心。
- 文风要和目标平台读者匹配。
- 非受众吐槽只能参考，不能全盘接收。

## 期待来源

- 读者期待熟悉人物在新世界继续强。
- 名侦探用细节压制对手。
- 神秘学给推理增加新变量。
- 不 OOC 带来同人安全感。
- 文风稳定让读者不出戏。

## 断档风险

- 只借福尔摩斯名字，不展示推理能力。
- 神秘世界里侦探显得迟钝。
- 日轻文风放错平台。
- 近期阅读同化导致没有自身文风。
- 推理难度超出作者掌控。

## 训练任务

写一场名侦探同人开篇，必须包含三个观察细节、一个错误嫌疑、一次反向压制。写完检查原型粉是否会觉得 OOC。
