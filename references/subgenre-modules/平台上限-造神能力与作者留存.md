# 平台上限-造神能力与作者留存

依据案例：`batch-216` 案例 1077。

## 读者承诺

判断平台时，不只看短期收益，还要看读者池、作者留存、行业地位和造神能力。

## 核心循环

```text
历史积累 -> 顶级作者留存 -> 庞大读者池 -> 行业地位 -> 破圈上限
```

## 开篇必备

- 明确作品目标是赚钱、成长还是破圈。
- 区分平台短期收益和长期上限。
- 观察头部作者为什么留下。
- 记录平台已有造神案例。

## 期待来源

作者知道平台选择会影响作品天花板，不再只按传闻或单一收益指标决策。

## 断档风险

- 把所有平台目标混为一谈。
- 只看买断或短期流量。
- 忽略头部作者聚集效应。
- 用少量评论当统计数据。

## 训练任务

对 3 个平台列读者池、头部作者、破圈案例、签约路径、收益方式和题材适配，分别判断赚钱目标与破圈目标。
