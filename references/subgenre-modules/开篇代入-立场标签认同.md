# 开篇代入-立场标签认同

依据案例：`batch-010` 案例 050。

## 读者承诺

开篇第一时间让目标读者觉得熟悉、认同、站队或舒适，从而愿意给后续剧情更多耐心。

## 核心循环

```text
目标读者 -> 熟悉处境/标签 -> 立场认同 -> 情绪波动 -> 进入剧情
```

## 开篇必备

- 明确目标读者。
- 选择他们熟悉的处境或标签。
- 立场服务题材受众。
- 一两笔完成，不抢剧情。

## 期待来源

- 读者觉得“这不就是我懂的东西”。
- 主角和读者站在同一阵营。
- 熟悉印象后接反差，更容易吊胃口。
- 立场筛选让目标读者更纯。

## 断档风险

- 想讨好所有读者。
- 立场误伤目标受众。
- 标签印象变成刻板冒犯。
- 代入技巧盖过剧情精彩。

## 训练任务

同一开篇写三版第一段：熟悉处境、标签印象、立场吐槽。选最能筛中目标读者的一版。
