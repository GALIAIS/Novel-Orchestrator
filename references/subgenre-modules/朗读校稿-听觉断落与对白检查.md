# 朗读校稿-听觉断落与对白检查

依据案例：batch-361 案例 1801

## 读者承诺

把写完的章节听一遍，用听觉发现肉眼漏掉的断落、跳脱、拗口和不自然对白，让读者读起来更顺。

## 核心循环

```text
导入朗读工具 -> 听出卡顿 -> 标记断句/转场/对白 -> 修改 -> 复听
```

## 开篇必备

- 选一个自然度足够的朗读声音。
- 边听边标问题，不边听边大改。
- 重点听开头、转场、对话和情绪段。

## 期待来源

- 句子更顺。
- 段落衔接更自然。
- 对话更像人说话。
- 情绪不突然断。

## 断档风险

- 语音太机械导致误判。
- 只听不标位置。
- 把朗读当替代判断。
- 改完不复听。

## 训练任务

听一章，只标拗口句、跳脱段、情绪断点三类问题。改完后复听一次。
