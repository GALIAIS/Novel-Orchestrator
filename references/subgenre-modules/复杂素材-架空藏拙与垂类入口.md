# 复杂素材-架空藏拙与垂类入口

依据案例：`batch-236` 案例 1176。

## 读者承诺

复杂专业素材要用垂类入口快速入门，同时对不懂处架空藏拙，避免考据卡死正文。

## 核心循环

```text
资料需求 -> 找垂类入口 -> 提取最低可用知识 -> 架空藏拙 -> 长期积累
```

## 开篇必备

- 明确最低可用知识。
- 找视频、图鉴、设定库等快速入口。
- 不懂的专业坑少写或模糊写。
- 资料服务剧情，不拖死写作。

## 期待来源

读者获得足够的专业感和幻想质感，作者不被炼金术、宗教、枪械等资料门槛卡住。

## 断档风险

- 只靠百科查资料。
- 硬装懂专业内容。
- 为资料停写正文。
- 不会藏拙，细节互相矛盾。

## 训练任务

做三层资料表：一句话入门、三个可直接用设定、三个暂时不写的专业坑。正文只用前两层。
