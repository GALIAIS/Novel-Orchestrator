# 开篇双型-逃脱监狱与无敌起飞

依据案例：`batch-099` 案例 495。

## 读者承诺

常见有效开篇有两类：逃脱监狱式，用下行末端和生死危机启动；无敌起飞式，用低冲突场景里的期望-满足链启动。两者都要让读者尽快看到上行。

## 核心循环

```text
逃脱监狱：绝境 -> 金手指可用 -> 破局 -> 上行
无敌起飞：期望 -> 满足 -> 新期望 -> 连续满足
```

## 开篇必备

- 逃脱式必须有当前危机。
- 金手指必须能立刻解决当前问题。
- 无敌式必须有连续小期待。
- 无敌最好是单领域无敌，不是全方位无敌。

## 期待来源

- 绝境反弹的爽感。
- 无敌碾压的轻松满足。
- 单领域无敌与其他领域成长并存。
- 身边人和群像承担阻力。

## 断档风险

- 逃脱式危机太慢，金手指不能用。
- 无敌式没有期待链，只剩平淡日常。
- 主角全能无阻力。
- 群像阻力变成读者不关心的配角成长。

## 训练任务

给同一个创意写两版开篇卡：逃脱监狱式写危机、金手指、破局；无敌起飞式写三组期望-满足链和一个仍需升级的领域。
