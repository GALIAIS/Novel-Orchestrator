# 修仙驱动-怕死到道心升级

依据案例：`batch-058` 案例 288。

## 读者承诺

这个模块用于升级、修仙、长生文的主角驱动力设计。它承诺的是让“怕死求生”在中后期升级为责任、守护、道心或归宿，避免主角越强越显得自私空洞。

## 核心循环

```text
弱小时求生 -> 变强获得安全感 -> 关系/责任出现 -> 主动承担风险 -> 道心或归宿成型 -> 长生意义升级
```

## 开篇必备

- 前期可以用活命压力启动。
- 早早埋下主角在乎的人、秩序或理想。
- 升级收益不只写自保。
- 中期要出现主角主动承担风险的节点。
- 长生要有精神含义，不只是怕死。

## 期待来源

- 读者理解主角为什么必须变强。
- 主角的安全感缺口不断被补偿。
- 关系和责任让主角不只是自保。
- 更高世界压迫带来长线升级。
- 道心让后期目标不空。

## 断档风险

- 大佬阶段仍只说“我要自保”。
- 主角为活命牺牲所有人，显得自私肤浅。
- 只有外挂、丹药、血脉，没有人格成长。
- 女性角色只做奖励或障碍。
- 永无止境升级但没有归宿。

## 训练任务

给主角写三次“可以逃但选择留下”的事件。第一次为亲近关系，第二次为所在秩序，第三次为道心或归宿。
