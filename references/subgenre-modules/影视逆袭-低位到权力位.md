# 影视逆袭-低位到权力位

依据案例：batch-286 案例 1427

## 读者承诺

低位受害者到权力位置的路径，靠强不公平伤害和高位结果望远镜持续拉情绪与期待。

## 核心循环

```text
身份跌落 -> 生存适应 -> 获得支持 -> 争夺话语权 -> 权力位置兑现
```

## 开篇必备

- 原本身份。
- 被伤害方式。
- 最低位置。
- 未来高位结果。
- 第一批支持者或资源。

## 期待来源

读者想看主角如何从被剥夺、被污名的位置，一步步获得尊严、资源和代表权。

## 断档风险

- 只写惨，不给上行望远镜。
- 高位结果太空，没有中间台阶。
- 群体支持来得无因。
- 逆袭只靠口号，不靠事件。

## 训练任务

写一个“从受害者到代表者”的三步纲：最低点、生存能力、第一次代表群体发声。
