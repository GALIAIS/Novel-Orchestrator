# 废土种田-建设解压发展执念

依据案例：`batch-054` 案例 268。

## 读者承诺

用荒芜到富足的建设上行满足读者解压、土地情结和发展执念；废土、第四天灾、群像都可以服务种田流核心。

## 核心循环

```text
荒芜困境 -> 开始建设 -> 资源改善 -> 秩序扩张 -> 群体富足 -> 新发展目标
```

## 开篇必备

- 先让读者看见荒芜和缺口。
- 种田主体要有行动热情。
- 第一项建设要快速见效。
- 群体收益要可视化。
- 防御、粮食、经济、科技至少开一条线。

## 期待来源

- 土地变好带来解压感。
- 物资增加带来富足感。
- 群体一起建设带来参与感。
- 根据地发展带来家国情绪。

## 断档风险

- 只有建设清单，没有危机和收益。
- 发展太顺，会失去冲突。
- 第四天灾只玩梗，不承担建设功能。
- 种田线不回收成生活改善，爽点会虚。

## 训练任务

设计废土种田前三章：荒芜危机、第一项建设、群体收益、下一步发展目标各写一段。
