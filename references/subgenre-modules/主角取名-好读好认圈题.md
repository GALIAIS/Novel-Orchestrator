# 主角取名-好读好认圈题

依据案例：`batch-090` 案例 449。

## 读者承诺

主角名先要好读好认，再考虑意境、谐音、设定暗示和独特性。名字是代入入口，不是炫技点。

## 核心循环

```text
好读好认 -> 排除生僻奇葩 -> 绑定主线/人设 -> 生成候选 -> 读者记住
```

## 开篇必备

- 不用生僻字。
- 不用奇葩到出戏的组合。
- 名字最好顺口。
- 可带主线谐音或题材暗示。
- 来源可以是诗词、现实昵称、音译、生成器。

## 期待来源

- 读者第一眼能读。
- 读者第二次能记住。
- 名字和主线、人设有轻微关联。
- 不打断代入感。

## 断档风险

- 为独特牺牲识别。
- 生僻字太多。
- 音译不像人名。
- 谐音梗太硬。

## 训练任务

列 20 个候选名，删掉难读、生僻、奇葩项；从剩余名字里选 3 个有主线或人设暗示的，朗读测试顺口度。
