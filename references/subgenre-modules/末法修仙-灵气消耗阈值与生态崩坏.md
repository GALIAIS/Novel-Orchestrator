# 末法修仙-灵气消耗阈值与生态崩坏

依据案例：`batch-395` 案例 1973

## 读者承诺

把修炼升级和废土求生锁在同一条资源因果上：旧修炼体系越成功，灵气存量和生态越接近阈值崩坏，主角必须建立有成本的替代路线。

## 核心循环

```text
修士与制度持续抽取灵气 -> 阈值灾害破坏生存资源 -> 旧修炼办法失效 -> 主角用低外部依赖传承破局 -> 替代成本引发新争夺 -> 资源秩序重组
```

## 开篇必备

- 灵气有明确来源、去向和再生速度。
- 第一个阈值有生活化先兆。
- 主角立即面对食水、避难或修炼资源危机。
- 旧体系在事件中真实失效。
- 另类传承降低一种依赖并增加另一种成本。
- 第一次升级直接改善生存处境。

## 期待来源

短线看主角如何活过下一次资源断裂，长线看他能否用替代修炼路线改写耗尽世界的文明制度。

## 断档风险

- 只把废土当视觉背景。
- 灵气枯竭没有收支和阈值规则。
- 高阶修士仍可无限挥霍力量。
- 金手指免费取消全部生存压力。
- 升级与求生分成两部互不相干的小说。
- 世界终局不回应资源制度。

## 训练任务

制作灵气收支表，列三种来源、五种消耗、两个阈值及生活后果；再写“灾害显影—旧法失效—替代法有代价破局”三章，确保修炼行为本身正在改变末世走向。
