# 冲突缓冲-支线爽点扩展

依据案例：`batch-149` 案例 743。

## 读者承诺

强冲突结束后，不急着立刻让更强敌人登场，而是用短支线、轻松笑点、扮猪吃虎或收获扩展，让读者缓一口气，再进入下一轮主线升级。

## 核心循环

```text
主线冲突释放 -> 获得收获 -> 短支线扩展爽点 -> 情绪缓冲 -> 老敌登场 -> 主线再升级
```

## 开篇必备

- 前一场冲突的明确收获。
- 缓冲支线的轻口味爽点。
- 支线自己的小矛盾。
- 篇幅短于前一场主线冲突。
- 回到“老的来了”的钩子。

## 期待来源

读者期待看到前段战利品如何变成金钱、实力、地位、道具、宠物或感情反馈，同时等下一轮更大冲突登场。

## 断档风险

- 缓冲支线过长，读者忘记主线。
- 收获没有变成可见爽点。
- 连续打斗不换口味，情绪疲劳。
- 支线升级和主线混在一起，节奏不清。
- 老敌登场缺少回钩。

## 训练任务

为一场主线战斗设计 3 章缓冲：第 1 章展示收获，第 2 章用收获无意装逼，第 3 章用老敌消息收尾。标注每章的情绪口味。
