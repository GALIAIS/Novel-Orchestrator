# 微观身体流-创意要接外部爽点

依据案例：`batch-334` 案例 1669

## 读者承诺

微观身体流的创意能吸引人，但必须接外部危机和主角爽点，不能只展示体内奇观。

## 核心循环

```text
外部生命危机 -> 主角缩小进入体内 -> 细胞队伍作战 -> 微观破局 -> 外界生命被救回
```

## 开篇必备

- 外部危机明确。
- 主角能力快速触发。
- 体内任务服务现实结果。
- 主角带队改变环境。
- 不服务救援的伏笔后置。

## 期待来源

读者期待主角用新奇设定解决医生解决不了的大问题，并得到现实反馈。

## 断档风险

- 单纯创意无法支撑长篇。
- 体内世界和外界割裂。
- 金手指触发前拖乱。
- 黄金前三章塞癌细胞伏笔和黑衣男人设。
- 主角没有成长和外部影响。

## 训练任务

重排前三章：外部危机、缩小触发、微观救援；所有不服务救援的伏笔后移。
