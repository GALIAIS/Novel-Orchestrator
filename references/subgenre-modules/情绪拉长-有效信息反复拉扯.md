# 情绪拉长-有效信息反复拉扯

依据案例：`batch-053` 案例 263。

## 读者承诺

章节变长不能靠水，要靠重点场景的情绪拉扯、见证者反应、后文铺垫和有效信息密度。

## 核心循环

```text
重点剧情 -> 情绪目标 -> 多角色见证 -> 反复拉扯 -> 铺垫后文 -> 有效变长
```

## 开篇必备

- 判断这段是不是高潮或重点。
- 明确要拉长的情绪。
- 围观者必须有不同功能。
- 每段反应都提供新信息。
- 重复对话和重复观点要删。

## 期待来源

- 重点场景写长，读者愿意接受。
- 见证者反应能放大装逼爽。
- 下一阶段人物被引出，后续冲突更合理。
- 断章位置越熟练，追读越稳。

## 断档风险

- 为凑字重复对话，读者会骂水。
- 非重点场景拉太长，会拖节奏。
- 围观群众只有倒吸凉气，没有信息功能。
- 装逼文去水无关做菜环节，是错位。

## 训练任务

扩写一个重点场景：新增 3 个见证者，每个见证者必须承担不同功能，并删掉所有重复表达。
