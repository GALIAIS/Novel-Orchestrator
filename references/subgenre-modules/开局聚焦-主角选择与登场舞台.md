# 开局聚焦-主角选择与登场舞台

依据案例：batch-368 案例 1836

## 读者承诺

开局只写主角不是写介绍书，而是在压力舞台里用连续选择展示人设，让配角和背景都服务主角成立。

## 核心循环

```text
聚焦主角 -> 压力舞台 -> 岔路选择 -> 人设显现 -> 剧情推进
```

## 开篇必备

- 主角性格先定。
- 登场舞台能逼出选择。
- 配角只做压力或反馈。
- 背景后置，不抢主角注意力。

## 期待来源

- 主角讨喜。
- 选择展示魅力。
- 读者注意力不分散。
- 每次选择都推动剧情。

## 断档风险

- 只写主角履历。
- 舞台太随便，没有选择。
- 配角喧宾夺主。
- 世界背景压过主角。

## 训练任务

写一章开局大纲，设计 3 次主角选择，每次选择都展示人设并推动事件。
