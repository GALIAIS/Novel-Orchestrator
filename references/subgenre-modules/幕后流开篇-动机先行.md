# 幕后流开篇-动机先行

依据案例：`batch-032` 案例 157。

## 读者承诺

幕后流开篇先让读者看懂主角为什么藏、怎么收割、有什么限制，再写震惊和装逼。

## 核心循环

```text
世界危险 -> 主角动机 -> 金手指前景 -> 幕后限制 -> 首个冲突 -> 情绪收割
```

## 开篇必备

- 主角视角聚焦。
- 世界风险具体。
- 金手指一句话说明。
- 第一章安排具体冲突。

## 期待来源

- 主角弱小但有策略。
- 幕后方案能规避危险。
- 召唤角色或小弟能制造事件。
- 限制让主角不能直接碾压。

## 断档风险

- 连续多章各方震惊。
- 主角目的不明。
- 许愿值和战力体系崩。
- 开局太无敌，后续没看点。

## 训练任务

写一个幕后流第一章小纲：前半交代动机和金手指，后半安排一次具体冲突收割情绪。
