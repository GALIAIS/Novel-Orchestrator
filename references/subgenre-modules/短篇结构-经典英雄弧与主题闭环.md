# 短篇结构-经典英雄弧与主题闭环

依据案例：batch-393 案例 1962

## 读者承诺

短篇以单一欲望和主要阻力完成目标、试炼、价值选择、代价与不可逆结果，视觉和主题通过人物后果落地。

## 核心循环

```text
常态缺口 -> 不可拒绝目标 -> 试炼抬高风险 -> 终局价值选择 -> 支付代价 -> 结果完成主题
```

## 开篇必备

- 一眼可懂的人物缺口和目标。
- 持续升级的一条主要阻力。
- 能代表人物价值观的终局选择。
- 由选择造成而非旁白宣布的主题结果。

## 期待来源

- 目标清楚后每次试炼都更接近终局。
- 旧办法失败迫使人物真正选择。
- 具体代价让成功或失败有重量。
- 结局回收欲望并留下主题余味。

## 断档风险

- 视觉奇观替代人物因果。
- 哲思说明替代终局选择。
- 多个概念争抢短篇时长。
- 只靠突然反转制造完整感。

## 训练任务

写十分钟短篇六拍纲，先完成无反转版本，再测试反转是否真正增加主题和因果，而非替代故事骨架。
