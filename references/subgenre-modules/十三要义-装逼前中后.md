# 十三要义-装逼前中后

依据案例：`batch-008` 案例 038。

## 读者承诺

读者要看到主角被低估、被压制、被挑衅，然后凭信息差、金手指和行动完成反转。爽点不只在打脸结果，而在前中后连续反应。

## 核心循环

```text
通俗动机 -> 对手压制 -> 观众站位 -> 信息差埋入 -> 装逼兑现 -> 多方反应 -> 新期待
```

## 开篇必备

- 一个容易代入的普通动机。
- 一个有理由挑衅的对手。
- 一组能放大情绪的观众或友军。
- 一个能兑现反转的金手指或信息差。
- 一个赌约、倒计时、考核、公开场合。

## 期待来源

- 对手越笃定，读者越期待反转。
- 观众反应越明确，装逼落点越清楚。
- 主角少解释、多留逼格，读者越想看他如何证明。
- 装逼后必须接新钩子，不能只停在“赢了”。

## 断档风险

- 开局苦情太重，读者代入成本过高。
- 对手无脑嘲讽，反而降低戏剧性。
- 主角逼格掉地，读者不愿代入。
- 只有结果，没有前反应、中反应、后反应。

## 训练任务

写一个 9 格装逼单元：动机、对手、观众、信息差、赌约、装逼前压制、装逼中反应、装逼后余波、下一章钩子。
