# 盘龙情绪折线-权力阻碍与连续上下行

依据案例：batch-361 案例 1802

## 读者承诺

用连续上下行牵引读者：局部希望刚出现，就用更高权力阻碍压下去，最后由主角或关键靠山完成兑现。

## 核心循环

```text
目标成立 -> 阻碍下压 -> 局部希望 -> 更强阻碍 -> 主角兜底 -> 最终兑现
```

## 开篇必备

- 核心目标清楚。
- 阻碍层级逐步变高。
- 上行给希望，下行给压力。
- 最低点后要兑现。

## 期待来源

- 两情相悦或目标正当。
- 皇权/强者压力。
- 主角准备兜底。
- 宣布前的最后压制。

## 断档风险

- 只有下行没有希望。
- 上行太轻，压不住读者焦虑。
- 下行不升级。
- 结尾兑现没回收前面压力。

## 训练任务

拆一段剧情，每 300-500 字标一次上行或下行，并写出改变情绪的事件。
