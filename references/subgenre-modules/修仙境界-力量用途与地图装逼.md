# 修仙境界-力量用途与地图装逼

依据案例：`batch-204` 案例 1020。

## 读者承诺

修仙境界名词要服务力量变化、地图扩展和装逼场景，不必死抠考据。

## 核心循环

```text
境界模板 -> 火书参考 -> 本书重编 -> 力量用途 -> 地图装逼
```

## 开篇必备

- 每个境界给可感知能力。
- 境界对应地图层级。
- 解释服务剧情，不堆玄词。
- 参考火书后再化用。

## 期待来源

读者期待每次升级都带来能看见的力量变化和更大的装逼舞台。

## 断档风险

- 境界名词玄而无用。
- 只考据，不服务剧情。
- 级数太多拖慢节奏。
- 新地图和新能力没有绑定。

## 训练任务

给“返虚、合道”各写 3 个解释，每个解释绑定一个新能力、一个新地图和一个装逼场景。
