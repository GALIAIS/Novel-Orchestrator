# 轻小说开篇-平台受众风格适配

依据案例：`batch-117` 案例 584。

## 读者承诺

轻小说风开篇要同时解决风格和平台问题。动漫味、人设、梗密度、名字、金手指翻盘，都要和目标平台受众匹配。

## 核心循环

```text
轻小说风开篇 -> 自审求评 -> 修流畅度/代入感 -> 强化金手指铺垫 -> 内投确认平台
```

## 开篇必备

- 语言简短流畅。
- 主角讨喜或至少不劝退。
- 梗不压过剧情。
- 名字不让目标读者出戏。
- 金手指翻盘前有作用铺垫。

## 期待来源

- 风格受众对味。
- 金手指作用提前被理解。
- 翻盘逻辑更容易接受。
- 平台选择降低错配风险。

## 断档风险

- 起点主站风和二次元站风混乱。
- 人设自大却没讨喜补偿。
- 打斗乱、句子绕。
- 梗密度高但不服务剧情。

## 训练任务

拿一章轻小说风正文，删绕句、降无功能梗、补两处金手指作用铺垫。再列出三个目标平台并判断受众适配。
