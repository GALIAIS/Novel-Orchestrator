# 灵异氛围-暧昧用词留白

依据案例：batch-264 案例 1317

## 读者承诺

灵异文要让读者先感觉“不对劲”，再逐步接近真相。暧昧用词、环境异常和身体细节比直说“鬼”更有效。

## 核心循环

```text
普通场景 -> 异常细节 -> 暧昧称呼 -> 威胁靠近 -> 恐惧补全
```

## 开篇必备

- 深夜或封闭环境。
- 光线/声音异常。
- 人物身体异常。
- 不直接命名的威胁。
- 角色迟疑反应。

## 期待来源

读者期待确认“那到底是什么”，不确定性本身制造恐惧。

## 断档风险

写得太清楚会干；直接说鬼会降低想象空间；可爱反差不能覆盖灵异氛围。

## 训练任务

写 500 字灵异开头，不准出现“鬼、妖、怪物”，只用环境和动作制造不安。
