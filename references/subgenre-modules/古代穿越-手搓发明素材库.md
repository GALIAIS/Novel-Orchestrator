# 古代穿越-手搓发明素材库

依据案例：`batch-331` 案例 1652

## 读者承诺

古代穿越发明要从素材变成剧情，必须拆材料、工具、工艺、试错、利益冲突和落地后果。

## 核心循环

```text
现代知识 -> 古代条件限制 -> 手搓试验 -> 发明落地 -> 利益格局变化
```

## 开篇必备

- 当前危机需要发明解决。
- 材料和工具限制明确。
- 有工匠或组织协作。
- 有失败成本。
- 成功后影响赚钱、军备、农业或权力。

## 期待来源

读者期待主角用现代知识在低技术环境里解决难题，并获得现实反馈。

## 断档风险

- 只写发明结果，不写落地过程。
- 主角像万能百科。
- 技术不服务主线。
- 没有社会阻力和利益冲突。
- 素材视频照抄成说明书。

## 训练任务

选一个发明写工艺卡：材料、工具、步骤、失败点、反对者、受益者、主线用途。
