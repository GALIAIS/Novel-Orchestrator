# 中长篇决策-百万字分界与编辑政策确认

依据案例：`batch-206` 案例 1028。

## 读者承诺

第一本中长篇是否切书，要同时看学习收益、完本履历和平台政策，不要只凭感觉。

## 核心循环

```text
连载中期 -> 收入与数据评估 -> 百万字结构训练 -> 问编辑政策 -> 决定切书或完本
```

## 开篇必备

- 记录当前字数、收入和更新。
- 判断是否第一本。
- 复盘长线结构问题。
- 直接询问编辑老作者/保底规则。

## 期待来源

作者把切书焦虑转成可判断的职业化决策。

## 断档风险

- 十几万、几十万字频繁切。
- 不问编辑，只靠感觉判断。
- 忽略百万字带来的结构暴露。
- 为完本履历拖太久，下一本没有准备。

## 训练任务

做“百万字前是否切书”表：收益、成本、结构复盘、完本履历、编辑政策、下一本准备度。
