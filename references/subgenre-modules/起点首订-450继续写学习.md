# 起点首订-450继续写学习

依据案例：batch-298 案例 1487

## 读者承诺

新人首订 450 不必立刻判死刑，要结合推荐轮次、收订比、后续涨幅和学习价值判断继续或切书。

## 核心循环

```text
首订出炉 -> 查收订比/推荐 -> 观察一个周期 -> 判断增长 -> 继续训练或切书
```

## 开篇必备

- 记录首订和收订比。
- 明确推荐是否跑完。
- 观察后续一个月涨幅。
- 写清当前书还能训练什么能力。

## 期待来源

- 作者从焦虑转为数据判断。
- 继续写能获得长篇经验。
- 切书也有明确触发条件。

## 断档风险

- 只看首订，不看收订比。
- 被编辑一句反馈吓到立刻切。
- 明明不涨还长期硬熬，没有复盘。

## 训练任务

做一张上架复盘表：首订、收订比、推荐轮次、一个月涨幅、继续写收益、切书阈值。
