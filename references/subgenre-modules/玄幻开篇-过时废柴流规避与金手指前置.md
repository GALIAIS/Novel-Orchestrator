# 玄幻开篇-过时废柴流规避与金手指前置

依据案例：`batch-199` 案例 995。

## 读者承诺

玄幻开篇不要让读者贴上废柴流和十年前旧套路标签；金手指要尽早露出可见效果。

## 核心循环

```text
弱势开局 -> 避免废柴标签 -> 金手指前置 -> 展示效果 -> 装逼期待
```

## 开篇必备

- 弱势不等于废柴。
- 第一章展示潜力或外挂。
- 金手指有可见效果。
- 主角要主动使用底牌。

## 期待来源

读者看到主角虽然弱，但已经有可成长的异常资源，愿意等他后续装逼。

## 断档风险

- 直接出现“废柴”词。
- 金手指藏太久。
- 主角只被动挨打。
- 开头有十年前玄幻旧味。

## 训练任务

改写一个废柴流开头：删掉废柴标签，第一章展示金手指效果，并让主角主动解决一个小问题。
