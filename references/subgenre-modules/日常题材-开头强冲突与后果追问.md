# 日常题材-开头强冲突与后果追问

依据案例：batch-369 案例 1843

## 读者承诺

日常、治愈、成长题材也要用开头强冲突抓住注意力，让读者立刻追问后果和人物变化。

## 核心循环

```text
温和题材 -> 开头强事件 -> 角色压力 -> 后果追问 -> 日常线接续
```

## 开篇必备

- 一个打破平静的动作。
- 主角受害或闯祸。
- 立刻后果。
- 角色改变的方向。

## 期待来源

- 主角为什么这么做。
- 后果如何处理。
- 日常关系如何修复。
- 主角会如何成长。

## 断档风险

- 题材温和就开头平。
- 只靠氛围不制造事件。
- 试看几分钟留不住人。
- 冲突和后续日常脱节。

## 训练任务

给一个日常治愈题材写三版开头：受害、闯祸、误会，每版标出后果追问。
