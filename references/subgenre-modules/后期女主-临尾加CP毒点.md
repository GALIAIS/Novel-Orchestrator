# 后期女主-临尾加CP毒点

依据案例：batch-307 案例 1531

## 读者承诺

长篇的女主/无女主倾向必须早定调。读者按无女主追了很久，临近结尾再强加女主或 CP，会被视为推翻原合同。

## 核心循环

```text
前期无感情承诺 -> 读者形成无女主预期 -> 后期突然加CP -> 类型承诺变更 -> 败人缘
```

## 开篇必备

- 30% 进度前明确感情线倾向。
- 单女主必须早埋不可替代性。
- 无女主就不要后期临时补恋爱完整性。
- 中期后尽量不新增重要人物。

## 期待来源

期待来自承诺稳定。读者知道自己追的是无女主、单女主还是弱 CP，后续才会按正确类型投入情绪。

## 断档风险

- 临近大结局才确定女主。
- 为完整故事临时加关系线。
- 大纲原本没有对应阶段，后期硬补。
- 重要新人物过晚登场。

## 训练任务

检查大纲感情线：写出女主/无女主承诺、首次明示章节、中期前关系节点、后期不能新增的关系类型。
