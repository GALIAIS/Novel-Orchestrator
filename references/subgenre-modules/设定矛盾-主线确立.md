# 设定矛盾-主线确立

依据案例：`batch-161` 案例 804。

## 读者承诺

把世界观设定、核心矛盾、势力目标、金手指和主角行动绑定起来，让设定直接推动主线。

## 核心循环

```text
核心设定 -> 终极矛盾 -> 势力目标 -> 主角锚点 -> 阶段收益 -> 新冲突
```

## 开篇必备

- 设定必须带出矛盾。
- 主角金手指要与核心矛盾相关。
- 第一阶段行动要有明确锚点。
- 收益必须改变利益格局。

## 期待来源

读者期待主角每一步行动都有必要性，且每次收益都会自然引出更深层冲突。

## 断档风险

- 设定漂亮但不推动剧情。
- 主角没有必须行动的理由。
- 副本与主线无关。
- 金手指和最终矛盾脱节。

## 训练任务

写一张设定-矛盾-主线表：终极矛盾、三方势力目标、主角金手指、前三卷收益和每次收益损害的对象。
