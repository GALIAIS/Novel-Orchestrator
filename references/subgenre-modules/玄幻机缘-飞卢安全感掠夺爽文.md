# 玄幻机缘-飞卢安全感掠夺爽文

依据案例：`batch-225` 案例 1124。

## 读者承诺

读者看主角抢机缘、打脸、稳定变强，同时确认主角处于不败之地。

## 核心循环

```text
倒计时困境 -> 胜算铺垫 -> 轻压迫 -> 夺机缘 -> 爽点直给
```

## 开篇必备

- 给一个清楚的倒计时压力。
- 冲突前铺主角某方面优势。
- 反派制造不公平伤害，但不能虐过头。
- 机缘夺取要立刻产生收益。

## 期待来源

安全感制造期待感：读者不是担心主角会不会输，而是等着看敌人怎么撞到主角强项上。

## 断档风险

- 压主过重导致读者弃书。
- 只抢机缘，没有胜算依据。
- 反派伤害和机缘争夺无关。
- 扫榜时先挑缺点，忽略作品留人点。

## 训练任务

仿写一个飞卢玄幻开头：倒计时困境、欺负主角的小反派、主角可夺机缘的明确胜算。再把一次压主桥段改成轻压迫、强安全感、直给反杀爽点。
