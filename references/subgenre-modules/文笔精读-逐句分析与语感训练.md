# 文笔精读-逐句分析与语感训练

依据案例：`batch-226` 案例 1129。

## 读者承诺

把文笔差拆成可训练问题，通过逐句精读高手文本，学习信息顺序、描写动机和表达节奏。

## 核心循环

```text
找样本 -> 逐句标注 -> 揣摩动机 -> 重写自己场景 -> 对照调整
```

## 开篇必备

- 先分清遣词造句、行文结构、爽点排列。
- 选自己真佩服的样本。
- 逐句记录信息和情绪。
- 把拆到的顺序迁移到自己场景。

## 期待来源

文笔从“乱乱的”变成可拆、可练、可对照的具体动作。

## 断档风险

- 泛读几眼，没做逐句分析。
- 只学词藻，不学思考顺序。
- 不迁移到自己的段落。
- 期待短期速成。

## 训练任务

选 3-5 个顺滑段落，逐句标注信息、情绪、位置原因；用同样顺序重写自己的一个场景，再对照删乱句、调顺序。
