# 资料工具-历史风物素材库

依据案例：`batch-164` 案例 818。

## 读者承诺

用风土人情和历史关系工具快速寻找背景、地域文化、人物关系和时间线素材。

## 核心循环

```text
查地域/时代 -> 提取风物/人物/事件 -> 交叉验证 -> 转化冲突 -> 写入场景
```

## 开篇必备

- 风物素材要服务场景气质。
- 历史素材要服务人物关系和利益冲突。
- 工具信息需要交叉验证。
- 不把资料名词堆成背景墙。

## 期待来源

读者期待地域和历史细节带来真实质感，同时推动人物选择和矛盾。

## 断档风险

- 工具资料不可靠。
- 只收藏链接，不转成剧情。
- 背景资料过多压住人物行动。
- 历史关系没有冲突功能。

## 训练任务

查一个地域和一个时代，各提取 3 条资料。每条资料都转化成一个人物冲突或场景动作。
