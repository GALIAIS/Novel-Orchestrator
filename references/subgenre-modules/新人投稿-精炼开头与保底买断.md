# 新人投稿-精炼开头与保底买断

依据案例：batch-355 案例 1774

## 读者承诺

新人投稿失败不一定是浪费。控制短稿成本、反复训练开头、获得编辑反馈和保底判断，可以把失败转化为职业化路径上的具体能力。

## 核心循环

```text
短稿投稿 -> 被拒反馈 -> 精炼开头能力提升 -> 编辑保底降低风险 -> 下本更有把握
```

## 开篇必备

- 每稿控制试错字数。
- 每稿只验证一种题材承诺。
- 记录拒签或反馈原因。
- 明确下一稿开头改进点。

## 期待来源

- 开头能力线性积累。
- 切书降低沉没成本。
- 保底买断缓解时间、竞争、精力压力。
- 后面崩了也能积累收入和判断。

## 断档风险

- 一个失败长篇硬耗太久。
- 投稿失败没有记录问题。
- 没有现实缓冲却盲目全职。
- 把坚持理解成不切书。

## 训练任务

做一份 5 稿投稿复盘表：题材、开头承诺、字数、反馈、失败原因、下一稿只改的一点。
