# 修仙种田冒险-异步产出与双环互馈

依据案例：`batch-417` 案例 2081

## 读者承诺

修仙种田的稳定收获与个人冒险的未知风险形成同一因果系统，而不是两部小说交替播放。冒险带回灵种、技术、劳力、市场和情报，生产提供丹药、装备、灵兽、信誉与谈判筹码；有成本的空间、自动化或属下代管让生产异步运行，资源分配和产出外溢再主动制造下一次冒险。

## 核心循环

```text
确定冒险或经营为主承诺 -> 冒险获取稀缺投入 -> 用有成本的空间、阵法、分身或属下启动生产 -> 主角离场时生产异步推进并积累故障或忠诚风险 -> 回归收获并在自用、出售、扩建和储备间取舍 -> 资源强化下一次冒险 -> 大量产出改变市场、宗门或生态 -> 外溢冲突迫使主角再次探索 -> 两环同步升级
```

主次承诺控制定位，异步机制解决时间冲突，双向资源流保证耦合，分配取舍提供策略，产出外溢把静态收获变成长线压力。

## 开篇必备

- 明确冒险或经营哪个是主承诺。
- 六步生产环与六步冒险环。
- 至少四处双向资源和信息互馈。
- 空间、自动化或属下代管的成本与故障。
- 代管者的能力、忠诚和利益。
- 稀缺投入、生产瓶颈和资源上限。
- 自用、出售、扩建和储备的真实取舍。
- 产出改变市场、宗门、生态或组织的后果。
- 回归收获与离场探索的节奏安排。

## 期待来源

- 冒险会带回何种不可替代的生产变量。
- 主角离场时基地会发生什么故障。
- 属下是否会忠实执行资源分配。
- 本轮收获用于个人突破还是势力扩张。
- 大量产出会打破哪项市场或宗门平衡。
- 家园安全感如何反过来提高冒险代价。
- 两条循环何时共同完成一次大升级。

## 断档风险

- 种田和冒险章节互不改变条件。
- 随身空间零成本、无限容量、永不故障。
- 主角定期回来领奖而无需决策。
- 冒险只带回重复货币，没有稀缺变量。
- 产出无限膨胀却不改变市场和组织。
- 随机强盗替代由资源结果产生的冲突。
- 属下代管没有能力、忠诚和利益问题。
- 主副承诺不明，两个受众各等待半本书。

## 训练任务

画出生产与冒险双环并标四处互馈。写一个 5000 字单元：冒险带回稀缺投入，有成本的异步机制推进生产，回归后作资源分配，产出再引发因果明确的外部冲突；禁止零成本无限空间和自动领奖。
