# 示弱诱敌-计划感暗示

依据案例：`batch-116` 案例 580。

## 读者承诺

主角可以主动示弱，但必须让读者知道主角有计划。隐藏的是具体底牌，不是隐藏主角智商和能动性。

## 核心循环

```text
强敌压制 -> 主角示弱 -> 暗示有计划 -> 敌人误判 -> 陷阱触发 -> 反杀兑现
```

## 开篇必备

- 交代敌我实力差。
- 给出示弱的必要性。
- 多次暗示主角有想法。
- 不提前暴露完整陷阱。
- 反杀必须回收前面的暗示。

## 期待来源

- 读者知道主角不是怂。
- 计划感让示弱变成钩子。
- 敌人高傲误判制造期待。
- 反杀带来震惊和爽点。

## 断档风险

- 遮掩太过，读者误判主角无能。
- 示弱时间太长没有计划提示。
- 反杀靠敌人降智。
- 前面暗示没有回收。

## 训练任务

写一个诱敌桥段，至少放三处“主角有计划”的暗示，但不说明底牌。反杀时逐一回收这些暗示。
