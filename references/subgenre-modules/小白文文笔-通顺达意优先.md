# 小白文文笔-通顺达意优先

依据案例：batch-354 案例 1769

## 读者承诺

小白文文笔的第一任务是让读者顺畅读懂，并让情节和爽点连续推进。华丽形容、比喻和铺陈都要让位于通顺、达意、分段、标点和阅读体验。

## 核心循环

```text
通顺达意 -> 分段标点顺读 -> 情节爽点连续 -> 少量关键句提神 -> 多看多写自然积累
```

## 开篇必备

- 句子无明显拗口。
- 段落长度适合手机阅读。
- 标点不妨碍理解。
- 爽点推进不被文笔拖慢。

## 期待来源

- 读者无阻力继续翻页。
- 关键场景有少量提神句。
- 情节和爽点比词藻更稳定。
- 长期积累让表达自然升级。

## 断档风险

- 强熬形容词和比喻。
- 学非小白强文笔作者后误套节奏。
- 文笔追求压过故事性。
- 以为白就是错，反而改得别扭。

## 训练任务

每天修 1000 字旧稿，只改通顺、达意、分段、标点，不增加比喻。连续做 7 天后再看阅读阻力是否下降。
