# 战争双线叙事-环境对照与伏笔回收

依据案例：`batch-207` 案例 1033。

## 读者承诺

战争题材双线叙事可以用环境差异制造悬疑和情绪回击，但每次切线都要提供新解释。

## 核心循环

```text
双线环境 -> 同一句话不同意义 -> 读者疑惑 -> 切线解释 -> 伏笔回收
```

## 开篇必备

- 两条线有明显环境差异。
- 正常对话埋异常反应。
- 切线时解释前一线疑惑。
- 至暗时刻回收伏笔。

## 期待来源

读者先困惑于角色反应，再通过另一条线理解其背后的阶层、战争和环境压力。

## 断档风险

- 视角切换只是重复。
- 世界观漏洞没有剧情内解释。
- 战争设定经不起读者挑刺。
- 伏笔太晚回收导致不信任。

## 训练任务

设计一个双线场景：一句关心在 A 线被误解，B 线解释原因，第 7 章回收为关键伏笔。
