# 女频开篇危机-不公平伤害替代生死

依据案例：batch-273 案例 1364

## 读者承诺

女频开篇不必强行物理生死危机。关系、名誉、事业、婚姻、资源、权力上的不公平伤害，同样能让读者站队并期待反击。

## 核心循环

```text
重要东西被夺 -> 不公平感建立 -> 主角被迫回应 -> 情绪站队 -> 反击期待
```

## 开篇必备

- 明确女主正在失去什么。
- 伤害要足够不公平。
- 第一章必须逼女主行动。

## 期待来源

读者期待女主夺回位置、洗清冤屈、摆脱压迫或重建关系。危机本质是人生位置被动摇。

## 断档风险

- 只是背景惨，主角没有回应。
- 危机太轻，读者不站队。
- 强塞生死危机，和题材情绪不匹配。

## 训练任务

写 5 个非生死开篇危机：失业、退婚、社死、被冤枉、被夺资源。每个都写主角第一反应。
