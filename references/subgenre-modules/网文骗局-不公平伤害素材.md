# 网文骗局-不公平伤害素材

依据案例：batch-296 案例 1479

## 读者承诺

把现实骗局转成有代入感的不公平伤害，让主角识破信息差并反击。

## 核心循环

```text
诱饵出现 -> 骗子伪装权威 -> 弱者受损 -> 主角发现破绽 -> 证据反击 -> 惩罚兑现
```

## 开篇必备

- 骗局诱饵足够具体。
- 受害者相信有合理原因。
- 损失明确可感。
- 主角能抓到一个可验证破绽。

## 期待来源

- 读者代入“我也可能被坑”的不公平感。
- 骗子越会钻规则空子，反击越爽。
- 惩罚兑现形成情绪回收。

## 断档风险

- 只嘲笑受害者蠢，削弱代入。
- 只做科普，不形成剧情压力。
- 主角反击没有证据链。

## 训练任务

写一个签约骗局桥段：诱饵、伪装、损失、破绽、惩罚五步各 50 字以内。
