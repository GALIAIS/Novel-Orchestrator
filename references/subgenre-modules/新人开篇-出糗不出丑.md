# 新人开篇-出糗不出丑

依据案例：`batch-028` 案例 136。

## 读者承诺

新人开篇先证明事件、基调和主角可追，不用低级笑料抢读者注意力。

## 核心循环

```text
异常开局 -> 有效事件 -> 主角出糗不出丑 -> 基调显形 -> 主线钩子前置
```

## 开篇必备

- 第一事件必须清楚。
- 事件要体现题材基调。
- 主角可以狼狈，但不能被写得招人嫌。
- 穿越反应和自嗨笑料要让位给主线钩子。

## 期待来源

- 读者快速知道这本书卖什么。
- 主角在尴尬处境中仍保留魅力。
- 世界线索比通用穿越反应更早出现。
- 第一章结尾有可追的危机或召唤。

## 断档风险

- 屎尿屁、裸奔、下三路定调。
- 作者自嗨，读者看不到主线利益。
- 穿越反应写太久。
- 有效钩子藏到太后面。

## 训练任务

把一个穿越开篇重排顺序：前 500 字必须出现事件、基调、主角目标和世界钩子，删掉不会影响主线的自嗨笑料。
