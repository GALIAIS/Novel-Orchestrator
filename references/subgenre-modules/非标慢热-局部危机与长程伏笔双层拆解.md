# 非标慢热-局部危机与长程伏笔双层拆解

依据案例：`batch-396` 案例 1978

## 读者承诺

用与机制生命周期匹配的尺度拆慢热作品：前三章看局部疑问和危机闭环，十章到首个大高潮追踪伏笔债务、身份矛盾和低存在感优势。

## 核心循环

```text
首读体验 -> 三章局部闭环表 -> 伏笔债务表 -> 扩展到十章三十章或大高潮 -> 验证规则与回收 -> 单项迁移并缩短兑现
```

## 开篇必备

- 每章至少改变疑问、人物或危机状态。
- 前三章完成一次小回收。
- 长程问题有种下和重提记录。
- 非对称优势先作为候选机制观察。
- 文本事实、阅读感受和作者意图分栏。
- 拆解范围覆盖目标机制的回收点。

## 期待来源

局部危机持续偿还小账，长程身份和规则问题不断被重提、改释，读者相信早期异常最终会进入主冲突。

## 断档风险

- 用前三章直接否定整部长结构。
- 把长期未回收自动解释为高级伏笔。
- 强行给模糊优势贴金手指标签。
- 把个人难受写成作者动机。
- 因整体难学而放弃所有局部技巧。
- 新人照搬超长延迟却没有小回收。

## 训练任务

为一部慢热作品建立前三章闭环表、前三十章伏笔债务表和首个高潮回收表；所有机制结论附章节证据，再只迁移一项技巧到自己的三章纲。
