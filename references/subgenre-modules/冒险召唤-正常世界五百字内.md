# 冒险召唤-正常世界五百字内

依据案例：`batch-318` 案例 1588

## 读者承诺

网文开篇里的正常世界越短越好，新人应在第一章前 500 字内点出基础信息并进入冒险、异常或主线事件。

## 核心循环

```text
基础身份 -> 500字内触发冒险 -> 行动中补背景 -> 读者追主线
```

## 开篇必备

- 主角职业、年龄或身份一句话。
- 一个私人缺口。
- 快速出现冒险召唤。
- 可后置的前史不要抢开篇篇幅。
- 背景在导师交流或危机行动里分批揭露。

## 期待来源

读者期待第一章就看到故事启动，而不是等作者铺完正常生活。

## 断档风险

- 正常世界拖一整章。
- 机械照搬英雄旅程，不适配网文节奏。
- 前史解释过多，主线迟迟不动。
- 冒险召唤出现太晚，读者离开。

## 训练任务

把一章正常生活压成 500 字，只保留职业、年龄、私人缺口，然后让冒险召唤立刻发生。
