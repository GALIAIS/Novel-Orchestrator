# 多重人格主角-开局好感爽点沉淀

依据案例：`batch-180` 案例 900。

## 读者承诺

多重人格或隐藏身份主角开局必须先建立主角好感，再让强人格爽点反哺主角成长。

## 核心循环

```text
悬疑开场 -> 主人格亮点 -> 强人格发威 -> 主人格获得信息/成长 -> 爽点沉淀
```

## 开篇必备

- 主人格前三章必须有闪光点。
- 弱人格不能只负责拖累。
- 强人格爽点要能被主角继承或理解。
- 陌生读者不应依赖粉丝背景才能看懂。

## 期待来源

读者期待悬疑之外还能喜欢主角。隐藏人格越强，越要让主人格获得成长收益，否则爽点会断片。

## 断档风险

- 主人格长期懦弱、碎嘴、无主动性。
- 强人格发威后主人格完全无记忆。
- 爽点只属于另一个人格。
- 粉丝向信息没有给陌生读者补足。

## 训练任务

设计双人格开局：弱人格给一个共鸣优点、一个专业能力、一次主动选择；强人格爽点必须留下可被弱人格使用的信息或成长。
