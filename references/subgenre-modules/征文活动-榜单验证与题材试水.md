# 征文活动-榜单验证与题材试水

依据案例：`batch-238` 案例 1187。

## 读者承诺

征文活动只能说明平台在收什么，不能直接证明什么最火；新人选题仍要优先看该平台销售榜。

## 核心循环

```text
看到征文 -> 误判热门 -> 查销售榜 -> 判断补库/试水/强势题材 -> 决定是否写
```

## 开篇必备

- 记录征文类型。
- 对照平台销售榜。
- 找同题材榜样书。
- 判断新人可写难度。

## 期待来源

作者用真实榜单校验活动公告，避免把平台供给需求误当成读者需求。

## 断档风险

- 把优先收稿当成热门保证。
- 不看当前榜单。
- 忽略平台补书库和试新方向的可能。
- 选择自己驾驭不了的题材。

## 训练任务

对任一征文题材找 3 本榜单同类书；找不到稳定榜样书时，标记为试水方向，不作为主力题材。
