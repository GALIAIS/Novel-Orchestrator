# 码字输入法-词库同步与备份

依据案例：batch-312 案例 1557

## 读者承诺

输入法词库是码字基础设施。长期积累的自造词、人物名、地名和常用表达要同步或备份。

## 核心循环

```text
长期码字 -> 输入法积累词库 -> 打字更顺 -> 换机/重装风险 -> 云同步或导出备份
```

## 开篇必备

- 输入法登录账号。
- 开启云同步。
- 记录自造词导出方式。
- 给作品专有名词单独留表。

## 期待来源

期待来自稳定产能。输入法越懂作者，码字越少被错字和候选词打断。

## 断档风险

- 重装系统前没备份词库。
- 作品专有名词只存在输入法里。
- 双拼/五笔切换期影响速度。
- 把打字慢误认为写作能力差。

## 训练任务

导出一次输入法自造词，建立作品专有名词表，并写下换电脑后的恢复步骤。
