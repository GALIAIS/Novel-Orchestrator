# 同人开头-败北前史与当前困境

依据案例：`batch-241` 案例 1204。

## 读者承诺

同人开头不宜大篇幅写主角败北前史；应从当前困境、被控制和自救压力开始。

## 核心循环

```text
前史战败 -> 主角弱化/被掳 -> 直接切当前困境 -> 展示自救目标 -> 带出金手指
```

## 开篇必备

- 找真正故事起点。
- 弱化前史战斗。
- 建立深陷重围的压力。
- 顺着原作画风写术语和能力。

## 期待来源

读者一开篇看到主角被压迫但仍有反制空间，期待他身在敌营如何自救。

## 断档风险

- 开头大篇幅写主角输了。
- 伏笔太多让读者云里雾里。
- 再教育过程拖慢冒险。
- 原作画风信号不足。

## 训练任务

删除失败战斗，只用 300 字交代前因，从主角醒来后的危险处境开始写。
