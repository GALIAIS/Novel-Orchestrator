# 阅读方法-实体书记录思考

依据案例：`batch-190` 案例 950。

## 读者承诺

看实体书或非网文小说时，笔记重点是记录和思考，把阅读收获转成可复用写作动作。

## 核心循环

```text
阅读 -> 标注/摘录 -> 写有效原因 -> 提炼技法 -> 改写练习
```

## 开篇必备

- 记录页码或章节。
- 摘录关键段落。
- 写清为什么有效。
- 对应到自己的写作场景。

## 期待来源

读者把非网文里的语言、结构、人物和情绪处理，转化为自己能用的技法。

## 断档风险

- 只抄金句。
- 摘录后不思考。
- 笔记散落不可检索。
- 不做改写练习，无法迁移。

## 训练任务

读一本实体小说 20 页，做 3 条笔记；每条必须包含摘录、有效原因和一个网文场景改写动作。
