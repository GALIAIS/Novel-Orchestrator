# 升级副本-起承转合与地位提升

依据案例：batch-364 案例 1816

## 读者承诺

升级副本要按目标、难度、破局、收获推进，并让能力升级转化为地位、待遇、名望和敌人层级变化。

## 核心循环

```text
目标 -> 难度铺垫 -> 主动破局 -> 打脸成功 -> 收获/地位提升
```

## 开篇必备

- 副本目标清楚。
- 难度和敌人被铺垫。
- 主角主动发现并解决问题。
- 成功后有资源或地位变化。

## 期待来源

- 每十章左右一轮晋升。
- 主角解决更高级问题。
- 待遇、名望、权利提升。
- 马甲开新地图。

## 断档风险

- 重复被动失败。
- 地位一开始太高，没提升空间。
- 收获只写宝物，不写社会反馈。
- 主角不像有能力的正常人。

## 训练任务

设计 10 章升级副本，每章写目标、阻碍、主动破局、收获和地位变化。
