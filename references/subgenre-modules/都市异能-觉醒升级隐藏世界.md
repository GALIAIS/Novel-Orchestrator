# 都市异能-觉醒升级隐藏世界

依据案例：`batch-130` 案例 648。

## 读者承诺

现代城市日常中出现超常能力，主角用能力解决现实小危机，再发现城市背后的组织、体系和更大公共危机。爽点来自能力首秀、身份变化、升级和世界扩张。

## 核心循环

```text
都市场景 -> 异能觉醒 -> 小危机解决 -> 隐藏世界证据 -> 组织/体系出现 -> 升级卷入大危机
```

## 开篇必备

- 从具体都市问题切入。
- 第一次异常要清楚。
- 能力有使用限制。
- 小危机能立刻展示能力价值。
- 结尾露出隐藏世界。

## 期待来源

- 普通生活被异常打破。
- 能力带来直接解决力。
- 城市背后还有更大规则。
- 主角从个体事件走向大局冲突。

## 断档风险

- 先讲异能百科。
- 能力没有限制。
- 都市场景只是背景板。
- 组织和大世界太晚出现。

## 训练任务

写一个 800 字开篇：主角在现实都市场景中第一次异常觉醒，解决一个小危机，并留下“城市并不普通”的证据。
