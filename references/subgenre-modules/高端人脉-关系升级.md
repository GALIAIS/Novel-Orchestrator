# 高端人脉-关系升级

依据案例：`batch-141` 案例 702。

## 读者承诺

主角不只靠自身升级打开世界，也会因为稀缺价值被更高层人物关注、拉拢、保护、试探或利用，从而让剧情尺度和位格持续上升。

## 核心循环

```text
等级体系清晰 -> 主角展现稀缺价值 -> 高层注意 -> 关系升级 -> 更大棋局压来 -> 主角仍亲自兑现
```

## 开篇必备

- 高层位格。
- 主角当前位格。
- 主角被关注的理由。
- 高层介入的代价。
- 主角不可被替代的任务。

## 期待来源

- 被高层看重带来的逼格。
- 世界尺度逐层打开。
- 主角身份或能力被重新解释。
- 高层局势制造更大压力。

## 断档风险

- 高层直接替主角解决问题。
- 等级体系不清，强者只像贴纸。
- 主角被大棋局压成工具人。
- 新人写得太高，逻辑和压迫感失控。

## 训练任务

设计同层伙伴、高一层贵人、顶层关注者三层关系。每层写出为什么看重主角，以及如何不替主角解决最终问题。
