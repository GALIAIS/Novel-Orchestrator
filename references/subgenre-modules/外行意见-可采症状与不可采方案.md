# 外行意见-可采症状与不可采方案

依据案例：batch-379 案例 1892

## 读者承诺

外行反馈要采症状，不直接采处方。读者说“不吸引”有价值，但“开车、玩梗、跟风”必须回到题材承诺和后续展开中重判。

## 核心循环

```text
外行感受 -> 提取症状 -> 丢弃乱处方 -> 结构定位 -> 改开头钩子
```

## 开篇必备

- 可被普通读者感知的吸引点。
- 冲突或代入源。
- 可持续展开的钩子。
- 三章内能兑现的承诺。

## 期待来源

- 读者说出想停的位置。
- 作者定位吸引力缺口。
- 金手指、冲突或设定形成代入。
- 改稿后能继续展开。

## 断档风险

- 全盘否定外行意见。
- 照抄外行热闹建议。
- 只追求炸裂，后续无展开。
- 把冲突当唯一吸引源。

## 训练任务

让非作者读开头，只问哪里想停、哪里没兴趣、哪里看不懂；把回答翻译成开篇钩子和前三章展开，不采纳对方具体写法。
