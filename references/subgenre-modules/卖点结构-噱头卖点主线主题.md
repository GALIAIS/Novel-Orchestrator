# 卖点结构-噱头卖点主线主题

依据案例：`batch-104` 案例 520。

## 读者承诺

噱头把读者拉进来，卖点让读者撑过铺垫，主线主题让长篇持续推进。三者分工不同，混成一团会导致开篇能点进来但留不住。

## 核心循环

```text
噱头吸引 -> 卖点留存 -> 铺垫有看头 -> 爽点爆发 -> 主线推进 -> 新主线伏笔
```

## 开篇必备

- 一句话噱头。
- 一个可延展卖点。
- 第一爽点前的看点。
- 长期主线。
- 下一个主线伏笔。

## 期待来源

- 新鲜猎奇。
- 反转和不常见。
- 卖点持续升级。
- 主线环环相扣。

## 断档风险

- 以为精彩剧情本身就是卖点。
- 噱头只能撑几百字。
- 卖点不能延展，爽完就没东西写。
- 主线切换没有提前埋线。

## 训练任务

写下本书噱头、卖点、主线各一句。再检查第一章是否三者都露面；缺一项就补一个对应桥段。
