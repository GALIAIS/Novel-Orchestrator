# 胜利收益-状态变化与下一轮杠杆

依据案例：batch-390 案例 1949

## 读者承诺

关键胜利必须让资源、能力、身份、关系、信息或选择空间发生可追踪变化，并让至少一项收益成为下一轮行动杠杆。

## 核心循环

```text
投入并承担风险 -> 赢下关键对抗 -> 收益代价同时入账 -> 状态发生差值 -> 新杠杆进入下一轮
```

## 开篇必备

- 明确的胜负确认动作。
- 与本轮冲突有因果的即时收益。
- 同时保留的伤势、人情或暴露代价。
- 下一章会被使用或争夺的状态变化。

## 期待来源

- 冲突压力获得可见释放。
- 风险与投入换来等价进步。
- 他人态度帮助读者理解收益价值。
- 新资源撬动更大的目标和争夺。

## 断档风险

- 写了赢却让战后状态完全归零。
- 只有奖励名称没有具体作用。
- 结算时自动忘记胜利代价。
- 收益后文再也不被使用。

## 训练任务

用“投入、胜负、收益、代价、下次使用”五列复查九章，给状态归零的关键胜利补结算，并在后两章实际调用收益。
