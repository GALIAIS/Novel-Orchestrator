# 隐藏大佬-旁人背书入场

依据案例：`batch-130` 案例 650。

## 读者承诺

角色低姿态入场，自称新人小白；旁人却用惊讶、外号和追问替他背书。读者会期待这个人到底是谁、为什么别人反应这么大。

## 核心循环

```text
低姿态报到 -> 旁人强烈反应 -> 抛出旧称/马甲 -> 本人继续回避 -> 身份悬念延长
```

## 开篇必备

- 本人姿态低。
- 旁人反应高。
- 至少出现一个旧称或外号。
- 第三方追问身份。
- 本人不立刻解释。

## 期待来源

- 自称和他人反应形成反差。
- 外号制造历史感。
- 围观者追问带动读者好奇。
- 身份揭示被延后。

## 断档风险

- 本人自己吹自己。
- 旁人反应没有证据感。
- 太早揭示真实身份。
- 反差只靠称号，没有后续兑现。

## 训练任务

写一个 300 字以内片段：主角以新人身份进入创作群、修行门派或职业圈，只说自己来学习；至少两名旁人用反应暗示他并不普通。
