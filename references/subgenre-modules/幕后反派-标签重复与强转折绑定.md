# 幕后反派-标签重复与强转折绑定

依据案例：batch-377 案例 1884

## 读者承诺

幕后反派标签要重复出现，并和强转折绑定。读者记住的不是标签本身，而是标签后面会发生什么。

## 核心循环

```text
角色标签 -> 多次重复 -> 强后果绑定 -> 条件反射 -> 正式登场
```

## 开篇必备

- 一个可识别标签。
- 标签对应的角色印象。
- 至少三次事件绑定。
- 与主角利益逐步靠近。

## 期待来源

- 读者看到标签就预感危险。
- 幕后感逐渐增强。
- 反派下场前已有存在感。
- 正式登场能引爆前文记忆。

## 断档风险

- 标签只装饰不影响剧情。
- 特征和角色作用不匹配。
- 长线铺垫超出掌控力。
- 反派行为不指向主角。

## 训练任务

设计一个幕后反派标签，安排 3 次只闻其名不见其人的事件，每次标签出现后都发生强转折。
