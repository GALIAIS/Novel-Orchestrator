# 大纲树干-主线回归

依据案例：`batch-141` 案例 703。

## 读者承诺

先确定一条清晰主线树干，让灵感、伏笔、小事件和装逼情节都能回归主线，保证长篇写作和阅读的连贯感。

## 核心循环

```text
一句话主线 -> 小事件挂枝 -> 判断能否回归主线 -> 可断枝叶删除 -> 伏笔服务爽点和转场
```

## 开篇必备

- 主角长期目标。
- 当前阶段目标。
- 能回归主线的小事件。
- 可删除的小事件。
- 伏笔回收位置。

## 期待来源

- 读者感到每段都在推进。
- 作者有标准筛选灵感。
- 高强度更新不散架。
- 伏笔回收后自然过渡。

## 断档风险

- 主线模糊，小事件越写越歪。
- 灵感随来随改，破坏方向。
- 伏笔随机塞，回收突兀。
- 枝叶删不掉，拖慢主线。

## 训练任务

写出一句主线树干；列 5 个小事件，标注回归主线、可删除或需改造；设计 2 个伏笔，要求都能在爽点后自然回收或转场。
