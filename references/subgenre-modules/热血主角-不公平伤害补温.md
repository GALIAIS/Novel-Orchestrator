# 热血主角-不公平伤害补温

依据案例：`batch-076` 案例 376。

## 读者承诺

这个模块用于冷静型作者写热血升级文。它承诺不用空喊热血，而是通过不公平伤害、保护对象和必须反抗的目标，让主角情绪从行动里升温。

## 核心循环

```text
不公平伤害 -> 牵连亲友/群体 -> 主角憋劲 -> 选择反抗 -> 行动兑现热血
```

## 开篇必备

- 主角当前不是单纯冷漠，而是有被压住的目标。
- 不公平伤害要具体。
- 伤害最好波及亲友、家族、伙伴或保护对象。
- 主角必须有不能退的理由。
- 反击动作要比心理独白更早兑现。

## 期待来源

- 读者替主角觉得不平。
- 亲友受害拉高必须反抗的正当性。
- 少年气来自不服输和向上冲。
- 保护对象让升级不只是为自己。
- 反击时释放前置下行。

## 断档风险

- 只写主角愤怒，不写伤害来源。
- 下行不够多维，反击不燃。
- 主角太冷，读者无法判断他在乎什么。
- 学热血样本却只学尬口号。
- 明明适合冷血主角，却硬改成热血。

## 训练任务

给主角设计一个三层不公平伤害：伤害自己、伤害亲友、堵死退路。再写一个反击动作，不能用大段心理独白解释热血。
