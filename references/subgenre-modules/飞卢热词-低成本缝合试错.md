# 飞卢热词-低成本缝合试错

依据案例：batch-293 案例 1461

## 读者承诺

用热点和热词缝合快速试水题材，几万字看反馈，火了扩展，不火止损。

## 核心循环

```text
热点 -> 热词缝合 -> 几万字试水 -> 数据反馈 -> 继续/放弃 -> 中期补机制
```

## 开篇必备

- 热点来源。
- 三个可见热词。
- 第一眼卖点。
- 前三章玩梗方式。
- 试水数据指标。
- 中期扩展机制。

## 期待来源

目标读者一眼识别热点，带着“这个梗会怎么玩”的好奇点进来。

## 断档风险

- 只有热点，没有中期机制。
- 单点创意撑不住长篇。
- 把飞卢试错法照搬到重视笔名信誉的平台。
- 试水成功后只会胡编。

## 训练任务

给一个热点题材写试水卡：热词、前三章、2 万字指标、中期扩展各一行。
