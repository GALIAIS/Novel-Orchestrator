# 黄金三章拆解-原文对读长文

依据案例：batch-312 案例 1558

## 读者承诺

黄金三章精拆要和原文逐段对读。只看结论容易把细节误读成套路口号。

## 核心循环

```text
选样本 -> 原文逐段读 -> 分析段落功能 -> 记录期待变化 -> 提炼可迁移手法
```

## 开篇必备

- 原文和分析并排看。
- 逐段记录功能，不先下总评。
- 关注信息释放顺序。
- 记录读者每一段的期待变化。

## 期待来源

期待来自细节复盘。学习者看见作者每一步怎么抓住读者，才能迁移到自己的开篇。

## 断档风险

- 只看分析，不看原文。
- 只摘结论，不拆段落。
- 只选爆款，不看偶然发现的优秀开头。
- 想用视频快速替代慢读。

## 训练任务

选一篇黄金三章，逐段写“原文功能、信息点、读者期待、可学手法”，不少于 20 条。
