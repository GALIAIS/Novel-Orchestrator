# 末日城开篇-主体先行现实延后

依据案例：batch-288 案例 1438

## 读者承诺

书名承诺末日城，开篇就要以末日城危机、救援和收益为主体，现实装逼放到收益之后。

## 核心循环

```text
末日危机 -> 主角上帝视角援助 -> 幸存者脱险感恩 -> 主角获得资源 -> 现实困境反打脸
```

## 开篇必备

- 末日城第一场危机。
- 幸存者的不公平遭遇。
- 主角援助方式。
- 可量化收益。
- 现实困境伏笔。
- 账目和细节校验。

## 期待来源

读者因末日城点进来，先看到末日生存与经营收益，再期待这些收益如何反哺现实。

## 断档风险

- 开头两章都写现实。
- 现实充大款没有收益来源。
- 金钱账目算错。
- 错别字和基础逻辑错误影响编辑观感。
- 两个世界横跳过早，主线变散。

## 训练任务

把现实装逼开头延后三章：先写末日危机、救援获利、收益入账，再安排现实打脸。
