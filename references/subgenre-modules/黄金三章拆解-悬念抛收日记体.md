# 黄金三章拆解-悬念抛收日记体

依据案例：`batch-316` 案例 1578

## 读者承诺

黄金三章不是固定套路，而是用书名简介筛读者、用开篇连续抛收悬念、用金手指解决问题，让读者确认这本书值得追。

## 核心循环

```text
书名简介承诺 -> 日记体抛悬念 -> 小回收保持顺滑 -> 金手指出现 -> 解决问题 -> 人设成立
```

## 开篇必备

- 开头每个信息点都服务继续阅读。
- 悬念要能抛，也要能收。
- 新手可先做到抛 3 收 2。
- 金手指要写使用方法、效果和收获。
- 人设要在事件里立住。

## 期待来源

读者期待每个小疑问都有回响，每次回收都带来新期待，第三章末尾确认主线吸引力。

## 断档风险

- 机械套黄金三章模板。
- 日记体只交代信息，没有悬念。
- 为赶节奏压缩到金手指讲不清。
- 悬念抛太多，收不回来。

## 训练任务

拆一章开篇，标出 3 个悬念和至少 2 个回收点，再写每个回收点带来的新期待。
