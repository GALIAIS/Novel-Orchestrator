# 换皮模仿-经典框架与适度改写

依据案例：`batch-241` 案例 1201。

## 读者承诺

新人第一本可以借成熟作品的成长线和拐点练结构，但要换掉题材逻辑、人物动机和具体事件。

## 核心循环

```text
缺长篇结构 -> 借经典框架 -> 抽取节点 -> 换题材事件 -> 练自己的大纲
```

## 开篇必备

- 选熟悉参考作品。
- 只抽成长线和关键拐点。
- 替换角色关系和事件表层。
- 建立自己的读者承诺。

## 期待来源

作者先获得能跑起来的长线骨架，再在具体题材里训练原创表达。

## 断档风险

- 照搬角色和桥段。
- 只换名词不换动机。
- 忽略新题材读者期待。
- 把模仿当成永久方法。

## 训练任务

提取一部经典作品的 5 个结构节点，改成自己题材的目标、冲突和阶段奖励。
