# 无敌文拆书-作者视角训练

依据案例：`batch-129` 案例 643。

## 读者承诺

新手拆无敌文，重点不是拆得对不对，而是训练从读者爽感转到作者动机。看主角为什么一直强、敌人如何衬托、信息差和节奏如何维持无敌感。

## 核心循环

```text
选择榜单样本 -> 判断无敌卖点 -> 拆前 10-15 章 -> 记录爽点 -> 追问作者意图
```

## 开篇必备

- 样本来自榜单或高成绩。
- 先判断题材和主角卖点。
- 每章记录剧情事件。
- 每章记录读者爽点。
- 每章追问作者为什么这样写。

## 期待来源

- 主角强势感稳定。
- 敌人反应衬托主角。
- 信息差带来预期优势。
- 新手逐渐形成作者视角。

## 断档风险

- 只判断自己爽不爽。
- 一开始就追求理论正确。
- 不看作者安排动机。
- 拆完不能反哺自己的写法。

## 训练任务

选一部榜单无敌文，只读前 10-15 章，手工记录三列：剧情事件、读者爽点、作者意图。最后写 300 字总结无敌感来自哪里。
