# 黑暗流私货-红线隔离

依据案例：`batch-072` 案例 357。

## 读者承诺

这个模块用于黑暗流、反派流和高争议题材。它承诺的是学习脑洞、力量体系和结构，不学习现实体制隐喻、组织无用论、粉丝身份动员等高风险私货。

## 核心循环

```text
黑暗爽点 -> 拆出可学结构 -> 识别现实隐喻 -> 删除私货 -> 改成虚构内部矛盾
```

## 开篇必备

- 暴力情节要有清晰语境。
- 不把现实政治和组织理论塞进网文。
- 不引导粉丝形成危险身份认同。
- 不把私货包装成世界真理。
- 可学点和红线点分栏处理。

## 期待来源

- 黑暗流提供差异爽感。
- 脑洞和力量体系可迁移。
- 虚构内部矛盾降低现实风险。
- 结构学习保留价值。
- 红线隔离保护连载。

## 断档风险

- 以为“黑暗”就是可以说任何话。
- 将社会体制隐喻写成主旨。
- 组织粉丝话术出现在章末。
- 热度起来后被集中举报。
- 学作品时连私货一起学走。

## 训练任务

拆一个黑暗流桥段，写两栏：可学的脑洞/结构、必须隔离的现实隐喻。再把桥段改成纯虚构势力内部矛盾。
