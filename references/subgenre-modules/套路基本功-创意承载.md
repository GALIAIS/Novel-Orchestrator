# 套路基本功-创意承载

依据案例：`batch-131` 案例 652。

## 读者承诺

创意不是替代套路，而是要被基础故事套路承载。先把基本剧情玩熟，才能把新设定、新脑洞整合成读者愿意看下去的故事。

## 核心循环

```text
掌握基础套路 -> 放入新设定 -> 明确主角目标 -> 设计阻碍代价 -> 完成单元闭环
```

## 开篇必备

- 先选基础套路。
- 主角目标清楚。
- 读者爽点清楚。
- 创意服务冲突。
- 第一单元能跑通。

## 期待来源

- 熟悉套路降低理解门槛。
- 新设定带来新鲜感。
- 基本功保证故事能持续。
- 创意不再漂浮。

## 断档风险

- 基本剧情玩不转。
- 只追求新东西。
- 看很多类型却不拆结构。
- 作者无脑写，读者无法无脑看。

## 训练任务

选一个新脑洞，把它套进废柴逆袭、强者归来、逃亡求生、破案解谜、事业升级任一基础套路，写出目标、阻碍、代价、第一单元爽点。
