# 平台作者门槛-知识边界避坑

依据案例：`batch-088` 案例 438。

## 读者承诺

平台要求作者有基础人文知识、价值观和社会认知，不是让作者写政治，而是要求作者别无意识踩线。新人仍要把开篇质量放在第一位。

## 核心循环

```text
看平台标准 -> 识别知识边界 -> 检查现实设定 -> 修正价值表达 -> 提升开篇质量 -> 接受改稿
```

## 开篇必备

- 开篇要先好看、清楚、可签。
- 涉及现实社会时，要检查历史、地理、政治、职业常识。
- 价值观表达要明确，不写低级误伤。
- 敏感内容能虚化就虚化。
- 过稿后准备接受编辑改稿。

## 期待来源

- 作品不因低级常识错误出戏。
- 不因边界模糊踩线。
- 开篇质量足以支撑签约。
- 编辑反馈能被快速吸收。

## 断档风险

- 把平台标准误解成必须写政治。
- 完全不关心边界，写出现实风险。
- 只补知识，不改开篇。
- 过稿后扛不住连续改稿。

## 训练任务

给开篇做现实边界检查。列出现实机构、历史地理、职业常识、价值判断四类内容，标注保留、虚化、删除或补资料。
