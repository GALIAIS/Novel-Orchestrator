# 拉片训练-微片段复盘

依据案例：`batch-016` 案例 079。

## 读者承诺

三分钟片段也能训练信息展示、节奏控制、情绪拉扯、氛围、人设和动机，不必一上来拆完整作品。

## 核心循环

```text
选短片段 -> 逐镜标注 -> 五栏复盘 -> 回连理论 -> 写作迁移
```

## 开篇必备

- 片段要短，便于反复看。
- 不复述剧情，直接标功能。
- 至少拆信息、节奏、情绪、人设、动机。
- 最后写一条可迁移技巧。

## 期待来源

- 理论能在片段里被看见。
- 新人训练成本低。
- 一个片段多次复盘会有新收获。
- 微单元能迁移到网文场景。

## 断档风险

- 贪多拆整季，最后只记剧情。
- 只写观后感，不标功能。
- 不回连理论。
- 不产出可迁移写法。

## 训练任务

找一个 3 分钟片段，填信息、节奏、情绪、人设、动机五栏，并写出一条能用于小说的技法。
