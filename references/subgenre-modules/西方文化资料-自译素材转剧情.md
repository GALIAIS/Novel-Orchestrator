# 西方文化资料-自译素材转剧情

依据案例：`batch-065` 案例 321。

## 读者承诺

这个模块用于西幻、巫术、驱魔、占卜、古文明、神秘学题材。它承诺的是把自译资料和文化素材转成可入戏的仪式、禁忌、职业、物品和冲突，而不是在正文里堆百科。

## 核心循环

```text
资料主题 -> 概念摘取 -> 剧情功能化 -> 事件承载 -> 世界观渐进露出
```

## 开篇必备

- 带着写作问题查资料。
- 每次只取能服务事件的概念。
- 资料要转成角色行动、规则或冲突。
- 第一章只放当前冲突需要的文化信息。
- 保留来源，避免照搬译者文本。

## 期待来源

- 陌生文化概念带来的新鲜感。
- 仪式、禁忌、巫术、占卜等进入事件。
- 古文明或神秘学资料制造谜团。
- 专题资料支撑世界质感。
- 资料逐步露出，形成探索感。

## 断档风险

- 囤资料但不写事件。
- 把自译文本原样搬进正文。
- 文化素材和主线冲突无关。
- 资料太杂，作者没有筛选问题。
- 第一章变成百科介绍。

## 训练任务

选一个西方文化概念，把它改造成一个仪式、一个禁忌和一个冲突。写出它们分别会怎样改变主角第一章的选择。
