# 新人签约-反捷径训练

依据案例：`batch-030` 案例 148。

## 读者承诺

签约焦虑不能靠代写解决，真正有用的是拆同类、练开篇、写长、复盘。

## 核心循环

```text
签约焦虑 -> 捷径诱惑 -> 风险识别 -> 回到训练 -> 长期能力增长
```

## 开篇必备

- 分清“想签约”和“能写完”。
- 明确当前真实短板。
- 拒绝包签和代写。
- 用 7 天任务替代焦虑。

## 期待来源

- 方向正确带来稳定感。
- 同类开篇拆解提供可改对象。
- 写到 3-5 万字后再回头校准开篇。
- 自己掌握后续质量。

## 断档风险

- 买开头导致后文接不上。
- 代写被识别，账号和笔名受损。
- 反复改开头把框架玩崩。
- 把签约当成最终目标。

## 训练任务

做 7 天签约焦虑替代计划：每天拆 1 个同类开篇或改 1 个钩子，禁止购买代写。
