# 废土狩猎-生存收益与主动觉醒

依据案例：`batch-396` 案例 1976

## 读者承诺

用一次高风险狩猎同时交付废土生存压力、主角行动能力和金手指首秀：主角先为了具体生活缺口主动冒险，再由危机中的选择触发能力。

## 核心循环

```text
生存缺口 -> 选定高价值猎物 -> 观察布置与交锋 -> 更强威胁击穿计划 -> 主动选择触发能力 -> 战利品和新能力引来下一争夺
```

## 开篇必备

- 交代主角为什么今天必须狩猎。
- 猎物能换算成食物、药物或居住时间。
- 主角通过准备和动作展示武夫能力。
- 意外威胁提高风险而非随机转场。
- 金手指回应主角的冒险选择。
- 结尾用收益解决部分生存问题。

## 期待来源

读者既担心主角能否带回维持生活的资源，也期待他的行动如何让特殊能力显效，并进一步改变废土中的地位。

## 断档风险

- 年代大事记压住当前危机。
- 远方人物惨史与主角无即时关系。
- 狩猎理由和猎物价值不清。
- 金手指靠随机天降而非人物选择。
- 第一章名字太多、性格都不鲜明。
- 高刺激苦难先制造入口排斥。

## 训练任务

写 1800 字狩猎开篇：300 字内交代断粮压力，600 字内开始行动，1200 字前出现超预期猎物，结尾由主角主动选择触发能力并获得可量化收益。
