# 动机公式-赌注风险可见

依据案例：`batch-061` 案例 305。

## 读者承诺

这个模块用于检查角色行动动机。它承诺的是把“我想要”改成“我押上了什么，失败会失去什么”，让行动有重量。

## 核心循环

```text
目标 -> 赌注 -> 风险 -> 行动必要性 -> 代价可见 -> 新赌注
```

## 开篇必备

- 主角目标要具体。
- 赌注必须是已经投入或必须押上的东西。
- 风险要让读者看得见。
- 行动要有“现在不做就来不及”的理由。
- 行动后产生新赌注。

## 期待来源

- 读者相信主角为什么行动。
- 失败代价制造紧迫。
- 赌注越明确，选择越有重量。
- 风险推动情绪拉扯。
- 新赌注接上下一段期待。

## 断档风险

- 只有口头目标，没有投入。
- 风险模糊。
- 角色想做什么和必须做什么脱节。
- 代价只在作者说明里。
- 套公式但没有事件。

## 训练任务

找一段主角行动，补三句：他押上什么，失败失去什么，为什么现在必须做。补不出来就重设冲突。
