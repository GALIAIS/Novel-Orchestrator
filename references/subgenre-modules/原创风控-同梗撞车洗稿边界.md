# 原创风控-同梗撞车洗稿边界

依据案例：`batch-186` 案例 927。

## 读者承诺

同梗撞车不是抄袭；文字搬运修改和大范围剧情照搬才是高风险洗稿。

## 核心循环

```text
公共题材梗 -> 自己组织剧情 -> 原创表达 -> 避免搬运 -> 平台风控降低
```

## 开篇必备

- 不复制文字。
- 不改词洗稿。
- 不大范围照搬剧情。
- 主角动机和事件组织自己设计。

## 期待来源

作者可以放心使用题材公共素材，把精力放在完成度和差异化表达上。

## 断档风险

- 换名字、倒词序。
- 桥段顺序照搬。
- 现实题材梗被误当私有。
- 没有自己的主角关系和冲突。

## 训练任务

拿一个公共梗，写出自己的主角、对手、冲突、结果和后续影响，再检查是否复制标杆桥段顺序。
