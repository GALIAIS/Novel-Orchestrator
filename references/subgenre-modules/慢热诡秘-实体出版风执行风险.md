# 慢热诡秘-实体出版风执行风险

依据案例：`batch-179` 案例 893。

## 读者承诺

慢热诡秘/实体出版风提供文笔、氛围、陌生世界适应过程和伏笔反转的沉浸体验。

## 核心循环

```text
陌生世界 -> 从容叙述 -> 氛围建立 -> 小反转 -> 伏笔回收 -> 世界继续展开
```

## 开篇必备

- 文笔和氛围能支撑慢节奏。
- 主角适应环境要有持续信息增量。
- 小反转要自然，伏笔不能太直白。
- 明确新人执行难度。

## 期待来源

读者期待沉浸、扎实和每章反转，而不是单纯即时爽点。慢热风有市场，但对笔力和结构控制要求高。

## 断档风险

- 节奏慢但文笔撑不住。
- 反套路过多后变成新套路。
- 伏笔太直白，读者提前猜到。
- 新人误以为慢热等于可以拖。

## 训练任务

写一个 3000 字慢热开头，用氛围、环境适应和一个小反转留住读者，并检查伏笔是否过于直白。
