# 反向阅读-不爽段铺垫取证

依据案例：`batch-043` 案例 215。

## 读者承诺

想从爽点里学结构，不能只盯高潮回味。要反向阅读爽点前那些不爽、慢、想跳读的段落，找出铺垫、转场和伏笔。

## 核心循环

```text
高潮爽点 -> 回看不爽段 -> 识别功能 -> 找铺垫伏笔 -> 迁移写法
```

## 开篇必备

- 先确定一个高潮。
- 不拆高潮本身。
- 回看高潮前的下行段。
- 标出每段结构功能。

## 期待来源

- 作者视角替代读者回味。
- 看清爽点如何被准备。
- 慢段不再只是拖。
- 训练出铺垫识别能力。

## 断档风险

- 把回味当拆书。
- 只摘爽点名场面。
- 不区分无效拖沓和必要下行。
- 没形成可仿写动作。

## 训练任务

选一个高潮，回看前 3000 字，标出 3 个本来想跳过但实际承担铺垫功能的段落。
