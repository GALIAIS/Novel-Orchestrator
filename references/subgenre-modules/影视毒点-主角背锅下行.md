# 影视毒点-主角背锅下行

依据案例：`batch-131` 案例 651。

## 读者承诺

下行剧情可以有，但不要让主角因犯蠢或圣母直接制造灾难。网文退出成本低，读者更难忍受“锅是主角的，苦也是读者吃的”。

## 核心循环

```text
外部压力制造下行 -> 主角承担补救 -> 反派/环境背锅 -> 主角力挽狂澜 -> 功劳回到主角
```

## 开篇必备

- 下行来源清楚。
- 主角不因低智制造灾难。
- 亲近角色死亡不能重复滥用。
- 主角必须有补救行动。
- 最终功劳要回到主角。

## 期待来源

- 读者替主角憋屈。
- 主角负责解决而不是背锅。
- 反派或环境造成压力。
- 补救成功产生爽感。

## 断档风险

- 主角圣母害死人。
- 本可解决却强行犯错。
- 成长节点重复剧情杀。
- 下行太久，主角只负责挨骂。

## 训练任务

设计一个 1000 字下行桥段：危机由反派或配角错误造成，主角承担补救责任并最终获得功劳。
