# 期待驱动-世界观成长与矛盾必然

依据案例：batch-352 案例 1756

## 读者承诺

读者追的不是单个悬念，而是主角每升一级、每换一个圈层后，世界都露出更高层的信息、资源和敌人。开篇先让读者好奇世界与主角，阶段推进时再把期待转成升级资源、新地图特点、配角回收和主线压力。

## 核心循环

```text
未知世界/主角 -> 阶段事件暴露信息 -> 目标资源出现 -> 矛盾必然爆发 -> 升级后打开更高期待
```

## 开篇必备

- 一个尚未说尽的世界缺口。
- 一个能让主角行动的阶段目标。
- 一个不靠主角没事找事也会爆发的矛盾。
- 至少一个可长期回收的配角或关系钩子。

## 期待来源

- 世界观尚未揭开的层级。
- 主角升级后能接触的新资源。
- 配角久别重逢与旧羁绊兑现。
- 极端反派、人设冲突、仇恨或幕后主线压力。

## 断档风险

- 开局把设定一次倒完，后续没信息可放。
- 主角目标只为当前一章服务，没有阶段延展。
- 反派冲突像临时碰瓷，不像必然爆发。
- 配角没有前史，重逢时无法放大情绪。

## 训练任务

给一个升级文设定 3 个阶段期待：村镇、宗门、上界。每阶段写“读者等什么、主角要什么、谁必然阻拦、升级后新增什么期待”。
