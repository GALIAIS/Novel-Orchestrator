# 都市爽文开篇-目标冲突赌注前置

依据案例：`batch-344` 案例 1717

## 读者承诺

都市爽文开篇要让读者三十秒内看清主角目标、奖励、冲突、赌注、金手指和第一记忆点。

## 核心循环

```text
主角目标 -> 可见奖励 -> 不公平阻碍 -> 金手指展示 -> 赌注兑现 -> 记忆点
```

## 开篇必备

- 目标要马上可见。
- 奖励金额或事件逻辑要合理。
- 金手指早展示。
- 冲突和赌注明确。
- 视角稳定，场景特征清楚。

## 期待来源

读者期待主角马上为了钱、校花、尊严或资源行动，并看到能力如何改变局面。

## 断档风险

- 18 元素打勾但不生效。
- 两章没有兑现点。
- 奖励设置不合逻辑。
- 分镜和视角混乱。
- 人物只有空白对话，没有动作表情。

## 训练任务

把开篇压成 500 字，只保留目标、奖励、阻碍、赌注、金手指首次效果和一个记忆点。
