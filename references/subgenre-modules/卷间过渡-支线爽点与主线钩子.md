# 卷间过渡-支线爽点与主线钩子

依据案例：`batch-209` 案例 1044。

## 读者承诺

卷间过渡要让读者喘息，但仍用支线爽点、战利品拓展或前置敌人扣回主线。

## 核心循环

```text
卷终收获 -> 轻松支线 -> 侧面装逼/搞笑 -> 战利品伏笔 -> 下一卷钩子
```

## 开篇必备

- 承接上一卷收获。
- 给一个轻松但有价值的支线爽点。
- 让战利品或配角线连接后文。
- 引出下一卷 boss 的前置痕迹。

## 期待来源

读者在高潮后获得喘息和收获感，同时看到下一段主线已经靠近。

## 断档风险

- 支线与主线无关。
- 写长像水文，写短没意义。
- 战利品只盘点不拓展。
- 下一卷没有提前露钩子。

## 训练任务

写 3 章卷间过渡小纲：收获爽点、支线侧面装逼、下一卷前置钩子，每章标注主线价值。
