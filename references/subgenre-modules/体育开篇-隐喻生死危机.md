# 体育开篇-隐喻生死危机

依据案例：`batch-153` 案例 762。

## 读者承诺

体育文开头不一定需要肉体生死危机，但必须有赛道内的前途危机：不突破，主角体育生涯、升学、阶层或人生路线就会结束。

## 核心循环

```text
低位困境 -> 前途危机 -> 天赋/机会 -> 抓住训练或比赛 -> 体育命运翻盘
```

## 开篇必备

- 主角低位处境。
- 不解决会失去的前途。
- 体育赛道内的机会。
- 第一次天赋或金手指信号。
- 第一场能证明自己的事件。

## 期待来源

读者期待主角从“没机会就完了”的困境里，通过体育天赋、训练或比赛抓住命运分岔点。

## 断档风险

- 冷门题材没有足够危机。
- 只写日常训练，不写前途代价。
- 把肉体死亡硬塞进不合适题材。
- 读者不明白为什么主角必须立刻行动。

## 训练任务

写一个体育文前三章设定：第一章给前途危机，第二章给天赋/机会，第三章给第一次证明。每章写清“不成功会失去什么”。
