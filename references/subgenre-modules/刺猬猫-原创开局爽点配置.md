# 刺猬猫-原创开局爽点配置

依据案例：`batch-194` 案例 969。

## 读者承诺

刺猬猫原创开篇要优先交付强冲突、主角装逼和女主/攻略期待，玩梗只能加分，不能替代爽点。

## 核心循环

```text
强冲突 -> 主角装逼 -> 女主期待 -> 圈层梗加味 -> 章末继续追
```

## 开篇必备

- 第一章有足够激烈的冲突。
- 主角有清晰优势展示。
- 女主或关系线有可期待方向。
- 梗不影响主线理解。

## 期待来源

读者能快速看到主角为什么值得追、关系线为什么可期待、作品为什么属于刺猬猫口味。

## 断档风险

- 只有萌点，没有冲突。
- 只有小女孩，没有主角爽点。
- 玩梗替代剧情。
- 原创开篇按同人方式依赖读者自带期待。

## 训练任务

写一个 8000 字四章小纲：每章分别承担强冲突、装逼、女主期待、章末钩子，梗只能作为氛围补充。
