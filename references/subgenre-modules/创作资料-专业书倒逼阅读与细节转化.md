# 创作资料-专业书倒逼阅读与细节转化

依据案例：`batch-205` 案例 1023。

## 读者承诺

写小说会倒逼作者阅读专业资料；资料要转成动作、道具、流程和决策，而不是正文科普。

## 核心循环

```text
题材需求 -> 阅读专业资料 -> 提取细节 -> 转成角色行动 -> 文本可信
```

## 开篇必备

- 明确题材需要哪些专业知识。
- 只提取能进入剧情的细节。
- 把术语转成动作或选择。
- 删除炫知识段。

## 期待来源

读者看到具体流程和工具，会相信角色真的懂行。

## 断档风险

- 为资料而资料。
- 大段科普拖节奏。
- 术语堆砌，没有剧情用途。
- 专业细节和人物行动脱节。

## 训练任务

选一本专业书摘 10 个知识点，每个转成一个角色动作、道具细节或剧情选择。
