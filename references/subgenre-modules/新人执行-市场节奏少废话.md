# 新人执行-市场节奏少废话

依据案例：`batch-021` 案例 105。

## 读者承诺

给新人一个最小执行闭环：看市场、定核心点、围绕主角视角和金手指写冲突，一个故事接一个故事，少说废话。

## 核心循环

```text
看市场 -> 定卖点 -> 主角视角 -> 金手指冲突 -> 故事闭环 -> 删废话 -> 写
```

## 开篇必备

- 投稿网站和市场风向。
- 主角视角。
- 金手指。
- 第一矛盾或冲突。
- 第一故事闭环。

## 期待来源

- 读者迅速知道看什么。
- 金手指和冲突直接发生反应。
- 每个故事闭环都接下一个故事。

## 断档风险

- 追求噱头，不看市场需求。
- 模仿大神慢展开。
- 开篇堆辞藻。
- 只总结不写。

## 训练任务

每天写前先填 8 行卡：目标网站、市场主流、主角、金手指、第一冲突、第一爽点、要删废话、今日字数。
