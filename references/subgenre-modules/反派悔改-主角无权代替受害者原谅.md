# 反派悔改-主角无权代替受害者原谅

依据案例：`batch-114` 案例 567。

## 读者承诺

反派悔改线承诺的是“复杂情绪下仍有清算”，不是“悲惨过去抵消罪行”。主角的选择会定义主角人设，不能替受害者原谅不可逆恶行。

## 核心循环

```text
反派作恶 -> 悔改增加复杂度 -> 读者等待代价 -> 主角做出处置 -> 人设被验证 -> 反派完成落幕
```

## 开篇必备

- 明确反派恶行边界。
- 区分同情遭遇和取消代价。
- 主角立场必须清楚。
- 被害者权益不能被主角一句话抹掉。
- 落幕要和主角选择有关。

## 期待来源

- 读者等待反派付出代价。
- 主角坚持原则带来人设稳定。
- 真悔改可以带来牺牲、自赎或赎罪张力。
- 清算完成后情绪闭环。

## 断档风险

- 主角放过十恶不赦反派。
- 用路人补刀修复主角的错误选择。
- 把悲惨过去写成免罪牌。
- 洗白角色前期作恶过重。
- 反派落幕与主角无关。

## 训练任务

给一个悔改反派设计三种代价：主角处决、反派自赎牺牲、活着赎罪。每种都写清主角做了什么选择，以及读者为什么不会觉得主角圣母。
