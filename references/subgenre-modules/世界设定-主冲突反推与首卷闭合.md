# 世界设定-主冲突反推与首卷闭合

依据案例：`batch-398` 案例 1988

## 读者承诺

世界设定做足但不做满：从主角能力、主要压力和首卷目标反推资源、制度与势力，只补足能解释本卷人物选择和后果的规则。

## 核心循环

```text
画风主角与首卷目标 -> 主冲突反推资源制度 -> 各剧情线补所需设定 -> 检查重大选择闭合 -> 按当前因果补缺 -> 卷后扩展世界
```

## 开篇必备

- 基本画风和类型语法清楚。
- 主角能力击中真实世界缺口。
- 主反派或结构压力可见。
- 第一卷必须改变的局部秩序明确。
- 关键地点和势力服务计划内事件。
- 暂不影响因果的内容只作占位。

## 期待来源

读者通过主角承受压力、使用能力和改变局部秩序逐层认识世界，而不是先阅读一份与行动无关的百科。

## 断档风险

- 开写前试图填满整个世界。
- 只有模板没有主冲突。
- 设定新奇却不改变任何选择。
- 每发现漏洞就扩出更多无关设定。
- 复制他作独特名词与事件链。
- 首卷结束仍未触及世界核心压力。

## 训练任务

画“主冲突—资源—制度—能力缺口”图，列首卷五个重大选择；只补足解释这些选择与后果的规则，其余灵感进入待扩清单，三章试写后再补一次。
