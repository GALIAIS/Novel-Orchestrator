# 小说封面委托-版权清单与缩略图验收

依据案例：`batch-416` 案例 2077

## 读者承诺

封面会在手机缩略图中清楚传达书名、主类型和核心情绪，并与简介及前三章保持同一合同。自制、套版、半定制或原创委托均须留下图片、字体、模板和生成素材的权利证据；交付不仅是一张大图，还包括无字底图、可编辑源文件、多平台裁切和明确修改、独占、广告及衍生范围。

## 核心循环

```text
写明书名、类型、读者和情绪的封面简报 -> 按预算、独特性、时限和权利风险选制作路径 -> 开工前核对图片、字体、模板与生成素材许可 -> 用草图确认构图和信息层级 -> 交付原图、无字底图、源文件与多尺寸版本 -> 在缩略图、灰度、低亮度和裁切中验收 -> 书面确认修改、独占、广告与衍生范围 -> 归档凭证并随包装变化复测
```

简报控制题材匹配，权利链降低下架风险，多尺寸测试对应真实点击场景，源文件和书面范围支持长期迭代。

## 开篇必备

- 书名、笔名、主副类型、核心情绪和目标读者。
- 必须出现与禁止出现的物件、人物和意象。
- 手机列表与不同平台的当前尺寸要求。
- 自制、套版、半定制或原创委托的选择理由。
- 图片、字体、模板和生成素材的来源与许可。
- 预算、时限、草图节点和修改轮次。
- 原图、无字底图、可编辑源文件和裁切版本。
- 商用、独占、广告、衍生和制作人展示权。
- 缩略图、灰度、低亮度和误导性验收结果。

## 期待来源

- 缩略图能否在一秒内读清书名。
- 画面与字体是否发出正确题材信号。
- 核心人物或物件会引出何种正文承诺。
- 不同裁切是否仍保留最重要信息。
- 包装变化时能否用源文件快速调整。
- 作品进入广告或衍生阶段时权利是否完整。

## 断档风险

- 只比较价格与交期，不核对题材和权利。
- 把聊天承诺当成完整商业许可。
- 使用来历不明的图片、字体或模板。
- 要求制作人照抄参考图。
- 只验收大图，列表缩略图无法辨认书名。
- 没有无字底图和源文件，改名必须重做。
- 独占、重复售卖和广告用途未书面约定。
- 沿用旧平台尺寸和旧商家价格。
- 封面吸引的人群与正文承诺不一致。

## 训练任务

写一页封面简报并制作自制、套版、原创三条路径的成本与权利表。准备两个方向稿，在原尺寸、120 像素宽、灰度、低亮度和两种裁切中测试；验收必须覆盖图片、字体、源文件、修改、独占、广告和衍生权限。
