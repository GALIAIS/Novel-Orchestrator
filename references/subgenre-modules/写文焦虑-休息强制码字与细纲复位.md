# 写文焦虑-休息强制码字与细纲复位

依据案例：batch-354 案例 1767

## 读者承诺

写文焦虑要先判断类型：疲劳就休息和转移，启动困难就用锁字数、拼字房、倒计时，故事卡住就重拉细纲。处理对症，产出才回得来。

## 核心循环

```text
焦虑/不想写 -> 判断原因 -> 休息转移/强制启动/细纲复位 -> 最低字数恢复产出
```

## 开篇必备

- 焦虑类型判断。
- 允许休息时间。
- 外部强制工具。
- 要复盘的细纲范围。

## 期待来源

- 暂时转移让脑子恢复。
- 外部倒计时激发胜负欲。
- 细纲重拉找回故事方向。
- 存稿承担低谷期风险。

## 断档风险

- 把逻辑卡住误判成心情不好。
- 一直休息变逃避。
- 强制码字但方向错。
- 没有存稿导致焦虑放大。

## 训练任务

建立一个三栏清单：疲劳时做什么、逃避时做什么、逻辑卡住时做什么。每栏只留 3 个动作，写不动时照表执行。
