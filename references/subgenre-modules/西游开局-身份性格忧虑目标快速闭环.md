# 西游开局-身份性格忧虑目标快速闭环

依据案例：batch-357 案例 1785

## 读者承诺

高效开局可以先交代主角来历、身份和性格，随后立刻给核心忧虑，再由点路者指出目标。读者快速知道“他是谁、怕什么、要去哪”。

## 核心循环

```text
来历身份性格 -> 核心忧虑 -> 智者点路 -> 长线目标 -> 冒险启动
```

## 开篇必备

- 主角来历。
- 主角身份和性格。
- 一个根本欲望或恐惧。
- 点明出路的人或事件。

## 期待来源

- 读者迅速认识主角。
- “不想死”这类根本忧虑天然有力。
- 成仙成佛式长线目标清晰。
- 目标出现后马上启动冒险。

## 断档风险

- 只介绍身份，不给行动理由。
- 忧虑太小，撑不起长线。
- 智者只解释世界，不点出目标。
- 学古典语言而忽略结构效率。

## 训练任务

写 300 字开局，按顺序完成：来历、性格、忧虑、出路、第一步行动。
