# 番茄首秀-好评燃料与数据考验

依据案例：batch-303 案例 1514

## 读者承诺

首秀前好评是写作燃料，不是过关结论；要把读者夸点转成后续强化方向。

## 核心循环

```text
收到好评 -> 分类夸点 -> 放大承诺 -> 首秀看数据 -> 复盘调整
```

## 开篇必备

- 记录好评数量。
- 摘出长评关键词。
- 分类读者喜欢点。
- 首秀后看完读、追读、评分变化。

## 期待来源

- 作者获得继续写的动力。
- 读者夸点指向作品优势。
- 数据验证承诺是否稳。

## 断档风险

- 把早期好评当最终胜利。
- 只开心不复盘。
- 首秀后数据变化不跟踪。

## 训练任务

整理 20 条评论，分成角色、爽点、节奏、设定、情绪五类，每类写一个加强动作。
