# 女性美感描写-联想通感避线

依据案例：`batch-042` 案例 206。

## 读者承诺

女性角色的美和吸引力可以成为关系爽点、奖励感和角色魅力的一部分，但表达要让读者产生联想，而不是用直白描写硬顶审核边界。

## 核心循环

```text
美感目标 -> 正面细节 -> 侧面证明 -> 联想通感 -> 关系期待
```

## 开篇必备

- 角色出场服务场景。
- 正面细节少而准。
- 有他人反应或社会证明。
- 吸引力指向关系、冲突或主角奖励感。

## 期待来源

- 读者自行脑补美感。
- 角色被场景中其他人注意。
- 美貌和性格、身份、危险或利益绑定。
- 读者期待主角与她发生关系变化。

## 断档风险

- 只堆身体部位。
- 正面描写过界。
- 侧面描写太虚，读者感受不到。
- 美貌与剧情、关系、冲突无关。

## 训练任务

写一个女性角色出场：只用 3 个正面细节、2 个他人反应和 1 个动作建立吸引力。
