# 飞卢节奏-下行点到即止

依据案例：`batch-020` 案例 097。

## 读者承诺

学习快节奏装逼文的轻压快回。下行只负责制造一瞬间误判，不负责长时间压抑读者。

## 核心循环

```text
优势铺垫 -> 一句误判 -> 快速兑现 -> 震惊反应 -> 更高目标
```

## 开篇必备

- 主角优势清楚。
- 误判角色清楚。
- 回升路径极短。
- 见证反应立刻出现。

## 期待来源

- 读者知道主角能打破误判。
- 路人和亲友轻微下压。
- 主角马上兑现，让上升曲线不断。

## 断档风险

- 把飞卢式下行写成长时间憋屈。
- 误判重复太多句。
- 震惊反应缺席。
- 只学夸张口吻，不学节奏密度。

## 训练任务

写 3 个 300 字片段，每个片段只允许一次下行，且下行后 100 字内必须开始回升。
