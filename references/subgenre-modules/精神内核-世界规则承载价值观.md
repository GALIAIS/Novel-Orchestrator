# 精神内核-世界规则承载价值观

依据案例：`batch-185` 案例 921。

## 读者承诺

爆款网文的价值观最好尽早出现，并通过世界规则、人物关系或不公平伤害变成剧情事件。

## 核心循环

```text
精神内核 -> 前三章触发 -> 世界规则承载 -> 情绪事件证明 -> 读者接受价值观
```

## 开篇必备

- 明确作品想表达的规则冲突。
- 前三章给出价值观入口。
- 避免主角直接说教。
- 用世界规则或不公平伤害承载主题。

## 期待来源

读者期待平凡者借规则漏洞或群众力量撬动高位格，获得打破旧秩序的爽感。

## 断档风险

- 价值观只停留在口号。
- 世界规则和主题无关。
- 人设承载主题但后续崩掉。
- 战斗描写只堆辞藻，不体现差距。

## 训练任务

设计一个世界规则，让低位格能借“公平/民意/天道”伤害高位格，并在前三章触发一次。
