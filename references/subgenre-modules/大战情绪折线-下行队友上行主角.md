# 大战情绪折线-下行队友上行主角

依据案例：`batch-321` 案例 1604

## 读者承诺

大战役可以多轮上下折线，但每次下行要证明敌人强，每次上行要让主角或阵营有效推进，主角全程不能无用。

## 核心循环

```text
强敌压迫 -> 阵营小上行 -> 敌人更强下行 -> 主角关键行动 -> 更大代价 -> 最终反转胜利
```

## 开篇必备

- 终极敌人足够强。
- 队友努力能制造代价和层次。
- 主角每轮关键行动有正面作用。
- 失败原因是敌人强，不是主角蠢。
- 最终反转回收前面绝望。

## 期待来源

读者期待希望不断出现又被压下，最后由主角承担最大风险完成反转。

## 断档风险

- 折线多但每轮没有推进。
- 队友只送死，不改变局势。
- 主角失败显得无能。
- 最终反转没回收前面铺垫。

## 训练任务

写一场终局战 6 段折线：3 次下行、3 次上行，并标注每次上行是谁的有效行动。
