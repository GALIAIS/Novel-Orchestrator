# 校园开篇-找茬理由与退路封死

依据案例：`batch-339` 案例 1691

## 读者承诺

校园开篇可以直接拉矛盾，但找茬理由要成立，主角退路要被封死，人物不能靠降智推动。

## 核心循环

```text
合理找茬理由 -> 主角不能离开 -> 重要角色到场 -> 冲突升级 -> 关系承诺显形
```

## 开篇必备

- 找茬理由有利益、面子或关系伤害。
- 主角不能简单走掉。
- 配角行为有成本逻辑。
- 对话像正常人。
- 女主或强敌到场升级关系链。

## 期待来源

读者期待开篇冲突自然卷住主角，并引出校园关系、竞争和情绪承诺。

## 断档风险

- 为冲突而冲突。
- 写真、传单、后援团等理由太轻。
- 主角可一走了之。
- 人物集体降智。
- 古早套路没有新处理。

## 训练任务

重写一个被找茬开头：给找茬硬理由、堵住主角退路、让重要角色到场升级冲突。
