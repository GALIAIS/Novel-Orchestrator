# 感情戏-不会写就扬长避短

依据案例：`batch-328` 案例 1636

## 读者承诺

不会写感情戏时，不要硬塞；可以拆素材后再写，也可以直接不写或改写友情。

## 核心循环

```text
不会写感情戏 -> 硬写成毒点 -> 拆素材/改友情/删除 -> 主线优势保住
```

## 开篇必备

- 判断这本书是否必须有爱情线。
- 找精彩素材，分出互动类型。
- 女性崇拜男主要有具体事件支撑。
- 写不好爱情时先写友情或战友情。
- 不让生硬感情戏打断主线爽点。

## 期待来源

读者期待关系互动能强化主角价值，而不是硬塞尴尬桥段。

## 断档风险

- 不会写却硬写。
- 女角色无理由崇拜。
- 感情戏脱离主线。
- 本来能靠主线火，却被失败感情线拖死。

## 训练任务

拆 3 段优秀互动，只记录事件、情绪、关系变化；拆不出来就先不写爱情线。
