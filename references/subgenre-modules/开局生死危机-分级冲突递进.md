# 开局生死危机-分级冲突递进

依据案例：`batch-195` 案例 972。

## 读者承诺

生死危机开局先解决第一级冲突，让主角脱离当前死局；复仇、反杀和更大势力可以后置。

## 核心循环

```text
眼前死局 -> 金手指救场 -> 一级冲突解决 -> 仇恨保留 -> 更大冲突牵出
```

## 开篇必备

- 当前危机足够清楚。
- 金手指能改变眼前处境。
- 主角获得继续行动资格。
- 反派或幕后势力留下后续钩子。

## 期待来源

读者先看到主角活下来并掌握新能力，再期待之后清算旧敌、牵出更大敌人。

## 断档风险

- 开局一次性清空所有矛盾。
- 金手指只展示不解决问题。
- 反派硬拖不清算，显得憋屈。
- 后置复仇没有明确钩子。

## 训练任务

写一个开局死局，限定第一章只解决“活下来”，不杀主犯，并在章末留下下一层敌人。
