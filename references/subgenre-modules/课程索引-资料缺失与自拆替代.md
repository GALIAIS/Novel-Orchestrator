# 课程索引-资料缺失与自拆替代

依据案例：`batch-222` 案例 1107。

## 读者承诺

当权威课程或内部资料不可得时，把“求资源”转成可执行的样本拆解路径，降低新人学习焦虑。

## 核心循环

```text
想学题材 -> 课程不可得 -> 选公开样本 -> 拆核心循环 -> 形成自用模板
```

## 开篇必备

- 明确想学的题材和能力缺口。
- 记录不可获取的课程或资料边界。
- 选择 3-5 个公开样本替代。
- 先拆前三章承诺，再拆长期循环。

## 期待来源

作者获得的不是课程原文，而是一条可重复使用的学习路线：样本、维度、记录表、输出模板。

## 断档风险

- 伪造无法读取的课程内容。
- 只收藏资料，不转成拆书任务。
- 样本选择太散，无法归纳题材规律。
- 只看技巧，不拆章节功能。

## 训练任务

选一本无限流样本拆 10 章，记录副本规则、主角目标、死亡风险、奖励反馈、队友关系、主线线索，并总结前三章模板。
