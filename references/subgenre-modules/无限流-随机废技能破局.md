# 无限流-随机废技能破局

依据案例：`batch-145` 案例 724。

## 读者承诺

主角每次副本前抽到随机技能，技能看似废物，任务却高压严肃。读者追看的核心是：这些离谱技能如何被组合成通关方案。

## 核心循环

```text
抽技能 -> 技能废物化展示 -> 高压任务 -> 寻找隐藏用途 -> 组合破局 -> 下一关更难
```

## 开篇必备

- 清楚的副本任务。
- 3 个表面无用的技能。
- 任务压力和技能反差。
- 技能参与破局的明确过程。
- 通关后抬高下一副本期待。

## 期待来源

读者期待解谜爽和反差爽：越不像能用的技能，越想看作者如何合理用它解决高压任务。

## 断档风险

- 废技能只是搞笑，不参与破局。
- 解法太牵强，读者觉得降智。
- 副本题材重复，疲劳。
- 只有主观推荐，没有客观成绩或具体桥段。

## 训练任务

设计一个原创无限流第一副本：任务是潜入或刺杀，主角只能使用 3 个生活类废技能。写出每个技能的隐藏用途和组合解法，并用一句话抬高下一关难度。
