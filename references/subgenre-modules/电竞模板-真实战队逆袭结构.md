# 电竞模板-真实战队逆袭结构

依据案例：`batch-225` 案例 1121。

## 读者承诺

把真实电竞队伍的草根起步、质疑、磨合、晋级和爆冷改写成虚构战队成长线。

## 核心循环

```text
草根起步 -> 外界质疑 -> 队内磨合 -> 小胜证明 -> 强敌挑战 -> 阶段晋级
```

## 开篇必备

- 虚构队伍有明确低起点。
- 给出外界不看好的理由。
- 队内至少有一个短板。
- 第一阶段目标清楚。

## 期待来源

读者期待弱队证明自己、团队磨合成型、战术升级后击败强敌。

## 断档风险

- 直接照搬真实战队人物和商标。
- 只写比赛操作，不写队伍成长。
- 没有阶段目标和舆论压力。
- 真实模板只抄事件，不抽结构。

## 训练任务

选一支真实电竞队伍或体育队伍，抽 8 个结构节点：起点、质疑、核心成员、第一次小胜、第一次大败、战术转折、关键强敌、阶段胜利，改写成虚构战队第一卷大纲。
