# 末世开局切入-秩序演化

依据案例：`batch-160` 案例 798。

## 读者承诺

根据后续主线选择末世开篇时间点，让第一章立刻兑现末世的崩坏、压迫、资源危机或新秩序冒险。

## 核心循环

```text
确定类型承诺 -> 选择降临/稳定后切入 -> 给出第一场危机 -> 主角崭露头角 -> 秩序演化
```

## 开篇必备

- 写末世降临，就要有倒计时、混乱和生存压力。
- 写稳定后世界，就要承认它更接近新秩序冒险。
- 第一章展示主角如何在危机中占据优势。
- 后续升级路径要与开篇承诺一致。

## 期待来源

读者期待从崩坏到求生，从求生到团队，从团队到势力，再到新秩序建立。

## 断档风险

- 名义末世，开篇却没有压迫感。
- 世界已稳定却还按灾难片写。
- 中段突然从打怪升级跳到种田争霸但没有过渡。
- 开头只问时间点，不问后续要写什么。

## 训练任务

把一个末世设定分别写成“降临当天”和“稳定后十年”两个开篇梗概，标出各自读者期待，并选择更适合主线的版本。
