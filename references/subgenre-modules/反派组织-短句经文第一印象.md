# 反派组织-短句经文第一印象

依据案例：`batch-228` 案例 1136。

## 读者承诺

用一句经文、口号或诗句快速立住反派组织的狂、疯、危险和价值观。

## 核心循环

```text
短句口号 -> 冒犯权威 -> 读者情绪反应 -> 组织印象成立 -> 后续兑现
```

## 开篇必备

- 口号体现组织信仰。
- 口气要有辨识度。
- 冒犯对象清楚。
- 后续剧情兑现它的狂妄和代价。

## 期待来源

读者通过一句话就判断这个组织危险、疯癫或作死，并期待它如何搅动剧情。

## 断档风险

- 只有设定说明，没有记忆点。
- 口号狂，但组织行动不兑现。
- 冒犯对象不清，读者无感。
- 诗句太虚，无法转成冲突。

## 训练任务

为一个邪教、反派组织或秘密社团写 5 条短口号。每条体现信仰、冒犯对象、危险程度，并让读者能用一个词概括第一印象。
