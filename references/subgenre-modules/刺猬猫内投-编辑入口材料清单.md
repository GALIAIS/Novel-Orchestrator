# 刺猬猫内投-编辑入口材料清单

依据案例：batch-303 案例 1511

## 读者承诺

刺猬猫内投先解决流程问题：投谁、投什么、字数和大纲是否符合要求。

## 核心循环

```text
确定平台 -> 查询编辑入口 -> 整理正文/大纲 -> 投递 -> 记录反馈
```

## 开篇必备

- 目标频道明确。
- 正文和大纲达到基本要求。
- 记录后台分配编辑。
- 备份私投入口和投递日期。

## 期待来源

- 投稿焦虑下降。
- 材料完整，反馈更可追踪。
- 流程和作品问题分开处理。

## 断档风险

- 把换编辑当成作品问题解法。
- 材料不齐就投。
- 联系方式过期或隐私外泄。

## 训练任务

做一张内投清单：编辑入口、正文文件、大纲文件、题材标签、投递日期、反馈状态。
