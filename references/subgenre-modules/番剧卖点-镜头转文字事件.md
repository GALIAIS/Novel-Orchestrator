# 番剧卖点-镜头转文字事件

依据案例：batch-261 案例 1303

## 读者承诺

番剧能用画面、音乐、镜头快速建立角色，网文要把这些卖点转成事件、反应和持续行动。读者通过文字事件认识人设。

## 核心循环

```text
视听卖点 -> 拆成角色特征 -> 转成具体事件 -> 前三章展示 -> 大纲持续兑现
```

## 开篇必备

- 参考作品的卖点。
- 角色最可感的行为。
- 一个能展示人设的事件。
- 开头三章的卖点展示。
- 后续大纲承接。

## 期待来源

读者期待在前几章就看到可爱、向往、成长或爽点，而不是等作者解释“这个角色以后会好看”。

## 断档风险

只搬镜头氛围，文字会空；只有开头卖点没有大纲，后面会迅速变水。

## 训练任务

把一个番剧镜头改成一个 500 字网文事件，禁止写“她很可爱”，必须让行为和他人反应证明。
