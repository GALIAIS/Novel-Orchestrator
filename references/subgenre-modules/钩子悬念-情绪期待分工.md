# 钩子悬念-情绪期待分工

依据案例：`batch-060` 案例 300。

## 读者承诺

这个模块用于区分钩子、悬念、拉情绪、拉期待。它承诺的是把“放置剧情的方式”和“剧情产生的作用”拆开，避免概念混用。

## 核心循环

```text
放置钩子/悬念 -> 触发情绪波动 -> 形成后续期待 -> 小高潮兑现部分 -> 新线索接大地图
```

## 开篇必备

- 先标注这是钩子还是悬念。
- 再标注它拉的是情绪、期待，还是两者兼有。
- 情绪点和期待点分开写。
- 假线索要有读者低估理由。
- 真线出现要接上更大地图。

## 期待来源

- 读者以为只是幌子，后面发现是真的。
- 低层级敌人牵出高层级世界。
- 情绪被反转拉起。
- 新地图或新势力形成追读。
- 主角人设处理局面带来核心期待。

## 断档风险

- 把钩子、悬念、情绪、期待混成一个词。
- 假线索没有成真价值。
- 真线出现太早，没有低估过程。
- 只拉情绪，不接新期待。
- 只设悬念，不服务主线。

## 训练任务

写一个虚晃一枪桥段。用四行标注：钩子是什么、悬念是什么、拉起的情绪是什么、形成的期待是什么。
