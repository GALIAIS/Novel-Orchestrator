# 主角正当性-必要之恶

依据案例：`batch-197` 案例 983。

## 读者承诺

主角做恶可以成立，但必须让读者看到实害、无替代方案、退路被堵死和代价。

## 核心循环

```text
危险对象 -> 实害/灾难证明 -> 主角无退路 -> 必要恶行 -> 承担代价
```

## 开篇必备

- 读者知道为什么非做不可。
- 对方危险不只停留在口头说明。
- 主角没有低成本替代方案。
- 恶行后有代价或心理压力。

## 期待来源

读者接受灰色主角，是因为他在解决无法回避的矛盾，而不是随意伤害无辜。

## 断档风险

- 让读者先代入受害者。
- 对方未作恶就被处刑。
- 只说能力危险，没有展示实害。
- 主角有办法却不救，压抑读者情绪。

## 训练任务

写一个必要之恶桥段，逐项补齐实害、无替代方案、退路被堵死和主角代价。
