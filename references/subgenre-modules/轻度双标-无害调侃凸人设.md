# 轻度双标-无害调侃凸人设

依据案例：`batch-034` 案例 170。

## 读者承诺

轻度双标可以做喜剧和人设，但前提是事件无害、关系安全、后果可接受。读者笑的是见色忘友式反差，不是主角道德破产。

## 核心循环

```text
小请求 -> 主角偏心 -> 朋友夸张控诉 -> 旁人劝架/吐槽 -> 气氛变轻
```

## 开篇必备

- 双标事项必须是小事。
- 被拒方不能受实质伤害。
- 关系基调轻松安全。
- 反应要夸张，后果要轻。

## 期待来源

- 读者预判主角会偏心。
- 基友吐槽放大笑点。
- 旁人劝架形成经典节奏。
- 人设在小事里变鲜明。

## 断档风险

- 把小事双标挪到救人、尊严、重大利益。
- 主角造成严重后果还无悔意。
- 被双标者真的受伤。
- 轻松调侃写成价值观崩坏。

## 训练任务

写一个轻喜剧小桥段：主角对朋友和暧昧对象双标，但只限一件小事，最后必须用吐槽或劝架收束。
