# 多女性角色感情线-关系承诺分层与选择闭合

依据案例：`batch-411` 案例 2051

## 读者承诺

多名高参与女性角色分别拥有独立目标和关系功能；亲密信号、排他优先级与共同代价形成不同强度承诺，每卷推进一次关系状态，最终以有代价选择、知情回应和明确边界闭合。

## 核心循环

```text
确定关系模式 -> 定义角色独立目标和关系功能 -> 建立亲密信号与边界 -> 共同选择累积代价 -> 每卷确认拒绝或转型 -> 主角作出未来选择 -> 各角色知情回应 -> 结局闭合边界与人物弧
```

## 开篇必备

- 单偶、多人、无恋爱或开放式关系合同。
- 主角对亲密关系的稳定价值。
- 每名主要角色的独立目标。
- 关系功能与恋爱强度分开记录。
- 暧昧的阶段增量和最迟决策点。
- 未被选择者的回应与后续路径。

## 期待来源

独特理解和共同选择先建立心动，未来不可兼得带来拉扯；最终行动证明主角愿为哪段关系支付代价，其他角色也能主动接受、拒绝或离开。

## 断档风险

- 所有出彩女性自动成为恋爱候选。
- 用戏份多少代替亲密因果。
- 几十万字重复吃醋却无状态变化。
- 古代背景被当作免除人物感受的理由。
- 结尾一句“知己”抹掉前期排他承诺。
- 未被选择者失去自己的目标和退出权。

## 训练任务

为主角与三名高参与角色各写关系卡，标出目标、信号、边界、共同代价和本卷变化；分别写单偶、无恋爱和开放结局的关键选择场景，保留与前期承诺最一致且所有角色仍有主体性的一版。
