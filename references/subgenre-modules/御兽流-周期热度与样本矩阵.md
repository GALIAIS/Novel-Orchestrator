# 御兽流-周期热度与样本矩阵

依据案例：batch-383 案例 1911

## 读者承诺

御兽流研究要同时查看历史先行者、成熟榜书和当期新书。只有逐本读正文后，才能区分长期机制、当期包装和单书特例。

## 核心循环

```text
跨周期选样 -> 逐本精读 -> 记录关系/成长/战斗 -> 比较共性差异 -> 判断切入方向
```

## 开篇必备

- 历史、成熟、新书三类样本。
- 数据日期和来源。
- 御兽关系与成长记录项。
- 共性和差异分栏。

## 期待来源

- 找到热度周期中的稳定需求。
- 看清现象级作品的独有创新。
- 识别新人当前切入方式。
- 避免照搬已疲劳包装。

## 断档风险

- 用一本爆款代表全部御兽流。
- 把推荐书单当成完成分析。
- 使用过期成绩判断当前市场。
- 未读正文就从书名概括特征。

## 训练任务

历史、成熟、新书各选一本，人工读前 20 章，记录御兽关系、成长、战斗和钩子，再区分三书共同项与单书特例。
