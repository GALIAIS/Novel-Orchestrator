# 目标渴望-冲突加深必要性

依据案例：batch-274 案例 1370

## 读者承诺

把普通目标变成主角非做不可的目标。冲突不是装饰，而是证明目标必要性、加深主角渴望和读者期待的工具。

## 核心循环

```text
普通目标 -> 冲突伤害 -> 缺口扩大 -> 目标变必要 -> 行动更坚决
```

## 开篇必备

- 先写主角想要什么。
- 再写不达成会失去什么。
- 冲突要贴近主角缺口。

## 期待来源

读者期待来自“非做不可”。赚钱、变强、复仇、得宝都要被冲突压成必要行动。

## 断档风险

- 目标只是想做，不够必要。
- 冲突和目标脱节。
- 代价不清，读者不着急。

## 训练任务

给赚钱、变强、得宝、进组织、打败敌人各加一个冲突，把它们改成必须达成的目标。
