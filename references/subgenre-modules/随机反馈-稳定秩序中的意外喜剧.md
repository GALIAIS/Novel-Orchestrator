# 随机反馈-稳定秩序中的意外喜剧

依据案例：`batch-222` 案例 1109。

## 读者承诺

正常规则被局部随机或反转反馈扭曲，主角用反常操作获得超常收益，读者持续期待下一次意外结果。

## 核心循环

```text
稳定规则 -> 局部异常反馈 -> 旁人认真承接 -> 主角发现规律 -> 反常操作收益
```

## 开篇必备

- 先建立正常预期。
- 明确异常只作用在掉落、奖励、好感、词条等反馈层。
- 人物性格和世界逻辑保持稳定。
- 主角要能摸到模糊规律并主动利用。

## 期待来源

乐趣来自“意外”而不是纯随机：读者知道世界大体稳定，因此关键反馈错位时才有喜剧冲击。

## 断档风险

- 到处随机，读者无法建立期待。
- 人物行为也随机，角色形象崩坏。
- 主角只被动抽奖，失去主观能动性。
- 异常反馈没有剧情承接，只剩段子。

## 训练任务

写 5 个正常行为得到反向反馈的桥段，每个包含正常预期、异常规则、实际反馈、旁人误读、主角收益；再写 3 个 NPC 严肃台词加随机物品错配的喜剧场面。
