# 女玄团宠无敌-代理出手与本体延迟兑现

依据案例：`batch-413` 案例 2063

## 读者承诺

高位或近乎无敌的女主可以延迟亲自出手，但每章仍要兑现阵营优势、能力线索或关系进展。萌宝、侍女、师父和部下先清理低阶阻碍，胜利必须由女主过去或当前的选择触发；每次代理兑现同时暴露边界，把“她为何必须亲自出手”推向更具体的强制条件。首次本体行动最终解决代理机制无法处理的规则、身份或情感问题。

## 核心循环

```text
展示女主的高位承诺 -> 代理人因其命令、关系或资源快速取胜 -> 小胜提供即时正反馈 -> 代理边界暴露更高层问题 -> 本体佯动部分兑现新线索 -> 双线交叉改变证据 -> 马甲升级解锁权限并增加责任 -> 代理方案失效迫使女主选择 -> 本体出手集中回收能力期待 -> 更高阶债务接棒
```

短线靠代理胜利供给，长线靠本体上限和强制出手条件拉动。延迟没有固定章数，只有“本次是否提供新兑现、是否让剩余债务更具体”的检验。

## 开篇必备

- 女主高位、无敌或多重身份承诺的可见证据。
- 至少一名与女主存在明确因果关系的代理人。
- 代理人可解决的低阶冲突和不可解决的边界。
- 一次快速胜利及其给女主带来的状态变化。
- 一次本体佯动所提供的部分兑现。
- 明确且逐步收紧的亲自出手条件。
- 两条各有目标、进展和局部结算的故事线。
- 马甲的权限、责任、敌人和升级代价。
- 第一次本体出手必须回收的前序铺垫。

## 期待来源

- 女主真正实力与代理人的差距。
- 她为何选择不亲自出手。
- 哪一类问题代理人一定无法处理。
- 每次佯动新增了何种能力证据。
- 两条信息差线何时首次交叉。
- 马甲升级会开放什么权限并招来什么责任。
- 谁能逼迫女主在不利条件下作出选择。
- 首次完整出手将怎样重写阵营关系。

## 断档风险

- 代理人自行完成目标和领取奖励，女主长期被动。
- 多次佯动只把同一高潮后移，没有部分兑现。
- 代理胜利不暴露边界，后续强敌只能凭空变强。
- 低阶反派重复出现，快速打脸变成同一节拍。
- 双线只靠误会拖延，没有独立进展和交叉后果。
- 马甲只有标签抬高，没有权限、责任和升级成本。
- 女主出手条件不断临时修改，读者无法预期。
- 首次本体行动解决的仍是代理人本可完成的小事。
- 拆书只统计爽点，不记录字数、状态变化和剩余债务。

## 训练任务

制作十章兑现表，安排三次代理胜利、两次本体佯动和一次完整出手。每次代理行动都要追溯到女主的选择，并暴露一种不同边界；每次佯动至少兑现能力、身份、关系或资源中的一项。为双线和马甲分别记录进展、代价、字数和剩余债务，禁止连续使用相同拖延理由。
