# 无限流-四驱动看点

依据案例：`batch-149` 案例 741。

## 读者承诺

无限流的吸引力来自四个驱动器：规则漏洞、空间本质、跨体系碰撞、轮回者 PVP。作品不必四项全满，但至少要有一两项足够强，让读者感觉这不是普通副本故事。

## 核心循环

```text
进入空间 -> 发现规则 -> 钻漏洞获利 -> 探索空间真相 -> 跨世界能力碰撞 -> 轮回者竞争与交易
```

## 开篇必备

- 一个能被主角利用的规则缝隙。
- 一个关于空间本质的长期疑问。
- 至少一种跨作品能力、装备或血统碰撞。
- 轮回者之间的竞争、合作或交易苗头。
- 副本任务本身仍要有清晰压力。

## 期待来源

读者期待主角发现系统不是铁板一块，并在多世界资源、轮回者人心和生存压力之间用脑子占便宜。

## 断档风险

- 只有任务，没有空间规则玩法。
- 主角完全接受主神安排，像被系统牵着走。
- 跨体系能力全是作者自造，缺少熟悉作品碰撞的新鲜感。
- 没有 PVP，危机只靠副本任务，篇幅容易显长。
- 只追求老式主神味，忽视时代包装变化。

## 训练任务

为一个无限流新书设计四驱动表：规则漏洞、空间本质、跨体系碰撞、PVP 竞争各写 1 个开篇伏笔，并说明哪一个是前三章主打卖点。
