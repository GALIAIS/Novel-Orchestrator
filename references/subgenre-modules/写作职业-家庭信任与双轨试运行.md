# 写作职业-家庭信任与双轨试运行

依据案例：batch-389 案例 1941

## 读者承诺

把家庭争执拆成可量化风险、可安排保障和不可强求的价值分歧，用真实双轨试运行证明生活能力，再自主选择并承担后果。

## 核心循环

```text
拆解家庭担忧 -> 建立保障与退出方案 -> 双轨记录真实负荷 -> 用连续结果沟通 -> 决策并承担后果
```

## 开篇必备

- 连续收入、支出和波动记录。
- 工作后真实可用的认知时段。
- 健康、社交与保障安排。
- 转全职及退出条件。

## 期待来源

- 抽象争吵变成可检查的风险表。
- 试运行暴露理想岗位与现实负荷差异。
- 连续履约逐步建立家庭信任。
- 作者获得有边界的职业自主。

## 断档风险

- 用单月收入证明长期稳定。
- 假设存在不必履职的闲岗。
- 用头部作者收入代替个人数据。
- 只设转入条件，不设失败回撤。

## 训练任务

执行十二周双轨记录，固定统计工作、通勤、写作、收入、睡眠和健康，并按预先写好的三种决策条件复盘。
