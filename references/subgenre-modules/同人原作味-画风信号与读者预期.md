# 同人原作味-画风信号与读者预期

依据案例：`batch-241` 案例 1202。

## 读者承诺

同人文第一门槛是让读者感觉“这就是那个原作世界”；没有原作味，剧情再完整也会别扭。

## 核心循环

```text
写同人 -> 读者期待原作味 -> 画风错位 -> 出戏 -> 重打磨识别信号
```

## 开篇必备

- 列原作核心名词。
- 抓战斗/成长/互动画风。
- 保留原作读者熟悉的机制。
- 避免只换 IP 外壳。

## 期待来源

读者一进入开头就看到熟悉的机制、称呼、能力和叙事气质。

## 断档风险

- 写数码却没有数码兽感觉。
- 同人变成其他类型换名。
- 表格自检代替原作味判断。
- 只做剧情，不做画风。

## 训练任务

列 10 个原作识别信号，开头至少自然使用 5 个，不用说明书式堆砌。
