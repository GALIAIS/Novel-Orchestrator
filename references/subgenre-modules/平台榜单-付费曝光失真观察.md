# 平台榜单-付费曝光失真观察

依据案例：`batch-188` 案例 936。

## 读者承诺

扫榜时要识别付费曝光对榜单的污染，避免把机制推上去的作品误当成真实市场标杆。

## 核心循环

```text
平台机制 -> 付费曝光 -> 榜单排名变化 -> 公信力下降 -> 多维度复核标杆
```

## 开篇必备

- 记录平台榜单规则。
- 标注是否存在宝箱、打赏、推荐票等付费上榜入口。
- 对异常时间点的榜单保持警惕。
- 同时查看评论、开篇质量和长期表现。

## 期待来源

作者通过更准确的榜单判断，找到真正值得学习的题材、书名、简介和开篇结构。

## 断档风险

- 只看榜单位置就选标杆。
- 把付费曝光当成读者真实选择。
- 忽略平台机制变化。
- 被短期异常榜单误导选题。

## 训练任务

找一个平台榜单，挑 5 本书判断它们的上榜原因：真实热度、付费曝光、题材红利或大神权重，并写出是否适合作为学习标杆。
