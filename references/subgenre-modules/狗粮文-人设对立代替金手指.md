# 狗粮文-人设对立代替金手指

依据案例：`batch-335` 案例 1674

## 读者承诺

狗粮文靠稳定男女主人设、对立感和互动甜点持续运转，人设加职业技能可以替代传统金手指。

## 核心循环

```text
固定男女主人设 -> 设置对立感 -> 小事件触发互动 -> 人设反差产糖 -> 关系推进一点
```

## 开篇必备

- 男女人设一句话说清。
- 两人对立感清楚。
- 第一事件能引发关系张力。
- 职业技能或身份特长能制造互动。
- 不引入第三暧昧对象。

## 期待来源

读者期待两个人下一次如何甜、谁先主动、谁先破防，而不是主角怎样打脸外人。

## 断档风险

- 套主站打脸装逼。
- 女主受辱处理不当。
- 男主下行。
- 别的女生暧昧爆毒。
- 人设不稳定导致糖点失效。

## 训练任务

设计一章恋爱日常开篇，不用系统和打脸，只用表白、合作或误会让男女主人设反差产出甜点。
