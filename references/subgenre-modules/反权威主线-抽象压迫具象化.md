# 反权威主线-抽象压迫具象化

依据案例：`batch-222` 案例 1106。

## 读者承诺

把无形压迫变成可被看见、质疑、击碎的权威象征，让读者获得反抗暴政和推翻神圣性的释放感。

## 核心循环

```text
抽象压迫 -> 具象权威 -> 主角受压 -> 质疑权威 -> 阶段反抗 -> 象征倒塌
```

## 开篇必备

- 明确一个读者能感知的不公。
- 给压迫安排具体载体：皇帝、宗门、公司、系统、规则。
- 展示普通人为何不敢反抗。
- 让主角先做出一次小规模质疑动作。

## 期待来源

读者期待的不是单纯杀 boss，而是看到压迫秩序被命名、被拆穿、被推翻。

## 断档风险

- 只喊宏大口号，没有具体受害场景。
- “皇帝”只是强敌，不承载制度压迫。
- 阶级矛盾理解不足，写成空泛说教。
- 主角反抗没有成本，权威失去重量。

## 训练任务

为玄幻、都市、规则怪谈各写一个“杀皇帝”结构，列出抽象压迫、具象载体、第一次受压、第一次质疑、第一卷末象征倒塌。
