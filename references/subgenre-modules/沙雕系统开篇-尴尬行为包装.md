# 沙雕系统开篇-尴尬行为包装

依据案例：`batch-043` 案例 214。

## 读者承诺

沙雕、变态、情绪值系统可以写离谱行为，但必须先把行为包装成主角不得不做的选择，再用压制和反应把尴尬转成爽点。

## 核心循环

```text
离谱承诺 -> 行为台阶 -> 反派压制 -> 尴尬行动 -> 反应兑现
```

## 开篇必备

- 书名和正文承诺一致。
- 主角行为有理由。
- 对手压制足够。
- 奖励不是轻飘飘获得。

## 期待来源

- 读者想看主角如何豁出去。
- 反派和旁观者误判。
- 离谱行为产生反转。
- 系统奖励带来后续用途。

## 断档风险

- 主角像脑子有问题。
- 争风吃醋冲突太低级。
- 配角心理活动太多。
- 情绪拉扯没到位就给奖励。

## 训练任务

重写一个尴尬沙雕桥段：补行为理由、反派压制、旁观者反应，再给系统奖励。
