# 影视拉片-目的重温与教科书样本

依据案例：batch-373 案例 1861

## 读者承诺

成熟影视作品可以用来训练期待、悬念、意外和爽点结构。重温时必须带目的，否则片单只会变成收藏。

## 核心循环

```text
选样本 -> 定训练目标 -> 标注期待悬念意外 -> 提炼机制 -> 改写网文结构
```

## 开篇必备

- 明确本次拉片目标。
- 标出观众继续看的理由。
- 区分机制和具体剧情。
- 写出网文迁移版本。

## 期待来源

- 悬念吊住观众。
- 意外改变局势。
- 人物对照产生张力。
- 爽点到位但不过白。

## 断档风险

- 只补片不拆结构。
- 照搬影视剧情。
- 样本太杂，训练目标不清。
- 只记录名场面，不记录牵引逻辑。

## 训练任务

重温一部熟片，每 10 分钟记录一次期待、悬念、意外，并改写成网文前三章提纲。
