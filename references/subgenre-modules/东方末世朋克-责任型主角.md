# 东方末世朋克-责任型主角

依据案例：`batch-146` 案例 729。

## 读者承诺

用东方玄幻和末世朋克混合基调制造压迫世界，再用底层生存中的同理心选择和承担责任，塑造“我来、我行、有我、我能”的主角爽点。

## 核心循环

```text
破败视觉 -> 废墟地点 -> 群体求生 -> 稀缺资源选择 -> 责任承担 -> 人物可信
```

## 开篇必备

- 一个强视觉幻想科技物件。
- 一个废墟或坟场式地点。
- 大量底层人的生存压力。
- 主角面对资源稀缺的选择。
- 选择必须符合处境和成本。

## 期待来源

读者期待在残酷世界里看到能扛事的人。责任型主角不是喊口号，而是在压力中做选择，并能承担结果。

## 断档风险

- 设定铺太久但没有人物选择。
- 善良脱离处境，变成圣母毒点。
- 只有责任口号，没有行动兑现。
- 老书桥段直接照搬，忽视时代毒点变化。

## 训练任务

写一个 800 到 1200 字开头：破败交通工具、废墟地点、群体抢资源。再写一个主角分出稀缺资源的选择，必须说明对象、成本、关系和处境。
