# 名著同人-公共认知强代入

依据案例：`batch-040` 案例 196。

## 读者承诺

名著同人用公共认知降低铺垫成本，用强反差身份制造点击和期待。读者会自动补齐人物关系、历史遗憾和原著冲突。

## 核心循环

```text
公共认知 -> 身份反差 -> 不公平伤害 -> 第一危机 -> 人设破局 -> 历史长线
```

## 开篇必备

- 原人物人设鲜明。
- 新身份有强处境。
- 第一危机立刻出现。
- 长线遗憾可继续展开。

## 期待来源

- 读者熟悉原著。
- 反差自带笑点和爽点。
- 弱势身份需要翻盘。
- 历史大局能拉长线。

## 断档风险

- 只借名著名气，不用人物性格破局。
- 前期强，中期基调乱转。
- 知识储备撑不起历史线。
- 同人信息优势没有转为事件。

## 训练任务

设计一个名人穿越弱势身份开篇：写明公共认知、第一危机、人设破局和长线遗憾。
