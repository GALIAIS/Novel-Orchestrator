# 长压抑情绪-下行毒点与回弹边界

依据案例：batch-374 案例 1867

## 读者承诺

压抑情绪可以促成人物成长，但下行过长会拉断读者情绪弦。读者珍视的核心承诺不能随意牺牲。

## 核心循环

```text
下行压迫 -> 成长代价 -> 情绪缓冲 -> 地图/时间重启 -> 回弹兑现
```

## 开篇必备

- 读者珍视的美好。
- 明确下行原因。
- 主角成长收益。
- 可预期的回弹方向。

## 期待来源

- 苦痛之后的反击。
- 主角心理完成变化。
- 缓冲后重新冒险。
- 失去带来的长期记忆点。

## 断档风险

- 压抑持续太长。
- 毁掉不可恢复的核心承诺。
- 只增加痛苦，不增加成长。
- 回弹奖励不够抵消毒点。

## 训练任务

写三章下行情节，每章都标出成长收益、缓冲点和回弹承诺；删掉只虐不长的段落。
