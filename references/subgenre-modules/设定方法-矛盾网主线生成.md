# 设定方法-矛盾网主线生成

依据案例：`batch-191` 案例 951。

## 读者承诺

设定不同等级的组织、集团和国家，本质上是在搭建矛盾网，为主线和长篇续航提供冲突燃料。

## 核心循环

```text
势力层级 -> 目标差异 -> 资源/规则冲突 -> 主线起点 -> 层级升级
```

## 开篇必备

- 至少有三层势力。
- 每层都有目标和利益。
- 势力之间有资源或规则冲突。
- 主角能从低层矛盾切入。

## 期待来源

读者看到主角解决局部矛盾后，牵出更高层势力和更大的主线问题。

## 断档风险

- 设定只是背景，不产冲突。
- 组织层级没有利益差。
- 主线起点和世界设定脱节。
- 解决一个事件后没有上层矛盾承接。

## 训练任务

为一个新书点子写三层势力表，每层列目标、资源、冲突和主角切入方式。
