# 网文开篇-新人模板强期待留存

依据案例：`batch-181` 案例 902。

## 读者承诺

新人开篇模板的承诺是降低写作难度，在前三章快速给读者可识别的主角、矛盾、金手指或小金手指、短目标和第一波兑现。

## 核心循环

```text
能力有限 -> 模板降难度 -> 期待源早出现 -> 小爽点兑现 -> 留存提高
```

## 开篇必备

- 主角处境清楚。
- 第一矛盾足够靠前。
- 金手指可以小，但期待源不能完全缺席。
- 前三章要有短期目标和小兑现。

## 期待来源

读者愿意追新人书，通常不是因为开头多高级，而是因为他知道接下来会爽在哪里、强在哪里、变在哪里。

## 断档风险

- 新人直接模仿大神慢热。
- 只学节奏名义，实则抄桥段。
- 金手指藏到读者完全感受不到。
- 开头没有短目标，只有世界观说明。

## 训练任务

把一个慢热开头改成简单模式：三章内交代主角处境、第一矛盾、期待源、短目标和一次小兑现，禁止搬运标杆剧情。
