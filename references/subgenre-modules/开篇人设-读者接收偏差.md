# 开篇人设-读者接收偏差

依据案例：`batch-147` 案例 733。

## 读者承诺

开篇诊断要看读者实际读到什么，而不是作者脑中设定什么。若作者想写冷血果断，读者却读成逗比，就要回到句子、动作和场景氛围中找偏差。

## 核心循环

```text
作者设定 -> 文本呈现 -> 读者接收 -> 偏差定位 -> 人设/氛围/金手指修正
```

## 开篇必备

- 主角性格用行为呈现。
- 场景情绪和人物表现一致。
- 金手指一句话能讲清。
- 多余能力延后。
- 有大纲或至少有后续方向。

## 期待来源

读者期待第一章就知道主角是什么人、处境有多紧、外挂怎么理解。表达偏差会直接影响代入。

## 断档风险

- 作者自认为冷血，读者读成搞笑。
- 高压逃命里无意写出逗比感。
- 金手指能力太杂，理解成本高。
- 没有后续规划，只靠脑热开头。

## 训练任务

把一个开头给 3 个读者，只问“你读到的主角性格是什么”。标出造成偏差的 5 个句子，再把多能力金手指压缩成一句话主能力。
