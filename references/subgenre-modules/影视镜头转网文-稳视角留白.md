# 影视镜头转网文-稳视角留白

依据案例：`batch-048` 案例 240。

## 读者承诺

借鉴影视的钩子、并列行动和留白，但转成网文时优先保证视角清晰、主线明确、信息不重复，让读者先跟住人物和目标。

## 核心循环

```text
影视片段 -> 提取钩子 -> 合并重复信息 -> 降低视角切换 -> 主角目标推进 -> 留一个可追问问题
```

## 开篇必备

- 不把电影式楔子直接当网文开篇模板。
- 明确主角当前要找谁、查什么、解决什么。
- 反派视角只保留必要应对，不重复解释同一信息。
- 开篇少切视角，等读者记住角色后再尝试复杂结构。
- 留白必须服务悬念，不能服务混乱。

## 期待来源

- 主角目标清晰，读者能跟。
- 反派应对出现，压力会变具体。
- 信息只讲一次，节奏会利落。
- 留一个关系谜题或线索谜题，推动翻页。

## 断档风险

- 影视多镜头照搬到网文，读者容易乱。
- 回溯如果用于补漏洞，会变成毒点。
- 简介或楔子提前透支整本书亮点，会削弱正文吸引力。
- 反派视角太早太多，会让主角线失焦。

## 训练任务

把一个电影多镜头段落改成 1500 字网文章节顺序：只允许切换 1 次视角，并写出删掉了哪些重复信息。
