# 简介封面-卖点包装同等重要

依据案例：`batch-316` 案例 1579

## 读者承诺

书名、封面、简介是正文前的第一轮转化。简介不是摘要，而是把核心卖点讲到读者一眼能识别。

## 核心循环

```text
作品卖点 -> 书名封面给信号 -> 简介重复强化 -> 读者识别期待 -> 点击正文
```

## 开篇必备

- 明确核心卖点。
- 学同题材排行榜简介。
- 第一时间给读者点击理由。
- 可以重复卖点，但要照顾阅读观感。

## 期待来源

读者期待在简介里快速判断这本书是否符合口味。

## 断档风险

- 简介写成剧情流水账。
- 卖点藏太深。
- 机械重复导致尴尬。
- 封面书名简介信号不一致。

## 训练任务

参考 5 本排行榜简介，给自己的作品写 3 版不同卖点侧重的简介。
