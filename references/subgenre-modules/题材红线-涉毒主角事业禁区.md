# 题材红线-涉毒主角事业禁区

依据案例：`batch-090` 案例 450。

## 读者承诺

涉毒不能作为主角事业线、升级线或爽点线。架空、穿越、爱情标签都不能稀释红线；需要涉及时只能放在反毒、调查、受害、惩戒链条中。

## 核心循环

```text
识别高危卖点 -> 判断是否违法红线 -> 删除主角经营线 -> 改反毒/调查视角 -> 保留冲突不保留爽点
```

## 开篇必备

- 标题不要把涉毒当卖点。
- 主角不能经营违法高危产业。
- 架空不能替代价值立场。
- 涉及毒品必须明确反毒。
- 高危内容要转为反派或案件。

## 期待来源

- 主角如何破案、救人、反制。
- 反派违法产业如何被摧毁。
- 受害者如何脱困。
- 正向价值如何兑现。

## 断档风险

- 把红线包装成穿越事业。
- 用谐音规避审核。
- 主角靠违法行为升级。
- 平台审核直接卡死。

## 训练任务

检查设定中所有黄赌毒和违法产业。凡是作为主角事业线的，全部改为反派线、案件线、受害线或惩戒线。
