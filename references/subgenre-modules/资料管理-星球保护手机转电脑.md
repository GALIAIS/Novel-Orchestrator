# 资料管理-星球保护手机转电脑

依据案例：`batch-187` 案例 932。

## 读者承诺

遇到星球保护导致电脑端看不到文件时，直接采用手机下载、转传电脑、再归档阅读的流程，避免把学习卡在平台限制上。

## 核心循环

```text
资料受保护 -> 确认限制不可关闭 -> 手机下载 -> 传电脑 -> 命名归档 -> 阅读复盘
```

## 开篇必备

- 知道限制来自平台强制机制。
- 准备手机端下载路径。
- 电脑端有统一资料目录。
- 文件转存后立刻命名和记录来源。

## 期待来源

作者不再反复寻找无法关闭的开关，而是把资料稳定纳入自己的学习库。

## 断档风险

- 继续等管理员关闭保护。
- 文件散落在聊天软件和下载目录。
- 只下载不命名，后续找不到。
- 资料没有阅读笔记和复盘结论。

## 训练任务

建立一个资料转存表，记录资料名、来源帖、下载设备、电脑路径、阅读状态和可复用结论，至少完成 3 份资料的归档。
