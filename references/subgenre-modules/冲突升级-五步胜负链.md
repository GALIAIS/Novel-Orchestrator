# 冲突升级-五步胜负链

依据案例：`batch-147` 案例 735。

## 读者承诺

用五步检查一个冲突是否成立：爆发、第一次升级、第二次升级、激烈对抗、决定胜负。冲突要和主角直接相关，并在胜负后产生情绪反馈。

## 核心循环

```text
冲突爆发 -> 第一次升级 -> 第二次升级 -> 激烈对抗 -> 决定胜负 -> 情绪反馈
```

## 开篇必备

- 一个和主角有关的小摩擦。
- 一方拒绝正常解决。
- 代价从事件升到态度、面子、钱、身份或规则。
- 伏笔支撑升级。
- 终局明确分出胜负。

## 期待来源

读者期待冲突越滚越大，并看到谁有底牌、谁被压服、围观者如何反应。

## 断档风险

- 冲突不冲着主角来。
- 从头吵到尾，没有升级。
- 升级无伏笔，显得硬转。
- 终局不分胜负，爽点落空。

## 训练任务

用“踩鞋”写一个网文冲突，必须包含五步。再选自己大纲中一场戏，按五步标注并补一个伏笔，让升级合理。
