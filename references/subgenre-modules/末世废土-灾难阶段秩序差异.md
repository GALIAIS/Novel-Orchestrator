# 末世废土-灾难阶段秩序差异

依据案例：batch-272 案例 1360

## 读者承诺

区分末世和废土的写作承诺：末世偏灾难发生与秩序崩塌进行时，废土偏灾难之后的新规则、新势力和重建争夺。

## 核心循环

```text
末世：正常秩序 -> 灾难爆发 -> 求生突变 -> 成因/解决
废土：灾难已过 -> 新秩序形成 -> 资源争夺 -> 重建/改写规则
```

## 开篇必备

- 先确定故事开始时灾难处于哪个阶段。
- 末世开篇突出异常、崩塌和即时求生。
- 废土开篇突出聚落、资源、势力和规则。

## 期待来源

末世读者期待看正常生活被灾难撕开；废土读者期待看人在残破新世界里争资源、建秩序、抢地盘。

## 断档风险

- 阶段不清，既不像末世也不像废土。
- 只写灾难背景，不写主角目标。
- 废土没有新规则，末世没有崩塌压力。

## 训练任务

同一灾难设定写两个开头：灾难爆发当天和灾难后十年。分别列出主角目标、第一敌人和资源压力。
