# 女频学习-教学源样本验证

依据案例：`batch-070` 案例 349。

## 读者承诺

这个模块用于男频作者或新人入门女频。它承诺的是先用靠谱教学源建立概念，再回到女频样本和读者评论验证，不把二手结论直接当写法。

## 核心循环

```text
找教学源 -> 记录概念 -> 找女频样本 -> 查读者反馈 -> 验证后入库
```

## 开篇必备

- 先选一个教学源。
- 每次只记 3 个概念。
- 每个概念找作品验证。
- 每个判断看读者评论。
- 未验证的结论不进入方法库。

## 期待来源

- 教学源降低入门成本。
- 样本避免空学概念。
- 读者评论校正理解。
- 多源验证减少偏差。
- 女频认知逐步成体系。

## 断档风险

- 只听 up 主，不看作品。
- 直接迁移男频判断。
- 把单一教学源当平台规则。
- 不看读者反馈。
- 概念太多，无法落地。

## 训练任务

看一个女频教学视频，记录 3 个概念。每个概念找 1 本样本和 5 条评论验证，写成一张学习卡。
