# 价值观小歪-点出问题与轻松化解

依据案例：batch-355 案例 1772

## 读者承诺

角色可以有小缺陷和小歪行为，但文本要让读者知道作者知道它不对。用角色质疑、反讽、轻松气氛或后果把行为限定为角色趣味，而不是作品立场。

## 核心循环

```text
小歪行为 -> 文本点出问题 -> 角色能力/幽默化解 -> 降低攻击性 -> 保留边界
```

## 开篇必备

- 明确这只是小歪，不是红线。
- 有人或叙述点出不妥。
- 有喜剧化或反讽处理。
- 必要时给轻微后果。

## 期待来源

- 角色缺陷带来趣味。
- 读者不会误解作者立场。
- 轻松处理让桥段不沉重。
- 边界清楚，后续还能继续用角色魅力。

## 断档风险

- 把问题行为写成理所当然。
- 美剧尺度直接照搬网文。
- 没有人点出问题。
- 触碰平台红线还想靠玩笑带过。

## 训练任务

写一个主角耍小聪明脱身场景，必须包含：旁人质疑、主角轻松化解、一个不严重但可见的后果。
