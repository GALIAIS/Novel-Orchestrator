# 都市修真养成-标签过多写不过来

依据案例：`batch-332` 案例 1656

## 读者承诺

都市修真养成不能把宠物、装备、伴侣全塞成主线；先定一个主养成对象，其他降级。

## 核心循环

```text
主养成对象 -> 投入资源 -> 阶段变化 -> 回馈主线 -> 下一轮更高需求
```

## 开篇必备

- 明确定义“养成”是什么。
- 主养成对象只保留一个。
- 养成资源和反馈清楚。
- 都市修真主线不被养成支线淹没。
- 找得到对标或相邻样本。

## 期待来源

读者期待对象逐步变强，并反过来帮助主角解决都市修真主线问题。

## 断档风险

- 标签太多，写不过来。
- 为了养成而养成。
- 宠物、装备、老婆抢同一份篇幅。
- 读者不知道该期待谁升级。
- 没有对标，组合太散。

## 训练任务

列出所有养成对象，只保留一个主对象，设计前三次升级的资源、变化、回报。
