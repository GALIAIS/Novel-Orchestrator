# 角色工具-个人调查表与立体化补缺

依据案例：`batch-213` 案例 1061。

## 读者承诺

角色调查表能帮助作者发现角色欲望、恐惧、关系和经历中的空白。

## 核心循环

```text
角色设想 -> 表格调查 -> 发现缺项 -> 补动机关系 -> 回填正文
```

## 开篇必备

- 表格字段服务行动和冲突。
- 填不出的项目要回到故事里补。
- 不把表格内容直接堆给读者。
- 写完章节后反查角色信息是否体现。

## 期待来源

作者通过可见字段检查角色，减少只有标签没有行动逻辑的扁平人物。

## 断档风险

- 填表变成设定堆砌。
- 表格信息不回流正文。
- 角色有资料但无选择。
- 表格不按题材调整。

## 训练任务

先写 300 字扁平角色出场，再填角色调查表，再重写 300 字，对比变化。
