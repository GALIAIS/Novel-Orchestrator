# 女频追妻-误会伤害赎罪复合

依据案例：`batch-181` 案例 901。

## 读者承诺

女频误会伤害后的追妻线，核心承诺是女主重新夺回关系主动权，男主以行动和代价赎罪，最终复合必须由女主主动选择。

## 核心循环

```text
误判伤害 -> 女主受损/退场 -> 信息差重启关系 -> 男主付代价 -> 女主清算情绪 -> 复合兑现
```

## 开篇必备

- 明确女主损失有多重。
- 男主不能只口头后悔。
- 小三或挑拨者必须被清算。
- 女主假失忆要制造主动信息差。

## 期待来源

读者期待的不是男主自我感动，而是他被迫面对伤害后果，并在女主控制节奏的情况下重新争取资格。

## 断档风险

- 女主轻易原谅。
- 男主没有实际代价。
- 清算全部由外力完成，女主无参与感。
- 假失忆只变成拖剧情，不产生甜虐拉扯。

## 训练任务

写一条追妻线：女主假失忆制造三层信息差，男主完成三次行动赎罪，女主亲自主导一次情绪清算，再决定是否复合。
