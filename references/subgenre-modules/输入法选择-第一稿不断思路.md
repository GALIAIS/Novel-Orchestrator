# 输入法选择-第一稿不断思路

依据案例：batch-375 案例 1875

## 读者承诺

第一稿优先不断思路。输入法选择要服从当前码字压力，赶稿用最稳方案，空窗期再练更高效工具。

## 核心循环

```text
判断码字压力 -> 选稳定输入法 -> 第一稿跳过卡字 -> 写完再修 -> 空窗训练
```

## 开篇必备

- 当前写作压力。
- 卡壳原因。
- 第一稿容错规则。
- 后期修订安排。

## 期待来源

- 情节流不断。
- 卡字不拖慢产出。
- 工具训练不影响更新。
- 后期统一修正质量。

## 断档风险

- 五笔字根不熟还硬用。
- 为完美字句打断思路。
- 把构思慢怪给输入法。
- 练工具占用赶稿期。

## 训练任务

写 2000 字第一稿，遇到不会打的字用同音字或括号跳过；写完统计卡壳次数，再决定是否换输入法。
