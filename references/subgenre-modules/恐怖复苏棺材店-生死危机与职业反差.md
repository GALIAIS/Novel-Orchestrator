# 恐怖复苏棺材店-生死危机与职业反差

依据案例：`batch-207` 案例 1035。

## 读者承诺

恐怖复苏开篇要让危机直接发生在主角身上，用职业反差和金手指破局，而不是先调查案件。

## 核心循环

```text
女鬼上门 -> 主角职业反常迎客 -> 生死危机 -> 棺材店规则/手艺 -> 破局爽点
```

## 开篇必备

- 危机第一时间落到主角店里。
- 主角反应和恐怖氛围形成反差。
- 棺材店规则或杀猪手艺尽早展示。
- 一万字内爽一发。

## 期待来源

读者期待恐怖末世里的主角不怕鬼，反而把鬼当生意和资源处理。

## 断档风险

- 开头先调查，主角不在危机中心。
- 世界观铺垫脱离行动。
- 金手指和职业没有绑定。
- 平台与女主/百合/无男主定位不匹配。

## 训练任务

把调查案件改成女鬼上门索命：1000 字内发生在主角店里，并展示棺材店规则或杀猪手艺反杀。
