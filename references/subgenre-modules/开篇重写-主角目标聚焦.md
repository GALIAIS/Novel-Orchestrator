# 开篇重写-主角目标聚焦

依据案例：`batch-019` 案例 093。

## 读者承诺

解决“文笔不错但剧情散”的开篇问题。先让主角面对冲突、获得优势、解决问题并装逼，再谈背景、设定和配角戏。

## 核心循环

```text
找爽画面 -> 定主角目标 -> 删配角废戏 -> 让主角行动 -> 用金手指兑现
```

## 开篇必备

- 主角正在面对的冲突。
- 主角短期目标。
- 金手指或优势。
- 读者最想看到的爽画面。

## 期待来源

- 读者知道主角能解决问题，但其他角色不知道。
- 主角与对手形成强对比。
- 冲突结果会改变主角处境。
- 爽点前每个铺垫都朝最终画面推进。

## 断档风险

- 主角沉睡或退场。
- 配角打架占据开篇。
- 作者觉得背景必要，读者却没有阅读理由。
- 设定、养崽、复仇、历劫、道具线同时塞进第一章。

## 训练任务

把一个失败开篇压成三句话：主角面对什么冲突、有什么优势、如何解决并装逼。三句话以外的第一章内容先全部删除。
