# 高产失效-反馈缺席与刻意练习重置

依据案例：`batch-404` 案例 2020

## 读者承诺

承认高产能训练耐力和完成，同时拒绝把总字数当成能力增长；每轮选一个可观察短板，以基线、挑战、反馈、修订和跨单元复现把产量转成刻意练习。

## 核心循环

```text
选择具体短板 -> 保存文本基线 -> 安排稍高难度任务 -> 收集针对性反馈 -> 定点修订对照 -> 下一单元复现 -> 卷末更新能力证据 -> 决定继续暂停或换方向
```

## 开篇必备

- 区分产量、项目结果和具体能力指标。
- 每轮只有一个可观察训练目标。
- 保留修改前的基线文本。
- 在章、单元、卷和项目设置检查点。
- 反馈问题直接指向当前短板。
- 至少跨两个单元复现后才确认学会。

## 期待来源

作者不再等待总字数自动触发进步，而能看到每轮具体学会什么；继续长篇和计划暂停都服务同一个证据闭环。

## 断档风险

- 用他人高产个案证明坚持必成或必败。
- 不核验材料便归因多开、水文或代笔。
- 反复练习已经熟悉的动作。
- 只看收藏、推荐或收入，不看文本能力。
- 生产压力挤掉全部反馈与修订。
- 每次换项目却没有跨情境复现。

## 训练任务

从最近三万字选择一个短板，保存基线，在下一万字安排两次针对性任务；每次以同一读者问题反馈并修订，卷末只按是否跨两次复现验收。
