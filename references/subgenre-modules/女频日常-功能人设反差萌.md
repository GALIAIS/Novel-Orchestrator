# 女频日常-功能人设反差萌

依据案例：`batch-123` 案例 613。

## 读者承诺

温馨日常文可以靠搞笑、造梗和反差萌留住读者。功能型角色不是工具人，只要缺点、反应和关系推进写细，就能变立体。

## 核心循环

```text
日常事件 -> 槽点出现 -> 不同角色反复反应 -> 反差萌放大笑点 -> 关系推进或冲突升级
```

## 开篇必备

- 每章至少有几个明确笑点。
- 槽点可被不同人、不同事复用。
- 金手指存在感可弱，但扩展性要强。
- 世界观拆散慢慢放。
- 功能角色要有缺点和可爱反差。

## 期待来源

- 平常事被讲得有趣。
- 同一槽点多次变奏。
- 打脸和吃糖并存。
- 冲突慢慢激化但不破坏温馨基调。

## 断档风险

- 节奏太慢，缺少每章反馈。
- 功能角色没有出路。
- 世界观靠新人解说，不靠展示。
- 反派压迫不足时冲突软。

## 训练任务

写一章温馨日常，安排 3 个笑点和 1 个可复用槽点。每个功能角色必须有一个缺点和一个反差萌反应。
