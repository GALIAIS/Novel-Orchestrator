# 现实题材-金羊毛改编

依据案例：`batch-031` 案例 155。

## 读者承诺

现实题材也能用金羊毛骨架，把真实困境改成任务链，再用道德代价完成主题转向。

## 核心循环

```text
现实压力 -> 任务召唤 -> 取物/获利 -> 组队扩大 -> 代价逼近 -> 主题转向
```

## 开篇必备

- 主角现实私欲目标。
- 可追的具体资源或宝物。
- 召唤者。
- 队伍成员功能。
- 后半段道德代价。

## 期待来源

- 主角先为自己解决困境。
- 旅程中不断获得资源。
- 队伍成员扩大故事面。
- 违法、情感、社会代价逐步压上来。

## 断档风险

- 现实题材写成散文记录。
- 类型骨架压过现实痛感。
- 只有赚钱上行，没有代价。
- 主题转向没有节点。

## 训练任务

设计一个现实题材金羊毛故事：先让主角为私欲取资源，再让资源牵出群体困境和道德选择。
