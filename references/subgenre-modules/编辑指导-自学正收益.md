# 编辑指导-自学正收益

依据案例：batch-299 案例 1495

## 读者承诺

新人不要等编辑一对一教写作，要把失败转成教程学习、拆书、写作和复盘。

## 核心循环

```text
签约/成绩失败 -> 放弃等编辑教学 -> 找教程拆书 -> 写正文试错 -> 复盘数据 -> 追求正收益
```

## 开篇必备

- 区分编辑职责和作者学习责任。
- 平台规则可问编辑，写法细节靠自学。
- 每周有明确写作和拆书任务。
- 复盘目标指向正收益。

## 期待来源

- 作者从抱怨转为行动。
- 教程打碎认知误区。
- 写作量和复盘带来真实进步。

## 断档风险

- 想太多，写太少。
- 把编辑当私人教练。
- 只看教程不产出正文。

## 训练任务

列本周四项任务：看一个教程、拆一本热销书、写三章正文、复盘一个数据指标。
