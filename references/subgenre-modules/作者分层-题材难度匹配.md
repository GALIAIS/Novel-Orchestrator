# 作者分层-题材难度匹配

依据案例：`batch-045` 案例 223。

## 读者承诺

作者要按自己的实力和认知选题材。编辑带作的核心是帮作者匹配用户、题材、难度和模板，再逐步补短板。

## 核心循环

```text
作者层级 -> 题材匹配 -> 固定模板 -> 补短板 -> 微创新
```

## 开篇必备

- 知道自己处在哪层。
- 选择能驾驭的题材。
- 明确目标用户群。
- 先用固定路线稳定产出。

## 期待来源

- 作者不再盲目挑战高难题材。
- 题材和用户匹配更清楚。
- 固定模板降低失败率。
- 微创新建立差异化。

## 断档风险

- 把模板当低级。
- 实力不够硬写复杂设定。
- 题材用户群不匹配。
- 只追吸量，不顾内容承接。

## 训练任务

做一次作者层级自测：写出自己稳定完成的最长篇幅、最熟题材、不能碰的题材和下一本固定模板。
