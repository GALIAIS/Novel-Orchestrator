# 历史锦衣卫开篇-题材敏感与审问钩子

依据案例：`batch-201` 案例 1001。

## 读者承诺

架空历史可以降低题材敏感，但锦衣卫开篇必须尽快给生死危机、权力压迫、审问交锋或金手指，不要把篇幅耗在无关画面。

## 核心循环

```text
架空背景 -> 权力身份 -> 抓捕/审问 -> 生死危机 -> 主角掌控局面
```

## 开篇必备

- 明确架空背景。
- 前几百字进入抓捕或审问。
- 给出主角权力位置和风险。
- 称呼体系保持历史质感。

## 期待来源

读者期待看到锦衣卫题材里的压迫感、审问交锋和主角如何在高风险权力局里掌控局面。

## 断档风险

- 只问题材能不能过审，忽略开篇吸引力。
- 画面描写过多，事件推进太慢。
- 没有生死危机或金手指。
- 称呼现代化导致出戏。

## 训练任务

写 1000 字锦衣卫开头：300 字内抓捕，700 字内审问，必须有一个生死风险和一个主角主动压场动作。
