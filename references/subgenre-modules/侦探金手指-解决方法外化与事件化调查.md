# 侦探金手指-解决方法外化与事件化调查

依据案例：`batch-411` 案例 2052

## 读者承诺

侦探金手指改变解决路径而非只提高职业数值；异常信号仍需解释，主角以独特人格进行验证，每次走访都变成有欲望、阻力、成本和多类状态变化的事件。

## 核心循环

```text
案件目标与职业假设 -> 能力捕捉异常信号 -> 现场人物阻止信息流动 -> 主角用人格化方法验证 -> 承担成本并改变局面 -> 线索关系风险更新 -> 敌我反应 -> 修正假设进入下一节点
```

## 开篇必备

- 主角职业基线和能力的二阶差异。
- 能力输入、盲区、噪声与误判。
- 每场调查的独立人物欲望。
- 人格化验证动作与现实成本。
- 线索、关系、风险或时间的状态变化。
- 能进入后续案件因果的反应链。

## 期待来源

常规调查遇阻后，能力提供一个可疑而非确定的信号；主角用非常规方法验证，局部事件给出线索与人物反应，新证据又推翻部分旧假设。

## 断档风险

- 只把本职观察力数值调高。
- 能力直接读出真凶和答案。
- 换地点询问却不改变假设。
- 临时小反派打完即消失。
- 围观赞叹不进入后续选择。
- 高潮由能力自动完成，主角不承担解释代价。

## 训练任务

把三个“到地点询问”的节点改成三场事件，每场包含独立欲望冲突、一个需解释的能力信号、一项人格化验证和两类状态变化；禁止直接读心及与案件无关的打脸靶子。
