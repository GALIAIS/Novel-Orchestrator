# 世界观问卷-表格化工具

依据案例：`batch-157` 案例 782。

## 读者承诺

把世界观问卷从文本清单变成表格，让作者更容易填写、筛选、检索和补设定。

## 核心循环

```text
文本问题 -> 表格字段 -> 填写答案 -> 标优先级 -> 补资料 -> 服务故事
```

## 开篇必备

- 问题项。
- 所属模块。
- 我的答案。
- 待补资料。
- 优先级。
- 是否影响主线。
- 是否必须写进正文。

## 期待来源

期待来自工具可执行：作者能用表格发现设定缺口，而不是被长清单压住。

## 断档风险

- 清单有价值但不好用。
- 表格只收藏不填写。
- 世界观答案直接塞正文。
- 没有区分作者知道和读者需要知道。

## 训练任务

把一份写作检查清单改成表格，并用已有故事试填 10 项，检查是否真的提高使用效率。
