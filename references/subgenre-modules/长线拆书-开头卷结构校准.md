# 长线拆书-开头卷结构校准

依据案例：`batch-167` 案例 834。

## 读者承诺

先细拆开头卷解决起步难，再简略拆长期结构，帮助作者看见自己大纲和结构成熟度不足的问题。

## 核心循环

```text
细拆前几章 -> 找起步难原因 -> 汇总材料 -> 简拆几卷 -> 回写大纲
```

## 开篇必备

- 起步难先拆开头卷。
- 细拆要记录耗时和收获。
- 拆完必须汇总。
- 长期结构可以简略拆，不必每章同精度。

## 期待来源

作者期待通过对标作品看见成熟结构如何撑住开局和长线，解决急着动笔导致后期兜不住的问题。

## 断档风险

- 只拆局部，不汇总。
- 一味细拆导致成本过高。
- 只看章节技巧，不看卷级结构。
- 拆书后不回写自己的大纲。

## 训练任务

细拆对标书前 5 章，简拆后续 2 卷目录，写出 3 个自己开局结构不成熟的具体问题。
