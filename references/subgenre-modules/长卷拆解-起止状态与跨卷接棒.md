# 长卷拆解-起止状态与跨卷接棒

依据案例：`batch-416` 案例 2080

## 读者承诺

长卷拆解以卷首和卷末的目标、身份、能力、资源、关系、规则认知及风险差异为核心，每个阶段都用章节证据证明不可逆变化。卷末高潮结算本卷中央债务，余波明确永久收益与损失；下一卷矛盾由本卷胜利、暴露、权力真空或规则后果触发，而不是突然添加的宣传式灾难。

## 核心循环

```text
记录卷首人物与世界状态 -> 触发事件建立期限和中央问题 -> 切分若干阶段状态门槛 -> 每阶段通过选择和代价改变至少两项状态 -> 中段反制修正主角方法 -> 卷末行动结算中央债务 -> 余波确认不可逆收益、损失和关系重排 -> 本卷结果触发下一卷敌方计划或世界变化 -> 新目标与迫近风险完成因果接棒
```

起止差提供可验证进度，阶段门槛防止中段散漫，高潮与余波分工保证结算可见，因果接棒把跨卷钩子变成旧结果的后果。

## 开篇必备

- 卷名、完整章节范围和实际阅读记录。
- 卷首主角目标、身份、能力、资源、关系和风险。
- 本卷触发事件、中央问题与期限。
- 五个左右可验证的阶段状态门槛。
- 每阶段至少两个章节证据和一项选择代价。
- 中段反制与主角认知模型修正。
- 卷末高潮动作及被结算的中央债务。
- 余波中的永久收益、损失和关系重排。
- 下一卷冲突与本卷结果的因果连接。
- 无法取得或无法验证的章节明确标记。

## 期待来源

- 卷首最大未知会在何阶段获得答案。
- 主角每段增加了哪项可行动资源。
- 中段反制会推翻哪项早期理解。
- 卷末高潮将共同回收哪些人物和规则债务。
- 胜利会留下何种无法撤销的损失。
- 哪个旧结果会启动下一卷敌方行动。
- 下一卷目标与本卷状态差如何连续。

## 断档风险

- 只列卷首和卷末名场面，没有中间证据。
- 用最终击败敌人代替所有人物与世界结算。
- 阶段只按章节数量切分，不看状态变化。
- 高潮后没有余波，收益和损失不可见。
- 下一卷灾难与本卷行动没有因果。
- 用名作常识补齐未取得的文章或章节。
- 评论中的题材标签替代文本证据。
- 卷内目标完成后人物仍能无损回到起点。
- 拆解不标注截断、缺章和不确定项。

## 训练任务

完整阅读一卷后制作卷首、五阶段和卷末状态表。每阶段引用两个章节证据并改变至少两项状态；卷末分开记录高潮动作、中央债务结算和余波。最后证明下一卷冲突由本卷结果触发，无法证明时标记为外加预告，禁止补写缺失内容。
