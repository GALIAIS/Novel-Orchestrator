# 阵营和解-立场转向与利益再契约

依据案例：`batch-404` 案例 2016

## 读者承诺

让长期对立阵营的和解同时完成三件事：用连续证据改变情感判断，承认双方责任与损失，并把原先的资源冲突改写成可执行的新权利义务。

## 核心循环

```text
稀缺资源冲突 -> 双方诉求与越界并置 -> 暴力升级暴露手段失控 -> 新证据促成有限停火 -> 双方认责 -> 谈判资源权利义务 -> 小合作验证 -> 违约风险延续
```

## 开篇必备

- 双方各有可理解诉求和可责行为。
- 敌方主体性在高潮前逐步展示。
- 主角态度变化有多次证据。
- 执行工具或共同敌人不替下令者免责。
- 伤亡、哀悼和不信任进入和解过程。
- 资源、边界、义务和违约规则明确。

## 期待来源

旧阵营判断被新证据逐步修正，临时停火之后仍要看双方能否承担责任、重新分配利益，并用第一次合作证明新关系。

## 断档风险

- 末刻一个勇敢动作完成全部洗白。
- 主角只怪执行工具、不承认自己下令。
- 共同敌人死亡后立即亲如一家。
- 象征性握手跳过资源冲突。
- 强制劳动被当作唯一道德答案。
- 死亡和损失被喜剧结尾擦除。
- 一面要求圆满，一面用荒诞跳过因果。

## 训练任务

把宿敌握手结局拆成双方越界、转向证据、认责、停火、利益谈判、合作验证和违约余波；分别写可信和解版与荒诞讽刺版，检查前置信号是否不同。
