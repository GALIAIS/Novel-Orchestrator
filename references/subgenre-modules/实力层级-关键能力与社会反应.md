# 实力层级-关键能力与社会反应

依据案例：`batch-223` 案例 1113。

## 读者承诺

境界提升不只是威力变大，而是行动自由、身体感知、社会地位和战斗方式发生可感知变化。

## 核心循环

```text
境界名词 -> 关键能力 -> 社会反应 -> 战斗方式变化 -> 新问题可解
```

## 开篇必备

- 每级给一个关键能力门槛。
- 用称呼、威压、恭敬程度表现等级差。
- 让突破带来新的行动自由。
- 少用纯数值威力说明。

## 期待来源

读者期待主角突破后能做以前不能做的事，比如飞行、神识外放、驱动法宝、抵抗威压、延长寿命。

## 断档风险

- 只写攻击力更强。
- 境界名很多，但功能差异不清。
- 他人反应缺失，等级没有社会重量。
- 过早真刀真枪比较，导致冲突失控。

## 训练任务

设计 5 个修真境界，每级写一个关键能力门槛、一个社会反应、一个战斗方式变化、一个主角突破后能解决的新问题。
