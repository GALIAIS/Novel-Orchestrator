# 都市脑洞-幽默代入开篇

依据案例：`batch-144` 案例 718。

## 读者承诺

都市脑洞开头先用幽默降低设定理解成本，让读者愿意进入荒诞或新奇设定；有效幽默不是一次性笑点，而是在正文中反复调用，形成稳定风格。

## 核心循环

```text
新奇设定 -> 幽默切入 -> 读者放松 -> 代入增强 -> 同类幽默复用 -> 风格稳定
```

## 开篇必备

- 一个有脑洞的都市异常。
- 一个普通人能理解的日常入口。
- 第一段内出现幽默处理。
- 幽默要服务设定，不要脱离剧情讲段子。

## 期待来源

读者期待的是“设定很怪但我能轻松看懂”。幽默让陌生设定变亲近，也能让主角声音更快立住。

## 断档风险

- 只讲设定，读者进入成本高。
- 只搞笑，不推进异常和冲突。
- 幽默只出现一次，后文风格断裂。
- 为了笑点破坏角色处境的压力。

## 训练任务

写一个 800 字都市脑洞开头：用一个日常场景切入异常设定，前 300 字内出现幽默处理，并在后文复用同一种幽默模式至少 2 次。
