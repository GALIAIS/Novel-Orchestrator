# 方法论-订阅向入门三幕式结构

依据案例：`batch-177` 案例 884。

## 读者承诺

把订阅向入门、三幕式课程、评论区补充和写作书资料整理成可复习、可实践的学习资产。

## 核心循环

```text
反复上课 -> 提炼框架 -> 整理导图 -> 拆分复习 -> 应用到开篇大纲
```

## 开篇必备

- 明确从混全勤转向冲订阅的目标。
- 把三幕式、订阅向和大纲方法放在同一张结构图里。
- 大图过长时拆分，便于查看和复习。
- 标注来源，保护知识产权。

## 期待来源

作者期待零散课程变成可执行结构，后续能直接指导开头、前三万字和订阅向目标。评论区的保存和求源文件说明这类资产复用价值高。

## 断档风险

- 只整理不实践。
- 导图过大，复习成本高。
- 只抄课程，不转成自己的开书步骤。
- 忽略来源和水印。

## 训练任务

把一门写作课整理成一页导图，只保留能指导下一本书开头和前三万字的节点，并立刻写一个开篇大纲。
