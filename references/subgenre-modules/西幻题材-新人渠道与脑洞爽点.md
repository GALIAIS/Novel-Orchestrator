# 西幻题材-新人渠道与脑洞爽点

依据案例：`batch-314` 案例 1569

## 读者承诺

新人写西幻要先评估市场难度和自身训练路径。纯西幻不容易出头，维多利亚黑魔法这类氛围题材必须转化成可见脑洞和爽点。

## 核心循环

```text
西幻想法 -> 判断渠道/受众 -> 识别新人难点 -> 加入脑洞爽点 -> 或转向更易训练题材
```

## 开篇必备

- 不只写氛围，要有明确爽点。
- 西幻元素要和主角利益、能力、冲突绑定。
- 前三章兑现一个可感知脑洞。
- 平台选择服从题材基本盘。
- 转型作者优先拆目标书仿写。

## 期待来源

读者期待的不是“像西方”，而是新规则下的利益、危险、能力、反差和爽点。

## 断档风险

- 纯西幻受众窄，冷启动难。
- 维多利亚背景只剩设定氛围。
- 黑魔法师没有爽点转化。
- 只问网站，不问读者为什么追。

## 训练任务

把一个西幻设定改写成前三章卖点表：第一章危机，第二章能力，第三章收益。
