# 社群互动-课程预告催更氛围

依据案例：`batch-184` 案例 919。

## 读者承诺

课程、视频或连载内容的催更互动，本质是期待管理和社群氛围维护。

## 核心循环

```text
读者等待 -> 发帖催更 -> 作者给进度 -> 评论玩梗 -> 期待维持
```

## 开篇必备

- 给出可信进度。
- 给出短期时间窗口。
- 允许轻量玩梗。
- 不夸大承诺。

## 期待来源

读者知道内容快来了，催更就能从焦虑变成互动。

## 断档风险

- 长期不回应。
- 给出不可信时间。
- 把低价值动态强行拔高成技法。
- 玩梗盖过正式信息。

## 训练任务

写一条内容预告：包含进度、预计时间、互动梗点和不夸大承诺的边界。
