# 女频废材-红绿折线拆解

依据案例：`batch-022` 案例 108。

## 读者承诺

用红绿标注拆女频废材流开篇，看每几百字如何完成一次情绪上行或下行。

## 核心循环

```text
受虐下行 -> 穿越上行 -> 家族压迫下行 -> 聪明反击上行 -> 男主压制下行 -> 契约暧昧上行 -> 新危机
```

## 开篇必备

- 原身受虐。
- 女主穿越后的接管。
- 家族、婚约或身体缺口。
- 男主强压和绑定关系。

## 期待来源

- 女主从废材处境里反击。
- 男主压制与暧昧交替拉扯。
- 每段都有情绪变化，不让读者平读。

## 断档风险

- 废材背景写太长。
- 暧昧戏不带转折。
- 超过 800 字没有情绪变化。
- 只复述剧情，不标上行下行。

## 训练任务

拆一本女频前三章，用红色标上行、绿色标下行。统计每次转折间隔字数，并找出最长平段。
