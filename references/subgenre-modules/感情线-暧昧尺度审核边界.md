# 感情线-暧昧尺度审核边界

依据案例：`batch-177` 案例 885。

## 读者承诺

暧昧情节可以服务人物塑造和关系推进，但必须控制尺度、观感和审核风险。

## 核心循环

```text
亲密互动 -> 尴尬张力 -> 角色反应 -> 性格显露 -> 隐晦处理/替换桥段
```

## 开篇必备

- 先确认桥段服务的人物功能。
- 避开具体生理反应和脖子以下细节。
- 严查期不试探尺度。
- 拿不准时问编辑。

## 期待来源

读者期待暧昧推进和人物反应，而不是被直白生理描写打断。低风险尴尬事件同样可以完成性格塑造。

## 断档风险

- 为性张力牺牲作品安全。
- 描写过直白导致审核或举报风险。
- 读者观感不适。
- 核心功能其实可用更低风险事件替代。

## 训练任务

把一个高风险暧昧桥段改成误会型、肢体接触型、台词尴尬型三个版本，保留人物性格功能，删掉具体生理描写。
