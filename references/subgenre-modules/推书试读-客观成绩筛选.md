# 推书试读-客观成绩筛选

依据案例：`batch-145` 案例 721。

## 读者承诺

推书不是凭个人口味闲聊，而是为作者筛出可学习样本。样本必须有客观成绩支撑，并说明网站、专区标签和可学习优点。

## 核心循环

```text
发现作品 -> 验证成绩 -> 标注标签 -> 提炼优点 -> 形成学习样本
```

## 开篇必备

- 书名和网站。
- 销售榜、月票榜、均订精品、热榜等成绩证据。
- 细分标签。
- 一个具体可学习点。

## 期待来源

读者期待获得“这本为什么值得学”的判断，而不是“这本我喜欢”。客观成绩降低样本噪音，细分标签方便按题材调用。

## 断档风险

- 只有主观喜欢，没有成绩证据。
- 只有书名，没有题材标签。
- 只说好看，不说学什么。
- 把读者向书单混成作者训练库。

## 训练任务

找 3 本同题材成绩达标作品，分别写推书试读卡。每张卡必须包含书名、网站、成绩、标签、一个可拆桥段和一个可学习技法。
