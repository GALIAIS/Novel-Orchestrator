# 完读数据诊断-口径拆分与观察窗止损

依据案例：`batch-404` 案例 2019

## 读者承诺

先确认完读指标的分母、终点、窗口、入口和样本量，再用章节漏斗定位流失；只做一次有限单变量修复，并按预设观察窗决定继续、暂停或止损。

## 核心循环

```text
核对指标口径 -> 固定来源观察趋势 -> 定位最早流失点 -> 提出单一可修复假设 -> 有限改稿复测 -> 评估时间收益与训练价值 -> 达标继续下一单元 -> 未达标复盘止损
```

## 开篇必备

- 分母、阅读终点和统计窗口清楚。
- 记录入口来源与有效样本量。
- 查看章节级留存而非只看总比例。
- 区分内容改善与样本筛选。
- 改稿前写明观察窗和退出门槛。
- 合同、账户和隐藏规则核验现行文本。

## 期待来源

低比例不再触发盲目硬撑或裸切；作者能指出数据究竟说明什么、最早问题在哪、哪项修改值得一次实验，以及何时停止。

## 断档风险

- 直接套用他人的固定百分比阈值。
- 连载完读与完结完读混为一谈。
- 比例上涨就认定内容改善。
- 同时改简介、开篇、题材和节奏。
- 观察窗内被每日波动反复改变决定。
- 用固定百万字数替代训练目标。
- 切书后不保留复盘和可复用资产。

## 训练任务

查清一个完读指标的口径并画前十章漏斗，只改最早流失点的一个变量；在改前写明观察窗、最低样本及继续和止损门槛，期满按预设执行。
