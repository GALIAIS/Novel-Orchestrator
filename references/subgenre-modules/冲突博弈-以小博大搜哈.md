# 冲突博弈-以小博大搜哈

依据案例：`batch-028` 案例 137。

## 读者承诺

主角明牌不大，但敢把赌注推到敌人不敢跟，从心理压迫里赢出爽点。

## 核心循环

```text
敌人优势 -> 主角不退 -> 赌注升高 -> 敌人误判 -> 第三方给台阶 -> 主角收获
```

## 开篇必备

- 敌人明面优势要成立。
- 主角人设要支撑敢押大赌注。
- 赌注升级后双方都有代价。
- 第三方打断要给敌人合理退路。

## 期待来源

- 弱势主角突然不退。
- 敌人骑虎难下。
- 主角底牌真假难辨。
- 敌人退场后主角气势变强。

## 断档风险

- 敌人秒怂显得降智。
- 主角人设不硬却强行莽。
- 第三方调停太机械。
- 主角没有实质收获。

## 训练任务

写一场主角真实底牌很小的对峙，让主角靠赌注烈度和人设气势逼退优势敌人。
