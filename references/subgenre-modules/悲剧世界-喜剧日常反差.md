# 悲剧世界-喜剧日常反差

依据案例：`batch-079` 案例 394。

## 读者承诺

这个模块用于写“世界很坏，但主角团生活很好笑”的反差型故事。它承诺稳定提供喜剧日常，同时让读者从细节里感到世界底色的残酷。

## 核心循环

```text
残酷世界底色 -> 主角团喜剧事件 -> 细节泄露真相 -> 欢乐不被摧毁 -> 余味加深
```

## 开篇必备

- 第一场事件必须按喜剧来写。
- 世界残酷只能露出少量细节。
- 主角团要有保护欢乐的关系或机制。
- 不能用突然死亡、血腥反转欺骗读者。
- 结尾保留“还能继续笑”的承诺。

## 期待来源

- 读者享受轻松互动。
- 读者意识到背景并不轻松。
- 喜剧承诺稳定，安全感不破。
- 残酷细节让笑点有余味。
- 奇迹和偶然让主角团继续活得开心。

## 断档风险

- 写成开头欺诈式虐转。
- 残酷世界只当装饰，没有细节。
- 悲剧压过喜剧，承诺变调。
- 主角团欢乐缺少合理保护。
- 只剩设定反差，没有可看的日常事件。

## 训练任务

写一个 1500 字日常喜剧场景，背景设定为末世、战乱或社会衰退。场景必须让主角团赢得一个小快乐，同时埋入 3 个证明世界残酷的细节。
