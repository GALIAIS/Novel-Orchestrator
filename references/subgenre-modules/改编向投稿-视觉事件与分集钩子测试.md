# 改编向投稿-视觉事件与分集钩子测试

依据案例：`batch-400` 案例 1998

## 读者承诺

在不牺牲小说可读性的前提下，把人物目标、能力规则和冲突外化成可画场景，并用分集任务与集尾问题检验跨媒介适配。

## 核心循环

```text
可视核心 -> 人物与能力外化 -> 场景动作转折 -> 分集任务 -> 集尾钩子 -> 样章与场景卡双重验证
```

## 开篇必备

- 一句话核心能形成视觉画面。
- 人物辨识度通过行为和造型显出。
- 关键冲突有具体对象、动作和空间。
- 能力规则与代价可重复视觉识别。
- 每集改变任务或关系状态。
- 集尾问题由本集结果推出。

## 期待来源

读者一眼理解核心设定，随人物完成具体任务，并被每集产生的新状态自然牵引到下一集。

## 断档风险

- 把改编向理解成小说可以写差。
- 重要冲突全靠内心旁白。
- 视觉场面没有人物目标。
- 每集只是均分字数。
- 钩子与本集结果无关。
- 用适配想象替代版权合同核验。

## 训练任务

把三章纲改成三集场景卡，每场写地点、目标、动作、转折、视觉点和离场状态；将一段内心戏改成道具、对话和空间行动三版。
