# 反派模板-屠龙者成恶龙

依据案例：`batch-323` 案例 1613

## 读者承诺

曾经的英雄变成反派不怕俗套，怕的是中间没有让读者信服的腐蚀、剧变或价值观滑坡。

## 核心循环

```text
正义履历 -> 主角尊敬/读者认可 -> 权力或诱惑腐蚀 -> 恶行败露 -> 主角背叛感 -> 决裂制裁
```

## 开篇必备

- 反派曾经真正做过的好事。
- 主角为什么尊敬或信任他。
- 腐蚀他的结构性压力或诱惑。
- 当前恶行的证据。
- 主角价值观与反派价值观的冲突。

## 期待来源

读者期待知道英雄为何堕落，也期待主角在惋惜中完成价值观决裂。

## 断档风险

- 只写“他变坏了”，不写转折逻辑。
- 过度洗白反派，削弱主角的善。
- 把反派辩解写得太满，冲淡制裁爽感。
- 主角背叛情绪不足，决裂就不疼。
- 惋惜写太多，立场变模糊。

## 训练任务

写一个三段式反派小传：昔日义举、当前败露、退场前一句未完全解释的经历。重点写主角如何从尊敬转为决裂。
