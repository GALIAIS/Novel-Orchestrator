# 写作自信-主干逻辑与蓝本精读

依据案例：batch-353 案例 1764

## 读者承诺

写作自信来自可执行的判断标准：知道自己写什么主干、参考哪本蓝本、哪些元素该删、哪些信息该动作化。把所有解释写出来只会堆信息，取舍清楚才会恢复掌控感。

## 核心循环

```text
自我否定/解释过多 -> 重列主干线 -> 设置阻碍和帮助 -> 精读蓝本建立标准 -> 削减杂音元素
```

## 开篇必备

- 一句话故事主干。
- 一个明确参考蓝本。
- 主角阶段目标和阻碍。
- 元素取舍清单。

## 期待来源

- 上下行填空让剧情重新可控。
- 蓝本对照提供取舍标准。
- 动作替代解释，节奏变轻。
- 伏笔设定文档减少重复说明。

## 断档风险

- 只处理心态，不处理故事逻辑。
- 扫榜过多，精读不足。
- 元素都想要，主题和类型混杂。
- 为怕读者不懂而过度解释。

## 训练任务

把当前构思所有元素列成表，标注“主干必需/后置/删除”。再选一本蓝本精读前三章，记录它如何交代设定而不堆解释。
