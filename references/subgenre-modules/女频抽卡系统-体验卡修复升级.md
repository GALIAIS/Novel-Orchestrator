# 女频抽卡系统-体验卡修复升级

依据案例：`batch-086` 案例 428。

## 读者承诺

女主用半损毁系统抽到名臣或工具人，以限时体验卡解决现实难题，再通过赚钱、经营和抱资源修复系统，解锁更高级卡池，完成身份和势力跃迁。

## 核心循环

```text
抽到体验卡 -> 限时解决难题 -> 获得资源/能量 -> 修复系统 -> 解锁高阶卡池 -> 更大局面
```

## 开篇必备

- 第一次召唤必须早出现。
- 体验卡要立刻解决一个现实问题。
- 系统损坏限制要制造阶段目标，而不是削弱爽点。
- 卡池分级要清楚：普通名臣、名将、稀有谋士、传说人物。
- 修复资源要和女主现实主线绑定。

## 期待来源

- 下一次会抽到谁。
- 限时人物能创造多大收益。
- 修复系统后能否永久召唤或延长时限。
- 女主如何借名臣能力抱上更大资源。
- 卡池升级会引出怎样的权谋、经营和情感变化。

## 断档风险

- 前期用不了几次系统，读者看不到系统流承诺。
- 抽卡只写设定，不写第一次收益。
- 赚钱修复和主线脱节。
- 卡池没有分级，抽奖缺少期待。
- 系统作为伏笔藏太久，开头亮点没写出来。

## 训练任务

写一个女频名臣抽卡开篇。前 300 字出现第一次召唤，前 700 字让体验卡人物解决现实难题，结尾给出下一档卡池诱惑。
