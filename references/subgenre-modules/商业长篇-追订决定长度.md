# 商业长篇-追订决定长度

依据案例：batch-279 案例 1392

## 读者承诺

商业网文的长度不由预设阅读时长决定，而由追订、收益、主线余量和新书机会决定。有追订就继续扩展，收益衰退或新书过关再收束。

## 核心循环

```text
连载成绩 -> 追订验证 -> 收益判断 -> 主线扩展 -> 完结/续写决策
```

## 开篇必备

- 记录追订和单章收益。
- 判断主线是否还能自然扩展。
- 观察读者疲劳信号。
- 准备新书承接。

## 期待来源

作者的商业安全感来自持续付费读者，而不是固定“最佳字数”。读者还追，作品就仍有商业生命。

## 断档风险

- 先验规定玄幻必须多少字。
- 用个人阅读疲劳替代追订数据。
- 新人成绩一般还强行拖长。
- 收益已低却没有新书计划。

## 训练任务

为一部连载设 3 个节点：30 万、100 万、300 万。每个节点写清继续写和完结的量化条件。
