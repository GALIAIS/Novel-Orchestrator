# 情绪折线-队友下行主角可靠

依据案例：batch-260 案例 1300

## 读者承诺

队友可以误判和失望，读者可以短暂下行，但主角必须保持可靠。下行的目的不是削弱主角，而是让读者等主角反转。

## 核心循环

```text
队友误判 -> 情绪下行 -> 主角沉稳 -> 读者保留期待 -> 主角反转兑现
```

## 开篇必备

- 队友看到的错误信息。
- 读者知道的主角能力。
- 主角可靠的信号。
- 下行幅度。
- 反转兑现点。

## 期待来源

期待来自“队友以为完了，但读者知道主角大概率有办法”。读者跟着下行一点，同时等主角证明自己。

## 断档风险

如果隐瞒金手指让主角也显得心虚，读者会失去安全感；如果队友情绪直接等同读者情绪，就会误伤主角代入。

## 训练任务

写一段队友视角误判主角失败的情节，同时给读者一个主角有把握的细节信号，最后用反转兑现。
