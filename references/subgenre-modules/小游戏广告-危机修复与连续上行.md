# 小游戏广告-危机修复与连续上行

依据案例：batch-371 案例 1854

## 读者承诺

小游戏广告的吸引力来自可视化危机、可操作修复、状态上行和更大危机；种田经营文可直接借用这条链。

## 核心循环

```text
破败处境 -> 修复操作 -> 状态好转 -> 新问题 -> 更大危机 -> 升级修复
```

## 开篇必备

- 角色受苦。
- 环境问题可视化。
- 修复动作明确。
- 每次修复有反馈。

## 期待来源

- 读者想帮角色。
- 破屋修好带来收获感。
- 老鼠、漏雨、暴风雨持续升级。
- 金手指力挽狂澜。

## 断档风险

- 危机不可视。
- 修复没有反馈。
- 问题重复不升级。
- 猪逼操作只让人烦，不引导期待。

## 训练任务

写一个经营开头：5 个破败问题逐个修复，每个给奖励，同时引出更大环境危机。
