# 后宫修罗场-关系推动剧情

依据案例：batch-295 案例 1474

## 读者承诺

后宫修罗场的剧情不是外部任务硬推，而是女主、主角、秘密、偏爱之间的关系变化自然催生事件。

## 核心循环

```text
人设差异 -> 关系诉求 -> 话题/误会触发 -> 主角被迫回应 -> 关系位置改变
```

## 开篇必备

- 至少两名角色有不同关系诉求。
- 主角当前偏向或秘密明确。
- 一个会改变关系位置的话题。
- 场景结束必须产生关系新状态。

## 期待来源

- 读者期待谁被偏爱、谁吃醋、谁结盟。
- 女主之间的竞争和试探带来张力。
- 主角每次回应都会制造下一轮关系钩子。

## 断档风险

- 只塞任务，不改变关系。
- 人设标签化，角色之间没有立场差。
- 修罗场只吵闹，不逼主角做选择或付代价。

## 训练任务

写一个双女主同场桥段：用同一称呼、同一资源或同一秘密触发碰撞，并写出碰撞后的关系变化。
