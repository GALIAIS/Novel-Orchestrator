# 贫民窟施舍-善良反智毒点

依据案例：`batch-137` 案例 681。

## 读者承诺

主角善良要体现为可靠地解决问题，而不是用不考虑环境的方式制造更大危险。帮助弱者要先判断对方能否守住帮助。

## 核心循环

```text
发现弱者 -> 判断环境风险 -> 选择低风险帮助 -> 保护受助者 -> 留下关系钩子
```

## 开篇必备

- 受助者处境明确。
- 环境危险明确。
- 帮助方式不能暴露对方。
- 主角表现出判断力。
- 后续关系或回报有钩子。

## 期待来源

- 主角善良且聪明。
- 低风险帮助更可信。
- 弱者得到实际改善。
- 读者认可主角人设。

## 断档风险

- 在贫民窟给弱者大额钱财。
- 善良变成圣母反智。
- 主角不考虑后果。
- 让读者替受助者担心。

## 训练任务

写 500 字施善桥段：主角帮助贫民窟小孩，禁止直接给大额钱财，必须写出环境判断和低风险帮助方式。
