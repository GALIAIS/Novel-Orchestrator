# 拆书素材包-结构折线复用

依据案例：`batch-051` 案例 251。

## 读者承诺

把拆书从零散笔记升级成可复用素材包：文字批注负责细节，结构折线负责大局观，脑图/图片负责复盘和迁移。

## 核心循环

```text
案例正文 -> 批注 -> 章节结构 -> 爽点折线 -> 冲突标注 -> 自己作品复用
```

## 开篇必备

- 明确拆解章节范围。
- 每章只抓核心事件、爽点和冲突。
- 用折线标出情绪上行下行。
- 文字批注补充细节原因。
- 工具只是辅助，不替代正文阅读。

## 期待来源

- 学习者能直观看到好书节奏。
- 多爽点密集出现时，折线能显示节奏压迫。
- 冲突和爽点放在同图，能提升结构感。
- 导出图片比专用源文件更容易传播复用。

## 断档风险

- 把素材包误当唯一拆书规则，会限制思路。
- 只看图不读正文，会学到空结构。
- 软件格式门槛太高，会降低复用率。
- 折线画得细但不总结迁移点，仍然学不会。

## 训练任务

拆一本书前 8 章，做一张“章节-爽点-冲突-情绪方向”表，再写出 3 条能迁移到自己开篇的技巧。
