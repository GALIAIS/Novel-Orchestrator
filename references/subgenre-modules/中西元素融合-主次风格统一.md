# 中西元素融合-主次风格统一

依据案例：`batch-042` 案例 210。

## 读者承诺

中西元素混搭不天然违和。只要分清主次，让世界观底盘和核心超凡元素在体验上统一，读者会优先关注事件和爽点。

## 核心循环

```text
现代底盘 -> 异质元素 -> 主次分明 -> 规则统一 -> 事件证明
```

## 开篇必备

- 明确什么是皮，什么是血肉。
- 不急着解释文化差异。
- 用事件证明元素能共存。
- 第一单元让读者看到核心爽点。

## 期待来源

- 现代都市降低进入门槛。
- 西幻/异质元素提供新鲜感。
- 统一规则减少违和。
- 主角用混搭规则解决具体问题。

## 断档风险

- 刻意强调文化冲突却处理不好。
- 只缝设定，不给事件。
- 主次不清，读者不知道看什么。
- 用热度替代可读性判断。

## 训练任务

写一个现代都市加西幻元素的事件卡：标明皮、血肉、统一规则和第一处爽点。
