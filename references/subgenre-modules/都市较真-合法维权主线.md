# 都市较真-合法维权主线

依据案例：`batch-171` 案例 852。

## 读者承诺

用现实受损事件激起代入，再让主角在合法规则内较真反击；爽点来自证据、流程、对方反扑和规则胜利，不是单纯金额膨胀。

## 核心循环

```text
现实委屈 -> 系统任务/资源 -> 合法证据 -> 对方反扑 -> 规则反击 -> 主线升级
```

## 开篇必备

- 第一事件要足够具体，最好是读者现实中能理解的委屈。
- 系统资源要有限制，不能随便撒钱。
- 主角反击必须踩在合法或平台规则上。
- 第一事件要能延展成一条长主线。

## 期待来源

读者期待自己现实中难以完成的较真被主角完成，并且看到对方在规则内一步步被反制。

## 断档风险

- 金额过大导致失真。
- 系统万能撒钱，现实规则失效。
- 第一事件太短，撑不起长线。
- 只写打脸，不写证据链和流程。

## 训练任务

设计一个“合法较真”都市开头：写出受损事件、3 个证据节点、2 次对方反扑、系统限制和第一主线结案方式。
