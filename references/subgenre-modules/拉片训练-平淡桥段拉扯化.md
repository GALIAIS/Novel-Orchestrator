# 拉片训练-平淡桥段拉扯化

依据案例：`batch-020` 案例 100。

## 读者承诺

把原本干巴巴、过得太快的桥段，扩成有误判、压制、见证和兑现的情绪拉扯段。

## 核心循环

```text
找爽点 -> 找草草带过处 -> 增加误判 -> 增加见证 -> 兑现反应 -> 接回主线
```

## 开篇必备

- 原桥段本来要满足的读者需求。
- 谁能误判主角。
- 谁能见证结果。
- 主角何时出手。

## 期待来源

- 读者想看主角证明、反击、解决或获得认同。
- 下行与误判延长期待。
- 多方反应让爽点更厚。

## 断档风险

- 把情绪展开误认为水文。
- 只赶主线，爽点没被充分消费。
- 扩写时新增无关设定。
- 没有见证者，兑现显得干。

## 训练任务

找一个 300 字以内解决的爽点，扩写成 1000 字。新增内容只能是误判、压制、见证、反应四类。
