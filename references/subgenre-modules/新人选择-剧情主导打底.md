# 新人选择-剧情主导打底

依据案例：`batch-056` 案例 280。

## 读者承诺

这个模块解决新人在人设主导和剧情主导之间摇摆的问题。它承诺的是先用剧情主导打底，保证目标、冲突和主线推进稳定，再让人物魅力在事件中显形。

## 核心循环

```text
新人选样本 -> 判断主导模式 -> 评估自身能力 -> 剧情主导先行 -> 人物在事件中表现 -> 稳定后再提高人物占比
```

## 开篇必备

- 第一单元有明确目标和冲突。
- 人物性格通过选择和行动展示。
- 不把日常互动当成唯一留人手段。
- 先保证主线推进，再补关系和人物魅力。
- 学人物流样本时，必须判断自己是否能复现其语言和日常质感。

## 期待来源

- 读者知道主角现在要解决什么。
- 冲突提供即时追读动力。
- 人物在压力下做选择，性格自然显形。
- 主线稳定后，关系互动不会散掉。
- 后续可以逐渐增加人物戏比例。

## 断档风险

- 新人误学人物流，只剩散淡日常。
- 没有危机、冲突和反转，读者不知道为什么继续看。
- 用万订样本证明自己也能人设主导，却忽略作者基本功。
- 冷门频道样本被误当成主流商业路径。
- 剧情主导写成工具人流水，忘记让人物在事件中变化。

## 训练任务

同一设定写两个 1000 字开篇：一版人物主导，一版剧情主导。分别写出读者继续看的理由；如果人物主导版说不清留人点，就先采用剧情主导版。
