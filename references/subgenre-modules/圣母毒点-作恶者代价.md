# 圣母毒点-作恶者代价

依据案例：`batch-105` 案例 525。

## 读者承诺

主角可以善良，但不能无条件原谅直接伤害自己的人。作恶者必须付代价，善后救人只能发生在清算之后。

## 核心循环

```text
主角受害 -> 反击/救援 -> 作恶者代价 -> 区分其余人 -> 恩威并施 -> 后续经营
```

## 开篇必备

- 作恶者、中立者、阻止者分清。
- 作恶者必须受惩罚。
- 不杀必须有硬理由。
- 救人要有读者能接受的代价或投名状。

## 期待来源

- 反杀和清算。
- 情绪平衡。
- 善良有原则。
- 后续经营不恶心读者。

## 断档风险

- 被吃还养活对方。
- 反派无代价变队友。
- 角色人设和救人方式冲突。
- 男频女频都可能爆圣母毒点。

## 训练任务

写一个村庄冲突处理表，把所有村民分三类并分别处理。直接作恶者不能进入“被救”名单。
