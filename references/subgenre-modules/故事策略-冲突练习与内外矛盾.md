# 故事策略-冲突练习与内外矛盾

依据案例：batch-365 案例 1825

## 读者承诺

冲突训练要同时检查外显行动冲突和内在情感冲突，并把教材概念转译成网文可执行任务，不必机械照搬练习。

## 核心循环

```text
强对手/高目标/高赌注 -> 外部冲突 -> 内在阻碍 -> 冲突升级 -> 有意义解决
```

## 开篇必备

- 可见的强阻力。
- 主角暂时够不到的目标。
- 明确赌注。
- 一个会影响选择的内在阻碍。

## 期待来源

- 冲突激烈可见。
- 危险不断升级。
- 主角必须先处理内心问题。
- 解决方式可信且有意义。

## 断档风险

- 只有外部打斗，没有内在选择。
- 冲突靠解释不可见。
- 赌注很大但读者不信。
- 照搬教材练习，不转成自己的类型场景。

## 训练任务

检查前三章：每章列一个外部冲突、一个内在阻碍、一个升级点和一个有意义解决；缺失处补事件。
