# 动作打斗-血腥尺度过审边界

依据案例：`batch-175` 案例 875。

## 读者承诺

动作打斗可以写出伤害、危险和代价，但不依靠重口血肉细节刺激读者。

## 核心循环

```text
冲突升级 -> 动作命中 -> 伤害结果 -> 角色反应/战术变化 -> 胜负推进
```

## 开篇必备

- 明确打斗的胜负目标。
- 用受伤体现危险，而不是停留猎奇。
- 普通刀伤、刺伤、流血可简洁呈现。
- 重口器官、碎肉、血浆描写要避开或压缩。

## 期待来源

读者期待打斗强度、角色选择、战术变化和胜负结果。伤害描写的作用是制造代价感，不是让阅读变成生理不适。

## 断档风险

- 长段描写血浆、内脏、肢解。
- 忽略平台、时期、审核人和举报差异。
- 受伤后没有战术后果，只剩血腥画面。
- 为了避审完全不写代价，导致打斗轻飘。

## 训练任务

写一段 800 字动作戏，只允许写伤口、流血、疼痛和动作变形，不写内脏、碎肉、血浆飞溅；最后检查受伤是否改变了战术选择。
