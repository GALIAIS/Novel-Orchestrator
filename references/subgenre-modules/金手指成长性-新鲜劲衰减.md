# 金手指成长性-新鲜劲衰减

依据案例：`batch-115` 案例 572。

## 读者承诺

强金手指不能只靠开篇新鲜。读者会期待玩法升级、适用场景升级和主要矛盾升级；如果后期只是重复同一规则，新鲜劲会快速衰减。

## 核心循环

```text
开篇强规则 -> 首次爽感 -> 玩法重复风险 -> 阶段升级 -> 解决更高矛盾 -> 维持长线兴趣
```

## 开篇必备

- 第一次使用要足够亮。
- 写清金手指边界。
- 预留成长维度。
- 主角身份变化后仍有适用空间。
- 不让金手指替主角做选择。

## 期待来源

- 读者期待下一种用法。
- 规则升级带来新问题。
- 身份提升后矛盾层级变化。
- 金手指继续服务主角行动。

## 断档风险

- 只有固定奖励或固定避坑。
- 当主角资源变多后金手指失去意义。
- 基调变化导致原卖点不适配。
- 强制剧情感压过主角选择。

## 训练任务

给你的金手指写三阶段玩法：初期保命、中期争资源、后期解决主要矛盾。每阶段必须有新用法。
