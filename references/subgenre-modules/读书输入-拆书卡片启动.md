# 读书输入-拆书卡片启动

依据案例：`batch-095` 案例 472。

## 读者承诺

读完理论书只是输入完成，不是能力形成。真正的写作训练要把理论变成拆书、拉片和卡片：每个概念都必须绑定样本、片段和下一次写作动作。

## 核心循环

```text
读理论 -> 选概念 -> 找样本 -> 拆片段 -> 写卡片 -> 回到正文练习
```

## 开篇必备

- 每次读书只保留少量概念。
- 每个概念都要找网文样本，不停在外国理论书原句。
- 卡片必须写“我下一章怎么用”。
- 不用追求一次读透，先形成输出闭环。

## 期待来源

- 新人输入后的启动感。
- 理论被样本验证后的清晰感。
- 卡片积累带来的素材安全感。
- 写作时能直接调用的结构模板。

## 断档风险

- 看了很多书但没有任何样本卡。
- 把理论名词当成能力。
- 卡片只摘抄，不写应用场景。
- 输入太久不写正文，热情耗尽。

## 训练任务

从今天读过的内容里选 1 个概念，找 1 个网文片段，写 1 张卡片。卡片只写三段：概念、样本、我下一章怎么用。超过 300 字就删。
