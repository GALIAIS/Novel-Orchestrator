# 团宠小可怜-下行必回弹

依据案例：`batch-033` 案例 163。

## 读者承诺

小可怜可以让读者心疼，但每次下行到下限都要有人出气，把情绪拉回上行。

## 核心循环

```text
可怜处境 -> 心疼铺垫 -> 下行到限 -> 大佬出气 -> 打脸张弛 -> 团宠兑现
```

## 开篇必备

- 女主可怜点具体。
- 侧面证明她值得被宠。
- 同章安排保护或出气。
- 打脸中继续插情绪牵拉。

## 期待来源

- 读者心疼女主。
- 等待舅舅/大佬出手。
- 可怜越实，出气越爽。
- 团宠关系持续兑现。

## 断档风险

- 只虐不回弹。
- 连续打脸没有张弛。
- 大佬宠爱来得没铺垫。
- 心疼感没有转化为爽点。

## 训练任务

写一个团宠小可怜单章：必须包含可怜、心疼、出气、打脸后亲情反馈四步。
