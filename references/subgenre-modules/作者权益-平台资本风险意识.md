# 作者权益-平台资本风险意识

依据案例：`batch-331` 案例 1655

## 读者承诺

作者除了学怎么赚钱，也要理解平台、资本、合同、版权和流量机制对长期利益的影响。

## 核心循环

```text
写读者爱看 -> 平台分发和合同介入 -> 收益/版权/账号受规则影响 -> 作者建立权益意识
```

## 开篇必备

- 知道收益来源。
- 看懂合同关键条款。
- 明确版权归属。
- 了解分成和推荐规则。
- 评估账号与内容限制风险。

## 期待来源

作者不只会产出内容，也能保护自己长期的作品、账号和收益。

## 断档风险

- 以为写好就必然赚到。
- 只服务读者，忽略平台规则。
- 不看版权和改编权条款。
- 被流量机制牵着改方向。
- 签约前没有风险表。

## 训练任务

为常用平台做权益表：合同、版权、分成、推荐、断更完结、改编权、账号风险，不懂项标红。
