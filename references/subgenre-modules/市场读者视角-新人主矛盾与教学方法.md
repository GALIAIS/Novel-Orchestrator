# 市场读者视角-新人主矛盾与教学方法

依据案例：`batch-253` 案例 1262。

## 读者承诺

市场读者视角模块要求作者按目标平台和题材的读者标准校准正文，而不是泛泛想象“读者会懂”。

## 核心循环

```text
作者脑内效果 -> 正文呈现落差 -> 对标市场读者标准 -> 检查看懂/想追/爽点 -> 回改文本
```

## 开篇必备

- 明确目标平台和题材。
- 找到同题材市场样本。
- 按读者接收信息逐段检查。
- 记录脑内效果和正文效果差距。

## 期待来源

文本达到目标市场读者熟悉的信息密度、节奏和爽点时，读者才会愿意继续追。

## 断档风险

- 脑内特效强，正文呈现弱。
- 找朋友看但无法得到市场判断。
- 放几天再看仍困在作者视角。
- 不区分读者层次。

## 训练任务

拆目标热书前三章，每 500 字记录读者知道什么、期待什么、情绪是什么，再用同表检查自己的正文。
