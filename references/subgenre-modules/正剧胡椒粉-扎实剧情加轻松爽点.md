# 正剧胡椒粉-扎实剧情加轻松爽点

依据案例：batch-297 案例 1483

## 读者承诺

在厚重正剧和复杂设定中插入轻松装逼调味，让读者既获得史诗感，也不被持续高压劝退。

## 核心循环

```text
重剧情推进 -> 阅读负荷上升 -> 轻松爽点释放 -> 顺带推进线索 -> 回到主线
```

## 开篇必备

- 主线剧情本身扎实。
- 轻松段落贴着主角行动。
- 装逼或趣味点要服务人物魅力。
- 调味段顺手推进一条线索。

## 期待来源

- 读者在复杂剧情后得到情绪释放。
- 主角魅力不被设定压住。
- 线索推进让轻松段不显水。

## 断档风险

- 只搞笑不推进主线。
- 正剧一直高压，读者疲劳。
- 调味过量，破坏原本史诗感。

## 训练任务

在一段严肃剧情后写一个轻松装逼尾声，必须同时推进一个新线索。
