# 双女主治愈-互救对称与长线成长发动机

依据案例：`batch-402` 案例 2009

## 读者承诺

让两位女主都拥有缺口、能力和不可代劳的成长任务；关系可以阶段性不对称，却要长期完成给予、接受、边界、分离和重新选择，以个人与关系双重变化支撑长篇。

## 核心循环

```text
两种孤独相遇 -> 小型交换形成互救 -> 共同压力升级关系 -> 依赖失衡暴露边界 -> 分离后各自行动 -> 更平等地重新选择 -> 共同面对下一阶段生活
```

## 开篇必备

- 两位主角各有外部困境和内部缺口。
- 双方拥有不同且有限的支持能力。
- 第一次相遇完成可见的小交换。
- 共同场域还能持续制造新事件。
- 每单元同时改变两人和关系状态。
- 文字用感官、物件和动作建立亲密证据。

## 期待来源

被看见和回赠带来温暖，依赖可能吞没主体性制造拉扯；两人最终能独立行动却仍选择彼此，形成更有重量的治愈。

## 断档风险

- 一人长期充当完整的救助者。
- 新难题加安慰不断重复。
- 关系只升温，没有边界冲突。
- 分离后仍由对方替代完成成长。
- 疾病、残障或创伤被关系神奇治愈。
- 文字只描述外形可爱，缺少事件证据。
- 包装没有准确预告双向关系承诺。

## 训练任务

设计八个单元，每单元分别记录 A、B 和关系的状态变化；中段必须因依赖失衡分离，两人各自完成一项对方不能代劳的选择后，才允许重新结盟。
