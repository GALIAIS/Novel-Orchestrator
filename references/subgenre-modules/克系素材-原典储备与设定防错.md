# 克系素材-原典储备与设定防错

依据案例：batch-381 案例 1905

## 读者承诺

克系素材要作为原典校验库使用。写前查出处、禁忌和能力边界，再决定忠实引用或二创变体。

## 核心循环

```text
素材包 -> 原典扫读 -> 设定索引 -> 正文校验 -> 二创标注
```

## 开篇必备

- 要使用的克系元素。
- 原典出处。
- 禁忌或能力边界。
- 二创改动理由。

## 期待来源

- 设定更有底气。
- 恐怖感来自准确禁忌。
- 视觉素材辅助气质。
- 读者质疑时有依据。

## 断档风险

- 只借怪名不查原典。
- COC 与原典混用不自知。
- 素材链接失效不归档。
- 二创改动没有边界。

## 训练任务

选一个克系元素写 300 字设定索引，再写 800 字正文片段，只保留一个准确设定点和一个二创变体。
