# 经验杂糅-新世界探索感

依据案例：`batch-064` 案例 316。

## 读者承诺

这个模块用于玄幻、奇幻、卡牌定制文、异世界冒险和强设定文。它承诺的是把长期积累的游戏、小说、题材经验重新组合成统一世界规则，让读者重新获得探索新世界、见到新设定的兴奋。

## 核心循环

```text
经验来源 -> 抽取机制 -> 统一包装 -> 第一扇门 -> 新设定出现 -> 冒险升级
```

## 开篇必备

- 至少有 2-3 个作者真正熟悉的经验来源。
- 每个来源只取机制，不照搬外壳。
- 名词、境界、地图和规则要统一包装。
- 第一章要出现可探索的新规则入口。
- 新设定不能只好看，必须改变主角行动。

## 期待来源

- 读者想看下一层世界规则。
- 熟悉机制被新包装后产生新鲜感。
- 冒险和升级同时推进。
- 主角每进入一层，就获得新规则、新资源或新危险。
- 多来源杂糅形成“发现新东西”的感觉。

## 断档风险

- 只有名词包装，没有机制差异。
- 把旧作品外壳拼贴在一起。
- 作者对来源不熟，杂糅后不统一。
- 设定不断增加，但不进入事件。
- 被赚钱焦虑催促，跳过积累。

## 训练任务

列 3 个熟悉来源，每个抽一个机制。把它们整合成一个世界规则，并写第一章主角第一次触碰规则时的目标、风险和收获。
