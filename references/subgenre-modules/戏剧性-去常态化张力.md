# 戏剧性-去常态化张力

依据案例：`batch-015` 案例 071。

## 读者承诺

戏剧性来自特定环境中的去常态化，让读者看到日常生活中难以遇到的人、事、物、能力和关系。

## 核心循环

```text
常态 -> 反常处理 -> 特定环境触发 -> 戏剧性 -> 读者兴趣
```

## 开篇必备

- 从戏剧性产生或即将产生的点进入。
- 明确常态是什么，反常在哪里。
- 去常态化要服务事业线、爱情线、设定或人设。
- 反常不能破坏常识和人物逻辑。

## 期待来源

- 低概率事件。
- 非常态能力。
- 非常态事态。
- 极端化人设或行为模式。
- 熟悉核心梗配新素材。

## 断档风险

- 只有日常，没有反常。
- 为了反常破坏真实感。
- 使用市场已脱敏套路。
- 戏剧性迟迟不出现，开篇像流水账。

## 训练任务

列一个平常场景，做 5 种去常态化处理，并标注每种处理带来的读者情绪和真实感风险。
