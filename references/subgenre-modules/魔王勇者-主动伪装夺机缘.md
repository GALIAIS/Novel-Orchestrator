# 魔王勇者-主动伪装夺机缘

依据案例：`batch-136` 案例 678。

## 读者承诺

魔王/勇者开头要让主角主动操盘。主角可以进入勇者队伍，但应是伪装、布局、夺机缘，而不是被控制成工具人。

## 核心循环

```text
魔王重生 -> 主动入队 -> 勇者误判 -> 主角暗中夺机缘 -> 勇者感激或背锅
```

## 开篇必备

- 主角主动目标。
- 金手指或重生优势。
- 加入队伍是计划。
- 勇者有误判。
- 第一章有夺机缘爽点。

## 期待来源

- 反勇者套路的新鲜感。
- 表面队友、实则收割的暗爽。
- 主角掌控局面。
- 游戏世界暗示带来悬念。

## 断档风险

- 第一章被精控。
- 主角无反制无计划。
- 魔王题材照搬日轻，读者难代入。
- 游戏设定后置太久，毒点先成立。

## 训练任务

写 1000 字开头小纲：主角主动加入勇者队伍，500 字内展示优势，设置一次“勇者以为被帮，实际主角夺机缘”的事件。
