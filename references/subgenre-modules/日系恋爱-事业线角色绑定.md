# 日系恋爱-事业线角色绑定

依据案例：`batch-188` 案例 938。

## 读者承诺

日系恋爱如果不走纯恋爱，就要把女主和主角事业线绑定，让恋爱对象成为伙伴、竞争对手或主角冲突的衍生点。

## 核心循环

```text
事业目标 -> 恋爱对象绑定 -> 共同事件 -> 情感反馈 -> 事业推进 -> 新关系压力
```

## 开篇必备

- 主角有明确事业或成长目标。
- 女主不是游离的装饰角色。
- 女主能帮忙但不抢主角关键决策。
- 女主冲突能牵动主角行动。

## 期待来源

读者一边看主角推进事业，一边看女主在事件中提供情绪反馈、帮助、竞争和关系升温。

## 断档风险

- 女主只卖萌，剧情可有可无。
- 女主太强抢戏。
- 女主太弱变花瓶。
- 竞争型女主真正伤害主角核心利益，导致感情线难回收。

## 训练任务

写一个单女主开篇设定：主角事业目标、女主功能、女主独立冲突、两人共同事件、章节末情感反馈各写一句。
