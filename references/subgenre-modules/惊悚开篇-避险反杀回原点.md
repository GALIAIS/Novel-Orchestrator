# 惊悚开篇-避险反杀回原点

依据案例：`batch-335` 案例 1675

## 读者承诺

惊悚开篇要让主角被迫避险、快速反杀小怪，再用回到原点或更大困境制造下一层恐惧。

## 核心循环

```text
日常入口 -> 异常暗示 -> 主角避险 -> 能力伏笔/执行计划 -> 小怪反杀 -> 更大困境回收
```

## 开篇必备

- 主角进入危险有合理需求。
- 异常信息逐步加码。
- 主角主动避险而非主动送死。
- 能力存在有轻量伏笔。
- 小怪战尽快给反杀或阶段结果。

## 期待来源

读者期待紧张下行后马上回血，再看到更大的规则、空间或怪异困境。

## 断档风险

- 金手指突然出现。
- 主角明知危险还下行。
- 为渲染气氛拖慢小怪。
- 只逃不反击导致憋屈。
- 回到原点没有新信息。

## 训练任务

写 800 字惊悚开篇：300 字日常与异常，500 字避险、反杀和更大困境回收；小怪必须本章处理。
