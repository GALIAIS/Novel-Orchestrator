# 主角优势-无金手指基本功

依据案例：`batch-013` 案例 062。

## 读者承诺

没有显性金手指，也必须让读者看到主角优势。优势可以来自脑子、认知、人设反差、价值观冲突和行动选择。

## 核心循环

```text
建立预期 -> 主角优势介入 -> 反转预期 -> 价值观碰撞 -> 行动结果 -> 他人反应
```

## 开篇必备

- 主角优势要通过行动证明，不靠旁白吹。
- 反差人设要在具体事件中呈现。
- 价值观冲突必须推动选择。
- 没有外挂时，更要有清晰的优越感来源。

## 期待来源

- 读者想看主角如何利用脑子赢。
- 读者想看预期被反转但逻辑自洽。
- 读者想看“正、赢、真”这类难题如何被暂时解决。
- 读者想看文风和思辨带来的高级感。

## 断档风险

- 没外挂也没优势，主角只是在解释观点。
- 思辨压过戏剧性，变成布道。
- 反差只有标签，没有事件证明。
- 新人直接照抄高难写法，缺少基本功支撑。

## 训练任务

写一个无系统场景：先让旁人产生错误预期，再让主角用一次选择反转，最后写出他人反应和读者优越感。
