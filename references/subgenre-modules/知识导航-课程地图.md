# 知识导航-课程地图

依据案例：`batch-002` 案例 009。

## 读者承诺

把大型知识库从“收藏夹”变成“可执行课程地图”，让新人能按阶段、问题、专项能力进入，而不是在碎片内容里乱翻。

## 核心循环

```text
总纲定地图 -> 新人规则定边界 -> 方法模块定能力 -> 案例目录定证据 -> 训练任务定落地 -> 复盘入口定迭代
```

## 开篇必备

- 一个总纲入口。
- 一个新人先读入口。
- 一组问题路由。
- 一组专项能力目录。
- 一组案例/训练入口。

## 期待来源

- 使用者知道自己当前该看什么。
- 每个知识点都有对应训练或案例。
- 卡住时能按问题回到路由，而不是继续堆资料。

## 断档风险

- 只有标题，没有说明解决什么问题。
- 只有理论，没有案例入口。
- 只有案例，没有训练顺序。
- 分类按作者收藏，而不是按使用场景。

## 训练任务

把任意 30 篇资料整理成三列：阶段入口、问题入口、专项能力入口。每篇只允许归入 1 个主入口，避免重复收藏制造假秩序。
