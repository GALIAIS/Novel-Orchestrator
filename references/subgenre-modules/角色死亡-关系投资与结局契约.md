# 角色死亡-关系投资与结局契约

依据案例：`batch-402` 案例 2007

## 读者承诺

不以共情次数或固定阈值决定角色能否死亡，而检查关系投资、未回收承诺、主体性、因果公平与长期余波；高投资提高证据门槛，不自动禁止悲剧。

## 核心循环

```text
盘点关系投资和未回收承诺 -> 让风险从既有因果逼近 -> 保留角色知情选择 -> 兑现或有意撕裂承诺 -> 死亡改变人物与世界 -> 追踪长期余波 -> 对照非死亡后果
```

## 开篇必备

- 记录高权重关系事件而非只数出场次数。
- 明确角色目标、未来信号和待回收关系。
- 提前建立危险、敌人能力和世界规则。
- 给角色可见退路和符合能力的选择。
- 区分真死、假死、复活和失踪契约。
- 预先设计立即、卷末和长期三层后果。

## 期待来源

读者知道角色仍有值得活下去的目标，也看见其价值把他推向不可回避的选择；失去很痛，但人物没有被作者临时降智或工具化。

## 断档风险

- 把共情次数当成统一分值。
- 用角色定位自动授予死亡许可。
- 临时降智或能力失效。
- 用完成一项目标宣布人物价值耗尽。
- 只让死亡刺激主角成长。
- 哭一章后关系与世界恢复原状。
- 复活反复稀释不可逆承诺。

## 训练任务

为同一高投资配角写死亡与永久决裂两版结局，均保留知情选择和三层后果；让陌生读者区分自己是不愿失去角色，还是认为因果、承诺或主体性被破坏。
