# 低境界打斗-武侠近身动作

依据案例：batch-260 案例 1299

## 读者承诺

低境界打斗要让读者看到身体、距离、兵器、破绽和代价。不要用高境界的宇宙级气势覆盖小尺度攻防。

## 核心循环

```text
有限能力 -> 近身试探 -> 攻防转换 -> 受伤/破绽 -> 胜负手
```

## 开篇必备

- 战斗空间。
- 双方能力边界。
- 第一招目的。
- 对手反制。
- 身体代价。

## 期待来源

读者期待每一招都影响下一招：距离变化、伤势累积、体力下降、破绽出现，最终形成可理解的胜负。

## 断档风险

低境界角色写出高境界灭世气势，会尺度错位；只写气势不写动作，会失去临场感。

## 训练任务

用武侠尺度写三次攻防：一招试探、一招反制、一招胜负手。每招都写清目的、动作、结果。
