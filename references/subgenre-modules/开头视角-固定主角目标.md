# 开头视角-固定主角目标

依据案例：batch-302 案例 1509

## 读者承诺

开头先固定主角视角、当前动作和目标，让读者抓住故事，而不是被名字、视角、设定淹没。

## 核心循环

```text
主角视角 -> 当前动作 -> 当前目标 -> 当前阻碍 -> 少量设定 -> 章末期待
```

## 开篇必备

- 先说明主角是谁。
- 让主角正在做一件事。
- 目标清楚。
- 设定名词少量释放。

## 期待来源

- 读者知道跟谁走。
- 当前目标提供阅读方向。
- 阻碍拉起期待。

## 断档风险

- 用配角视角展示男主。
- 视角频繁切换。
- 文学性描述过多，主线不清。
- 设定问题一次性涌出。

## 训练任务

重写一个开头：前 500 字只用一个主角视角、一个目标、一个阻碍，删除多余形容词。
