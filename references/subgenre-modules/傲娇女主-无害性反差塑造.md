# 傲娇女主-无害性反差塑造

依据案例：`batch-181` 案例 905。

## 读者承诺

傲娇女主的承诺不是让主角忍受伤害，而是让读者看见高傲外壳下的柔软内核，并期待她被理解、被哄、露出反差。

## 核心循环

```text
骄傲外壳 -> 无害小性子 -> 柔软内核露头 -> 男主理解诱导 -> 反差甜点兑现
```

## 开篇必备

- 傲只是表层，娇必须尽早露出迹象。
- 小性子不能损害主角核心利益。
- 傲娇行为要分场合。
- 给出骄傲来源和防御理由。

## 期待来源

读者想看的不是拔掉她的刺，而是看她在安全场合一点点露出柔软、嘴硬和依赖。

## 断档风险

- 只剩傲，没有娇。
- 不分场合拖累主角。
- 用伤害别人取悦自己。
- 主角无底线忍让，显得像跪舔。

## 训练任务

设计一个落魄傲娇公主：列出骄傲来源、三个无害嘴硬行为、一个绝不能越界的场合和一次柔软内核外露。
