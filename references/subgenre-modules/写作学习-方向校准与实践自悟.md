# 写作学习-方向校准与实践自悟

依据案例：batch-387 案例 1931

## 读者承诺

外部指导用于校准方向和基础方法，自主实践用于形成具体判断。任何学习资源都要通过小规模正文结果验证。

## 核心循环

```text
明确卡点 -> 方法校准 -> 实写验证 -> 结果修正 -> 自主内化
```

## 开篇必备

- 一个具体写作问题。
- 外部方法的可执行步骤。
- 同一文本的对照验证。
- 是否继续投入的判断标准。

## 期待来源

- 常识性错误被提前避开。
- 漫长试错被缩短。
- 方法在正文中产生可见变化。
- 作者逐步形成自己的判断。

## 断档风险

- 把购买当作已经进步。
- 因价格或名气直接判断价值。
- 依赖群聊而不实写。
- 把他人方法当不可质疑公式。

## 训练任务

对同一段文字分别用旧方法和外部方法改写，盲测目标体验后再决定是否继续投入。
