# 职业写作-高产反馈与质量线

依据案例：batch-287 案例 1435

## 读者承诺

高产建立在长期训练、套路内化、商业反馈和质量把控之上，新人应拆机制而不是盲目对标字数。

## 核心循环

```text
长期训练 -> 类型模板熟练 -> 读者/收入反馈 -> 更新压力 -> 高产输出 -> 继续强化熟练度
```

## 开篇必备

- 当前稳定质量字数。
- 已掌握的类型套路。
- 卡文原因记录。
- 读者或数据反馈。
- 提速训练计划。
- 不能牺牲的质量线。

## 期待来源

作者看到职业高产背后的反馈系统，就能把焦虑转成可训练项目：先保证故事能看，再逐步提高字数。

## 断档风险

- 新人直接追日万，导致水文。
- 把码字房压力当成能力本身。
- 只看大佬字数，不看大佬多年训练。
- 没有反馈时过度透支。
- 提速时牺牲冲突、爽点和可读性。

## 训练任务

连续 7 天记录计划字数、实际字数、删改比例、最卡的段落和一个可复用套路，判断自己的稳定质量字数上限。
