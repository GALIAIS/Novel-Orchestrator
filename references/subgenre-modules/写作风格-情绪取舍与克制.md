# 写作风格-情绪取舍与克制

依据案例：`batch-236` 案例 1177。

## 读者承诺

风格来自对同一场景的稳定取舍：写什么情绪、写到多细、哪些明明能写但不写。

## 核心循环

```text
多个可写角度 -> 判断当前需要 -> 选择一个主情绪 -> 删除多余表现 -> 风格稳定
```

## 开篇必备

- 场景目标清楚。
- 可写情绪列出来。
- 选择最服务主线的一种。
- 克制“想到什么都写”的欲望。

## 期待来源

读者获得稳定观感，不会被作者随手展开的霸气、无奈、历史感等多重情绪拉散。

## 断档风险

- 新手式放肆，什么都写。
- 为装一次逼破坏主线节奏。
- 反复犹豫导致写作卡住。
- 把风格误解成辞藻。

## 训练任务

同一场景写三版：霸气、无奈、历史厚重。只选一版入正文，并写明另外两版为什么不该写。
