# 被装逼配角-冲突升级复用

依据案例：`batch-143` 案例 712。

## 读者承诺

让被主角打脸的配角多次出现，靠的是配角自身计划与主角计划反复冲突并逐步升级，而不是硬安排龙套活着挨打。

## 核心循环

```text
配角有目标 -> 与主角冲突 -> 被打脸 -> 升级资源/关系 -> 再冲突 -> 更高层人物入局
```

## 开篇必备

- 配角利益目标。
- 配角资源。
- 与主角第一次冲突。
- 被打脸后的升级方式。
- 后续更高层牵引。

## 期待来源

- 反复误判带来连续爽感。
- 配角升级让打脸不重复。
- 更高层人物提供新压迫。
- 主角每次打脸都有更大收益。

## 断档风险

- 配角只是为装逼而出场。
- 动机只有仇恨。
- 情敌转友方拿捏不好成毒点。
- 多次打脸没有升级，只是重复。

## 训练任务

设计一个可复用配角，写出利益目标、三次冲突、每次升级资源和每次被打脸后继续入局的理由。禁止只用仇恨驱动。
