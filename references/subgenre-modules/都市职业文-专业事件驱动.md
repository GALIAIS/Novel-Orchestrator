# 都市职业文-专业事件驱动

依据案例：`batch-035` 案例 174。

## 读者承诺

都市职业文承诺的是专业能力带来的新鲜感、可信感和爽感。行业细节必须进入事件，而不是停在科普。

## 核心循环

```text
职业难题 -> 外行误判 -> 主角专业操作 -> 内行认可 -> 资源/地位收益
```

## 开篇必备

- 明确职业流程。
- 给出读者能懂的专业难题。
- 主角能力通过操作证明。
- 结果带来可见收益。

## 期待来源

- 外行看懂“这事很难”。
- 主角展示稀缺专业判断。
- 内行或权威认可形成爽点。
- 下一次事件难度升级。

## 断档风险

- 作者不了解行业。
- 只写术语，不写冲突。
- 专业细节和主线无关。
- 职业文变成普通都市爽文换皮。

## 训练任务

为一个职业写 5 个事件源，每个都包含流程、难题、主角操作、误判者和结果收益。
