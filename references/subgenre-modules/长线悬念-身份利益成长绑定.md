# 长线悬念-身份利益成长绑定

依据案例：batch-262 案例 1306

## 读者承诺

长线悬念要持续牵引读者，必须和主角身份、利益、未来成长前景高度相关。谜底不只是世界背景，而是会改变主角自身。

## 核心循环

```text
早期异常 -> 反复暗示 -> 阶段线索 -> 暂不揭底 -> 下一卷继续追问
```

## 开篇必备

- 主角身上的异常。
- 异常与身份的关系。
- 异常与利益的关系。
- 异常与未来成长的关系。
- 第一卷至少三次暗示。

## 期待来源

读者期待知道主角到底是谁、为什么特殊、未来能到什么层级。每次线索都让谜更重要。

## 断档风险

只埋不提，读者会忘；只吊不推进，读者会烦；和主角无关，读者不会长期关心。

## 训练任务

给主角设计一个三卷悬念，每卷写 3 次暗示位置和一次局部线索兑现。
