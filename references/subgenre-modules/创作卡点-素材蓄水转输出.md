# 创作卡点-素材蓄水转输出

依据案例：`batch-038` 案例 187。

## 读者承诺

卡文或卡选题不是空转。把卡点期间冒出的副素材、题目和参考输入记录下来，主任务完成后可转为连续输出。

## 核心循环

```text
复杂任务卡住 -> 记录副素材 -> 补外部输入 -> 想通关键点 -> 主任务完成 -> 素材爆发
```

## 开篇必备

- 写明卡在哪个问题。
- 记录副素材。
- 标出可转化题目。
- 设定主任务完成条件。

## 期待来源

- 卡点仍有积累。
- 素材池能反哺后续文章或章节。
- 读者知道进度。
- 外部输入帮助破局。

## 断档风险

- 只说卡住，不沉淀。
- 灵感散掉。
- 副素材喧宾夺主。
- 主任务无限延期。

## 训练任务

卡文时列 10 个副素材，选 3 个转成章节标题或专栏题目，并写清释放顺序。
