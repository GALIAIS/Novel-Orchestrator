# 平台二次元-刺猬猫订阅收入中层生存

依据案例：`batch-178` 案例 887。

## 读者承诺

垂直二次元平台的价值不只在头部天花板，也在题材精准后中层作者相对容易稳定吃饭。

## 核心循环

```text
平台读者精准 -> 题材适配 -> 订阅转化 -> 榜单分层 -> 中层生存/头部冲高
```

## 开篇必备

- 判断平台主流读者。
- 判断作品题材是否适配。
- 区分中层目标和头部目标。
- 认清无渠道收入带来的天花板。

## 期待来源

作者期待现实收入预期。平台型二次元文要先服务明确读者，再谈冲榜和月收入。

## 断档风险

- 只盯月十万头部个例。
- 忽略平台内部收入分层。
- 把主站逻辑直接套到垂直站。
- 没有方法论却期待稳定月万。

## 训练任务

调查目标平台榜单前 20，记录题材、更新、订阅/可见收入和原创同人比例，给自己定一个中层可持续目标。
