# 全球求生模拟器-现实行动承接

依据案例：`batch-040` 案例 197。

## 读者承诺

全球求生模拟器文不能只让主角连续模拟。模拟提供情报，现实行动改变未来，全球流系统提供世界和社交反馈。

## 核心循环

```text
现实危机 -> 模拟情报 -> 现实行动 -> 系统反馈 -> 新未来 -> 全球频道/交易承接
```

## 开篇必备

- 模拟有冷却、次数或代价。
- 主角现实里使用新能力。
- 全球流系统元素出现。
- 区域频道有功能角色。

## 期待来源

- 读者等模拟结果。
- 读者等现实破局。
- 全球频道提供气氛和对照。
- 交易、兵种、建筑拉长线。

## 断档风险

- 主角只模拟不行动。
- 没限制导致原地刷天赋。
- 全球流只剩面板。
- 陌生世界和模拟器设定同时挤爆第一章。

## 训练任务

写一个第一单元：最多模拟 2 次，每次后必须现实行动，并安排区域频道和交易钩子。
