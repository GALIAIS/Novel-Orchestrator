# 节奏学习-影视标杆扒节奏

依据案例：`batch-165` 案例 822。

## 读者承诺

通过高节奏影视或网文样本，拆出小兑现和大兑现的间隔，解决“剧情节奏慢”的问题。

## 核心循环

```text
找到标杆样本 -> 标目标 -> 标阻碍 -> 标小兑现 -> 标大兑现 -> 迁移章节节拍
```

## 开篇必备

- 先确认自己慢在目标晚、阻碍弱，还是兑现间隔长。
- 选择节奏强的样本，不选择纯氛围样本。
- 同时拆影视转折和网文章节钩子。
- 每章至少有一个可感知事件功能。

## 期待来源

读者期待短周期有小冲突，中周期有大压力，章尾能接上新问题。

## 断档风险

- 只说“节奏快”，不统计兑现间隔。
- 小冲突太密但主线不推进。
- 大冲突太远，读者中途失去追读理由。
- 影视节奏直接照搬到网文章节，缺少章尾钩子。

## 训练任务

拆一个 10 节点样本，给每个节点标注目标、阻碍、兑现。最后写出自己的“三章小兑现、五章大兑现”版本。
