# 仙侠玄幻-长生仙气边界

依据案例：`batch-069` 案例 342。

## 读者承诺

这个模块用于区分仙侠和玄幻。它承诺的是用等级体系、文化气质、终极目标和能力来源判断题材边界，避免书名说仙侠、正文却写成自由混搭玄幻。

## 核心循环

```text
题材定位 -> 检查仙/长生/古典文化 -> 检查能力来源 -> 统一世界气质 -> 确定仙侠或玄幻
```

## 开篇必备

- 等级体系是否出现仙。
- 世界气质是否古典。
- 主角目标是否指向长生或超脱。
- 能力是否来自修炼和学习。
- 外来宗教、科技、异族元素是否破坏仙侠感。

## 期待来源

- 仙侠读者期待仙气、侠意和求道。
- 长生目标带来长线牵引。
- 古典文化提供稳定氛围。
- 玄幻读者接受更自由的天赋和血脉。
- 题材边界清楚后，读者知道该看什么。

## 断档风险

- 仙侠里乱塞科技和西式宗教。
- 玄幻能力却硬标仙侠。
- 长生只是口号，不进主线。
- 修炼体系和 bug 天赋混杂无规则。
- 佛教元素使用过重导致受众和敏感风险。

## 训练任务

把同一个“少年修炼复仇”设定分别写成仙侠版和玄幻版，各 150 字。仙侠版必须有长生/求道，玄幻版必须有自由能力体系。
