# 肢体语言-情绪外显潜台词

依据案例：batch-269 案例 1345

## 读者承诺

把肢体语言当作潜台词工具，让读者从动作推断紧张、撒谎、不安、开放和防备。

## 核心循环

```text
真实情绪 -> 微动作外显 -> 读者推断 -> 对话产生潜台词 -> 后续揭露
```

## 开篇必备

- 每场只选一两个关键动作。
- 动作要出现在压力点上。
- 不要把动作和心理一一死绑定。
- 动作必须服务对话或关系变化。

## 期待来源

期待来自读者比角色更早察觉异常：他说没事，但手抓袖口；她说坦诚，但杯子一直挡在胸前。

## 断档风险

- 肢体语言堆太多，像资料搬运。
- 动作和场景压力无关。
- 每个角色都用同一种动作。
- 直接解释心理，动作失去意义。

## 训练任务

写一段谈判，不许写心理词。用喉结、手臂、手掌、杯子等 3 个动作制造潜台词。
