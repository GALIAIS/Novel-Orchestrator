# 开头降筛选-少人名强行动线

依据案例：`batch-142` 案例 710。

## 读者承诺

投稿前先降低开头筛选强度：少人名、少复杂线、单主角、单目标、强行动线，让新读者先看懂并留下来。

## 核心循环

```text
前两章信息盘点 -> 删除/延后人名和设定 -> 聚焦主角目标 -> 强行动推进 -> 复杂故事后置释放
```

## 开篇必备

- 单主角视角。
- 一个明确目标。
- 少量必要人名。
- 一条清晰行动线。
- 第 5 章后再释放复杂背景。

## 期待来源

- 读者进入门槛低。
- 推荐期拉人效果更稳。
- 主角目标明确。
- 复杂故事仍可后续展开。

## 断档风险

- 第二章开始人物太多。
- 故事太绕，读者眼花。
- 试图靠放缓节奏解决复杂度。
- 新手开头过早展示大设定。

## 训练任务

标出前两章所有人名、势力名、设定名，判断理解负担。把梗概改成单主角、单目标、少人名、强行动线版本，并列出延后到第 5 章后的内容。
