# 无敌出山-边写边构思乱麻开篇

依据案例：`batch-107` 案例 534。

## 读者承诺

无敌出山开篇要让读者快速看到身份差和第一次碾压，而不是陪作者在正文里构思大纲。灵感可以多，但正文只能保留服务第一条阅读线的内容。

## 核心循环

```text
出山原因 -> 身份差亮相 -> 第一场冲突 -> 无敌能力验证 -> 都市目标开启
```

## 开篇必备

- 一句话说清主角为何出山。
- 前三行给读者抓手。
- 两千字内兑现一次无敌感。
- 人物名和关系保持一致。
- 临时灵感先放大纲，不抢正文。

## 期待来源

- 山中修道与都市秩序的身份差。
- 读者等待主角第一次出手。
- 快速碾压带来的确定爽感。
- 后续都市修仙路线的目标感。

## 断档风险

- 环境描写拖沓，爽点迟迟不到。
- 边写边构思，把大纲写进正文。
- 称呼、口语和低俗词破坏气氛。
- 老题材没有新切入，修道年限反差不足。

## 训练任务

把一个无敌出山开篇压缩成 4 段：出山、入城、冲突、碾压。删掉不能服务这 4 段的设定和描写，另存为素材。
