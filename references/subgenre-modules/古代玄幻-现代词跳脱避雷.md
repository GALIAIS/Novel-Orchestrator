# 古代玄幻-现代词跳脱避雷

依据案例：batch-306 案例 1526

## 读者承诺

古代玄幻、武侠、仙侠读者期待稳定的语感和世界感。严肃土著语境里突然出现现代科学词、现代时间单位，会打断代入。

## 核心循环

```text
古代玄幻语境 -> 检查现代词 -> 判断说话者身份 -> 替换为古风/中性表达 -> 维持代入
```

## 开篇必备

- 土著、旁白、功法口诀尽量不用现代科学词。
- 时间单位用“七日”“数日”等更贴合语境的表达。
- 穿越者吐槽和搞笑桥段可以放宽，但要有明确理由。
- 替换词服务清晰，不堆砌难懂文言。

## 期待来源

期待来自统一世界感。读者相信这个世界，才会继续相信功法、升级、宗门和强者体系。

## 断档风险

- 功法里写“细胞”等现代科学词。
- 古代严肃语境里频繁出现“一周”等现代单位。
- 小白文误以为语感可以随便破。
- 为了古风而写得晦涩，反而影响理解。

## 训练任务

把一段古代玄幻修炼描写中的现代词全部圈出，分成可保留、需替换、必须删除三类，并改写成不出戏的版本。
