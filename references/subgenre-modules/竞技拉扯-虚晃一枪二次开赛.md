# 竞技拉扯-虚晃一枪二次开赛

依据案例：`batch-018` 案例 088。

## 读者承诺

让竞技装逼不只靠一次挑衅直接碾压，而是先小试能力、再退出、再被反派升级局面拉回场，形成更自然的虚晃一枪和二次期待。

## 核心循环

```text
新能力出现 -> 小试证明 -> 主角退出 -> 反派升级 -> 二次开赛 -> 技术兑现
```

## 开篇必备

- 主角获得或激活的新能力。
- 竞技场景的可见规则。
- 反派的初始优势或挑衅。
- 主角第一次小试的合理收益。

## 期待来源

- 读者想看新能力效果。
- 主角退出让读者短暂落空。
- 反派升级装备、赌注、面子或围观压力，把期待重新拉起。
- 最终兑现靠具体竞技细节，而不是一句“系统显威”。

## 断档风险

- 主角刚得能力却没有试验动机。
- 虚晃只靠口头拒绝，没有行动落空。
- 反派二次挑衅没有新增筹码。
- 技术兑现没有细节，导致爽点空泛。

## 训练任务

写一场赛车、格斗、电竞或考试竞技桥段。先让主角小胜一轮，再准备退出；反派必须拿出新筹码，逼出第二轮终局。
