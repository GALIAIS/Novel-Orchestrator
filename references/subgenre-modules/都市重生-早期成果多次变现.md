# 都市重生-早期成果多次变现

依据案例：`batch-135` 案例 674。

## 读者承诺

主角早期一个小成果不要一次用完，要在多个场景、多类人物反应中持续变现，并自然牵出后续主线。

## 核心循环

```text
提前铺行动理由 -> 完成小成果 -> 多角色反馈 -> 多场景再兑现 -> 引出下一主线
```

## 开篇必备

- 主角为什么做这件事要提前传递。
- 小成果要有明确完成场景。
- 权威者、同伴、关键关系人分层反馈。
- 每次反馈推动关系或行动。
- 成果牵出下一条主线。

## 期待来源

- 行动合理顺畅。
- 成果被反复确认。
- 不同人物反应放大爽点。
- 当前主线自然接下一主线。

## 断档风险

- 小成果一次夸完就丢。
- 过渡章不铺后续。
- 反复反馈只是重复夸主角。
- 主角行动理由出现太晚。

## 训练任务

选一个早期小成果，设计 5 次后续变现：权威者、同伴、关键异性、家庭/学校、下一主线。每次必须推动关系或行动。
