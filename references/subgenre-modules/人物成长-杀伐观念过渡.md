# 人物成长-杀伐观念过渡

依据案例：`batch-029` 案例 143。

## 读者承诺

现代人穿越后的狠辣要用事件完成观念转变，不靠一句设定强行变冷血。

## 核心循环

```text
文明惯性 -> 生死威胁 -> 犹豫压制 -> 差点被反杀 -> 反杀复盘 -> 第二次果断
```

## 开篇必备

- 主角原本三观。
- 第一次威胁足够具体。
- 犹豫和身体反应要写出来。
- 事后复盘给出新世界规则。

## 期待来源

- 读者理解主角为什么不能立刻冷血。
- 威胁升级逼主角突破底线。
- 反杀后主角完成第一次成长。
- 后续杀伐有可信基础。

## 断档风险

- 现代人开局无心理成本杀人。
- 用原主灵魂影响制造后续复杂坑。
- 只写狠，不写代价和复盘。
- 成长事件后没有行为变化。

## 训练任务

写第一次杀人和第二次杀人的对照：第一次手抖复盘，第二次更果断但仍符合主角价值观。
