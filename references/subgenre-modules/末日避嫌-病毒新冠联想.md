# 末日避嫌-病毒新冠联想

依据案例：`batch-099` 案例 493。

## 读者承诺

末日可以写，但病毒传播、病害扩散、疫情管理等设定容易联想到现实新冠，审核和影射风险高。要么避开，要么换成完全不同的灾难机制。

## 核心循环

```text
末日设想 -> 检查现实联想 -> 识别疫情风险 -> 替换灾难源 -> 保留求生压力
```

## 开篇必备

- 不使用现实疫情词。
- 不写现实传播链管理细节。
- 灾难机制和新冠联想切开。
- 保留末日求生、秩序崩坏、资源争夺等核心。

## 期待来源

- 生存压力。
- 秩序崩坏。
- 未知灾难。
- 主角资源和能力破局。

## 断档风险

- 读者或审核联想到现实疫情。
- 用隔离、核酸、封控等现实词。
- 病毒设定过近，形成影射。
- 为避嫌删掉末日压力，题材变弱。

## 训练任务

把一个病毒末日设定改成非疫情灾难：异界雾、规则污染、太阳异常任选。写出灾难规则、求生压力、主角第一目标。
