# 末日预知-书名卖点与倒计时

依据案例：`batch-219` 案例 1095。

## 读者承诺

书名直接承诺末日将临、主角预知、信息差布局和未来改变。

## 核心循环

```text
末日题材 -> 倒计时压力 -> 预知能力 -> 提前布局 -> 灾难验证
```

## 开篇必备

- 书名融合题材词、危机词和金手指词。
- 第一章展示未来信息。
- 未来线索保留缺口。
- 主角只能改变有限变量。

## 期待来源

读者期待先知优越感、倒计时紧迫感、布局爽和众人后悔爽。

## 断档风险

- 书名文艺但卖点不清。
- 预知过完整导致流水账。
- “所有人不信我”无限复用。
- 看到未来和加速控制没有边界。

## 训练任务

写 20 个末日预知书名，再设计 3 条不完整未来信息，并写 500 字开局。
