# 网文文笔-顺畅底线与商业主矛盾

依据案例：batch-348 案例 1739

## 读者承诺

新人先把文字做到清楚、顺畅、不碍读，再优先训练故事、矛盾、爽点和节奏；文笔上限靠长期积累。

## 核心循环

```text
文字清楚 -> 读者顺利进入剧情 -> 矛盾爽点驱动追读 -> 长期积累提升上限
```

## 开篇必备

- 句子不挡理解。
- 长短句配合基础节奏。
- 段落排版服务扫读。
- 重点先放故事目标、阻碍和爽点。
- 词句积累作为长期训练，不抢主线训练时间。

## 期待来源

读者读得顺，故事欲望持续推进，文笔作为加分项出现。

## 断档风险

- 把文笔焦虑当成不写故事的理由。
- 追求漂亮句子导致节奏拖慢。
- 背词句但不会构造矛盾。
- 短期强改语感，越急越慢。
- 忽视文笔底线，句子已经影响理解。

## 训练任务

一章稿只改两轮：先改顺，再补强目标阻碍爽点。禁止加漂亮形容词，观察可读性变化。
