# 自我表达-市场输入与完本优先

依据案例：batch-377 案例 1881

## 读者承诺

想写自我价值和留名可以，但商业网文先要理解读者、读火书、写完本。理想不能替代市场输入。

## 核心循环

```text
自我表达冲动 -> 市场输入缺口 -> 读火书拆机制 -> 先完本 -> 再判断商业化
```

## 开篇必备

- 当前阅读偏好。
- 市场样本阅读计划。
- 完本目标。
- 自我表达放置边界。

## 期待来源

- 个人理想提供长期动力。
- 火书输入校准读者需求。
- 完本积累真实经验。
- 兼职保留试错空间。

## 断档风险

- 不爱读网文却想全职。
- 第一书未完就想留名。
- 把自嗨当市场特质。
- 没准备迎合市场就辞职。

## 训练任务

读完一本不喜欢但成功的火书前三十章，记录读者留下理由和自己排斥点，分清市场机制和个人口味。
