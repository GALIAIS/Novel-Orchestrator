# 正剧网文-幽默边界控制

依据案例：`batch-199` 案例 994。

## 读者承诺

正剧需要正常搞笑来调节节奏，但不建议使用出戏的烂梗和恶搞桥段。

## 核心循环

```text
正剧压力 -> 自然笑点缓冲 -> 情绪回落 -> 回到主线
```

## 开篇必备

- 笑点来自人物性格或处境。
- 不破坏主线质感。
- 不插网络烂梗。
- 笑点后能回到正剧情绪。

## 期待来源

读者在严肃剧情里获得呼吸感，同时不被作者玩梗拽出故事。

## 断档风险

- 烂梗出戏。
- 恶搞破坏角色可信度。
- 全程紧绷导致疲劳。
- 网络梗加速文本过时。

## 训练任务

给一段正剧情节加入两个自然笑点：一个人物性格笑点，一个处境反差笑点。
