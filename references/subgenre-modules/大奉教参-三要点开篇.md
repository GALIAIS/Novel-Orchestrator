# 大奉教参-三要点开篇

依据案例：`batch-154` 案例 768。

## 读者承诺

新人开头如果没底，就用成熟样本校准：快速交代主角时地、快速抛冲突、快速展示主角破局资本。

## 核心循环

```text
主角时地 -> 生死/前途危机 -> 金手指或优势资本 -> 冲向第一兑现 -> 解决危机
```

## 开篇必备

- 主角是谁。
- 时间和地点。
- 第一冲突。
- 和书名简介同向的期待。
- 主角的装逼资本。
- 主角马上要采取的行动。

## 期待来源

读者期待主角带着可见资本解决眼前危机，完成第一次爽点兑现。

## 断档风险

- 迟迟不交代主角处境。
- 冲突和书名简介不一致。
- 金手指或优势资本出现太晚。
- 只讲设定，不让主角行动。

## 训练任务

写一个 1500 字开头：220 字内交代主角、时间、地点；800 字内抛冲突；1400 字内展示破局资本和下一步行动。
