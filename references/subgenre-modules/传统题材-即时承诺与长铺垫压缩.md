# 传统题材-即时承诺与长铺垫压缩

依据案例：batch-388 案例 1939

## 读者承诺

传统题材保留长线剧情厚度，但要在第一场代表性事件中提前证明主角差异、秩序失衡和后续核心体验。

## 核心循环

```text
传统核心体验 -> 最小失衡事件 -> 主角独特应对 -> 小结果与长疑问 -> 背景分批展开
```

## 开篇必备

- 可代表长期主线的小事件。
- 主角有代价的差异化选择。
- 当前秩序的一处可见失衡。
- 短闭环和明确长线问题。

## 期待来源

- 熟悉题材迅速出现具体变化。
- 主角的应对证明人物能动性。
- 小结果交付第一口题材体验。
- 长问题为后续铺垫标明方向。

## 断档风险

- 把脑洞包装当唯一路线。
- 开头先讲完整背景和体系。
- 为求速度牺牲题材气质。
- 第一事件与后续主线无关。

## 训练任务

给同一传统题材写背景先行版和事件先行版各 600 字，盲测读者能否说清主角下一步与全书长期看点。
