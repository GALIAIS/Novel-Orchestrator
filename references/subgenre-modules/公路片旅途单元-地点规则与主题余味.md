# 公路片旅途单元-地点规则与主题余味

依据案例：`batch-203` 案例 1014。

## 读者承诺

旅途/公路片网文靠“新地点、新规则、新遭遇、离开后的主题余味”驱动，需要长期路线目标来支撑追读。

## 核心循环

```text
上路 -> 到达新地点 -> 遭遇当地问题 -> 选择介入或离开 -> 带走变化
```

## 开篇必备

- 给主角一个长期路线目标。
- 每个地点有独立规则。
- 单元冲突能体现主题。
- 离开时留下情绪余味或线索。

## 期待来源

读者期待下一站的新奇见闻，以及主角在每一站的选择如何改变自己。

## 断档风险

- 只有旅行，没有单元冲突。
- 单元之间没有积累。
- 主角只是旁观，缺少选择压力。
- 没有长期路线钩子。

## 训练任务

设计 5 个旅途单元：地点、规则、冲突、主角选择、主题余味、下一站钩子。
