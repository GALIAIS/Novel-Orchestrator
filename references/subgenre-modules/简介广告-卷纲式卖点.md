# 简介广告-卷纲式卖点

依据案例：`batch-051` 案例 253。

## 读者承诺

把简介当广告：书名、简介和开篇几章一起筛选读者、抛出卖点、承诺第一卷主线；卷纲式简介适合新人快速压缩主矛盾。

## 核心循环

```text
书名卖点 -> 简介承诺 -> 卷纲压缩 -> 开篇兑现 -> 读者确认可追
```

## 开篇必备

- 简介只写核心卖点，不堆设定。
- 第一卷矛盾要能压成一句广告钩子。
- 前三章必须兑现简介承诺。
- 现代神话复苏要提前控制都市尺度。
- 简介保留悬念，不剧透结果。

## 期待来源

- 书名先吸引目标读者。
- 简介告诉读者这本书看什么。
- 卷纲式简介能给读者方向感。
- 开篇兑现后，广告信任成立。

## 断档风险

- 简介只列元素，读者不知道冲突。
- 简介承诺和开篇不一致，会劝退。
- 神话复苏写成社会黑暗面，会偏离主卖点。
- 卷纲式简介写太满，会失去悬念。

## 训练任务

给自己的书写一版卷纲式简介：包含主角、第一卷矛盾、题材卖点、前三章兑现点，不超过 200 字。
