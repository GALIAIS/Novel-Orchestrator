# 言情开篇-能力救人绑定关系

依据案例：batch-291 案例 1453

## 读者承诺

言情金手指可以是医术等专业能力，开篇用救人同时展示能力和绑定关系。

## 核心循环

```text
高压场景 -> 关键人物濒危 -> 女主能力救人 -> 秘密/恩情绑定 -> 关系线启动
```

## 开篇必备

- 女主能力。
- 高压场景。
- 被救对象身份。
- 救人代价。
- 能力暴露程度。
- 救命后的误会或试探。

## 期待来源

读者看到女主有主动权，同时期待被救者如何回报、怀疑或纠缠她。

## 断档风险

- 能力只自夸，不落到事件。
- 救人后没有关系钩子。
- 金手指和情感线割裂。
- 高身份人物被救却无后续反应。

## 训练任务

写一个女主救高身份人物的开篇梗概，让能力展示和关系绑定同时发生。
