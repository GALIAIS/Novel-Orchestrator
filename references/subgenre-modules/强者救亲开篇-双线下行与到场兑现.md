# 强者救亲开篇-双线下行与到场兑现

依据案例：`batch-402` 案例 2008

## 读者承诺

先证明保护者有能力却暂时缺席，再让弱势亲人以主动选择建立关系投资；双线每次切换都新增危险、赶赴行动和更短期限，最终由核心人物亲自完成救援与身份反转。

## 核心循环

```text
能力与缺席代价 -> 亲人主动守护 -> 弱线危险升级 -> 强线获得信息并赶赴 -> 两线交替缩短窗口 -> 本人到场兑现 -> 保护追责与关系修复
```

## 开篇必备

- 保护者的能力基线和缺席原因可信。
- 亲人有目标、判断与有限行动。
- 每次下行都增加新状态而非重复惨状。
- 两线在信息或后果接点切换。
- “会来”的承诺由不同证据递进。
- 到场后由核心人物解决主矛盾。

## 期待来源

读者知道强者有能力解决，却不知道能否及时赶到；弱势角色不断守住关系承诺，距离缩短与期限收紧共同放大最终释放。

## 断档风险

- 只堆儿童或亲人受虐强迫共情。
- 弱势角色没有任何主动选择。
- 双线切换不传递信息和后果。
- 同一句到场口号机械重复。
- 配角震惊长期替代本人行动。
- 强者临时失能破坏人物基线。
- 救援后不处理情绪债与关系后果。

## 训练任务

写六拍双线救援纲：弱线三拍各有新行动和损失，强线三拍各有赶赴选择和成本；第七拍必须由保护者本人兑现，删除所有不改变状态的重复惨状与震惊。
