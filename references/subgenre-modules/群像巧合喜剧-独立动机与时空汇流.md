# 群像巧合喜剧-独立动机与时空汇流

依据案例：`batch-412` 案例 2057

## 读者承诺

三至五方以独立目标和错误认知行动，共享物件、地点或时间窗口使路线必然靠近；动作跨线传递误解并产生多重后果，最终在可回溯的共同瓶颈汇流。

## 核心循环

```text
各方独立目标与资源 -> 共享瓶颈迫使路线靠近 -> 信息不全下合理行动 -> 物件或误解跨线移交 -> 新视角重释后果 -> 单一动作改变多线 -> 共同汇流 -> 各方分别结算得失
```

## 开篇必备

- 三方各自成立的目标、资源和时间压力。
- 一个共享物件、身份、地点或窗口。
- 作者唯一的时空因果矩阵。
- 只在重释、移交或新后果处切换视角。
- 至少一次影响两条以上线路的动作。
- 汇流后各方不同的收获和代价。

## 期待来源

读者分别理解各方计划后，比人物更早看见共享连接；误解和物件不断跨线，路线逐渐逼近同一瓶颈，所有偶然最终都能追溯到人物自己的行动。

## 断档风险

- 随机把人物传送到一起。
- 多条线没有独立目标和资源。
- 视角切换不提供新信息或重释。
- 一炮多响只停留在功能标签。
- 用段子长期遮掩剧情未推进。
- 汇流时用一段解释抹平所有代价。

## 训练任务

设计三方、一件共享物品和十二小时故事，逐小时记录地点、已知、动作和物件去向；每方至少制造一次影响另外两线的后果，汇流后证明每个巧合都能回溯到早先选择。
