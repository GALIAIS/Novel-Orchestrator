# 弱外挂剧情-特工任务链

依据案例：batch-284 案例 1417

## 读者承诺

弱外挂文要让任务、阴谋、关系反转承担主要爽点。外挂只提供入局资格和破局工具，不能替代剧情压力。

## 核心循环

```text
遗物入局 -> 追杀逃亡 -> 组织接入 -> 能力揭示 -> 特工任务 -> 关系反转
```

## 开篇必备

- 一个能找上主角的遗物或人物。
- 直接追杀压力。
- 清晰组织入口。
- 弱外挂的具体用途。
- 第一卷任务目标。

## 期待来源

读者想看普通人被卷入世界级阴谋后，依靠专业能力、选择和队友关系完成高压任务。

## 断档风险

- 世界观百科压过主角危机。
- 外挂太弱但任务不够强。
- 特工技能只是设定，不进入行动。
- 关系反转没有前置铺垫。

## 训练任务

写一卷特工任务链：入局、追杀、组织、能力、任务、反转六步，每一步写出主角的选择和代价。
