# 主角缺席-存在感维持与舆论下行释放

依据案例：`batch-211` 案例 1055。

## 读者承诺

主角短暂缺席时，也要靠提及、误解、对照和穿插主角线维持存在感，并安排情绪释放。

## 核心循环

```text
配角危机 -> 主角被提及 -> 舆论误解 -> 主角线穿插 -> 真相反转 -> 情绪释放
```

## 开篇必备

- 缩短配角单线篇幅。
- 配角嘴里或局势里持续提到主角。
- 舆论压力要有误导源。
- 主角胜利要靠特质、金手指或反制逻辑。

## 期待来源

读者虽然暂时看不到主角行动，但能持续等待主角介入和打脸释放。

## 断档风险

- 两三章完全无主角。
- 普通人抨击只制造憋屈不释放。
- 配角惨到显废。
- 主角轻松赢导致战力体系塌。

## 训练任务

把配角团灭压缩成一章穿插结构，并检查主角存在感、舆论来源、释放对象、敌人逼格、配角不废。
