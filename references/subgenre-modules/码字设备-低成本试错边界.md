# 码字设备-低成本试错边界

依据案例：`batch-080` 案例 397。

## 读者承诺

这个模块用于处理新人码字设备焦虑。它承诺先保证能写和能活下去，再谈硬件升级；兼职试错阶段用最低可行设备即可。

## 核心循环

```text
生存优先 -> 最低设备可写 -> 连续产出 -> 稿费验证 -> 再升级硬件
```

## 开篇必备

- 先确认生活费和工作是否稳定。
- 列出已有设备。
- 手机加蓝牙键盘作为最低方案。
- 购买设备前先证明自己能持续写。
- 只用稿费或稳定预算升级。

## 期待来源

- 写作门槛被压低。
- 作者不再把没设备当借口。
- 低成本试错降低失败损失。
- 稿费反哺形成正循环。
- 工具选择服务产出。

## 断档风险

- 吃饭问题没解决就全职赌网文。
- 买设备代替写作训练。
- 追求机械键盘、电脑配置、软件工具。
- 兼职阶段投入过重。
- 没有产出却继续升级硬件。

## 训练任务

用现有设备完成 7 天码字记录。每天写明设备、字数、耗时、最大阻碍。第 7 天判断是否真的需要买设备。
