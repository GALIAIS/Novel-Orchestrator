# 内投失败复盘-第一章特色抛出

依据案例：`batch-044` 案例 217。

## 读者承诺

内投失败后先查第一章是否抛出了独特卖点。普通危机、普通杀怪、普通逃生都不够，读者要看到主角与其他书不同的能力和前途。

## 核心循环

```text
投稿失败 -> 定位期待缺口 -> 强化独特能力 -> 书名直给 -> 第一章重写
```

## 开篇必备

- 第一章出现特色能力。
- 特色能力能变成资源。
- 书名或简介直给卖点。
- 世界压力能放大能力价值。

## 期待来源

- 主角不是普通杀怪。
- 怪物、敌人或困境能被转化为资源。
- 读者看到乱世出头可能。
- 编辑能一眼识别差异点。

## 断档风险

- 公开处刑没大问题就误以为能过。
- 书名深藏不露。
- 第一章只写普通危机。
- 独特能力藏到后面。

## 训练任务

把一个普通怪物危机改写成特色能力危机：主角必须把怪物变成资源、手下或优势。
