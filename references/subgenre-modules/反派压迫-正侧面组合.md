# 反派压迫-正侧面组合

依据案例：`batch-060` 案例 296。

## 读者承诺

这个模块用于反派、怪物、强敌首次登场。它承诺的是用侧面铺垫、环境压迫、外形呈现、他人反应和当面伤害组合出可信威胁。

## 核心循环

```text
侧面传闻 -> 他人恐惧 -> 环境异常 -> 反派现身 -> 当面制造代价 -> 主角产生逃脱/反制目标
```

## 开篇必备

- 反派出场前先有人怕他。
- 环境要先压低安全感。
- 外形或声音要有可视化记忆点。
- 第一次行为必须证明威胁。
- 主角当下不能轻松解决反派。

## 期待来源

- 读者想知道主角如何活下来。
- 反派越压迫，反制期待越强。
- 配角恐惧让危险先入为主。
- 当面代价让威胁可信。
- 压迫感成为后续升级目标。

## 断档风险

- 只用形容词说反派恐怖。
- 没有侧面铺垫，出场突兀。
- 反派只摆造型，不造成代价。
- 主角太早压过反派。
- 模仿强风格作品，只学反转不学铺垫。

## 训练任务

写一个反派登场：先用三个人的不同反应铺垫，再让反派出场造成一个不可逆损失。写完删除所有“可怕、恐怖、压迫”形容词，检查是否仍然成立。
