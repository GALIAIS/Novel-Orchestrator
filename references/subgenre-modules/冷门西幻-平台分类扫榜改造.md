# 冷门西幻-平台分类扫榜改造

依据案例：`batch-089` 案例 443。

## 读者承诺

西幻、轻克苏鲁、法师精灵等冷门元素不能只靠情怀开书。先扫榜确认平台和分类，再把核心卖点向都市异能、副本、网游模拟器、同人或其他热门标签改造。

## 核心循环

```text
冷门题材 -> 扫平台榜 -> 判断分类 -> 借热门标签 -> 改开篇 -> 前五万验证
```

## 开篇必备

- 明确不是纯西幻还是混合题材。
- 找到平台上最近同类样本。
- 选择读者能识别的分类。
- 保留核心气质，改造入口标签。
- 前五万必须足够出彩。

## 期待来源

- 魔物复苏如何影响现代秩序。
- 种族传承如何带来升级。
- 副本如何连接中世纪魔幻世界。
- 主角能力和现实身份如何绑定。

## 断档风险

- 纯西幻逆潮流硬上。
- 平台分类选错。
- 不扫榜就问别人去哪发。
- 50 万字大纲锁死，无法改造。

## 训练任务

起点、番茄、小站各扫 5 本相近题材，记录分类、标签、首章卖点和读者反馈。把自己的西幻构思改成一个更容易被平台识别的版本。
