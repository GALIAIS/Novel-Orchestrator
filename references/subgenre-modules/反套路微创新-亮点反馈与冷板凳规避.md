# 反套路微创新-亮点反馈与冷板凳规避

依据案例：`batch-204` 案例 1019。

## 读者承诺

反套路微创新能提供亮点、吸引读者反馈、提高下限和上限，避免新人把字数耗在过时套路冷板凳上。

## 核心循环

```text
新题材/新金手指 -> 读者被亮点吸引 -> 反馈增多 -> 迭代更快 -> 商业回报提高
```

## 开篇必备

- 旧套路中找可翻新的点。
- 新金手指或核心梗一眼可见。
- 书名简介先传达亮点。
- 前三章兑现创新承诺。

## 期待来源

读者希望看到新鲜核心梗，而不是等更一本没有新意的旧套路。

## 断档风险

- 用“基本功不足”逃避创新。
- 只有创意，正文不兑现。
- 继续写过时套路导致长期单机。
- 亮点没有包装到书名简介和开篇。

## 训练任务

选一个老套路，写 3 个微创新方案：新金手指、书名卖点、前三章展示方式、读者反馈入口。
