# 期待感四法-素材库驱动剧情

依据案例：`batch-197` 案例 985。

## 读者承诺

期待感结构必须由新鲜素材承载；没有素材，逻辑正确也会无聊。

## 核心循环

```text
压制/奖励/未知/围观 -> 素材承载 -> 期待形成 -> 兑现升级 -> 素材库更新
```

## 开篇必备

- 明确使用哪种期待法。
- 给出能撑住兴趣的素材。
- 压制-准备-反制优先练熟。
- 神秘奖励、未知、围观要有新鲜事件。

## 期待来源

读者不是只等结构兑现，而是在等一个具体、有趣、新鲜的素材结果。

## 断档风险

- 只套方法，没有素材。
- 素材过时，不符合市场。
- 未知世界不新鲜。
- 围观事件没有吃瓜价值。

## 训练任务

用四种期待感各写一个 200 字桥段，并标注素材来源；无聊的桥段必须换素材。
