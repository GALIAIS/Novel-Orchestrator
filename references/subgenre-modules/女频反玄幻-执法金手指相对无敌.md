# 女频反玄幻-执法金手指相对无敌

依据案例：`batch-094` 案例 469。

## 读者承诺

主角穿进弱肉强食的玄幻/修仙世界，用执法金手指审判传统恶习：滥杀、夺宝、灭门、废天赋、垄断修行知识。读者要看的不是普通升级，而是现代秩序感对旧玄幻恶习的碾压、打脸、劳改和关系推进。

## 核心循环

```text
发现恶习 -> 受害者被压迫 -> 主角依据规则执法 -> 相对无敌碾压 -> 惩罚/劳改/剥夺力量 -> 秩序收益扩大 -> CP 关系推进
```

## 开篇必备

- 开场展示一个传统玄幻恶习，最好足够不公平。
- 执法金手指要能立刻改变局面。
- 主角不是靠普通升级赢，而是靠规则权限和审判权赢。
- CP 或原龙傲天要尽早进入事件，承接女频情感期待。

## 期待来源

- 反传统玄幻的解气感。
- 相对无敌带来的低压装逼打脸。
- 判官笔、法律书、监狱等道具的规则感。
- 弱肉强食世界被重新建立秩序的长线收益。
- CP 对主角执法方式的误解、配合、心动和立场变化。

## 断档风险

- 写成普通男频升级，女频平台承诺被稀释。
- 金手指太无敌但没有案件升级，后继无力。
- 只讲法律设定，不把法律变成事件。
- 情感线太晚进入，耽美读者缺少关系牵引。

## 训练任务

设计 5 个“玄幻恶习案件”。每个案件写清：恶习、受害者、施害者、执法规则、惩罚方式、围观反应、CP 关系变化、下一层更大秩序问题。
