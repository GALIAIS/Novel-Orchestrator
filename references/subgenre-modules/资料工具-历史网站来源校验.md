# 资料工具-历史网站来源校验

依据案例：`batch-172` 案例 856。

## 读者承诺

历史资料网站可以做快速入口，但创作者必须校验来源、立场和适用范围，避免把单一平台叙述直接写成作品事实。

## 核心循环

```text
资料入口 -> 快速查背景 -> 来源立场校验 -> 多源交叉 -> 剧情素材化
```

## 开篇必备

- 记录资料入口和查询主题。
- 重大历史判断至少交叉两个来源。
- 区分事实、观点和创作改编。
- 涉及战争、制度、民族、政治时尤其谨慎。

## 期待来源

读者期待历史题材有可信质感。可信来自关键事实和制度逻辑稳，而不是只堆百科名词。

## 断档风险

- 把资料网站当唯一依据。
- 忽略平台立场和叙述偏向。
- 架空改编没有标清哪些是事实、哪些是虚构。
- 查到资料但没有转成冲突。

## 训练任务

选一个历史事件，用一个资料网站做入口，再找两个来源交叉核对。提炼 5 条可用事实和 3 条争议风险。
