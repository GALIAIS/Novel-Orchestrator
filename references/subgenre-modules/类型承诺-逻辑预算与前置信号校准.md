# 类型承诺-逻辑预算与前置信号校准

依据案例：`batch-402` 案例 2010

## 读者承诺

用书名、简介、标签和前三章稳定示范主要体验与夸张许可；类型可以降低现实写实要求，却不能豁免人物、规则和前提之后的内部因果。

## 核心循环

```text
包装给出类型信号 -> 开篇连续兑现核心乐趣 -> 明确夸张前提与保护不变量 -> 后续维持内部因果 -> 解释资源集中于关键判断 -> 转调前重新校准 -> 目标读者反馈修约契约
```

## 开篇必备

- 明确首要类型、目标读者和核心乐趣。
- 包装与前三章发出一致信号。
- 只要求读者接受清楚的夸张前提。
- 列出不能破坏的保护不变量。
- 人物知识、动机和选择保持连续。
- 类型转调前给出可识别的预警。

## 期待来源

读者不必按现实尺度检查每个过场，却能相信核心关系、胜负、能力和结果不会被作者随意改写；规则稳定后，夸张变奏反而更自由。

## 断档风险

- 出现漏洞后才临时宣布“不必认真”。
- 把现实偏差与内部因果断裂混为一谈。
- 漏洞替主角解决核心困难。
- 角色为笑点突然失去既有判断。
- 包装许诺轻松，正文长期转为高伤害正剧。
- 用作者猜测代替目标读者反馈。
- 把类型许可理解成无需铺垫。

## 训练任务

审计一个轻松或热血开篇，列出一个夸张前提、三个保护不变量和五个关键因果；让三名目标读者分别标记可接受夸张与破坏体验的漏洞，并指出受损的是笑点、人物、胜负还是承诺。
