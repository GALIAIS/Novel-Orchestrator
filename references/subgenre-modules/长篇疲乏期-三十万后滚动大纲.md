# 长篇疲乏期-三十万后滚动大纲

依据案例：`batch-234` 案例 1168。

## 读者承诺

长篇真正难点常在三十万字后；大纲要滚动修，卷尾要提前规划，存稿不能当永久安全垫。

## 核心循环

```text
新书期顺畅 -> 中段疲乏 -> 卷尾卡点 -> 修大纲 -> 续上下一卷
```

## 开篇必备

- 前期设定和大纲尽量充分。
- 每卷目标清楚。
- 大纲边写边修。
- 关注卷尾和下一卷入口。

## 期待来源

读者在中后期仍能看到主线推进、卷尾爆发和下一卷新期待。

## 断档风险

- 只盯开头，忽视中段。
- 存稿被卡文消耗光。
- 卷尾不知道收什么。
- 大纲写完后长期不更新。

## 训练任务

做卷尾体检：当前卷目标、剩余 5 个事件、卷尾高潮、下一卷开场、存稿余量、大纲待修点。缺项先补规划。
