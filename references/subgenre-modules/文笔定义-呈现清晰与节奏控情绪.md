# 文笔定义-呈现清晰与节奏控情绪

依据案例：`batch-343` 案例 1715

## 读者承诺

文笔是把具体内容呈现得流畅、清晰、有画面，并通过句式和节奏服务读者情绪。

## 核心循环

```text
细纲内容 -> 人物目的行动 -> 清晰画面 -> 句式节奏 -> 情绪推进
```

## 开篇必备

- 先明确人物目的。
- 描写要清晰流畅。
- 场景要有画面感。
- 长短句服务节奏。
- 文笔不能拖慢情节。

## 期待来源

读者期待看懂发生了什么、人物为什么动、画面如何变化，并被节奏推着继续读。

## 断档风险

- 把文笔理解成扩写字数。
- 辞藻像散文但剧情寡淡。
- 只重遣词造句。
- 忽略节奏和情绪。
- 文笔与笔力混为一谈。

## 训练任务

把 50 字细纲扩成 500 字，只检查目的、画面、句式节奏和情绪推进四项。
