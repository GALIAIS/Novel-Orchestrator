# 变百卖点-性转被爱读者画像

依据案例：`batch-191` 案例 955。

## 读者承诺

变百和性转文的核心卖点可不是严密逻辑，而是性转美少女带来的被爱、被宠、低责任和亲密幻想。

## 核心循环

```text
平台受众 -> 热标签筛选 -> 性转美少女 -> 被动接受宠爱 -> 情绪满足
```

## 开篇必备

- 书名直接外显性转或变身。
- 简介补足白毛、龙化、百合、后宫、逆推等热标签。
- 主角弱受或被动接受逻辑成立。
- 早期给出被宠爱和贴贴反馈。

## 期待来源

目标读者期待从现实身份责任中逃离，进入一个只要可爱、弱势或被动，就能被美少女环绕和宠爱的梦境。

## 断档风险

- 只写设定，不给被爱反馈。
- 用传统升级逻辑压过核心情绪。
- 标签和平台受众不匹配。
- 误把逻辑严密当成第一优先级。

## 训练任务

拆一个平台榜单变百作品，列书名标签、简介标签、主角身份变化、被爱反馈和现实心理缺口。
