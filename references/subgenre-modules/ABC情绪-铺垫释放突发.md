# ABC情绪-铺垫释放突发

依据案例：`batch-015` 案例 073。

## 读者承诺

用 ABC 情绪结构把爽文单元拆成铺垫下行、行动释放和意外续接，让新人先能写出可循环的情绪段落。

## 核心循环

```text
A铺垫下行 -> B行动释放 -> C伏笔突发 -> A2下一轮
```

## 开篇必备

- A 要拉期待，不只是交代准备。
- B 要有主角行动和反应层。
- C 要能接下一个故事。
- ABC 都要吸引读者。

## 期待来源

- A 中的压制、嘲讽、困难和误判。
- B 中的主角解决和各方反应。
- C 中的意外、新问题、新伏笔。
- 循环接续带来的不断档。

## 断档风险

- A 一路下行，没有小上行或期待。
- B 只有结果，没有反应。
- C 只是突然，不接下一轮。
- 用 ABC 水字数，但没有情绪功能。

## 训练任务

把一个普通小事写成 ABC。A 至少 3 个铺垫点，B 至少 2 个反应层，C 必须接下一轮 A2。
