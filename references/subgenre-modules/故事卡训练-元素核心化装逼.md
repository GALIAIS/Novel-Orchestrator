# 故事卡训练-元素核心化装逼

依据案例：`batch-014` 案例 069。

## 读者承诺

故事卡训练要求把给定元素变成剧情核心，而不是装饰。装逼卡要有压制、对比、展示、见证和反应延伸。

## 核心循环

```text
抽元素 -> 定故事卡 -> 元素核心化 -> 误判压制 -> 主角展示 -> 多方反应 -> 毒点修补
```

## 开篇必备

- 元素 A 和 B 都要有不可替代功能。
- 装逼对象要能形成有效对比。
- 主角展示要解决当前困境。
- 见证者反应要足够具体。

## 期待来源

- 读者看到别人误判主角。
- 读者期待主角用元素反转。
- 媒体、网友、路人、专业人士反应继续发酵。
- 元素和场景越贴合，爽点越自然。

## 断档风险

- 元素换掉也成立，说明没核心化。
- 只有身份曝光，没有压制和反应。
- 主角行为像反派，或女主人设有毒。
- 为了装逼牺牲常识，需要额外补丁。

## 训练任务

随机抽两个元素和一个故事卡，写梗概后做替换测试：任一元素可替换则重写，直到两个元素都不可替代。
