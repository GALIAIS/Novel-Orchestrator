# 受害工具人支线-配角主体性与救援有效性

依据案例：`batch-408` 案例 2038

## 读者承诺

短戏份配角仍拥有目标、边界和主动选择；反派之恶通过持续利益逻辑显形，主角介入改变正在发生或未来可能发生的结果，并为亲缘决裂与制裁支付代价。

## 核心循环

```text
配角以主动选择建立人格 -> 反派借权势逼近边界 -> 配角抵抗并留下行动路径 -> 主角获得可行动信息 -> 介入改变结果 -> 庇护网络反扑 -> 主角付出代价决裂制裁 -> 配角保有收尾选择
```

## 开篇必备

- 配角在受害事件之外的独立目标。
- 一次可见的善意、勇气或拒绝。
- 反派稳定的利益逻辑与重复模式。
- 危险仍可被改变的时间窗口。
- 主角获得信息和介入的可信路径。
- 亲缘、名誉或安全层面的决裂代价。

## 期待来源

读者先关心一个能行动的人，再担心其边界被权势侵害；主角及时改变结果形成释放，庇护网络与亲缘压力把一次救援升级为有代价的正面决裂。

## 断档风险

- 配角只承担激怒主角的工具功能。
- 用极端伤害替代反派塑造。
- 主角事后宣誓却不创造前向价值。
- 受害者没有抵抗、证据、恢复或选择。
- 一件惨案自动授予主角所有行动正当性。
- 用旧评论代替当前平台规则核验。

## 训练任务

写一条不依赖性暴力和死后触发的受害支线：配角先完成主动选择，主角在场景结束前改变一项结果，并为与亲属反派决裂付出可见代价；删掉主角后，配角仍须保有完整行动链。
