# 新人开篇-主角缺位与大场面失焦

依据案例：batch-259 案例 1294

## 读者承诺

第一章先让读者抓住主角、处境、目标和危机，再让世界观、大场面、群像和反转逐步进入。新人开篇的核心不是证明设定宏大，而是建立代入入口。

## 核心循环

```text
主角登场 -> 当前目标清楚 -> 阻碍压迫 -> 主角行动 -> 世界观作为行动背景逐步展开
```

## 开篇必备

- 第一屏出现主角或明确主角视角。
- 前三段说明主角处境和当下目标。
- 有名角色数量控制在读者能记住的范围。
- 大场面只保留服务主角目标的部分。
- 章尾给出下一步行动，而不是只堆设定。

## 期待来源

读者期待跟着主角解决眼前问题，然后顺手理解更大世界。场面越宏大，越要用主角的具体目标把读者钉住。

## 断档风险

如果开头同时写神界大战、仙魔乱斗、深山老林、多人文戏和复杂反转，读者会先问“谁是主角”。一旦代入失败，后面的宏大设定都会变成负担。

## 训练任务

拿一段大场面序章，删到只剩一个主角、一个对手、一个目标。把世界观解释压缩成三句，剩余信息全部后置。
