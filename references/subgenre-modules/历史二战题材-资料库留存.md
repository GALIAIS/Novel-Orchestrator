# 历史二战题材-资料库留存

依据案例：`batch-197` 案例 981。

## 读者承诺

二战资料入口要沉淀成可检索素材卡，服务军事历史、架空战争、谍战和年代文。

## 核心循环

```text
发现资料 -> 记录入口 -> 标注适用题材 -> 写作前核验 -> 转化成场景细节
```

## 开篇必备

- 资料主题明确。
- 来源链接和提取码记录完整。
- 标注当前是否可访问。
- 不把未读资料写成事实。

## 期待来源

作者需要历史细节时能快速找到资料入口，并知道哪些内容还需要查证。

## 断档风险

- 网盘链接失效。
- 没有标签，后续找不到。
- 未读资料被当作已掌握知识。
- 历史细节未经核验直接使用。

## 训练任务

为一份二战资料做素材卡，只填主题、来源、可用状态、适用题材和待核验问题。
