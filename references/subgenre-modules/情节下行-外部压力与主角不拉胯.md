# 情节下行-外部压力与主角不拉胯

依据案例：`batch-221` 案例 1103。

## 读者承诺

情绪下行应来自外部压力和局势恶化，而不是主角犯蠢、迷茫或拉胯。

## 核心循环

```text
外部压力 -> 局势下行 -> 主角判断在线 -> 操作反击 -> 逆风翻盘
```

## 开篇必备

- 下行来源不是主角犯错。
- 主角判断和能力保持在线。
- 困境压迫感来自敌人、资源、信息差。
- 反击节点清楚。

## 期待来源

读者看主角像看高手操作，在逆风局里用更强判断解决问题。

## 断档风险

- 主角迷茫靠配角拉回。
- 主角捅篓子制造危机。
- 幻境/回忆提前制造压抑。
- 队友鲁莽让读者迁怒团队。

## 训练任务

把主角犯错导致危机改成外部压力导致危机，要求主角全程判断在线并逆风翻盘。
