# 新人实践-火题材反馈驱动学习

依据案例：`batch-206` 案例 1027。

## 读者承诺

新人不必等“准备好”才写热门题材；读者多、反馈多，反而更容易暴露问题并进步。

## 核心循环

```text
新人写作 -> 选择火题材 -> 获得更多读者 -> 收集反馈 -> 复盘短板 -> 继续改进
```

## 开篇必备

- 明确目标热门题材。
- 设定前三万字验证点。
- 每周收集真实反馈。
- 边写边补基础技能。

## 期待来源

作者通过更多读者反馈和收入可能，获得学习动力和方向校正。

## 断档风险

- 用“基础不足”阻止实践。
- 冷题材长期单机，反馈不足。
- 只学习不写，永远没有真实问题。
- 写火题材扑街后不复盘。

## 训练任务

做一个“边写边学”计划：题材、前三万字验证点、反馈渠道、每周复盘项、下一轮修改动作。
