# 斩神都市脑洞-高成绩样本

依据案例：`batch-146` 案例 726。

## 读者承诺

用高收入、高口碑的都市脑洞作品作为拆书样本，帮助同题材作者确认什么值得优先学习，并把样本按平台、成绩和题材标签沉淀。

## 核心循环

```text
高成绩作品 -> 题材定位 -> 推荐拆书 -> 评论背书 -> 扩展同类样本
```

## 开篇必备

- 书名和平台。
- 可验证的成绩或影响力。
- 明确题材：都市脑洞、无敌文等。
- 学习对象：想写同类书的人。

## 期待来源

读者期待的是“这本为什么值得拆”。成绩给可信度，题材标签给方向，评论背书提高优先级。

## 断档风险

- 只有成绩，没有可学习方向。
- 只说好看，不说平台。
- 把短推荐当完整拆书。
- 没有同类样本扩展，无法形成训练库。

## 训练任务

选 3 本都市脑洞或无敌文高成绩样本，分别写一张样本卡：平台、成绩、题材、评论背书、适合谁拆、优先拆哪一块。
