# 素材库-打人七公式参考

依据案例：batch-307 案例 1534

## 读者承诺

素材库要服务具体场景。写打斗、冲突、职业、物件时，资料应转化为动作、反馈、环境和后果。

## 核心循环

```text
描写卡点 -> 检索素材库 -> 抽取动作公式 -> 改写进当前场景 -> 复盘沉淀
```

## 开篇必备

- 先写清当前需要什么素材。
- 按动作、受击、环境、旁人反应分类摘取。
- 只取骨架，不原样堆砌。
- 写后给素材打标签，方便复用。

## 期待来源

期待来自具体感。动作有路径、受击有结果、环境有变化，读者才会感到打击和冲突真实。

## 断档风险

- 只收藏素材，不做索引。
- 素材和当前场景不匹配。
- 公式照搬，段落变机械。
- 缺少受击和后果，动作没有重量。

## 训练任务

做一张打人素材卡：7 个动作、7 个受击反馈、7 个环境反应。任选 3 组改写成一段冲突。
