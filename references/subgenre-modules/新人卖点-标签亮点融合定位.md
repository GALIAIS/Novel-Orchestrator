# 新人卖点-标签亮点融合定位

依据案例：`batch-188` 案例 940。

## 读者承诺

新人想靠网文赚钱，先用热门标签定位读者入口，再把真实长处、主角标签、风格标签和金手指标签融合成可外显卖点。

## 核心循环

```text
扫标签 -> 拆标杆 -> 盘点长处 -> 融合卖点 -> 小纲测试 -> 开篇兑现
```

## 开篇必备

- 明确目标平台和热门标签。
- 至少有风格、主角、金手指或世界设定中的组合点。
- 不选群像等高难结构。
- 书名简介能看出卖点。

## 期待来源

读者通过标签找到书，再通过书名简介和前三章确认这本书能提供自己想看的爽点和亮点。

## 断档风险

- 自嗨写作，却按商品路线期待赚钱。
- 自我感觉良好，误判自己长处。
- 热门标签只蹭词，不懂底层逻辑。
- 霸榜书拉高标签热度，误以为套路成熟。

## 训练任务

列 3 个喜欢且认真拆过的类型，每个写出风格标签、主角标签、金手指/设定标签和自己的长处，再各写 500 字小纲测试哪个最顺。
