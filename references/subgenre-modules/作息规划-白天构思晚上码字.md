# 作息规划-白天构思晚上码字

依据案例：batch-300 案例 1496

## 读者承诺

把低干扰时段留给正文，把白天碎片留给学习、构思、资料和复盘，形成稳定产出。

## 核心循环

```text
白天学习/构思 -> 晚上固定码字 -> 留下接续点 -> 次日继续
```

## 开篇必备

- 明确每天低干扰码字窗口。
- 白天任务不强行替代正文。
- 晚上启动前已有场景目标。
- 收尾时写下明天接续点。

## 期待来源

- 作者减少启动选择。
- 正文有连续时间推进。
- 白天杂事不再吞掉全部写作计划。

## 断档风险

- 全天都在“准备写”，晚上无正文。
- 作息跟精力峰值冲突。
- 每天结束不留接续点，次日启动困难。

## 训练任务

安排一周作息：每天白天一个学习/构思任务，晚上一个正文目标，睡前留下一句明天要写的开头。
