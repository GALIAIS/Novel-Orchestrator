# 命名工具-二次元风格与素材筛选

依据案例：batch-371 案例 1851

## 读者承诺

二次元命名工具适合批量启动灵感，但最终名字要人工筛选，匹配题材、世界观和角色气质。

## 核心循环

```text
工具生成 -> 风格筛选 -> 题材匹配 -> 人工改写 -> 素材库归档
```

## 开篇必备

- 明确题材风格。
- 明确角色气质。
- 生成足够候选。
- 筛掉跳戏名字。

## 期待来源

- 名字提前释放角色风格。
- 二次元网感。
- 轻小说气味。
- 命名效率提升。

## 断档风险

- 工具名直接照用。
- 现实题材过度二次元化。
- 世界观命名体系混乱。
- 名字比角色本身更跳。

## 训练任务

生成 50 个候选名，分成可用、需改写、只适合网名、弃用四类，并为 5 个角色各选 3 个备选。
