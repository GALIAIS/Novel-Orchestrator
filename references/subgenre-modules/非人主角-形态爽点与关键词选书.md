# 非人主角-形态爽点与关键词选书

依据案例：batch-381 案例 1902

## 读者承诺

非人主角要让形态持续产生限制、优势和反差。选样本时用形态关键词、平台标签和成绩筛选，不要只泛问书名。

## 核心循环

```text
非人形态 -> 初始限制 -> 形态优势 -> 进化升级 -> 社会反差
```

## 开篇必备

- 明确非人形态。
- 交流或身份限制。
- 初始能力优势。
- 进化或化形期待。

## 期待来源

- 读者好奇非人限制如何解决。
- 形态优势制造爽点。
- 升级解锁新形态。
- 与人类社会产生反差。

## 断档风险

- 非人只是噱头。
- 形态限制没有戏剧作用。
- 没有升级路线。
- 样本检索不看成绩和平台。

## 训练任务

选一个非龙形态，设计前三章：限制、优势、进化路线各出现一次。
