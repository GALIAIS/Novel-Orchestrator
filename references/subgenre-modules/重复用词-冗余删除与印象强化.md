# 重复用词-冗余删除与印象强化

依据案例：batch-350 案例 1746

## 读者承诺

重复用词不必一律雕琢，优先删除冗余连接词，保留能强化人物和物象印象的描写性重复。

## 核心循环

```text
发现重复 -> 判断功能 -> 冗余词删除 -> 功能性重复保留 -> 不牺牲产能
```

## 开篇必备

- “然后”“接着”先判断能否删除。
- 比喻不一定需要喻词。
- 描写性关键词可重复强化印象。
- “突然”“忽然”可用视角放大替代。
- 不为同义词替换拖慢全章。

## 期待来源

读者读得顺，不被冗余词打断，也能记住关键形象。

## 断档风险

- 过度追求换词导致产能下降。
- 用连接词硬推句子。
- 比喻词堆叠。
- 所有重复都删，反而削弱意象。
- 作者比读者更敏感，误判优先级。

## 训练任务

一章稿只查两类词：可删连接词、影响阅读的重复词。其他近义词替换暂时不做。
