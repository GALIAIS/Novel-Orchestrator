# 怪兽高武-IP抽奖翻盘

依据案例：`batch-143` 案例 713。

## 读者承诺

在怪兽高武危机中，让低位新人靠热门 IP 抽奖能力完成第一次详细翻盘；读者既获得熟悉 IP 的识别快感，也看到能力在新世界里的实战变形。

## 核心循环

```text
新人受不公 -> 怪兽危机 -> IP 能力抽奖 -> 详细实战使用 -> 翻盘打脸 -> 势力线升级
```

## 开篇必备

- 主角低位原因。
- 不公平任务。
- 怪兽压迫。
- 第一章内金手指。
- 熟悉 IP 能力。
- 翻盘后的升级线。

## 期待来源

- 读者认出热门 IP 能力。
- 生死压力逼出第一次使用。
- 能力过程写细，形成画面爽。
- 队伍、上级、导师或家族线继续升级。

## 断档风险

- 只蹭 IP 名字，不写使用过程。
- 金手指翻盘太简略。
- 不公平对待理由不成立。
- 爽点被小纲篇幅限制，没放大情绪。

## 训练任务

写 1500 到 2500 字开头：前 500 字出现不公平危险任务，第一章出现金手指，抽到熟悉 IP 能力，并用 600 字以上写清第一次实战过程。
