# 落笔细节-配角反应与情绪血肉

依据案例：`batch-250` 案例 1249。

## 读者承诺

落笔细节模块把抽象情绪折线转成配角言辞、行动、心理活动和关系层级，让读者实际感到上行、下行和震惊。

## 核心循环

```text
桥段目标 -> 关系铺垫 -> 威胁证明 -> 多层配角反应 -> 主角行动释放
```

## 开篇必备

- 主角和被保护者关系具体。
- 威胁由多个角色反应证明。
- 配角反应分亲近者、旁观者、对立者和高位者。
- 最后必须收束到主角行动。

## 期待来源

读者通过旁人恐惧、劝退、震惊和误判确认事件重量，主角解决时爽点更足。

## 断档风险

- 只说“情绪上行”，正文没有反应细节。
- 配角反应重复，变成水字数。
- 威胁不具体，主角行动价值不足。
- 多层震惊不收束，抢走主线注意力。

## 训练任务

把一句“主角打赢敌人”扩成 600 字，必须写亲友、旁观者、敌人、高位者四类反应，并让每类反应只服务一个情绪变化。
