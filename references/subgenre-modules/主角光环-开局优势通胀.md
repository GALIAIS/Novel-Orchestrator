# 主角光环-开局优势通胀

依据案例：`batch-165` 案例 824。

## 读者承诺

使用高配开局制造入口刺激，同时用限制、代价和外部强压防止主角光环把剧情压平。

## 核心循环

```text
高配身份/血统 -> 快速爽感 -> 遇到优势无法解决的问题 -> 付出代价 -> 外部压力升级 -> 优势重新校准
```

## 开篇必备

- 主角可以有一个强优势，但不能所有优势同时拉满。
- 优势来源要清楚。
- 写出优势不能解决的问题。
- 尽快给出代价或限制。

## 期待来源

读者期待主角强，但也期待强者面对足够硬的阻碍。爽感来自优势和阻碍碰撞，不来自作者直接发胜利。

## 断档风险

- 顶级身份、顶级资源、顶级小弟同时白送。
- 所有危机都被设定自动解决。
- 读者只看到作者安排，看不到主角行动。
- 优势持续通胀，后续没有更强压力可写。

## 训练任务

写一张高配开局限制卡：一个顶级优势、三个解决不了的问题、一个代价、一个能压住主角的敌对结构。
