# 基础理念-执行力优先

依据案例：batch-276 案例 1376

## 读者承诺

把写作训练拉回基础：稳定产出、读懂反馈、完成章节、按问题改稿。高级技巧只有在基础执行力上才有意义。

## 核心循环

```text
理念混乱 -> 回到基础 -> 稳定训练 -> 作品变好 -> 再加高级技巧
```

## 开篇必备

- 先确认每天能写、能改、能复盘。
- 每周只抓一个基础短板。
- 用作品质量验证训练有效。

## 期待来源

作者期待来自可验证进步。不是懂了多少概念，而是章节能不能更顺、更稳、更可读。

## 断档风险

- 收藏技巧但不产出。
- 基础执行力差，却追求复杂理论。
- 把失败归因给外部，忽略训练态度。

## 训练任务

连续 7 天记录日更、改稿、拆书和反馈吸收情况，只改最低分的一项。
