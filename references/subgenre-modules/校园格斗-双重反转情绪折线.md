# 校园格斗-双重反转情绪折线

依据案例：batch-281 案例 1402

## 读者承诺

校园格斗小剧情用两轮上行下行压情绪：同学撑腰失败，主角智取失败，最终在孤立无援处用金手指反杀。

## 核心循环

```text
群情激奋 -> 敌人威慑 -> 主角预案 -> 敌人反预案 -> 最低点 -> 反杀见证
```

## 开篇必备

- 明确被堵人的起因。
- 第一批支援者。
- 敌人的威慑和经验。
- 主角底牌或金手指。

## 期待来源

读者先期待群体撑腰，再期待主角智取，两次期待都被压下去后，主角亲手反杀的爽感更足。

## 断档风险

- 混混只会放狠话，没有反制能力。
- 主角预案直接成功，缺少最低点。
- 死党受辱不足，情义压力不够。
- 围观者没有见证结果。

## 训练任务

写 1800 字校园冲突，必须包含两次希望和两次失败，结尾用围观者反应展示主角战果。
