# 震惊写法-情绪ABC压榨

依据案例：`batch-010` 案例 048。

## 读者承诺

震惊不是反复写旁人张嘴，而是用铺垫、压制、反应、收获和余波，把一个情节的情绪价值榨干。

## 核心循环

```text
事件 -> 情节排序 -> A段铺垫 -> 爆发 -> 多方反应 -> 余波接钩
```

## 开篇必备

- 先区分事件和情节。
- A段要铺动机、困境和期待。
- 震惊对象要有层次。
- 余波要接下个情节。

## 期待来源

- 读者知道为什么值得震惊。
- 不同人物反应提供不同爽感。
- 收获阶段继续释放情绪。
- 旧情节余波能穿插新情节铺垫。

## 断档风险

- 连续堆“众人震惊”。
- 没有事前压制和期待。
- 震惊只发生在高潮一刻。
- 情节结束后没有余波。

## 训练任务

拿一个打脸桥段，拆成“前反应、中反应、后反应”。每层写不同人物、不同情绪、不同作用。
