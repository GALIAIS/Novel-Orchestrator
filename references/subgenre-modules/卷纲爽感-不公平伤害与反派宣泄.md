# 卷纲爽感-不公平伤害与反派宣泄

依据案例：`batch-160` 案例 797。

## 读者承诺

在卷纲事件中集中读者情绪，让不公平伤害、反派归因、期待拉升和爽感兑现形成同一条宣泄链。

## 核心循环

```text
反派制造不公平 -> 主角/代入阵营受害 -> 期待被拉起 -> 主角准备反击 -> 反派被解决 -> 情绪宣泄
```

## 开篇必备

- 反派责任要清楚。
- 主角或读者代入阵营要遭受不公平伤害。
- 解决方式要提前铺垫可行性。
- 暴力、智谋、取巧都要服务情绪宣泄。

## 期待来源

读者期待压迫越清楚，归因越集中，主角反击时越有释放感。

## 断档风险

- 反派实力升级但情绪没有升级。
- 伤害对象分散，读者愤怒不集中。
- 主角不参与关键解决。
- 只追求暴力场面，忽略情绪煽动和拉期待。

## 训练任务

拆一卷三个事件，分别标出伤害对象、施害反派、解决方式和爽感兑现。把最弱事件改到“同一反派同时压迫主角和代入阵营”。
