# 公开处刑-投稿平台与新媒体匹配

依据案例：`batch-225` 案例 1123。

## 读者承诺

公开处刑不只看稿件好坏，还要判断标题、系统、开篇脑洞、叙事手法和目标平台是否匹配。

## 核心循环

```text
带稿求评 -> 看标题卖点 -> 看前三章爽点 -> 看平台读者 -> 调整投递方向
```

## 开篇必备

- 明确稿件字数和开篇卖点。
- 标出系统或金手指。
- 判断叙事短板。
- 对照目标平台的读者偏好。

## 期待来源

作者获得的是平台语境中的修改优先级，而不是泛泛一句好或不好。

## 断档风险

- 附件不可读却强行评价正文。
- 有练习字数就误判为成熟稿。
- 不区分平台风格。
- 只问投哪里，不看开篇是否匹配。

## 训练任务

为一个两万字开头做平台匹配表：标题、题材、金手指、前三章爽点、叙事短板、目标平台、平台读者偏好、需要调整的一处开篇。
