# 反派主角-坏而不恶狠而不毒

依据案例：`batch-327` 案例 1633

## 读者承诺

市场能接受坏主角，但要有代入、底线、格局；坏可以，恶不行，狠可以，毒不行。

## 核心循环

```text
现实不满/强目标 -> 灰黑手段 -> 保留底线 -> 达成更大结果 -> 坏英雄爽感
```

## 开篇必备

- 给主角可代入的目标。
- 说明他为什么不走正常路。
- 灰黑手段服务目标，而不是享受伤害。
- 保留底线。
- 写出格局和审美，避免低贱。

## 期待来源

读者期待主角用非常规手段反击压迫或达成大目标。

## 断档风险

- 主角享受纯粹作恶。
- 杀人流血奸淫掳掠被当爽点。
- 没有普适价值观和带入感。
- 坏得低贱，缺少格局。
- 只像坏狗蛋，不像坏英雄。

## 训练任务

写一个反派主角桥段：违规手段、正当目标、不可触碰底线、读者可代入理由四项齐全。
