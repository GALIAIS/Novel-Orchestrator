# 废土升级开头-金手指盘活与生存奖励

依据案例：batch-361 案例 1805

## 读者承诺

废土升级开头要把下行处境、金手指亮点、快速反击和衣食住行改善绑在一起。苦难不是目的，改善生存才是爽点。

## 核心循环

```text
生存困境 -> 敌人压迫 -> 金手指显露 -> 快速反击 -> 食物/钱财/安全改善
```

## 开篇必备

- 废土困境关乎主角当前问题。
- 金手指足够早、足够新。
- 下行后尽快有上行。
- 衣食住行改善作为奖励。

## 期待来源

- 兄妹生存压力。
- 看似不可战胜的敌人。
- 一次性道具或能力限制。
- 资源改善带来的代入爽。

## 断档风险

- 只写苦难，第三章才上行。
- 金手指不清楚。
- 危机后修炼太久，情绪弦松掉。
- 角色太多分散焦点。
- 废土风土人情拖慢主线。

## 训练任务

写 1000 字废土开头：前 300 字威胁，前 600 字亮金手指，前 1000 字完成小反击并改善一个生存条件。
