# 推进类型-人物推动与情节推动

依据案例：`batch-227` 案例 1133。

## 读者承诺

新人先判断自己靠人物互动推进，还是靠固定主线推进，避免两边都想满配。

## 核心循环

```text
选择推进类型 -> 配置重点能力 -> 放弃次要复杂度 -> 稳定开篇
```

## 开篇必备

- 人物推动要有人设差异和互动冲突。
- 情节推动要有固定主线和阶段节点。
- 新人先写好一种。
- 情节推动文至少先把主角写好。

## 期待来源

人物推动给陪伴和互动期待；情节推动给目标、节奏和阶段推进。

## 断档风险

- 人物和情节都想写满，结果都弱。
- 群像难度过高。
- 人物行动有逻辑但不有趣。
- 情节推动没有清晰目标。

## 训练任务

为同一个创意写两版开篇：人物推动版写 3 个角色的人设差异和互动冲突；情节推动版写 1 个固定主线目标和 5 个阶段节点。
