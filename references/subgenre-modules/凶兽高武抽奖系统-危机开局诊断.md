# 凶兽高武抽奖系统-危机开局诊断

依据案例：`batch-175` 案例 874。

## 读者承诺

主角在凶兽高武危机中凭借独有金手指、理智判断和主动计划，从人为不公与怪物外压中翻盘变强。

## 核心循环

```text
高武危机 -> 人为不公 -> 金手指可见证据 -> 主角有所恃 -> 主动反制 -> 因果闭环 -> 变强期待
```

## 开篇必备

- 金手指影响剧情前先给可感知证据。
- 主角提出要求前，要写清依仗和第二手准备。
- 怪物行动、反派操作、爆炸结果都要有因果链。
- 反派动机先给最重要的一条，后文再扩展。
- 不公平伤害要拉出读者情绪。

## 期待来源

读者期待主角不是被环境推着走，而是看清局势、提前准备、抓住机会果断反制。高武危机的爽点来自“危险压迫 + 主角有脑子 + 独有金手指兑现”。

## 断档风险

- 金手指像突然想起，读者觉得作者临时加设定。
- 主角行动缺依仗，看起来鲁莽或开挂硬推。
- 危机因果缺环，读者替作者脑补。
- 反派动机塞太多人名，拖慢开头。
- 主角一直被环境驱使，缺少主角感。

## 训练任务

把危机开局前三章拆成“读者已知信息表”，检查每个反转、金手指使用、怪物反应和主角决策是否提前给过证据。
