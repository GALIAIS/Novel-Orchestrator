# 兼职写作-碎片构思与晚间成稿

依据案例：`batch-251` 案例 1255。

## 读者承诺

兼职写作模块把白天碎片构思和晚上集中成稿分离，降低日更启动成本。

## 核心循环

```text
白天构思 -> 手机记录 -> 晚上筛成熟点 -> 集中写正文 -> 次日接续
```

## 开篇必备

- 白天有固定记录入口。
- 每天至少记录 3 条剧情点。
- 晚上只写已成熟方案。
- 统计真实可用字数。

## 期待来源

兼职作者不用等整块灵感，白天先把剧情燃料准备好，晚上直接进入成稿。

## 断档风险

- 晚上坐下才开始想剧情。
- 用存几十万字逃避投稿验证。
- 只靠热情，不测真实作息。
- 记录太散，晚上无法转正文。

## 训练任务

连续 7 天执行“白天 3 条剧情笔记，晚上 1 小时成稿”，记录启动耗时、字数和卡点，得出兼职日更最低线。
