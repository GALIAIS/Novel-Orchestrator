# 大纲自审-角色背景剧情三轴

依据案例：`batch-048` 案例 238。

## 读者承诺

把“用心写大纲”变成可执行自检：角色看性格与成长，背景看动机与映射，剧情看爽点、悬念和钩子，三轴共同排查毒点。

## 核心循环

```text
大纲初稿 -> 角色自审 -> 背景自审 -> 剧情自审 -> 毒点定位 -> 修改动作
```

## 开篇必备

- 明确主角性格如何推动第一阶段事件。
- 明确主角何时小成长、何时大改变。
- 给配角安排独立性格和功能。
- 背景不要只说明，要能映射到具体剧情。
- 每章至少检查一个爽点、一个悬念或一个钩子。

## 期待来源

- 角色行为符合性格，读者才信。
- 背景能提供动机，世界才不空。
- 爽点密度足，章节才不断档。
- 钩子锋利，读者才愿意翻下一章。

## 断档风险

- 角色成长过早，会丢掉主角定位。
- 角色成长过晚，会让节奏单调。
- 配角只搬运信息，会显得工具人。
- 背景和剧情脱节，会变成设定展示。
- 悬念定义不清，会让读者不知道该期待什么。

## 训练任务

对一卷大纲做三轴检查：角色、背景、剧情各列 5 个问题，每个问题后面写一个具体修改动作。
