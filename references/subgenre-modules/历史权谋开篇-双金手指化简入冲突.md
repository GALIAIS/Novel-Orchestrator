# 历史权谋开篇-双金手指化简入冲突

依据案例：`batch-314` 案例 1566

## 读者承诺

历史权谋开篇可以有复杂背景和双金手指，但第一章必须让读者立刻明白死局、能力和冲突。复杂设定要压成几句话，先让主角入局反击。

## 核心循环

```text
身份死局 -> 金手指一句话说明 -> 敌方压迫 -> 主角亮出筹码 -> 第一轮权力反击
```

## 开篇必备

- 第一句给身份、悬念或生死压力。
- 金手指只讲当前能用的功能。
- 穿越前信息只保留会影响后续剧情的部分。
- 主角从弱到狠需要过渡。
- 皇权弱势反击要写清筹码。

## 期待来源

读者期待弱皇帝不是乱撞，而是凭未来信息、命运预警、身份筹码，在朝堂死局中撬开第一道口子。

## 断档风险

- 梦境、前世、书页、朝堂背景同时铺开。
- 主角转变过快，缺少心理桥。
- 反派行为太直，显得降智。
- 主角反击看不出依仗，像跳梁小丑。

## 训练任务

把双金手指开篇改成 500 字：死局、能力、筹码、反击四项必须全部出现。
