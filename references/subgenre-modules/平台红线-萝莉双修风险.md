# 平台红线-萝莉双修风险

依据案例：`batch-135` 案例 672。

## 读者承诺

有受众不等于能写。涉及未成年外形、性暗示、双修等元素时，先看平台红线和编辑口径，风险高于爽点收益。

## 核心循环

```text
敏感设定 -> 判断外形年龄 -> 判断性暗示 -> 查询平台口径 -> 放弃或改写
```

## 开篇必备

- 区分设定年龄和外形年龄。
- 识别性/双修暗示。
- 明确目标平台。
- 判断近期整治方向。
- 准备安全替代设定。

## 期待来源

- 强者反差可以保留。
- 关系张力可通过身份、实力、阵营制造。
- 安全设定让作品能长期连载。
- 红线规避减少封禁风险。

## 断档风险

- 用“实际年龄很大”规避萝莉外形问题。
- 把小众受众当成平台许可。
- 严查期硬碰双修题材。
- 不问编辑就拿整本书赌。

## 训练任务

把一个萝莉双修设定改成安全版：保留强者反差和关系张力，删除未成年外形、性暗示和审核敏感点。
