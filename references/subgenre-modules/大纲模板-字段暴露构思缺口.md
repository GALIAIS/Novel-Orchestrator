# 大纲模板-字段暴露构思缺口

依据案例：`batch-110` 案例 549。

## 读者承诺

大纲模板的作用不是替作者想剧情，而是用固定字段暴露构思缺口。哪里填不出来，哪里就需要继续拆书、查资料或补冲突。

## 核心循环

```text
套模板 -> 填字段 -> 暴露空白 -> 补素材/冲突 -> 重填大纲
```

## 开篇必备

- 先写核心梗。
- 再填主角目标和主要矛盾。
- 每卷必须有阶段目标。
- 空字段不能糊弄过去。
- 填完后再决定是否写正文。

## 期待来源

- 散乱灵感变得可检查。
- 缺口提前暴露。
- 主线和卷目标更清楚。
- 正文开写前减少崩盘。

## 断档风险

- 把模板当创意来源。
- 字段填得漂亮但没有冲突。
- 空字段跳过不管。
- 填完模板不回到章节节奏。

## 训练任务

用模板填一个新脑洞，专门列出所有填不出来的字段。每个空字段后写一个补齐动作。
