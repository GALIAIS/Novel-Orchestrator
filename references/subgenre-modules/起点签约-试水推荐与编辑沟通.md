# 起点签约-试水推荐与编辑沟通

依据案例：batch-356 案例 1777

## 读者承诺

签约后推荐异常要先沟通编辑、确认流程，再决定继续、上架吃全勤、完结重开或换账号。推荐和追读的顺序要理清，裸奔状态下很难谈有效追读。

## 核心循环

```text
签约无推荐 -> 问编辑 -> 确认试水推/追读/全勤路径 -> 评估反馈价值 -> 决定继续或重开
```

## 开篇必备

- 当前签约和字数状态。
- 是否经历试水推。
- 编辑明确回复。
- 下一本投递和笔名方案。

## 期待来源

- 推荐提供真实读者反馈。
- 编辑沟通降低误判。
- 完结重开保留学习节奏。
- 换账号或换编辑减少旧路径焦虑。

## 断档风险

- 不问编辑只猜平台。
- 裸奔数据当成追读判断。
- 为全勤继续单机太久。
- 把换笔名当成解决作品问题的核心。

## 训练任务

写一张签约后沟通清单：无推荐、追读低、编辑建议全勤、想换笔名、想重开五种情况分别该问什么。
