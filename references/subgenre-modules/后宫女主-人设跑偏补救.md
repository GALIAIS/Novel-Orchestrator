# 后宫女主-人设跑偏补救

依据案例：batch-292 案例 1457

## 读者承诺

后宫女主人设跑偏时，先判断偏差类型，再用外因、主角影响或新角色承接来补救。

## 核心循环

```text
发现跑偏 -> 判断偏差类型 -> 解释外因/主角影响 -> 后续行为校正 -> 稳定新状态
```

## 开篇必备

- 女主原承诺。
- 跑偏行为。
- 偏差类型。
- 是否同人角色。
- 修正理由。
- 后续两场证明戏。

## 期待来源

读者看到角色变化有因果，而不是作者忘了人设。

## 断档风险

- 不问偏法就乱补。
- 同人角色小细节偏差无解释。
- 主要女主随便废掉。
- 只口头解释，后续行为不稳定。

## 训练任务

写一张女主修正卡：原人设、跑偏点、外因解释、两场拉回人设的剧情。
