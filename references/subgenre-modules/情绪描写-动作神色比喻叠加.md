# 情绪描写-动作神色比喻叠加

依据案例：batch-265 案例 1325

## 读者承诺

在玄幻、仙侠、团队战、宗门战等高情绪场面中，把悲壮、绝望、孤注一掷拆成可见层次，而不是只用旁白宣布情绪。

## 核心循环

```text
绝境压力 -> 身体动作 -> 面部神色 -> 处境比喻 -> 短促台词 -> 群体/局势回应
```

## 开篇必备

- 先给人物一个无法轻易解决的绝境。
- 情绪爆发前必须有身体动作。
- 神态词要服务当下压力。
- 比喻要来自人物处境。
- 台词要短，放在铺垫之后。

## 期待来源

期待来自情绪层层加压后的爆点。读者看见人物站起、失态、低吼、求援，才会相信那句台词不是口号，而是被绝境挤出来的选择。

## 断档风险

- 只写台词，不铺情绪，像喊口号。
- 比喻脱离处境，显得堆辞藻。
- 神态描写泛泛，不能推进情绪。
- 为了“精简”删掉必要铺垫，导致大场面没重量。

## 训练任务

把一句“请救救我们”扩成 500 字。必须按“动作、表情、比喻、台词、旁人反应”五步写，最后删掉所有直接评价情绪的词。
