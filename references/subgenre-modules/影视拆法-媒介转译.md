# 影视拆法-媒介转译

依据案例：`batch-152` 案例 757。

## 读者承诺

电影动漫更直观，适合快速拆剧情手法；但小说不能照搬镜头、明星和导演光环，必须把手法功能转译成文字里的矛盾、收获和悬念。

## 核心循环

```text
影视片段 -> 看清手法功能 -> 提炼矛盾/收获/悬念 -> 识别不能照搬处 -> 小说化改写
```

## 开篇必备

- 一个影视片段。
- 片段制造的期待。
- 矛盾、收获、悬念三项功能。
- 影视表现方式。
- 小说文字转译方式。

## 期待来源

作者期待用影视降低拆解成本，更快看清剧情推进和情绪设计，再把功能搬到小说里。

## 断档风险

- 直接照搬镜头语言。
- 忽视明星/导演光环和大神作者光环的差异。
- 只说画面燃，不拆功能。
- 学影视开头花活，小说开篇却没读者承诺。

## 训练任务

拆电影开头 10 分钟，提炼 3 个机制。每个机制写出影视手法、小说可替代写法、适合放在第几章。
