# 分形结构-宏观套娃情绪优先

依据案例：`batch-066` 案例 330。

## 读者承诺

这个模块用于长篇分卷、三幕式、四幕式和单元结构设计。它承诺的是用分形思路帮助故事写长：卷和单元可以近似自相似并逐级升级；但拆到场景时，优先看情绪折线和叙事缓急，不强行套完整结构。

## 核心循环

```text
全书主题 -> 分卷三幕 -> 每卷升级 -> 单元复用 -> 场景情绪拉扯
```

## 开篇必备

- 全书主题先定。
- 每卷有建置、冲突、解决。
- 每卷解决的问题要升级。
- 单元结构可复用，但不能机械填表。
- 场景层优先看情绪变化。

## 期待来源

- 每卷都有阶段目标。
- 卷与卷之间问题升级。
- 结构相似让作者可控。
- 情绪折线让小场景好看。
- 主题统合所有卷。

## 断档风险

- 为了套娃而套娃。
- 每个场都强行三幕完整。
- 结构正确但情绪平。
- 只有宏观规划，没有小单元拉扯。
- 忽视主题统合。

## 训练任务

设计三卷长篇，每卷写建置、冲突、解决、升级点。再选一个场景，只画情绪折线，不填三幕表。
