# 冲突爆发-高潮兑现

依据案例：batch-270 案例 1347

## 读者承诺

让冲突爆发真正成为高潮：矛盾摊牌、主角选择、胜负转折和结果兑现连在一起。

## 核心循环

```text
矛盾积累 -> 导火索 -> 爆发 -> 主角选择 -> 结果兑现
```

## 开篇必备

- 爆发前要有矛盾积累。
- 爆发时主角必须做选择或出手。
- 高潮不只是吵架，要有胜负变化。
- 结果要当场或很快兑现。

## 期待来源

期待来自读者等了很久的矛盾终于摊牌，并看到主角把局面推向结果。

## 断档风险

- 冲突爆发只有争吵，没有结果。
- 前文铺垫不足，爆发没重量。
- 主角在高潮里旁观。
- 结果拖太久，情绪散掉。

## 训练任务

写一个冲突爆发段，必须包含导火索、主角选择、对手反应、结果兑现四项。
