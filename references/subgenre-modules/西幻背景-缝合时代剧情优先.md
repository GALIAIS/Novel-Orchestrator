# 西幻背景-缝合时代剧情优先

依据案例：`batch-062` 案例 309。

## 读者承诺

这个模块用于西幻、类西幻、蒸汽、魔法、异族、衍化同人等背景取舍。它承诺的是不把考据当卖点主体，而是用低成本架空或套皮方案，让设定服务剧情、人物压力和事件生产。

## 核心循环

```text
西幻兴趣 -> 市场风险确认 -> 背景缝合降成本 -> 三条规则进事件 -> 剧情先跑通 -> 细节再加分
```

## 开篇必备

- 先判断是否必须纯西幻。
- 只保留会立刻改变主角处境的规则。
- 用架空时代缝合替代完整历史复刻。
- 第一章要有普通读者能理解的目标和代价。
- 细节只服务事件，不做大段展示。

## 期待来源

- 陌生规则带来的处境变化。
- 魔法、阶层、异族、宗教、蒸汽等元素进入冲突。
- 主角用现代认知、信息差或特殊身份破局。
- 末世皮、都市往返、同人衍化降低进入门槛。
- 世界细节在事件中逐步露出。

## 断档风险

- 开篇堆时代考据。
- 设定很完整但剧情不动。
- 模仿维多利亚外壳却没有对应故事能力。
- 纯西幻小众风险被低估。
- 读者看不懂主角现在要什么。

## 训练任务

把西幻背景删到只剩三条会影响主角行动的规则，再写一个第一章事件。事件必须包含目标、阻碍、代价和一个设定带来的转折。
