# 开头试读-重复语感自检

依据案例：`batch-166` 案例 827。

## 读者承诺

通过公开试读反馈，把“有没有读下去欲望”和“具体哪里卡住”拆开，先修重复词、拗口句和语感阻滞。

## 核心循环

```text
开头试读 -> 读者续读判断 -> 重复词定位 -> 保留有效氛围 -> 朗读复查 -> 下一轮修改
```

## 开篇必备

- 先问读者是否愿意继续读。
- 收集第一个卡点。
- 圈出高频重复词和相近句式。
- 同时保留已经成立的氛围和文笔优点。

## 期待来源

作者期待开头能顺畅建立氛围，让读者继续读；读者期待文字不要在同一小段反复撞词。

## 断档风险

- 只听“好看/不好看”，没有可执行修改点。
- 全盘否定开头，删掉有效氛围。
- 只改错别字，不处理语感重复。
- 附件不可读时臆测正文问题。

## 训练任务

检查 500 字开头，统计重复词、重复形容和相近句式。删改后朗读，记录修改前后的续读欲望变化。
