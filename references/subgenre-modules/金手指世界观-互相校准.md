# 金手指世界观-互相校准

依据案例：batch-294 案例 1466

## 读者承诺

金手指和世界观要互相提供价值：能力有舞台，世界有适配冲突。

## 核心循环

```text
金手指 -> 使用场景 -> 世界压力 -> 能力解决问题 -> 世界规则再调整
```

## 开篇必备

- 核心金手指。
- 它最适合解决的问题。
- 世界观提供的压力。
- 第一次使用场景。
- 长期升级空间。

## 期待来源

读者看到能力在环境里持续有效，就会期待下一次更高阶使用。

## 断档风险

- 军事才能放进和平世界。
- 电脑天才穿到无用武之地。
- 世界观只是背景展示。
- 金手指只是标签，不制造事件。

## 训练任务

为一个金手指反推三种必须存在的世界压力，并删掉一个不适配设定。
