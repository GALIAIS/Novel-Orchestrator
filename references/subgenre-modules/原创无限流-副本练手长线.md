# 原创无限流-副本练手长线

依据案例：`batch-092` 案例 457。

## 读者承诺

原创无限流难在后期持续副本设计。新人先用小规模副本练手，参考成熟样本，再决定是否开长篇。

## 核心循环

```text
确定无限类型 -> 参考样本 -> 设计副本规则 -> 通关收益 -> 副本外成长 -> 长线串联
```

## 开篇必备

- 明确原创、诸天、同人或主神空间方向。
- 第一个副本规则清楚。
- 任务目标和失败代价明确。
- 通关收益能推动主线。
- 至少准备 3 个副本差异。

## 期待来源

- 新副本规则有什么变化。
- 主角如何破解规则。
- 通关收益如何升级。
- 副本外长线如何推进。

## 断档风险

- 只写开头，后期副本难以持续。
- 副本重复。
- 奖励和主线脱节。
- 原创世界规则失控。

## 训练任务

设计 3 个原创副本，每个写规则、任务、失败代价、通关收益和主线关系，再判断能否支撑 10 万字。
