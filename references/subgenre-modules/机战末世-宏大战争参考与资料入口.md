# 机战末世-宏大战争参考与资料入口

依据案例：`batch-251` 案例 1252。

## 读者承诺

机战末世模块把机甲、战争、外星异种、末世拆成多个资料维度，跨媒介取材后再网文化重组。

## 核心循环

```text
组合题材 -> 拆要素 -> 找小说/漫画/动画/游戏参考 -> 提取机制 -> 回到主角线
```

## 开篇必备

- 人类文明面临异种压迫。
- 机甲承担明确战场功能。
- 战争目标和损失可见。
- 主角有进入战场的短期任务。

## 期待来源

读者期待的是绝境中组织化抵抗、技术武装、牺牲代价和主角在大战中的成长位置。

## 断档风险

- 只堆机甲标签，没有战场功能。
- 宏大战争只有规模，没有目标和代价。
- 异种没有规则差异。
- 参考跨媒介后忘了转成网文节奏。

## 训练任务

为项目找 5 个参考源，每个只拆一个维度：机体限制、异种规则、战术调度、末世秩序、主角任务。
