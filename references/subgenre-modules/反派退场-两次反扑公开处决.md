# 反派退场-两次反扑公开处决

依据案例：batch-285 案例 1424

## 读者承诺

高仇恨反派退场要让读者看见他用尽后手仍逃不掉，最后被主角公开、确定、不可逆地处决。

## 核心循环

```text
仇恨累积 -> 读者催杀 -> 反派后手逃生 -> 主角压制 -> 公开处决 -> 新钩子补位
```

## 开篇必备

- 反派仇恨来源。
- 已交锋次数。
- 读者期待的惩罚。
- 两个逃生后手。
- 主角王炸和见证者。

## 期待来源

读者想看的不是反派简单死亡，而是他挣扎、反扑、似乎能跑，最后仍被主角按住。

## 断档风险

- 反派蹦跶太久。
- 主角每次赢但没有实质惩罚。
- 反派死后追读空缺。
- 处决不公开、不确定、不解气。

## 训练任务

写一个反派退场章纲：两次逃生失败，一次王炸处决，章末接新地图小冲突。
