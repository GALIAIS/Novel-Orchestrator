# 同人受众-规则适配

依据案例：`batch-031` 案例 153。

## 读者承诺

同人文的段落、梗和行文要先适配平台与原作受众，不要硬套通用网文规则。

## 核心循环

```text
平台受众 -> 原作信仰 -> 梗密度 -> 行文习惯 -> 故事流畅
```

## 开篇必备

- 目标平台。
- 原作受众是否强。
- 读者是否熟悉角色和梗。
- 行文规则是否来自同一受众。

## 期待来源

- 原作角色和世界观自带吸引力。
- 梗和口吻让读者确认“懂原作”。
- 大段或高密度内容被特定受众接受。
- 故事能顺着原作期待走。

## 断档风险

- 外行硬拆同人文笔。
- 把通用段落规则当铁律。
- 只学形式，不懂原作梗。
- 平台受众不一致却照搬。

## 训练任务

拆一本同人文，只记录平台、原作梗、受众习惯、段落密度和你不能照搬的原因。
