# 一级结构-情绪阐发

依据案例：`batch-004` 案例 016。

## 读者承诺

在起承转合之外，进一步解释单元故事为什么能产生情绪，以及这个单元背后的核心梗能否换壳复用。

## 核心循环

```text
表层事件 -> 情绪缺口 -> 主角释放 -> 核心梗抽象 -> 换壳复用
```

## 开篇必备

- 当前单元的情绪目标。
- 读者期待的兑现画面。
- 主角和读者共情的进入点。
- 可抽象的核心梗。

## 期待来源

- 读者知道自己在等什么情绪被释放。
- 事件不是随机发生，而是服务一个可复用情绪链。
- 每次换壳都保留同一核心满足。

## 断档风险

- 只有事件，没有情绪目标。
- 只会套起承转合，不知道爽点来自哪里。
- 抽象过度，核心梗脱离具体矛盾。

## 训练任务

给 3 个已写单元补“情绪目标”和“核心梗抽象”。写不出来的单元先回炉。
