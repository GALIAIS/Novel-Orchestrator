# 空间素材-全景家居与场景卡

依据案例：`batch-255` 案例 1271。

## 读者承诺

空间素材模块把“想不起来宅邸豪宅怎么写”转成可搜索、可复用、能触发剧情的场景卡。

## 核心循环

```text
空间画面缺失 -> 找短视频/全景/书单 -> 提取房间家具 -> 设计行动路线 -> 写成场景戏
```

## 开篇必备

- 先确定古代宅邸还是现代豪宅。
- 用视频和全景解决方位感。
- 用资料书单补器物名称。
- 每个物件都要服务身份、行动或冲突。

## 期待来源

读者能在可信空间里看见人物移动、观察、藏匿、对峙，场景因此有沉浸感。

## 断档风险

- 只收藏素材，不转成自己的场景卡。
- 只堆陈设名词，没有人物行动。
- 空间方位混乱。
- 古代和现代物件风格混搭。

## 训练任务

为一个宅邸或豪宅做场景卡：10 个物件名、5 个空间名、3 条行动路线、2 个由空间触发的剧情事件。
