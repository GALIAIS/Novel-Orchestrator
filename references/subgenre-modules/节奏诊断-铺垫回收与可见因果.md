# 节奏诊断-铺垫回收与可见因果

依据案例：`batch-394` 案例 1968

## 读者承诺

把“开局慢”诊断为材料与当前事件的连接问题：每段信息都让人物判断、行动或局面发生可见变化，读者始终知道为什么现在要看它。

## 核心循环

```text
单一当前事件 -> 材料连接压力信息或选择 -> 主角行动 -> 局面变化 -> 铺垫短程回收 -> 推出下一步
```

## 开篇必备

- 开局只有一个可复述的当前事件。
- 背景解释眼前选择，不独立展开前史。
- 设定改变当前解法、风险或收益。
- 伏笔在当下形成问题。
- 关键铺垫有明确短程回收位置。
- 本章结束时人物或局面状态已改变。

## 期待来源

读者不断看见前文如何影响下一步，因而相信新信息会被使用，并期待主角的选择怎样改变局面。

## 断档风险

- 用节点提前和短句替代因果修复。
- 作者知道后续用途，读者却看不见连接。
- 每段都精彩却服务不同事件。
- 伏笔只储存秘密，不形成当前问题。
- 只能解释“以后有用”的材料占据开头。
- 删除段落后当前事件完全不受影响。

## 训练任务

为慢开头逐段标注“当前事件—本段作用—造成变化—回收位置”，移出只能服务远期的段落；让陌生读者每 500 字复述一次当前问题，确保始终沿一条因果线推进。
