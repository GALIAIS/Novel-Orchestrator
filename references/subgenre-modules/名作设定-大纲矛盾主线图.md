# 名作设定-大纲矛盾主线图

依据案例：`batch-157` 案例 781。

## 读者承诺

用成熟名作的复杂设定，反推大纲、矛盾和主线如何成立，并通过图表把关系讲清。

## 核心循环

```text
名作设定 -> 矛盾关系 -> 主线作用 -> 图表化 -> 方法迁移
```

## 开篇必备

- 样本作品。
- 核心设定。
- 阵营和利益冲突。
- 主角目标。
- 主线推进作用。
- 可视化字段。

## 期待来源

读者期待从熟悉名作中看到可复制的结构，而不是只听剧情赏析。

## 断档风险

- 只复述剧情，不提炼结构。
- 题目太大却没有图表。
- 设定和矛盾、主线脱节。
- 名作案例无法迁移到自己的作品。

## 训练任务

选一部长篇作品，做一张三列表：设定元素、触发的矛盾、推动的主线作用。
