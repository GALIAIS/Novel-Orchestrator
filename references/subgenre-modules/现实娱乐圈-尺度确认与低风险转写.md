# 现实娱乐圈-尺度确认与低风险转写

依据案例：batch-369 案例 1844

## 读者承诺

现实娱乐圈涉及潜规则、幕后故事和真实影射时，先问编辑确认平台尺度，再把敏感点转成低风险事业线或架空表达。

## 核心循环

```text
现实素材 -> 敏感点识别 -> 编辑确认 -> 架空/轻量化 -> 可发卖点
```

## 开篇必备

- 明确平台。
- 明确敏感内容。
- 先确认尺度。
- 保留低风险爽点。

## 期待来源

- 事业升级。
- 资源博弈。
- 作品爆红。
- 圈内规则但不直贴黑料。

## 断档风险

- 熟悉素材就直接写。
- 潜规则写成主卖点。
- 不问编辑就开书。
- 现实影射引发审核和争议。

## 训练任务

把一个潜规则情节改成现实短篇版和起点长篇事业线版，比较风险收益。
