# 萌新卖座-扫榜噱头与极致收获

依据案例：batch-370 案例 1848

## 读者承诺

新人想找简单卖座路径，先扫目标网站新书榜，抓第一句话噱头和高频收获感，而不是凭空找捷径。

## 核心循环

```text
目标平台 -> 新书榜样本 -> 第一眼噱头 -> 极致收获 -> 持续验证
```

## 开篇必备

- 平台样本。
- 第一句话噱头。
- 低门槛金手指。
- 高频奖励反馈。

## 期待来源

- 开头新奇。
- 情绪先被调动。
- 签到/打卡等奖励。
- 每章都有收获。

## 断档风险

- 以为简单等于容易。
- 不扫榜空想方向。
- 有金手指但没笔力。
- 照抄题材不拆机制。

## 训练任务

扫 20 本目标平台新书，记录第一句话噱头和前三章收获感，提炼 5 个最低成本机制。
