# 世界观关系图-伏笔冲突可视化

依据案例：`batch-171` 案例 853。

## 读者承诺

把复杂世界观从文字堆转成节点和箭头，让伏笔、冲突、来源、敌对和演化关系一眼可查，减少长篇设定遗忘。

## 核心循环

```text
设定节点 -> 箭头关系 -> 伏笔位置 -> 冲突入口 -> 写作调用
```

## 开篇必备

- 先区分宇宙规则、势力阵营、角色关系三层。
- 每条箭头要标明关系类型。
- 每个关键节点至少绑定一个冲突或伏笔。
- 图只承载关系，大段解释放大纲或设定文档。

## 期待来源

读者期待世界观有因果和层次。作者用关系图维护好节点后，正文里的冲突会更自洽，伏笔也更容易回收。

## 断档风险

- 只画名词，没有关系类型。
- 图越来越大却不分层。
- 伏笔没有绑定节点关系。
- 画完不用，正文仍然写散。

## 训练任务

把一个世界观画成三层图，每层至少 5 个节点、8 条箭头。每条箭头写明关系类型和一个可写冲突。
