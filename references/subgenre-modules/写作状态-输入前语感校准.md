# 写作状态-输入前语感校准

依据案例：batch-278 案例 1387

## 读者承诺

用短时同类样本阅读校准语感，再立刻写自己的正文；同时管理本章说、断章、命名和写前输入，避免语言串味和自嗨。

## 核心循环

```text
明确题材氛围 -> 短读同类样本 -> 立刻写正文 -> 检查串味/节奏 -> 沉淀可复用表达
```

## 开篇必备

- 写前只读少量同类样本。
- 不长时间沉入参考作品世界观。
- 写前避开酒精和不匹配内容输入。
- 明确本章互动点和断章承接。

## 期待来源

读者读到的是稳定、贴合题材的语言节奏，而不是作者刚被别的内容带偏后的情绪残留。

## 断档风险

- 参考读太多，世界观被带跑。
- 看日漫后写非二次元，语气词和表达串味。
- 饮酒写作，情绪阈值失真。
- 断章反转没有下一章内容承接。
- 本章说互动失控成争吵。

## 训练任务

连续 3 次写作前各读 5000 字同类型文本，立刻写 1000 字正文。每次记录一个可保留句式、一个需要删掉的串味表达。
