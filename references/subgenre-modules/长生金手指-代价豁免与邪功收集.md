# 长生金手指-代价豁免与邪功收集

依据案例：batch-352 案例 1759

## 读者承诺

长生类金手指不只承诺“活得久”，更强的写法是豁免世界里高收益资源的代价。折寿魔功、残缺功法、速成邪法原本没人敢碰，主角因为长生和保持巅峰，能把这些有毒资源变成专属升级路线。

## 核心循环

```text
表层不老 -> 发现折寿代价可豁免 -> 收集危险功法 -> 快速获得收益 -> 更高代价资源继续出现
```

## 开篇必备

- 金手指表层限制，保留死亡压力。
- 一个世界公认的高收益高代价资源。
- 主角身份能接触这类资源。
- 第一次揭示“豁免代价”的实证场景。

## 期待来源

- 鸡肋能力突然变成关键钥匙。
- 低价值邪功/魔功被重新估价。
- 职业场景持续供给资源。
- 后续不断出现更危险、更高收益的功法。

## 断档风险

- 长生写成纯苟，没有即时收益。
- 豁免代价过强，死亡压力消失。
- 资源来源不稳定，收集线断掉。
- 新人只写设定，不写细微情绪和收益反馈。

## 训练任务

列 8 个“高收益但有代价”的修行资源，再给主角设计一个只能豁免其中一种代价的金手指，写出第一章如何证明它有效。
