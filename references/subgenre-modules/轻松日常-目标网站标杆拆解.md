# 轻松日常-目标网站标杆拆解

依据案例：`batch-173` 案例 864。

## 读者承诺

轻松日常没有通用口号，必须先确定目标网站，再拆该网站标杆作品的结构、主题和感情线。

## 核心循环

```text
个人偏好 -> 目标网站 -> 标杆作品 -> 拆结构/主题/感情线 -> 开书
```

## 开篇必备

- 先确定平台读者吃哪种轻松日常。
- 拆标杆的章节结构，不只看文风。
- 记录关系推进和低伤害冲突。
- 确认自己想写的轻松感与平台匹配。

## 期待来源

读者期待稳定陪伴、关系推进、小事件和轻松情绪，而不是完全没有冲突。

## 断档风险

- 只凭个人喜欢开书。
- 轻松写成无事发生。
- 不拆目标平台，误判读者口味。
- 只模仿语气，不学结构。

## 训练任务

在目标平台找 3 本轻松日常标杆，各拆前 10 章，记录每章事件、关系推进、轻松点、冲突强度和章末期待。
