# 敏感行为-未成年烟酒弱化处理

依据案例：`batch-213` 案例 1065。

## 读者承诺

涉及未成年烟酒等敏感行为时，不强调年龄绑定，不美化行为，必要时用替代表现。

## 核心循环

```text
敏感行为 -> 必要性判断 -> 弱化标签 -> 不美化 -> 后果/替代
```

## 开篇必备

- 判断情节是否必须保留。
- 不把未成年身份和烟酒行为反复绑定。
- 不写成酷点或卖点。
- 能替换就用更安全的压力表现。

## 期待来源

作者既能表达人物压力或环境，又降低审核与读者反感风险。

## 断档风险

- 明确强调未成年抽烟喝酒。
- 把敏感行为写成魅力展示。
- 非必要还保留。
- 忽视平台尺度变化。

## 训练任务

把“未成年主角抽烟耍酷”改成删除版、替代表现版、保留但写后果版，并标注风险。
