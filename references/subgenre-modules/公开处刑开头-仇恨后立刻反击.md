# 公开处刑开头-仇恨后立刻反击

依据案例：batch-279 案例 1391

## 读者承诺

开头仇恨已经拉起时，读者要的是主角立刻反击和反差爽，不是回忆解释。前史最多点到为止，当前冲突必须优先。

## 核心循环

```text
当前欺压 -> 仇恨积累 -> 主角爆发 -> 敌人惊恐 -> 爽点兑现
```

## 开篇必备

- 当前不公平或欺压。
- 明确可恨对象。
- 主角反击动作。
- 反击后敌人的恐惧或失控。

## 期待来源

读者看到仇恨后，会自然期待主角用行动回应。爽点来自压迫后的立刻兑现。

## 断档风险

- 用回忆抢当前冲突。
- 仇恨拉起后继续解释人物过去。
- 相同欺压场面重复，没有动作升级。
- 反击太轻，辜负读者期待。

## 训练任务

删掉一段开头回忆，只保留 3 句背景。用 1000 字写完一次当前欺压和主角反击，重点写敌人惊恐。
