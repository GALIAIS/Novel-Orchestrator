# 番茄稿费-题材完读与单价错觉

依据案例：`batch-343` 案例 1714

## 读者承诺

番茄稿费不能用固定在读单价粗算，题材、完读、留存、字数阶段都会造成收入差异和单价错觉。

## 核心循环

```text
在读数字 -> 题材粘性 -> 完读留存 -> 字数阶段 -> 后台收入 -> 复盘选题
```

## 开篇必备

- 不用传闻单价做决策。
- 同时看流量上限和留存。
- 区分热门题材和小众题材。
- 关注完读率。
- 用真实后台校准。

## 期待来源

作者期待在读能换收入，但平台收益更依赖读者质量和题材表现。

## 断档风险

- 一万在读等于固定收入。
- 只追高单价题材。
- 忽略小众题材流量上限。
- 忽略热门题材留存问题。
- 用 UP 传闻替代后台数据。

## 训练任务

选三个题材，分别写流量上限、留存可能、单价传闻和最大风险，再决定是否值得开。
