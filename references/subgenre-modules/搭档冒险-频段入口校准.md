# 搭档冒险-频段入口校准

依据案例：batch-283 案例 1411

## 读者承诺

搭档冒险不能只靠“双主角关系”和设定元素开局，必须先让目标频段读者识别自己能获得的主承诺：男频看上行爽点，女频看伤害、复仇、情感助力。

## 核心循环

```text
确定频段 -> 选大火标签 -> 主视角承诺前置 -> 搭档关系服务爽点/情感 -> 持续兑现
```

## 开篇必备

- 目标平台和频段。
- 主视角是谁。
- 第一章的不公平伤害或强冲突。
- 主角优势和主线目标。
- 搭档关系对主线的具体功能。

## 期待来源

读者不是为“师姐弟搭档”这个概念留下，而是为明确的复仇、升级、冒险、恋爱助力或身份反差留下。

## 断档风险

- 男女频都想要，导致两边都不明确。
- 女主重生但没有前世伤害和复仇目标。
- 男主天才但没有上行爽点。
- 搭档关系只是人设装饰，不推动主线。

## 训练任务

把同一对搭档分别写成男频开篇和女频开篇，各 300 字。检查哪一版的目标读者、主线目标和情绪承诺更清楚。
