# 编辑偏好-成绩更新敏感避雷

依据案例：`batch-108` 案例 539。

## 读者承诺

想获得编辑重视，不靠讨好，而靠作品能赚钱、更新稳定且少制造审核麻烦。作者能控制的是作品质量、更新量、读者反馈和敏感内容排查。

## 核心循环

```text
作品可卖 -> 稳定更新 -> 读者反馈变好 -> 编辑更愿意投入资源 -> 作者继续稳定交付
```

## 开篇必备

- 先看作品是否有读者。
- 明确自己的日更能力。
- 不承诺做不到的更新量。
- 开书前排查敏感内容。
- 和编辑沟通时讲作品指标，不讲讨好话术。

## 期待来源

- 成绩提升带来自然关注。
- 更新稳定降低编辑风险。
- 少审核麻烦让合作成本更低。
- 作者把焦虑转向可控指标。

## 断档风险

- 把编辑当成需要讨好的人脉。
- 作品没成绩却期待特殊照顾。
- 拍胸脯承诺高更新但做不到。
- 敏感内容反复给编辑添负担。

## 训练任务

写一张编辑沟通前自检表：当前成绩、日更能力、存稿、敏感内容、下一周更新计划。只写自己能兑现的项。
