# 末世女性角色-人设驱动装逼反馈

依据案例：`batch-196` 案例 979。

## 读者承诺

末世女角色被主角救后，可以提供装逼反馈和关系期待，但反应必须符合人设，不必无脑白给。

## 核心循环

```text
主角救场 -> 女角色按人设反应 -> 承认主角强 -> 保留性格阻力 -> 关系自然推进
```

## 开篇必备

- 女角色救场前有基本能力或性格。
- 主角救场动作足够强。
- 女角色反馈服务主角爽点。
- 暧昧节奏服从人设。

## 期待来源

读者既看到主角被承认，又相信女角色不是降智工具人，从而期待后续互动。

## 断档风险

- 火书常见桥段照抄。
- 救一次立刻白给。
- 人设和反应冲突。
- 暧昧太早压过末世危机。

## 训练任务

写一个末世救场桥段，让女强人承认主角强，但保留她的判断力、戒备和互动阻力。
