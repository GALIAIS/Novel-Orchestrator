# 菠萝包百合-调情欺负爽点

依据案例：batch-278 案例 1389

## 读者承诺

把百合文中的“欺负”理解为轻度调情和角色互动爽点，通过病娇、社恐、傲娇等鲜明属性制造贴贴、拉扯和关系推进。

## 核心循环

```text
角色属性 -> 轻欺负/调情 -> 可爱反应 -> 暧昧升级 -> 日常关系推进
```

## 开篇必备

- 主视角女主有明确可反应属性。
- 女二或后宫角色有鲜明互动属性。
- “欺负”不能造成真实沉重伤害。
- 每次互动都推进关系距离。

## 期待来源

读者追的是角色魅力、暧昧反馈和贴贴拉扯，不一定是打怪装逼。女主被“欺负”后的反应本身就是甜点。

## 断档风险

- 把调情写成真羞辱。
- 用男频打脸装逼节奏套百合日常。
- 角色属性不鲜明，互动无燃料。
- 收藏虚高被误读成所有套路都有效。

## 训练任务

写一个 1000 字百合日常段：女二轻欺负女主，女主炸毛或嘴硬，结尾两人关系更近。标出调情边界，避免真实伤害。
