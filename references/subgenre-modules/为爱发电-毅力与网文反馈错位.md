# 为爱发电-毅力与网文反馈错位

依据案例：`batch-237` 案例 1182。

## 读者承诺

坚持写很多年不等于适配网文读者；为爱发电可以自洽，想要网文成绩就必须学习市场和反馈。

## 核心循环

```text
长期坚持 -> 成绩不动 -> 分辨爱好/商业目标 -> 学习读者反馈 -> 调整写法
```

## 开篇必备

- 明确写作目标。
- 区分表达自我和争取读者。
- 找到可学习的榜样书。
- 建立反馈复盘。

## 期待来源

读者看到作者不只是堆年限和字数，而是在每轮创作里修正题材、节奏、爽点和表达。

## 断档风险

- 把毅力当成质量证明。
- 不认可网文套路却期待网文成绩。
- 长期无反馈仍不复盘。
- 用“为爱发电”回避作品适配问题。

## 训练任务

把自己的写作目标分成“爱好表达”和“网文成绩”两栏；凡是写在成绩栏的目标，都必须配一个榜样书和一个可观察反馈指标。
