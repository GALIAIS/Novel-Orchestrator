# 卷间承接-二八遗留多线升级

依据案例：`batch-052` 案例 259。

## 读者承诺

大高潮后不让读者期待清空：解决上一卷 80%，留下 20% 引出下一卷，用目标管理控制战力，用多升级线和 7:3 主支线配比延续追读。

## 核心循环

```text
卷末高潮 -> 兑现主要期待 -> 遗留问题 -> 下一卷新冲突 -> 多线升级 -> 新期待建立
```

## 开篇必备

- 上一卷结尾不要把所有伏笔一次清空。
- 奖励够用即可，别让战力崩。
- 下一卷提纲要提前存在。
- 遗留问题能自然引出新敌人或新压力。
- 升级线不只本体战力，还包括势力、经济、装备、情感。

## 期待来源

- 读者满足上一卷高潮后，还能看到新问题。
- “打了小的来了老的”简单有效。
- 多线升级降低打怪爆宝疲劳。
- 不同风格支线能补充期待和爽点。

## 断档风险

- 新剧情像新单元，会产生陌生感。
- 奖励太强，后文只能反派降智。
- 伏笔太多不埋，会反噬信任。
- 单一升级循环到中期会疲劳。

## 训练任务

为一个卷末设计承接表：80% 已解决、20% 遗留、新冲突、战力上限、三条升级线、一个支线爽点。
