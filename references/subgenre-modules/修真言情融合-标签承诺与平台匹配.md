# 修真言情融合-标签承诺与平台匹配

依据案例：`batch-214` 案例 1070。

## 读者承诺

现代修真加日常言情时，要么把日常修真化，要么按言情画风找平台。

## 核心循环

```text
标签承诺 -> 正文体验 -> 类型错位检查 -> 同类扫榜 -> 平台选择
```

## 开篇必备

- 明确核心阅读体验。
- 修真元素改造恋爱日常事件。
- 平台选择看同画风样本成绩。
- 不只靠标签包装。

## 期待来源

读者点进现代修真能看到修真规则参与恋爱日常，而不是只有普通恋爱披皮。

## 断档风险

- 标签写修真，正文全是普通言情。
- 平台凭印象选。
- 修真元素只当背景板。
- 日常桥段没有题材化。

## 训练任务

把约会、吵架、逛街、帮朋友、冷战 5 个恋爱日常桥段各写普通版和修真版，并扫 3 个平台同类样本。
