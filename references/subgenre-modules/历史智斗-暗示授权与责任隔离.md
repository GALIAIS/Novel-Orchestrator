# 历史智斗-暗示授权与责任隔离

依据案例：`batch-396` 案例 1979

## 读者承诺

让读者看懂君主真实意图，却让局中人只收到可否认信号；代理人按自身利益执行并产生偏差，主角在公开面具、幕后控制和责任风险间博弈。

## 核心循环

```text
隐秘目标 -> 模糊信号授权 -> 代理人利益化执行 -> 对手误判责任中心 -> 主角公开切割私下善后 -> 证据累积逼近真实权力
```

## 开篇必备

- 主角有不能公开表达的目标。
- 暗示通过制度和信息链传递。
- 代理人有自己的收益和风险。
- 读者比局中人多知道一层意图。
- 第一次执行产生可见偏差。
- 公开面具服务具体战略。

## 期待来源

读者期待代理人会怎样理解一句暗示、对手会把责任追向哪里，以及主角的否认空间何时被证据耗尽、被迫公开站队。

## 断档风险

- 所有人都完美理解主角暗示。
- 代理人没有独立利益，只是遥控工具。
- 甩锅不依赖制度与证据。
- 对手永远不会从结果中学习。
- 表面昏聩让主角长期没有行动。
- 多轮成功不增加暴露代价。

## 训练任务

写一场三方朝堂局：主角只能用一句模糊问话授权，代理人多做一步，对手追错责任派系；结尾留下一项能反推主角的证据，并在下一场迫使他付出善后代价。
