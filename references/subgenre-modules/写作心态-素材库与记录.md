# 写作心态-素材库与记录

依据案例：batch-297 案例 1481

## 读者承诺

把写作卡顿从“我没天赋”改成可命名、可记录、可防范的工作流问题。

## 核心循环

```text
发现卡点 -> 命名心态问题 -> 记录素材/技巧 -> 做最低写作动作 -> 留下明天接续点
```

## 开篇必备

- 区分自嗨和教条两种极端。
- 建立素材库和技巧记录。
- 每天保留最低写作动作。
- 中断前写下下一步。

## 期待来源

- 作者知道自己为什么卡。
- 素材库降低正文启动成本。
- 复盘减少无序进步。

## 断档风险

- 把正文困难直接归因天赋。
- 只看技巧不写正文。
- 写累就中断，没有接续点。

## 训练任务

每天写一张四行卡：今日卡点、可用素材、可用技巧、明天第一句。
