# 虚晃一枪-不出手到被迫打脸

依据案例：`batch-240` 案例 1197。

## 读者承诺

打脸桥段要先制造“主角暂时不会出手”的预期，再用连续挑衅把主角推到必须爆发。

## 核心循环

```text
主角有能力 -> 暂不出手 -> 反派误判 -> 挑衅升级 -> 主角打脸
```

## 开篇必备

- 主角不出手原因。
- 反派误判点。
- 连续挑衅。
- 最后一根稻草。

## 期待来源

读者先被压住期待，再看到主角被迫出手，爽点弹性更强。

## 断档风险

- 一上来就直接打脸。
- 没有明显虚晃一枪。
- 主角不出手理由太弱。
- 爽点只靠数值评分，不靠人物关系。

## 训练任务

给一个打脸剧情补三句：主角为什么忍、反派为什么误判、什么触发主角出手。
