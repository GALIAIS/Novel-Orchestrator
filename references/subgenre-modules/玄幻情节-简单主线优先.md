# 玄幻情节-简单主线优先

依据案例：batch-296 案例 1478

## 读者承诺

玄幻新人先保证目标、敌人、升级、收益清楚，再少量加入因果嵌套。

## 核心循环

```text
明确目标 -> 明确阻碍 -> 主角升级 -> 击败阻碍 -> 获得收益 -> 下一目标
```

## 开篇必备

- 主角现在要什么。
- 谁挡住主角。
- 主角缺什么能力或资源。
- 打赢后有什么可见收益。

## 期待来源

- 读者持续知道自己在等什么。
- 升级和战斗兑现直接。
- 少量隐藏因果提供后续钩子。

## 断档风险

- 因果网过早压过主线。
- 作者觉得复杂，读者只觉得费脑。
- 隐藏信息太多，当前爽点不清楚。

## 训练任务

把一个复杂玄幻设定压成一章：目标、敌人、升级、收益各一句，再补一个隐藏因果钩子。
