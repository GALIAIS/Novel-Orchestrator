# 长篇拆书-卷级训练路径

依据案例：`batch-156` 案例 778。

## 读者承诺

拆书训练不能只拆几章，至少要拆到卷级结构，才能反哺自己的大纲和第一卷写作。

## 核心循环

```text
选择适合样本 -> 拆完整卷/多卷 -> 横向扫榜 -> 补自己大纲 -> 回看理论 -> 动笔验证
```

## 开篇必备

- 样本选择理由。
- 已拆章节和卷数。
- 拆出的剧情方向。
- 横向对比样本。
- 补进自己大纲的内容。
- 动笔验证节点。

## 期待来源

期待来自长期基本功：从高成绩样本中拆出卷级推进，再落到自己的第一卷。

## 断档风险

- 只拆开头，误以为理解整本。
- 表格做完但不回流大纲。
- 只拆一本，不横向对比。
- 拆书替代动笔。

## 训练任务

拆完一本高成绩书第一卷，再横向拆 3 本同题材第一卷，把共性整理成自己的第一卷大纲。
