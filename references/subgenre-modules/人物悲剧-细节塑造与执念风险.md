# 人物悲剧-细节塑造与执念风险

依据案例：`batch-336` 案例 1680

## 读者承诺

情景细节能把人物写活，但悲剧必须服务成长、主题或系列承诺，不能只服务作者执念。

## 核心循环

```text
情景细节 -> 人物变活 -> 悲剧预兆 -> 死亡/分离爆发 -> 成长或主题收益验证
```

## 开篇必备

- 用情景展示人物，不直接说明。
- 细节能让读者得出性格判断。
- 悲剧前有足够铺垫。
- 悲剧后有成长或主题收益。
- 检查是否破坏读者承诺。

## 期待来源

读者期待人物痛得难忘，但也要痛得值得，不能只是被作者强按进 badend。

## 断档风险

- 作者执念压过故事。
- 角色死亡不推动成长。
- 悲剧只为衬托悲剧色彩。
- 单卷悲剧破坏系列后续。
- 旧时代虐法不适配当下读者。

## 训练任务

写一个人物场景，不直接写性格，只靠三个细节让人物变活；若安排悲剧，列明不可替代收益。
