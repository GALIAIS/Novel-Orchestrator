# 三幕式副本-危机变形后怕收尾

依据案例：batch-270 案例 1350

## 读者承诺

把三幕式落到冒险副本节点：误判、危机、应对、升级、环境解法和后怕收尾连续变化。

## 核心循环

```text
进入副本 -> 误判危险 -> 真危机出现 -> 多次应对 -> 危机强化 -> 环境解法 -> 后怕收尾
```

## 开篇必备

- 入口要让角色进入未知环境。
- 第一次危险最好带误判。
- 危机至少变形两次。
- 最终解法来自环境或前文道具。

## 期待来源

期待来自敌人一次次超出预期。每次以为逃掉，又发现危险换形态逼近。

## 断档风险

- 三幕式只停留在标签。
- 危机一次出现一次解决。
- 最终解法凭空出现。
- 没有后怕镜头，敌人压迫感不足。

## 训练任务

设计一个 8 节点小副本：入口、误判、真危机、应对1、升级、应对2、最终解法、后怕收尾。
