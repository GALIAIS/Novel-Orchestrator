# 五折线格斗-后门虚晃与慌乱弱化

依据案例：`batch-247` 案例 1235。

## 读者承诺

校园格斗五折线可以用逃生方案落空做虚晃，但主角慌乱要控制，不能削弱能力期待。

## 核心循环

```text
朋友危机 -> 后门方案 -> 锁死落空 -> 翻墙上行 -> 埋伏再压 -> 招式脱身
```

## 开篇必备

- 朋友危机明确。
- 主角给出方案。
- 方案合理落空。
- 能力最终证明主线价值。

## 期待来源

读者看到逃法失败后，会期待主角正在学习的能力真正派上用场。

## 断档风险

- 虚晃落空太随意。
- 主角慌乱过重。
- 对手没有准备。
- 脱身不强化主线动力。

## 训练任务

写一个校园格斗五折线，先让逃生方案落空，再用主角正在学习的能力脱身。
