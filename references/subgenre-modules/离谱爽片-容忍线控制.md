# 离谱爽片-容忍线控制

依据案例：`batch-049` 案例 242。

## 读者承诺

用强情绪和强期待拉住读者，让夸张桥段卡在“离谱但能接受”的线上；只修会打断情绪的漏洞，不为每个细节牺牲节奏。

## 核心循环

```text
明确情绪 -> 放大场面 -> 暴露小漏洞 -> 情绪压过推敲 -> 期待继续前进
```

## 开篇必备

- 先确定这一段要燃、爽、感动还是解气。
- 离谱动作必须服务目标情绪。
- 给读者一个最低限度的接受理由。
- 下一步期待要立刻跟上。
- 只补关键逻辑，不补无关枝节。

## 期待来源

- 夸张场面带来视觉化爽感。
- 情绪足够强，读者愿意暂时放过小 bug。
- 节奏快，读者会追下一次升级。
- 容忍线掌握得好，离谱会变成风格。

## 断档风险

- 越过容忍线，爽点会变成毒点。
- 情绪不强时，漏洞会被放大。
- 全部合理化会拖慢写作。
- 完全不补关键逻辑，会显得作者糊弄。

## 训练任务

写一个离谱爽点，列出读者可能吐槽的 3 个点；只补最影响情绪的 1 个，并写出下一步期待。
