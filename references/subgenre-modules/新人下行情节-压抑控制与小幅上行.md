# 新人下行情节-压抑控制与小幅上行

依据案例：`batch-209` 案例 1043。

## 读者承诺

下行情节可以压主角，但要给读者看见翻盘依据和阶段性回拉。

## 核心循环

```text
压力下行 -> 底牌可见 -> 小幅上行 -> 再压迫 -> 爽点兑现
```

## 开篇必备

- 标出读者会憋屈的点。
- 提前露出金手指或翻转伏笔。
- 大趋势下行中安排小胜。
- 控制压抑持续字数。

## 期待来源

读者愿意忍受压力，是因为相信压抑正在为反转和爽点蓄力。

## 断档风险

- 下行写成连续受虐。
- 作者自嗨，读者只剩憋屈。
- 没有可见底牌。
- 没有小幅上行回拉。

## 训练任务

做一张 2000 字下行情绪表，每 500 字标一次读者情绪，并安排金手指可见、翻转伏笔和小幅上行。
