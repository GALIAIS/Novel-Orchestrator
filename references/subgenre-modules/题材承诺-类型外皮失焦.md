# 题材承诺-类型外皮失焦

依据案例：`batch-166` 案例 826。

## 读者承诺

强题材标签不仅负责吸引读者，还必须持续参与世界压迫、资源结构、战斗方式和主线矛盾。

## 核心循环

```text
题材标签 -> 核心体验 -> 主要冲突 -> 题材元素参与解决 -> 中期持续兑现
```

## 开篇必备

- 明确题材给读者的核心体验。
- 列出必须持续出现的题材元素。
- 让题材元素参与第一批冲突。
- 提前规划中期不被其他套路吞掉。

## 期待来源

读者期待开局承诺的类型氛围在后续仍然有效，而不是开局赛博、中期都市异能、后期修行乱斗。

## 断档风险

- 题材只做外皮和热度入口。
- 作者熟悉套路吞掉题材承诺。
- 战斗方式与世界观资源不匹配。
- 单个设定能解释，整体氛围仍然断裂。

## 训练任务

为一个强题材写 5 个不可丢失的核心元素，并设计中期三场冲突，要求每场冲突都必须使用这些元素。
