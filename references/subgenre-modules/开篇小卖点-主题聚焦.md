# 开篇小卖点-主题聚焦

依据案例：`batch-162` 案例 807。

## 读者承诺

在前三章有限篇幅里集中展示最强小卖点，让读者先被一个核心体验抓住。

## 核心循环

```text
确定主题 -> 选择小卖点 -> 集中展示 -> 暂缓其他元素 -> 后续轮动
```

## 开篇必备

- 明确整本书大卖点或主题。
- 前三章只主攻一个最强小卖点。
- 世界、人设、剧情、金手指不要平均分配。
- 小卖点必须服务主题。

## 期待来源

读者期待第一口就吃到明确体验：世界新奇、剧情刺激、主角讨喜、金手指有趣，至少要有一个足够突出。

## 断档风险

- 开篇面面俱到但没有重点。
- 强行套跌宕剧情，忽视世界或人设才是真卖点。
- 小卖点和整本书主题脱节。
- 后续轮动时丢掉主承诺。

## 训练任务

把前三章每段标注为世界、剧情、人设、金手指或情绪。保留最强类别为主线，删减分散注意力的辅助信息。
