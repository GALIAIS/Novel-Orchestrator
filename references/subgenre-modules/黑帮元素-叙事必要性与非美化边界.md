# 黑帮元素-叙事必要性与非美化边界

依据案例：batch-392 案例 1956

## 读者承诺

犯罪组织只在承担主线压力时进入故事，并通过受害、调查、脱离和后果呈现复杂选择；架空套皮不代替现行官方规则核验。

## 核心循环

```text
地下权力制造压迫 -> 主角调查抵抗或脱离 -> 组织以利益和威胁反制 -> 选择付出法律关系代价 -> 阶段结果改善受害处境
```

## 开篇必备

- 删除后主线会受损的必要功能。
- 清楚的受害、调查或反犯罪观察位置。
- 犯罪收益背后的胁迫与退出成本。
- 发布前现行官方规范门禁。

## 期待来源

- 地下权力与公共秩序发生真实冲突。
- 主角在利益和底线间作出选择。
- 调查与反制不断改变证据局面。
- 阶段胜利让责任和后果落地。

## 断档风险

- 用黑帮标签只为让主角耍酷。
- 架空、外国或卧底被当作免责词。
- 展示可复现违法步骤。
- 违法收益长期没有代价。

## 训练任务

把同一梗概改成受害者求生、调查者取证、成员脱离三版，删除纯耍酷细节；现行平台规则未核验前不作可发布结论。
