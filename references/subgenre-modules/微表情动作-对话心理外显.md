# 微表情动作-对话心理外显

依据案例：batch-263 案例 1314

## 读者承诺

对话里的心理状态可以通过肢体动作、微表情、停顿和台词矛盾外显，让读者自己推断人物真实想法。

## 核心循环

```text
真实心理 -> 外显动作 -> 台词不一致 -> 压力追问 -> 破绽暴露
```

## 开篇必备

- 人物真实目的。
- 表面台词。
- 一个可观察动作。
- 动作与台词的矛盾。
- 对手抓住破绽。

## 期待来源

读者期待从细节中判断谁在撒谎、谁在隐瞒、谁在掌控局面。

## 断档风险

机械堆“眼神闪躲”会俗；动作脱离语境会误导；写成心理学科普会破坏剧情。

## 训练任务

写一段审讯或谈判，禁止直接说明撒谎，用三处动作和一次停顿暴露心理。
