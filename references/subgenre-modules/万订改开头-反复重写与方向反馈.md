# 万订改开头-反复重写与方向反馈

依据案例：batch-355 案例 1771

## 读者承诺

好开头常常来自多轮推翻和重写。真正有效的反馈不是一句“不行”，而是指出思路哪里不成立，帮助作者重建读者承诺、矛盾入口和主角行动路线。

## 核心循环

```text
开头初稿 -> 方向反馈 -> 推翻旧思路 -> 多轮重写 -> 达到发布标准 -> 成绩验证
```

## 开篇必备

- 明确开头承诺。
- 18 元素基础检查。
- 反馈指出具体方向。
- 每轮重写只解决明确问题。

## 期待来源

- 思路重建后，读者进入更快。
- 多轮否定压出更清晰的第一事件。
- 高水平反馈能缩短试错。
- 成绩反过来验证开头有效。

## 断档风险

- 只改句子，不改思路。
- 只收到“不行”，没有方向。
- 反复重写变成情绪消耗。
- 基础 18 元素都未达标就追求万订质感。

## 训练任务

拿一个开头连续做 3 版重写，每版写明“本版解决的唯一问题”。禁止同时大改所有东西。
