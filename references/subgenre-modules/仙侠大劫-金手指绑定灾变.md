# 仙侠大劫-金手指绑定灾变

依据案例：`batch-176` 案例 878。

## 读者承诺

主角被顶层斗争外溢出的灾变牵连，通过调查局部异常、解决灾祸和触发金手指，逐步接近大劫真相。

## 核心循环

```text
顶层斗争 -> 位面动荡 -> 众生受害 -> 主角介入 -> 金手指触发 -> 仇恨扩大 -> 大劫真相显露
```

## 开篇必备

- 顶层斗争要能影响底层众生。
- 主角介入点必须具体。
- 金手指触发动作要绑定题材。
- 每次收益同时制造新冲突。

## 期待来源

读者期待主角从局部灾祸查到大劫根源，同时用有题材辨识度的金手指升级。剑仙斩剑、炼丹夺天赋等机制都能把成长和冲突绑在一起。

## 断档风险

- “天资+努力”表达太白，缺少可感知机制。
- 金手指只给收益，不制造冲突。
- 灾变只是背景板，和主角成长无关。
- 升级速度没有限制，早早失控。

## 训练任务

设计一个仙侠金手指：写清触发动作、收益、限制、反噬、与大劫的关系，以及中期二次揭露方式。
