# 新人入门-本命技能专精

依据案例：`batch-009` 案例 044。

## 读者承诺

新人先打磨一个能理解、能拆、能写的本命技能，而不是同时追逐所有理论。专精让训练有反馈，反馈再决定下一步。

## 核心循环

```text
筛方向 -> 选样本 -> 集中阅读 -> 拆同类 -> 写同类 -> 反馈修正
```

## 开篇必备

- 选择自己能看懂的题材。
- 找到 3 到 5 本同类样本。
- 只练一个核心技能。
- 用拆书连接理论和实战。

## 期待来源

- 训练目标清晰，不被教程淹没。
- 同类样本越多，套路越容易看出来。
- 能先写出糙但完整的商业桥段。
- 小范围反馈比大范围焦虑更有效。

## 断档风险

- 今天学开篇，明天学文笔，后天学名著。
- 没有同题材样本，只收藏理论。
- 追求面面俱到，结果一个技能也不能用。
- 过早追求高级表达，忽略商业可读性。

## 训练任务

两周只练一个本命技能。每天读同类书 20 章，记录 3 个桥段，仿写 1 个小单元。
