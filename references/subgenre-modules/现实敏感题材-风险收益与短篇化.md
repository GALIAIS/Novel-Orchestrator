# 现实敏感题材-风险收益与短篇化

依据案例：batch-364 案例 1818

## 读者承诺

现实高敏议题情绪强，但审核、争议和平台风险大。长篇开头慎用，必要时短篇化、架空化或私人困境化。

## 核心循环

```text
现实痛点 -> 风险评估 -> 短篇/架空替代 -> 保留核心情绪 -> 避免长篇拖累
```

## 开篇必备

- 先判断审核风险。
- 评估读者争议是否会压过故事。
- 不直贴热点。
- 保留情绪，降低现实指向。

## 期待来源

- 强痛点。
- 私人困境。
- 架空制度压迫。
- 短篇快速兑现主题。

## 断档风险

- 招娣、八孩女等现实热点直接开局。
- 涉现实政治和战争影射。
- 审核讨论压过剧情。
- 风险大但收益小。

## 训练任务

把一个现实高敏开头改成 3000 字短篇版和架空长篇版，比较风险与期待集中度。
