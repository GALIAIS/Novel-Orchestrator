# 高端慢开局-悬念折线铺垫

依据案例：`batch-029` 案例 144。

## 读者承诺

慢开局不是拖，而是用异常细节、日常代入和悬念折线持续拉期待。

## 核心循环

```text
异常细节 -> 抛出疑问 -> 日常代入 -> 小回收 -> 更大疑问 -> 节点爆发
```

## 开篇必备

- 第一章必须有反常信息。
- 日常段落承担代入或人设。
- 每个答案后要有新问题。
- 明确新人是否能承受这个节奏。

## 期待来源

- 读者想知道异常背后的真相。
- 金手指逼格先被抬高。
- 人设节点提前埋伏。
- 小回收制造继续追读。

## 断档风险

- 只学慢，不学悬念密度。
- 日常没有代入功能。
- 悬念只抛不回收。
- 新人投稿照搬大神节奏。

## 训练任务

拆慢开局前三章，每章列出异常细节、读者疑问、小回收和下一层悬念。
