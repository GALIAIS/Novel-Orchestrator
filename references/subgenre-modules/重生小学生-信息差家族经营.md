# 重生小学生-信息差家族经营

依据案例：batch-304 案例 1518

## 读者承诺

重生小学生靠信息差逐层经营校园、家庭、商业和人脉，但亲情线不能写成操控家人。

## 核心循环

```text
重生信息差 -> 校园低位经营 -> 家庭信任授权 -> 商业/人脉扩张 -> 家族共同受益
```

## 开篇必备

- 小学生身份形成反差。
- 第一层目标不要过大。
- 信息差要落成具体收益。
- 家人是互惠关系，不是工具。

## 期待来源

- 小孩身份降维打击。
- 未来知识带来连续经营。
- 家庭和事业一起变好。

## 断档风险

- 金手指太普通，剧情构建撑不住。
- 把爷爷、父母写成工具人。
- “暗中控制家族”触发孝道毒点。

## 训练任务

写三段家族经营：校园声望、父母信任、爷爷授权。每段都写互惠收益。
