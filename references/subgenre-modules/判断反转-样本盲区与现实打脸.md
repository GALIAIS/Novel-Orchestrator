# 判断反转-样本盲区与现实打脸

依据案例：batch-385 案例 1921

## 读者承诺

角色基于有限样本作出自信判断，随后由快速、客观、不可辩解的结果完成反证。打脸后还要产生认知更新。

## 核心循环

```text
有限样本 -> 武断结论 -> 结果出现 -> 事实反证 -> 认知更新
```

## 开篇必备

- 角色看见的有限样本。
- 明确公开的预测。
- 被忽略的关键变量。
- 可验证的客观结果。

## 期待来源

- 读者察觉判断可能有盲区。
- 预测和结果形成强反差。
- 时间差短，兑现迅速。
- 角色必须处理认错或嘴硬后果。

## 断档风险

- 个性化推荐被当成总体市场。
- 单个反例又被当成整体证明。
- 靠群体羞辱替代事实反证。
- 打脸后角色毫无认知变化。

## 训练任务

写一个 1200 字场景：角色基于三个有限样本预测失败，二十四小时内被客观结果推翻，禁止群体嘲讽。
