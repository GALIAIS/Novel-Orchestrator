# 歌词叙事-配角人设原型

依据案例：`batch-099` 案例 491。

## 读者承诺

叙事型歌词能提供完整的人物命运线。把歌词里的伤害、目标、主动行动、环境阻力和情义证明拆出来，就能生成有故事感的配角或女主人设。

## 核心循环

```text
叙事歌词 -> 标注情绪节点 -> 提炼人物核心 -> 换题材外壳 -> 写成角色线
```

## 开篇必备

- 选有完整故事的歌词，不选纯意识流碎片。
- 拆叙事节点，不只摘金句。
- 保留人物主动行动。
- 把情义、伤痛或坚持转成角色核心。

## 期待来源

- 不公平伤害带来的同情。
- 主动寻人/守诺带来的可敬。
- 恶劣环境反衬情义。
- 民间故事原型带来的熟悉感。

## 断档风险

- 只搬歌词句子，没拆人物线。
- 换皮后丢掉主动行动。
- 情绪只靠悲，不给角色选择。
- 现代意识流歌词缺完整剧情。

## 训练任务

选一首叙事老歌，拆成角色卡：伤害、目标、行动、阻力、代价、情义证明、结局。再把它改成当前题材里的一个配角。
