# 开篇人设-主持公道侧写不公

依据案例：`batch-320` 案例 1599

## 读者承诺

老套金手指也能靠快节奏、伏笔和选择立住开篇；“别人不敢我敢”的主持公道能迅速建立主角价值观。

## 核心循环

```text
具体不公 -> 主角选择主持公道 -> 危机追杀 -> 顺带交代势力设定 -> 核心悬念拉期待
```

## 开篇必备

- 不公待遇用细节和选择侧写。
- 主角要有别人不敢做的动作。
- 金手指老套时，更要靠事件节奏补足。
- 第一次追杀同时带出势力和规则。
- 核心悬念反复强调，不一次讲完。

## 期待来源

读者期待主角在不公环境里做出有价值观的选择，并通过危机看到世界规则和后续悬念。

## 断档风险

- 只让主角口头感叹不公。
- 天赋金手指老套，事件还慢。
- 追杀只打斗，不带出设定和势力。
- 核心悬念没有持续拉期待。

## 训练任务

写 800 字开篇：金手指只是天赋好，但主角必须通过一次主持公道立人设，并在危机里带出一个核心悬念。
