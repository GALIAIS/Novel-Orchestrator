# 同人开篇-原著规则与身份用词

依据案例：`batch-228` 案例 1140。

## 读者承诺

同人开篇既要有通用网文钩子，也要守住原著设定、年龄身份和常识。

## 核心循环

```text
通用钩子 -> 原著校验 -> 身份用词校验 -> 删除违和 -> 代入稳定
```

## 开篇必备

- 有不公平伤害、冒险理由、系统或出发节点。
- 原著时间线清楚。
- 年龄和形容词匹配。
- 能力来源不违和。

## 期待来源

同人读者先确认熟悉规则被尊重，再接受新故事爽点。

## 断档风险

- 通用钩子有了，原著规则崩。
- 十岁角色用成人化形容。
- 血缘、宗族、觉醒时间线说不清。
- 附件不可读却硬评正文。

## 训练任务

重写同人开篇设定表：主角年龄、原著时间线、能力来源、冒险动机、被伤害事件、出发节点。每项写原著依据/不违和理由，并删掉年龄身份不匹配的形容词。
