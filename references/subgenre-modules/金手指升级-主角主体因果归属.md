# 金手指升级-主角主体因果归属

依据案例：`batch-184` 案例 916。

## 读者承诺

后期金手指升级必须让读者觉得是主角催化了能力，而不是主角只是预设外挂的载体。

## 核心循环

```text
旧能力 -> 主角特质催化 -> 能力进化 -> 主角功劳放大 -> 长篇情绪资产保值
```

## 开篇必备

- 旧能力要有清晰功能。
- 升级触发要和主角选择、意志或资质有关。
- 回溯设定不能打脸前期势力行为。
- 新能力要延伸旧能力，而不是吞掉旧能力。

## 期待来源

读者期待后期升级证明主角一路成长有意义，而不是证明他早就被钦定。

## 断档风险

- 神级外挂早已预设，主角可有可无。
- 前期敌人行为因新设定变蠢。
- 主角努力被血统或宿命吞掉。
- 旧金手指和新金手指断裂。

## 训练任务

把一个普通金手指升级成神级能力：必须写出主角独有触发条件和前期设定不被打脸的解释。
