# 理论自检-课程大纲与知识映射

依据案例：batch-372 案例 1860

## 读者承诺

外部课程大纲可以作为创作体系自检表。先映射已有知识和案例，再判断缺口，避免把学习焦虑误当成新方法需求。

## 核心循环

```text
外部大纲 -> 条目拆分 -> 映射已有模块 -> 标记缺口 -> 生成训练清单
```

## 开篇必备

- 一个完整外部大纲。
- 已有知识模块清单。
- 对应案例或实践证据。
- 掌握程度判断。

## 期待来源

- 把模糊焦虑转成清单。
- 发现已有体系覆盖范围。
- 找到真正缺失的二三成。
- 让学习回到训练任务。

## 断档风险

- 只收藏大纲，不做映射。
- 看到新名词就以为必须重学。
- 缺少案例证明掌握程度。
- 买课代替拆书和实战。

## 训练任务

找一份 10 条以上课程大纲，逐条写出“已有对应知识、案例证据、缺口、训练任务”，只把无法解释的条目标成补课对象。
