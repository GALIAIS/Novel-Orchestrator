# 手弩理论-可用非必用

依据案例：batch-270 案例 1346

## 读者承诺

把手弩理论当作增强冲突高潮的技巧，而不是每个冲突都必须套用的硬模板。

## 核心循环

```text
目标 -> 阻力 -> 情绪下压 -> 主角抉择 -> 底牌解决 -> 高潮回弹
```

## 开篇必备

- 先判断冲突重量。
- 大冲突可多次下压，小冲突可直接解决。
- 下压必须来自真实阻力。
- 底牌要承担情绪回弹。

## 期待来源

期待来自读者看见主角在压力下仍能达成目标。压力越有效，底牌越有爽感。

## 断档风险

- 把“可以用”误解成“必须用”。
- 小冲突硬套完整流程拖慢节奏。
- 下压点硬塞，读者觉得憋屈。
- 底牌没有提前建立。

## 训练任务

同一个冲突写两版：一版完整手弩流程，一版快速解决。比较哪一版更符合章节重量。
