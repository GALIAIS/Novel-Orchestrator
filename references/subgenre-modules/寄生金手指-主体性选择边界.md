# 寄生金手指-主体性选择边界

依据案例：`batch-337` 案例 1683

## 读者承诺

寄生类金手指可以强化身体和资源，但主角人格必须独立，性格变化要来自选择、经历和力量处境。

## 核心循环

```text
外力强化 -> 主角保留选择 -> 经历改变行为 -> 原有特质放大 -> 爽点归属主角
```

## 开篇必备

- 金手指能做什么清楚。
- 人格边界清楚。
- 主角有选择余地。
- 原本人格特质有伏笔。
- 首次变化归因于主角行动。

## 期待来源

读者期待外力帮助主角变强，但最终决定、代价和爽点都属于主角本人。

## 断档风险

- 外挂强行改性格。
- 主角像傀儡或被夺舍。
- 怂炮慢慢成长太拖。
- 性格变化没有原有底色。
- 金手指主导行动，主角只执行。

## 训练任务

把一个“外挂改变性格”的设定改成“外挂给三个选择，主角主动选择最锋利的一条”，写清收益和代价。
