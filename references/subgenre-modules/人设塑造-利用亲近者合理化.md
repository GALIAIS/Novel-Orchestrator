# 人设塑造-利用亲近者合理化

依据案例：`batch-155` 案例 774。

## 读者承诺

主角可以算计和利用亲近者，但必须提前铺垫困境、需求和补偿，让读者接受这是复杂选择而非无意义伤害。

## 核心循环

```text
亲近者困境 -> 主角不得不利用 -> 利用带来补偿 -> 情绪不快被化解 -> 后续反转继续推进
```

## 开篇必备

- 被利用者的当前困境。
- 被利用者的隐藏需求。
- 主角不得不做的理由。
- 利用后的补偿。
- 读者不快点的化解。
- 后续反转伏笔。

## 期待来源

读者期待主角既有算计能力，又不会无意义损害自己人。

## 断档风险

- 只写主角利用，不写对方需求。
- “为你好”只靠台词，没有结果证明。
- 被利用者没有补偿。
- 主角冷酷到读者失去认同。

## 训练任务

写一个主角利用队友的桥段，先列对方困境、需求、主角理由和补偿结果，再写正文。
