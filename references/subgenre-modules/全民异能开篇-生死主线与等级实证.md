# 全民异能开篇-生死主线与等级实证

依据案例：batch-391 案例 1955

## 读者承诺

用生死限制聚焦觉醒主线，先以常规高等级能力建立具体效果标尺，再让主角越级结果同时缓解危机并引来长期麻烦。

## 核心循环

```text
生存倒计时 -> 选择觉醒路径 -> 常规高阶能力立标尺 -> 主角能力越级实证 -> 部分获救并被更大势力关注
```

## 开篇必备

- 能约束主角语气和行动的生死问题。
- 一条清楚而非并发的觉醒路径。
- 高等级能力的物理或社会效果参照。
- 前后统一的疾病、药物、觉醒和能力术语。

## 期待来源

- 生存时间压力迫使主角选择。
- 标尺让能力等级不再只是字母。
- 主角越级获得即时上行情绪。
- 解决部分危机同时产生新关注和代价。

## 断档风险

- 穿越、系统、神赐和绝症等量抢戏。
- 连续对话不改变局势。
- 用环境字数配额掩盖转场无因果。
- 简介承诺的异能或游戏玩法后文停用。

## 训练任务

把前三章压成生存、觉醒、标尺、越级、部分获救五步；重写一段功能性对话和两个因果转场，再用术语表统一设定名称。
