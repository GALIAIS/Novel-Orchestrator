# 工具人推进-好心办坏事

依据案例：`batch-029` 案例 142。

## 读者承诺

当主角人设不适合主动触发冲突时，用好心办坏事的配角把主角带入局中，同时保住主角价值观。

## 核心循环

```text
主角原则 -> 配角误推 -> 主角入局 -> 能力兑现 -> 主角补救 -> 配角受罚
```

## 开篇必备

- 主角不能主动做某事的理由。
- 配角推动事件的合理动机。
- 配角不是纯恶意拖后腿。
- 结尾有代价回收。

## 期待来源

- 主角被迫进入不合人设的冲突。
- 读者期待主角如何处理。
- 装逼兑现后还能保持原则。
- 配角因错误承担后果。

## 断档风险

- 工具人太蠢太讨厌。
- 主角被动得像没脑子。
- 装逼后不修复人设。
- 配角不付代价，读者憋气。

## 训练任务

写一个正派主角被配角拖进灰色冲突的桥段，结尾必须让主角补救价值观，并让配角承担小损失。
