# 傲娇人设-仪式小动作

依据案例：`batch-138` 案例 689。

## 读者承诺

高冷角色在亲密仪式中用极小动作泄露真实期待，让读者吃到“傲”和“娇”同时存在的满足感。

## 核心循环

```text
高冷外壳 -> 关键物件 -> 亲密仪式 -> 小动作泄露 -> 关系确认
```

## 开篇必备

- 一个关键物件。
- 女主表面克制。
- 男主完成任务。
- 仪式性互动。
- 极小身体动作。

## 期待来源

- 物件承载关系期待。
- 冷淡外壳和主动小动作反差。
- 读者自己识别真实情绪。
- 关系推进不靠直白表白。

## 断档风险

- 只有傲，没有娇。
- 直接台词表白，失去克制感。
- 小动作太大，变成崩人设。
- 物件没有情感含义。

## 训练任务

写 300 字男女主互动：女主交出关键物件，男主完成任务后归还或佩戴，女主只能通过一个极小动作暴露期待。
