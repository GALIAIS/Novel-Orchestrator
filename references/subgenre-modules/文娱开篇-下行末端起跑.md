# 文娱开篇-下行末端起跑

依据案例：`batch-020` 案例 096。

## 读者承诺

让文娱开篇从“主角持续受苦”改成“主角已经在下行末端，马上走向作品展示和装逼”。读者点进来要看才华兑现，不是连续三章憋屈。

## 核心循环

```text
困境压缩交代 -> 作品准备 -> 现场误判 -> 才华展示 -> 反派/导师反应
```

## 开篇必备

- 主角处于什么下行末端。
- 马上要展示的作品或能力。
- 现场谁不看好主角。
- 作品如何回应当前言论。

## 期待来源

- 读者知道主角有文娱金手指。
- 现场评价、导师偏见、反派嘲讽等待被打脸。
- 作品选择和当前场景高度相关。

## 断档风险

- 连续写主角苦逼，不给上行路径。
- 歌曲和现场冲突无关。
- 选秀里没有可踩对象。
- 爽点弱却只想着换歌，不补前置拉扯。

## 训练任务

写一个 1500 字选秀开篇：前 300 字交代困境，后 1200 字必须进入准备、登台、误判和作品回应。
