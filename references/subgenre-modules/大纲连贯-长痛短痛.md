# 大纲连贯-长痛短痛

依据案例：`batch-130` 案例 646。

## 读者承诺

提前打磨大纲很痛苦，但能换取长篇主线贯穿、支线回流、伏笔可控和人物行动合理。短痛解决连贯性，避免中后期烂尾长痛。

## 核心循环

```text
主线框架 -> 支线功能 -> 人物行动理由 -> 伏笔回收 -> 反复增删改 -> 长篇可持续
```

## 开篇必备

- 全书主线明确。
- 卷目标能接主线。
- 支线必须有功能。
- 人物行动符合处境和性格。
- 前期设定能支撑后期回收。

## 期待来源

- 剧情前后连贯。
- 支线能推动主线。
- 人物做事有理由。
- 后续修改不至于全局崩盘。

## 断档风险

- 想到哪写到哪。
- 支线不回流主线。
- 为设定抬杠浪费精力。
- 人物行动和处境性格不匹配。

## 训练任务

把一个 10 万字以上的大纲拆成主线、卷目标、支线功能、人物行动理由四栏，找出 3 个不连贯点并写修复方案。
