# 仙侠书单-古典旁门类型索引

依据案例：`batch-176` 案例 880。

## 读者承诺

通过代表作品建立古典仙侠、凡人流、旁门左道和诡异修仙的参照系，帮助作者选样本拆书。

## 核心循环

```text
收集书单 -> 判断开头气质 -> 追踪核心循环 -> 区分子类型 -> 选择章节精拆
```

## 开篇必备

- 古典仙侠看仙味、道统、入道体验。
- 凡人流看资源、谨慎、境界推进。
- 旁门左道看异术、代价、偏门法脉。
- 诡异修仙看污染、恐怖、认知错乱。

## 期待来源

读者期待不同仙侠味道带来的审美体验。作者用书单找标靶，再通过章节精拆确定这本书真正靠什么留住读者。

## 断档风险

- 只看开头氛围就下分类结论。
- 把书单当方法论，缺少后续拆解。
- 古典仙侠和凡人流混淆。
- 旁门异术只剩猎奇，没有修行循环。

## 训练任务

选三本推荐书，各读前 10 章，分别标注古典、凡人、旁门、诡异特征，并写出最值得拆的一个章节。
