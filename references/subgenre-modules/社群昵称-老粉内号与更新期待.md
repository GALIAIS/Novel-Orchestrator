# 社群昵称-老粉内号与更新期待

依据案例：`batch-241` 案例 1203。

## 读者承诺

社群昵称和老粉内号能沉淀共同记忆，但需要让新成员能理解来源。

## 核心循环

```text
老粉用昵称 -> 新粉疑惑 -> 成员解释 -> 共同记忆形成 -> 更新期待增强
```

## 开篇必备

- 记录昵称来源。
- 给新粉解释入口。
- 明确更新时间。
- 保持自然使用。

## 期待来源

用户不只是等内容，也在等一个熟悉的人和熟悉社群的持续更新。

## 断档风险

- 内部梗让新粉看不懂。
- 强行造梗。
- 只玩梗不兑现更新。
- 老称呼和新品牌割裂。

## 训练任务

整理作者社群的常用称呼与来源表，给新成员一个能快速理解的入口。
