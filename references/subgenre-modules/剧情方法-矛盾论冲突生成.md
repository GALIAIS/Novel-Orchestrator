# 剧情方法-矛盾论冲突生成

依据案例：`batch-186` 案例 930。

## 读者承诺

把矛盾论转化为剧情工具，可以从人物目标、资源、规则和观念中稳定生成冲突。

## 核心循环

```text
核心矛盾 -> 多层冲突 -> 具体场景 -> 解决代价 -> 下一层矛盾
```

## 开篇必备

- 先抓核心矛盾。
- 每层矛盾都要有双方目标。
- 冲突要落到具体场景。
- 解决要付代价。

## 期待来源

读者持续看到问题升级、选择变难、代价变重，剧情就不会空转。

## 断档风险

- 只写事件，不写矛盾。
- 矛盾停留在抽象词。
- 没有层级，长篇很快断粮。
- 冲突没有代价。

## 训练任务

为一个新书点子列人物内在、人际、资源、规则、世界观五层矛盾，每层写一个具体冲突场景。
