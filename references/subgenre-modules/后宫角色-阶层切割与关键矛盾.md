# 后宫角色-阶层切割与关键矛盾

依据案例：`batch-219` 案例 1092。

## 读者承诺

用社会阶层和群体身份生成差异化角色，让每个重要角色绑定一条关键矛盾。

## 核心循环

```text
世界阶层 -> 代表身份 -> 利益立场 -> 关键矛盾 -> 牵动主角
```

## 开篇必备

- 先切世界群体，再选角色身份。
- 身份标签要绑定社会关系。
- 关键人物对应关键矛盾。
- 后宫数量要控制。

## 期待来源

读者看到不同身份角色时，同时看到她背后的资源、义务、敌人和主线冲突。

## 断档风险

- 只用性格标签区分角色。
- 后宫角色太多，矛盾过载。
- 身份不参与剧情。
- 标签鲜明但事件不兑现。

## 训练任务

切出 6 个社会群体，各设计 1 个女性身份和牵连主角的矛盾，再删到最能推动主线的 3 个。
