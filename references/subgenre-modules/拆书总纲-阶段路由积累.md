# 拆书总纲-阶段路由积累

依据案例：`batch-092` 案例 460。

## 读者承诺

没写过书时先看书积累，不急着拆复杂理论；开始写后，带着具体短板去拆书、拉片和积累素材，才能真正吸收。

## 核心循环

```text
没写过 -> 先看同题材书 -> 开始写 -> 暴露短板 -> 对应拆书拉片 -> 积累素材 -> 回到实写
```

## 开篇必备

- 判断自己是否写过书。
- 没写过先输入同题材。
- 写作后再确定拆什么。
- 阅读量不足可以边写边补。
- 总纲只做路由，不一次性吞完。

## 期待来源

- 拆书目标更明确。
- 素材积累能服务实写。
- 阅读和写作同步增长。
- 工具只辅助，不替代动手拆。

## 断档风险

- 没写过就沉迷拆书理论。
- 等看完一亿字才动笔。
- 链接和工具失效不维护。
- 拆书不对应自己的短板。

## 训练任务

判断阶段：没写过列 10 本同题材待读；刚写列 3 个短板；已有作品选 1 个短板，对应拆 1 本书和 1 段影视片段。
