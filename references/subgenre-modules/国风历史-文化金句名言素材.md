# 国风历史-文化金句名言素材

依据案例：`batch-178` 案例 890。

## 读者承诺

文化类金句素材帮助作者提升历史、文物、国风和家国主题的表达质感。

## 核心循环

```text
文化语料 -> 提取句式/意象 -> 分类归档 -> 剧情语境改写 -> 台词/标题/卷首语落地
```

## 开篇必备

- 素材按守护、传承、时间、器物、家国分类。
- 宏大表达要落到具体人物和器物。
- 名言只学句式和情绪，不直接硬塞。
- 使用前必须改写进当前剧情。

## 期待来源

读者期待文化表达有厚度、有画面、有情绪。素材库提供句式来源，真正有效的是场景化改写。

## 断档风险

- 截图收藏，不分类。
- 大词堆砌，没有人物和器物承接。
- 生搬节目台词，语境不合。
- 文化素材不服务剧情。

## 训练任务

整理 10 句文化类台词，按意象和适用场景分类，每类改写一句放入小说台词、标题或卷首语。
