# 输入疲劳-换媒介恢复阅读

依据案例：batch-310 案例 1550

## 读者承诺

作者阶段性看不进小说时，不必硬逼自己扫榜，可以换电影、动漫或更高质量作品，保持输入不断。

## 核心循环

```text
小说输入疲劳 -> 换媒介 -> 继续吸收结构/节奏/人物 -> 记录方法 -> 回到写作
```

## 开篇必备

- 承认短期疲劳正常。
- 换媒介但不停止输入。
- 只学方法，不照搬细节。
- 记录看完后能用于写作的一条结构经验。

## 期待来源

期待来自恢复刺激。换媒介能让创作脑从文字疲劳里出来，重新吸收故事。

## 断档风险

- 把短期疲劳当成能力废掉。
- 硬扫榜，越看越烦。
- 学细节过多，产生抄袭风险。
- 只娱乐不复盘。

## 训练任务

做七天换媒介输入表：两部电影、两集动漫、一本小说前三章。每个样本只记一个结构方法。
