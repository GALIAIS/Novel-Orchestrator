# 新人综合征-理论挑刺与百万字消化

依据案例：`batch-237` 案例 1183。

## 读者承诺

理论会先提高挑刺能力，再慢慢转化为写作能力；新人不要因几万字用不上全部知识就停笔自责。

## 核心循环

```text
学习理论 -> 挑刺能力提升 -> 自写落差 -> 问题过载 -> 长篇实践消化
```

## 开篇必备

- 接受挑刺和写作不同步。
- 每次只选一个最高优先级问题。
- 用 18 元素表做筛选，不做一次性全修。
- 设置几十万字后的复盘点。

## 期待来源

作者从“我懂但写不好”的焦虑，转向“我先写长，再逐项把理论变成本能”的长期训练。

## 断档风险

- 看别人书时过度自信。
- 看自己稿时完全失焦。
- 18 元素表一照就全盘否定。
- 一直学习不继续写。

## 训练任务

连续 7 天只盯 18 元素表中的一项写作或改稿；其他问题只记录，等下一个周期再处理。
