# 动画拉片-一级结构轻量复盘

依据案例：`batch-018` 案例 090。

## 读者承诺

用短动画、童话、喜剧片段训练一级结构识别能力。目标不是照搬角色，而是提取欲望、阻碍、转折、情绪缺口和兑现。

## 核心循环

```text
选短片 -> 找欲望 -> 标阻碍 -> 拆尝试 -> 识别转折 -> 改写网文事件
```

## 开篇必备

- 角色想得到什么。
- 对手或环境为什么阻碍。
- 观众/读者担心或期待什么。
- 第一次尝试如何失败或变形。

## 期待来源

- 简单欲望带来的低门槛理解。
- 多次尝试和反击制造情绪折线。
- 小转折让同一情绪缺口保持新鲜。
- 最终兑现满足前面持续积累的需求。

## 断档风险

- 只复述剧情，不抽事件功能。
- 把动画表面桥段硬搬到网文。
- 忽略情绪缺口，只看笑料。
- 过度细分，导致看不见主单元。

## 训练任务

选一个 5-10 分钟短片，拆成“欲望、阻碍、三次尝试、一次转折、兑现”。再把它改成同题材网文桥段。
