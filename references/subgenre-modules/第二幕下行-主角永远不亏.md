# 第二幕下行-主角永远不亏

依据案例：batch-289 案例 1442

## 读者承诺

第二幕可以下行，但主角不能空手而归；原目标暂失时，要换到资源、信息或权限。

## 核心循环

```text
主角立功 -> 原收益暂失 -> 主动谈判 -> 换取补偿 -> 第三幕兑现
```

## 开篇必备

- 主角付出的功劳。
- 暂时失去的目标。
- 主角主动选择。
- 可交换资源。
- 读者可见的不亏证据。
- 后续兑现方式。

## 期待来源

读者接受短期下行，是因为看见主角没有被拿捏，而是在用功劳换更大的局。

## 断档风险

- 用昏迷剥夺主角选择权。
- 战利品被拿走却无补偿。
- 主角一直被官方拿捏。
- 下行只剩憋屈，没有新收益。

## 训练任务

把“战利品被官方保管”改成主角主动交易，列出三项官方必须付出的补偿。
