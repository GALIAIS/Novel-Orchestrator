# 剧情杀败北-假死逃脱拿好处

依据案例：`batch-334` 案例 1668

## 读者承诺

主角败北可以写，但要提前铺垫强敌和保命手段，让败北变成战略转移而不是被施舍。

## 核心循环

```text
强敌压迫 -> 保命伏笔 -> 表面败北 -> 假死/逃脱 -> 计划中拿好处
```

## 开篇必备

- 强敌强大提前证明。
- 主角有假死或逃脱伏笔。
- 好处来自主角计划。
- 败北后 500 字内给补偿。
- 避免恶心反派白给好处。

## 期待来源

读者期待主角在不可硬刚的局面里保住主动，并从败局里拿到收益。

## 断档风险

- 真反派打惨主角还赏好处。
- 主角无准备被放过。
- 败北只有羞辱没有成长。
- 好处像作者硬塞。
- 毒点恐惧导致想象力僵化。

## 训练任务

写败北小纲：强敌证据、保命伏笔、表面败北、真实收益、情绪补偿。
