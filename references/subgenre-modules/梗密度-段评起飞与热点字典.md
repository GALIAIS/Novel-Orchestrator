# 梗密度-段评起飞与热点字典

依据案例：`batch-244` 案例 1218。

## 读者承诺

高梗密度能提升段评互动，但必须依靠持续收集、筛选和贴合章节情绪。

## 核心循环

```text
收集热梗 -> 匹配章节情绪 -> 角色口吻承载 -> 读者识别 -> 段评接力
```

## 开篇必备

- 有梗素材库。
- 梗贴合角色口吻。
- 梗不挤占关键剧情。
- 预留段评可接的位置。

## 期待来源

读者在正文里认出熟悉语境，会产生参与感和会心一笑。

## 断档风险

- 一章塞太多梗，剧情变碎。
- 梗不符合人物。
- 只追热点，不服务情绪。
- 小众梗识别门槛太高。

## 训练任务

给一章轻松剧情准备 8 个梗，最终只留下 3 个：台词、旁白、段末互动各一个。
