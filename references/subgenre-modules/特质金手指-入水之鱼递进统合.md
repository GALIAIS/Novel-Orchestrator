# 特质金手指-入水之鱼递进统合

依据案例：`batch-184` 案例 918。

## 读者承诺

主角自身特质与冒险世界高度适配，本身就是入水之鱼式金手指；后续新能力最好与旧特质递进统合。

## 核心循环

```text
主角特质 -> 世界适配 -> 优势兑现 -> 新能力加入 -> 整体感增强
```

## 开篇必备

- 主角特质具体可见。
- 世界规则能放大该特质。
- 前三章要有特质兑现。
- 新金手指尽量与旧特质相连。

## 期待来源

读者期待主角不是靠外物偶然变强，而是因为这个世界正好需要他这种人。

## 断档风险

- 特质只在简介里说，正文不兑现。
- 新金手指和旧优势毫无关系。
- 强行关联导致别扭。
- 忘记爽感优先。

## 训练任务

设计一个特质型金手指：列出主角特质、世界适配点、前三次兑现和后续新能力的递进关系。
