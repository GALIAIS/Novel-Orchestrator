# 女频迪化-反向攻略恋爱

依据案例：`batch-148` 案例 737。

## 读者承诺

主角不断选择最差恋爱选项，却被外界迪化误读成高明、深情或强大，最后把男频打怪升级爽点转译成女频攻略成功爽点。

## 核心循环

```text
最差选项 -> 迪化误读 -> 主角生气 -> 结果有利 -> 攻略反馈 -> 主线推进
```

## 开篇必备

- 主角明确不想谈恋爱或想避开剧情。
- 一个最差选项。
- 男主或配角的误读。
- 喜剧气氛。
- 攻略成功反馈。
- 扎实主线，避免只靠段子。

## 期待来源

读者期待反差：主角越想搞砸，外界越觉得她厉害，恋爱线越被反向推进。

## 断档风险

- 迪化重复导致疲劳。
- 只有梗，没有主线。
- 最差选项没有合理动机。
- 男频爽点直接搬运，没转成攻略反馈。

## 训练任务

写 5 个“最差恋爱选项 -> 被误读成高明行为”的桥段。再把一个男频打怪升级爽点改写成女频攻略成功爽点，保留目标、阻碍、反馈和升级。
