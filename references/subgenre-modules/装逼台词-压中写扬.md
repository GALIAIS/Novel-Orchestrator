# 装逼台词-压中写扬

依据案例：batch-282 案例 1409

## 读者承诺

装逼台词不靠华丽辞藻，而靠先压低主角愿望，再把出手责任推给敌人，最后完成审判和惩罚宣告。

## 核心循环

```text
我本不想 -> 只想平凡 -> 是你们逼我 -> 道德审判 -> 惩罚兑现
```

## 开篇必备

- 主角原本的低愿望。
- 敌人破坏这个愿望。
- 主角不愿但必须出手。
- 台词后有实际行动兑现。

## 期待来源

读者感到敌人唤醒了不该唤醒的人。压低越真，扬起越有逼格。

## 断档风险

- 主角直接自吹。
- 台词华丽但没有审判重量。
- 只说不做。
- 责任没有推给敌人，显得爱装。

## 训练任务

写 8 句强者归来台词：前三句压低愿望，中间两句归责敌人，后三句审判并宣告惩罚。
