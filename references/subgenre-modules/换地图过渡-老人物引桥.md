# 换地图过渡-老人物引桥

依据案例：`batch-159` 案例 794。

## 读者承诺

换地图、开新卷或进入新剧情时，用老人物关系先拉住读者，让陌生场景顺滑过渡到新目标和新冲突。

## 核心循环

```text
新地图 -> 老人物登场 -> 旧关系唤起期待 -> 引出新人物/新冲突 -> 短期目标接管 -> 老人物收场
```

## 开篇必备

- 选一个读者记得住标签的老人物。
- 让旧关系立刻产生小冲突或小麻烦。
- 用这段关系引出新地图真正问题。
- 三章内安排老人物退场、转岗或进入新关系。

## 期待来源

熟悉人物带来代入，旧关系带来情绪预判，新地点提供新鲜感，新冲突接管后续阅读动力。

## 断档风险

- 新地图直接堆新名词、新势力、新人物。
- 老人物桥段太长，压住真正短期目标。
- 只怀旧，不推动新冲突。
- 老人物没有收场，变成拖线。

## 训练任务

写一份新卷前三章表：第一章老人物带入新地点，第二章旧关系触发新冲突，第三章短期目标明确并安排老人物收场。
