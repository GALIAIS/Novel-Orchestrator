# 苟而不怂-避锋后必清算

依据案例：`batch-109` 案例 541。

## 读者承诺

主角可以稳、可以苟、可以暂时避开强敌，但读者必须相信这不是逃避。真正的承诺是主角会积累实力，等时机成熟后回来清算旧敌。

## 核心循环

```text
敌强我弱 -> 暂避锋芒 -> 暗中成长 -> 反击窗口 -> 痛快清算
```

## 开篇必备

- 明确主角为什么现在打不过。
- 避让必须是策略，不是怯懦。
- 给出未来清算承诺。
- 暂避期间有实际成长。
- 最终敌人必须付出代价。

## 期待来源

- 读者等主角回来。
- 强敌压迫越重，清算越爽。
- 主角的稳被证明有用。
- “苟”服务复仇，而不是取消复仇。

## 断档风险

- 主角一直逃，没有反击计划。
- 作者把怂写成稳。
- 旧敌长期无代价。
- 身边高智商角色为了衬托主角而降智。

## 训练任务

写一段主角撤退剧情，结尾必须留下三个东西：复仇目标、成长路径、第一次反击机会。
