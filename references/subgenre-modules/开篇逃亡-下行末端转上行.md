# 开篇逃亡-下行末端转上行

依据案例：`batch-032` 案例 156。

## 读者承诺

逃亡开局只能短用，最好从下行末端切入，马上让主角主动反转进入上行。

## 核心循环

```text
不公平伤害 -> 下行末端 -> 主角计划 -> 金手指转机 -> 复仇/脱离 -> 主线上行
```

## 开篇必备

- 下行不能持续太久。
- 主角必须主动。
- 第一章内给转机。
- 逃亡要服务复仇或主线目标。

## 期待来源

- 读者看到压迫后马上等反打。
- 金手指出现带来安全感。
- 主角不是被追着跑，而是在执行计划。
- 逃亡结束后进入提升或复仇。

## 断档风险

- 七八章连续逃亡。
- 主角完全被动挨打。
- 自爆重生只是瞎蒙活下来。
- 解决一波追兵又来一波追兵。

## 训练任务

把一个逃亡开头改成下行末端切入：500 字内出现金手指或主动计划，章末必须转上行。
