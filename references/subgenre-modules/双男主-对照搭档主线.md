# 双男主-对照搭档主线

依据案例：`batch-024` 案例 118。

## 读者承诺

两个男主必须有明显立场、心性和行为差异，并通过共同目标不断冲突与协作。

## 核心循环

```text
差异登场 -> 目标绑定 -> 方法冲突 -> 被迫协作 -> 共同完成 -> 关系/事业推进
```

## 开篇必备

- 明确目标平台。
- 是否有 CP 或感情脑补空间。
- 双主角对照标签。
- 第一个共同目标。

## 期待来源

- 差异人物碰撞。
- 同目标下的方法冲突。
- 协作完成事件后的关系变化。
- 事业线或感情线持续推进。

## 断档风险

- 男频无 CP 双男主却按单主爽文卖。
- 不扫对应平台榜单。
- 两个主角功能重复。
- 没有共同目标，只是并排出场。

## 训练任务

写一张双男主对照表：A 和 B 的规则、底线、解决问题方式必须完全不同，再用一个共同目标把他们绑进同一事件。
