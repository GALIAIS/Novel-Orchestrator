# 同人改写-原著偏离后自造

依据案例：`batch-066` 案例 329。

## 读者承诺

这个模块用于名著同人、影视同人、无限流副本改写。它承诺的是前期借原著框架降低门槛，后期承认原剧情被主角改到面目全非后，必须自造合理新剧情。

## 核心循环

```text
原著熟悉感 -> 主角介入 -> 改写旧事件 -> 连锁偏离 -> 自造新剧情 -> 维持原著合理性
```

## 开篇必备

- 读者熟悉原著事件。
- 主角介入点明确。
- 改写结果要符合原著人物逻辑。
- 每次改写都记录后果。
- 后期要有脱离原著后的新主线。

## 期待来源

- 熟悉事件被改写。
- 原著遗憾被补偿。
- 人物关系产生新走向。
- 主角影响越来越大。
- 后期新剧情仍然合理。

## 断档风险

- 以为同人只需稍改原著。
- 随便加新人物新事件，读者不买账。
- 改写旧事件不考虑后续连锁。
- 力量体系越写越乱。
- 后期原剧情偏离后不会自造。

## 训练任务

选一个原著事件，写主角介入后的结果。连续推演三个后果，并标出哪条原著线从此不能照旧写。
