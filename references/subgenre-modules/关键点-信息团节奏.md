# 关键点-信息团节奏

依据案例：`batch-005` 案例 024。

## 读者承诺

用关键点和信息团控制节奏，让读者脑中持续出现有戏剧性的核心信息。

## 核心循环

```text
核心信息 -> 戏剧性表达 -> 信息团 -> 关键点 -> 期待/情绪/矛盾同步升级
```

## 开篇必备

- 每章有明确关键点。
- 每个信息团都有核心信息。
- 核心信息必须对目标读者有戏剧性。
- 关键点之间有序列关系。

## 期待来源

- 读者不断接收到有价值的戏剧性信息。
- 期待感、情绪、矛盾同步递进。
- 信息团可以跨场景，也可以在一个场景里连续出现。

## 断档风险

- 信息很多，但核心信息不戏剧。
- 戏剧性不是目标读者想看的。
- 只按字数机械放期待点。
- 情绪升级和矛盾升级脱节。

## 训练任务

拆一章正文，标出所有信息团。每个信息团写一句核心信息，删掉无戏剧性且不服务关键点的内容。
