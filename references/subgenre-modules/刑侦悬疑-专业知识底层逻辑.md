# 刑侦悬疑-专业知识底层逻辑

依据案例：`batch-107` 案例 531。

## 读者承诺

刑侦悬疑的吸引力来自“真实流程下破案”。读者要看到案件不是靠主角灵光一闪解决，而是在程序、证据、权限和技术边界中一步步逼近真相。

## 核心循环

```text
案件出现 -> 进入流程 -> 证据受限 -> 研判方向 -> 新线索突破 -> 更深真相
```

## 开篇必备

- 明确案件类型和第一现场。
- 给出一个真实流程限制。
- 主角获得第一条可验证线索。
- 让读者知道现有证据不能直接破案。
- 专业细节服务悬念，不堆术语。

## 期待来源

- 证据链逐步补全。
- 程序限制制造阻碍。
- 错误研判被推翻。
- 专业动作带来可信突破。

## 断档风险

- 把刑侦写成玄学直觉。
- 只有尸体和凶手，没有侦查流程。
- 专业名词堆砌但不影响剧情。
- 主角权限无限，导致案件没有阻力。

## 训练任务

拆一个刑侦科普资料，把 5 个真实流程节点改写成 5 个剧情阻碍。每个阻碍后必须配一个可推进的新证据。
