# 爽文毒点-主角背义与中后期疲劳

依据案例：`batch-222` 案例 1110。

## 读者承诺

爽文主角可以做艰难权衡，但不能撕毁读者已经投入的人设、三观和情义账。

## 核心循环

```text
关系投入 -> 战略分歧 -> 主角选择 -> 三观校验 -> 情绪保全/断裂
```

## 开篇必备

- 先确认主角底线和情义观。
- 盟友关系一旦建立，就要记录读者投入点。
- 牺牲关系必须有不可逆冲突、巨大代价或后手补偿。
- 升级流中期要推新规则，不只换更强敌人。

## 期待来源

读者期待主角强，也期待主角值得追随；能力爽和三观信任缺一不可。

## 断档风险

- 主角为自保背弃盟友。
- 用“符合大局”压过读者恶心感。
- 盟友关系没有情绪清算。
- 中后期只重复升级打怪，设定不再推新。

## 训练任务

把主角杀盟友保全自己改成 3 个低毒版本：盟友先背弃底线、主角被迫二选一且付出巨大代价、主角表面背叛实则保留后手；再设计一个升级流中期新规则。
