# 公开处刑-开篇主角期待校准

依据案例：`batch-163` 案例 811。

## 读者承诺

公开诊断开篇时，不只看节奏是否顺，还要看主角第一印象、路线模式和金手指期待是否成立。

## 核心循环

```text
开篇求评 -> 节奏检查 -> 主角期待检查 -> 路线模式校准 -> 金手指意义确认
```

## 开篇必备

- 第一章让读者知道主角是谁、要干什么。
- 主角弱可以，但要有补偿或成长承诺。
- 穿越者要体现穿越优势。
- 无敌文和成长流不能混成弱鸡穿弱鸡。

## 期待来源

读者期待主角的存在有意义：要么强得能爽，要么弱得有清晰成长路径。

## 断档风险

- 主角第一印象弱气、学生味重。
- 穿越设定没有实际优势。
- 书名简介和开篇类型错位。
- 设定前后矛盾。

## 训练任务

把开篇主角分别按无敌模式和成长模式重写 300 字，选择期待感更明确的一版。
