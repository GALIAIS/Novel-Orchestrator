# 观念理论-心理学调情绪

依据案例：`batch-132` 案例 656。

## 读者承诺

不靠背具体桥段调情绪，而是先找到读者已有观念，再设计情节触碰、违背或满足这个观念，让情绪能跨题材迁移。

## 核心循环

```text
读者观念 -> 情节触碰观念 -> 判断产生情绪 -> 角色行动放大情绪 -> 结果释放情绪
```

## 开篇必备

- 明确目标情绪。
- 明确读者默认观念。
- 给角色一个能触碰该观念的处境。
- 让事件有判断空间，不只是信息说明。
- 准备一个释放点。

## 期待来源

- 读者感觉“这事该这样/不该这样”。
- 角色行为顺应或挑战读者判断。
- 情绪不是作者宣布，而是读者自己下判断。
- 同一观念可替换到不同题材。

## 断档风险

- 只复刻故事套路，不知道情绪从哪里来。
- 观念太抽象，无法落到角色行动。
- 情节只讲道理，没有冲突和释放。
- 跨题材后机制失效。

## 训练任务

选一个常见爽点，写出读者观念、触发事件、情绪判断、释放方式，再把它迁移到另一个题材。
