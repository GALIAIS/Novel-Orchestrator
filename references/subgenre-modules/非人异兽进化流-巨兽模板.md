# 非人异兽进化流-巨兽模板

依据案例：`batch-132` 案例 659。

## 读者承诺

让主角保持非人异兽身份，在生态链中通过捕食、吞噬、融合、进化获得压迫感和成长爽点。新鲜感来自身体、感官和生态位变化。

## 核心循环

```text
低生态位求生 -> 捕食/吞噬 -> 获得能力 -> 体型血脉升级 -> 挑战更高生态位
```

## 开篇必备

- 主角不是换皮人类，要有非人身体限制。
- 给出生态链压力。
- 第一场捕食要直接改变能力。
- 明确下一阶段进化目标。
- 坚持不化形，至少前期不破坏卖点。

## 期待来源

- 非人视角的新鲜感。
- 捕食和进化的可见变化。
- 巨兽体型与能力压迫。
- 更强怪物带来的生态位目标。

## 断档风险

- 写成普通冒险，只是主角叫恐龙。
- 过早化人形。
- 不写感官、身体和捕食逻辑。
- 冷门设定没有套入成熟爽文循环。

## 训练任务

写 800 字开篇：幼年恐龙在异世界底层求生，通过第一次吞噬获得能力，并挂出下一阶段进化目标。全程不化人形。
