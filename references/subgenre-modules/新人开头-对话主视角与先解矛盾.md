# 新人开头-对话主视角与先解矛盾

依据案例：batch-358 案例 1786

## 读者承诺

新人开头要让读者先进入故事和主角选择。用对话、自语、日记等交流方式建立代入，主视角锁住主角，第一矛盾推进后再放回忆和设定。

## 核心循环

```text
交流入口 -> 主角选择 -> 第一矛盾推进 -> 正向反馈/危机结果 -> 再补设定
```

## 开篇必备

- 前段有交流痕迹。
- 主视角固定在主角。
- 配角信息服务主角。
- 第一个矛盾先被推进或解决。

## 期待来源

- 读者听见人物在交流。
- 主角做出选择并承担结果。
- 危机先动起来。
- 设定后置，阅读阻力低。

## 断档风险

- 纯叙述干巴巴。
- 配角心理视角乱飞。
- 危机临头先回忆五六百字。
- 自称个人风格但市场不买。

## 训练任务

把一个开头改成“前 200 字有对话或自语，前 500 字只跟主角，第一矛盾先推进”的版本。
