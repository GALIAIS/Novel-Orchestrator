# 无限流副本-类型基调一致

依据案例：`batch-051` 案例 254。

## 读者承诺

无限流可以换世界，但不能反复换掉主承诺；书名、简介和第一个副本筛来的读者，后续副本要继续满足他们的核心期待。

## 核心循环

```text
开篇定基调 -> 第一个副本筛读者 -> 后续副本延续主承诺 -> 风格变化有边界 -> 期待持续上行
```

## 开篇必备

- 先明确书名简介承诺。
- 第一个副本不要随便承诺后文做不到的风格。
- 后续副本可以换元素，但不能降级期待。
- 主世界和副本世界要有稳定连接。
- 每次换世界前检查会筛掉哪些读者。

## 期待来源

- 读者期待新世界，但更期待同一主爽点持续兑现。
- 风格变化带来新鲜感。
- 基调稳定带来安全感。
- 副本和主世界互相支撑，会形成长线追读。

## 断档风险

- 奇幻科幻期待突然降到低武，会造成期待下行。
- 主世界和副本割裂，会打断阅读惯性。
- 每个世界都换风格，读者越筛越少。
- 大神能扛的家庭矛盾和慢热，新人未必能扛。

## 训练任务

设计三个副本，每个副本题材不同，但写出同一个主承诺、同一种情绪上行方式，以及绝不能跨过的风格边界。
