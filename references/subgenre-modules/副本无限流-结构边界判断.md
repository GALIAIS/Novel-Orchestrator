# 副本无限流-结构边界判断

依据案例：batch-275 案例 1372

## 读者承诺

判断含副本作品是否是无限流，重点看故事推进是否依赖多世界/多副本循环，而不是只看有没有任务、规则或副本元素。

## 核心循环

```text
副本元素 -> 推进主体判断 -> 无限流/副本都市分流 -> 开篇承诺兑现
```

## 开篇必备

- 明确现实主线和副本主线哪个占主导。
- 若承诺无限流，前三章要有副本规则与任务压力。
- 若承诺都市异能，现实身份和组织关系要站住。

## 期待来源

无限流读者期待副本循环、规则破解、任务奖励；副本都市读者期待能力成长、现实关系和长期主线。

## 断档风险

- 标签承诺和前三章兑现不一致。
- 副本太少却吸引无限流读者。
- 现实主线太弱，都市异能读者没抓手。

## 训练任务

列前三个大事件，标出发生在现实还是副本、是否结算、是否推动长期主线，再判断题材归类。
