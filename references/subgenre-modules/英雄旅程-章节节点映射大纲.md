# 英雄旅程-章节节点映射大纲

依据案例：batch-383 案例 1913

## 读者承诺

结构书节点可以帮助新人起大纲，但每个节点必须翻译成人物压力、选择、代价和后果，并由正文样本核对。

## 核心循环

```text
理论节点 -> 人物压力 -> 卷章功能 -> 样本核对 -> 题材化改写
```

## 开篇必备

- 当前人物欲望。
- 结构节点承担的功能。
- 必须做出的选择。
- 选择造成的代价。

## 期待来源

- 空白大纲获得阶段方向。
- 每个节点推动人物变化。
- 样本提供可验证参照。
- 理论暴露功能缺口。

## 断档风险

- 把节点名称直接当剧情。
- 强迫一章对应一个节点。
- 先信模板再曲解样本。
- 照搬参照作品具体情节。

## 训练任务

选五个相邻结构节点，逐项填写欲望、障碍、选择、代价和后果，再用一部熟悉作品的正文证据核对。
