# 外链资料-不可访问证据处理

依据案例：batch-281 案例 1404

## 读者承诺

只有外链、无正文、无评论时，不凭链接猜内容。先记录访问状态和证据边界，等取得正文后再拆。

## 核心循环

```text
发现外链 -> 检查访问 -> 未取得正文 -> 标注限制 -> 暂不内容拆解
```

## 开篇必备

- 外链地址。
- 访问结果。
- 是否取得正文。
- 当前可用证据。

## 期待来源

系统可靠性来自不编造。缺证据的材料被明确标记，后续补证时再进入内容层。

## 断档风险

- 链接可打开就当内容已读。
- 根据标题猜文档内容。
- 用同作者其他帖子替代当前证据。
- 为凑模块强行拆解未知内容。

## 训练任务

整理 5 条外链资料，标注“已读正文/仅链接可见/不可访问/需登录”，只拆“已读正文”。
