# 开篇动机-卷入事件规避愚蠢

依据案例：`batch-123` 案例 614。

## 读者承诺

开篇危机可以下行，但不能因为主角犯蠢下行。想让主角获得收益，就让他被迫卷入合理事件，再靠主动选择和金手指破局。

## 核心循环

```text
主角有燃眉之急 -> 外部事件逼近 -> 主角被迫卷入 -> 合理选择行动 -> 金手指破局 -> 收益兑现
```

## 开篇必备

- 主角需求要合理。
- 危机触发不能靠主角蠢。
- 外部逼迫要明确。
- 主角选择要有脑子。
- 收益要服务当前困境。

## 期待来源

- 读者理解主角为什么必须行动。
- 被迫卷入保住合理性。
- 金手指破局带来爽点。
- 额外收获开启下一步。

## 断档风险

- 主角主动招惹远强于自己的危险。
- 下行由愚蠢行为导致。
- 只套套路，不找读者期待点。
- 收益和燃眉之急脱节。

## 训练任务

把一个“主角犯蠢惹祸”的桥段改为“主角被外部事件逼入局”。保留收益，但重写动机和触发。
