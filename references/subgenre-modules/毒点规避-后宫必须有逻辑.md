# 毒点规避-后宫必须有逻辑

依据案例：`batch-318` 案例 1589

## 读者承诺

后宫爽点可以成立，但必须有关系铺垫、选择理由和主角边界；无脑种马和他人越界会迅速触发毒感或 NTR 联想。

## 核心循环

```text
关系铺垫 -> 主角选择理由 -> 边界清楚 -> 情绪回收 -> 后宫爽点成立
```

## 开篇必备

- 女性角色不是单纯奖励或羞辱工具。
- 收人前有利益、情绪或命运绑定。
- 他人不能随意越过主角关系边界。
- 主角反应必须维护读者默认秩序。
- 重口味桥段要确认目标读者能接受。

## 期待来源

读者期待后宫带来占有感、情绪反馈和关系推进，而不是廉价调戏、侵犯或主角边界失守。

## 断档风险

- 无逻辑收人。
- 女性角色工具化。
- 他人触碰准女主，引发 NTR。
- 主角像种马，读者恶心弃书。
- 局部截图几句话就足以自毁。

## 训练任务

把“主角随手收美女”改成有逻辑版本：写清利益绑定、情绪铺垫、主角选择理由和他人不得越界的动作。
