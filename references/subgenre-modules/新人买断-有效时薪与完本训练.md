# 新人买断-有效时薪与完本训练

依据案例：`batch-397` 案例 1983

## 读者承诺

把第一份买断从单一千字价改成现金流、有效时薪、合同权利和完本训练的综合决策，建立可持续交付能力而不是灌水习惯。

## 核心循环

```text
名义报价 -> 核对有效计费和义务 -> 测稳定净字产能 -> 算月毛收入与有效时薪 -> 审查版权退出 -> 对照生活与训练目标决策
```

## 开篇必备

- 明确哪些字数真正计费。
- 奖励按可达成条件折算。
- 用数周中位产能而非单日峰值。
- 总工时包含构思、修改和沟通。
- 核对版权、独家、完本与违约。
- 写清本书承担的完本训练目标。

## 期待来源

新人获得稳定交付、编辑协作和完本证据，下一本能以真实产能和作品质量争取更好条件。

## 断档风险

- 直接用名义单价乘冲刺日更。
- 假定全勤和全部字数必然计费。
- 忽略税费、结算和拒收。
- 为短期收入训练灌水。
- 用练笔理由接受过宽版权。
- 高估自己能长期维持的更新量。

## 训练任务

连续两周记录合格净字数与全部工时，计算保守、中位、冲刺三档有效时薪；再逐项核对计费、奖励、版权和退出，缺失条款一律标记待确认。
