# 二次元轻小说-受众平台定位

依据案例：`batch-046` 案例 226。

## 读者承诺

二次元、轻小说、同人和平台频道要先分清。写法核心通常是人设优先、轻松语气、脑洞清奇，并匹配青少年或对应 IP 受众。

## 核心循环

```text
概念澄清 -> 扫榜样本 -> 受众识别 -> 人设钩子 -> 平台匹配
```

## 开篇必备

- 明确原创还是同人。
- 明确目标平台。
- 第一章给出人设钩子。
- 语言有轻松感和独特思维。

## 期待来源

- 读者被角色差异吸引。
- 脑洞有新鲜感。
- 同人 IP 提供进入门槛。
- 青少年向情绪能被准确满足。

## 断档风险

- 概念没分清就专攻。
- 只看教程不扫榜。
- 平台频道差异被忽略。
- 人设和文风不匹配目标受众。

## 训练任务

拆一本二次元/轻小说榜单样本：标出人设钩子、轻松语气、脑洞点、目标受众和前三章承诺。
