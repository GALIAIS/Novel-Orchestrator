# 终局主角-合法性防篡位

依据案例：`batch-141` 案例 704。

## 读者承诺

全书情感和期待押在主角身上，终局必须由主角完成关键兑现；更强者可以出现，但不能替代主角成为真正答案。

## 核心循环

```text
读者投资主角 -> 后期出现高位者 -> 高位者制造镜像压力 -> 主角完成最终选择 -> 证明不可替代
```

## 开篇必备

- 主角终局承诺。
- 新高位角色的功能。
- 提前伏笔。
- 主角不可替代的价值。
- 最终兑现点。

## 期待来源

- 读者想看主角完成长期承诺。
- 高位者反衬主角短板。
- 主角在选择、代价或价值上胜出。
- 终局不靠外人代打。

## 断档风险

- 结尾新增强者抢走兑现。
- 高位者证明主角不配。
- 新角色像续命产物。
- 读者长期投资被打碎。

## 训练任务

写 A/B 两版终局大纲：A 版让高位者篡位，B 版让高位者证明主角不可替代。对比最终兑现点由谁完成。
