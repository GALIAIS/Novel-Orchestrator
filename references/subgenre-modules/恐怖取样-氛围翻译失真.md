# 恐怖取样-氛围翻译失真

依据案例：`batch-058` 案例 287。

## 读者承诺

这个模块用于恐怖题材学习取样。它承诺的是避开翻译和文化差异导致的恐怖点失真，优先拆国内作品、国外影视、新怪谈规则和可转译的异常设定。

## 核心循环

```text
选样本 -> 判断氛围是否依赖原文 -> 判断恐怖点是否跨文化 -> 提取异常规则 -> 转成悬念/期待 -> 避开审核风险
```

## 开篇必备

- 不把外国恐怖小说原文氛围当作可直接迁移对象。
- 先判断目标读者是否能感到同样恐怖。
- 恐怖元素最好服务悬念和期待。
- SCP、新怪谈、规则怪谈要注意标签风险。
- 架空、游戏、副本背景更适合降低现实敏感度。

## 期待来源

- 读者看到异常规则后想知道真相。
- 怪物和环境制造不确定性。
- 恐怖不是终点，而是拉出问题。
- 规则背后的代价形成追读。
- 国内读者能无注释理解不安来源。

## 断档风险

- 学外国小说学到翻译腔。
- 恐怖点依赖宗教、血统或旧时代文化。
- 只写氛围，没有事件推进。
- 纯恐怖导致审核和商业风险同时升高。
- SCP 标签或现实灵异表达被平台限制。

## 训练任务

把一个外国恐怖桥段改成游戏副本：删掉需要文化注释的恐怖点，只保留异常规则、失败代价和主角必须解开的悬念。
