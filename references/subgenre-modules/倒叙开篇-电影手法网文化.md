# 倒叙开篇-电影手法网文化

依据案例：`batch-091` 案例 455。

## 读者承诺

电影式倒叙可以用，但网文必须快速解释、分段兑现、及时切回主线。长悬念不能让读者等几卷才知道为什么。

## 核心循环

```text
未来高能 -> 短闪回解释 -> 反向塑造悬念 -> 分段兑现 -> 切回主线 -> 结算预告
```

## 开篇必备

- 未来场面必须高能。
- 1000 字内给出部分原因。
- 闪回不能无限延长。
- 开头出现的对象要在回忆中反向塑造。
- 需要金手指、人设、剧情至少一项强吸引。

## 期待来源

- 主角为什么做出异常行为。
- 开头被杀/毁掉的人或物，过去为何重要。
- 未来结果和回忆真相如何反差。
- 主时间线何时回归。

## 断档风险

- 必死危机后从穿越第一天慢慢讲。
- 悬念结算跨度过长。
- 倒叙只有形式，没有反向塑造。
- 金手指和人设撑不住读者耐心。

## 训练任务

写一个网文化倒叙开篇：1000 字内解释部分原因，3000 字内切回主线，至少设置 3 个开头结果和回忆真相相反的反转点。
