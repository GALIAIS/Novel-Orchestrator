# 无敌救场-战力标尺与价值抬升

依据案例：`batch-395` 案例 1974

## 读者承诺

胜负已知仍保留强期待：用逐级战力标尺、主角合理缺席和不断扩大的附带损失蓄压，救场后再以力量使用方式抬升人物价值。

## 核心循环

```text
能力伏笔 -> 威胁击穿逐级标尺 -> 主角合理缺席 -> 配角抵抗增加信息与代价 -> 主角到场绝对反转 -> 胜后价值选择
```

## 开篇必备

- 读者早已相信主角具备压倒性能力。
- 主角缺席理由可信且有限时。
- 每层配角失败揭示不同敌人规则。
- 配角至少完成救人、拖延或情报贡献。
- 场内误判持续维持主角存在感。
- 救场后还有一项非战力选择。

## 期待来源

读者不担心主角能否赢，而是等待他何时抵达、到场前会失去什么，以及拥有绝对力量的人最终怎样对待弱者、荣誉与功劳。

## 断档风险

- 配角只排队受虐而不提供新信息。
- 为拖延强行让主角失联。
- 重复下行不增加风险。
- 主角到场后继续磨蹭兑现。
- 碾压摧毁全部配角尊严。
- 只复制现成角色外形和桥段。

## 训练任务

写三层标尺的救场戏，每层失败必须揭露一种新能力并产生贡献；主角用最短动作反转后，再安排一个不靠武力的选择，让读者对他产生第二次评价升级。
