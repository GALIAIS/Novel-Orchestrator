# 手弩短视频-多压抑后爆发

依据案例：`batch-081` 案例 404。

## 读者承诺

这个模块用于写短视频式危机反转和网文小高潮。它承诺用多次压抑把情绪压到低点，再用一次扣扳机式爆发集中兑现。

## 核心循环

```text
危机开局 -> 望远镜拉期待 -> 连续压抑 -> 至暗时刻 -> 转机 -> 扣扳机爆发
```

## 开篇必备

- 开头直接进入危机。
- 先露出最终爆发的影子。
- 至少三次压抑。
- 计划必须看起来几乎不可行。
- 爆发要一次性清算前面的压力。

## 期待来源

- 观众知道主角有可能反杀。
- 压力一层层加码。
- 队友牺牲或代价抬高情绪。
- 至暗时刻让爆发更爽。
- 最终动作回收开头望远镜。

## 断档风险

- 危机不够快。
- 压抑次数不够。
- 计划太容易成功。
- 转机没有代价。
- 爆发只解决表面问题，没有清算全部压力。

## 训练任务

写一个危机小高潮：100 字内给危机和望远镜，正文安排 3 次压抑，最后用一个动作完成爆发。标出每次压抑的数字。
