# 不公平伤害-现实新闻反派标签

依据案例：`batch-184` 案例 920。

## 读者承诺

现实新闻中的不公平情绪可以转化为反派标签、多重恶性 buff 和主角惩罚爽点。

## 核心循环

```text
现实不公 -> 标签化反派 -> 多重恶性叠加 -> 读者愤怒 -> 主角惩罚兑现
```

## 开篇必备

- 受害者要具体。
- 反派行为要可见。
- 标签必须落到行动上。
- 最好叠加身份反差和二次伤害。

## 期待来源

读者期待小说替现实里无法立刻伸张的公道，完成情绪兑现。

## 断档风险

- 只贴标签，不写恶行。
- 未核验新闻直接照搬。
- 单一渣点情绪不够。
- 惩罚力度不匹配读者火气。

## 训练任务

提炼一个现实不公平素材：写受害者、反派标签、三重恶性 buff、二次拱火角色和惩罚爽点。
