# 开篇结构-ABCDEF短矛盾转长期矛盾

依据案例：`batch-208` 案例 1039。

## 读者承诺

开篇可以用 ABCDEF 完成短冲突闭环，并在尾声把它升级为长期目标、长期敌人或长期约定。

## 核心循环

```text
A动作 -> B背景 -> C复杂化 -> D激化 -> E爆发解决 -> F尾声转长线
```

## 开篇必备

- A 用动作和现场矛盾入场。
- B 只补当前矛盾所需背景。
- C 让危机变复杂。
- D 让矛盾继续激化。
- E 给出有主角能动性的解决。
- F 留下长期承诺。

## 期待来源

读者先被短期冲突抓住，再被尾声中的长期矛盾、约定或目标拉住追读。

## 断档风险

- A 变成设定说明。
- B 背景过长压住矛盾。
- E 只是外力解决。
- F 没有生活变化或长线目标。

## 训练任务

用 ABCDEF 排一个 7 章开篇：前 6 章推进并解决短矛盾，第 7 章必须引出长期矛盾，并估算可支撑字数。
