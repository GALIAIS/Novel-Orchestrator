# 理论适用域-显性冲突与隐性阻碍映射

依据案例：`batch-401` 案例 2001

## 读者承诺

把写作公式当作可证伪的诊断模型：先声明分析尺度和适用域，再识别人物珍视状态的显性或隐性摩擦；真正不匹配时切换体验模型，不靠扩大概念维护万能解释。

## 核心循环

```text
声明问题与尺度 -> 标记目标、摩擦、选择、结果 -> 映射信息/时间/身份/关系/规则等隐性约束 -> 检查共享上下文 -> 不匹配时切换体验模型 -> 用读者反馈修正适用域
```

## 开篇必备

- 明确本次分析针对作品、卷、场景还是单次爽点。
- 写出目标读者已知与未知的信息。
- 将欲望落实为人物珍视的可观察状态。
- 把战斗外的时间、身份、关系与价值限制纳入检查。
- 标明该理论对阅读反应的具体预测。
- 预设能够推翻预测的反例证据。

## 期待来源

表面优势持续兑现，深层限制却迫使人物作出有成本选择；读者既获得类型承诺，又能看见每轮状态真实变化。

## 断档风险

- 用辱骂、权威或个案收入替代理论论证。
- 把所有叙事现象重新命名为冲突。
- 把阻碍窄化成战斗力更强的敌人。
- 忽略无敌角色在信息、关系和时间上的限制。
- 不区分共享上下文与信息缺失。
- 只会事后解释，无法提前预测。
- 遇到反例无限扩大概念边界。

## 训练任务

用同一张诊断表分析常规升级、无敌流和盘点/高光各一段；允许至少一段不适用，并为它改用奇观、陪伴、揭晓或模式识别模型，再设计陌生读者测试。
