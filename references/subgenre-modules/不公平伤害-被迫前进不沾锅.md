# 不公平伤害-被迫前进不沾锅

依据案例：`batch-328` 案例 1637

## 读者承诺

不公平伤害让主角本想安稳却被迫前进，从而给反击、变强和打爆敌人提供正当性。

## 核心循环

```text
只想安稳 -> 外界逼迫 -> 无法退让 -> 被迫反击 -> 主角不沾锅
```

## 开篇必备

- 先写清主角想安稳。
- 外部伤害必须不公平。
- 主角不是主动找茬。
- 反击前有无法退让的理由。
- 反击后保留“都是被逼的”证据。

## 期待来源

读者期待主角被逼到极限后合理爆发。

## 断档风险

- 主角主动惹事，却硬说被迫。
- 不公平伤害没有落到主角身上。
- 主角安稳诉求不清。
- 反击缺少正当性。

## 训练任务

写一段主角只想安稳但被逼反击的开卷剧情，结尾让读者觉得主角不沾锅。
