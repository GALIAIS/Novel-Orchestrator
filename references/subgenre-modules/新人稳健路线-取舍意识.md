# 新人稳健路线-取舍意识

依据案例：`batch-159` 案例 793。

## 读者承诺

帮助新人放弃“全都要”的创作幻觉，先选择最能保护读者承诺、最少犯错、最符合当前能力的写法。

## 核心循环

```text
识别主基调 -> 列出想加入元素 -> 判断冲突 -> 主动放弃 -> 保留长板 -> 稳定执行
```

## 开篇必备

- 明确本书最重要的读者承诺。
- 只放入服务主基调的元素。
- 高难技巧和人物成长不要同时堆满。
- 一旦基调确立，前三章不要突然换方向。

## 期待来源

读者期待一本书持续兑现同一种核心体验，而不是开头轻松、中段沉重、后期又强行改成另一种书。

## 断档风险

- 同时想写轻松、悲剧、权谋、成长、群像和宏大重编。
- 为了致敬原型而牺牲自己作品的主线。
- 半路觉得另一个元素更高级，于是改变基调。
- 新人误把复杂当成熟。

列出自己项目中最想加入的 5 个元素，只保留最服务主承诺的 2 个。剩下 3 个写明删除理由，不允许用“以后可能有用”保留。
