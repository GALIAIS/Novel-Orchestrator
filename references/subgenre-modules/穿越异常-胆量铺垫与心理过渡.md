# 穿越异常-胆量铺垫与心理过渡

依据案例：batch-308 案例 1538

## 读者承诺

穿越主角可以勇敢，但普通人遇到怪物和异常不能毫无心理反应。勇敢需要心理过渡，冷静需要经历支撑。

## 核心循环

```text
穿越遇怪 -> 本能恐惧 -> 无退路压力 -> 心理转折 -> 行动反击
```

## 开篇必备

- 先写正常恐惧或震惊。
- 给出无退路的明确原因。
- 用一句心理转折把怕转成行动。
- 若主角极冷静，提前给身份经历补丁。

## 期待来源

期待来自“怕但敢动”。读者既觉得主角像人，又觉得他有主角感。

## 断档风险

- 普通人穿越后完全不怕。
- 只有行动，没有心理过渡。
- 用“没退路”解释所有冷静反应。
- 主角长期懦弱，失去爽文入口。

## 训练任务

写两版遇怪开篇：普通人版写恐惧和转折，老兵版写经验判断和冷静反击。
