# 书娘设定-知识库差异化金手指

依据案例：`batch-113` 案例 564。

## 读者承诺

书娘世界的金手指必须让主角与普通人拉开差距。穿越者旧世界知识库应带来数量、稀有度或组合优势，而不是人人都能读书获得同样书娘。

## 核心循环

```text
世界规则 -> 普通人限制 -> 主角知识优势 -> 多书娘获取 -> 翻盘兑现
```

## 开篇必备

- 普通人获取书娘很难。
- 主角拥有旧世界知识。
- 第一场事件展示差距。
- 设定少量抛出。
- 用翻盘证明金手指有用。

## 期待来源

- 稀有书娘连续出现。
- 主角数量优势压过常人。
- 旧世界诗词典籍变资源。
- 读者期待下一本书娘化。

## 断档风险

- 人人都能得到书娘。
- 大篇幅讲设定。
- 幕后高人太早露面。
- 主角没遭遇不公平就被看中。

## 训练任务

写一个书娘开篇事件：普通人一生一只书娘，主角凭旧世界知识连续触发三只，并用其中一只翻盘。
