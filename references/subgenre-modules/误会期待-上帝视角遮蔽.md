# 误会期待-上帝视角遮蔽

依据案例：`batch-076` 案例 377。

## 读者承诺

这个模块用于身份错位、秘密恋情、敌我不知身份等误会线。它承诺让读者先知道真相，角色暂时不知道，通过差点揭穿和合理遮蔽制造期待。

## 核心循环

```text
读者知道 -> 角色不知道 -> 差点揭穿 -> 巧合遮蔽 -> 再次接近真相 -> 最终爆发
```

## 开篇必备

- 读者掌握的真相要清楚。
- 角色不知道真相的原因要合理。
- 每次遮蔽都要具体，不要纯作者硬拦。
- 最终揭露必须引爆冲突。
- 避开主角核心雷点。

## 期待来源

- 读者等角色发现真相。
- 真相越接近，紧张越强。
- 差点揭穿强化长线伏笔。
- 最终揭露带来冲突爆发。
- 身份规则或怪力乱神能替代现实伦理雷点。

## 断档风险

- 误会压太久，读者觉得拖。
- 巧合太硬，像作者不让揭穿。
- 误会内容踩中 NTR 等主角雷点。
- 配角误会占太多笔墨。
- 最终揭露没有足够后果。

## 训练任务

写一个三次差点揭穿的身份误会。每次都写遮蔽原因、读者知道什么、角色误判什么，最后写揭穿后谁会受伤、谁会行动。
