# 末世吃人流-标签叠加与金手指卖点

依据案例：`batch-215` 案例 1073。

## 读者承诺

杀人变强的末世设定要叠加清晰热门标签，让读者一眼知道冷酷生存、实力跃迁和规则碾压在哪里。

## 核心循环

```text
末世崩坏 -> 杀人变强 -> 重生信息差 -> 系统反馈/无敌压制 -> 标签放大入口
```

## 开篇必备

- 明确主金手指如何变强。
- 把末世残酷和杀伐选择绑定。
- 用重生、系统、无敌等标签增强识别度。
- 人设标签作为基础配置，不替代核心卖点。

## 期待来源

读者期待主角靠冷酷选择抢资源、避坑复仇、获得清晰反馈，并在残酷秩序里快速拉开差距。

## 断档风险

- 只追求金手指新奇，忽略读者识别。
- 杀伐果断停留在人设口号。
- 多金手指功能重复。
- 杀人变强脱离末世秩序，变成空噱头。

## 训练任务

设计 3 版末世吃人流卖点组合，每版写书名式卖点、升级反馈、前三章杀伐兑现和中后期续航问题。
