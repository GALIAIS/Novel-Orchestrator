# 秩序冲突-既得利益维护规则

依据案例：batch-282 案例 1410

## 读者承诺

势力冲突不必从抢钱抢地盘开始。让主角触碰既有秩序，维护秩序的既得利益者自然会出手。

## 核心循环

```text
主流秩序缺位 -> 地方势力补位 -> 秩序产生利益 -> 主角破坏规矩 -> 新旧秩序冲突
```

## 开篇必备

- 当地已有规矩。
- 维护规矩的人。
- 从规矩中受益的人。
- 主角破坏规矩的行为。

## 期待来源

读者看到的不只是利益争夺，而是谁有资格定义“规矩”。新势力崛起就是新秩序替换旧秩序。

## 断档风险

- 反派只会抢钱，动机浅。
- 忽略普通居民也可能需要秩序。
- 只写权力，不写维护秩序的责任。
- 主角破坏旧秩序后没有新方案。

## 训练任务

写一个地方势力冲突：主角做了一件好事，但破坏当地规矩，引发既得利益者和居民的不同反应。
