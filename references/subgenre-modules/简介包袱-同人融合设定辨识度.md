# 简介包袱-同人融合设定辨识度

依据案例：`batch-189` 案例 942。

## 读者承诺

同人或融合设定的简介包袱，要让读者一眼识别新设定和原设定的差别，并立刻产生爽点想象。

## 核心循环

```text
熟悉体系 -> 对话铺垫 -> 新名词揭晓 -> 辨识差异 -> 爽点期待
```

## 开篇必备

- 借用体系足够熟悉。
- 新增名词和原设定明显区分。
- 最后一句有包袱。
- 包袱能扩展后续能力和冲突。

## 期待来源

读者先被熟悉体系带入，再被新职业、新序列或新能力的反差吸引。

## 断档风险

- 新名词像原设定，读者发愣。
- 包袱只有笑点，没有后续剧情空间。
- 蹭同人体系但没有新规则。
- 现实语义强但爽点方向不清。

## 训练任务

写 3 个同人融合简介末句，分别揭示一个新序列、新职业或新能力，并说明它和原体系哪里不同。
