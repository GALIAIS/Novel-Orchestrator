# 敏感词-女性描写高压线

依据案例：batch-304 案例 1516

## 读者承诺

女性角色魅力要靠气质、动作、关系反馈写，不要以诱人和身体细写为目标踩审核线。

## 核心循环

```text
女角色登场 -> 写气质/动作 -> 他人反馈 -> 推动关系 -> 避开露骨身体描写
```

## 开篇必备

- 不直写敏感身体部位。
- 不强调荷尔蒙、诱人、露出。
- 用角色能力和关系张力承载魅力。
- 审核风险高时宁可少写。

## 期待来源

- 角色有魅力但不打断剧情。
- 关系互动产生痒点。
- 章节不过审风险降低。

## 断档风险

- 把“诱人”当描写目标。
- 露骨身材细写。
- 用别的平台尺度误判网文审核。

## 训练任务

把一段女性身材描写改成气质、动作、他人反应三版，不出现具体敏感部位。
