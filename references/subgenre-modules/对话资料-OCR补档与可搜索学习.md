# 对话资料-OCR补档与可搜索学习

依据案例：batch-379 案例 1891

## 读者承诺

对话资料的价值不在收藏，而在可搜索、可摘录、可回到正文局部改写。OCR 补档让作者能带着具体问题检索技法。

## 核心循环

```text
对话问题 -> OCR 检索 -> 摘出技法 -> 局部改写 -> 正文验证
```

## 开篇必备

- 当前对话症状。
- 明确检索关键词。
- 可对照的正文片段。
- 一次小范围改写。

## 期待来源

- 资料能快速定位问题。
- 改写后角色声音更清楚。
- 对话冲突和潜台词增强。
- 搜索结果能反复复用。

## 断档风险

- 只收藏不检索。
- 从头泛读但不改稿。
- 搜索词太空泛。
- 摘录技法不验证。

## 训练任务

选一段平淡对话，列出三个问题关键词，检索资料后只改 500 字以内，检查角色目标、潜台词和打断是否更有效。
