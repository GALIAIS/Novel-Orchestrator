# 十三要义-章纲到正文分镜

依据案例：`batch-012` 案例 058。

## 读者承诺

把章纲里的干信息团转成可读正文，靠的不是硬扩写，而是用故事卡包装内容，再用十三要义拆骨、填肉、排层次、落分镜。

## 核心循环

```text
干信息团 -> 故事卡包装 -> 骨肉拆分 -> 反应层次 -> 情绪折线检查 -> 分镜关键词 -> 正文
```

## 开篇必备

- 先确认本章要表达的内容，不直接把信息团当正文。
- 主角动机、对手动机、场景、时间、观众只做骨架。
- 前反应、后反应、收获、发酵、余温要重点展开。
- 每个层次都要有镜头和人物反应。

## 期待来源

- 读者先看到基础表现，再发现这只是测试或铺垫。
- 反应层层升级，制造“原来还没到真正高潮”的期待。
- 触发点在章末接住下一章目标。
- 镜头关键词降低正文空转，让动作、语言、神态、描述和旁白各有功能。

## 断档风险

- 只把章纲改写成说明文。
- 十三要义变成填空表，反而破坏情绪流动。
- 反应层次不足，只有作者说好，没有人物证明好。
- 章末没有触发，读者看完本章不知道追什么。

## 训练任务

将一段章纲拆成 4 个分镜层次，每层写明骨架信息、前后反应和关键词，最后补一个章末触发。
