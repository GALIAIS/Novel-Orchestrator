# 题材选择-阅读扫榜定方向

依据案例：`batch-117` 案例 583。

## 读者承诺

不知道写什么时，不靠空想选题材。先回到自己的阅读偏好，再去对应平台扫榜，用真实样本收敛方向。

## 核心循环

```text
题材迷茫 -> 回看阅读偏好 -> 平台扫榜 -> 记录重复元素 -> 选择可操作题材
```

## 开篇必备

- 从自己常看的题材开始。
- 从自己熟悉的平台扫榜。
- 记录榜单共性。
- 判断自己能不能写事件。
- 新人可优先考虑自带世界观/流量的方向。

## 期待来源

- 输入变多后题材判断更稳。
- 平台样本提供读者口味。
- 自身兴趣降低长期写作阻力。
- 同人等方向降低入门成本。

## 断档风险

- 凭空想题材。
- 跨平台乱抄榜。
- 只看热度不看自己能不能写。
- 没样本就开长篇。

## 训练任务

扫一个常看平台的同榜 20 本书，只记题材、主角身份、金手指、开篇事件。用这 20 条决定你的试写方向。
