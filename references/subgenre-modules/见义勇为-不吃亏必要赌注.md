# 见义勇为-不吃亏必要赌注

依据案例：`batch-098` 案例 487。

## 读者承诺

爽文可以写主角见义勇为，但不能写主角无脑吃亏。出手必须有必要赌注，过程要聪明，结果要让好人有好报、恶人有恶报。

## 核心循环

```text
发现不平 -> 建立不得不管 -> 主角计谋介入 -> 不吃亏解决 -> 被救者回馈 -> 新剧情打开
```

## 开篇必备

- 被救者与主线有关。
- 主角有能力或计谋，不是硬送。
- 读者能理解主角为什么管。
- 结果必须有收获或情绪补偿。

## 期待来源

- 现实中不敢做的事由主角潇洒完成。
- 弱者被救的正义爽感。
- 恶人立刻付出代价。
- 被救者带来线索、人情或资源。

## 断档风险

- 打不过硬上。
- 受伤吃亏却无收益。
- 只为证明主角善良，剧情不推进。
- 把爽文桥段写成现实憋屈新闻。

## 训练任务

写一个救人桥段，列出被救者价值、主角不得不管的理由、主角不吃亏的解决方法、救人后的超额收获。
