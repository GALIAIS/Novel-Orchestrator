# 主反派塑造-优秀对手抬高胜利

依据案例：`batch-255` 案例 1272。

## 读者承诺

主反派塑造模块让高潮对手有能力、有品格、有压迫感，从而抬高主角胜利的含金量。

## 核心循环

```text
前置误判 -> 对手优秀出场 -> 主角压力升级 -> 正面对决 -> 胜利盖章
```

## 开篇必备

- 区分炮灰和主要对手。
- 主要对手要有正面证据，不只靠恶。
- 让读者感到“这个人确实强”。
- 主角必须有不可替代优势。

## 期待来源

对手越优秀，主角赢下来的爱情、地位或资源越有分量，爽点也更扎实。

## 断档风险

- 高潮反派写成无脑炮灰。
- 对手优秀到主角没有胜利理由。
- 只靠女主台词硬选主角，前文没铺垫。
- 反派塑造抢走主角光环。

## 训练任务

写一个优秀竞争者：3 个正面行为、2 个压迫主角的能力、1 个主角不可替代优势、1 句胜利盖章台词。
