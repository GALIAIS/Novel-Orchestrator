# 反派洗白-死得其所赎罪

依据案例：batch-299 案例 1494

## 读者承诺

罪恶反派可以临终醒悟，但要通过救人而死回应旧罪，避免无成本洗白。

## 核心循环

```text
旧罪累积 -> 醒悟触发 -> 选择救人 -> 死亡付代价 -> 悲壮赎罪
```

## 开篇必备

- 旧罪具体且不可抹。
- 醒悟有触发。
- 死亡方式和旧罪相关。
- 主角不能轻飘飘放过恶人。

## 期待来源

- 读者看到反派承担不可逆代价。
- 救人行为让最后选择有价值。
- 死亡保留人物复杂性，也封住毒点。

## 断档风险

- 反派随便死，不回应罪行。
- 作者让主角直接原谅。
- 牺牲没有救人意义，只是机械退场。

## 训练任务

写一个反派死得其所的场景：救的人、救的方式、对应旧罪、主角态度各一句。
