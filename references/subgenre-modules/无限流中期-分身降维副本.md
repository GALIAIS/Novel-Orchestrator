# 无限流中期-分身降维副本

依据案例：`batch-173` 案例 861。

## 读者承诺

当无限流主角中期偏无敌时，可以用分身降维进入副本，但必须保留赌注、风险和新的装逼玩法。

## 核心循环

```text
本体过强 -> 分身受限 -> 赌注保压 -> 经验补偿 -> 花样装逼
```

## 开篇必备

- 解释本体为什么不能直接入场。
- 分身弱化要有明确规则。
- 失败要损失真实资源、权限、身份或人命。
- 分身要保留主角经验、意识或特殊权限。

## 期待来源

读者期待主角被限制后仍能靠判断、经验和规则理解赢，而不是继续粗暴碾压。

## 断档风险

- 分身只是作者强行削弱。
- 没有赌注，失败无所谓。
- 分身弱化后爽点消失。
- 本体随时救场，副本压力作废。

## 训练任务

设计一个分身副本：本体不能进，分身只有三成实力。写明限制、赌注、三次风险升级和一次靠规则理解的反杀。
