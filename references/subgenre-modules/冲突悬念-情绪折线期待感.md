# 冲突悬念-情绪折线期待感

依据案例：`batch-332` 案例 1660

## 读者承诺

故事场景必须有冲突和悬念；落到网文，就是情绪折线和期待感。

## 核心循环

```text
人物目标 -> 阻力出现 -> 结果未知 -> 情绪波动 -> 期待兑现
```

## 开篇必备

- 人物有目标。
- 场景有阻力。
- 结果暂不确定。
- 情绪有上行下行。
- 章末留待解决问题。

## 期待来源

读者因为冲突未解、结果未知而继续翻页。

## 断档风险

- 对话只交换信息，没有立场冲突。
- 场景没有阻力。
- 结果一眼确定。
- 情绪平直。
- 没有章末期待。

## 训练任务

检查一章正文，每场戏标出目标、阻力、未知结果、情绪变化；标不出来就补或删。
