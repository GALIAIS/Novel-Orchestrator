# 开篇元素审查-表格排查闭环

依据案例：`batch-111` 案例 555。

## 读者承诺

开篇是否合格不能只靠感觉，要靠元素审查。题材承诺、主角状态、当前目标、第一冲突、金手指、情绪钩子和毒点风险都要逐项检查。

## 核心循环

```text
开头完成 -> 填元素表 -> 找缺失项 -> 对照毒点 -> 修改前3000字 -> 复查
```

## 开篇必备

- 题材承诺清楚。
- 主角目标明确。
- 第一冲突出现。
- 金手指或能力展示有位置。
- 情绪钩子能让读者留下。

## 期待来源

- 开篇问题可定位。
- 缺失元素能补到具体字数内。
- 毒点和缺项一起排查。
- 改稿有闭环。

## 断档风险

- 只看表格不填。
- 填不出来却说后面再讲。
- 元素齐了但没有情绪。
- 修改后不复查。

## 训练任务

填一张开篇元素表。所有空项都必须写补救方案，且限定补到前 3000 字。
