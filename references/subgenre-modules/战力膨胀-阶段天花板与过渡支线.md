# 战力膨胀-阶段天花板与过渡支线

依据案例：`batch-345` 案例 1725

## 读者承诺

长篇必然面对战力和情绪膨胀，要用阶段天花板、支线过渡、换地图或维度变化控制失控。

## 核心循环

```text
主角变强 -> 阶段天花板 -> 支线延缓 -> 伏笔过渡 -> 换地图/换维度 -> 新天花板
```

## 开篇必备

- 每阶段有清晰上限。
- 越级边界不能乱。
- 换地图提前埋伏笔。
- 支线要有意义。
- 爽点不只靠数字变大。

## 期待来源

读者期待新挑战持续出现，但挑战可以来自规则、概念、情绪、支线和地图，而不只是更高战力数字。

## 断档风险

- 高一个大境界也随便打赢。
- 数值无限放大。
- 感动和高潮不断加码到麻木。
- 换地图突兀。
- 支线只水字数。

## 训练任务

设计 4 个阶段天花板，每阶段写越级边界、过渡支线、换地图伏笔和下一阶段战斗维度。
