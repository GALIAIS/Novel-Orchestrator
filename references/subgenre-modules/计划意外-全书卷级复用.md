# 计划意外-全书卷级复用

依据案例：batch-263 案例 1315

## 读者承诺

计划-意外可以作为整本书的卷级发动机。重复的是结构，不应重复计划内容和意外表皮。

## 核心循环

```text
制定计划 -> 执行计划 -> 意外打断 -> 调整策略 -> 阶段结果
```

## 开篇必备

- 本卷计划目标。
- 计划预期收益。
- 意外来源。
- 意外造成的新局势。
- 主角调整后的结果。

## 期待来源

读者期待看主角有准备地行动，再看意外如何逼出临场应对和成长。

## 断档风险

如果每卷意外只是换名阻碍，会显得似曾相识；意外不改变局势，只是拖延，就会水。

## 训练任务

写三卷计划-意外表，每卷的计划目标、意外来源、调整方式都必须不同。
