# 角色出场-被动搭话视角五感观察

依据案例：`batch-203` 案例 1012。

## 读者承诺

被搭话视角下，新角色出场应从打断注意力的感知细节开始，再用局部印象和对话补足性格。

## 核心循环

```text
注意力被打断 -> 声音/气味/脚步/局部 -> 抬头回应 -> 对话展示性格
```

## 开篇必备

- 明确视角人物正在做什么。
- 用一个感知细节打断。
- 只挑 1-2 个强局部印象。
- 让第一句对话体现人物特色。

## 期待来源

读者跟随视角人物从被打断到观察对方，逐步形成角色印象。

## 断档风险

- 被搭话时还长篇铺环境。
- 外貌标签堆砌，没有性格。
- 细节太多，破坏回应节奏。
- 出场顺序不符合视角习惯。

## 训练任务

写同一角色两版出场：主动观察版和被动搭话版，各不超过 500 字。
