# 期待三角-困境目标金手指

依据案例：`batch-006` 案例 030。

## 读者承诺

用困境制造缺口，用目标指向方向，用金手指或能力提供达成可能，让读者相信下一章值得点开。

## 核心循环

```text
困境 -> 情绪缺口 -> 目标 -> 达成可能性 -> 下一章期待
```

## 开篇必备

- 主角当前困境。
- 明确短期目标。
- 主角有某种达成可能性。
- 章末读者知道自己在等什么。

## 期待来源

- 困境让读者在意。
- 目标让读者知道方向。
- 金手指/能力/资源让读者相信能成。
- 未兑现部分驱动下一章。

## 断档风险

- 有困境没目标，只剩压抑。
- 有目标没困境，缺乏驱动力。
- 有目标没可能性，读者不信。
- 有情绪但没期待，读者爽完退出。

## 训练任务

检查前三章，每章填困境、目标、达成可能性、下一章期待。缺两项以上就重写核心事件。
