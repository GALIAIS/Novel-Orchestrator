# 资料获取-先读再纠结版本

依据案例：`batch-319` 案例 1594

## 读者承诺

写作资料拿到可读版本后就该进入阅读和转化，不要把购买渠道、纸质或 PDF 的选择变成学习拖延。

## 核心循环

```text
确定要读资料 -> 找到可读版本 -> 停止渠道内耗 -> 阅读做笔记 -> 转成作品修改动作
```

## 开篇必备

- 明确本次阅读要解决的问题。
- 选择一个立刻能读的版本。
- 纸质书和 PDF 按阅读习惯选择。
- 星球已有资源先用现成资源。
- 读后必须形成作品动作。

## 期待来源

作者期待通过《作家之旅》《故事策略》等资料解决结构、冲突、开篇、人物问题，而不是只完成资料收集。

## 断档风险

- 纠结渠道太久，不开始读。
- 收 PDF 代替学习。
- 不知道不同平台可能有不同译名。
- 读完没有笔记，也没有修改作品。

## 训练任务

限定 30 分钟选定一本写作书的阅读版本，并写下本次只解决的一个问题和读后要修改的一个作品位置。
