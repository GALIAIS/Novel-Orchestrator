# 写作睡眠-剧情卸载与固定作息

依据案例：batch-386 案例 1928

## 读者承诺

睡前灵感用最短记录卸载，床铺不继续承担构思任务，并以固定作息保护第二天产出。持续失眠不应自行用药硬扛。

## 核心循环

```text
剧情冒出 -> 最短记录 -> 明确明日入口 -> 停止扩写 -> 固定作息
```

## 开篇必备

- 一句核心桥段。
- 三个关键词。
- 明天第一步。
- 明确记录结束时间。

## 期待来源

- 灵感不会丢失。
- 未完成感得到卸载。
- 第二天能快速恢复创作。
- 睡眠和产出形成稳定循环。

## 断档风险

- 备忘录变成夜间长写。
- 把床铺长期当构思场所。
- 将持续失眠正常化。
- 凭群聊自行服用助眠产品。

## 训练任务

连续七天执行一分钟剧情卸载并记录作息；问题持续或明显影响白天功能时寻求专业帮助。
