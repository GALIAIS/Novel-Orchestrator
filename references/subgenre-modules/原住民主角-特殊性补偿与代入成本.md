# 原住民主角-特殊性补偿与代入成本

依据案例：batch-376 案例 1877

## 读者承诺

原住民主角可以写，但要补偿穿越自带的特殊性、代入感和信息差。前三章必须证明主角凭什么值得跟。

## 核心循环

```text
原住民身份 -> 缺少外来代入 -> 补强金手指/人设/身份冲突 -> 事件证明优势
```

## 开篇必备

- 主角特殊性。
- 读者代入点。
- 可替代穿越的信息差或能力。
- 第一冲突中的优势展示。

## 期待来源

- 主角不是普通路人。
- 原住民身份能带来本土关系网。
- 强人设或强金手指补位。
- 事件中直接证明价值。

## 断档风险

- 只有土著身份，没有特殊性。
- 文笔不足以支撑慢代入。
- 读者吐槽没代入感。
- 编辑和市场更偏穿越重生。

## 训练任务

同一开篇写穿越者版和原住民版，比较两版代入来源、特殊性和第一冲突破局理由。
