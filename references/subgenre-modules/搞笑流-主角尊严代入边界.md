# 搞笑流-主角尊严代入边界

依据案例：`batch-177` 案例 883。

## 读者承诺

搞笑文可以轻松、荒诞、反差强，但必须保护主角基本尊严和代入感。

## 核心循环

```text
情境错位 -> 反差/误会/互动出笑点 -> 主角保有体面 -> 读者继续代入
```

## 开篇必备

- 笑点来自情境，不来自主角自我贬低。
- 主角可以翻车，但不能长期降智。
- 配角吐槽不能把主角打成小丑。
- 搞笑后仍要保留主角主动性。

## 期待来源

读者期待轻松有趣，同时仍愿意代入主角。主角有体面，笑点才不会伤害爽感。

## 断档风险

- 主角主动扮丑装傻。
- 笑点建立在主角尊严崩塌上。
- 长期降智导致读者不想代入。
- 只有热闹，没有角色魅力。

## 训练任务

写三个搞笑桥段，分别用误会、情境反差和配角吐槽制造笑点，要求主角不扮丑、不装傻、不失去主动性。
