# 心理描写-动机行动改变链条

依据案例：`batch-214` 案例 1069。

## 读者承诺

心理描写要服务动机、行动和变化，不能变成无节制灌水。

## 核心循环

```text
心理 -> 动机 -> 行动 -> 结果 -> 心理改变
```

## 开篇必备

- 行动和台词优先。
- 关键处用心理补足。
- 配角心理主要写变化。
- 只写能补爽点或期待感的心理。

## 期待来源

读者在关键位置理解角色为什么行动，以及行动后心理如何变化。

## 断档风险

- 每个人都大量内心独白。
- 心理不推动行动。
- 龙套心理过多。
- 心理戏成灌水。

## 训练任务

写一场支线戏的两版：行动台词版、加入 3 段心理版。逐段判断是否补爽点、期待或人物变化。
