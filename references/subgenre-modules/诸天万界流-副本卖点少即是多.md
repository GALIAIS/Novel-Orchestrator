# 诸天万界流-副本卖点少即是多

依据案例：`batch-206` 案例 1026。

## 读者承诺

诸天万界流每个世界只抓一两个核心卖点，围绕原著大事件切入改变，而不是事事复述。

## 核心循环

```text
原著世界 -> 选核心卖点 -> 切大事件 -> 主角改变 -> 副本收益反哺主世界
```

## 开篇必备

- 明确副本世界的原著大事件。
- 每个副本只选 1-2 个卖点。
- 主角必须制造新变化。
- 原著角色反应不能崩。

## 期待来源

读者期待熟悉剧情被主角改变，并看到副世界收益带回主世界形成成长。

## 断档风险

- 照抄原著，读者不如看原作。
- 偏离太远，读者觉得没入主题。
- 副本卖点太多，大纲散乱。
- 原著人设反应失真。

## 训练任务

选一个 IP，列 3 个大事件；每个只写一个改变点、主角切入方式、原著角色反应和副本收益。
