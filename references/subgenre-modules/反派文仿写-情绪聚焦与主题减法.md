# 反派文仿写-情绪聚焦与主题减法

依据案例：`batch-230` 案例 1146。

## 读者承诺

反派文开头要先选定一个强情绪主轴，笔力不足时少塞元素，避免生死危机、女角色、金手指互相抢注意力。

## 核心循环

```text
识别原文主轴 -> 选择仿写主轴 -> 删除分散元素 -> 强化第一情绪 -> 再展开支线
```

## 开篇必备

- 第一千字只有一个主卖点。
- 生死危机和金手指能互相承接。
- 女角色若不是主轴，先延后展开。
- 支线先概括，不抢压迫感。

## 期待来源

读者进入开头后迅速抓住一个强情绪：爽、压抑、情感或女人，而不是被多个半成品卖点分散。

## 断档风险

- 把工作室反馈的多个方向同时塞进开头。
- 生死关头还描写女徒弟漂亮，削弱危机。
- 仿写只换皮元素，没有学聚焦方式。
- 金手指提前抢走危机压力。

## 训练任务

给仿写开头逐段标注服务的主轴，只保留一个主轴相关段落。其余内容压成一句或移到第三章以后。
