# 开篇自审-题材单元十问

依据案例：`batch-010` 案例 046。

## 读者承诺

开篇不是填满要素，而是在第一个单元故事里准确兑现题材承诺。自审的目的，是防止作者写偏题材、写成填空、写成自嗨。

## 核心循环

```text
题材定位 -> 对标拆样 -> 第一个单元 -> 十问反查 -> 修正开篇
```

## 开篇必备

- 先确定主对标题材。
- 先确定第一个单元故事范围。
- 审核题材方向是否偏。
- 删除填空式要素。

## 期待来源

- 读者一开始就知道这是什么书。
- 第一个单元能展示核心卖点。
- 情绪和爽点贴合目标读者。
- 作者知道自己不是在写万能模板。

## 断档风险

- 死卡黄金三章。
- 把十问写成十个填空项。
- 不拆对标，自己臆造开篇尺度。
- 抽象要素齐了，但题材承诺错了。

## 训练任务

拆主对标第一个单元故事，记录字数、事件、卖点、情绪兑现。再按同一尺度反查自己的开篇。
