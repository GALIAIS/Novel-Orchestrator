# 短剧爽文-场景台词人设拉情绪

依据案例：`batch-078` 案例 389。

## 读者承诺

这个模块用于龙王赘婿、歪嘴战神式短剧爽点。它承诺用场景、台词、背景和人设的高密度配置，快速完成压低、误判、反转、震惊。

## 核心循环

```text
低位入场 -> 台词羞辱 -> 权力压制 -> 身份反转 -> 众人震惊 -> 情绪兑现
```

## 开篇必备

- 场景能立刻看出阶层差。
- 主角表面低位。
- 反派压制台词直接。
- 旁观者见证。
- 反转触发物明确。

## 期待来源

- 被低估者真实身份。
- 反派误判越深，反转越爽。
- 旁观者态度翻转。
- 台词压制积累情绪。
- 短平快节奏不留废笔。

## 断档风险

- 只学浮夸，不学配置。
- 羞辱过度导致毒。
- 反转触发物太硬。
- 场景不支持权力差。
- 逻辑弱到情绪也撑不住。

## 训练任务

设计一个短剧打脸桥段：写场景、羞辱台词、误判原因、反转证据、三类旁观者反应。控制在 1500 字内。
