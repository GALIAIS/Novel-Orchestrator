# 现实新闻-不公平伤害开篇

依据案例：`batch-155` 案例 773。

## 读者承诺

现实新闻里天然有读者秒懂的不公平伤害，可转化为开篇的愤怒、站队和反击期待。

## 核心循环

```text
主角付出 -> 强者承诺 -> 主角完成成果 -> 强者翻脸夺利 -> 证据反杀
```

## 开篇必备

- 主角长期付出。
- 对方明确承诺。
- 主角完成的成果。
- 对方翻脸动作。
- 主角损失。
- 可反杀的证据。

## 期待来源

读者期待主角拿回被夺走的成果，让背信者付出代价。

## 断档风险

- 只写主角惨，不写主角付出。
- 没有明确承诺，伤害不够不公平。
- 对方翻脸太抽象。
- 没有证据或反击抓手。

## 训练任务

找一条现实纠纷新闻，改成 1500 字开篇，必须包含付出、承诺、翻脸、损失和证据。
