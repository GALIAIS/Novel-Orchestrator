# 开篇危机-外显冲突走钢丝

依据案例：`batch-130` 案例 647。

## 读者承诺

开篇危机不能只让主角在内心痛苦和机关选择里打转。时间限制、当前目标、金手指使用和外部敌人正面冲突要尽快外显，让读者看到主角在走钢丝。

## 核心循环

```text
时间限制 -> 当前小目标 -> 外部阻碍压迫 -> 金手指首次有效使用 -> 正面冲突 -> 爽点/逃生进展
```

## 开篇必备

- 明确时间限制。
- 明确当前小目标。
- 第一困难由金手指参与解决。
- 第一章有外部冲突。
- 世界观说明尽量后移。

## 期待来源

- 读者知道主角要马上做什么。
- 外部敌人带来动态压迫。
- 金手指展示用途和潜力。
- 断章前情绪被打出去。

## 断档风险

- 纯内心斗争太久。
- 主角只和机关较劲。
- 外面敌人出现却不交锋。
- 设定名词破坏氛围。
- 压抑时间过长，爽点不来。

## 训练任务

把一个封闭空间自救开篇改成小纲：第一章内出现外部敌人、金手指第一次有效使用、主角主动行动和一次正面冲突。
