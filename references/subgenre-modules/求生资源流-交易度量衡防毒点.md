# 求生资源流-交易度量衡防毒点

依据案例：batch-267 案例 1335

## 读者承诺

在全民求生、末世资源、无限物资等题材中，维持交易尺度和主角尊严，避免因为随手给资源或负面关系绑定导致读者判定主角掉价。

## 核心循环

```text
资源体系 -> 统一估价 -> 交易对象筛选 -> 条件交换 -> 主角不吃亏 -> 稀有物收益兑现
```

## 开篇必备

- 先建立基础物资和稀有物的价值尺度。
- 每次交易都要符合前文价格区间。
- 无法估价时用附加条件、分期、试用，不要随便豪掷。
- 与主角关联的女性人设要避开福利姬/公交等负面联想。

## 期待来源

期待来自主角掌控资源市场：他可以大方，但必须显得精明、有规则、有主动权。

## 断档风险

- 无限物资导致作者忽视价值体系。
- 给价忽高忽低，读者觉得规则崩。
- 主角被负面女性占便宜，尊严受损。
- 推流期低星放大毒点，后续难救。

## 训练任务

写一张资源流价格表，再写一个稀有物交易场景。主角必须先估价，再压价或设置附加条件，不能直接豪掷物资。
