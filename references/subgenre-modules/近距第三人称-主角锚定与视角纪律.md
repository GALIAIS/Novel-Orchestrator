# 近距第三人称-主角锚定与视角纪律

依据案例：`batch-399` 案例 1993

## 读者承诺

以第三人称称谓保持叙述清晰，同时把感知、记忆、推断和心理限制在当前焦点人物内，让读者共享主角的问题和选择。

## 核心循环

```text
确认焦点人物 -> 当前需求建立信息边界 -> 观察记忆推断引入世界 -> 心理判断推动动作 -> 结果修正认知 -> 明确边界后切换视角
```

## 开篇必备

- 尽快确认跟随谁和他在意什么。
- 外部镜头明确指向主角主线。
- 主角未知结论不进入近距叙述。
- 人名可读、可区分并尽快稳定。
- 心理活动只属于当前视角人物。
- 视角切换有分段和新锚点。

## 期待来源

读者与主角共享有限信息、形成判断并通过行动验证，因认知变化而自然追问下一步。

## 断档风险

- 宏观开场迟迟不落到主角。
- 一段内无提示进入多人内心。
- 作者结论伪装成主角感知。
- 同一人物频繁更换代称。
- 内心独白替代外部行动。
- 把某种人名和人称当唯一规则。

## 训练任务

逐句标记一章信息属于观察、记忆、推断还是作者全知，删除焦点人物不可能知道的部分；统一人物称呼，并给每次视角切换补明确新锚点。
