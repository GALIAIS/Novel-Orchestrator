# 转行写作-全职风险试错

依据案例：`batch-090` 案例 447。

## 读者承诺

没出成绩前不要轻易全职写网文。先保生存底线，用兼职写作验证能力，再考虑文案、编剧、公众号或全职写作转向。

## 核心循环

```text
职业痛苦 -> 检查存款和压力 -> 找糊口工作 -> 兼职写作试错 -> 迁移文案能力 -> 达标再全职
```

## 开篇必备

- 存款能支撑至少一年。
- 家庭压力低。
- 写作本身有快乐。
- 有实际作品或收入证明。
- 有止损时间。

## 期待来源

- 写作能力能迁移到文案表达。
- 兼职试错成本可控。
- 作品数据逐渐证明可行。
- 转行不是裸奔。

## 断档风险

- 把不喜欢本专业等同于适合写网文。
- 没成绩就全职。
- 低估月入五千难度。
- 把爱看、想写、能赚钱混为一谈。

## 训练任务

做一张 12 个月试错表：存款、生活费、工作收入、写作字数、投稿结果、最低收入线、止损时间。
