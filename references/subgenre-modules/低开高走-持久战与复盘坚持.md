# 低开高走-持久战与复盘坚持

依据案例：`batch-240` 案例 1196。

## 读者承诺

低开不等于死刑，但坚持必须配合复盘；持久战是长期前提，不是万能理由。

## 核心循环

```text
低数据开局 -> 阵痛期动摇 -> 找可比样本 -> 继续更新观察 -> 按反馈修正
```

## 开篇必备

- 记录当前数据。
- 找可比低开样本。
- 明确还值得继续的证据。
- 写出接下来要改的问题。

## 期待来源

作者从短期焦虑回到长期视角，继续写，同时知道自己要修哪里。

## 断档风险

- 把个例当规律。
- 只喊坚持不改问题。
- 低开后完全否定自己。
- 没有观察节点和止损条件。

## 训练任务

做一张持久战表：当前数据、继续理由、需修问题、观察节点、止损条件，每项都写具体。
