# 同人提前线-人设世界观取舍

依据案例：`batch-075` 案例 374。

## 读者承诺

这个模块用于同人文提前原著时间线的开局判断。它承诺先判定原作核心吸引来自人设还是世界观，再决定是否能远离原主角时代。

## 核心循环

```text
识别原作承诺 -> 判断人设/世界观权重 -> 选择时间线 -> 设置熟悉锚点 -> 提供新事件
```

## 开篇必备

- 明确读者为什么爱原作。
- 提前时间线要有熟悉锚点。
- 原著角色出现必须有功能。
- 不出现主角团时要给补偿卖点。
- 开篇要立刻证明“这仍是那个世界”。

## 期待来源

- 熟悉人物的新命运。
- 原作未展开时代的新鲜感。
- 世界观规则被重新利用。
- 读者知道未来大势，等待主角改变。
- 原著名场面被提前铺垫或改写。

## 断档风险

- 原作靠主角团人设，却长时间不出现主角团。
- 只借壳世界观，读者认不出原味。
- 提前线没有新事件，只是拖到原著开始。
- 熟悉角色只是客串，不承担期待。
- 照搬原著或过度远离原著，两头不讨好。

## 训练任务

给一个同人题材做提前线评估。写出原作吸引权重、提前时间、熟悉锚点、新事件、主角团缺席补偿，以及前三章必须兑现的原作味。
