# 异世开局-身份金手指期待补全

依据案例：`batch-182` 案例 908。

## 读者承诺

异世第一章必须让读者知道主角是谁、从哪里来、在世界中处于什么位置，以及凭什么值得期待。

## 核心循环

```text
异世场景 -> 主角身份锚点 -> 当前处境 -> 金手指/期待源 -> 追读问题
```

## 开篇必备

- 时间、地点、人物只是底线。
- 主角来源要清楚：穿越者或原住民。
- 社会身份和当前处境要具体。
- 金手指或期待源要尽早出现。

## 期待来源

读者先代入主角身份，再根据金手指和困境判断他如何破局。

## 断档风险

- 只有环境，没有主角锚点。
- 只写危险，不写能力期待。
- 金手指出场过慢。
- 读者看不出主角在世界秩序中的位置。

## 训练任务

检查异世第一章：缺主角来源、社会身份、金手指、第一矛盾任意两项，就重排开篇信息。
