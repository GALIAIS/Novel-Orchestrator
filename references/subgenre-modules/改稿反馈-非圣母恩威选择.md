# 改稿反馈-非圣母恩威选择

依据案例：`batch-105` 案例 523。

## 读者承诺

当主角想保留善意时，先让作恶者付代价，再给可救者选择。善意必须有边界，宽恕不能跳过清算。

## 核心循环

```text
极端伤害 -> 救援到场 -> 作恶者清算 -> 可救者区分 -> 给选择 -> 继续主线
```

## 开篇必备

- 明确谁直接作恶。
- 明确谁仍可救。
- 惩罚先发生。
- 选择带威慑。
- 善意不等于无条件原谅。

## 期待来源

- 坏人付代价。
- 受害者获得主动权。
- 善意有边界。
- 后续经营线不膈应。

## 断档风险

- 直接放过伤害者。
- 用“善良”跳过清算。
- 全体村民被同质化。
- 读者情绪没被平衡。

## 训练任务

把一个原谅桥段改成三步：清算、区分、选择。每一步都写一句执行动作。
