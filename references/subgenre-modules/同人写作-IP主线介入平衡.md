# 同人写作-IP主线介入平衡

依据案例：`batch-165` 案例 823。

## 读者承诺

在保留原作熟悉感的同时，让原创主角真正改变事件结果，避免写多了 OOC、写少了透明。

## 核心循环

```text
原作熟悉元素 -> 原创主角变量 -> 介入关键事件 -> 保留人物底层动机 -> 改写结果 -> 生成新主线
```

## 开篇必备

- 明确读者因哪个 IP 元素进入。
- 选定一个原作事件作为介入点。
- 原创主角要有不可替代功能。
- 原作人物的底层动机不能随便改。

## 期待来源

读者期待“我熟悉的世界出现新变量”，既能见到原作名场面，又能看到原创主角带来的不同结果。

## 断档风险

- 原创情节过重导致人物 OOC。
- 太贴原著导致主角旁观透明。
- 只蹭 IP 热度，不会构筑新矛盾。
- 无敌化后取消了事件压力。

## 训练任务

选一个原作事件，写出主角介入点、不能改的三条人物动机、可以改的一条结果，以及改写后衍生的新矛盾。
