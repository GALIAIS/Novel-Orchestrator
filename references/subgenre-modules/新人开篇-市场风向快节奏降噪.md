# 新人开篇-市场风向快节奏降噪

依据案例：`batch-190` 案例 946。

## 读者承诺

新人开篇先贴合平台市场，再用主角视角、情绪拉扯、金手指和冲突快速起势，少写废话。

## 核心循环

```text
市场风向 -> 开篇卖点 -> 主角压力 -> 冲突/金手指 -> 连续事件
```

## 开篇必备

- 明确目标平台。
- 第一章快速切主题。
- 主角视角和压力清楚。
- 有冲突、金手指或期待钩子。

## 期待来源

读者快速知道这本书卖什么、主角遇到什么、后面会爽在哪里。

## 断档风险

- 学大神慢热铺细节。
- 创意只追噱头，不看市场。
- 堆辞藻和背景。
- 一个事件结束后没有下一个故事。

## 训练任务

重写自己的第一章前 500 字，删掉不服务主角压力和期待的句子，并在前 300 字出现明确冲突。
