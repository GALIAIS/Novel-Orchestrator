# 写作续航-鸡汤读物行动化

依据案例：`batch-174` 案例 868。

## 读者承诺

写作鸡汤读物提供情绪续航和身份认同，但必须转成具体行动，不能替代拆书、写作和反馈。

## 核心循环

```text
写作疲惫 -> 鸡汤读物 -> 情绪续航 -> 行动清单 -> 继续产出
```

## 开篇必备

- 区分情绪支持和技法支持。
- 读完鸡汤后立刻安排写作动作。
- 不把励志句子当商业方法。
- 续航材料要服务持续产出。

## 期待来源

作者期待获得继续写下去的动力。真正有效的续航，是读完之后能回到训练和作品。

## 断档风险

- 只读不写，短暂兴奋后归零。
- 把鸡汤误当技法教程。
- 情绪续航没有明确产出目标。
- 阅读材料越囤越多，作品没有增加。

## 训练任务

读一本写作鸡汤书 30 分钟，立刻写 7 天行动清单：每天字数、拆书章节、反馈位置和修改目标。
