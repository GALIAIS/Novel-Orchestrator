# 传统玄幻-不公平伤害与平台适配

依据案例：batch-357 案例 1783

## 读者承诺

传统玄幻仍可写，但不能照搬旧套路。它需要强情绪、清晰逆袭目标和平台适配；《斗破苍穹》式开篇核心不是慢快，而是不公平伤害连续酝酿后集中爆发。

## 核心循环

```text
无辜受损 -> 族人轻视 -> 退婚羞辱 -> 牵连亲友 -> 宣言爆发 -> 逆袭期待
```

## 开篇必备

- 主角无辜受损。
- 多层不公平伤害。
- 亲友被牵连。
- 一个集中爆发的宣言。

## 期待来源

- 读者替主角问“凭什么”。
- 三章情绪逐层叠高。
- 退婚/废材/亲情共同压迫。
- 爆发后自然期待逆袭。

## 断档风险

- 没变化没脑洞地照搬斗破。
- 情绪不到位，只剩旧设定。
- 平台读者不匹配。
- 没好剧本硬写传统玄幻。

## 训练任务

写一个传统玄幻前三章情绪递进表，每章只新增一层“不公平”，第三章给出一句主角宣言。
