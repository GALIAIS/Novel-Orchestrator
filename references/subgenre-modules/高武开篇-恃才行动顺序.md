# 高武开篇-恃才行动顺序

依据案例：`batch-022` 案例 106。

## 读者承诺

让高武主角的强势行动有凭据。主角提出要求、顺势反击之前，读者必须看到他凭什么敢这么做。

## 核心循环

```text
遇险 -> 看到金手指 -> 心里有所恃 -> 主动提要求 -> 不公平伤害 -> 装逼兑现
```

## 开篇必备

- 主角当前危险。
- 金手指预兆。
- 主角第二手准备。
- 反派对主角的不公平点名。

## 期待来源

- 读者知道主角不是乱莽。
- 反派看主角好欺负，制造反击需求。
- 主角的理智选择带来装逼期待。

## 断档风险

- 金手指先影响剧情，后面才解释。
- 反派动机一次塞太多人名。
- 主角一直被环境驱使。
- 因果链缺环，读者需要自己脑补。

## 训练任务

写一个高武开篇小纲，每个主角关键选择前都补一句“他凭什么敢”。缺凭据的行动删掉或前置铺垫。
