# 体内力量-自主权防夺舍

依据案例：batch-259 案例 1291

## 读者承诺

主角体内可以有远超当前层级的力量，但读者必须确认主角不是被别人替代的傀儡。爽点要落在主角的意志、选择、抵抗和后续掌控上。

## 核心循环

```text
危机压迫 -> 体内力量苏醒 -> 主角保留意识并干预 -> 失控换来胜利与代价 -> 主角追求真正掌控
```

## 开篇必备

- 主角当前弱点和危机。
- 体内力量出现的触发条件。
- 主角能保留的意识、选择或抵抗动作。
- 失控代价或夺舍风险。
- 后续掌控目标。

## 期待来源

读者期待主角从“借力活下来”走向“把力量变成自己的”。每次爆发都既要解决危机，也要抬高力量背后的秘密和掌控难度。

## 断档风险

如果主角昏迷、失忆、完全不知情，胜利就会被读者算到外挂身上；如果体内存在太独立且无制衡，就会变成夺舍毒点。

## 训练任务

写三版同一场爆种：完全被接管、半清醒抵抗、主动协商借力。比较哪一版最能让胜利归属主角。
