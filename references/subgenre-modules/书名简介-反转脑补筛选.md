# 书名简介-反转脑补筛选

依据案例：`batch-027` 案例 131。

## 读者承诺

书名简介要在读者点击前完成钩子、脑补和受众筛选。

## 核心循环

```text
意外反转 -> 脑补片段 -> 关键词筛选 -> 点击期待 -> 开篇兑现
```

## 开篇必备

- 一个反常问题。
- 一个可脑补场景。
- 明确类型关键词。
- 第一章对应兑现点。

## 期待来源

- 反转让读者提问。
- 小剧场让读者预演爽点。
- 标签筛掉非目标读者。
- 开篇迅速接矛盾。

## 断档风险

- 书名只有设定，没有问题。
- 简介只讲世界观，不给脑补片段。
- 关键词错配受众。
- 第一章不兑现书名钩子。

## 训练任务

给一个故事写三版简介：新媒体、脑洞爽文、女频小剧场。每版都标出反转、脑补片段和目标读者。
