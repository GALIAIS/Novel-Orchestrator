# 幕后系统-收益与人设一致

依据案例：`batch-019` 案例 094。

## 读者承诺

用“主角成为系统代理”制造幕后经营爽点，同时保证主角的收益逻辑、价值观和情绪表达一致。

## 核心循环

```text
现实压力 -> 代理系统权限 -> 发任务 -> 抽成变现 -> 升级商城 -> 现实打脸
```

## 开篇必备

- 主角现实困境。
- 代理系统能做什么。
- 幸存者为什么愿意接受任务。
- 主角如何从任务中获利。

## 期待来源

- 主角像系统一样操控末日幸存者。
- 幸存者劳动产出反哺现实。
- 商城差价、抽成、任务奖励带来经营爽点。
- 现实债务和聚会打脸形成双线兑现。

## 断档风险

- 主角突然共情到哭，情绪过快。
- 前面圣母，后面剥削，价值观打架。
- 催债、反派、同学会压力太弱。
- 幕后收益没有规则，像作者随手送钱。

## 训练任务

设计一个代理系统设定，写清三项：主角抽成比例、幸存者获得什么、主角对幸存者的底线。再写一个现实兑现桥段。
