# 扫榜疲劳-恢复阅读热情

依据案例：`batch-236` 案例 1178。

## 读者承诺

扫榜学习把热情耗光时，先停止功利阅读，找回能看下去、喜欢小说的情绪。

## 核心循环

```text
机械扫榜 -> 阅读麻木 -> 暂停学习任务 -> 找喜欢的书 -> 恢复热情
```

## 开篇必备

- 识别自己是否麻木。
- 暂停“必须学习”的阅读。
- 找一本真能看下去的书。
- 必要时短期远离网文。

## 期待来源

作者重新获得对故事的兴趣和创作燃料，再回到扫榜时不只是机械完成任务。

## 断档风险

- 强迫自己继续扫榜。
- 学习目标压垮兴趣。
- 每个念头都被问题清单浇灭。
- 把短期休息误判成放弃。

## 训练任务

停止功利扫榜 3 天，只读一本喜欢的书。每天记录“为什么愿意继续看”，不拆结构、不总结套路。
