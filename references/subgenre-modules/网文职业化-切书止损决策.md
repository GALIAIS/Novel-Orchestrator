# 网文职业化-切书止损决策

依据案例：`batch-197` 案例 982。

## 读者承诺

成绩不好可以切书，但切书应基于目标、收入、推荐反馈和下一本准备度，而不是情绪崩盘。

## 核心循环

```text
成绩不佳 -> 判断目标 -> 评估收益和反馈 -> 准备下一本 -> 止损切换
```

## 开篇必备

- 区分练笔目标和赚钱目标。
- 记录推荐轮次和上架反馈。
- 计算全勤或保底收益。
- 切前准备下一本题材。

## 期待来源

作者能把失败项目转成经验和下一本题材，而不是陷入反复裸切。

## 断档风险

- 没想好下一本就切。
- 为了全勤无限拖烂书。
- 新人练笔期过早否定自己。
- 商业目标明确却不止损。

## 训练任务

做一张切书判断表，列出继续写、扛到上架、吃完全勤、立刻切四种触发条件。
