# 开篇行动链-第一冲突同向与金手指实证

依据案例：batch-376 案例 1878

## 读者承诺

第一章铺垫出的主角最急目标，必须决定第二章第一冲突。金手指要在行动链上实证，而不是只亮相。

## 核心循环

```text
核心铺垫 -> 最急目标 -> 行动链 -> 反派阻碍 -> 金手指实证
```

## 开篇必备

- 主角最急目标。
- 目标对应行动。
- 反派挡路理由。
- 金手指当场效果。

## 期待来源

- 读者关心既有危机。
- 冲突正好阻碍主角目标。
- 金手指表现出书的差异。
- 强反派压出后续打脸期待。

## 断档风险

- 第二章换成无关英雄救美。
- 反派像弱智工具人。
- 金手指只介绍不使用。
- 新造概念太多，标签混乱。

## 训练任务

删掉不在主角第一目标行动链上的冲突，重写第一个反派阻碍，并让金手指实证一次。
