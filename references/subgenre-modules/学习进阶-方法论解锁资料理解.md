# 学习进阶-方法论解锁资料理解

依据案例：`batch-344` 案例 1720

## 读者承诺

系统方法和真实点评会让旧资料从看不懂变成可迁移，进步标志是能判断好坏并落到下一稿。

## 核心循环

```text
旧资料看不懂 -> 学方法 -> 被点评纠错 -> 建立框架 -> 回看资料 -> 转成作品动作
```

## 开篇必备

- 方法论先形成判断框架。
- 点评指出自身盲区。
- 资料要转成书名、简介、开篇或冲突。
- 眼界提高后还要落笔。
- 普通作者可先套结构。

## 期待来源

作者期待从迷茫变成知道哪里错、为什么错、下一稿怎么改。

## 断档风险

- 买资料但没有转化框架。
- 眼界提高后只觉得别人不过如此。
- 不整理点评。
- 只学技巧不落地。
- 和天才抓热点硬比。

## 训练任务

重读一份旧资料，提炼一个书名元素、一个开篇动作、一个冲突模板，并写进当前作品计划。
