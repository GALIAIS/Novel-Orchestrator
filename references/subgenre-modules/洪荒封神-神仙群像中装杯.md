# 洪荒封神-神仙群像中装杯

依据案例：batch-293 案例 1462

## 读者承诺

在神仙群像里，主角装杯靠信息差、因果差、站位和借势，而不是硬压所有强者。

## 核心循环

```text
大劫格局 -> 主角低位入局 -> 信息差 -> 借势布局 -> 小人物撬动大势
```

## 开篇必备

- 大劫或天命变化。
- 主角初始低位。
- 已知信息差。
- 强者限制。
- 第一场借势装杯。
- 后续因果代价。

## 期待来源

读者看到众神登场后，期待主角如何在强者规则缝隙里改变局面。

## 断档风险

- 主角被神仙群像淹没。
- 只靠口嗨装杯。
- 战力体系压不住强者却硬打。
- 只写神仙打架，不写王朝、天命和民生秩序。

## 训练任务

写一场封神式小装杯：主角用一个天命节点救下小人物，让旁观神仙第一次重视他。
