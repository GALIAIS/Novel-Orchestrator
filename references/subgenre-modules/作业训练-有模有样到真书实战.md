# 作业训练-有模有样到真书实战

依据案例：`batch-238` 案例 1189。

## 读者承诺

作业训练的终点不是作业区表现，而是把已练成的技法应用到真实作品，接受市场反馈。

## 核心循环

```text
做作业 -> 基础动作成型 -> 获得反馈 -> 迁移到正文 -> 用读者反馈验收
```

## 开篇必备

- 明确训练能力。
- 记录作业反馈。
- 选择可迁移章节。
- 设置读者反馈指标。

## 期待来源

作者感到自己确实进步，并把这种进步转化为能支撑连载开头的真实能力。

## 断档风险

- 永远停在练习模式。
- 作业写得好，正文不用。
- 把内部反馈当成市场反馈。
- 技法迁移后不检查读者承诺。

## 训练任务

把最近作业里最稳定的一个技法放进真实开头第一章，并用读者承诺、冲突、爽点三项复查。
