# 傀儡皇帝开头-身份处境与期待前置

依据案例：`batch-248` 案例 1238。

## 读者承诺

身份复杂的开头要先让读者明白主角是谁、被什么压住、接下来想做什么。

## 核心循环

```text
身份前置 -> 处境压迫 -> 金手指/信息差 -> 反抗目标 -> 第一次冲突
```

## 开篇必备

- 前一千字讲清主角身份。
- 给一句强冲突钩子。
- 明确当前处境。
- 尽快给期待感。

## 期待来源

读者知道主角是傀儡皇帝并看到反抗可能，才会继续吃铺垫。

## 断档风险

- 氛围多，信息少。
- 作者知道但读者不知道。
- 前几段无情绪起伏。
- 金手指和目标出现太晚。

## 训练任务

用 300 字重写复杂身份开头：第一段交代身份、压迫和反抗目标，第二段补金手指。
