# 主线暗线-底层矛盾串联

依据案例：`batch-170` 案例 846。

## 读者承诺

用明线提供每章可见目标，用暗线解释全书底层矛盾，让主角、配角、反派的行动最终回收到同一个核心秘密或核心物。

## 核心循环

```text
明线升级/任务 -> 暗线物反复牵动局势 -> 配角动机显影 -> 反派目标揭开 -> 终局合流兑现
```

## 开篇必备

- 主角要有清楚的明线目标。
- 暗线可以只露一个异常物、身份谜或历史缺口。
- 前期暗线不必帮主角，但要制造危机或解释他为何被盯上。
- 配角靠近主角的理由要能被暗线回收。

## 期待来源

读者先追明线爽点，后期期待暗线揭开时重新解释前面所有危机，并把终局高潮推到更大层级。

## 断档风险

- 暗线只藏不动，长期不影响剧情。
- 反派动机和暗线无关，终局像临时换题。
- 明线太散，读者不知道主角每卷在做什么。
- 暗线前期只当设定，不制造行动压力。

## 训练任务

设计一个暗线核心物，列出它如何影响 3 个配角、3 次反派行动、2 次主角危机和 1 次终局反转。
