# 御兽召唤-人形召唤物与污名开局

依据案例：`batch-219` 案例 1091。

## 读者承诺

人形召唤物要被包装成高等种族、战力功能和世界规则，而不是敏感噱头。

## 核心循环

```text
人形召唤 -> 世界误判 -> 主角污名 -> 能力证明 -> 召唤战绩
```

## 开篇必备

- 解释人形为什么代表高等种族。
- 主角边界干净，避免真变态感。
- 误会要尽快转成能力价值。
- 第一只召唤物要有明确战力或功能。

## 期待来源

读者期待外界误判被战绩反打，主角用稀有召唤能力证明自己。

## 断档风险

- “萝莉”“恋童癖”等词造成平台和读者风险。
- 召唤物只剩属性标签。
- 污名化拖太久，爽点不兑现。
- 世界规则解释不够，像作者恶趣味。

## 训练任务

把高风险召唤能力名改写成 3 个低风险版本，并设计第一只人形召唤物的功能、误判和反转战绩。
