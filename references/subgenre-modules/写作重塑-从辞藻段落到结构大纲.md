# 写作重塑-从辞藻段落到结构大纲

依据案例：`batch-232` 案例 1159。

## 读者承诺

新人要从局部辞藻和段落叙述，转向整体结构、法则运用和详细大纲能力。

## 核心循环

```text
局部文字强 -> 长篇失控 -> 学结构法则 -> 练详细大纲 -> 再写整本
```

## 开篇必备

- 承认结构能力缺口。
- 先练事件和大纲。
- 不用华丽辞藻遮住主线问题。
- 接受重组期痛苦。

## 期待来源

作者能持续制造期待、安排事件、控制长篇，而不是只写漂亮片段。

## 断档风险

- 把文笔当成全部能力。
- 学理论后不练事件设计。
- 一上来就写整本，继续失控。
- 重组期痛苦就放弃。

## 训练任务

7 天结构练习：每天设计 1 个事件，写目标、阻碍、选择、代价、结果、下一期待；第 7 天串成小大纲。
