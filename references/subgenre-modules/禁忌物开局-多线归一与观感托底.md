# 禁忌物开局-多线归一与观感托底

依据案例：`batch-230` 案例 1148。

## 读者承诺

禁忌物开局可以多线并发，但前三章必须让读者看见清晰因果和当前观感回报。

## 核心循环

```text
代价危机 -> 使用原因 -> 内幕线索 -> 当前回报 -> 多线归一
```

## 开篇必备

- 主角使用禁忌物有走投无路或误启理由。
- 代价和危机有因果线索。
- 敌人 A 不抢走敌人 B 的主线权重。
- 三章内给出爽点、感情或世界观回报。

## 期待来源

读者接受暂时下行和多线，是因为每章都知道主角为何受难、接下来要解决什么、当前卖点在哪里。

## 断档风险

- 多线后期才归一，前三章观感先散掉。
- 主角主动犯错却没有充分理由。
- 第五章才逆转，前四章缺少回报。
- 用结构自洽替代读者观感。

## 训练任务

写三章表：第一章代价危机，第二章使用原因，第三章观感回报。每章末尾写一句读者期待，三句不连贯就重排线索。
