# 多人对白-电影台词拆解训练

依据案例：`batch-176` 案例 876。

## 读者承诺

多人对白要让读者看见立场冲突、信息流动和权力变化，而不是角色轮流闲聊。

## 核心循环

```text
多人入场 -> 各自目的 -> 关键台词交锋 -> 信息揭露 -> 上风变化 -> 剧情推进
```

## 开篇必备

- 每个参与者都有自己的目的。
- 台词要符合身份、处境和心理。
- 关键台词承担信息量或象征意义。
- 镜头切换按冲突焦点走，不随意跳。

## 期待来源

读者期待谁会说破真相、谁被压住、谁露出破绽、哪一句话改变局面。会议、朝堂、答辩、谈判等场景都靠这条期待链成立。

## 断档风险

- 多人对白写成平均发言。
- 每句都符合人设，但没有推动剧情。
- 镜头视角乱跳，读者不知道该看谁。
- 缺少台词素材积累，关键对白写成说明书。

## 训练任务

拆一段商业电影多人争论戏，逐句标注人物目的、台词动机、信息量和上风变化，再仿写一个三人争论场景。
