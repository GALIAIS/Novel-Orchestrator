# 猫站签约-申请审核等待

依据案例：`batch-136` 案例 679。

## 读者承诺

刺猬猫签约流程先由作者点击申请，再等待编辑审核。不要把等待误解为统一安排，也不要臆造具体周期。

## 核心循环

```text
看到签约入口 -> 点击申请 -> 编辑审核 -> 等待后续签约
```

## 开篇必备

- 明确是否需要作者主动点击。
- 明确下一步是编辑审核。
- 提醒编辑人少可能慢。
- 不写未经证实的天数。
- 以后以后台和编辑通知为准。

## 期待来源

- 操作动作清楚。
- 等待对象清楚。
- 现实预期被管理。
- 减少重复询问。

## 断档风险

- 等统一安排而不点击。
- 以为点击后自动签约。
- 编造审核天数和通过率。
- 忽略平台流程变化。

## 训练任务

写一条 FAQ：猫站签约需要等统一安排吗？答案包含点击申请、等待编辑审核、编辑人少可能慢，禁止加入具体审核天数。
