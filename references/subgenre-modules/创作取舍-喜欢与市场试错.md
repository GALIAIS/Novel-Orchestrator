# 创作取舍-喜欢与市场试错

依据案例：`batch-094` 案例 468。

## 读者承诺

这个模块不是鼓励自嗨，也不是强迫跟风，而是把“喜欢”和“市场”变成可试错的创作决策。作者必须先判断现实收入压力，再决定是先写喜欢的校准成绩，还是先服从市场确保生存。

## 核心循环

```text
明确现实压力 -> 列出喜欢题材 -> 列出市场题材 -> 找交集改造 -> 写短开篇测试 -> 用成绩和写作状态复盘 -> 调整下一本
```

## 开篇必备

- 不能只说“我喜欢”，要写出读者能得到什么。
- 不能只说“市场热”，要确认自己能持续写。
- 若写市场文痛苦到效率崩溃，必须降低跟风强度。
- 若必须靠稿费生活，不能完全无视榜单和付费读者。

## 期待来源

- 作者真实兴趣带来的稳定表达欲。
- 市场题材提供的可识别卖点。
- 榜单样本验证读者需求。
- 失败成绩反向校准作者认知。

## 断档风险

- 纯喜欢但没有外部承诺，读者不知道追什么。
- 纯市场但作者强烈排斥，更新和完本崩掉。
- 学大神慢热，却忽视粉丝基础差异。
- 写一半才发现题材不适合自己，沉没成本过高。

## 训练任务

给同一个创意写两版 500 字开篇：一版按自己最喜欢的节奏，一版按榜单快节奏改造。写完分别评估“我愿不愿写 50 万字”和“读者第一章能不能知道爽点”。只保留两项都过线的版本。
