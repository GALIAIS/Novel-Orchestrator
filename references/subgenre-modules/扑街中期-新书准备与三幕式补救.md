# 扑街中期-新书准备与三幕式补救

依据案例：`batch-207` 案例 1034。

## 读者承诺

中期扑街时，老书操作空间有限；准备新书恢复动力，同时用三幕式练当前书的爽点高潮。

## 核心循环

```text
成绩低迷 -> 准备新书 -> 评估切书 -> 老书练剧情构建 -> 下本优化开局
```

## 开篇必备

- 判断当前字数和成绩趋势。
- 识别开书元素带来的受众上限。
- 准备新书题材设定大纲。
- 用三幕式组织老书剩余高潮。

## 期待来源

作者不再只靠全勤硬熬，而是把低迷期转成下一本的准备和剧情构建训练。

## 断档风险

- 三十五万字后幻想小修逆天改命。
- 不准备新书，只消耗动力。
- 忽略开书元素决定受众上限。
- 不练剧情构建能力。

## 训练任务

做双轨表：老书 3 个剧情构建训练点；新书题材、金手指、前三章验证点、切书条件。
