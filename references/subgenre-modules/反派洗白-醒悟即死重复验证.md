# 反派洗白-醒悟即死重复验证

依据案例：batch-300 案例 1500

## 读者承诺

反派旧罪越重，醒悟越不能当清账；高权重处理是醒悟后用死亡救人封住强洗毒点。

## 核心循环

```text
旧罪累积 -> 醒悟出现 -> 读者警惕强洗 -> 救人而死 -> 有限赎罪成立
```

## 开篇必备

- 旧罪具体。
- 醒悟有触发。
- 牺牲回应旧罪。
- 主角不轻易原谅。

## 期待来源

- 读者看见不可逆代价。
- 死亡阻断恶人无成本回归。
- 悲壮感来自最后选择。

## 断档风险

- 用操控或误会抹掉全部罪行。
- 让恶人活着享受原谅。
- 牺牲对象和旧罪无关。

## 训练任务

把一个活着洗白桥段改成醒悟即死：列旧罪、救人对象、死亡方式、主角态度。
