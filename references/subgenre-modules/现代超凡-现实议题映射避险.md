# 现代超凡-现实议题映射避险

依据案例：`batch-190` 案例 947。

## 读者承诺

现代超凡可以借现实议题制造新鲜感，但必须弱化政治、宗教、国际局势等直指现实的隐喻。

## 核心循环

```text
旧超凡体系 -> 现代命名 -> 检查现实指向 -> 抽象换皮 -> 保留能力内核
```

## 开篇必备

- 超凡体系有清楚原型。
- 现代命名不直接映射现实组织和事件。
- 敏感词先替换成抽象词。
- 能力规则比现实影射更重要。

## 期待来源

读者看到熟悉体系在现代语境下变形，但不会因为现实隐喻过强而出戏或触发风险。

## 断档风险

- 政治隐喻过强。
- 宗教、革命、国际局势直指现实。
- 脑洞强但编辑不过。
- 现实事件变化后反向触发下架风险。

## 训练任务

写 5 个现代超凡途径名，每个都给出高危直指版和抽象避险版，并删除能直接联想到现实事件的词。
