# 校园短剧-桥段素材转写

依据案例：`batch-084` 案例 417。

## 读者承诺

这个模块用于青春校园小说找素材。它承诺从短剧中拆出真实、轻松、可直接转写的桥段，而不是复述整部作品。

## 核心循环

```text
短剧观看 -> 桥段截取 -> 关系/场景标注 -> 轻松真实感提炼 -> 小说事件转写
```

## 开篇必备

- 选择青春校园短剧。
- 每次只拆一个桥段。
- 标注人物关系。
- 标注校园场景。
- 写明可转成的章节事件。

## 期待来源

- 素材真实。
- 氛围轻松。
- 桥段短，转写成本低。
- 能补校园日常细节。
- 可服务友情、误会和小成长。

## 断档风险

- 照搬剧情。
- 只记台词，不记关系功能。
- 桥段缺少小说事件。
- 素材太长，不好转写。
- 校园感不真实。

## 训练任务

看一集校园短剧，拆 5 个桥段。每个桥段只写“场景、关系、尴尬/笑点、小说事件”。
