# 行动收益-奖励期待链

依据案例：batch-263 案例 1313

## 读者承诺

主角行动成功后要有明确收益。奖励不仅是结算，更是下一段剧情的期待来源。

## 核心循环

```text
行动成功 -> 奖励落袋 -> 能力/资源变化 -> 后续使用 -> 新期待形成
```

## 开篇必备

- 本场行动目标。
- 成功后的即时收益。
- 长线收益或底牌。
- 奖励使用限制。
- 下一次使用场景。

## 期待来源

读者期待主角拿到奖励后变强、翻盘或解锁更高层问题。

## 断档风险

只有打脸没有收益，情绪会断；奖励和后续无关，就会像空结算。

## 训练任务

给一个胜利桥段设计三种奖励：立刻用、三章后用、十章后揭开。
