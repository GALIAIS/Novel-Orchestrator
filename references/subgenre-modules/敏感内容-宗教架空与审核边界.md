# 敏感内容-宗教架空与审核边界

依据案例：`batch-240` 案例 1200。

## 读者承诺

宗教等敏感内容要优先降真实指向；必须写时用架空名、架空教派和弱联想处理。

## 核心循环

```text
题材需要信仰背景 -> 真实指向有风险 -> 架空替代 -> 弱化现实联想 -> 保留剧情功能
```

## 开篇必备

- 判断敏感元素是否必要。
- 删除真实名称。
- 改成架空教派或外置体系。
- 评估国内现实联想。

## 期待来源

故事保留信仰、组织、仪式和人物动机的功能，同时降低审核和现实群体风险。

## 断档风险

- 真实宗教名称直写。
- 架空但强联想到现实宗教。
- 为现实投射硬闯雷区。
- 忽略平台审核变化。

## 训练任务

把一个宗教设定改成删除版、弱化版、完全架空版，选能保留剧情功能且真实指向最低的一版。
