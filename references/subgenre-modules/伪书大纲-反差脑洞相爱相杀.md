# 伪书大纲-反差脑洞相爱相杀

依据案例：`batch-102` 案例 510。

## 读者承诺

用伪推书大纲制造强吸引力：先建立读者熟悉的题材预期，再用黑暗反转、荒诞喜剧、病娇关系和单元任务，把一个脑洞扩成可追的长线循环。

## 核心循环

```text
常规预期 -> 黑暗反转 -> 有代价金手指 -> 固定追杀循环 -> 单元任务装逼 -> 更大真相揭露
```

## 开篇必备

- 熟悉的开局模板。
- 第一次强反转。
- 金手指真实代价。
- 主角独特反应。
- 固定关系压力源。

## 期待来源

- 恐怖和搞笑反差。
- 金手指既有用又危险。
- 主角当机立怂的喜剧感。
- 病娇妹妹梦境追杀的固定循环。
- 案件线与世界观真相逐步扩大。

## 断档风险

- 只堆脑洞，没有循环。
- 反差元素互不服务。
- 金手指只有爽，没有代价。
- 单元任务和核心关系脱节。

## 训练任务

写一个 1000 字伪书大纲，必须包含常规题材预期、第一次反转、固定追杀循环、单元任务舞台和更大真相。最后标出每个单元如何回到核心关系。
