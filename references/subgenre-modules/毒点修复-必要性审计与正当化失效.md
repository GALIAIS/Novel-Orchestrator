# 毒点修复-必要性审计与正当化失效

依据案例：`batch-414` 案例 2067

## 读者承诺

高风险行为不会因为附带一个慈善、家国或大义理由就被叙事自动洗白。作者先定位读者厌恶来自哪项承诺、主体性、规则或价值观损伤，再证明情节是否不可替代；只有删除、降伤、行为替代和视角替代都无法保留核心功能时才进入保留方案。人物可以自我合理化，事实、受影响者和后果必须让叙事保持诚实。

## 核心循环

```text
记录具体退出点和厌恶症状 -> 定位被破坏的承诺或主体性 -> 写清高风险情节的结构功能 -> 依次尝试删除、降伤、替代行为和替代视角 -> 无低损方案时建立因果压力与真实选择 -> 让人物给出自我合理化理由 -> 用事实和受影响者行动显露理由裂缝 -> 结算不可撤销责任与修复成本 -> 目标读者复测信任变化
```

稳定项是功能与后果，不能把理由强度当作修复量。正面理由只在人物塑造层有效，不会自动抵扣行为造成的合同损伤。

## 开篇必备

- 目标读者认为“毒”的具体动作和退出位置。
- 被伤害的核心承诺、关系底线或世界规则。
- 该情节必须承担的唯一结构功能。
- 删除、降伤、替代行为和替代视角四版比较。
- 人物当时掌握的信息和真实可选方案。
- 人物自我合理化中的真诚成分与自利成分。
- 受影响者拒绝、反击或改变结果的机会。
- 不可撤销后果、责任和后续修复成本。
- 目标读者对角色信任的复测方法。

## 期待来源

- 人物会否承认自己借高尚理由满足私欲。
- 受影响者能否识破并改变局势。
- 角色是否愿意承担无法追回的损失。
- 修复行动会不会要求放弃真正珍视的资源。
- 同一压力再次出现时，人物是否作出不同选择。
- 读者能否在不认同行为的情况下继续相信人物复杂性。

## 断档风险

- 一看见毒点就先寻找“高大上理由”。
- 把人物辩解当成叙事事实。
- 用后续补偿抹掉已经发生的责任。
- 声称情节不可避免，却没有测试替代方案。
- 为制造冲突而剥夺受影响者全部主体性。
- 后果只停留在内疚旁白，不改变关系和资源。
- 作者询问理由是否感人，却不记录退出点和信任。
- 目标读者仍然认为合同被破坏时继续升级说教。

## 训练任务

选一个高风险情节，先分别写删除、降伤、行为替代和视角替代方案；保留版必须证明四种方案为何损失核心功能。随后写 1800 字场景：人物给出看似正面的理由，受影响者通过行动揭开理由裂缝，并让结果产生不可撤销代价。用三名目标读者复测退出点和角色信任。
