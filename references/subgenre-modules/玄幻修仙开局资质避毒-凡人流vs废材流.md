# 玄幻修仙开局资质避毒-凡人流vs废材流

依据案例：`batch-195` 案例 973。

## 读者承诺

主角可以弱、穷、低地位，但不能被读者识别成不能修炼、低于普通人的废材；开局要尽早给出普通资质或明确潜力。

## 核心循环

```text
弱势处境 -> 普通资质/隐藏潜力 -> 金手指亮相 -> 未来可见 -> 读者愿意等成长
```

## 开篇必备

- 主角至少具备普通修炼资格。
- 第一章给出潜力证据。
- 金手指或天赋尽早出现。
- 弱势来自资源、地位、敌人或环境，而非彻底废。

## 期待来源

读者相信主角能变强，只是在等待他用潜力和金手指突破当前处境。

## 断档风险

- 写成不能修炼的废材。
- 让读者等很多章才知道主角不废。
- 开头被废再靠外挂起飞，触发老套路疲劳。
- 只写低地位，不给高潜力。

## 训练任务

把一个废材开局改成凡人流开局：保留低地位和压力，删掉不能修炼，补上普通资质、性格优势和第一章金手指。
