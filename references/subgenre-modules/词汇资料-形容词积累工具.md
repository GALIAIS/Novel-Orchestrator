# 词汇资料-形容词积累工具

依据案例：`batch-108` 案例 537。

## 读者承诺

词汇资料的价值是减少写作时的表达卡顿，让作者在人物、环境、动作、情绪和氛围描写中有更多可选词。它只能辅助准确表达，不能替代观察、场景设计和人物视角。

## 核心循环

```text
表达重复 -> 查词汇资料 -> 按语境筛选 -> 小幅替换 -> 复盘常用词
```

## 开篇必备

- 先知道自己哪类描写卡词。
- 按描写对象分类整理。
- 每个词都检查语境和视角。
- 替换后读一遍是否更准确。
- 不为了华丽强行换词。

## 期待来源

- 描写不再重复同一批词。
- 情绪和氛围更精确。
- 写作卡顿减少。
- 自己的常用词库逐步形成。

## 断档风险

- 把词汇表当万能写作课。
- 只换词，不改观察和动作。
- 用词不符合人物视角。
- 为了华丽牺牲清楚。

## 训练任务

找一段自己的描写，标出 5 个重复形容词。每个词最多换一个更准确的词，并写明替换理由。
