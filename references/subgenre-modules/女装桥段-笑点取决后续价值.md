# 女装桥段-笑点取决后续价值

依据案例：`batch-319` 案例 1592

## 读者承诺

女装伪装不天然是毒点，只有能绕过危险、引出新麻烦或回收身份暴露笑点时，才值得放在主角身上。

## 核心循环

```text
危险需要伪装 -> 女装过度成功 -> 引发额外麻烦 -> 主角主动处理 -> 身份暴露回收笑点
```

## 开篇必备

- 给出不得不伪装的压力。
- 主角人设能承受搞笑或骚浪贱桥段。
- 明确不会发展真实暧昧。
- 被喜欢只用于制造麻烦或反转。
- 身份暴露要推进主线或回收笑点。

## 期待来源

读者期待看到主角用非常规手段过关，并在误认、幻灭、身份暴露中获得喜感。

## 断档风险

- 主角代入被女装梗伤到。
- 只有玩梗，没有后续情节价值。
- 被喜欢写成真实感情线，读者尴尬。
- 主角尊严持续被踩，笑点变毒点。
- 放在男配身上更稳的桥段硬放主角。

## 训练任务

写 600 字女装伪装桥段：危险动机、误认麻烦、边界说明、身份暴露笑点和主线推进缺一不可。
