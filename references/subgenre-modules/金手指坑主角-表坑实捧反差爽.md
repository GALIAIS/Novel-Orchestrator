# 金手指坑主角-表坑实捧反差爽

依据案例：batch-272 案例 1359

## 读者承诺

让金手指看似打断主角装逼，实际给主角带来更大收益或更新鲜的赢法。读者先错愕发笑，随后确认主角没亏，爽点换轨兑现。

## 核心循环

```text
装逼预期 -> 表面翻车 -> 短暂笑点 -> 隐性收益 -> 另一种打脸兑现
```

## 开篇必备

- 先建立明确装逼预期。
- 表面坑不能造成真实憋屈。
- 后续必须证明主角实际得利。

## 期待来源

读者期待的是新鲜感和安全感同时存在。表面翻车打破套路，实际收益保证爽点不丢。

## 断档风险

- 真让主角吃亏，读者不爽。
- 反派嘲讽没有被回收。
- 主角厚脸皮变成无能，缺少主动姿态。

## 训练任务

写一个测天赋桥段：预期装逼、表面失败、隐藏收益、反派二次破防四步都要出现。
