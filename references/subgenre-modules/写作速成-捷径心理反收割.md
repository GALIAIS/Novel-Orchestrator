# 写作速成-捷径心理反收割

依据案例：`batch-109` 案例 544。

## 读者承诺

写作速成资料最容易收割想走捷径的新人。真正有用的资料可以提供行业简介或基础知识，但不能替代阅读、拆书、动笔、反馈和复盘。

## 核心循环

```text
写作焦虑 -> 速成承诺 -> 低价诱惑 -> 识别内容类型 -> 回到实战训练
```

## 开篇必备

- 看清资料是否承诺速成。
- 区分行业简介和训练体系。
- 降低购买和试错成本。
- 不把教程当保证成绩的路径。
- 每个知识点都要实战验证。

## 期待来源

- 新人少花冤枉钱。
- 学习目标从捷径改为训练。
- 资料只承担参考功能。
- 判断力逐渐提高。

## 断档风险

- 相信“21天成才”。
- 囤教程代替写作。
- 只看宣传不看实际内容。
- 把浅层行业简介当成完整方法论。

## 训练任务

拆一个写作教程宣传页：列出它的速成承诺、可验证知识点、不能替代的训练。最后判断是否值得继续看。
