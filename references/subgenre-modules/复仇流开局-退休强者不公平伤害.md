# 复仇流开局-退休强者不公平伤害

依据案例：`batch-182` 案例 907。

## 读者承诺

退休强者复仇流的承诺是平静生活被极不对等伤害摧毁后，隐藏强者重新拿起旧武器，把读者愤怒转成反杀期待。

## 核心循环

```text
退休强者 -> 平静锚点 -> 不公平伤害 -> 旧能力显影 -> 复仇推进
```

## 开篇必备

- 主角过去强，但当前已退场。
- 平静生活要有情感锚点。
- 反派伤害必须不对等。
- 旧武器或旧身份要快速显影。

## 期待来源

读者期待主角把被摧毁的秩序打回来，并让无端施害者付出远超其预期的代价。

## 断档风险

- 退休身份只说不显。
- 伤害没有打到主角核心。
- 反派动机不够低劣，不公平感不足。
- 复仇目标太散，第一卷方向不清。

## 训练任务

设计一个退休强者开局：写出平静锚点、不公平伤害、旧能力显影和第一复仇目标。
