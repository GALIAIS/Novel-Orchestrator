# 冲突烈度-损害面叠加与价值积累

依据案例：batch-384 案例 1920

## 读者承诺

冲突烈度来自读者在意的价值可能受损。可以横向叠加关系、事业等多面，也可以用长期投入抬高一个核心价值。

## 核心循环

```text
建立价值 -> 威胁多面或核心一面 -> 迫使取舍 -> 落下损失 -> 承受后果
```

## 开篇必备

- 表层争夺物。
- 读者已在意的价值面。
- 过去为核心价值付出的代价。
- 无法全部保住的选择。

## 期待来源

- 一个事件同时影响多条生活线。
- 长期建立的关系受到威胁。
- 人物必须决定优先级。
- 损失改变后续局面。

## 断档风险

- 添加未铺垫的价值标签。
- 冲突变长但没有更难取舍。
- 让人物最终全部保住。
- 用争吵音量代替损害后果。

## 训练任务

把一个物品争夺分别写成单一损害版和关系、事业叠加版；第二版必须落下不可逆损失。
