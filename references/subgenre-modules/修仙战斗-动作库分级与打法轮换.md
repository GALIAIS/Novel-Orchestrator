# 修仙战斗-动作库分级与打法轮换

依据案例：batch-364 案例 1817

## 读者承诺

正式战斗要提前设计打法，建立动作库并分级。每场重要战斗赢法不同，旧打法隔期升级复用。

## 核心循环

```text
战法归类 -> 精彩分级 -> 分配战斗 -> 差异化胜利 -> 旧打法升级复用
```

## 开篇必备

- 重要战斗不重复。
- 每场有明确战法。
- boss 战保留高等级战法。
- 旧打法复用时必须升级。

## 期待来源

- 怪异法器。
- 偷袭、埋伏、属性克制。
- 新功法、嗑药、临时破境。
- 智取方式变化。

## 断档风险

- 同一攻击套路用三遍。
- 只靠文笔描写招式。
- 高精彩战法浪费在小兵战。
- 没动作库，临场硬编。

## 训练任务

列 20 种战法并分普通、精彩、压轴三级，安排 8 场正式战斗且打法不重复。
