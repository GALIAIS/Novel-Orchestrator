# 末世杀手开篇-人设行为一致性

依据案例：`batch-094` 案例 470。

## 读者承诺

末世、神话复苏、异能和杀手背景可以提供强烈悬念，但开篇必须先让主角行为符合读者对“顶级杀手”的默认期待。冷静、专业、果断、会反击，是底线；如果主角主动丢掉核心优势，还没有更大的秘密或代价，读者会先判定降智。

## 核心循环

```text
组织压迫 -> 主角短期目标 -> 敌方反制 -> 主角用专业能力破局 -> 获得收获/线索 -> 暴露更大末世秘密
```

## 开篇必备

- 主角职业标签和第一行动必须一致。
- 敌方组织要有控制主角的合理手段。
- 主角要有一个明确短期目标，而不是被动逃。
- 不公平伤害要逼主角动用能力反击。
- 任何“放弃优势”的行为，都要先写出更大代价。

## 期待来源

- 顶级杀手的专业反击。
- 神血、陨石、神话复苏等悬念逐层升级。
- 组织压迫带来的不公平伤害。
- 主角用能力解决冲突后的收获。
- 背景设定从事件中露出，而不是整章说明。

## 断档风险

- 主角口头是杀手，行动像普通善良少年。
- 敌方组织像慈善机构，没有反制和成本。
- 背景设定压过当前目标。
- 开篇只有逃亡下行，没有上行反击。
- 名字、术语、现实敏感词混乱导致读者出戏。

## 训练任务

写一个 1000 字杀手隐退开篇。要求：第一场冲突里主角必须遭遇不公平伤害，必须用专业能力反击，必须拿到一个短期收获。写完逐项检查主角行为是否符合“顶级杀手”四字。
