# 长篇网文主线工程-脊椎骨架心脏血肉灵魂

依据案例：`batch-195` 案例 975。

## 读者承诺

长篇网文用主线作脊椎、支线作骨架、爽点作心脏、细节作血肉、立意作灵魂，避免只靠单一设定硬撑。

## 核心循环

```text
主线贯穿 -> 支线填充 -> 爽点供能 -> 细节成形 -> 立意定方向
```

## 开篇必备

- 至少有一条贯穿全文的长线目标。
- 支线能填充阶段剧情。
- 每段支线都有爽点供能。
- 立意不替代剧情推进。

## 期待来源

读者持续知道主角在追什么，又能在升级、寻宝、装逼、破案等支线里获得阶段满足。

## 断档风险

- 只有立意，没有爽点。
- 只有支线，没有贯穿主线。
- 主线太虚，不能驱动章节。
- 细节堆积但不服务目标。

## 训练任务

为一本长篇列 3 条主线、5 类支线、10 个爽点和 1 句立意，检查每个支线是否服务至少一条主线。
