# 战后尾声-风险计划收获

依据案例：`batch-131` 案例 655。

## 读者承诺

大事件结束后，不只告诉读者赢了，还要交代风险、隐患、下一步计划和超额收获。尾声要把旧期待收掉，同时抛出新期待。

## 核心循环

```text
胜利结果 -> 风险/赌注回收 -> 隐患点名 -> 下一步计划 -> 超额收获结算
```

## 开篇必备

- 明确本次胜利。
- 回收承担过的风险。
- 留一个未解决隐患。
- 给出下一步计划。
- 给一个超额收获。

## 期待来源

- 胜利满足感。
- 风险被承认，胜利不轻飘。
- 隐患保留悬念。
- 超额收获提升爽感。
- 下一步计划防止断档。

## 断档风险

- 大高潮后只写庆功。
- 没有代价和风险回收。
- 没有新目标。
- 奖励只是常规结算。
- 伏笔隐患被忘掉。

## 训练任务

写一个 300-500 字大事件尾声：包含本次风险/赌注、一个未解决隐患、下一步计划、一个超额收获。
