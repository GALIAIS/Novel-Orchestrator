# 历史开局-妥协冲突与标题期待

依据案例：`batch-229` 案例 1143。

## 读者承诺

历史或权谋开局可以让主角暂时妥协，但必须让读者看见标题承诺、身份悬念和后续显圣舞台。

## 核心循环

```text
强权压制 -> 主角暂退 -> 标题托底 -> 身份/线索铺垫 -> 后续舞台显圣
```

## 开篇必备

- 主角为什么不能立刻赢。
- 妥协不是软弱，而有个人理由。
- 标题承诺能解释当前铺垫。
- 三章内有明确显圣或翻身方向。

## 期待来源

读者接受暂时不赢，是因为知道更大的权力舞台正在靠近，主角的身份或信息差会在那里爆发。

## 断档风险

- 主角妥协却没有后续胜利指向。
- 标题承诺和开篇事件断开。
- 权力压制只剩憋屈，没有布局感。
- 技法不足，读者把暂退读成窝囊。

## 训练任务

写一个历史开局冲突表：外部压制、主角个人理由、标题承诺、身份线索、三章内显圣舞台。每项缺一不可，再写 300 字开局收束段。
