# 卖点标签-金手指读者画像

依据案例：`batch-115` 案例 574。

## 读者承诺

金手指是卖点、标签和读者画像的交汇点。读者看书名简介时，往往是在判断这个金手指类型是否合口味，以及它能不能持续制造主角与众不同。

## 核心循环

```text
选标签 -> 定金手指 -> 包装书名简介 -> 开篇展露 -> 吸引目标读者 -> 持续解决主要矛盾
```

## 开篇必备

- 金手指能凸显主角不同。
- 金手指和世界环境有紧密联系。
- 第一章能看出标签味道。
- 金手指不替主角做决定。
- 预留资源提升后的适用场景。

## 期待来源

- 标签让读者快速筛选。
- 金手指类型给出明确口味。
- 反差让卖点更突出。
- 长线主要矛盾可持续被解决。

## 断档风险

- 只选老标签，不打磨金手指。
- 金手指和世界设定脱节。
- 书名简介看不出核心卖点。
- 主角变强后金手指失效。

## 训练任务

写 5 个金手指标签方案，每个方案用一句话说明目标读者、第一章展示方式和中期主要矛盾。
