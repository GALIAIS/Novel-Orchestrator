# 废材主角-低谷必须给成长苗头

依据案例：`batch-327` 案例 1631

## 读者承诺

废材主角可以低谷、卖惨、被嘲，但必须早给成长苗头；只写废物不写未来，读者不会等。

## 核心循环

```text
低谷/实惨 -> 共情 -> 成长苗头 -> 阶段性改变 -> 翻身期待
```

## 开篇必备

- 低谷要能让读者共情，不是主角自作自受。
- 前 1000 字给出成长苗头。
- 三章内看到能力、心性或局势变化。
- 配角不能长期抢走主角光。
- 小情绪场面不能牺牲主线推进。

## 期待来源

读者期待现在的废材低谷会换来后续成长和翻身。

## 断档风险

- 只卖惨，不成长。
- 成长苗头藏到读者走光后才给。
- 主角长期巨婴化。
- 配角太强，主角没有存在感。
- 为小场面放弃大故事线。

## 训练任务

写一个废材开篇，前半段低谷，后半段必须出现一个可验证成长苗头。
