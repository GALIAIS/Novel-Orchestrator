# 玄幻大纲-灵感到手册

依据案例：`batch-102` 案例 509。

## 读者承诺

把玄幻灵感变成大纲，不靠灵感乱飞，而靠固定顺序：热点标签、金手指、世界背景联系、反差、矛盾网、情绪折线逐步落地。

## 核心循环

```text
灵感 -> 热点标签 -> 金手指 -> 背景联系/反差 -> 矛盾网 -> 情绪折线 -> 第一卷大纲
```

## 开篇必备

- 灵感对应的题材标签。
- 金手指和世界背景有关。
- 金手指与背景存在反差。
- 至少一张矛盾网。
- 第一卷情绪折线。

## 期待来源

- 标签给读者入口。
- 金手指提供成长和爽点。
- 背景反差制造新鲜感。
- 矛盾网提供持续事件。
- 情绪折线避免流水账。

## 断档风险

- 只有灵感，没有构建顺序。
- 金手指和世界背景脱节。
- 矛盾网没搭起来，剧情只能靠偶遇。
- 情绪折线缺失，章节只有设定推进。

## 训练任务

给一个玄幻点子补齐六项：标签、金手指、背景联系、背景反差、矛盾网、情绪折线。缺一项就先不写正文。
