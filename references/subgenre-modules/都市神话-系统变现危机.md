# 都市神话-系统变现危机

依据案例：`batch-131` 案例 653。

## 读者承诺

现实缺钱的人获得神话/修仙系统，用超凡资源先完成可信小额变现，再被修仙者或神话势力追踪卷入危机。爽点来自现实翻盘和超凡入侵现实。

## 核心循环

```text
现实压力 -> 系统早出 -> 超凡物品小额验证 -> 现实变现爽点 -> 超凡势力追踪 -> 新危机
```

## 开篇必备

- 金手指前 500 字内出现。
- 第一次奖励可验证。
- 变现流程可信。
- 情绪压迫对象明确。
- 修仙者上门有合理目标和手段。

## 期待来源

- 缺钱困境被逆转。
- 超凡资源进入现实市场。
- 小额试探降低较真风险。
- 神话势力接入都市危机。

## 断档风险

- 打游戏掉装备铺太久。
- 银锭直接换巨款。
- 古董价和材料价混乱。
- 修仙者上门方式太 low。
- 为凑字强行加戏。

## 训练任务

写 1500 字以内开篇：主角现实缺钱，金手指前 500 字出现；第一次奖励是可小额验证的超凡物品；完成低风险变现，并埋下修仙者追踪线索。
