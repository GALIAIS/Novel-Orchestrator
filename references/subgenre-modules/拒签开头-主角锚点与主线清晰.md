# 拒签开头-主角锚点与主线清晰

依据案例：batch-355 案例 1775

## 读者承诺

动作开头、梦境开头和谜团开头都必须先给主角锚点、行动理由和清晰主线。读者不知道主角是谁、为什么非这样做、目标是什么，就不会关心死人和谜团。

## 核心循环

```text
开头危机 -> 给主角轮廓 -> 堵死替代路线 -> 明确主线目标 -> 给可见收益/爽点
```

## 开篇必备

- 主角第一锚点：职业、性格、特长或主动心态。
- 当前危机与目标。
- 关键物不能丢的理由。
- 第一章结尾的主线方向。

## 期待来源

- 读者先抓住主角。
- 分叉被堵死后行动显得必然。
- 谜团背后有可追目标。
- 变化带来好处或爽点。

## 断档风险

- 学大神只学表象。
- 上来死人但读者不认识主角。
- 梦境现实来回跳，节奏割裂。
- 只给画面感，不给主线和收益。

## 训练任务

写一个 1000 字动作开头大纲，前 100 字给主角锚点，前 500 字堵死替代路线，结尾给出主线目标和一个可见收益。
