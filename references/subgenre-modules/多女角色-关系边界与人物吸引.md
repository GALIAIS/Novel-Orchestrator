# 多女角色-关系边界与人物吸引

依据案例：batch-387 案例 1932

## 读者承诺

多人女性角色不等于后宫。每名角色要有独立目标、关系类型和拒绝边界，吸引力来自能力、个性、选择与互动。

## 核心循环

```text
独立角色目标 -> 与主角形成具体关系 -> 关键选择建立魅力 -> 关系边界结算 -> 团队继续变化
```

## 开篇必备

- 每名角色的独立利益。
- 清楚的非恋爱关系类型。
- 她能够拒绝主角的边界。
- 一次展示能力与选择的场景。

## 期待来源

- 多种关系产生不同冲突。
- 女性角色能独立解决问题。
- 互动改变信任和合作。
- 关系承诺及时得到确认。

## 断档风险

- 用低俗性化描写替代人物。
- 长期暧昧却拒绝结算关系。
- 所有女性只围绕主角存在。
- 把审核个案当规避教程。

## 训练任务

设计部下、盟友、家人式三名女性角色，每人写拒绝主角、独立解题和关系确认各一场。
