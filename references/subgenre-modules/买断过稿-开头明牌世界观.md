# 买断过稿-开头明牌世界观

依据案例：`batch-138` 案例 686。

## 读者承诺

复杂设定开头要先让读者看懂，再谈慢慢揭开世界观。生死危机、主角手段、金手指关键作用和差异化世界观都要清楚。

## 核心循环

```text
生死危机 -> 主角求生 -> 金手指关键作用 -> 世界观明牌 -> 读者留存
```

## 开篇必备

- 直面危机。
- 主角用手段活下来。
- 立出主角人设。
- 金手指参与关键解决。
- 用 NPC 或旁白解释核心差异。

## 期待来源

- 危机带来钩子。
- 主角能力可见。
- 金手指有用。
- 世界观差异清楚。
- 读者不被新名词劝退。

## 断档风险

- 设定全藏起来，读者看不懂。
- 新名词太多。
- 只主角不知道，读者也不知道。
- 世界观有差异但展示不明白。

## 训练任务

写 800 字开头小纲：包含生死危机、求生手段、金手指关键作用、一次爆发，并用 100 字解释核心世界观差异。
