# 外挂开篇-三百字钩子首秀

依据案例：`batch-337` 案例 1684

## 读者承诺

外挂文开篇要在前三百字抛出钩子，用当场危机逼出外挂首秀，并让能力改变局面。

## 核心循环

```text
失败样本 -> 当场危机 -> 主角被迫上场 -> 外挂显效 -> 局面改变 -> 章末期待
```

## 开篇必备

- 前三百字有危险或异常钩子。
- 危机足够近，不靠远期倒计时。
- 工具人戏份压缩。
- 外挂中段至少露一次作用。
- 章末不是只亮名字，而是改变局面。

## 期待来源

读者期待看到书名承诺马上兑现：外挂不是设定名词，而是真能救场、破局、带来新可能。

## 断档风险

- 开头太闷。
- 藏钩子。
- 第一章只在章末亮外挂名。
- 危机不够凶险。
- 工具人加戏抢主角。

## 训练任务

重排一章：300 字内给失败样本，1500 字内让主角上场，中段展示外挂作用，章末让外挂改变危机。
