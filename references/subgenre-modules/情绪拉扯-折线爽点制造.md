# 情绪拉扯-折线爽点制造

依据案例：`batch-014` 案例 070。

## 读者承诺

情绪拉扯通过下行压制和上行释放制造爽点，核心难点是尺度：压得够不够、转得准不准、读者是否愿意忍到释放。

## 核心循环

```text
情绪基准 -> 下行 -> 期待积蓄 -> 转折 -> 上行 -> 爽点释放 -> 新期待
```

## 开篇必备

- 先知道读者当前情绪基准。
- 下行要能制造期待，不只是虐。
- 上行触发点要清楚。
- 爽点释放后要接新期待。

## 期待来源

- 读者知道主角有可能翻盘。
- 压制让读者想看反击。
- 转折由主角主动推动。
- 释放能回收前面的压制。

## 断档风险

- 下行不够，爽点没力度。
- 下行过度，读者直接弃。
- 上行点找不准，剧情显得平。
- 只套一种拉扯方法，不拆书积累变化。

## 训练任务

写一场 800 字情绪拉扯，标出情绪最低点、主角主动转折点、爽点释放点和下一轮期待。
