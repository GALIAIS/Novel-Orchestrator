# 毒点判断-争议不等于翻车

依据案例：`batch-320` 案例 1597

## 读者承诺

争议桥段不等于真正翻车；判断毒点强度要看它是否破坏核心期待、造成实际流失，以及市场成绩是否证明读者无法承受。

## 核心循环

```text
角色争议行为 -> 读者预期损失 -> 评论区反弹 -> 成绩/追读验证 -> 判断争议还是翻车
```

## 开篇必备

- 识别物品是否绑定情感价值。
- 判断主角是否主动造成损失。
- 观察读者预期失去哪些未来好处。
- 区分评论区骂声和核心读者流失。
- 新手避开老作者可承受的争议点。

## 期待来源

读者期待主角不要亲手毁掉情感信物、女角色好感或未来收益。

## 断档风险

- 把争议误判为可随便模仿。
- 主角主动犯蠢，损失未来好处。
- 赠物或遗物被轻慢处理。
- 只看骂声，不看追读和成绩。

## 训练任务

找一个主角毁掉赠物的桥段，列出三种读者预期损失，并写一个保留情节但降低毒感的替代方案。
