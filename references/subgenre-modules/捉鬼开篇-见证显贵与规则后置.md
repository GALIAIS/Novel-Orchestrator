# 捉鬼开篇-见证显贵与规则后置

依据案例：`batch-340` 案例 1700

## 读者承诺

捉鬼、驱邪、玄奇开篇要让主角第一次出手被有效见证，并用金手指首秀改变局面，规则说明后置。

## 核心循环

```text
公开鬼患 -> 众人怀疑/求助 -> 危机逼近 -> 主角出手 -> 金手指显效 -> 见证者态度反转
```

## 开篇必备

- 事件放在有见证者的场域。
- 见证者前后态度可反转。
- 鬼患要当场制造压力。
- 金手指第一次战斗要展示效果。
- 货币、规则、世界观只给当前剧情所需。

## 期待来源

读者期待看到主角有本事、别人看见并改变态度；第一次战斗要先兑现能力，再慢慢解释规则。

## 断档风险

- 只有单个 NPC 见证。
- 二三章解释设定拖节奏。
- 一开始硬写迪化。
- 货币数值定太死。
- 古代文风和现代语法混杂。

## 训练任务

把偏僻地点单人见证的驱鬼开头，改成村内公开事件：列 5 个见证者、一次危机、一次能力首秀和一个态度反转。
