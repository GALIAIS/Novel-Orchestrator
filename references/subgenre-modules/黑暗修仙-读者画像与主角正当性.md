# 黑暗修仙-读者画像与主角正当性

依据案例：`batch-187` 案例 934。

## 读者承诺

黑暗修仙可以写狠主角，但要先锁定读者画像，并为主角越界行为提供正当理由和足够该死的目标对象。

## 核心循环

```text
残酷世界 -> 主角越界 -> 正当性过滤 -> 目标对象过滤 -> 读者画像过滤 -> 爽感/毒点分流
```

## 开篇必备

- 明确目标读者是否接受弱肉强食。
- 主角第一次越界要有充分理由。
- 被惩罚对象不能是无辜者。
- 旁观评价要服务读者判断。

## 期待来源

目标读者期待主角在残酷世界里果断、强硬、有仇必报，但仍能站在可代入的利益和情绪位置上。

## 断档风险

- 把滥杀包装成杀伐果断。
- 欺凌弱小导致读者反感。
- 叛族、背义、伤害无辜触发强毒点。
- 小众价值观出圈后被大众批判。

## 训练任务

写一个主角狠辣出手桥段，列出目标对象罪状、主角出手理由、读者可代入利益和出圈风险；缺任意一项，就降低越界程度。
