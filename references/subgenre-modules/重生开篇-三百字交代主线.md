# 重生开篇-三百字交代主线

依据案例：`batch-313` 案例 1563

## 读者承诺

重生文开篇要快速给出“我回来了、我知道灾难、我能改命”的主线承诺；外部负评要筛选，只修补真实影响承诺的小缺口。

## 核心循环

```text
醒来异常 -> 物证确认重生 -> 未来灾难压迫 -> 情绪中转 -> 改命宣言 -> 第一行动
```

## 开篇必备

- 具体年份或时代物证。
- 一个短期会发生的灾难。
- 主角知道前世结果。
- 从震惊到振奋之间有一拍心理过渡。
- 明确的改命目标。

## 期待来源

读者的期待来自信息差补偿：主角知道未来，能用前世记忆改写家庭、事业、命运。

## 断档风险

- 只写重生震惊，不写未来危机。
- 主角振作过快，缺少“可能性”意识。
- 被单条挑刺负评带偏，推翻有效结构。
- 300 字内没有主线目标。

## 训练任务

写 300 字重生开篇，必须包含物证、年份、未来灾难、心理中转、改命宣言。
