# 文学价值网文-人物抉择铺垫

依据案例：`batch-048` 案例 237。

## 读者承诺

在商业网文框架里吸收文学质感：让每个重要人物都有动机、选择和代价，让世界压力推动人物行动，再用长线伏笔在卷末集中兑现。

## 核心循环

```text
人物动机 -> 压力逼近 -> 做出抉择 -> 影响主线 -> 伏笔沉淀 -> 卷末集中兑现
```

## 开篇必备

- 主角不能长期无主动性，除非作者信誉足够强。
- 至少给主角一个当下目标，承接慢热铺垫。
- 重要配角一出场就要有自己的欲望或恐惧。
- 文学表达要和当前冲突绑定，不要先讲理念。
- 第一卷末必须兑现前期忍耐的收益。

## 期待来源

- 读者期待人物选择带来真实后果。
- 配角不是工具人，会让世界更厚。
- 长线伏笔集中触发，会产生“原来如此”的顿悟爽。
- 人物动机导出剧情，比作者强推事件更有可信度。

## 断档风险

- 新人直接模仿大神慢热，会在开篇流失读者。
- 主角一直被算计，会削弱商业网文的主动爽感。
- 结局突然背离作品承诺，会让读者觉得被背刺。
- 文青表达没有铺垫，会变成作者自嗨。

## 训练任务

列出 5 个有名字的配角：每人写一个欲望、一个恐惧、一个必须做出的选择，以及这个选择如何影响主角。
