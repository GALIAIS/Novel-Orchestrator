# 书名欲望钩子-LSP向点击承诺

依据案例：`batch-195` 案例 974。

## 读者承诺

书名可以直接钩住特定读者欲望，但正文前三章必须兑现标题暗示，至少给出同类情绪补偿。

## 核心循环

```text
欲望书名 -> 点击进入 -> 寻找兑现 -> 情绪补偿 -> 留存或流失
```

## 开篇必备

- 明确标题钩住的欲望。
- 前三章安排对应桥段或氛围。
- 人物行为逻辑不能为钩子崩坏。
- 不直给时要给同类替代满足。

## 期待来源

读者因为标题产生明确想看内容，并在开篇确认作者不会挂羊头卖狗肉。

## 断档风险

- 标题强钩子，正文完全不兑现。
- 只骗点击，不给情绪补偿。
- 改名后剧情方向和标题承诺错位。
- 主角逻辑不成立导致留存失败。

## 训练任务

写 3 个欲望型书名，并为每个书名列出前三章兑现点；没有兑现点的标题直接淘汰。
