# 双线开篇-社会人设与宏大异象汇流

依据案例：batch-388 案例 1940

## 读者承诺

日常线让读者在误读、关系和选择中关心人物，异象线让读者看见威胁逼近；两线在预先锁定的地点与时间汇流。

## 核心循环

```text
日常选择立人设 -> 异象状态升级 -> 两线交替各自推进 -> 共同坐标汇流 -> 普通关系网被整体卷入
```

## 开篇必备

- 日常线中的人物误读和关键选择。
- 读者与场内角色之间的信息差。
- 异象线至少两次状态变化。
- 提前出现的共同地点、时间或行动。

## 期待来源

- 社会互动建立人物关心。
- 克制的信息差带来短线回收。
- 远端异常逐步逼近熟悉世界。
- 汇流同时兑现奇观和人物代价。

## 断档风险

- 日常线只剩生活流水账。
- 异象线重复展示同一奇观。
- 切线不新增信息或改变局势。
- 两线靠巧合突然相撞。

## 训练任务

按日常、异象、日常、异象、汇流写五场短戏，保证每次切线推进一个状态，并检查任一线删除后是否损失独立功能。
