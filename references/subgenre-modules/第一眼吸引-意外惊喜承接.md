# 第一眼吸引-意外惊喜承接

依据案例：`batch-037` 案例 185。

## 读者承诺

书名、简介、前三章要先吸量，后续故事和基本功负责留存。意外惊喜要有独到切入点，也要能被基本功承接。

## 核心循环

```text
书名简介 -> 第一眼兴趣 -> 快速主线 -> 意外惊喜 -> 基本功承接 -> 留存
```

## 开篇必备

- 书名有吸量点。
- 简介承诺清楚。
- 前三章快速进主线。
- 意外和惊喜不是旧梗复刻。

## 期待来源

- 读者第一眼愿意点。
- 开头验证承诺。
- 独到切入点制造新鲜感。
- 后续故事证明不是噱头。

## 断档风险

- 包装没做，读者直接 pass。
- 只有脑洞，没有基本功。
- 梗大众化，不再惊喜。
- 追求吸眼球而主线迟迟不进。

## 训练任务

做一个第一眼测试：3 个书名、3 条简介钩子、前三章主线进入点，逐一标注意外和惊喜。
