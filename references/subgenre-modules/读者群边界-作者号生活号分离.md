# 读者群边界-作者号生活号分离

依据案例：`batch-326` 案例 1629

## 读者承诺

读者群扩大后，作者要分离作者号和生活号；付费读者也不能越界骚扰作者私生活。

## 核心循环

```text
读者群扩大 -> 私信骚扰增加 -> 设置账号边界 -> 屏蔽/拉黑/踢出 -> 创作环境恢复
```

## 开篇必备

- 作者号和生活号分开。
- 读者群规则提前写清。
- 关闭或限制陌生私信。
- 越界读者先屏蔽，严重时踢出。
- 必要时保留证据并报警。

## 期待来源

作者期待保留正常读者交流，同时不让少数越界者消耗创作状态。

## 断档风险

- 因为对方付费就不敢处理。
- 用生活号承接所有读者互动。
- 怕评论区搞事，提前牺牲私生活。
- 对骚扰装作正常互动。
- 没有规则，处理时显得随意。

## 训练任务

写 8 条读者群规则，覆盖催更、私信、指导作者、群内攻击、打赏纠纷、拉黑踢群标准。
