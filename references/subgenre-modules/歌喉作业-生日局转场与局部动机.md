# 歌喉作业-生日局转场与局部动机

依据案例：`batch-245` 案例 1224。

## 读者承诺

大神级歌喉作业的主折线成立后，还要补足生日宴转 KTV、朋友不爽、反派嘲讽等局部动机。

## 核心循环

```text
生日局上行 -> 离场虚晃 -> KTV邀约 -> 反派嘲讽 -> 女主维护 -> 歌喉兑现
```

## 开篇必备

- 主角为什么来。
- 主角为什么要走。
- 众人为什么转去 KTV。
- 朋友为什么替女主不爽。

## 期待来源

读者顺着社交动机进入唱歌局，后面的打脸才不会因为前置别扭而降爽。

## 断档风险

- 转场理由缺失。
- 女主朋友不爽没有触发点。
- 反派安排显刻意。
- 虚晃离场理由待定。

## 训练任务

给五折线作业每个社交转场补一句动机：谁提出、为什么、谁反对、反对理由是什么。
