# 恋爱修罗场-日常甜感与多角张力

依据案例：batch-367 案例 1833

## 读者承诺

恋爱日常修罗场要用多角色样本训练：日常负责甜感，信息差和同场碰撞负责张力，每次碰撞都要推进关系。

## 核心循环

```text
多角色承诺 -> 日常互动 -> 信息差/误会 -> 同场碰撞 -> 关系推进
```

## 开篇必备

- 至少两个关系对象。
- 每人独立吸引点。
- 一个共同场景。
- 一个轻度误会或信息差。

## 期待来源

- 多角色都值得喜欢。
- 谁更进一步的悬念。
- 公开场合的轻尴尬。
- 碰撞后的甜感补偿。

## 断档风险

- 角色只当醋意工具。
- 修罗场只有尴尬没有推进。
- 冲突过重破坏日常轻感。
- 不扫样本凭空想技巧。

## 训练任务

拆一本恋爱修罗场样本的 10 场同场戏，标出信息差、醋意来源、尴尬点和关系推进。
