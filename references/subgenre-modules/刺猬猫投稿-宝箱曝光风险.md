# 刺猬猫投稿-宝箱曝光风险

依据案例：`batch-174` 案例 869。

## 读者承诺

刺猬猫等轻小说平台投稿前，要先判断平台生态、原创/同人机会、热点题材和宝箱曝光的投入风险。

## 核心循环

```text
平台调研 -> 题材匹配 -> 数据观察 -> 是否投放 -> 曝光转化 -> 止损
```

## 开篇必备

- 先看当前榜单题材生态。
- 区分原创和同人机会。
- 投放前检查点收比、活人反馈和推荐票。
- 明确预算和止损条件。

## 期待来源

作者期待平台曝光带来读者，但真正有效的是题材匹配和转化能力，不是单纯花钱买收藏。

## 断档风险

- 平台信息过期仍照用。
- 盲目砸宝箱换死收。
- 原创不吃平台热度还硬投。
- 没有预算和止损线。

## 训练任务

投稿前拆榜单前 30：统计原创/同人比例、热点 IP、题材标签，再写自己的投放预算和止损条件。
