# 体育小众-情绪缺口折线

依据案例：`batch-016` 案例 080。

## 读者承诺

小众体育文也要用大众爽文结构：简介挖情绪缺口，金手指提供主角优势，第一章用合理压制和情绪折线展示机制。

## 核心循环

```text
情绪缺口 -> 主角优势 -> 合理嘲讽/压制 -> 机制展示 -> 竞技兑现
```

## 开篇必备

- 简介写出情绪缺口。
- 主角优势要能被读者畅想。
- 压制要符合角色关系和地位。
- 第一章展示金手指机制。

## 期待来源

- 读者想看弱势主角靠优势翻盘。
- 体育竞技天然有胜负反馈。
- AI 预判等机制能制造专业爽点。
- 小众题材用大众情绪降低门槛。

## 断档风险

- 只写专业，不造情绪缺口。
- 无关角色硬嘲讽。
- 金手指机制展示太晚。
- 简介没有满足后的畅想。

## 训练任务

给一个小众题材写标准简介，包含情绪缺口、主角优势和满足畅想；再设计第一章合理压制。
