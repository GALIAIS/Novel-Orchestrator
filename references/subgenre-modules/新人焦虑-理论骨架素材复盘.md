# 新人焦虑-理论骨架素材复盘

依据案例：`batch-070` 案例 346。

## 读者承诺

这个模块用于学理论后更焦虑、长期未签约的新手。它承诺的是把“理论能不能让我签约”的压力，改成近期火书细读、素材积累和作者思维复盘。

## 核心循环

```text
学理论 -> 降低结果焦虑 -> 找近期火书 -> 记录剧情和技巧 -> 积累素材 -> 再写作验证
```

## 开篇必备

- 理论只当骨架，不当签约保证。
- 选两年内、自己能看下去的火书。
- 阅读时记录剧情记忆，不只留下情绪记忆。
- 复盘布局、情绪和期待。
- 保持慢就是快。

## 期待来源

- 理论提供基本结构。
- 近期火书提供市场样本。
- 剧情记忆反哺写作。
- 笔记复盘减少空转。
- 积累过程替代签约恐惧。

## 断档风险

- 学完理论立刻期待结果。
- 看很多书但想不起剧情。
- 只记“爽”，不记作者怎么写爽。
- 盲目跨分类阅读。
- 焦虑导致不敢下笔。

## 训练任务

选一本两年内火书，拆前 3 万字。每 3000 字记录一个期待点、一个情绪点、一个可复用素材。
