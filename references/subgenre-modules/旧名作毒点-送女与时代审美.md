# 旧名作毒点-送女与时代审美

依据案例：batch-370 案例 1846

## 读者承诺

旧名作里的牺牲爱情、成人之美和虐感桥段，放到当前男频爽文语境可能变成送女毒点。

## 核心循环

```text
经典桥段 -> 时代审美变化 -> 当前读者承诺 -> 关系被让出 -> 毒点反噬
```

## 开篇必备

- 判断原桥段成立时代。
- 判断当前读者承诺。
- 检查是否伤主角尊严。
- 检查是否破坏重要关系。

## 期待来源

- 主角守住关系边界。
- 主角尊严不被背刺。
- 悲剧压力不等于送女。
- 经典机制被现代化改写。

## 断档风险

- 盲学名作光环。
- 把送女包装成君子成全。
- 用旧武侠虐感写现代爽文。
- 忽视读者对情感线的占有期待。

## 训练任务

改写一个旧武侠虐恋桥段，保留悲剧压力，但不能让主角主动送出重要关系。
