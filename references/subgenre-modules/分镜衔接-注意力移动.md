# 分镜衔接-注意力移动

依据案例：`batch-016` 案例 078。

## 读者承诺

分镜衔接的本质是读者注意力移动。镜头从哪里切到哪里，取决于下一步要承载的信息、情绪或人设。

## 核心循环

```text
当前注意点 -> 关联动作/物件/声音 -> 注意力移动 -> 新信息/新情绪
```

## 开篇必备

- 明确当前镜头注意点。
- 明确下一镜头功能。
- 用动作、眼神、声音、物件承接。
- 不为转场而转场。

## 期待来源

- 读者跟着注意力自然移动。
- 分镜同时服务情绪拉扯。
- 镜头细节展示人设。
- 景别变化控制信息密度。

## 断档风险

- 只记转场术语，不知道功能。
- 镜头移动生硬。
- 分镜没有情绪或信息任务。
- 景别混乱，读者不知道看哪里。

## 训练任务

选 30 秒片段，逐镜写“注意力从哪里到哪里”，再改写成 300 字小说分镜。
