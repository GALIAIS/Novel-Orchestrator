# 主角动机-必要赌注风险

依据案例：`batch-103` 案例 512。

## 读者承诺

主角行动要让读者认同，必须有成功收益和失败损失。主角不是为了推动剧情而动，而是因为不动会失去，动了有机会得到。

## 核心循环

```text
目标 -> 成功收益 -> 失败损失 -> 不行动后果 -> 读者认同 -> 行动推进
```

## 开篇必备

- 主角当前想要什么。
- 为什么现在必须做。
- 成功能得到什么。
- 失败会失去什么。
- 不行动会发生什么。

## 期待来源

- 风险带来紧张。
- 收益带来期待。
- 不行动后果带来必要性。
- 读者认同主角后才愿意跟行动。

## 断档风险

- 作者觉得合理，读者不认同。
- 主角只是被剧情推着走。
- 风险空泛，收益模糊。
- 学了理论但没填到文本里。

## 训练任务

给主角第一章所有行动列动机表。任何一项没有收益、风险或不行动后果，就合并动作或重写赌注。
