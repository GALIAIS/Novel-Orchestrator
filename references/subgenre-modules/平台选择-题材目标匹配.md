# 平台选择-题材目标匹配

依据案例：`batch-150` 案例 749。

## 读者承诺

新人选平台时，不只看稿费透明或反馈快，而要先判断题材偏主站还是渠道、金手指新旧、逻辑强度、是否需要长期流量和读者反馈。

## 核心循环

```text
确定题材 -> 判断主站/渠道 -> 判断金手指和逻辑 -> 比较反馈/流量/全勤/长期收益 -> 选择平台
```

## 开篇必备

- 作品类型和读者画像。
- 主站或渠道倾向。
- 金手指是否新。
- 作者是否愿写热门题材。
- 对反馈、全勤、长期收益的排序。

## 期待来源

作者期待降低平台选择焦虑，把“哪个网站更安全”转换成“我的项目更适合哪类流量和读者”。

## 断档风险

- 只凭情绪厌恶平台制度。
- 只看短期反馈，不看题材匹配。
- 偏主站逻辑文却用渠道完本思路决策。
- 只考虑第一本写完，不考虑长期收获。

## 训练任务

为一个新书项目比较两个平台，写出题材倾向、金手指新旧、逻辑强度、反馈需求、全勤收益、长期收益。最后用一句话说明首发选择。
