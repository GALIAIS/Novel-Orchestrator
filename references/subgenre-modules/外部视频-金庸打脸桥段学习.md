# 外部视频-金庸打脸桥段学习

依据案例：`batch-100` 案例 499。

## 读者承诺

把外部视频从“收藏了”转成“能写出来”：只取一个专项技巧，例如打脸桥段，观看后立刻拆成可复用结构，再迁移到自己的题材中。

## 核心循环

```text
找到高质量分析 -> 聚焦一个桥段功能 -> 拆轻视/压制/证明/反应 -> 换题材复写 -> 回到正文验证
```

## 开篇必备

- 学习目标只选一个，不能泛刷。
- 看完必须产出桥段表。
- 表里必须有轻视理由、证明动作和旁观者反应。
- 迁移时只换题材壳，不丢桥段功能。

## 期待来源

- 被轻视制造压抑。
- 主角隐藏优势制造期待。
- 证明动作兑现爽点。
- 旁观者反应放大情绪。
- 后续名望/关系变化延长收益。

## 断档风险

- 只记“这个视频不错”，没有拆桥段。
- 把金庸的语境硬套到网文题材。
- 只有打脸结果，没有前置压制。
- 只有主角赢，没有旁观者传播。

## 训练任务

选一个公开视频或章节，拆 1 个打脸桥段。必须写清：谁轻视、为什么轻视、主角如何被压、主角用什么动作证明、谁看见、看见后关系如何变化。
