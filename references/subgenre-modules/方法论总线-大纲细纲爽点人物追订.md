# 方法论总线-大纲细纲爽点人物追订

依据案例：`batch-185` 案例 925。

## 读者承诺

商业网文生产要把大纲、细纲、节奏、爽点、人物、事件链和反馈收获串成一条总线。

## 核心循环

```text
全文大纲 -> 当前卷细纲 -> 小事件链 -> 爽点兑现 -> 人物成长 -> 事后反馈 -> 追订维持
```

## 开篇必备

- 主角是谁、在哪。
- 金手指是什么。
- 遇到什么困难。
- 如何用金手指解决困难。

## 期待来源

读者持续看到冲突、解决、收获、成长和下一轮问题，才会追长篇。

## 断档风险

- 没大纲只靠灵感。
- 细纲过细耗尽激情。
- 事件没有动机、手法、收获。
- 爽点弱却不扫榜扒大纲。
- 人物没有稳定特质和成长事件。

## 训练任务

拆一卷 5 个小事件，每个写起因、经过、结果、主角动机、手法、收获、节奏类型和事后反馈。
