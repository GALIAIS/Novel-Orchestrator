# 异能开篇-信息行动化与人设一致

依据案例：batch-346 案例 1727

## 读者承诺

全民异能开篇要让读者迅速看到身份差、能力差、压力、目标、首次正反馈，并相信主角的行动符合人设。

## 核心循环

```text
现实压力 -> 觉醒/系统触发 -> 能力差异暴露 -> 外部阻碍加压 -> 主角选择证明人设 -> 爽点正反馈
```

## 开篇必备

- 时间、地点、人物、目标、阻碍用场景带出。
- 金手指或异能首次出现后，必须给读者正反馈。
- 嘲讽、打赌、入宿舍等冲突要有合理风险。
- 主角的善良、孝顺、义气、聪明要落实在选择上。
- 穿越亲情动机要用时间或事件补足说服力。

## 期待来源

读者期待低开高走、废柴异能反转、主角用能力改变评价。

## 断档风险

- 信息全靠旁白交代。
- 同学全员降智嘲讽。
- 主角获得外挂后像幼稚玩闹。
- 主角拉朋友涉险却被包装成正派。
- 检查表打勾，但读者没有感到风险和爽点。

## 训练任务

写一个 1200 字全民异能开篇，只允许通过电话、课堂、觉醒台、同学反应和主角选择交代信息，禁止单独设定说明段。
