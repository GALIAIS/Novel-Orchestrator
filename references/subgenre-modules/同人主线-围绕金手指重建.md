# 同人主线-围绕金手指重建

依据案例：`batch-106` 案例 529。

## 读者承诺

同人不能只换主角截胡原著。新主角的金手指不同，主线就必须围绕新的变强方式、解锁条件和目标重建。

## 核心循环

```text
新金手指 -> 如何变强 -> 解锁条件 -> 主线事件 -> 原著事件取舍 -> 差异化同人
```

## 开篇必备

- 金手指一句话概括。
- 明确“如何变强”。
- 第一阶段解锁条件。
- 与原著主线的差异。
- 哪些原著事件要删改。

## 期待来源

- 熟悉世界的新路线。
- 新金手指带来的差异化。
- 原著事件被重新利用。
- 解锁和升级推动主线。

## 断档风险

- 只截胡原著机缘。
- 主角换人但剧情没变。
- 金手指和主线脱节。
- “变强”太笼统，没有具体方式。
