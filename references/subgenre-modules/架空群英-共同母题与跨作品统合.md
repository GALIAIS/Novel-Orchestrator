# 架空群英-共同母题与跨作品统合

依据案例：`batch-401` 案例 2005

## 读者承诺

先用共同母题、制度基线和叙事风格统一世界，再按政治功能与价值立场筛选人物；所有角色进入新世界后重新获得利益、限制和选择，主角始终承担格局决策。

## 核心循环

```text
确定共同母题与风格 -> 建立资源制度基线 -> 按功能和立场选角色原型 -> 重建新世界利益与限制 -> 产生互斥方案 -> 主角选择并付代价 -> 联盟和秩序变化
```

## 开篇必备

- 资源、交通、军制、治理与合法性规则清楚。
- 有一个能统合群英的共同母题。
- 角色按功能和价值差异进入，而非按知名度收集。
- 每人拥有新世界出身、利益、底线和能力盲区。
- 导入机制符合严肃或轻松风格。
- 主角必须决定资源与路线并承担后果。

## 期待来源

熟悉能力或原型快速建立预测，新世界利益又迫使人物偏离旧印象；互斥方案让主角每次选择都改变联盟和制度。

## 断档风险

- 先列客串名单，后补世界规则。
- 多历史条件彼此矛盾却不解释。
- 角色只复刻标签和名场面。
- 召唤机制自动保证忠诚与合作。
- 群英轮流为主角提供无成本能力。
- 删除角色也不影响任何决策。
- 未核验受保护角色、改编形象与平台规则。

## 训练任务

不用现成名字，围绕一个共同母题设计五个原创角色原型，写明功能、利益、底线、盲区和互斥方案；删掉不改变首卷决策的角色，直到每个人都进入主线因果。
