# 冲突模型-三观视角差替代生死危机

依据案例：`batch-187` 案例 931。

## 读者承诺

当剧情不适合生死危机时，用三观、视角和路线差异制造强阻碍，能保持冲突强度同时避免人物失真。

## 核心循环

```text
主角目标 -> 阴影角色阻碍 -> 判断敌人/反派 -> 选择危机强度 -> 用行动证明路线
```

## 开篇必备

- 主角目标清楚。
- 阻碍者有自己的立场。
- 双方差异能落到具体行为。
- 冲突强度和场景类型匹配。

## 期待来源

读者期待主角在不把对手强行恶人化的情况下，用能力、选择或结果证明自己的路线正确。

## 断档风险

- 所有阻碍都写成生死危机。
- 都市、家庭、职场桥段过度恶人化。
- 反派没有自己的视角。
- 冲突升级后人物可信度崩掉。

## 训练任务

把一个“被人看不起”的桥段改成三观/视角差冲突：对手不骂人、不犯罪，只用职业经验或价值判断阻碍主角，最后让主角用结果完成反击。
