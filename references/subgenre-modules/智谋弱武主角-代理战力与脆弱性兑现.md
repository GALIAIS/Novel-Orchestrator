# 智谋弱武主角-代理战力与脆弱性兑现

依据案例：`batch-400` 案例 1996

## 读者承诺

让主角以判断、组织和说服解决核心矛盾，身体弱点真实限制行动却不夺走主体性；强力盟友基于独立利益协作，而非充当无条件外接战力。

## 核心循环

```text
非战斗能力识别矛盾 -> 脆弱限制正面方案 -> 说服独立盟友协作 -> 代理行动出现偏差 -> 主角修正完成胜利 -> 增强自保但保留弱武边界
```

## 开篇必备

- 核心矛盾确实需要主角的判断能力。
- 身体弱点改变路线、时间或筹码。
- 计划关键点对读者公平可见。
- 强力盟友有欲望、底线和拒绝权。
- 合作需要主角支付资源或关系成本。
- 第一场胜利由主角现场修正完成。

## 期待来源

读者知道主角不能硬闯，因而期待他如何找到利益缝隙、组织他人并在计划偏离时重新夺回主动。

## 断档风险

- 事后用“早已料到”强行显智。
- 每卷靠被抓等待救援。
- 弱点从不造成真实损失。
- 盟友无条件忠诚且没有利益。
- 所有对手为显智集体降智。
- 后期全能力补满抹掉核心辨识度。

## 训练任务

写一场主角不能亲自战斗的营救，两名盟友利益不同且一人偏离计划；主角只能靠公开信息、谈判和预案修正，结尾补自保能力但不获得正面战力。
