# 天才样本-成就事实与可学证据分层

依据案例：batch-389 案例 1943

## 读者承诺

承认杰出作者可能拥有罕见天赋，同时把年龄传记、作品结果、解释假设和文本机制分层，只训练能被重复观察的部分。

## 核心循环

```text
天才样本引发比较 -> 核验事实与结果 -> 保留天赋为解释假设 -> 拆文本机制 -> 与自己的旧稿对照训练
```

## 开篇必备

- 待核验的年龄和职业事实。
- 可观察而非传闻化的作品结果。
- 对天赋解释的置信度边界。
- 连续文本中可重复的具体动作。

## 期待来源

- 传记震撼被转成证据层级。
- 不可控能力差异得到诚实承认。
- 文本分析提取出可练动作。
- 比较对象从天才年龄转回自己的旧稿。

## 断档风险

- 用年龄成就直接解释成功原因。
- 为安慰自己否认真实能力差异。
- 用“天才”停止全部文本分析。
- 把特殊样本当普通新人进度表。

## 训练任务

拆杰出作者连续三章，把每项发现分成不可复制条件和七天训练动作，评分时只与自己的旧章节比较。
