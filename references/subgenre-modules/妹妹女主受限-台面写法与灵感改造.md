# 妹妹女主受限-台面写法与灵感改造

依据案例：`batch-252` 案例 1256。

## 读者承诺

妹妹女主受限模块把高风险亲密关系灵感改造成可公开包装的安全关系，同时保留保护欲和陪伴感。

## 核心循环

```text
敏感灵感 -> 平台风险确认 -> 撤下台面卖点 -> 改成青梅/邻家关系 -> 保留情绪功能
```

## 开篇必备

- 明确不能显性包装的敏感标签。
- 找到要保留的情绪：保护、信任、陪伴。
- 换成低风险关系标签。
- 删除禁忌和露骨暗示。

## 期待来源

读者吃到的是特殊亲近感，不一定非要靠妹妹标签；青梅关系也能提供长期陪伴和唯一性。

## 断档风险

- 表面换标签，实际继续写敏感内容。
- 只规避审核，丢掉原本情绪功能。
- 用二次元作品经验直接套网文平台。
- 把高风险关系放进书名简介台面包装。

## 训练任务

把一个妹妹女主设定改成青梅女主设定，列出保留情绪、删除内容、替代互动和第一章关系信号。
