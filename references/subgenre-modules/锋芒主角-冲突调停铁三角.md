# 锋芒主角-冲突调停铁三角

依据案例：`batch-030` 案例 149。

## 读者承诺

锋芒主角负责把冲突推高，反派负责承压，调停者负责在失控前把爽点拉回主线。

## 核心循环

```text
反派压制 -> 主角激化 -> 反派害怕 -> 调停者收束 -> 主线继续
```

## 开篇必备

- 主角有锋芒人设。
- 反派有明确压制动作。
- 调停者有合理身份。
- 每轮冲突后必须回到主线。

## 期待来源

- 主角不退反进。
- 反派从压制变成害怕。
- 调停者防止主角显得无脑。
- 高频小下行和上行制造爽感。

## 断档风险

- 没调停者导致主角莽出不可挽回后果。
- 主角锋芒没有人设依据。
- 反派只会送脸。
- 冲突密集但信息不清。

## 训练任务

写 3 个短冲突，每个用一句动作完成下行，再用主角一句话或动作完成上行，最后由调停者收束。
