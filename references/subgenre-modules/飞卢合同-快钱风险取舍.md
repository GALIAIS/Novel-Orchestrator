# 飞卢合同-快钱风险取舍

依据案例：batch-302 案例 1507

## 读者承诺

平台选择要按当前阶段取舍快钱、题材适配、手速要求和合同风险。

## 核心循环

```text
查看合同 -> 标出风险条款 -> 判断自身阶段 -> 评估短期收益 -> 决定试错或转平台
```

## 开篇必备

- 记录版权、笔名、续约等条款。
- 判断自己是否适合快节奏平台。
- 明确短期目标是练节奏还是赚快钱。
- 保留转平台预案。

## 期待来源

- 作者决策不靠幻想。
- 快钱平台有清楚的收益边界。
- 长线版权风险被提前记录。

## 断档风险

- 不看合同盲签。
- 只做大神梦，忽视当前阶段。
- 把非法律讨论当专业结论。

## 训练任务

做一张平台取舍表：短期收益、合同风险、题材适配、手速要求、退出预案。
