# 竞技翻盘-至暗五杀高潮

依据案例：batch-306 案例 1530

## 读者承诺

竞技爽点不是主角顺风虐菜，而是在队友失心、敌方强势、局面崩盘时，由主角打出关键操作，带队逆风翻盘并收获双方认可。

## 核心循环

```text
逆风崩盘 -> 队友质疑 -> 主角关键操作 -> 希望回升 -> 决胜资源 -> 团战翻盘 -> 认可回收
```

## 开篇必备

- 先给清晰劣势：资源差、队友差、敌方强点。
- 安排队友不信任或嘲讽，制造证明空间。
- 中段给一个希望点，如五杀、抢龙、极限防守。
- 最终高潮必须由团队胜利兑现。

## 期待来源

期待来自低谷后的反转。队友越不信，敌人越强，主角带队赢后的认可越爽。

## 断档风险

- 只写主角强，缺少逆风压力。
- 五杀直接结束，少了最终团战兑现。
- 敌方太弱，胜利含金量不足。
- 队友和敌方没有认可回收。

## 训练任务

写一段竞技翻盘小纲：开局三重劣势，中段一个希望点，末段最终团战，并安排队友、敌方各一次认可。
