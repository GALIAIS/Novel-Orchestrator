# 主线显性化-事件收束与下一目标

依据案例：`batch-229` 案例 1145。

## 读者承诺

作者心里的主线必须在文本里反复显形：阶段事件结束后，用收获、困境和下一目标把读者重新带回方向。

## 核心循环

```text
事件结束 -> 盘点收获 -> 分析困境 -> 明确下一目标 -> 行动围绕主线
```

## 开篇必备

- 每个阶段事件有收束段。
- 主角明确知道下一步要做什么。
- 行动围绕当前目标展开。
- 读者能一句话复述主线。

## 期待来源

读者在每次事件结束后都知道主角获得了什么、还缺什么、接下来要去哪，于是愿意进入下一事件。

## 断档风险

- 作者心里有主线，但正文不说。
- 主角行为和口头目标不一致。
- 长期布局遮蔽当前方向。
- 事件结束后直接跳新事件，读者觉得散。

## 训练任务

给一章结尾补 200 字收束段：先盘点收获，再写当前困境，最后明确下一目标。写完后检查读者能否一句话说出主角接下来要做什么。
