# 游戏制作文-关系改观与团队竞技冲突

依据案例：`batch-216` 案例 1076。

## 读者承诺

用男女主被迫合作完成项目制造关系改观，再把冲突升级为团队竞技。

## 核心循环

```text
情感隔阂 -> 被迫合作 -> 小项目成果 -> 女主改观 -> 团队对抗
```

## 开篇必备

- 女主抗拒原因清楚。
- 被迫合作理由站得住。
- 小项目能证明男主能力。
- 外部团队冲突承接主角团成形。

## 期待来源

读者期待女主从不信任到改观，同时期待主角团在项目竞技里持续证明自己。

## 断档风险

- 男女主矛盾过火，损伤主角团期待。
- 小项目没有真实成果。
- 情感线和事业线分离。
- 外部竞技出现太晚或太突兀。

## 训练任务

写游戏制作文前三章小纲，分别标注关系进展、项目进度、冲突升级和读者爽点。
