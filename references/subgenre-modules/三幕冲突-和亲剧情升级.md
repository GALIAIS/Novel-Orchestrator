# 三幕冲突-和亲剧情升级

依据案例：`batch-106` 案例 526。

## 读者承诺

一卷三幕不是主角内心转三次，而是同一个核心冲突不断升级。和亲剧情要从观念冲突升级到异族、朝廷、权臣等多方压力。

## 核心循环

```text
冲突爆发 -> 第一解决 -> 外部升级 -> 伪胜利/伪危机 -> 最过激手段 -> 最终解决
```

## 开篇必备

- 核心冲突对象。
- 第一幕只解决一个小爆发。
- 第二幕引入升级压力。
- 第三幕让各方出最重手段。
- 提前埋下一卷伏笔。

## 期待来源

- 公主和亲的命运悬念。
- 异族和朝廷双重压力。
- 主角保护与建功并行。
- 每次解决后还有更大压力。

## 断档风险

- 把内心转变误当三幕。
- 救完人后没有升级事件。
- 新矛盾突兀加入。
- 没想下一卷就开始发正文。

## 训练任务

写一卷和亲剧情升级表：每一幕都必须新增一个外部压力，并改变主角行动方案。
