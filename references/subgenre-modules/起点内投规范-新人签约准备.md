# 起点内投规范-新人签约准备

依据案例：`batch-159` 案例 792。

## 读者承诺

把新人内投前的字数、大纲、卖点和材料可读性整理成清单，降低“我写了但不知道能不能投”的焦虑。

## 核心循环

```text
准备样章 -> 写外部可读大纲 -> 提炼卖点 -> 对齐平台规范 -> 找人复述 -> 修改再投
```

## 开篇必备

- 样章能展示题材承诺和主角优势。
- 简介能说明这本书好看在哪里。
- 大纲不是私人笔记，要让没看过设定的人也能读懂。
- 主线、阶段目标、冲突和结局方向要清楚。

## 期待来源

编辑或外部读者看材料时，能快速判断题材、卖点、主角、冲突和后续可写性。

## 断档风险

- 大纲只有作者自己看得懂。
- 只堆设定，不写主角目标和核心冲突。
- 只关心格式，忽略项目卖点。
- 不查平台规范，凭道听途说准备材料。

## 训练任务

把项目材料压缩成四块：300 字简介、100 字卖点、500 字大纲、前三章钩子。让一个不了解设定的人复述，复述不出主线就重写。
