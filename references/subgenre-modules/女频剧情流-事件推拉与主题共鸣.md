# 女频剧情流-事件推拉与主题共鸣

依据案例：batch-360 案例 1796

## 读者承诺

女频剧情流经验可迁移为事件推动、关系推拉、价值判断和情绪回收。剧情流重事件目标，感情流重人设关系，但顶尖作品都要让人物选择触发读者情绪。

## 核心循环

```text
人物需求 -> 事件阻碍 -> 关系/位置变化 -> 读者价值判断 -> 情绪回收
```

## 开篇必备

- 人物有可见需求。
- 事件或关系制造阻碍。
- 读者能判断谁被亏待、谁该反击。
- 爽点或痛点和人物选择绑定。

## 期待来源

- 事件升级。
- 关系位置变化。
- 价值判断带来的共鸣。
- 升级打脸被拆成可感情绪。

## 断档风险

- 只堆事件不看人物需求。
- 只讲主题不让人物选择体现。
- 照搬女频审美，不转化为本题材结构。
- 高级干货没有落成检查表。

## 训练任务

重拆一个升级打脸桥段，写清需求、阻碍、位置变化、读者价值判断、情绪回收五项。
