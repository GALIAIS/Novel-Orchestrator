# 平台选择-稿费类型完读差异

依据案例：`batch-103` 案例 514。

## 读者承诺

查稿费不能只问平台统一标准，要按平台、类型、完读转换和收入模式分别判断。尤其番茄，不同类型的完读转化差异会直接影响收益预期。

## 核心循环

```text
确定平台 -> 确定类型 -> 查公开经验 -> 对照完读/转化 -> 判断题材适配 -> 决定投稿
```

## 开篇必备

- 目标平台。
- 目标类型。
- 收入模式。
- 同类型样本。
- 完读或追读要求。

## 期待来源

- 发书前降低平台误判。
- 同类型稿费经验更可靠。
- 完读转换能解释番茄收益差异。
- 起点和番茄适合不同写法。

## 断档风险

- 问一个统一稿费标准。
- 拿都市经验套别的类型。
- 只看高收入截图，不看完读和转化。
- 平台机制和自己题材不匹配。

## 训练任务

为同一本书填起点和番茄两张平台表。只允许使用同类型资料，最后选择一个平台并写明最大风险。
