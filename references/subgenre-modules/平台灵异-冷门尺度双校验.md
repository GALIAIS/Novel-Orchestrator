# 平台灵异-冷门尺度双校验

依据案例：`batch-053` 案例 261。

## 读者承诺

灵异、规则类、都市神话复苏先做平台尺度和市场冷热双校验；能过审不代表有流量，有热度也不代表能碰现实敏感边界。

## 核心循环

```text
灵异题材 -> 查平台尺度 -> 查榜单热度 -> 判断冷门风险 -> 必要时转题材皮
```

## 开篇必备

- 明确目标平台。
- 查同平台灵异/规则类样本。
- 问清封建迷信、都市现实、后宫暧昧边界。
- 判断是否冷门单机。
- 准备玄幻皮或异能皮转译方案。

## 期待来源

- 规则类有现成好奇心。
- 灵异有恐惧和禁忌吸引力。
- 神话复苏有都市秩序崩坏期待。
- 转玄幻皮后能降低现实限制。

## 断档风险

- 冷门题材在流量平台可能没曝光。
- 都市灵异容易碰官僚腐败、社会毒瘤和封建迷信。
- 平行宇宙不一定能规避都市限制。
- 写到几万字才查规则，沉没成本高。

## 训练任务

给一个灵异题材做双校验：列出平台样本、尺度风险、市场风险和一个转译成玄幻/异能皮的版本。
