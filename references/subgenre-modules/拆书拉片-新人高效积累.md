# 拆书拉片-新人高效积累

依据案例：`batch-091` 案例 454。

## 读者承诺

初级新人不要等速成结论。人设、打斗、台词、视角切换拉经典电影；结构、开头、桥段拆大神书。自己动手拆，才能把规律变成体悟。

## 核心循环

```text
识别短板 -> 电影拉片 -> 大神书拆解 -> 写迁移练习 -> 复盘 -> 再拆
```

## 开篇必备

- 先选一个具体短板。
- 拉片学局部技法。
- 拆大神书学市场结构。
- 每次拆完都要写迁移练习。
- 不期待别人直接给速成结论。

## 期待来源

- 填充剧情更顺。
- 台词、打斗、人设更具体。
- 开头和结构更贴市场。
- 经验积累能反哺实写。

## 断档风险

- 只收藏教程不动手。
- 只看总结不拆样本。
- 拆完不写迁移。
- 初级阶段硬莽长篇。

## 训练任务

连续 3 天拆解：一天电影对白，一天电影打斗，一天大神书前三章。每天写 300 字迁移练习。
