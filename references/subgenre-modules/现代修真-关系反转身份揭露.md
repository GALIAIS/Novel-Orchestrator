# 现代修真-关系反转身份揭露

依据案例：`batch-217` 案例 1085。

## 读者承诺

现代关系被修真身份击穿，主角获得清晰行动目标和超凡爽点。

## 核心循环

```text
现实矛盾 -> 身份裂缝 -> 修真证据 -> 主角主动反应 -> 关系/世界改写
```

## 开篇必备

- 标签与开篇体验一致。
- 单视角进入关系冲突。
- 尽快给出现代修真证据。
- 主角必须推动剧情。

## 期待来源

读者因“刚要离婚发现老婆是仙师”等高概念反转，期待现实关系和超凡世界同时爆开。

## 断档风险

- 现代修真标签下写成言情轻小说。
- 多视角、梦境、动物视角乱跳。
- 主角不讨喜且无剧情功能。
- 长句和作文式描写劝退编辑。

## 训练任务

用“刚要离婚，发现老婆是仙师”写 800 字单视角开头，包含现实矛盾、修真证据、主动反应和结尾钩子。
