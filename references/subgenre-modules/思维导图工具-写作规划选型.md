# 思维导图工具-写作规划选型

依据案例：`batch-173` 案例 862。

## 读者承诺

写作规划工具要按用途选择：层级大纲、自由关系图、长期资料库、本地安全和导出维护，不是功能越多越好。

## 核心循环

```text
规划需求 -> 工具选型 -> 建立结构 -> 写作调用 -> 长期维护
```

## 开篇必备

- 先判断是层级大纲还是网状关系。
- 明确是否需要本地保存和导出。
- 长期使用要考虑会员成本和文件管理。
- 工具要服务主线、角色、伏笔和势力层级。

## 期待来源

作者期待工具减少混乱。真正有效的工具能让长篇信息在写作时被快速调用。

## 断档风险

- 只追求漂亮界面，不考虑维护。
- 在线保存带来安全和控制焦虑。
- 工具换来换去，资料分散。
- 图做完不回写剧情。

## 训练任务

用两个工具分别整理同一项目：一个做层级大纲，一个做自由关系图。比较主线、角色、伏笔、势力哪个更易维护。
