# 治愈情绪-上下行折线拉扯

依据案例：`batch-223` 案例 1114。

## 读者承诺

平淡、病痛、陪伴类题材也能靠情绪上下行产生触动、代入和释放。

## 核心循环

```text
下行 -> 小幅回暖 -> 再下行 -> 再上行 -> 高潮释放 -> 美好结局
```

## 开篇必备

- 先给一个平淡但有压力的初始状态。
- 病情、关系、氛围绑定同一条情绪线。
- 回暖不能太顺，要有再次跌落。
- 高潮释放要回扣长期陪伴。

## 期待来源

治愈不是一路变好，而是在反复下坠和被接住中，让最终好结局有重量。

## 断档风险

- 直线恶化导致压抑。
- 直线上升导致无聊。
- 笑点泪点彼此孤立，没有进入同一条折线。
- 结局美好但没有代价和积累。

## 训练任务

为一个治愈、病痛或亲密关系题材写 8 个情绪节点，至少两次“刚回暖又跌落”，最后给出高潮释放点。
