# 打斗描写-招式服务人物目的

依据案例：`batch-320` 案例 1596

## 读者承诺

打斗要服务人物形象、剧情目的和情绪强度；普通过场合理略写，情绪宣泄和高潮战斗才值得详写。

## 核心循环

```text
战斗目的 -> 人物气质 -> 招式风格 -> 详略取舍 -> 情绪/剧情回收
```

## 开篇必备

- 先明确战斗在剧情中的作用。
- 招式特效贴合人物设定。
- 体系词统一，不混乱切换武侠/玄幻概念。
- 视角稳定，画面连贯。
- 根据读者期待决定详写或略写。

## 期待来源

读者期待通过战斗看懂角色是什么人、此战解决什么问题，以及战斗后关系或局势发生什么变化。

## 断档风险

- 打斗脱离前后剧情，只剩精致动作。
- 女频非战斗主线里过度写招式。
- 普通过场详写拖节奏。
- 灵力、真气等体系混用导致画面破碎。
- 招式不服务人物形象。

## 训练任务

为一场战斗写目的卡：过路、夺宝、保护、复仇或情绪宣泄。再按目的决定 200 字略写或 800 字详写。
