# 低密度缓冲-收获盘点

依据案例：`batch-005` 案例 025。

## 读者承诺

在高密度剧情后给读者换气，同时用收获盘点、日常互动和信息整理继续提供爽点。

## 核心循环

```text
高密度冲突 -> 信息疲倦 -> 低密度缓冲 -> 收获盘点 -> 新期待埋入 -> 下一轮冲突
```

## 开篇必备

- 前一段确实有高密度信息或激烈冲突。
- 缓冲段要整理主角现状、收获和下一步。
- 日常不能空转，要承担情绪换气和新期待。

## 期待来源

- 读者享受战后收获。
- 读者整理刚发生的信息。
- 配角羡慕、耍宝、吹牛形成轻爽点。
- 缓冲末尾埋下下一轮目标。

## 断档风险

- 主角一直动，读者信息疲倦。
- 缓冲全是无效闲聊，没有收获和新期待。
- 战后不盘点，爽点少一半。
- 激烈冲突和日常不切开，情绪转换生硬。

## 训练任务

在高潮后写一段 800-1500 字缓冲，必须包含收获、反应、整理、新期待四项。
