# 新人分级-证据门槛与阶段任务路由

依据案例：`batch-400` 案例 1999

## 读者承诺

不用写作年限和一次成绩笼统定义新人，而以完成因果、承诺单元、长线产能、商业复现四级证据路由当前唯一主任务。

## 核心循环

```text
收集作品证据 -> 判定最低稳定阶段 -> 选择唯一主任务 -> 明确门槛验收 -> 维持旧能力 -> 升级或回退补缺
```

## 开篇必备

- 一级能完成可复述的三章因果单元。
- 二级能明确并连续兑现核心承诺。
- 三级能完成卷级回收且产能可持续。
- 四级能用数据、合同和完本复盘机制。
- 每阶段只有一个主训练任务。
- 升级依据可重复作品证据。

## 期待来源

创作者从“是否来得及”的不可回答焦虑，转向下一项可完成、可验证的能力门槛。

## 断档风险

- 按写作年限自动升级。
- 一次签约或爆款直接跳级。
- 同时训练过多复杂能力。
- 用日更数字掩盖结构问题。
- 商业阶段忽略合同和复现。
- 不允许回退修补退化能力。

## 训练任务

拿最近作品按四级门槛逐项附文本证据，以最低项确定当前级别；未来两周只训练一个主任务，由陌生读者或卷级结果验收。
