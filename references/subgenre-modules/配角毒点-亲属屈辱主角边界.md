# 配角毒点-亲属屈辱主角边界

依据案例：`batch-340` 案例 1698

## 读者承诺

亲属或近关系配角承受接盘、背叛、舔狗式屈辱时，必须处理它与主角尊严的距离，否则会污染读者代入。

## 核心循环

```text
配角屈辱设定 -> 判断与主角距离 -> 判断是否污染主角尊严 -> 给出切割/清算/反转 -> 降低毒性
```

## 开篇必备

- 标清受辱对象和主角关系。
- 判断读者是否会替主角丢脸。
- 叙事态度要清楚。
- 主角不能认可窝囊。
- 近关系屈辱要尽快处理。

## 期待来源

读者期待主角保持尊严和能动性；如果近亲属毒点出现，就期待主角切割、清算或反转。

## 断档风险

- 把屈辱当笑料。
- 主角替窝囊背书。
- 近关系毒点长期悬置。
- 苦情没有补偿。
- 配角问题污染主线爽感。

## 训练任务

写一个亲属配角黑历史，分别设计“弱化距离”“主角切割”“主角清算”三种处理方案，比较毒性变化。
