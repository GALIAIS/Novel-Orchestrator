# 玄幻转折-反派乱入与环境突变

依据案例：batch-380 案例 1897

## 读者承诺

玄幻意外转折要让已有局势被人物乱入、危机背景或环境规则改写。意外可以突然，但最好有一句伏笔。

## 核心循环

```text
稳定局势 -> 伏笔压力 -> 突然入场/突变 -> 规则改变 -> 主角重选策略
```

## 开篇必备

- 当前稳定目标。
- 已提过的人物或危机。
- 能改变规则的触发物。
- 主角必须重新选择的局面。

## 期待来源

- 读者以为局势可控。
- 新因素打断原计划。
- 旧判断失效。
- 主角被迫换策略。

## 断档风险

- 转折完全无根。
- 只吓人，不改变局势。
- 新反派入场后不影响目标。
- 环境突变没有规则后果。

## 训练任务

写一个秘境转折：主角刚占优势，前文提过一句的仇家入场，同时秘境规则突变，迫使主角重选目标。
