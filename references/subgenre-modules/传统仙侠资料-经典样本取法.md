# 传统仙侠资料-经典样本取法

依据案例：batch-271 案例 1351

## 读者承诺

为想写传统仙侠但世界观薄弱的新人提供样本取法：不是堆资料，而是用经典源头、现代标杆、轻松变体、侠义侧重分别补齐世界气质、规则压力、人物关系和事件生成。

## 核心循环

```text
世界观缺口 -> 选择样本功能 -> 拆出规则/气质/关系 -> 改写成自己的事件 -> 第一卷验证
```

## 开篇必备

- 明确这本仙侠的主气质：仙、侠、轻松、厚重、奇观或因果。
- 第一章要让读者看到世界规则如何压到主角身上。
- 不要用名词表开篇，要用事件展示门派、修行、恩怨或奇遇。

## 期待来源

读者期待的是一个值得神往、可持续冒险的仙侠世界。经典样本提供想象力，现代样本提供长线结构，轻松样本提供陪伴感，侠义样本提供选择压力。

## 断档风险

- 只抄境界、法宝、门派名，缺少事件功能。
- 把源头作品的旧节奏直接搬到现代网文。
- 世界观很大，但第一卷没有可执行目标。
- 仙气有了，主角选择和读者爽点不足。

## 训练任务

找三部仙侠样本，每部只拆一个功能，并写成“我第一卷可以怎么用”的事件句。
