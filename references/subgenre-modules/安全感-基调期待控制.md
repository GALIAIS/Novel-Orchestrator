# 安全感-基调期待控制

依据案例：`batch-015` 案例 072。

## 读者承诺

读者代入主角后感到局面可控还是失控，会决定作品基调。安全感越高，越偏轻松、无敌、碾压；安全感越低，越偏高压、危机、悬疑。

## 核心循环

```text
展示底牌 -> 建立安全感 -> 危机压迫 -> 期待反杀 -> 底牌兑现
```

## 开篇必备

- 明确目标安全感高低。
- 让读者知道主角至少有一个优势。
- 危机要压低情绪，但不能无凭据压垮读者。
- 基调前后一致。

## 期待来源

- 高安全感：读者想看主角怎么反杀。
- 低安全感：读者想看主角如何补回安全。
- 小无敌前置制造局部可控。
- 看似失控、实则可控形成爽点。

## 断档风险

- 高安全感文频繁让主角失控，基调崩。
- 低安全感文突然长期无敌，最大期待消失。
- 威胁没有情绪下行，安全感变化无效。
- 读者不知道主角底牌，只剩憋屈。

## 训练任务

同一场危机写两个版本：高安全感版先给小无敌，低安全感版只给一线生机。比较两版基调。
