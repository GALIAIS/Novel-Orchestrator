# 新人训练-精准方向理论实践循环

依据案例：`batch-087` 案例 431。

## 读者承诺

新人不靠盲目加码和天赋幻想成长，而是先找准写作方向，用理论和实践反复校正动作，像健身练标准姿势一样练出稳定能力。

## 核心循环

```text
确定方向 -> 学习理论 -> 小任务实践 -> 发现问题 -> 修正动作 -> 再实践
```

## 开篇必备

- 明确当前训练题材或能力点。
- 降低难度，避免一上来挑战超大题材。
- 找一个可参考高手，但不把他当假想敌。
- 每轮训练必须有反馈和修正。

## 期待来源

- 每天能看见一个动作变准。
- 理论能在实写中落地。
- 错误被及时发现，而不是靠字数堆积。
- 高手样本提供方向感。

## 断档风险

- 用天赋论给不练习找借口。
- 只冲字数，不校正姿势。
- 盲目模仿高手题材。
- 每天都在准备，迟迟不行动。

## 训练任务

做 7 天精准方向训练。每天只练一个能力点：开篇、矛盾、对话、爽点、转折、收尾、复盘。每天写 1000 字并记录一个修正动作。
