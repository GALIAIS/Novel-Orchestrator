# 拆书方法-多遍批注题材适配

依据案例：`batch-051` 案例 255。

## 读者承诺

拆书先完整读，再多遍批注；用结构图和工具辅助理解，但最终要按题材差异形成自己的方法，不把单一模板硬套所有文。

## 核心循环

```text
第一遍读完 -> 判断是否值得拆 -> 第二遍批注 -> 第三遍看结构 -> 题材适配 -> 方法沉淀
```

## 开篇必备

- 第一遍先读感受，不急着拆。
- 判断这本书是否适合自己学习。
- 第二遍标注爽点、铺垫、情绪拉扯。
- 第三遍看整体结构和细节联动。
- 拆出的模板要检查题材适配性。

## 期待来源

- 多遍阅读会不断发现新东西。
- 批注能抓住细节。
- 折线图能看清大局。
- 题材适配能避免模板变枷锁。

## 断档风险

- 没读完就拆，会拆偏。
- 只拆成绩好的书，可能学不来。
- 把“计划-意外”硬套日常恋爱，会丢掉题材味道。
- 模板凌驾作者思路，容易写成四不像。

## 训练任务

对同一本书前三章做三遍笔记：第一遍写阅读感受，第二遍写爽点和铺垫，第三遍写结构线；最后写 3 条不能直接套用的原因。
