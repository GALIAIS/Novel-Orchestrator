# 同人书名简介-原著主题日常钩子

依据案例：`batch-344` 案例 1716

## 读者承诺

同人书名和简介要用一句话召回原著大主题，再用日常反差钩子建立新期待和主角牌面。

## 核心循环

```text
原著主题 -> 日常小钩子 -> 主角身份实力 -> 原著人物重遇 -> 新爽点成立
```

## 开篇必备

- 目标读者熟悉原著主题。
- 钩子要和原著主题形成反差。
- 书名或简介暗示主角实力。
- 新身份能打开多种写法。
- 原著资产服务新期待。

## 期待来源

读者期待熟悉强角色和原著主题被日常规则重新冒犯、重新评价，产生新鲜爽感。

## 断档风险

- 只堆 IP 名词。
- 日常钩子只是玩梗，没有剧情承诺。
- 主角身份不清。
- 复刻原著，没有新问题。
- 私货压过原著读者期待。

## 训练任务

写 5 个“原著大主题 + 日常小钩子”的同人简介句，每句都暗示主角身份和原著人物会怎样被重新评价。
