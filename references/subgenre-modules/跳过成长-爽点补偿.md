# 跳过成长-爽点补偿

依据案例：`batch-031` 案例 152。

## 读者承诺

跳过初次成长可以扬长避短，但必须立刻补足当前动机和爽点，否则读者会觉得缺了一块。

## 核心循环

```text
跳过成长 -> 代入缺口 -> 当前节点 -> 动机补足 -> 爽点补偿
```

## 开篇必备

- 明确跳过了什么成长。
- 当前节点必须有强事件。
- 主角动机不能含糊。
- 避免主动认怂、主动投靠毒点。

## 期待来源

- 读者很快看到当前主线利益。
- 跳过部分后续可用回忆或短反应补。
- 表面投靠转成潜入或利用。
- 爽点足够强，带走缺失感。

## 断档风险

- 只是偷懒圆设定。
- 跳过成长后没有爽点。
- 让主角先主动投降。
- 解释太多，不用剧情说话。

## 训练任务

写一个跳过初次杀人的开篇，前 500 字必须说明当前目标，并把“投靠敌人”改成“潜入利用”。
