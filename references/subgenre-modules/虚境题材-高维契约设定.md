# 虚境题材-高维契约设定

依据案例：`batch-155` 案例 775。

## 读者承诺

虚境题材的吸引力来自未知高维世界、契约交换、现实影响和势力规则，而不是“虚境”这个标签本身。

## 核心循环

```text
进入虚境 -> 接触高维规则/存在 -> 签订契约或完成任务 -> 获得加成 -> 付出代价 -> 影响现实
```

## 开篇必备

- 虚境入口。
- 高维世界本质。
- 主角第一任务。
- 契约对象或规则。
- 加成收益。
- 代价。
- 现实世界危机。

## 期待来源

读者期待主角在未知规则中交换力量，并承担代价影响现实处境。

## 断档风险

- 只堆概念，不落到任务。
- 只讲世界观，不给主角选择。
- 契约没有代价。
- 势力设定与剧情冲突无关。

## 训练任务

设计一个虚境开篇：写清进入方式、第一任务、加成、代价、现实危机；再为一个势力填入思潮、政体、国策和起源。
