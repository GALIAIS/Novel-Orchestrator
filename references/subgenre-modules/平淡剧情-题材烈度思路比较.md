# 平淡剧情-题材烈度思路比较

依据案例：`batch-089` 案例 445。

## 读者承诺

剧情平淡先查题材烈度和选材吸睛度，再查笔力、大纲布局、期待和伏笔。拆书不是照搬，而是比较自己和优秀作者的思路差异。

## 核心循环

```text
诊断平淡 -> 提高题材烈度 -> 布局期待伏笔 -> 拆样本逻辑 -> 比较思路差异 -> 改大纲再实写
```

## 开篇必备

- 开头要先吸引人，不只追求读完后精彩。
- 题材卖点尽早抛出。
- 期待线和伏笔线写进大纲。
- 拆书时记录优秀作者为什么这样写。
- 比较自己的思路和样本思路。

## 期待来源

- 选材本身有惊人点。
- 读者前几段就知道看什么。
- 期待和伏笔有明确布局。
- 作者每次改稿都有样本参照。

## 断档风险

- 题材烈度低还压着不写。
- 拆两本就期待立刻见效。
- 写着写着回到旧思路。
- 只评价样本，不比较构建逻辑。

## 训练任务

选一个平淡片段，判断是题材、笔力、大纲还是拆书迁移问题。找一个样本事件，比较双方思路差异，再重写自己的大纲。
