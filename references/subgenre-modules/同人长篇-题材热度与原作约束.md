# 同人长篇-题材热度与原作约束

依据案例：`batch-119` 案例 593。

## 读者承诺

同人长篇承诺的是原作记忆点的新满足，而不是简单借壳。读者既要看到熟悉角色和设定被尊重，也要看到新主线能长期推动。

## 核心循环

```text
选择原作 -> 扫榜确认热度 -> 拆参照作品 -> 标出原作约束 -> 设计新主线 -> 验证长篇续航
```

## 开篇必备

- 原作核心记忆点必须尽早出现。
- 主角切入点不能破坏原作人设底线。
- 开篇矛盾要接住同人读者的熟悉期待。
- 新主线要能脱离单个名场面继续推进。
- 作者必须足够熟悉原作。

## 期待来源

- 原作遗憾被改写。
- 熟悉角色在新局面中产生新反应。
- 主角利用原作信息差破局。
- 榜单同类作品证明读者仍有需求。

## 断档风险

- 作者没读懂原作，只查设定开写。
- 原创套路迁入后变成同人毒点。
- 只有开头名场面，没有百万字主线。
- 题材热度已过，缺少当前参照。

## 训练任务

为一个同人题材写 10 行风控清单：3 个不能违背的原作点、3 个榜单参照、2 个长期矛盾、2 个自己最可能写崩的位置。
