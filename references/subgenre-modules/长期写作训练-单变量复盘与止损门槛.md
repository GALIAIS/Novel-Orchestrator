# 长期写作训练-单变量复盘与止损门槛

依据案例：`batch-417` 案例 2082

## 读者承诺

长期写作不会用累计字数冒充技能增长，也不会用一次商业失败判断天赋。完成、技巧、签约和收入目标分别建账；每个项目只验证一组明确假设，在预设里程碑获取编辑、目标读者和行为反馈。继续、转向或停止依据事先写下的门槛，旧项目的耐力、流程和素材资产被迁移，时间、财务、睡眠与健康边界得到保留。

## 核心循环

```text
把完成、技能、签约和收入目标分轨 -> 用旧稿建立基线和重复症状 -> 为当前项目写目标读者、核心承诺和单项训练假设 -> 进行短场复写和新场迁移 -> 在预设字数或时间取得外部反馈 -> 比较改动前后的具体结果 -> 按预设继续、转向或停止门槛决策 -> 归档耐力、流程与可复用资产 -> 下一项目携带已验证能力
```

目标分轨防止混账，单变量练习产生可归因学习，预设门槛减少情绪决策，资产迁移承认过去价值，安全边界保证长期可持续。

## 开篇必备

- 当前目标属于完成、技能、签约或收入中的哪一轨。
- 目标读者、核心承诺和基线样本。
- 多个旧项目重复出现的一个具体症状。
- 本轮唯一训练能力和理论假设。
- 三版同场复写与一版新场迁移。
- 编辑和目标读者的症状反馈。
- 预设字数、时间与复盘里程碑。
- 继续、转向和停止的明确条件。
- 时间、金钱、睡眠和债务上限。
- 当前项目可迁移的素材、流程和耐力资产。

## 期待来源

- 单项改写是否真的改变目标读者反应。
- 新方法能否迁移到陌生场景。
- 编辑与读者是否重复指出同一症状。
- 里程碑信号会要求继续还是转向。
- 停止当前项目后哪些资产仍然有效。
- 下一实验能否避免重复同一错误。
- 长期产出如何转成可见能力曲线。

## 断档风险

- 用总字数证明自己必然进步。
- 只读理论，没有成稿和迁移验证。
- 同时修改多项变量，无法判断原因。
- 每本书都坚持到百万字才复盘。
- 在低谷临时决定退出，在高峰临时加码。
- 把商业信号与个人完成价值混成一项。
- 用个体收入截图判断自身天赋。
- 将财务危险、睡眠不足和健康透支称为坚持。
- 停止项目时丢弃全部可复用资产。

## 训练任务

从旧稿选一个重复问题，做三版同场复写和一版新场迁移。事先写明目标读者、单变量假设、两周时间上限、反馈样本和继续/转向/停止条件；复盘分别记录技能变化、商业信号与可迁移资产，禁止用累计字数替代结果。
