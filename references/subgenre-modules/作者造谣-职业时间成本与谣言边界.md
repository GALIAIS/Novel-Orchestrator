# 作者造谣-职业时间成本与谣言边界

依据案例：`batch-258` 案例 1288。

## 读者承诺

作者造谣模块提醒写作学习回到公开文本和可验证经验，不把冒名、八卦、谣言当方法证据。

## 核心循环

```text
公开笔名 -> 冒名造谣 -> 社群误导 -> 作者澄清 -> 回到文本证据
```

## 开篇必备

- 区分公开文本和社群传言。
- 作者身份、后台、作品归属都要核验。
- 私生活八卦不能作为写作证据。
- 方法库只吸收可验证经验。

## 期待来源

学习者不被“大神内幕”带偏，把注意力放回作品、更新、公开发言和可复核方法。

## 断档风险

- 轻信冒名授课。
- 把后台截图和八卦当证据。
- 用谣言解释作品成败。
- 忽略职业作者高强度时间成本。

## 训练任务

列 5 类写作信息来源，标记公开文本或社群传言。传言类全部标为待核验，不进入方法库。
