# 反向高概念-身份反常与逻辑补洞

依据案例：`batch-218` 案例 1087。

## 读者承诺

用常识身份加反常行为制造书名钩子，并提前补好逻辑债。

## 核心循环

```text
常识身份 -> 反常行为 -> 高概念钩子 -> 好奇点击 -> 逻辑补洞
```

## 开篇必备

- 一句话能看懂反向亮点。
- 第一章解释为什么必须反常行动。
- 写清跟随者利益和心理。
- 区分搞笑版和严肃版逻辑要求。

## 期待来源

读者被荒诞问题吸引，然后追看主角如何把不可能的反向行为变成可执行计划。

## 断档风险

- 只有书名钩子，没有执行逻辑。
- 严肃文忽略军心、利益、合法性。
- 身份教育和人物思维不自洽。
- 喜剧和权谋写法混用。

## 训练任务

写 5 个身份反向行为书名钩子，并分别补最大逻辑漏洞、搞笑版处理和严肃版处理。
