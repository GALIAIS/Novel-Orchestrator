# 撒钱羞辱-后悔情绪替代恶心动作

依据案例：`batch-327` 案例 1635

## 读者承诺

撒钱、羞辱、低俗爽点要控制尺度；惩罚要匹配罪责，用后悔和心理崩溃替代生理不适动作。

## 核心循环

```text
轻视主角 -> 资源反压 -> 心理防线逐步崩 -> 后悔屈服 -> 主角好感保住
```

## 开篇必备

- 明确对方过错。
- 惩罚强度匹配罪责。
- 金钱数值贴近后续可写性。
- 用加价、话语和心理描写递进。
- 避免生理不适动作。

## 期待来源

读者期待主角用资源和话术击溃对方，而不是看主角做过分恶心的事。

## 断档风险

- 对方罪责不重，惩罚过狠。
- 低俗恶心盖过爽点。
- 主角好感被败掉。
- 撒钱数值过度膨胀，后续难写。
- 高潮来太快，缺少情绪拉扯。

## 训练任务

改写一个撒钱羞辱桥段：删掉恶心动作，保留轻视、加价施压、后悔屈服三段。
