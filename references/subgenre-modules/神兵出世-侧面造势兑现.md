# 神兵出世-侧面造势兑现

依据案例：`batch-160` 案例 799。

## 读者承诺

让神兵利器出场前已经拥有重量，出场后用实战效果兑现，而不是只靠外形和天象特效硬写厉害。

## 核心循环

```text
传闻造势 -> 众人争夺/畏惧 -> 神兵出现 -> 第一次实战 -> 局势改变
```

## 开篇必备

- 先给神兵名号和传闻。
- 写旁人争夺、避让、巴结或杀人夺宝。
- 出世可以安静，价值由铺垫承载。
- 第一次实战必须改变局势。

## 期待来源

读者先相信这件东西很重要，再等待它出现和兑现效果。

## 断档风险

- 只写光效、颜色、天气。
- 没有任何人因神兵改变行动。
- 出世后不展示效果。
- 传闻和实战能力不匹配。

## 训练任务

为一件神兵写 3 段：江湖传闻、旁人争夺、第一次实战。外形描写总字数不得超过 80 字。
