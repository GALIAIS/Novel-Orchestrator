# 游戏搞笑桥段-意外翻车与画龙点睛

依据案例：`batch-209` 案例 1042。

## 读者承诺

游戏搞笑桥段要让离谱操作一路失控，并用最后一次翻车完成笑点封口。

## 核心循环

```text
离谱目标 -> 作死操作 -> 连续失控 -> 以为结束 -> 最后补刀
```

## 开篇必备

- 目标简单可懂。
- 动作失败因果清晰。
- 过程至少两次失控。
- 结尾追加一次摔死、误伤或机关反噬。

## 期待来源

读者先看角色作死，再看局面失控，最后被意外补刀放大笑点。

## 断档风险

- 只堆圈内梗，没玩过游戏看不懂。
- 失败没有动作因果。
- 结尾没有更高一级的意外。
- 笑点只靠台词，不靠事件。

## 训练任务

写一个 600 字第四天灾副本搞笑桥段，结尾必须用一次意外翻车做画龙点睛。
