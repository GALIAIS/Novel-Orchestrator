# 轻松文毒点-连续下行反噬

依据案例：`batch-151` 案例 755。

## 读者承诺

轻松爽文可以下压主角，但不能连续多章只让主角狼狈、无实质反抗、无补偿信号。读者要看到转折、蓄力或核心卖点升级。

## 核心循环

```text
反派压迫 -> 主角受挫 -> 反抗/筹码信号 -> 转折释放 -> 核心卖点补偿
```

## 开篇必备

- 明确作品轻松基调。
- 下行情绪的章数上限。
- 每章主角主动行动。
- 关键变故铺垫。
- 章尾不能卡在纯狼狈点。
- 反杀或升级信号。

## 期待来源

读者期待短暂压抑后快速释放。如果持续下行，至少要相信主角正在蓄力，而不是作者在折磨主角和读者。

## 断档风险

- 连续七八章反派针对，主角没有明显反抗。
- 吐槽替代行动。
- 队友魔化等关键变故铺垫不足。
- 更新停在主角狼狈逃走节点。
- 轻松爽文突然变压抑虐文。

## 训练任务

检查一段 3 章危机：每章写情绪值、主角行动、补偿点、章尾节点。若连续两章下行，第三章必须加入反击、蓄力或金手指升级。
