# 新人平台-赚钱心态先行

依据案例：`batch-168` 案例 840。

## 读者承诺

新人问平台前，先校准赚钱心态、写作目标、题材方向和持续产出能力，再判断平台匹配。

## 核心循环

```text
平台焦虑 -> 赚钱心态校准 -> 题材/读者/产能自查 -> 平台匹配
```

## 开篇必备

- 先明确是否以赚钱为目标。
- 写清题材和目标读者。
- 写清日更产能和试错周期。
- 不在信息不足时直接求平台答案。

## 期待来源

作者期待找到更适合自己的投稿位置，但平台选择必须建立在作品类型和商业心态之上。

## 断档风险

- 把平台当成成功原因。
- 没有题材和产能信息就问推荐。
- 用换平台逃避作品问题。
- 未看完基础心态材料就急着开书。

## 训练任务

写一页平台选择前置自查：题材、目标读者、日更能力、赚钱预期、可接受试错周期、反馈承受能力。
