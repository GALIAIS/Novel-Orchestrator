# 平淡焦虑-鉴赏提升与曲线校准

依据案例：`batch-336` 案例 1677

## 读者承诺

越写越觉得平淡，可能是鉴赏力先提升；解决方式是用情绪曲线定位缺口，而不是整章推倒。

## 核心循环

```text
拆大神书 -> 发现自己平淡 -> 画情绪曲线 -> 定位缺口 -> 局部重写验证
```

## 开篇必备

- 区分退步和审美提高。
- 不把删稿当训练成果。
- 每章有下行、转折、上行和章末期待。
- 只改最平的一处。
- 改后重新检查曲线。

## 期待来源

作者期待把自我否定转成可执行修订项，逐步让章节有起伏。

## 断档风险

- 拆书后只焦虑。
- 写几千字就全删。
- 不画曲线，只凭感觉。
- 表达能力暂时追不上审美就放弃。
- 重写没有验证标准。

## 训练任务

给一章稿子画五点情绪曲线，只重写最平的一段，让它承担下行、转折或上行功能。
