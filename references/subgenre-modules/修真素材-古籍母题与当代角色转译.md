# 修真素材-古籍母题与当代角色转译

依据案例：batch-385 案例 1925

## 读者承诺

古籍和笔记小说提供修真母题，成熟网文提供机制参照；作者要把二者转译成符合当代审美的人物压力、选择和升级反馈。

## 核心循环

```text
古籍母题 -> 出处核验 -> 网文机制参照 -> 当代角色转译 -> 场景验证
```

## 开篇必备

- 有出处的神话或异闻母题。
- 清楚的现代规则改造。
- 人物欲望和使用代价。
- 可见的阶段反馈。

## 期待来源

- 熟悉又陌生的文化意象。
- 古规则直接压迫人物选择。
- 修炼机制产生稳定反馈。
- 更深出处逐步打开世界。

## 断档风险

- 只堆神仙妖怪名词。
- 不核验古籍版本和语境。
- 把术数材料当现实指导。
- 素材多但角色不符合当代审美。

## 训练任务

从两部古籍各选一个母题，核对出处并转成设定卡，再组合成一个有选择、代价和升级反馈的修真场景。
