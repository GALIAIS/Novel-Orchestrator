# 练级过程-玩法任务化与阶段解锁预告

依据案例：batch-392 案例 1957

## 读者承诺

提前让读者知道下一阶段会解锁的新选择，让练级过程同时推进故事，并在合理短程内用关键首秀兑现能力变化。

## 核心循环

```text
阶段解锁预告 -> 练级任务化推进剧情 -> 获得技能形态或权限 -> 关键首秀改变局势 -> 新能力打开下一阶段问题
```

## 开篇必备

- 当前能力边界与下一解锁内容。
- 练级同时承担的剧情或人物功能。
- 解锁后旧办法无法解决的首秀压力。
- 新能力的限制、代价和后续责任。

## 期待来源

- 等级变化对应新的玩法选择。
- 训练过程本身产生冲突和关系变化。
- 技能获得形成阶段满足。
- 关键首秀把面板承诺变成可见结果。

## 断档风险

- 只有数字增长没有能力变化。
- 重复修炼可整段概括。
- 新技能长期不使用。
- 首秀无压力、无限制、无后果。

## 训练任务

用三章完成解锁预告、任务化练级和技能首秀；删掉等级数字后，读者仍能从选择变化看出升级才算通过。
