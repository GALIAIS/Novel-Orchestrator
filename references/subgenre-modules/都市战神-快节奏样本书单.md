# 都市战神-快节奏样本书单

依据案例：batch-300 案例 1498

## 读者承诺

研究快节奏都市战神文，先建立样本池，再拆前三章的身份反差、羞辱、反击和钩子。

## 核心循环

```text
选样本 -> 拆前三章 -> 提取身份反差 -> 记录第一次打脸 -> 复用节奏
```

## 开篇必备

- 主角有隐藏强者身份。
- 开篇出现羞辱或压迫。
- 第一波反击来得快。
- 章末继续抛出关系或危机。

## 期待来源

- 强者归来的身份反差。
- 战神/龙王/医术标签的能力兑现。
- 妻女、家族、恩怨压力推着主角出手。

## 断档风险

- 铺垫太慢，迟迟不打脸。
- 只学标签，不学节奏。
- 样本太少，误判类型规律。

## 训练任务

拆《天神殿》《龙王令》《龙王医婿》前三章：每本只记录身份、羞辱、反击、章末钩子。
