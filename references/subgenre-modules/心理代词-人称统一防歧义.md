# 心理代词-人称统一防歧义

依据案例：`batch-073` 案例 361。

## 读者承诺

这个模块用于第三人称心理描写。它承诺的是让“他”“自己”等代词指向清晰，避免读者在主角心理段落里卡一下。

## 核心循环

```text
心理描写 -> 检查代词指向 -> 统一同段称谓 -> 朗读验证 -> 代入顺滑
```

## 开篇必备

- 同一段里代词不要来回跳。
- “他”可能误指别人时改成“自己”或主角名。
- 心理活动可单开段落。
- 必要时用引号或特殊符号标识。
- 用语音朗读检查顺口。

## 期待来源

- 读者不用判断代词指谁。
- 主角心理连续。
- 人设语言习惯稳定。
- 阅读速度不被语法打断。
- 细节顺滑提升代入。

## 断档风险

- 同段“他/自己”乱换。
- 主语省略过多。
- 心理独白像作者旁白。
- 角色说了不属于他人设的话。
- 自己读通但读者读不通。

## 训练任务

取一段 200 字心理描写，标出所有代词，逐个写指向。凡是有歧义的地方，改成主角名或统一代称。
