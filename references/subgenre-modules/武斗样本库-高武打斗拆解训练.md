# 武斗样本库-高武打斗拆解训练

依据案例：`batch-233` 案例 1162。

## 读者承诺

高武打斗要靠高质量样本拆解，重点学攻防变化、压力递进、胜负代价和爽感兑现。

## 核心循环

```text
选样本 -> 截战斗章节 -> 拆攻防节点 -> 提炼技法 -> 写原创战斗
```

## 开篇必备

- 区分高武、法术、近战类型。
- 只拆高质量战斗。
- 每场拆胜负节点。
- 不只学招式名。

## 期待来源

读者能看懂战斗压力、攻防选择和胜负变化，愿意耐心看完并觉得爽。

## 断档风险

- 战斗走马观花。
- 只堆技能名和爆炸特效。
- 法术打斗没有样本还硬学。
- 读完整本却不截取战斗节点。

## 训练任务

拆 3 场高武战斗，每场 8 节点：优势、试探、升级、受伤、变化、反转、胜负、收益。再写 1200 字原创战斗。
