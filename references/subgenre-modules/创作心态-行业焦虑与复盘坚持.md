# 创作心态-行业焦虑与复盘坚持

依据案例：batch-365 案例 1821

## 读者承诺

新人面对行业劝退和变现焦虑时，要把恐惧转成风险校准、生活兜底和失败复盘，而不是把一次心凉当成退场理由。

## 核心循环

```text
劝退信息 -> 收益焦虑 -> 现实校准 -> 保留退路 -> 扑街复盘 -> 下一本调整
```

## 开篇必备

- 明确焦虑来源。
- 承认行业不稳定。
- 保留工作和生活退路。
- 设定下一阶段目标。

## 期待来源

- 失败不是终点，而是反馈。
- 业余写作可以先从外快和兴趣开始。
- 每次扑街都能拆出可修正变量。
- 真正的长期性来自复盘习惯。

## 断档风险

- 用白金级收益要求新人阶段。
- 被单篇劝退文直接击穿心态。
- 扑街后只换题材不复盘。
- 过早全职导致生活压力吞掉创作。

## 训练任务

写一份扑街复盘表，必须列出失败证据、读者承诺误判、下一本保留项和下一本调整项。
