# 修仙战力-叠盒子尺度与体系自洽

依据案例：`batch-221` 案例 1105。

## 读者承诺

修仙战力可以升级格局，但必须和力量体系、社会结构、冲突尺度自洽。

## 核心循环

```text
战力膨胀 -> 格局升级 -> 自洽压力 -> 强者限制 -> 冲突回扣
```

## 开篇必备

- 确定最高战力尺度。
- 解释普通社会如何存在。
- 给强者行为限制。
- 战力升级回扣剧情冲突。

## 期待来源

读者期待更高境界带来更大格局，同时世界规则仍能解释为什么故事还能成立。

## 断档风险

- 只堆破坏单位。
- 高战力压垮日常和社会结构。
- 强者无约束却不碾压，显得不合理。
- 剧情、文笔、社会构造、善良主角全要导致难度过高。

## 训练任务

设计一个修仙战力体系，写每级破坏力、强者限制、普通社会存在理由，再删掉 2 个不必要战力单位。
