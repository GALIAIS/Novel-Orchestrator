# 稳态长篇-长短目标排布

依据案例：`batch-128` 案例 639。

## 读者承诺

慢节奏长篇也能持续留人，前提是长期目标、阶段目标和当前短目标始终清楚。读者可以接受稳，但不能接受不知道主线往哪走。

## 核心循环

```text
长期目标定方向 -> 阶段目标承接 -> 当前短目标抓注意力 -> 目标状态变化 -> 预告下一阶段
```

## 开篇必备

- 全书长期目标可感知。
- 当前阶段目标明确。
- 每几章都有短目标变化。
- 慢章节服务目标状态。
- 读者知道当下关心什么。

## 期待来源

- 长线方向稳定。
- 短线追读理由不断更新。
- 慢节奏不失焦。
- 阶段目标自然引向后续布局。

## 断档风险

- 只有日常，没有目标变化。
- 长期目标太远，短目标断档。
- 慢章节不服务任何推进。
- 读者注意力被支线分散。

## 训练任务

写一个 30 章大纲：列全书长期目标、前三个阶段目标，并要求每 3 章至少更新一次短期目标状态。
