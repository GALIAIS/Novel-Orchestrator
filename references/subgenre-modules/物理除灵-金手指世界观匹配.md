# 物理除灵-金手指世界观匹配

依据案例：`batch-331` 案例 1653

## 读者承诺

物理除灵的卖点是主角独有的世界观错位优势，而不是把法剑换成枪。

## 核心循环

```text
传统除灵失效 -> 主角被轻视/遇险 -> 现代物理手段登场 -> 粗暴破局 -> 围观迪化
```

## 开篇必备

- 世界默认除灵规则清楚。
- 主角物理除灵是特有能力。
- 传统方法先失败。
- 主角有危机和被轻视。
- 物理工具有前置特写。

## 期待来源

读者期待严肃除灵流程被主角简单粗暴打破，形成爽点、反差和搞笑迪化。

## 断档风险

- 人人都能物理打鬼，职业体系崩。
- 火器风格偏西幻，和本土除灵不匹配。
- 第一章解决太快，没有危机。
- 主角没有不公平伤害或被轻视。
- 金手指只是普通操作。

## 训练任务

写一个除灵开头：传统除灵者失败，主角被轻视，章末只写到他掏出物理工具准备反杀。
