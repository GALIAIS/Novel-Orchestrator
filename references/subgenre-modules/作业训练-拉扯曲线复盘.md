# 作业训练-拉扯曲线复盘

依据案例：`batch-018` 案例 087。

## 读者承诺

把情绪拉扯从“听懂概念”变成“能在章纲和正文里执行”。核心不是多学术语，而是通过作业、批改和复盘修正平坦剧情。

## 核心循环

```text
写作业 -> 标折线 -> 找平段 -> 批改反馈 -> 只改一个拐点 -> 下次复练
```

## 开篇必备

- 作业题目限定。
- 当前情节原始版本。
- 上行、下行、兑现标记。
- 明确要练的单一问题。

## 期待来源

- 平坦情节被改造成曲折情节。
- 原本只按“开端发展高潮结局”的章纲，变成能针对爽点突破的情绪曲线。
- 每次批改都能形成下一轮可执行动作。

## 断档风险

- 只看教程不写作业。
- 一次复盘想解决所有短板。
- 只会拉高情绪，不会积累和回收。
- 把转场、对白、知识储备等正文问题误判成结构问题。

## 训练任务

拿一个章纲做三色标记：上行、下行、兑现。只挑最平的一段补一个反转点，然后重写 500 字正文验证是否更有期待。
