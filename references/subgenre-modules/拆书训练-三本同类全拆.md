# 拆书训练-三本同类全拆

依据案例：`batch-330` 案例 1646

## 读者承诺

新人不要只在自己开头里打转，先拆三本同类标杆，把人设、背景、剧情和节奏拆成素材库。

## 核心循环

```text
选同类标杆 -> 拆开头/人物/故事/节奏 -> 建立对照表 -> 回看自己稿子 -> 定点修改
```

## 开篇必备

- 至少三本同类样本。
- 每本拆前三章事件链。
- 记录主角人设和开篇承诺。
- 记录节奏节点和章末钩子。
- 用标杆反查自己缺口。

## 期待来源

作者先看见成熟作品怎么组织期待，再把可复用结构迁移到自己的项目。

## 断档风险

- 把拆书当普通看书。
- 只拆黄金三章，不拆整体架构。
- 不做同类对比。
- 素材不入库，拆完就忘。
- 闭门造车，自创低效结构。

## 训练任务

选 3 本同类书，每本拆前三章五项：承诺、人设、事件、情绪折线、章末钩子。
