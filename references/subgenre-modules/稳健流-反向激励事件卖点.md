# 稳健流-反向激励事件卖点

依据案例：`batch-124` 案例 620。

## 读者承诺

稳健流的激励事件要服务“稳”和“苟”的卖点。主角越不想冒险，外部事件越要逼他在保持稳健的前提下行动。

## 核心循环

```text
稳健卖点 -> 常规危机不适配 -> 反向激励事件 -> 主角被迫行动 -> 行动仍兑现稳健 -> 结果留下新张力
```

## 开篇必备

- 主角核心卖点要明确。
- 激励事件不能破坏基调。
- 外部事件要和主角欲望反向拉扯。
- 过程要继续证明主角稳健。
- 结果要给下一轮剧情加压。

## 期待来源

- 读者期待主角如何稳中破局。
- 反向动机制造戏剧张力。
- 卖点进入事件，而不是只停在标签。
- 基调稳定增强长期信任。

## 断档风险

- 稳健主角被写成热血莽夫。
- 激励事件照搬普通升级文。
- 题材噱头强但主线构架弱。
- 反套路只有设定，没有过程拉扯。

## 训练任务

给一个稳健型主角设计激励事件：写清他不想做什么、外部事件如何逼他做、他如何用稳健方式解决。
