# 签约路径-内投普通发书同待遇

依据案例：`batch-095` 案例 474。

## 读者承诺

内投不是推荐特权，也不是签约后的待遇加成。它只是更早让编辑判断作品，甚至可能因为编辑更熟悉作者短板而更严格。新人要把精力放在作品长进，而不是路径幻想。

## 核心循环

```text
准备投稿 -> 选择内投/普通发书 -> 编辑判断作品 -> 签约后待遇一致 -> 反馈问题 -> 扫榜/换题材/大改
```

## 开篇必备

- 不把内投当捷径。
- 内投前先准备可解释的卖点、题材和前三章。
- 接受被毙时要问作品问题，不问待遇问题。
- 普通发书过签也不代表下一本内投能过。

## 期待来源

- 作者希望更早得到编辑反馈。
- 内投能提前暴露题材和开篇问题。
- 签约后进入同一推荐机制。
- 反馈倒逼作者扫榜和改题材。

## 断档风险

- 把内投失败理解成人情不够。
- 签约后期待额外待遇，忽视数据和更新。
- 上一本过签，下一本没长进。
- 编辑建议扫榜换题材，但作者只改小细节。

## 训练任务

准备一份内投自检表：题材卖点、前三章钩子、同题材对标、可能被毙理由、可大改方案。每项只写一句，写不清就先别投。
