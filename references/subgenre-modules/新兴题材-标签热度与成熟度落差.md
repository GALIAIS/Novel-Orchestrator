# 新兴题材-标签热度与成熟度落差

依据案例：`batch-232` 案例 1157。

## 读者承诺

新兴题材标签热度高不等于市场成熟；代表作少意味着模板少、套路库弱、试错成本高。

## 核心循环

```text
看到热标签 -> 查代表作 -> 判断成熟度 -> 借邻近类型 -> 决定是否进入
```

## 开篇必备

- 区分热度和成熟度。
- 找到可拆代表作。
- 没代表作时借邻近类型承诺。
- 不只写题材皮相。

## 期待来源

作者知道自己是在写成熟套路、改造邻近类型，还是承担新题材试错。

## 断档风险

- 以为热度高就一定好写。
- 没有代表作可拆。
- 只写视觉元素，不写剧情承诺。
- 新人缺少自造结构能力。

## 训练任务

评估一个新题材：找 5 本参考作；找不到就列 3 个邻近成熟类型，并写出可借的主线承诺和前三章结构。
