# 末世开局-人性冲突保压

依据案例：`batch-174` 案例 866。

## 读者承诺

末世文不能只靠打怪升级。主角开挂后，要持续用人性、资源、秩序和同类反派保住冲突压力。

## 核心循环

```text
末世降临 -> 主角开挂领跑 -> 人类冲突升级 -> 反派也有优势 -> 主角果断处理
```

## 开篇必备

- 开挂后仍要有人类威胁。
- 怪物是外压，人性是长线冲突。
- 反派可以有重生、系统、背景、资源等优势。
- 亲友、小弟和基地都能成为后期冲突入口。

## 期待来源

读者期待主角在秩序崩坏中既能打怪，也能处理圣母、坑货、背叛、资源争夺和同类强敌。

## 断档风险

- 主角太快领先，普通人威胁失效。
- 后期只刷怪刷副本，像网游文。
- 末世人性和秩序崩坏没有参与主线。
- 反派太弱，装逼对象不足。

## 训练任务

写一个末世第一卷纲：至少安排 3 次人类冲突、1 个挂逼反派、1 次资源争夺、1 次亲友危机和 1 次怪物外压。
