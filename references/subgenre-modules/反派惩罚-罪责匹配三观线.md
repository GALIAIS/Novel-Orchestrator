# 反派惩罚-罪责匹配三观线

依据案例：`batch-125` 案例 621。

## 读者承诺

主角报复必须和反派罪责匹配。对方不够可恨时，主角下重手会从爽点变成三观毒点。

## 核心循环

```text
反派过错 -> 罪责确认 -> 主角受害 -> 报复理由成立 -> 惩罚匹配 -> 读者情绪释放
```

## 开篇必备

- 写清反派是故意还是无意。
- 写清主角受害程度。
- 报复强度要匹配罪责。
- 反派可有闪光点，但主角清算理由要够。
- 角色人设变化必须有过渡。

## 期待来源

- 读者等待恶有恶报。
- 反派越明确害人，清算越稳。
- 主角三观稳，代入感才不崩。
- 罪责和惩罚匹配带来情绪闭合。

## 断档风险

- 反派只是无意过错，主角却重度报复。
- 读者同情反派，转而讨厌主角。
- 原剧情改了罪行，后续惩罚没同步改。
- 人设突然反转，无理由崩盘。

## 训练任务

为一个反派写罪行清单和惩罚清单。逐项检查惩罚是否强于罪行，强了就补故意害人证据或降低惩罚。
