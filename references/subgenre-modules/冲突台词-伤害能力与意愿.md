# 冲突台词-伤害能力与意愿

依据案例：batch-300 案例 1497

## 读者承诺

冲突台词要同时让对方相信“你做得到”和“你真的会做”，才有压迫感。

## 核心循环

```text
冲突对峙 -> 展示能力 -> 表达意愿 -> 对方重新判断 -> 行动验证台词
```

## 开篇必备

- 说话者具备真实能力。
- 台词表达明确意愿。
- 对方知道自己会受影响。
- 话后局势产生变化。

## 期待来源

- 威胁不是空喊。
- 保护承诺也有能力和意愿支撑。
- 台词后接行动，读者更代入。

## 断档风险

- 只有狠话，没有能力证明。
- 有能力但不愿行动，压力不足。
- 日常对白滥用冲突台词，显得用力过猛。

## 训练任务

写一句威胁和一句保护承诺，每句都标出能力、意愿、行动验证。
