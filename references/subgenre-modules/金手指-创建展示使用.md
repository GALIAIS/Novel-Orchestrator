# 金手指-创建展示使用

依据案例：`batch-005` 案例 021。

## 读者承诺

金手指负责第一眼吸引、主角特别感、问题解决和长线收获，是网文能不能有人看、能不能写长的核心动力之一。

## 核心循环

```text
卖点出现 -> 特别感展示 -> 解决矛盾 -> 收获盘点 -> 阶段变化 -> 新用法
```

## 开篇必备

- 金手指第一眼能被读者看懂。
- 第一次展示必须证明主角特别。
- 第一次使用要解决当前问题并带来收获。
- 不能让金手指喧宾夺主，主角仍是使用工具的人。

## 期待来源

- 读者期待主角用金手指钻空子。
- 读者期待别人发现主角特别。
- 读者期待金手指升级、变化或阶段性新用法。
- 读者期待收获感持续滚动。

## 断档风险

- 强金手指透支爽点，后期没东西可写。
- 弱金手指基本功不够，开篇吸引弱。
- 金手指只有设定，没有解决问题。
- 金手指骑到主角头上，损伤代入。

## 训练任务

写三次金手指展示：吸引、解决、升级/变化。每次必须有不同矛盾和不同收获。
