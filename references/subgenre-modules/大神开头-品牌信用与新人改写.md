# 大神开头-品牌信用与新人改写

依据案例：batch-371 案例 1852

## 读者承诺

大神能靠作者信用承受慢热和前期难懂；新人必须在前三章更早给清楚追读理由。

## 核心循环

```text
作者信用 -> 慢热开头 -> 读者坚持 -> 后续兑现 -> 口碑反哺
```

## 开篇必备

- 识别作者信用加成。
- 标出开头理解成本。
- 找到首次兑现位置。
- 改写成新人版早兑现。

## 期待来源

- 作者名本身的承诺。
- 老读者信任。
- 后续惊喜。
- 新人版则靠目标、冲突、金手指或人物。

## 断档风险

- 把大神慢热当通用技巧。
- 新人要求读者坚持十章。
- 开头听不懂还不补偿。
- 只学形式不学信用前提。

## 训练任务

拆一个大神慢热开头，标出信用支撑点，再改写成第一章内有明确追读理由的新人版。
