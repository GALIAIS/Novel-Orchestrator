# 写手健康-体能恢复码字保护

依据案例：`batch-088` 案例 437。

## 读者承诺

写手长期产出需要身体底盘：头脑清醒、手指稳定、关节不伤、第二天能恢复。健康管理不是健身炫技，而是保护码字能力。

## 核心循环

```text
评估身体 -> 合适训练 -> 拉伸恢复 -> 手指小臂保护 -> 睡眠恢复 -> 稳定码字
```

## 开篇必备

- 运动量符合自身水平。
- 肌肉训练辅以有氧。
- 体重大时避免硬跑，可选游泳。
- 每次运动后拉伸 5 到 10 分钟。
- 码字前做手指和小臂放松。

## 期待来源

- 第二天精神状态更稳。
- 手指疲劳减少。
- 睡眠和恢复改善。
- 长期码字不被伤病打断。

## 断档风险

- 有氧过量导致疲惫易怒。
- 关节不适还硬练。
- 只拉手指，不处理小臂。
- 韧带损伤不听医生建议。
- 什么都想做，最后恢复不足。

## 训练任务

做 7 天写手身体记录：运动量、拉伸、睡眠、手指疲劳、小臂紧张、第二天精神状态。目标是找出最不影响码字的运动量。
