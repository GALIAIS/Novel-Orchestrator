# 体质无敌流-被动爽点主动化

依据案例：`batch-343` 案例 1711

## 读者承诺

体质无敌流不能连续依靠站着挨打不破防，必须把被动抗伤转成主动破局、误判设局和成长用法。

## 核心循环

```text
高体质 -> 首次抗伤展示 -> 敌人误判 -> 主角主动利用 -> 反击破局 -> 新用法升级
```

## 开篇必备

- 第一场展示体质强度。
- 第二场必须换玩法。
- 主角要有主动选择。
- 敌人可恨点要明确。
- 困境解决不能拖太久。

## 期待来源

读者期待看到“打不动我”之后，主角进一步用这点反杀、硬闯、碰瓷、救人或诱敌。

## 断档风险

- 连续三章重复挨打。
- 主角没有魅力和能动性。
- 反派疯癫但不可恨。
- 数值单调无成长。
- 奇幻背景削弱折磨冲击力。

## 训练任务

写三段体质无敌递进：被动抗伤、利用敌人误判、主动进入危险场景破局。
