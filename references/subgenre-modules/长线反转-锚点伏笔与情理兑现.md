# 长线反转-锚点伏笔与情理兑现

依据案例：batch-387 案例 1934

## 读者承诺

先确定终局反转锚点，再倒推必要事实和双义线索。揭晓要出乎意料，回看又能完整成立。

## 核心循环

```text
反转锚点 -> 必要事实 -> 双义线索 -> 合理误导 -> 揭晓回看
```

## 开篇必备

- 最终真相。
- 读者当前的表层解释。
- 至少三条可感知事实。
- 能同时支持两种解释的线索。

## 期待来源

- 表层故事本身合理。
- 细小异常持续积累。
- 关键动作推翻旧解释。
- 旧线索在揭晓时同时亮起。

## 断档风险

- 突然新增设定冒充反转。
- 线索只存在于作者脑中。
- 为隐藏真相故意让角色失智。
- 揭晓后无法回看验证。

## 训练任务

先写终局真相，再倒推三条事实和两条双义线索，分散到五章并让读者初读、揭晓后各解释一次。
