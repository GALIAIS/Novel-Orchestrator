# 双向不公平伤害-生活冲突提炼

依据案例：`batch-041` 案例 201。

## 读者承诺

真实争执常常不是一方纯错，而是双方都认为自己受了不公平伤害。把这种双向委屈提炼出来，冲突会更真实。

## 核心循环

```text
信息误差 -> 双方委屈 -> 各自反击 -> 冲突升级 -> 暂时收束
```

## 开篇必备

- 双方都有合理理由。
- 误会来源清楚。
- 第一爆点具体。
- 反击不是无理取闹。

## 期待来源

- 读者理解双方。
- 冲突自然升级。
- 台词有真实火气。
- 后续可修复也可恶化。

## 断档风险

- 一方被写成纯坏。
- 真实素材未经艺术加工。
- 只有吵架没有目标。
- 误会没有后续结果。

## 训练任务

写一个双方都有理的争执桥段：每方至少有一个合理委屈和一个过激反应。
