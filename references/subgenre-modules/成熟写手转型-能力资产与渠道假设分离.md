# 成熟写手转型-能力资产与渠道假设分离

依据案例：`batch-404` 案例 2018

## 读者承诺

把旧成绩拆成可迁移能力与旧环境依赖，用成功作和近期失利作建立同维度对照；每轮只验证一个内容变量，内容成立后再单独核验渠道。

## 核心循环

```text
盘点能力资产与环境依赖 -> 对照旧成功和近期失败 -> 建立竞争失败假设 -> 单变量短样章验证 -> 目标读者反馈复现 -> 渠道契约独立核验 -> 新机制并入生产流程
```

## 开篇必备

- 保留完成、节奏和生产组织等可迁移能力。
- 标出依赖旧题材、旧读者和旧入口的惯例。
- 同维度比较成功作与失败作。
- 一轮只调整一个主要内容变量。
- 对照阅读直接服务已定位缺口。
- 当前渠道以现行合同、受众和数据核验。

## 期待来源

写手不用否定全部过去，也不再靠整体换风格碰运气；每个短实验都指出哪项旧能力仍有效、哪项假设需要替换。

## 断档风险

- 把历史收入等同于方法永久有效。
- 同时换题材、文风、渠道和更新方式。
- 看完样本凭感觉立即开长篇。
- 只学表面节奏，不测人物与中段发动机。
- 内容未验证就归因平台。
- 反复中止导致长期结构没有能力证据。

## 训练任务

用一部旧成功作和两部近期失利作比较开篇承诺、人物差异、单元循环、中段升级和完本能力；只选最低一项写两个对照样章，内容结论出来前不更换渠道变量。
