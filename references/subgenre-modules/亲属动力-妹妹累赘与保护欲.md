# 亲属动力-妹妹累赘与保护欲

依据案例：batch-298 案例 1486

## 读者承诺

用妹妹等亲属快速调动保护欲，但必须让被保护对象后续产生价值，避免长期拖油瓶。

## 核心循环

```text
亲属建立 -> 弱势遇险 -> 主角保护出手 -> 情绪兑现 -> 亲属获得后续价值
```

## 开篇必备

- 主角和亲属的情感关系可信。
- 遇险能逼出主角行动。
- 被保护对象不能只会哭和被抓。
- 中期有成长、身份、秘密或资源反哺。

## 期待来源

- 读者天然同情弱小亲属。
- 主角保护行为建立担当。
- 后续反哺让亲情线不拖剧情。

## 断档风险

- 妹妹遇险过于老套。
- 年龄太小导致长期无戏份。
- 女朋友遇险容易被骂拖油瓶。
- 弟弟保护感弱，除非特别小。

## 训练任务

设计一个妹妹保护线：开篇遇险、中期成长、后期反哺主角，各写一个事件。
