# 结构套娃-五折线稳定下限

依据案例：batch-283 案例 1412

## 读者承诺

五折线套娃的价值是稳定长期更新的剧情下限，让卷、单元、章节、段落都保持目标、意外、下行、反击和阶段满足。

## 核心循环

```text
上层目标 -> 小层计划 -> 意外打断 -> 情绪下沉 -> 积累反击 -> 回到上层推进
```

## 开篇必备

- 卷级目标先清楚。
- 小情节目标服务卷级目标。
- 下行点不脱离主角已有矛盾。
- 爽点最好调用金手指或前文收获。

## 期待来源

读者持续看到计划推进、意外改变、压力加深和主角用积累翻盘，获得稳定的追读节奏。

## 断档风险

- 只套结构名词，事件不服务主线。
- 小折线太多导致主目标模糊。
- 下行无因，爽点无铺垫。
- 追求妙手偶得，长期供给不稳定。

## 训练任务

写一个卷级五折线，再抽其中一章写章节五折线。标注章节里的每一步如何服务卷级目标。
