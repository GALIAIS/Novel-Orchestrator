# 开篇复盘-迫切不公情绪三项

依据案例：`batch-113` 案例 561。

## 读者承诺

开篇复盘要把问题拆成可改的项：迫切性够不够、不公平伤害有没有落到主角身上、情绪有没有被压住再反向抬高。

## 核心循环

```text
对照教程 -> 找缺项 -> 写复盘 -> 转修改动作 -> 再改稿
```

## 开篇必备

- 迫切性必须具体。
- 不公平伤害要实际发生。
- 情绪下压要有对象。
- 迪化前要有看不起或误判。
- 复盘写出来再改。

## 期待来源

- 作者能定位问题。
- 读者更快感到主角处境危险。
- 不公平让读者站队。
- 反向抬高制造爽点。

## 断档风险

- 只有困境设定，没有现实伤害。
- 迫切性停在可能会怎样。
- 大佬迪化前没有对比。
- 看教程后立刻乱改。

## 训练任务

对一个开头写三项复盘：迫切性、不公平伤害、情绪对比。每项给出一个可执行修改。
