# 智谋高潮-连锁瓦解替代大战

依据案例：`batch-048` 案例 236。

## 读者承诺

让读者看到主角不靠正面大战，也能凭奖励、布局和预判把敌方部署一层层拆掉，爽点是“敌人越补救越证明主角早已看穿”。

## 核心循环

```text
敌方部署 -> 主角暗中布置 -> 小计划受阻 -> 敌方补救 -> 反制显形 -> 敌方全盘瓦解
```

## 开篇必备

- 先让读者知道敌方部署强在哪里。
- 提前展示主角获得的奖励或资源，不要卷末凭空出现。
- 安排敌方至少 2 次补救动作。
- 每次补救都对应一个主角提前布下的点。
- 最后用敌方反应承认主角压制。

## 期待来源

- 读者期待大战爆发，结果看到更高级的不战而胜。
- 敌方每次失控都制造一次小爽点。
- 补救连续失败，会把主角智谋层级越抬越高。
- 反派从自信到崩溃，是高潮情绪线。

## 断档风险

- 只写结果不写敌方补救，会像作者直接宣布胜利。
- 主角布置没有前置，容易变成机械降神。
- 敌方太蠢，智谋爽会变成降智爽。
- 没有现场连锁反应，不大战会显得高潮不足。

## 训练任务

设计一个不爆发大战的卷末高潮：写出敌方原计划、3 个补救动作、3 个主角预判反制，以及最后的敌方崩溃反应。
