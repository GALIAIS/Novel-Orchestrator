# 章纲素材-写作提速流

依据案例：`batch-127` 案例 632。

## 读者承诺

写作提速先解决“写什么”，再解决“怎么展示”。章节越清楚每段要传递什么有用信息，临场憋字越少；素材越充足，正文越容易快速成形。

## 核心循环

```text
确定本章信息 -> 拆段落目的 -> 准备素材 -> 快速写草稿 -> 删除无用表达 -> 积累下次素材
```

## 开篇必备

- 本章有清楚目的。
- 每段有具体信息功能。
- 先追求易懂有用。
- 素材服务段落目的。
- 写完后删掉不服务目标的句子。

## 期待来源

- 作者不再被“好好写”卡住。
- 读者获得清楚信息和有效剧情。
- 章纲降低组织成本。
- 素材库降低表达阻力。

## 断档风险

- 只追求辞藻华丽。
- 章纲只有剧情名，没有段落目的。
- 素材不足，写一句想一句。
- 看完工具书期待立刻脱胎换骨。

## 训练任务

给一章列 8 个段落目的，每段配一个素材或动作。限时写草稿，再删掉所有不服务段落目的的句子。
