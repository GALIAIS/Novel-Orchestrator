# 分卷节奏-快慢校准

依据案例：`batch-133` 案例 661。

## 读者承诺

分卷推进既不让事件快到失重，也不让铺垫慢到无进展。每一卷都要有清楚目标、事件密度和情绪回报。

## 核心循环

```text
上一卷节奏问题 -> 本卷调速 -> 检查事件密度 -> 检查情绪回报 -> 微调快慢
```

## 开篇必备

- 明确上一卷留下的问题。
- 明确本卷新目标。
- 每章有至少一种可见推进。
- 铺垫要服务后续事件。
- 提前设置读者反馈点。

## 期待来源

- 读者知道本卷要解决什么。
- 放慢后信息更扎实。
- 提速后事件更有回报。
- 每章都有目标或情绪变化。

## 断档风险

- 第一卷太快，第二卷机械拉慢。
- 只有铺垫，没有新事件。
- 只有事件，没有代入和因果。
- 求评问题太泛，读者无法反馈。

## 训练任务

给一份分卷大纲，逐章标出新信息、新冲突、新收益、新目标，并判断该章该提速还是降速。
