# 平台福利-小站收益止损

依据案例：`batch-116` 案例 576。

## 读者承诺

平台福利变化会影响作者收益预期，但不应立刻情绪化停书。先判断当前书还有没有推荐、完结、练手和下本迁移价值，再决定是否转平台。

## 核心循环

```text
福利缩水 -> 收益预期下降 -> 横向比较平台 -> 当前书尽人事 -> 下本选择更适配平台
```

## 开篇必备

- 了解当前平台榜单和推荐规则。
- 判断当前书是否还有大推或曝光机会。
- 记录备选平台门槛。
- 当前书仍按开篇准则和连载质量执行。
- 下本再做平台迁移。

## 期待来源

- 作者不被单次规则变化打乱。
- 平台选择变成可比较表格。
- 当前书能保留完结和练手价值。
- 下本选择更稳。

## 断档风险

- 看到福利缩水立刻烂尾。
- 只看奖金，不看推荐和题材适配。
- 转平台前没准备符合平台口味的新书。
- 忽视开头几章对收益的决定性影响。

## 训练任务

给当前书写一张平台止损表：当前平台福利、推荐机会、完结价值、下本候选平台、迁移前需要补的开篇能力。
