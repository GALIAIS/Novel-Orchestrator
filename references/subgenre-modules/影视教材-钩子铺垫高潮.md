# 影视教材-钩子铺垫高潮

依据案例：`batch-040` 案例 199。

## 读者承诺

影视作品可作为网文钩子、铺垫和高潮爆发教材。重点不是复述剧情，而是拆钩子如何铺、如何误导、如何集中解决。

## 核心循环

```text
开场钩子 -> 中段铺垫 -> 误导加压 -> 高潮爆发 -> 情绪满足
```

## 开篇必备

- 记录钩子。
- 标出铺垫。
- 找误导点。
- 拆高潮如何解决前文。

## 期待来源

- 观众等待谜底。
- 铺垫持续加压。
- 高潮集中回收。
- 人性议题通过事件呈现。

## 断档风险

- 只写观后感。
- 只学题材，不学结构。
- 高潮脱离铺垫。
- 电影节奏不转译成网文章节。

## 训练任务

拆一部电影前 60 分钟：3 个钩子、3 个铺垫、1 个误导、1 个高潮预备点，并改成 5 章小纲。
