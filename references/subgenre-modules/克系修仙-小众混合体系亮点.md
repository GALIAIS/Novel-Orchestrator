# 克系修仙-小众混合体系亮点

依据案例：batch-269 案例 1343

## 读者承诺

在克苏鲁、修仙、SCP、架空近代等小众混合题材里，先证明体系和剧情好看，再让克系元素锦上添花。

## 核心循环

```text
混合元素 -> 避敏架空 -> 核心体系 -> 开篇亮点 -> 主角行动 -> 元素兑现
```

## 开篇必备

- 不强调真实地名国名。
- 前三章给出核心体系玩法。
- 克系元素要服务压力或规则。
- 开篇亮点必须比题材标签更具体。

## 期待来源

期待来自“这套混合题材怎么玩”。读者被标签吸引后，要尽快看到主角如何在规则里获得压力、收益和选择。

## 断档风险

- 元素堆叠，体系不清。
- 硬搬不可名状，网文阅读违和。
- 架空小众却没有强亮点。
- 克系理解不到位，变成毒点。

## 训练任务

把小众混合题材压成 3 个元素、1 套规则、1 个开篇事件。删掉不能推动剧情的氛围设定。
