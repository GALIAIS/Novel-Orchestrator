# 精拆取材-文字版OCR工作流

依据案例：`batch-053` 案例 262。

## 读者承诺

不可复制的小说也要变成可批注材料：文字版、TXT、OCR、截图批注都服务于多遍精读和复盘。

## 核心循环

```text
正版阅读 -> 不可复制 -> 文字版/OCR/TXT -> WPS 批注 -> 核对原文 -> 精拆沉淀
```

## 开篇必备

- 先确认正版阅读位置。
- 找可编辑文字来源或做 OCR。
- 导入 WPS 或文档工具。
- 对关键段落做批注。
- 回正版页面核对错漏。

## 期待来源

- 可编辑文本降低批注成本。
- 多遍标注能沉淀拆书证据。
- 截图批注可保留原页面语境。
- 工具链稳定后，精拆效率更高。

## 断档风险

- 文字版可能错字缺章。
- OCR 可能识别错误。
- 只处理文本不阅读，会变成工具替代精拆。
- 来源混乱会影响拆书结论。

## 训练任务

选一章不可复制正文，做 500 字 OCR 或文字版导入，批注 3 个点，并对照原文核查错漏。
