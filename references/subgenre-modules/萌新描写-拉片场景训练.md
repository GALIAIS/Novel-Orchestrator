# 萌新描写-拉片场景训练

依据案例：`batch-171` 案例 854。

## 读者承诺

萌新写不出人物、环境和过渡时，优先训练视角起点、动作顺序和镜头衔接；拉片比空泛多读更快暴露场景组织问题。

## 核心循环

```text
场景卡壳 -> 拉片拆镜头 -> 确定视角起点 -> 动作环境衔接 -> 改写成文字
```

## 开篇必备

- 每段先定视角人物。
- 先写人物感知到的第一信息。
- 小动作按顺序展开，不急着堆形容词。
- 环境描写要服务情绪或剧情。

## 期待来源

读者期待顺滑进入场景。朴素文字只要视角稳定、动作清楚，也比华丽但混乱的描写更有效。

## 断档风险

- 把描写问题误判成单纯词汇量问题。
- 反复润句，但视角起点仍不清。
- 环境描写和人物动作断开。
- 只看不写，经验无法形成。

## 训练任务

选 3 个电影片段，每个拉片 1 分钟，记录镜头、动作、环境、情绪。再选一个改写成 500 字主角视角文字。
