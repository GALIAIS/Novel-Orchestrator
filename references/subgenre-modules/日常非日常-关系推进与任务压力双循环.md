# 日常非日常-关系推进与任务压力双循环

依据案例：`batch-400` 案例 1997

## 读者承诺

让外部任务和共同生活双向供能：任务带来期限、秘密与危险，日常产生关系资源和软肋，两类内容都持续改变下一轮选择。

## 核心循环

```text
任务迫使维持关系 -> 日常目标与秘密冲突 -> 关系状态变化 -> 新资源软肋进入任务 -> 任务后果回流生活 -> 秘密与关系阶段升级
```

## 开篇必备

- 非日常任务有目标、期限和失败后果。
- 日常关系是完成任务的必要条件。
- 每场日常有小目标和不一致需求。
- 温馨或笑点留下关系状态变化。
- 任务真实使用日常建立的关系资源。
- 预先列出秘密揭露与关系阶段。

## 期待来源

读者同时等待任务是否完成、秘密是否暴露，以及临时关系何时从表演变成真实牵挂。

## 断档风险

- 日常章删掉后剧情毫无变化。
- 非日常危机随机插入提神。
- 秘密只重复制造同一种误会。
- 任务期间关系成长全部冻结。
- 温馨场景不产生承诺或债务。
- 事件发动机耗尽仍强拖篇幅。

## 训练任务

设计一章任务与两章日常：任务产生生活后果，两章日常分别生成关系资源和软肋，下一任务必须同时调用；删除任一章会断因果才通过。
