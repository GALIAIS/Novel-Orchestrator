# 第一章精拆-生死主动望远镜

依据案例：`batch-036` 案例 179。

## 读者承诺

强第一章要让主角立刻处在重大冲突里，用身份能力化被动为主动，并在短线逃生后递出长线望远镜。

## 核心循环

```text
生死冲突 -> 不公平伤害 -> 主角身份能力 -> 主动求生 -> 短线收益 -> 长线谜团
```

## 开篇必备

- 冲突发生在主角身上。
- stakes 事关生死或前途。
- 主角身份能解释行动。
- 至少回收一部分悬念。

## 期待来源

- 弱小对比让读者站队。
- 不公平伤害制造情绪。
- 主角主动选择带来代入。
- 魔纹、秘密、阴谋拉长线。

## 断档风险

- 悬念只堆不回收。
- 主角一直被剧情推着走。
- 行动没有身份逻辑。
- 短线解决后没有新期待。

## 训练任务

拆一本同题材第一章：标 5 个悬念、2 个回收、1 个主角主动选择和 1 个长线望远镜。
