# 古武机甲-情怀大场面与体系冲突

依据案例：batch-352 案例 1760

## 读者承诺

古武机甲混搭承诺的是科技极盛后的反差热血：机甲、星舰、科技文明都扛不住末日敌人时，古法传承以一人一剑补上最后缺口。它吃情怀和大场面，也必须回答科技与修行各自为什么存在。

## 核心循环

```text
科技体系先证明强 -> 敌人突破科技上限 -> 古法传承登场 -> 一人一剑反差逆转 -> 两体系重新分工
```

## 开篇必备

- 先证明机甲和科技不是废物。
- 给出科技失败的特定条件。
- 埋下古法能解决该条件的理由。
- 大场面前要有角色或传承铺垫。

## 期待来源

- 末日尺度和星空战场。
- 古法与高科技的时代反差。
- 神明机甲、星核碎片、古修传承等名词张力。
- 一人一剑对抗万妖神魔的极端画面。

## 断档风险

- 只堆大词，没有前文铺垫。
- 科技和修行强弱关系混乱。
- 机甲被写成纯陪衬，题材承诺浪费。
- 情怀段落单看燃，连载中却无法解释续航。

## 训练任务

写一个体系设定表：科技能解决什么，修行能解决什么，两者共同面对什么敌人。每栏至少写 5 条，禁止出现“都可以”。
