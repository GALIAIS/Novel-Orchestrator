# 终局反派-逼格兑现与胜利代价

依据案例：batch-382 案例 1910

## 读者承诺

长期抬高的反派必须在终局兑现战力和计划。主角可以赢，但胜利要支付与铺垫相称的代价，不能把所有成果无损打包带走。

## 核心循环

```text
抬高反派 -> 积累兑现债 -> 高烈度决战 -> 有损胜利 -> 长期余波
```

## 开篇必备

- 反派逼格证据。
- 长期计划和核心筹码。
- 主角团可能失去的东西。
- 不可逆胜利代价。

## 期待来源

- 反派本体兑现战力。
- 多年计划形成真正难题。
- 主角只能做艰难取舍。
- 胜利留下新秩序和后遗症。

## 断档风险

- 反派多年计划被轻易摘桃。
- 前期激烈，终局异常顺利。
- 主角同时杀敌、夺成果、收盟友。
- 己方损失被全部复活或恢复。

## 训练任务

重写一个终局：杀反派、保队友、夺成果三选二，留下不可逆损失，并让反派完成一次有效反制。
