# 情绪拉扯-聊天式吸引

依据案例：`batch-016` 案例 077。

## 读者承诺

把网文当成和读者聊天：先让读者感兴趣，再通过回应、撤退、再推进，持续制造情绪波动。

## 核心循环

```text
搭话 -> 引起兴趣 -> 给一点回应 -> 小撤退 -> 再抛钩子 -> 继续追读
```

## 开篇必备

- 第一句要能搭上读者兴趣。
- 不连续灌输信息。
- 每段都要有推进或反转。
- 语气服务吸引，而不是作者自嗨。

## 期待来源

- 读者感觉作者在和自己互动。
- 读者想看下一句怎么转。
- 轻松调侃降低理解压力。
- 小撤退制造继续追问的欲望。

## 断档风险

- 比喻误用到现实追人。
- 聊天感变成废话。
- 只有调侃，没有信息增量。
- 情绪波动不服务故事。

## 训练任务

把一段 500 字说明文改写成聊天式表达，每 200 字至少有一个兴趣点、一个转折或一个钩子。
