# 长篇大纲-大章主题与魂骨皮

依据案例：`batch-202` 案例 1010。

## 读者承诺

长篇大纲需要整书总主题和大章阶段主题；文笔是皮，剧情是骨，主题是魂。

## 核心循环

```text
总主题 -> 大章主题 -> 阶段矛盾 -> 事件取舍 -> 人物选择 -> 主题落点
```

## 开篇必备

- 先写整书总主题。
- 每个大章有一句阶段主题。
- 主题对应核心矛盾和人物选择。
- 细纲围绕主题筛选事件。

## 期待来源

读者会感到每个大阶段都有方向和内涵，而不是单纯事件堆叠。

## 断档风险

- 主题停在口号，不落到事件。
- 只写矛盾，不知道阶段内涵。
- 细纲枝节太多，主方向变弱。
- 文笔和设定盖过剧情骨架。

## 训练任务

给 5 卷长篇做主题表：整书总主题、每卷阶段主题、核心矛盾和一个必须出现的人物选择。
