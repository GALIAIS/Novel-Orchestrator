# 异类开局-世界观融梗适配

依据案例：`batch-095` 案例 475。

## 读者承诺

兽人、妖怪、狐仙、妖兽等异类开局不是不能写，而是旧热度不能原样复活。异类身份必须绑定一个当下读者能理解的世界观和目标，让“非人”变成事件发动机，而不是单纯外貌设定。

## 核心循环

```text
选择异类身份 -> 判断旧热度是否过气 -> 融入新世界观 -> 设计可代入目标 -> 用身份制造冲突 -> 兑现反差爽点
```

## 开篇必备

- 第一章就说明异类身份带来的具体麻烦或优势。
- 给主角一个人类读者能理解的目标：考编、上岸、赚钱、升学、求生。
- 世界观要承认异类存在，并给出规则压力。
- 旧西幻味、魔兽味不能单独当卖点。

## 期待来源

- 异类身份与现代目标的反差。
- 妖怪进入制度、职场、校园、考试等熟悉场景。
- 非人能力解决现实化问题。
- 旧元素被新梗重新包装后的新鲜感。

## 断档风险

- 只写兽人/妖怪外貌，没有事件功能。
- 读者无法代入主角目标。
- 旧题材热度退潮，作者仍按老套路写。
- 融梗只停在概念，没有转成第一章冲突。

## 训练任务

写一个异类主角开篇卡：身份、世界规则、现实化目标、第一麻烦、第一优势、第一场冲突。用这张卡写 500 字正文，要求异类身份必须推动冲突发生。
