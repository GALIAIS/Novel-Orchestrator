# 拉片素材-先定短板再选片

依据案例：`batch-322` 案例 1606

## 读者承诺

拉片前先确定自己要学什么，再选择对应素材；喜欢、热门和经典都只是入口，最终要服务当前写作短板。

## 核心循环

```text
当前短板 -> 学习目标 -> 选择同类型素材 -> 拉片总结 -> 改写进自己的作品
```

## 开篇必备

- 先写清要学的桥段。
- 优先同类型素材。
- 自己喜欢的素材更容易坚持。
- 市场热门可校准当下口味。
- 拉完必须总结并转写。

## 期待来源

作者期待通过拉片解决具体能力缺口，而不是积累片单。

## 断档风险

- 先找片再想学什么，本末倒置。
- 看完不总结。
- 只搬结构，没有加入自己的东西。
- 经典素材与当前题材完全不匹配。

## 训练任务

写一个当前短板，选一部对应作品拉 30 分钟，拆出 3 个可迁移动作。
