# 帅气主角-扫榜确认XP

依据案例：`batch-126` 案例 626。

## 读者承诺

想写帅主角，先用扫榜确认自己真正兴奋的爽点。帅不是形容词，要落到言行一致、绝境反杀、幕后迪化和升级节奏上。

## 核心循环

```text
扫榜 -> 确认 XP -> 找同类样本 -> 拆帅点场景 -> 提炼可仿写动作
```

## 开篇必备

- 主角第一印象要立住。
- 帅点必须通过行动展示。
- 绝境反杀要有低谷。
- 迪化幕后要有他人脑补反馈。
- 样本选择优先同平台榜单。

## 期待来源

- 读者期待主角强、稳、狠或神秘。
- 弱者绝境反杀提供反弹爽。
- 幕后迪化提供逼格爽。
- 言行一致建立可信帅感。

## 断档风险

- 只说主角帅，不给行为证据。
- 书单越列越多，不拆样本。
- 跟风别人的爽点，丢掉自己 XP。
- 只看设定，不看升级节奏。

## 训练任务

拆 3 本帅主角样本前 5 章，每本记录 3 个“帅的行为证据”，不要记录形容词。
