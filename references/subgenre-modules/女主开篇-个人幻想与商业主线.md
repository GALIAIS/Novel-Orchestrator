# 女主开篇-个人幻想与商业主线

依据案例：batch-375 案例 1873

## 读者承诺

女主可以写，但必须服务主线。作者个人幻想和暧昧爽感，不能分散开篇当前矛盾。

## 核心循环

```text
主线矛盾 -> 女主介入 -> 推动剧情/制造压力 -> 情绪收益 -> 主线继续上升
```

## 开篇必备

- 当前主矛盾。
- 女主出场功能。
- 与主角关系压力。
- 不分散节奏的边界。

## 期待来源

- 女主推动主线。
- 关系变化带来情绪。
- 分别后更好重逢。
- 善与美通过剧情证明。

## 断档风险

- 作者自嗨幻想。
- 暧昧戏抢走冲突。
- 女主成不能伤害的娇花。
- 个人表达压过读者口味。

## 训练任务

删掉一个女主出场桥段中只满足作者幻想的内容，只保留推动主线、制造压力或改变关系的部分。
