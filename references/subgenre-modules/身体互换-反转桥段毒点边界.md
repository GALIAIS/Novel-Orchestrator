# 身体互换-反转桥段毒点边界

依据案例：`batch-039` 案例 194。

## 读者承诺

身体互换能制造反转、搞笑和互相理解，但在男频网文里是双刃剑。平台受众和桥段边界必须先判断。

## 核心循环

```text
身体互换 -> 反差笑点 -> 彼此理解 -> 共同破局 -> 换回主线
```

## 开篇必备

- 平台受众能接受。
- 桥段功能明确。
- 篇幅短。
- 性别边界不越线。

## 期待来源

- 反差喜剧。
- 角色互相理解。
- 用对方身份解决问题。
- 换回后关系变化。

## 断档风险

- 受众不吃变身桥段。
- 猎奇压过主线。
- 异性身体使用引发毒点。
- 拖太久不换回。

## 训练任务

写三章身体互换小单元：笑点、理解、破局、换回、回主线五步必须齐全。
