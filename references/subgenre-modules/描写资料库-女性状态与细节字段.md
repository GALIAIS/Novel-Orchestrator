# 描写资料库-女性状态与细节字段

依据案例：batch-351 案例 1752

## 读者承诺

把女性描写资料转成状态字段库，用少量细节服务身份、情绪、关系和剧情功能。

## 核心循环

```text
资料样例 -> 提炼观察维度 -> 匹配人物状态 -> 压缩网文化 -> 服务场景
```

## 开篇必备

- 先定人物状态。
- 细节要指向剧情功能。
- 描写控制长度。
- 不照搬资料句式。
- 每个细节要能被后文使用。

## 期待来源

读者通过少量细节判断人物此刻的力量、脆弱、伪装或压迫感。

## 断档风险

- 堆漂亮词。
- 所有女性角色同质化。
- 资料句式不适合网文节奏。
- 描写无后续用途。
- 描写遮住冲突。

## 训练任务

做 5 类女性状态卡：惊艳、病弱、威压、疲惫、伪装。每类写 4 个可用细节。
