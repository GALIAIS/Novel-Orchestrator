# 开头小纲-结构化公开处刑

依据案例：`batch-138` 案例 690。

## 读者承诺

用结构化小纲和排查表替代直接丢正文，让公开处刑先聚焦剧情结构、金手指冲突、主角主动性和人设展示。

## 核心循环

```text
填 5 章小纲 -> 18 元素自查 -> 发公开处刑 -> 聚焦剧情反馈 -> 迭代表格
```

## 开篇必备

- 预备书名和简介。
- 前 5 章小纲。
- 第一场冒险。
- 金手指解决冲突。
- 主角主动选择。
- 风险、赌注、收益。

## 期待来源

- 评论者快速看懂结构。
- 作者先自查一遍。
- 问题集中在剧情内容。
- 新人少用花哨开头。

## 断档风险

- 倒叙、插叙、蒙太奇开局炫技。
- 没有金手指装逼冲突。
- 主角行动动机不合理。
- 类型不适配表格却硬套。

## 训练任务

填一份前 5 章小纲，并标出第一个金手指装逼冲突。按主角主动性、冲突强度、金手指作用、风险收益、人设展示五项自查。
