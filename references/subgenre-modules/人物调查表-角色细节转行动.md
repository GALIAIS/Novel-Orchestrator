# 人物调查表-角色细节转行动

依据案例：`batch-110` 案例 550。

## 读者承诺

人物调查表能提高角色细节密度，但最终必须转成行动、选择和冲突。读者记住角色，不是因为表格填得多，而是因为细节影响了他怎么做事。

## 核心循环

```text
填人物字段 -> 提炼欲望/弱点 -> 选择登场细节 -> 影响行动 -> 制造冲突
```

## 开篇必备

- 明确角色核心欲望。
- 明确最大弱点。
- 选 1-3 个可见习惯。
- 第一次登场只展示关键细节。
- 细节必须影响选择。

## 期待来源

- 角色像活人。
- 配角有独立行动逻辑。
- 细节能制造记忆点。
- 人物关系更容易产生冲突。

## 断档风险

- 表格填很多，正文用不上。
- 背景细节不影响行动。
- 第一次登场堆履历。
- 角色只有功能，没有欲望。

## 训练任务

给一个配角填 10 个调查字段，再删到 3 个登场细节。保留的每个细节都必须能改变他的一次选择。
