# 章节字数-两千字单章与四千日更

依据案例：`batch-234` 案例 1166。

## 读者承诺

常见新人更新策略是 2000 字一章、一天两到三章，至少保证约 4000 字日更量。

## 核心循环

```text
平台全勤 -> 日更底线 -> 2000字单章 -> 双更/三更 -> 稳定追更
```

## 开篇必备

- 区分单章字数和日更总量。
- 单章节奏可控。
- 日更量满足平台底线。
- 保留存稿余量。

## 期待来源

读者每天获得稳定更新，作者也能用较短单章控制小目标和结尾钩子。

## 断档风险

- 单章硬撑 3000 字导致节奏拖。
- 日更量不足影响全勤。
- 无存稿导致断更。
- 只追字数不管章节功能。

## 训练任务

把 6000 字事件切成 3 个 2000 字章节，每章写小目标、阻碍和结尾钩子。
