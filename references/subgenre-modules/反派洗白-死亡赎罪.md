# 反派洗白-死亡赎罪

依据案例：batch-296 案例 1480

## 读者承诺

罪恶反派可以临终醒悟，但必须用不可逆代价赎罪，避免主角放过恶人的毒点。

## 核心循环

```text
旧罪累积 -> 醒悟触发 -> 无法清账 -> 主动救人 -> 牺牲死亡 -> 人物升华
```

## 开篇必备

- 反派罪行必须被读者记住。
- 醒悟有触发点，不是突然变好。
- 旧债无法被一句道歉抵消。
- 最后牺牲救人要和罪行形成回应。

## 期待来源

- 读者担心强行洗白。
- 反派主动承担代价。
- 死亡让赎罪有分量。

## 断档风险

- 主角轻易原谅罪恶滔天者。
- 反派活着回归正常生活。
- 牺牲和旧罪无关，像机械退场。

## 训练任务

列出反派三条罪行，再设计一个救人牺牲动作，让牺牲对象或方式回应其中一条罪。
