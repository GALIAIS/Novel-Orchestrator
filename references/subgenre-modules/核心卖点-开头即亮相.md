# 核心卖点-开头即亮相

依据案例：batch-261 案例 1304

## 读者承诺

作品要让读者尽早知道“这本书主要爽什么”。十八元素和情绪拉扯都必须服务核心卖点，不能替代核心卖点。

## 核心循环

```text
明确核心卖点 -> 第一章亮相 -> 情绪围绕卖点拉扯 -> 大纲围绕卖点扩展 -> 追读理由稳定
```

## 开篇必备

- 一句话核心卖点。
- 第一章可见证据。
- 第一爽点。
- 情绪拉扯目的。
- 后续围绕卖点的事件链。

## 期待来源

读者期待开头就看到独特基调或玩法，并相信后文会围绕它持续展开。

## 断档风险

没有缺点不等于有卖点；元素齐全但卖点不亮，读者会觉得没必要继续看。

## 训练任务

检查第一章前三屏：是否已经出现核心卖点的动作证据。没有就重写一个展示卖点的事件。
