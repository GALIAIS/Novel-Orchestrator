# 社区秩序-挂人与奇事共赏边界

依据案例：`batch-323` 案例 1614

## 读者承诺

创作社群可以讨论离谱案例和抵制不良写作行为，但讨论必须回到学习、避坑和规则边界，不能滑向人身羞辱。

## 核心循环

```text
看到离谱案例 -> 判断是否有公共避坑价值 -> 隐去个人攻击 -> 提炼文本问题 -> 沉淀社群规则
```

## 开篇必备

- 明确讨论目的不是看乐子。
- 只描述文本行为和风险。
- 区分抄袭照搬、恶意嘲讽、普通低水平作品。
- 避免把求反馈者变成被围观对象。
- 最后落到自己以后如何避坑。

## 期待来源

成员期待社群既能保持学习秩序，也能对明显不良行为形成共识。

## 断档风险

- 带入贴吧式乐子人语气。
- 批评作品时羞辱作者。
- 炫耀成绩压人，不给具体文本建议。
- 把公共案例讨论变成挂人泄愤。
- 只围观，不沉淀任何避坑规则。

## 训练任务

把一个想挂人的案例重写成社群避坑条目：删去身份信息，只保留行为、风险、读者后果、正确做法。
