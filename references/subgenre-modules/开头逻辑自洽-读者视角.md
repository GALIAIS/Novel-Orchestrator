# 开头逻辑自洽-读者视角

依据案例：`batch-162` 案例 809。

## 读者承诺

让开头的动作、战力、动机和技能规则在当前页面成立，避免作者脑内设定和读者理解错位。

## 核心循环

```text
关键动作 -> 当前信息解释 -> 读者直觉校验 -> 毒点补偿 -> 规则自洽
```

## 开篇必备

- 动作受力符合直觉。
- 战力变化必须当场有信号。
- 人物动机不能全靠后文解释。
- 高毒点要立刻有补偿。
- 技能距离和限制要支撑情节。

## 期待来源

读者期待第一章不用读作者脑内设定，也能理解人物为什么这样行动、为什么能赢或会输。

## 断档风险

- 节奏快但逻辑断。
- 关键设定藏到第四章。
- 主角开头下跪、挨打等毒点缺少补偿。
- 技能规则临时服务剧情。

## 训练任务

抽第一章 5 个关键动作，逐条写读者当前能看到的解释。凡是解释依赖后文的，前置一个信号或删掉该动作。
