# 外貌描写-通感动作名字清单

依据案例：`batch-319` 案例 1595

## 读者承诺

外貌描写要给读者可记忆的特质：初见两点特征，通感或生活联想强化记忆，后续用动作、气味、名字继续补形象。

## 核心循环

```text
人物总纲 -> 初见两点特征 -> 通感生活联想 -> 动作互动补描 -> 名字贴合/反差强化
```

## 开篇必备

- 不写空泛的“很美”。
- 初见只挑两点最特别的外貌或气质。
- 用一个生活物或季节感建立联想。
- 小动作和体香要服务人物印象。
- 名字可贴合气质，也可制造反差。

## 期待来源

读者期待新角色一出场就有可召回的标签，并在后续互动中逐渐变立体。

## 断档风险

- 初见堆一大段辞藻。
- 美貌没有具体特质。
- 男频写女性只剩尴尬凝视。
- 没有人物总纲，后续描写不稳定。
- 细节不进动作和关系，只停在静态肖像。

## 训练任务

写 150 字女性角色初见：两处外貌特征、一个生活联想、一个小动作，并让读者记住她的名字或气质。
