# 东京妖怪开篇-设定压缩代入优先

依据案例：`batch-034` 案例 166。

## 读者承诺

都市异能、东京妖怪、灵气复苏等复合开篇，第一承诺不是“设定很多”，而是主角在清晰困境中用金手指打开新世界。

## 核心循环

```text
主角困境 -> 金手指可见 -> 少量设定服务危机 -> 主角主动求变 -> 爽点/新世界钩子
```

## 开篇必备

- 主角和金手指占主要篇幅。
- 每章只新增必要设定。
- 情绪先铺再爆。
- 配角推动冲突时必须有正常动机。

## 期待来源

- 读者看懂主角为什么被逼改变。
- 金手指先制造期待，再解决问题。
- 东京、妖怪、机械风等元素被同一主轴串起来。
- 主角决断可信，代入不断。

## 断档风险

- 元素堆满但主轴不清。
- 原主憋屈无法转化为当前主角代入。
- 主角突然杀伐果断，读者出戏。
- 官方或女配为推动剧情强行降智。

## 训练任务

写三章都市异能小纲：每章只加一个设定，主角和金手指合计占三分之二，第一波爽点前必须完成情绪铺垫。
