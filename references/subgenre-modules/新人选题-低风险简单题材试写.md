# 新人选题-低风险简单题材试写

依据案例：`batch-186` 案例 926。

## 读者承诺

新人选题先承认能力边界，优先找简单易上手题材试写，跑通完整流程。

## 核心循环

```text
能力评估 -> 教程参照 -> 平台扫题材 -> 低风险试写 -> 反馈校准
```

## 开篇必备

- 明确自己短板。
- 避开高难混合题材。
- 找平台标杆。
- 先用短开头验证主角、金手指和冲突。

## 期待来源

作者先获得可完成的正反馈，再逐步挑战更复杂题材。

## 断档风险

- 把观望变成拖延。
- 灵感冲动开高难题材。
- 只看教程不试写。
- 题材简单但卖点缺失。

## 训练任务

列 3 个想写题材，选最简单的一个写 3000 字，验证主角、金手指、冲突和第一波期待。
