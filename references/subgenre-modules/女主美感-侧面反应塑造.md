# 女主美感-侧面反应塑造

依据案例：batch-307 案例 1532

## 读者承诺

女主的美感不是外貌词堆出来的，而是角色完整性、气质、关系重量和他人反应共同证明出来的。

## 核心循环

```text
女主登场 -> 不直接喊美 -> 行为显气质 -> 他人反应付代价 -> 读者感到不可替代
```

## 开篇必备

- 给女主一个清晰气质核心。
- 用行为选择展示她的重量。
- 让配角因她改变行动。
- 外貌描写少而准，避免标签堆叠。

## 期待来源

期待来自“她为什么值得被记住”。当女主能影响他人选择、关系走向和主角命运，美感才会从描述变成感受。

## 断档风险

- 只有萌点标签，没有完整人格。
- 外貌描写很多，但没人因她改变。
- 作者反复夸美，读者没有证据。
- 女主和主线关系轻，失去不可替代性。

## 训练任务

写一个女主登场场景，不写五官，只写三个人的反应和一个实际后果。
