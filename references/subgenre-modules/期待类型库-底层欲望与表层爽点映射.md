# 期待类型库-底层欲望与表层爽点映射

依据案例：`batch-411` 案例 2055

## 读者承诺

按掌控、地位、正义、自由、获得、真相、归属、亲密、保护、支持和身份等底层期待组织爽点；先建立欠缺与情绪债，再选择多样表层动作，以可观察状态变化证明兑现。

## 核心循环

```text
选择底层期待 -> 建立欠缺与情绪债 -> 发出未来承诺信号 -> 阻碍持续否定 -> 主角通过表层动作突破 -> 资源身份关系或规则改变 -> 确认兑现 -> 未偿余量和新代价进入下一轮
```

## 开篇必备

- 一个清楚的主期待和少量次期待。
- 主角当前缺少的具体状态。
- 读者能识别的情绪债和承诺信号。
- 与人物一致的多种兑现动作备选。
- 可观察的外部状态证明。
- 已偿、未偿和升级代价记录。

## 期待来源

读者先看见主角缺少重要状态，再由信号知道未来可能获得；阻碍制造压抑，人物选择让现实真的改变，余量和新代价继续推动下一轮。

## 断档风险

- 把表面打脸动作当成完整分类。
- 没有欠缺和阻力便直接发奖励。
- 高潮汇合过多期待而没有主次。
- 只用旁观者赞叹证明爽点。
- 强者帮助完全替代主角行动。
- 亲密奖励缺少双方主体性、同意和边界。

## 训练任务

选“被认可”一种底层期待，分别用战斗、谈判和救援兑现；再把同一次公开胜利分别铺成下克上、正义清算和身份揭露，比较欠缺、阻碍与状态变化是否一致。
