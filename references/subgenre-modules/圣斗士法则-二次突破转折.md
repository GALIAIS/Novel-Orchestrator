# 圣斗士法则-二次突破转折

依据案例：`batch-125` 案例 622。

## 读者承诺

二次突破爽点来自“旧招失败、规则压制、主角改变后打破规则”。老套路仍有效，因为它稳定制造低谷和转折。

## 核心循环

```text
闯关目标 -> Boss 强大 -> 第一次技能失败 -> 主角被压到低谷 -> 定律宣布 -> 二次改变 -> 打破规则
```

## 开篇必备

- 关卡目标明确。
- Boss 强大要先展示。
- 第一次方案必须失败。
- 定律或规则要压出绝望感。
- 第二次突破要有改变，不是简单再来一次。

## 期待来源

- 读者等主角如何过关。
- 单挑焦点制造集中期待。
- 失败和低谷拉高反弹。
- 打破规则满足“例外主角”期待。

## 断档风险

- 第一次失败不够痛。
- 定律太弱，打破不爽。
- 第二次没有新理解或新代价。
- 只套皮，不写情绪压低。

## 训练任务

写一个 Boss 战十步表：抵达、展示、试探、失败、被压、生死、规则、改变、突破、胜利。
