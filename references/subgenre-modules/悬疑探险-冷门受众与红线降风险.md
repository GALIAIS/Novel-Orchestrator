# 悬疑探险-冷门受众与红线降风险

依据案例：`batch-222` 案例 1108。

## 读者承诺

用未知空间、禁忌线索、临场探险和真相分层提供紧张好奇，同时控制受众窄、主站节奏差异和恐怖灵异红线。

## 核心循环

```text
进入未知空间 -> 发现异常规则 -> 线索推进 -> 危险升级 -> 真相露一层 -> 更深谜团
```

## 开篇必备

- 给主角一个必须进入危险空间的理由。
- 第一章出现异常和禁忌。
- 危险来自线索推进，不只靠吓人。
- 恐怖元素准备现实化、机关化或人为化替代方案。

## 期待来源

读者期待空间秘密逐层打开：每次解开一个谜，都露出更大的未知。

## 断档风险

- 直接套主站升级打脸节奏，破坏悬疑气质。
- 第一人称样本照搬到不匹配的叙事结构。
- 灵异恐怖元素踩平台红线。
- 冷门题材商业盘小，长篇续航不足。

## 训练任务

拆一段悬疑探险开篇，标出进入场景、第一异常、第一规则、第一危险、第一层真相、章尾悬念；再把灵异元素改成机关、历史误会、未知生物或人为阴谋。
