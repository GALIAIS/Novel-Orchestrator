# 新闻式开篇-世界观铺垫与主角视角锚定

依据案例：`batch-205` 案例 1025。

## 读者承诺

新闻可以铺世界观和反派，但必须尽早锚定主角视角，否则读者会误认新闻人物为主角。

## 核心循环

```text
新闻背景 -> 反派信息 -> 主角看新闻/接任务 -> 视角固定 -> 冲突爆发 -> 能力展示
```

## 开篇必备

- 新闻只承担背景和引线。
- 500 字内出现主角行动。
- 主角报名字后固定视角。
- 金手指或能力尽早钩住读者。

## 期待来源

读者理解世界危险后，马上跟随主角进入任务和冲突。

## 断档风险

- 反派比主角更像主角。
- 新闻人物特殊能力太抢戏。
- 主角只剩开朗自来熟标签。
- 世界观铺开但代入点缺失。

## 训练任务

改写一段新闻式开篇：保留新闻，500 字内让主角看新闻、接任务或行动，并展示能力钩子。
