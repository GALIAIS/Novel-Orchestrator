# 市场东风-热度标签与省力入场

依据案例：`batch-235` 案例 1172。

## 读者承诺

蹭热度可以理解为借东风：借市场已有题材接受度，降低新人教育读者的成本。

## 核心循环

```text
识别热题材 -> 借标签入口 -> 顺应读者选择 -> 做差异化 -> 降低起步阻力
```

## 开篇必备

- 承认网文是商品。
- 找到平台当前热标签。
- 借热度不等于抄书。
- 热题材内仍有自家卖点。

## 期待来源

读者已经愿意点这个题材，作者不用从零解释为什么这个故事值得看。

## 断档风险

- 文青式抗拒市场。
- 逆行冷门题材硬冲。
- 只蹭标签，没有内容差异。
- 把热度当成唯一成功原因。

## 训练任务

列 5 个目标平台热标签。每个写读者点击理由、可借东风、雷点、自己的差异化卖点。
