# 法师体系-规则升级与打击感

依据案例：batch-288 案例 1437

## 读者承诺

法师题材要同时满足规则可信、升级可追、施法有代价、战斗有打击感。

## 核心循环

```text
魔力来源 -> 施法规则 -> 升级路径 -> 代价限制 -> 战斗展示 -> 破解限制
```

## 开篇必备

- 魔力来源。
- 施法媒介。
- 可理解升级线。
- 第一种代价。
- 第一章可视化法术。
- 读者期待的法师味。

## 期待来源

读者想看知识、规则和远程火力带来的压制感，同时期待主角用理解力破解施法限制。

## 断档风险

- 参考型月和魔禁却没有升级线。
- 只把修真术语换成魔法术语。
- 战斗全靠脑补，没有打击感。
- 近战法师反差过早压过法师承诺。
- 体系太杂，第一章看不懂。

## 训练任务

写一张法师体系卡：魔力来源、升级阶段、三类法术、施法代价、第一章展示法术各一行。
