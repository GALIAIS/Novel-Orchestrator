# 情绪拆解-主体分层

依据案例：`batch-027` 案例 134。

## 读者承诺

拆情绪上下行时先确认主体：主角、读者、反派、配角、围观群众可以分别上行或下行。

## 核心循环

```text
确定主体 -> 标注上行/下行 -> 观察交叉影响 -> 形成期待 -> 兑现反转
```

## 开篇必备

- 场景类型。
- 至少两个情绪主体。
- 情绪升降理由。
- 升降如何转化为读者期待。

## 期待来源

- 主角下行不等于读者下行。
- 围观误判可变成装逼期待。
- 反派上行可强化后续打脸。
- 多主体反转带来情绪拉扯。

## 断档风险

- 只写“上行/下行”，不写是谁。
- 把主角情绪等同读者情绪。
- 忽略围观群众和配角情绪。
- 兑现时没有主体反转。

## 训练任务

拆一个装逼场景，分别写主角、反派、围观群众、读者四方的上行和下行。
