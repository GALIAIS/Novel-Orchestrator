# 敏感词转译-平台位置与语义指向

依据案例：batch-377 案例 1885

## 读者承诺

敏感词风险取决于平台、出现位置和语义指向。目标是保留剧情功能，不是死保原词。

## 核心循环

```text
原词需求 -> 平台判断 -> 位置判断 -> 语义判断 -> 低风险转译
```

## 开篇必备

- 平台尺度。
- 标题/简介/正文位置。
- 现实映射风险。
- 替代表达。

## 期待来源

- 主题功能保留。
- 审核风险降低。
- 架空语境更稳。
- 敏感现实指向被隔离。

## 断档风险

- 标题使用高风险词。
- 现实政治映射太直。
- 平台尺度变化未复查。
- 只问能不能，不看语义。

## 训练任务

把含“革命/革命军”的设定改成标题、简介、正文三版低风险表达，并标注删改理由。
