# 写作笔记-前五万字自检

依据案例：`batch-051` 案例 252。

## 读者承诺

前五万字用硬自检提高成活率：砍无效铺垫、压缩纯心理描写、简化超出能力的设定、控制角色数量，并保持持续接触文本。

## 核心循环

```text
写出章节 -> 检查推进 -> 删除水分 -> 修改前五万字 -> 保持手感 -> 理论校准
```

## 开篇必备

- 铺垫要服务当前剧情或未来剧情。
- 心理描写不要连续过长。
- 设定复杂到写不顺，就改设定。
- 重要角色数量要可控。
- 每章都要有推动剧情的有效动作。

## 期待来源

- 读者更快进入剧情。
- 角色少，读者更容易记住。
- 设定顺，阅读阻力低。
- 作者持续接触文本，思路不容易断。

## 断档风险

- 慢节奏会让新人前期流失读者。
- 复杂设定超出表达能力，会拖垮整本书。
- 长时间不碰正文，手感和思路会断。
- 理论完全不合时还硬撑，问题多半在文本。

## 训练任务

用本模块检查前三章：每章列出无效铺垫、最长心理描写、出场角色数、一个必须修改的地方。
