# 暗黑仙侠-妖魔乱世选样

依据案例：batch-261 案例 1301

## 读者承诺

暗黑仙侠要给读者妖魔横行、秩序崩坏、普通人被怪力压迫的世界承诺，同时给主角一条能活下去并变强的生路。

## 核心循环

```text
乱世压迫 -> 妖魔威胁 -> 主角找到生路 -> 代价升级 -> 逐步理解世界秩序
```

## 开篇必备

- 明确的妖魔或怪力压迫。
- 普通秩序失效。
- 主角的求生目标。
- 一条可拆样本的开篇入口。
- 暗黑氛围服务行动，而不是只做背景。

## 期待来源

读者期待在恐怖压迫中看主角破局：先活下来，再变强，再摸清妖魔世界的规则。

## 断档风险

只堆暗黑氛围会显得阴沉无推进；只写升级会丢掉西游降魔/倩女幽魂式妖魔乱世承诺。

## 训练任务

拆 3 个暗黑仙侠样本第一章，记录妖魔威胁、主角处境、生路提示和世界规则暗示。
