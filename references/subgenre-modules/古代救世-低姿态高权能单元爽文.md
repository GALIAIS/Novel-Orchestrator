# 古代救世-低姿态高权能单元爽文

依据案例：`batch-227` 案例 1134。

## 读者承诺

小人物遭遇不公时，高能力救世者以低姿态入场，惩恶救善，释放公道感。

## 核心循环

```text
民间不公 -> 疯癫入场 -> 识破真相 -> 戏谑惩戒 -> 好人获救
```

## 开篇必备

- 先给小人物困境。
- 主角外在不靠谱，内在有权能。
- 恶人有短期优势。
- 裁决要正能量且有趣。

## 期待来源

读者不一定代入救世者，而是期待有人出来救济不公。

## 断档风险

- 主角只疯癫，没有智慧和善心。
- 惩恶无趣，缺少戏谑反差。
- 民间冤屈不够具体。
- 高权能没有约束，单元失去悬念。

## 训练任务

写 3000 字单元：主角表面不靠谱，实际看穿地方恶霸骗局。包含被误解、反转显能、恶人自食其果、群众情绪释放。
