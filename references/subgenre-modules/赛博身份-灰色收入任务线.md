# 赛博身份-灰色收入任务线

依据案例：`batch-124` 案例 617。

## 读者承诺

赛博无身份危机要让主角聪明生存，不要靠抢无辜者或低智赌场作弊过关。灰色委托能同时保留风险、收入和主线推进。

## 核心循环

```text
无身份被通缉 -> 缺启动资金 -> 接灰色任务 -> 金手指完成 -> 获得收入 -> 身份线索推进
```

## 开篇必备

- 身份危机要贴身。
- 资金缺口要具体。
- 主角不能越过读者底线。
- 第一个灰色任务必须用金手指完成。
- 收入要连接身份主线。

## 期待来源

- 读者等主角如何在城市夹缝生存。
- 灰色任务提供赛博味。
- 金手指首秀兑现生存能力。
- 任务成果换身份带来上行。

## 断档风险

- 主角直接抢钱崩三观。
- 赌场作弊让敌人显得弱智。
- 无身份状态拖太久但没有收益。
- 灰色任务和主线不交汇。

## 训练任务

写一个民不举官不究的赛博委托，让主角用金手指完成任务，拿到启动资金和身份线索。
