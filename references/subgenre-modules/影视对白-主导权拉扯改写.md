# 影视对白-主导权拉扯改写

依据案例：`batch-091` 案例 451。

## 读者承诺

对白不是信息传递，而是人物在关系、地位、动作和规则中争夺主导权。改变人物性格，同一场对白会变成另一种故事。

## 核心循环

```text
关系定位 -> 台词压迫 -> 动作抢位 -> 规则反压 -> 主导权转移 -> 性格改写
```

## 开篇必备

- 明确双方地位和目的。
- 每句台词都能看出谁占主导。
- 用距离、沉默、动作辅助台词。
- 让规则和身份约束进入对白。
- 改写时让性格变化带动剧情变化。

## 期待来源

- 谁会压住谁。
- 弱势方是否会夺回主导权。
- 台词背后暴露什么性格变化。
- 同阵营是否也有内部拉扯。

## 断档风险

- 对白只交代信息。
- 人物性格和台词不匹配。
- 动作与权力关系脱节。
- 改写只换台词，不换主导权。

## 训练任务

选一段双人对白，标出每句的主导权。把弱势方改成强势性格重写，要求动作、沉默和规则都跟着变化。
