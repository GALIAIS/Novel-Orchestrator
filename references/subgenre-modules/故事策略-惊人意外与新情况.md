# 故事策略-惊人意外与新情况

依据案例：batch-299 案例 1492

## 读者承诺

区分主线级转折和短期小变化，避免大纲里每个事件力度混乱。

## 核心循环

```text
事件变化 -> 判断能否退缩 -> 判断是否改处境 -> 分配主线/短期目标 -> 推进对应剧情
```

## 开篇必备

- 惊人意外必须落到主角身上。
- 惊人意外让主角不能退缩。
- 新情况可以是短期目标。
- 新情况可以发生在别人身上。

## 期待来源

- 主线转折让读者知道最终矛盾。
- 短期新情况让当前段落有事做。
- 事件强度和目标大小匹配。

## 断档风险

- 主线触发写得像小插曲。
- 小矛盾硬拔成最终 Boss。
- 读者不知道哪个目标重要。

## 训练任务

同一个事件写两版：一版只制造短期麻烦，一版彻底改变主角处境并指向最终敌人。
