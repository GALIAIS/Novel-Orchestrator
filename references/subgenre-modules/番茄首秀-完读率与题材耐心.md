# 番茄首秀-完读率与题材耐心

依据案例：`batch-240` 案例 1199。

## 读者承诺

番茄首秀完读率低要谨慎；热门脑洞更看短期起量，传统玄幻可能需要更长字数观察。

## 核心循环

```text
首秀放量 -> 完读偏低 -> 结合题材判断 -> 设观察节点 -> 继续或止损
```

## 开篇必备

- 记录首秀天数和放量。
- 看完读率与追读比例。
- 区分题材类型。
- 设定继续观察和切书条件。

## 期待来源

作者不被单日数据吓崩，也不靠幻想硬拖，而是按题材和趋势判断。

## 断档风险

- 2.5% 完读还盲目乐观。
- 用个例安慰所有低数据。
- 热门题材起不来还硬熬。
- 传统玄幻过早切掉。

## 训练任务

记录 3 天、7 天、14 天的放量、完读、追读、收入，并写清继续到多少字或低于什么指标切书。
