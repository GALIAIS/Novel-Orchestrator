# 理论内化-生活记忆非模板

依据案例：`batch-068` 案例 339。

## 读者承诺

这个模块用于学习《作家之旅》等结构理论。它承诺的是可以用个人经历帮助记忆理论，但真正写作时必须把理论拆成可变化、可组合的故事功能。

## 核心循环

```text
理论概念 -> 个人经历对应 -> 加深记忆 -> 拆成功能零件 -> 多题材变化组合
```

## 开篇必备

- 先承认个人对应只是一种理解。
- 找出它帮助记忆的概念。
- 拆出理论节点的功能。
- 至少准备三种不同题材用法。
- 不把生活经历当唯一模板。

## 期待来源

- 切身经历降低理论门槛。
- 功能拆解让理论可迁移。
- 多种组合避免模板化。
- 理论服务故事压力。
- 记忆和应用分开处理。

## 断档风险

- 只会讲自己的经历。
- 把理论当固定流程。
- 写作时照搬生活线。
- 忽视不同题材的变化。
- 理论理解深刻但无法生成故事。

## 训练任务

选一个理论节点，写 1 个个人经历例子和 3 个不同题材例子。每个例子只保留同一个功能，不允许复用相同人物关系和情境。
