# 码字设备-习惯成本核验

依据案例：`batch-057` 案例 281。

## 读者承诺

这个模块服务长期写作产能维护。它不把设备升级当成捷径，而是检查设备是否适配作者的输入习惯、久坐负担和更新节奏。

## 核心循环

```text
记录码字状态 -> 识别身体/效率痛点 -> 小范围试用设备 -> 评估适应成本 -> 固化适合自己的产能配置
```

## 开篇必备

- 先记录现有键盘、椅子、时长和身体痛点。
- 判断打字手法是否会被工学键盘强行重置。
- 游戏和码字需求分开看。
- 椅子关注坐垫、支撑和长期变形。
- 任何设备更换都要预留适应期。

## 期待来源

- 设备减少身体负担，支持长期更新。
- 输入效率不会因盲目换设备崩掉。
- 作者知道自己的产能瓶颈在哪里。
- 坐姿、键盘、节奏形成稳定组合。
- 设备选择回到数据，不被营销词牵着走。

## 断档风险

- 只因“人体工学”就购买。
- 不正规但高效的打字习惯被新键盘打断。
- 码字和游戏共用设备导致两边都不舒服。
- 坐垫变形后继续硬撑，影响久坐。
- 把设备问题误当成写作能力问题。

## 训练任务

连续 7 天记录码字设备、坐姿、字数、时长、身体不适和输入错误。第 8 天只改一个变量，比如键盘或椅垫，比较两组数据后再决定是否长期更换。
