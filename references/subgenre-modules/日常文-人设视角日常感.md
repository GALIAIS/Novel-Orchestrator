# 日常文-人设视角日常感

依据案例：`batch-086` 案例 429。

## 读者承诺

日常文承诺的是舒服、真实、轻松或酸甜的生活体验。结构可以弱，但人设、互动、视角和细节必须撑起读者继续看的理由。

## 核心循环

```text
普通场景 -> 人设差异露出 -> 自然互动 -> 视角捕捉细节 -> 情绪微变化 -> 关系继续推进
```

## 开篇必备

- 至少一个能体现人物差异的小动作或说话方式。
- 场景要有真实生活动作，不只靠对话空转。
- 叙述视角要贴近当前情绪。
- 细节要具体到光影、汗珠、食物、声音、动作等可感知对象。
- 每场日常都要有一点关系或情绪变化。

## 期待来源

- 人物相处是否有趣。
- 关系是否从陌生、熟悉、暧昧、陪伴中推进。
- 普通生活是否有真实质感。
- 细节是否带来酸甜、治愈、轻松或代入。

## 断档风险

- 贴日常标签，却没有日常卖点。
- 人物脸谱化，场景像模板。
- 同事吹水、舍友聊天写成无效闲聊。
- 只有心理描写，没有可感知细节。
- 结构弱还没有人物魅力，读者会觉得没事发生。

## 训练任务

写一个 800 字日常小场景。必须包含 3 个生活动作、2 个感官细节、1 次关系或情绪微变化；禁止用大事件推动。
