# 拉仇恨-少杀讨喜配角

依据案例：batch-294 案例 1467

## 读者承诺

提升冲突不必反复杀主角身边讨喜角色，可用被俘、被废、假死、夺宝等方式拉仇恨。

## 核心循环

```text
角色受威胁 -> 非死亡伤害 -> 主角被迫反击 -> 仇恨升级 -> 爽点兑现
```

## 开篇必备

- 被伤害角色。
- 读者感情强度。
- 非死亡伤害方式。
- 反派拉仇恨动作。
- 主角反击理由。
- 兑现高潮。

## 期待来源

读者担心喜欢的角色出事，同时期待主角救回或反击，而不是被强行喂刀。

## 断档风险

- 反复杀女角色。
- 杀讨喜角色后只靠报仇补偿。
- 杀不讨喜角色又没共鸣。
- 死亡不配大高潮分量。

## 训练任务

把“杀师妹复仇”改成被俘、假死、被废三版，各写一个反击爽点。
