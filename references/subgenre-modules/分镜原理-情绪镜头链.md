# 分镜原理-情绪镜头链

依据案例：`batch-008` 案例 039。

## 读者承诺

每个镜头都承载情绪、信息或氛围。读者不是看角色空说话，而是看角色在场景刺激下产生反应、判断、行动和下一轮碰撞。

## 核心循环

```text
场景刺激 -> 生理反应 -> 主观判断 -> 语言/动作 -> 对方反应 -> 情绪升级
```

## 开篇必备

- 每个关键人物当前情绪明确。
- 人物分镜优先写情绪变化。
- 辅助分镜补足信息和氛围。
- 对话前后必须有动作、心理、关联或反应。

## 期待来源

- 读者能感到人物情绪正在变化。
- 每个镜头都把冲突往前推一点。
- 对方反应会引出下一句、下一招、下一场景。
- 分镜越清楚，正文越容易写长且不水。

## 断档风险

- 对话删掉后，镜头没有任何意义。
- 情绪只有标签，没有生理反应和主观判断。
- 辅助分镜只写景，不服务信息和氛围。
- 连续镜头之间没有情绪递进。

## 训练任务

拿一段 500 字对话，删掉全部引号内容。若剩下内容不能构成情绪链，就按“场景刺激、生理反应、主观判断、对方反应”补足分镜。
