# 公开处刑规则-两万字五章省评审

依据案例：`batch-344` 案例 1719

## 读者承诺

公开处刑要求两万字或五章以上，是为了让评审看见开篇承诺、节奏续航和核心问题。

## 核心循环

```text
按规则整理 -> 两万字/五章 -> 材料易读 -> 星友愿意看 -> 反馈更多更准
```

## 开篇必备

- 正文够两万字或五章。
- 书名简介小纲齐全。
- 文件手机和电脑都易打开。
- 作者疑问明确。
- 不让评审猜背景和格式。

## 期待来源

作者期待发书前获得一轮真实反馈；评审期待材料省事、范围足够、问题明确。

## 断档风险

- 两三章就求评。
- 链接或文件打不开。
- 没有小纲和问题。
- 评审无法判断续航。
- 提交成本省了，反馈质量也低。

## 训练任务

整理一次公开处刑提交包，让陌生读者检查能否快速打开、知道你问什么、读到五章并判断节奏。
