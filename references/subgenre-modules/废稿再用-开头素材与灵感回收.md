# 废稿再用-开头素材与灵感回收

依据案例：`batch-242` 案例 1210。

## 读者承诺

废弃开头不是失败垃圾，而是可以被保存、复盘、魔改并转化为新书开头的素材库。

## 核心循环

```text
反复打磨开头 -> 产生废稿 -> 保存旧稿 -> 过后翻看 -> 提取元素 -> 魔改成新开头
```

## 开篇必备

- 保留废稿原始版本。
- 标注失败原因。
- 提取仍有价值的元素。
- 换题材、人物或矛盾重新使用。

## 期待来源

作者不再把写坏开头视为纯损耗，而是把每次失败沉淀成下一次开头的燃料。

## 断档风险

- 原样复制废稿问题。
- 只囤不复盘。
- 反复磨一个开头导致停滞。
- 没区分可保留元素和必须删除的问题。

## 训练任务

整理 5 个废弃开头，每个提取一个可保留元素，并改写成完全不同题材的 100 字新开头。
