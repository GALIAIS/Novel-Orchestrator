# 困境领袖-声望服从建立

依据案例：`batch-088` 案例 440。

## 读者承诺

权力真空中的主角不是被白送领导权，而是通过解决困境、建立群众基础、拉亲信、压反对者和持续表现从容，逐步让众人服从。

## 核心循环

```text
权力真空 -> 初步解困 -> 声望上升 -> 拉亲信 -> 服从性测试 -> 排除异己 -> 更大危机掌权
```

## 开篇必备

- 原高层死亡或失能，形成权力真空。
- 群体有共同困境。
- 主角至少一项能力过人。
- 第一次解决问题不要全解决。
- 必须有反对者和服从性测试。

## 期待来源

- 主角如何证明自己。
- 群众何时开始依赖他。
- 反对者如何挑战。
- 主角如何压住内乱。
- 更大危机中如何正式掌权。

## 断档风险

- 群众无理由拥戴。
- 主角没有过人之处。
- 问题一次解决，后续没领导空间。
- 权斗写得过深，作者驾驭不了。

## 训练任务

设计 5 章上位小单元：权力真空、初步解困、拉亲信、压反对者、更大危机正式掌权。
