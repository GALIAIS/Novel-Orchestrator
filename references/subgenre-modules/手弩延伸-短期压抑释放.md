# 手弩延伸-短期压抑释放

依据案例：`batch-136` 案例 676。

## 读者承诺

用负面情绪拉高爽感，但压抑和释放必须在读者耐受范围内完成。读者可以被拉弦，不能被拉断。

## 核心循环

```text
危机预感 -> 亲友/敌人拉弦 -> 主角触发行动 -> 短期释放 -> 爽感回收
```

## 开篇必备

- 拉弦对象与主角关系紧密。
- 锅由敌人、环境或配角因素承担。
- 主角有行动触发点。
- 配角塑造要短而完整。
- 释放不要拖太久。

## 期待来源

- 亲近关系带来的代入。
- 危机将至的压迫。
- 主角觉悟或成长。
- 反击释放负面情绪。

## 断档风险

- 负面情绪反复拉到读者断弦。
- 主角背锅过重变毒点。
- 压抑跨太久，爽点失效。
- 配角塑造拖成支线。

## 训练任务

设计一章配角危机桥段：用过去/现在对比完成塑造，由敌人或环境拉起负面情绪，并在同章或当天更新内释放。
