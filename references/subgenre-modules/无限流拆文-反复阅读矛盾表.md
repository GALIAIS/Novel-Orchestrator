# 无限流拆文-反复阅读矛盾表

依据案例：`batch-153` 案例 763。

## 读者承诺

无限流拆文迷茫时，不先求抽象答案，而是把一卷反复读两三遍，用矛盾冲突表把主线、副本线和情绪递进填出来。

## 核心循环

```text
读一遍 -> 填副本目标/收获 -> 再读 -> 补矛盾层级/情绪递进 -> 主要矛盾显形
```

## 开篇必备

- 全书或卷级主要矛盾。
- 每个副本目标。
- 副本内部一二三级矛盾。
- 轮回者/队友冲突。
- 空间规则压力。
- 副本间情绪递进。

## 期待来源

作者期待从一堆理论回到文本证据，真正看懂无限流如何层层推进。

## 断档风险

- 只看理论，不填表。
- 拆一遍就急着下结论。
- 副本目标和主线矛盾分不清。
- 只拆事件，不拆情绪递进。

## 训练任务

选一卷无限流读两遍。第一遍填副本目标和收获，第二遍填一二三级矛盾和情绪递进。最后用一句话概括卷级主要矛盾。
