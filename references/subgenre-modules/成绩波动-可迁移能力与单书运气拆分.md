# 成绩波动-可迁移能力与单书运气拆分

依据案例：`batch-395` 案例 1971

## 读者承诺

不让一次爆款或扑街替作者下总判决，而是拆分基本功、市场适配、执行和外部波动，用跨项目复测确认真正可迁移的创作能力。

## 核心循环

```text
单书结果拆因 -> 建立可观察指标 -> 上线前短测 -> 正式项目收集反馈 -> 抽取候选机制 -> 换题材复做 -> 更新能力账本
```

## 开篇必备

- 明确目标读者和本书承诺。
- 基本功指标能由读者行为观察。
- 市场假设与文本执行分栏记录。
- 书名、简介、首章和前三章设短测门。
- 正式连载指标区分点击、追读和长线兑现。
- 成功机制准备跨题材复测。

## 期待来源

作者每轮都知道正在验证哪一项能力，读者则持续获得更清楚、更稳定的题材承诺和兑现节奏。

## 断档风险

- 把一次高收入直接解释成稳定实力。
- 把一次失败归结为全盘没有天赋。
- 用热点掩盖首章与结构问题。
- 只写十几万字后才第一次收反馈。
- 把短测当成真实连载的完整替代。
- 坚持重复同一错误却不更换假设。

## 训练任务

对最好与最差的旧项目分别做四栏归因，抽取一个候选机制换题材写 3000 字；只有新样本再次产生目标读者体验，才把该机制记入可迁移能力账本。
