# 救世主神豪-低位逆袭开篇

依据案例：`batch-024` 案例 117。

## 读者承诺

先让主角从现实低位逆袭，再把个人逆袭升级成拯救世界。

## 核心循环

```text
低位压迫 -> 绝境边缘 -> 系统降临 -> 第一份资源 -> 身边打脸 -> 长期救世目标
```

## 开篇必备

- 主角最惨的现实状态。
- 具体压迫者，如房东、同学、邻居。
- 金手指延后到情绪最低点。
- 第一份能立刻改善处境的奖励。

## 期待来源

- 标题给出宏大身份，正文给出低位反差。
- 读者期待压迫者被反打。
- 金钱、身体、资源奖励带来即时爽感。
- 长线从自救转向救世，有成长空间。

## 断档风险

- 主角像 NPC，只旁观设定。
- 末日开局嘻嘻哈哈，基调虚假。
- 系统一上来解释太多。
- 世界目标太大，前三章没有身边爽点。

## 训练任务

写一个救世主开篇小纲，第一章不许解释世界末日，只写主角低位；第二章再让系统降临并兑现第一轮逆袭。
