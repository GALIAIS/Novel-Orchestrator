# 写作结构-三柱代入基调

依据案例：`batch-134` 案例 670。

## 读者承诺

用剧情、人物、世界三根柱子承载情绪，再用代入感、基调、舒适亲切、流畅精确托住阅读体验。写作问题可以按层定位。

## 核心循环

```text
读者心理 -> 情绪拉扯 -> 剧情/人物/世界 -> 代入感 -> 基调 -> 流畅精确
```

## 开篇必备

- 明确目标读者心理。
- 明确主要情绪。
- 剧情有波折和布线。
- 人物满足预期且有特色。
- 世界设定服务情绪。
- 文字流畅精确。

## 期待来源

- 基调吸引试读。
- 画面和视角建立代入。
- 剧情、人物、世界持续拉扯情绪。
- 文字贴近目标读者认知。

## 断档风险

- 只堆设定，不服务情绪。
- 人物没特色或不满足类型预期。
- 画面感弱，读者进不去。
- 文笔不流畅、不精确，读者卡住。

## 训练任务

选一章正文，按读者心理、情绪、剧情、人物、世界、代入感、基调、流畅、精确九项自检，每项写一句问题和一句修改动作。
