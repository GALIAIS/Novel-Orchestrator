# 长篇冲突-主要冲突卷级递进

依据案例：`batch-127` 案例 631。

## 读者承诺

长篇读者要看到每卷不是随机冒险，而是在同一个主要冲突路标下逐步逼近最终舞台。卷级冲突可以换敌人、换地图、换任务，但解决后必须让主角离整本书目标更近。

## 核心循环

```text
主要冲突定路标 -> 拆卷级次要冲突 -> 用次要矛盾合理化 -> 卷内爆发解决 -> 推近最终舞台
```

## 开篇必备

- 明确主角长期目标。
- 给出主角为什么必须选择这条路。
- 第一卷冲突能代表全书方向。
- 卷内敌人不是随机路人。
- 胜利后能看见下一阶段舞台。

## 期待来源

- 读者知道主角终点。
- 每卷都有阶段性胜利。
- 次要冲突持续服务主要冲突。
- 境界、地图、势力递进带来长线推进感。

## 断档风险

- 主线目标不清，卷级剧情乱漂。
- 每卷只换名字打脸，没有舞台递进。
- 用主要矛盾硬拆每卷，操作绕远。
- 次要冲突解决后不靠近最终问题。

## 训练任务

写一个长篇三卷表：整本书主要冲突、前三卷次要冲突、每卷次要矛盾、卷尾主角离最终目标更近的一步。
