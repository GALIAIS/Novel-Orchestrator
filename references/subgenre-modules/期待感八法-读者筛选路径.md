# 期待感八法-读者筛选路径

依据案例：`batch-047` 案例 231。

## 读者承诺

期待感方法要服务目标读者。先选网站和读者，再选择高潮、好奇心、矛盾、危机、代入、向往、引诱或系统推动。

## 核心循环

```text
目标读者 -> 期待类型 -> 高潮逆推 -> 伏笔埋线 -> 读者筛选
```

## 开篇必备

- 明确目标网站。
- 明确目标读者。
- 选一种主期待。
- 第一章让读者看懂并想追。

## 期待来源

- 高潮点提前被暗示。
- 好奇心被噱头调动。
- 冲突和危机带来压力。
- 系统推动降低新人操作难度。

## 断档风险

- 怪读者看不懂。
- 没选目标读者。
- 期待类型混乱。
- 只设计高潮，不倒推伏笔。

## 训练任务

为同一题材写 8 个开篇钩子，分别对应八种期待感塑造法。
