# 主角动机-考试作弊与三观风险

依据案例：batch-298 案例 1489

## 读者承诺

涉及作弊、透视、偷看等金手指时，主角动机必须正当，否则读者会先讨厌主角。

## 核心循环

```text
主角困境 -> 金手指诱惑 -> 正当动机约束 -> 付出代价 -> 读者认同逆袭
```

## 开篇必备

- 主角提升成绩有正当压力。
- 金手指不能只用来占便宜。
- 主角需要付出努力或承担代价。
- 目标最好关联家庭、尊严、改变命运。

## 期待来源

- 读者认同主角为什么必须变强。
- 小聪明服务奋斗，而不是猥琐。
- 金手指有限制，逆袭更可信。

## 断档风险

- 作弊提分为了把妹。
- 透视加痞子主角的过时模板。
- 主角轻浮，读者不愿追随。

## 训练任务

把一个作弊金手指改成“救急但有代价”的版本：写动机、限制、付出、读者认同点。
