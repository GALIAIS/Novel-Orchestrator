# 新人入行-扑街反馈与理论内化

依据案例：batch-376 案例 1880

## 读者承诺

新人不能靠等人喂饭和抄作业入行。扑街反馈能逼迫作者回到理论、拆书、实写和复盘。

## 核心循环

```text
自信开写 -> 市场扑街 -> 认识差距 -> 理论拆书 -> 实写复盘
```

## 开篇必备

- 明确学习大纲。
- 每天拆书和实写。
- 失败反馈记录。
- 周期复盘。

## 期待来源

- 从失败中校准方向。
- 先写大众故事。
- 再融入个人故事。
- 能独立判断问题。

## 断档风险

- 只问不写。
- 视频不看、文不码。
- 抄作业替代能力。
- 加入社群却不按大纲行动。

## 训练任务

做 30 天入行计划，每天安排理论、拆书、写作、复盘，7 天一次用反馈调整方向。
