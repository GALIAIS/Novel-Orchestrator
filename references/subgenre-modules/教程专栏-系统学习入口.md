# 教程专栏-系统学习入口

依据案例：`batch-126` 案例 627。

## 读者承诺

新手问题多时，先走系统教程入口。按顺序学习能建立共同语言，比东拼西凑碎片技巧更有效。

## 核心循环

```text
问题很多 -> 读置顶教程 -> 按顺序学习 -> 做笔记 -> 按规矩提问 -> 进入训练
```

## 开篇必备

- 先找到教程入口。
- 按顺序看，不跳着拼。
- 学前判断是否适合自己。
- 做笔记而不是只刷视频。
- 提问前看发帖规矩。

## 期待来源

- 系统路线降低迷茫。
- 共同术语提高讨论效率。
- 修订教程提供最新入口。
- 学完后问题会更具体。

## 断档风险

- 到处搜碎片技巧。
- 看过视频却没有笔记和训练。
- 没看规则就发散提问。
- 把教程当万能答案，不做实践。

## 训练任务

列出当前 10 个写作问题，逐一对应教程章节。按顺序学完后，只保留 3 个仍无法解决的问题再提问。
