# 恐怖氛围-怪异陌生化虚晃

依据案例：`batch-336` 案例 1679

## 读者承诺

恐怖气氛要把熟悉日常陌生化，用怪异、依靠失效、虚晃一枪和危险逼近剥夺安全感。

## 核心循环

```text
正常日常 -> 怪异违背常理 -> 主角寻找解释 -> 依靠或解释失效 -> 危险逼近
```

## 开篇必备

- 熟悉场景或物件。
- 一处违反常理的怪异。
- 主角给出合理化解释。
- 解释被反转推翻。
- 逃避空间或心理安全感缩小。

## 期待来源

读者期待自己刚建立的安全判断被推翻，继续追问这个世界到底哪里不对。

## 断档风险

- 只写黑暗、冷风、怪声。
- 太早摆出鬼。
- 怪异没有递进。
- 依靠对象永远安全。
- 小说照搬电影音效吓人。

## 训练任务

写 600 字灵异场景，不写黑夜、冷风、鬼影，只让一个日常物件按不合逻辑的方式运行，并推翻主角两次解释。
