# 硬汉战神-真金手指直给

依据案例：`batch-043` 案例 213。

## 读者承诺

硬汉、卧底、燕双鹰式战神文不能只靠“能打”。真正的新鲜卖点必须在开篇直给，让读者知道主角靠什么超出普通战神。

## 核心循环

```text
硬汉身份 -> 无解困境 -> 真金手指出现 -> 爽点破局 -> 长线用途
```

## 开篇必备

- 书名说清卖点。
- 第一章出现真金手指或强预告。
- 打斗前有代入关系。
- 解说功能用自然人物承载。

## 期待来源

- 普通能力解决不了当前困境。
- 新金手指改变战局。
- 读者期待高科技/能力后续用途。
- 身份反差和保护关系形成爽点。

## 断档风险

- 真卖点藏太久。
- 主角能打但没有差异化。
- 科幻背景破坏兵王代入。
- 助手只为解释而存在。

## 训练任务

写一个硬汉战神第一章困境：必须让普通战斗失效，再用真金手指解决。
