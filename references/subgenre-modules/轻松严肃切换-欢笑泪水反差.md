# 轻松严肃切换-欢笑泪水反差

依据案例：`batch-321` 案例 1601

## 读者承诺

轻松主基调可以承载严肃高潮，但严肃情绪要从前期喜剧角色、关系和伏笔里自然生长，并在危机解除后回落。

## 核心循环

```text
轻松日常 -> 喜剧关系铺垫 -> 危机加剧 -> 严肃高潮回收 -> 危机解除后回落主基调
```

## 开篇必备

- 稳定主基调。
- 喜剧桥段同时承担关系铺垫。
- 严肃危机由剧情压力推动。
- 高潮回收前期滑稽或轻松素材。
- 设计危机后的回落桥段。

## 期待来源

读者期待前期的欢笑在后期变成情感重量，哭完之后作品仍回到原本可追的主基调。

## 断档风险

- 严肃高潮像换了一个作者。
- 高潮后切不回轻松。
- 前期搞笑只玩梗，不承担关系。
- 只想升华主题，忽略读者原本来看的口味。

## 训练任务

写一个“先笑后哭”的小纲：3 个轻松桥段、1 个严肃高潮、1 个危机解除后的回落桥段。
