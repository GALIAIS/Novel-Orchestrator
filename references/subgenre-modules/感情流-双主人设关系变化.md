# 感情流-双主人设关系变化

依据案例：`batch-097` 案例 485。

## 读者承诺

感情流写的是人物、感情和关系的变化。剧情不是独立热闹，而是为了丰满两位主角、推动他们互相看见、互相改变、互相治愈或互相伤害。

## 核心循环

```text
双主人设核心 -> 相遇理由 -> 危难纠缠 -> 各自高光 -> 关系变化 -> 离别/误会 -> 再次选择 -> 情感结局
```

## 开篇必备

- 两位主角都要有核心魅力。
- 两人都不能只有恋爱，必须有自己的目标和生命。
- 初遇要有纠缠理由。
- 剧情必须同时服务人设或关系推进。

## 期待来源

- 双主角互补和反差。
- 危难中相爱。
- 用对方视角看见高光。
- 离别和误会拉开关系变化。
- 伤痛、目标、秘密带来的故事感。

## 断档风险

- 一人出彩，另一人黯淡。
- 剧情突然上线下线，工具味重。
- 作者上帝视角让角色爱得太理所当然。
- 爱意写得太直白，苏感流失。
- 人设设太细，写作中反而被束缚。

## 训练任务

做一张 CP 双主人设卡。每人写核心、目标、伤痛、缺口、高光。再写 5 个剧情事件，每个事件都要标注推动了谁的人设、推动了哪一段关系变化。
