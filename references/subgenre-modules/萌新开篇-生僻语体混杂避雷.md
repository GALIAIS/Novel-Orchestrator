# 萌新开篇-生僻语体混杂避雷

依据案例：batch-311 案例 1554

## 读者承诺

萌新开篇先保证读者读得懂、代入得进。生僻字、谐音脏话、古风句式和轻小说口癖混杂，会抢走主线注意力。

## 核心循环

```text
开篇第一段 -> 去掉生僻/谐音/突兀语体 -> 补齐处境冲突期待 -> 代入稳定
```

## 开篇必备

- 少用生僻字。
- 谐音脏话能不用就不用。
- 语体统一，不乱混古风和现代吐槽。
- 先对齐 18 元素。

## 期待来源

期待来自清楚的处境和冲突。读者先顺利进入故事，才会接受金手指和设定。

## 断档风险

- 第一页就让读者查字。
- 谐音词显得幼稚或出戏。
- 古风句式突然插进现代吐槽。
- 节奏乱，代入感差。

## 训练任务

把前三百字改成直白版：只保留主角醒来、当前冲突、目标和期待，删除所有抢戏语体。
