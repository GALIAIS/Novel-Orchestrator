# 长线伏笔-道具压轴高潮

依据案例：`batch-131` 案例 654。

## 读者承诺

关键道具可以提前很久埋下，让读者猜用途、挂期待，最终在高潮中成为胜负手。爽感来自“原来早就准备好了”的回看满足。

## 核心循环

```text
关键道具前置 -> 读者猜用途 -> 小高潮抬势 -> 至暗压低 -> 道具压轴兑现 -> 尾声收束
```

## 开篇必备

- 道具辨识度高。
- 首次出现时不立刻用完。
- 读者能猜到至少两种用途。
- 高潮有至暗低点。
- 最终胜负手来自前文伏笔。

## 期待来源

- 长线挂期待。
- 读者参与猜测。
- 明线冲突和暗线伏笔合流。
- 大高潮兑现前文准备。

## 断档风险

- 道具临时出现。
- 伏笔太隐，读者忘记。
- 高潮平推，没有低点。
- 兑现和前文无关。

## 训练任务

写一个 2000 字小剧情：主角前期获得一个用途不明的物件，中段展示至少两种可能用途，结尾按小高潮、至暗时刻、大高潮、尾声完成兑现。
