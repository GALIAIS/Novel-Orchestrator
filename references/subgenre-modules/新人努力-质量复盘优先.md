# 新人努力-质量复盘优先

依据案例：`batch-335` 案例 1672

## 读者承诺

新人努力要进入写作、反馈、复盘、修正闭环，更新量和多开不能替代质量判断。

## 核心循环

```text
写一段 -> 拿反馈 -> 定位短板 -> 对标样本 -> 修一处 -> 再验证
```

## 开篇必备

- 明确当前最短板能力。
- 不用多开稀释精力。
- 更新量服从质量底线。
- 每次扑街都写复盘结论。
- 用同赛道样本校准方向。

## 期待来源

读者期待作者每轮练习都解决一个真实问题，而不是靠日更字数感动自己。

## 断档风险

- 把高字数当进步。
- 反复写会写的内容。
- 扑街后不总结。
- 四开、多开、当枪手消耗精力。
- 写百万字仍不面对短板。

## 训练任务

选一个扑街章节，标出三个弃读点，只重写最致命的一处，并用榜单样本验证改动方向。
