# 都市异能-设定事件化开篇

依据案例：`batch-021` 案例 103。

## 读者承诺

让都市异能开篇先用事件展示金手指，而不是连续两章讲世界观、图书馆、系统和伏笔。

## 核心循环

```text
日常善行 -> 积分反馈 -> 兑换能力 -> 进入冒险 -> 翻盘奖励
```

## 开篇必备

- 一个能触发金手指的日常行动。
- 金手指第一次反馈。
- 马上可用的兑换物或能力。
- 第一场冒险困境。

## 期待来源

- 读者看到能力如何解决问题。
- 设定通过行动一点点露出。
- 冒险奖励拉起后续目标。

## 断档风险

- 世界观介绍太长。
- 金手指说明太长。
- 冲突偏平淡，蒙混过关不爽。
- 伏笔太早，占掉开篇爽点。

## 训练任务

把 1000 字设定说明改成一个事件：主角做事得积分、兑换能力、进入危险场景、用能力翻盘。
