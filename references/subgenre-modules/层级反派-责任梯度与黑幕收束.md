# 层级反派-责任梯度与黑幕收束

依据案例：`batch-396` 案例 1977

## 读者承诺

让反派苦衷扩大因果而不稀释问责：每层敌人都有外部压力、主动选择和对应代价，局部案件先兑现，再由证据把主角引向更高黑幕。

## 核心循环

```text
局部伤害 -> 追查动机与异常 -> 当前反派按责任付代价 -> 证据指向上层 -> 幸存者转盟友 -> 敌我双线围绕共同目标收束
```

## 开篇必备

- 第一层伤害和受害者损失具体。
- 低层反派的苦衷不取消主动选择。
- 本层结束前完成问责或补偿。
- 上层黑幕留下可重复的异常痕迹。
- 主角基于证据提出假设并主动追查。
- 隐藏反派有独立目标和行动进度。

## 期待来源

读者先获得局部正义释放，再发现相似恶行背后存在共同结构；每拿下一层，都得到更接近终极责任者的证据、盟友和范围升级。

## 断档风险

- 每层都用苦衷完成洗白。
- “被控制”免除所有个人责任。
- 主角随机撞进下一个势力。
- 局部反派未处理就急着升级。
- 每次战后集中播放背景回忆。
- 终极黑幕没有早期行动轨迹。

## 训练任务

设计四层反派，为每层写压力、主动伤害、应付代价和上层证据；用三个事件验证“先问责、再升级”，并让一名低层角色以技能和债务加入主线。
