# 西幻开篇-短句事件威胁优先

依据案例：`batch-321` 案例 1605

## 读者承诺

西幻开篇也要先让读者看懂事件、威胁、不公和钩子；语言先主谓宾清楚，再谈风格。

## 核心循环

```text
直接进事件 -> 给威胁/不公 -> 金手指或替代钩子 -> 短句承载画面 -> 类型语感稳定
```

## 开篇必备

- 第一章讲什么要清楚。
- 生死威胁或不公待遇要可感知。
- 金手指未出场时必须有强钩子。
- 西幻语感不要被现代口语破坏。
- 长句病句先改成主语、动作、结果。

## 期待来源

读者期待先进入事件和压力，再接受设定与风格。

## 断档风险

- 二改只删支线，主线仍看不懂。
- 翻译腔和病句劝退。
- “这波我的”一类口语破坏西幻质感。
- 形容词和比喻堆太多。
- 没威胁、没不公、没金手指。

## 训练任务

拿一段西幻开篇，删掉不影响事件的形容词和比喻，把长句改成“主语-动作-结果”的短句。
