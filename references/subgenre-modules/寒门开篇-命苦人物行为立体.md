# 寒门开篇-命苦人物行为立体

依据案例：`batch-061` 案例 302。

## 读者承诺

这个模块适用于寒门、赘婿、种田、历史日常里的弱势人物开篇。它承诺的是用命苦处境、性格矛盾和被迫行为，让人物立起来并激发读者保护欲。

## 核心循环

```text
弱势处境 -> 被人贬低 -> 读者同情 -> 被迫大胆选择 -> 性格回拉 -> 人物立体
```

## 开篇必备

- 快速抛出故事主体人群和处境。
- 人物困境要具体，不只说命苦。
- 行为反差必须由压力逼出来。
- 行为后要有回拉细节，证明人设稳定。
- 主角选择要回应读者的帮助欲。

## 期待来源

- 读者想看命苦人物被善待。
- 弱势处境形成情绪缺口。
- 反差行为让人物鲜活。
- 主角介入带来改变期待。
- 落差和反转带来爽感。

## 断档风险

- 只用旁白说人物可怜。
- 行为反差没有压力支撑，像崩人设。
- 设定一次性太多但不服务期待。
- 只写美貌，不写困境。
- 情绪营造不到位，读者不想帮。

## 训练任务

设计一个命苦女主或配角：写一个缺陷、一个被贬低标签、一个被迫大胆行为、一个行为后的羞怯回拉。
