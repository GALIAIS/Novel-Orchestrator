# 克系开篇-诡异卖点吸引力诊断

依据案例：`batch-186` 案例 929。

## 读者承诺

克系或诡异开篇必须提供未知压迫、规则代价、机制新奇或金手指差异化，不能只贴标签。

## 核心循环

```text
诡异标签 -> 未知压迫 -> 规则代价 -> 金手指差异 -> 破局期待
```

## 开篇必备

- 第一章有惊悚或压迫感。
- 规则有代价。
- 金手指有明确爽点。
- 设定和同类作品有差异。

## 期待来源

读者期待主角在未知规则下，用独特能力找到破局方式。

## 断档风险

- 克系不惊悚。
- 诡异青年无机制亮点。
- 金手指鸡肋。
- 新奇感和压迫感都弱。

## 训练任务

写一个克系小纲：未知压迫、规则代价、金手指差异化和第一章破局期待缺一不可。
