# 支线创新-回绑主线框架

依据案例：`batch-045` 案例 221。

## 读者承诺

支线可以制造意外和惊喜，但必须最终协助推进主线。临场灵感要回绑大纲，不能让长篇结构失控。

## 核心循环

```text
平直主线 -> 不可逆牵绊 -> 新支线 -> 关键作用 -> 回流主线
```

## 开篇必备

- 原目标清楚。
- 新牵绊合理且不可逆。
- 变化意外但不毒。
- 支线有主线回流点。

## 期待来源

- 读者以为会逃，结果必须潜伏。
- 新支线带来新风险。
- 主角目标更复杂。
- 卷尾主线因此获得关键推进。

## 断档风险

- 突发灵感冲断主线。
- 支线只敲边鼓。
- 大纲不更新。
- 换地图没有提前证明看点。

## 训练任务

把一个逃离剧情改成潜伏支线，并写明支线在卷尾主线中的关键作用。
