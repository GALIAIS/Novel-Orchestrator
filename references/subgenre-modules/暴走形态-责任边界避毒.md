# 暴走形态-责任边界避毒

依据案例：`batch-137` 案例 684。

## 读者承诺

主角可以暴走，但不能让读者替他背无辜者死亡的道德债。暴走桥段要前置消解风险，别事后硬洗。

## 核心循环

```text
反派作恶 -> 主角爆发 -> 目标合法化 -> 暴走清算 -> 获得成长/能力
```

## 开篇必备

- 明确被打对象是否该死。
- 主角保留底线或后手。
- 反派责任清楚。
- 暴走服务爽点和人设。
- 不靠事后洗白。

## 期待来源

- 愤怒指向真正恶人。
- 主角力量爆发。
- 读者不用承担伦理负担。
- 暴走后有成长和掌控。

## 断档风险

- 主角被完全控制。
- 误杀本该拯救的人。
- 后续硬洗显得伪善。
- 影视桥段直接搬到文字。

## 训练任务

写同一暴走桥段三版：毒点版、改设定版、改动因版。标出哪一版最不容易让读者觉得主角伪善。
