# 多标签题材-主承诺精炼

依据案例：`batch-047` 案例 235。

## 读者承诺

题材标签越多，越要精炼主承诺。脑洞都市、人族妖族、复兴、轻松、热血、争霸不能全都抢第一位。

## 核心循环

```text
标签堆叠 -> 冲突识别 -> 主线基调 -> 对标关键词 -> 一句话承诺
```

## 开篇必备

- 删掉互相冲突的标签。
- 主线基调和剧情匹配。
- 平台和对标关键词明确。
- 第一章兑现主承诺。

## 期待来源

- 读者知道这是哪类书。
- 高武/灵气复苏等对标清楚。
- 轻松和热血按阶段切换。
- 复兴主线有足够铺垫。

## 断档风险

- 标签太多让人无从判断。
- 严肃复兴和轻松日常互相打架。
- 热血只剩喊口号。
- 找样本前没有明确承诺。

## 训练任务

把一个五标签题材压缩成一句话承诺，只保留主线、主基调、目标读者和核心脑洞。
