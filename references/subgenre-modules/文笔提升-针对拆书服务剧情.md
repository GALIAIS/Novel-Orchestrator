# 文笔提升-针对拆书服务剧情

依据案例：`batch-339` 案例 1694

## 读者承诺

文笔训练要先定位哪里白，再拆同类样本仿写，最终服务剧情清楚和阅读体验。

## 核心循环

```text
定位白点 -> 找同类样本 -> 拆动作/细节/台词 -> 仿写一招 -> 回看剧情是否更好读
```

## 开篇必备

- 区分平述、描写弱、情绪弱、剧情弱。
- 不靠堆形容词解决。
- 摘抄后要举一反三。
- 练措辞也要保推进。
- 剧情优先于花团锦簇。

## 期待来源

作者期待表达更有画面和节奏，但读者期待的是更清楚、更有情绪的故事。

## 断档风险

- 空问文笔技巧。
- 只加比喻和形容词。
- 为漂亮句子拖慢剧情。
- 摘抄变照搬。
- 把剧情白误判成文笔白。

## 训练任务

选一段白描写，找同类优秀样本，拆 3 个动作或细节写法，仿结构重写并检查剧情是否更清楚。
