# 天赋认知-积累学习而非自我否定

依据案例：`batch-107` 案例 532。

## 读者承诺

新人训练不能把失败归结为“我没天赋”。所谓天赋常常是阅读、思考、表达和行业理解的长期积累，能学会并持续训练，本身就是可用能力。

## 核心循环

```text
写作受挫 -> 错误归因天赋 -> 拆出积累差距 -> 转成训练项 -> 复盘下一轮结果
```

## 开篇必备

- 明确自己失败的具体位置。
- 区分不会写、没输入、没训练、没兴趣。
- 找出一项可补的积累。
- 不用天赋判断替代复盘。
- 把焦虑转成一周内任务。

## 期待来源

- 自我否定变成可执行清单。
- 抽象天赋变成具体输入差距。
- 训练后能看到局部改善。
- 行业认知减少被课程收割的概率。

## 断档风险

- 用“没天赋”逃避拆问题。
- 只羡慕新人爆发，不看长期积累。
- 没看同类书就急着挣钱。
- 把签约课当成上岗证。

## 训练任务

写一张“天赋归因校准表”：列出 5 个你以为是天赋的问题，再分别改写成阅读、拆书、日更、反馈、行业认知中的一个训练动作。
