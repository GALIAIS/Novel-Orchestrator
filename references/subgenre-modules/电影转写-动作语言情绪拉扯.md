# 电影转写-动作语言情绪拉扯

依据案例：batch-298 案例 1490

## 读者承诺

用电影片段训练场景功能、动作语言顺序和情绪拉扯，但最终要脱离影片写原创正文。

## 核心循环

```text
看片段 -> 拆承上启下 -> 记录动作语言 -> 标注情绪升降 -> 原创同功能场景
```

## 开篇必备

- 场景承上和启下明确。
- 主角目标、阻力人物、信息提供者明确。
- 情绪升级有动作支点。
- 情绪释放服务主角性格。

## 期待来源

- 读者看见主角在压力下做选择。
- 信息场景也有冲突和看点。
- 克制后的释放更有爽感。

## 断档风险

- 只复述镜头，不会原创。
- 只有信息传递，没有戏剧性。
- 情绪释放和主角价值观无关。

## 训练任务

拆一个电影冲突场景，再写一个原创场景：同样完成信息获取、人物塑造、情绪释放三件事。
