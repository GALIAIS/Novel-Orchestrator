# 计划意外式-小剧情情绪折线

依据案例：batch-280 案例 1396

## 读者承诺

用“计划-意外”搭建 2000 字级别小剧情情绪折线：每次计划带来希望，每次意外制造下行，最后用搏命方案兑现爆发。

## 核心循环

```text
目标 -> 计划推进 -> 意外打断 -> 新计划 -> 更强意外 -> 最终爆发
```

## 开篇必备

- 小剧情目标明确。
- 第一计划看似可行。
- 意外必须来自敌人、环境或信息差。
- 最终爆发要回收前面下压。

## 期待来源

读者每次看到新计划，都会短暂相信主角有机会；意外打断后，最终反杀才有重量。

## 断档风险

- 意外像作者硬塞。
- 计划失败靠主角降智。
- 一路下压没有小希望。
- 最终爆发和前面计划无关。

## 训练任务

写一个三轮计划的 2000 字冲突段。每轮标出上行理由、下行原因和最终爆发的回收点。
