# 多线折线-错峰下行与情绪交织

依据案例：`batch-243` 案例 1212。

## 读者承诺

多条故事线可以各有折线，但整体阅读体验要保持起伏，避免多个下行同时压住读者。

## 核心循环

```text
A线下行 -> B线上行托底 -> B线下行 -> A线回升 -> 整体情绪持续摆动
```

## 开篇必备

- 列出正在推进的故事线。
- 标出每条线的上行和下行。
- 找出同时下行点。
- 给其中一条线安排托底上行。

## 期待来源

读者能承受压力，是因为另一条线仍提供希望、奖励或推进。

## 断档风险

- 多条线同时下行，观感变虐。
- 每条线单看成立，合起来压抑。
- 机械套折线，忽略整体情绪。
- 托底上行太弱，撑不住主下行。

## 训练任务

把一章拆成三条线，标出上行/下行；同章两条以上下行时，补一个托底上行或移动一个下行点。
