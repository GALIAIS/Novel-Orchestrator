# 套娃折线-成熟主角先行

依据案例：batch-285 案例 1421

## 读者承诺

新人练套娃折线时，先用成熟主角承载结构，等目标、意外、下行、底牌、高潮掌握后，再处理角色成长。

## 核心循环

```text
上层目标 -> 小结构套娃 -> 大局下行 -> 主角个人上行 -> 情绪高潮
```

## 开篇必备

- 上层目标。
- 小情节目标。
- 主角成熟能力。
- 大局下行点。
- 主角个人上行动作。

## 期待来源

读者看的是主角如何在更坏局面中打出价值，而不是每个小结构都完成一次成长。

## 断档风险

- 新人同时硬塞成长、三幕式、五折线。
- 把下行误解为主角必须憋屈。
- 小结构不服务大局。
- 为套格式牺牲自然节奏。

## 训练任务

写一个主角失败但读者爽的桥段：大目标失败，主角亮底牌并让敌人付出代价。
