# 前世身份-涉黑风险与杀伐替代

依据案例：`batch-224` 案例 1117。

## 读者承诺

用低风险身份解释主角杀伐果断，保留行动合理性，删除涉黑和代入阻力。

## 核心循环

```text
需要果断行动 -> 选择前世身份 -> 合规/代入校验 -> 替换身份 -> 保留能力与底线
```

## 开篇必备

- 明确想解释的行为是果断、战斗经验还是冷静判断。
- 删除黑帮等敏感标签。
- 用军人、特工、救援、执法、战场幸存者等替代。
- 第一场战斗展示底线而非凶残。

## 期待来源

读者期待主角敢行动、能承担，但不希望先被涉黑标签劝退。

## 断档风险

- 用黑帮老大当主角底色。
- 模糊带过后又不断强化涉黑人设。
- 把杀伐果断写成无底线凶残。
- 前世身份太具体，筛掉大量读者。

## 训练任务

把黑帮老大穿越降妖除魔改成 3 个低风险版本：退役军人、特殊救援队、边境缉毒警家属。每版写保留能力、删除风险、第一场战斗底线选择。
