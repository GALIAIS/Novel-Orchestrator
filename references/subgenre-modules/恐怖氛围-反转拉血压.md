# 恐怖氛围-反转拉血压

依据案例：`batch-332` 案例 1659

## 读者承诺

恐怖文前期要用氛围持续压迫，再用反转不断推翻读者判断，让血压上升。

## 核心循环

```text
异常细节 -> 初步解释 -> 反转推翻 -> 危险更近 -> 新异常继续
```

## 开篇必备

- 空间和声音等氛围细节。
- 信息不完整。
- 每章有错误解释。
- 反转改变读者理解。
- 章末保留未解问题。

## 期待来源

读者期待自己刚看懂的规则被新信息推翻，从而继续追下一层真相。

## 断档风险

- 只摆怪物，不铺氛围。
- 太早解释清楚全部规则。
- 反转不影响局势。
- 恐怖细节和后文无关。
- 前十章压迫感断掉。

## 训练任务

设计恐怖前三章，每章写异常、错误解释、反转，第三章结尾推翻前两章理解。
