# 新人分层-阅读拆书写作路径

依据案例：`batch-011` 案例 054。

## 读者承诺

新人先定位自己卡在哪一层，再补对应短板。阅读不足、拆书不足、理论不足、长篇结构不足，训练方案不同。

## 核心循环

```text
阶段诊断 -> 单点补短板 -> 同类样本 -> 拆书/写作 -> 反馈迭代
```

## 开篇必备

- 先看阅读量。
- 再看写作字数和成绩。
- 判断是不会看、不会拆、不会写，还是不会长篇。
- 一次只补一个主问题。

## 期待来源

- 训练不再乱投医。
- 同类阅读能补题材语感。
- 精拆大神能补结构能力。
- 写作反馈验证训练方向。

## 断档风险

- 所有短板一起补。
- 阅读量不足就急着开书。
- 能签约但不拆中期结构。
- 执念大于市场反馈。

## 训练任务

写一张新人诊断卡：阅读量、完整字数、签约情况、成绩、最大短板、下月唯一训练重点。
