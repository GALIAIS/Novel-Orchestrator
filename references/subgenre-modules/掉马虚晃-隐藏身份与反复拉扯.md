# 掉马虚晃-隐藏身份与反复拉扯

依据案例：`batch-246` 案例 1230。

## 读者承诺

隐藏身份类故事用“快掉马但没掉”持续拉扯读者期待，让读者享受信息差。

## 核心循环

```text
建立马甲 -> 快要暴露 -> 找理由遮掩 -> 留新破绽 -> 再次快掉马
```

## 开篇必备

- 读者知道真实身份。
- 关键角色不知道。
- 每次虚晃都有强线索。
- 没掉后留下新破绽。

## 期待来源

读者比角色知道更多，因此既期待曝光，又享受差一点暴露的抓心感。

## 断档风险

- 只拖不兑现。
- 遮掩理由太硬。
- 虚晃没有新线索。
- 配角反应不推进关系。

## 训练任务

设计 5 次掉马虚晃，每次写线索、差点知道的人、没掉原因和新破绽；第 3 次给小兑现。
