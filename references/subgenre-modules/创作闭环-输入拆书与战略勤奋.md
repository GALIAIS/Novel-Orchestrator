# 创作闭环-输入拆书与战略勤奋

依据案例：batch-368 案例 1838

## 读者承诺

码字只是输出环节，不能用字数勤奋掩盖输入、拆书和剧情思考不足；写作训练要形成完整闭环。

## 核心循环

```text
阅读输入 -> 拆书吸收 -> 剧情思考 -> 码字输出 -> 反馈复盘
```

## 开篇必备

- 每天有输入。
- 每天有拆解。
- 每天有构思。
- 输出后有复盘。

## 期待来源

- 判断力变准。
- 素材储备增加。
- 剧情设计更清楚。
- 码字不再闭门造车。

## 断档风险

- 只统计字数。
- 闷头写错方向。
- 没有阅读和拆书。
- 战术勤奋掩盖战略懒惰。

## 训练任务

做一周记录表：输入、拆书、构思、码字、复盘五栏缺一不可。
