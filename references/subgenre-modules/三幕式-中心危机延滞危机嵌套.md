# 三幕式-中心危机延滞危机嵌套

依据案例：`batch-090` 案例 446。

## 读者承诺

中心危机负责给故事主压力，延滞危机负责在主角半成功时爆发下坠。多个小危机用伪胜利和伪失败串联，让一卷故事写长写大。

## 核心循环

```text
中心危机 -> 小危机 -> 伪胜利/伪失败 -> 小高潮半成功 -> 延滞危机爆发 -> 至暗时刻 -> 第三幕翻盘
```

## 开篇必备

- 先明确中心危机。
- 延滞危机要能暂时藏住。
- 小危机必须改变大局。
- 伪胜利和伪失败要连接下一危机。
- 延滞危机在半成功后爆发。

## 期待来源

- 主角能否解决当前小危机。
- 每次胜败如何改变中心危机。
- 半成功后会不会崩盘。
- 至暗时刻能否翻盘。

## 断档风险

- 小危机和中心危机无关。
- 伪胜利只是热闹，不改变局势。
- 延滞危机爆发太早或太晚。
- 至暗时刻没有翻盘资源。

## 训练任务

写一个中心危机、一个延滞危机和三个小危机。每个小危机后写一次伪胜利或伪失败，并说明它如何推动第三幕。
