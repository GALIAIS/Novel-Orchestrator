# 现实爽文-天才反差公交卡

依据案例：`batch-315` 案例 1571

## 读者承诺

把现实天才新闻转化为爽文结构：高门槛难题先压住所有人，再让低调天才轻松解决，最后用朴素酬谢制造强反差。

## 核心循环

```text
团队无解 -> 天才出手 -> 快速解决 -> 朴素回报 -> 能力崇拜+生活反差
```

## 开篇必备

- 难题门槛足够高。
- 失败者足够专业。
- 主角能力有公众认知或前置铺垫。
- 解决过程显得轻松但不空。
- 酬谢或生活细节形成反差。

## 期待来源

读者期待看到“别人不行，他行”的能力碾压，同时被朴素细节拉近距离。

## 断档风险

- 没有垫高难题，天才出手不爽。
- 只写崇拜，没有反差记忆点。
- 机械降神缺少前置能力标签。
- 现实素材未经核验就写成事实科普。

## 训练任务

写一个天才解决难题桥段，必须包含失败团队、天才动作、低调酬谢三个要素。
