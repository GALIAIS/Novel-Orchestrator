# 第一章卖点优先-危机辅助

依据案例：`batch-043` 案例 211。

## 读者承诺

第一章要先让读者看到本书最值得看的卖点。生死危机和不公平伤害是辅助工具，不是开篇目的本身。

## 核心循环

```text
卖点直给 -> 读者识别 -> 危机加压 -> 使用/预告卖点 -> 形成追读
```

## 开篇必备

- 题材和卖点清楚。
- 金手指或核心能力可见。
- 危机服务卖点。
- 第一章结束前有使用期待。

## 期待来源

- 读者知道这本书看什么。
- 主角马上要用卖点解决问题。
- 危机让卖点更有必要。
- 不公平伤害补充情绪压力。

## 断档风险

- 只有危机，没有卖点。
- 为套模板写无关不公平伤害。
- 卖点藏到第二单元以后。
- 作者不清楚想让读者期待什么。
