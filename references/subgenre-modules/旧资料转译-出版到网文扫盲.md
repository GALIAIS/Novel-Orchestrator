# 旧资料转译-出版到网文扫盲

依据案例：`batch-059` 案例 291。

## 读者承诺

这个模块用于处理旧出版、编剧、畅销小说培训资料。它承诺的是提炼基础认知，再用网文案例校准，而不是直接照搬旧体系。

## 核心循环

```text
读取旧资料 -> 提炼情节/市场/结构方法 -> 标出出版语境 -> 找网文案例验证 -> 转成新人训练项
```

## 开篇必备

- 先确认资料正文是否可见。
- 可见内容只按证据摘取。
- 重点找情节策略、市场定位、开篇方法、大纲范本。
- 每条旧方法都要用网文样本校准。
- 不把出版节奏当成网文追读节奏。

## 期待来源

- 新人得到系统扫盲，不只学零碎技巧。
- 作者理解市场和结构的基础逻辑。
- 旧资料提供编辑沟通和大纲模板。
- 网文案例负责验证当前可用性。
- 资料学习转化为实际训练任务。

## 断档风险

- 附件不可见还强行脑补内容。
- 照搬出版方法导致开篇慢。
- 只收藏资料，不做转译。
- 把年代久远的市场判断当现行规则。
- 缺少网文案例验证。

## 训练任务

从一份旧写作资料中摘 5 条方法，每条配一个网文案例和一个改写任务。配不出案例的，暂不进入自己的写作流程。
