# 市场反馈-理想表达与读者买账

依据案例：`batch-226` 案例 1128。

## 读者承诺

先把理想表达包装成读者愿意买账的第一部作品，再根据反馈决定扩展。

## 核心循环

```text
表达欲 -> 可卖包装 -> 第一部反馈 -> 续作空间 -> 长线表达
```

## 开篇必备

- 区分作者想表达和读者先看到的卖点。
- 第一阶段目标要能被市场验证。
- 反馈不足时不盲目扩展。
- 迎合读者不等于放弃核心。

## 期待来源

市场反馈给作品续命，读者买账后，作者才有更大空间展开理想。

## 断档风险

- 只谈梦想，不管读者。
- 把迎合市场理解成毫无表达。
- 第一部没验证就铺超长系列。
- 忽略销量、追读、反馈等信号。

## 训练任务

把一个理想题材写成两版简介：作者表达版、读者买点版。比较哪版更适合第一部开局，并写出什么反馈达标后再扩展第二卷。
