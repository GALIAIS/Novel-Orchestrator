# 民俗恐怖-资料改编捏造

依据案例：`batch-065` 案例 323。

## 读者承诺

这个模块用于民俗恐怖、诡异仙侠、传统灵异转型。它承诺的是从游戏、书籍、影视、网文、县志、口述、古早论坛和灵异节目中有方向地积累素材，再通过换角度改编，把普通民俗或故事变成恐怖规则和事件。

## 核心循环

```text
定方向 -> 多渠道取材 -> 摘仪式/禁忌/口传 -> 换角度恐怖化 -> 捏造规则 -> 放入升级/无敌骨架
```

## 开篇必备

- 明确民俗恐怖的地方或主题。
- 至少准备三个资料渠道。
- 第一章要有一个具体禁忌或仪式。
- 恐怖素材要服务主角目标。
- 可用升级文或无敌文骨架承载。

## 期待来源

- 熟悉民俗被反常解释。
- 禁忌被触犯后的代价。
- 地方口传和县志带来真实感。
- 普通故事被恐怖化。
- 主角在诡异规则中升级或破局。

## 断档风险

- 只找网文二手素材。
- 风土人情资料无法转成事件。
- 资料不足就急着开书。
- 民俗元素只是装饰。
- 恐怖感和升级爽点互相脱节。

## 训练任务

找一个习俗或童话，改成恐怖规则：写原始版本、反常解释、禁忌、触犯代价、主角介入方式。
