# 毒点逻辑-表达重心错位

依据案例：batch-283 案例 1415

## 读者承诺

毒点不只来自逻辑漏洞，也来自表达重心和角色选择让读者无法感同身受。悲剧、极端选择、反转都必须让读者相信角色当时别无他路。

## 核心循环

```text
前文重心 -> 角色极端选择 -> 读者追问后果 -> 铺垫回应疑问 -> 悲剧成立或毒点爆发
```

## 开篇必备

- 角色最在乎的人或事。
- 角色承受的痛苦强度。
- 可选方案及其失败原因。
- 极端选择的当时认知。
- 选择后果的回应。

## 期待来源

读者愿意接受悲剧，是因为能理解角色为什么走到那里，而不是因为作者告诉他这很深刻。

## 断档风险

- 前期重点写母子深情，结尾却不回应孩子后果。
- 角色有明显更优选择却不选。
- 逻辑能解释，但情绪不能说服。
- 为了反转牺牲角色动机。

## 训练任务

给一个悲剧结尾列出 5 个读者会骂的问题，再逐条补铺垫。补不上的问题，说明这个结尾会形成毒点。
