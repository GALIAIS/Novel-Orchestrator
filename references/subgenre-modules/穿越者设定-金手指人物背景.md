# 穿越者设定-金手指人物背景

依据案例：`batch-104` 案例 516。

## 读者承诺

穿越不是随手加的流行标签，而是金手指和人物背景。它会改变主角三观、知识、吐槽、选择和剧情路线。

## 核心循环

```text
是否穿越 -> 三观差异 -> 知识差异 -> 与世界冲突 -> 关键选择 -> 剧情分叉
```

## 开篇必备

- 穿越给主角带来什么能力或视角。
- 主角和本世界土著有什么差异。
- 这种差异如何影响第一场选择。
- 不穿越版本是否仍成立。

## 期待来源

- 现代视角看异世界。
- 知识差带来的破局。
- 三观碰撞。
- 玩梗和读者代入。

## 断档风险

- 穿越只是身份标签。
- 主角选择和土著完全一样。
- 读者追问“穿越者为什么不这样做”。
- 穿越与金手指、世界背景脱节。

## 训练任务

同一剧情写穿越者版和土著版，各列三个不同选择。若没有差异，就删掉穿越设定。
