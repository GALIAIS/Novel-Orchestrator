# 读者视角自审-作者滤镜与反馈综合

依据案例：batch-292 案例 1456

## 读者承诺

用读者视角检查文本，把作者脑内默认信息补到文本里，并综合反馈判断毒点。

## 核心循环

```text
作者滤镜 -> 读者已知检查 -> 动机利弊补足 -> 删除无效铺垫 -> 综合反馈判断
```

## 开篇必备

- 读者已知信息。
- 读者未知但必须知道的信息。
- 主角动机。
- 行动利弊。
- 是否上帝视角。
- 高潮后的观众反应。
- 反馈样本数量。

## 期待来源

读者理解主角为什么行动，并在爽点后看到足够反应来延长满足。

## 断档风险

- 作者默认读者知道脑内设定。
- 大纲写作漏掉思考过程。
- 场景铺垫变成形容词堆叠。
- 单条差评被误判为毒点。

## 训练任务

给一章成稿标注“读者已知/作者脑补/必须补写”，再删三处无效铺垫。
