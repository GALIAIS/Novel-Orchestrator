# 现实苦难-新闻共情素材

依据案例：`batch-185` 案例 923。

## 读者承诺

现实苦难素材能快速触发共情，因为读者会联想到自己和家人的脆弱处境。

## 核心循环

```text
现实压力 -> 具体崩溃瞬间 -> 普通人代入 -> 共情痛苦 -> 故事行动出口
```

## 开篇必备

- 压力类型要贴近日常。
- 崩溃瞬间要具体。
- 读者能联想到自身或亲人。
- 后续必须给行动出口。

## 期待来源

读者希望故事回应现实中难以解决的痛苦，而不是只展示苦难。

## 断档风险

- 只消费真实苦难。
- 没有故事出口。
- 抽象卖惨，没有具体人物。
- 未核验新闻直接照搬。

## 训练任务

提炼一个现实苦难新闻的情绪结构，改写成虚构人物开头，并给主角一个可执行的帮助动作。
