# 新人求稳-成熟套路优先

依据案例：`batch-150` 案例 748。

## 读者承诺

新人先掌握热销榜上普适、稳妥、不容易出幺蛾子的成熟套路，避开拍脑袋造成的毒点和雷点。创新可以有，但最好放到签约、千订、完本或能力提高之后。

## 核心循环

```text
研究成熟榜书 -> 拆金手指和开篇 -> 规避毒点 -> 写俗但稳的桥段 -> 完成签约/完本 -> 再尝试创新
```

## 开篇必备

- 目标平台的成熟题材。
- 多人追风过的金手指。
- 可拆的高订完本或榜书。
- 前 20 章开篇样本。
- 明确要避开的毒点。

## 期待来源

新人期待先活下来：用成熟套路换签约、千订、完本和稳定经验，而不是用尚未掌握的灵光一闪把读者毒跑。

## 断档风险

- 误以为套路都被读者看腻了。
- 选择全网只有少数火书的孤例设定。
- 写目标平台却拆错平台风格。
- 不肯下功夫拆书，只想一炮而红。
- 把“俗”当失败，忽略赚钱和完本本身的价值。

## 训练任务

选一个成熟题材，找 3 本同平台高成绩样本，各拆前 20 章。输出一版“俗但稳”的前三章方案，必须包含成熟金手指、第一冲突、第一收获、毒点规避。
