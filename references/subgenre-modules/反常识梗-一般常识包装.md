# 反常识梗-一般常识包装

依据案例：batch-311 案例 1553

## 读者承诺

真实可行不等于读者会信。反常识梗要按一般读者常识包装，必要时用铺垫、演示或搞笑语气降低抗拒。

## 核心循环

```text
反常识动作 -> 读者不信 -> 提前铺垫条件 -> 场内演示 -> 让梗变成爽点/笑点
```

## 开篇必备

- 先判断一般读者会不会信。
- 严肃剧情要给物理条件和前置演示。
- 搞笑剧情可以放大荒诞感。
- 旁人反应要替读者吐槽。

## 期待来源

期待来自“离谱但能接受”。读者既惊讶，又能顺着文本给的证据或语气继续看。

## 断档风险

- 作者知道真实，读者却不信。
- 严肃桥段没有铺垫。
- 解释太晚，已经被读者判定离谱。
- 反常识动作破坏世界可信度。

## 训练任务

写“法棍打晕人”两版：严肃版铺垫硬度和角度，搞笑版让角色吐槽并把吐槽变成笑点。
