# 人物描写资料-女性细节与情绪功能

依据案例：batch-348 案例 1737

## 读者承诺

女性描写资料要转化为功能化细节：外貌、神态、动作、环境共同服务人物状态和场景情绪。

## 核心循环

```text
人物状态 -> 选择细节 -> 指向情绪功能 -> 读者形成判断 -> 后续剧情使用印象
```

## 开篇必备

- 先定场景需要的读者感受。
- 再选外貌、神态、动作或环境细节。
- 描写必须服务身份、处境、关系或情绪。
- 网文中要压缩名著式长描写。
- 避免只写漂亮、清冷、妩媚等空标签。

## 期待来源

读者通过少量细节立刻理解人物状态，而不是只收到形容词。

## 断档风险

- 照搬名著句式。
- 描写过长拖慢节奏。
- 只堆美词，没有剧情功能。
- 所有女性角色都同一种美。
- 细节不被后续剧情使用。

## 训练任务

同一女性角色写三版 120 字描写：惊艳、病弱、压迫。每版标注三个细节分别服务什么情绪。
