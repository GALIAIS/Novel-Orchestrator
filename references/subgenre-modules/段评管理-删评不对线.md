# 段评管理-删评不对线

依据案例：batch-312 案例 1560

## 读者承诺

首秀后评论暴涨，作者要保护创作状态。杠精评论直接删，别扣字对线。

## 核心循环

```text
首秀曝光 -> 段评暴涨 -> 识别杠精/有效反馈 -> 删评降噪 -> 固定时间复盘
```

## 开篇必备

- 先定删除标准。
- 有效批评记录，不即时争辩。
- 杠精、辱骂、带节奏直接处理。
- 每天固定时间看评论。

## 期待来源

期待来自稳定连载。评论区少消耗，作者才能继续更新。

## 断档风险

- 和读者逐条对线。
- 被负评打断当天写作。
- 把有效批评和恶意杠混为一谈。
- 禁言或争吵激化矛盾。

## 训练任务

写一份段评处理规则：保留、删除、记录不回复、集中复盘四类，各列 5 条标准。
