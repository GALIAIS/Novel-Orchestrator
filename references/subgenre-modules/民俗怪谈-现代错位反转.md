# 民俗怪谈-现代错位反转

依据案例：`batch-141` 案例 705。

## 读者承诺

把传统禁忌和现代生活逻辑撞在一起，让高压怪谈规则被语言、流程、身份或常识错位轻巧绕开，形成章节内的小情绪反转。

## 核心循环

```text
摆出传统规则 -> 制造回答压力 -> 给出可回收细节 -> 主角用现代错位破局 -> 紧张变笑点
```

## 开篇必备

- 读者熟悉的禁忌规则。
- 主角必须回答或行动的压力。
- 一个可利用细节。
- 一句反转台词。

## 期待来源

- 读者先按怪谈规则紧张。
- 主角用日常逻辑打断仪式。
- 低成本破局带来荒诞感。
- 评论区容易继续玩梗。

## 断档风险

- 规则没立住就反转。
- 反转靠解释，不靠一句话。
- 现代梗太硬，破坏故事基调。
- 小反转替代大主线。

## 训练任务

写 5 个传统禁忌加现代错位破局的微桥段，每个 150 字内。要求先压迫，中段埋细节，结尾一句话反转。
