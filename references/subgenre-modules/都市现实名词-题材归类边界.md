# 都市现实名词-题材归类边界

依据案例：`batch-072` 案例 356。

## 读者承诺

这个模块用于不完全架空都市、地球开局异世界和混合题材归类。它承诺的是现实名词作为背景锚点可以谨慎使用，但题材标签必须按主玩法和读者期待来定。

## 核心循环

```text
现实锚点 -> 风险判断 -> 主玩法识别 -> 标签归类 -> 避免误导读者
```

## 开篇必备

- 区分地理名词和真实敏感映射。
- 地球开局不等于都市主玩法。
- 奇幻元素不等于正统西幻。
- 多世界穿梭优先考虑无限诸天。
- 标签服务读者筛选。

## 期待来源

- 现实名词降低理解门槛。
- 异世界提供冒险新鲜感。
- 诸天裂缝带来长期地图期待。
- 准确标签减少错位。
- 主玩法清楚后读者知道追什么。

## 断档风险

- 用真实名词牵涉敏感机构或政治。
- 标签写西幻，正文是无限诸天。
- 地球开局后迅速丢失都市承诺。
- 元素太杂但没有主玩法。
- 读者被错误标签吸引进来。

## 训练任务

给一个混合题材写 3 个标签和一句承诺，说明每个标签会吸引什么读者，以及哪个标签最容易造成错位。
