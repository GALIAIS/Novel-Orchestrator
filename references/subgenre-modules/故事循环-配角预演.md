# 故事循环-配角预演

依据精拆：`case-library/精拆/batch-001.md` 案例 003。

## 适用范围

适合新人快速写长，也适合副本、规则怪谈、玄幻换地图、都市打脸、诱惑测试、末世生存。

## 读者承诺

读者先看见规则或后果，再看主角进入同类局面，因此自然期待主角会如何应对，以及主角和别人有什么不同。

## 核心循环

```text
配角遭遇 -> 配角失败/付出代价 -> 读者理解规则 -> 主角遭遇同类事件 -> 主角差异化破局 -> 新外壳重复
```

## 开篇必备

- 配角预演必须服务主角，不是单独水剧情。
- 配角失败结果要清楚，让读者知道代价。
- 主角上场时要有差异化优势：信息、能力、意志、金手指、身份。
- 每轮循环换外壳，不换核心情绪。

## 常见用法

- 规则怪谈：别人违反规则死亡，主角识别漏洞。
- 副本流：别人艰难通关，主角用特殊解法碾过。
- 诱惑流：别人被诱惑失败，主角抵抗并获得奖励。
- 换地图：每层地图有更高阶同构压迫。

## 断档风险

- 每轮只是重复同一个场面。
- 配角预演太长，主角存在感弱。
- 主角破局方式没有差异，只是更强。
- 新循环没有抬高代价或扩大世界。

## 操作卡

```text
本轮规则：
配角预演：
失败代价：
主角得知的信息：
主角差异化优势：
主角破局：
下一轮外壳：
```

## 训练任务

写三轮同构事件：第一轮用校园，第二轮用公司，第三轮用副本。保持“配角失败 -> 主角破局”的结构，但每轮换压力来源和破局方式。
