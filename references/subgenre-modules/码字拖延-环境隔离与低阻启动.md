# 码字拖延-环境隔离与低阻启动

依据案例：batch-380 案例 1900

## 读者承诺

拖延时先处理环境和任务门槛。移走手机、断网、用小黑屋类工具；写不动正文时降级写大纲，先恢复写作动作。

## 核心循环

```text
卡住 -> 隔离诱惑 -> 降低任务 -> 小完成 -> 回到正文
```

## 开篇必备

- 当前卡点。
- 最大诱惑源。
- 物理隔离办法。
- 最低可完成任务。

## 期待来源

- 写作动作重新开始。
- 小任务降低焦虑。
- 开头磨稿被正常化。
- 正文产出逐步恢复。

## 断档风险

- 自责替代行动。
- 开头磨稿变成刷手机。
- 旧书全勤和新书开头混成一团。
- 只换工具不隔离诱惑。

## 训练任务

做一次 90 分钟断网写作：手机放远，前 20 分钟写大纲，后 60 分钟写正文，最后 10 分钟只标注问题不修改。
