# 文笔教学-视角流畅性修正

依据案例：batch-311 案例 1551

## 读者承诺

文笔提升要让读者读得顺。视角清楚、信息承接自然，比单纯词句漂亮更重要。

## 核心循环

```text
读者卡顿 -> 检查视角主体 -> 检查句间承接 -> 调整信息顺序 -> 情节变丝滑
```

## 开篇必备

- 每段先确定视角主体。
- 动作、感受、信息来源要同向。
- 转场必须给提示。
- 修文时优先解决读者看不懂。

## 期待来源

期待来自无阻力阅读。读者不用反复确认谁在看、谁在动，就能跟上情节。

## 断档风险

- 视角跳动没有提示。
- 句子漂亮但承接断裂。
- 情节信息顺序混乱。
- 只改辞藻，不改叙事流。

## 训练任务

选一段卡顿文字，逐句标出视角主体和动作。凡主体切换无提示处，补转场或重排信息。
