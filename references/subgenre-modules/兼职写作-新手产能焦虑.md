# 兼职写作-新手产能焦虑

依据案例：`batch-127` 案例 635。

## 读者承诺

兼职新手写得慢、词穷、时间不够，是多重负荷叠加，不是直接证明不适合写作。先用检索和工具降低表达阻力，再靠长期练习把剧情组织和语言表达自动化。

## 核心循环

```text
写得慢崩溃 -> 拆出卡点 -> 检索补表达 -> 工具降低阻力 -> 小目标练习 -> 逐步自动化
```

## 开篇必备

- 承认当前实际可用时间。
- 区分剧情卡、表达卡、时间卡。
- 每天只设最小写作目标。
- 卡表达时立即检索。
- 长期积累常用表达和桥段。

## 期待来源

- 新手焦虑被正常化。
- 有具体工具能立刻减压。
- 小目标让兼职写作可持续。
- 题海练习带来长期提速。

## 断档风险

- 把慢等同于没天赋。
- 同时要求构思、学习、修改、日更全满。
- 只收藏工具，不形成表达积累。
- 还没签约就被日更恐惧压垮。

## 训练任务

记录 7 天写作：每天写作时长、实际字数、最大卡点、用过的检索词、次日最小目标。第 7 天复盘卡点是否从表达卡转为剧情卡或时间卡。
