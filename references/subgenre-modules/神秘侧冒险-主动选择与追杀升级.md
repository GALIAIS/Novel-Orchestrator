# 神秘侧冒险-主动选择与追杀升级

依据案例：batch-288 案例 1440

## 读者承诺

神秘侧开篇要让主角在组织压迫和不公平伤害下主动选择，并用金手指反击后引出追杀升级。

## 核心循环

```text
神秘威胁 -> 组织设局 -> 主角受害 -> 金手指触发 -> 主动反杀 -> 追杀升级
```

## 开篇必备

- 神秘威胁的规则。
- 组织行动的利益逻辑。
- 主角遭受的不公平伤害。
- 金手指触发条件。
- 主角第一次主动选择。
- 反击后的新压力。

## 期待来源

读者看到未知组织和影咒危险后，期待主角在伤害中激活能力，并从被动猎物变成主动反击者。

## 断档风险

- 第一章后节奏越写越散。
- 主角一直被带着走。
- 组织为什么不直接杀人说不通。
- 金手指展示太弱。
- 神秘侧名词多，但危险规则不清。

## 训练任务

把“组织带主角回基地”改成五步：设局、击毙/重伤、金手指触发、主角反杀、组织追杀。
