# 玄幻无线-同质套路旧中求新

依据案例：batch-307 案例 1535

## 读者承诺

无线玄幻开篇的同质套路能反复出现，通常因为它们容易制造不公平伤害、反击期待和后续展开。破局不是弃用底层结构，而是在旧套路里换新触发。

## 核心循环

```text
成熟不公平伤害 -> 保留反击期待 -> 替换压迫来源 -> 加强逻辑/世界观/金手指 -> 旧中求新
```

## 开篇必备

- 第一章给出具体不公平伤害。
- 三章内给出反击期待。
- 换掉过熟的女友利用、挖骨等表层包装。
- 用世界观、金手指和逻辑细节做差异。

## 期待来源

期待来自“主角被亏待后怎么讨回来”。读者熟悉套路，但仍会为具体压迫和爽快反击继续阅读。

## 断档风险

- 为反套路放弃开篇动力。
- 只换名词，不换压迫来源和事件逻辑。
- 不公平伤害太轻，读者不愤怒。
- 反击太慢，憋屈变添堵。

## 训练任务

把一个挖骨开篇改成三种变体：兄弟背叛、殃及池鱼、势力献祭。每种写第一章伤害和第三章反击。
