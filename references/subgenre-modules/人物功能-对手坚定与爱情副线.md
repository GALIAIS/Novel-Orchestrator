# 人物功能-对手坚定与爱情副线

依据案例：`batch-022` 案例 110。

## 读者承诺

按故事需要构造人物。对手要坚定阻止主角，爱恋对象要带来故事改变和情绪过渡。

## 核心循环

```text
主角独特命运 -> 对手坚定阻碍 -> 险境成立 -> 爱恋对象带来意外 -> 共同完成 -> 下行过渡
```

## 开篇必备

- 主角独一无二的能力或命运。
- 对手为什么坚信要阻止主角。
- 对手能让主角如何失败。
- 爱恋对象带来的意外。

## 期待来源

- 主角与对手形成强对比。
- 对手不是纸片坏人，而是有自己目标。
- 爱恋对象推动故事变化，让主角成长。

## 断档风险

- 对手动机被主角视角弱化。
- 反派良知太多，险境不够。
- 爱恋对象只负责好看，不推动故事。
- 爱情下行没有内外因结合。

## 训练任务

给一个反派写三句话：他坚信什么、为什么主角必须被阻止、他能让主角失去什么。再给爱恋对象写一次意外和一次共同完成事件。
