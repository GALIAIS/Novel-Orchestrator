# 主角决策-主战主和不背锅

依据案例：`batch-332` 案例 1657

## 读者承诺

强人设主角面对高风险决策时，不能显怂、不能背锅，还要让读者看见他暗中控局。

## 核心循环

```text
旁人提出保守方案 -> 主角表面接受 -> 暗中观察布局 -> 敌人露破绽 -> 主角反击拿功
```

## 开篇必备

- 风险方案由旁人或环境提出。
- 主角不轻易背锅。
- 主角有独立判断。
- 表面顺从有战术意义。
- 最终功劳归主角。

## 期待来源

读者期待主角不是莽夫，也不是怂人，而是在限制中保持掌控感。

## 断档风险

- 兵王主角主动提怂方案。
- 主角只等敌人出尔反尔，显被动。
- 冲突锅落在主角身上。
- 被迫加入事件后，主角失去思考。
- 反击高光没有提前布局。

## 训练任务

写一段谈判：旁人提妥协，主角表面同意，暗中找头目和破绽，章末反制。
