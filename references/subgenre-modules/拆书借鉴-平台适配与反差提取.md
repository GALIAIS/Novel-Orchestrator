# 拆书借鉴-平台适配与反差提取

依据案例：batch-288 案例 1436

## 读者承诺

拆书借鉴时先判断原书卖点能否适配目标平台，再提取人物反差和机制，而不是照搬设定。

## 核心循环

```text
原书卖点 -> 平台期待 -> 节奏差异 -> 人物反差 -> 开篇改造 -> 快速兑现
```

## 开篇必备

- 原书核心卖点。
- 目标平台读者期待。
- 可借反差机制。
- 需要加速的冲突。
- 前三章兑现点。

## 期待来源

读者看到熟悉类型中的错位人物和反差互动，快速理解趣味在哪里。

## 断档风险

- 只觉得设定新颖，不分析平台。
- 照搬慢热节奏。
- 反穿越只写设定，不写人物反应。
- 原书不成熟处也一起借来。

## 训练任务

拆一本目标书，写出可借机制和不可照搬点，再把它改成目标平台前三章节奏。
