# 新人题材-都市入口与陌生设定降负

依据案例：batch-384 案例 1916

## 读者承诺

现代城市可为修仙、异能、科幻等题材提供低解释入口。先利用现实共识建立代入，再逐层导入陌生世界与规则。

## 核心循环

```text
现实共识 -> 快速代入 -> 首个异常元素 -> 分层解释 -> 扩张类型世界
```

## 开篇必备

- 读者熟悉的现实场景。
- 无需解释的共识物件和关系。
- 第一个异常元素。
- 清楚的元素导入顺序。

## 期待来源

- 主角处境容易理解。
- 对白保持自然语境。
- 陌生规则逐次打开。
- 现代入口通向更大世界。

## 断档风险

- 把旧市场判断当当前禁令。
- 都市开篇和主要分类承诺错位。
- 一次塞入过多专有名词。
- 为降低难度而放弃题材核心。

## 训练任务

为同一创意写纯异世界版和都市入口版各 1000 字，比较解释量、专有名词与首个冲突位置。
