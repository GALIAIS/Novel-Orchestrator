# 开书流程-目标层级与执行拆解

依据案例：batch-368 案例 1839

## 读者承诺

开书先从题材、卷数、上下行、节点目标和段落目标往下拆，再让反派、配角、战斗和对白服务目标。

## 核心循环

```text
题材卷数 -> 卷级上下行 -> 节点目标 -> 段落目标 -> 章纲 -> 角色/战斗/对白
```

## 开篇必备

- 题材和体量。
- 第一卷目标。
- 最终目标。
- 第一章自查。

## 期待来源

- 每章都影响段落目标。
- 反派阻碍主角目标。
- 配角影响节点推进。
- 战斗形式不重复。

## 断档风险

- 先想零碎剧情。
- 每章和段落目标无关。
- 女主没有深远影响。
- 战斗和对白只是填充。

## 训练任务

写一页开书表：先写目标和上下行，再写节点、段落、章纲，最后补反派、配角、战斗库。
