# 西幻开篇-信息量控制与上行情绪

依据案例：batch-359 案例 1793

## 读者承诺

复杂世界开篇要先让读者看见主角处境、目标、行动和不同之处。设定只给当下理解所需，情绪下行必须被及时反打或希望压住。

## 核心循环

```text
陌生世界 -> 主角行动 -> 最低必要解释 -> 小上行/反打 -> 亮点扩展世界
```

## 开篇必备

- 主视角锁定主角。
- 第一目标清楚。
- 设定跟随行动释放。
- 生死危机中少解释、多动作。
- 第一处上行情绪足够明确。

## 期待来源

- 主角能动而不是挨打。
- 金手指或不同之处开始显露。
- 世界风味有但不压情节。
- 信息包装在冲突和选择中。

## 断档风险

- 西幻名词和背景过早堆叠。
- 影视化描写导致文字停滞。
- 第二章强下行压过第一章希望。
- 指望后文补救前文退场。

## 训练任务

把一个复杂设定开头改成 1500 字版本：前 300 字交代处境和目标，前 800 字出现主动反打，设定解释只保留动作理解所需。
