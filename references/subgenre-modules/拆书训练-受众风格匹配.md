# 拆书训练-受众风格匹配

依据案例：`batch-150` 案例 747。

## 读者承诺

拆书的目的不是填固定表格，而是看见作品如何服务目标受众。高频爽点、密集收获、快速升级是否“苍白”，要看目标读者是否接受并追读。

## 核心循环

```text
阅读样本 -> 标记升级收获 -> 判断逻辑链 -> 对照目标受众 -> 学习套路 -> 决定自己的节奏
```

## 开篇必备

- 明确拆书对象和受众。
- 记录每次升级、收获、身份变化。
- 判断每个刺激点是否有逻辑。
- 区分个人不适和读者不适。
- 给出可学习与可调整两套结论。

## 期待来源

作者期待通过拆高订老书看懂成熟套路：为什么密集刺激能让目标读者继续翻页，以及自己是否要改成更重铺垫和拉扯的风格。

## 断档风险

- 把个人口味误判成市场问题。
- 只说爽点多，不记录爽点逻辑。
- 用固定表格代替真实分析。
- 学老套路时不考虑当前热门题材外壳。

## 训练任务

拆一本升级流前 20 章，逐章列出升级、收获、压力、读者反馈。最后写两个结论：目标受众为什么吃这一套；如果自己写，哪些地方会放慢节奏。
