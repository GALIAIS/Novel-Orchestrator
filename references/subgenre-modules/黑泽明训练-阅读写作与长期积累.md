# 黑泽明训练-阅读写作与长期积累

依据案例：batch-377 案例 1883

## 读者承诺

创作没有绕过阅读和写作的捷径。写作痛苦是训练成本，积累和坚持才会让输出变稳。

## 核心循环

```text
想创作 -> 阅读输入 -> 写作输出 -> 生活观察 -> 长期积累
```

## 开篇必备

- 每日阅读。
- 每日写作。
- 观察记录。
- 坚持周期。

## 期待来源

- 输入变厚。
- 输出变稳。
- 素材来自生活。
- 痛苦被训练化。

## 断档风险

- 只等灵感。
- 只看课程不写。
- 觉得写作痛苦就是不适合。
- 跨媒介学习只看热闹。

## 训练任务

连续 14 天读 30 分钟、写 1000 字、记录 3 个生活观察，只统计完成，不评价好坏。
