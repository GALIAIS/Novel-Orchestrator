# 故事策略-价值观冲突奖励闭环

依据案例：`batch-313` 案例 1565

## 读者承诺

把长篇网文的发动机整理成可执行闭环：价值观和基调确定读者站队，冲突和不公平伤害制造投入，险境逼迫选择，金手指与主角行动破局，最后用力、利、名、权兑现奖励。

## 核心循环

```text
价值观/基调 -> 强烈目标 -> 阻碍升级 -> 不公平伤害+险境 -> 金手指/行动破局 -> 力利名权奖励
```

## 开篇必备

- 一套稳定价值观，而不是单句主题。
- 主角有强烈、具象、可被阻碍的目标。
- 不公平伤害同时具备强制性和价值不对等。
- 险境能导致主角失去重要资格、名声、能力、生活或生命。
- 前三章展示勇气与能力。
- 金手指要独特，并由主角行动触发或使用。

## 期待来源

读者期待价值观被践行、不公平被纠正、险境被突破，并在每轮冲突后获得力量、利益、名声或权力位置的可见增益。

## 断档风险

- 价值观空泛，不能决定站队。
- 只有危险，没有不公平。
- 只有金手指，没有主角行动。
- 奖励不具体，读者感受不到赢。
- 意外没有信息差支撑，显得作者硬拧。

## 训练任务

用“目标、阻碍、增强阻碍、不公平伤害、险境、金手指、行动、奖励”八格，设计一个第一章冲突。
