# 简单金手指-低位认知差与成长拓展

依据案例：`batch-257` 案例 1282。

## 读者承诺

简单金手指模块让低位身份、外界低估和真实底牌形成持续爽点，不靠花哨设定撑场。

## 核心循环

```text
低位身份 -> 外界低估 -> 经验底牌破局 -> 认知差打脸 -> 能力池拓展
```

## 开篇必备

- 金手指一句话能说清。
- 第一次危机要立刻验证能力。
- 外界必须有错误评估。
- 后续用途不能只剩战斗力。

## 期待来源

读者知道主角被当成消耗品，但他有别人不知道的经验和变强路径，因此期待低位翻盘。

## 断档风险

- 简单能力没有后续拓展。
- 异常规则只靠战力硬打。
- 外界低估没有被打脸。
- 类 SCP 元素未架空转译。

## 训练任务

用一句话设计金手指，再列 5 个拓展用途，其中至少 2 个不能靠纯战斗解决，并写一个低位打脸场景。
