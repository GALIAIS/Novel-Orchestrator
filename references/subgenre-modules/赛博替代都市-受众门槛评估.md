# 赛博替代都市-受众门槛评估

依据案例：`batch-038` 案例 189。

## 读者承诺

赛博近未来不是现代都市的无损替代。它增加设定成本、缩小受众，必须用更强冲突和爽点补偿。

## 核心循环

```text
现代都市承诺 -> 赛博替换 -> 设定成本上升 -> 新爽点补偿 -> 受众验证
```

## 开篇必备

- 新设定服务主线。
- 读者能快速理解。
- 赛博元素制造新冲突。
- 平台受众匹配。

## 期待来源

- 新背景带来新鲜感。
- 科技水平改变冲突方式。
- 都市异能/灵异获得差异化。
- 设定不只是换皮。

## 断档风险

- 受众较小。
- 设定门槛过高。
- 赛博和灵异双重负担。
- 新背景没有新增爽点。

## 训练任务

把现代都市异能开篇改成赛博版，只准加 3 个设定；每个设定必须制造冲突或爽点。
