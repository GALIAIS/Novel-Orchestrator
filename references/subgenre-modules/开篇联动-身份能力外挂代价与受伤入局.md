# 开篇联动-身份能力外挂代价与受伤入局

依据案例：batch-388 案例 1936

## 读者承诺

身份能力、外挂与代价、不公平伤害、行动动机要组成一条因果链，并由首次破局共同验证，而不是开篇信息清单。

## 核心循环

```text
身份形成认知优势 -> 外挂给出可能与代价 -> 不公平伤害迫使选择 -> 首次破局 -> 收益与新麻烦并存
```

## 开篇必备

- 能在事件中实证的职业或身份能力。
- 具体可见的外挂作用和使用代价。
- 与主线同向的不公平伤害。
- 足以解释立即行动的动机。

## 期待来源

- 主角用不同认知看见普通人忽略的机会。
- 能力可以破局但不能无成本解决全部问题。
- 读者替无辜损失追讨情绪债。
- 第一次选择同时指向人设和长线。

## 断档风险

- 职业只写在履历中。
- 代价不改变任何选择。
- 连续羞辱却没有伤害升级。
- 任务替主角决定一切，人物没有表态。

## 训练任务

写 900 字开头，让职业能力、外挂代价、不公平损失和不可撤回选择依次改变下一步行动，逐句删除无因果信息。
