# 文笔精修-的字密度控制

依据案例：batch-309 案例 1541

## 读者承诺

通过检查“的”字密度，压缩无效修饰链，让段落更干净，读者更容易快速吞下信息。

## 核心循环

```text
粗稿段落 -> 统计“的”字 -> 判断必要性 -> 删除无效修饰 -> 主谓宾靠近
```

## 开篇必备

- 每段先看“的”字数量。
- 超过 2 个就检查修饰链。
- 必要关系词保留。
- 无效形容和重复限定优先删。

## 期待来源

期待来自阅读顺滑。句子少绕路，信息就更快进入读者脑子。

## 断档风险

- 机械删掉所有“的”。
- 修饰链过长，主谓宾隔太远。
- 为文笔堆辞藻，拖慢节奏。
- 只统计不重写。

## 训练任务

选 10 个段落，统计“的”字，超过 2 个的段落重写为主谓宾更近的版本。
