# 短篇结构-起承转合与长篇续航

依据案例：batch-364 案例 1820

## 读者承诺

短篇也适用三幕式和起承转合，能训练完整故事闭环；但长篇还需要持续承诺、阶段升级和多轮期待。

## 核心循环

```text
短篇冲突 -> 起承转合闭环 -> 情绪收束 -> 长篇新增多轮期待
```

## 开篇必备

- 一个核心冲突。
- 起承转合清楚。
- 转折有效。
- 结尾收束。

## 期待来源

- 快速进入问题。
- 短篇一轮兑现。
- 长篇可扩成多轮冲突。
- 训练结构肌肉。

## 断档风险

- 会短篇就以为会长篇。
- 短篇冲突过多。
- 长篇缺续航。
- 只练闭环，不练阶段承诺。

## 训练任务

写一万字短篇大纲，只允许一个核心冲突；再列出扩成长篇必须新增的三条长线。
