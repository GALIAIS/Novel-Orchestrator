# 双金手指-主动挑事名利双收

依据案例：`batch-021` 案例 104。

## 读者承诺

用前世经验加情绪系统构成双金手指，让主角主动挑事、算计别人，并在结尾名利双收。

## 核心循环

```text
前世判断 -> 系统触发 -> 主动挑事 -> 配角反应 -> 奖励到账 -> 名利双收
```

## 开篇必备

- 主角前世大佬的判断力。
- 第二金手指的奖励规则。
- 主角主动挑事的目标。
- 这件事如何体现人设。

## 期待来源

- 读者看主角算计成功。
- 配角越夸张，反应越有反馈。
- 助人为乐、情绪收集、名声收益形成闭环。

## 断档风险

- 双金手指互相抢戏。
- 前世大佬导致危机感全无。
- 主动挑事不符合人设。
- 台词只喊口号，不体现人设。

## 训练任务

写一个女主主动挑事桥段：前世经验判断对方破绽，系统奖励推动她出手，结尾必须有名声和实际收益。
