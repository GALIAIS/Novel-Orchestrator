# 大神序章-新人不可学的慢进主线

依据案例：`batch-205` 案例 1022。

## 读者承诺

大神可以靠作者信用、世界观和人设让读者等待主线，新人第一章不要轻易写关系不明的序章短篇。

## 核心循环

```text
作者信用 -> 序章吸引 -> 主线延后 -> 人设世界观硬抗 -> 新人模仿流失
```

## 开篇必备

- 新人优先写清主角。
- 1000 字内给当前冲突。
- 能力或特殊身份尽早出现。
- 少用“序章”降低进入感。

## 期待来源

新人读者需要马上知道看谁、看什么冲突、为什么继续追。

## 断档风险

- 学大神慢进主线。
- 用序章拖延正篇。
- 世界观和人设不足以硬抗。
- 主角能力太晚出现。

## 训练任务

把一个序章短篇改成新人第一章：保留氛围，1000 字内出现主角、当前冲突、能力或特殊身份。
