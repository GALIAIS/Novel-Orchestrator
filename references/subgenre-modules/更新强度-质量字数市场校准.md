# 更新强度-质量字数市场校准

依据案例：batch-307 案例 1533

## 读者承诺

稳定更新是网文竞争的基础承诺之一。质量让读者留下，稳定产能让读者放心追。

## 核心循环

```text
确定平台节奏 -> 建立稳定日更 -> 按内容划分章节 -> 保持质量底线 -> 读者形成追更信任
```

## 开篇必备

- 明确平台和题材的常见更新节奏。
- 设定能长期执行的最低日更。
- 给改文和大纲预留时间。
- 大章小章按内容节点划分。

## 期待来源

期待来自稳定供给。读者知道作品会持续更新，且每章有足够内容，就更愿意订阅、投票和追读。

## 断档风险

- 只和大神比字数，忽视个人能力。
- 用质量为王当拖更借口。
- 大章小章只看技巧，不看阅读体验。
- 更新量上去了，故事质量掉线。

## 训练任务

设计一周可执行更新表：每日字数、章纲时间、改文时间、章节节点和最低质量检查项。
