# 飞卢节奏-点到即止下行与快速震惊

依据案例：`batch-246` 案例 1228。

## 读者承诺

飞卢式爽文可以有下行，但下行短、轻、快，马上用展示和震惊反馈拉回上升。

## 核心循环

```text
强期待 -> 一句质疑 -> 主角展示 -> 路人震惊 -> 爽感继续上升
```

## 开篇必备

- 金手指给强预期。
- 下行只点一下。
- 迅速给展示动作。
- 立刻安排震惊反馈。

## 期待来源

读者想持续看主角起飞，小阻力只是为了让展示更亮。

## 断档风险

- 下行过长。
- 质疑太重。
- 展示反馈太慢。
- 学到装逼味，没学到顺滑节奏。

## 训练任务

写 500 字爽点段，下行不超过 80 字，200 字内必须回到主角展示和旁人震惊。
