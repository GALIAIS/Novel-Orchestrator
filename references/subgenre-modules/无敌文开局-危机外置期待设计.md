# 无敌文开局-危机外置期待设计

依据案例：`batch-183` 案例 915。

## 读者承诺

无敌文开局不必削弱主角制造生死危机，可以把危机外置到配角、后人、名誉、秩序或计划上，再用无敌金手指制造期待。

## 核心循环

```text
主角无敌 -> 危机外置 -> 不可退让后果 -> 金手指制造机会 -> 爽点兑现
```

## 开篇必备

- 主角无敌点要清楚。
- 承压对象要和主角有关。
- 后果要不可退让。
- 主角解决方式不能降格。

## 期待来源

读者期待无敌主角用高位能力解决别人解决不了的问题，而不是被强行拉低到普通生死危机。

## 断档风险

- 强行让无敌主角怕死。
- 配角危机和主角无关。
- 主角直接碾压导致无波澜。
- 只有冲突，没有金手指期待。

## 训练任务

设计一个无敌文开局：配角遭遇社会性死亡风险，主角不直接代打，而是用无敌能力制造翻盘机会。
