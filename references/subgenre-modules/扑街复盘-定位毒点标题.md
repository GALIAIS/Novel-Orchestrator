# 扑街复盘-定位毒点标题

依据案例：batch-297 案例 1484

## 读者承诺

扑街后按开篇毒点、市场定位、类型主线、标题吸量四项拆解，而不是笼统归因运气。

## 核心循环

```text
数据扑街 -> 查开篇完读 -> 查类型承诺 -> 查升级线 -> 查书名噱头 -> 扫榜修正
```

## 开篇必备

- 前三章主角不能出现高风险退缩。
- 类型承诺要集中。
- 玄幻脑洞必须有升级线。
- 书名能提供点击理由。

## 期待来源

- 读者点进来能马上得到承诺。
- 主角行为符合爽文期待。
- 标题和正文卖点一致。

## 断档风险

- 玄幻里混太多商战都市，承诺错配。
- 开篇主角退缩，完读崩。
- 书名平，流量入口弱。

## 训练任务

复盘一本扑街书前三章：标出主角退缩点、类型错配点、升级线缺口、标题噱头缺口。
