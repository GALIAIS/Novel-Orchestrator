# 高武模拟器-精简人物煽动

依据案例：`batch-139` 案例 694。

## 读者承诺

在高武都市里给主角一个修仙模拟器外挂，用异质成长路线制造降维逆袭；开局少人物、少闲聊，把注意力集中在主角劣势、外挂激活和第一次验证。

## 核心循环

```text
高武弱势 -> 修仙模拟器激活 -> 获得异质成长 -> 外部压力逼验证 -> 主角破局打脸
```

## 开篇必备

- 高武世界的基础压力。
- 主角当前劣势。
- 修仙模拟器首次触发。
- 一个阻碍者。
- 一个亲近者或旁观压力源。
- 第一次必须验证外挂的冲突。

## 期待来源

- 读者想看修仙外挂如何碾压高武规则。
- 主角劣势越清晰，变强反馈越直接。
- 外界反对和轻视为打脸蓄力。
- 第一次模拟收益立刻服务冲突。

## 断档风险

- 第二章开始人物过多。
- 对白太多，替代了行动和事件推进。
- 群众反感、家人反对只是噪音，没有服务破局。
- 金手指设定可用，但迟迟没有验证爽点。

## 训练任务

写 300 字开局梗概，只允许主角、阻碍者、亲近者三人进入核心镜头。三段分别完成：高武劣势、模拟器激活、必须马上验证外挂的冲突。
