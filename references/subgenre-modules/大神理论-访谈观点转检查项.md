# 大神理论-访谈观点转检查项

依据案例：batch-350 案例 1749

## 读者承诺

大神访谈的价值不是崇拜作者，而是把写作理念转成可检查、可复盘、可应用的具体动作。

## 核心循环

```text
访谈观点 -> 对应写作问题 -> 转成检查项 -> 应用到章节 -> 复盘效果
```

## 开篇必备

- 摘出明确观点。
- 对应自己的章节问题。
- 转成一句检查问句。
- 找应用位置。
- 复盘是否改善读者感受。

## 期待来源

新人看到大神也有理论和长期失败积累，从而相信方法训练有意义。

## 断档风险

- 收藏访谈但不应用。
- 把理论当口号。
- 迷信天赋。
- 只复刻题材，不学方法。
- 爽点不爽时只加力度，不补铺垫。

## 训练任务

把一个访谈观点改成检查项，立刻检查自己一章稿，记录修改前后的爽点变化。
