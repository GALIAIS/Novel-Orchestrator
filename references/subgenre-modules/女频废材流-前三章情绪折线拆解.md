# 女频废材流-前三章情绪折线拆解

依据案例：`batch-204` 案例 1018。

## 读者承诺

女频废材流前三章要用密集的上行/下行情绪折线抓人，套路老也能靠情绪密度成立。

## 核心循环

```text
惨境下压 -> 穿越反击 -> 再遇压迫 -> 男主/金手指拉扯 -> 章末危机
```

## 开篇必备

- 每章约 2000 字稳定推进。
- 每 300-500 字有情绪节点。
- 上行和下行交替。
- 暧昧拉扯服务情绪转折。

## 期待来源

读者持续经历受压、反击、拉扯、危机，形成快节奏追读。

## 断档风险

- 情绪长时间平直。
- 暧昧戏只占字数，不产生转折。
- 金手指和危机节点太疏。
- 只拆剧情，不标情绪。

## 训练任务

拆一本女频热门书前三章：每 300-500 字标剧情功能和“上行/下行/平稳”，画出折线。
