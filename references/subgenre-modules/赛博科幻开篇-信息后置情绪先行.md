# 赛博科幻开篇-信息后置情绪先行

依据案例：batch-266 案例 1330

## 读者承诺

在赛博朋克、硬科幻、严肃科幻等高设定题材里，先让读者抓住主角、冲突和情绪，再逐步接收世界观信息。

## 核心循环

```text
陌生世界 -> 主角处境 -> 当下冲突 -> 情绪代入 -> 最小设定解释 -> 后续背景展开
```

## 开篇必备

- 第一屏让读者知道主角是谁和正在遇到什么。
- 专有名词限量，优先解释会影响当下行动的设定。
- 人物关系要先给最小暗示，不能全部拖到后文。
- 主角强大要通过行动展示，不只靠设定说明。

## 期待来源

期待来自“我看懂了这个人在这个世界里遇到的麻烦”。风格和设定越陌生，越需要清晰的情绪入口。

## 断档风险

- 信息轰炸带来质感，但读者不知道剧情线。
- 关系和动机拖到第三、四、五章才解释。
- 主角很强却缺少当场证明。
- 科幻氛围压过网文开篇钩子。

## 训练任务

重写一个高设定开篇，只保留 3 个专有名词。第一屏写处境，第二屏写冲突，第三屏才补最少背景。
