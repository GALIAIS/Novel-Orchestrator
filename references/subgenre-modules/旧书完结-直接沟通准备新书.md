# 旧书完结-直接沟通准备新书

依据案例：`batch-328` 案例 1640

## 读者承诺

旧书扑街或长期停更时，完结可以正常沟通；下一本能否过，关键看新书亮点和持续写下去的可能。

## 核心循环

```text
旧书扑街/停更 -> 直接说明完结 -> 复盘教训 -> 准备新书亮点 -> 重新投稿
```

## 开篇必备

- 按编辑要求沟通完结。
- 完结说明简短、诚实。
- 复盘旧书赶热度、边想边写的问题。
- 新书至少达到及格线。
- 给出亮点和可持续性。

## 期待来源

作者期待放下旧书心理负担，把精力转到更有准备的新项目。

## 断档风险

- 因心虚一直拖着旧书。
- 新书仍然边想边写冲保底。
- 只想换项目，不复盘旧书问题。
- 轻小说频道竞争被低估。
- 新书没有亮点和持续写下去的证明。

## 训练任务

写一段 150 字以内的完结说明，再写新书亮点、前三章承诺、可持续写作证明。
