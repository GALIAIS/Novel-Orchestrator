# 书评学习-主观批评与商业成绩校验

依据案例：`batch-217` 案例 1083。

## 读者承诺

从书评中提炼可用技法，但不用主观审美替代商业成绩和读者反馈。

## 核心循环

```text
书评观点 -> 受众定位识别 -> 成绩事实校验 -> 读者爽点提炼 -> 写法转化
```

## 开篇必备

- 区分文学批评和商业写作教程。
- 记录作品成绩事实。
- 分析读者为什么付费。
- 把评价转成具体写法动作。

## 期待来源

作者既能获得书评角度，又不会被 UP 主个人审美带偏开书判断。

## 断档风险

- 把主观暴论当真理。
- 只学骂点，不学成绩来源。
- 忽略评论者自己的受众和运营策略。
- 分析停在好坏判断，不落到写法。

## 训练任务

选一个热门书评视频，拆受众、评价标准、成绩证据、读者爽点和 3 个可执行写法动作。
