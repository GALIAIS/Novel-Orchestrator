# 新媒体赘婿-委屈反转流

依据案例：`batch-023` 案例 115。

## 读者承诺

把现实生活中的家庭、阶层、婚姻和工作委屈放大，再用隐藏身份或资源反转释放。

## 核心循环

```text
低位身份 -> 公开羞辱 -> 读者知道底牌 -> 众人误判 -> 小反转打脸 -> 新委屈接上
```

## 开篇必备

- 公开场合。
- 低位身份，如赘婿、家庭主妇、失意职员。
- 亲友或同事的不公平对待。
- 读者提前知道的底牌。
- 一个具体物件或需求。

## 期待来源

- 现实压力容易共鸣。
- 公开羞辱放大释放需求。
- 角色不知道底牌，读者提前期待。
- 反转落到钱、车、房、证、工作或婚姻。

## 断档风险

- 把受众理解成“没脑子”，写不到真实压力。
- 爽点太离地，脱离生活委屈。
- 只有羞辱没有底牌。
- 反转后没有新压迫接续。

## 训练任务

写一个 800 字新媒体开篇：寿宴、家庭聚会、公司会议三选一；必须有公开羞辱、隐藏底牌和一次小反转。
