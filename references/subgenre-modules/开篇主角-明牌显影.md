# 开篇主角-明牌显影

依据案例：`batch-129` 案例 641。

## 读者承诺

第一章必须让读者知道谁是主角、主角有什么卖点、题材承诺何时出现。伪装和误导可以骗配角，不能骗到读者无法代入。

## 核心循环

```text
书名承诺 -> 主角明牌登场 -> 题材卖点亮相 -> 第一冲突压上 -> 读者建立代入
```

## 开篇必备

- 500 字内明确主角。
- 书名卖点尽快出现。
- 主角第一动作讨喜或有吸引力。
- 金手指/能力不能拖太久。
- 伪装不伤害读者判断。

## 期待来源

- 读者知道该关注谁。
- 题材承诺尽快兑现。
- 主角卖点清楚。
- 第一冲突能服务主角展示。

## 断档风险

- 第一章不见主角。
- 大叔、黑衣人等角色抢走判断。
- 书名有法师，正文迟迟不点法师。
- 任务链先行，主角魅力缺席。

## 训练任务

把一个主角伪装登场的开篇改成明牌版：500 字内让读者知道谁是主角、他有什么卖点、他正在面对什么问题。
