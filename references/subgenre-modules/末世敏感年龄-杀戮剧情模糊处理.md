# 末世敏感年龄-杀戮剧情模糊处理

依据案例：`batch-219` 案例 1093。

## 读者承诺

末世生存里涉及年龄和杀戮时，模糊敏感身份，把焦点放在生存压力和被迫选择。

## 核心循环

```text
敏感年龄 -> 模糊表述 -> 生存压力 -> 被迫行动 -> 后果承担
```

## 开篇必备

- 不精确强调未成年身份。
- 用“和平年代也就是上高中的年纪”等模糊表述。
- 杀戮必须有生存必要性。
- 不把敏感点当噱头。

## 期待来源

读者关注主角如何活下去，而不是被年龄风险和敏感描写打断。

## 断档风险

- 反复强调年龄。
- 把杀戮写成炫耀。
- 用“刚满十八岁”硬凑。
- 忽略具体平台规则差异。

## 训练任务

删除具体年龄，写 3 句模糊表达，补 3 个生存压力证据，并把杀戮改成被迫防卫。
