# 都市脑洞-主线凝聚桥段

依据案例：`batch-055` 案例 274。

## 读者承诺

这个模块解决都市脑洞、神豪、较真、商业、异能等题材中“装逼桥段很多但剧情很散”的问题。它承诺的是：先用金手指和主线给桥段分工，再让爽点作为事件过程中的携带物出现。

## 核心循环

```text
金手指定位 -> 可识别外皮 -> 主线终局 -> 桥段分类 -> 事件服务主线 -> 爽点兑现 -> 身份/事业/关系升级
```

## 开篇必备

- 先说清金手指或核心脑洞。
- 给作品一个市场能识别的外皮：神豪、职业、商业、异能、灵气复苏、轻小说等。
- 设定作者安排的终局，不只写主角嘴上想要什么。
- 第一批装逼桥段必须服务同一条主线。
- 每个桥段要推进身份、事业或关系，不能只负责打脸。

## 期待来源

- 主角每次打脸都让长期身份更清晰。
- 金手指不断制造不同层级的现实反应。
- 旁人误判和主线目标绑定，爽点不孤立。
- 事业、关系、资源随事件推进发生变化。
- 读者能预见主角最终会到达某个更高位置。

## 断档风险

- 只堆 KTV、豪车、楼盘、阔少、国家站台等孤立桥段。
- 不说明金手指，导致类型无法判断。
- 桥段只有爽点，没有目的性。
- 前期爽点强，后期只能重复同类打脸。
- 金手指太复杂但没有可识别外皮，市场无法归类。

## 训练任务

列出 10 个都市装逼桥段，把它们分别标成身份推进、事业推进、关系推进。每类只保留最能服务主线的 2 个，并写出该桥段结束后主角实际获得了什么长期变化。
