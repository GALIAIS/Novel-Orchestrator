# 情绪放大-多角度重复三遍

依据案例：`batch-258` 案例 1286。

## 读者承诺

情绪放大模块用多个角度重复同一核心信息，让爽感、压迫感或震惊感逐层增强。

## 核心循环

```text
核心情绪 -> 本人表现 -> 行动后果 -> 旁人反应 -> 敌人反馈 -> 强度放大
```

## 开篇必备

- 先确定唯一核心信息。
- 每次重复换一个视角。
- 每层都要提供新信息。
- 通过反馈判断是否啰嗦。

## 期待来源

读者从本人、旁人、敌人等多个参照系同时确认强度，情绪被稳稳放大。

## 断档风险

- 原地同义重复。
- 每层不增加强度。
- 只为凑字，不服务爽点。
- 重复过量，读者觉得啰嗦。

## 训练任务

写 5 个角度表达同一个信息，每个角度必须有新信息，最后删掉最弱的一条。
