# 新人同人-开书准备导向

依据案例：`batch-149` 案例 745。

## 读者承诺

新人第一次写同人时，最需要的不是零散技巧，而是一个开书前准备入口：先知道该看什么、想什么、准备什么，再开始写第一章。

## 核心循环

```text
表达冲动 -> 写不好焦虑 -> 准备路径缺失 -> 系统教程导向 -> 建立开书判断 -> 再动笔
```

## 开篇必备

- 明确新人想写的原作或同人方向。
- 承认表达欲和焦虑并存。
- 不现场碎答过大问题。
- 指向完整教程或系统材料。
- 要求看完后形成基本判断。

## 期待来源

新人期待的是降低不确定性：知道自己不是“马上写或放弃”，而是可以先按教程建立路线感。

## 断档风险

- 把宽泛问题拆成零散鸡汤。
- 未读外链却假装掌握全部步骤。
- 只鼓励码字，不让新人先确认读者期待和前三章兑现。
- 系统材料太多但没有入口顺序。

## 训练任务

列一张同人开书前自问清单：原作吸引点、同人读者想补偿什么、主角如何介入原剧情、前三章兑现什么、要先阅读哪些教程或样本。清单未填完前不写正文。
