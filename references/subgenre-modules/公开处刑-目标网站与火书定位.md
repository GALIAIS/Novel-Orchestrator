# 公开处刑-目标网站与火书定位

依据案例：batch-380 案例 1898

## 读者承诺

公开求评前先写清目标网站、分类、标签和相似火书。反馈要对准目标读者，避免把原本的核心卖点改没。

## 核心循环

```text
样稿 -> 定位卡 -> 目标赛道反馈 -> 筛选建议 -> 保住卖点改问题
```

## 开篇必备

- 目标网站。
- 目标分类和标签。
- 三本相似火书。
- 需要保住的核心卖点。

## 期待来源

- 评者知道按什么标准看。
- 作者重新确认卖点。
- 同赛道建议更精准。
- 错位审美更容易被过滤。

## 断档风险

- 没定位就公开处刑。
- 被非目标读者带偏。
- 改成四不像。
- 只求被骂，不筛建议。

## 训练任务

发样稿前写定位卡：目标网站、分类、三个标签、三本相似火书、核心卖点、三个反馈问题；只采纳不破坏定位的建议。
