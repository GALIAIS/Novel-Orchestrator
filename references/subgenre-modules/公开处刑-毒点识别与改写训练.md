# 公开处刑-毒点识别与改写训练

依据案例：batch-354 案例 1768

## 读者承诺

公开处刑文给新人提供高可见度错误样本。通过挑毒点、分析出戏原因、私下改写，作者能把别人的问题变成自己的自查预警。

## 核心循环

```text
处刑样本 -> 毒点标注 -> 出戏原因分析 -> 改写练习 -> 自己正文快速识别
```

## 开篇必备

- 明确毒点位置。
- 说明读者为什么出戏。
- 给出一版改写方案。
- 把毒点写入自查表。

## 期待来源

- 错误比优秀更容易拆。
- 改写让判断变成肌肉记忆。
- 摘下滤镜后能当天发现自己问题。
- 公开样本降低试错成本。

## 断档风险

- 只嘲笑不改写。
- 只看别人错，不对照自己。
- 毒点不归类，无法复用。
- 把处刑当成羞辱而不是训练。

## 训练任务

每周选 5 个处刑毒点，每个写 100 字改写，并归类到“逻辑、代入、节奏、设定、爽点”五类之一。
