# 冲突赌注-观众期待与失去风险

依据案例：batch-370 案例 1850

## 读者承诺

赌注不是危险本身，而是冲突威胁到读者已经在意、已经下注、不愿失去的人、物或关系。

## 核心循环

```text
建立期待 -> 读者在意 -> 冲突威胁 -> 失去可能 -> 赌注成立
```

## 开篇必备

- 先让读者在意。
- 明确主角与赌注关系。
- 冲突真实威胁赌注。
- 胜败结果情绪不同。

## 期待来源

- 美女接近主角后可能远离。
- 救命药剂能救在意的人。
- 主角重要关系被威胁。
- 读者觉得自己也在下注。

## 断档风险

- 只有危险，没有读者在意。
- 配角赌注和主角无关。
- 家族破产等危机没铺垫意义。
- 套电影公式忽视网文主角中心。

## 训练任务

给一个普通战斗补赌注：先写读者在意对象，再写冲突如何威胁它，最后写胜败情绪差。
