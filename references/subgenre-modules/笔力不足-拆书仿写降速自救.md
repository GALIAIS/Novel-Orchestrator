# 笔力不足-拆书仿写降速自救

依据案例：batch-272 案例 1357

## 读者承诺

把“笔力不够”拆成可训练的表达短板：动作、神态、场景、打斗、节奏。通过拆书和相似段落仿写，减少无参照反复修改导致的写作降速。

## 核心循环

```text
写不顺 -> 找相似样本 -> 拆表达功能 -> 仿写小段 -> 回写自己的剧情
```

## 开篇必备

- 先定位具体短板，不要笼统说笔力差。
- 样本要找功能相似的段落。
- 每次只仿一个技术点。

## 期待来源

作者期待来自“我知道该怎么改”。有样本结构后，修改从感觉判断变成动作顺序、句长节奏、视角落点的具体调整。

## 断档风险

- 没有样本地反复自改。
- 仿大神辞藻，不仿段落功能。
- 一次想补齐所有短板，训练失焦。

## 训练任务

拆一段 500 字样本，标出动作、神态、环境、心理、结果，再按同样比例写自己的桥段。
