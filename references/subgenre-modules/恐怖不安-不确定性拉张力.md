# 恐怖不安-不确定性拉张力

依据案例：`batch-037` 案例 182。

## 读者承诺

恐怖感不只靠怪物杀人，而靠“看起来不对劲但不能立刻下结论”的不确定性。读者和主角一起不安，张力才会持续。

## 核心循环

```text
异常外形 -> 主角不安 -> 现实理由压住判断 -> 猪队友推进 -> 恐怖升级兑现
```

## 开篇必备

- 异常细节具体。
- 主角直觉不安。
- 不能马上确认危险。
- 旁人无视或反驳线索。

## 期待来源

- 读者知道会出事。
- 主角没有证据，只能忍耐。
- 掌权者推进错误决定。
- 恐怖兑现后再升级。

## 断档风险

- 怪物太早确认。
- 只写惊吓，不写真实细节。
- 猪队友没有现实功能。
- 不安没有升级兑现。

## 训练任务

写 800 字恐怖桥段：前 500 字只允许不安和细节，不允许怪物直接伤人。
