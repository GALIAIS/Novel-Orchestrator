# 校园格斗-受害者同情与正义出手

依据案例：`batch-239` 案例 1192。

## 读者承诺

校园格斗爽点要让读者支持主角出手；被救对象越无辜，主角越有正义感。

## 核心循环

```text
朋友遭威胁 -> 主角机智缓解 -> 危机升级 -> 受害者无辜性确认 -> 格斗爆发
```

## 开篇必备

- 明确被救对象。
- 保留受害者同情心。
- 让主角先有聪明上行。
- 最后用能力完成正义爽点。

## 期待来源

读者看到恶人压迫无辜者，等待主角把风险扛起来并赢回尊严。

## 断档风险

- 受害者家人自作自受。
- 主角出手像多管闲事。
- 下行让主角背锅。
- 混混威胁和债务原因不可信。

## 训练任务

同一段替朋友出头剧情，分别写“赌债版”和“被胁迫版”，比较哪版更能让读者站主角。
