# 资料管理-军事资料留存

依据案例：`batch-196` 案例 980。

## 读者承诺

军事资料帖要记录来源、标签、可用状态和适用题材；未读或失效资料不能当作已吸收知识。

## 核心循环

```text
发现资料 -> 记录链接 -> 补标签 -> 标记可用状态 -> 写作前二次核验
```

## 开篇必备

- 资料主题清楚。
- 来源和访问方式完整。
- 标注是否可访问。
- 写明适用题材和场景。

## 期待来源

作者需要军事细节时能快速定位资料，并知道哪些内容还要核验。

## 断档风险

- 网盘链接失效。
- 没有标签，后续找不到。
- 未读资料被当成事实引用。
- 军事细节凭印象乱写。

## 训练任务

做一张军事资料卡，字段只保留主题、来源、可用状态、适用题材、可写场景和核验提醒。
