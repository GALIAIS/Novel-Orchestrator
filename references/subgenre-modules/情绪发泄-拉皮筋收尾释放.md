# 情绪发泄-拉皮筋收尾释放

依据案例：`batch-112` 案例 558。

## 读者承诺

情绪发泄不只靠打脸，也可以靠死亡、告别、埋葬和启程。关键是先把情绪拉紧，再用关键选择释放，最后用收尾动作让情绪平息。

## 核心循环

```text
铺垫人物 -> 拉起情绪 -> 关键释放 -> 收尾仪式 -> 新期待
```

## 开篇必备

- 明确主角目标。
- 目标有自身、伙伴、事件三重阻碍。
- 关键事件改变主角选择。
- 收尾动作分别解决阻碍。
- 平息后开启下一段期待。

## 期待来源

- 情感压抑得到圆满句号。
- 角色选择有重量。
- 日常收尾也有爽感。
- 下一段冒险被自然打开。

## 断档风险

- 只有悲伤，没有释放。
- 收尾日常和目标无关。
- 人物刻画不足，死亡没重量。
- 释放后没有新期待。

## 训练任务

写一个“导师牺牲后主角启程”的小单元：接受召唤、埋葬导师、告别家乡三个动作分别解决一个阻碍。
