# 金钱体系-收入支出闭环

依据案例：batch-281 案例 1403

## 读者承诺

金钱体系要像升级体系一样提前设计：收入增长必须匹配支出、损耗、建设、金手指收费和新目标，否则财富膨胀会让故事失去压力。

## 核心循环

```text
赚钱 -> 支出升级 -> 换取资源/进展 -> 新目标变贵 -> 再赚钱
```

## 开篇必备

- 当前阶段收入上限。
- 必要支出项目。
- 金手指或能力的收费点。
- 下阶段更贵的目标。

## 期待来源

读者既看到主角赚钱成功，也看到钱立刻转化成更大行动力，财富不会只变成空数字。

## 断档风险

- 一高兴就写几个亿。
- 只涨收入，不涨支出。
- 强行清空财富像作者硬削。
- 货币设定过复杂，读者只觉得麻烦。

## 训练任务

做 5 阶段金钱表：每阶段写收入、必要支出、剧情收益和下一阶段更贵目标。
