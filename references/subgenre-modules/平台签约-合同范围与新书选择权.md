# 平台签约-合同范围与新书选择权

依据案例：`batch-255` 案例 1275。

## 读者承诺

平台签约模块把“平台是不是签人”的焦虑拆成合同范围、独家期限和新书选择权。

## 核心循环

```text
签约传闻 -> 查看合同文本 -> 标出限制范围 -> 询问编辑确认 -> 决定平台策略
```

## 开篇必备

- 区分签作品、签笔名、签作者主体。
- 看清独家期限和新书限制。
- 问清违约责任。
- 不把社群经验当法律结论。

## 期待来源

作者知道自己签约后还能不能开新书、去哪里开、承担什么成本，项目规划更稳定。

## 断档风险

- 只听传闻，不看合同。
- 觉得执行成本低就忽略条款。
- 签前不问新书限制。
- 把旧经验当当前平台规则。

## 训练任务

做一张签约风险表：合同对象、独家期限、新书限制、违约责任、收益分成、终止条件，每项写一个确认问题。
