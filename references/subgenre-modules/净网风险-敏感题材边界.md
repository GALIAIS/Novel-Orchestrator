# 净网风险-敏感题材边界

依据案例：batch-280 案例 1398

## 读者承诺

净网期主动识别涉政、未成年、骨科、师生恋、炼铜、现实争议等高风险元素，发书前降级或替换，避免封章、限流和上架后大改。

## 核心循环

```text
风向变化 -> 敏感元素排查 -> 平台尺度判断 -> 降级/删除/替换 -> 连载风险降低
```

## 开篇必备

- 恋爱关系年龄和身份检查。
- 国内时事政治检查。
- 暴力、色情、亲属关系检查。
- 平台近期风向记录。

## 期待来源

作者保住连载稳定性，读者不会因为章节被封、情节大改或作品断更而流失。

## 断档风险

- 以“别人能写”为安全证据。
- 火书尺度套到新人书。
- 发书时过审就以为永久安全。
- 高风险元素没有替代表达。

## 训练任务

做一张净网风险表：每个敏感元素标注保留、降级、删除或替换，并写一句替代方案。
