# 影视节奏转网文-上下行升级与重点文字化

依据案例：batch-355 案例 1773

## 读者承诺

影视作品适合学习上下行、至暗时刻、救场爆发和连续升级，但文字不能照搬镜头信息量。改成网文时，只保留情绪因果和爽点必要画面。

## 核心循环

```text
影视片段 -> 标注上下行 -> 找连续升级 -> 选择重点画面 -> 转成文字因果和爽点
```

## 开篇必备

- 明确下行如何加深。
- 明确至暗时刻。
- 明确救场爆发点。
- 明确哪些镜头信息不写。

## 期待来源

- 读者持续感到危机升级。
- 救场前压迫足够。
- 前期铺垫在爆发时收束。
- 文字节奏紧，不被镜头细节拖慢。

## 断档风险

- 把所有镜头都翻译成文字。
- 只学燃点，不学铺垫和限制。
- 作者表达高出读者太多。
- 没有必要解释，读者无法接受爆发。

## 训练任务

拆一个 3 分钟影视片段，只保留 5 个文字节点：铺垫、下行、至暗、救场、升级。每个节点不超过 80 字。
