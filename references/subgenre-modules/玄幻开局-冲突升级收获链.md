# 玄幻开局-冲突升级收获链

依据案例：`batch-219` 案例 1094。

## 读者承诺

玄幻开局要快速形成机会、冲突、装逼、收获和成长可视化。

## 核心循环

```text
机会出现 -> 他人争夺 -> 主角反制 -> 装逼兑现 -> 资源收获 -> 成长升级
```

## 开篇必备

- 场景被写出来就要有变故。
- 机会和冲突绑定。
- 收获能推动境界、资源或身份提升。
- 等级或战绩让成长可视。

## 期待来源

读者期待主角从压迫和争夺中反击，拿到资源并进入更大的修行冲突。

## 断档风险

- 反套路时删掉冲突和收获。
- 只有套路表面，没有功能兑现。
- 亲情线只当负担。
- 成长没有可见反馈。

## 训练任务

把“地摊买宝被抢”改成 3 个非地摊场景，保留机会暴露、争夺、反制和收获。
