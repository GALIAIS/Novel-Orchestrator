# 玄幻开篇-不公平伤害与危机叠加

依据案例：`batch-243` 案例 1214。

## 读者承诺

玄幻开篇用不公平伤害制造共鸣，再叠加生死危机时，必须给足等量上行补偿。

## 核心循环

```text
权利被夺 -> 读者不平 -> 生死危机加压 -> 金手指开启 -> 更高身份/传承补偿
```

## 开篇必备

- 写清本该属于主角的东西。
- 写清剥夺者和剥夺理由。
- 追加危机时控制下行重量。
- 用金手指给出更大补偿。

## 期待来源

读者对“本该属于我却被抢走”更容易共鸣，并期待主角拿到更大的新位置。

## 断档风险

- 只虐主角，不给上行。
- 生死危机叠太重。
- 被夺权利不够具体。
- 金手指只保命，不补身份和尊严。

## 训练任务

写一个被夺权开篇，再叠一个暗杀危机，最后补一个比原权利更高的能力、传承或身份。
