# 学习顺序-先拆小说再读理论

依据案例：`batch-324` 案例 1617

## 读者承诺

理论书读不进去时，不要硬啃；先拆同赛道小说建立结构直觉，等写作遇到具体问题，再回头查理论。

## 核心循环

```text
目标题材 -> 拆同赛道作品 -> 记住剧情结构 -> 写作遇到卡点 -> 回查理论 -> 转成动作
```

## 开篇必备

- 先选 3 本同题材已验证作品。
- 只拆前三章的场景、危机、钩子。
- 不把理论笔记当作写作成果。
- 带着具体卡点翻理论书。
- 每次只把一个理论概念转成一个写作动作。

## 期待来源

作者期待通过可模仿样本降低入门阻力，再用理论补足自己已经遇到的问题。

## 断档风险

- 把读完理论书当成开写前置条件。
- 记了很多概念，写时完全不会用。
- 没看够同赛道作品，就急着套理论。
- 学习顺序太重，导致写作停在准备阶段。

## 训练任务

拆 3 本同题材前三章，每章只记“场景、危机、钩子”；等自己写卡住，再查一个理论概念解决卡点。
