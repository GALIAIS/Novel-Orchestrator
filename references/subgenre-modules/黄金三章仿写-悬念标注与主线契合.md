# 黄金三章仿写-悬念标注与主线契合

依据案例：batch-357 案例 1781

## 读者承诺

黄金三章仿写能训练结构感，但悬念、配角反应、规则铺垫和金手指必须服务自己的主线。只照着标注悬念，不处理因果和主线契合，会让有趣设定变成突兀雷点。

## 核心循环

```text
仿写结构 -> 标注悬念 -> 补规则和人设铺垫 -> 金手指接入主线 -> 三章留住读者
```

## 开篇必备

- 本书自己的长期主线。
- 第一悬念和展示悬念。
- 配角行为的价值观铺垫。
- 主角冒险的安全规则或风险解释。

## 期待来源

- 反差简介带来兴趣。
- 悬念连续展开。
- 金手指有趣且能推动洗白主线。
- 配角反应怪但有逻辑。

## 断档风险

- 模仿痕迹大于自家主线。
- 好感度变化没有铺垫。
- 主角挑衅强者却没有安全理由。
- 视角第一人称和第三人称混乱。

## 训练任务

写一份黄金三章标注表，每章至少包含“悬念、铺垫、规则、主线功能”四栏。
