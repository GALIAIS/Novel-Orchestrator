# 长篇提纲-防烂尾三级结构

依据案例：`batch-010` 案例 047。

## 读者承诺

长篇不是靠开书前完美提纲撑住，而是靠阶段目标、局势变化、人物变化和高潮兑现不断接力。

## 核心循环

```text
阶段目标 -> 局势变化 -> 主角变化 -> 配角变化 -> 高潮兑现 -> 下一阶段钩子
```

## 开篇必备

- 只规划够当前阶段使用的提纲。
- 给阶段设置明确收获。
- 提前埋外部局势变化。
- 让配角和主角同步变化。

## 期待来源

- 读者看到世界不是静止背景。
- 主角苟着发育时，外部压力也在升级。
- 阶段高潮能回收前面变化。
- 每个阶段结束都有新风险。

## 断档风险

- 开书前追求完美长纲，迟迟不开写。
- 主角发育，局势和配角不动。
- 高潮只靠临时事件，不回收阶段铺垫。
- 追求统一理论，耽误实际推进。

## 训练任务

把一个卷纲切成 3 阶段。每阶段填“目标、局势、主角、配角、高潮、下一钩子”六项。
