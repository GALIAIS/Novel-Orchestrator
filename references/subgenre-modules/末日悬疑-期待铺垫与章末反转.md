# 末日悬疑-期待铺垫与章末反转

依据案例：batch-385 案例 1924

## 读者承诺

末日悬疑不必每章给爽点，但每章都要提供部分答案、扩大问题或改变局势，并用章末反转连接长线谜团。

## 核心循环

```text
危机谜团 -> 多章铺垫 -> 章末反转 -> 阶段爽点 -> 更远危险
```

## 开篇必备

- 核心生存危机。
- 可持续追问的长线谜团。
- 金手指未来可能。
- 设定章也能改变局势的反转。

## 期待来源

- 每章获得一部分答案。
- 新问题不断扩大世界。
- 伙伴与设定改变危机判断。
- 阶段脱险后仍有长线危险。

## 断档风险

- 机械复制两爽六铺垫比例。
- 设定章只有静态说明。
- 只拆十章就判断长伏笔。
- 初步感悟未经后文回收验证。

## 训练任务

人工拆一部末日悬疑至少十万字，逐章记录答案、新问题、局势变化和章末拉力，并回标伏笔回收。
