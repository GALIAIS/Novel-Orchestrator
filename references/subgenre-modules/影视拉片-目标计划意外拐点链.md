# 影视拉片-目标计划意外拐点链

依据案例：batch-267 案例 1332

## 读者承诺

把影视作品拆成可迁移的冲突链：目标、计划、意外、拐点、新计划、至暗时刻和终局方案，而不是只写剧情简介。

## 核心循环

```text
目标 -> 计划 -> 意外失败 -> 拐点资源 -> 新计划 -> 更大失败 -> 新方案
```

## 开篇必备

- 先写最终目标和失败代价。
- 每个意外都要改变角色选择。
- 每个拐点都要给新资源或新信息。
- 至暗时刻要对应核心目标失败。

## 期待来源

期待来自计划不断被打断又不断重组。读者会跟着角色在损失中寻找新方案，而不是看事件流水账。

## 断档风险

- 拉片只复述剧情，没有结构标签。
- 意外不改变计划，冲突无效。
- 拐点只是新事件，不提供新选择。
- 终局方案和前文资源无关。

## 训练任务

拆一部电影，写 12 行以内的冲突链。每行格式固定为“计划/意外/拐点：事件；角色选择变化：……”
