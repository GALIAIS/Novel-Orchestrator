# 玄幻境界命名-语义轴与类型语感统一

依据案例：`batch-406` 案例 2027

## 读者承诺

先确定每级独有的行动质变，再选择力量来源、变化动作或社会身份作为主语义轴；名称要统一、易排、易读，并在升阶事件中证明含义。

## 核心循环

```text
列等级质变 -> 选择主语义轴 -> 统一词根与递进后缀 -> 检查联想发音和长度 -> 读者盲排气质 -> 升阶事件证明 -> 正式名俗称分层使用
```

## 开篇必备

- 相邻等级先有功能差异。
- 一套体系只用一条主命名语法。
- 词根与前后缀能解释递进。
- 常见名称优先降低认知成本。
- 正式名、俗称和人物用语区分。
- 首次突破用行动与社会后果实证。

## 期待来源

读者从名称预判成长方向，再在突破和第一次应用中确认能力与身份质变；统一语义让长阶梯容易记忆。

## 断档风险

- 功能相同只换更响亮名称。
- 器官、方位和官职随机混搭。
- 为好听牺牲发音和排序。
- 把玄幻仙侠语感当成绝对边界。
- 名称承担全部说明却没有事件。
- 角色口语机械复述正式全称。

## 训练任务

先写五级行动质变，再用来源、动作、身份三种语义轴各命名一套；让五名读者盲排顺序和气质，只保留最一致且能猜到功能的一套。
