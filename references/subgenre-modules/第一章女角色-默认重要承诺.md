# 第一章女角色-默认重要承诺

依据案例：`batch-337` 案例 1685

## 读者承诺

男频第一章出现女角色，不一定是女主，但会被读者默认重要；不是女主也要给足功能或降低特写。

## 核心循环

```text
女角色开篇出场 -> 读者默认加权 -> 态度/感情线判断 -> 戏份或功能兑现 -> 避免投资浪费
```

## 开篇必备

- 明确她是否女主。
- 明确她对主角的态度。
- 控制感情线暗示。
- 非女主也要有不可替代功能。
- 工具人就一笔带过。

## 期待来源

读者期待第一章给到特写的女角色后续有关系推进、情绪价值或关键功能。

## 断档风险

- 给强特写后随手丢弃。
- 非女主却写出女主暗示。
- 伤害主角还想当女主。
- 工具人女性占用开篇注意力。
- 读者感情投资被浪费。

## 训练任务

检查第一章女角色：不是女主就写出前三章功能和退场/转关系方案；只是工具人就降特写。
