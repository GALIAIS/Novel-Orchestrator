# 写作参考-五类工具书分工

依据案例：`batch-340` 案例 1696

## 读者承诺

写作参考书要转成问题工具链，分别解决结构、冲突、人物、对话和诊断，而不是停在买书收藏。

## 核心循环

```text
当前卡点 -> 选择对应工具书 -> 抽出检查项 -> 套到作品片段 -> 用读者反馈验证
```

## 开篇必备

- 先列当前作品卡点。
- 每本书绑定一个写作环节。
- 阅读后必须产出检查项。
- 检查项要能立刻改稿。
- 不把版本、价格、收藏当成学习成果。

## 期待来源

作者期待参考书给方法，真正有效的是把方法落成开篇钩子、人物选择、冲突升级、对话潜台词和章节结尾检查表。

## 断档风险

- 只买书不拆书。
- 读完只记概念。
- 同时看太多书却不改稿。
- 用宏观理论逃避具体桥段。
- 不区分结构问题和表达问题。

## 训练任务

选 3 本参考书，各写 5 条可直接用于改稿的检查项，并标注它们分别服务哪个写作环节。
