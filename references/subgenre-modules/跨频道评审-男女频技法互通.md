# 跨频道评审-男女频技法互通

依据案例：`batch-197` 案例 984。

## 读者承诺

男女频可以互相交流，但评审要拆成通用写作层和频道专属层，避免把频道差异误判成错误。

## 核心循环

```text
跨频道投稿 -> 通用层评审 -> 频道层补充 -> 声明标准 -> 给出有效反馈
```

## 开篇必备

- 明确投稿频道和目标读者。
- 通用层看冲突、期待、节奏、动机。
- 频道层看关系推进和情绪价值。
- 不直接套不适用标准。

## 期待来源

作者既能得到基础写作反馈，也能保留频道特有的阅读承诺。

## 断档风险

- 男频标准硬套女频。
- 不懂术语却直接下结论。
- 把关系节奏误判为不推进。
- 忽略女频情绪价值。

## 训练任务

给一篇女频开头做双层评审：通用层 4 项，频道层 3 项，每项只写一个可修改动作。
