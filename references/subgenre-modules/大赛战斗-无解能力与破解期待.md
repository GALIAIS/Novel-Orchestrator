# 大赛战斗-无解能力与破解期待

依据案例：batch-378 案例 1888

## 读者承诺

大赛篇不靠连续打架取胜，而靠新能力、工具人验证、主角破解和战后收获持续刷新期待。

## 核心循环

```text
新敌人 -> 无解能力 -> 工具人验证 -> 主角破解 -> 线索/能力收获
```

## 开篇必备

- 清楚比赛目标。
- 读者熟悉的战斗体系。
- 一个强敌新能力。
- 战斗后的剧情收益。

## 期待来源

- 无解能力怎么破。
- 主角是否领悟新用法。
- 幕后干扰带来不公平伤害。
- 比赛外线索牵引后续。

## 断档风险

- 连续战斗无变化。
- 只写招式不写收益。
- 战斗体系读者不熟。
- 没有战前铺垫和战后结果。

## 训练任务

设计三场比赛：每场一个新能力、一个工具人验证、一个破解方式、一个剧情收获；第三场加入幕后干扰。
