# 熟人CP素材-群体磕点与原创化转写

依据案例：`batch-213` 案例 1064。

## 读者承诺

群体反复磕的熟人互动，往往已经具备可转写的关系张力和事件素材。

## 核心循环

```text
真实互动 -> 群体起哄 -> 关系张力 -> 原创化角色 -> 连续事件
```

## 开篇必备

- 提炼互动机制，不照搬隐私。
- 明确双方欲望、误会和反差。
- 写旁观者起哄如何放大关系。
- 做匿名化、原创化处理。

## 期待来源

读者容易被真实互动转化出的暧昧、笑点、反差和围观感吸引。

## 断档风险

- 未经边界处理直接写真人。
- 只有熟人梗，陌生读者看不懂。
- 关系没有持续推进。
- 原创设定输给真实互动张力。

## 训练任务

从一个群聊梗提炼欲望、误会、反差、旁观者起哄点，改写成完全原创 1000 字关系开篇。
