# 无敌流-配角铺垫压制

依据案例：`batch-026` 案例 127。

## 读者承诺

无敌主角的爽感来自别人复杂挣扎后的绝对压制，而不是无铺垫秒杀。

## 核心循环

```text
反派布局 -> 小弟周旋 -> 底牌暴露 -> 主角幕后拆解 -> 前台打脸 -> 身份加深
```

## 开篇必备

- 主角无敌来源。
- 反派误判。
- 小弟或好友前台行动。
- 主角暂不出手的理由。

## 期待来源

- 反派越复杂，压制越爽。
- 读者知道主角比局势更高一层。
- 搞笑好友代装逼制造喜剧。
- 神秘身份不断加深。

## 断档风险

- 主角一上来秒，缺少铺垫。
- 配角讨人厌，读者不想看他代装逼。
- 反派底牌太弱，压制没有反差。
- 无敌变成无事可写。

## 训练任务

写一段幕后无敌桥段：反派准备两层底牌，主角只拆关键点，让配角在前台完成一次误打误撞的装逼。
