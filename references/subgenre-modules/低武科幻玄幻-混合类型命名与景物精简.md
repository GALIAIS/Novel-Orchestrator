# 低武科幻玄幻-混合类型命名与景物精简

依据案例：`batch-201` 案例 1002。

## 读者承诺

赛博低武玄幻要让读者同时看见科幻压迫和玄幻反差，不能只剩财阀、机甲和景物描写。

## 核心循环

```text
主角行动 -> 财阀压迫 -> 玄幻招式/低武能力 -> 科幻武器对照 -> 类型反差成立
```

## 开篇必备

- 第一段从人物行动开始。
- 快速展示财阀或赛博压迫。
- 玄幻命名要进入招式、称呼或规则。
- 科幻元素要和玄幻元素产生对比。

## 期待来源

读者期待低武/玄幻力量在高科技财阀秩序里制造反差破局。

## 断档风险

- 景物太多，读者没看到事件。
- 玄幻信号不足，像普通低武科幻。
- 经济逻辑不清，老兵征召显得不合理。
- 网络词或人物反应破坏沉浸。

## 训练任务

写 800 字赛博低武玄幻开头：第一段必须行动起笔，同时出现财阀压迫、玄幻招式名、科幻武器/机甲名。
