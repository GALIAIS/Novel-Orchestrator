# 起点内投-正确投稿方式入口

依据案例：batch-287 案例 1434

## 读者承诺

把平台投稿资料入口纳入创作系统，减少新人因渠道、材料和格式不清造成的无效投递。

## 核心循环

```text
准备投稿 -> 找官方资料 -> 核对材料 -> 整理正文大纲卖点 -> 正确渠道提交 -> 记录反馈
```

## 开篇必备

- 目标平台。
- 官方或作家专区资料链接。
- 投稿渠道。
- 正文和大纲准备状态。
- 简介、卖点、联系方式。
- 投递日期和反馈记录。

## 期待来源

作者确认流程后，能把焦虑从“怎么投”转回“作品是否达到可投标准”。

## 断档风险

- 只收藏资料，不建立投稿核对表。
- 作品质量没达标，却期待流程解决签约问题。
- 链接散落，临投找不到。
- 未记录投递时间和反馈结果。
- 把平台规则和故事结构混在一起，导致系统不可查。

## 训练任务

为当前作品做一张投稿核对表：平台、渠道、资料链接、正文进度、大纲状态、简介卖点、投递时间、反馈处理。
