# 金手指-背景空子适配

依据案例：`batch-004` 案例 019。

## 读者承诺

金手指必须让主角在当前范围内特别，并能通过世界背景里的空子持续获得收益、解决矛盾。

## 核心循环

```text
确定特别赛道 -> 设计背景空子 -> 金手指钻空子 -> 获得收益 -> 解决矛盾 -> 特别感扩大
```

## 开篇必备

- 主角在哪个赛道与众不同。
- 背景中有什么空子适合金手指发挥。
- 金手指第一次使用能带来明确收益。
- 收益能解决当前冲突。

## 期待来源

- 读者想看主角如何利用空子。
- 读者想看别人发现主角特别。
- 读者想看金手指收益如何滚大。

## 断档风险

- 同赛道有人比主角更特别。
- 世界设定压死金手指。
- 金手指能用但没收益。
- 收益和当前矛盾无关。

## 训练任务

给自己的金手指写 3 个世界常见矛盾，并证明金手指收益能自然解决它们。
