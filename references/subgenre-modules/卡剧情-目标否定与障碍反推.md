# 卡剧情-目标否定与障碍反推

依据案例：`batch-409` 案例 2043

## 读者承诺

卡剧情时先写清可判定的卷目标，再断言主角目前做不到；从外部、信息、关系、资源和内心五类障碍反推行动、局部失败、新信息与更高代价，生成选择链。

## 核心循环

```text
写成可判定目标句 -> 否定当前可达性 -> 选择互相强化的障碍 -> 找到既有因果来源 -> 主角合理尝试 -> 局部结果暴露新信息 -> 代价升级 -> 两难选择迫使策略或人物变化
```

## 开篇必备

- 包含行动、结果或期限的卷目标。
- 明确的失败损失与共情点。
- 二至三项互相强化的障碍。
- 每项障碍的既有来源和可见迹象。
- 合理尝试留下的局部收获。
- 最终要求人物付代价的不可逆选择。

## 期待来源

读者知道主角必须完成什么，又逐层看见旧方案为何不够；每次失败都带来新信息并提高代价，直到外部任务与内心弱点在一次选择中汇合。

## 断档风险

- 目标无法判定是否完成。
- 横向堆积十个不相关麻烦。
- 随机事故只延迟篇幅。
- 主角失败后没有任何状态变化。
- 临时新增规则制造困难。
- 障碍不触碰人物价值和选择。

## 训练任务

为卡住的一卷写目标句和五个“做不到”原因，只保留互相强化的三项；每项写一场合理尝试、局部结果、新信息和升级代价，第三场必须迫使不可逆选择。
