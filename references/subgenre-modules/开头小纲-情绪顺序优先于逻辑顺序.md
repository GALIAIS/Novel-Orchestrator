# 开头小纲-情绪顺序优先于逻辑顺序

依据案例：`batch-187` 案例 933。

## 读者承诺

开头小纲不能只按事件发生顺序铺排，要按读者情绪上下行重排，让危机、反击、救人和异常发现形成可追读波形。

## 核心循环

```text
列事件 -> 标注上行/下行 -> 检查连续压抑 -> 提前能力感 -> 保留章节末悬念
```

## 开篇必备

- 第一章下行后有上行补偿。
- 主角能力感尽早出现。
- 重伤、逃生、获得道具都有可信因果。
- 伏笔尽量通过动作和台词呈现。

## 期待来源

读者先感到压力，再看到主角能处理压力，随后遇到更大的异常，从而愿意继续追。

## 断档风险

- 开局长期下行。
- 只按逻辑顺序写，情绪曲线太平。
- 危机因果不可信。
- 金手指或道具出现过早，消耗成长期待。

## 训练任务

把第一章拆成 6 个情绪点，标注上行、下行或平；如果连续两个下行没有补偿，就调整事件顺序或提前主角反击。
