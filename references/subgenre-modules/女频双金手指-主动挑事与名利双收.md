# 女频双金手指-主动挑事与名利双收

依据案例：`batch-214` 案例 1066。

## 读者承诺

女主双金手指可以成立，主动挑事也可以成立，关键是让她算计别人后名利双收。

## 核心循环

```text
前世大佬 -> 情绪系统 -> 主动挑事 -> 算计破局 -> 名利双收
```

## 开篇必备

- 两个金手指功能分工清楚。
- 主动挑事符合女主人设。
- 过程有布局、语言交锋和资源转化。
- 结果兑现名利双收。

## 期待来源

读者看强势女主主动布局、操控局面，并从冲突中拿到实际收益。

## 断档风险

- 双金手指重复解决同一问题。
- 主角太强导致过程无聊。
- 配角夸张但没有功能。
- 台词只发泄，不体现人设和推进剧情。

## 训练任务

写 5 场女主主动挑事小纲，每场标注使用哪个金手指、配角反应、读者爽点和名利双收结果。
