# 青少年文风-平台样张诊断

依据案例：`batch-046` 案例 228。

## 读者承诺

文风适合哪个网站，要拿真实样张判断。青少年向文本可以轻快、任性、二次元，但仍要补足情节设置和情绪调动。

## 核心循环

```text
文本样张 -> 文风识别 -> 受众年龄 -> 平台候选 -> 剧情短板训练
```

## 开篇必备

- 有 2000 字左右样张。
- 判断目标读者年龄层。
- 冲突服务受众口味。
- 开头目标清楚。

## 期待来源

- 读者认同主角语气。
- 同人/IP 提供加成。
- 平台受众和文本气质吻合。
- 冲突升级合理。

## 断档风险

- 文笔够用但剧情弱。
- 开头目标散。
- 冲突升级过快导致尴尬。
- 任性人设不符合平台受众。

## 训练任务

拿两个开头样张做平台诊断：写出文风、受众年龄、平台候选、冲突问题和下一步训练项。
