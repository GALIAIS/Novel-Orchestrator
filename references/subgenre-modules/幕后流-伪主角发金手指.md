# 幕后流-伪主角发金手指

依据案例：`batch-142` 案例 707。

## 读者承诺

幕后主角作为金手指源头，把能力、信息或任务发给群像伪主角；读者既看伪主角装逼，也看幕后主角位格被反向抬高。

## 核心循环

```text
幕后主角布局 -> 伪主角获得金手指 -> 伪主角进入爽文局 -> 外界震惊 -> 幕后主角收束结果
```

## 开篇必备

- 幕后主角掌握的资源。
- 第一个伪主角的困境。
- 发放的金手指或任务。
- 伪主角装逼场景。
- 回收到幕后主角的结果。

## 期待来源

- 伪主角像传统主角一样爽。
- 读者知道真正源头在幕后。
- 上帝视角制造掌控感。
- 多棋子铺开群像。

## 断档风险

- 配角不像伪主角，只是工具人。
- 幕后主角长期没有回收和证明。
- 只铺群像，不抬高主角位格。
- 伪主角欲望和压力不足。

## 训练任务

设计 3 个伪主角，每人一个困境、一个幕后主角发放的金手指、一场装逼。最后写 500 字幕后回收，证明三场装逼都是布局。
