# 国术直播流-强者日常曝光

依据案例：`batch-045` 案例 225。

## 读者承诺

国术直播流可以迁移“隐世强者日常被直播”的结构。核心爽点是现代观众误判日常，再被国术能力不断震惊。

## 核心循环

```text
直播日常 -> 观众误判 -> 能力外露 -> 弹幕震惊 -> 身份挖掘
```

## 开篇必备

- 有隐世强者。
- 有连接直播的后代/主播。
- 第一场误判清楚。
- 国术能力事件化展示。

## 期待来源

- 现代视角不理解传统能力。
- 弹幕承担见证体系。
- 强者日常不断被放大。
- 身份和传承持续被挖。

## 断档风险

- 只讲拳理，不给事件。
- 直播反应太水。
- 主角能力无法验证。
- 只换皮，不迁结构。

## 训练任务

写一个国术直播开篇：误判、能力验证、弹幕震惊、身份钩子四项必须齐全。
