# 拆书感悟-接地气功锅剧情驱动

依据案例：`batch-341` 案例 1701

## 读者承诺

拆书要拆出接地气细节、功锅分配和剧情对话驱动，避免设定悬浮、主角背锅和无意义打斗。

## 核心循环

```text
夸张设定 -> 接角色背景 -> 配角承担风险动作 -> 对话剧情推进 -> 收益反馈兑现
```

## 开篇必备

- 设定夸张也要从人物处境出发。
- 主角拿功，锅交给配角或外部压力。
- 道德风险动作不要让主角主动提出。
- 打斗前后要有剧情意义。
- 高潮后要写收益和观众反馈。

## 期待来源

读者期待在陌生设定里找到熟悉的人性逻辑，并看到主角合理获得收益和评价变化。

## 断档风险

- 假大空、不接地气。
- 主角主动提出违背公序良德的方案。
- 打斗无意义堆字数。
- 设定解释挤占剧情。
- 高潮后没有反馈。

## 训练任务

拆一个大神桥段，标出接地气细节、谁拿功、谁背锅、哪句对话推进期待、高潮后谁给反馈。
