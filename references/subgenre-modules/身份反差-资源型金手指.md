# 身份反差-资源型金手指

依据案例：`batch-112` 案例 560。

## 读者承诺

金手指不一定是系统或超能力。身份、财富、地位、资源和权限的突变，也能形成金手指，核心是让主角获得新的行动可能。

## 核心循环

```text
原身份困境 -> 穿越高位身份 -> 资源权限暴增 -> 解决新危机 -> 持续利用反差
```

## 开篇必备

- 穿越前困境清楚。
- 穿越后身份足够反差。
- 新资源能立刻产生爽点。
- 新身份也有麻烦。
- 第一章展示身份带来的行动权限。

## 期待来源

- 负债者变世界首富。
- 读者期待财富和权力怎么用。
- 旧困境被新身份反转。
- 新身份隐藏危机打开长线。

## 断档风险

- 只说变富，不写权限。
- 没有新危机，剧情停住。
- 反差只爽一次。
- 身份资源和主线无关。

## 训练任务

设计一个身份型金手指：写清穿越前困境、穿越后资源、第一件爽事和第一个身份危机。
