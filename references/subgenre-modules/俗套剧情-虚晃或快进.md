# 俗套剧情-虚晃或快进

依据案例：`batch-098` 案例 490。

## 读者承诺

读者一眼能猜到结尾的俗套桥段，要么用伏笔支撑非常规反转，要么快进略过，把篇幅留给新期待。拖着写常规打脸，只会显得无聊。

## 核心循环

```text
识别俗套桥段 -> 判断读者是否已猜到 -> 虚晃反转/快进略过 -> 接新期待 -> 继续上行
```

## 开篇必备

- 先判断读者会不会看开头知结尾。
- 反转必须提前埋伏笔。
- 快进要短，不拖过程。
- 快进后必须给新目标或新悬念。

## 期待来源

- 虚晃一枪打破预期。
- 俗套快速兑现不浪费耐心。
- 新期待接住读者。
- 阻碍和转折让老套路重新有波动。

## 断档风险

- 临时花活，没有伏笔。
- 伏笔太深，读者看不懂。
- 快进后没有新期待。
- 炮灰、围观、主角配置太熟，情绪没起来。

## 训练任务

把一个打脸桥段写成两个版本：虚晃版提前列 2 个伏笔，快进版 300 字内完成打脸并抛出新目标。
