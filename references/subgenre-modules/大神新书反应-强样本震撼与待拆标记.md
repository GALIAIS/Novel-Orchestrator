# 大神新书反应-强样本震撼与待拆标记

依据案例：`batch-253` 案例 1264。

## 读者承诺

大神新书反应模块把强烈阅读震撼记录为待拆样本，而不是停留在感叹。

## 核心循环

```text
强烈反应 -> 标记样本 -> 找触发段落 -> 拆前后铺垫兑现 -> 转成可学写法
```

## 开篇必备

- 记录第一处强反应。
- 回到原文定位段落。
- 只拆触发点前后文本。
- 反推期待和情绪来源。

## 期待来源

即时震撼能帮助拆书者定位真正击中自己的强点，再从强点反推结构。

## 断档风险

- 只说大神牛，不拆证据。
- 事后合理化第一反应。
- 没有原文段落，结论空泛。
- 拆太大范围，找不到触发点。

## 训练任务

读一个强开篇，标记第一处震撼段落，只拆前后 1000 字，写出触发期待的具体句段。
