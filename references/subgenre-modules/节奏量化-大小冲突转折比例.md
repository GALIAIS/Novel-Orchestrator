# 节奏量化-大小冲突转折比例

依据案例：batch-362 案例 1808

## 读者承诺

用大小冲突和大小转折的数量比例研究节奏，让拆书从“感觉快慢”变成可标注的变化密度。

## 核心循环

```text
标大冲突 -> 标小冲突 -> 标大转折 -> 标小转折 -> 判断节奏比例
```

## 开篇必备

- 区分阶段推进和局部抓手。
- 记录每次变化。
- 看平均多少字一次转折。
- 比较同类作品。

## 期待来源

- 小冲突保持注意。
- 大冲突推动阶段。
- 小转折制造场景变化。
- 大转折改变局势。

## 断档风险

- 全是大冲突，节奏太赶。
- 全是小冲突，故事太碎。
- 没有转折，阅读变平。
- 只复述剧情，不标变化点。

## 训练任务

拆 3 章榜单文，标大冲突、小冲突、大转折、小转折，并比较比例。
