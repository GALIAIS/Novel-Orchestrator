# 双核心-金手指外核与情感内核

依据案例：batch-386 案例 1926

## 读者承诺

金手指和目标提供可持续的外在事件循环，情感与价值提供人物选择标准。二者应在行动、代价和后果中反复汇合。

## 核心循环

```text
能力驱动事件 -> 价值决定选择 -> 选择产生代价 -> 后果深化能力意义 -> 双核心升级
```

## 开篇必备

- 清楚的金手指规则。
- 主角外在目标。
- 一项可被选择检验的情感价值。
- 能力使用带来的第一次代价。

## 期待来源

- 能力持续升级和组合。
- 同一能力引出不同价值选择。
- 爽点附带关系与情感重量。
- 人物成长改变能力用法。

## 断档风险

- 所有事件只为展示能力。
- 内核只靠旁白说明。
- 主题表达变成角色说教。
- 新人同时控制过多抽象层次。

## 训练任务

为同一金手指写三次使用，逐步加入利益与关系取舍，第三次必须为内核付出不可逆代价。
