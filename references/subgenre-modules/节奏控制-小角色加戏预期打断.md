# 节奏控制-小角色加戏预期打断

依据案例：`batch-193` 案例 965。

## 读者承诺

小角色文戏必须嵌入主角当前预期链，先回答读者正在等待的问题，再承担人物塑造，否则容易被追更读者感知为临时加戏。

## 核心循环

```text
主角形成疑问 -> 读者等待证据 -> 小角色提供关键确认 -> 文戏解释动机 -> 主线期待继续推进
```

## 开篇必备

- 先明确主角当前疑问。
- 小角色出场要带来主线信息。
- 文戏放在读者需要答案之后。
- 章末回到主角行动或反派威胁。

## 期待来源

读者看到小角色戏份时，会认为它是在回答主线问题、强化反派威胁或推进主角行动，而不是作者突然展示表达欲。

## 断档风险

- 主角还没怀疑，先切小角色支线。
- 文戏漂亮但不服务当前问题。
- 临时加戏打断追更读者章内期待。
- 养书党能接受，追更党却掉订。

## 训练任务

把一段小角色投诚戏放入主角视角链：主角先怀疑，获得线索，再看到小角色动机，最后立刻做出下一步行动。
