# 聊天群文-主角连接群像

依据案例：`batch-038` 案例 186。

## 读者承诺

聊天群文像群像剧，主角的核心功能是把不同人物联系起来，让他们的认知差、身份差和时代差发生事件。

## 核心循环

```text
群成员差异 -> 主角连接 -> 群像碰撞 -> 现实收益 -> 新成员/新议题
```

## 开篇必备

- 主角不是旁观者。
- 群成员差异清楚。
- 群聊影响现实主线。
- 主角从连接中获得收益。

## 期待来源

- 名人互相反应。
- 历史认知差制造笑点或震惊。
- 主角调度资源。
- 新人物带来新冲突。

## 断档风险

- 群聊变成段子合集。
- 主角没有行动和收益。
- 人物太多焦点散。
- 只玩梗不推进事件。

## 训练任务

设计一个聊天群第一单元：3 个群成员、1 个议题、1 次群像碰撞、1 个主角现实收益。
