# 悬疑开头-危险实证与新人节奏

依据案例：batch-367 案例 1832

## 读者承诺

悬疑开头要给禁忌、危险实证、非去不可的目标和冒险奖励；新人不能照搬大神老书的慢热铺垫。

## 核心循环

```text
禁忌 -> 危险实证 -> 必须冒险 -> 正面冲突 -> 金手指解决 -> 收获
```

## 开篇必备

- 不能做的事。
- 工具人或事件证明危险。
- 主角非做不可的理由。
- 冒险后的明确奖励。

## 期待来源

- 禁忌带来的未知。
- 死亡或损失证明危险。
- 主角仍然要去。
- 危险兑现后得到收益。

## 断档风险

- 只有悬念，没有实证。
- 只有危险，没有目标。
- 大神笔名能撑慢热，新人撑不住。
- 拆老书不结合当下节奏。

## 训练任务

写一个“不能进入某地”的前三章大纲，必须包含危险实证、非去不可、正面冲突和奖励。
