# 多修炼体系-核心力量只能一个

依据案例：`batch-333` 案例 1662

## 读者承诺

多个修炼体系可以写，但底层核心力量只能有一个，并且要逐步引出。

## 核心循环

```text
统一核心力量 -> 不同修行路径 -> 人设/阶层/资源差异 -> 体系碰撞 -> 主角选择
```

## 开篇必备

- 核心力量统一。
- 第一卷逐步引出。
- 每个体系有人设气质。
- 体系差异带来冲突。
- 阶级划分有逻辑。

## 期待来源

读者期待不同体系代表不同人物道路，并在冲突中体现优劣。

## 断档风险

- 多体系一股脑倒出来。
- 核心力量不统一。
- 体系只是名词不同。
- 阶级和体系关系有漏洞。
- 每次换开头就推倒世界观。

## 训练任务

做修炼体系表：核心力量、使用方式、代表人群、资源门槛、优劣势，删掉不能制造冲突的体系。
