# 敏感金手指-审核风险转译

依据案例：`batch-050` 案例 247。

## 读者承诺

保留金手指最刺激的能力展示，但把高风险、导向不良或现实敏感的用法，转译成黑吃黑、公益合作、沙雕变现等更稳表达。

## 核心循环

```text
极端能力 -> 极端用法 -> 审核风险 -> 正向转译 -> 能力仍然被凸显
```

## 开篇必备

- 先识别最可能触审的表达。
- 区分“能力爽点”和“危险包装”。
- 主角立场尽量正向。
- 替代方案仍要凸显金手指。
- 给转译后的收益安排后续用途。

## 期待来源

- 极端能力天然带来好奇。
- 黑吃黑能提供解气爽。
- 公益合作能提供人脉和资源。
- 沙雕变现能降低沉重感。

## 断档风险

- 只追求刺激，容易踩审核雷。
- 转译后不凸显能力，会丢掉卖点。
- 主角导向不正，会让读者和平台都不舒服。
- 现实敏感联想太强，最好直接绕开。

## 训练任务

选择一个高风险金手指用法，写出风险点、能力爽点、3 个替代方案，并判断哪个最适合前三章。
