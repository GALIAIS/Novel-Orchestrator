# 学习社群-保底炫耀与现实打脸

依据案例：`batch-324` 案例 1620

## 读者承诺

学习社群里，保底、入行和口头自信不能替代作品质量；真正站得住的是文本、读者反馈和可验证成绩。

## 核心循环

```text
中肯意见 -> 被炫耀/质疑反驳 -> 真实反馈形成反差 -> 证据回收 -> 自我反省
```

## 开篇必备

- 明确社群目标是学习，不是炫耀。
- 把保底和成绩放回真实含金量里判断。
- 记录作品真实反馈，如切书、无人看、收入水平。
- 批评别人前先自查文本能力。
- 将争执转成可执行反省清单。

## 期待来源

围观者期待看到虚高姿态被真实证据校正；作者本人更应期待从羞耻点中找到改进方向。

## 断档风险

- 用低价保底当入行资本。
- 作品反馈很差却嘲讽别人。
- 把社群学习变成身份展示。
- 只享受现实打脸情绪，不做自查。

## 训练任务

写一个现实打脸桥段：中肯意见、嘲讽反驳、证据回收、自查规则四步齐全。
