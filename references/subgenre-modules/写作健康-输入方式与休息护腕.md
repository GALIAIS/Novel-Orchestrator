# 写作健康-输入方式与休息护腕

依据案例：`batch-330` 案例 1649

## 读者承诺

高强度码字要保护手部，输入设备、休息、护腕和替代输入都属于长期产能管理。

## 核心循环

```text
持续码字 -> 手部负担 -> 疼痛预警 -> 工具/休息/输入方式调整 -> 稳定产能
```

## 开篇必备

- 记录疼痛位置。
- 控制连续输入时间。
- 准备更舒适的键盘或护腕。
- 语音输入作为备选。
- 疼痛加重时降低字数目标。

## 期待来源

作者能长期稳定更新，而不是靠一次极限爆发换来数月停摆。

## 断档风险

- 腱鞘炎继续硬写。
- 只换键盘，不休息活动。
- 单日字数冲刺没有恢复计划。
- 语音输入不适合自己还硬用。
- 疼痛出现后没有停写规则。

## 训练任务

做 7 天记录：字数、连续输入时间、疼痛位置、休息次数、输入方式；据此设定单日字数上限。
