# 恐怖搞笑-风格反差筛读者

依据案例：batch-310 案例 1549

## 读者承诺

严肃恐怖外壳可以承载搞笑反差，但必须让世界仍危险、主角仍在求生，不能把风格切成两本书。

## 核心循环

```text
恐怖世界压迫 -> 普通人求生 -> 主角反向吓到怪物 -> 搞笑反差 -> 生存升级继续
```

## 开篇必备

- 先立住恐怖世界和生存压力。
- 主角要聪明坚韧，不是纯玩闹。
- 反差笑点来自危险被反向破解。
- 风格转向要保留世界威胁。

## 期待来源

期待来自“危险里搞笑”。读者既担心主角死，又期待他用反常规操作反吓怪物。

## 断档风险

- 前期严肃，后期彻底变无厘头。
- 搞笑破坏恐怖设定可信度。
- 只写反差，不写求生和升级。
- 风格混合筛掉前期留下的读者。

## 训练任务

写一个恐怖搞笑开篇：前段严肃求生，后段主角无意反吓怪物，但结尾保留更大危险。
