# 少年感拆书-热血单类型深挖

依据案例：`batch-343` 案例 1713

## 读者承诺

新人拆少年感应先深挖同类型样本，提炼不服输、热血、伙伴和爆发结构，再谨慎跨媒介参考。

## 核心循环

```text
目标少年感 -> 同类型样本 -> 受压不服输 -> 伙伴/挫败 -> 爆发兑现 -> 跨媒介补套路
```

## 开篇必备

- 明确要写的少年感类型。
- 先盯同类型死薅。
- 拆情绪结构，不抄桥段。
- 动漫长线可参考。
- 短期桥段要适配网文章节。

## 期待来源

读者期待少年主角弱而不认命，遍体鳞伤仍坚持，最终在伙伴或目标牵引下爆发。

## 断档风险

- 新人贪多胡学。
- 跨类型样本看完不会用。
- 直接照搬动漫节奏。
- 只学热血台词，不学受压和代价。
- 少年感变中二空喊。

## 训练任务

拆 3 个少年感样本，各写受辱点、不服输动作、伙伴反馈、挫败代价、第一次爆发。
