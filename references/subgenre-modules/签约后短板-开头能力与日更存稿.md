# 签约后短板-开头能力与日更存稿

依据案例：`batch-236` 案例 1179。

## 读者承诺

开头被编辑认可只是入场券；长线节奏、日更 4000 和存稿才决定能否持续走下去。

## 核心循环

```text
开头可签 -> 暴露长线短板 -> 补节奏 -> 练日更 -> 存稿开书
```

## 开篇必备

- 识别开头优势。
- 复盘上本切书原因。
- 训练剧情节奏。
- 开书前先存稿。

## 期待来源

编辑认可开头后，作者能用稳定更新和长线大纲把签约机会转成上架机会。

## 断档风险

- 误以为会写开头就能写完整本。
- 日更 4000 不稳定。
- 没存稿直接开书。
- 长线节奏没补，20 万字后再崩。

## 训练任务

开书前存 3-5 万字，列前 20 万字 4 个大事件，并连续 7 天日写 4000 字达标。
