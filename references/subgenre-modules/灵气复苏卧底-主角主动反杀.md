# 灵气复苏卧底-主角主动反杀

依据案例：batch-266 案例 1326

## 读者承诺

在灵气复苏、刑侦、异能、都市悬疑等框架里使用身边卧底时，让读者享受“威胁贴身但主角更早察觉”的安全感和反杀爽点。

## 核心循环

```text
卧底贴身 -> 异常露出 -> 主角察觉 -> 暗中准备 -> 卧底出手 -> 主角反杀
```

## 开篇必备

- 卧底接近主角必须有明确职务或任务理由。
- 主角要在其他人之前发现一处异常。
- 伏笔不能太隐蔽，读者回看要能成立。
- 小高潮要证明主角不是被动撞运气。

## 期待来源

期待来自贴身危机与主角主动性的并存：读者知道主角身边有人不干净，但更期待主角什么时候掀桌。

## 断档风险

- 主角长期无察觉，会显得迟钝。
- 伏笔太少，小高潮反转像硬编。
- 卧底只制造麻烦，不服务主角高光。
- 诬陷和生死危机没有组织逻辑。

## 训练任务

设计一个卧底桥段，先写 3 个异常细节，再写主角暗中准备的 2 个动作，最后写小高潮反杀方式。
