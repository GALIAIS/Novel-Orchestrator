# 起点内投-责编月初与无回复止损

依据案例：batch-260 案例 1298

## 读者承诺

内投选择要服务稿件，而不是迷信编辑玄学。优先找题材匹配的责编，月初投，等待明确期限；无回复就止损，不把焦虑耗在猜测上。

## 核心循环

```text
确认题材 -> 选择责编 -> 月初投递 -> 等待反馈 -> 有反馈修改/无反馈止损
```

## 开篇必备

- 题材和卖点。
- 目标责编或编辑组。
- 投递时间。
- 等待期限。
- 无回复后的处理方案。

## 期待来源

作者期待通过清晰投递流程减少焦虑，把注意力放回稿件质量和下一稿，而不是无限比较通过率表。

## 断档风险

历史通过率表可能过期；主编要求更高且不回概率更大；无回复不是可执行反馈，长期等待会拖慢创作。

## 训练任务

写一张内投决策卡：题材、编辑、投递日期、等待到哪天、收到反馈怎么改、没回复写什么新稿。
