# 游戏剧本参考-RPG任务与AVG分支

依据案例：`batch-246` 案例 1229。

## 读者承诺

游戏剧本资料能帮助网文作者理解任务触发、分支选择、事件奖励和熟悉感。

## 核心循环

```text
任务触发 -> 角色选择 -> 阻碍升级 -> 奖励反馈 -> 章末钩子
```

## 开篇必备

- 借游戏结构，不搬说明书语气。
- 让选择影响事件。
- 给失败代价。
- 把奖励变成读者期待。

## 期待来源

读者熟悉 RPG/AVG 的任务和分支感，会自然期待下一步选择与奖励。

## 断档风险

- 只写设定说明。
- 分支不影响剧情。
- 任务像菜单，不像事件。
- 游戏术语压过人物压力。

## 训练任务

把一个 RPG 任务改成网文章节，写出触发、选择、阻碍、奖励和章末钩子。
