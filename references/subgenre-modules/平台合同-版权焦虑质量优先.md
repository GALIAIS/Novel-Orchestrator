# 平台合同-版权焦虑质量优先

依据案例：batch-277 案例 1381

## 读者承诺

解决新人投稿前的合同与平台焦虑：先判断作品是否已经具备读者价值，再评估平台读者规模、反馈密度和版权条款，而不是把“平台是否剥削”当成拖延写作的主问题。

## 核心循环

```text
平台收入样本 -> 合同/版权担心 -> 读者反馈环境评估 -> 当前文本短板定位 -> 小项目试水/投稿决策
```

## 开篇必备

- 明确作品最适合的读者群。
- 确认平台是否有同题材读者。
- 找到可验证合同来源，而不是听二手传言。
- 把开篇、节奏、卖点和更新能力列为优先项。

## 期待来源

作者的安全感来自“我知道这本书为什么能被读者看下去”，而不是来自抽象平台选择。平台只提供场域，留存仍由文本承担。

## 断档风险

- 用版权焦虑逃避文本竞争。
- 看到稿费截图就误判平台普遍收益。
- 为避开大站内卷而进入低反馈单机场。
- 只比较合同，不比较同题材读者密度。
- 把未来版权收益放在当前签约能力之前。

## 训练任务

列出 3 个候选平台，为每个平台写 100 字判断：读者匹配度、反馈密度、合同可接受程度、当前作品胜算。最后只选一个平台做 3 万字试水计划。
