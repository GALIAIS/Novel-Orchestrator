# 女性角色-欣赏不舔狗智商不降

依据案例：`batch-110` 案例 546。

## 读者承诺

男频主角可以欣赏女性、喜欢女性，但不能因为女性登场就智商下降、行动失控或莫名帮忙。读者要看到主角保持目标、利益和判断力。

## 核心循环

```text
女性登场 -> 主角欣赏 -> 理性判断 -> 按目标行动 -> 关系后续推进
```

## 开篇必备

- 女性角色有明确功能。
- 主角第一反应不能舔。
- 帮忙必须有利益、因果或目标理由。
- 主角表面稳得住。
- 关系推进靠对手戏或共同事件。

## 期待来源

- 主角仍然强势。
- 女性角色不拖低剧情智商。
- 读者接受“欣赏美”但拒绝“跪舔美”。
- 强对手戏带来关系张力。

## 断档风险

- 一见美女就脸红、看呆。
- 莫名其妙为了美女牺牲利益。
- 假正经写成真舔狗。
- 不会写女主却硬写大女主。

## 训练任务

写一个女性角色初登场片段：主角可以称赞她漂亮，但下一句必须回到任务、利益或风险判断。
