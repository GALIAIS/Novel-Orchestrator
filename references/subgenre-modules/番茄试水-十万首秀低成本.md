# 番茄试水-十万首秀低成本

依据案例：`batch-342` 案例 1710

## 读者承诺

番茄十万字首秀能降低新人试错成本，但平台选择仍要看文风适配、反馈节点和长期收益模式。

## 核心循环

```text
新人入行 -> 选早反馈平台 -> 十万字首秀 -> 数据判断切续 -> 校准文风和下一本
```

## 开篇必备

- 明确自己的文风。
- 知道平台核心模式。
- 设定首个数据节点。
- 预先写切书阈值。
- 不把收入图当个人预期。

## 期待来源

作者期待低成本知道自己能不能留住读者，平台数据能提供比空想更早的反馈。

## 断档风险

- 只看收入榜。
- 不看文风适配。
- 首秀后没有切续标准。
- 把免费模式收益想得太确定。
- 忽略长期职业规划。

## 训练任务

给同一题材写飞卢、起点、番茄三版前十万字策略，分别标出爽点频率、故事占比、关键数据节点和切书阈值。
