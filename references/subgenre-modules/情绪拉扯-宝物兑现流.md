# 情绪拉扯-宝物兑现流

依据案例：`batch-002` 案例 008。

## 读者承诺

让主角新获得的宝物、技能、身份或能力不只是“设定很强”，而是在兑现前不断加杠杆，让读者主动期待它如何装大。

## 核心循环

```text
展示功能 -> 误判鸡肋 -> 适配强敌 -> 他人失败 -> 主角看穿 -> 众人不看好 -> 主角兑现 -> 震惊收获
```

## 开篇必备

- 宝物/能力的明确功能。
- 至少一个信息不全的误判者。
- 一个正好能被该能力克制的难题。
- 一个见证主角兑现的人群或权威。

## 期待来源

- 信息差：读者比配角更懂宝物价值。
- 对比差：别人用更强外壳失败，主角用“鸡肋”成功。
- 反应差：围观者越不看好，兑现后越爽。
- 收获差：宝物从鸡肋变神器，主角获得资源反馈。

## 断档风险

- 只说宝物强，不给使用场景。
- 没有误判者和见证者。
- 反派不适配，导致宝物像强行派上用场。
- 兑现太晚，中间没有再次提醒宝物价值。

## 训练任务

写一个 8 拍章纲：功能、误判、反派、失败者、主角方案、围观不信、兑现、收获。每一拍写一句具体事件。
