# 电影拉片-盗梦第一幕表格

依据案例：batch-301 案例 1503

## 读者承诺

用表格拆电影第一幕，训练目标、阻力、规则释放和信息差的开篇组织能力。

## 核心循环

```text
观看第一幕 -> 表格列节点 -> 标目标/阻力/信息 -> 提炼场景功能 -> 改写原创开篇
```

## 开篇必备

- 每个节点只记录功能。
- 重点看规则如何边行动边释放。
- 标出情绪转折和信息差。
- 最后必须转写原创场景。

## 期待来源

- 作者学会让读者边看行动边懂设定。
- 第一幕目标清楚。
- 规则和危机同步推进。

## 断档风险

- 拉片变剧情复述。
- 表格有节点但无功能判断。
- 永远只拆电影，不写原创。

## 训练任务

拆 8 个第一幕节点，每个填目标、阻力、信息释放、情绪变化四栏。
