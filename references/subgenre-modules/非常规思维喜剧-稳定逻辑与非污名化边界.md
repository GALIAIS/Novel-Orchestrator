# 非常规思维喜剧-稳定逻辑与非污名化边界

依据案例：`batch-399` 案例 1991

## 读者承诺

不用真实精神疾病充当搞笑快捷标签，而以稳定的非常规优先级制造可回推的意外行动，让反差既有笑点也有任务、关系和成长后果。

## 核心循环

```text
常规预期 -> 非常规内部逻辑解读 -> 意外但一致的行动 -> 任务与关系后果 -> 同伴多样反馈 -> 人物学习边界后继续变体
```

## 开篇必备

- 写清角色核心欲望和异常优先级。
- 特殊判断有稳定触发条件。
- 角色保留能力、底线与主体性。
- 第一处笑点真实改变局面。
- 同伴反馈不只负责捧哏。
- 不使用未经研究的医学标签包装人设。

## 期待来源

读者能预测人物会用另一套逻辑处理问题，却无法预知它在新情境中会产生何种意外结果和关系成本。

## 断档风险

- 随机发疯代替稳定人设。
- 把疾病痛苦或失控当笑点。
- 用诊断暗示危险或无道德底线。
- 所有旁人都无条件纵容。
- 行动不产生现实后果。
- 人物每章重置、从不学习。

## 训练任务

不用诊断词写出角色的欲望、触发、优先级、底线、优势和成本；用同一逻辑完成三场不同场景喜剧，删除无法回推到规则的随机笑点。
