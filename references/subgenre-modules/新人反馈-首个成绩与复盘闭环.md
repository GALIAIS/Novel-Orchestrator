# 新人反馈-首个成绩与复盘闭环

依据案例：`batch-224` 案例 1116。

## 读者承诺

新人第一次获得可量化成绩时，要把兴奋转成复盘，确认哪些学习动作开始有效。

## 核心循环

```text
潜水学习 -> 实践发布 -> 指标出现 -> 社群确认 -> 复盘有效动作
```

## 开篇必备

- 记录指标类型，区分收藏、订阅、追读等。
- 写清作品题材、平台和时间。
- 只提炼 1-3 个可能有效动作。
- 给下一轮设置一个改进目标。

## 期待来源

成绩节点让作者确认自己不是只在空学，而是进入了学习、实践、反馈、复盘的循环。

## 断档风险

- 神化单一成绩数字。
- 报喜后不复盘。
- 混淆收藏和订阅等指标。
- 把一次正反馈误判为长期稳定能力。

## 训练任务

为自己的一个成绩节点写 300 字复盘：指标是什么、从哪里来、哪三件事可能有效、哪两件事仍没验证、下一轮只保留一个改进动作。
