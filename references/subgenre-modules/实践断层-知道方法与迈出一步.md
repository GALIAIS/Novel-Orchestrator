# 实践断层-知道方法与迈出一步

依据案例：`batch-246` 案例 1226。

## 读者承诺

写作方法只有被拿去写正文，才算真正开始转化；资料收集不能替代落笔。

## 核心循环

```text
想写好 -> 找资料 -> 待办膨胀 -> 迟迟不写 -> 作业逼出最小行动
```

## 开篇必备

- 明确当前写作问题。
- 限定资料范围。
- 定一个最小写作动作。
- 设置反馈和期限。

## 期待来源

作者从“我知道方法”切换到“我今天写了一段”，学习才进入可反馈状态。

## 断档风险

- 用准备替代写作。
- 资料越找越多。
- 追求即时反馈。
- 没有外部作业压力。

## 训练任务

选一个已知方法，今天只写 300 字应用它，禁止继续找资料。
