# 子流派细分模块

本目录用于把 8 个大题材继续拆成可操作的子流派/结构流派模块。

要求：

- 每个模块必须绑定人工精拆案例。
- 不用关键词脚本生成正文。
- 每个模块必须给出读者承诺、核心循环、开篇必备、期待来源、断档风险、训练任务。
