# 历史文-弥补缺憾代入感

依据案例：`batch-321` 案例 1602

## 读者承诺

历史文的专业知识服务可信度，真正驱动读者的是对历史缺憾的弥补、治愈和补偿感。

## 核心循环

```text
资料可信 -> 找历史缺憾 -> 主角介入改变 -> 弥补悲剧 -> 读者获得代入补偿
```

## 开篇必备

- 明确历史时期和读者熟悉的遗憾。
- 主角有可信介入位置。
- 改变不是乱整活。
- 资料服务故事，不堆史料。
- 情绪方向清楚，是爽、治愈还是热血。

## 期待来源

读者期待看到熟悉历史中的遗憾被主角以可信方式改写，获得“如果当时这样就好了”的补偿。

## 断档风险

- 只展示学历和史料，不给情绪。
- 电波系感情路线不适配目标读者。
- 新人误学个案整活。
- 历史严谨压过故事可读性。

## 训练任务

选一个历史遗憾，写清真实缺憾、主角介入点、改写方式和读者获得的补偿感。
