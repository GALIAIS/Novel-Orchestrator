# 断更追订-不可逆新书止损

依据案例：`batch-099` 案例 492。

## 读者承诺

长时间更新不稳会破坏追读习惯，追订崩盘后通常难以救回。不可抗力断更的现实策略是止损、养身体、准备新书和存稿。

## 核心循环

```text
生病断更 -> 追订下滑 -> 判断旧书难救 -> 拆书复盘 -> 新书存稿 -> 稳定更新
```

## 开篇必备

- 不把健康透支当努力。
- 发新书前留存稿缓冲。
- 有请假和病休预案。
- 旧书救不回时尽快复盘止损。

## 期待来源

- 稳定更新维持读者习惯。
- 存稿降低突发风险。
- 新书重启数据。
- 身体恢复保证长期创作。

## 断档风险

- 抱着旧书不放，错过新书窗口。
- 无存稿硬发，重复断更。
- 身体没恢复就继续透支。
- 把追订崩盘归咎于单个剧情问题。

## 训练任务

给下一本书制定发书前存稿计划：至少写清存稿字数、日更量、病休缓冲天数、断更通知方案。
