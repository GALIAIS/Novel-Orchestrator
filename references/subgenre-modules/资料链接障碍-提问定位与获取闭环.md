# 资料链接障碍-提问定位与获取闭环

依据案例：`batch-252` 案例 1259。

## 读者承诺

资料链接障碍模块把“链接打不开”转成可复现、可定位、可解决的求助信息。

## 核心循环

```text
资料打不开 -> 定位帖子和链接类型 -> 描述设备/错误 -> 他人复现 -> 获取替代路径
```

## 开篇必备

- 给出所在帖子或评论楼层。
- 说明链接类型。
- 描述设备和客户端。
- 提供错误截图或现象。

## 期待来源

新人能把资料获取障碍从模糊抱怨变成具体问题，学习链路不被入口问题卡死。

## 断档风险

- 只说打不开，不给链接位置。
- 混淆附件、图片、短链、外部网页。
- 不说明设备和登录状态。
- 没有记录备用路径。

## 训练任务

按“帖子、楼层、链接类型、设备、错误表现、截图”写一条链接求助，确认别人能复现。
