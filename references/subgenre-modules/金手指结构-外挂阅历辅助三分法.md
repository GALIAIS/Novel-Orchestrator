# 金手指结构-外挂阅历辅助三分法

依据案例：`batch-213` 案例 1063。

## 读者承诺

主角优势可拆成外挂、阅历、辅助三类，各自承担爽点、能动性和冲突丰富度。

## 核心循环

```text
外挂承载爽点 -> 阅历提供选择 -> 辅助丰富打法 -> 奖励伏笔续航
```

## 开篇必备

- 外挂优势要直观可感。
- 阅历优势要让主角主动。
- 辅助资源要服务前中期危机。
- 三类金手指分工清楚。

## 期待来源

读者既能看到强力外挂，也能看到主角自身判断和持续获得新资源。

## 断档风险

- 主角变成外挂工具人。
- 阅历优势没有参与关键选择。
- 辅助资源只是摆设。
- 奖励和伏笔不回收。

## 训练任务

给主角设计外挂、阅历、辅助三层优势，并写出它们在前三章、中期危机和大冲突后的用途。
