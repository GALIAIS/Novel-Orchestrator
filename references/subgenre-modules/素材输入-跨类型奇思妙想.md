# 素材输入-跨类型奇思妙想

依据案例：`batch-105` 案例 524。

## 读者承诺

奇思妙想来自网文之外的素材厚度。科幻、推理、民俗、志怪、奇幻、武侠、心理学都能提供网文新卖点。

## 核心循环

```text
外部阅读 -> 异常素材 -> 类型知识 -> 网文结构化 -> 新奇卖点 -> 题材方向
```

## 开篇必备

- 一个外部素材来源。
- 一个可迁移异常点。
- 一个适配的网文题材。
- 一个情绪或结构用途。

## 期待来源

- 新鲜知识。
- 陌生设定。
- 跨类型组合。
- 读者没见过的题材入口。

## 断档风险

- 只扫网文导致同质化。
- 外部素材只收藏不转化。
- 照搬实体书内容。
- 新奇点不能进入人物和事件。

## 训练任务

读一个非网文短篇或章节，摘一个异常素材，改成一个网文开篇事件。必须写清主角如何被这个素材影响。
