# 外部刊物-资料入口观察

依据案例：`batch-119` 案例 592。

## 读者承诺

外部刊物、评论集和行业资料的价值不在收藏，而在把观点转化为创作问题、样本验证和训练动作。资料入口必须回到作品正文，否则只会增加信息负担。

## 核心循环

```text
发现资料 -> 判断类型 -> 提取一个观点 -> 找作品验证 -> 转成训练任务 -> 回到写作复盘
```

## 开篇必备

- 先标注资料类型：理论、评论、访谈、行业观察或样本集。
- 只取当前最相关的一个问题。
- 必须绑定至少一个作品正文样本。
- 结论要能转成改稿或仿写动作。

## 期待来源

- 作者期待找到更高质量资料源。
- 外部观点能补足盲区。
- 作品验证让理论变得可操作。
- 训练任务把阅读投入转回产出。

## 断档风险

- 只囤资料不验证。
- 用宏观评论替代正文拆解。
- 同时吸收太多观点，训练目标失焦。
- 找不到当前作品问题的对应证据。

## 训练任务

选一份网文评论或行业资料，摘一个观点，用一本榜单作品验证。最后写出一个“我下一章能怎么用”的动作，不超过 100 字。
