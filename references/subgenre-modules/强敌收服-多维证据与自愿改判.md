# 强敌收服-多维证据与自愿改判

依据案例：`batch-415` 案例 2073

## 读者承诺

强力候选属下拥有基于经验、利益和价值观的合理不服，不会因反复受辱突然忠诚。主角在专业、勇气、资源、责任和底线等不同冲突中提供累积证据，候选人也能反驳、获胜、救场并迫使主角修正；高潮揭开其毕生追求的错误前提，却保留有效核心，最终由他在多个真实选项中付出代价、自愿改判。

## 核心循环

```text
建立候选人合理不服的信念模型 -> 不同事件分别检验专业、勇气、资源、责任和价值 -> 候选人提出反证并保留优势 -> 双方相互贡献和修正 -> 多场结果持续更新同一信念变量 -> 价值危机拆开错误前提与有效核心 -> 主角提供可验证新路径 -> 候选人在三种可行选择中付费站队 -> 入队后以独立能力改变主线
```

证据累积替代羞辱驯化，对等贡献保护强敌价值，选择成本证明忠诚，独立任务兑现得力属下而非赞叹工具。

## 开篇必备

- 候选人的专业、欲望、底线和过去成功经验。
- 三条足以支持其不服主角的合理证据。
- 一个跨场持续更新的核心信念变量。
- 五场不同直接原因和能力维度的验证。
- 至少一次候选人获胜、一次救场、一次有效反证。
- 主角因对方证据修改计划的节点。
- 毕生追求中的错误前提与有效核心。
- 旧路、中立和站队三个真实可行选项。
- 站队会失去的利益、身份或关系。
- 入队后的职责、否决权和首次独立任务。

## 期待来源

- 候选人的不服是否会被事实动摇。
- 哪个维度仍然是他胜过主角的领域。
- 主角会否承认并修正自己的错误。
- 毕生追求究竟错在目标还是前提。
- 候选人是否愿意为新判断背叛旧利益。
- 入队后双方原则冲突会如何继续。
- 第一次独立任务能否证明收服没有削弱人物。

## 断档风险

- 每场都是围观者嘲讽后主角碾压。
- 不服只来自候选人愚蠢或嫉妒。
- 多次打脸没有更新同一信念变量。
- 候选人从不胜利，也不向主角提供价值。
- 高潮把其一生全部写成笑话。
- 主角只给更宏大口号，没有可验证新路径。
- 站队没有替代选项和实际成本。
- 入队后失去专业、原则和独立行动。
- 把装逼或计划意外变成所有主线的唯一算法。

## 训练任务

设计一名强敌候选属下和五场多维验证。至少安排他胜过主角一次、救场一次、用反证迫使主角改计划一次；高潮只能否定毕生追求的错误前提。让他在旧路、中立和站队中付出代价选择，随后写一场不依赖主角的独立任务。
