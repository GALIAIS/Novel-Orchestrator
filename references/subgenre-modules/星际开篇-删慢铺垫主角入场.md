# 星际开篇-删慢铺垫主角入场

依据案例：batch-293 案例 1464

## 读者承诺

新人星际开篇要用主角视角快速进入压力和选择，少用实体小说式慢铺垫。

## 核心循环

```text
删背景 -> 主角视角 -> 几百字危机 -> 明确目标 -> 第一选择 -> 期待建立
```

## 开篇必备

- 前三百字主角处境。
- 当前压力。
- 主角目标。
- 背景三句话以内。
- 第一选择。
- 第一危险或收益。

## 期待来源

读者先代入主角压力，再逐步接受星际世界观。

## 断档风险

- 上帝视角讲宇宙。
- 节奏慢且语病多。
- 没有期待感。
- 前几百字不能让读者感兴趣。
- 新人模仿慢热沉重风格。

## 训练任务

把星际开篇 1000 字压缩成 300 字，只留主角、压力、目标、选择。
