# 建筑家具资料-时代风格与搜索词入口

依据案例：`batch-254` 案例 1269。

## 读者承诺

建筑家具资料模块把泛泛查资料改成时代、地域、风格、空间功能四步定位，让场景细节服务人物行动。

## 核心循环

```text
确定时代地域 -> 选风格搜索词 -> 查住宅园林家具 -> 提炼空间功能 -> 写成场景戏
```

## 开篇必备

- 先确定中式/西式、古代/现代、具体时期。
- 中式住宅可从四合院或近似四合院结构入手。
- 园林和家具按时代搜索。
- 西式与现代建筑要用明确风格词，不要乱混。

## 期待来源

读者进入场景后，空间结构、家具陈设和人物行动一致，沉浸感和时代可信度自然增强。

## 断档风险

- 只堆建筑名词，不转化为行动路线。
- 不同时代风格混搭，场景像资料拼贴。
- 家具孤立出现，不反映阶层和生活方式。
- 查资料没有服务剧情事件。

## 训练任务

给一个宅院或别墅场景列 5 个搜索词、3 个空间功能、3 件家具陈设，并写出 2 个由空间触发的剧情事件。
