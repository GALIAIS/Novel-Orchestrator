# 反派密谋-篇幅比例与主角视角回流

依据案例：batch-379 案例 1893

## 读者承诺

反派密谋要提供新增信息和信息差，并及时回到主角团。读者看密谋，是为了期待陷阱如何撞上主角，而不是旁听反派开会。

## 核心循环

```text
反派目标 -> 阴谋信息 -> 读者信息差 -> 主角视角回流 -> 陷阱碰撞
```

## 开篇必备

- 反派具体目标。
- 陷害或夺宝手段。
- 主角团未知点。
- 回到主角视角的节点。

## 期待来源

- 读者提前知道危险。
- 主角团暂时不知道。
- 陷阱有明确触发条件。
- 主角即将接近危险。

## 断档风险

- 一整章只写反派商讨。
- 密谋没有新增信息。
- 比例过长导致主线断开。
- 反派只说狠话不行动。

## 训练任务

写一段 1500 字章节结构：前 900 字反派布置陷阱，后 600 字转回主角团，让主角主动走向其中一个危险点。
