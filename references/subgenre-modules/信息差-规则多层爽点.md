# 信息差-规则多层爽点

依据案例：`batch-014` 案例 068。

## 读者承诺

用题材规则定义强弱，再用信息差制造误判，让主角底牌分层释放，一次打脸获得多重收益。

## 核心循环

```text
规则具体化 -> 主角隐藏优势 -> 他人误判 -> 下行压制 -> 底牌释放 -> 多层收益
```

## 开篇必备

- 先明确本题材什么算强。
- 主角至少有一个他人不知道的底牌。
- 打压要围绕规则，不是无脑嘴臭。
- 下行前要让读者知道主角有胜算。

## 期待来源

- 读者知道反派误判了。
- 读者期待主角什么时候亮底牌。
- 一个底牌能解决多个危机。
- 胜利同时带来资源、身份、关系和名声。

## 断档风险

- 规则不具体，爽点无从衡量。
- 下行时读者也不知道主角能赢，只剩憋屈。
- 底牌一次全放完，缺少层次。
- 收益单一，爽点短。

## 训练任务

设计一场打脸戏：写清题材规则、误判者、主角两个底牌、三项收益和读者提前知道胜算的证据。
