# 写作环境-奖励图像启动器

依据案例：`batch-116` 案例 579。

## 读者承诺

写作环境可以降低启动成本。背景图、稿费图和目标提示不是创作能力本身，但能让作者更容易打开软件，进入第一步。

## 核心循环

```text
不想打开软件 -> 设置刺激背景/奖励图 -> 增加启动欲望 -> 进入写作界面 -> 记录实际产出
```

## 开篇必备

- 背景不能影响正文阅读。
- 刺激物要能让自己想打开软件。
- 稿费图比空泛鸡血更具体。
- 启动器只负责开始，不负责剧情质量。
- 用记录验证是否有效。

## 期待来源

- 即时刺激减少拖延。
- 收入目标提供长期想象。
- 每天打开软件变容易。
- 写作行为比空想更多。

## 断档风险

- 把环境优化当成写作训练。
- 背景图干扰码字。
- 只换图不记录产出。
- 目标图变成焦虑来源。

## 训练任务

设置一个写作启动背景或稿费目标图，连续 7 天记录打开软件用时和实际字数。无效就换，不做玄学解释。
