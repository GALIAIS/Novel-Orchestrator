# 飞卢仿写-爽点力度与情绪拉扯

依据案例：`batch-229` 案例 1142。

## 读者承诺

飞卢式爽点不是机械完成铺垫、打压、打脸、收尾、收益，而是让主角在强不公平和强危机下压住情绪，再在公开舞台集中反击。

## 核心循环

```text
明确目标 -> 强压迫 -> 情绪蓄力 -> 公开舞台 -> 反杀打脸 -> 收益兑现
```

## 开篇必备

- 主角有马上要达成的目标。
- 反派压迫落在资格、尊严、前途或生死上。
- 打脸对象最好是前期铺垫过的欺压者。
- 主角反击前有可见的压抑情绪。

## 期待来源

读者先感到主角被不公压住，再期待他在众人面前翻身，爽感来自情绪释放。

## 断档风险

- 主角主动招事，读者不觉得他受害。
- 危机只是小打小闹。
- 临时敌人替代长期欺压者。
- 主角被坑后无奈一笑，情绪下行。
- 女角色或金手指过度工具化，作者自己也代入不足。

## 训练任务

重写一个打脸段落：让反派前文两次欺压主角，主角反击前写一个压抑动作，打脸舞台放在测试、考核或审判现场，结尾必须有收益兑现。
