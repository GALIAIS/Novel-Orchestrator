# 一级结构模板-事业爱情八式

依据案例：`batch-009` 案例 045。

## 读者承诺

把空泛起承转合替换成事业线和爱情线的具体模板，让每个单元都有目标、阻碍、兑现和余韵。

## 核心循环

```text
目标/关系变化 -> 行动 -> 阻碍/危机 -> 解决 -> 收获/余韵
```

## 开篇必备

- 判断本单元是事业线还是爱情线。
- 事业线先找目标，爱情线先找关系变化。
- 从八个模板中选一个主模板。
- 收尾要有收获或余韵。

## 期待来源

- 读者清楚主角当前要什么。
- 阻碍和危机让过程有转折。
- 装逼可能提前产生期待。
- 余韵盘点关系变化和下一步钩子。

## 断档风险

- 只写“起承转合”，没有具体操作。
- 目标不清，阻碍就无意义。
- 爱情线只有互动，没有关系变化。
- 模板组合后自己都理不顺。

## 训练任务

把一章剧情填进八式之一。填不进去就先重写目标或关系变化，不要硬套起承转合。
