# 漫改征文-宣传承诺与合同交付核验

依据案例：`batch-397` 案例 1985

## 读者承诺

面对高稿费与同步漫改宣传，先把愿景拆成主体、入选、收入、权利、交付和退出六组书面条件，用完成案例和小项目验证后再投入长篇。

## 核心循环

```text
宣传吸引 -> 六组条款拆解 -> 规则合同逐项取证 -> 团队与完成案例核验 -> 短项目试水 -> 条件成立后扩大投入
```

## 开篇必备

- 确认活动仍在有效期。
- 区分征集、签约、制作和发行主体。
- 核对“起”价的计费与验收条件。
- 漫改权与小说权分项审查。
- 写清立项、制作、上线和中止规则。
- 长篇前先选择可隔离风险的短项目。

## 期待来源

作者不再靠平台品牌和宣传标题想象结果，而能知道谁在什么条件下支付、制作、上线，以及项目失败时如何退出。

## 断档风险

- 把历史截图当成当前机会。
- 把“同步漫改”理解成保证上线。
- 只看名义千字价。
- 默认平台流量等于专业制作能力。
- 未审权利就按漫改要求重写长篇。
- 社区口碑替代书面合同。

## 训练任务

将一份征文规则按主体、入选、收入、权利、交付、退出六栏填写证据；没有漫改触发条件、期限和中止处理时保持“未证明”，不以宣传语补全。
