# 拆书方法-主线逆推过渡章节

依据案例：`batch-187` 案例 935。

## 读者承诺

拆书时用主线逆推找长期因果，用旁观者视角拆爽感爆发，用过渡章功能检查换地图是否稳固。

## 核心循环

```text
确定卖点 -> 逆推主线 -> 拆情绪结构 -> 标注过渡功能 -> 反哺自己的大纲
```

## 开篇必备

- 先确认作品核心卖点。
- 设定展示尽量找人物言行和节日等载体。
- 战斗拆旁观者误判和读者情绪。
- 换地图章节要承担具体功能。

## 期待来源

作者不只知道标杆“写得好”，还能看出设定、主线、爽感和过渡章各自如何工作。

## 断档风险

- 只摘桥段，不逆推主线。
- 设定全靠作者硬讲。
- 战斗只拆招式，不拆旁观者情绪。
- 过渡章被当成可有可无的水文。

## 训练任务

拆一本标杆的一个换地图段落：列主线逆推节点、设定展示载体、旁观者误判、过渡章承担的小事和配角退场功能。
