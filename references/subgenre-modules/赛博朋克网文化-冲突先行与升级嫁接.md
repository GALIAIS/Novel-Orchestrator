# 赛博朋克网文化-冲突先行与升级嫁接

依据案例：`batch-217` 案例 1082。

## 读者承诺

保留赛博朋克味道，但用冲突危机、欲望兑现和升级体系承载商业网文体验。

## 核心循环

```text
赛博审美 -> 读者体验校准 -> 危机前置 -> 元素后置解释 -> 升级体系嫁接
```

## 开篇必备

- 先写主角处境和危机。
- 少术语，后解释。
- 明确读者图什么。
- 用权限、义体、网络等转成升级路径。

## 期待来源

读者先被危机和目标抓住，再逐渐接受赛博世界观和个人审美。

## 断档风险

- 文学腔和翻译腔压过故事。
- 纯正赛博受众小，新人风险高。
- 主角、金手指、冲突都不清楚。
- 个人理想和商业目标混在一起。

## 训练任务

把纯赛博开篇改成三章网文事件流：危机、能力救命、权限升级；每章只保留 3 个术语。
