# 魂穿设定-身份血脉与爽感兜底

依据案例：`batch-231` 案例 1151。

## 读者承诺

魂穿文要处理身体、灵魂、血脉的身份归属；感情和子嗣线越重，越需要一句设定兜底。

## 核心循环

```text
魂穿爽点 -> 身份追问 -> 血脉/伦理质疑 -> 设定兜底 -> 回到爽感
```

## 开篇必备

- 明确主角与原身的关系。
- 感情线中说明女主喜欢的是当前人格。
- 子嗣线中避免血脉归属脏感。
- 可用转世/记忆复苏兜底。

## 期待来源

读者安心接受主角就是这个身份，再看他变强、恋爱、夺资源，而不是卡在原身与主角的归属问题上。

## 断档风险

- 写后宫或子嗣却不解释身体归属。
- 读者把女主感情理解成喜欢原身。
- 极端逻辑质疑污染爽点。
- 为解释设定写太多，反而拖慢节奏。

## 训练任务

给魂穿设定写三句身份兜底：身体与灵魂关系、原身与主角是否同源、恋爱/子嗣归属。再判断是否需要放入正文前三章。
