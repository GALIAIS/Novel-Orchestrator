# 后宫败犬迫害-关系错位破防

依据案例：batch-277 案例 1383

## 读者承诺

提供后宫文中败犬属性女主的情绪迫害模型：不是单纯虐角色，而是让嘴硬、迟疑、傲娇的女主因为关系证据、竞争者施压和男主撤回关注而破防。

## 核心循环

```text
女主已动心但嘴硬 -> 男主与竞争者亲密 -> 女主目睹/误读 -> 竞争者夸大战果 -> 男主冷处理 -> 女主后悔破防
```

## 开篇必备

- 先建立败犬女主的魅力。
- 让读者知道她其实在意男主。
- 给她一个嘴硬或迟疑的理由。
- 安排竞争者拥有可被误读的亲密证据。

## 期待来源

读者期待她承认心意，但角色因为面子、傲娇或身份继续错过。迫害的快感来自“她明明有机会，却亲手让机会溜走”。

## 断档风险

- 没建立角色魅力就开始迫害，读者不会心疼。
- 把迫害写成身体伤害或恶意霸凌。
- 男主无缘无故冷处理，显得人设崩坏。
- 竞争者炫耀过度，变成低级雌竞。
- 破防后关系状态没有变化。

## 训练任务

写一场 1200 字修罗场：女主 B 先嘴硬拒绝男主，再误以为男主和 A 已经确定关系，最后发现男主不再主动找她。重点写 B 的自我怀疑和面子崩塌。
