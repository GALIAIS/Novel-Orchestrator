# 新人开篇-倒叙逃亡门槛

依据案例：`batch-089` 案例 441。

## 读者承诺

新人可以写逃亡、倒叙和长悬念，但前提是读者先看懂主角处境，并被金手指、人设或短期危机吸住。没有强吸引力时，复杂开篇会先劝退。

## 核心循环

```text
能力评估 -> 选择低门槛开篇 -> 短期危机先结算 -> 金手指给吸引力 -> 再尝试倒叙/双时间线
```

## 开篇必备

- 前 300 字看懂主角处境。
- 前 1000 字看见金手指或强人设。
- 逃亡式开头要有短期结算。
- 倒叙不能让悬念拖太久。
- 双时间线要及时切回主线。

## 期待来源

- 当前危机能否解决。
- 开头预告和回忆内容哪里不对等。
- 金手指为何有趣。
- 主角如何从危机中反杀或复仇。

## 断档风险

- 新人照大神写隐藏最大危机。
- 逃亡过程太长。
- 倒叙悬念跨度太大。
- 金手指吸引力不足还玩结构。

## 训练任务

检查自己的开篇是否倒叙或逃亡。写出短期危机、结算点、金手指吸引点和最长允许悬念长度。
