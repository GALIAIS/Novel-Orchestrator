# 有效慢节奏-戏剧性信息团

依据案例：`batch-007` 案例 032。

## 读者承诺

慢节奏可以成立，但每个信息团都必须吸引目标读者，持续拉期待、做对比、给小满足。

## 核心循环

```text
单元进度放缓 -> 信息团保持戏剧性 -> 小爽点持续 -> 情绪拉扯完整 -> 高潮兑现
```

## 开篇必备

- 明确当前单元不是无效拖延。
- 每个信息团都有目标读者感兴趣的核心信息。
- 有持续对比或小爽点。

## 期待来源

- 读者虽然等高潮，但每一步都有可看的变化。
- 对比持续强化主角优势。
- 情绪拉扯让慢进度也有张力。

## 断档风险

- 信息团多但不吸引人。
- 只有铺垫，没有小满足。
- 慢节奏没有题材素材支撑。
- 生搬硬套结构，缺乏同题材拆文。

## 训练任务

对一章慢节奏正文逐段标“有效/无效信息团”。无效段落必须删除或补戏剧性。
