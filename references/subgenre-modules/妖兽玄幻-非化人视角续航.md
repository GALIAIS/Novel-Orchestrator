# 妖兽玄幻-非化人视角续航

依据案例：`batch-344` 案例 1718

## 读者承诺

妖兽玄幻的核心卖点是非人视角和妖兽三观；不化人能保留卖点，但必须设计长期爽点变化。

## 核心循环

```text
非人开局 -> 野性视角 -> 血脉成长 -> 领地族群 -> 外部征战 -> 妖王秩序
```

## 开篇必备

- 明确非人视角差异。
- 前三章展示野性求生。
- 不化人理由要成立。
- 阶段玩法提前规划。
- 世界背景尺度匹配节奏。

## 期待来源

读者期待主角不是人也能凭野性、血脉和妖族秩序成长为王。

## 断档风险

- 化人后卖点丢失。
- 不化人但桥段重复。
- 只写捕猎升级。
- 仙味慢写但没有事件。
- 十万字后无续航方案。

## 训练任务

设计幼兽求生、血脉觉醒、领地争夺、族群责任、太古征战五阶段，每阶段写一个不同爽点。
