# 学生写作-阅读拆书与退路规划

依据案例：batch-366 案例 1830

## 读者承诺

学生阶段写网文要先补阅读量和拆书量，建立周期化写作，同时保留学历和技能退路，避免用热情堵死现实选择。

## 核心循环

```text
写作理想 -> 经验不足 -> 阅读拆书 -> 周末写作 -> 复盘调整 -> 学历/技能兜底
```

## 开篇必备

- 当前可用时间。
- 阅读输入计划。
- 拆书分析计划。
- 不影响学业的最低码字目标。

## 期待来源

- 输入量增加后更敢动笔。
- 拆大卖作品获得结构经验。
- 周末集中写作有产出。
- 现实退路降低焦虑。

## 断档风险

- 不看够数就硬写。
- 假期日万但没有结构准备。
- 完全放弃休息导致厌烦。
- 未成年或学生阶段过早孤注一掷。

## 训练任务

做一张一周表：平日阅读拆书和灵感记录，周末码字与复盘，并写出不影响学业的最低指标。
