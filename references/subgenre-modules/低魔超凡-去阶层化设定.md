# 低魔超凡-去阶层化设定

依据案例：batch-295 案例 1472

## 读者承诺

在低魔或现实幻想里，让超凡者有用但不能天然凌驾社会，形成能力、制度、资源之间的博弈。

## 核心循环

```text
超凡职业出现 -> 能力受限 -> 社会资源牵制 -> 世俗权力吸纳 -> 职业/阶层博弈
```

## 开篇必备

- 不同超凡职业有不同称谓和利益。
- 超凡者不能脱离社会供给。
- 凡人或政权有制衡手段。
- 主角切入一个具体职业或权力缝隙。

## 期待来源

- 读者期待超凡能力如何被制度利用。
- 职业之间互相竞争，带来冲突。
- 主角用专业能力突破世俗限制。

## 断档风险

- 超凡者强到凡人毫无反制，世界自然变统治阶层。
- 给所有人统一称谓，强化共同身份。
- 假装能力者不会获得资源倾斜，社会逻辑失真。

## 训练任务

设计三个超凡职业，各写一项社会依赖、一项内部矛盾、一项世俗权力制衡方式。
