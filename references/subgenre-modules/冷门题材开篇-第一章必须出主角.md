# 冷门题材开篇-第一章必须出主角

依据案例：`batch-317` 案例 1581

## 读者承诺

冷门复合世界观可以保留新鲜感，但第一章必须让读者确认主角是谁、他在哪里承压、下一步要面对什么。

## 核心循环

```text
冷门题材 -> 主角早出场 -> 设定挂在遭遇上 -> 读者绑定视角 -> 愿意继续理解世界观
```

## 开篇必备

- 前 300 字内让主角露出。
- 第一章给主角明确处境和压力。
- 专有名词数量要克制。
- 世界观只露出能制造危机或悬念的一角。
- 开头小纲先检查“谁、在哪、怕什么、要做什么”。

## 期待来源

读者期待先跟住一个人，再通过这个人的遭遇理解克苏鲁、神话、异世界或其他复杂设定。

## 断档风险

- 第一章只铺氛围，读者不知道跟谁走。
- 主角第二章才出现，开篇代入断开。
- 设定名词太密，冷门题材门槛继续升高。
- 作者误把“干巴巴”当成文笔问题，实际是缺事件和人物。

## 训练任务

把一个复杂世界观开篇重写成 800 字，只保留 1 个专有名词，并在前 300 字内让主角出场、承压、做选择。
