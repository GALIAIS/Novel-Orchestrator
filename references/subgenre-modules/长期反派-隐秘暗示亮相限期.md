# 长期反派-隐秘暗示亮相限期

依据案例：`batch-085` 案例 423。

## 读者承诺

这个模块用于设计一生之敌或长期反派。它承诺让反派早期隐秘、逐步揭露、亮相后限期解决，避免变成拖沓的狗皮膏药。

## 核心循环

```text
长期利益冲突 -> 暗示存在 -> 逐步揭露 -> 清晰亮相 -> 三卷内解决
```

## 开篇必备

- 先设计贯穿全书的利益冲突。
- 早期不要正面冲突太明。
- 用暗示制造幕后感。
- 中后期再清晰认知。
- 亮相后给解决期限。

## 期待来源

- 幕后压力持续存在。
- 信息揭露逐步加深。
- 反派魅力支撑等待。
- 其他内容提供爽感缓冲。
- 解决时有长期积累的释放。

## 断档风险

- 一开始说明白却拖很久。
- 明确打脸主角后不解决。
- 反派反复逃跑升级。
- 没有贯穿利益冲突。
- 阿蒙式反派能力不足硬学。

## 训练任务

写一个长期反派表：5 次暗示、1 次亮相、亮相后三卷内解决方案。写不出解决方案就不要让他早期正面冲突。
