# 童话机制转译-异类能力与都市事件循环

依据案例：`batch-398` 案例 1990

## 读者承诺

从童话寓言中只抽取“非人能力如何击中社会缺口”的机制核，重做载体、时代、欲望、成本、对手和主题，形成独立的都市异能事件循环。

## 核心循环

```text
抽取能力与社会缺口 -> 六轴原创改造 -> 建立有限规则 -> 都市事件实证 -> 使用后果引来追查 -> 伙伴关系和主线升级
```

## 开篇必备

- 非人角色拥有明确但有限的能力。
- 能力击中一个具体社会或生活缺口。
- 主角有独立于来源故事的新欲望。
- 使用能力会支付信息、伦理或关系成本。
- 非人伙伴拥有自身目标。
- 第一事件产生后续追查和关系变化。

## 期待来源

异类视角让熟悉都市显出新规则，能力解决普通方法无解的问题，却不断制造新的社会后果和伙伴矛盾。

## 断档风险

- 直接复用原角色名与事件链。
- 只换时代地点。
- 信息型能力没有边界。
- 非人伙伴只是工具。
- 异类视角只负责吐槽。
- 单元事件不推进长期关系和主线。

## 训练任务

从两个童话各抽一张“能力—社会缺口”卡，再重做载体、时代、欲望、成本、对手和主题六项；新梗概若仍能立即对应原角色与原事件，继续重写。
