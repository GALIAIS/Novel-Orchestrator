# 御兽背景-性价比不可替代

依据案例：`batch-058` 案例 286。

## 读者承诺

这个模块适用于御兽、召唤、灵兽、守护灵题材。它承诺的是先证明“养兽比只修自己更划算”，再谈古代、修真、赛博或洪荒外皮。

## 核心循环

```text
世界制度承认御兽 -> 第一只兽不可替代 -> 收集/养成/进化 -> 对战破局 -> 新兽/新形态期待
```

## 开篇必备

- 说明御兽是不是社会核心制度。
- 第一只兽必须有主角自修替代不了的功能。
- 宠物性价比要直观。
- 背景外皮不能吞掉御兽主线。
- 第一单元要展示收集、培养或对战的核心乐趣。

## 期待来源

- 读者想看不同兽的能力组合。
- 宠物成长能带来明确收益。
- 主角和兽之间有策略配合。
- 下一只兽或下一次进化形成追读。
- 世界规则迫使御兽成为主流路径。

## 断档风险

- 写成普通仙侠宠物坐骑。
- 古代神怪氛围削弱御兽反常感。
- 主角自己修炼更强，宠物变累赘。
- 御兽读者和仙侠读者承诺错位。
- 只借山海经名字，不给养成和对战价值。

## 训练任务

设计一个古代御兽开篇，先写“主角不养兽就必败”的危机，再写第一只兽如何用不可替代能力破局。
