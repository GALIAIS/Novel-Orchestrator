# 虐剧情绪-作者共情与读者尴尬风险

依据案例：`batch-249` 案例 1245。

## 读者承诺

虐剧情绪模块把作者自己的强共情转化为读者能接住的关系铺垫、损失压力和克制表达。

## 核心循环

```text
作者找到虐点 -> 补关系积累 -> 放大具体损失 -> 逼出人物选择 -> 克制表达让读者共情
```

## 开篇必备

- 虐点前先让读者拥有这段关系。
- 损失必须具体，不只写“很痛苦”。
- 人物要有选择和代价。
- 情绪词要克制，用动作和未完成约定承载。

## 期待来源

读者被打动，不是因为作者哭了，而是因为他们提前理解了人物关系，并亲眼看到不可逆的损失发生。

## 断档风险

- 作者情绪很满，读者铺垫不足。
- 直接大段抒情，事件压力不够。
- 生离死别只写哭，不写代价。
- 角色关系没建立，读者觉得被强行煽情。

## 训练任务

写一个 800 字虐戏，限制“哭”或同义表达只出现一次。其余情绪必须通过动作、道具、未完成承诺和人物选择体现。
