# 无限流换卷-下卷钩子

依据案例：`batch-156` 案例 777。

## 读者承诺

无限流换副本时不断档，靠上一卷提前埋下下一卷目标、道具、能力或情报用途。

## 核心循环

```text
当前卷收获 -> 下卷钩子悬置 -> 换卷承接 -> 新副本兑现 -> 伏笔感形成
```

## 开篇必备

- 当前卷未用完的收获。
- 下卷目标。
- 道具或能力用途。
- 新副本进入理由。
- 下卷开篇承接点。

## 期待来源

读者期待上个副本留下的东西在下个副本发挥作用，从而愿意跨卷追读。

## 断档风险

- 每卷孤立结束。
- 新副本只有背景，没有目的。
- 道具和线索长时间不用。
- 下卷开篇不承接上卷期待。

## 训练任务

为当前卷设计 3 个下卷钩子：目标、道具或能力、情报或人物关系；分别写出当前卷埋法和下卷用法。
