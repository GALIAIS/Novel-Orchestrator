# 人物成长-外显成长替代内弧

依据案例：batch-295 案例 1471

## 读者承诺

给新人一个低风险人物成长模型：先让读者看见主角等级、身份、资源、技能、他人态度在变，再考虑复杂心理弧线。

## 核心循环

```text
低位主角 -> 外显成长事件 -> 新权限/新资源 -> 他人态度变化 -> 下一次成长目标
```

## 开篇必备

- 主角初始低位或能力不足。
- 一个清晰可见的成长目标。
- 第一次成长后立刻有外部反馈。
- 主角核心性格保持稳定，不硬写心理拧巴。

## 期待来源

- 读者看见数值、身份、装备、关系的进度。
- 周围人从轻视到认可，提供情绪反馈。
- 技能和资源能不断解决新问题。

## 断档风险

- 强行写性格缺陷修复，变成矫情和毒点。
- 身份变了，语言行为完全没变化。
- 只升级数值，没有他人态度和事件反馈。

## 训练任务

写三次外显成长：一次技能提升，一次身份提升，一次周围人态度变化。每次都用一个场景兑现。
