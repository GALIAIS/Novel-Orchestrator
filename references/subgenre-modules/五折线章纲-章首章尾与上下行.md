# 五折线章纲-章首章尾与上下行

依据案例：batch-348 案例 1738

## 读者承诺

用章首章尾锁定变化，再用五折线安排上下行情绪，帮助章节从平铺事件变成有推进的阅读体验。

## 核心循环

```text
定章首 -> 定章尾变化 -> 填五个转折 -> 标上下行 -> 检查是否教条
```

## 开篇必备

- 本章开头局面清楚。
- 本章结尾必须有变化。
- 中段至少有希望与阻碍的交替。
- 每折都要服务目标、风险或爽点。
- 允许根据章节功能删减变通。

## 期待来源

读者在一章内感到局势起伏，并在结尾获得下一章期待。

## 断档风险

- 机械套五段。
- 只有结构，没有爽点。
- 上下行不影响人物选择。
- 每章节奏雷同。
- 章尾没有状态变化。

## 训练任务

把一个平铺章纲改成五折线，标注每折上行或下行，再删掉最像硬套模板的一折。
