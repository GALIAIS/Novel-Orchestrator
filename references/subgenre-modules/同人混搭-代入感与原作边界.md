# 同人混搭-代入感与原作边界

依据案例：`batch-227` 案例 1131。

## 读者承诺

同人混搭要同时满足原作受众、人设一致性和新故事期待，不能只靠 IP 壳。

## 核心循环

```text
原作背景 -> 新体系改造 -> 人设校验 -> 代入点建立 -> 第一期待
```

## 开篇必备

- 明确主角是原作角色还是原创角色。
- 写出原作人设不可违背处。
- 第一章先给读者期待。
- 无关设定后移。

## 期待来源

读者既想看到熟悉元素被正确使用，也想看到新故事带来的爽点。

## 断档风险

- 原作角色人设崩。
- 力量体系互换只有设定，没有故事。
- 受众过窄，读者无法代入。
- 事件都有，但没有期待。

## 训练任务

把一个火影修仙开头改成两版：原作角色版、原创主角诸天流版。分别写前三章期待、主角代入点、原作元素边界。
