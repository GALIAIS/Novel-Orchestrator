# 飞卢旧书-不续写直接开新书

依据案例：`batch-333` 案例 1664

## 读者承诺

飞卢旧书断更太久且窗口期已过，一般不要续写，复盘后直接开新书。

## 核心循环

```text
旧书断更 -> 推荐热度消失 -> 续写唤醒困难 -> 复盘经验 -> 新书抢窗口
```

## 开篇必备

- 判断断更时长。
- 看旧书推荐窗口是否还在。
- 复盘首订和切书原因。
- 不被沉没成本拖住。
- 新书重新扫榜选题。

## 期待来源

作者把旧书经验变成新书优势，而不是把时间浪费在唤醒旧坑上。

## 断档风险

- 因为旧书首订还行就想续。
- 断更太久，读者和推荐都回不来。
- 补完旧书挤占新书准备。
- 沉没成本影响判断。
- 回飞卢前不扫榜。

## 训练任务

给旧书写一页复盘：题材、开篇、上架节点、首订、切书原因、可复用经验；然后列新书选题。
