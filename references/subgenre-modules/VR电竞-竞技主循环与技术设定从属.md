# VR电竞-竞技主循环与技术设定从属

依据案例：`batch-404` 案例 2017

## 读者承诺

以训练、战术、团队和职业后果作为电竞主循环，让 VR 只负责创造独有交互、位置和信息差；技术不能给主角破坏公平竞争的独占捷径。

## 核心循环

```text
公平规则建立 -> VR 产生独有选择 -> 战队训练形成方案 -> 对手读取与反制 -> 临场团队调整 -> 胜负改变生涯资源 -> 新版本迫使体系进化
```

## 开篇必备

- 先确定电竞职业或游戏冒险的主次。
- 胜利条件、资源和地图对非玩家可读。
- VR 交互删除后会损失真实战术。
- 双方共享公平边界。
- 位置职责和团队短板明确。
- 每场胜负改变排名、关系或职业资源。

## 期待来源

读者先看懂规则，再见主角和队伍在规则内证明能力；个人技巧被对手针对后，团队补位和战术更新完成更高层兑现。

## 断档风险

- VR 只提供视觉包装。
- 独占黑科技直接决定胜负。
- 现代背景被误当成无需世界观。
- 专业术语不解释当前选择。
- 逐帧操作淹没战术因果。
- 多种运动元素互相争夺主循环。
- 使用真实 IP 却未核验权利与平台规则。

## 训练任务

设计一项原创 VR 团队竞技，写清胜利条件、三种资源、五个职责和 VR 独有交互；用三次关键决策完成一场短赛，每次都要出现对手反制和赛后职业后果。
