# 直播弹幕-观众震惊与玩梗水法

依据案例：`batch-242` 案例 1206。

## 读者承诺

用直播间弹幕把主角表现变成可围观的群体震惊，让读者获得热闹、玩梗、低估反转和刷屏爽感。

## 核心循环

```text
围观入口 -> 弹幕玩梗低估 -> 主角表现超预期 -> 弹幕震惊刷屏 -> 主线爽点被放大
```

## 开篇必备

- 设定可围观场景。
- 给弹幕一个初始态度。
- 写出主角被低估或被调侃的点。
- 用表现触发弹幕集体转向。

## 期待来源

读者一边看主角做事，一边看围观群众从看乐子变成被震住。

## 断档风险

- 弹幕太多只剩注水。
- 玩梗过期或太内部。
- 弹幕不服务主角表现。
- 纯直播主线长期事件不足。

## 训练任务

写 20 条弹幕，前半低估和玩梗，后半震惊和扩散；删掉重复句，只保留能推进气氛的 10 条。
