# 更新速度-竞速内卷与幸存者偏差

依据案例：`batch-315` 案例 1572

## 读者承诺

更新速度是连载竞争的一部分，但日更三万、四万属于极端样本。作者要识别读者比较压力，也要识别幸存者偏差。

## 核心循环

```text
读者催更比较 -> 高速作者形成标杆 -> 作者产生压力 -> 判断可持续产能 -> 管理读者预期
```

## 开篇必备

- 明确自身稳定日更线。
- 区分短期冲刺和长期连载。
- 不把顶端个例当行业平均。
- 用作品质量和节奏管理读者期待。

## 期待来源

读者期待稳定、够看、不断粮；作者期待在可持续范围内维持竞争力。

## 断档风险

- 被日更三万样本带偏。
- 误以为写小说普遍暴利。
- 为追速度牺牲身体和质量。
- 只看成功者收入，不看大量失败者。

## 训练任务

写一份自己的更新策略：稳定字数、冲刺字数、休息规则、读者催更回复方式。
