# 开篇反转-从死到生金手指翻盘

依据案例：`batch-114` 案例 569。

## 读者承诺

开篇金手指反转承诺的是强烈命运翻盘：先让读者相信主角下一秒就要完蛋，再用金手指把死局反成生局，并额外给主角资源、身份、名声或线索。

## 核心循环

```text
死局压低 -> 难度加码 -> 金手指首次使用 -> 生死反转 -> 额外获利 -> 连续爽点接上
```

## 开篇必备

- 危机必须足够具体。
- 读者能预见最坏结果。
- 金手指解决的是核心死局。
- 翻盘后不只脱险，还要有收获。
- 后续几章能接连续爽点。

## 期待来源

- 下行越明确，反转越强。
- 金手指第一次使用证明价值。
- 从死到生带来强情绪跃迁。
- 额外收益打开后续剧情。

## 断档风险

- 危机只是轻微麻烦。
- 金手指只解围不获利。
- 反转没有难度铺垫。
- 翻盘后没有连续爽点承接。

## 训练任务

设计一个两段式开篇：第一段把主角压到必死或必败，第二段用金手指翻盘。写清翻盘后主角额外得到的一个好处。
