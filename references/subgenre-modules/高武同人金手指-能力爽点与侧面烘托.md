# 高武同人金手指-能力爽点与侧面烘托

依据案例：`batch-256` 案例 1279。

## 读者承诺

高武同人金手指模块让外挂能力既能救命、升级、反杀，也能通过旁人反应把爽点放大。

## 核心循环

```text
外部压迫 -> 金手指觉醒 -> 能力救命 -> 旁人见证 -> 阴招反用 -> 阶段反杀
```

## 开篇必备

- 平台样本先验证同人/蹭热度边界。
- 能力第一次兑现要清楚。
- 死里逃生要写出没能力会死。
- 旁人反应要放大差异。

## 期待来源

读者看到主角从被迫送死，到用能力活下来，再把对手阴招反用，憋屈得到回账。

## 断档风险

- 只给能力设定，不放大实际爽点。
- 主角被压后只忍，没有记账和回收。
- 旁人不震惊，读者感受不到差异。
- 只纠结标签，不对标平台火书处理。

## 训练任务

重写一个死里逃生桥段：没能力会死、能力差异、2 次旁人反应、1 次反用对手阴招、结尾阶段性回账。
