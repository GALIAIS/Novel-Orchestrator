# 开篇卷拆书-快节奏金手指逆转

依据案例：`batch-170` 案例 849。

## 读者承诺

开篇卷要快速聚焦主角、困境和金手指，尽早完成第一次逆转，再把世界观、人物和规则拆进后续剧情。

## 核心循环

```text
主角入场 -> 困境压迫 -> 金手指抛出 -> 第一次逆转 -> 断章拉高期待 -> 卷纲承接
```

## 开篇必备

- 几千字内进入故事，不先铺大段世界观。
- 主角、困境、金手指、第一次逆转要尽快建立。
- 前期角色少，不一次性铺人物表。
- 紧张剧情中介绍后移，保持节奏。

## 期待来源

读者期待看到主角如何用新机制解决当前困境，并相信这个机制之后还能打开更大爽点。

## 断档风险

- 开局说明书太长，主角迟迟不动。
- 金手指出现后没有立刻兑现。
- 人物靠形容词贴标签，不靠选择呈现。
- 断章位置随意，不能形成追读。

## 训练任务

把一个三章开头重排成“主角入场、困境、金手指、逆转、断章”五步，每章只保留一个核心期待，其余设定后移到剧情触发时再讲。
