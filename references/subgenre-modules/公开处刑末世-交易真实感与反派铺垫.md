# 公开处刑末世-交易真实感与反派铺垫

依据案例：`batch-254` 案例 1267。

## 读者承诺

公开处刑末世模块让资源筹备场景同时完成末世倒计时、交易真实感、反派种子和主角占先机。

## 核心循环

```text
末世倒计时 -> 抢先筹备 -> 交易阻力 -> 反派暴露 -> 末日兑现资源价值
```

## 开篇必备

- 主角知道末世信息，行动要快。
- 买装备不能像购物清单，要有验货、抬价、风险。
- 反派特质要在交易动作中显出来。
- 资源筹备必须被末日爆发证明有用。

## 期待来源

读者期待主角利用未来信息抢占资源、保护关键人物，并在末日降临后看到所有准备变成优势。

## 断档风险

- 交易像买白菜，削弱末世真实感。
- 反派只靠旁白介绍，没有现场压迫。
- 装备清单太长，倒计时压力消失。
- 绑架女主卖点没有末日兑现托底。

## 训练任务

改写一个买装备桥段：加入 1 次交易试探、1 个反派危险细节、1 个主角未来信息优势、1 个后续回收钩子。
