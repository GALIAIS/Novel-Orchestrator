# 克系标签-未知恐怖锦上添花

依据案例：`batch-096` 案例 480。

## 读者承诺

克系在网文里更适合当未知恐怖和神秘感的增味标签，而不是纯粹主承诺。大众读者仍然需要王道追读线：目标、成长、调查、求生、反击。纯克系压抑和旧日概念硬搬，很容易小众化或两头不讨好。

## 核心循环

```text
主流王道底盘 -> 加入未知恐怖 -> 控制压抑浓度 -> 保持主角目标 -> 用克系提升差异化 -> 回到追读线
```

## 开篇必备

- 先有主角目标，再放克系氛围。
- 未知恐怖一次只出现一个核心异常。
- 克系概念不硬搬原著或爆款设定。
- 压抑场景不能连续挤掉剧情推进。

## 期待来源

- 未知恐怖带来的神秘感。
- 主流王道线提供稳定追读。
- 克系元素制造差异化背景。
- 调查、升级、求生中的真相层层揭开。

## 断档风险

- 克系内容少，被标签读者骂挂羊头卖狗肉。
- 克系内容太多，大众读者看不下去。
- 直接搬旧日或《诡秘》概念，底层逻辑驾驭不住。
- 只写掉 san 氛围，没有事件和目标。

## 训练任务

写一个 600 字克系点缀开篇。必须先写主角当前目标，再写一个未知恐怖异常，最后回到主角下一步行动。禁止整段只堆氛围。
