# 至暗时刻-飞卢爽与起点虐压

依据案例：`batch-317` 案例 1583

## 读者承诺

至暗时刻要先确认目标爽感：飞卢式偏轻压迫快回弹，起点式可以重代价慢回弹，但主角都必须不怂且进退有据。

## 核心循环

```text
敌人压迫 -> 主角保持判断 -> 损失制造下行 -> 保留反击空间 -> 胜利回弹
```

## 开篇必备

- 明确敌人为什么强。
- 主角不能因压迫降智或怂化。
- 损失对象要有刻画基础。
- 轻压迫适合快节奏爽文。
- 重牺牲需要读者先喜欢相关角色。

## 期待来源

读者期待主角在强敌面前仍能行动、判断、承担代价，并在反击时把前面的压迫转成爽感回弹。

## 断档风险

- 为了至暗而写工具人死亡，读者无感。
- 压迫太重，目标读者觉得憋屈。
- 主角只挨打不判断，爽点断掉。
- 新手误以为越惨越有爽感。

## 训练任务

同一场危机写两版：飞卢式轻压迫快反击，起点式重代价慢回弹，并说明两版分别适合哪类读者。
