# 题材被否-卖点亮点重包装

依据案例：`batch-212` 案例 1060。

## 读者承诺

题材被否后先重做卖点和亮点，不要马上盲目追热点。

## 核心循环

```text
题材被否 -> 拆被否原因 -> 重做卖点 -> 对比热点 -> 决定重开/改造
```

## 开篇必备

- 记录原题材被否原因。
- 写一句话卖点。
- 拆前三章亮点。
- 判断自己能否写出热点差异点。

## 期待来源

作者从“什么题材火”转向“我能给读者什么点击理由”。

## 断档风险

- 一被否就换赛道。
- 热门题材没有自己的卖点。
- 原题材可改造却直接放弃。
- 投稿前没有一句话测试。

## 训练任务

给被否题材写 3 版重包装：原题材保留版、热点融合版、完全换题材版。
