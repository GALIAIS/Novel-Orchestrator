# 人物塑造-表层反写与代价选择显核

依据案例：`batch-405` 案例 2024

## 读者承诺

先用由同一欲望产生的表层策略形成合理初判，再揭示真实处境；最终让人物在不能两全的压力下付出代价，用选择证明内核并重释早期行为。

## 核心循环

```text
确定人物内核 -> 设计表层策略 -> 形成合理初判 -> 露出真实处境与成本 -> 压力禁止两全 -> 代价选择显核 -> 早期细节和关系被重释
```

## 开篇必备

- 最终要证实的内核清楚。
- 表层与内核由同一欲望或防御连接。
- 先用行动建立初判，不用旁白贴标签。
- 至少一处细节提示表层并非全部。
- 压力场景迫使人物放弃一项需要。
- 选择造成资源、关系或自我代价。

## 期待来源

读者先误判，再发现旧行为比想象中更昂贵；人物用不可两全的选择让核心价值自行显现，反差转成理解和共情。

## 断档风险

- 把随机反常行为当作反写。
- 表层与内核没有共同原因。
- 配角对照替代人物自身选择。
- 揭示只靠旁白宣布“其实是好人”。
- 关键选择没有任何代价。
- 作者意图与读者实际推断不核对。

## 训练任务

禁止直接使用一个目标性格词，先写造成相反初判的动作，再揭示处境，最后安排不能两全的代价选择；让三名读者只按动作复述人物最珍视什么。
