# 爽点分类-能力反转与情绪清算双轴

依据案例：`batch-406` 案例 2029

## 读者承诺

用能力/因果掌控轴和情绪/关系清算轴组织爽点，让两轴在关键行动汇合；大兑现之间以信息、进度、成长和认可小兑现保持希望。

## 核心循环

```text
目标与未偿债 -> 阻力持续加压 -> 小进度兑现 -> 能力或规则突破 -> 行动触及旧问题 -> 情绪清算 -> 状态变化 -> 新目标与代价
```

## 开篇必备

- 主角目标、能力限制和情绪债清楚。
- 小兑现连接当前主线。
- 能力突破来自理解、选择或成长。
- 清算回应具体伤害和关系。
- 弱势角色拥有独立目标与行动。
- 兑现后资源、身份或关系改变。

## 期待来源

读者既看见主角真正掌控因果，也等到长期压迫、误解或亏欠获得回应；两轴同一拍兑现时形成高强度释放。

## 断档风险

- 把个人二分当成完整爽点理论。
- 中段插入与主线无关的救人桥段。
- 弱势角色只充当情绪燃料。
- 固定章数机械安排爽点。
- 连续承诺却迟迟不兑现。
- 连续兑现却没有新阻力。
- 清算升级成无边界报复。

## 训练任务

为十二章画能力轴和情绪轴，标出债务、小兑现与大兑现；设计两次双轴汇合，并删除所有不改变状态的插入式爽点。
