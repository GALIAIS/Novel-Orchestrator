# 兵王人设-退敌不退让与锅功分配

依据案例：`batch-245` 案例 1222。

## 读者承诺

兵王主角可以理智谨慎，但不能让读者觉得他主动退让；保守方案可以别人提，功劳必须回到主角。

## 核心循环

```text
雇主求稳 -> 主角表面配合 -> 暗中提防 -> 危机爆发 -> 主角判断正确
```

## 开篇必备

- 写清兵王善战底色。
- 保守方案由他人提出。
- 主角保留真实判断。
- 后续让危机证明主角。

## 期待来源

读者期待顶级强者有血性、有准备、有判断，而不是主动把尊严和主动权交出去。

## 断档风险

- 主角主动出钱消灾。
- 理智写成怂。
- 血性写成无脑鲁莽。
- 冲突爆发后主角背锅。

## 训练任务

写一段护送冲突：雇主要求息事宁人，主角表面配合、暗中布置，最后危机证明主角判断正确。
