# 写作工具-描写素材检索润色

依据案例：`batch-176` 案例 879。

## 读者承诺

描写素材工具帮助作者扩展词汇、意象和句式，但最终必须按当前场景重写成自己的表达。

## 核心循环

```text
表达贫乏 -> 关键词检索 -> 素材筛选 -> 语境改写 -> 场景落地 -> 标签归档
```

## 开篇必备

- 先明确要描写的人物、物件和场景。
- 同义词解决词汇单薄。
- 古诗文解决意象和气质。
- 反向词典解决“知道意思但想不起词”。
- 素材必须改写后使用。

## 期待来源

作者期待从贫乏表达进入可选表达池。读者期待描写准确、有画面、有信息量，而不是好词堆砌。

## 断档风险

- 复制好词好句，语境不匹配。
- 资料库太大，缺少标签，真正写作时找不到。
- 只追求华丽，不服务人物和动作。
- 临时检索替代长期积累。

## 训练任务

写一个人物首次出场，按声音、大场景、中场景、近景、动作、外貌细节、小动作、台词推进，并归档 5 个可复用词句。
