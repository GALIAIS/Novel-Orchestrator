# 融梗玩梗-剧情融合原则

依据案例：`batch-147` 案例 732。

## 读者承诺

桥段和梗只在符合基调、设定、场景和角色动机时使用。看懂的人会心一笑，看不懂的人也不影响阅读；剧情永远是主体。

## 核心循环

```text
发现梗 -> 判断基调 -> 判断剧情需要 -> 补动机 -> 融入场景 -> 不合适就暂存
```

## 开篇必备

- 当前剧情确实需要这个桥段。
- 角色动机成立。
- 不破坏原有情绪。
- 梗不过时、不烂、不冒犯目标读者。

## 期待来源

读者期待的是自然会心，不是作者强行炫梗。最好的融梗删掉梗名后，剧情仍然成立。

## 断档风险

- 不为剧情服务，只为玩梗。
- 看不懂会影响主线理解。
- 烂梗、过时梗破坏观感。
- 场景和对象不合适，得罪观众。

## 训练任务

列 5 个想用的桥段或梗，逐个判断基调、设定、动机、风险。只允许使用 1 个最适合的，写成 300 字剧情片段，并确认删掉梗名后剧情仍成立。
