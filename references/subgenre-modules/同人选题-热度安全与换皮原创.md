# 同人选题-热度安全与换皮原创

依据案例：batch-291 案例 1454

## 读者承诺

同人选题先看安全风险和原生受众；热度不足时，战力体系可换皮原创或直接换热门题材。

## 核心循环

```text
封禁原因 -> 同人生态 -> 原生受众 -> 同人/换皮原创/换题材 -> 开篇承接
```

## 开篇必备

- 原 IP 安全风险。
- 现有同人数。
- 现有同人成绩。
- 可借战力机制。
- 换皮方式。
- 目标平台热度。

## 期待来源

作者知道自己是在蹭已有读者，还是在从零解释一个原创体系。

## 断档风险

- 政治风险 IP 硬写同人。
- 原生受众少却期待同人自带流量。
- 只借设定，不处理版权和敏感风险。
- 冷门同人不如直接换热门标签。

## 训练任务

给一个旧 IP 做五项判断：安全、受众、同人成绩、可换皮机制、原创成本。
