# 主角光环-被动解毒主动化

依据案例：batch-312 案例 1559

## 读者承诺

危机脱险最好来自主角主动判断。中毒后刚好滚到解药边，容易显得作者硬救。

## 核心循环

```text
主角中毒 -> 已知线索/现场观察 -> 主动冒险找解药 -> 付出代价 -> 合理解毒
```

## 开篇必备

- 前置一个草药或丹药知识。
- 危机中给主角观察机会。
- 主角主动选择行动路线。
- 脱险要有代价。

## 期待来源

期待来自求生能力。读者看到主角靠知识和选择活下来，会比看运气救命更爽。

## 断档风险

- 全靠滚到正确地点。
- 解药毫无前置。
- 危机解决太省事。
- 奇遇过老套，没有主角贡献。

## 训练任务

重写一个中毒脱险桥段：主角必须凭前置知识和现场观察找到解药，不能靠随机摔落。
