# 第一卷细纲-卷纲章纲与前百章规划

依据案例：`batch-221` 案例 1101。

## 读者承诺

第一卷规划要从大纲、卷纲、章纲逐层拆开，让前百章持续兑现类型承诺。

## 核心循环

```text
灵感 -> 大纲 -> 卷纲 -> 章纲 -> 正文推进 -> 卷末兑现
```

## 开篇必备

- 整本主线清楚。
- 第一卷目标明确。
- 每章承担冲突、信息、爽点、期待。
- 范文只拆功能，不照抄事件。

## 期待来源

读者在第一卷看到开篇承诺、金手指启动、阶段冲突和首次大收获逐步兑现。

## 断档风险

- 只要范文，不建自己的卷目标。
- 章纲脱离大纲。
- 前百章只有事件，没有阶段任务。
- 提纲到正文时丢失爽点。

## 训练任务

写一个 30 章第一卷细纲，每章列冲突、信息、爽点、章尾期待。
