# 同人原著-CP情感资产保护

依据案例：`batch-179` 案例 894。

## 读者承诺

同人改设要尊重原著高权重情感资产，让读者先相信作者懂原著，再接受新展开。

## 核心循环

```text
原著记忆 -> 核心关系检查 -> 改设铺垫 -> 情绪补偿 -> 读者接受/完读下滑
```

## 开篇必备

- 列出原著核心 CP、遗憾和羁绊。
- 改设前给足铺垫。
- 不轻易拆高权重情感链。
- 用完读率验证读者是否接受。

## 期待来源

同人读者期待熟悉资产被尊重。高权重关系被硬改，会让读者怀疑作者理解原著。

## 断档风险

- 过早让主角得到女神青睐。
- 拆原著 CP 没铺垫。
- 破坏名场面和核心遗憾。
- 完读率狂掉还继续解释。

## 训练任务

为一个同人感情改设列出原著 CP、遗憾、羁绊、名场面，再写改设铺垫和补偿爽点。
