# 历史文-官场腐败风险边界

依据案例：`batch-193` 案例 962。

## 读者承诺

历史文中的官场腐败不等同现实官场文，关键是把腐败写成历史秩序、制度积弊和剧情阻力，而不是现实影射。

## 核心循环

```text
历史背景 -> 权力腐败阻力 -> 主角破局 -> 旧秩序松动 -> 权谋/改革爽点兑现
```

## 开篇必备

- 明确朝代或架空背景。
- 腐败行为服务主角目标。
- 阻力来自制度、地方利益或旧秩序。
- 表达重心放在历史冲突和剧情推进。

## 期待来源

读者期待主角在历史权力结构中破局，打掉贪官、救民、升迁、改革或借腐败局势完成权谋反杀。

## 断档风险

- 写成现实官场影射。
- 只批判不推进剧情。
- 腐败对象没有制度来源。
- 风险尺度只靠经验判断，不看平台规则。

## 训练任务

写一个贪官侵吞赈灾粮桥段，要求腐败链条、主角破局方式和历史制度背景同时出现。
