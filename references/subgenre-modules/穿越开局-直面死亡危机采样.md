# 穿越开局-直面死亡危机采样

依据案例：`batch-183` 案例 914。

## 读者承诺

醒来或穿越即直面死亡危机的开局，承诺第一时间给读者压力、判断和行动。

## 核心循环

```text
醒来/穿越 -> 死亡现场 -> 快速判断 -> 逃生/搏杀 -> 章末生机
```

## 开篇必备

- 危险已经发生。
- 失败后果明确。
- 主角马上做判断。
- 信息只服务活下去。

## 期待来源

读者期待主角在极短时间内活下来，并通过行动展示能力、性格和金手指可能。

## 断档风险

- 先铺世界观，危险被冲淡。
- 危机只是远处预告。
- 场面混乱，读者看不懂主角判断。
- 只有恐慌，没有动作。

## 训练任务

300 字写“醒来即将死亡”：必须有威胁、倒计时、第一判断、第一动作、失败后果。
