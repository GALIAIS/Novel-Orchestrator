# 情绪转写-自我标签拆成触发矛盾与动作

依据案例：`batch-401` 案例 2004

## 读者承诺

暂停“矫情、混乱、难过”等抽象自我标签，先保留具体材料，再用触发、相反需要、现实任务和动作偏差把情绪转成可参与的场景。

## 核心循环

```text
暂停标签与删改 -> 收集触发、身体、念头和冲动 -> 找到两种相反需要 -> 绑定现实任务与物件 -> 动作偏差泄露 -> 选择付出关系成本 -> 完稿后删重复解释
```

## 开篇必备

- 具体到时间、地点、事件和关系对象。
- 记录身体最先出现的反应。
- 写出想做与阻止自己做的两股力量。
- 给人物一个必须完成的现实任务。
- 选择一个能被触碰、躲避或误用的物件。
- 情绪最终改变动作、选择或关系。

## 期待来源

读者先从小动作看出异常，再等待人物在相反需要之间失控、克制或选择，并承担对方看得见的后果。

## 断档风险

- 用自我贬低替代具体诊断。
- 初稿尚未形成就反复删除。
- 只有情绪形容词，没有触发和动作。
- 长段解释不改变任何选择。
- 所谓留白没有可推断证据。
- 情绪与现实任务互不影响。
- 对真实发言者作心理或人格诊断。

## 训练任务

把一句抽象情绪标签拆成 15 条事实与反应，再写 800 字场景；人物必须完成普通任务，并在两种相反需要间作出有关系成本的选择，正文禁止出现原标签。
