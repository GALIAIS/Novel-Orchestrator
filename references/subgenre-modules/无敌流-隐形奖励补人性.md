# 无敌流-隐形奖励补人性

依据案例：`batch-071` 案例 354。

## 读者承诺

这个模块用于满级大佬、无敌流和强者下新手村。它承诺的是主角不再靠力量升级，而靠亲情、友情、爱情、救赎和社会感知等隐形奖励持续成长。

## 核心循环

```text
战力已满 -> 暴露人性缺口 -> 事件解决 -> 获得关系/情感 -> 更像完整的人
```

## 开篇必备

- 明确主角已经不缺战力。
- 写出主角的情感或社会缺损。
- 表面毒点要有后续合理化。
- 每个事件给隐形奖励。
- 长线目标是补完人性。

## 期待来源

- 碾压提供安全感。
- 缺损提供悬念。
- 关系奖励替代力量奖励。
- 毒点转合理带来反转。
- 神逐渐变成人形成长线期待。

## 断档风险

- 只写碾压，没有内在变化。
- 外界奖励全是钱和地位。
- 主角像机器人但不解释。
- 人性补完没有阶段节点。
- 无敌导致事件没有代价。

## 训练任务

设计 5 个无敌流单元，每个单元只能奖励一种情感或关系，不能奖励战力。写清主角补完了哪一块人性。
