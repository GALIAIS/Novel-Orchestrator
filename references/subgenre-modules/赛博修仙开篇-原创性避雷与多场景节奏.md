# 赛博修仙开篇-原创性避雷与多场景节奏

依据案例：`batch-199` 案例 992。

## 读者承诺

赛博修仙可以借鉴强风格作品，但第一章必须替换标志性意象和场景功能，避免读者一眼认出处。

## 核心循环

```text
借鉴方向 -> 排查标志性细节 -> 替换原创意象 -> 主场景承载钩子 -> 章末切景
```

## 开篇必备

- 一个主场景承载世界观和冲突。
- 标志性借鉴细节全部替换。
- 多场景切换服务同一钩子。
- 主角能力或异常点清楚。

## 期待来源

读者看到的是新鲜赛博修仙口味，而不是某本强风格作品的影子。

## 断档风险

- 复刻原作第一章意象。
- 一章内换景过多但钩子不统一。
- 世界观没立稳就切场景。
- 原创性风险压过节奏讨论。

## 训练任务

列出借鉴对象第一章 5 个标志性细节，全部换成原创意象，再写一个只切一次场景的开头小纲。
