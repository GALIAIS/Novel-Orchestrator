# 马甲化名-身份反差与误判爽点

依据案例：`batch-218` 案例 1089。

## 读者承诺

主角用马甲分离行为权限，制造外界误判、隐藏实力和名字战绩爽点。

## 核心循环

```text
本体人设 -> 马甲行为 -> 外界误判 -> 战绩沉淀 -> 身份反转
```

## 开篇必备

- 马甲有明确用途。
- 马甲行为风格和本体区分。
- 读者知道真相，局中人不知道。
- 第一次使用就留下战绩或传闻。

## 期待来源

读者看到马甲名出现，就期待局中人误判、主角隐藏实力和战绩兑现。

## 断档风险

- 马甲只是换名字，没有行为差异。
- 遮蔽理由不合理。
- 第二人格替代马甲导致毒点。
- 马甲太多但没有战绩记忆。

## 训练任务

设计本体和马甲，写行为边界、外界印象和一场 800 字误判戏。
