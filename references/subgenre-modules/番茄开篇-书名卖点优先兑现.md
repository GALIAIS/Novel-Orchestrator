# 番茄开篇-书名卖点优先兑现

依据案例：batch-305 案例 1525

## 读者承诺

强标题作品的前三章必须优先兑现标题最大卖点。读者因“僵尸老婆”点进来，就要尽快看到僵尸老婆被唤醒、压棺、互动、危机或明确指向，不能被画符和世界观支线抢走注意力。

## 核心循环

```text
书名给出强卖点 -> 读者带期待进入 -> 第一章指向卖点 -> 支线绑定卖点 -> 前三章兑现核心画面
```

## 开篇必备

- 写出书名承诺的第一期待。
- 第一章必须出现或强指向该承诺。
- 支线只能作为主卖点的工具。
- 世界观和对白压缩到服务当前危机。
- 主角要有主动计划，不能只被事件推着走。

## 期待来源

期待来自“标题说了什么，正文马上证明什么”。标题越强，正文越不能绕远；每一场戏都要让读者感觉离主卖点更近。

## 断档风险

- 卷纲完整，但正文前三章主卖点迟迟不出现。
- 支线小高潮占比过大，读者误以为被骗。
- 世界观对白过长，冲淡标题期待。
- 角色形象先行，角色作用后置。

## 训练任务

用一句话写出你的书名最大承诺，再检查前三章每场戏：出现主卖点、强指向主卖点、服务主卖点、不服务。把“不服务”的场景改写或删除。
