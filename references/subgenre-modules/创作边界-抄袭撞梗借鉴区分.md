# 创作边界-抄袭撞梗借鉴区分

依据案例：`batch-185` 案例 924。

## 读者承诺

新人应区分抄袭、撞梗和借鉴：常见设定可共用，复制文本和结构才是高危。

## 核心循环

```text
参考作品 -> 拆功能位 -> 自己重组动机冲突 -> 新表达落地 -> 避免复制
```

## 开篇必备

- 不复制文本。
- 不只换人名。
- 主角动机要重新设计。
- 第一冲突和解决方式要有自己的组织。

## 期待来源

作者放下撞梗恐惧，专注把常见类型素材写出自己的完成度。

## 断档风险

- 调色盘一对比高度一致。
- 剧情结构完全照搬。
- 拆解学习停在复刻。
- 因怕相似而不敢写常见好用设定。

## 训练任务

用同一个常见设定写三个开局，要求主角动机、敌人、冲突和解决方式全部不同。
