# 新人长篇-十万字门槛莽穿

依据案例：batch-283 案例 1414

## 读者承诺

新人第一本到十万字时，首要任务常常是穿过真实连载门槛，获得完整反馈闭环，而不是因审美反噬立刻弃坑。

## 核心循环

```text
写到十万字 -> 回看发现缺陷 -> 焦虑动摇 -> 继续完成阶段目标 -> 数据和经验反哺下一本
```

## 开篇必备

- 当前平台验证节点。
- 已有真实数据。
- 继续写到的最小阶段目标。
- 可复盘的能力清单。
- 明确止损条件。

## 期待来源

作者需要通过真实连载知道自己在哪些地方掉读者、哪里能写动，而不是只在自我否定里反复重开。

## 断档风险

- 把“我写得差”当成唯一判断。
- 审美提升后马上弃坑。
- 没有阶段复盘指标。
- 80 万字大纲带来的远期恐惧压倒当前验证。

## 训练任务

写十万字节点复盘表：继续写到哪个节点、要验证什么能力、哪些缺陷留给下一本、哪些数据出现才止损。
