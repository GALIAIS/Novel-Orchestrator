# 科幻网文-叙事目标与故事性

依据案例：batch-304 案例 1520

## 读者承诺

科幻网文要把概念、文风和思想落到主角事件上，先讲清故事，再谈风格。

## 核心循环

```text
科幻概念 -> 主角目标 -> 当前阻碍 -> 事件推进 -> 设定随行动释放
```

## 开篇必备

- 主角明确。
- 主线目标明确。
- 视角集中。
- 每段文字至少推动人设、关系或情节之一。

## 期待来源

- 读者跟随主角经历事情。
- 科幻设定服务冲突。
- 文风建立氛围但不压过故事。

## 断档风险

- 只有总结没有故事。
- 主角不明确、主线不明确、视角散。
- 翻译味和文青感拉远读者。

## 训练任务

把一段科幻概念开篇改成主角行动：前 1000 字只写主角完成一件事。
