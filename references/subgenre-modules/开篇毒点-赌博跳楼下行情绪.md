# 开篇毒点-赌博跳楼下行情绪

依据案例：batch-301 案例 1505

## 读者承诺

开篇可以让主角惨，但不能让主角主动烂赌作死导致下行，否则读者会先判主角活该。

## 核心循环

```text
主角困境 -> 外部不公平伤害 -> 主角受压 -> 读者同情 -> 主角反击
```

## 开篇必备

- 下行来源尽量来自外部压迫。
- 主角不主动黄赌毒作死。
- 如果涉及赌博，必须有陷害、救亲、被迫等缓冲。
- 更稳是直接改掉毒设定。

## 期待来源

- 读者同情主角被害。
- 反派或环境承担锅。
- 主角保住基本人设，后续逆袭才成立。

## 断档风险

- 赌博欠债跳楼。
- 主角找小三、低能作死、玩物丧志。
- 作者抓着毒设定硬圆。

## 训练任务

把赌博跳楼开篇改成外部陷害版：写陷害者、损失、主角无辜点、反击目标。
