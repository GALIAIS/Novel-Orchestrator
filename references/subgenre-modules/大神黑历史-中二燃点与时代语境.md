# 大神黑历史-中二燃点与时代语境

依据案例：`batch-256` 案例 1276。

## 读者承诺

大神黑历史模块把作者多年后的尴尬感和当年有效的燃点机制分开看，不用羞耻感否定结构。

## 核心循环

```text
群体压力 -> 使命托付 -> 高声量台词 -> 主角承接 -> 情绪爆发
```

## 开篇必备

- 先铺群体困境。
- 主角必须有承接资格。
- 托付台词要落在情绪最高点。
- 模仿时拆结构，不只抄台词。

## 期待来源

读者吃的是铺垫后的集体期待和主角承压爆发，不是孤立的中二台词。

## 断档风险

- 没铺垫直接喊口号。
- 只看多年后尴尬，忽略当年读者语境。
- 作者自我感动但读者没有被推到。
- 高声量台词和人物身份不匹配。

## 训练任务

拆一个热血名场面：3 个群体压力、2 个主角承接理由、1 句高声量台词，再说明台词脱离铺垫为何尴尬。
