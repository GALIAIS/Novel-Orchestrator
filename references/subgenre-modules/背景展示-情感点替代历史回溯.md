# 背景展示-情感点替代历史回溯

依据案例：`batch-210` 案例 1049。

## 读者承诺

主角过去不要靠对白完整讲述，要用场景、物品、行为和选择让读者体会。

## 核心循环

```text
过去背景 -> 当下痕迹 -> 行为反应 -> 选择证明 -> 读者推断
```

## 开篇必备

- 只保留当前剧情必须知道的信息。
- 用物品和生活痕迹暗示过去。
- 用主角选择表现人物。
- 直说信息要短，并和当前冲突相关。

## 期待来源

读者通过推断参与人物理解，比被对白灌输前史更容易产生情感投入。

## 断档风险

- 开头大段历史回溯。
- 对白里完整讲过去。
- 只给信息点，没有情感点。
- 背景设定压住当前剧情。

## 训练任务

把 500 字前史说明改成场景展示，用 5 个物品、动作或选择让读者推断角色过去。
