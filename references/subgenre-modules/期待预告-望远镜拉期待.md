# 期待预告-望远镜拉期待

依据案例：`batch-021` 案例 101。

## 读者承诺

在情绪拉扯前，先让读者看见远处值得追的目标，并让阻碍者足够讨厌。

## 核心循环

```text
远处目标 -> 阻碍出现 -> 读者生恨 -> 暂不兑现 -> 拉扯训练
```

## 开篇必备

- 读者能看见的好处、答案或反击。
- 一个阻碍者。
- 阻碍者做了让人恨的事。
- 当前暂时不能兑现的理由。

## 期待来源

- 读者知道后面有值得看的东西。
- 可恨行为制造反击需求。
- 暂缓兑现让期待积累。

## 断档风险

- 只写冲突，不写远处目标。
- 阻碍者不可恨，读者无感。
- 还没拉期待就急着兑现。

## 训练任务

写一个 300 字片段，只负责拉期待：出现目标、阻碍者和恨意，不许打脸。
