# 飞卢爽文-情绪折线与多重爆发

依据案例：`batch-231` 案例 1152。

## 读者承诺

飞卢爽文的核心是短打压、快上行、多重爆发、即时收益；每句都要服务情绪或收益。

## 核心循环

```text
拉仇恨 -> 主角出手 -> 正面爽 -> 侧面震惊 -> 即时收益 -> 下一轮期待
```

## 开篇必备

- 一个下行后尽快上行。
- 主角正面发威不可缺。
- 路人、同伴、反派、大佬提供侧面爽。
- 冲突后必须有看得见的收益。

## 期待来源

读者被短暂打压调动情绪后，立刻看到主角杀伐、无敌、收获和名望上升。

## 断档风险

- 连续下行压太久。
- 只有主角动作，没有多方震惊放大。
- 冲突没有即时收益。
- 句子没有功能，拖慢短平快节奏。

## 训练任务

把 2000 字爽文拆成“剧情、作用、调动情感”三列表。没有明确情绪或收益作用的句子直接删除。
