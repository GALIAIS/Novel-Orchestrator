# 人脉关系-升级剧情入口

依据案例：batch-301 案例 1501

## 读者承诺

人脉像战力一样升级：主角能调动的人、资源、情报和圈层越来越高级。

## 核心循环

```text
低位关系 -> 互惠事件 -> 关系信任 -> 资源兑现 -> 更高层入口
```

## 开篇必备

- 主角初始关系资源有限。
- 关键人物有明确需求。
- 主角提供一次价值。
- 关系升级后必须在主线里兑现。

## 期待来源

- 读者看到人脉反哺主线。
- 关系升级带来安全感和布局感。
- 新圈层打开新任务和新资源。

## 断档风险

- 只认识大佬，没有互惠。
- 通讯录堆人名，不产生剧情。
- 人脉帮忙没有代价和边界。

## 训练任务

写三层人脉升级，每层一个互惠事件和一次关键兑现。
