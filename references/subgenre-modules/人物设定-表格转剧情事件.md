# 人物设定-表格转剧情事件

依据案例：`batch-338` 案例 1689

## 读者承诺

人物设定表只在能生成选择、冲突和章节事件时有价值，不能停留在自我满足的档案。

## 核心循环

```text
人物印象 -> 追求/恐惧/缺点 -> 逼迫选择的事件 -> 行动显形 -> 章节小纲
```

## 开篇必备

- 一句话人物印象。
- 至少一个现实参照。
- 追求、恐惧、缺点能转事件。
- 设定同步生成小剧情。
- 删除无法表现的装饰设定。

## 期待来源

读者期待人物在事件中表现不同，而不是看到作者填了多少设定题。

## 断档风险

- 机械填 40 题。
- 设定太多记不住。
- 人物丰满停留在作者脑内。
- 世界观设定不推剧情。
- 还没写正文就被设定消耗完。

## 训练任务

只取人物设定表里的追求、恐惧、缺点、最不可能做的事、生活习惯五项，每项必须生成一个剧情事件。
