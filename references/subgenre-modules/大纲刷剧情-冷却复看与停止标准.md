# 大纲刷剧情-冷却复看与停止标准

依据案例：batch-349 案例 1741

## 读者承诺

大纲通过冷却复看和试写验证提高可写性，但必须设置停止标准，避免无限刷剧情。

## 核心循环

```text
全书架构 -> 卷纲小纲 -> 冷却复看 -> 试写开头 -> 修正路线 -> 达标开写
```

## 开篇必备

- 先有全书终局和第一卷目标。
- 第一卷小纲要能支撑第一章开写。
- 每次修改后留出冷却期。
- 试写开头验证路线是否落地。
- 提前定义“够好就停”的标准。

## 期待来源

作者获得可执行的大纲，而不是永远更好的幻想版本。

## 断档风险

- 没有停止标准，越改越没完。
- 新点子不断替换主线。
- 只做大纲不试写。
- 试写偏离后全盘推翻。
- 追求完全满意，拖延开书。

## 训练任务

给当前小纲写 3 条验收线。冷却两天后只改不达标部分，达标后立即写第一章。
