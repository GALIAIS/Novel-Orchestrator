# 反派洗白-宰掉最低风险

依据案例：batch-297 案例 1485

## 读者承诺

反派洗白是高风险工程；没有充分准备时，让反派死亡赎罪通常比活着被原谅更稳。

## 核心循环

```text
反派罪行累积 -> 作者想洗白 -> 检查铺垫/代价 -> 不足则死亡退场 -> 情绪风险收束
```

## 开篇必备

- 反派罪行和读者仇恨强度明确。
- 早期有可追溯铺垫。
- 受害者不能被忽略。
- 代价必须不可逆。

## 期待来源

- 读者看到恶人付出成本。
- 醒悟不等于清账。
- 死亡让赎罪有分量。

## 断档风险

- 临时洗白，前文无准备。
- 主角轻易原谅大恶人。
- 反派活着享受新生活，受害者被遗忘。

## 训练任务

给一个待洗白反派打分：铺垫、代价、受害者处理、主角态度。任一项低于及格，就改写成死亡退场。
