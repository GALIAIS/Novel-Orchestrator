# 写作自查-目标读者与追订反馈

依据案例：batch-383 案例 1915

## 读者承诺

自查要先固定写作目的、目标读者和少量评价标准，再通过完整作品与追订反馈校准，不能无限拆框架。

## 核心循环

```text
明确目的 -> 锁定读者 -> 设定标准 -> 完成并上架 -> 反馈校准
```

## 开篇必备

- 一个首要写作目的。
- 一个目标读者群。
- 三项以内评价标准。
- 明确暂不追求的效果。

## 期待来源

- 判断标准不再反复漂移。
- 自查结论能转成改稿动作。
- 完整作品产生真实反馈。
- 追订帮助校准读者承诺。

## 断档风险

- 天天自查却不继续写。
- 同时追求所有优点。
- 模仿神作的多目标例外。
- 没有作品反馈就不断细分理论。

## 训练任务

写一页自查契约：一个目的、一个读者群、三项标准、三项暂不追求；完成十章后结合实际反馈只复盘一次。
