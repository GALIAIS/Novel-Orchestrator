# 开篇人设-言行合一去回忆

依据案例：batch-281 案例 1401

## 读者承诺

开篇先用当前危机验证人物，不用大段回忆解释人物。主角台词和行动必须服务核心人设，不能为了有趣把冷静孤狼写成憨憨。

## 核心循环

```text
核心人设 -> 当前危机 -> 台词/行动验证 -> 回忆后置 -> 人物可信
```

## 开篇必备

- 主角核心人设一句话。
- 当前追兵或危机。
- 符合人设的第一反应。
- 回忆压缩到最低。

## 期待来源

读者通过危险中的选择相信主角，而不是通过作者解释相信主角。

## 断档风险

- 前文塑造冷静，后文台词憨。
- 为了逗趣牺牲人设一致性。
- 开篇写鞋子、师傅、功法来历。
- 回忆压过当前危机。

## 训练任务

把开篇回忆删到 3 句内，重写主角前 5 句台词，确保每句都符合核心人设。
