# 大纲细纲-新人路线图

依据案例：`batch-143` 案例 711。

## 读者承诺

新人先用大纲确定方向，再用细纲补足每章目标、冲突、动作、情绪点和钩子，避免现想现写导致跑偏。

## 核心循环

```text
大纲定路线 -> 正文暴露细节缺口 -> 细纲补步骤 -> 按细纲执行 -> 写后修正路线图
```

## 开篇必备

- 一句话主线。
- 卷级目标。
- 章节目标。
- 本章冲突。
- 本章结尾钩子。

## 期待来源

- 作者知道往哪写。
- 每章执行更稳。
- 新灵感不容易带偏主线。
- 细节缺口提前暴露。

## 断档风险

- 误以为有大纲就等于能写正文。
- 照搬大神无外显大纲。
- 现想现写时一激动跑偏。
- 日常文过度细纲化，丢失味道。

## 训练任务

把一个 5 章大纲扩成细纲。每章写清主角目标、人物、冲突、关键动作、情绪点和结尾钩子，并标出原大纲没覆盖的细节。
