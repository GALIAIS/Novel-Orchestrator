# 人脉矛盾网-资源与冲突同步

依据案例：batch-303 案例 1515

## 读者承诺

人脉升级要放进矛盾网：获得资源的同时，引入新的势力、利益和冲突。

## 核心循环

```text
结识人脉 -> 获得资源 -> 接入势力利益 -> 触发反对者 -> 矛盾网升级
```

## 开篇必备

- 人脉属于某个势力。
- 对方有自己的利益。
- 帮主角会影响别人。
- 新资源带来新冲突。

## 期待来源

- 贵人相助不再像外挂。
- 资源兑现同时扩大格局。
- 新敌人和新圈层自然出现。

## 断档风险

- 人脉只负责送资源。
- 大佬无代价帮主角。
- 关系升级不改变矛盾网。

## 训练任务

画三节点关系：主角、帮助者、反对者。写出资源流向、利益冲突、下一层人脉入口。
