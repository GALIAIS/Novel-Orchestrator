# 双现实惊悚-证据对冲与情感锚点

依据案例：`batch-411` 案例 2054

## 读者承诺

两套现实各有可行动证据和竞争解释，主角以跨世界稳定欲望主动测试；每次含混结果都迫使不可逆选择，并让另一现实中的珍贵关系承担情感代价。

## 核心循环

```text
稳定欲望跨越两种现实 -> 世界A给证据 -> 世界B以关系反证 -> 主角主动测试 -> 结果支持一边又保留另一解释 -> 不可逆行动 -> 跨模型情感代价 -> 重估可信对象 -> 共同异常逼近真相
```

## 开篇必备

- 两个世界各自稳定规则。
- 跨世界成立的欲望与关系锚点。
- 支持双方的证据、反证和未知项。
- 能改变行动的可观察测试。
- 每轮摇摆留下的不可逆状态。
- 局部安全、答案和主动计划。

## 期待来源

读者在一边站稳后，另一边以熟悉关系推翻判断；主角测试却得到双重解释，只能为守护一边作出会伤害另一边的选择，继续追查两边共同异常。

## 断档风险

- 只有混乱，没有可理解的当前判断。
- 一边证据全真、另一边纯作者欺骗。
- 每轮用“都是梦”抹掉状态变化。
- 持续压抑却不给局部目标和喘息。
- 主角被动承受，从不设计测试。
- 把现实精神疾病等同暴力、邪恶或怪物。

## 训练任务

为同一事件写 A、B 两套解释，各列三项支持和一项反证；让主角因稳定欲望进行一次可被两边解释的测试，并作出会伤害另一边关系的选择，最后审查无后果重置与精神疾病污名化。
