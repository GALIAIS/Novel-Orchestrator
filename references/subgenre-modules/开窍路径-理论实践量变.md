# 开窍路径-理论实践量变

依据案例：`batch-096` 案例 478。

## 读者承诺

开窍不是玄学，而是理论、实践、复盘积累到一定量后，突然理解某个理论如何落到正文。开窍可以发生很多次，每次打通一个具体能力点。

## 核心循环

```text
学理论 -> 写片段 -> 失败复盘 -> 再写 -> 某个点突然连通 -> 应用到正文 -> 进入下一个能力点
```

## 开篇必备

- 训练要锁定一个具体理论点。
- 理论必须落到正文片段。
- 每次写完要复盘哪里落成、哪里没落成。
- 不等整体顿悟，只追单点突破。

## 期待来源

- 抽象理论变成可写动作。
- 片段练习带来即时反馈。
- 复盘发现稳定错误。
- 单点突破后正文趣味性变强。

## 断档风险

- 只学理论不写。
- 只写正文不复盘。
- 把开窍想成一次性终局。
- 追求捷径，跳过积累。

## 训练任务

选一个理论点练 7 天。每天写 300 字正文片段，再写 3 行复盘：我用了什么理论、哪一句落成了、下一次要改什么。
