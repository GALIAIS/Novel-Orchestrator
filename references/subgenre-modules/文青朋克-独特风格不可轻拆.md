# 文青朋克-独特风格不可轻拆

依据案例：`batch-258` 案例 1289。

## 读者承诺

文青朋克模块把个人喜欢和新人可学分开，避免直接模仿主线不清、风格独特的高门槛样本。

## 核心循环

```text
喜欢神作 -> 识别吸引来源 -> 判断是否可迁移 -> 剥离不可学风格 -> 换可学样本
```

## 开篇必备

- 先判断吸引力是否依赖作者个人风格。
- 新人开书优先保证主线和承诺清晰。
- 年代早、文青风、风格独特的样本谨慎拆。
- 朋克风也要有明确读者钩子。

## 期待来源

作者保留审美偏好，但学习时选择更可复制、更贴近当下市场的结构样本。

## 断档风险

- 把“我喜欢”误当“我能学”。
- 只学到主线不清。
- 意外推动替代主角目标。
- 文青氛围盖过商业承诺。

## 训练任务

给一本喜欢的风格神作列 3 个可学点和 3 个不可学点，再找一本同题材当下榜单书做替代模板。
