# 平台选择-造神能力与收益定位

依据案例：`batch-212` 案例 1056。

## 读者承诺

选平台前先分清自己要赚钱、曝光、稳定投稿，还是白金级影响力。

## 核心循环

```text
作者目标 -> 平台生态 -> 赚钱/造神分离 -> 预期校准 -> 投稿策略
```

## 开篇必备

- 明确当前目标级别。
- 区分赚钱平台和造神平台。
- 评估平台读者规模和榜单生态。
- 不把出版路径和网文路径混用。

## 期待来源

作者不再靠“哪里竞争小”选平台，而是按目标选择承受对应代价。

## 断档风险

- 以为小平台更容易成大神。
- 把赚钱机会误判成造神路径。
- 只逃避内卷，不看平台上限。
- 出版和网文标准混淆。

## 训练任务

给 3 个平台做目标匹配表：赚钱机会、造神可能、竞争强度、适合题材、失败代价。
