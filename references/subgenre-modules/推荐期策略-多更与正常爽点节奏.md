# 推荐期策略-多更与正常爽点节奏

依据案例：`batch-254` 案例 1268。

## 读者承诺

推荐期策略模块把推荐曝光转化为更新供给和稳定追读，不为了推荐临时扭曲主线高潮。

## 核心循环

```text
推荐曝光 -> 多更承接 -> 正常情绪折线 -> 稳定爽点 -> 新读者追读
```

## 开篇必备

- 先保证更新量和存稿缓冲。
- 保持原有情绪折线。
- 每章继续兑现既有爽点。
- 高潮卡点留给上架等更适合的节点。

## 期待来源

新读者从推荐入口进入后，能连续读到稳定更新和顺滑爽点，降低追读成本。

## 断档风险

- 临时强推高潮导致铺垫不足。
- 为推荐改乱原本节奏。
- 更新不足，曝光无法承接。
- 只追求大爆点，日常爽点断供。

## 训练任务

做一张 7 天推荐期更新表：每天写明更新量、当日爽点、章末期待，不能新增临时大高潮。
