# 无限流-副本外矛盾网主线

依据案例：`batch-193` 案例 964。

## 读者承诺

无限流不必把每个副本强行编进同一张矛盾网，而是用副本内单元危机提供即时可读性，用副本外主线提供长期方向和后果。

## 核心循环

```text
副本任务 -> 单元危机解决 -> 副本外势力变化 -> 旧敌介入新本 -> 长线矛盾升级
```

## 开篇必备

- 第一个副本有清楚基础任务。
- 副本外有主线方向或势力压力。
- 主角每次通关都会改变外部关系。
- 至少一个外部后果能进入后续副本。

## 期待来源

读者既能享受每个副本独立电影式的任务压力，也能期待副本外旧仇、势力下注和终点主线逐步靠近。

## 断档风险

- 硬套传统升级矛盾网，结构过重。
- 每个副本完全散装，没有长期后果。
- 只有主神任务，没有主角方向。
- 副本外势力不影响副本内危机。

## 训练任务

设计三个副本，每个副本写一条内部任务，再写一条副本外势力如何改变下一副本难度。
