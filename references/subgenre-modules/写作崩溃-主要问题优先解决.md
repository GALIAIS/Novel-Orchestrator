# 写作崩溃-主要问题优先解决

依据案例：batch-356 案例 1780

## 读者承诺

长期没成绩时，焦虑会让作者到处学、到处改，像无头苍蝇。恢复方向的办法是先抓主要问题，只训练最影响追读的一个短板，解决后再换下一个。

## 核心循环

```text
心态崩溃 -> 冷静列问题 -> 选主要问题 -> 单点训练 -> 反馈后选新问题
```

## 开篇必备

- 当前最痛的作品问题。
- 套路、期待、人物、爽感等候选分类。
- 本周只解决的一项。
- 对应参考案例。

## 期待来源

- 问题从一团乱变成一个训练点。
- 单点进步带来掌控感。
- 爽感和短期期待先撑起基本盘。
- 厚积薄发变成可执行迭代。

## 断档风险

- 同时修所有缺点。
- 看教程做笔记替代行动。
- 只靠咬牙坚持，不做问题排序。
- 把非核心缺点当成第一优先级。

## 训练任务

本周只训练一个问题。每天写 500 字同类桥段，晚上只检查这个问题是否改善，其他缺点暂时不管。
