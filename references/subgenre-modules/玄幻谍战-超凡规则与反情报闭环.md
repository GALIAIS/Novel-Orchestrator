# 玄幻谍战-超凡规则与反情报闭环

依据案例：batch-390 案例 1948

## 读者承诺

超凡能力必须真实改变潜入、伪装和情报操作，同时留下成本与痕迹供敌方反制，使玄幻手段和谍战不确定性形成同一循环。

## 核心循环

```text
接受情报任务 -> 超凡手段破局 -> 能力留下痕迹 -> 敌方反侦察 -> 主角调整身份与计划
```

## 开篇必备

- 一个组织、一层掩护身份和一项明确任务。
- 能改变任务执行方式的核心能力。
- 能力的代价、痕迹或使用边界。
- 敌方理解规则后的合理核验与反制。

## 期待来源

- 超凡手段提供现实谍战没有的新选择。
- 每次使用能力都增加暴露风险。
- 真假情报迫使双方持续博弈。
- 阶段成功升级更深身份危机。

## 断档风险

- 玄幻和谍战只并排贴标签。
- 万能搜魂易容消灭全部悬念。
- 设定并发量超过新人控场能力。
- 附件缺失时猜测原方案内容。

## 训练任务

为同一潜入任务写两版开场，每版只用一种能力和一种反制；让读者复述任务、代价与暴露点，选择最清楚且可继续循环的一版。
