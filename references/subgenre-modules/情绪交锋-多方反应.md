# 情绪交锋-多方反应

依据案例：`batch-006` 案例 027。

## 读者承诺

情绪不是直接说出来，而是在事件影响、人物交锋和多方反应里自然生成。

## 核心循环

```text
刺激事件 -> 立场被触动 -> 交锋过招 -> 多方反应 -> 情绪结果
```

## 开篇必备

- 主角在乎的东西。
- 对手或环境施加的压力。
- 至少一轮来回交锋。
- 旁观者或关系人的不同反应。

## 期待来源

- 读者想看主角如何回应刺激。
- 读者通过多方反应判断局势。
- 情绪在交锋中升级，而不是靠作者解释。

## 断档风险

- 只写主角感受，情绪曲线单调。
- 直接堆情绪词，没有事件化学反应。
- 加太多反应导致繁琐。
- 没有立场差，交锋不成立。

## 训练任务

把一句情绪描写改成 800 字交锋，至少写主角、对手、旁观者三方反应。
