# 都市异能开篇-设定渐透视角过渡

依据案例：batch-280 案例 1397

## 读者承诺

都市异能开篇先让读者站到主角视角里，再通过小事件逐步透出非凡规则；不要开场硬塞设定、战力体系和大打斗。

## 核心循环

```text
普通视角 -> 小异常 -> 小冲突 -> 非凡规则露出 -> 主角选择 -> 大主线
```

## 开篇必备

- 主角身份和日常处境。
- 一个能引出异常的小事件。
- 清楚的镜头转换和因果。
- 设定随冲突出现。

## 期待来源

读者和主角一起发现非凡世界，代入感来自“我知道他为什么被卷入”，爽点来自普通世界被撕开后的选择。

## 断档风险

- 第一章先抛大量设定。
- 人物没立住就开打。
- 战力体系还没建立，战斗无爽感。
- 镜头转换混乱，因果断裂。

## 训练任务

重写一个都市异能开头：1000 字内只写一个异常小事件和主角反应，设定不超过 3 句。
