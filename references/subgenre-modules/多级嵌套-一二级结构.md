# 多级嵌套-一二级结构

依据案例：`batch-005` 案例 023。

## 读者承诺

把单元故事、二级节点、卷结构串起来，让长篇不是一堆孤立事件，而是多级期待折线嵌套。

## 核心循环

```text
一级单元 -> 二级节点 -> 卷目标 -> 长线期待 -> 下一级单元回收
```

## 开篇必备

- 每个一级单元有独立期待和兑现。
- 多个一级单元服务同一个二级节点。
- 二级节点服务卷目标。
- 结构用法灵活，不按字数硬套。

## 期待来源

- 小单元有即时追读。
- 二级节点有阶段目标。
- 卷目标有长线牵引。
- 嵌套关系让兑现后仍有上层期待。

## 断档风险

- 单元彼此孤立，爽完就断。
- 二级结构只是字数划分，没有目标。
- 故事卡用法死板，反而卡住写作。
- 学结构后只会否定旧稿，不继续写。

## 训练任务

把一个 10 万字卷纲拆成 4 个二级节点，每个节点下列 3 个一级单元，并写出上下层期待如何衔接。
