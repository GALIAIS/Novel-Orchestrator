# 小纲正文-对照诊断

依据案例：`batch-138` 案例 688。

## 读者承诺

小纲说明作者意图，正文验证执行效果。公开处刑要把两者对照，才能判断作者是否真正写到了点上。

## 核心循环

```text
小纲承诺 -> 正文呈现 -> 对照偏差 -> 定位视角/画面/重点问题 -> 修改
```

## 开篇必备

- 小纲写清核心事件。
- 正文提供对应片段。
- 检查重点是否一致。
- 检查视角和画面。
- 检查情绪是否兑现。

## 期待来源

- 评审不被概括误导。
- 作者能发现执行偏差。
- 正文问题更具体。
- 修改建议更落地。

## 断档风险

- 只看小纲，忽略执行。
- 小纲总结失真。
- 正文视角偏移。
- 画面和重点没写出来。

## 训练任务

给一组小纲和正文，判断小纲是否抓重点、正文是否兑现、是否有视角/画面/重点偏移，并写一条修改建议。
