# 现代高武-反差升级开篇

依据案例：`batch-146` 案例 728。

## 读者承诺

主角以为重生回熟悉现代，准备用旧经验发财；世界却反转成以武为尊，旧经验失效。读者追看主角如何在新规则下重新找到上升通道。

## 核心循环

```text
重生补偿感 -> 世界规则反转 -> 旧经验失效 -> 高武反差钩子 -> 行动目标 -> 升级入口
```

## 开篇必备

- 熟悉现代入口。
- 主角对旧经验的补偿期待。
- 高武世界反差信息。
- 旧经验失效的打击。
- 一个可执行目标，如考武科、报名费、资格。

## 期待来源

读者期待“套路不对”后的重新上升。没有早出金手指时，世界观反差、通俗行文和明确目标必须承担留人。

## 断档风险

- 把重生身份误当有效金手指。
- 旧经验失效后没有新目标。
- 只讲设定，不给行动。
- 新人硬学大神慢出金手指。

## 训练任务

写一个 1500 字开篇：主角以为重生回熟悉年代，中段发现世界以武为尊，旧经验失效，章末给出一个明确行动目标。金手指可不启用，但必须有替代留人机制。
