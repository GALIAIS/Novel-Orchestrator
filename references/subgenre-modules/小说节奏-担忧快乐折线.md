# 小说节奏-担忧快乐折线

依据案例：`batch-313` 案例 1564

## 读者承诺

用最小情绪折线组织桥段：先让读者替主角或主角在乎的人担忧，再让读者为主角的反击快乐。

## 核心循环

```text
威胁出现 -> 主角暂时缺席/受限 -> 读者担忧 -> 虚晃一枪 -> 主角关键行动 -> 情绪上行
```

## 开篇必备

- 有明确被威胁对象。
- 有反派或外部压力。
- 主角不能立刻解决，必须让担忧成立。
- 转折点要靠主角行动，不靠旁白宣布。
- 回收要具体，最好有打脸动作。

## 期待来源

读者期待坏结果被阻止，并期待主角用行动证明“他能赶上、能解决、能让反派付代价”。

## 断档风险

- 只有威胁没有回收，变压抑。
- 主角出现太早，担忧不足。
- 主角出现太晚，读者疲劳。
- 反击太轻，快乐不足。

## 训练任务

写一段救场桥段：用 400 字制造担忧，用 400 字完成出现、反击、打脸。
