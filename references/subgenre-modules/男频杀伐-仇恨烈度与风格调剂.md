# 男频杀伐-仇恨烈度与风格调剂

依据案例：batch-381 案例 1901

## 读者承诺

杀伐果断要服务题材烈度和读者宣泄。仇恨值拉高后，主角必须给奖惩；长期严肃后，要用缓和、情感或配角欢乐调剂。

## 核心循环

```text
题材预期 -> 反派拉仇恨 -> 读者等宣泄 -> 主角果断奖惩 -> 战后调剂
```

## 开篇必备

- 题材和平台烈度。
- 反派伤害对象。
- 主角可克制的边界。
- 必须果断的触发点。

## 期待来源

- 反派伤害主角重视的人。
- 读者等待快意恩仇。
- 主角既能怂也能莽。
- 战后有情绪缓冲。

## 断档风险

- 仇恨值拉高却轻放。
- 纯都市乱套玄幻杀伐。
- 主角性格剧烈变化破坏风格。
- 长期绷紧导致疲倦。

## 训练任务

写一个冲突场景：先拉高仇恨值，主角一次克制后果断奖惩，结尾用配角或情感段落缓和。
