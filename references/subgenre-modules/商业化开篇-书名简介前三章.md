# 商业化开篇-书名简介前三章

依据案例：`batch-040` 案例 200。

## 读者承诺

商业化开篇要求书名、简介、前三章同向。书名下钩子，简介拉情绪和主角目标，前三章兑现承诺。

## 核心循环

```text
书名题材/事件/风格 -> 简介主题/情绪 -> 前三章承接 -> 主线推进 -> 留存
```

## 开篇必备

- 书名有题材、事件或风格钩子。
- 简介说明主角要干什么。
- 前三章承接包装承诺。
- 视角中心服务主角和主线。

## 期待来源

- 扫榜时愿意点进来。
- 简介确认情绪方向。
- 前三章快速验证承诺。
- 主线笔墨集中。

## 断档风险

- 淡如水开篇，新人无读者宽容。
- 无关配角吃笔墨。
- 火书段落盲学。
- 书名简介和正文风格不一致。

## 训练任务

写一个商业化开篇包：5 个书名、2 版简介、前三章承接表，并删掉不能服务主线的配角描写。
