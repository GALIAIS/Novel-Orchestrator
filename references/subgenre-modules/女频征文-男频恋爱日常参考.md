# 女频征文-男频恋爱日常参考

依据案例：batch-312 案例 1556

## 读者承诺

女频甜宠征文可以参考男频恋爱日常，但主线必须是男女主感情发展，其他元素只能做支线。

## 核心循环

```text
征文规则 -> 感情主线 -> 男频日常节奏参考 -> 支线偶尔波动 -> 甜宠回收
```

## 开篇必备

- 明确征文禁止主线。
- 男女主关系必须先启动。
- 打脸、脑洞、穿越只能点缀。
- 日常狗粮要连续供给。

## 期待来源

期待来自关系推进。读者看的是两个人如何靠近、误会、互动、确认，而不是外部爽文主线。

## 断档风险

- 借甜宠壳写打脸爽文。
- 穿越/穿书元素主导剧情。
- 男频样本节奏照搬，忽略女频情绪细腻度。
- 支线抢走男女主互动篇幅。

## 训练任务

把一个男频恋爱日常桥段改成女频甜宠征文版：每场戏都必须推动男女主关系。
