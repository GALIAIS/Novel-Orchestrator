# 网文输入-先补同赛道阅读

依据案例：`batch-322` 案例 1610

## 读者承诺

想写网文就要补同赛道网文输入；动画能提供画面感，但不能替代章节节奏、爽点回收和读者期待训练。

## 核心循环

```text
创作痛苦 -> 发现输入不足 -> 破除作者滤镜 -> 补读同题材大神 -> 建立网文语感
```

## 开篇必备

- 承认自己的主要输入来源。
- 找出缺少的网文经验。
- 用自己的文暴露问题。
- 补读同题材大神前三章。
- 每本只拆一个具体能力点。

## 期待来源

作者期待通过补输入建立可调用的类型节奏、章节钩子和语言模板。

## 断档风险

- 不看网文却想写网文。
- 写作后看不进小说，输入断裂。
- 只靠动画画面感，没有网文章法。
- 读书不拆，无法转成写法。

## 训练任务

选 3 本同题材大神网文，每本拆前三章的章节钩子、爽点回收、语言密度和一个可模仿写法。
