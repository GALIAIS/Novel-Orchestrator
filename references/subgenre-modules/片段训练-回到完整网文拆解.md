# 片段训练-回到完整网文拆解

依据案例：`batch-097` 案例 482。

## 读者承诺

孤立片段能练一点描写，但学不到商业网文的系统承诺。描写、上行、爽点、情绪都依赖前后文对比。训练片段后，必须回到完整网文拆前因、行动、信息和结果。

## 核心循环

```text
片段练习 -> 发现白描问题 -> 找完整样本 -> 拆前因后果 -> 行动带信息 -> 回写正文
```

## 开篇必备

- 片段描写必须有行动，不只写外貌站位。
- 行动要释放信息。
- 训练后要找完整章节对照。
- 判断好坏要看是否服务书名、卖点、基调和前后期待。

## 期待来源

- 行动带来的即时画面。
- 信息释放带来的继续读动力。
- 前后文对比带来的上行情绪。
- 市场验证样本降低自嗨风险。

## 断档风险

- 沉迷给孤立片段打分。
- 只练白描，不写事件。
- 自己模拟练习无人评判。
- 把上行当独立片段，忽略铺垫。

## 训练任务

写一个 300 字对峙片段后，找一本完整网文中的对峙章节，拆出前因、冲突、行动、信息、结果。再重写自己的片段，要求每 100 字至少有一个动作推动信息。
