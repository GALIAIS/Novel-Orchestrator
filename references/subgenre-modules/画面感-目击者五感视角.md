# 画面感-目击者五感视角

依据案例：`batch-122` 案例 606。

## 读者承诺

画面感来自读者跟着某个目击者逐步看见，而不是作者站在高处把图片细节全报出来。有限视野、五感、误判和反应能让场景更像正在发生。

## 核心循环

```text
第一感官信号 -> 目击者误判 -> 细节逐步揭露 -> 身体反应 -> 认知翻转 -> 主线后果
```

## 开篇必备

- 固定一个目击者视角。
- 先写声音、触感、气味或震动。
- 让角色先误判，再看清。
- 描写只保留主线相关细节。
- 删除空转语气词和解释词。

## 期待来源

- 读者跟着目击者逐步拼出画面。
- 误判制造小悬念。
- 五感让怪物或场景更真实。
- 角色反应放大压迫感。

## 断档风险

- 上帝视角铺陈过多。
- 细节越写越满，读者无法想象。
- 描写和剧情无关。
- 翻译腔和闲聊式旁白压过事件。

## 训练任务

用一个路人视角写 500 字怪物登场，必须包含第一感官、误判、看清、身体反应、主线后果五项。
