# 观点辩论-前提定义与角色合理性

依据案例：`batch-257` 案例 1283。

## 读者承诺

观点辩论模块先锁定辩题定义和角色前提，再写理念碰撞，避免偷换概念造成智性不适。

## 核心循环

```text
定义辩题 -> 明确双方前提 -> 观点交锋 -> 暴露诡辩/差异 -> 角色理念留存
```

## 开篇必备

- 先分清讨论的是治国、气量、责任还是欲望。
- 角色立场必须来自人设。
- 诡辩可以存在，但文本要知道那是诡辩。
- 配角或结果要校正偷换概念。

## 期待来源

读者能理解角色为什么这么说，也能看清观点冲突真正争的是什么。

## 断档风险

- 国情、责任、气量混成一团。
- 作者把偷换概念当真理。
- 强角色只靠气势压人。
- 读者回头细想发现论证链断裂。

## 训练任务

写一场 800 字观点冲突：先列双方各 3 条前提，正文允许一方诡辩，但必须指出他偷换了哪个概念。
