# 亲近女性毒点-背景设定删雷

依据案例：batch-261 案例 1305

## 读者承诺

主角强关联女性不能用性/情感屈辱做廉价背景。母亲、养母、姐姐、女主这类人物一旦让读者产生被冒犯感，爽文代入会断。

## 核心循环

```text
亲近女性负面设定 -> 读者代入受损 -> 厌恶感上升 -> 爽感断裂 -> 改为无性化困境
```

## 开篇必备

- 关联人物与主角关系。
- 设定是否涉及被占便宜。
- 对主线是否必要。
- 可替代的困境。
- 最终删改决策。

## 期待来源

读者期待主角身份带来安全感和尊严感。亲近女性的苦难可以有，但不能写成羞辱主角代入的雷。

## 断档风险

这类毒点后续补丁难救；一句背景也足以让读者退出。

## 训练任务

扫一遍开篇亲属和女角色背景，凡涉及被占便宜、绿帽、强迫、性羞辱，全部改成债务、疾病、阶层压迫或仇恨来源。
