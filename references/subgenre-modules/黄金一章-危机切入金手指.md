# 黄金一章-危机切入金手指

依据案例：`batch-112` 案例 559。

## 读者承诺

新人开篇要尽快进状态。境界、前情、父亲关系、过多心理活动可以后移，第一章先用生死危机触发金手指，并用旁人反应放大爽点。

## 核心循环

```text
直接危机 -> 金手指触发 -> 主角脱险 -> 能力展示 -> 旁人反应
```

## 开篇必备

- 第一段进入危机。
- 前三段触发金手指。
- 一章内完成脱险。
- 返程或结尾展示收益。
- 后移境界和家族前情。

## 期待来源

- 读者马上看到危险。
- 金手指救场明确。
- 主角能力有外部反馈。
- 街坊邻居羡慕嫉妒恨放大爽点。

## 断档风险

- 第一章讲境界。
- 两章内容才进状态。
- 心理活动和打斗过多。
- 老虎搏斗等非核心事件抢篇幅。

## 训练任务

把两章压成一章：删掉境界说明和父亲前情，开头直接采药遇险，结尾写回村反馈。
