# 情绪拉扯-入门索引路径

依据案例：`batch-016` 案例 076。

## 读者承诺

新人学习情绪拉扯，先建立节奏、情绪折线、铺垫、爽点的索引路径，再进入具体案例。

## 核心循环

```text
基础疑问 -> 概念地图 -> 汇总帖/视频 -> 样本拆解 -> 最小练习
```

## 开篇必备

- 先分清情绪折线、铺垫、爽点、节奏。
- 不孤立背定义。
- 看一个视频案例。
- 拆一个同题材开篇。

## 期待来源

- 新人知道该从哪里学。
- 抽象概念能落到文本位置。
- 学完马上有最小练习。

## 断档风险

- 只问“怎么写”，不看体系入口。
- 看教程不拆样本。
- 会说术语，不会标注情绪位置。

## 训练任务

写出情绪拉扯 100 字定义，并在一个开篇中标注下行、期待、爽点三处。
