# 新人期-平台数据解读

依据案例：`batch-194` 案例 967。

## 读者承诺

新人期少量投资、推荐票、收藏波动都只是弱反馈，不能被当成作品崛起或失败的强证据。

## 核心循环

```text
早期发书 -> 出现小数据 -> 作者情绪波动 -> 判断反馈强弱 -> 回到更新和观察
```

## 开篇必备

- 记录当前字数和签约状态。
- 区分投资、推荐票、收藏、追读、签约等不同信号。
- 小样本不做大结论。
- 把数据反应用到下一步行动。

## 期待来源

作者获得平台数据感，不因一两个按钮互动过度兴奋，也不因短期没反馈立刻放弃。

## 断档风险

- 把投资当成作者收益。
- 把弱反馈误判成强市场信号。
- 用单日数据判断整本书。
- 不理解后台指标含义。

## 训练任务

列出新书前 3 万字的 10 个可能数据，分别标注弱反馈、过程信号、强信号，并写出对应动作。
