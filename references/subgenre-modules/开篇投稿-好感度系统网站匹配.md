# 开篇投稿-好感度系统网站匹配

依据案例：`batch-192` 案例 959。

## 读者承诺

低武灵气复苏加好感度系统，开篇完成度之外还要看风格、读者画像和网站匹配。

## 核心循环

```text
开篇结构 -> 风格判断 -> 网站匹配 -> 读者反馈 -> 桥段细修
```

## 开篇必备

- 剧情结构完整。
- 主角行动动机顺。
- 女角色好感变化可信。
- 目标平台读者能接受风格。

## 期待来源

作者能把完成度不错的开篇投到合适平台，并根据编辑和读者反馈改动细节。

## 断档风险

- 结构可用但平台错配。
- 英雄救美后一眼倒贴。
- 女角色能力和反应矛盾。
- 伏笔解释太晚，前期先显得逻辑不通。

## 训练任务

检查一个英雄救美桥段：女角色是否需要帮助、是否符合人设、好感为何产生、目标平台读者是否吃这一套。
