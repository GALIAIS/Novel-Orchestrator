# 武侠转译-旧味新节奏

依据案例：`batch-068` 案例 340。

## 读者承诺

这个模块用于武侠、武侠味玄幻、江湖感仙侠和权谋。它承诺的是保留侠义、江湖关系和人情锋芒，同时用现代网文的爽点尺度、人物效率和章节节奏重写。

## 核心循环

```text
旧武侠味 -> 现代题材容器 -> 爽点尺度升级 -> 人物更鲜明 -> 节奏更追读 -> 江湖感复活
```

## 开篇必备

- 江湖关系要进入事件。
- 打斗要有情绪和胜负代价。
- 主角目标不能只靠行侠情怀。
- 章节节奏符合现代追读。
- 武侠味要转成爽点或悬念。

## 期待来源

- 侠义选择带来道德张力。
- 江湖人情带来关系网。
- 升级后的战力尺度带来爽感。
- 门派、恩怨、名声可转成长线期待。
- 旧味放进新题材降低过时风险。

## 断档风险

- 只剩情怀，没有商业承诺。
- 打斗干涩，没有画面和情绪。
- 人物刻板。
- 节奏停在旧式连载。
- 武侠脑洞没有升级。

## 训练任务

把一个“少侠替人出头”的旧桥段改成现代网文版本。要求保留侠义选择，但加入更强代价、见证反应和下一章钩子。
