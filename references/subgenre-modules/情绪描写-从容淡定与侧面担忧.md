# 情绪描写-从容淡定与侧面担忧

依据案例：`batch-244` 案例 1220。

## 读者承诺

主角从容淡定不能靠重复描述堆出来，要靠外部压力、配角担忧和反派不解来衬托。

## 核心循环

```text
建立威胁 -> 配角担忧 -> 反派不解 -> 主角少量反应 -> 结果证明底气
```

## 开篇必备

- 小 boss 有压迫力。
- 配角不知道主角底牌。
- 反派误判主角自信过头。
- 主角直接描写克制。

## 期待来源

局面越值得紧张，主角越少反应，读者越期待他凭什么这么稳。

## 断档风险

- 同一情绪反复写，读者烦。
- 主角太早消解威胁。
- 没有信息差。
- 从容没有后续结果证明。

## 训练任务

写一个主角从容面对小 boss 的场景，主角直接描写不超过 3 句，其余用配角和反派反应衬托。
