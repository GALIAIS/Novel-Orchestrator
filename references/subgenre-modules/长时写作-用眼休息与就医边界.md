# 长时写作-用眼休息与就医边界

依据案例：batch-383 案例 1914

## 读者承诺

长时写作要预设用眼休息，而不是难受后才停。酸涩、异物感持续、加重或疑似发炎时，应接受专业检查。

## 核心循环

```text
定时用眼 -> 主动休息 -> 记录症状 -> 必要时就医 -> 调整工作环境
```

## 开篇必备

- 三十至四十分钟休息提醒。
- 远眺、闭眼或起身动作。
- 睡眠和症状记录。
- 清楚的就医触发条件。

## 期待来源

- 疲劳信号被及早处理。
- 写作节奏更可持续。
- 工作姿势和环境得到调整。
- 专业检查替代自行猜测。

## 断档风险

- 把不适归咎于意志薄弱。
- 揉眼、眯眼或强撑。
- 只凭帖子自行用药。
- 症状持续仍不检查。

## 训练任务

连续三天执行三十至四十分钟用眼加短休息并记录症状；持续、加重或疑似发炎时停止自行试药并就医。
