# 飞卢大事件-拉情绪装逼

依据案例：`batch-132` 案例 657。

## 读者承诺

用连续大事件快速拉起情绪，让主角通过金手指变化和压场装逼持续兑现爽点。成长线服务装逼结果，不拖成慢铺垫。

## 核心循环

```text
新危机 -> 大事件入场 -> 金手指变化 -> 主角压场装逼 -> 下一件更大的事
```

## 开篇必备

- 第一时间给出可期待的大事件。
- 明确本次装逼对象和见证者。
- 金手指变化要带来可见结果。
- 事件结束后立刻挂下一件事。

## 期待来源

- 大事件密度高。
- 读者马上看到主角变强。
- 围观者反应提供情绪放大。
- 爆点单元可以连续串联。

## 断档风险

- 为了铺成长拖慢事件。
- 金手指变化没有装逼结果。
- 大事件彼此孤立，无法接成长篇。
- 情绪拉起后没有下一钩子。

## 训练任务

设计 3 个连续大事件，每个写清危机、金手指变化、压场结果和下一事件钩子。
