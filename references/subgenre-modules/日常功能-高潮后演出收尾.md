# 日常功能-高潮后演出收尾

依据案例：`batch-093` 案例 465。

## 读者承诺

日常不是把战况再变紧张，而是在高潮后清点战果、释放情绪、补人设、铺伏笔、给金手指，为下一次激励事件做准备。

## 核心循环

```text
高潮结束 -> 清点战果 -> 震惊欢呼 -> 人设互动 -> 伏笔/道具准备 -> 过渡新地点 -> 激励事件
```

## 开篇必备

- 放在冲突结束后。
- 先确认战果和收获。
- 让配角反应提供情绪回报。
- 日常里混入笑点、甜点或泪点。
- 最后过渡到下一目标。

## 期待来源

- 读者确认上一场爽点价值。
- 角色关系变得更饱满。
- 新道具或能力埋下期待。
- 新舞台自然出现。

## 断档风险

- 用战况急转直下冒充日常。
- 处处爽导致没有爽点。
- 日常无功能，只是闲聊。
- 没有过渡到下一剧情。

## 训练任务

写 1500 字高潮后日常：战果清点、他人反应、人设互动、一个伏笔、一个下个剧情会用的道具或能力。
