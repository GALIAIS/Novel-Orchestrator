# 轻小说后宫-卖点突出壳子弱化

依据案例：`batch-079` 案例 395。

## 读者承诺

这个模块用于判断轻小说/后宫文里哪些东西该重、哪些可以轻。它承诺让卖点压过外壳：读者追的是人设互动和关系反馈时，修仙、玄幻等外层设定可以只承担舞台功能。

## 核心循环

```text
外层题材壳 -> 核心人设卖点 -> 关系反馈 -> 修罗场/占有欲升级 -> 下一轮互动期待
```

## 开篇必备

- 前三章亮出核心关系卖点。
- 主角身份和后宫角色的关系要清楚。
- 至少一次角色反馈证明标签有效。
- 外层设定只交代行动必要信息。
- 判断读者是追升级还是追关系。

## 期待来源

- 病娇、师妹、反派大师兄等标签立即兑现。
- 多角色围绕主角形成关系压力。
- 修仙事件提供见面、冲突和误会舞台。
- 角色反应比设定说明更密。
- 每章留下下一次互动钩子。

## 断档风险

- 壳子写太重，卖点出场慢。
- 后宫角色只有标签，没有行为反馈。
- 读者期待修仙升级，正文却只写关系。
- 关系互动重复，没有升级。
- 世界设定弱到无法提供事件舞台。

## 训练任务

写一个“修仙外壳 + 后宫卖点”的前三章节拍表。每章必须有一次人设反馈、一次关系压力、一个下一章互动钩子；修炼设定只写必要舞台。
