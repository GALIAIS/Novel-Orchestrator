# 现实辩论-立场论据要害对话

依据案例：`batch-121` 案例 601。

## 读者承诺

现实强对话能训练网文角色交锋。好用的不是长篇说理，而是立场先行、证据加码、抓住对方要害、被打断后反夺主导权。

## 核心循环

```text
亮明立场 -> 抛出事实证据 -> 身份/代表性加码 -> 对方打断 -> 反驳定性 -> 换位到对方利益 -> 给解决方案
```

## 开篇必备

- 角色必须有不可退的立场。
- 每句论据都要压中对方责任或利益。
- 打断不是废话，要改变主导权。
- 反驳后要马上给解决方向。
- 对话结尾要有情绪或权力变化。

## 期待来源

- 读者等弱势方能否站住脚。
- 证据层层加码提升压迫。
- 对方打断制造即时冲突。
- 反驳成功带来主导权爽点。

## 断档风险

- 照搬现实长篇发言，剧情停滞。
- 只有观点，没有对方要害。
- 对话双方不改变局势。
- 角色说理脱离身份和当前目标。

## 训练任务

拆一段现实强对话，提取 5 个主导权节点。再写成网文争执场景，要求每个节点都让一方更接近目标。
