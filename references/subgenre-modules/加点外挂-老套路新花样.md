# 加点外挂-老套路新花样

依据案例：batch-289 案例 1445

## 读者承诺

加点外挂是成熟老套路，新人要用资源来源、代价、融合效果和第一章冲突展示新花样。

## 核心循环

```text
老加点能力 -> 新花样 -> 快速撞冲突 -> 主角主动选择 -> 加点解决问题 -> 新期待
```

## 开篇必备

- 基础加点能力。
- 与旧套路的差异。
- 加点资源来源。
- 加点代价。
- 第一章冲突。
- 主角锋芒。
- 简介里的外挂作用。

## 期待来源

读者看到熟悉爽点后，又发现这个外挂有新的代价、玩法或融合效果。

## 断档风险

- 只写属性点和武学融合，没有新意。
- 第一章买药拖节奏。
- 主角谨慎写成怂包。
- 简介写空泛选择，不展示外挂作用。

## 训练任务

给一个普通加点系统加三条新花样，并写出第一章如何用它解决冲突。
