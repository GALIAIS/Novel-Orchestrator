# 教程排期-冲突大纲卖点题材

依据案例：`batch-118` 案例 587。

## 读者承诺

系统学习可以先分成四条主线：冲突、全书大纲、卖点亮点、题材演化。每次只选当前最卡的一条，避免同时追所有教程。

## 核心循环

```text
定位问题 -> 归入四类 -> 选择教程 -> 拆一个样本 -> 做一个训练 -> 复盘
```

## 开篇必备

- 冲突解决单元推进。
- 大纲解决长线布局。
- 卖点解决读者入口。
- 题材演化解决方向判断。
- 当前只选一个主问题。

## 期待来源

- 学习路线不再零散。
- 教程与写作问题对应。
- 每个方向都有训练产物。
- 不靠收藏资料制造进步幻觉。

## 断档风险

- 四个方向同时学。
- 只看教程不做训练。
- 用大纲问题掩盖卖点问题。
- 用题材演化替代当前章节修复。

## 训练任务

把当前问题写成一句话，归入冲突、大纲、卖点、题材四类之一。只做一个对应训练，完成后再换方向。
