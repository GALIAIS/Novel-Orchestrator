# 主角责任危机-作死避坑

依据案例：`batch-159` 案例 791。

## 读者承诺

让主角犯错引发危机时，读者感受到的是压力、代价和成长，而不是作者强行让主角降智推动剧情。

## 核心循环

```text
主角选择 -> 外部压力/信息差限制 -> 危机扩大 -> 主角付代价 -> 主角补救 -> 责任转化为成长
```

## 开篇必备

- 先交代主角为什么只能这样选。
- 给出足够强的外部压力、情感压力或信息差。
- 让危机后果真实影响主角和身边人。
- 避免让更合理的替代方案明晃晃摆在读者眼前。

## 期待来源

读者期待主角不是永远正确，而是在错误后承担代价、补救局面，并获得更成熟的判断力。

## 断档风险

- 危机全靠主角低智触发。
- 主角牵连无辜者却没有心理和现实代价。
- 反派、规则、环境都不强，只靠主角作死制造剧情。
- 影视 IP 的情怀补偿被误当成小说也能用的结构补偿。

## 训练任务

写一个 1000 字危机场景，要求主角必须犯错，但读者能理解他为什么会选错。写完后列出三项：当时信息差、外部压力、事后代价。
