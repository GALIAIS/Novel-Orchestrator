# 单招吃透-期待循环实战

依据案例：`batch-003` 案例 012。

## 读者承诺

新人先把一个有效动作练熟：拉期待、满足期待、再拉期待。先让作品能跑，再逐步扩展更多技巧。

## 核心循环

```text
选一招 -> 拆样本验证 -> 连续章节复用 -> 追读反馈 -> 再扩展第二招
```

## 开篇必备

- 一个明确的信息差。
- 一个明确的反差。
- 一个本章满足点。
- 一个满足前埋下的新期待。

## 期待来源

- 信息差让读者等真相揭开。
- 反差让读者等误判翻转。
- 兑现后立刻补新期待，避免爽完即走。

## 断档风险

- 同时套太多理论，导致章节不成形。
- 只收藏方法，不写作验证。
- 每章只有满足，没有下一轮期待。
- 期待方式重复但没有换人物、场景、层级。

## 训练任务

连续 5 章只检查“信息差”和“反差”。每章写完后，用一句话标注下一章读者为什么还要看。
