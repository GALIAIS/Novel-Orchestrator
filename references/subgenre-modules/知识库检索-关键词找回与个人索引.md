# 知识库检索-关键词找回与个人索引

依据案例：`batch-211` 案例 1051。

## 读者承诺

写作方法帖要能被找回，不能只靠看过后的模糊印象。

## 核心循环

```text
方法帖 -> 关键词 -> 收藏/索引 -> 问题场景 -> 搜索复用
```

## 开篇必备

- 记录原文核心短语。
- 给方法帖贴问题标签。
- 标注适用章节场景。
- 保留平台搜索路径。

## 期待来源

创作者在写到具体问题时，能快速回到对应方法，而不是重新求助。

## 断档风险

- 看过不收藏。
- 只记大概意思，不记关键词。
- 个人知识库没有分类。
- 需要复用时找不到原帖。

## 训练任务

整理 10 条写作方法帖索引：标题、3 个关键词、解决问题、适用章节、复用路径。
