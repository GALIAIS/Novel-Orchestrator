# 免费平台-作者收益不可见风险

依据案例：`batch-333` 案例 1661

## 读者承诺

免费平台会把作者收入从读者订阅转成平台/资本分配，核心风险是收益不可见和议价能力弱。

## 核心循环

```text
免费阅读 -> 平台拿流量/广告 -> 平台分配稿费 -> 作者缺少透明凭据 -> 权益风险上升
```

## 开篇必备

- 分清收费订阅和免费分配。
- 看清结算口径。
- 确认数据是否可查。
- 判断平台扣款空间。
- 评估自己议价能力。

## 期待来源

作者选择平台时，不只看短期能不能赚钱，也看长期收益是否可追踪。

## 断档风险

- 以为免费平台必然对作者更好。
- 收入路径不可查询。
- 小作者完全看平台脸色。
- 只看到眼前稿费，忽略长期黑箱。
- 不看合同和结算规则。

## 训练任务

对比一个收费平台和一个免费平台：收入来源、可查数据、分成、结算、扣款空间、申诉渠道。
