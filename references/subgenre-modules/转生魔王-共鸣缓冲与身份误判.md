# 转生魔王-共鸣缓冲与身份误判

依据案例：`batch-118` 案例 590。

## 读者承诺

转生魔王想写不公平伤害，必须先处理“魔王通常该被讨伐”的读者默认印象。先证明主角善良或被误判，再写受害，否则共鸣会断。

## 核心循环

```text
高风险身份 -> 证明善良/无辜 -> 遭遇不公平伤害 -> 转生低位 -> 缓冲身份变化 -> 建立新目标
```

## 开篇必备

- 不直接要求读者同情魔王。
- 先给善良、正义或被误判证据。
- 不公平伤害要有前因后果。
- 转生后身份变化要有缓冲。
- 情绪转折不能生硬。

## 期待来源

- 读者理解主角为何值得同情。
- 身份反差成立。
- 低位受辱不再像自作自受。
- 后续目标建立在共鸣上。

## 断档风险

- 魔王被砍却说是不公平。
- 只按农民身份理解主角，忽略魔王前史。
- 强冲突缺少铺垫。
- 中期才说是游戏，导致读者觉得白担心。

## 训练任务

写一个魔王转生开篇，先用 500 字证明魔王没有作恶或被误判，再让他被伤害。检查读者是否会觉得“他活该”。
