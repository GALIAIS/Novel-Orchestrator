# 打斗资料-关键词检索与素材找回

依据案例：batch-374 案例 1869

## 读者承诺

打斗资料要能被检索、找回和转写。链接收藏只是入口，素材卡才是可复用成果。

## 核心循环

```text
写作卡点 -> 关键词检索 -> 找回资料 -> 标注用途 -> 转成打斗素材卡
```

## 开篇必备

- 卡点类型。
- 检索关键词。
- 资料来源。
- 可套用写法。

## 期待来源

- 动作更清楚。
- 镜头顺序更顺。
- 力道反馈更具体。
- 情绪节奏更有起伏。

## 断档风险

- 只存链接不整理。
- 卡住后重复从零找。
- 资料和实际场景无关。
- 不区分动作、镜头、反馈、情绪。

## 训练任务

整理 5 张打斗素材卡，每张只保留一个可立即套用的动作或镜头写法。
