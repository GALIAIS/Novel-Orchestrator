# 新人避坑-情节下行诊断入口

依据案例：`batch-224` 案例 1120。

## 读者承诺

把“新人为什么扑街”压缩成可排查的情节下行问题，再用情绪拉扯修复期待。

## 核心循环

```text
新人扑街 -> 找下行位置 -> 保住主角形象 -> 改成情绪拉扯 -> 增加期待
```

## 开篇必备

- 标出让读者失望的位置。
- 判断主角形象是否受损。
- 把挫折改成外部压力或期待延迟。
- 让下行之后仍有下一章吸引力。

## 期待来源

新人需要一个诊断入口：不是笼统说不好看，而是判断剧情是否让期待、爽感、主角魅力持续下降。

## 断档风险

- 把视频入口当成完整教程。
- 附件不可读却引用附件细节。
- 只制造挫折，不制造反弹期待。
- 下行直接损害主角能力和人设。

## 训练任务

拿一段新人开篇大纲，标出三处可能让情绪下行的位置；每处改写成挫折后仍增加期待的版本，并说明主角形象如何不受损。
