# 影视拆书-支线主角推动主线

依据案例：`batch-189` 案例 943。

## 读者承诺

拆影视主支线时，把支线人物当成支线主角，看他自己的目标如何通过结果或影响推动主线。

## 核心循环

```text
主线节点 -> 切入支线 -> 支线人物推进目标 -> 产生影响 -> 推动主线
```

## 开篇必备

- 主线目标明确。
- 支线人物有独立目标。
- 支线结果对主线有影响。
- 视角切换发生在合适主线节点。

## 期待来源

读者看到每条支线都不是闲笔，而是在不同角度推进主线、揭示悬念或改变局面。

## 断档风险

- 支线人物只是主角附庸。
- 支线只补设定，不改变主线。
- 视角切换随意，叙事脉络断裂。
- 身份悬念揭开却不推动事件。

## 训练任务

拆一部电影，列主线和 2 条支线；每条支线写出“支线主角目标、切入节点、结果、对主线影响”四项。
