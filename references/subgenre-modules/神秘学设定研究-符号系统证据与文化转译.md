# 神秘学设定研究-符号系统证据与文化转译

依据案例：`batch-413` 案例 2064

## 读者承诺

符文、字母、数字和仪式系统具有可追溯的来源层级、清楚的不确定性和明确的小说改编边界。不同传统先独立建档，再从原始材料、学术研究、现代实践和流行再造中提取可核验证据；虚构效果、触发、代价与误用另栏设计，并通过解释权、版本控制和组织利益进入剧情，而不是把现实传统混成装饰性魔法词库。

## 核心循环

```text
拆分独立文化与资料域 -> 给来源分级并记录语言、版本、页码和作者资历 -> 区分资料主张与证据支持 -> 保存争议和不确定性 -> 提取符号形态与历史用途 -> 另栏设计虚构触发、效果、代价和误用 -> 分配解释权与版本控制 -> 规则改变人物利益和组织制度 -> 剧情结果反向暴露新的来源问题
```

研究循环的完成标准不是资料数量，而是每项采用内容都能追溯，每项原创改编都有标记，每条符号规则都能迫使人物作出有代价的选择。

## 开篇必备

- 独立传统的名称、语言、时代和地域边界。
- 至少一项原始材料或可靠译本线索。
- 一项注明引文、方法和争议的学术研究。
- 现代实践和流行再造与历史层的分栏。
- 作者、出版机构、版次、页码和资历记录。
- “资料声称”与“证据支持”两个字段。
- 虚构规则的触发、效果、成本、禁忌和误用。
- 掌握解释权的具体人物或组织。
- 现实借鉴与原创部分的显式标识。

## 期待来源

- 不同版本为何对同一符号给出冲突解释。
- 谁从某种解释中获利，谁试图压制其他版本。
- 一次仪式失败究竟源于误译、伪造还是规则代价。
- 流行解释掩盖了哪项原始证据。
- 主角能否证明某个权威版本并不完整。
- 符号系统的使用成本如何限制强者。
- 改写规则会触发何种宗教、政治或身份后果。

## 断档风险

- 凭商品封面、标题或二手帖子补写历史结论。
- 把不同传统默认成同源、同义和互通。
- 不记录语言、版本和页码，后续无法追溯。
- 用确定语气隐藏学界争议和资料空白。
- 把现代实践说成统一的古代事实。
- 虚构规则与现实借鉴不分栏。
- 只有符号名词，没有触发、成本和剧情后果。
- 所有人对符号拥有同等解释权，世界缺少制度。
- 现实文化只被用作邪恶或异域装饰。

## 训练任务

选择两个不同传统，分别建立三层证据卡：原始或可靠译本、学术研究、流行再造。每张卡填写语言、版本、页码、主张、证据和争议，禁止合并传统。随后只取一个符号另栏设计虚构规则，并写一场 1500 字冲突，让误译、使用代价和解释权同时改变人物选择。
