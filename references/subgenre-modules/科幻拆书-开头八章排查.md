# 科幻拆书-开头八章排查

依据案例：`batch-118` 案例 588。

## 读者承诺

科幻开篇拆书要结合原文前八章，检查钩子、文风、设定事件化和开头排查表元素。榜单样本的价值在于证明同题材读者如何被留下。

## 核心循环

```text
扫榜找样本 -> 拆前八章 -> 记录每章钩子 -> 对照开头排查表 -> 提炼仿写动作
```

## 开篇必备

- 选同题材榜单样本。
- 限定前八章范围。
- 每章记录情节钩子。
- 检查开头排查表元素。
- 观察科幻设定如何进入事件。

## 期待来源

- 榜单成绩提供验证。
- 钩子让读者一章章追。
- 轻松文风降低设定门槛。
- 科幻征文有可拆样本。

## 断档风险

- 脱离原文看拆书结论。
- 只拆设定，不拆钩子。
- 学科幻概念，不学开篇留人。
- 样本和目标平台不匹配。

## 训练任务

拆一本同题材前八章，每章只写两个字段：钩子是什么、满足了开头排查表哪一项。最后仿写一个开篇钩子。
