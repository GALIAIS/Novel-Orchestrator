# 新站投稿-买断征文与分成单机风险

依据案例：`batch-204` 案例 1017。

## 读者承诺

新站人少时，除非有买断或征文把握，否则分成发书容易单机。

## 核心循环

```text
新站机会 -> 流量小 -> 买断/征文评估 -> 有把握才投 -> 无保障不分成单机
```

## 开篇必备

- 先评估读者规模。
- 确认是否买断。
- 看征文是否有把握。
- 算同稿投大站的机会成本。

## 期待来源

作者希望新站给机会，但真正值得投入的是有收益保障或明确活动机会的平台。

## 断档风险

- 被新站机会感吸引。
- 分成模式没有读者池支撑。
- 没有买断仍押主力稿。
- 征文没把握还投入长篇。

## 训练任务

做新站投稿风险表：买断、征文、流量、编辑反馈、分成预期、机会成本六项评级。
