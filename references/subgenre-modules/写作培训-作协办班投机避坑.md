# 写作培训-作协办班投机避坑

依据案例：`batch-100` 案例 497。

## 读者承诺

帮助新人把“学习写作”从迷信头衔改成核验证据：能学的是真实作品、真实经验、可复盘案例和清晰方法；要避开的是靠作协、活动、资源、人脉包装自己，却缺少作品和从业经验的收费投机者。

## 核心循环

```text
新人焦虑 -> 权威包装出现 -> 核验作品/经验 -> 拆穿承诺过度 -> 保留可学经验或直接避坑
```

## 开篇必备

- 先问对方有没有公开作品。
- 再问经验是否能被核验。
- 警惕“保证提升”“资源背书”“内部关系”。
- 只把课程当经验来源，不把它当成绩保证。

## 期待来源

- 新人想少走弯路。
- 老作者可提供套路和状态启动经验。
- 可核验案例能降低学习风险。
- 避坑本身能节省时间和钱。

## 断档风险

- 把作协/鲁院/网作协名头当成写作能力证明。
- 看到活动资源就以为有稳定提升路径。
- 付费前没有检查对方作品。
- 被概念包装牵走，忘记回到同题材文本。

## 训练任务

任选一个写作课程介绍，填“作品、经验、案例、承诺、收费导向、避坑信号”六格。若公开作品和真实案例两项为空，直接判为不适合作为主要学习对象。
