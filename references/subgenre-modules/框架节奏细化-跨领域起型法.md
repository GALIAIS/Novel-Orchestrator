# 框架节奏细化-跨领域起型法

依据案例：`batch-046` 案例 229。

## 读者承诺

非高手写长篇，要像画画先起型一样先搭框架，再修节奏，最后细化。沉迷局部灵感会让整体越来越散。

## 核心循环

```text
整体框架 -> 节奏修正 -> 单元细化 -> 稳定发挥 -> 长期打磨
```

## 开篇必备

- 先有整体框架。
- 第一卷目标清楚。
- 节奏问题提前暴露。
- 只细化当前单元。

## 期待来源

- 作者知道每卷要完成什么。
- 中后期不轻易崩。
- 每卷发挥稳定。
- 局部灵感能回到整体。

## 断档风险

- 目的性太强导致急躁。
- 沉迷某个好桥段。
- 没有大纲直接写长篇。
- 框架问题拖到后期才暴露。

## 训练任务

用“起型-修型-定型”重做一个大纲：整体框架、节奏问题、第一单元细化三步分开写。
