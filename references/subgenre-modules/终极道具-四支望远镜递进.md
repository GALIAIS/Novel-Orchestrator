# 终极道具-四支望远镜递进

依据案例：batch-294 案例 1468

## 读者承诺

终极道具要通过传言、前人失败、位置确认、最后门槛逐层递望远镜，再由主角成为唯一解。

## 核心循环

```text
大危机 -> 道具传言 -> 前人失败 -> 位置确认 -> 门槛失效 -> 主角破局
```

## 开篇必备

- 终极目标。
- 时间压力。
- 道具唯一价值。
- 前人失败证据。
- 最后门槛。
- 主角金手指解法。

## 期待来源

读者不断看到希望更重要、更难得、更接近，又在最后一刻断掉，等待主角补上唯一解。

## 断档风险

- 只说神器重要，不增加难度。
- 前人失败证据不足。
- 没有最后门槛。
- 主角破局和金手指无关。

## 训练任务

设计四支望远镜：传言、前人失败、位置确认、钥匙失效，最后用金手指修复。
