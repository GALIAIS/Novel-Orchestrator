# 神话原型-工具书样本回读

依据案例：batch-275 案例 1373

## 读者承诺

把《千面英雄》这类抽象理论书转成可操作工具：先用熟悉影视和小说样本理解英雄成长、挑战和角色功能，再回读理论。

## 核心循环

```text
理论概念 -> 具体样本 -> 拆人物/挑战/功能 -> 回读理论 -> 网文化节点
```

## 开篇必备

- 不直接背概念，先找熟悉故事。
- 拆主角起点、挑战、导师、敌人和变化。
- 最终转成卷级或章节节点。

## 期待来源

作者期待从“看不懂神话故事”转成“我知道这个节点如何服务主角成长”。理论必须落回桥段。

## 断档风险

- 把理论当玄学。
- 没有样本积累，硬啃抽象概念。
- 只记英雄旅程名称，不会改写成网文节奏。

## 训练任务

用一部熟悉电影拆七个英雄旅程节点，再把每个节点改写成网文一卷中的事件。
