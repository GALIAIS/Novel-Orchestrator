# 穿越身份-行为暗示与一句确认

依据案例：batch-382 案例 1906

## 读者承诺

穿越身份可以先用言行暗示，但最好用一句自然确认消除歧义。身份信息要嵌进当前动作，并立刻服务信息差和剧情。

## 核心循环

```text
场景动作 -> 异常习惯/判断 -> 一句确认穿越 -> 信息差起效
```

## 开篇必备

- 当前场景目标。
- 来自前世的习惯或知识。
- 不绕口的确认句。
- 第一次信息差作用。

## 期待来源

- 读者先发现反应异常。
- 简短确认建立共同前提。
- 前世知识参与当前行动。
- 信息差后续还能升级。

## 断档风险

- 只暗示导致读者误判。
- 用“蓝色星球”式陈词滥调。
- 为简单信息制造无意义悬念。
- 交代后不产生剧情作用。

## 训练任务

写 500 字开篇：先用行为暗示，随后一句确认穿越，最后让前世信息帮助或误导一次行动。
