# 摸鱼系统-反向迪化与后期续航

依据案例：`batch-214` 案例 1067。

## 读者承诺

摸鱼系统的爽点是主角越想偷懒，世界越误判他深谋远虑，并把收益推给他。

## 核心循环

```text
逃避痛苦 -> 系统误判 -> 摸鱼行为 -> 外界迪化 -> 收益扩大
```

## 开篇必备

- 职场痛苦足够共鸣。
- 系统逻辑荒诞但清楚。
- 摸鱼行为和成功结果形成反差。
- 外界误判持续升级。

## 期待来源

读者既爽于反加班共鸣，也爽于所有人把主角偷懒解读成高明策略。

## 断档风险

- 只靠一个梗撑全书。
- 后期场景重复。
- 主角没有真实选择。
- 迪化误判没有新规则支撑。

## 训练任务

设计 10 个“摸鱼反而成功”事件，分别写清偷懒点、系统误判、外界迪化、实际收益和升级方式。
