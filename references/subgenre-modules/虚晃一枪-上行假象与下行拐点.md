# 虚晃一枪-上行假象与下行拐点

依据案例：`batch-250` 案例 1248。

## 读者承诺

虚晃一枪模块用情绪上行制造解决假象，再让同一铺垫在拐点处引发下行恶化。

## 核心循环

```text
上行希望 -> 平安铺垫 -> 拐点触发 -> 意外恶化 -> 情绪拉扯增强
```

## 开篇必备

- 先让读者相信事情会变好。
- 平安证据要具体。
- 意外必须从前文铺垫里来。
- 下行后要保留后续补偿期待。

## 期待来源

读者刚放松时被抽走安全感，会产生“差一点就好了”的刺痛，比单纯坏事更有拉扯。

## 断档风险

- 没有上行假象，虚晃变成普通事故。
- 意外没有因果，读者觉得作者强行捣乱。
- 下行过重，后续补偿不明。
- 为反转而反转，忘了服务情绪。

## 训练任务

写一个短桥段：前半段给 3 个安全信号，后半段让其中 1 个安全信号反过来制造危机。
