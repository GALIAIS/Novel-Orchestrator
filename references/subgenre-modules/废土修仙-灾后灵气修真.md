# 废土修仙-灾后灵气修真

依据案例：`batch-132` 案例 660。

## 读者承诺

把废土的生存压力和修仙的境界成长绑在一起：资源稀缺、秩序崩塌、污染灵气和遗迹争夺共同推动主角升级。

## 核心循环

```text
灾后生存压力 -> 发现稀缺修炼资源 -> 争夺/净化资源 -> 境界提升 -> 改变生存处境
```

## 开篇必备

- 说明灾变如何影响灵气。
- 给主角一个现实生存危机。
- 第一份修行资源必须稀缺且有代价。
- 废土组织或敌人参与争夺。
- 升级结果要解决生存问题。

## 期待来源

- 修仙成长提供上行希望。
- 废土环境持续压低安全感。
- 资源争夺天然制造冲突。
- 旧宗门、遗迹、黑市带来探索感。

## 断档风险

- 废土只是背景皮。
- 修仙体系照搬，没有污染/稀缺改造。
- 升级和生存分成两条线。
- 没有代表作验证就直接长篇投入。

## 训练任务

写 300 字开篇设定：灾变来源、资源稀缺原因、主角第一个修行机会。要求升级机会同时解决一个废土生存危机。
