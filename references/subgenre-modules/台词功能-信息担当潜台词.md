# 台词功能-信息担当潜台词

依据案例：batch-286 案例 1429

## 读者承诺

台词要为内容服务，最好同时承担信息和情绪、氛围或人设功能，并用潜台词压缩表达。

## 核心循环

```text
情节内容 -> 信息担当 -> 叠加情绪/人设 -> 潜台词表达 -> 推动行动
```

## 开篇必备

- 这句台词要交代的信息。
- 附带功能。
- 说话人的立场。
- 不能直说的内容。
- 对话后的行动变化。

## 期待来源

读者在对话里获得信息、感受关系变化，并期待下一步行动。

## 断档风险

- 无病呻吟。
- 只表现人设，不承担内容。
- 台词脱离情节线。
- 直白说破，缺少潜台词。

## 训练任务

把 5 句直白台词改成潜台词版本，每句至少承担两个功能，其中一个必须是信息。
