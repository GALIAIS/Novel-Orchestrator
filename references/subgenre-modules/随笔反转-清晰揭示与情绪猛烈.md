# 随笔反转-清晰揭示与情绪猛烈

依据案例：batch-367 案例 1835

## 读者承诺

网文反转要让读者当场看懂真相和意义；隐晦可以有，但关键因果、选择和奖励不能藏到只剩困惑。

## 核心循环

```text
表层任务 -> 隐藏测试 -> 关键选择 -> 反转揭示 -> 奖励认可
```

## 开篇必备

- 主角明确目标。
- 表层任务有压力。
- 隐藏测试有可见线索。
- 结尾奖励即时兑现。

## 期待来源

- 主角受考验。
- 读者等待真相。
- 关键选择证明人设。
- 峰回路转得到认可。

## 断档风险

- 怕太明显而藏太深。
- 读者看不懂结尾因果。
- 反转后没有奖励。
- 情绪不够猛烈，只剩文笔流畅。

## 训练任务

把一个隐晦反转改成网文版：埋 3 个清晰线索，结尾一句话揭示真相并立刻给奖励。
