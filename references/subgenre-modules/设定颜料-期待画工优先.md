# 设定颜料-期待画工优先

依据案例：`batch-120` 案例 597。

## 读者承诺

设定是颜料，期待结构是画工。读者不是因为世界观足够炫才留下，而是因为铺垫、误判、阻力、兑现形成了好看的情绪弧线。

## 核心循环

```text
起点 -> 铺垫期待 -> 阻力/误判 -> 小兑现 -> 新阻力 -> 大兑现
```

## 开篇必备

- 先写主角起点和目标。
- 设定只服务当前期待。
- 至少有一次阻力改变路径。
- 爽点前要有铺垫。
- 结尾要留下下一段弧线。

## 期待来源

- 读者想知道主角如何到达目标。
- 误判和转折让路径不再直线。
- 小兑现提供持续反馈。
- 大兑现回收前面铺垫。

## 断档风险

- 只堆金手指、背景、人设和升级体系。
- 两点一线直达目标。
- 转折是硬加的，不改变情绪。
- 玩梗和搞笑替代剧情张弛。

## 训练任务

选一个已有设定，写出“起点、目标、阻力、小兑现、再阻力、大兑现”六格。每格只写事件，不新增设定名词。
