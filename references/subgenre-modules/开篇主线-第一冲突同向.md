# 开篇主线-第一冲突同向

依据案例：`batch-023` 案例 114。

## 读者承诺

第一章给出的最大急迫目标，第二章第一冲突必须继续推进或阻断它。

## 核心循环

```text
急目标 -> 主角行动 -> 反派挡路 -> 金手指主动证明 -> 目标推进 -> 新期待
```

## 开篇必备

- 主角最急目标。
- 行动路径。
- 反派挡路点。
- 金手指主动用途。
- 新造设定的一句话爽点解释。

## 期待来源

- 读者看到主角没有忘记急事。
- 反派压迫和主线目标同向。
- 金手指在事件中证明价值。
- 章尾留下下一步行动。

## 断档风险

- 第一章救妹妹，第二章去管无关英雄救美。
- 需要大量解释才能证明主角动机。
- 金手指只是被动救命，没有主动效果。
- 新造概念太多，读者不知道看点是什么。

## 训练任务

把一个第二章事件改写成“反派直接挡在主角急目标上”。如果原事件不能改，就把它挪到主线目标稳定之后。
