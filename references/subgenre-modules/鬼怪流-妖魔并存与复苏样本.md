# 鬼怪流-妖魔并存与复苏样本

依据案例：`batch-325` 案例 1623

## 读者承诺

鬼怪流大致是妖魔鬼怪与人类共同存在，主角在异常事件、鬼怪规则和生存代价中持续处理危机。

## 核心循环

```text
鬼怪真实存在 -> 异常事件爆发 -> 规则/代价揭露 -> 主角应对 -> 更大鬼怪秩序浮出
```

## 开篇必备

- 第一章出现具体异常事件。
- 鬼怪有害人规则或侵蚀方式。
- 主角有初始应对能力或信息优势。
- 解决事件要付出代价。
- 事件背后露出更大的鬼怪体系。

## 期待来源

读者期待未知压迫、规则破解、生存代价和主角处理鬼怪事件的能力成长。

## 断档风险

- 只写吓人氛围，没有事件循环。
- 鬼怪没有规则，危机随意。
- 主角缺少应对方式，只能被动逃。
- 开篇先讲世界观，不先给异常。
- 只模糊说灵异，不拆代表书前三章。

## 训练任务

按《神秘复苏》《这里有妖气》的方向，设计三章：异常、规则、代价回收，每章一个章末钩子。
