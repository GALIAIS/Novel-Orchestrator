# 生存游戏智斗-规则剧情化

依据案例：batch-273 案例 1361

## 读者承诺

用残酷规则、人性博弈和主角智斗破局制造追读。规则介绍必须变成剧情压力，不能停在说明书。

## 核心循环

```text
残酷游戏 -> 主角入局 -> 规则威胁 -> 智斗破局 -> 人性反转/收益
```

## 开篇必备

- 第一章先让主角或核心角色遇事。
- 规则要通过威胁、选择、代价展示。
- 主角必须尽快展示能力或性格。

## 期待来源

读者期待主角用智商压过规则和对手，也期待规则暴露人性。看懂推理后，爽点才会成立。

## 断档风险

- 规则说明太长，剧情停住。
- 主角开篇缺席，读者无法带入。
- 照搬影视节奏，文字缺少画面承托。
- 女主或队友拖油瓶化，拉低智斗逼格。

## 训练任务

把一段规则说明改写成一次小型博弈：角色遇险、规则露出、主角判断、反打兑现四步齐全。
