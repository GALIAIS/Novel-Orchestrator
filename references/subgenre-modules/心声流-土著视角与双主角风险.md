# 心声流-土著视角与双主角风险

依据案例：`batch-325` 案例 1625

## 读者承诺

土著主角听见穿越者心声可以写，但必须让读者站在土著这边，并控制穿越者抢主角位的风险。

## 核心循环

```text
土著听见心声 -> 获得穿越者信息差 -> 套话/借力/反制 -> 主角获益 -> 穿越者不抢位
```

## 开篇必备

- 明确真正主角是谁。
- 第一场戏让土著利用心声获得收益。
- 给土著足够强的目标、人格魅力或正当性。
- 如果需要，给穿越者拉仇恨，避免读者代入错位。
- 负面情绪要有卸力和回收。

## 期待来源

读者期待土著用心声信息差反制、套话、借力，获得传统穿越者本该占有的好处。

## 断档风险

- 穿越者比土著更像主角。
- 现代人优越感让读者天然代入穿越者。
- 变成双主角后，另一个主角塑造失败。
- 只靠心声标签，没有持续事件。
- 反道德、OOC、负面情绪不卸力，变成毒点。

## 训练任务

写一个试章：土著听见穿越者心声，套出一个关键信息并获益；结尾检查读者是否仍站在土著一边。
