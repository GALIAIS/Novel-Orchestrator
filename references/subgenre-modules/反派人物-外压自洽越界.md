# 反派人物-外压自洽越界

依据案例：`batch-338` 案例 1686

## 读者承诺

反派要有从初始缺口到越界选择的路径，让读者理解其自洽逻辑，但不替他洗白。

## 核心循环

```text
初始缺口 -> 外部挤压 -> 自我解释扭曲 -> 越界选择 -> 反派人格成型
```

## 开篇必备

- 反派有初始弱点或缺口。
- 外部压力连续而非突发。
- 反派有自己的解释系统。
- 第一次越界不可逆。
- 主角立场压住三观。

## 期待来源

读者期待看见反派如何一步步走到现在，而不是只看一个功能性恶人。

## 断档风险

- 只贴疯批标签。
- 一件事突然黑化。
- 反派魅力变成洗白。
- 电影人物手法没有转成网文选择。
- 主角立场被反派魅力冲散。

## 训练任务

设计一个反派三次受压、一次自我解释、一次越界的成型链，并写清主角如何回应。
