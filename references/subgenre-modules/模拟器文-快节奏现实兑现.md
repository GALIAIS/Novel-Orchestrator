# 模拟器文-快节奏现实兑现

依据案例：`batch-033` 案例 164。

## 读者承诺

模拟器文的爽点在于模拟信息快速改变现实选择，而不是长篇展示平行经历。

## 核心循环

```text
现实问题 -> 模拟取证 -> 获得信息/收益 -> 现实兑现 -> 新风险 -> 下一轮模拟
```

## 开篇必备

- 现实问题先出现。
- 模拟成本清楚。
- 模拟段控制篇幅。
- 回到现实立刻改变局面。

## 期待来源

- 读者等模拟揭示风险。
- 模拟带来路线选择。
- 现实快速反杀或避坑。
- 下一轮模拟有新钩子。

## 断档风险

- 模拟段太长。
- 模拟和现实脱节。
- 没有现实收益。
- 节奏慢，读者跳模拟。

## 训练任务

写一章模拟器文小纲：现实困境、模拟信息、现实兑现、下一轮风险四段必须连起来。
