# 三视角拆书-事件复盘预测

依据案例：batch-273 案例 1363

## 读者承诺

把无脑阅读改成主动拆书：用读者、作者、编辑三视角复盘事件，并通过预测对比训练结构感。

## 核心循环

```text
阅读事件 -> 停顿记录 -> 三视角复盘 -> 预测后续 -> 对比作者处理 -> 素材归档
```

## 开篇必备

- 以一个完整事件为最小单位。
- 先记录关键情节，再分析功能。
- 第一遍就尝试预测作者下一步。

## 期待来源

作者端期待来自“看懂别人怎么写”。读者感受被转译成作者手段和编辑判断后，素材才能被自己调用。

## 断档风险

- 一口气读完才总结，细节已经丢失。
- 只写喜欢不喜欢，不拆功能。
- 截图不分类，后续找不到。

## 训练任务

读完一个小事件后写 8 行复盘：情节、感受、手段、卖点、风险、预测、对比、可迁移写法。
