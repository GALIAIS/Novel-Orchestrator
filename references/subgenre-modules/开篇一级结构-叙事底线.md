# 开篇一级结构-叙事底线

依据案例：`batch-013` 案例 064。

## 读者承诺

开篇第一个一级结构要先像网文：低理解负担、主角快速登场、连贯叙事、切入点明确、第一章有期待和卖点。

## 核心循环

```text
少人物 -> 主角登场 -> 当前矛盾 -> 主角行动 -> 切入点 -> 情绪上行 -> 新期待
```

## 开篇必备

- 主要人物不超过 3 个。
- 300 字内叙事主角登场。
- 1000 字内给期待点或爽点。
- 不插叙、不乱切视角、不大篇幅梦境回忆。
- 第一章写明卖点。

## 期待来源

- 读者知道主角下一步要干什么。
- 读者看到切入点改变主角生活。
- 读者能预料喜闻乐见结局，但猜不到过程。
- 主角用自己的行动推动第一场变化。

## 断档风险

- 开头人物太多，关系复杂。
- 介绍性文字不能直接连接当前矛盾。
- 主角只是被审视对象，迟迟不行动。
- 情绪长时间下行或平直，没有上行。
- 第一章没有卖点，读者不知道看什么。

## 训练任务

检查前三章：标出主角登场字数、切入点、第一处期待、第一处爽点和第一章卖点。缺一项就重写对应段落。
