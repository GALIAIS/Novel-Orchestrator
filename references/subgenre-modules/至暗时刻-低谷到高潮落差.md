# 至暗时刻-低谷到高潮落差

依据案例：batch-294 案例 1470

## 读者承诺

爽感来自至暗低谷到高潮高峰的落差，冲突要逐级加压。

## 核心循环

```text
初始压力 -> 冲突升级 -> 至暗时刻 -> 反派情绪巅峰 -> 主角反转 -> 爽感兑现
```

## 开篇必备

- 初始压力。
- 一级冲突。
- 二级冲突。
- 三级冲突。
- 至暗时刻。
- 主角底牌。
- 高潮收益。

## 期待来源

读者看到主角被压到最低点，随后用底牌反转，情绪落差变成爽感。

## 断档风险

- 无敌文全程平推。
- 没有低谷，高潮无力。
- 冲突不升级。
- 反派情绪没到巅峰就被打掉。

## 训练任务

写一段相对无敌桥段：主角强，但被规则或人质压到至暗，再反转。
