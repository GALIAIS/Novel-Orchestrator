# 三国同人-人物认知约束

依据案例：`batch-153` 案例 761。

## 读者承诺

三国同人借用历史和演义人物资产，读者期待熟悉人物按默认认知行动。人设可改，但反常行为必须有足够理由。

## 核心循环

```text
登场历史人物 -> 触发读者默认印象 -> 行动符合或有理由反常 -> 维持熟悉感 -> 推进剧情
```

## 开篇必备

- 角色默认印象。
- 核心能力与性格。
- 不能随便写的行为。
- 若反常，给出理由。
- 关键历史/演义细节。

## 期待来源

读者期待典韦就是武力高、愚忠、少质疑命令；曹操刺董等熟知桥段也要尊重常识。熟悉感本身就是同人爽点。

## 断档风险

- 人物行为违背印象且无理由。
- 小纲流畅但正文角色别扭。
- 借 IP 省设定，却忘了 IP 约束。
- 基础典故写错，削弱可信度。

## 训练任务

选一个三国人物，列出默认认知、不能做的事、可反常的理由。写 500 字桥段，确保反常行为有因果支撑。
