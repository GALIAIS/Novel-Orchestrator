# 游戏文垂类平台-买断与流量评估

依据案例：`batch-202` 案例 1007。

## 读者承诺

小众垂类平台题材精准，但是否值得投，要看流量规模、买断/保底和机会成本。

## 核心循环

```text
垂类匹配 -> 流量评估 -> 买断/保底判断 -> 试投或回大站
```

## 开篇必备

- 明确平台用户规模。
- 明确是否买断或保底。
- 判断精品门槛。
- 准备大站备选方案。

## 期待来源

作者期待题材读者更精准，但需要收益兜底才能抵消小平台流量不足。

## 断档风险

- 只看题材匹配，忽略人少。
- 没买断还押主力稿。
- 精品门槛高导致试错周期长。
- 放弃大站曝光。

## 训练任务

做投稿平台评估表：用户规模、题材匹配、买断/保底、门槛、反馈速度、机会成本，最后给“主投/试投/不投”。
