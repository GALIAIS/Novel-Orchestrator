# 开头借鉴-结构相似不等于撞车

依据案例：`batch-173` 案例 863。

## 读者承诺

成熟开头会共享叙事节奏。学习结构不等于抄袭，关键是替换题材、主角、规则、谜题和情绪细节。

## 核心循环

```text
拆热门开头 -> 提取共通结构 -> 替换核心设定 -> 保留有效节奏 -> 做出差异化
```

## 开篇必备

- 明确借鉴的是结构，不是句子和关键桥段。
- 共通节奏要服务类型承诺。
- 主角身份、异常来源、金手指规则要不同。
- 相似节点下必须写出自己的新信息。

## 期待来源

读者熟悉结构后更快进入类型，但会期待设定、人物和谜题有新东西。

## 断档风险

- 把具体场景、句子、谜底也搬过来。
- 因怕撞车而放弃有效结构。
- 只学外形，不懂节点功能。
- 差异化只改名词，不改冲突逻辑。

## 训练任务

拆两个热门开头的共通结构，再写自己的版本。保留 5 个结构节点，但替换主角身份、异常来源、规则、第一悬念和情绪落点。
