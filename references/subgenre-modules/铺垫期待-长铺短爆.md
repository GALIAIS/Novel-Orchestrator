# 铺垫期待-长铺短爆

依据案例：`batch-033` 案例 162。

## 读者承诺

铺垫负责拉长期待，爽点负责短促爆发；背景和爆发短，铺垫最长但不能无聊。

## 核心循环

```text
核心卖点 -> 背景告知 -> 反派误判 -> 拉仇恨 -> 期待满格 -> 短爆发
```

## 开篇必备

- 读者先知道主角底牌。
- 反派踩中对应痛点。
- 铺垫过程持续加码。
- 爽点爆发短而狠。

## 期待来源

- 读者比反派知道更多。
- 反派越嚣张，翻车越值得等。
- 主角先不急着打脸。
- 每一步铺垫都更接近爆发。

## 断档风险

- 没铺垫就打脸，软绵绵。
- 铺垫和核心卖点不在一条线。
- 铺垫长但不拉期待。
- 爽点爆发拖太长。

## 训练任务

写一个“长铺短爆”桥段：铺垫至少三次加码，爆发只用一个动作或一句话完成。
