# 专业题材资料-医学刑侦工具

依据案例：`batch-162` 案例 808。

## 读者承诺

用人体结构、药品疾病等专业工具补足医学、刑侦、伤病描写的可信细节。

## 核心循环

```text
发现专业细节缺口 -> 查工具 -> 提取可写信息 -> 转成动作/判断/冲突 -> 标注边界
```

## 开篇必备

- 专业细节要服务剧情。
- 人体结构用可视化工具核对位置和动作。
- 药品疾病用结构化资料查适应症、用法、不良反应。
- 不把写作素材当真实医疗建议。

## 期待来源

读者期待专业题材有可信细节，但不会被资料堆砌拖慢故事。

## 断档风险

- 只靠想象写专业场景。
- 查到资料后原样堆进正文。
- 医学判断越界。
- 细节不服务冲突。

## 训练任务

写一个伤病或刑侦场景前，列出 5 个需要核对的专业点。只把最影响冲突的 2 个写进正文。
