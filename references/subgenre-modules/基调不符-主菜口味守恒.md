# 基调不符-主菜口味守恒

依据案例：`batch-111` 案例 552。

## 读者承诺

开篇定下的基调就是给读者上的主菜。日常恋爱、渣男文、玄幻种田、奇幻冒险都可以有支线，但后续不能突然换成另一种核心口味。

## 核心循环

```text
开篇定口味 -> 读者留下 -> 支线服务主菜 -> 变化提前缓冲 -> 基调守恒
```

## 开篇必备

- 明确主基调。
- 知道读者为什么留下。
- 支线不能抢主线口味。
- 转向前要有长期铺垫。
- 后续事件持续强化主菜。

## 期待来源

- 读者得到承诺中的体验。
- 支线丰富而不夺权。
- 口味稳定形成基本盘。
- 转折不显得被欺骗。

## 断档风险

- 日常恋爱突然变重战斗。
- 种田突然变纯升级。
- 搞笑和悲伤反复乱跳。
- 老梗加戏但没有情绪价值。

## 训练任务

列出你的前三章主菜口味，再列后续 10 个事件。每个事件标注它是在强化主菜，还是偏到别的桌。
