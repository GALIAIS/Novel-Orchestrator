# 虚晃情绪-双下行后兑现

依据案例：`batch-074` 案例 369。

## 读者承诺

这个模块用于“读者知道终会成功，但不知道怎么成功”的桥段。它承诺通过两次可承受的落空拉高期待，再用第三次意外机会兑现，让成功更像小高潮，而不是平铺发奖。

## 核心循环

```text
稀缺资格 -> 第一次希望 -> 落空 -> 第二次希望 -> 再落空 -> 意外新机会 -> 兑现
```

## 开篇必备

- 资格、宝物、机会必须稀缺。
- 主角必须有强烈需要。
- 第一次落空承担基础下行。
- 第二次落空承担虚晃一枪。
- 第三次机会要有意外感，但不能完全凭空。

## 期待来源

- 读者知道主角终会拿到，仍想知道过程。
- 落空越合理，兑现越有弹性。
- 第二次希望让读者误以为要兑现。
- 第三次意外把低情绪拉回上行。
- 兑现后要立即给反应和阶段性奖励。

## 断档风险

- 两次落空完全重复，读者觉得拖。
- 下行成本过高，角色动机显得离谱。
- 牺牲物和目标没有因果，只剩煽情。
- 网文开篇拉太久，读者在兑现前离开。
- 兑现后没有立刻产生新局面。

## 训练任务

写一个“考试名额、秘境门票或试镜资格”的双下行兑现桥段。两次落空必须各有新信息，第三次兑现必须同时带来惊喜和下一阶段目标。
