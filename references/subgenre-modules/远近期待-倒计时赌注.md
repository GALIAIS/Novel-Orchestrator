# 远近期待-倒计时赌注

依据案例：`batch-101` 案例 504。

## 读者承诺

开篇要同时完成吸入和留存：近期期待靠危机、情绪和兑现，远期期待靠倒计时、马甲、反差和主角无法拒绝的赌注。

## 核心循环

```text
近危机 -> 当前兑现 -> 远目标露出 -> 倒计时 -> 赌注明确 -> 读者等待下一节点
```

## 开篇必备

- 一个眼前危机。
- 一个短期兑现。
- 一个远期目标。
- 一个明确时间节点。
- 一个主角不能拒绝的赌注。

## 期待来源

- 危机强情绪。
- 装逼打脸或破局兑现。
- 三年之约、宗门大比、追杀期限等倒计时。
- 马甲和身份反差。
- 失败代价。

## 断档风险

- 只有倒计时概念，没有赌注。
- 远期目标太虚。
- 当前爽点和远期目标互不相干。
- 套路烂大街但没有新意和笔力。

## 训练任务

给开篇设计一条 3 章倒计时和一条 30 章倒计时。每条都写清：时间、赌注、失败代价、主角为什么不能逃、第一次推进事件。
