# 童话副本-假目标与倒计时拉扯

依据案例：`batch-239` 案例 1193。

## 读者承诺

童话副本可以利用读者熟悉感制造虚晃一枪：目标看似简单，假答案和倒计时让真相变危险。

## 核心循环

```text
进入原型故事 -> 目标看似明确 -> 倒计时压迫 -> 假目标出现 -> 揭真相脱离
```

## 开篇必备

- 选读者熟悉的原型。
- 设置保命约定。
- 给出看似简单的任务。
- 安排假答案和不可调和矛盾。

## 期待来源

读者先借原故事产生判断，再被变体打破预期，跟随主角解出真正目标。

## 断档风险

- 只复述童话，没有变体。
- 假目标太明显。
- 倒计时没有压力。
- 真相揭露不改变主角处境。

## 训练任务

选一个童话，设计一个假目标和一个倒计时，让主角必须在熟悉剧情里找出不同真相。
