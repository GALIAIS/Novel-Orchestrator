# 女频迁移-根源理论转译

依据案例：`batch-028` 案例 138。

## 读者承诺

女频可以吸收冲突、爽点、情绪这些根源理论，但必须回到女频样本里转成感情推进和关系变化。

## 核心循环

```text
通用理论 -> 女频样本 -> 感情线主导 -> 情绪转译 -> 平台口味校准
```

## 开篇必备

- 先分清通用机制和男频特例。
- 女频样本必须真实拆。
- 感情线是主轴，其他线为辅助。
- 爽点要转译成偏爱、清算、补偿或关系确认。

## 期待来源

- 关系推进带来的情绪波动。
- 误会、亏欠、偏爱、补偿的持续变化。
- 男女主第一章就形成强关系牵引。
- 外部事件服务感情线变化。

## 断档风险

- 照搬男频打脸和升级。
- 只学理论，不拆女频案例。
- 感情线弱，事件线喧宾夺主。
- 平台口味混用。

## 训练任务

把一个男频“压抑后打脸”桥段改成女频现言桥段，打脸结果必须推动男女主关系变化。
