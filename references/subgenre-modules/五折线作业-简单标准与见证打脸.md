# 五折线作业-简单标准与见证打脸

依据案例：`batch-244` 案例 1216。

## 读者承诺

五折线作业不必复杂，只要每个折点功能清楚，简单校园格斗也能完成期待和打脸。

## 核心循环

```text
朋友受威胁 -> 主角有信心 -> 同伴不信 -> 虚晃逃脱 -> 退路被堵 -> 当众打脸
```

## 开篇必备

- 明确被保护对象。
- 明确外部威胁。
- 金手指给主角信心。
- 最后安排见证人反馈。

## 期待来源

读者看到所有人都不信主角能赢，于是等待主角用行动证明。

## 断档风险

- 折点标注清楚但事件没有变化。
- 虚晃只是路过，没有改变期待。
- 威胁不够具体。
- 打脸缺见证人。

## 训练任务

写 5 句五折线，每句对应一个折点；检查每句是否改变了读者对结果的判断。
