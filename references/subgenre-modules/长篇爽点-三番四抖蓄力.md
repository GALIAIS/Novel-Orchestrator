# 长篇爽点-三番四抖蓄力

依据案例：`batch-161` 案例 802。

## 读者承诺

用多次压情绪和最终释放组织长篇爽点，同时保留后续升级空间，避免前期把最高层级爽点耗尽。

## 核心循环

```text
第一番压迫 -> 第二番加压 -> 第三番合流 -> 第四次释放 -> 后续台阶保留
```

## 开篇必备

- 前三次主要压情绪，不急着打脸。
- 参与者要有不同利益和反应。
- 释放点要有权威或结果认证。
- 当前场景只解决当前等级问题。

## 期待来源

读者期待压力越压越集中，最后一次释放既打回当前质疑，也打开下一层舞台。

## 断档风险

- 每次都写震惊，情绪单一。
- 高中场景提前写完世界级爽点。
- 打脸层级不分，后续只能重复膨胀。
- 直接照搬短平快网站技巧到长篇。

## 训练任务

把一个装逼打脸桥段拆成三次压情绪和一次释放。写完后标出本场没有消耗的下一层爽点。
