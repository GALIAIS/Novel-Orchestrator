# 迪化流-配角脑补反差爽

依据案例：batch-308 案例 1540

## 读者承诺

迪化流的爽点来自配角把主角的随手行为脑补成高深布局，并主动补全逻辑、送上资源或推动剧情。

## 核心循环

```text
主角无心行为 -> 配角误读 -> 脑补成大棋 -> 主动行动 -> 主角顺水推舟/怕穿帮 -> 反差爽
```

## 开篇必备

- 读者要知道主角真实想法。
- 配角脑补必须自洽。
- 误会要转化为行动收益。
- 主角反应要制造喜剧或爽感。

## 期待来源

期待来自上帝视角。读者同时看到真相和误读，享受配角自我攻略带来的反差和收益。

## 断档风险

- 配角只是降智吹捧。
- 脑补不产生剧情动作。
- 主角真实想法不清，反差不成立。
- 误会长期不升级，梗被耗空。

## 训练任务

写一个迪化桥段：主角一句无心话，配角推出三层计划并送上资源。标出真实意图、脑补逻辑和实际收益。
