# 精神低谷-低压写作恢复

依据案例：`batch-075` 案例 371。

## 读者承诺

这个模块用于作者处在长期低动力、卡文、压力过载或健康风险时的写作安排。它承诺不把低谷粗暴归咎为懒，而是先降压，再用小细纲和最低任务恢复可执行性。

## 核心循环

```text
状态识别 -> 降低自责 -> 拆小任务 -> 细纲辅助 -> 短时行动 -> 记录反馈
```

## 开篇必备

- 先判断是否需要专业帮助。
- 今日任务小到能开始。
- 只列下一段，不规划全年产能。
- 用情绪折线辅助想剧情。
- 写作恶化状态时停止加码。

## 期待来源

- 小任务让作者重新获得可控感。
- 细纲减少坐下后的空白压力。
- 情绪折线把卡文转成下一步判断。
- 生活输入帮助写作恢复热度。
- 持续记录能区分技术性卡文和健康问题。

## 断档风险

- 用宏大目标继续施压。
- 把精神健康问题当普通拖延处理。
- 只靠热情，不做细纲。
- 身体已经强烈排斥仍硬压产量。
- 用写作建议替代现实治疗。

## 训练任务

连续三天做低压写作记录：每天只写最低字数、三步细纲和一个情绪变化。若任务仍明显加重痛苦，就暂停加量并优先处理现实健康支持。
