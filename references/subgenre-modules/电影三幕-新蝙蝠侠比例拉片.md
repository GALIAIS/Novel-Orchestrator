# 电影三幕-新蝙蝠侠比例拉片

依据案例：batch-303 案例 1513

## 读者承诺

拉片先看三幕功能和比例，再拆目标变化；中段长不是水，而是持续追查和压力升级。

## 核心循环

```text
总长度 -> 三幕比例 -> 各幕功能 -> 目标段落 -> 中段升级 -> 最终兑现
```

## 开篇必备

- 标出总时长或总章数。
- 划定三幕边界。
- 写清每幕功能。
- 第二幕重点拆目标变化。

## 期待来源

- 第一幕建立承诺。
- 第二幕持续发现和升级。
- 第三幕集中真相和回收。

## 断档风险

- 只写剧情概述，不拆功能。
- 误以为三幕必须均分。
- 中段没有目标变化，变水。

## 训练任务

选一部电影，标三幕比例，并列出第二幕至少五次目标变化。
