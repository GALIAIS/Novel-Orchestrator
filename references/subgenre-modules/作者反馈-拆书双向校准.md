# 作者反馈-拆书双向校准

依据案例：`batch-053` 案例 265。

## 读者承诺

用作者反馈校准拆书：创作者的直觉写法经拆书显性化后，可以反过来验证结构、情绪折线和前后呼应是否分析准确。

## 核心循环

```text
作品有效 -> 拆书显性化 -> 作者反馈确认 -> 方法可信 -> 学习者迁移
```

## 开篇必备

- 记录作者确认了哪些分析。
- 找出前后呼应证据。
- 标出情绪上行和下行。
- 区分作者直觉与拆书总结。
- 把隐性逻辑写成可训练动作。

## 期待来源

- 作者反馈增强拆书可信度。
- 情绪折线比单纯三幕更贴近实际写作。
- 下行服务下一次上行，读者期待不断续上。
- 创作和分析双向受益。

## 断档风险

- 把互相肯定当成方法本身，会空。
- 只看作者天赋，不拆可训练部分，会学不到。
- 三幕式不清晰时，不能误判为没有结构。
- 只顾爽不复盘，阅读积累难转写作能力。

## 训练任务

选一段成功桥段，写出作者可能无意识做对的 3 点、拆书可显性化的 3 点，并画出情绪上下行。
