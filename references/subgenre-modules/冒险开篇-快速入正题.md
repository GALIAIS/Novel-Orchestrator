# 冒险开篇-快速入正题

依据案例：`batch-096` 案例 477。

## 读者承诺

冒险题材的第一承诺是尽快让读者看到未知世界、危险和收益。读者不想先看一大段主角冒险前的生活说明；进入方式可以简单，关键是快速踢进正题。

## 核心循环

```text
开局触发 -> 主角进入冒险世界 -> 立刻遇到危险/收益 -> 留下前置谜团 -> 后续多次少量补信息
```

## 开篇必备

- 100 到 300 字内出现冒险相关异常。
- 第一章必须让主角接触冒险世界。
- 前置背景只留钩子，不完整解释。
- 进入方式服务正题，不喧宾夺主。

## 期待来源

- 异世界或未知区域的新鲜感。
- 第一危险带来的紧张。
- 第一收益带来的追读。
- 前置谜团的多次少量补充。

## 断档风险

- 开头解释主角冒险前太多生活。
- 为了进入方式自然，拖慢题材承诺。
- 前置设定一口气写完。
- 第一章结束还没进入冒险正题。

## 训练任务

写 3 个冒险开局，每个 300 字以内。要求第一段就出现异常，第二段主角进入冒险场域，第三段给出危险或收益。
