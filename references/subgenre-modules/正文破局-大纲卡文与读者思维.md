# 正文破局-大纲卡文与读者思维

依据案例：`batch-240` 案例 1198。

## 读者承诺

大纲磨到脑袋空时，进入正文能逼出细节和逻辑；写作要从作者上帝视角转成读者逐步接收。

## 核心循环

```text
大纲卡住 -> 先写正文 -> 细节澄清逻辑 -> 复盘读者理解 -> 回头删设定
```

## 开篇必备

- 区分作者已知和读者已知。
- 前三章先勾兴趣。
- 设定随剧情展开。
- 写完后做减法。

## 期待来源

读者先被冲突、人物或矛盾拉住，再逐步理解设定，而不是被说明书劝退。

## 断档风险

- 永远磨大纲不开写。
- 作者以为读者都懂。
- 设定堆太细。
- 关键情节一笔带过。

## 训练任务

卡细纲时直接写 1500 字正文，再标出哪些信息读者已经知道、哪些只是作者脑内知道。
