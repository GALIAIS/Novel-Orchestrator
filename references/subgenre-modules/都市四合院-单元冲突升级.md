# 都市四合院-单元冲突升级

依据案例：`batch-179` 案例 895。

## 读者承诺

都市、四合院和碎片爽文不一定靠宏大战力线，也可以靠连续加码的小冲突和即时情绪偿还留住读者。

## 核心循环

```text
小矛盾 -> 对手加码 -> 主角反击 -> 后果升级 -> 撕破脸/反杀 -> 下一单元
```

## 开篇必备

- 初始冲突足够生活化或利益明确。
- 每轮对手层级、后果或情绪压力升级。
- 主角反击要形成偿还。
- 弱主线也要强情绪拉扯。

## 期待来源

读者期待小事不断加码，最后形成打脸、反杀、撕破脸或名声清算。都市娱乐文可升级公司层级，四合院文可升级邻里矛盾。

## 断档风险

- 只有东扯西扯，没有升级。
- 情景剧式单元没有情绪偿还。
- 飞卢短爽透支太快，长线乏力。
- 剪辑文只搬高潮，缺少自己的剧情承载。

## 训练任务

设计一个不靠战力的 4 级冲突升级链：小事、误会、利益压迫、撕破脸，每级都写清对手加码和主角反击。
