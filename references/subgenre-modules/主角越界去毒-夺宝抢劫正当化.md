# 主角越界去毒-夺宝抢劫正当化

依据案例：`batch-180` 案例 898。

## 读者承诺

主角需要夺宝、抢资源或下狠手时，要让读者仍觉得主角站在爽点和道义高位。

## 核心循环

```text
资源需求 -> 正常获取失败 -> 目标对象先恶/先动手 -> 主角反击夺取 -> 爽感无毒
```

## 开篇必备

- 主角越界前先处理道义风险。
- 目标对象不能是无辜者。
- 最稳的是对方先招惹主角。
- 主角行为最好落在反击或见义勇为上。

## 期待来源

读者期待主角获得关键资源，同时不希望主角欺负无辜。黑吃黑、反杀和见义勇为能保护主角人品。

## 断档风险

- 主角主动抢无辜者。
- 只靠给对方泼脏水，仍显得俗套。
- 替天行道太刻意。
- 主角处于道义下风。

## 训练任务

同一个夺宝桥段写两版：目标恶贯满盈版、目标先动手版。比较哪一版更不伤主角人品。
