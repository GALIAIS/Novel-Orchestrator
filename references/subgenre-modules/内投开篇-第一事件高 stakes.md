# 内投开篇-第一事件高 stakes

依据案例：`batch-033` 案例 165。

## 读者承诺

内投开篇第一事件就要事关前途、生死或信念，压主必须合理，不能靠主角死脑筋硬撑。

## 核心循环

```text
高 stakes 第一事件 -> 合理压迫 -> 主角主动求胜 -> 失败有成长 -> 金手指转机 -> 爽点兑现
```

## 开篇必备

- 第一事件不能无关痛痒。
- 主角失败要有原因和成长。
- 赌注、风险、规则要顺。
- 金手指出现前主角先主动。

## 期待来源

- 读者立刻知道为什么重要。
- 不公平待遇让读者站队。
- 主角主动策略带来代入。
- 金手指不是天降摆烂奖励。

## 断档风险

- 读者等不到第二第三事件。
- 主角连输几十场显得蠢。
- 受虐奖励金手指难写长。
- 书名第一眼劝退。

## 训练任务

重写一个擂台开篇：第一场比赛必须影响主角前途，主角失败前要做出聪明选择，失败后立刻获得可见成长。
