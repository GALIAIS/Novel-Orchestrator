# 拆书选样-男频搞笑恋爱标签

依据案例：batch-309 案例 1543

## 读者承诺

拆搞笑文或恋爱文前，先限定男频/女频、平台和标签。样本要和自己的读者承诺一致。

## 核心循环

```text
确定类型 -> 限定男频女频 -> 用平台标签筛样本 -> 看成绩 -> 拆前三章机制
```

## 开篇必备

- 先写明目标平台。
- 区分男频恋爱、女频恋爱、日轻恋爱。
- 用平台标签页找候选。
- 拆成绩好且读者承诺同向的样本。

## 期待来源

期待来自选样准确。拆对样本，学到的是目标读者会买单的机制。

## 断档风险

- 男频作者拆女频恋爱后直接套用。
- 只看书好，不看平台和读者。
- 搞笑文只拆段子，不拆情境和人物关系。
- 恋爱文只拆互动，不拆关系承诺。

## 训练任务

用平台标签筛 10 本男频搞笑/恋爱样本，选 3 本高成绩同向书，拆前三章笑点或关系钩子。
