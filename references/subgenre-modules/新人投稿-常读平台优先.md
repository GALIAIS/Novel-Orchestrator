# 新人投稿-常读平台优先

依据案例：`batch-316` 案例 1580

## 读者承诺

新人选投稿平台，先投自己经常看书的网站。熟悉读者口味和文风，比盲目听平台标签更可靠。

## 核心循环

```text
常读平台 -> 熟悉文风节奏 -> 拆同题材榜单 -> 写近似口味开篇 -> 投稿
```

## 开篇必备

- 先列常读网站。
- 找同题材榜单样本。
- 对齐开篇节奏和文风。
- 陌生平台先拆书再投稿。

## 期待来源

作者期待降低口味偏差，平台读者期待看到符合本站阅读习惯的开篇。

## 断档风险

- 只看平台传闻，不看读者口味。
- 盲投陌生站。
- 题材和文风不匹配。
- 常读但不拆，仍停留在读者视角。

## 训练任务

按平台整理最近读过的 10 本书，选数量最多的平台，拆 3 本同题材前三章。
