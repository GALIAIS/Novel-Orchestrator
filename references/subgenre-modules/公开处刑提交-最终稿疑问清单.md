# 公开处刑提交-最终稿疑问清单

依据案例：`batch-343` 案例 1712

## 读者承诺

公开处刑要提交当前能力上限的最终稿，并附书名简介小纲、18 元素自检和具体疑问。

## 核心循环

```text
自查基础错误 -> 改到当前上限 -> 列疑问清单 -> 集中评审 -> 大改复盘成方法
```

## 开篇必备

- 错字病句标点先自查。
- 书名简介小纲先改到位。
- 18 元素缺口要标出。
- 提交作者自己的思路。
- 用 1、2、3 提具体问题。

## 期待来源

评文者期待看到作者已认真思考过，反馈才能集中打到作者看不见的核心问题。

## 断档风险

- 随手发草稿。
- 基础错误等别人抓。
- 只写求毒舌锐评。
- 反复小修反复求评。
- 作者不主动总结。

## 训练任务

准备一次公开处刑提交包：最终稿、书名简介、小纲、18 元素自检、3 个具体疑问。
