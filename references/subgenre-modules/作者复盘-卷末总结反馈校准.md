# 作者复盘-卷末总结反馈校准

依据案例：`batch-184` 案例 917。

## 读者承诺

卷末复盘用于校准长篇写作：回看本卷目标、结构、人物、节奏、优缺点和下一卷调整。

## 核心循环

```text
一卷结束 -> 总结目标 -> 评估兑现 -> 标记问题 -> 结合反馈 -> 调整下一卷
```

## 开篇必备

- 写清本卷原定功能。
- 分析结构和节奏。
- 记录人物与世界推进。
- 分辨可靠反馈和噪音反馈。

## 期待来源

作者持续复盘，才能让长篇不是靠惯性推进，而是阶段性修正。

## 断档风险

- 只庆祝完结，不复盘问题。
- 平台反馈滞后导致误判。
- 把被平台机制掩盖的问题当成优点。
- 下一卷不吸收本卷教训。

## 训练任务

写一份卷末复盘：本卷目标、实际完成、情绪曲线、3 个优点、3 个缺点、下一卷 3 个调整。
