# 矛盾网-利益冲突反派针对

依据案例：`batch-103` 案例 511。

## 读者承诺

反派针对主角必须有利益根。读者可以接受反派傲慢、误判、嘴臭，但不能接受反派无视已知信息、没有利益冲突还硬骂主角。

## 核心循环

```text
利益冲突 -> 信息误判 -> 嘲讽/针对 -> 主角反击 -> 认知更新 -> 矛盾升级
```

## 开篇必备

- 反派为什么在场。
- 主角碰了什么利益。
- 反派知道什么、不知道什么。
- 反派为什么敢先动嘴或动手。
- 反击后反派是否改变策略。

## 期待来源

- 压抑来自合理轻视。
- 爽感来自主角反击。
- 后续来自矛盾网扩散。
- 反派认知更新能避免降智。

## 断档风险

- 主角变成移动嘲讽机器。
- 反派刚见识主角能力还继续骂废物。
- 只有情绪模板，没有利益根源。
- 反派智商被强行压低。

## 训练任务

写一个 500 字反派嘲讽桥段，先列利益冲突和误判来源。写完删掉所有没有利益支撑的嘲讽句。
