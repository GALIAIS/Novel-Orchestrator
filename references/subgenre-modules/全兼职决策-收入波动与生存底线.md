# 全兼职决策-收入波动与生存底线

依据案例：batch-386 案例 1930

## 读者承诺

兼职或全职要由自己的连续收入、更新时间、健康成本和生活底线决定，不能用头部收入或单月成绩代替职业风险评估。

## 核心循环

```text
记录真实收入与产出 -> 核算生存和健康成本 -> 评估波动风险 -> 设转入/退出条件 -> 定期复盘
```

## 开篇必备

- 连续多月净收入。
- 各收入来源占比。
- 最低生活与健康成本。
- 转全职和退出条件。

## 期待来源

- 职业梦想变成可验证条件。
- 单月波动不再主导决策。
- 兼职与全职路线各有边界。
- 收入中断时已有预案。

## 断档风险

- 用头部年入故事估算自己。
- 把版权收入预设为常规收入。
- 忽略更新对健康和质量的成本。
- 没有退出条件就贸然全职。

## 训练任务

用连续多月个人数据填写收入、更新、工时、健康、支出和波动；样本不足时只记录，不做全职结论。
