# 惊悚副本开篇-规则锚定与金手指延迟一拍

依据案例：`batch-400` 案例 2000

## 读者承诺

先让读者理解被卷入方式、生存规则、任务和普通能力基线，再让金手指以一项清楚偏差出现，并在第一任务立即实证。

## 核心循环

```text
日常关联 -> 异常转移 -> 世界内规则告知 -> 准备窗口建立普通基线 -> 金手指偏差 -> 第一任务实证与代价
```

## 开篇必备

- 主角为何被卷入以及主动/被迫性质清楚。
- 失败代价和当前任务明确。
- 规则由可信世界内来源传达。
- 普通参与者的能力与资源先建立。
- 金手指比基线延迟一拍出现。
- 第一任务迅速验证优势和限制。

## 期待来源

读者先知道普通人为什么难活，再立即理解主角多出的优势，并等待他能否在真实任务中正确利用。

## 断档风险

- 从拖拉过度修成信息跳级。
- 规则未明就直接开金手指。
- 没有普通基线，能力差异不可感。
- 游戏术语不符合世界内认知。
- 准备阶段只讲设定、不改变决策。
- 第一任务迟迟不验证能力。

## 训练任务

用六拍各 300 字以内重写：日常、转移、规则、准备、能力、任务；让陌生读者在能力出现前先复述普通人生存方法，再判断主角优势。
