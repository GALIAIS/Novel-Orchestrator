# 写作基本功-语感节奏训练

依据案例：`batch-177` 案例 881。

## 读者承诺

语感不是玄学，而是作者对人物、对白、节奏和表达顺滑度的具体判断力。

## 核心循环

```text
阅读成熟文本 -> 标注顺/不顺 -> 反复复读 -> 模仿改写 -> 写作时形成判断
```

## 开篇必备

- 先把语感拆成贴人物、对白、节奏和句子顺滑度。
- 用成熟作者文本做训练材料。
- 反复阅读重要段落。
- 把感受落到具体句子。

## 期待来源

作者期待从“说不清哪里差”变成“能判断哪句不贴人物、哪段节奏拖”。语感训练的价值在于让修改有抓手。

## 断档风险

- 只谈概念，不看具体文本。
- 只读不写，不做改写。
- 把语感当成天赋，放弃训练。
- 追求漂亮句子，忽略人物和节奏。

## 训练任务

选 3 页成熟文本，标注顺、贴人物、有节奏、别扭四类句子，再重写自己 5 句别扭表达。
