# 素材收藏-行动转化与一万字卡点

依据案例：batch-377 案例 1882

## 读者承诺

素材收藏不是成果。素材必须转成设定、对话、冲突或正文，否则只会变成逃避写作的安全感。

## 核心循环

```text
收藏资料 -> 选择一份 -> 绑定任务 -> 写出正文/小纲 -> 复盘卡点
```

## 开篇必备

- 素材类型。
- 今日输出任务。
- 对应正文位置。
- 完成后的复盘。

## 期待来源

- 资料能降低启动难度。
- 输出能验证资料价值。
- 卡点能暴露真实问题。
- 小任务能穿过逃避期。

## 断档风险

- 只收藏不写。
- 网盘过期无归档。
- 一万字后卡住就逃避。
- 资料没有转成行动。

## 训练任务

任选一份素材，30 分钟整理后立刻写 800 字正文或 1 个小纲，禁止只收藏。
