# 网文至暗时刻-短下行快上行

依据案例：`batch-038` 案例 190。

## 读者承诺

网文低谷要短，下行后迅速拐头上行。低谷、反思、斗争和正确选择最好在一章内完成，并给出更大期待。

## 核心循环

```text
低谷打击 -> 短反思 -> 成长判断 -> 正确选择 -> 绝处逢生 -> 更大期待
```

## 开篇必备

- 明确读者耐受度。
- 下行篇幅短。
- 主角迅速行动。
- 结尾给更大期待。

## 期待来源

- 低谷证明代价。
- 反思证明成长。
- 正确选择证明主角进步。
- 绝处逢生带来上行。

## 断档风险

- 照搬传统剧本长低谷。
- 爽文读者误判为虐主。
- 反思太长没有行动。
- 下行后没有更大期待。

## 训练任务

写一个 1500 字至暗时刻小纲：低谷、反思、斗争、拐头上行四段必须在一章内完成。
