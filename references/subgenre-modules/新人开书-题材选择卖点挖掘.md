# 新人开书-题材选择卖点挖掘

依据案例：`batch-178` 案例 888。

## 读者承诺

新人开书前先明确题材、亮点和卖点，避免方向错了再用大纲和更新硬补。

## 核心循环

```text
候选题材 -> 目标读者 -> 亮点提炼 -> 卖点包装 -> 作业验证 -> 开书
```

## 开篇必备

- 题材要对应明确读者。
- 亮点要能在前三章兑现。
- 卖点要能一句话说清。
- 方法学习后要配合作业实践。

## 期待来源

新人期待开书方向更稳。选题、亮点和卖点清楚，后续开篇才知道该兑现什么。

## 断档风险

- 等课程等到不实践。
- 题材热门但自己没有卖点。
- 亮点只在作者脑内，前三章看不见。
- 作业反馈不好却硬开。

## 训练任务

给 3 个候选题材各写一句话卖点、前三章兑现点和差异化亮点，拿反馈最清楚的一个继续做大纲。
