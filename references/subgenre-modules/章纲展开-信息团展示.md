# 章纲展开-信息团展示

依据案例：`batch-006` 案例 026。

## 读者承诺

把提纲里一句干巴巴的信息，展开成有情绪、有反应、有细节、有期待的正文。

## 核心循环

```text
提纲信息 -> 核心信息 -> 展示方式 -> 情绪拉扯 -> 反应扩展 -> 章末期待
```

## 开篇必备

- 明确这一段要让读者知道什么。
- 至少一个误判、压迫、质疑或阻碍。
- 至少一个角色心理活动或反应链。
- 展开后仍要服务本章关键点。

## 期待来源

- 读者看到信息如何影响人物。
- 读者等待误判被纠正。
- 读者等待主角如何回应压力。

## 断档风险

- 只加字数，不加戏剧性。
- 心理活动脱离冲突，变成空转。
- 展开后忘记本段核心信息。
- 没有对标拆书，只凭感觉水。

## 训练任务

把一个 100 字章纲点扩成 1500 字，必须包含误判、心理活动、情绪拉扯和章末期待。
