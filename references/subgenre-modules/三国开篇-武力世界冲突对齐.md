# 三国开篇-武力世界冲突对齐

依据案例：`batch-341` 案例 1705

## 读者承诺

三国异能开篇要让考试、羞辱、金手指和世界规则对齐，主角合理反击，反派承担作弊坏规则。

## 核心循环

```text
世界规则 -> 开篇冲突 -> 主角受压 -> 反派背锅 -> 金手指破局 -> 打脸反馈
```

## 开篇必备

- 冲突符合世界核心规则。
- 文试或武试要和生死后果建立必然关系。
- 主角不主动作弊坏规则。
- 反派承担负面行为。
- 背景设定边推进边交代。

## 期待来源

读者期待废物主角在合理压迫下觉醒或破局，靠金手指翻身，而不是先质疑规则和主角三观。

## 断档风险

- 强者为尊世界用文试定生死却无解释。
- 太守公子被随意羞辱不合理。
- 主角作弊导致三观崩。
- 三国背景解释太多。
- 装逼前下行情绪没拉满。

## 训练任务

把“考试定生死”分别改成武试版和文试金手指版，要求世界规则自洽、反派背锅、主角只做合理反击。
