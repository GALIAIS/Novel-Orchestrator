# 中点结构-短下行长期上行

依据案例：`batch-067` 案例 334。

## 读者承诺

这个模块用于卷纲、三幕式和中段节奏设计。它承诺的是把电影中点的伪胜利/伪失败转译成网文可追读结构：中点制造变化，但后续以短下行和长期上行维持爽点密度。

## 核心循环

```text
前半卷推进 -> 中点伪结果 -> 短促下行戳破 -> 调整计划 -> 长期上行 -> 第二幕小高潮
```

## 开篇必备

- 卷目标先明确。
- 中点必须改变局势。
- 伪胜利或伪失败要能被戳破。
- 下行要短，不能压太久。
- 第二幕结尾的小高潮保留。

## 期待来源

- 中点让读者感到局势变了。
- 短促下行提供反转和压力。
- 长期上行保证推进感。
- 小高潮维持网文爽感阈值。
- 中点后的计划调整带来新期待。

## 断档风险

- 把电影结构硬套成长时间人物低谷。
- 有中心危机就取消小高潮。
- 伪胜利后一直向好，没有被戳破。
- 伪失败后一直向坏，读者失去爽感。
- 中点只做概念解释，不改变主角行动。

## 训练任务

给一卷写 8 个节点：前半卷目标、中点伪结果、短促下行、计划调整、两段长期上行、第二幕小高潮、卷末新期待。检查下行是否超过两个节点。
