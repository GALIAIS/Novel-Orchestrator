# 长线矛盾-章卷书层级服务

依据案例：`batch-124` 案例 618。

## 读者承诺

长篇不散，靠章节矛盾服务卷，卷矛盾服务书。多事件可以热闹，但必须回流核心矛盾。

## 核心循环

```text
书核心矛盾 -> 卷核心矛盾 -> 章节局部事件 -> 事件结果回流卷 -> 卷结果丰富整书
```

## 开篇必备

- 本卷核心矛盾先定清。
- 章节事件必须能回流本卷。
- 多支线不能只增加热闹。
- 中段前要铺垫支线和核心的关系。
- 主线要沿着期待感构思。

## 期待来源

- 局部事件有独立期待。
- 事件结果推动卷核心变紧。
- 卷与卷共同丰富整本书。
- 世界观在回流中逐步完善。

## 断档风险

- 上一代恩怨、新人怀疑等事件不服务主线。
- 章节直接对标整书，跨度过大。
- 强行分化矛盾，写得细碎无聊。
- 不是所有冲突都能当主线。

## 训练任务

列一个卷的 5 个章节事件，每个事件写“如何服务本卷核心矛盾”。写不出回流关系的事件直接删改。
