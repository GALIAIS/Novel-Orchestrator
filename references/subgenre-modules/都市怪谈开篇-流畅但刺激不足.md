# 都市怪谈开篇-流畅但刺激不足

依据案例：`batch-315` 案例 1575

## 读者承诺

都市怪谈开篇不能只求流畅，还要在黄金三章内持续发生事件、抬高情绪、展示金手指和怪物威胁。

## 核心循环

```text
地球日常 -> 夜街异常 -> 怪物伪装 -> 主角能力预埋 -> 拉锯战 -> 击退回收
```

## 开篇必备

- 先明确故事发生在地球。
- 每章都要有事件。
- 怪物伪装和主角靠近要有动机。
- 金手指提前埋线，不能临时旁白。
- 怪物强度要衬托主角爽点。

## 期待来源

读者期待日常城市里冒出异常威胁，并看到主角凭特殊能力处理怪谈。

## 断档风险

- 第一章流畅但情绪波动小。
- 第二章只有实验能力和背景说明。
- 方言、错别字、血腥味漏洞降低信任。
- 主角行为不符合警惕逻辑。
- 怪物太弱，金手指无逼格。

## 训练任务

改一个都市怪谈开篇：第一章必须有异常、危险线索、金手指展示、击退怪物四项。
