# 描写删重点-心理环境与选择分析压缩

依据案例：`batch-220` 案例 1099。

## 读者承诺

心理、环境和选择分析只保留影响读者理解剧情的重点。

## 核心循环

```text
写完重读 -> 判断是否影响理解 -> 删除冗余 -> 提前重点 -> 推进下一信息
```

## 开篇必备

- 重点放在明显位置。
- 删掉不影响剧情理解的句子。
- 环境描写服务危险或选择。
- 心理描写服务动机和情绪。

## 期待来源

读者快速获得关键信息，马上进入下一个冲突、选择或爽点。

## 断档风险

- 指望读者耐心找重点。
- 心理描写变自言自语。
- 环境描写和剧情选择无关。
- 重点藏太深。

## 训练任务

拿 1000 字描写逐句标注“必须理解剧情”“强化情绪”“可删”，删完后把重点提前。
