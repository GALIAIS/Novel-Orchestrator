# 上架数据-首订榜单观察

依据案例：`batch-093` 案例 464。

## 读者承诺

首订榜单用来验证题材和卖点，不是只围观数字。要同时看分类、收藏、首订、收订比、同期排名和数据来源。

## 核心循环

```text
找榜单 -> 记录收藏首订 -> 算收订比 -> 对照分类 -> 看同期样本 -> 反推卖点反馈
```

## 开篇必备

- 记录数据来源。
- 不只看首订。
- 看收订比。
- 对照同日上架作品。
- 结合题材卖点判断。

## 期待来源

- 哪些题材转化强。
- 公共认知角色是否有效。
- 同人式改命是否有市场反馈。
- 低收藏高转化样本能否学习。

## 断档风险

- 不同来源数据有差异。
- 只看破线不看分类。
- 用单本数据下绝对结论。
- 忽略同期竞争。

## 训练任务

找 10 本同日上架书，记录分类、收藏、首订、收订比和卖点，比较高转化与低转化样本。
