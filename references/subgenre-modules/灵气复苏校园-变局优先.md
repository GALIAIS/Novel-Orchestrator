# 灵气复苏校园-变局优先

依据案例：`batch-026` 案例 126。

## 读者承诺

校园提供代入，灵气复苏提供变化；第一章必须让变局进入校园事件。

## 核心循环

```text
校园日常 -> 变局打破 -> 主角发现异常 -> 金手指必要使用 -> 公开解决 -> 同学关系发酵
```

## 开篇必备

- 灵气复苏带来的第一变化。
- 主角在主线上的低位。
- 主角高潜力或可见底牌。
- 普通办法解决不了的问题。

## 期待来源

- 熟悉校园被超凡变局打破。
- 读者知道主角有底牌。
- 同学见证装逼打脸。
- 校园关系能持续产生活动和喜剧。

## 断档风险

- 前几章只写普通校园生活。
- 校园代入抢走灵气复苏卖点。
- 金手指解决普通问题，显得没必要。
- 下行很重，打脸很轻。

## 训练任务

写一个灵气复苏校园开篇：第一场冲突必须由超凡异常引起，主角不用金手指就无法识破或解决。
