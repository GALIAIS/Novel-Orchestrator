# 末日资料-三大神书素材库

依据案例：batch-301 案例 1504

## 读者承诺

末日、穿越、生存、基建题材需要把实用资料转成具体生产、医疗、防卫任务。

## 核心循环

```text
生存问题 -> 查资料库 -> 抽具体技术 -> 设置执行阻碍 -> 解决危机 -> 基地升级
```

## 开篇必备

- 建立资料来源卡。
- 优先抽粮食、医疗、防卫、建筑、组织管理。
- 每项技术都对应一个剧情问题。
- 解决后给出生活质量或势力升级。

## 期待来源

- 读者看到废墟里恢复秩序。
- 技术方案具体可信。
- 基地一步步升级。

## 断档风险

- 大段抄目录。
- 技术没有执行阻碍。
- 主角像百科播音，不像解决问题。

## 训练任务

写三项末日建设任务：粮食、医疗、防卫。每项写问题、资料技术、阻碍、收益。
