# 女频咸鱼人设-被宠爱优待承诺

依据案例：`batch-100` 案例 500。

## 读者承诺

咸鱼女主不是“什么都不做”，而是承诺一种低欲望人物被世界、强者或亲密关系看见并优待的体验。她不主动争抢，但外部压力会把她卷入核心关系，读者看的就是她在不卷中被宠爱、被重视、被特殊对待。

## 核心循环

```text
低欲望咸鱼 -> 外部压力卷入 -> 被强者/关系看见 -> 获得特殊优待 -> 糖刀拉扯 -> 再次被卷入
```

## 开篇必备

- 说明主角为什么想咸鱼。
- 给她一个无法完全拒绝的高压关系或事件。
- 让她的低主动性和外部压力形成反差。
- 第一章要有被选中、被误会、被推入核心关系的动作。
- 优待必须具体，不能只写“他很宠”。

## 期待来源

- 女频读者对被宠爱、被重视、被优待的情感需求。
- 咸鱼低欲望带来的松弛感。
- 高压世界对主角形成反衬。
- 糖刀比例制造情绪拉扯。
- 短篇验证后扩成长篇的人设稳定性。

## 断档风险

- 主角一直躺平，外部没有压力。
- 只有宠爱，没有事件和代价。
- 只写事业爽点，忽略情感反馈。
- 糖刀失衡，甜文读者或虐文读者都接不住。

## 训练任务

写一个 600 字咸鱼女主开篇：主角必须不想争，但必须被卷入一个无法拒绝的核心关系。文后列出 3 个优待证据和 1 个刀点，确保每个甜点前都有压力对照。
