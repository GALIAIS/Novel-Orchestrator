# 写改分离-日更拉片改文闭环

依据案例：`batch-225` 案例 1122。

## 读者承诺

先保证初稿产量，再用畅销电影拉片校准对话、剧情和爽点，最后回到改文。

## 核心循环

```text
先写初稿 -> 拉片输入 -> 找可用技巧 -> 对照改文 -> 记录提升
```

## 开篇必备

- 初稿阶段不反复打断。
- 拉片对象必须是畅销、成熟样本。
- 每次只找对话、剧情、爽点中的具体可用点。
- 改文只改一个明确卡点。

## 期待来源

作者不用空想怎么改，而是拿成熟样本做参照，把卡点转成可模仿动作。

## 断档风险

- 边写边改导致产量崩。
- 只看电影不拆具体技法。
- 样本和自己项目不匹配。
- 改文没有前后对照。

## 训练任务

连续 3 天执行写改分离：每天先写 3000-8000 字不改；再拆一部畅销电影的 3 个对话、3 个情节点、3 个爽点；最后只改一个卡点并记录差异。
