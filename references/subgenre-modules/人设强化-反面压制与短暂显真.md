# 人设强化-反面压制与短暂显真

依据案例：`batch-397` 案例 1981

## 读者承诺

不用旁白反复宣布特质，而是让角色因反向表象付出代价，再由关键危机短暂显真，以结果推翻旧评价并留下不可逆的知情关系。

## 核心循环

```text
目标特质 -> 有代价的反向表象 -> 矛盾证据 -> 危机迫使显真 -> 见证者行动更新 -> 重新遮蔽但状态不可逆
```

## 开篇必备

- 目标特质有可观察后果。
- 遮蔽原因属于世界规则或人物选择。
- 维持伪装会损失关系、荣誉或资源。
- 读者先看到至少一条矛盾证据。
- 第一次显真解决具体问题。
- 显真后留下知情人或追查风险。

## 期待来源

读者知道表面评价有误，却要等待角色何时被迫证明自己，以及真相出现后哪些关键人物会真正改变选择。

## 断档风险

- 反向表象只为强拖揭面。
- 所有人突然无条件追捧。
- 真相只靠外貌或旁白宣布。
- 重戴面具后社会状态完全重置。
- 每次揭露重复同一套震惊反应。
- 道德特质只靠一次显真证明。

## 训练任务

选一个可遮蔽特质，写 1500 字“误判—矛盾证据—危机显真—关系改变”；限制见证者为一个关键群体，并让角色重新遮蔽后至少留下一个不可逆后果。
