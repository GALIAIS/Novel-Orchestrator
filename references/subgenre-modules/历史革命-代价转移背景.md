# 历史革命-代价转移背景

依据案例：`batch-025` 案例 123。

## 读者承诺

让革命、变法、改朝换代不是口号，而是基层承担代价后被逼出的结构性行动。

## 核心循环

```text
资源攫取 -> 代价下压 -> 基层破产 -> 主角看见 -> 组织行动 -> 既得利益者反扑
```

## 开篇必备

- 谁获利。
- 谁承担代价。
- 基层具体痛苦。
- 主角最先遇到的压迫事件。

## 期待来源

- 旧秩序的不可维持。
- 主角行动有现实根基。
- 革命目标指向制度重组。
- 既得利益者反扑形成长线冲突。

## 断档风险

- 只写朝堂，不写基层。
- 只讲理论，不写具体生活压力。
- 主角革命动机像现代口号穿越。
- 背景资料堆砌，不转化成事件。

## 训练任务

写一个王朝末期开篇：一户人家被税粮、地租、兵役三重压迫逼到卖地，主角从这件事看见制度问题。
