# 投稿转站-信息不足决策

依据案例：`batch-136` 案例 677。

## 读者承诺

转站选择不能只凭“玄幻脑洞文”这类大标签。必须先补正文、章纲、人设和卖点，才能判断哪个平台更匹配。

## 核心循环

```text
投稿未过 -> 想转站 -> 补齐作品信息 -> 判断平台读者 -> 决定投放
```

## 开篇必备

- 原平台反馈。
- 核心卖点。
- 主角人设。
- 前三章章纲。
- 正文片段。
- 备选平台读者匹配。

## 期待来源

- 作品信息足够具体。
- 平台建议可被验证。
- 转站不是情绪逃离。
- 类型改造有方向。

## 断档风险

- 只报大类题材。
- 不提供正文和章纲。
- 把起点不过等同于平台不适合。
- 纯玄幻误投二次元平台。

## 训练任务

补全一份转站求助材料：100 字卖点、主角人设、前三章章纲、300 字正文片段，再分别判断 17K、刺猬猫、番茄是否匹配。
