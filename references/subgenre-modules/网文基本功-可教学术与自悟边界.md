# 网文基本功-可教学术与自悟边界

依据案例：`batch-232` 案例 1160。

## 读者承诺

网文基本功没有统一定义，应拆成可训练的剧情、画面、文字、故事设计和表达能力。

## 核心循环

```text
概念模糊 -> 拆成能力项 -> 单项训练 -> 复盘短板 -> 继续练
```

## 开篇必备

- 不追求唯一权威定义。
- 把基本功拆成可练项目。
- 先练一个能力，不同时全抓。
- 区分可教学的术和需要自悟的部分。

## 期待来源

新人不再被“基本功”这个词吓住，而能每天练一个具体能力。

## 断档风险

- 基本功停留在空泛口号。
- 同时练太多，看不出进步。
- 把教程当成全部答案。
- 忽视阅读积累和市场嗅觉。

## 训练任务

做基本功周练表：打斗画面、对话画面、场景画面、人物刻画、情感渲染、剧情转折、文字压缩。每天只检查一个能力。
