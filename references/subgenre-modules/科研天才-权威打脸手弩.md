# 科研天才-权威打脸手弩

依据案例：`batch-164` 案例 819。

## 读者承诺

用权威压迫、同伴质疑、敌人轻视和硬证据验证，制造科研天才打脸爽感。

## 核心循环

```text
发现权威错误 -> 三重压力 -> 反复验证 -> 公开证据 -> 权威认错 -> 声望升级
```

## 开篇必备

- 权威来源要够强。
- 主角发现异常要有专业依据。
- 同伴和敌人分别施压。
- 打脸必须靠数据、实验、订正或回信。

## 期待来源

读者期待主角不是嘴上藐视权威，而是用可验证真理让权威低头。

## 断档风险

- 主角只靠宣言赢。
- 权威太弱，压迫不足。
- 验证过程缺失。
- 打脸后没有更高层伏笔。

## 训练任务

写一个科研打脸桥段，前三段压情绪，第四段用公开证据释放爽点。
