# 人物描写-生活不公平伤害与真实积累

依据案例：`batch-206` 案例 1029。

## 读者承诺

最有力的人物描写常来自真实生活积累；生活本身的不公平可以成为强共情伤害。

## 核心循环

```text
真实观察 -> 具体处境 -> 不公平伤害 -> 克制情绪落点 -> 读者共情
```

## 开篇必备

- 写具体职业或生活处境。
- 写清不公平从哪里来。
- 不用硬造反派。
- 用一句克制判断收束情绪。

## 期待来源

读者从真实生活压力中识别人物处境，并自然共情。

## 断档风险

- 只写标签，不写具体人。
- 为虐而虐，缺少生活质感。
- 过度煽情削弱真实感。
- 主角变强后仍套底层伤害逻辑。

## 训练任务

写 3 个生活观察人物速写：职业/处境、不公平伤害、克制情绪落点，各不超过 300 字。
