# 女频投稿-平台形态分流

依据案例：`batch-091` 案例 452。

## 读者承诺

女频投稿不能只问是不是晋江。先判断作品是无线、新媒体、脑洞、传统言情还是其他形态，再选晋江、番茄、七猫或其他站点。

## 核心循环

```text
女频构思 -> 判断形态 -> 对照平台生态 -> 扫样本 -> 首投 -> 备用平台
```

## 开篇必备

- 明确作品形态。
- 不把女频等同于晋江。
- 区分新人机会和中层作者机会。
- 扫同题材平台样本。
- 准备备用投稿路径。

## 期待来源

- 平台读者是否吃这个题材。
- 新人能否拿到反馈。
- 是否需要梗、车、脑洞或强情绪。
- 哪个平台更适合第一本试错。

## 断档风险

- 只按作者性别选平台。
- 只知道晋江、番茄、七猫。
- 不区分无线和新媒体。
- 不扫榜就投稿。

## 训练任务

拿一个女频构思，判断形态后找 3 个平台样本。写出首投平台、备用平台和不投某平台的理由。
