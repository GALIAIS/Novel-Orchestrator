# 人物情节-小传大纲整合

依据案例：`batch-129` 案例 642。

## 读者承诺

人物小传不是独立资料卡，必须和情节三幕结构互相影响。主角的核心性格、重大事件、关键时刻和高潮选择，要能解释他为什么这样行动。

## 核心循环

```text
人物属性 -> 重大事件 -> 初始状态 -> 诱发事件 -> 关键选择 -> 高潮行为
```

## 开篇必备

- 主角核心性格清楚。
- 重大事件影响当前选择。
- 初始状态可被诱发事件打破。
- 关键时刻要出现在高潮前。
- 人物变化必须改变高潮行为。

## 期待来源

- 角色行动有心理依据。
- 情节推进能逼人物选择。
- 高潮不是机械打赢，而是人物变化兑现。
- 大纲和小传互相咬合。

## 断档风险

- 小传只写标签。
- 情节只按事件推，不改变人物。
- 关键选择和高潮无关。
- 序幕只是炫设定。

## 训练任务

给当前主角做一页小传：核心性格、重大事件、初始状态、诱发事件、关键时刻、高潮选择。再写 200 字说明人物变化如何改变最终高潮。
