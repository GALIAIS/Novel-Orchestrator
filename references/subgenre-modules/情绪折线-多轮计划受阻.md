# 情绪折线-多轮计划受阻

依据案例：batch-276 案例 1378

## 读者承诺

用“计划可行-计划受阻-新计划-再受阻-底牌兑现”的节奏制造情绪折线，让读者预期不断上升、下压、再冲顶。

## 核心循环

```text
目标 -> 计划一 -> 意外失败 -> 计划二 -> 强阻碍 -> 底牌 -> 高潮
```

## 开篇必备

- 主角目标清楚。
- 每轮计划都要看似可行。
- 下行来自真实阻碍，不靠降智。

## 期待来源

读者期待主角不断换方案，直到最终底牌解决问题。每次失败都让高潮更有重量。

## 断档风险

- 计划失败原因牵强。
- 多轮计划不围绕同一冲突。
- 底牌没有铺垫，高潮显得硬。

## 训练任务

写一个三轮计划冲突段，标出每一轮的上行情绪和下行情绪来源。
