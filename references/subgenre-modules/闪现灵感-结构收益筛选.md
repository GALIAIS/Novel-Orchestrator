# 闪现灵感-结构收益筛选

依据案例：`batch-156` 案例 780。

## 读者承诺

突然闪现的灵感不能凭喜欢就插入正文，要看它是否增强主线、伏笔、人物或高潮，并衡量改动成本。

## 核心循环

```text
灵感出现 -> 判断结构收益 -> 衡量改动成本 -> 选择插入/后置/入库/放弃
```

## 开篇必备

- 灵感内容。
- 影响章节。
- 影响结构层级。
- 对主角目标的帮助。
- 对冲突和高潮的帮助。
- 改动成本。

## 期待来源

期待来自灵感真正服务后续兑现，而不是让当前正文失控。

## 断档风险

- 一有灵感就改大纲。
- 只看单点有趣，不看结构收益。
- 伏笔牵动过大，拖慢写作。
- 没有灵感库，导致反复打断正文。

## 训练任务

建立灵感决策表。每个新灵感先标注结构收益和改动成本，只在同时增强主线或高潮且成本可控时插入。
