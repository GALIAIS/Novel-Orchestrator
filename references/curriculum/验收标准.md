# 验收标准

## 通过线

| 阶段 | 最低通过标准 |
|---|---|
| L0 | 能说清题材承诺和目标读者 |
| L1 | 能拆出第一单元的六要素和短期期待 |
| L2 | 能写出起承转合扣完整事件 |
| L3 | 能把目标转化为具体期待和爽点兑现 |
| L4 | 能保持两长一短，高潮后不断档 |
| L5 | 能进入题材专项模块稳定产出事件 |
| L6 | 能用证据复盘并形成下一轮修改 |

## 不通过信号

- 只能复述剧情，不能说结构功能。
- 只说“爽”“有趣”，不能说期待如何形成。
- 只写设定，不写事件。
- 主角目标存在，但读者收益不明确。
- 高潮结束后才开始想下一个目标。
- 拆书表很多，自己的章节没有变化。

## 验收产物

每一级必须留下一个文件：

- L0：`读者承诺卡`
- L1：`开篇六要素表`
- L2：`单元故事卡`
- L3：`期待链路表`
- L4：`期待库存表`
- L5：`题材专项拆解表`
- L6：`诊断报告`
