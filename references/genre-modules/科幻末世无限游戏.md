# 科幻末世无限游戏专项模块

## 子流派

- 科幻星际：技术、文明、战争、伦理、探索。
- 末世生存：资源、秩序崩坏、基地、人性压力。
- 无限诸天：副本任务、多世界规则、团队生存。
- 游戏电竞：机制理解、竞技成长、版本信息差。
- 第四天灾：玩家群体、系统任务、势力经营、荒诞反馈。

## 读者承诺

主角在高压规则、资源稀缺或复杂机制中识别规律，利用能力或信息差建立优势，持续打开新地图、新副本、新科技或新玩法。

## 核心循环

```text
规则/资源压力 -> 主角识别机制 -> 制定策略 -> 通过任务/战斗/经营破局 -> 解锁更大系统
```

## 开篇必备

- 核心规则或危机的最小展示。
- 主角与普通人的差异。
- 第一个任务、副本、版本或生存目标。
- 失败代价。
- 通关或破局后的奖励和新入口。

## 期待来源

- 规则漏洞被主角识别。
- 副本奖励推动长线成长。
- 团队或玩家反应放大主角操作。
- 新科技、新地图、新敌人提前露出。
- 版本变化制造不断档。

## 断档风险

- 规则复杂但不影响选择。
- 副本结束后主线归零。
- 团队角色没有功能。
- 奖励堆叠但没有玩法变化。
