# 都市现实职业专项模块

## 子流派

- 都市神豪：资源跃迁、阶层差、消费场景、关系重估。
- 都市职业：医生、律师、警察、主播、厨师、鉴宝、文娱等专业能力。
- 都市日常：生活质感、关系推进、轻冲突、稳定陪伴。
- 文娱娱乐圈：作品发布、舆论反馈、行业资源、粉丝增长。
- 年代商业：时代信息差、资源倒卖、创业、家庭关系。

## 读者承诺

主角在现代社会秩序里获得能力、资源、专业地位或信息差，改变现实关系和阶层处境，让读者看到具体生活变好、旁人改观、旧压迫被反转。

## 核心循环

```text
现实处境/关系压力 -> 专业能力或资源入场 -> 解决具体问题 -> 关系和地位反馈 -> 更大资源或行业场景开启
```

## 开篇必备

- 主角所在行业或生活处境。
- 一个具体可感的现实困境。
- 能力/金手指如何解决现实问题。
- 见证人：老板、客户、亲友、同行、前任、网友。
- 目标达成后的可视化收益。

## 期待来源

- 被看不起后人前显圣。
- 专业能力解决别人解决不了的问题。
- 资源跃迁带来生活画面。
- 舆论、榜单、客户、粉丝放大反馈。
- 旧关系重新洗牌。

## 断档风险

- 钱或能力只是数字，没有生活和关系反馈。
- 专业细节太硬，没转化成事件。
- 神豪只消费不解决问题。
- 文娱只发作品不写评价、传播、资源变化。
