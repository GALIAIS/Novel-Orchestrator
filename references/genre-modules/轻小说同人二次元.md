# 轻小说同人二次元专项模块

## 子流派

- 同人：原作角色、原作事件、信息差、改写名场面。
- 综漫：多世界串联、角色碰撞、设定兼容。
- 校园青春：社团、竞赛、友情、暗恋、成长。
- 轻幻想日常：吐槽、角色魅力、轻设定、低压事件。
- 宅文二次元：梗文化、角色互动、反套路。

## 读者承诺

读者期待熟悉元素被重新组合，主角用信息差、身份差或性格差改变原有关系和事件，同时获得角色互动、梗反馈和轻快节奏。

## 核心循环

```text
熟悉场景/角色出现 -> 主角带着差异入场 -> 名场面偏转 -> 角色关系变化 -> 新原作节点或跨界碰撞
```

## 开篇必备

- 明确是哪类同人/轻小说语境。
- 主角与原作或轻设定的差异。
- 第一个可识别名场面或角色互动。
- 主角改变事件的能力。
- 改变后带来的关系期待。

## 期待来源

- 原作角色对主角产生新反应。
- 名场面被反转或补完。
- 主角提前知道未来。
- 多角色关系网提供持续互动。
- 梗和吐槽降低阅读阻力。

## 断档风险

- 只复述原作，没有偏转。
- 梗太密，事件不推进。
- 角色只贴标签，没有行为差异。
- 综漫世界规则不兼容。
