# 悬疑灵异规则怪谈专项模块

## 子流派

- 规则怪谈：规则文本、漏洞、代价、认知陷阱。
- 灵异民俗：禁忌、仪式、地方传说、因果报应。
- 悬疑推理：线索、公平性、嫌疑、反转。
- 惊悚恐怖：空间压迫、未知威胁、心理崩坏。
- 刑侦破案：案件、证据链、职业协作、真相分层。

## 读者承诺

读者期待主角在未知、危险或规则压迫中保持认知优势，识别真相、破解规则、规避代价，并逐步触碰更大的世界秘密。

## 核心循环

```text
异常规则/案件出现 -> 主角发现矛盾点 -> 试探与取证 -> 危险升级 -> 破解/反杀/揭晓 -> 更深层规则露出
```

## 开篇必备

- 异常规则或危险的明确呈现。
- 主角为什么能活得更久或看得更深。
- 第一条可验证线索。
- 违反规则的代价。
- 真相背后的更大疑问。

## 期待来源

- 读者比角色先看到危险。
- 主角识别规则漏洞。
- 错误理解规则的人付出代价。
- 旧线索在转折处回收。
- 副本外系统或幕后秩序逐步显影。

## 断档风险

- 规则只吓人，不参与解法。
- 真相靠作者宣布，不靠证据回收。
- 主角没有主动试探。
- 每个副本结束后缺少长线谜团。
