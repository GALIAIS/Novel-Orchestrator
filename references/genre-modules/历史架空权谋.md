# 历史架空权谋专项模块

## 子流派

- 架空争霸：势力扩张、制度重组、战争、外交。
- 历史穿越：先知信息差、改命、技术和制度优势。
- 科举官场：阶层上升、规则博弈、名声和权力。
- 种田基建：资源经营、人口、生产、组织效率。
- 宫斗宅斗权谋：规则压迫、联盟、背叛、身份和名分。

## 读者承诺

主角利用信息、制度、资源、人心或谋略，在旧秩序中取得位置，重组权力关系，并让读者看到小人物撬动大结构的过程。

## 核心循环

```text
旧秩序压迫 -> 主角发现规则缝隙 -> 谋划/经营/结盟 -> 阶段性胜利 -> 权力反噬或更高规则出现
```

## 开篇必备

- 时代或架空秩序的最小规则。
- 主角身份和受限处境。
- 可利用的信息差、制度差或资源差。
- 第一场规则内博弈。
- 阶层改变或权力反馈。

## 期待来源

- 小人物用规则反杀大人物。
- 现代信息差改变历史结果。
- 生产力提升带来势力扩张。
- 名声、官位、爵位、资源逐步兑现。
- 朝堂、家族、敌国提供长线压力。

## 断档风险

- 背景讲太多，事件启动太慢。
- 权谋只有作者解释，没有可验证选择。
- 基建只造物，没有人和制度反馈。
- 穿越信息差一次用尽，后续没有新结构。
