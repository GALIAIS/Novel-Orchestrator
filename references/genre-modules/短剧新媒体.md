# 短剧新媒体专项模块

## 子流派

- 男频短剧：高手下山、神豪、战神、复仇、逆袭。
- 女频短剧：追妻、萌宝、豪门、真假千金、复仇成长。
- 漫剧/听书：强钩子、强反应、低门槛信息。
- 知乎/短篇爽文：标题承诺、快速反转、密集情绪。
- 番茄新媒体：高频钩子、清楚设定、强类型反馈。

## 读者承诺

用最短路径建立冲突、身份差或信息差，快速压制、快速反转、快速反馈，让读者持续获得明确的情绪兑现。

## 核心循环

```text
强处境/强误判 -> 快速压制 -> 身份/能力/证据反转 -> 多方震惊后悔 -> 新误判或更大危机
```

## 开篇必备

- 第一屏就能看懂主角处境。
- 冲突对象和压制方式明确。
- 身份差、资源差或证据差尽快露出。
- 每一场都有情绪目标。
- 台词和行为直接服务反转。

## 期待来源

- 读者知道主角真实身份，对方不知道。
- 压制越具体，反转越有力。
- 旁观者、亲友、反派密集反应。
- 每集或每章末尾留新误判、新证据、新危机。

## 断档风险

- 铺垫太慢。
- 情绪目标不清。
- 台词解释过多，行动反转过少。
- 反转后没有立刻接新冲突。
