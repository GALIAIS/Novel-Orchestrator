## 使用顺序

1. 先用总控判断是否真是题材问题。
2. 进入对应题材模块，确认读者承诺和核心循环。
3. 打开对应案例索引，选 3-5 个同流派样本。
4. 回到 `modules/module-拆书取证.md` 做定向拆解。
