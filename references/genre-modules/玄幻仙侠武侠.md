# 玄幻仙侠武侠专项模块

## 子流派

- 玄幻升级：血脉、宗门、异世、境界、资源竞争。
- 仙侠修真：练气筑基金丹、长生、飞升、因果、道统。
- 武道高武：气血、武技、妖魔、城市/宗门秩序、强者压迫。
- 凡人流：弱起点、资源稀缺、谨慎选择、阶段性突破。
- 洪荒封神：神话秩序、阵营、因果、先知信息差。
- 神诡修行：诡异代价、禁忌规则、能力污染、世界秘密。

## 读者承诺

主角从低位进入强者秩序，通过能力、资源、信息差或意志选择不断提升，获得地位、力量、敬畏、自由和更高层世界入口。

## 核心循环

```text
低位压迫/资源缺口 -> 获得机会或能力 -> 付出代价修炼/争夺 -> 突破或破局 -> 见证体系改变态度 -> 更高层压力出现
```

## 开篇必备

- 主角初始身份和低位压力。
- 修炼或力量体系的最小可懂单位。
- 金手指或核心优势如何介入升级。
- 第一个敌人、困境或资源缺口。
- 升级后的具体好处：地位、资源、安全、复仇、自由。

## 期待来源

- 突破后能进入更高阶层。
- 被轻视者重新评价主角。
- 宗门、家族、城池、榜单提供见证。
- 高阶强者或世界秘密提前露出。
- 旧资源不足逼出新地图。

## 断档风险

- 只升级数值，没有地位和关系变化。
- 战力膨胀太快，等级差异消失。
- 换地图后旧见证体系断裂。
- 金手指只给奖励，不制造事件。
